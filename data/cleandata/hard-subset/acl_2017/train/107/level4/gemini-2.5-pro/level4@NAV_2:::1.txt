**Summary and Contributions**

This paper investigates weakly supervised methods for cross-lingual Named Entity Recognition (NER), focusing on transferring knowledge from a resource-rich language (English) to target languages with no annotated data. The authors propose and evaluate two primary approaches: 1) an annotation projection method that uses comparable corpora and introduces a heuristic scheme to select high-quality projected labels, and 2) a representation projection method that maps target-language word embeddings into the English embedding space, allowing a single English NER model to be applied directly. Finally, two co-decoding schemes are introduced to combine the outputs of these distinct systems.

The main contributions of this work are:

1.  A strong and empirically validated system that combines annotation and representation projection, achieving state-of-the-art results on the standard CoNLL benchmark for cross-lingual NER. This provides a valuable new baseline for the community.
2.  A simple yet effective heuristic data selection mechanism for annotation projection that significantly improves performance, particularly in scenarios with low-quality word alignments.
3.  A thorough evaluation of these methods across multiple languages and two different settings: a public benchmark with few entity types and an in-house dataset with a large, fine-grained entity set.

**Strengths**

1.  The paper presents strong empirical results. The final combined system is shown to outperform two recent and strong state-of-the-art approaches on the CoNLL test data (Table 5). This is a significant result and establishes a new, robust baseline for weakly supervised cross-lingual NER.
2.  The evaluation is comprehensive and convincing. The authors test their methods on eight language pairs in total, using both the public CoNLL datasets and a proprietary in-house dataset. The use of an in-house dataset with a much larger label set (>50 entity types) demonstrates that the proposed methods are not just tailored to the standard 4-class NER problem but can generalize to more complex, fine-grained tasks.
3.  The proposed data selection scheme for annotation projection is well-motivated and its effectiveness is clearly demonstrated. As shown in Table 2, the scheme provides substantial gains (+13.9 to +20.0 F1) for languages like Japanese and Korean where word alignment quality is lower, validating the importance of filtering noise from projected annotations.
4.  The idea of combining two philosophically different projection approaches (label-space vs. representation-space) via co-decoding is sound. The results consistently show that the combined models outperform the individual components, highlighting the complementary nature of the two views.

**Weaknesses**

1.  The methodological novelty of the individual components is somewhat limited. Annotation projection and representation projection (cross-lingual embeddings) are well-established paradigms for cross-lingual transfer. The primary contribution here is not in proposing a fundamentally new technique, but rather in the careful engineering, effective combination, and extensive evaluation of these existing ideas for the cross-lingual NER task.
2.  Crucial details regarding hyperparameter tuning are missing, which impacts reproducibility. For the data selection scheme, the paper states that "we have tried different threshold parameters q and n and reported the best numbers in Table 2." It is unclear how these optimal thresholds were found—whether on a development set or the test set itself. For a truly unsupervised setting, where no labeled data exists in the target language, it is not obvious how a user would tune these critical parameters.
3.  The description of the in-house dataset is too brief. While its use strengthens the evaluation, the lack of detail regarding its domain (e.g., newswire, web text), size, and creation process makes the results presented in Table 3 difficult to fully interpret and contextualize.

**Questions to Authors**

1.  Could you please clarify the procedure used to select the quality (`q`) and entity number (`n`) thresholds for the data selection scheme presented in Table 2? Was a labeled development set used? How would you recommend tuning these parameters for a new language in a real-world scenario where no labeled data is available?
2.  The co-decoding schemes are key to achieving the paper's best results. Could you please provide a more detailed description or pseudo-code for the "exclude-O confidence-based" and "rank-based" co-decoding algorithms to improve clarity?
3.  Could you provide more details about the in-house NER dataset, such as its text domain, overall size, and any other relevant characteristics that would help in understanding the results in Table 3?
4.  Since combining two systems proved very effective, have you experimented with combining more than two models (e.g., Annotation Projection + NN1 + NN2) to see if further gains are possible?
