**Summary of the Paper**
This paper presents two weakly supervised methods for cross-lingual named entity recognition (NER) and proposes to combine them. The first method, "Annotation Projection," is a standard approach that projects NER labels from a source language (English) to a target language using aligned corpora. The main contribution here is a heuristic filtering scheme to select higher-quality projected sentences based on entity-label frequency counts, which is shown to improve performance when alignment quality is low. The second method, "Representation Projection," is a direct model transfer approach. An NER model (a feed-forward neural network) is trained on English word embeddings. For a target language, word embeddings are projected into the English embedding space via a learned linear transformation, and the English NER model is applied directly. Finally, the paper proposes two simple heuristic "co-decoding" schemes to combine the outputs of the annotation projection and representation projection systems. Experiments on in-house and CoNLL datasets show that the combined system outperforms the individual components and two previous state-of-the-art systems.

**Contributions**
1.  The primary contribution is an empirical demonstration that combining a traditional annotation projection pipeline with an embedding-based direct transfer model yields strong results for cross-lingual NER, achieving state-of-the-art performance on the standard CoNLL benchmarks.
2.  A secondary, practical contribution is the data selection heuristic for the annotation projection method. While simple, it effectively mitigates noise from poor-quality alignments, leading to significant performance gains, particularly for language pairs with less reliable alignments.

**Strengths**
1.  **Strong Empirical Results:** The paper's main strength lies in its results. The proposed combined system outperforms two established and strong baselines (Täckström et al., 2012; Tsai et al., 2016) on the CoNLL test sets. Achieving state-of-the-art on a standard benchmark is a significant result and provides a valuable new baseline for the community.
2.  **Thorough Evaluation:** The methods are evaluated across a diverse set of seven target languages and two different datasets: a fine-grained in-house dataset and the standard CoNLL dataset. This demonstrates the robustness and general applicability of the proposed combination strategy.
3.  **Well-Motivated Combination of Orthogonal Approaches:** The paper effectively leverages two distinct and complementary sources of cross-lingual signal: explicit alignments for annotation projection and distributional semantics for representation projection. The observation that one system (annotation projection) tends to be high-precision while the other is more balanced provides a solid rationale for combining them, and the results bear this out.

**Weaknesses**
1.  **Limited Methodological Novelty:** The core components of this work are adaptations of well-established methods, and the paper overstates its novelty.
    *   **Annotation Projection:** This is a classic cross-lingual transfer technique. The "heuristic data selection scheme" is a straightforward filtering step based on frequency, which can be seen as a simple confidence measure. It is a useful engineering improvement but lacks significant research contribution.
    *   **Representation Projection:** This is a direct application of the linear mapping technique for cross-lingual embeddings proposed by Mikolov et al. (2013b). This technique has already been applied to several other cross-lingual tasks like parsing and POS tagging. Its application to NER is a logical next step but not a novel approach in itself.
    *   **Co-Decoding:** The proposed combination schemes are simple heuristics (e.g., prefer non-O tags, or trust one system's outputs over the other's). These are standard ensembling strategies and lack a more principled, learned combination mechanism.
2.  **Unrealistic Parameter Tuning Protocol:** The experimental setup seems to rely on labeled data in the target language for tuning, which contradicts the paper's premise of "no human annotation in the target language." For the annotation projection method, the quality and entity count thresholds (q, n) are critical hyper-parameters. The paper reports results with the best settings without specifying how these were found. If they were tuned on the target test set (or a held-out portion), this constitutes a data leak. Similarly, the choice of the best co-decoding scheme and neural network architecture (NN1 vs. NN2) is justified by performance on the CoNLL development set. In a true zero-shot setting, there is no clear way to make these crucial model selection decisions.
3.  **Insufficient Analysis and Ablation:** The paper presents the methods and their final results but lacks deeper analysis. For instance, the representation projection method's performance varies significantly between the NN1 and NN2 architectures across different datasets, but no clear explanation is provided. Why does the smoothing layer in NN2 help on the in-house data but hurt on CoNLL data? Furthermore, the analysis of the representation projection is limited to F1 scores. A qualitative analysis showing, for example, the quality of the embedding space alignment could provide much-needed insight into the method's behavior and limitations.

**Questions to Authors**
1.  Could you please clarify the procedure for tuning the hyperparameters (q, n) for the data selection scheme? Specifically, were these tuned using labeled data from the target language? If so, how do you propose setting these parameters in a practical scenario where no such data exists?
2.  Your model selection (e.g., choosing between NN1/NN2 and confidence/rank-based co-decoding) appears to rely on a labeled development set in the target language. How would a user of your method make these decisions in a truly zero-shot setting? Is there a way to estimate performance or select the best configuration using only unlabeled target data?
3.  The performance of the representation projection approach is highly dependent on the quality of the cross-lingual embedding mapping. Could you provide some intrinsic evaluation or qualitative analysis of this mapping? For example, showing the translation accuracy on the dictionary or providing nearest-neighbor examples for projected words could strengthen the claim that this is an "effective" representation projection.
