### Summary of the Paper ###
This paper presents a comprehensive framework for weakly supervised cross-lingual named entity recognition (NER), designed to build NER systems for target languages without requiring any annotated data. The work introduces and evaluates two distinct projection-based approaches. The first is an annotation projection (AP) method, which transfers NER labels from a source language (English) to a target language using comparable corpora. Its key innovation is a heuristic data selection scheme that filters noisy projected labels to create a higher-quality training set for a target-language model. The second approach is a representation projection (RP) method, which performs direct model transfer. It learns a linear mapping from a target language's word embedding space to the source language's space, allowing a single source-language neural NER model to be applied directly to the projected target-language representations without retraining. Finally, the paper proposes two co-decoding schemes that effectively combine the outputs of the complementary AP and RP systems. The methods are evaluated extensively on both in-house, fine-grained NER datasets and standard CoNLL benchmark datasets, demonstrating state-of-the-art performance.

### Main Contributions ###
1.  **An effective engineering framework combining two orthogonal projection methods.** The primary contribution is the successful integration of a data-level transfer method (annotation projection) with a model-level transfer method (representation projection). By analyzing their complementary error profiles (e.g., high precision from AP, balanced performance from RP) and developing simple yet effective co-decoding schemes, the work provides a robust and high-performing solution to the cross-lingual NER problem.
2.  **A novel data selection heuristic to improve annotation projection.** The proposed quality scoring mechanism for filtering noisy projected data is a significant contribution. It provides a simple and intuitive way to address the critical problem of noise amplification in annotation projection, especially when using comparable corpora with potentially low-quality alignments. The empirical results show this technique yields substantial performance gains.
3.  **A strong and comprehensive empirical evaluation.** The work is validated across six target languages and two distinct types of NER tasks (fine-grained and coarse-grained). This thorough evaluation, including comparisons to strong recent baselines on standard benchmarks, provides convincing evidence for the effectiveness and generalizability of the proposed framework.

### Strengths ###
1.  **State-of-the-Art Performance:** The combined system achieves impressive results, outperforming strong prior work from Täckström et al. (2012) and Tsai et al. (2016) on the CoNLL test data. This demonstrates a clear advance in the field of weakly supervised cross-lingual NER.
2.  **Effective Data Filtering for Annotation Projection:** The proposed data selection scheme is a simple but powerful idea. It directly tackles the noise problem inherent in annotation projection. The reported F1 score improvements of +20.0 for Korean and +13.9 for Japanese (Table 2), where alignment quality is noted to be lower, strongly validate the utility of this contribution.
3.  **Well-Motivated System Combination:** The design of two distinct systems and their subsequent combination is well-reasoned. The paper provides a clear analysis of the differing characteristics of the AP and RP systems (precision/recall trade-offs) which provides a solid justification for the co-decoding strategies, particularly the rank-based scheme.
4.  **Clarity and Methodological Soundness:** The paper is well-written, and the proposed methods are described clearly. The distinction between the annotation projection and representation projection approaches is well-articulated, and the experimental setup is sound. The use of both in-house and public benchmark datasets strengthens the conclusions.

### Weaknesses ###
1.  **Limited Technical Novelty in Representation Projection:** The representation projection method is a direct application of existing cross-lingual embedding mapping techniques (e.g., Mikolov et al., 2013b) to the NER task. While shown to be effective, the contribution is more in the application and integration into the broader framework rather than a novel projection technique itself.
2.  **Reliance on Heuristics and Parameter Tuning:** Several key components of the framework are heuristic in nature. This includes the data selection thresholds (`q`, `n`), the choice of co-decoding strategy (confidence-based vs. rank-based), and the fixed size of the dictionary for learning the embedding mapping. The paper would be strengthened by a more principled discussion of how these parameters could be set in a true zero-resource scenario (e.g., for the in-house data, how were `q` and `n` chosen without a development set?) and an analysis of the system's sensitivity to these choices.
3.  **Unexplained Inconsistency in NN Model Performance:** Two neural network architectures (NN1 and NN2) are proposed for the representation projection system. However, NN1 performs better on the CoNLL data, while NN2 is superior on the in-house data. The paper does not offer an analysis or hypothesis for this discrepancy. A deeper investigation into why the smoothing layer in NN2 helps for fine-grained entity types but not for coarse-grained ones would provide valuable insight.
4.  **Absence of Qualitative Error Analysis:** The evaluation is based entirely on aggregate metrics (P, R, F1). A qualitative error analysis would be highly beneficial. For example, what kinds of errors are specific to the AP system versus the RP system? Are there systematic errors that persist even after co-decoding? Such an analysis would offer a clearer picture of the remaining challenges and guide future work.

### Questions to Authors ###
*   The data selection thresholds `(q, n)` in Table 2 appear to be tuned per language. In a practical scenario without access to target-language annotated data for tuning, how might these thresholds be set? Could the paper elaborate on the method used to select these values for the in-house experiments?
*   The rank-based co-decoding scheme is motivated by the observation that the AP system yields higher precision. Was this characteristic observed consistently across all languages and datasets? It would be useful to understand the conditions under which this assumption holds and how the system might adapt if it does not.
*   The paper reports that larger bilingual dictionaries (over 5-6K word pairs) can harm the performance of the representation projection model. This is a counter-intuitive finding. Is there a hypothesis for this effect? For example, could it be related to overfitting the linear mapping or introducing noise from less reliable word pairs?
