### Summary and Contributions

This paper proposes a neural network-based method for event detection (ED) that explicitly leverages annotated argument information from the training data. The central argument is that prior work, particularly joint models for event detection and argument extraction, fails to effectively use argument information to improve the ED subtask. The proposed solution is a supervised attention mechanism. During training, the model's attention layer is guided by a "gold" attention distribution constructed from the locations of annotated event arguments. This is incorporated into the learning objective via a joint loss function, combining the standard classification loss with a squared error loss for the attention weights. Experiments on the ACE 2005 dataset show that this approach significantly outperforms an unsupervised attention baseline and achieves state-of-the-art results compared to previous methods.

In my view, the main contributions of this work are:

1.  **A novel application of supervised attention for Event Detection:** The primary contribution is the method of using a supervised attention mechanism to explicitly guide an ED model to focus on context words that serve as event arguments. This provides a direct and effective way to incorporate argument information into the ED task, which the authors convincingly argue was underutilized in prior work.
2.  **Strong empirical results and insightful analysis:** The paper presents state-of-the-art performance on the ACE 2005 benchmark, demonstrating the practical value of the proposed method. The authors also provide a clear analysis of the shortcomings of previous joint learning methods and use an effective oracle experiment (Section 4.2) to validate their core assumption about the utility of argument information.

### Strengths

1.  **Strong Motivation and Intuitive Approach:** The paper is built on the clear and compelling hypothesis that argument information is a crucial clue for disambiguating event triggers. The example of the trigger "fired" (End-Position vs. Attack) effectively illustrates this point. The proposed solution—to explicitly supervise the model's attention mechanism to focus on arguments—is a logical and direct way to inject this information into the model.

2.  **Thorough and Convincing Experimental Evaluation:** The empirical results are a significant strength. The proposed model (ANN-S2) achieves a new state-of-the-art F1 score on the standard ACE 2005 dataset, outperforming a range of recent and competitive systems. The evaluation is well-structured, including a comparison against an important unsupervised attention baseline, which highlights the impact of the supervision signal. The inclusion of experiments augmenting the training data with FrameNet further demonstrates the robustness and potential of the approach.

3.  **Insightful Ablation and Oracle Analysis:** Section 4.2 provides a valuable analysis that strengthens the paper's claims. By comparing a baseline model to "oracle" models (ANN-Gold1/2) that use gold argument information at both training and test time, the authors effectively demonstrate the potential performance gain from using this information. This validates their research direction and provides a clear motivation for developing a mechanism to learn this focus.

4.  **Clarity and Readability:** The paper is well-written and easy to follow. The problem statement, motivation, model architecture, and experimental setup are all described clearly and logically. The structure of the paper guides the reader effectively from the initial hypothesis to the final results.

### Weaknesses

1.  **Limited Technical Novelty of the Core Mechanism:** While the application is novel and effective, the core technical component—supervised attention—is an existing technique (e.g., Mi et al., 2016, which is cited). The novelty lies in using argument locations as the supervision signal for the ED task. The underlying neural architecture (an ANN classifier with an attention-weighted context vector) is also quite standard. Therefore, the primary contribution is in the problem formulation and application rather than a new model or mechanism.

2.  **Dependency on Argument Annotations During Training:** The method's effectiveness is contingent on the availability of event argument annotations in the training corpus. This is a significant prerequisite that may limit the model's applicability to datasets where only trigger annotations are present or where argument annotation is sparse. The paper would benefit from a more explicit discussion of this limitation and the potential adaptability of the method to less richly annotated settings.

3.  **Unclear Handling of Instances Without Arguments:** The paper is not entirely clear on how the model handles training instances where a trigger candidate has no annotated arguments in its context. The S1 strategy description mentions a heuristic of forcing attention over all entities, but it is unclear if this is used to generate a gold standard for the loss function in the main supervised models (ANN-S1/S2). If these instances are simply ignored for the attention loss term, it could bias the model. Clarification on this point is needed, as is an analysis of performance on event types that frequently appear without arguments.

4.  **Potential Mismatch Between Training Signal and Test-Time Goal:** The model is trained to attend to argument locations, but at test time, its goal is to classify the trigger type. While the results suggest a strong correlation, there is a risk that the model learns to overfit to features predictive of "argument-hood" rather than learning a more generalizable representation for trigger disambiguation. An error analysis or qualitative visualization of attention weights on test examples could help confirm that the learned attention is indeed focusing on semantically relevant clues for the final classification task.

### Questions to Authors

1.  Could you please clarify how gold attention vectors are constructed for the supervised loss function (`D(θ)`) during training when a trigger candidate has no annotated arguments in its context? Is the attention loss term simply set to zero for these instances, or is a different heuristic (such as the one mentioned for S1 involving all entities) used to create a gold standard?

2.  The performance gap between the unsupervised attention model (ANN, F1=64.6) and the supervised model (ANN-S2, F1=71.7) is substantial. This suggests the unsupervised attention mechanism fails to learn a useful signal from the ACE data alone. Could you provide a qualitative analysis, perhaps by visualizing the attention distributions of both the unsupervised and supervised models on a few illustrative test set examples, to show how the supervision concretely changes what the model focuses on?

3.  Given the model's reliance on argument annotations for training, have you considered how this approach might be extended to settings with less supervision? For instance, could the model be used in a bootstrapping or semi-supervised framework where an argument extraction system first proposes candidate arguments to generate a noisy supervision signal for the attention mechanism?
