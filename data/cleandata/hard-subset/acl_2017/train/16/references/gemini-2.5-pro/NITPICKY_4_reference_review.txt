### Summary and Contributions

This paper addresses the task of Event Detection (ED), arguing that existing pipeline and joint learning methods fail to effectively leverage argument information to improve the identification and classification of event triggers. The authors propose a neural network model that incorporates a supervised attention mechanism. The core idea is to create "gold standard" attention vectors from the annotated arguments in the training data, and then train the model's attention component to replicate these gold attentions via an auxiliary loss function. This forces the model to focus on words that are arguments of the event, which the authors hypothesize provides crucial context for disambiguating trigger words. The main contributions of this work, as I see them, are:

1.  The application of a supervised attention mechanism to the ED task, using annotated arguments as the source of supervision. This provides a direct and explicit method for injecting argument information into the trigger classification process.
2.  An empirical demonstration that this supervised approach significantly outperforms an unsupervised attention baseline and achieves a new state-of-the-art F1 score on the standard ACE 2005 dataset.

### Strengths

1.  **Well-Motivated Premise:** The paper's core motivation is clear, intuitive, and compelling. The example of the trigger "fired" (disambiguated between *End-Position* and *Attack* events by the argument "former protege") is an excellent illustration of the problem. The analysis in the introduction, which points out that joint models have historically improved Argument Extraction (AE) but not ED, provides a strong rationale for developing a method that focuses explicitly on using argument information *for* ED.

2.  **Simple and Effective Method:** The proposed solution—supervised attention—is an elegant and direct way to address the stated problem. Rather than relying on complex joint inference or feature engineering, the method injects the desired inductive bias directly into the model's learning objective. The exploration of two different strategies for generating gold attentions (hard attention vs. Gaussian-smoothed) is a sensible and useful analysis.

3.  **Strong Empirical Results:** The proposed model, particularly ANN-S2, achieves a new state-of-the-art result on the widely used ACE 2005 benchmark, outperforming several strong recent systems. The ablation study in Section 4.2 provides convincing evidence for the central claim: using gold argument information (ANN-Gold1/2) drastically outperforms both the unsupervised baseline (ANN) and a baseline focused only on entities (ANN-ENT), confirming that it is specifically the *argument* information that is beneficial.

### Weaknesses

Despite the strong results, the paper is marred by a significant number of flaws in its technical description, experimental rigor, and presentation. These issues undermine the clarity and reproducibility of the work.

1.  **Ambiguity in Model Architecture and Attention Mechanism:** The description of the model in Section 3 is critically underspecified and contains inconsistencies.
    *   **Attention Calculation:** In Section 3.1.3, the calculation of the entity attention vector `αe` is opaque. The paper states `we` is calculated by "transforming w from the word space into the entity type space," but the mechanism for this transformation (e.g., a learned projection matrix) is never defined. It is unclear why this indirect method is used instead of simply using the entity type embedding of the candidate word (if available).
    *   **Attention Combination:** The paper states `α = αw+αe`. It is not specified if this is a simple element-wise sum, and if the resulting vector `α` is re-normalized (e.g., with a softmax). More importantly, this combined vector `α` is then used to weight both the contextual words (`cw = CwαT`) and contextual entities (`ce = CeαT`). This is a highly questionable design choice, as it forces the model to learn a single attention distribution over positions that is optimal for both word and entity type representations, which seems unlikely. A more standard approach would be to use `αw` and `αe` to separately weight their respective contexts.
    *   **Input to Detector:** Section 3.2.1 states the input to the event detector is a concatenation of "context (including contextual words and entities) and the current candidate trigger". This description of a large, concatenated vector of all context embeddings seems to directly contradict Figure 1 and Equations 4-5, which clearly depict the context being compressed into two fixed-size vectors, `cw` and `ce`. The actual input to the final classifier is therefore ambiguous.

2.  **Insufficient Experimental Details and Comparisons:**
    *   **Weak Baseline:** The `ANN` baseline, with unsupervised attention, scores an F1 of 64.6. This is "unexpectedly poor" as the authors note, and is substantially lower than other neural baselines like Chen et al. (2015) at 69.1. This raises concerns that the base model architecture is weak, which could artificially inflate the perceived gains from adding supervised attention. The paper would be much stronger if the baseline were more competitive.
    *   **Significance Testing:** The authors claim their method "significantly outperforms all of the compared methods" based on a t-test. A single t-test is not appropriate for multiple comparisons. More rigorous statistical tests (e.g., paired bootstrap or permutation tests) are required to substantiate this strong claim. Furthermore, the paper does not report results over multiple random seeds, nor does it provide standard deviations, making it impossible to assess the stability and true significance of the reported gains.
    *   **Table 1 Presentation:** Table 1 is poorly formatted and its logic is confusing. Comparing a joint model to the SOTA pipeline "when the corresponding joint method was proposed" is an unusual and potentially misleading way to frame the comparison. A direct comparison of the best contemporary pipeline vs. joint methods would be more transparent.

3.  **Serious Presentation and Formatting Issues:** The paper exhibits a lack of meticulousness that is not up to the standards of a top-tier publication.
    *   **Reference Section:** The bibliography is completely unformatted, appearing as raw, unprocessed text with BibTeX-like artifacts (e.g., `et al\.,?`, `\Q2010\E`). Multiple entries are duplicated and malformed. This is a major formatting failure.
    *   **Typos and Awkward Phrasing:** The text is littered with errors, such as "exiting methods" (Introduction), "join-t methods" (Table 1), "Ngyuen" (Table 3), "p 6 0.05" (Section 4.3), and "on earth benefits" (Section 4.2).
    *   **Inconsistent Notation and Footnotes:** Footnote 2 refers to a superscript '2' in the text that is not clearly explained. Footnote 4 makes an empirical claim ("Only 10% of triggers...") without any citation. The notation for context window `[w−n, w−n 2 +1, ...]` is confusing; `w_{-n/2}` would be clearer.

### Questions to Authors

1.  Could you please provide a precise mathematical definition for the transformation of the trigger word embedding `w` into the entity-space vector `we` used for attention calculation?
2.  Could you clarify how the attention vectors `αw` and `αe` are combined into `α`? Specifically, is the sum re-normalized via softmax? Furthermore, could you justify the design choice of using the same attention vector `α` to weight both word and entity type embeddings, as opposed to using `αw` and `αe` separately?
3.  The performance of your unsupervised attention baseline `ANN` (64.6 F1) is notably lower than prior neural ED models. Do you have a hypothesis for this performance gap? Could you report results over multiple runs with standard deviations for your main models in Table 3 to help assess the stability of the improvements?
