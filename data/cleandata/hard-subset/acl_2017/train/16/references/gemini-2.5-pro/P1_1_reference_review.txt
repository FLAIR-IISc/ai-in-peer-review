### **Summary and Contributions**

This paper addresses the task of Event Detection (ED), which involves identifying and classifying event trigger words. The authors make a compelling argument that existing methods, including joint event extraction models, fail to effectively leverage annotated event arguments to improve ED performance. The paper's core contribution is a novel method to explicitly incorporate this argument information via a supervised attention mechanism. In this framework, a neural network is trained not only to classify trigger candidates correctly but also to ensure its attention mechanism focuses on words that are annotated as arguments in the training data. This is achieved by adding a loss term that minimizes the difference between the model's attention distribution and a pre-defined "gold" attention distribution derived from argument labels. The authors demonstrate that this approach achieves state-of-the-art results on the standard ACE 2005 benchmark.

The main contributions of this work, as I see them, are:

1.  **A novel application of supervised attention for knowledge injection in Event Detection.** The key idea of using argument locations to create a "gold" attention vector and directly supervising the attention mechanism is an innovative way to force the model to learn the connection between specific contextual clues (arguments) and event types. This provides a more direct and explicit way of using argument information for ED than prior joint models.
2.  **Strong empirical validation and a new state-of-the-art result on a standard benchmark.** The proposed method significantly outperforms previous systems on the ACE 2005 dataset, demonstrating its effectiveness. The paper includes a thorough evaluation with well-designed ablation studies that support the authors' central hypothesis.
3.  **A clear analysis and motivation for the problem.** The introduction provides a convincing explanation for why previous joint models have struggled to improve ED, citing data imbalance and the information flow in those models. This framing effectively highlights the gap that the current work aims to fill.

### **Strengths**

1.  **Well-motivated and elegant approach.** The central premise of the paper—that arguments provide crucial context for disambiguating triggers (e.g., "fired" as Attack vs. End-Position)—is highly intuitive and well-supported by the example provided. The proposed solution directly and elegantly addresses this by supervising the part of the model responsible for weighing contextual importance (the attention mechanism). This makes the model's "reasoning" more transparent and aligned with human intuition.

2.  **Thorough and convincing experimental setup.** The authors conduct a series of well-designed experiments. Section 4.2 ("Correctness of Our Assumption") serves as an excellent oracle experiment, showing that access to gold argument information dramatically improves performance and confirming that the benefit comes from arguments specifically, not just entities. The main results in Table 3 show a substantial improvement over a range of strong, recent baselines. The further experiments with FrameNet data augmentation add another layer of robustness to their claims.

3.  **Significant performance improvement.** The proposed method (ANN-S2) achieves an F1 score of 71.7, which is a notable improvement over the previous best-reported result of 71.3 (and 70.7 from a comparable model). The increase in precision is particularly impressive (78.0 vs. ~75-77 in other top systems), suggesting the model is indeed making more confident and accurate predictions, likely due to the supervised attention.

4.  **The paper is clearly written and easy to follow.** The motivation is laid out very well in the introduction, the model architecture is clearly explained with a helpful diagram (Figure 1), and the experimental sections are well-structured. This clarity makes the paper's contributions easy to understand and appreciate.

### **Weaknesses**

1.  **The chicken-and-egg problem at test time is not analyzed.** The model is trained using gold argument annotations, but at test time, it must rely on the *learned* attention to hopefully identify argument-like words on its own. The paper provides no analysis of whether the attention mechanism actually succeeds in doing this. Does the trained model learn to attend to semantically plausible arguments in unseen sentences? A qualitative analysis showing attention maps on test examples (especially for ambiguous triggers) would be crucial to fully validate that the mechanism is working as intended, rather than just acting as a complex form of regularization that happens to work well.

2.  **The performance of the unsupervised baseline (`ANN`) seems unusually low.** The unsupervised attention model (`ANN`) scores only 64.6 F1, which is significantly lower than many of the prior state-of-the-art systems it is compared against. The authors attribute this to data sparsity, but this weak performance makes the gains from supervised attention appear larger. It is possible that the specific attention formulation used is simply not effective without supervision. A comparison against a stronger, more established unsupervised attention baseline would make the claims about the necessity of supervision more robust.

3.  **The concept of "argument information" is limited to location.** The supervision signal is derived solely from the position of argument words. However, arguments also have roles and entity types, which contain rich semantic information. The model could potentially be improved by incorporating this information into the supervision. For example, for a `Die` event, attending to a `Person` argument might be more important than attending to a `Place` argument. The current approach treats all arguments equally, which seems like a missed opportunity for more fine-grained supervision.

4.  **Minor lack of clarity in the model description.** In Section 3.1.3, the calculation of the entity attention vector αe uses a term `we`, which is described as "transforming w from the word space into the entity type space". This description is vague. It is unclear if this involves a new learned projection matrix or some other operation. Clarifying the exact mathematical formulation is necessary for reproducibility.

### **Questions to Authors**

1.  Could you provide a qualitative analysis of the learned attention weights on test set examples? For instance, on a sentence with an ambiguous trigger like "fired", does the model without gold argument labels learn to place high attention on words that are semantically plausible arguments (e.g., a person being fired, a weapon)? This would significantly strengthen the claim that the model learns the intended behavior.

2.  The unsupervised `ANN` baseline's performance is substantially lower than other recent models. Do you have any insights into why this specific architecture performs poorly with unsupervised attention? Could this be related to hyperparameter tuning, or is this attention mechanism inherently reliant on direct supervision to be effective on this task?

3.  Regarding the entity attention calculation (Eq. 3), could you please clarify precisely how `we` is derived from the word `w`? Is there a separate learned linear projection `W_e` such that `we = f(wW_e)`?
