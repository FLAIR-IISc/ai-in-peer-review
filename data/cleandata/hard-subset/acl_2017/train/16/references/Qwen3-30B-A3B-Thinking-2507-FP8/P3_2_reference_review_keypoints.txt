- Summary: Proposes supervised attention using annotated arguments to guide event detection (ED), offering two "gold attention" strategies (S1: arguments only; S2: arguments + context) and showing improvements on ACE 2005.

- Contribution 1: Highlights that argument information is underutilized for improving ED in existing joint ED/AE methods.

- Contribution 2: Introduces a simple supervised attention mechanism that explicitly leverages argument annotations for ED.

- Contribution 3: Provides comprehensive experiments showing statistically significant F1 improvements (71.7 on ACE 2005) over prior state-of-the-art.

- Strength 1: Identifies and analyzes a genuine limitation in joint models (they mainly boost AE, not ED), with plausible causes like data imbalance.

- Strength 2: Strong experimental validation, including ablations (ANN-Gold1/2 outperform baselines) and statistical significance.

- Strength 3: Clear, effective methodological design; S1 yields higher precision while S2 yields better recall.

- Strength 4: Thorough analysis explaining why prior joint approaches fail to aid ED (data imbalance and indirect use of arguments).

- Weakness 1: Lacks theoretical justification for why supervised attention outperforms unsupervised attention beyond a plausibility argument about data sparsity.

- Weakness 2: Comparison set is limited, missing recent transformer-based baselines (e.g., BERT, RoBERTa), which weakens claims of state-of-the-art.

- Weakness 3: Ambiguity in technical novelty—insufficiently clear how this approach is fundamentally different from prior joint methods that use argument information indirectly.

- Weakness 4: FrameNet augmentation is superficial; potential mismatches with ACE argument structures and noise from automatic labeling are not addressed.

- Question 1: Request for a clear technical distinction between this supervised attention approach and previous joint methods using argument information.

- Question 2: Why were recent transformer-based ED methods (post-2016) not included in comparisons?

- Question 3: Request for detailed analysis of which event types (especially ambiguous triggers) benefit most from argument-informed attention.

- Question 4: How does using the RPI Joint Information Extraction System to label entities affect the quality and noise of the FrameNet-augmented data?

- Overall assessment: Strong conceptual and empirical contribution but needs comparisons with modern baselines and stronger theoretical/technical clarification; recommendation: Weak Accept pending revisions.
