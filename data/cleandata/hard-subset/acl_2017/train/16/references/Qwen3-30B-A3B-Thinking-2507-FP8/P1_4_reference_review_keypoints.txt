- Proposes supervised attention mechanisms that explicitly use argument information to improve event detection (ED).
- Introduces two "gold attention" strategies: S1 (arguments only) and S2 (arguments plus surrounding words).
- Reports improved ED performance on ACE 2005 (e.g., ANN-S2 71.7% F1 vs. prior 69.4%), with statistical significance.
- Key insight: joint ED+AE methods underutilize arguments for ED and mainly improve argument extraction (AE), not triggers.
- Method is simple, easy to integrate into existing ED systems, and yields clear empirical gains.
- Thoroughly compares S1 and S2, showing trade-offs (S1 higher precision, S2 higher recall).
- Weakness: lacks deeper theoretical justification for why argument information helps trigger identification beyond examples.
- Weakness: comparisons omit recent neural baselines (e.g., BERT-based models), limiting relevance to current SOTA.
- Weakness: insufficient ablation study to isolate the effect of supervised attention from other model components.
- Weakness: limited discussion of practicality when argument annotations are unavailable; FrameNet augmentation provided only marginal gains.
- Weakness: novelty is somewhat overstated since supervised attention has prior use in other NLP tasks.
- Question: How does the model handle non-entity triggers if entity information is used yet only ~10% of triggers are entities?
- Question: Why does large external augmentation with FrameNet yield only a 0.2 F1 improvement for ANN-S2?
- Question: Would the supervised attention mechanism still add value when combined with modern architectures like BERT?
- Overall: Solid, well-executed contribution demonstrating value of argument information for ED; recommended accept with minor revisions.
