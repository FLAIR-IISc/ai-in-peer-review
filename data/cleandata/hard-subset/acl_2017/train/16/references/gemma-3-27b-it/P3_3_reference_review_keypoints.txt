- Paper proposes supervised attention to incorporate annotated argument information into event detection, showing improvements on ACE 2005.  
- Core idea: train attention to focus on argument words using "gold" attention weights derived from annotations.

- Empirically demonstrates that incorporating argument information improves event detection performance.  
- Introduces a supervised attention framework for integrating argument cues into ED.  
- Provides comprehensive experimental evaluation including baselines, ablations, and FrameNet augmentation.

- Clear motivation and well-articulated problem statement with an illustrative example.  
- Strong, statistically significant experimental results and useful ablation studies.  
- Well-structured, readable paper with informative figures, tables, and reproducibility details.  
- Thoughtful critique of prior joint ED/AE work and practical implementation details.  
- Data augmentation using FrameNet strengthens the evaluation.

- Methodological novelty is limited since supervised attention is not new; novelty lies in applying it to ED.  
- Heavy reliance on quality of argument annotations; paper lacks analysis of annotation noise impact.  
- Experiments limited to ACE 2005, raising questions about generalizability to newer or more diverse datasets.  
- Lacks detailed error analysis to show where the model fails or which event types remain challenging.

- How sensitive is model performance to noisy or imperfect argument annotations?  
- Have you evaluated on other datasets (e.g., SemEval, recent news or social media corpora) to test generalizability?  
- Can you visualize learned attention weights to illustrate how argument information is used?  
- What is the computational overhead of the supervised attention mechanism versus baselines?

- Expand related work to include recent attention advances and their NLP applications.  
- Consider incorporating argument role information into the attention mechanism, not just mention locations.

- Overall: solid and well-executed work that convincingly leverages arguments for ED; recommend acceptance with minor revisions addressing novelty positioning, annotation robustness, dataset generalizability, and error analysis.
