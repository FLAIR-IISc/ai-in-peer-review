- Paper proposes supervised attention on argument words to improve event detection (ED) and shows gains on ACE 2005, with further improvement using FrameNet augmentation.
- Empirically demonstrates that explicitly incorporating argument information can improve ED, challenging prior findings about joint models.
- Introduces a simple, practical supervised attention mechanism that yields measurable performance boosts.
- Provides systematic evaluation and ablation studies (attention variants, FrameNet augmentation) to analyze method components.
- Clear motivation and problem definition, with an illustrative example showing how arguments disambiguate triggers.
- Strong empirical results against strong baselines, with statistical significance testing.
- Well-structured, readable paper with clear figures illustrating the architecture.
- Thorough experiments that investigate different attention strategies and data augmentation effects.
- Core method has limited novelty: supervised attention is straightforward and not a major technical breakthrough.
- Evaluation is limited to ACE 2005, raising concerns about generalizability across domains and schemas.
- Lacks detailed error analysis to identify failure modes and event types where the method helps or hurts.
- Hyperparameter sensitivity (especially the lambda in the joint loss for S2) is not fully explored or justified.
- Requests for authors: provide detailed error analysis by event type.
- Requests for authors: include sensitivity analysis or learning curves for the lambda hyperparameter.
- Requests for authors: comment on potential of more advanced attention mechanisms (e.g., transformers/self-attention).
- Requests for authors: report which event types benefit most from FrameNet augmentation and why.
- Minor formatting/reference consistency issues and suggestion to expand related work on recent attention advances.
- Overall recommendation: accept — solid empirical contribution with clear presentation, minor weaknesses addressable in revision.
