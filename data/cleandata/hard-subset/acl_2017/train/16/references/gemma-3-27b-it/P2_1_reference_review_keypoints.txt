- Proposes supervised attention using argument information to improve event detection and shows gains on ACE 2005.
- Empirically demonstrates that explicitly leveraging argument information improves event detection performance.
- Introduces a supervised attention framework that uses gold argument annotations to guide attention.
- Provides comprehensive experiments on ACE 2005, compares to multiple baselines, and explores FrameNet augmentation.
- Reports strong, statistically significant improvements over state-of-the-art methods.
- Clearly motivates the problem with helpful examples (e.g., “fired”) and a readable presentation.
- Paper is well-structured, with a clear architecture diagram and reproducible experimental details.
- Compares different gold-attention strategies (S1 vs S2), offering insight into attention design choices.
- Core techniques (neural nets, attention, supervised learning) are not novel; novelty is in the application to arguments for ED.
- Method depends heavily on the quality of argument annotations; annotation errors could harm performance.
- ACE 2005 is a limited, small testbed; broader evaluation on larger/diverse datasets is needed.
- Lacks a detailed ablation study on the attention mechanism (e.g., removal or alternative attention functions).
- Request analysis of annotation errors and their impact, including error analysis of attention weights.
- Ask whether the approach generalizes to other datasets and what adaptation challenges exist.
- Request attention visualizations to qualitatively show how the model uses argument information.
- Ask for clarification on the computational overhead introduced by the supervised attention mechanism.
- Notes some references appear incomplete and suggests adding discussion of broader applications (e.g., argument or relation extraction).
- Overall assessment: a strong, well-motivated paper with compelling empirical results; recommend acceptance with minor revisions.
