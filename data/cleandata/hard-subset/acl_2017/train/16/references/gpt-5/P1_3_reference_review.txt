Summary and main contributions
This paper addresses ACE-style event detection (ED) and proposes to explicitly exploit annotated event arguments to guide a supervised attention mechanism over the context of each trigger candidate. The model computes attention from the candidate token to contextual words and entity types, and learns to match gold attention distributions constructed from argument annotations (two strategies: S1 focusing only on arguments; S2 smoothing attention to neighbors via a Gaussian). The attention loss is combined with the standard ED cross-entropy. Experiments on ACE 2005 show gains over several reported baselines, and augmentation with FrameNet-derived events further improves recall.

As I see it, the main contributions are:
1) A simple, supervised attention formulation for ED that uses argument supervision to bias attention toward argument tokens (and their neighborhood), yielding consistent gains over an unsupervised attention baseline and matching/exceeding prior reported results on ACE 2005.
2) A set of diagnostic experiments that isolate the contribution of argument supervision (including oracle “Gold” attention), providing evidence that arguments are useful for ED beyond generic entity information.
3) A practical data augmentation with FrameNet events adapted to the proposed setting, demonstrating additional recall gains.

Strengths
- Clear and practical idea with measurable impact: Using argument annotations as supervision for attention is straightforward yet effective. The improvements over the unsupervised attention baseline are sizable (e.g., ANN 64.6 F1 → ANN-S2 71.7 F1), and the model achieves or exceeds prior reported scores on ACE 2005.
- Diagnostic analysis supports the core claim: The comparison among ANN, ANN-ENT, and ANN-Gold1/2 helps distinguish the effect of arguments vs. generic entity presence. The large gap between ANN-ENT and ANN-Gold variants suggests argument supervision is the key signal.
- Simple, efficient architecture: The approach adds a light-weight attention supervision loss to a feed-forward ED classifier, making it easy to reproduce and potentially to plug into other architectures.
- Sensible attention strategies: The comparison between S1 (argument-only) and S2 (argument + Gaussian smoothing) is informative. The precision-recall trade-off aligns with intuition: S1 yields higher precision, S2 yields higher recall.
- External data augmentation: Adapting FrameNet events, even with noisy labels, is a useful contribution for addressing data sparsity, and the results show consistent recall improvements.

Weaknesses
- Limited novelty relative to supervised attention literature and joint EE: Supervised attention is not new conceptually, and using gold argument spans to supervise attention for ED is a fairly direct application. The paper positions itself in contrast to joint models but does not demonstrate that supervised attention offers advantages beyond what a carefully implemented joint model with loss reweighting or trigger-focused pre-pruning could achieve. The stated reasons why joint models “favor AE” are plausible but untested in controlled comparisons.
- Weak/dated baselines and inconsistent reporting: The internal ANN baseline underperforms substantially relative to prior neural baselines, and there is limited ablation against stronger architectures (e.g., CNN/RNN encoders). The Skip-CNN result is reported partially (N/A, 71.3), making direct comparison unclear. It would be stronger to implement a competitive neural ED baseline (e.g., CNN/LSTM-based with entity features) and show gains over that.
- Reliance on gold entities and unclear test-time assumptions: The model uses entity type embeddings in context; it appears these are gold ACE entity annotations at both train and test time. This limits applicability and may inflate performance compared to real-world settings with predicted entities. The paper does not report performance with automatically detected entities, nor the sensitivity of gains to entity accuracy.
- Limited analysis of attention behavior and per-type effects: Beyond the diagnostic oracle experiments, there is little qualitative/quantitative analysis of learned attention (e.g., how often attention aligns with true arguments at test time, or which event subtypes benefit most). This limits insight into when and why the method helps (e.g., on ambiguous triggers).
- Reproducibility and experimental details: The random selection of dev docs should be fixed and documented (doc IDs). Hyperparameter λ differs across strategies but tuning details are sparse. The statistical test is mentioned without specifying the test set granularity or variance estimation. Additional ablations (e.g., removing entity-based attention, using only word-based attention, varying context window) would strengthen claims.

Questions to authors
- Do the gains persist when using predicted entities at test time? Please report results with an off-the-shelf ACE entity tagger to assess robustness.
- How sensitive are results to λ, the attention window size, and Gaussian parameters (μ, σ)? Please include a sensitivity plot.
- What is the effect of removing entity-based attention (i.e., using only word-based attention) and vice versa? Is the simple sum αw + αe optimal, or does a learned combination help?
- Can you provide per-event-type P/R/F1 to illustrate where the method helps most (e.g., ambiguous triggers like “fired”)?
- How often does the learned attention peak on true arguments at test time (without access to gold arguments)? A quantitative alignment analysis would be helpful.
- Please clarify the exact dev/test document IDs and any preprocessing differences relative to compared work (tokenization, entity labels).
- Could a competitive CNN/LSTM encoder with the same attention supervision further improve results? If not, why did you choose a feed-forward architecture?

Overall assessment
This is a clear, well-motivated application of supervised attention to ED with solid empirical gains and useful diagnostics showing that argument supervision helps. The methodological novelty is moderate, and the comparative evaluation could be stronger (baselines, entity assumptions, analyses). With clarifications and extended experiments (especially with predicted entities and stronger baselines), this would be a strong contribution. As it stands, I lean toward acceptance if space permits, given the practical impact and simplicity, but I encourage the authors to address the above concerns to strengthen the case.
