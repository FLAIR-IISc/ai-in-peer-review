Summary and contributions
This paper addresses event detection (ED) in ACE 2005 by explicitly leveraging annotated argument information through supervised attention. The model computes attention over contextual words and entity types using a dot-product mechanism with the current trigger candidate, and trains the attention to match “gold” attention derived from argument annotations (two strategies: focusing only on argument tokens, or on arguments plus nearby tokens via a Gaussian spread). A simple feed-forward classifier then predicts the event type. Experiments show substantial gains over an unsupervised-attention baseline and improvements over several reported state-of-the-art systems; augmenting with weakly labeled FrameNet events yields small additional gains.

As I see it, the main contributions are:
1) A supervised attention mechanism that uses annotated arguments to guide which contextual tokens should be emphasized for ED, leading to improved performance on ACE 2005.
2) Empirical validation (including ablations with gold-attention at train/test and two supervision strategies) that argument information, beyond entity types alone, substantially benefits ED.
3) A straightforward, efficient architecture achieving competitive or state-of-the-art results on ACE 2005, with analysis of precision/recall trade-offs between attention strategies.

Strengths
- Clear and well-motivated idea: Using arguments to disambiguate triggers is intuitive and well justified (e.g., “fired” example). The supervised attention formulation operationalizes this intuition directly, in contrast to joint models where argument information often only indirectly helps ED.

- Strong empirical gains over a controlled baseline: The supervised-attention variants (ANN-S1/S2) significantly outperform the unsupervised-attention baseline (64.6 F1 → 71.7 F1), indicating that the key improvement comes from attention supervision, not just from the architecture.

- Competitive against prior art: ANN-S2 slightly surpasses or matches reported SOTA on ACE 2005 (e.g., beating Nguyen and Grishman 2016 and Liu et al. 2016a). The paper also reports a t-test indicating significance (though more details would help; see questions).

- Insightful ablations demonstrating the role of arguments: The comparisons with ANN-ENT (entities only) versus ANN-Gold1/2 (using gold arguments) convincingly show that entity types are insufficient by themselves and that argument-specific supervision matters.

- Practicality at test time: The model does not require arguments during inference—only during training to supervise attention—so the approach is usable in standard ED settings where only triggers are predicted at test time.

Weaknesses
- Limited novelty in method: Supervised attention guided by gold annotations is a known idea in other tasks (e.g., NMT). Applying it to ED with argument positions is a natural extension but conceptually straightforward. The attention mechanism itself (dot-product; summing word/entity attentions) is standard and somewhat simplistic. The work’s contribution is more empirical than methodological.

- Dependence on gold resources during training and gold entities during inference: The approach explicitly requires argument annotations for attention supervision during training. While ACE provides them, many ED datasets do not, limiting generalizability. Moreover, the model uses entity type embeddings and states it exploits ACE annotated entities; it is unclear whether gold entity mentions/types are used at test time, which would inflate realism and comparability. Prior work varies on using gold vs. predicted entities.

- Comparability and significance unclear: The paper claims following prior splits, but details are sparse and some numbers (e.g., Skip-CNN showing only one score) are incomplete. The t-test is mentioned without reporting the number of runs/seeds or variance. Given small margins vs. some SOTA (e.g., 71.7 vs. 71.3), more rigorous statistical reporting is needed.

- Missing analyses and ablations: There is no error analysis (per-type improvements, confusion patterns for ambiguous triggers), no visualization or qualitative examples of learned attention, and no ablations for key design choices: using only word attention vs. only entity attention; effect of the context window size n; effect of λ; alternative attention formulations; what happens when no arguments are in the window (the S1 fallback “average over entities” seems ad hoc).

- Data processing for FrameNet augmentation introduces uncertainty: FN events are augmented by auto-labeling entities with an external system and keeping only core FEs. Though reasonable, this introduces noise and domain mismatch; the reported gains are small (e.g., 71.7 → 71.9), making the added section less compelling without deeper analysis of when and why it helps.

Questions to authors
- Entities at test time: Do you use gold entity mentions/types or predicted ones when evaluating on the ACE test set? If predicted, which NER/typing system and with what performance? Please clarify to ensure fair comparability to prior work.

- Context window: What is the fixed context length n used? How sensitive are results to n? Do many arguments fall outside the window, and how does this affect S1/S2?

- Attention design: Why is a single shared attention vector α used to weight both word and entity contexts (α = αw + αe), rather than separate attention and subsequent fusion? Did you try ablations with only αw or only αe?

- Supervision strength and robustness: How many random seeds did you run for each model, and what are mean/std F1? Please provide details of the t-test and whether improvements over Skip-CNN and Liu et al. (2016a) remain significant across seeds.

- No-argument cases: In S1, for candidates without any annotated arguments in context, you “force all entities to average the attention.” Why is this preferable to a learned or uniform attention over all words? Did you try alternatives?

- Generalizability: Could you train the attention supervision using predicted arguments (from a separate AE model) to avoid requiring gold arguments during training? Any preliminary results?

Additional comments (minor)
- Clarity could be improved (typos, figure references, and consistent notation). Please report complete metrics for all baselines (P/R/F1) and include qualitative attention examples. Code release would greatly aid reproducibility.

Overall recommendation
A solid empirical paper with a simple, effective idea and competitive results. Novelty is moderate, but the demonstration that supervised attention via argument supervision can materially improve ED is valuable. Addressing the concerns about entity usage, statistical rigor, and additional analyses would strengthen the work.
