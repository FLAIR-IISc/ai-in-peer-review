### Summary and Contributions

This paper addresses the task of Event Detection (ED), aiming to improve performance by explicitly leveraging event argument information from the training data. The authors argue that previous methods, particularly joint models for event detection and argument extraction, fail to effectively use argument information to improve the ED subtask. To remedy this, they propose a neural network model that incorporates a supervised attention mechanism. During training, this mechanism is explicitly guided by a loss function that encourages the model's attention weights to match a "gold" attention distribution derived from the annotated event arguments. The authors experiment with two strategies for generating this gold supervision signal. Experiments on the ACE 2005 dataset show that this approach significantly improves performance over an unsupervised attention baseline and achieves a new state-of-the-art result.

In my view, the main contributions of this work are:

1.  **A novel formulation for incorporating argument information into Event Detection using supervised attention.** This is the core technical contribution. The idea of using annotated arguments to create a supervisory signal for an attention mechanism is an elegant and effective way to force the model to focus on the most salient contextual clues for disambiguating event triggers.
2.  **Strong empirical results and a new state-of-the-art on the ACE 2005 benchmark.** The proposed model significantly outperforms previous work, demonstrating the practical value of the proposed method.
3.  **A thorough analysis of the proposed method.** The paper provides valuable ablations that validate the core assumption (that argument information is key), compare different supervision strategies, and demonstrate that the approach is complementary to data augmentation techniques using external resources like FrameNet.

### Strengths

1.  **Well-Motivated and Novel Approach:** The paper clearly identifies a plausible weakness in prior work—the ineffective use of rich argument information for the ED task. The motivation, highlighted by the analysis of joint models in Table 1, is compelling. The proposed solution of using supervised attention is a direct and clever way to inject this information into the model, guiding it to learn the connections between triggers and their argument patterns. This is a significant conceptual contribution to the event detection literature.

2.  **State-of-the-Art Performance:** The proposed method achieves a new state-of-the-art F1 score of 71.7 on the standard ACE 2005 test set, which is a significant improvement over several strong, contemporary baselines. The fact that the method also achieves a notably high precision (78.0) compared to prior work suggests that the argument information provides strong, unambiguous evidence that helps reduce false positives. These strong results are compelling evidence for accepting the paper.

3.  **Thorough and Insightful Experiments:** The authors conduct a well-designed set of experiments that go beyond simply presenting a final number.
    *   The experiments in Section 4.2 ("Correctness of Our Assumption") are particularly effective. By comparing against an oracle system using gold attentions at test time (ANN-Gold1/2), they convincingly demonstrate the utility of argument information and provide a soft upper-bound for this type of approach.
    *   The comparison between the two gold attention strategies (S1 and S2) provides valuable insight, showing that a softer attention distribution that includes context around arguments (S2) leads to better recall and overall F1-score.
    *   The successful integration of FrameNet data shows that the model is robust and its benefits are complementary to other techniques for addressing data sparsity.

### Weaknesses

1.  **Potentially Weak Baseline Model:** The base model (ANN) uses a simple feed-forward network over a fixed context window. Its performance (64.6 F1) is substantially lower than other contemporary neural baselines like Skip-CNN (71.3 F1). While the improvement from adding supervised attention is impressive (+7.1 F1), it is unclear how much of this gain is compensating for a relatively weak context encoder. The paper would be more convincing if the supervised attention mechanism was applied to a stronger base architecture (e.g., a BiLSTM or CNN) to demonstrate that the gains are additive and not just an artifact of the baseline's limitations.

2.  **Dependence on Rich Annotations:** The core contribution relies on the availability of event argument annotations during training. This limits the method's applicability to datasets like ACE 2005. The approach cannot be used on corpora that only contain trigger annotations, which may be more common or easier to create. While this does not invalidate the contribution, it is a practical limitation that restricts the method's generality, and this trade-off is not discussed.

3.  **Indirect Comparison to Joint Models:** The paper motivates its work by critiquing the failure of joint ED and AE models to improve ED. However, the proposed model is not a joint model itself but rather a pipeline-style ED classifier. A more direct way to test the paper's hypothesis would be to integrate the proposed supervised attention mechanism into an existing joint model (e.g., Nguyen et al., 2016) and measure the impact on its ED performance. As it stands, the comparison is less about "fixing" joint models and more about showing that a pipeline model can better leverage argument information for the ED stage.

### Questions to Authors

1.  The base ANN model's performance is significantly below other neural models from the same period. Could you comment on whether you believe the impressive gains from supervised attention would be of a similar magnitude if applied to a stronger base architecture like a BiLSTM or the Skip-CNN model? Is it possible the attention mechanism is primarily effective because it helps a simple ANN focus on important features that a more powerful encoder might capture automatically?

2.  Your method of augmenting data with FrameNet requires automatically labeling entities and mapping frame elements to arguments, which inevitably introduces noise. The results in Table 4 show that using this noisy data improves recall and F1 but hurts precision. Could you provide some analysis on the quality of these automatic annotations and discuss how this noise might impact the training of the supervised attention mechanism?

3.  Given that your method requires argument annotations for training, do you have any thoughts on how this approach could be adapted to settings with only trigger annotations? For instance, could the "gold" attention signal be generated via distant supervision or in a semi-supervised fashion?
