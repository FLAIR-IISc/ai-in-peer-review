- Proposes a novel encoder–decoder MT variant that starts and ends at the character level while internally using morpheme/character representations.
- Releases source code and trained models for fr-en, cs-en, and en-cs to enable verification and reproduction.
- Produces reasonably strong translations after one training epoch, with further improvements in later epochs.
- Learns sensible morphological segmentations and can handle unseen or nonce words by backing off to morphemic units.
- Fails to state which WMT test/dev sets were used in the paper (README shows test=newstest2015, dev=newstest2013); this should be explicit in the manuscript.
- Software README needs improvement: include concrete training/testing command examples and add a --help flag to the program.
- Figure 2 and its caption don’t clearly reflect the described six-level architecture; the full layer stack should be shown or explained (appendix if needed).
- Comparisons omit non-character-based state-of-the-art MT systems reported on WMT/matrix.statmt.org and should acknowledge/discuss higher-performing results.
- Contains several minor English, spelling, and LaTeX issues (e.g., reversed quotation marks) that should be corrected.
- Overall assessment: a solid contribution to character-based neural MT.
