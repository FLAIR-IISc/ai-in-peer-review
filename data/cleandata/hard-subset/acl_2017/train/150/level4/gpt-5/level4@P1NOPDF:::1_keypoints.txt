- Novel encoder–decoder MT that reads and writes characters while using internal morpheme- and character-level representations.
- Authors release code and trained models for fr–en, cs–en, and en–cs, enabling validation and replication.
- System attains reasonable translation quality after one epoch and improves with more training.
- Model appears to learn sensible morphological segmentations and handles unseen/nonce words by backing off to morphemes.

- Paper fails to state which WMT development/test sets were used (found only in README: test=newstest2015, dev=newstest2013); this should be explicit in the paper.
- Software README is adequate but needs explicit end-to-end command-line examples and a working --help flag for usage.
- Paper describes a six-level architecture but Figure 2 shows fewer layers; caption should clarify and include a full-layer schematic (e.g., in an appendix).
- Results are compared to other character-based systems but not situated against state-of-the-art MT on the same datasets; the performance gap should be acknowledged and discussed.
- Minor English, spelling, and LaTeX issues (e.g., reversed quotation marks) need correction.

- Overall: a useful and well-executed contribution to character-based neural MT.
