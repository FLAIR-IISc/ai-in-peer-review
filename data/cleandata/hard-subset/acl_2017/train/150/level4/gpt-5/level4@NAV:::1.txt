- Strengths:
The authors propose a novel variant of encoder–decoder neural MT that begins and ends at the character level, while internally representing sequences in terms of morphemes and characters.
They release both the source code and the trained models for fr-en, cs-en, and en-cs, aiding verification and enabling others to reproduce and extend the work.
The system yields reasonably strong translations after just one training epoch, with continued improvements over subsequent epochs.
The model learns sensible morphological segmentations and can handle unseen words (including nonce items) by implicitly backing off to morphemic units.

- Weaknesses:
The paper does not explicitly specify which WMT test and dev sets are used, complicating comparisons with prior results (e.g., those at matrix.statmt.org). This reviewer found the details only in the code supplement’s README, which states the test set is newstest2015 and the dev set is newstest2013. This should be stated clearly in the paper.
The software README is adequate but could be improved: the training and testing sections should include concrete command examples, and the program should support a --help flag, which it currently lacks.
The text describes a six-level architecture, yet Figure 2 appears to depict fewer than six layers. The caption should clarify this; if some layers are omitted, there should be a figure (even in an appendix) showing the full stack.
The comparisons focus on other character-based neural systems but omit state-of-the-art results from other MT paradigms. WMT (and matrix.statmt.org) reports performance for additional systems on these datasets, and the top results appear substantially higher than those presented here. This should be acknowledged and, ideally, discussed.
There are several minor English issues, misspellings, and small LaTeX problems (e.g., reversed quotation marks) that should be corrected.

- General Discussion:
This paper is a solid contribution to the literature on character-based neural MT.
