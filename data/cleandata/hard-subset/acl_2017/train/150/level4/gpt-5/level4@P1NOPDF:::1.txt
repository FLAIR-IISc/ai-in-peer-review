- Strengths:
The paper presents a novel adaptation of encoder–decoder neural MT that reads and generates characters but uses internal representations at the morpheme and character levels.
The authors release both their code and trained models for fr-en, cs-en, and en-cs, which aids validation and enables others to replicate and extend the work.
The system produces reasonable translation quality after the first training epoch and continues to improve with further epochs.
It appears to learn sensible morphological segmentations and can handle previously unseen (including nonce) words by effectively backing off to morphemes.

- Weaknesses:
The paper does not explicitly state which WMT development and test sets were used. This omission makes it difficult to compare results to existing work (e.g., matrix.statmt.org). The only place this reviewer found the information was in the code README, which indicates that the test set is newstest2015 and the development set is newstest2013. This should be clearly stated in the paper.
The software README is adequate but could be improved. The training and testing sections should include explicit, end-to-end command-line examples. The software should also respond to a --help flag with usage information, which it currently does not.
The text describes a 6-level architecture, but Figure 2 appears to depict fewer than six layers. The caption should clarify exactly what is shown. If the figure is schematic and omits components, the paper should include (even in an appendix) a figure showing all layers.
The results compare to other character-based neural systems but do not situate performance relative to state-of-the-art MT systems on these datasets. WMT and matrix.statmt.org report substantially higher scores for other approaches on the same test sets. This gap should be acknowledged and, ideally, discussed.
There are a few minor issues with English phrasing, spelling, and LaTeX (e.g., reversed quotation marks). These should be corrected.

- General Discussion:
This is a useful and well-executed contribution to the literature on character-based neural MT.
