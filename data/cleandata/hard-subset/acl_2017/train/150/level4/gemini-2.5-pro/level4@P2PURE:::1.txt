### Strengths:
*   The paper introduces a novel encoder-decoder NMT architecture that operates on characters at the input and output layers, while utilizing intermediate representations of both morphemes and characters.
*   A significant strength is the public release of the source code and pre-trained models for the French-English, Czech-English, and English-Czech language pairs. This greatly facilitates validation, replication, and extension of the work.
*   The model demonstrates rapid learning, achieving reasonable translation quality after only a single training epoch and showing continued improvement in subsequent epochs.
*   The system appears to learn effective morphological segmentations and capably handles out-of-vocabulary words, including nonce words, by leveraging its morpheme-level representations.

### Weaknesses:
*   The paper does not specify which WMT test and development sets were used for evaluation. This makes it difficult for readers to compare the reported scores with existing results, such as those on matrix.statmt.org. This crucial information was only found in the README file of the supplementary code and should be stated explicitly in the paper.
*   The software documentation could be improved. The README file, for instance, lacks explicit command-line examples for training and testing. Furthermore, the software does not respond to a standard `--help` flag, which would improve its usability.
*   There is a discrepancy between the architecture described in the text (6 levels) and the diagram in Figure 2, which appears to show fewer than six layers. The figure caption should be updated to clarify this, or a new diagram illustrating the complete architecture should be included, possibly in an appendix.
*   The experimental results are only compared against other character-based neural systems. The paper fails to acknowledge or discuss the overall state-of-the-art results for these datasets (available from WMT), which are substantially higher than those reported. Including this broader context is essential.
*   The manuscript has a number of minor English grammar errors, misspellings, and LaTeX formatting issues (e.g., inverted quotation marks) that require correction.

### General Discussion:
This paper is a valuable contribution to the literature on character-based neural machine translation.
