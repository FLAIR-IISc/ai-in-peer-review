- Strengths:
The paper introduces an innovative encoder-decoder NMT architecture that operates on characters at the input and output layers, while internally processing representations derived from both morphemes and characters.
The public release of the source code and the final trained models for the fr-en, cs-en, and en-cs language pairs is commendable. This facilitates the verification of the reported results and encourages future work that builds upon this research.
The proposed system demonstrates impressive training efficiency, achieving a respectable translation quality after only a single epoch and showing steady improvement in subsequent epochs.
A significant strength is the model's ability to learn plausible morphological segmentations. This allows it to effectively translate out-of-vocabulary words, including nonce words, by leveraging its morpheme-level representations.
- Weaknesses:
The manuscript fails to specify the WMT development and test sets used for evaluation, which complicates direct comparison with established benchmarks (e.g., matrix.statmt.org). This critical information—newstest2013 for development and newstest2015 for testing—was only found in the supplementary code's documentation and should be explicitly stated in the main paper.
The accompanying software documentation could be improved. Specifically, the README file would benefit from concrete examples of the training and testing commands. Furthermore, the software lacks a standard --help flag for command-line usage, which should be implemented.
There is a notable inconsistency between the text, which describes a 6-level architecture, and Figure 2, which appears to depict fewer layers. This discrepancy is confusing. The figure's caption should be clarified, and a complete diagram of the full architecture should be provided, perhaps in an appendix.
The experimental results are compared only against other character-based neural models, omitting a comparison with the broader state-of-the-art in machine translation. As results on WMT datasets from other system types are significantly higher, the paper should acknowledge this performance gap and provide some discussion to contextualize its contributions.
The manuscript requires a thorough proofreading to correct several minor grammatical errors, typos, and formatting issues (e.g., incorrect LaTeX quotation marks).
- General Discussion:
Overall, this paper represents a valuable contribution to the field of character-level neural machine translation.
