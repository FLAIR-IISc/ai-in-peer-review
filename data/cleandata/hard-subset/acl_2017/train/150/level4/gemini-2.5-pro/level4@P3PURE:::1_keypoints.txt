- Novel encoder-decoder NMT variant that uses characters at input/output while internally representing both morphemes and characters.  
- Public release of source code and trained models for French-English, Czech-English, and English-Czech, enabling verification and future work.  
- System achieves reasonable translation quality after a single epoch and continues to improve with more training.  
- Model learns sensible morphological segmentations, helping handle out-of-vocabulary and nonce words via morpheme representations.  
- Paper fails to state which WMT dev/test sets were used (only found in README: newstest2013 dev, newstest2015 test); this should be explicit in the paper.  
- Documentation is lacking: README needs explicit command-line training/testing examples and the software should implement a standard --help flag.  
- Discrepancy between the claimed 6-level architecture and Figure 2’s depiction; figure caption should clarify or a full architecture diagram should be provided.  
- Experimental comparison focuses on other character-based systems and omits broader state-of-the-art WMT systems; the paper should acknowledge and contextualize the performance gap.  
- Contains minor grammatical errors, typos, and formatting issues (e.g., LaTeX quotation marks) that require thorough proofreading.  
- Overall assessment: a solid and interesting contribution to character-based neural machine translation.
