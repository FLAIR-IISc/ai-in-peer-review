### Strengths:

*   The paper presents a novel adaptation of the encoder-decoder NMT framework. The proposed model operates on characters at the input and output layers, while internally utilizing representations of both morphemes and characters.
*   The public release of the source code and the final trained models for French-English, Czech-English, and English-Czech is a significant strength. This facilitates verification of the results and encourages future work that can replicate and extend this research.
*   The proposed system demonstrates the ability to produce reasonable translation quality after only a single epoch of training, with performance continuing to improve in subsequent epochs.
*   A key advantage of the model is its ability to learn sensible morphological segmentations. This allows it to handle out-of-vocabulary words, including nonce words, by implicitly backing off to its learned morpheme representations.

### Weaknesses:

*   The paper does not explicitly state which WMT development and test sets were used for evaluation. This omission makes it difficult for readers to compare the reported results with those of other systems, such as those listed on matrix.statmt.org. This information was only found in the README file of the supplementary code (newstest2013 for development and newstest2015 for testing) and should be clearly stated in the main paper.
*   The documentation accompanying the source code could be improved. Specifically, the README file would benefit from explicit command-line examples for training and testing. Additionally, the software should implement a standard `--help` flag to display usage information, which it currently lacks.
*   There is a discrepancy between the text, which describes a 6-level architecture, and Figure 2, which appears to depict fewer than six layers. The figure's caption should be clarified to explain this. If the figure is a simplification, a complete diagram of the full architecture should be provided, perhaps in an appendix.
*   The experimental evaluation compares the proposed model primarily to other character-based NMT systems. However, it omits a comparison with the overall state-of-the-art for these language pairs, which, according to WMT results, is substantially higher than the scores reported here. The paper should acknowledge this performance gap and ideally include a discussion contextualizing its results within the broader MT landscape.
*   The paper contains several minor grammatical errors, typos, and formatting issues (e.g., incorrect LaTeX quotation marks). A thorough proofreading is recommended to correct these.

### General Discussion:

Overall, this paper presents a solid and interesting contribution to the literature on character-based neural machine translation.
