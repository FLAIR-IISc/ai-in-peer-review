**Strengths:**

The authors present a new implementation of the Encoder-Decoder Neural MT, which adopts an approach that starts and ends with characters, but works with morphemes and character representations in between.

The authors also release their code for fr-en, cs-en, and en-cs, as well as their final learned models. This helps validate their work, as well as for others who want to replicate and expand on it.

The demonstrated system appears to produce reasonably good translation results even after the first training epoch, and continues to improve in future epochs.

The system seems capable of learning proper morphological tokenization, and also appears capable of handling previously unseen words (even non-native words) by implicitly going back to morphemes.

**Weaknesses:**

In the paper, the authors do not explicitly state which WMT test and dev set their results were reported on. This is problematic for readers who want to compare the reported results with existing work (for example, results on matrix.statmt.org). The only way this reviewer could obtain this information was to look in the code supplement's README, which indicates that the test set was newstest2015 and the dev test was newstest2013. This should have been clearly described in the paper.

The instructions in the software README are fine, but not very good. Each of the training and testing sections could be enhanced with clear examples of how to run the relevant commands. The software itself should respond to a --help flag, which it currently does not.

The paper describes a 6-layer architecture, but the diagram in Figure 2 shows fewer than 6 layers. What is happening? The caption should be more explicit, and if the figure is not showing all the layers, there should be a figure somewhere (even if it's in an appendix) that shows all the layers.

The results show comparisons with other character-based neural systems, but do not show state-of-the-art results for other types of MT systems. WMT (and matrix.statmt.org) have reported results for other systems on these datasets, and it seems that the state-of-the-art is much higher than any of the results reported in this paper. This should be acknowledged, and ideally discussed.

There are a handful of minor English errors, incorrect spellings, and minor LaTeX issues, such as inverted quotation marks. These should be corrected.

**General Discussion:**

The paper is a good contribution to the existing literature on character-based neural MT.
