- Introduces an encoder-decoder NMT that starts and ends with characters while using morpheme and character representations in intermediate layers.
- Provides open-source code and trained models for fr-en, cs-en, and en-cs, enabling replication and validation.
- Shows the system produces reasonable translations after the first epoch with improvements in later epochs.
- Learns morphological tokenizations and can handle unseen or nonce words by reverting to morpheme-level representations.
- Fails to state which WMT test/dev sets were used in the paper (only found in the code README: newstest2015/test, newstest2013/dev).
- README training/testing instructions are adequate but need explicit command examples and a missing `--help` flag.
- Figure 2 and caption do not clearly match the described six-layer architecture; the full architecture should be illustrated or explained.
- Omits comparisons to state-of-the-art (non-character-based) MT results on WMT datasets and should acknowledge/discuss the performance gap.
- Contains minor English phrasing, typographical, and LaTeX formatting errors that require correction.
- Overall, a valuable contribution to character-based neural machine translation research.
