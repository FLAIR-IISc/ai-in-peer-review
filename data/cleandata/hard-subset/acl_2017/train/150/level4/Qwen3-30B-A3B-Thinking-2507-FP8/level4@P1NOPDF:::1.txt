

Strengths:  
The authors propose a novel encoder-decoder neural MT framework that processes input and output at the character level while leveraging morpheme-based representations internally. This approach effectively bridges character-level and morpheme-level modeling. The authors release both code and final models for fr-en, cs-en, and en-cs, enabling reproducibility and facilitating further research. The system achieves reasonable translation quality even after the first training epoch, with steady improvement across subsequent epochs. Notably, it demonstrates robust morphological tokenization capabilities and effectively handles unseen words—including nonce words—by implicitly decomposing them into learned morphemes.

Weaknesses:  
The paper omits explicit specification of the WMT test and development sets used for evaluation. This omission hinders direct comparison with existing work (e.g., results on matrix.statmt.org), as the necessary details were only discoverable in the code supplement’s README (newstest2015 for test, newstest2013 for dev). This information must be clearly stated in the main text.  

The software README provides adequate instructions but lacks concrete examples for training and inference commands. The tool should also support a `--help` flag, which it currently does not.  

Figure 2 depicts a 6-level architecture but appears to show only four layers. The caption must explicitly clarify this discrepancy, and if the figure omits layers, a full architectural diagram (e.g., in an appendix) is required.  

The results compare favorably against character-based neural MT systems but omit state-of-the-art results for other MT approaches on the same datasets. Given that WMT benchmarks report significantly higher scores for non-character-based systems (e.g., on matrix.statmt.org), this gap must be acknowledged and discussed.  

Minor language issues persist, including disfluencies, spelling errors, and LaTeX formatting inconsistencies (e.g., reverse quotation marks), which should be corrected.  

General Discussion:  
The paper makes a valuable contribution to character-based neural machine translation, offering a well-motivated architectural innovation with strong empirical support. The release of code and models further enhances its utility to the community.
