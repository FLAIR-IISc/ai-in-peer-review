

Strengths:  
The paper introduces a novel encoder-decoder neural machine translation (NMT) architecture that processes input and output at the character level while leveraging morpheme-level representations internally. This approach demonstrates clear innovation in bridging character-based and morpheme-based modeling. The authors provide comprehensive reproducibility resources, releasing both code and trained models for French-English, Czech-English, and English-Czech translation tasks, significantly aiding validation and future extensions. The system achieves reasonable translation quality even after the first training epoch, with consistent improvement across subsequent epochs. Notably, the model learns meaningful morphological tokenizations and effectively handles unseen words—including nonce words—by implicitly decomposing them into morphemes during inference.

Weaknesses:  
The paper omits explicit specification of the WMT test and development sets used for evaluation, which is critical for readers to contextualize results against existing benchmarks (e.g., matrix.statmt.org). The reviewer was forced to infer the test set as newstest2015 and development set as newstest2013 from the code README, a practice that undermines clarity and should be addressed in the main text. The software README offers functional but insufficient instructions; training and testing sections would benefit from concrete command examples, and the tool should implement a `--help` flag for immediate user guidance. Figure 2 depicts a 6-level architecture but visually shows fewer layers than described. The caption must clarify this discrepancy, and if the figure omits layers, an appendix diagram should be added. Crucially, the results compare only to character-based NMT systems but omit state-of-the-art results from other MT paradigms (e.g., word-level or subword-based systems) on the same datasets. Given that WMT benchmarks report significantly higher scores for SOTA systems (e.g., 30+ BLEU for some tasks), this omission requires explicit acknowledgment and discussion. Minor issues include typographical errors (e.g., reverse quotation marks), inconsistent LaTeX formatting, and minor English disfluencies that should be corrected.

General Discussion:  
This work makes a valuable contribution to character-based NMT research by introducing a morpheme-aware architecture that effectively handles morphology and out-of-vocabulary words. The reproducibility efforts and early training performance are particularly commendable. However, the lack of explicit dataset references, incomplete architectural visualization, and absence of SOTA comparisons weaken the paper’s impact. Addressing these points would substantially strengthen the submission. The core idea is promising and merits publication with revisions, but the current presentation requires refinement to meet ACL’s standards for clarity and scholarly rigor.
