

- Strengths:  
The authors present a novel adaptation of encoder-decoder neural machine translation (MT) that operates entirely at the character level for input and output, while utilizing morpheme and character representations in intermediate layers. The release of both code and final learned models for fr-en, cs-en, and en-cs translation tasks significantly enhances reproducibility and facilitates further research. The system achieves reasonable translation quality even after the first training epoch, with consistent improvement across subsequent epochs. Furthermore, the model successfully learns meaningful morphological tokenizations and handles unseen words—including nonce words—by implicitly backing off to character-level representations.  

- Weaknesses:  
The paper fails to explicitly specify the WMT test and development sets used for evaluation, which impedes direct comparison with prior work (e.g., results on matrix.statmt.org). The required information was only discoverable in the code supplement’s README (indicating newstest2015 for test and newstest2013 for dev), a detail that should be clearly stated in the main text.  
The software README provides adequate but insufficient instructions; training and testing sections would benefit from explicit command examples. The software itself lacks a --help flag, which is a standard expectation for user-facing tools.  
The paper describes a 6-level architecture, yet Figure 2 depicts fewer than six layers. The caption must clarify this discrepancy, and if the figure omits layers, a supplementary figure (e.g., in an appendix) should explicitly show all six levels.  
The results compare only against character-based neural MT systems but omit state-of-the-art results for other MT approaches (e.g., word-level or subword-based systems) on the same datasets. Given that matrix.statmt.org reports significantly higher scores for current SOTA systems, this gap must be acknowledged and discussed.  
Minor grammatical errors, English disfluencies, misspellings, and LaTeX issues (e.g., backwards quotation marks) require correction.  

- General Discussion:  
This paper makes a valuable contribution to the literature on character-based neural machine translation.
