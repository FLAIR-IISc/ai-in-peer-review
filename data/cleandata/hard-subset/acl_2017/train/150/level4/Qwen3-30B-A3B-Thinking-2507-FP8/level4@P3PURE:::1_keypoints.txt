Strengths:
- Novel character-level encoder-decoder MT that uses morpheme and character representations in intermediate layers.
- Code and final models for fr-en, cs-en, and en-cs released, improving reproducibility.
- Reasonable translation quality after the first epoch with consistent improvements thereafter.
- Learns meaningful morphological tokenizations and handles unseen/nonce words by backing off to character-level representations.

Weaknesses:
- Paper fails to explicitly state WMT test/dev sets used (only found in README: newstest2015 test, newstest2013 dev), hindering direct comparison with prior work.
- README lacks explicit training/testing command examples and the software lacks a standard --help flag.
- Figure 2 contradicts the described 6-level architecture; caption should clarify and a full six-level figure should be provided (e.g., in an appendix).
- Results only compare to character-based MT systems and omit state-of-the-art word/subword baselines (e.g., scores on matrix.statmt.org), which should be acknowledged and discussed.
- Contains minor grammatical errors, English disfluencies, misspellings, and LaTeX issues that need correction.

General:
- Overall, the paper makes a valuable contribution to the literature on character-based neural machine translation.
