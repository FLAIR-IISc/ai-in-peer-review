- Novel encoder-decoder NMT that starts/ends at the character level and uses morpheme and character representations internally.  
- Authors publicly release code and trained models for fr-en, cs-en, and en-cs, enabling replication and validation.  
- System achieves reasonable translation quality after the first epoch and improves consistently in later epochs.  
- Learns morphological tokenization effectively and handles unseen/nonce words by reverting to morpheme-level processing.  
- Manuscript fails to explicitly state the WMT test/dev sets used ( reviewer found newstest2015/newstest2013 only in the README).  
- Software README is suboptimal—needs explicit training/testing command examples and a --help flag.  
- Figure 2/caption is unclear about the claimed 6-level architecture; if not all layers are shown, provide a supplementary figure.  
- Results compare only to character-based systems and omit stronger state-of-the-art MT benchmarks reported on WMT/matrix.statmt.org.  
- Minor issues: English disfluencies, spelling mistakes, and LaTeX problems (e.g., reversed quotation marks).  
- Overall, a valuable contribution to the literature on character-based neural machine translation.
