## Review

**Strengths:**

The authors present a novel adaptation of encoder-decoder neural machine translation (MT) that uniquely combines character-level processing at the input and output with morpheme- and character-level representations in between. 

The release of both code and trained models for fr-en, cs-en, and en-cs is a significant contribution, facilitating validation of the work and enabling replication and extension by other researchers.

The reported system achieves reasonably high-quality translation results even after the initial training epoch, with continued improvement observed in subsequent epochs.

The system demonstrates an ability to learn plausible morphological tokenizations and effectively handles out-of-vocabulary words, including nonce words, by implicitly falling back to morphemic representations.

**Weaknesses:**

The paper lacks explicit specification of the WMT test and development sets used for evaluation. This omission hinders comparison of the reported results with existing work, such as those available at matrix.statmt.org. I located this information only within the code supplement’s README file, which indicates the use of newstest2015 for the test set and newstest2013 for the development set. This detail should have been clearly stated in the paper itself.

While the instructions in the software README are adequate, they could be improved. Providing explicit examples of how to execute the training and testing commands would be beneficial. The software should also respond to a `--help` flag, which is currently not implemented.

The paper describes a 6-layer architecture, but Figure 2 appears to illustrate fewer than six layers. This discrepancy requires clarification. The figure caption should be more descriptive, and if the figure does not depict all layers, an additional figure (potentially in the appendix) showcasing the complete architecture should be included.

The results section compares the system’s performance against other character-based neural systems but does not include comparisons to state-of-the-art results from other MT approaches. WMT and matrix.statmt.org report results from a broader range of systems on these datasets, and the current state-of-the-art significantly surpasses the results presented in this paper. This should be acknowledged and discussed.

The paper contains minor English errors, including disfluencies, misspellings, and LaTeX issues (e.g., incorrect quotation marks). These should be addressed during final polishing.

**General Discussion:**

This paper represents a valuable contribution to the existing literature on character-based neural MT.
