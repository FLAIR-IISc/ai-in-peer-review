## Review

**Strengths:**

This paper introduces a novel neural machine translation (MT) approach based on an encoder-decoder architecture. A key feature is its utilization of characters at the input and output layers, combined with morphemic and character-level representations internally. The authors’ decision to publicly release both the code and trained models for fr-en, cs-en, and en-cs translation is highly commendable. This facilitates verification of the research and provides a valuable resource for future work in this area. The reported system demonstrates promising translation quality, achieving noticeable improvements even from the initial training epoch and continuing to refine performance with subsequent iterations. Furthermore, the system exhibits a capacity for effective morphological tokenization, and appears capable of handling out-of-vocabulary words – including novel terms – by leveraging implicit fallback mechanisms to morphemic units.

**Weaknesses:**

The paper lacks clarity regarding the specific WMT test and development sets used for evaluation.  This omission hinders direct comparison of the reported results with existing publications (e.g., those available on matrix.statmt.org).  Identifying this information required consulting the code supplement’s README file, which indicates the use of newstest2015 for the test set and newstest2013 for development. Explicitly stating these details within the paper is crucial. While the software documentation in the README is adequate, it could be improved. Providing concrete examples for running the training and testing commands would be beneficial.  Moreover, the software should implement a `--help` flag to enhance usability.  The paper describes a 6-level architecture, however, the diagram in Figure 2 depicts fewer than six layers. The figure caption requires clarification. If the figure does not illustrate the complete architecture, an additional figure – potentially in the appendix – showcasing all layers should be included. The results section presents comparisons to other character-based neural systems but does not benchmark against state-of-the-art results from other MT approaches. Considering that WMT and matrix.statmt.org report higher scores for different systems on these datasets, this context should be acknowledged and discussed. Finally, the manuscript contains some minor issues related to English grammar, spelling, and LaTeX formatting (e.g., incorrect quotation mark direction) which should be addressed.

**General Discussion:**

This paper represents a valuable contribution to the field of character-based neural machine translation.
