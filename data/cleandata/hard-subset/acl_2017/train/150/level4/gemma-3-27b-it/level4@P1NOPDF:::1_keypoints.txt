- Novel model: combines character-level input/output with morpheme- and character-level intermediate representations in an encoder-decoder MT architecture.  
- Reproducibility: authors release code and trained models for fr-en, cs-en, and en-cs.  
- Fast learning: system achieves high-quality translations after the first epoch and improves with further training.  
- Robustness: model learns plausible morphological tokenizations and handles OOV and nonce words via morphemic fallback.  
- Missing dataset info: paper does not state WMT test/dev sets used (found in README as newstest2015 test / newstest2013 dev) and should report this explicitly.  
- Usability: README is adequate but needs explicit train/test command examples and a working --help flag.  
- Architecture clarity: paper claims a 6-layer model but Figure 2 shows fewer layers; figure caption is insufficient and a full-architecture figure is needed.  
- Baseline comparisons: results are compared only to other character-based systems and do not acknowledge or compare to WMT/matrix.statmt.org state-of-the-art systems that outperform this work.  
- Presentation issues: minor English disfluencies, misspellings, and LaTeX/quotation problems need polishing.  
- Overall: paper is a valuable contribution to character-based neural MT.
