- Strengths:
The authors present a novel adaptation of encoder-decoder neural machine translation (MT) that operates on characters at the input and output levels, but utilizes representations of both morphemes and characters internally. 
The release of both the code and the final learned models for French-English, Czech-English, and English-Czech is a valuable contribution, as it facilitates the validation of their work and enables others to replicate and extend this research.
The reported system produces translation results of reasonable quality even after the first training epoch, with continued progress in subsequent epochs.
Furthermore, the system appears to learn reasonable morphological tokenizations and is able to handle previously unseen words (including nonce words) by implicitly backing off to morphemes.

- Weaknesses:
A notable limitation of the paper is that it does not explicitly specify which WMT test and development sets the reported results are based on. This omission makes it challenging for readers to compare the results to existing work (e.g., the results available at matrix.statmt.org). This reviewer had to consult the README file in the code supplement to determine that the test set was newstest2015 and the development set was newstest2013. This information should have been clearly stated in the paper.
The instructions provided in the software README are adequate but could be improved. Specifically, the training and testing sections would benefit from explicit examples of how to run the respective commands. Additionally, the software should respond to a --help flag, which is currently not implemented.
The paper describes a 6-level architecture, but the diagram in Figure 2 appears to depict fewer than 6 layers. This discrepancy should be addressed, and the caption should be more explicit. If Figure 2 does not show all the layers, a supplementary figure (possibly in an appendix) should be included to provide a complete illustration of the architecture.
The results are compared to other character-based neural systems, but do not include state-of-the-art results for other types of MT systems. The WMT results (and those reported on matrix.statmt.org) suggest that the state-of-the-art performance is significantly higher than the results reported in this paper. This discrepancy should be acknowledged and discussed.
Finally, there are several minor English disfluencies, misspellings, and LaTeX issues (e.g., reverse quotation marks) that should be corrected.

- General Discussion:
This paper is a valuable contribution to the existing literature on character-based neural MT, and its findings have the potential to advance research in this area.
