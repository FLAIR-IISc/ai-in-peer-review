**Strengths:**

The authors introduce a novel neural machine translation (MT) approach based on an encoder-decoder architecture. This work uniquely integrates characters with morphemic and character-level representations. The public release of both code and trained models for fr-en, cs-en, and en-cs translation pairs is a significant benefit, facilitating validation and further research. The system demonstrates promising translation quality, achieving noticeable improvements even after initial training and continuing to progress with subsequent epochs. The system effectively learns morphological tokenizations and exhibits a capacity to handle out-of-vocabulary words, including novel terms, through implicit morphemic decomposition.

**Weaknesses:**

The paper lacks explicit specification of the WMT test and development sets used for evaluation. This omission hinders direct comparison of reported results with existing literature, such as those available on matrix.statmt.org. Identifying this information required consulting the code supplement’s README file, which indicates newstest2015 for the test set and newstest2013 for the development set; this detail should be included directly in the paper. While adequate, the software instructions in the README could be improved with concrete examples demonstrating command execution for both training and testing. Implementing a `--help` flag for the software would further enhance usability. The paper references a 6-level architecture, but Figure 2 depicts fewer layers. The figure caption requires clarification, and a comprehensive diagram illustrating all layers – potentially in an appendix – should be provided if the current figure is incomplete. The results are benchmarked against other character-based neural systems, but lack comparison to state-of-the-art results from other MT approaches. Published results on these datasets, available through WMT and matrix.statmt.org, indicate significantly higher performance levels than those reported here, a discrepancy that should be acknowledged and discussed. Finally, the manuscript contains minor issues with English grammar, spelling, and LaTeX formatting (e.g., incorrect quotation marks) that require correction.

**General Discussion:**

This paper represents a valuable addition to the body of work on character-based neural machine translation.
