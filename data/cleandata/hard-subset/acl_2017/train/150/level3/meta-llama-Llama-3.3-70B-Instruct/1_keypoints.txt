- Proposes a novel multigraph-based model for recognizing overlapping entities in text.  
- Releases code and learned models to allow validation and replication.  
- System yields reasonable translation results after the first training epoch and learns morphological tokenizations.  
- Main contributions: new multigraph model, higher F1-scores vs prior models, and theoretical proof resolving spurious-structure issues.  
- Strength: handles overlapping entities common in many datasets.  
- Strength: model is non-deficient, addressing spurious structures of prior SOTA.  
- Strength: empirical improvements across several datasets.  
- Weakness: authors do not explicitly state which test/dev sets were used, hindering comparison.  
- Weakness: software README is incomplete and unclear.  
- Weakness: diagram of the 6-level architecture is unclear and appears to show fewer than six layers.  
- Question: request for more details on features used and the feature selection process.  
- Question: how the model handles multiple overlapping entities with different types.  
- Question: ask for deeper analysis of the spurious-structures issue and how the proposed model resolves it.  
- Overall: a useful contribution to character-based neural MT with promising results, but needs clearer documentation and more thorough analysis.
