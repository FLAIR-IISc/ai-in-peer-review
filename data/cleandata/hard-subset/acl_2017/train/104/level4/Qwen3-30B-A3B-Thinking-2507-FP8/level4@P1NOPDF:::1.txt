

Strengths:  
* The model outperforms ALIGN on the supervised entity linking task, demonstrating that its jointly learned text and knowledge representations are more effective.  
* The direct comparison with a closely related approach using identical input data strengthens the validity of the results.  
* The analysis of the smoothing parameter is particularly valuable, as the impact of entity popularity remains a longstanding challenge in entity linking.  

Weaknesses:  
* The comparison with ALIGN requires clarification. ALIGN uses a context window size of 10 versus the current work’s 5, and a vector dimension of 500 versus 200. Crucially, it is unclear whether $N(e_j)$ includes only entities linking *to* $e_j$ (as defined for ALIGN, where context comprises entities linking to a target). The graph is directed (using Wikipedia outlinks), but adjacency appears defined as in an undirected graph. If $N(e_j)$ differs between methods, the observed performance gap on the entity similarity task could stem from this implementation difference rather than the core approach.  
* The terminology around "mention" is ambiguous. The paper uses "mention embedding," yet embeddings are learned only for *mention senses*, not for raw string types or document-specific mentions. This inconsistency complicates interpretation of the model’s inputs and outputs.  
* Without comparison to other unsupervised entity linking methods, the impact of the sense disambiguation order remains undetermined. This omission limits the assessment of whether the proposed ordering strategy is a meaningful contribution.  

General Discussion:  
The paper makes a compelling case for jointly learning text and knowledge representations, with the entity linking improvements and smoothing parameter analysis representing solid contributions. However, the methodological inconsistencies in the ALIGN comparison—particularly the differing context window sizes, vector dimensions, and neighborhood definitions—undermine the confidence in attributing performance gains solely to the proposed framework. Clarifying these implementation details would significantly strengthen the evaluation. Additionally, resolving the ambiguity around "mention" terminology and benchmarking against unsupervised baselines would address key gaps in the analysis. While the core idea is promising, the current comparison framework prevents a definitive conclusion about the method’s relative advantages.
