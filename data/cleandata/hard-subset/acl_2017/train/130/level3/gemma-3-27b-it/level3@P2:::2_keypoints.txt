- Core idea: Model speech transcripts as word co-occurrence graphs enriched with word embeddings and use graph topological features to detect MCI.
- Primary contribution: Proposes complex network representations as a lightweight, potentially language-independent feature set for MCI detection.
- Novelty: Applies graph-based text representation specifically to MCI detection and shows competitive performance versus traditional linguistic features.
- Multi-dataset evaluation: Tests the approach on three datasets (English and Portuguese) with different discourse types, suggesting some generalizability.
- Feature combination: Explores ensemble and multi-view learning to combine network, linguistic, and bag-of-words features, improving results on some datasets.
- Clear methodology: Provides a reasonably clear description of graph construction, embedding enrichment, and topological feature extraction.
- Clinical relevance: Addresses an important problem (early MCI detection) using speech transcripts from neuropsychological assessments.
- Writing quality: Manuscript has numerous grammatical errors, awkward phrasing, and inconsistent terminology that hinder readability.
- Data description unclear: Dataset and instance definitions are confusing; transcript characteristics and participant demographics are inconsistently reported.
- Graph enrichment justification weak: Motivation for using embeddings to enrich co-occurrence networks is not well justified relative to simpler semantic methods.
- Limited use of domain tools: Paper overlooks existing biomedical NLP tools and clinical-text resources that could improve feature extraction.
- Inappropriate metric: Reporting only accuracy is misleading given possible class imbalance; need F1, precision, recall, AUC.
- No significance testing: Results lack statistical significance testing and report over-precise accuracy values without justification.
- Coh-Metrix feature choices unclear: Unclear why specific Coh-Metrix-Dementia features were discarded—no justification provided.
- Embedding threshold unexplained: Rationale and optimization for the cosine similarity threshold used to add edges are not described.
- Instance definition question: Clarify whether the classification instance is a full narrative, a segment, a sentence, or other.
- Class distribution question: Provide counts of MCI vs. control subjects for each dataset.
- Feature selection question: Explain whether exclusions of certain features were based on experiments or theory.
- Evaluation metrics question: Explain why alternative metrics (F1, precision, recall, AUC) were not reported or provide them.
- Citation and supplement issues: Citation style is inconsistent and supplementary material contains irrelevant content; add discussion of limitations and future work.
