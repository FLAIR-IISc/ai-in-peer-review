- Summary: Proposes modeling speech transcripts as complex networks enriched with word embeddings and using network topological features for MCI detection; evaluated on English and Portuguese datasets against linguistic and BoW baselines.

- Contribution 1: Novel application of complex network analysis (with embeddings) to MCI detection from speech.

- Contribution 2: Empirical evaluation across three datasets (two English, one Portuguese) demonstrating generalizability.

- Contribution 3: Shows performance gains when combining network features with traditional linguistic features and bag-of-words.

- Strength 1: Interesting and logical extension of network-based text analysis by incorporating word embeddings.

- Strength 2: Solid experimental setup using 5-fold cross-validation and multiple classifiers across languages.

- Strength 3: Clear, detailed methodological description enabling reproducibility.

- Strength 4: Competitive results, with combined features achieving the best performance (e.g., 75% on Cinderella dataset).

- Strength 5: Practical clinical relevance for automated speech-transcript analysis and early MCI diagnosis.

- Weakness 1: Individual components (networks, embeddings) are not novel; novelty is mainly in their application to this domain.

- Weakness 2: Lacks detailed error analysis to explain types and causes of misclassifications.

- Weakness 3: Insufficient detail on parameter sensitivity and threshold optimization (e.g., cosine similarity cutoff).

- Weakness 4: Dataset heterogeneity (narrative length, elicitation) not thoroughly analyzed for its influence on results.

- Weakness 5: Limited comparison to state-of-the-art MCI-from-speech methods.

- Question 1: Request for details on cosine similarity threshold optimization and its impact on performance.

- Question 2: Request for an in-depth error analysis linking misclassifications to linguistic patterns.

- Question 3: How robust are results to training set size variations (smaller/larger datasets)?

- Question 4: What is the impact of manually revised transcripts and how to handle noisy, unrevised real-world transcripts?

- Question 5: Clarify selection and exclusion rationale for Coh-Metrix linguistic features.

- Additional comment 1: Writing is generally good but some sentences are convoluted.

- Additional comment 2: Supplementary material is useful for dataset and feature details.

- Additional comment 3: Provide code and data links to facilitate reproducibility.

- Overall assessment: Solid, well-executed paper; recommend accept with minor revisions to address weaknesses.
