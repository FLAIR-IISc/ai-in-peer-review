- Paper tackles MCI detection from speech transcripts across three datasets using a graph-based co-occurrence method with lexical/syntactic features and reports 5-fold CV results with multiple classifiers.  
- Writing quality is poor with grammatical errors and awkward phrasing, reducing readability.  
- The machine learning formulation is unclear: the unit of classification (transcript, sentence, etc.) isn’t specified and dataset descriptions focus on demographics rather than instance-level data.  
- Reported dataset numbers are inconsistent (e.g., section 4.1.1 vs. Tables), raising doubts about data reporting.  
- The motivation for enriching co-occurrence graphs with word embeddings is not justified; an alternative of connecting nodes by vector similarity is not discussed.  
- The approach ignores domain-specific biomedical NLP tools or knowledge despite using biomedical datasets.  
- Class distributions (MCI vs. control) are not provided, making accuracy an unreliable metric; F1 would be more appropriate for the binary task.  
- Results are reported to excessive precision on very small datasets without statistical significance testing, so reported gains may be meaningless.  
- Request to clarify what constitutes a single classification instance (transcript, sentence, etc.).  
- Request to provide class counts (MCI and control) for each dataset.  
- Request to explain the rationale and advantage of enriching co-occurrence networks with embeddings versus building graphs from vector similarity.  
- Request to state whether domain-specific NLP tools/resources were considered or used, and why/why not.  
- Request to provide statistical significance tests (e.g., paired t-tests) for performance comparisons given small sample sizes.  
- Suggest revising dataset descriptions to highlight attributes relevant to the classification task.  
- Recommend using more descriptive variable names in code for readability.  
- Ask for clarification of preprocessing steps applied to each dataset, particularly disfluency removal and sentence segmentation.
