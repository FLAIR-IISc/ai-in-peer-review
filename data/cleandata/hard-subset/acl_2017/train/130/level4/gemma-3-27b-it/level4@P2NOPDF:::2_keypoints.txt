- Paper investigates MCI detection from speech transcripts using a graph-based method built from word co-occurrence and extracted graph features for classification.
- Evaluated on three datasets with 5-fold cross-validation and multiple classifiers, showing varying performance across datasets.
- Addresses a clearly defined problem and uses appropriate datasets.
- Manuscript has numerous grammatical and stylistic errors that hinder readability.
- The ML task is not clearly articulated; dataset descriptions focus on study demographics rather than transcript-level classification details.
- Tables report irrelevant study-level demographics (e.g., patient age) instead of features pertinent to the classification task.
- Inconsistencies in reported dataset numbers (e.g., participant counts vs. table totals) raise concerns about data reporting.
- Motivation for using word co-occurrence to build graphs is unclear; no justification for not using semantic/vector similarity connections.
- No use of domain-specific biomedical NLP tools or resources that could improve performance.
- Class distributions per dataset are not reported, making accuracy a potentially inappropriate metric; F1-score should be reported for binary classification.
- Results are reported with unjustified precision given small sample sizes, and no statistical significance tests were performed to validate differences.
