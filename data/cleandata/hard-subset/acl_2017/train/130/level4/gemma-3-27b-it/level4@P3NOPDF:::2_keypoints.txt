- Studies MCI detection from speech transcripts using three datasets and a graph-based method leveraging word co-occurrence.
- Encodes graph features using lexical, syntactic, and other characteristics and evaluates with 5-fold cross-validation across several classifiers.
- Demonstrates variable performance across datasets and uses appropriate data sources for the problem.
- Manuscript has many English grammar and spelling errors and needs thorough proofreading.
- The machine learning task is poorly articulated; the unit of classification (e.g., per-transcript instance) is not clearly stated.
- Dataset tables are misleading—should describe dataset instances, not the source studies—and include mostly irrelevant demographic details.
- Reported dataset counts are inconsistent (e.g., Section 4.1.1 claims 326 individuals while Table 1’s gender totals are under 100).
- The motivation for using co-occurrence in the graph is inadequately justified; alternative node-linking by vector similarity is not discussed.
- The paper does not exploit domain-specific biomedical NLP tools or lexicons that could improve performance.
- Class distributions are unclear, making accuracy an unsuitable metric; F1-score is recommended for binary/imbalanced cases.
- Results are reported to four decimal places despite small sample sizes, and no statistical significance testing is provided.
- Authors should include appropriate significance tests (e.g., paired t-test or Wilcoxon signed-rank) to validate reported improvements.
