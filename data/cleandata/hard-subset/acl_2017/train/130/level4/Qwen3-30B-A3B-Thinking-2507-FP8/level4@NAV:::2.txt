

Strengths:  
This paper addresses the problem of identifying Mild Cognitive Impairment (MCI) patients through analysis of speech transcripts from three distinct datasets. It proposes a graph-based methodology that utilizes co-occurrence patterns between words in transcripts, with features derived from lexical, syntactic, and other graph properties. Performance is evaluated via 5-fold cross-validation across multiple classifiers, revealing dataset-specific model variations. The work targets a well-defined clinical problem using appropriate datasets.  

Weaknesses:  
The manuscript has several significant limitations:  
1.  Poor readability due to grammatical errors and spelling mistakes; a thorough language review is essential before publication.  
2.  The core classification task is inadequately described. It is unclear whether each transcript was classified as MCI or non-MCI. If so, dataset descriptions should report transcript-level statistics (not study-level details), as presented in Tables 1–3. Patient age is irrelevant to the classification task and unnecessarily occupies significant space (≈2 pages). A discrepancy exists in reported participant counts: Section 4.1.1 cites 326 participants, yet Table 1’s male/female totals sum to <100.  
3.  The rationale for enriching the graph structure is unexplained. Why not model words as nodes connected by vector similarity (ignoring co-occurrence patterns)?  
4.  Despite the biomedical domain context, no domain-specific linguistic or clinical tools were applied.  
5.  Unclear class distribution precludes justification for using accuracy as the sole metric; F1-score would be more appropriate for binary classification.  
6.  Results are reported to four decimal places for small datasets (e.g., 43 transcripts) without statistical significance testing, making claimed performance gains unverifiable.
