- Language-independent framework that models speech transcripts as word co-occurrence networks enriched with word embeddings to detect MCI without manual correction of disfluencies.  
- Empirical claim that complex network features outperform traditional linguistic features across three datasets (English and Portuguese).  
- Systematic analysis of ensemble and multi-view learning to improve classification accuracy.  
- Novel application of complex network theory to a clinical MCI detection problem, showing cross-language robustness.  
- Strong empirical validation across three datasets with reported improvements (up to 10% on Cinderella) using ensemble/multi-view methods.  
- Practical significance in eliminating costly manual transcript correction, aiding automated clinical deployment.  
- Poor writing, inconsistent terminology, and unclear problem formulation (uncertainty whether transcripts or utterances are classification instances; confusing dataset tables).  
- Insufficient motivation/justification for enriching co-occurrence networks with word embeddings and unclear explanation of how this addresses “linear” short-text networks.  
- Questionable evaluation practices: reporting accuracy to four decimals on small datasets, no statistical tests, inappropriate metric choice given potential class imbalance.  
- Neglect of domain-specific linguistic/clinical features (e.g., MCI-specific disfluency patterns) that could improve relevance.  
- Inadequate benchmarking against recent state-of-the-art neural approaches (e.g., transformers).  
- Request to quantify how semantic enrichment mitigates network linearity with network metrics.  
- Request to clarify the classification unit (transcript vs. utterance) and how dataset counts (e.g., 43 MCI Cookie Theft) map to instances.  
- Request to justify metric choice, report class distributions, and provide statistical validation (e.g., F1/AUC, significance tests).  
- Request to estimate time/cost savings from avoiding manual transcript correction.  
- Overall: promising idea but undermined by writing and methodological gaps; recommend a thorough rewrite and explicit comparison to prior network-based methods (e.g., Amancio et al. 2012).
