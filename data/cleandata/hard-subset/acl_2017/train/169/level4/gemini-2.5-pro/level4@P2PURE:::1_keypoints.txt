- Addresses a useful application for teachers and learners and enables fine-grained comparison of GEC systems.
- System description is highly superficial; core rule set lacks detail (examples, number, complexity, ordering).
- Paper devotes excessive space to CoNLL-2014 system evaluation instead of crucial system details.
- Evaluation is flawed: annotators reviewed pre-annotated output rather than creating an independent gold standard.
- Reported inter-annotator agreement is unconventional and potentially misleading (averaging individual rater scores).
- Lack of clarity on distribution of test-data error categories and which categories the system successfully identifies.
- Vague statement about “unusual” edit boundaries without examples or clear implications.
- Claim that rule-based system is less domain-dependent is questionable given Hunspell coverage and manual rule biases.
- Rules are language-dependent, which is a disadvantage compared to many ML approaches.
- Evaluation uses data from a single domain (student essays), weakening domain-independence claims.
- Motivation for a new error tagset is unclear; existing tagsets (e.g., Nicholls 2003) already support general-category searches.
- Authors did not justify not using the CoNLL-2014 tagset, preventing direct comparison with CoNLL gold standard.
- Primary contribution is ambiguous: new system, new tagset, or superficial CoNLL analysis—none are sufficiently supported.
- Typos and formatting issues need correction (e.g., “c.f.” → “cf.”, M2 formatting, unclear “50-70 F1”, capitalization in references).
