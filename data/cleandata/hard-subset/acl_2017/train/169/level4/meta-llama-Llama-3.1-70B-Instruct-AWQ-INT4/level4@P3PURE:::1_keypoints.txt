- Useful application: provides fine-grained comparison of GEC systems valuable for teachers and learners.
- System description is superficial; core relies on manual rules but authors omit examples, counts, complexity, and ordering.
- Excessive space devoted to CoNLL-2014 results, with repetitive text that merely restates table entries.
- Evaluation method flawed: annotators rated system output instead of independently creating a gold standard for the 200 test sentences.
- Rater treatment flawed: averaging individual rater percentages is inappropriate and may hide that many edits were rated "bad" by some annotators.
- Missing test-data details: no information on number of error categories present or which categories are covered.
- Vague claim "edit boundaries might be unusual" needs precise definition and examples because it could be problematic.
- Domain-independence claim is unsubstantiated: manual rules and resources like Hunspell can be domain-dependent; test sets are all student essays.
- Unclear justification for designing a new error tagset when existing sets (Nicholls 2003 or CoNLL-2014) could be used.
- Overall motivation unclear: is the paper about a new system, a new tagset, or an evaluation of CoNLL-2014 systems?
- Numerous typos and formatting issues (e.g., "c.f." -> "cf.", M2 -> M^2, percent notation, reference casing, Fleiss' kappa).
