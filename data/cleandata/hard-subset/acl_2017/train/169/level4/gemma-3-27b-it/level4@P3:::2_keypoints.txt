- Presents a novel GEC evaluation allowing error-type precision as well as recall, addressing lack of error-tagged system outputs.
- Important contribution for analyzing GEC system behavior and includes evaluation across multiple systems.
- Advantages: computes precision by error type, independent of manual annotation, and handles multi-token errors.
- Human experts mostly approved automatically selected error tags despite noisy automatic edit extraction.
- Key rules for deriving error types are not described, leaving a major methodological gap.
- Unclear basis for error categories and possible dependence of rules on the alignment algorithm; examples of rules are needed.
- Classifier evaluation lacks detailed error analysis and actionable directions to improve the classifier.
- Human rater evaluation may be biased by noisy boundaries and lacked an intermediate “Acceptable” option; re-evaluation with manual boundaries is needed.
- Unclear how well approach generalizes beyond English; rule sets, preprocessing, and error categories may require language-specific adaptation.
- Pre-processing yields continuous error spans, problematic for discontinuous edits (e.g., German separable verbs); classifier’s handling of discontinuous spans is uncertain.
- Reporting of significance tests is unclear (which metrics tested); using F0.5 for significance is questionable due to weak discriminative power.
- Comparison between gold and auto references seems focused on boundaries, not on classification mismatches.
- Some systems’ failures on certain error types may reflect system design limits rather than evaluation failure; analyses did not account for system intent.
- Results in Table 6 lack raw TP/FP/TN/FN counts, obscuring low-instance issues and alternative comparisons.
- Multi-token error analysis is useful but needs interpretation given some systems’ inherent inability to correct such errors and training-data limitations.
- Paper should acknowledge that some systems can already output error annotations via specialized submodules, while stressing the proposed approach’s unifying benefit.
- Minor issues: reference formatting errors and missing bibliographic details.
