- Useful application for teachers and learners; enables fine-grained comparison of GEC systems.  
- System description is highly superficial and omits core implementation details.  
- Authors fail to provide illustrative examples, the number/complexity of rules, or rule ordering/interaction.  
- Excessive emphasis on CoNLL-2014 results (Table 6) with superficial presentation and reiteration of obvious facts.  
- Evaluation lacks an independently annotated gold standard for the 200 test sentences.  
- Small test set and fixed tagset make creation of a gold standard feasible and necessary.  
- Averaging individual rater scores is atypical and may mask dispersed errors; 18.5% “bad” edits conflicts with reported 3.7% average.  
- Missing information about test-data coverage: which error categories exist and which received “good” ratings.  
- “Unusual edit boundaries” is undefined; examples and assessment of its impact are needed.  
- Claim of reduced domain-dependence is questionable: Hunspell and manual rules are domain/language dependent, and tests use only student essays.  
- Rationale for a new error-category set is unclear given existing tagsets (e.g., Nicholls, CoNLL) that could support the same functionality and evaluation.  
- Central motivation is ambiguous: unclear whether the contribution is a new system, a new tagset, or an evaluation of CoNLL systems.  
- Multiple typos and reference formatting errors (e.g., cf., M^2, ESL, Fleiss/Kappa) require correction.
