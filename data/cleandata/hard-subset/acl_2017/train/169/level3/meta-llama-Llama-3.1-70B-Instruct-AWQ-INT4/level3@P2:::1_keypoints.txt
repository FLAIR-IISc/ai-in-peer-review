- Paper proposes automatically annotating parallel error-correction data with explicit edit spans and error types to enable more than recall-based GEC evaluation.

Contributions:
- Proposes an automatic annotation method producing edit spans and error-type labels for GEC data, enabling finer evaluation.
- Introduces a dataset-independent, rule-based error-type classifier that uses only automatic info (POS, lemmas).
- Performs the first comprehensive per-error-type analysis across all CoNLL-2014 shared task systems.

Strengths:
- Enables fine-grained comparison useful for teachers and learners to identify system strengths/weaknesses.
- Provides detailed error-type analyses that can guide researchers to improve overall GEC performance.
- Dataset-independence of the error-type classification supports evaluation across different datasets.

Weaknesses:
- System description is superficial and omits examples, rule counts, rule complexity, and ordering details.
- Paper overemphasizes CoNLL-2014 table results and repeats visible facts without deeper analysis.
- Evaluation relies on rater judgments for 200 test sentences rather than an independent gold standard, risking bias.
- Aggregating rater scores by averaging individual percentages is misleading and may misrepresent performance.
- Lacks information about test-data composition (number of error categories and coverage).
- Makes vague claims (e.g., impact of unusual edit boundaries) without precise definitions or examples.
- Overall motivation and key details are unclear, making the contribution hard to assess.

Questions to authors:
- Please provide core rule-set details: examples, counts, complexity, and rule-ordering.
- How will you address unusual edit boundaries and their potential evaluation impact?
- What is the test-data composition (how many error categories and which are covered)?
- Why adopt a new error-tagset instead of existing tagsets like Nicholls or CoNLL-2014?
- Can you provide deeper analysis of CoNLL-2014 results, highlighting systems' strengths and weaknesses?
