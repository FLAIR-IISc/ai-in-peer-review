- Strength: Paper is well written and easy to follow; proposed methods and results are interesting.
- Weakness: Evaluation methodology and reliability of reported results are potentially problematic.
- Contribution: Proposes end-to-end argumentation mining using neural nets via sequence labeling and dependency parsing, plus multitask learning.
- Motivation: Clearly argues advantage over ILP/manual feature pipelines by joint learning to reduce error propagation.
- Clarity: Methods are clearly described, with a few missing details noted.
- Experiments: Substantial experiments and appropriate comparisons; results are interesting.
- Data size concern: Training set (~320 essays) and unstated dev set are small relative to BiLSTM model capacity, risking unreliable parameter estimation.
- Overfitting: Paper lacks analysis of overfitting; should include training/dev loss curves.
- Model stability: Must report means and standard deviations over multiple runs and perform statistical significance tests to assess reliability.
- Regularization: Discussion and application of regularization are insufficient given dataset size and should be addressed thoroughly.
- Hyperparameter tuning: Recommend replacing current procedure with Bayesian optimization.
- Supplementary material: Move key details (pre-trained embeddings, error analysis) from supplement into main paper.
- Dataset context: Include inter-annotator agreement scores to contextualize dataset difficulty and headroom.
- Figure clarity: Make Figure 1 readable in black-and-white using distinct colors or patterns.
- Post-edit request: After revisions, reviewer raised score to 4 but requests reporting F1-score ranges and means/variances across runs.
- Remaining concern: Analyze large variance in settings (e.g., Kiperwasser); report score ranges (e.g., F1 [0.56,0.61]) to aid replication.
