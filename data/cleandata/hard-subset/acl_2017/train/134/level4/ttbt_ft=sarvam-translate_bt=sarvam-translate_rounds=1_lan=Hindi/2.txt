This work describes a combined neural approach for logic mining. It explores several approaches, including:

1) Presenting the problem as a dependency parsing problem (trying out many different parsers)
2) Presenting the problem as a sequence labeling problem
3) Multitask learning (based on the sequence labeling model below)
4) An out-of-the-box neural model for labeling entities and relations (LSTM-ER)
5) An ILP-based state-of-the-art model

All approaches are evaluated using F1 defined on concepts and relations. Dependency-based solutions do not work well, sequence labeling solutions are effective.

The out-of-the-box LSTM-ER model performs very well, especially at the paragraph level.

Both sequence labeling and LSTM-ER models perform better than the ILP approach.

A very comprehensive supplement was provided, including all the technical aspects of training the model, optimizing hyper-parameters, etc.

It was also shown that sequence labeling models can be greatly improved by a multitask approach (the dependency-based task helps more than the relation-based task).

This is a very thorough examination of neural-based approaches for end-to-end logic mining.

- Key Comments
 - My only concern is with the dataset; I wonder if it's a problem that the essays in the train set and test set can be on the same topics, as a result of which the authors may use the same or similar arguments in both essays, which could lead to information leakage from the train to the test set. In turn, this could give overly optimistic performance estimates. However, I think the same issues exist for ILP models, so your model does not have an undue advantage. Still, it may be worth discussing.
 - My second concern is whether one of your best models, LSTM-ER, is actually an out-of-the-box application of a model from a related task. However, given the relative success of sequence-based models and all the experiments and useful lessons learned, I think this task is publishable.
