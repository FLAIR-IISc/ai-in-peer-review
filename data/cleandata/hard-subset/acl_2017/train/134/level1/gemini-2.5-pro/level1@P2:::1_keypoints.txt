- Comprehensive benchmark and new SOTA for neural end-to-end argumentation mining, showing BiLSTM taggers outperform a strong ILP baseline.
- Clear negative result: token-level dependency parsing is empirically ineffective for AM despite theoretical appeal.
- Novel sequence-tagging formulation that encodes component boundaries, types, and relation distance/type into a single complex tag.

- Systematic, rigorous comparison across multiple neural framings of the full end-to-end AM task.
- Strong empirical gains and clear SOTA improvement (paragraph-level overall F1 improved from 55.23% to 60.72% with LSTM-ER).
- Insightful analysis explaining model behaviors (e.g., LSTM-ER’s essay-level relation drop due to quadratic candidate pairs).
- Novel inductive bias in the sequence-tagging scheme using component-distance encoding to linearize graph prediction.

- Limited generality: STagT assumes one outgoing link per premise (tree-like), which doesn’t hold for many argumentation graphs.
- Ambiguity in distance d definition—unclear how component counts are measured in complex/interleaved text.
- Scalability concerns: tag set size grows with max distance d, risking large/sparse label spaces for long-range dependencies.
- Insufficient qualitative error analysis of top models (common error types and failure modes not deeply explored).

- How exactly is distance d computed (e.g., based on component start token order), and how are forward links (positive d) handled?
- How would STagT be adapted to non-tree structures where components can have multiple incoming/outgoing links?
- Was the maximum distance d capped in experiments; if so, what was the cap and how were relations beyond it handled?
