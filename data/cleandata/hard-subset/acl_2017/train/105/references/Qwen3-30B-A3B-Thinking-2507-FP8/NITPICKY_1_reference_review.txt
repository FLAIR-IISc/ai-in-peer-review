

# Review of "Morphological Inflection Generation with Hard Monotonic Attention"

## Summary

This paper proposes a model for morphological inflection generation that uses a "hard monotonic attention" mechanism. The authors claim their model outperforms previous neural and non-neural approaches, especially on small datasets. The model uses pre-determined alignments (from an external character alignment process) to guide a neural encoder-decoder network, with a control mechanism that alternates between writing output characters and advancing the attention pointer.

## Main Contributions (as I see them)

1. The paper's primary contribution is not the "hard attention" model as described (which doesn't actually learn attention weights), but rather a model that uses pre-determined alignments for morphological inflection generation.
2. The comprehensive evaluation across three datasets (CELEX, Wiktionary, and SIGMORPHON) with different dataset sizes.
3. The insightful analysis of learned alignments and representations that provides valuable insights into how neural models work for morphological inflection.

## Strengths

1. **Comprehensive Evaluation**: The paper provides a thorough evaluation across three well-established morphological inflection datasets with varying sizes. This demonstrates the model's performance across different resource settings, which is valuable for understanding the model's strengths and limitations.

2. **Insightful Analysis**: The analysis of learned alignments and representations (Figures 3 and 4) is particularly strong. It provides valuable insights into how neural models for morphological inflection encode linguistic features, which is important for understanding the models' behavior beyond just performance metrics.

3. **Clear Motivation**: The motivation for the model is well-explained, highlighting the importance of morphological inflection generation for NLP tasks and the limitations of previous approaches, especially with small training sets.

## Weaknesses

1. **Fundamental Misunderstanding of "Hard Attention"**: This is the most critical flaw. The paper misuses the term "hard attention" to describe a model that doesn't learn attention weights but uses pre-determined alignments. In the neural sequence transduction literature, "hard attention" typically refers to a model that selects a single source position at each step (rather than using a weighted sum), but still learns the attention distribution. The authors' model doesn't learn attention at all - it uses externally provided alignments. This is a fundamental misunderstanding that undermines the paper's main contribution. The paper states: "The hard attention mechanism allows the network to jointly align and transduce while using a focused representation at each step," but this is incorrect. The attention mechanism isn't hard in the traditional sense, and the model doesn't jointly align and transduce - it uses pre-determined alignments.

2. **Inconsistent Data Description**: The paper initially states that the CELEX dataset "includes only 500 training examples for each of the four inflection types," but then describes the data splits as "five folds, each consisting of 500 training, 1000 development and 1000 test examples." This implies that the total training data for each inflection type is 2,500 examples (5 folds × 500 training examples), not 500 as initially stated. This inconsistency is confusing and undermines the paper's credibility.

3. **Formatting and Presentation Issues**: The paper contains multiple formatting inconsistencies that are distracting and unprofessional:
   - Reference list has inconsistent formatting (e.g., "Ahlberg et al.,? 2015, Ahlberg et al.," instead of "Ahlberg et al., 2015").
   - Figure 3 is referenced but not included in the submission.
   - Table 3's caption ("suffixing+stem changes circ. suffixing+agg.+v.h. c.h. templatic") doesn't match the table content.
   - The paper claims implementation details are in the supplementary material, but it's not included in the submission.

4. **Undermined Claim**: The analysis in Section 5 shows that the soft attention model also learns monotonic alignments, which directly undermines the authors' main motivation for their model. The authors state: "First, we notice the alignments found by the soft attention model are also monotonic, supporting our modeling approach for the task." This contradicts their main claim that their model's advantage comes from enforcing monotonic alignment, as both models learn monotonic alignments. The paper doesn't adequately explain why their model performs better if both models learn similar alignments.

5. **Lack of Clear Contribution**: The paper doesn't clearly distinguish its contribution from previous work that uses external alignments. The Neural Transducer model by Jaitly et al. (2015) and the work by Yu et al. (2016) also use external alignments, and the paper doesn't adequately explain how this work differs from those approaches.

## Questions to Authors

1. The paper uses the term "hard attention" throughout, but the model doesn't learn attention weights - it uses pre-determined alignments. How do you respond to the criticism that this is not "hard attention" but rather a model using external alignments? What is the precise technical difference between your approach and the "hard attention" models in the literature?

2. In Table 1, your model achieves 89.44% accuracy on CELEX, while the LAT model achieves 88.3%. The paper claims this is "the first neural model to surpass the performance of the weighted FST model with latent variables specifically tailored for the task by Dreyer et al. (2008)." However, the paper by Rastogi et al. (2016) also used a neural WFST approach. Did you compare your results with Rastogi et al.'s model on the CELEX dataset? If so, why didn't you include these results in the paper?

3. In Figure 2, the learning curves for the soft and hard attention models show similar performance on the training set, but the hard attention model performs better on the dev set. The paper attributes this to the hard attention model's ability to harness monotonic alignments. However, the analysis in Figure 3 shows that the soft attention model also learns monotonic alignments. How do you explain this performance difference if both models learn similar alignments?

4. The paper claims implementation details are in the supplementary material, but it's not included in the submission. Will you be able to provide this supplementary material during the author response period?

5. The paper states "the network can then be trained using straightforward cross-entropy loss." However, the training procedure described in Section 2.3 uses a sequence of step and write actions derived from pre-determined alignments. This is not standard cross-entropy training for sequence generation. How does this differ from standard sequence-to-sequence training with attention?

## Recommendation

I recommend rejection of this paper. While the paper presents a well-motivated approach and provides a thorough evaluation across multiple datasets, the fundamental misunderstanding of the term "hard attention" is a significant methodological flaw that undermines the paper's main contribution. The paper's claims about "hard attention" are incorrect, and the analysis shows that the soft attention model also learns monotonic alignments, which directly contradicts the paper's main motivation.

The formatting inconsistencies and data description issues further diminish the paper's credibility. While the evaluation and analysis are strong, they are overshadowed by these fundamental methodological and presentation flaws. The paper would require substantial revision to correct these issues before it could be considered for publication.
