- Proposes Neural Belief Tracker (NBT) that removes need for hand-crafted semantic lexicons by using pre-trained word vectors.  
- Shows Paragram-SL999 semantic-specialized embeddings substantially improve dialogue state tracking without domain-specific lexicons.  
- Demonstrates neural models can handle lexical variation via distributed representations instead of exact matching.  
- Presents a scalable architecture that avoids manual semantic dictionary creation for new domains.  
- Empirical results: NBT matches lexicon-enhanced baselines and outperforms them when lexicons are unavailable (e.g., +10.9% on WOZ 2.0).  
- Analysis shows semantic-specialized vectors are crucial, with Paragram-SL999 beating GloVe and random vectors by up to 8.5% on DSTC2.  
- Clear scalability advantage for real-world deployment due to no domain-specific lexicon requirement.  
- Limited novelty: core technique of using word vectors for semantic matching has prior work in other NLP tasks.  
- Evaluation gap: insufficient assessment of ASR robustness; performance degrades using ASR hypotheses vs. clean transcripts.  
- Baseline comparison limited: does not include recent state-of-the-art models (e.g., transformer-based approaches).  
- Clarification requested: how NBT handles morphological/antonymous variations (e.g., “expensive” vs. “inexpensive”) compared to explicit dictionaries?  
- Clarification requested: what is NBT’s computational complexity versus standard baselines, especially with pre-trained word vectors?
