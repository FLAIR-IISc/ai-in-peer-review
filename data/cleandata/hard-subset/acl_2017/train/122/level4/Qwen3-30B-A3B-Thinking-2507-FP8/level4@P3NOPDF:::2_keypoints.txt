- Proposes neural framework for DST using pre-trained word vectors and a binary decision network to predict slot-values.  
- Evaluates two encoders (NBT-DNN, NBT-CNN) and shows strong gains over a delexicalized baseline on DSTC2.  
- Clear goal, reasonable methodology, and empirically validated improvements—solid technical contribution.  
- Poor organization: DST definition is buried and should precede technical details; introduction confuses readers by discussing dialogue system architecture before defining DST.  
- Recommend reordering: include a high-level dialogue system diagram in Section 1 and a precise DST problem statement immediately after.  
- Notation is inconsistent (symbols introduced before definition); add a dedicated early notation section.  
- Terminology issue: calling the model "joint" is misleading if it only handles DST and not SLU; clarify scope or rename.  
- Baseline comparison lacks context about DSTC2 dataset preprocessing/corrections; explicitly compare using the same dataset versions.  
- Model choice needs justification: explain why DNN/CNN were chosen over sequential models (RNNs/LSTMs) commonly effective in SLU.  
- Provide statistics/costs for the baseline’s semantic dictionary (size, manual effort) to assess scalability and practicality.  
- Include error analysis with concrete examples where the proposed model corrects baseline failures to show why representation learning helps.  
- Overall: technically sound but presentation and missing contextual details obscure contribution; substantial revisions required before publication.
