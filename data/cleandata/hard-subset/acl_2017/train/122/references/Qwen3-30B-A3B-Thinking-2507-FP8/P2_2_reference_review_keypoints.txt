- Proposes Neural Belief Tracker (NBT-DNN, NBT-CNN) using pre-trained word vectors to replace hand-crafted semantic dictionaries for dialogue state tracking.  
- Claims a joint SLU/DST framework that avoids manual lexicons and improves practical scalability.  
- Shows semantically specialized Paragram-SL999 vectors significantly boost DST performance over GloVe and random vectors.  
- Clearly motivates the scalability bottleneck caused by manual semantic dictionaries.  
- Strong empirical evaluation on DSTC2 (with ASR errors) and WOZ 2.0 (clean text) with statistically significant results.  
- Ablation study demonstrates that word vector quality materially affects DST accuracy.  
- Provides sufficient implementation details and clear architecture diagrams for reproducibility.  
- Justifies design choices (element-wise multiplication, cumulative n-gram representations) convincingly.  
- Core approach has limited novelty—combines existing techniques (word vectors, n-grams, CNNs) in a straightforward way.  
- Evaluation is restricted to a simple restaurant domain with only three informable slots, limiting claims of scalability.  
- Does not compare against more recent state-of-the-art DST models, weakening baseline claims.  
- Lacks analysis of failure cases or specific error types the model struggles with.  
- Omits discussion of computational efficiency and runtime/resource trade-offs for real-world deployment.  
- Asks authors to clarify novelty relative to prior embedding-based DST work (e.g., Mesnil 2015, Sun 2016).  
- Requests clarification on whether reported baselines already include semantic dictionaries or if "baseline + sem. dict" is a different system.  
- Seeks insight on how NBT would scale to domains with many more slots/values (e.g., 50+ slots).  
- Overall recommendation: accept—important practical contribution with convincing results despite limited novelty and evaluation scope.
