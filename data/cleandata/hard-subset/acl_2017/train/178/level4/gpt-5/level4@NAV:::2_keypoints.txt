- Summary: Proposes a framework embedding words, atomic phrases, and concepts using an ontology-linked skip-gram approach, evaluated on UMLS and YAGO for concept similarity/relatedness.

- Strength: Addresses lexically dissimilar but semantically close phrases and non-compositionality, proposing a plausible phrase embedding model.

- Strength: Resulting embeddings are competitive or superior on some concept similarity tasks.

- Strength: Released software could be useful to biomedical NLP researchers.

- Weakness: Limited novelty — the model is essentially a small modification of skip-gram.

- Weakness: The full model often underperforms simpler baselines (e.g., Choi) on Mayo datasets and is only competitive with Chiu on UMNSRS.

- Concern: The term “distant supervision” is misleading — linking phrases to concepts provides context but not supervision in the usual predictive-task sense.

- Concern: Notation introduced in Section 3.2 (e.g., E_W) is unused elsewhere and unnecessary.

- Concern: Using a single global β to control phrase compositionality is surprising and likely inappropriate; some reported β values (zero) contradict compositionality assumptions.

- Clarification needed: Experimental setup for Table 4 is unclear — how are similarities computed when concept–phrase mapping is many-to-many, and are concept vectors the only viable option?

- Critique: The approx_phr method (averaging phrase vectors) performs best, which questions the added value of using the concept ontology versus starting from a seed set of phrases.
