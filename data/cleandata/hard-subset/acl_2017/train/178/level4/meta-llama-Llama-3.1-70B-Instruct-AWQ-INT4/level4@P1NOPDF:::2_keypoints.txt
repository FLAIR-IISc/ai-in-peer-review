- Proposes embedding words, phrases, and ontology concepts via mapping phrases to concepts and a skip-gram–style training, evaluated on UMLS and Yago concept similarity/relatedness.
- Addresses the problem that semantically close phrases may lack lexical similarity and non-compositional phrases need special treatment.
- Presents a plausible model for phrase embeddings and releases software useful for biomedical NLP researchers.
- Model lacks novelty—essentially a tweak to the skip-gram method.
- The full proposed model is not the top performer in results (Table 4); Choi baseline substantially outperforms it on the two Mayo datasets.
- Similar performance issues appear in Table 6; on the larger UMNSRS dataset the model is at best competitive with simpler models (Chiu).
- Calling the use of known phrases “distant supervision” is misleading, as occurrences only provide context rather than true supervised labels.
- Notation introduced in Section 3.2 (e.g., E_W) is not used elsewhere in the paper.
- Using a single global β to control phrase compositionality is surprising and likely unrealistic.
- Cross-validated β values are odd (β=0 for PM+CL and WikiNYT), which effectively argues against compositionality.
- Experimental procedure for Table 4 is unclear—uncertainty about how phrase/word vectors are used when concept↔phrase mapping is many-to-many.
- Table 5 shows the “approximate phr” method (averaging phrase vectors) performs best, calling into question the need for the ontology versus just using a seed set of phrases.
