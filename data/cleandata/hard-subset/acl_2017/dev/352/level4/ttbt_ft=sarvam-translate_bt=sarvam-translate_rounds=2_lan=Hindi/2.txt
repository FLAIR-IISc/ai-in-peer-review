**Paper Summary**

This paper presents a method for learning well-divided shared and task-specific feature spaces for an LSTM text classifier. Multi-class adversarial training encourages shared space representation so that a differential classifier cannot identify the source of the task (and thus generalize). The model evaluation is full-shared, shared-private, and adversarial shared-private - the latter ASP model is one of the main contributions. They also use orthogonality constraints to help reward shared and private spaces that are different. The ASP model has a lower error rate compared to single-task and other multi-task neural models. They also experiment with a task-level cross-validation to find out whether shared representation can be transferred to tasks, and it seems to be adapted accordingly. Finally, there is some analysis of shared layer activations that suggest the ASP model is not being misled by strong weights learned on a specific (inappropriate) task.

**Review Summary**

Good ideas, well-expressed, and tested. Some minor comments.

**Strengths**

* It is a good set of well-working ideas. I particularly like the focus on explicitly trying to create a useful shared representation. These have been quite successful in the CV community, but it seems they need a lot of work to be built into NLP.
* Sections 2, 3, and 4 are very clearly expressed.
* Section 5.5 has a good way to evaluate task-level cross-validation transfer.
* There is an implementation and data.

**Weaknesses**

Especially to highlight the benefit of adversarial training between SP-MLT and ASP-MLT results. **General Discussion**
