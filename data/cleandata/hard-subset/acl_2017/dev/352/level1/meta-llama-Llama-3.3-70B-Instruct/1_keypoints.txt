- Proposes an adversarial multi-task learning framework to learn task-invariant shared features while preventing task-specific contamination.  
- Introduces orthogonality constraints to reduce redundant latent representations and separate shared vs. private encoders.  
- Extends binary adversarial training to multi-class settings for joint training of multiple tasks.  
- Demonstrates adversarial training effectively learns task-invariant features and boosts multi-task learning.  
- Reports improved performance over several baselines across 16 text classification tasks.  
- Provides qualitative analysis and visualizations of learned features for additional insight.  
- Model is complex, combining adversarial training, orthogonality constraints, and multi-task elements, making training and tuning challenging.  
- Limited interpretation of how learned features map to specific tasks and datasets.  
- Insufficient comparison to other state-of-the-art multi-task learning models.  
- Asks how the approach can be extended to other NLP tasks (e.g., sequence labeling, MT).  
- Requests more detailed analysis linking learned features to particular tasks/datasets.  
- Queries how authors will reduce model complexity and improve training/tuning efficiency.
