- Proposes a new evaluation framework for reading comprehension datasets based on two metric classes: prerequisite skills and readability.  
- Refines previous prerequisite-skill taxonomies using insights from psychology of human text comprehension.  
- Annotates six existing RC datasets with these metrics and analyzes dataset characteristics and inter-metric correlations.

Main contributions:
- Introduces a combined prerequisite-skills and readability metric set for RC dataset evaluation.  
- Extends and refines Sugawara et al.'s prerequisite skills to be more comprehensive and psychologically grounded.  
- Demonstrates the metrics by annotating six RC datasets and reporting dataset-level analyses.

Strengths:
- Provides a comprehensive, multi-faceted analysis framework to expose dataset strengths and weaknesses.  
- Improves prior skill taxonomies, increasing applicability across diverse RC datasets.  
- Offers practical guidance useful for new dataset construction and RC system development.

Weaknesses:
- Annotation is time-consuming and relies on human annotators, limiting scalability.  
- Lacks clear guidance on how to apply the metrics to develop or evaluate RC systems in practice.  
- Does not deeply analyze the relationship between prerequisite skills and readability metrics.

Questions to authors:
- How should researchers select or combine the proposed metrics to guide RC system development and evaluation?  
- How will you address scalability of annotation (e.g., automated tools or crowdsourcing)?  
- Can you provide deeper analysis or insights on how prerequisite skills and readability interact and influence task difficulty?
