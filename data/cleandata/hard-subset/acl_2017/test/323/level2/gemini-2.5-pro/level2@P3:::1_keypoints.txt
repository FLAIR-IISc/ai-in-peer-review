- Summary: Proposes a CNN-based neural architecture over distributed entity-grid role embeddings to model local text coherence, trained end-to-end with pairwise ranking and evaluated on discrimination, insertion, and summary rating tasks with state-of-the-art results.
- Contribution 1: Introduces a CNN applied to the entity grid to overcome data sparsity and task-agnostic feature engineering via dense representations and end-to-end learning.
- Contribution 2: Provides a strong, comprehensive empirical evaluation showing consistent, significant improvements over non-neural baselines across multiple benchmarks.
- Strength 1: Elegant, well-motivated model that naturally captures longer-range entity transitions by learning salient transition patterns with convolutional filters.
- Strength 2: Convincing empirical gains (e.g., ~4% absolute on discrimination) with the Extended Grid-CNN outperforming non-neural counterparts using fewer entity-specific features.
- Strength 3: Sound, reproducible methodology—large standard dataset (WSJ), strong baselines, hyperparameter details, averaged runs, pretraining for small tasks, and code release commitment.
- Weakness 1: Limited qualitative analysis of learned embeddings and convolutional filter patterns, leaving model interpretability and insights into learned features lacking.
- Weakness 2: Incomplete comparison to other neural coherence models (e.g., Li & Hovy 2014) due to reproduction issues, leaving gaps in positioning relative to contemporaneous neural approaches.
- Weakness 3: Simple feature integration by creating composite vocabulary symbols (e.g., PERSON-S) inflates vocabulary and may not generalize; separate embeddings for each feature type could be more scalable.
- Question 1: Was the full set of nine entity-specific features tested with the neural model, and if so, how did it affect performance?
- Question 2: Was the reverse pretraining/fine-tuning experimented with (e.g., pretrain on discrimination then fine-tune on insertion), especially given insertion’s higher difficulty?
