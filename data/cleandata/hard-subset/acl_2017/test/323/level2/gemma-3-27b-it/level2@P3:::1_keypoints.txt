- Proposal: CNN-based neural local coherence model operating on entity grids with distributed representations and end-to-end pairwise ranking training.  
- Primary contribution: successful application of a neural architecture to entity grids yielding improved performance across coherence tasks.  
- Secondary contribution: demonstration that end-to-end training learns task-specific features within the entity-grid framework.  
- Strong empirical results: achieves state-of-the-art on discrimination, insertion, and summary coherence rating tasks (notably >4% gain on summary rating).  
- Addresses limitations: replaces discrete representations and decoupled feature computation with distributed embeddings and joint training.  
- Clear motivation and presentation: ties to Centering Theory and provides a helpful model diagram.  
- Code release: authors provide code to support reproducibility.  
- Limited ablation study: lacks systematic analysis of convolution filter sizes, pooling strategies, and per-component contributions.  
- Sparse hyperparameter details: brief description of tuning, unclear search space and rationale for chosen settings.  
- Narrow baselines: primarily compares to entity grid methods, lacks comparison to non-grid neural models (e.g., transformers).  
- Dataset limitation: summary coherence rating dataset is small, so gains should be interpreted cautiously.  
- Question on hyperparameters: request for elaboration on filter sizes and pooling choices.  
- Question on pretraining: ask for the magnitude of pretraining gains and dependence on pretraining dataset size.  
- Question on entity features: inquire whether other entity-specific features were tried and their outcomes.  
- Question on generalization: request performance results on more complex or diverse writing styles.  
- Notation issue: Equation 5 needs clarification.  
- Complexity discussion: suggest adding computational complexity comparison with traditional entity-grid methods.  
- Bibliography error: incomplete reference for "Barzilay and Lapata.,? 2005".
