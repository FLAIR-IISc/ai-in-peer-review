- Proposes a CNN-based neural model over entity-grid representations to assess local text coherence, capturing long-range entity transitions.
- Uses a pairwise ranking approach for end-to-end training and evaluates on discrimination, insertion, and summary coherence rating tasks with released code.
- Neuralizes the entity grid to use distributed representations and model longer transitions without overfitting.
- Learns task-specific high-level features via end-to-end training instead of using task-agnostic discrete features.
- Demonstrates strong empirical validation with state-of-the-art results across three benchmark coherence tasks.
- Novel contribution addressing dimensionality and feature-extraction limitations of traditional entity-grid models.
- Consistent, statistically significant performance improvements over non-neural baselines, validated across multiple runs.
- Code and hyperparameter release improve reproducibility.
- Comprehensive evaluation across sentence ordering and summary coherence tasks shows model versatility.
- Incorporates some entity-specific features (e.g., named entity types) to enhance real-world applicability.
- Does not include the full set of entity-specific features from prior work, potentially limiting performance in some scenarios.
- Model is not directly trained for the insertion task, which may explain smaller gains there.
- Evaluation is focused mainly on the WSJ corpus, raising questions about generalizability to other domains or genres.
- Summary coherence rating relies on pretraining from discrimination, which may be infeasible in low-resource settings.
- Could performance on insertion improve with task-specific training rather than discrimination pretraining?
- Have the authors tested generalization to more diverse datasets (e.g., conversational or informal text)?
- How does the model perform when entity-specific features are sparse or missing, and does that materially affect results?
- Overall: innovative, well-validated neuralized entity-grid model with SOTA results and minor limitations; recommended for acceptance.
