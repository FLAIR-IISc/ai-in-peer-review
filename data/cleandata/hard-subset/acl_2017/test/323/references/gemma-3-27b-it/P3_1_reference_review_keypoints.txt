- Summary: Proposes a neural network version of the entity grid using distributed representations and end-to-end training, improving coherence evaluation on discrimination, insertion, and summary-rating tasks.

- Contribution 1: Neuralizes the entity grid to learn richer distributed representations of entity transitions instead of discrete features.

- Contribution 2: Uses end-to-end training to automatically learn task-specific coherence features.

- Contribution 3: Demonstrates state-of-the-art empirical results across three standard coherence tasks.

- Strength 1: Clearly motivates the problem and links to coherence theory (Centering Theory).

- Strength 2: Methodology is sound; CNN with wide convolutions and max-pooling is a logical choice for feature extraction.

- Strength 3: Comprehensive evaluation on multiple tasks with standard WSJ dataset and baselines.

- Strength 4: Code release enhances reproducibility.

- Strength 5: Consistent performance gains, notably on the challenging insertion task.

- Weakness 1: Neural architecture (CNN + max-pooling) lacks novelty; novelty is mainly in application to entity grids.

- Weakness 2: Entity-specific feature integration is simplistic (concatenation) and could use more advanced mechanisms.

- Weakness 3: Ablation studies are limited; more analysis of embedding size, filter sizes, pooling, etc., is needed.

- Weakness 4: Missing qualitative error analysis to explain failure cases and limitations.

- Question 1: Why choose CNNs over RNNs or Transformers, and were alternatives tried?

- Question 2: Were more sophisticated entity feature integration methods (attention, gating) experimented with?

- Question 3: Can the authors provide examples and analysis of cases where the model fails?

- Question 4: How much did pre-training on WSJ improve summary-rating performance and how does performance scale with pre-training data size?

- Additional comment 1: Related-work discussion is solid but could mention recent transformer-based discourse work.

- Additional comment 2: Paper is generally well-written and easy to follow.

- Overall assessment: Strong, well-motivated contribution with effective execution and state-of-the-art results; recommend acceptance.
