### **Summary and Contributions**

This paper presents a neural network model for assessing local text coherence, designed as a direct successor to the influential entity grid model. The core of the proposed method is a convolutional neural network (CNN) that operates not on raw text, but on an entity grid where grammatical roles are represented as distributed vectors (embeddings). This architecture is trained end-to-end using a pairwise ranking objective to distinguish more coherent texts from less coherent ones.

I see the main contributions of this work as follows:
1.  **A Novel Neural Architecture for a Classic Representation:** The primary contribution is the successful "neuralization" of the entity grid. By applying embeddings and a CNN to this structured representation, the authors provide a principled way to overcome the well-known limitations of earlier discrete models, namely the curse of dimensionality that prevents modeling long-range entity transitions and the reliance on hand-crafted, task-agnostic feature engineering.
2.  **Strong Empirical Validation and State-of-the-Art Results:** The paper provides a thorough empirical evaluation on standard coherence tasks (discrimination, insertion, summary rating) using a large, appropriate dataset (WSJ). The proposed model significantly outperforms its direct non-neural counterparts and achieves state-of-the-art results, demonstrating the practical effectiveness of the neural approach.

### **Strengths**

1.  **Elegant and Well-Motivated Model:** The paper does an excellent job of motivating the work by clearly articulating the limitations of existing entity grid models. The proposed neural architecture is a direct and elegant solution to these problems. Using embeddings to create dense representations of roles and a CNN to learn location-invariant transition patterns is a natural and powerful combination that is perfectly suited to the task. The model design is simple, yet highly effective.

2.  **Rigorous and Convincing Evaluation:** The experimental setup is exemplary. The authors use standard, well-understood evaluation tasks and a large-scale dataset (WSJ) that is more general than the domain-specific corpora used in some prior work. The choice of baselines is excellent, particularly the inclusion of the direct non-neural counterparts (`Grid-all nouns` and `Extended Grid`), which allows for a clean ablation of the contribution of the neural components. The reporting of results, including hyperparameter tuning on a development set and averaging over multiple runs, adheres to best practices.

3.  **Significant and Consistent Performance Gains:** The model achieves substantial and statistically significant improvements across all three evaluation tasks. The ~4% absolute gain in the discrimination task over a strong baseline is impressive. Crucially, the `Extended Grid-CNN` achieves its superior performance while using only a subset of the entity-specific features from the `Extended Grid` model, which strongly supports the authors' claim that the neural model learns more effective, task-specific feature representations.

4.  **Clarity and Reproducibility:** The paper is exceptionally well-written, with clear explanations of both the prior work and the proposed methodology. Figure 2 provides a helpful visualization of the architecture and training process. The authors' decision to release their code is a significant contribution that strengthens the paper and will benefit the research community.

### **Weaknesses**

While this is a very strong paper, there are a few areas where the work could be extended or clarified.

1.  **Simplicity of Entity-Specific Feature Integration:** The method for incorporating entity-specific features—by creating new atomic symbols in the vocabulary (e.g., `PERSON-S`)—is simple but perhaps not optimal. This approach results in a larger vocabulary and does not explicitly allow the model to learn separate representations for grammatical roles and entity features. An alternative, potentially more powerful approach would be to concatenate the embedding for the grammatical role with embeddings for each feature type (e.g., `emb(S) || emb(PERSON) || emb(salient)`). This would allow for more compositional learning of feature interactions.

2.  **Inherited Independence Assumption:** The model inherits the core assumption from the entity grid that entity transition columns are independent. While this is a standard simplification, it ignores potential interactions between entities (e.g., a subject-to-subject transition for the main character might make an object-to-other transition for a secondary character more likely). A neural framework seems uniquely positioned to potentially model such cross-entity dependencies, for example by using 2D convolutions over the grid, which could be an interesting direction for future work.

3.  **Lack of Qualitative Analysis of Learned Representations:** The paper convincingly shows *that* the model works, but provides limited insight into *why* it works beyond high-level intuition. An analysis of the learned components would strengthen the paper. For instance, visualizing the embeddings could show whether roles like 'S' and 'O' are learned to be closer than 'S' and '-'. More importantly, analyzing the learned convolutional filters to identify the specific n-gram transition patterns they become sensitive to would provide concrete evidence of what coherence patterns the model has captured, moving beyond the black-box nature of the results.

### **Questions to Authors**

1.  Regarding the integration of entity-specific features: Did you experiment with alternative methods, such as concatenating separate feature embeddings for roles and entity types, instead of creating combined symbols? It seems this could offer more compositional power and prevent vocabulary explosion if many features were used.

2.  The reported performance for the Li and Hovy (2014) model is surprisingly low (19.34% accuracy). I appreciate the transparency in reporting this difficulty. Do you have any hypotheses for this result? Does it suggest that for these tasks, the explicit, syntax-aware structure of the entity grid provides a much stronger inductive bias than models operating on distributed sentence representations alone?

3.  Could you provide any qualitative analysis of what the model has learned? For example, are there particular entity transition patterns (e.g., `S -> S -> O`, `S -> - -> S`) that the convolutional filters learn to fire on most strongly? This would provide valuable insight into the linguistic phenomena captured by your model.
