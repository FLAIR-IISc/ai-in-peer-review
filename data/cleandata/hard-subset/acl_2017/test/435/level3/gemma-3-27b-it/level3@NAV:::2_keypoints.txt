- Paper addresses identifying causal vs. non-causal clause relations without explicit connectives using the AltLex dataset and an LSTM-based model.  
- Main contribution: empirical validation that LSTMs outperform CNNs for implicit causality detection on AltLex, providing a new benchmark.  
- Secondary contribution: detailed experimental analysis with examples showing model generalization.  
- Strength: clear problem statement motivating the need to detect implicit causality beyond explicit connectives.  
- Strength: empirical improvements over baseline on AltLex (modest 0.5–1.5% F1 gains) that appear plausible.  
- Strength: detailed analysis of model behavior through illustrative examples.  
- Strength: well-written and well-organized presentation.  
- Weakness: potential overfitting because the best configuration was reportedly selected using test set performance instead of a dev set.  
- Weakness: absence of statistical significance testing limits confidence in claimed improvements given small dataset size.  
- Weakness: bootstrapping procedure to extend the corpus is undocumented, raising concerns about data quality and bias.  
- Weakness: unclear dataset access/redistribution rights and an unspecified coding scheme for TSV files.  
- Weakness: limited baselines—lack of simpler LSTM variants or other ablations to justify model complexity.  
- Question: request details of the bootstrapping procedure, selection criteria, and label assignment.  
- Question: whether a development set was used for hyperparameter tuning or if model selection was done on the test set; ask for dev-set results if applicable.  
- Question: clarification of the TSV coding scheme (0-1-2) and confirmation of rights to redistribute the dataset.  
- Question: request for statistical significance testing (e.g., paired t-test) to support superiority claims.  
- Question: why the model favors precision over recall and analysis of error types causing this imbalance.
