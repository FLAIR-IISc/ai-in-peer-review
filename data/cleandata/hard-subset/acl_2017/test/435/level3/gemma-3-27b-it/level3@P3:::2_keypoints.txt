- Empirically shows LSTM-based architectures outperform CNNs for detecting implicit causal clause relations on the AltLex dataset (modest 0.5–1.5% F1 gain).  
- Provides a solid baseline and comprehensive experimental analysis for the AltLex dataset, including qualitative examples.  
- Clearly defines the task of causal meaning disambiguation and motivates the need to go beyond explicit connectives.

- Paper is well-written, clearly motivated, and technically easy to follow.  
- Strong motivation demonstrating limitations of explicit-connective and lexical-feature approaches for implicit causality.  
- Thorough experimental evaluation exploring multiple LSTM configurations and both bootstrapped and non-bootstrapped AltLex variants.  
- Useful qualitative analysis showing examples where the proposed model succeeds and prior work fails.  
- Authors identify and discuss dataset challenges such as class imbalance and train/test differences.

- Model selection appears to have been done using the test set, risking overfitting and optimistic results.  
- No statistical significance testing reported to support the claimed performance gains.  
- Concerns about annotation quality and noise/bias from English and Simple English Wikipedia sources.  
- Novelty is incremental since LSTMs’ superiority for sequential tasks is expected.  
- Bootstrapping procedure for the expanded AltLex corpus is not described, leaving its validity unclear.

- Was the final configuration chosen using a held-out development set or the test set; if the latter, please re-evaluate with proper dev validation.  
- Please provide details on the bootstrapping procedure: selection criteria and quality assessment of bootstrapped instances.  
- Why were simpler baselines (e.g., single-layer LSTM) not included in experiments?  
- Can you analyze why the best model favors precision over recall and whether dataset biases explain this?  
- Are code and data available, and what does the 0-1-2 coding scheme in the TSV files represent; any redistribution restrictions?

- Reference list appears incomplete with missing information for several citations.  
- Suggest exploring attention mechanisms within the LSTM to help focus on relevant input spans and potentially improve performance.
