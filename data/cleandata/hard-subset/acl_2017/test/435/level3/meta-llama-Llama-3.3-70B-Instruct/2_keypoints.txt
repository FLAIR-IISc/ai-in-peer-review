- Proposes an LSTM-based neural method for clause-level causal relation detection, improving previous SVM-based results by 0.5–1.5% F1.  
- Introduces a novel, larger, and more challenging dataset for causal relation detection.  
- Demonstrates the method’s effectiveness on both monolingual and cross-lingual datasets.  
- Achieves state-of-the-art performance, outperforming prior methods.  
- Provides thorough evaluation across multiple datasets with baseline comparisons.  
- Releases code and data for reproducibility.  
- Fails to clearly distinguish this task from implicit connective recognition or explain why prior methods are inapplicable.  
- Bootstrapping used to extend the corpus is not clearly explained; corpus construction needs more detail.  
- Best model configuration was selected on the test set rather than the dev set, and reported improvements may not be statistically significant.  
- Gold-standard annotations sourced from English/Simple English Wikipedia may be unreliable, potentially affecting results.  
- Request: provide detailed description of the bootstrapping process used to extend the corpus.  
- Request: explain how gold-standard annotation quality was ensured and steps taken to address errors or inconsistencies.  
- Request: offer more analysis of performance across types of causal relations, especially implicit versus explicit relations.
