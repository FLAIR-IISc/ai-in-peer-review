- Task: Causal meaning disambiguation using an LSTM to encode context for lexical markers (e.g., "because," "since") without relying on lexical resources.
- Contribution: Shows a neural LSTM model can outperform prior feature-based systems for this task.
- Contribution: Provides analysis of ambiguous markers, demonstrating context-dependent causal vs. temporal interpretations.
- Contribution: First to apply the AltLex corpus for neural causality disambiguation.
- Strength: Neural model beats Hidey & McKeown (2016) by 2.13% F1 on the Causal class and correctly handles some ambiguous markers the baseline misclassifies.
- Strength: Clear evidence (figures/tables) that context matters for disambiguating markers like "since" and "after."
- Strength: Addresses a relatively neglected task with practical relevance to discourse analysis and event extraction.
- Weakness: Novelty is limited—approach is similar to prior neural work (e.g., Kruengkrai et al. 2017, Melamud et al. 2016) and architecture innovations are not clearly justified.
- Weakness: Dataset size (AltLex ~8–9k) is small for neural methods; no data augmentation or transfer learning explored.
- Weakness: Evaluation is narrow (two versions of same corpus, no cross-dataset tests or cross-validation), limiting generalizability.
- Weakness: No thorough ablation studies to attribute gains to specific model components (LSTM vs dense layers, etc.).
- Question: How does this method compare to recent neural baselines (e.g., BERT) and what are the concrete differences/improvements?
- Question: How was class imbalance handled in AltLex, and how would results change with a more balanced dataset?
- Question: How does the model handle out-of-vocabulary markers (e.g., "which") and how would such cases be addressed in production?
- Overall assessment: Reasonable contribution demonstrating neural context encoding helps, but novelty and evaluation weaknesses limit significance.
- Recommendation: Weak/marginal accept contingent on stronger comparisons to recent neural methods, larger/more diverse evaluation, and deeper analysis (ablation).
