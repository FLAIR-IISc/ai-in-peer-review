- Task: Disambiguating whether lexical markers (e.g., "since," "after") signal causal relations by modeling context rather than relying on syntax or lexical lists.  
- Claim: Causality is context-dependent, so disambiguation should be treated as a contextual problem, not a syntactic/lexical-resource problem.  
- Method: Proposes an LSTM-based neural architecture ("Pair LSTM") that encodes context and avoids reliance on lexical/syntactic features.  
- Contribution 1: Strong theoretical argument challenging prior work that assumes fixed lexical/syntactic causality.  
- Contribution 2: Neural model that improves precision and outperforms state-of-the-art baselines, especially reducing false positives.  
- Contribution 3: Thorough evaluation on the AltLex corpus, including ablations and analysis of ambiguous markers.  
- Strength: Well-supported theoretical foundation citing prior work to argue causality is not syntactically fixed.  
- Strength: Comprehensive experimental evaluation showing consistent neural improvements and a best configuration ("Pair LSTM Mean Adam") with higher precision.  
- Strength: Practical examples demonstrating that context encoding resolves ambiguities that prior models (B2) miss.  
- Weakness: Fails to discuss limitations like multi-sentence causality, long-context handling, and possible memorization vs. genuine learning.  
- Weakness: Corpus and scalability concerns — AltLex is small and bootstrapped instances may break train/test distributions and generalization to unseen markers.  
- Weakness: Insufficient analysis of what contextual features the model learns and weak evidence that it generalizes beyond marker class distributions.  
- Question 1: How to handle precision-recall trade-offs when recall is more critical in applications like information extraction?  
- Question 2: Can the model handle implicit causality with no lexical markers, or would architecture changes be needed?  
- Question 3: Please provide more examples where the model correctly handles markers that are mostly causative in training but non-causal in testing.  
- Overall: Valuable contribution demonstrating context importance, but needs stronger analysis of model behavior and clearer limitations; recommend acceptance with major revision.
