- Paper proposes LSTM-based classifier (GloVe embeddings → LSTM → 3 dense tanh layers → softmax) to detect causal vs non-causal clause relations without explicit connectives.  
- Uses AltLex dataset (Hidey & McKeown 2016) to identify implicit causal relations; addresses cases where connectives are absent or ambiguous.  
- Authors claim LSTMs outperform CNNs for representing clause-level features.  
- Best system configuration improves over Hidey & McKeown’s SVM by ~0.5–1.5% F1; some qualitative examples show correct generalization.  
- Paper is well written and problem description is clear.  
- Needs clarification on how this task differs from implicit connective recognition and why existing methods aren’t directly applicable.  
- Code provision is commendable, and dataset upload is helpful, but the 0-1-2 TSV coding is unexplained.  
- Must clarify whether the authors have rights to repost Hidey & McKeown’s data.  
- The “bootstrapping” step that expands the corpus by ~15% is insufficiently described.  
- Experimental methodology flaw: best configuration was chosen on the test set rather than a development set, inflating reported improvements.  
- Statistical significance of the reported gains is not provided and should be included given dataset size.  
- Concerns about gold-standard reliability (English/SimpleEnglish Wikipedia annotations) mean scores should be interpreted cautiously.  
- Novelty is limited—neural methods outperforming feature-based classifiers for binary tasks is expected; include simpler baselines (e.g., single-layer LSTM) and analyze precision vs recall.
