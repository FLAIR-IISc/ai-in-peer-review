- Proposes an LSTM-based neural approach to detect causal relations between clauses without explicit discourse connectives.  
- Leverages the AltLex dataset (Hidey & McKeown, 2016) as the primary resource for training and evaluation.  
- Model architecture: GloVe embeddings → LSTM layer → three dense tanh layers → softmax classification.  
- Reports modest F1 improvements over the Hidey & McKeown SVM baseline (0.5–1.5%), with a best-case 2.13% gain.  
- Includes useful qualitative analysis showing generalization beyond training indicator words.  
- Writing and problem description are clear and well-presented.  
- Needs clearer distinction from implicit connective recognition and explanation why those methods aren’t directly applicable.  
- Code and dataset release is appreciated, but the 0-1-2 TSV coding scheme needs clarification relative to the stated binary task.  
- Authors should verify and explicitly state redistribution rights for the AltLex dataset.  
- The bootstrapping procedure that expands the corpus (~15%) is under-described and requires more implementation detail.  
- Reported improvement’s significance is unclear because best configuration appears selected on the test set; should use a development set and report held-out test performance.  
- Statistical significance testing and discussion of annotation noise (e.g., Wikipedia-derived errors) and inter-annotator agreement are missing.  
- Results feel confirmatory rather than novel; experiments with simpler baselines (e.g., single-layer LSTM) are recommended.  
- Analysis should explore why the system favors precision over recall and investigate the precision/recall tradeoff.
