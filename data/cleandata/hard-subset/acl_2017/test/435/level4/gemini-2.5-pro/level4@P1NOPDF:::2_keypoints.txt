- Paper proposes an LSTM-based neural method (GloVe embeddings → LSTM → 3 dense tanh layers → softmax) to detect causal vs. non-causal clause relations, focusing on unmarked cases using the AltLex dataset.  
- Authors argue LSTMs are better than CNNs for clause-level features and report modest F1 gains over the SVM baseline (≈0.5–2.13%).  
- Includes both quantitative results and a useful qualitative analysis showing some generalization (e.g., correctly reclassifying indicator words).  
- Paper is well written with a clear problem description.  
- Needs clarification on how this task differs from implicit discourse relation classification and why prior methods may not apply.  
- Reproducibility concern: TSV data use a 0-1-2 coding scheme that conflicts with the described binary task and needs explanation.  
- Data rights: must clarify whether the authors are allowed to redistribute Hidey and McKeown (2016) data.  
- Bootstrapping: the reported ~15% corpus extension via bootstrapping is insufficiently described and should be detailed.  
- Evaluation concern: best model appears to be selected using the test set rather than the development set, risking overfitting and unfair comparison.  
- Statistical significance of the reported improvement is not tested and should be provided given dataset size.  
- Annotation reliability/noise: inter-annotator agreement and dataset noise (from Wiki/SimpleWiki) should be discussed and cited (AltLex paper).  
- Novelty concern: neural outperforming hand-crafted features is expected; authors should include simpler neural baselines (e.g., single-layer LSTM) to pinpoint source of gains.  
- Analysis suggestion: investigate why the model favors precision over recall to better understand behavior.
