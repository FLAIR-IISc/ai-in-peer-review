- Proposes an LSTM-based neural model (GloVe embeddings → LSTM → three tanh dense layers → softmax) for binary clause-level causal relation detection without explicit connectives.  
- Uses the AltLex dataset (Hidey and McKeown, 2016) for the causal vs. non-causal classification task.  
- The network is not particularly deep despite being labeled “deep learning.”  
- Best configuration yields a small 0.5–1.5% F1 improvement over Hidey and McKeown’s SVM baseline.  
- Provides useful qualitative analysis with examples showing the model generalizes (e.g., identifying non-causal uses of indicator words).  
- Paper is well-written with a clear problem description.  
- Needs clarification on how this task differs from implicit connective recognition and why prior methods don’t apply.  
- Code and data are included, but the 0-1-2 coding in TSV files is unexplained.  
- Authors must confirm they have rights to redistribute Hidey and McKeown (2016) data.  
- The bootstrapping procedure that increases corpus size by ~15% is mentioned but not described.  
- Experimental methodology appears flawed: model selection/hyperparameter tuning seems to have been done on the test set rather than the development set.  
- Small F1 improvement’s significance is unclear; statistical significance tests should be reported.  
- Gold labels derived from Wikipedia alignments may be noisy, so small gains should be treated with caution.  
- Contribution is unsurprising given neural models’ general advantages; need simpler neural baselines (e.g., single-layer LSTM) to isolate architecture effects.  
- Recommend deeper analysis of results (e.g., why the model favors precision over recall compared to the baseline).
