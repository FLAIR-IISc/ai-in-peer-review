- Proposes LSTM-based clause causality classifier using GloVe embeddings and AltLex data to detect causal vs. non-causal relations without explicit connectives.
- Authors argue LSTMs outperform CNNs and use GloVe -> LSTM -> three dense tanh layers -> softmax architecture.
- Best system reports modest F1 gains over Hidey & McKeown (0.5–1.5% in one place; 2.13% reported elsewhere).
- Paper shows useful qualitative generalizations, e.g., indicator words labeled non-causal in test despite being causal in training.
- Writing and problem description are clear; qualitative and quantitative analyses are thorough.
- Needs clarification on how this task differs from implicit connective recognition and why prior methods are unsuitable.
- Public code and dataset release is appreciated, but reviewer didn’t execute the code and questions remain about the 0-1-2 TSV coding.
- Unclear whether authors have permission to redistribute the Hidey & McKeown dataset.
- “Bootstrapping” that expands the corpus by ~15% is mentioned but not explained and needs detail.
- Model selection appears to have used the test set (improper); best configuration should be chosen on the dev set.
- Statistical significance of reported improvements is not provided and should be reported given dataset size.
- Contribution is incremental (neural methods already often beat hand-crafted features); include simpler baselines (e.g., 1-layer LSTM) and analyze why the model favors precision over recall.
