- Introduces an LSTM-based neural classifier (GloVe embeddings → LSTM → three tanh dense layers → softmax) for clause-level causal relation detection using the AltLex dataset, claiming small F1 gains (0.5–1.5%) over an SVM baseline.  
- Provides strong qualitative examples showing generalization (e.g., correctly labeling non-causal instances of words seen as causal in training).  
- Paper is well-written with a clear problem description but needs to clarify how this task differs from implicit connective recognition and why existing methods are unsuitable.  
- Authors shared code and dataset, but the meaning of the 0-1-2 TSV coding is unclear and must be explained.  
- Need clarification on legal/ethical right to redistribute Hidey & McKeown data and note that an older dataset version is included.  
- The bootstrapping procedure (adding ~15% to the corpus) is not defined or described and should be explained.  
- Experimental setup concerns: best configuration appears selected on the test set (not dev), undermining fairness; reported 2.13% F1 gain may be overfitted.  
- No statistical significance testing is reported despite dataset size making it feasible; significance should be established.  
- Gold-standard annotations (from English/Simple English Wikipedia) show inconsistencies, raising reliability concerns that affect score interpretation.  
- Contribution is incremental—neural methods outperforming engineered features is expected; authors should compare to simpler baselines (e.g., single-layer LSTM) and analyze why their model favors precision over recall.
