- Proposes an LSTM-based neural model for clause-level causal relation detection without explicit discourse connectives.  
- Uses the AltLex dataset (Hidey & McKeown, 2016) and GloVe embeddings feeding an LSTM, three tanh dense layers, and a softmax output.  
- Reports a modest 0.5–1.5% F1 improvement over the Hidey & McKeown SVM baseline (claimed 2.13% in places).  
- Provides qualitative analysis showing the model can generalize beyond lexical causal indicators.  
- Paper is well-written with a clear problem statement.  
- Must clarify distinction from implicit connective recognition (ICR) and explain why ICR methods cannot be adapted for this binary causal classification.  
- Authors share code and the AltLex dataset, which supports reproducibility.  
- TSV files use 0-1-2 coding that contradicts the paper’s binary classification claim and must be explained.  
- Unclear whether authors have permission to redistribute the AltLex data—this should be explicitly stated.  
- Reported best configuration appears to be selected using the test set (test-set leakage), undermining the validity of results.  
- No statistical significance testing (p-values/confidence intervals) is provided despite small dataset and marginal gains.  
- Gold-standard annotations (English/Simple English Wikipedia) contain inconsistencies, so reported scores should be interpreted cautiously.  
- Novelty is limited because prior work already shows neural methods outperform handcrafted features; contribution is mostly confirmatory.  
- Recommend adding simpler baselines (e.g., single-layer LSTM), investigating why the model favors precision over recall, and clarifying bootstrapping and other technical details before acceptance.
