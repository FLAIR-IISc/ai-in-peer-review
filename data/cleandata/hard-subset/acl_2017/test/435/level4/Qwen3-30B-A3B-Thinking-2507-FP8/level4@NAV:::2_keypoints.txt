- Introduces an LSTM-based neural approach for clause-level causal relation detection using GloVe embeddings and dense layers.  
- Tackles causal vs non-causal detection when no explicit discourse connective is present, using the AltLex dataset.  
- Argues LSTMs capture clause features better than CNNs; model: LSTM → three tanh dense layers → softmax.  
- Best model yields a 0.5–1.5% F1 improvement over Hidey & McKeown (2016) SVM baseline.  
- Demonstrates generalization with examples where training indicator words are correctly classified as non-causal in test.  
- Provides both qualitative and quantitative analyses and a clear problem description.  
- Needs clarification on how this task differs from implicit connective recognition and why prior methods are unsuitable.  
- Code and dataset (including older data) are publicly released — appreciated.  
- TSV files use a 0–1–2 coding scheme that needs explanation given the paper frames a binary task.  
- Must address authors’ rights to repost the AltLex dataset originally from Hidey & McKeown.  
- “Bootstrapping” expands the corpus by ~15% but is not defined or described in the paper.  
- Model selection appears to use test data (choosing best among 4–7 configs) instead of dev set, undermining comparisons.  
- Should report statistical significance for the reported F1 gains given dataset size.  
- Gold-standard annotations from English/Simple English Wikipedia may be imperfect; acknowledge reliability concerns.  
- Improvements largely confirm the known trend that neural nets beat hand-engineered features; novelty limited.  
- Recommend including simpler baselines (e.g., 1-layer LSTM) and analyzing why the model trades recall for precision.
