- Proposes a neural network architecture with two inputs (first event; lexical marker + second event) using LSTM and dense layers to disambiguate causal lexical markers by encoding sentence meaning.
- Evaluates the system on the challenging AltLex corpus and reports improvements over Hidey and McKeown (2016).
- Provides qualitative and quantitative analyses with example generalizations where the system succeeds.
- Clear problem statement and description of the proposed solution.
- Well-organized experiments with comparisons to a baseline (Hidey & McKeown) and result analysis.
- Code and data are released to support reproducibility.
- Paper lacks a clear explanation of the corpus bootstrapping process used to extend data.
- Comparison is limited to a single prior work and omits other baseline methods.
- Improvement over the state of the art is reported without clear statistical significance testing.
- Asks for a clearer description of the bootstrapping procedure.
- Requests details on how the best system configuration was selected and whether a development set was used.
- Seeks more analysis of gold-standard annotation reliability and its impact on results.
- Suggests evaluating simpler network baselines (e.g., 1-layer LSTM) to justify the proposed architecture.
