### Summary and Contributions
This paper proposes a neural sequence modeling approach for Japanese Predicate Argument Structure (PAS) analysis, with a focus on resolving omitted arguments (zero arguments). The key idea is to model interactions between multiple predicates within a sentence to improve argument identification, without relying on syntactic features from external parsers, thereby avoiding error propagation. The authors present two models: (i) a "Single-Sequence" model using stacked bidirectional RNNs to analyze each predicate independently, and (ii) a "Multi-Sequence" model using a Grid-RNN architecture to jointly model all predicates and their argument candidates in a sentence.

As I see them, the main contributions are:
1.  The proposal of a Grid-RNN architecture to jointly model multi-predicate interactions for Japanese PAS analysis. This is a novel application of this architecture to the task and directly addresses a known challenge in the field.
2.  An empirical demonstration that neural models operating on word sequences alone can outperform previous state-of-the-art systems that rely on syntactic features, on the standard NAIST Text Corpus 1.5 benchmark.
3.  A strong baseline Bi-RNN model ("Single-Sequence") which, by itself, surpasses prior syntax-based work, highlighting the power of contextual representations for this task.

### Strengths
1.  **Well-motivated Approach:** The motivation for the work is clear and compelling. Moving away from syntax-dependent pipelines to an end-to-end neural model is a well-established trend, and the paper correctly identifies error propagation as a key problem in prior work. The central hypothesis—that modeling predicate interactions can help resolve difficult cases like zero arguments—is linguistically plausible and builds logically upon prior joint models (e.g., Ouchi et al., 2015).

2.  **Novel Architecture for the Task:** The use of a Grid-RNN to capture interactions between predicate-specific sequences is a clever and appropriate architectural choice. It provides a principled way to allow information to flow both along the sentence length (time dimension) and across the different predicates being analyzed (predicate dimension). This is a more sophisticated approach than simple feature-based joint modeling.

3.  **Strong Empirical Results:** The paper achieves a new state-of-the-art result on a standard dataset. The improvements are particularly pronounced for the challenging "Zero" argument category, which directly supports the authors' claims. The fact that even their simpler Single-Sequence model outperforms the previous SOTA (Ouchi et al., 2015) is a strong result in its own right and validates the general approach of using RNNs for contextual encoding in this task.

### Weaknesses
Despite the novel ideas and strong results, the paper suffers from a substantial number of minor, yet collectively significant, flaws in its presentation, methodological description, and analysis.

1.  **Serious Presentation and Formatting Issues:** The paper exhibits a concerning lack of attention to detail.
    *   The bibliography is completely malformed (e.g., "Bastien et al.,? 2012, Bastien et al.,", "Carreras and Màrquez.,? \Q2005\E, Carreras and Màrquez.,"). This is unacceptable for a formal submission and makes it difficult to verify citations.
    *   The source code URL is a placeholder ("http:xxx") in Footnote 1.
    *   Key architectural details are relegated to footnotes. For instance, the choice of GRU as the recurrent unit (Footnote 3) is a fundamental detail that belongs in the main text of Section 3.2.
    *   Typographical errors are present (e.g., "As an solution" in Section 2.2).
    *   Figure 2 is referenced in the introduction before the models it depicts have been introduced, which disrupts the flow of the paper.

2.  **Insufficient Analysis of Key Results:** The paper reports several intriguing results but fails to provide adequate analysis.
    *   The most striking result is the massive F-measure improvement of ~30 points for dative (DAT) arguments over the SOTA baseline (Table 3). This is an extraordinary gain. The paper's explanation—"the neural models can robustly learn it"—is superficial and uninformative. A detailed analysis is required. What linguistic phenomena or data artifacts cause this? Are previous models' syntactic features particularly poor for dative arguments? Without this analysis, the result is impressive but not understood.
    *   The main contribution, the Multi-Sequence model, provides a statistically significant but practically modest improvement of 0.27 F1 over the Single-Sequence model (81.15 to 81.42). The authors should discuss this trade-off: is the significant added complexity of the Grid-RNN justified by this relatively small gain?

3.  **Lack of Clarity in Experimental Details:** Several hyperparameters and implementation choices are not justified.
    *   The mini-batch sizes are extremely small ({2, 4, 8}). This is highly unusual and warrants an explanation. Is it due to memory limitations of the Grid-RNN structure, which scales with the number of predicates? If so, this is a practical limitation of the model that should be discussed.
    *   The GRU parameter dimensions differ between the single-sequence (32x32) and multi-sequence models (input-related: 64x32, others: 32x32). The reason for this architectural difference is not provided.
    *   It is not explicitly stated that the Single-Sequence model is run independently for each predicate in a sentence, which is a crucial detail for understanding its computational profile compared to the joint Multi-Sequence model.

4.  **Vague or Unsubstantiated Claims:**
    *   Footnote 2 states that for Inter-Zero arguments, "The F-measure remains 10-20%." This is a strong claim presented without context (e.g., which systems, what specific evaluation?).
    *   The authors dismiss a direct comparison with other neural Japanese PAS models (Shibata et al., 2016; Iida et al., 2016) by stating they use "unpublished datasets and evaluated the performance with different experimental settings". While a direct quantitative comparison might be impossible, a more detailed qualitative discussion of architectural differences and potential relative merits is expected and currently absent.

### Questions to Authors
1.  Could you provide a detailed error analysis explaining the extraordinary ~30 point F-measure improvement for dative (DAT) arguments? What specific types of dative arguments are the neural models correctly identifying that the syntax-based models miss?
2.  Please clarify the reason for using very small mini-batch sizes (2, 4, 8). Does the Grid-RNN architecture impose significant memory constraints that prevent the use of larger, more standard batch sizes?
3.  The performance gain of the Multi-Sequence model over the Single-Sequence model is statistically significant but small in absolute terms (0.27 F1). Could you comment on the computational cost and complexity trade-off? In a practical scenario, would the simpler Single-Sequence model be preferable?
4.  Why were different GRU dimensions chosen for the single and multi-sequence models? Was this decision based on tuning on the development set, and if so, what was the observed effect?
