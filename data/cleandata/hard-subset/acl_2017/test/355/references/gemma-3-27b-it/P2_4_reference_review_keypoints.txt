- Proposes using Grid-RNNs to model multi-predicate interactions for Japanese PAS, avoiding reliance on syntactic parsing.  
- Shows empirical benefit of explicitly modeling multi-predicate interactions on the NAIST Text Corpus.  
- Adapts Grid-RNNs to capture long-range dependencies between predicates.  
- Presents a clear single-sequence baseline alongside the multi-sequence Grid-RNN model.  
- Clearly motivates the task by highlighting Japanese argument omission and parsing limitations.  
- Reports strong, statistically significant improvements, especially for zero argument identification.  
- Paper is well-structured and readable, with helpful architectural figures.  
- Source code is publicly released to support reproducibility.  
- Includes an ablation study examining network depth and residual connections.  
- Individual components (RNNs, Grid-RNNs) are not novel; novelty is in their application and combination.  
- Lacks a detailed error analysis showing where the model still fails.  
- Hyperparameter sensitivity and search details are not extensively explored.  
- Comparisons to stronger or more recent baselines (e.g., pretrained language models) are limited.  
- Requests from reviewer: provide detailed error analysis, report sensitivity to window size C, report experiments with pre-trained embeddings, and comment on Grid-RNN training time and scalability.  
- Recommends adding discussion of limitations and a concrete example illustrating how Grid-RNNs capture predicate interactions.  
- Overall assessment: solid contribution with convincing results; recommend acceptance with minor revisions.
