This paper proposes a joint neural modeling approach for Japanese language PAS analysis based on Grid-RRN, which compares it in various ways with a conventional single-sequence RRN approach.

It is a well-functioning paper, targeting a well-established work in Japanese language, but achieving state-of-the-art results and presenting the work in a way that is almost comprehensible to those who are not fluent in Japanese. However, I feel you could have discussed the work in a little more detail, such as talking about the inherent ambiguity between the NOM and ACC arguments of the first rule, as in your example in Figure 1, as the NOM argument of the second rule, and how the work contrasts with SRL (mostly through the ambiguity of the zero pronoun). I would have liked to see some statistics on the ratio of zero pronouns that are actually inter-contextually solvable, as this complicates the subject of the defined work (i.e., the distinction between inherent and inter-contextual zero anaphors must be made). Here, I was not sure about one thing: what representation do you use for an inter-contextual zero pronoun for a given rule's argument? Is there any identification of that argument, or is it identified as an empty argument? In my opinion, it is the first one, in which case there is no clear representation of the zero pronoun, which seems like a slightly flawed representation (which probably affects the model's ability to hold zero pronouns); it would have been good if there had been some discussion on this matter.

There are some limitations that are not captured in the model (which clearly models some ILP-based approaches for SRL, such as): (1) a given rule will usually have only one argument of a specific type (especially NOM and ACC); and (2) a given argument usually fills only one argument slot for a given rule. I would have liked to see some analysis of the model's output to see how well the model has learned such limitations. More generally, considering the mix of numbers between single-sequence and multi-sequence in Table 3 (where there is really no improvement for multi-sequence, it is only NOM), I would have liked to see some discussion on the relative difference between the outputs of the two models: are they basically identical, or very different but almost the same overall, for example? In what context do you notice the difference between the two models? Such an analysis would have made a difference between a hard and a strong paper, shedding light on the internal issues of the models, and this is the main area where I think the paper could be improved (except for including the results of SRL, but that would require much more work).
