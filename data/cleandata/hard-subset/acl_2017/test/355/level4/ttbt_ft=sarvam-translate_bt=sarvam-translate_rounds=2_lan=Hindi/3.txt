This paper proposes a combined neural modeling approach based on grid-RNA for PAS analysis in the Japanese language, compared to the traditional single-sequence RNA approach.

It is a solidly executed paper, targeting a well-established task in the Japanese language, but achieving cutting-edge results in the task, and presenting the task in a mostly accessible way for those who are not fluent in the Japanese language. Having said that, I felt you could have talked a little about the complexity of the task, for example, about the ambiguity inherent between the NOm and ACC arguments in Figure 1, as the NOm argument of the second predicate, and better describe how the task contrasts with SRL (mostly through ambiguity in zero pronouns). I would also like to see some figures about the ratio of zero pronouns that are actually intra-sentential हल-able, as this makes the defined task more complex (i.e., the need to implicitly distinguish between intra- and inter-sentential zero anaphors). One thing I was not sure of here: what representation do you use in the case of an inter-sentential zero pronoun for a given predicate argument? Is there no notation for that argument, or is it marked as an empty argument? My point in reading this paper is that it is a precursor, which does not have a clear representation of the fact that there is a zero pronoun, which seems like a slightly flawed representation (which potentially affects the model's ability to capture zero pronouns); some discussion on this would be appreciated.

There are some constraints that are not captured in the model (which are clearly models for some ILP-based SRLs, for example): (1) a given predicate will generally have only one type of argument (especially NOm and ACC); and (2) a given argument will generally fill only one argument slot for a given predicate. I would like to see some analysis of the model's output to see how well the model was able to learn these types of constraints. More generally, given the mix of numbers in Table 3 between single-sequence and multi-sequence (where it is actually only NOm where there is improvement for multi-sequence), I would like to see some discussion on the relative difference in the output of the two models: are they quite similar, or quite different but almost similar, for example? In what contexts do you see the difference between the two models? Such an analysis would have made the difference between a solid and a strong paper, shed light on the internal parts of the models, and this is the main area where I think the paper could be improved (besides including the results of SRL, but that would require a lot more work).
