**Summary of the Paper**

This paper proposes a neural model using Grid-type Recurrent Neural Networks (Grid-RNN) for Japanese predicate argument structure (PAS) analysis. The model automatically induces features sensitive to multi-predicate interactions from word sequence information of a sentence, without relying on syntactic information predicted by parsers. The authors introduce a basic single-sequence model and extend it to a multi-sequence model using Grid-RNNs, which captures the interactions between all predicates in a sentence. The experiments on the NAIST Text Corpus show that the proposed model outperforms the state-of-the-art Japanese PAS analyzer.

**Main Contributions**

1. A novel neural model using Grid-RNNs for Japanese PAS analysis, which effectively captures multi-predicate interactions without relying on syntactic information.
2. A comprehensive evaluation of the proposed model on the NAIST Text Corpus, demonstrating its state-of-the-art performance.
3. An in-depth analysis of the model's performance on different argument types, particularly on zero arguments, which are prevalent in Japanese and challenging for PAS analysis.

**Strengths**

* The paper presents a sophisticated application of Grid-type Recurrent Neural Nets to the task of determining predicate-argument structures (PAS) in Japanese.
* The approach does not use any explicit syntactic structure and outperforms the current state-of-the-art systems that do include syntactic structure.
* The authors give a clear and detailed description of the implementation and of the results.
* The paper pays close attention to the performance on dropped arguments, zero pronouns, which are prevalent in Japanese and especially challenging with respect to PAS.
* The multi-sequence model, which takes all of the predicates in the sentence into account, achieves the best performance for these examples.
