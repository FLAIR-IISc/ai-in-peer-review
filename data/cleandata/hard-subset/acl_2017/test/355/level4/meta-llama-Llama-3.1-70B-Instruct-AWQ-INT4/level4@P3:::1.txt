**Summary**

This paper proposes a neural model for Japanese predicate argument structure (PAS) analysis, which automatically induces features sensitive to multi-predicate interactions from word sequence information of a sentence. The model uses grid-type recurrent neural networks (Grid-RNN) to capture the interactions between multiple predicates without relying on syntactic information. The authors evaluate their model on the NAIST Text Corpus and demonstrate that it outperforms the state-of-the-art Japanese PAS analyzer.

**Main Contributions**

1. A neural model that effectively captures multi-predicate interactions for Japanese PAS analysis without relying on syntactic information.
2. A grid-type neural architecture that can jointly predict arguments for all predicates in a sentence at a time.
3. State-of-the-art results on the NAIST Text Corpus, particularly for zero argument identification, which is a challenging task in Japanese PAS analysis.

**Strengths**

* The paper presents a sophisticated application of Grid-type Recurrent Neural Nets to the task of determining predicate-argument structures (PAS) in Japanese.
* The approach does not use any explicit syntactic structure, and outperforms the current SOA systems that do include syntactic structure.
* The authors give a clear and detailed description of the implementation and of the results.
* The paper pays close attention to the performance on dropped arguments, zero pronouns, which are prevalent in Japanese and especially challenging with respect to PAS.
* The multi-sequence model, which takes all of the predicates in the sentence into account, achieves the best performance for these examples.

**Weaknesses**

* Minor typos and grammatical errors throughout the paper, which can be corrected to improve English fluency.
* The use of boldface in Table 2 is not clear.
* It would be helpful to illustrate the point about the PRED including context around the "predicate" with an example.

**Questions to Authors**

* Can you provide more details on how the Grid-RNN architecture is able to capture long-distance dependencies in the sentence?
* How do you plan to extend this work to other languages, particularly those with different syntactic structures?
* Have you considered using other types of neural architectures, such as transformers, for this task?
