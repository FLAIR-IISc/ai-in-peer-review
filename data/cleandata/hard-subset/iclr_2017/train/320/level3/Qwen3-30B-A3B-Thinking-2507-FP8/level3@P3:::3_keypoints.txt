- Proposes visual servoing in pre-trained deep feature space using a bilinear visual dynamics model and reinforcement learning for robust target-following.

- Claims ImageNet pre-trained deep features work well for servoing when combined with a bilinear predictive model.

- Demonstrates substantial performance gains over multiple baselines, including methods using ground-truth bounding boxes.

- Introduces a principled method to learn multi-scale visual feature weights via efficient trust-region fitted Q-iteration (FQI).

- Reports remarkable sample efficiency, learning effective policies with only ~20 training trajectories.

- Releases an open-source virtual city environment for benchmarking visual servoing.

- Decision: Accept (with minor revisions).

- Paper addresses a meaningful robotics problem and is well-motivated versus hand-designed features and analytic dynamics.

- Empirical results are rigorous, with careful baselines and multiple evaluation metrics supporting the sample-efficiency claim.

- Method is well-situated in existing literature and clearly differentiates from prior work.

- Suggest evaluating on more complex, diverse, and realistic visual conditions to strengthen generalization claims.

- Recommend investigating end-to-end fine-tuning of the convolutional network to potentially improve performance on novel targets.

- Ask to improve reproducibility: simplify implementation details in the paper, provide a clear README and step-by-step instructions.

- Fix typos noted on page 2.

- Add a brief discussion of real-world deployment limitations (e.g., sim-to-real domain shift, camera calibration sensitivity).

- Questions for authors: detail the diversity of car models used (appearance, size, shape).

- Question why VGG conv5_3 performs worse than shallower features — is it spatial invariance or other factors?

- Ask why FQI yields most gains after the first iteration and why additional sampling gives diminishing returns.

- Request specific quantitative comparisons to the best hand-designed feature approaches in the literature.
