- Decision: Accept.
- Novel approach: combines learned visual features, learned predictive dynamics, and reinforcement learning for visual servoing.
- Method: servo in learned feature space using pre-trained deep features and a bilinear predictive model for camera-motion effects.
- Policy learning: uses sample-efficient fitted Q-iteration that automatically selects multi-scale visual feature weights.
- Contributions: use of pre-trained deep features, principled learning of multi-scale feature weights, and strong sample efficiency.
- Experiments: simulated quadcopter following a car; approach outperforms several baselines including ground-truth bounding-box methods.
- Reproducibility and clarity: paper is well-written with clear explanations and proper citations.
- Limitation: evaluation limited to simulated, relatively simple visual conditions.
- Suggestion: evaluate in more complex, diverse, and realistic visual settings (clutter, distractors, varied appearances/3D models).
- Suggestion: explore end-to-end training/fine-tuning of the convnet to improve representation learning and generalization.
- Suggestion: improve reproducibility by simplifying implementation details and releasing concise open-source code.
- Minor fixes: correct typos quoted on p.2.
- Questions: plans to extend to more complex visual conditions?
- Questions: request for more implementation details on the bilinear predictive model and fitted Q-iteration.
- Questions: how will the method handle distractors and irrelevant objects in the scene?
