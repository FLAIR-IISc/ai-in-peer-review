- Proposes visual servoing for target following using multi-scale spatial feature maps from pre-trained convnets combined with bilinear one-step visual dynamics and an RL policy optimized on predicted feature distances.
- Systematic simulation experiments evaluating the usefulness of pre-trained deep features for visual servoing.
- Shows performance improvements over multiple baselines, including methods using ground-truth bounding boxes.
- Introduces a principled method to learn multi-scale visual feature weights, supported by an efficient trust-region fitted Q-iteration to handle distractors.
- Achieves high sample efficiency by combining a Q-function approximator with model-based one-step visual feature dynamics.
- Releases an open-source virtual city environment to facilitate visual servoing benchmarking.
- Recommend using a more complex benchmark with increased clutter, distractors, diverse appearances and motions, and a larger set of varied 3D car models.
- Suggest evaluating end-to-end training/fine-tuning of the convolutional network to test representation learning benefits and improve generalization.
- Advise consolidating and clarifying practical implementation details in the main text and providing an open-source implementation link for reproducibility.
- Typo corrections: p.2 "learning is a relative[ly] recent addition" and "be applied [to] directly learn".
- Conclusion: paper is compelling and well-executed, and authors appear capable of addressing suggested improvements.
