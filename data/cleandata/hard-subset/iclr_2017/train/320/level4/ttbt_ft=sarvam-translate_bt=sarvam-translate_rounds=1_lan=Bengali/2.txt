This paper discusses the advantages of visual servoing using a learned visual representation. The authors first propose a conditional binary model of visual features (obtained from a pre-trained VGG net) for learning, from which a policy can be generated using simplification of the dynamics. A multi-scale, multi-channel, and locally connected type of binary model is presented. Since the binary model only predicts the dynamics one step further, this paper proposes a weighted objective that incorporates the long-term values of the current policy. The evaluation problem is solved using a fit-value method.

The paper is well-written, mathematically robust, and conceptually complete. The experiments also demonstrate the benefits of using a weighted objective, and this is a significant contribution to this paper. This paper may have been the first to outline a trust-region fit-Q iteration algorithm. The use of pre-trained visual features also helps in general, which is empirically proven.

Overall, I recommend this paper because it will be useful for many robotics researchers. However, in the context of this conference, I believe the contribution on the "presentation" problem is limited. It shows that a pre-trained VGG representation is useful, but it does not consider end-to-end learning. This does not mean it should be end-to-end, but proportionally speaking, the paper spends more time on the control problem than on the representation learning.

Also, the policy representation is static, and the values are approximated in a linear form using problem-specific features. This does not make the paper less valuable, but it is probably less consistent with the core concept of ICLR.
