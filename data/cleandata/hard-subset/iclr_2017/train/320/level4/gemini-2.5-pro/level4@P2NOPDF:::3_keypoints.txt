- Proposes a visual-servoing method that uses spatial CNN feature maps (pretrained on classification), RL for the servoing policy, and a learned multi-scale bilinear model for one-step visual-feature dynamics.
- Controlled simulation experiments effectively quantify the benefit of using pretrained deep features for target following.
- Shows clear performance gains over multiple baselines, including methods using ground-truth bounding boxes.
- Introduces a principled method to learn weights of multi-scale visual features via a trust-region fitted Q-iteration algorithm that handles visual distractors.
- Achieves high sample efficiency through the chosen Q-function approximator and the one-step feature dynamics model.
- Releases an open-source virtual city environment that can serve as a benchmark for future work.
- Suggests increasing benchmark complexity (more visual diversity, clutter, distractors, varied 3D car models, non-target vehicles) to better validate robustness claims.
- Recommends evaluating end-to-end training / fine-tuning of the feature extractor to study representation learning and improve generalization.
- Urges consolidating implementation details and heuristics into a single section and providing open-source code to improve reproducibility.
- Notes minor typos on p.2 ("relative[ly]" and "be applied [to] directly learn").
- Concludes the paper is interesting and solid, strengthened by revisions, and that addressing the suggestions would make it an even stronger contribution.
