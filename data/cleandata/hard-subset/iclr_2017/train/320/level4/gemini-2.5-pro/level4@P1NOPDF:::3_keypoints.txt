- Tackles visual servoing for target following using pre-trained convolutional spatial feature maps, a bilinear one-step visual dynamics model at multiple scales, and an RL policy minimizing a regularized weighted distance to predicted features.
- Controlled simulation experiments quantify benefits of using pre-trained deep features for visual servoing.
- Method outperforms several strong baselines, including those using ground-truth bounding boxes.
- Proposes a principled method to learn multi-scale visual feature weights via an efficient trust-region fitted Q-iteration to handle distractors.
- Achieves high sample efficiency due to the chosen Q-function approximator and model-based one-step visual dynamics.
- Releases an open-source virtual city environment for benchmarking visual servoing.
- Suggests strengthening experiments with more complex, realistic benchmarks (more clutter, distractors, varied 3D car models, moving/parked vehicles).
- Recommends exploring end-to-end fine-tuning and representation learning to improve generalization to harder visual conditions.
- Advises consolidating scattered implementation details and providing open-source code to improve reproducibility.
- Notes minor typos: "relative[ly]" and "be applied [to] directly learn."
- Overall assessment: solid and interesting paper; authors responsive to feedback and can feasibly address suggested improvements.
