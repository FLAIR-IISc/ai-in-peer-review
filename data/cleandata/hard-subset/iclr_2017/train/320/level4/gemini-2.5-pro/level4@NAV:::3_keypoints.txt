- Presents a visual-servoing method using spatial CNN feature maps (pre-trained for classification) combined with reinforcement learning and multi-scale bilinear one-step dynamics models.  
- Trains a policy by minimizing a regularized, weighted average of distances to feature states predicted by the dynamics model.  
- Quantitative simulation results show pre-trained deep features are effective for target-following visual servoing.  
- Method significantly outperforms several baselines, including some that use ground-truth bounding boxes.  
- Introduces a principled, learned weighting for multi-scale visual features using trust-region fitted Q-iteration to handle distractors.  
- Demonstrates high sample efficiency attributable to the Q-function approximator choice and model-based one-step visual dynamics.  
- Provides an open-source virtual city environment for benchmarking visual servoing algorithms.  
- Suggests strengthening experiments with more challenging visual scenarios: more clutter, denser/more varied distractors, varied target appearance/motion, and more realistic 3D car models.  
- Recommends evaluating end-to-end training (fine-tuning CNN) to test generalization and study representation learning for visual servoing.  
- Calls for clearer, consolidated implementation details for reproducibility and an open-source implementation with complete details.  
- Notes a few minor typos and concludes the paper is compelling and well-executed, with suggested changes likely to substantially strengthen it.
