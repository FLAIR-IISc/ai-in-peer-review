- Addresses visual servoing (target following) using spatial feature maps from pre-trained convnets combined with a bilinear one-step visual dynamics model and RL to learn a servoing policy.
- Policy trained by minimizing a regularized, weighted average of distances to target features predicted by the visual dynamics model.
- Controlled simulation experiments quantify the utility of pre-trained deep features for visual servoing.
- Demonstrates clear performance gains over multiple sensible baselines, including methods using ground-truth bounding boxes.
- Proposes a principled method to learn weights of multi-scale visual features using an efficient trust-region fitted Q-iteration algorithm to handle distractors.
- Shows good sample efficiency attributed to the chosen Q-function approximator and the model-based one-step visual dynamics.
- Provides an open-source virtual city environment for benchmarking visual servoing tasks.
- Recommends strengthening the benchmark with more visual complexity: clutter, more distractors, varied appearances/motions, and more 3D car models.
- Suggests investigating end-to-end training/fine-tuning of the convolutional network to evaluate representation learning and improve generalization.
- Requests improved reproducibility: summarize critical implementation choices in the paper and release an open-source implementation.
- Notes minor typos to fix (examples on p.2).
- Overall assessment: interesting, solid paper; authors' revisions improved the work and they can address suggested improvements.
