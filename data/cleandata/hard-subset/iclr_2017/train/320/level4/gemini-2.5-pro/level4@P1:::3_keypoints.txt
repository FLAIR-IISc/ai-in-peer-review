- Tackles visual servoing for target following using spatial feature maps from pre-trained convolutional networks.  
- Combines a bilinear one-step multi-scale visual feature dynamics model with reinforcement learning to learn a servoing policy.  
- Policy is optimized by minimizing a regularized, weighted distance between desired target features and predicted features.  
- Recommendation: Accept.  
- Strength: Principled combination of pre-trained features, learned dynamics, and sample-efficient RL yielding strong performance.  
- Strong empirical evaluation against multiple relevant baselines.  
- Controlled simulations show substantial benefits of pre-trained deep features over classical methods and ground-truth bounding-box baselines.  
- Technical contribution: efficient method to learn multi-scale feature weights via fitted Q-iteration, which helps focus on relevant features and handle distractors.  
- Impressive sample efficiency—learning from only ~20 trajectories—attributed to model-based one-step dynamics and the chosen Q-function approximator.  
- Useful community contribution: open-source virtual city benchmarking environment.  
- Suggested improvements: evaluate under more complex visual conditions (more clutter, distractors, varied object appearance/motion) to substantiate robustness claims.  
- Suggested improvements: investigate end-to-end fine-tuning of features for better generalization and clearer assessment of representation learning.  
- Suggested improvements: consolidate critical implementation details in the main paper and refer to open-source code for full specifics.  
- Minor typos to fix: p.2 "learning is a relative[ly] recent addition" and p.2 "be applied [to] directly learn".
