- Proposes visual servoing for target following using spatial feature maps from pre-trained CNNs, bilinear one-step feature dynamics, and an RL policy minimizing weighted feature-distance.

- Controlled simulation experiments show pre-trained deep features are effective for visual servoing and provide quantitative insights.

- Method outperforms sensible baselines, including those using ground-truth bounding boxes.

- Introduces principled multi-scale visual feature weighting learned with a trust-region fitted Q-iteration algorithm to handle distractors.

- Choice of Q-function approximator plus model-based one-step feature dynamics yields good sample efficiency.

- Provides an open-source virtual city environment as a visual servoing benchmark.

- Suggests testing on more complex visual conditions (clutter, more distractors, varied 3D car models) to improve realism and diversity.

- Recommends investigating end-to-end training/fine-tuning of the convolutional network to evaluate representation learning and generalization.

- Calls for clearer organization of implementation tricks and practical details, or providing an open-source implementation to aid reproducibility.

- Notes minor typos on page 2 that should be corrected.

- Overall assessment: well-written, solid paper with potential for further improvement and strong authors’ responsiveness to feedback.
