- Summary: Proposes visual servoing via multi-scale spatial feature maps from pre-trained convnets, using bilinear one-step feature dynamics and RL to learn a servoing policy that minimizes predicted feature distances.

- Contribution: Provides controlled simulation experiments quantifying effectiveness of pre-trained deep features for visual servoing.

- Contribution: Demonstrates performance gains over sensible baselines, including methods using ground-truth bounding boxes.

- Contribution: Introduces a principled method to learn multi-scale visual feature weights with a trust-region fitted Q-iteration algorithm to mitigate distractors.

- Contribution: Shows good sample efficiency due to the chosen Q-function approximator and model-based one-step visual feature dynamics.

- Contribution: Releases an open-source virtual city environment for benchmarking visual servoing.

- Suggestion: Expand benchmarking to more complex visual conditions (clutter, distractors, varied appearances/motion, more diverse 3D car models) to test robustness.

- Suggestion: Investigate end-to-end training/fine-tuning of the convnet to measure benefits of representation learning, especially under harder visual scenarios.

- Suggestion: Improve reproducibility by consolidating and clarifying implementation details, summarizing key tricks in the main text, and providing open-source code.

- Typos: Fix minor typos (e.g., "learning is a relative[ly] recent addition", "be applied [to] directly learn").

- Conclusion: Paper is interesting and solid with promising results; authors appear able to address suggested improvements to strengthen the work.
