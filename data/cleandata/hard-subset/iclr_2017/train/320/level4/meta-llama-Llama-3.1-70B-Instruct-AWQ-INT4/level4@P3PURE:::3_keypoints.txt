- Proposes a visual servoing/target-following method using spatial feature maps from pretrained CNNs, bilinear one-step feature dynamics, and reinforcement learning to minimize weighted feature prediction errors.
- Demonstrates controlled simulation experiments quantifying the value of pretrained deep features for visual servoing.
- Shows clear performance gains over several sensible baselines, including methods using ground-truth bounding boxes.
- Introduces a principled multi-scale feature-weight learning approach using trust-region fitted Q-iteration to handle distractors effectively.
- Achieves good sample efficiency via the chosen Q-function approximator and model-based one-step visual feature dynamics.
- Provides an open-source virtual city environment to benchmark visual servoing.
- Suggests evaluating in more complex, realistic environments with greater clutter, distractors, motion/appearance diversity, and more diverse 3D models to test robustness.
- Recommends investigating end-to-end training/fine-tuning of the convolutional network to assess benefits of representation learning.
- Requests improved reproducibility by consolidating implementation details in the main text and providing a code link.
- Notes minor typos in the paper and concludes the work is interesting and solid, with authors capable of addressing suggested improvements.
