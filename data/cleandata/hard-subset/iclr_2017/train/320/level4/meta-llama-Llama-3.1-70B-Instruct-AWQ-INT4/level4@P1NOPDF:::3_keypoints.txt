- Proposes visual servoing for target following using spatial feature maps from pre-trained convnets, bilinear one-step visual dynamics at multiple scales, and an RL policy minimizing weighted distances to predicted features.  
- Controlled simulation experiments quantify the usefulness of pre-trained deep features for visual servoing.  
- Method outperforms several baselines, including those using ground-truth bounding boxes.  
- Introduces a principled learning scheme for multi-scale visual feature weights via trust-region fitted Q-iteration to handle distractors.  
- Demonstrates good sample efficiency attributed to the chosen Q-function approximator and model-based one-step visual dynamics.  
- Provides an open-source virtual city environment to benchmark visual servoing.  
- Experiments need richer visual complexity (clutter, distractors, varied appearances/motions, more diverse 3D car models and traffic scenarios) to test robustness.  
- Suggests investigating end-to-end training/fine-tuning of the convnet to improve generalization and explore representation learning benefits.  
- Reproducibility could be improved by reorganizing and summarizing practical implementation details and releasing code.  
- Minor typos noted that should be corrected.  
- Overall assessment: interesting, solid paper with authors capable of addressing suggested improvements.
