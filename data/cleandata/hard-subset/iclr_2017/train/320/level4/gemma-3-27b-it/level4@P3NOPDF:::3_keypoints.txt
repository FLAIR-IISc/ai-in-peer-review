- Proposes visual servoing for target following using multi-scale spatial feature maps from pre-trained CNNs combined with a reinforcement learning policy.  
- Learns bilinear one-step visual feature dynamics to predict feature changes for control.  
- Optimizes a policy by minimizing a regularized, weighted average of distances to predicted features.  
- Empirically validates that pre-trained deep features are effective for visual servoing in simulation.  
- Demonstrates strong performance against several baselines, including ones using ground-truth bounding boxes.  
- Introduces a principled multi-scale feature weighting method using trust-region fitted Q-iteration to reduce distractor influence.  
- Achieves high sample efficiency by combining a Q-function approximator with model-based one-step feature dynamics.  
- Releases an open-source virtual city benchmark environment for visual servoing research.  
- Suggests increasing environmental visual complexity (more diverse 3D models, more static/dynamic distractors) to better test robustness.  
- Recommends exploring end-to-end and representation learning (fine-tuning CNNs during RL) to improve generalization.  
- Advises consolidating implementation details and providing public code; notes minor editorial fixes.  
- Overall: strong, well-motivated paper with recommended acceptance and scope for further improvements.
