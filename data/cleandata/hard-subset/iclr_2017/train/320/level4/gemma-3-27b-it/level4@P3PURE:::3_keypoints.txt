- Proposes a visual-servoing method that uses pre-trained CNN spatial feature maps combined with bilinear one-step feature dynamics and RL to learn a servoing policy.
- Controlled simulation experiments quantify benefits of pre-trained deep features for visual servoing.
- Demonstrates clear performance improvements over several sensible baselines, including those using ground-truth bounding boxes.
- Introduces a principled multi-scale feature-weight learning approach and an efficient trust-region fitted Q-iteration algorithm to handle visual distractors.
- Achieves good sample efficiency via the chosen Q-function approximator and model-based one-step visual feature dynamics.
- Provides an open-source virtual city environment as a benchmark for visual servoing research.
- Suggests evaluating on a more challenging and diverse benchmark with increased clutter, distractors, and varied object appearances (e.g., more 3D car models, traffic).
- Recommends investigating end-to-end training/fine-tuning of the convolutional network to assess representation learning benefits and improve generalization.
- Advises improving reproducibility by consolidating implementation details in the main text and providing a complete open-source implementation.
- Notes minor typos to correct: "relative[ly]" and "be applied [to] directly learn".
- Concludes the paper is an interesting, solid contribution and that authors have addressed pre-review feedback well, indicating potential for a stronger final paper.
