- Introduces a visual servoing method that uses spatial feature maps from pre-trained convnets, bilinear one-step feature dynamics at multiple scales, and RL to learn a servoing policy minimizing weighted distances to predicted features.  
- Provides controlled simulation experiments quantifying benefits of pre-trained deep features for visual servoing.  
- Demonstrates significant performance improvements over several baselines, including ones using ground-truth bounding boxes.  
- Proposes a principled method to learn multi-scale visual feature weights using an efficient trust-region fitted Q-iteration to handle visual distractors.  
- Shows good sample efficiency due to the chosen Q-function approximator and model-based one-step visual feature prediction.  
- Releases an open-source virtual city environment as a benchmark for visual servoing research.  
- Suggests expanding the benchmark complexity (more clutter, distractors, diverse object appearances/motion, more 3D car models, additional vehicles).  
- Recommends exploring end-to-end training/fine-tuning of convnets to test representation learning benefits and improve generalization.  
- Calls for better reproducibility by consolidating key implementation details in the main text and providing an open-source implementation link.  
- Notes minor typos to fix (e.g., "relative[ly]" and "be applied [to] directly learn").  
- Overall assessment: interesting and solid paper; authors’ revision and responses indicate they can address suggested improvements to strengthen the final submission.
