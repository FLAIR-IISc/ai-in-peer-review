This paper discusses the benefits of visual servoing using a learning visual representation. The authors initially propose a conditional binary model (obtained from a pre-trained VGG net) of visual features for learning, from which a policy can be generated using simplified kinematics. A multi-scale, multi-channel, and locally connected binary model type is presented. Since the binary model only estimates one-step kinematics, this paper proposes a weighted objective that incorporates the long-term values of the current policy. Evaluation problems are solved using a fit-value method.

The paper is well-written, mathematically robust, and conceptually complete. The experiments also demonstrate the benefits of using a weighted objective, and this is a significant contribution to this paper. This paper may be the first to outline a trust-region fit-Q iteration algorithm. The use of pre-trained visual features also helps in general, which is empirically proven.

Overall, I recommend this paper because it will be useful for many robotics researchers. However, in the context of this conference, I believe the contribution of the "presentation" problem is limited. It shows that a pre-trained VGG representation is useful, but it does not consider end-to-end learning. This does not mean it should be end-to-end, but to be proportionate, the paper spends more time on control problems than on representation learning.

Also, the values are approximated in a linear form using stable and problem-specific features in the policy representation. This does not make the paper less valuable, but it is probably less consistent with the core concept of ICLR.
