- Summary: Uses pre-trained convnet spatial feature maps plus bilinear one-step visual-dynamics models and RL to learn a multi-scale visual servoing policy minimizing predicted feature distances.

- Contribution: Provides careful, controlled simulation experiments quantifying value of pre-trained deep features for visual servoing.

- Contribution: Shows clear performance gains over sensible baselines, including ones using ground-truth bounding boxes.

- Contribution: Introduces principled learning of multi-scale visual feature weights via a trust-region fitted Q-iteration algorithm to handle distractors.

- Contribution: Demonstrates good sample efficiency due to the chosen Q-function approximator and model-based one-step dynamics.

- Contribution: Releases an open-source virtual city environment for benchmarking visual servoing.

- Suggestion: Use a more challenging, diverse benchmark with increased clutter, stronger distractors, more varied car models, and greater environmental diversity to better test robustness.

- Suggestion: Evaluate end-to-end and representation learning (convnet fine-tuning) to assess benefits for generalization under harder visual conditions.

- Suggestion: Improve reproducibility by grouping/simplifying implementation details, summarizing key practical tricks in a dedicated section, and linking open-source code.

- Minor: Fix typos on p.2 (“learning is a relatively recent addition”; “be applied to directly learn”).

- Conclusion: Paper is interesting and solid with a strong revision and likely able to address suggested improvements.
