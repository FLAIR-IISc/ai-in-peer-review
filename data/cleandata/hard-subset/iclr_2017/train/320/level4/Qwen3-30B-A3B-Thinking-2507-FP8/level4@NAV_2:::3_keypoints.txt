- Summary: Uses pre-trained convolutional feature maps and multi-scale locally connected bilinear one-step dynamics with a model-based RL policy that minimizes a regularized weighted sum of distances to predicted features for visual target following.
- Decision: Accept — significant contribution showing robust target following with exceptional sample efficiency.
- Pre-trained features: Controlled simulations show pre-trained deep features are highly effective for visual servoing.
- Performance vs baselines: Method substantially outperforms many baselines, including those using ground-truth bounding boxes.
- Multi-scale weighting: Introduces a principled approach to learn multi-scale feature weights via an efficient fitted Q-iteration, helping handle distractors.
- Sample efficiency: Achieves remarkable sample efficiency (over two orders of magnitude better than standard model-free deep RL) due to one-step dynamics and an appropriate Q-function approximator.
- Benchmark: Provides an open-source virtual city environment for benchmarking visual servoing methods.
- Benchmark complexity feedback: Recommend increasing visual complexity (more clutter, diverse distractors, varied appearances/motion, more 3D car models) to better test robustness.
- End-to-end learning feedback: Suggest exploring fine-tuning or end-to-end representation learning to study generalization in harder visual conditions.
- Reproducibility feedback: Recommend streamlining and summarizing implementation details in the main text and linking to open-source code to improve reproducibility.
- Typos: Minor typographical errors noted on page 2.
- Question — fine-tuning: How would performance change if the convolutional network were fine-tuned, especially in more complex visual environments?
- Question — dynamics model: What specific challenges arose implementing the locally connected bilinear dynamics model, and how do they affect scalability to higher-dimensional visual representations?
- Question — occlusions: What are the method’s limitations for extreme or prolonged occlusions, and how might it be extended to handle them?
