- Summary: Uses spatial feature maps from pre-trained convnets and bilinear one-step visual dynamics across scales with RL to learn a visual-servoing (target-following) policy minimizing predicted feature-distance.  
- Contribution: Demonstrates quantitatively in simulation that pre-trained deep features are useful for visual servoing.  
- Contribution: Shows substantial performance gains over multiple baselines, including methods using ground-truth bounding boxes.  
- Contribution: Learns multi-scale visual feature weights via a trust-region fitted Q-iteration algorithm, effectively mitigating distractors.  
- Contribution: Achieves high sample efficiency through the chosen Q-function approximator and model-based one-step visual feature dynamics.  
- Contribution: Provides an open-source virtual city environment for visual servoing benchmarking.  
- Suggestion: Increase benchmark visual complexity (clutter, more distractors, diverse 3D car models and scenes) to better test robustness to appearance variation.  
- Suggestion: Evaluate end-to-end training / convnet fine-tuning to assess necessity of representation learning for generalization.  
- Suggestion: Improve reproducibility by consolidating/organizing implementation details and summarizing key aspects in the main text with a link to code.  
- Suggestion: Fix typos (examples: p.2 "learning is a relatively recent addition"; p.2 "be applied to directly learn").  
- Conclusion: Despite limited experimental scope, the paper is a compelling, robust contribution and authors appear able to address the suggested improvements.
