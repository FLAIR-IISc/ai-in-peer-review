- Proposes servoing in pre-trained deep feature space using bilinear predictive dynamics and reinforcement learning to reduce manual feature design and model engineering.  
- Shows effective visual servoing learned from minimal data (≈20 trajectories) with orders-of-magnitude better sample efficiency than model-free RL.  
- Empirical results outperform pixel- and hand-designed-feature baselines and match classical approaches with far less data.  
- Employs a bilinear model for feature dynamics and sample-efficient fitted Q-iteration to learn feature weights, with sound justification.  
- Implementation details are clear and code/environment are publicly released, aiding reproducibility.  
- Suggests expanding benchmarks to more complex visuals (distractors, varied lighting, realistic 3D models) to test robustness.  
- Recommends exploring end-to-end fine-tuning of convolutional features for task-specific performance gains.  
- Advises streamlining appendix implementation details and providing clear links to the public code for accessibility.  
- Questions how performance scales with increased visual complexity (distractors, lighting variation, occlusion).  
- Questions the computational requirements versus classical IBVS and how to reduce them for real-time deployment.  
- Questions whether the bilinear dynamics model can be extended to handle object deformation or large viewpoint changes.
