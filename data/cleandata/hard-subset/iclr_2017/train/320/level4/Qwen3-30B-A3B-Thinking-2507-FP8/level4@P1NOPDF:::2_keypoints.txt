- Proposes an action-conditional bilinear model of pre-trained VGG visual features and derives a policy via linearization of the dynamics.  
- Introduces a multi-scale, multi-channel, locally-connected variant of the bilinear model.  
- Bilinear model predicts only one-step dynamics; paper addresses long-term behavior with a weighted objective incorporating value estimates.  
- Uses a fitted-value approach to compute the long-term value weighting; experiments show this value-weighted objective is beneficial.  
- Appears to be the first formalization of a trust-region fitted-Q iteration algorithm.  
- Empirical results: pre-trained visual features improve generalization and validate the control innovations.  
- Paper is well-written, mathematically rigorous, and conceptually thorough.  
- Reviewer recommends acceptance, noting the work’s value to robotics researchers.  
- Major criticism: contribution to representation learning is limited—visual features remain fixed (VGG) with no end-to-end adaptation.  
- Policy representation uses a fixed linear value approximation with problem-specific features, limiting representation novelty.  
- Imbalance: substantial emphasis on control methodology versus modest advances in representation learning.  
- For a venue prioritizing representation learning (e.g., ICLR), the limited representation contribution weakens the paper’s fit.
