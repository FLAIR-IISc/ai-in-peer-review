- Problem & approach: Visual servoing/target following using pre-trained CNN spatial feature maps, bilinear multi-scale one-step visual dynamics, and an RL policy minimizing weighted distances to predicted features.  
- Validation: Quantitative simulation results demonstrating the utility of pre-trained deep features for visual servoing.  
- Performance: Strong results compared to multiple baselines, including methods using ground-truth bounding boxes.  
- Feature-weight learning: Principled multi-scale visual feature weighting via an efficient trust-region policy iteration that mitigates distractors.  
- Sample efficiency: High sample efficiency attributed to the Q-function approximator and model-based one-step visual dynamics.  
- Benchmark: Release of an open-source virtual city environment for visual servoing benchmarking.  
- Suggestion — benchmark complexity: Increase visual complexity (clutter, diverse distractors, more realistic 3D car models, moving/parked vehicles) to better test robustness.  
- Suggestion — representation learning: Evaluate end-to-end fine-tuning of the CNN to assess whether deeper representation learning improves generalization.  
- Suggestion — reproducibility: Consolidate and streamline implementation details and link to code to improve accessibility.  
- Minor fixes: Correct typos on p.2 ("learning is a relatively recent addition" and "be applied directly to learn").  
- Overall: Solid and interesting contribution; authors responsive to feedback and likely to improve the work for publication.
