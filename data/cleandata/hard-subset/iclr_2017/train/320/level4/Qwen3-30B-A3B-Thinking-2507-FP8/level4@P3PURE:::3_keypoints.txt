- Paper tackles target-following visual servoing using spatial feature maps from CNNs pretrained on classification, combining multi-scale bilinear one-step visual feature dynamics with an RL policy learned by minimizing a regularized weighted feature-distance loss.  
- Demonstrated that pretrained deep features are useful for visual servoing in controlled simulation experiments.  
- Achieves significant performance gains over strong baselines, including methods using ground-truth bounding boxes.  
- Proposes a principled method to learn multi-scale visual feature weights via a trust-region fitted Q-iteration algorithm, reducing distractor effects.  
- Shows improved sample efficiency thanks to the Q-function approximator and model-based one-step visual feature dynamics.  
- Provides an open-sourced virtual city environment for benchmarking visual servoing.  
- Suggests increasing benchmark visual complexity (more clutter, diverse distractors, varied object appearances/motions, larger set of realistic 3D car models, traffic/parked vehicles) to better evaluate robustness.  
- Recommends assessing end-to-end training/fine-tuning of the convolutional backbone to improve representation learning and generalization.  
- Calls for improved reproducibility: reorganize/summarize implementation details in the main text and release complete open-source implementation.  
- Notes minor typos to fix (two examples on p.2).  
- Concludes the paper is a solid, interesting contribution; authors addressed pre-review comments and with suggested refinements the work could be much stronger.
