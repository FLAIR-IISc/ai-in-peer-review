This paper examines the benefits of visual servoing using a learned visual representation. The authors propose learning a first action-conditioned bilinear model of visual features (obtained from Pre-Trained VGD Net), which can be used to derive a policy using the linearization of dynamics. A multi-level, multi-channel, and locally connected version of the bilinear model is presented. Since the bilinear model only predicts dynamics one step ahead, this paper proposes a weighted objective that incorporates the long-term values of the current policy. The evaluation problem is addressed using a fitted-value approach. This paper is well-written, mathematically sound, and conceptually complete. The experiments also demonstrate the benefits of using a value-weighted objective, and this is a significant contribution to this paper. It seems that this paper is the first to outline a trust-area fitted-Q iteration algorithm. The use of pre-trained visual features to help generalize empirically is also shown. Overall, I recommend this paper as it will benefit many researchers in robotics. However, in the context of this conference, I find the contribution on the "representation" problem particularly limited. It shows that pre-trained VGD representation is useful, but it does not consider end-to-end learning. This does not mean it should be end-to-end, but proportionally speaking, the paper spends more time on the control problem than on the representation learning. Also, the policy representation is fixed, and the values are estimated linearly using problem-specific features. This does not make the paper less valuable, but perhaps it is less aligned with what I think should be a CLR.
