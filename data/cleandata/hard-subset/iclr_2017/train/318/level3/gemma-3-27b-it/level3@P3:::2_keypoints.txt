Recommendation: accept the paper despite lacking substantial performance gains over existing methods.

Core contribution: proposing a differentiable graph-structured memory (GGT-NN) for dialog state representation and reasoning.

Model capability: GGT-NN learns to construct and modify graphs from text to capture complex relationships between entities.

Expressiveness: graph memory is more expressive than prior memory structures (stacks/lists/memory networks).

Empirical results: preliminary—comparable to strong memory networks but inferior to rule-induction methods on bAbI tasks.

Rule-discovery strength: successfully learned complex update rules in cellular automata and Turing machine tasks, showing relational reasoning ability.

Technical soundness: thorough literature positioning and detailed implementation (graph transformations, attention, GRU updates).

Clarity and documentation: improved manuscript clarity and useful appendices detailing graph formats for tasks.

Model complexity: significant increase in complexity versus established memory architectures, argued to be justified by theoretical benefits.

Computational concerns: quadratic scaling with graph size—needs discussion of optimizations (sparse graphs, limited-attention propagation) and cost comparisons to other memory models.

Direct reference effect: direct reference improves performance; the cause (initialization, gradient flow, etc.) should be investigated.

Presentation suggestion: clarify interplay among graph transformations and include a visual example tracing information flow for a simple task (e.g., bAbI).

Questions for authors: request detailed trade-offs of graph transformation configurations, quantification/limits of learned rule complexity, sensitivity to initial graph structure and pretraining, remedies for worse performance with graph sequences (e.g., sequence attention), and clarification on edge-update toggling and its gradient implications.
