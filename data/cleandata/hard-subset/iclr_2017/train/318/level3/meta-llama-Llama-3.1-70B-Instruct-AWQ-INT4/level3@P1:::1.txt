This paper proposes a novel neural network architecture, the Gated Graph Transformer Neural Network (GGT-NN), which learns to construct and modify graphs based on textual input. The GGT-NN model is an extension of the Gated Graph Sequence Neural Network (GGS-NN) and uses graph-structured data as an intermediate representation. The model can learn to generate graph sequences and can be used for tasks such as question-answering and rule discovery.

The GGT-NN model is designed to work with unstructured input and can modify a graphical structure based on the input. The model uses a set of differentiable graph transformations, including node addition, node state update, edge update, propagation, and aggregation. These transformations can be combined to process a graph in complex ways. The model also uses a recurrent input model to incrementally construct a graph given natural language input.

The authors evaluate the GGT-NN model on the bAbI tasks, a set of simple natural-language tasks, and demonstrate its ability to outperform state-of-the-art models on these tasks. The model is also evaluated on two additional tasks, a cellular automaton task and a Turing machine task, and shows promising results.

Overall, the GGT-NN model is a novel and powerful architecture for learning graph-structured data and has the potential to be useful in solving many other types of problems.

**Decision:** Accept

1. The paper proposes a novel and well-motivated architecture for learning graph-structured data.
2. The model is evaluated on a variety of tasks and shows promising results, including outperforming state-of-the-art models on the bAbI tasks.

3. The paper could benefit from more discussion of the limitations of the model and potential future work.

**Questions for the authors:**
