- Proposes Gated Graph Transformer Neural Network (GGT-NN) that learns to construct and modify graphs from textual input.
- GGT-NN is an extension of Gated Graph Sequence Neural Network (GGS-NN) using graph-structured intermediate representations.
- Introduces differentiable graph transformations: node addition, node state update, edge update, propagation, and aggregation.
- Uses a recurrent input model to incrementally construct graphs from natural language.
- Can generate graph sequences and is applicable to tasks like question-answering and rule discovery.
- Evaluated on bAbI tasks and reported to outperform state-of-the-art models.
- Also evaluated on cellular automaton and Turing machine tasks with promising results.
- Decision: Accept.
- Main reasons: novel, well-motivated architecture and strong empirical results across multiple tasks.
- Supporting: clear, well-written introduction and careful model design addressing graph complexities.
- Supporting: thorough bAbI evaluation and additional tasks that bolster evidence of capability.
- Suggestion: include more visualizations of graph structures and transformations.
- Suggestion: strengthen evaluation with more baselines and task-wise performance comparisons.
- Suggestion: include discussion of model limitations and future work.
- Questions for authors: how will scalability to larger graphs be addressed?
- Questions for authors: provide implementation details on differentiable graph transformations and how they are learned.
- Questions for authors: plans to extend the model to more complex tasks and larger datasets?
