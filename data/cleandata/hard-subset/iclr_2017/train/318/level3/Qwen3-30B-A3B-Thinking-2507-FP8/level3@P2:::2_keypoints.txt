- Introduces Gated Graph Transformer Neural Network (GGT-NN) to learn construction and modification of graph-structured representations from text.  
- Demonstrates strong performance on bAbI (≈95% on most tasks) and on rule-discovery tasks (cellular automata, Turing machines).  
- Argues graph-structured memory is a useful, differentiable representation for modeling relationships between entities.  
- Decision: Accept — novel extension of graph-based neural nets with meaningful contributions and viable results.  
- Well-motivated framework that enables learning graphs from unstructured input and is technically sound.  
- Experimental evaluation is rigorous, includes ablations, matches state-of-the-art on bAbI and outperforms others on pathfinding.  
- Represents the first implementation of a differentiable graph memory learned from unstructured input.  
- Lacks detailed analysis of computational complexity and training time given acknowledged quadratic graph operations.  
- Needs clearer guidance on when graph representations are preferable to other structures (stacks, trees) with concrete examples.  
- Unclear exactly how much graph-level supervision was provided versus learned — clarification needed.  
- Requires more detailed analysis of the failure case on task 17 (Positional Reasoning).  
- Clarifying questions requested: specify graph supervision for bAbI; explain how rules were learned without textual input; identify tasks where direct reference most helped and why.
