- Introduces Gated Graph Transformer Neural Network (GGT-NN) that represents dialog as a differentiable graph-based memory.  
- Framework learns and modifies graph structures during inference with long-term learning of graph-transformation parameters and sentence encodings.  
- Claimed first implementation of differentiable memory as a graph, offering more flexibility than fixed-structure representations.  
- Reviewer recommends acceptance.  
- Novel, well-motivated approach filling a gap in structured memory representations.  
- Marginal performance gains on bAbI tasks versus memory networks, but clear advantages on graph-aligned tasks (e.g., pathfinding, task 19).  
- Technical content is sound and clearly presented; revised manuscript improved clarity.  
- Comprehensive empirical evaluation across bAbI, cellular automaton, and Turing machine domains.  
- Demonstrates ability to learn complex state transitions that match problem structure.  
- Main limitation: not yet significantly superior in learning or generalization compared to existing methods.  
- Promising approach for tasks with natural graph structure.  
- Suggested future work: compare with rule-based/symbolic methods, reduce graph-level supervision, improve computational efficiency, and demonstrate scalability to larger real-world problems.
