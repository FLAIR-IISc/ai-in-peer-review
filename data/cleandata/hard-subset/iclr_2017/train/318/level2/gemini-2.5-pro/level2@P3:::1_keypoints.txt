- Introduces Gated Graph Transformer Neural Network (GGT-NN) that builds and manipulates an explicit graph-structured internal state from text via differentiable graph transformations.  
- Core design: five modular differentiable transformations (addition, state update, edge update, propagation, aggregation) for dynamic node/edge creation and information flow.  
- Achieves strong results on bAbI QA: state-of-the-art or near state-of-the-art on 19/20 tasks using only 1k examples; particularly strong on pathfinding and induction tasks.  
- Demonstrates rule discovery for 1D cellular automata and Turing machines, learning deterministic state transitions without explicit rules.  
- Novelty: bridges fixed-graph GNNs and unstructured differentiable memory by producing an explicit, interpretable symbolic representation.  
- Technical exposition is clear and detailed, especially the soft graph representation and transformation classes.  
- Empirical validation is diverse and convincing; authors are transparent about using ground-truth graph supervision during training.  
- Comparison to prior work is comprehensive, emphasizing the advantage of an explicit graphical state over implicit vector memories (e.g., DNCs, Memory Networks).  
- Primary limitation: quadratic time/space complexity in number of nodes; sparse connections or attention-based edge updates are suggested but need expansion.  
- Recommends more extensive ablation studies (e.g., sensitivity to number of propagation steps and the role of intermediate propagation in specific bAbI tasks).  
- Suggests leveraging interpretability by visualizing learned graph evolution on bAbI stories to illustrate node/edge creation and reasoning.  
- Notes practical limitation of strong step-by-step graph supervision and suggests exploring weaker supervision schemes (final-graph supervision or RL).  
- Clarification requested: how the maximum number of nodes for T_add was chosen and how sensitive performance is to this hyperparameter.  
- Clarification requested: deeper analysis of failure on bAbI task 17—whether due to embedding capacity or incorrect relational structure, via inspection of generated graphs.
