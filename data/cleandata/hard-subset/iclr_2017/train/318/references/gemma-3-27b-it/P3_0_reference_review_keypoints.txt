- Paper proposes Gated Graph Transformer Neural Network (GGT-NN) that enables differentiable graph transformations for dynamic graph-structured representations and outputs.  
- Main contributions: differentiable node addition, state update, edge update, propagation, aggregation combined with a recurrent input model.  
- Evaluated on bAbI tasks, a cellular automaton task, and Turing machine simulations, showing ability to learn complex update rules and represent state information.  
- Recommendation: Accept with minor revisions.  
- Novelty: Significant advance by learning graph structure alongside content, extending typical GNNs that use fixed graphs.  
- Empirical validation: Strong results, including high accuracy on bAbI (notably task 19) and success on rule-discovery tasks; ablation studies are informative.  
- Presentation: Well-written, clear explanations, useful Algorithm 1 and detailed Appendix B on transformations.  
- Related work: Thorough and insightful comparison situating GGT-NN within GNN and memory-augmented models.  
- Computational complexity: Paper notes quadratic scaling with graph size; needs deeper discussion of bottlenecks and mitigation strategies (e.g., sparse representations, approximations).  
- Hyperparameter sensitivity: Suggests adding analysis of sensitivity to learning rate, hidden sizes, and propagation steps.  
- Visualization: Recommends visualizations of learned graphs (e.g., from bAbI) to improve intuition about internal representations.  
- Code availability: Encourages ensuring code is accessible and well-documented for reproducibility.  
- Clarify graph construction: Requests clearer, diagrammatic explanation of how cellular automaton and Turing machine are converted to graphs and the meaning of edge types.  
- Questions for authors: clarify scalability bottlenecks and mitigation strategies.  
- Questions for authors: provide visualizations of learned graphs and insights from them.  
- Questions for authors: explain rationale and selection process for the five graph transformations and whether others were tried.  
- Questions for authors: justify using sum of partial representations for direct reference and whether alternatives (e.g., attention) were considered.  
- Questions for authors: provide error analysis for lower performance on task 17 (Positional Reasoning) to pinpoint reasoning failures.
