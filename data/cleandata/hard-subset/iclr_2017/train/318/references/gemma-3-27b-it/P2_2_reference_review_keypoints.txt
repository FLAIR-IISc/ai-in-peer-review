- Paper introduces Gated Graph Transformer Neural Network (GGT-NN) for dynamically constructing and transforming graph-structured internal representations.  
- Core contribution: differentiable graph transformations (node add, state update, edge update, propagation, aggregation).  
- Demonstrated effectiveness on bAbI tasks, 1D cellular automaton, and Turing machine simulation with generalization to longer sequences.  
- Decision: Accept with minor revisions.  
- Novelty: Learning and manipulating graph structures within a neural network is a significant contribution.  
- Motivation: Strong rationale that many data types benefit from dynamic graph reasoning.  
- Empirical validation: Strong bAbI results and compelling automaton/Turing machine experiments showing learned transformations.  
- Clarity: Well-written with helpful pseudocode (Algorithm 1).  
- Ablation: Comparison with/without direct reference demonstrates the benefit of that feature.  
- Complexity: Need more thorough time and space complexity analysis, especially given quadratic scaling concerns.  
- Visualization: Recommend visualizing learned graphs during training for better interpretability.  
- Hyperparameter sensitivity: Paper lacks discussion of hyperparameter choices and their impact.  
- Error analysis: Suggest detailed error analysis on bAbI, especially task 17 where performance lags.  
- Code release: Code availability is positive; ensure it is well-documented and reproducible.  
- Questions for authors: strategies for scalability (sparse connections, sampling), methods to interpret contributions of each transformation, risks/alternatives to direct reference (e.g., attention), and causes of poor performance on task 17.
