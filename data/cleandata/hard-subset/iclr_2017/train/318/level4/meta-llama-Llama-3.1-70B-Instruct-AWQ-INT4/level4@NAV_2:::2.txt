This paper proposes learning on the fly to represent a dialog as a graph, which acts as the memory, and is first demonstrated on the bAbI tasks. Graph learning is part of the inference process, though there is long term representation learning to learn graph transformation parameters and the encoding of sentences as input to the graph. This seems to be the first implementation of a differentiable memory as graph: it is much more complex than previous approaches like memory networks without significant gain in performance in bAbI tasks, but it is still very preliminary work, and the representation of memory as a graph seems much more powerful than a stack.

**Summary of the Paper's Contribution**

The paper proposes a novel approach to learning graph-structured data, introducing the Gated Graph Transformer Neural Network (GGT-NN) model. This model can learn to construct and modify graphs in sophisticated ways based on textual input, and can use the graphs to produce a variety of outputs. The GGT-NN model is demonstrated on the bAbI tasks and shows promising results, particularly in tasks that require reasoning about temporal sequences of events.
