- Introduces Gated Graph Transformer Neural Network (GGT-NN) that combines graph-structured intermediate representations with a recurrent input model to incrementally build graphs from natural language.
- Extends Gated Graph Sequence Neural Network (GGS-NN) to allow graphs to be constructed and modified from unstructured input.
- Tackles the problem of learning graph-structured state transitions from textual input.
- Well-motivated approach that introduces differentiable graph transformations enabling graph->graph learning with gradient descent.
- Provides strong experimental evidence on bAbI tasks, a 1D cellular automaton, and a 4-state Turing machine, showing effectiveness and generalization.
- Paper is well-written with clear explanations of the model and thorough presentation of experimental results.
- Recommends acceptance based on novelty, motivation, and empirical results.
- Suggests adding more visualizations of learned graph structures and state transitions to clarify model behavior.
- Suggests deeper analysis of learned graph transformations and how they map to specific task rules (e.g., cellular automaton and Turing machine).
