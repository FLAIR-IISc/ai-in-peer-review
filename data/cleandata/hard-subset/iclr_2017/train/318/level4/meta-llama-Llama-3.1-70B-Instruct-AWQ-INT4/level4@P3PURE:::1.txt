The main contribution of this paper appears to be the introduction of a set of differential graph transformations that enable the learning of graph-to-graph classification tasks using gradient descent. This concept naturally maps to the task of learning a cellular automaton represented as a sequence of graphs. In this task, the graph of nodes expands at each iteration, with nodes pointing to neighbors and special nodes representing values (0/1). The proposed architecture allows for the learning of this sequence of graphs, although in the experiment, this task (Rule 30) was far from being fully solved.

This idea is combined with concepts from previous papers (GGS-NN) to enable the model to produce textual output rather than graph output, using graphs as an intermediate representation. This approach allows the model to achieve state-of-the-art results on the BaBi tasks.
