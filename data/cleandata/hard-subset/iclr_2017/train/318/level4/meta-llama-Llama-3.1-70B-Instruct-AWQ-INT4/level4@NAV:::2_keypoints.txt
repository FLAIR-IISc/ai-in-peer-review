- Introduces a novel dialog representation using an on-the-fly learned graph as memory, demonstrated on bAbI tasks.
- Integrates graph learning into inference and uses long-term representation learning for graph transformation parameters and sentence encoding.
- First differentiable-memory implementation as a graph, which is conceptually more complex than prior memory network approaches.
- Increased complexity does not deliver substantial performance gains on bAbI compared to top memory networks.
- Graph-based memory appears more expressive than a traditional stack and has untapped potential.
- Revised paper substantially improved clarity, making it accessible and a valuable contribution.
- Preliminary results do not conclusively show the graph memory outperforms alternatives in learning or generalization.
- Performance is comparable to top memory networks but still lags behind traditional rule induction methods.
- Overall technically sound, thought-provoking, and worthy of publication.
