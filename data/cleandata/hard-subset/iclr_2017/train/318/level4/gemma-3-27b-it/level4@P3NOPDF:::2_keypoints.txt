- Proposes a novel dialog representation: dialog history as a dynamically constructed graph serving as a differentiable memory.  
- Demonstrated initially on bAbI tasks.  
- Integrates graph learning into inference alongside long-term learning of graph transformation parameters and sentence encodings.  
- Claimed first implementation of a fully differentiable graph-based memory structure.  
- Conceptually more complex than prior memory network architectures.  
- Performance gains on bAbI are modest—competitive with memory networks but behind rule-induction methods.  
- Graph memory has strong potential for richer relational modeling in more complex dialog scenarios.  
- Manuscript improved substantially from the original submission; authors were responsive to feedback.  
- Some methodological aspects remain hard to follow—especially the interplay between learned graph transformations and sentence encodings.  
- Needs clearer, more intuitive explanation of the graph transformation process.  
- Lacks detailed justification of graph construction and traversal choices and their effect on performance.  
- Evaluation is limited to bAbI; broader testing on realistic, challenging dialog datasets is necessary.  
- Should discuss strategies for scaling to larger/complex dialog datasets.  
- Should include a brief discussion of computational cost compared to alternative approaches.  
- Overall assessment: technically sound, novel, and promising; recommend acceptance with minor revisions.
