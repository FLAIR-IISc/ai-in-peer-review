- Proposes a novel, fully differentiable graph-structured memory that is dynamically built and manipulated during inference for dialog reasoning.
- Claims this is the first implementation of a fully differentiable graph memory, a more complex approach than prior memory networks.
- Performance gains on bAbI tasks are modest, but the graph-based memory shows promising potential as preliminary work.
- Author substantially improved clarity between versions, making the paper more accessible.
- Recommendation: Accept — the paper is technically sound and thought-provoking.
- Demonstrates the GGT-NN model on diverse tasks (bAbI, cellular automaton, Turing machines), suggesting potential generalizability.
- Raises a complexity vs. performance concern: graph complexity does not yet yield clear advantages over simpler memory networks.
- Notes scalability issues: graph manipulation is computationally costly for large graphs and needs discussion of optimizations (e.g., sparse representations).
- Requests ablation studies to quantify the contributions of components like direct reference and intermediate propagation.
- Requests visualizations of learned graphs on example inputs to help interpret the model’s internal representations.
- Asks authors to clarify implementation limitations preventing superior bAbI performance and to describe plans for scalability improvements, ablations, and example graph outputs.
