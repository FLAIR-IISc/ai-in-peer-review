This paper proposes a method for representing a dialog in a graph (which acts as memory) for immediate learning, and it has been demonstrated in the first bAbI tasks. Graph learning is part of the process of inference, although there are also long-term representation learning tasks such as graph transformation parameters and learning sentence encoding as input to the graph. This seems to be the first implementation of a differentiable memory as a graph: it is much more complex than previous methods such as memory networks without significant performance improvement in bAbI tasks, but it is still a preliminary work, and representation of memory as a graph seems much more powerful than a stack. Clarity is a major problem, but from an initial version that was constructive and better read by a computer than by a human, the author later proposed a vastly improved version. This fundamental, technically sound (in my understanding) and thought-provoking paper is worthy of publication.
