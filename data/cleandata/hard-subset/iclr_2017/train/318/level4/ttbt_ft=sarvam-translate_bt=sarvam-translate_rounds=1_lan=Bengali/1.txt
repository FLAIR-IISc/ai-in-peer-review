The main contribution of this paper is the introduction of a set of differential graph transformations that will help you learn the graph->graph classification task using gradient descent. This is naturally related to a cellular automaton learning task represented as a sequence of graphs. In that task, the graph of nodes grows with each iteration, where the nodes point to their neighbors, and special nodes represent 0/1 values. The proposed architecture helps to learn this sequence of graphs, although in the experiment, this task (rule 30) was far from being solved.

This concept is combined with ideas taken from previous papers (GGS-NN) so that the model can generate textual output instead of graph output and use the graph as an intermediate representation, which helps it to beat state-of-the-art methods in BaBi tasks.
