This paper proposes a method for representing a dialog in a graph (which acts as memory) for immediate learning and has been demonstrated in the first bAbI tasks. The graph is part of the process of learning inference, although there are also long-term representation learning tasks such as learning graph transformation parameters and learning sentence encoding as input to the graph. It seems to be the first implementation of a differentiable memory as a graph: it is much more complex than previous methods such as memory networks without significant performance improvement in bAbI tasks, but it still seems to be a preliminary work and representation of memory as a graph is much more powerful than a stack. Clarity is a major problem, but from an initial version that was constructive and read better by computers than by humans, the author later proposed a vastly improved version. This fundamental, technically sound (in my understanding) and thought-provoking paper is worthy of publication.
