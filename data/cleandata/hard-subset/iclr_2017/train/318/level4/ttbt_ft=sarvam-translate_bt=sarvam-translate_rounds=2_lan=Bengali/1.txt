The main contribution of this paper is the introduction of a set of differential graph transformations that will help you learn the graph->graph classification task using gradient descent. This is naturally related to the cellular automaton learning task presented as a sequence of graphs. In that task, the graph of nodes grows with each iteration, where nodes point to their neighbors, and special nodes represent 0/1 values. The proposed architecture helps to learn this sequence of graphs, although in the experiment, this task was far from being solved (Rule 30).

This concept combines ideas taken from the previous paper (GGS-NN) so that the model can generate text output instead of graph output and use the graph as an intermediate representation, which helps it to defeat state-of-the-art methods in BaBi tasks.
