- Introduces a novel differentiable graph memory for dialog modeling, constructed dynamically during inference.  
- Presents the first implementation of graph-based differentiable memory and demonstrates it on the bAbI tasks.  
- Graph structure is more expressive than simpler memories (e.g., stacks/vectors) but substantially more complex.  
- Current implementation shows no clear performance gains on bAbI compared to standard memory networks.  
- Model learns graph transformation parameters and sentence encodings long-term, with graph construction integrated into inference.  
- Revised paper greatly improved clarity and readability over the initial draft.  
- Technical approach is sound and the core idea is thought-provoking and potentially impactful.  
- Preliminary empirical results do not establish advantages in learning capacity or generalization over existing methods.  
- Performance matches state-of-the-art memory networks but is inferior to rule-based baselines (e.g., Weston et al. 2015).  
- Main contribution is conceptual novelty: a dynamic, learnable graph memory for dialog rather than fixed structures.  
- Work is preliminary and focuses on feasibility rather than empirical superiority; complexity may not yet be justified.  
- Paper needs stronger contextualization versus prior memory approaches, including theoretical analysis or ablation studies.  
- Authors should clarify how the dynamic graph avoids pitfalls of static memory structures (e.g., fixed-size buffers).  
- Overall valuable and novel work that merits publication with minor revisions to address performance and theoretical comparisons.
