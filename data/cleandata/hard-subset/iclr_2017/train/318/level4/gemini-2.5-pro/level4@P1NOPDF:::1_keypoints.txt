- Introduces a set of differentiable graph transformations for graph-to-graph learning using gradient descent.
- Framework naturally maps to learning cellular automata represented as a sequence of growing graphs.
- Graph representation uses nodes pointing to neighbors and special nodes to encode binary values 0 and 1.
- Proposed architecture is intended to learn the graph sequence, but experiments (Rule 30) indicate the task remains far from solved.
- Combines this idea with concepts from GGS-NNs to use graphs as an intermediate representation for producing text.
- Using graphs as intermediates yields state-of-the-art performance on the bAbI tasks.
