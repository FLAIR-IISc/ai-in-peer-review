The main contribution of this paper is the introduction of a set of differentiable graph transformations designed to solve graph-to-graph learning tasks using gradient descent. This framework maps naturally to the task of learning a cellular automaton, which can be represented as a sequence of graphs. In this application, the graph grows at each iteration, with nodes pointing to their neighbors and special nodes representing the binary values 0 and 1. The proposed architecture is designed to learn this sequence of graphs, although the experimental results show that this task (specifically, Rule 30) was far from solved.

This idea is combined with concepts from previous papers (GGS-NNs), enabling the model to produce textual output rather than graph-based output by using graphs as an intermediate representation. This approach allows the model to achieve state-of-the-art performance on the bAbI tasks.
