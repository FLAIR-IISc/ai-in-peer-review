- Proposes dynamically representing dialogue as a learned graph serving as the model's memory.  
- Demonstrates the approach on the bAbI dialogue tasks.  
- Graph structure is learned during inference; model also learns long-term parameters for graph transformations and sentence encoding.  
- Appears to be the first implementation of a differentiable memory structured as a graph.  
- Model is significantly more complex than prior approaches (e.g., memory networks).  
- Does not show a clear performance improvement on bAbI compared to memory networks.  
- Conceptually promising: graph memory seems more powerful than simpler structures (e.g., stacks), though preliminary.  
- Initial submission had major clarity issues; revised version is substantially more readable.  
- Overall assessment: original, thought-provoking, technically accurate, and worthy of publication.  
- Empirical results are inconclusive about superior learning/generalization; performance matches best memory networks but lags behind traditional rule-induction methods.
