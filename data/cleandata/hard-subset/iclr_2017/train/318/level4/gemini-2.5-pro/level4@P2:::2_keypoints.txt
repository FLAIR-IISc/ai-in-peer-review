- Proposes Gated Graph Transformer Neural Network (GGT-NN): a differentiable model that builds and modifies an explicit graph memory from text.  
- Decision: Accept.  
- Novelty: First fully differentiable model that dynamically constructs an explicit graph as memory via composable graph transformations.  
- Empirical strength: Strong results on bAbI and successful rule discovery for cellular automata and Turing machines, showing broad applicability.  
- Architectural contribution: Five differentiable graph transformations provide a principled, general toolkit for mutable relational structure.  
- bAbI performance: Comparable to SOTA overall and significantly better on graph-native tasks (Task 19 pathfinding, Task 16 induction) vs supervised Memory Network baseline.  
- Rule-discovery results: Learns precise deterministic update rules (Turing machines, CAs), evidencing capability for symbolic-like computations.  
- Supervision limitation: Current implementation relies on strong, per-step ground-truth graph supervision, limiting end-to-end applicability.  
- Complexity concern: Quadratic time/space in node count; needs more prominent discussion and formal complexity analysis.  
- Direct reference mechanism: Provides major performance boost; should be compared to learned attention and explored as a learnable soft mechanism.  
- Visualization request: Add step-by-step graph visualizations (including failure cases) for bAbI to illustrate learned representations.  
- Ablation needs: Further ablations recommended (e.g., effect of propagation steps T_prop, necessity of each transformation).  
- Questions for authors: Plans to reduce supervision (semi-supervised or regularization approaches), explanation for poor Task 17 (positional reasoning) performance, and whether learned node/edge types are human-interpretable.
