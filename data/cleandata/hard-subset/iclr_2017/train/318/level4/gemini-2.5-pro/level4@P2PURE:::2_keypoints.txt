- Proposes a novel method that learns to represent a dialog as a dynamic graph, using the graph as the system's memory.
- Demonstrated the approach on the bAbI dialog tasks.
- The graph structure is learned on-the-fly during inference, while long-term parameters govern graph transformations and sentence encoding.
- Claimed to be the first implementation of a differentiable memory structured as a graph.
- Model is considerably more complex than prior approaches (e.g., memory networks).
- Does not yet show significant performance improvement on bAbI tasks compared to state-of-the-art memory networks.
- Preliminary results are insufficient to conclude that the graph-based memory yields better learning or generalization than existing methods.
- Conceptually, graph-structured memory is more powerful than simpler structures (e.g., stack) and shows significant potential.
- Initial submission suffered from clarity issues; the revised version is vastly improved.
- Overall assessment: original, technically sound, thought-provoking, and worthy of publication despite preliminary empirical gains.
