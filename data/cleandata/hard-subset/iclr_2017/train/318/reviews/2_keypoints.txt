- Proposes on-the-fly learning to represent dialog as a graph that serves as differentiable memory, demonstrated on bAbI tasks.
- Graph learning occurs during inference, with long-term learning for graph transformation parameters and sentence encodings.
- Appears to be the first implementation of a differentiable memory represented as a graph.
- Approach is substantially more complex than prior methods like memory networks.
- No significant performance gain on bAbI tasks compared to memory networks.
- Results are still preliminary and do not yet demonstrate superior learning or generalization capacity.
- Graph memory representation seems more powerful than simpler structures like a stack.
- Clarity was a major issue in earlier drafts, but a much improved later version was provided.
- Paper is technically accurate (as understood by reviewer) and thought-provoking.
- Reviewer considers the work constructive and worth publishing.
- Performance on bAbI is comparable to top memory networks but worse than traditional rule induction.
