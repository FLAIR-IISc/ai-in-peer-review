- Addresses few-shot learning by meta-learning an LSTM-based optimizer that tailors parameter updates for the few-shot regime.
- Frames few-shot learning as a sequential meta-learning task: sequences of training examples followed by test evaluation.
- Key insight: LSTM cell state represents base-learner parameters, with input/forget gates acting like adaptive learning rates and momentum.
- Method jointly learns base-learner initialization and task-specific parameter update strategy.
- Contribution 1: Novel interpretation of gradient-descent updates as LSTM cell-state updates.
- Contribution 2: Demonstrates LSTM meta-learner can learn optimization strategies that outperform standard baselines.
- Contribution 3: Achieves competitive results with state-of-the-art metric learning (Matching Networks) on Mini-ImageNet for 1-shot and 5-shot.
- Recommendation: Accept — approach is promising, well-motivated, and empirically convincing.
- Experiments are rigorous: matched meta-train/meta-test conditions, appropriate baselines, and detailed ablation studies.
- Provides practical implementation insights: initialization strategies, weight sharing, and gradient/loss preprocessing.
- Major weakness: clarity and presentation need improvement; task description is abstract and would benefit from concrete examples/diagrams.
- Unclear details: how N-class meta-training relates to the base learner softmax output size need clarification.
- Visualization and analysis are insufficient: Figure 3 gate plots lack interpretation of how learned gates improve optimization.
- Requests for authors: clarify N vs. softmax sizing and provide deeper analysis comparing learned updates to SGD with momentum, especially in 1-shot.
