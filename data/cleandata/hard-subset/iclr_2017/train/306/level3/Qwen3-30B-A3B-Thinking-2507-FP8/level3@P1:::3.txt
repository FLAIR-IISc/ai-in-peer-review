

This paper addresses the challenging few-shot learning problem by proposing a novel meta-learning approach where an LSTM-based meta-learner learns an optimization algorithm specifically tailored for the few-shot regime. The authors frame few-shot learning as a sequential meta-learning problem where each task consists of a sequence of training examples followed by a test evaluation. The key insight is that the LSTM meta-learner's cell state can directly represent the parameters of a base learner (a CNN classifier), with the input and forget gates learning adaptive update rules analogous to learning rates and momentum. This formulation allows the meta-learner to simultaneously learn an optimal initialization for the base learner and an effective parameter update strategy for the few-shot setting.

The paper makes several important contributions: (1) it provides a novel interpretation of gradient descent update rules as LSTM cell state updates; (2) it demonstrates that an LSTM meta-learner can learn task-specific optimization strategies that outperform standard baselines; (3) it achieves competitive results with state-of-the-art metric learning methods like Matching Networks on Mini-ImageNet for both 1-shot and 5-shot classification tasks.

I recommend acceptance. The paper makes a compelling case that meta-learning the optimization algorithm itself is a promising approach for few-shot learning, with results that are both convincing and competitive with recent work. The key insight about the relationship between LSTM memory cells and gradient descent updates is novel and well-justified through empirical results. The approach is well-motivated by the limitations of standard gradient-based optimization in few-shot settings and properly positioned within the meta-learning literature.

The results are scientifically rigorous: the authors carefully design their experiments to match meta-training and meta-testing conditions, employ appropriate baselines (nearest-neighbor and fine-tuning), and provide detailed ablation studies. The competitive performance against Matching Networks (which had previously achieved state-of-the-art results) is particularly convincing. The paper also provides valuable practical insights for implementing LSTM meta-learners, including initialization strategies, weight sharing, and preprocessing techniques for gradients and losses.

However, the paper would benefit from significant improvements in clarity and presentation. The task description is abstract and would be greatly improved with a concrete example and diagram illustrating the few-shot learning setup. The details of the N-class few-shot setup (which classes are seen during meta-training, how the softmax output size is determined) are unclear and need clarification. The writing is often opaque, particularly regarding how the meta-learner's output connects to the base learner's parameters. The computational graph in Figure 2 is uninformative, and the discussion around it lacks insight into how the meta-learner's learned parameters actually function.

The paper would also benefit from more detailed analysis of the meta-learner's learned behavior. The visualization in Figure 3 shows gate values but doesn't provide meaningful interpretation of how these values lead to better performance. A more thorough discussion of the relationship between the LSTM gates and the optimization process would strengthen the paper's contribution.

I have two specific questions for the authors: (1) Could you clarify how the number of classes N in the meta-training set relates to the softmax output size of the base learner network? (2) Could you provide more detailed analysis of how the meta-learner's learned update rules differ from standard optimization algorithms like SGD with momentum, particularly in the 1-shot setting where the performance gap with Matching Networks is smallest?

The paper presents a compelling approach to few-shot learning that is both theoretically interesting and empirically effective. While the current presentation needs improvement, the core contribution is strong and the results are convincing. With minor revisions to improve clarity and presentation, this would be a valuable contribution to the meta-learning literature. The authors have successfully demonstrated that learning an optimization algorithm specifically for the few-shot regime is a promising direction, and their LSTM-based meta-learner provides a practical and effective solution that achieves competitive performance with state-of-the-art methods.
