- Novel approach: frames few-shot learning as sequential meta-learning using an LSTM meta-learner whose cell state represents base-learner parameters and whose updates mimic gradient descent.  
- Empirical claim: LSTM meta-learner learns good initialization and update rules, achieving competitive performance and outperforming Matching Networks on 1- and 5-shot Mini-ImageNet (notably in 5-shot).  
- Practical contributions: provides training insights (initialization strategies, weight sharing analogous to shared learning rates, normalization techniques).  
- Conceptual contribution: highlights a novel connection between LSTM memory cells and gradient-descent-style updates.  
- Decision: Accept — originality, effectiveness, and empirical results justify acceptance despite presentation issues.  
- Distinctness: method differs from prevalent metric-learning approaches, offering a valuable alternative route to few-shot learning.  
- Experimental support: Mini-ImageNet results substantiate the paper’s claims.  
- Rebuttal effect: reviewer score increased after authors clarified the meta-learning task, showing responsiveness improved confidence.  
- Clarity issue: Task description (Section 2) is abstract and needs a concrete example and a visual diagram of an episode.  
- Experimental detail missing: N-class few-shot setup unclear — how classes are selected for meta-training/validation/testing and whether there is overlap.  
- Model detail missing: unclear what the softmax output layer size is for the base learner in each N-class task.  
- Figure critique: Figure 2 is uninformative; plots show no clear patterns and lack insightful interpretation.  
- Writing quality: prose is often opaque and should be rewritten for clarity and conciseness.  
- Question for authors: Provide a specific example illustrating a complete meta-training episode and data flow between base learner, meta-learner, and loss.  
- Question for authors: Clarify class selection strategy and whether classes are disjoint across meta-train/val/test.  
- Question for authors: State the softmax output layer size of the base learner for each task (N classes).  
- Question for authors: Explain what Figure 2 is intended to convey and whether any gate-value patterns correlate with successful or unsuccessful episodes.  
- Question for authors: Have you ablated the gradient-independence assumption, and would including second-order gradients materially affect performance despite added cost?
