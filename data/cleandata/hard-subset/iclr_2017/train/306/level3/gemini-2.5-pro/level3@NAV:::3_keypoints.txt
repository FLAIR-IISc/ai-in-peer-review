- Proposes framing few-shot learning as a sequential meta-learning problem using an LSTM-based meta-learner.  
- Core contribution: LSTM meta-learner that learns an optimization algorithm for a base ConvNet classifier.  
- Insightful connection drawn between LSTM cell updates and gradient descent, using LSTM memory to model base-learner parameters.  
- Meta-learner learns both parameter initialization and an effective update strategy for fast convergence from few examples.  
- Recommendation: accept, based on novelty and strong empirical results.  
- Empirical strength: competitive results on Mini-ImageNet and superior performance in the 5-shot case versus Matching Networks.  
- Strengths: elegant formulation, convincing experimental validation, and demonstration that meta-learning an optimizer is viable for SOTA performance.  
- Practical contributions: useful training details (weight sharing, initialization) that aid reproducibility and future work.  
- Major revision needed: improve clarity and presentation—writing is opaque and task description is highly abstract.  
- Suggestion: include a concrete, step-by-step example of a single training episode and add a diagram to improve accessibility.  
- Criticism: Figure 2 plots are uninformative and should be revised or replaced to better illustrate the learned optimization behavior.  
- Clarification requested: explain handling of the final softmax layer when N (number of classes) varies between meta-training and meta-testing or across episodes.  
- Clarification requested: state whether classes seen during meta-training are disjoint from those used in meta-testing and provide clearer class split details.
