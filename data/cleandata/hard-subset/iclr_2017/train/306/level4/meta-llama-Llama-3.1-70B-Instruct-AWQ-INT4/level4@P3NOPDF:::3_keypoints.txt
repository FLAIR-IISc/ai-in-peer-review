- Reviewer updated score to 8 after authors' responsiveness and clearer meta-learning task description.  
- Paper formulates few-shot learning as a sequential meta-learning problem where each example is a sequence of training batches followed by a test batch.  
- Uses an LSTM meta-learner that takes base-learner outputs and parameters as inputs and uses second-layer memory cells to model parameter updates.  
- Motivates LSTM use by analogy between LSTM memory updates and gradient descent.  
- Meta-learner is trained using loss on the final test batch and assumes weight sharing across second-layer cells (like a shared learning rate).  
- Recreates Mini-ImageNet and achieves competitive/outperforming results vs. Matching Networks on 1- and 5-shot tasks.  
- Base learner is a simple ConvNet classifier (not embedding + nearest-neighbor), showing a different successful approach.  
- Novel and interesting insight about relationship between LSTM memory cells and gradient-descent updates.  
- Provides practical, replicable training/design insights (initialization, coordinate weight sharing, normalization/rescaling of loss/gradients/parameters).  
- Writing is sometimes opaque and the abstract task description is hard to follow; a concrete example and diagram would help.  
- Unclear details about the N-class few-shot setup and Mini-ImageNet splits — which classes are seen during meta-training and softmax size are ambiguous.  
- Figure 2 plots are not self-explanatory and the discussion of them is limited.  
- Overall: interesting paper with convincing results and a clear accept recommendation, contingent on improved presentation.
