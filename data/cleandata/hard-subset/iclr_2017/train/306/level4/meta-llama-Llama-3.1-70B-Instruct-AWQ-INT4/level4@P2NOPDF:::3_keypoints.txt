- Reviewer updated score to 8 after authors' responsiveness and clarification of the meta-learning task.
- Paper frames few-shot learning as a sequential meta-learning problem where each example is a sequence of training batches followed by a test batch.
- Inputs to the meta-learner include base-learner outputs (loss, gradients) and the base learner's current parameters.
- An LSTM meta-learner models base-learner parameter updates via its inner memory cells, noted to be analogous to gradient descent updates.
- The meta-learner is trained using the base learner's prediction loss on the final test batch.
- Authors make simplifying assumptions such as sharing weights across second-layer LSTM cells (like using the same learning rate for all parameters).
- Experiments recreate Mini-ImageNet and show the LSTM meta-learner is competitive with or outperforms Matching Networks on 1- and 5-shot tasks.
- Using a simple ConvNet base learner (rather than nearest-neighbor on learned embeddings) makes the results more compelling.
- The relationship between LSTM memory cells and gradient descent is an interesting, novel insight.
- Paper provides practical implementation tips (initialization, coordinate weight sharing, normalization/rescaling) that aid replication.
- Writing is often opaque and would benefit from a concrete example and a diagram to improve accessibility.
- Details about the N-way few-shot setup are unclear (e.g., Mini-ImageNet label splits and softmax output size during meta-training).
- Figure 2 plots are uninformative and the discussion around them lacks insight.
- Overall assessment: interesting and convincing work, clear accept if presentation is improved; reviewer may raise score with better writing.
