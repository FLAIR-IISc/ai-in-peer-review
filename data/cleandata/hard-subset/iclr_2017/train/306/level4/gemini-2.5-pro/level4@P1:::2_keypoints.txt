- Core idea: Reframe optimization as a learnable task by training an LSTM-based meta-learner to produce parameter updates for a conventional neural network learner.  
- Decision: Accept.  
- Main strengths: Original, elegant conceptual contribution framing the optimizer itself as a learnable model.  
- Empirical strength: Strong results on Mini-ImageNet, competitive with or superior to state-of-the-art metric-learning approaches.  
- Mechanism: Meta-learner learns both a shared initialization and a dynamic per-parameter update rule enabling fast adaptation with few examples.  
- Interpretation: The LSTM-style gates provide per-parameter adaptive learning rates and learned regularization, directly optimizing for rapid generalization.  
- Experimental comparisons: Solid baselines (fine-tuning, nearest-neighbor) and Matching Networks were used, showing clear benefits over standard fine-tuning.  
- Practical contributions: Useful implementation insights (batch normalization handling, meta-learner initialization) and commitment to release code.  
- Model formulation concern: The proposed update resembles a GRU more than an LSTM; the paper should acknowledge or justify using an LSTM framing.  
- Clarity issue: Description of class partitioning (meta-train/val/test) is hard to follow; a diagram of the data split would improve comprehension and Figure 2 could be replaced.  
- Experimental suggestion: Include ablation studies to isolate contributions of the input and forget gates (e.g., fix ft=1 or replace it with conventional adaptive rates).  
- Minor points: Fix small typo ("each coordinate has it" → "each coordinate has its") and note appreciation for promising code release.
