- Proposes framing few-shot learning as sequential meta-learning using an LSTM meta-learner that updates a base learner from loss, gradients, and parameters.
- Draws a novel parallel between LSTM memory cell updates and gradient descent, offering a new perspective on meta-learning.
- Achieves competitive or state-of-the-art results on 1- and 5-shot Mini-ImageNet, matching/surpassing Matching Networks.
- Uses a fundamentally different base learner (CNN) rather than embedding+nearest-neighbor approaches.
- Provides practical implementation guidance (initialization, weight sharing, input normalization) useful for training LSTM meta-learners.
- Writing and presentation are dense and unclear, especially Section 2’s abstract notation, reducing accessibility.
- Experimental setup details are ambiguous: unclear whether only 64 classes are seen during meta-training and how the base learner’s softmax output size is handled.
- Figure 2 plots are limited and under-discussed, offering little interpretive value.
- Overall: compelling and impactful work with convincing results, recommended for acceptance if clarity and experimental/setup explanations are improved.
