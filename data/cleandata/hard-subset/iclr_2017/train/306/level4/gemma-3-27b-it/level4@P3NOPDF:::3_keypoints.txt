- Frames few-shot learning as a sequential meta-learning problem using an LSTM meta-learner to model parameter updates of a CNN base learner.
- Demonstrates competitive or superior performance on 1-shot and 5-shot Mini-ImageNet, outperforming Matching Networks in some cases.
- Presents a novel and insightful analogy between LSTM memory cell updates and gradient descent.
- Offers practical guidance for designing and training LSTM meta-learners (initialization, weight sharing, input normalization).
- Provides a valuable contrast to embedding-based approaches by using a convolutional base learner.
- Presentation is unclear and dense (abstract and Section 2); needs a concrete example and a clear diagram to improve accessibility.
- Ambiguity in the N-class few-shot task description and Mini-ImageNet usage—unclear handling of unseen classes and softmax output sizing.
- Figure 2 visualizations are not informative and lack sufficient interpretation.
- Overall: valuable technical contribution; reviewer updates score to 8 and recommends improving clarity to raise the score further.
