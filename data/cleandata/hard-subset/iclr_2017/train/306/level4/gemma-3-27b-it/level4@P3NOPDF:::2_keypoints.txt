- Novel framing: SGD update rule is interpreted as a gated recurrent model with trainable parameters, offering an original meta-learning perspective.
- Feasible and promising: The approach is plausible and opens opportunities for learning task-specific update rules.
- Competitive results: Empirical performance is competitive with state-of-the-art meta-learning algorithms.
- Practical guidance: Paper provides useful implementation recommendations that enhance applicability.
- Architecture clarity: The model is more akin to a GRU than an LSTM; the absence of an LSTM-like cell state should be clarified.
- Meta-set separation unclear: Description of meta-training vs. meta-testing data splits is hard to follow; a visual illustration is recommended.
- Limited sensitivity analysis: Missing thorough experiments on key hyperparameters (notably the learned update parameters 'it' and 'ft') to assess robustness.
- Figure 2 weak: Figure 2 adds little insight and its inclusion should be reconsidered or improved.
- Minor typo: Section 3.2 contains "it" where "its" is intended.
- Code release positive: Authors’ promise to release code is encouraging for reproducibility.
- Overall recommendation: Promising, recommend acceptance conditional on addressing clarity, parameter analysis, and figure issues.
