- Reviewer raised score to 8 after authors clarified the meta-learning task and updated the manuscript.  
- Paper frames few-shot learning as a sequential meta-learning problem where each example is a sequence of training batches followed by a test batch.  
- Inputs at each step include the base learner’s outputs (loss, gradients) and its current parameters; an LSTM models this sequence.  
- Second-layer LSTM memory cells directly model the base learner’s updated parameters, with parallels drawn to gradient descent.  
- Meta-learner is trained using the base learner’s test-batch prediction loss.  
- Key simplification: weight sharing across second-layer cells (analogous to a uniform learning rate).  
- Authors recreate Mini-ImageNet and obtain competitive state-of-the-art results on 1- and 5-shot tasks.  
- Using a simple ConvNet base learner (rather than nearest-neighbor) shows strong performance via a distinct methodology.  
- Insight linking LSTM memory updates to gradient descent is novel and compelling.  
- Paper provides practical implementation guidance (initialization, weight sharing, input normalization), aiding reproducibility.  
- Writing and notation are unclear in places; abstract task description uses non-standard language and is hard to follow.  
- Problem definition for Mini-ImageNet is ambiguous (unclear whether meta-training sees only 64 classes or all 100, and implications for softmax size).  
- Figure 2’s plots are uninformative and the discussion around them offers little insight.  
- Overall: strong empirical results and technical contributions but requires significant presentation and clarity improvements.
