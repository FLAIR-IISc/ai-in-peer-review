- Explores automated architecture search; currently computationally intensive but trade-off likely to improve.  
- Evaluates both vision and text tasks across many benchmarks, showing gains beyond standard RNN/CNN search spaces.  
- Demonstrates architectures competitive with or surpassing human designs, enabling dataset-specific architectures without heavy hand-engineering.  
- Well-written paper with strong results; reviewer recommends acceptance.  
- Proposes Neural Architecture Search using reinforcement learning with a recurrent controller — novel and well-motivated.  
- CIFAR-10 results: competitive error rate, outperforms prior similar models and offers speed improvements.  
- Penn Treebank results: discovered recurrent cell outperforms LSTMs and transfers well to character language modeling with substantial perplexity gains.  
- Paper is well-structured, clear methodology, includes control experiments, and acknowledges limitations and future directions.  
- Requests more transparency on CIFAR-10 grid search hyperparameter ranges (learning rates, weight decays, batchnorm epsilons, decay epochs).  
- Requests Penn Treebank details on dropout rates (recurrent, embedding, etc.) and whether embedding sharing was used.  
- Asks for a detailed description of the weight initialization scheme and release of a base hyperparameter set for reproducibility.  
- Parameter server setup description is brief; requests more detail on synchronization strategy and effects of asynchronous updates on stability.  
- Recommends clarifying “skip connections” and how the controller predicts them, ideally with an illustrative example.
