- Paper addresses automating architecture search; currently computationally intensive but expected to improve with technology.
- Strong empirical results on both vision and text benchmarks, demonstrating gains beyond conventional RNN/CNN search spaces.
- Method can match or surpass human-designed architectures, suggesting dataset-specific automated tailoring without heavy hand-engineering.
- Well-written paper with a clear contribution; reviewer recommends acceptance.
- Novel formulation: neural architecture design framed as a reinforcement learning task using a recurrent network controller.
- Achieves state-of-the-art or near state-of-the-art from scratch (e.g., 3.65% error on CIFAR-10, 62.4 perplexity on Penn Treebank).
- Approach is flexible, supporting skip connections and automatic generation of recurrent cell architectures.
- Reviewer requests more detailed hyperparameters for each experiment, including specific dropout types (e.g., recurrent, embedding).
- Suggestion to release a base set of hyperparameters (as in Zaremba et al.) to aid reproducibility.
- Request for detailed description of the grid search process, including the ranges explored for each hyperparameter.
- Question about the parameter server: what was the network bandwidth between controller replicas and the server, and was bandwidth limiting?
- Request precise training settings (learning rate, dropout, batch size, etc.) used for the character language model achieving 1.214 perplexity.
