This paper presents a new framework for solving SR problems – an abstracted MAP estimation and a pre- अनुमानित learning affinity projection layer to ensure that the output is consistent with the LR. It also proposes three different methods to solve the problem of reducing cross-entropy. Overall, it is an excellent paper. However, I have a few comments:

1) The proposed abstracted MAP estimation is new and different from previous SR methods. Combined with GAN, this framework can achieve commendable and good results. Compared to another GAN-based SR method – using a photo-realistic single image super-resolution using a Generative Adversarial Network – the question may arise as to what this new formulation adds to the latest technology.
2) Using an affinity projection architecture as an obstacle, the model does not require any corresponding {HR, LR} image pairs for training. However, when training the affinity projection layer, we still need {HR, LR} image pairs. Does this mean we transfer this training process to affinity projection training?
3) The paper presents several results of the framework, including results from natural images from ImageNet. Can the authors also provide results from Set5, Set14, or BSD100, which are traditional test datasets for SR, so that we can make a proper comparison with previous work?
4) I noticed that the size of the nature image results presented in this paper is limited to 128 * 128. Can this framework perform well on larger image sizes? Because SR will have to face input of arbitrary size.
5) In a typical GAN, a noise word will be a hidden place, so that it can be better represented as a distribution learning. Do the authors try a noise vector?

Overall, this paper provides a new framework for SR with solid theoretical analysis. The idea is new, and the authors explore several methods. However, questions such as the need for more experimentation still exist. I think this work will give good inspiration to the community.
