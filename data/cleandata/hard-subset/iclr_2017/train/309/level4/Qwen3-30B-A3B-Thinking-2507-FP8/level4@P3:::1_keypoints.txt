- Introduces UNREAL: an RL agent that jointly learns auxiliary control tasks (pixel/feature-based) alongside primary reward maximization.  
- Proposes a unified framework where auxiliary tasks build a shared representation and a reward prediction loss focuses task-relevant features.  
- Empirical results: UNREAL achieves 880% human-normalized performance on Atari, surpassing prior state-of-the-art.  
- Empirical results: On Labyrinth, UNREAL reaches 87% human performance with a 10× learning speedup over A3C across 13 levels.  
- Ablation studies show auxiliary tasks are necessary — removing them degrades performance significantly.  
- Core novelty: the reward prediction mechanism that accelerates representation learning without biasing the main policy.  
- Key hypothesis: controlling sensorimotor features (auxiliary control tasks) yields more useful representations than reconstruction/prediction.  
- Validation is thorough: Atari results over 57 games and 50 hyperparameter settings; Labyrinth evaluation includes diverse levels and ablations.  
- Analysis shows reward prediction (with skewed sampling) accelerates reward-relevant feature learning and combines synergistically with pixel/feature control.  
- Recommendation: Accept — paper makes a strong contribution to representation learning in RL with robust empirical support.  
- Minor feedback: clarify UNREAL’s computational cost versus A3C (per-step/episode time and memory overhead).  
- Minor feedback: justify specific auxiliary task design choices (e.g., 20×20 pixel grids) and discuss task-design trade-offs.  
- Minor feedback: provide analysis for why pixel control outperforms feature control (e.g., perceptual salience of feature units).  
- Questions for authors: how does the 3-frame immediate reward prediction scale to environments with long-delayed rewards (10+ steps)?  
- Questions for authors: would combining feature control with learned/self-supervised feature extractors improve performance further?
