- Proposes UNREAL: augment A3C with auxiliary tasks (auxiliary control, reward prediction, value replay) that shape a shared CNN-LSTM representation.  
- Decision: Accept.  
- Core contribution: combining on-policy main-task learning with off-policy auxiliary control/prediction tasks to improve representation learning and data efficiency.  
- Empirical result: strong performance on Atari and 3D Labyrinth with significant final-score gains and ~10x mean speedup in learning.  
- Novelty: framing auxiliary tasks as separate control problems (e.g., pixel control) encourages learning features tied to controllability rather than passive prediction.  
- Ablations: Labyrinth ablation studies show each auxiliary component contributes meaningfully, supporting the architecture design.  
- Robustness: UNREAL shows improved learning speed and robustness to hyperparameters compared to baselines.  
- Computational-cost request: paper should report wall-clock training overhead and which auxiliary components are most expensive.  
- Clarity request: clarify that each auxiliary control task learns a distinct policy (avoid language implying a single composite policy).  
- Feature-control concern: empirical validation for feature control is weak and may be unstable due to non-stationary targets—authors should comment or stabilize it.  
- Hyperparameter λ concern: justify how auxiliary loss weights (λ_VR, λ_PC, λ_RP) were chosen and whether they were tuned.  
- Training-duration suggestion: run top agents longer since curves still trend upward to reveal ultimate performance.  
- Atari ablation suggestion: include detailed ablation studies on Atari similar to Labyrinth to confirm consistency across domains.  
- Off-policy clarification: clarify that auxiliary tasks use an off-policy n-step return variant rather than standard Q-learning.  
- Minor fixes: correct notation/typos (policy-loss sign, λ_C placement, return notation r(c)t+k).  
- Figure/reference fixes: fix missing/incorrect references (Figure 1(d), Figure 2, Figure 3 panels, Figure 5 level counts) and clarify legend terms (e.g., "Clip").  
- Grammar/style fixes: correct sentences like "the features discovered in this manner is shared" → "are shared".  
- Appendix clarity: state where supplementary materials are located.
