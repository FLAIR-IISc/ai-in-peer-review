- Proposes UNREAL: augment A3C with domain-independent auxiliary tasks (pixel/feature control, reward prediction, value replay) to shape shared CNN+LSTM representations.  
- Auxiliary tasks shape representation only (except value replay) and are not executed as options.  
- Experiments on Labyrinth and 57 Atari games show large performance gains, improved robustness, and much better data efficiency than A3C.  
- Decision: Accept.  
- Key reasons for acceptance: well-motivated placement in literature and strong, convincing empirical results (data efficiency and robustness).  
- Problem/motivation: tackles sparse extrinsic reward by using auxiliary RL objectives better aligned with control than reconstruction.  
- Approach: combines on-policy A3C with off-policy n-step Q-learning on replay, reward prediction with rebalanced sampling, and value replay; includes deconvolutional heads for pixel control.  
- Empirical support: UNREAL doubles human-normalised Labyrinth performance vs A3C and achieves up to 18× speedups; on Atari achieves 880% mean and 250% median human-normalised performance.  
- Ablations on Labyrinth help isolate individual component contributions.  
- Request to quantify computational cost: report wall-clock slowdown and which auxiliary components dominate.  
- Clarify wording: explicitly state separate policies are learned for each auxiliary control task in abstract/intro.  
- Feature-control: empirical gains are limited; discuss instability from changing target features and possible mitigations (slower updates, target networks, layer selection).  
- Training horizon: run best agents longer to determine asymptotic behavior and whether performance gaps change.  
- Auxiliary loss weighting: explain why λ-weights were not tuned per domain and whether chosen values were validated.  
- Off-policy updates: explicitly note auxiliary control uses off-policy n-step Q-learning rather than 1-step off-policy Q-learning.  
- Minor correction: describe policy gradient loss as a loss to be minimized (sign error).  
- Equation fixes: consider placing λc inside the sum in Eq. 1 and fix r(c) t → r(c) t+k.  
- Figure and reference fixes: reference Figure 2; cite Figure 1(d) in §3.3; correct Figure 3 references/labels and clarify “Clip” in legend.  
- Grammar and notation fixes: fix subject-verb agreement (“are shared”), define LPC around Eq. 2 and ensure consistent notation.
