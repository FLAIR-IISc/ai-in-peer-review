- Introduces a novel RL training approach using auxiliary tasks to learn more robust features.  
- Proposes two pseudo-control tasks: controlling pixel-intensity changes and activation of latent features.  
- Adds a supervised reward-prediction regression task learned offline via skewed sampling from the replay buffer.  
- Agents with these auxiliary tasks reach baseline performance in roughly 10x fewer iterations on discrete-action–continuous-space tasks.  
- Approach contrasts with passive unsupervised/model-based methods by leveraging agent control to learn useful representations.  
- Method is innovative and intuitively explained, though some notation (e.g., L_PC) should be introduced earlier rather than only in the appendix.  
- Methodology is sound; using top-3 measurements helps find strong hyperparameters if they lie within the search ranges.  
- Main limitation: insufficient in-depth experimental analysis of how each auxiliary task (especially pixel and feature control) affects learning.  
- Empirical nuance: in Labyrinth, A3C+PC outperforms most methods except UNREAL, suggesting task-specific effects worth probing.  
- Recommendation: perform deeper analyses and metrics beyond RL performance to understand the mechanisms and contributions of auxiliary tasks.
