- Novel approach: agents learn multiple pseudo-reward (auxiliary) tasks alongside the primary reward to build richer representations.  
- Recommendation: reviewer recommends accepting the paper.  
- Empirical gains: achieves state-of-the-art results on Atari and Labyrinth with markedly improved sample efficiency.  
- Motivation: addresses sparse extrinsic rewards by providing richer learning signals.  
- Auxiliary tasks: introduces pixel change, network feature control, and reward prediction as auxiliary objectives.  
- Integration: auxiliary tasks are unified with the A3C algorithm into the UNREAL agent.  
- Performance: UNREAL consistently outperforms A3C and baselines in final performance and learning speed.  
- Labyrinth results: especially large improvements in average/median performance and learning speed on Labyrinth.  
- Robustness: includes hyperparameter sensitivity analysis that strengthens the contribution.  
- Suggestion — losses: paper should state specific loss functions (e.g., L_PC) more explicitly in the main text.  
- Suggestion — ablations: provide deeper analysis of individual auxiliary tasks and their interactions; pixel control’s strong effect in Labyrinth needs exploration.  
- Suggestion — representation metrics: evaluate learned representations with metrics beyond RL performance (transfer, visualizations).  
- Question — reward-prediction skew: how was the 0.5 skewing ratio chosen and is it environment-sensitive?  
- Question — feature-control layer: how was the hidden layer for feature control selected and were alternatives tested?  
- Question — compute overhead: what's the computational/training-time cost per environment step versus A3C?  
- Question — result consistency: are Labyrinth performance gaps consistent across random seeds and hyperparameter settings?
