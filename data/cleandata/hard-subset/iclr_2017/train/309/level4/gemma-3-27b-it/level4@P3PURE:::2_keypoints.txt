- Overall assessment: paper is well-written, ideas are sound, results convincing, recommendation to accept.  
- Method: augmenting policy optimization with auxiliary tasks (control, reward prediction, value replay) to shape shared CNN+LSTM features.  
- Empirical claim: improved data efficiency on Atari and Labyrinth compared to A3C.  
- Computational cost: request to report additional training cost and which components dominate runtime.  
- Multi-policy clarification: abstract/introduction should explicitly state distinct policies are learned per task to avoid misinterpretation.  
- Feature control validation: current empirical support is weak and may raise stability/convergence concerns as features and tasks co-evolve.  
- Extended training: recommend continuing training for top agents since performance was still improving.  
- Auxiliary task weights: question fixed λ_* = 1; suggest treating these as tunable hyperparameters and report experiments justifying the choice.  
- Off-policy training: note auxiliary tasks are trained off-policy with multi-step empirical rewards, not “true” Q-learning—please acknowledge.  
- Request ablations: more Atari ablations to assess individual task contributions similar to Labyrinth analyses.  
- Minor correction: state that policy gradient loss should be described as minimizing L_pi = -… not maximizing.  
- Equation/notation fixes: include λ_c inside summation in Eq.1; change rt^(c) to rt+k^(c) below Eq.1; clarify value of λ_PC (assumed 1).  
- Missing/undefined terms: L_PC referenced in text is not defined or shown in Eq.2—fix and define.  
- Figure references and legends: reference Figure 2 in text; cite Figure 1(d) in Section 3.3; clarify Figure 3 legend percentages refer to Labyrinth; explain “Clip” in dueling-networks legend.  
- Figure labeling errors: correct several incorrect figure references (e.g., “Figure 3 (right)” actually top-left; “Figure 3 Top” should be “Top Right”; Figure 5 caption/text inconsistencies).  
- Grammar/typos: fix “the features discovered in this manner is shared” → “are shared” and other awkward phrasing (Section 4.2 sentence).  
- Appendix/supplementary: specify where supplementary materials are located.  
- Acknowledgement: some issues already addressed in OpenReview comments, per reviewer edit.
