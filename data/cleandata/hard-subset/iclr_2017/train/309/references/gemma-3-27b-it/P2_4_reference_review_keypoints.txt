- Summary: Introduces UNREAL, an A3C-based agent using pixel control, feature control, and reward prediction auxiliary tasks to improve representation learning and RL performance.  
- Decision: Recommend accept with minor revisions due to strong empirical contribution and clarity.  
- Strong empirical results: Demonstrates >2x improvement on Labyrinth and significant gains on Atari, with robust performance across hyperparameters.  
- Well-motivated approach: Auxiliary tasks are justified as promoting robust feature learning and better exploration, building on prior representation work.  
- Comprehensive ablations: Ablation studies show each auxiliary task contributes and that pixel control outperforms pixel reconstruction and immediate reward prediction.  
- Clear presentation: Paper is well-written, organized, and supplemented by a helpful agent performance video.  
- Computational cost: Request for discussion on added computational/training time overhead from auxiliary losses and network complexity.  
- Reward prediction details: Ask for exact implementation specifics (sampling ratios, network architecture) for the skewed reward prediction.  
- Hyperparameter sensitivity: Suggest systematic sensitivity analysis for λVR, λPC, and λRP to quantify robustness.  
- Error bars: Recommend including error bars on learning curves to show statistical significance.  
- Learning rate/entropy sampling: Ask why log-uniform sampling was chosen and how sampling ranges were determined.  
- Deconvolution choice: Request motivation for using a deconvolutional network for pixel control instead of simpler linear projections.  
- Scaling auxiliary tasks: Question whether adding more auxiliary objectives yields diminishing returns or harms performance.  
- Reward prediction bias: Ask how λRP was tuned to avoid biasing the main policy and causing detrimental effects.
