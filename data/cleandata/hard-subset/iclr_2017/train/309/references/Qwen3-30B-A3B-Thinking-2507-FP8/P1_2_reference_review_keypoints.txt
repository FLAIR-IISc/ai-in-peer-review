- Paper introduces UNREAL: an RL agent that adds auxiliary tasks (pixel/feature control, reward prediction) and prioritized replay to improve representation learning.
- UNREAL builds on A3C and integrates auxiliary control tasks, an auxiliary reward prediction task, and prioritized experience replay.
- Pixel control learns to maximize changes in visual inputs; feature control targets network activations; reward prediction shapes representations via predicting immediate rewards.
- Authors report large empirical gains: Labyrinth 87% vs A3C 54% human-normalized mean and ~10x learning speedup.
- On 57 Atari games UNREAL achieves 880% mean and 250% median human-normalized performance, outperforming prior state-of-the-art.
- Ablation studies show each auxiliary component contributes and pixel/feature control outperform simple reconstruction or change prediction baselines.
- UNREAL is reported to be more robust to hyperparameter choices than vanilla A3C (robustness analysis highlighted).
- Literature review positions UNREAL as focusing on representation learning rather than using auxiliaries solely for planning or separate value functions.
- Recommendation: reviewer recommends Accept based on significant contribution, strong results, and thorough experiments.
- Suggestion: clarify how the shared/unified representation is constructed and visualized within the architecture.
- Suggestion: more explicitly explain interaction between reward prediction loss and A3C loss and why reward prediction doesn’t bias the policy.
- Suggestion: expand analysis on why feature control works and what features are being controlled.
- Suggestion: add visualizations showing what auxiliary tasks focus on (e.g., pixel control attention maps).
- Suggestion: clarify the “more than twice” performance claim (e.g., 87% vs 54%) and which metric is being compared.
- Questions for authors: details on baseline implementations (pixel reconstruction, input change prediction) and how they differ from reward prediction.
- Questions for authors: precise training procedure for the reward prediction task and how L_RP is applied.
- Questions for authors: how reward prediction differs mechanistically from adding an auxiliary loss to the main RL objective.
- Questions for authors: how UNREAL behavior compares to an optimal exploration/exploitation policy in procedurally-generated navigation levels.
- Questions for authors: how human-normalized Atari scores were computed and normalized for the 880% claim.
