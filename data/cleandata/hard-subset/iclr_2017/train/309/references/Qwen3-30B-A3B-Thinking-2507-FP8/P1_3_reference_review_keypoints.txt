- Proposes UNREAL: an agent using auxiliary tasks (pixel control, feature control) and reward prediction to improve RL representations.
- Uses auxiliary control tasks to maximize changes in visual regions (pixel control) or network features (feature control).
- Introduces reward prediction as an auxiliary task to shape representations without biasing the main policy.
- Integrates auxiliary tasks with A3C in a unified framework.
- Empirically doubles human-normalized Labyrinth scores (54% → 87%) and achieves 880% mean human-normalized performance on 57 Atari games.
- Demonstrates ~10× faster learning than A3C and increased robustness to hyperparameter settings.
- Recommendation: accept.
- Contribution: significant empirical improvement in data efficiency and robustness for deep RL.
- Well-motivated and clearly articulated framework for representation learning via auxiliary tasks.
- Strong experimental evaluation across Labyrinth and Atari, with ablation studies showing each auxiliary task’s effect.
- Reward prediction sampling carefully balances zero and non-zero rewards to address reward sparsity.
- Architecture and Appendix B provide sufficient implementation details for reproducibility.
