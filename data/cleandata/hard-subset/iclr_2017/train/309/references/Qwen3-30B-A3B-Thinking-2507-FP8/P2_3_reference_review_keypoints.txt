- Introduces UNREAL: an A3C-based agent augmented with auxiliary tasks (pixel/feature control, reward prediction, replay) to improve data efficiency and robustness.
- Demonstrates major empirical gains: >2× human-normalized performance over A3C on Labyrinth and strong state-of-the-art results on 57 Atari games (880% mean, 250% median).
- Provides extensive ablation studies showing the contribution of each auxiliary task and comparisons to baselines like pixel reconstruction and input-change prediction.
- Shows strong methodological rigor: evaluations on two domains, many hyperparameter settings (45–50) and multiple random seeds for statistical robustness.
- Claims auxiliary tasks improve representations and learning speed (≈10× data efficiency) and increase hyperparameter robustness.
- Reviewer recommends acceptance based on clear contribution and comprehensive empirical support.
- Suggest clarifying how human-normalized scores were computed for reproducibility.
- Suggest adding analysis (e.g., t-SNE or feature similarity) to concretely demonstrate improved representations from auxiliary tasks.
- Request quantification/discussion of computational overhead (inference time, memory) introduced by auxiliary objectives.
- Ask for clearer explanation of how reward prediction is used only for shaping features without biasing the main policy.
- Question why feature control underperforms pixel control and whether alternative combinations were explored.
- Ask for evidence that input reconstruction hurts performance (e.g., visualizations of agent focus) to support the claim.
- Request details on skewed reward-sampling: how the 0.5 non-zero reward probability was chosen and whether other ratios were tested.
- Overall conclusion: significant and well-supported contribution that advances data-efficient, robust RL, with a few clarifications requested.
