- Introduces UNREAL: uses auxiliary control and reward-prediction tasks to improve representation learning, data efficiency, and robustness in deep RL.  
- Two auxiliary task types: pixel/feature control (maximize changes) and reward prediction (predict immediate reward onset); used only for representation learning.  
- Empirical claim: UNREAL significantly outperforms A3C on Labyrinth and Atari, achieving 87% vs. 54% human-normalized on Labyrinth, 10× faster learning, and strong Atari gains.  
- Decision: Accept.  
- Key acceptance reasons: addresses sparse-reward RL with a novel, well-grounded approach; strong empirical improvements across diverse environments and baselines.  
- Strong literature placement and clear differentiation from prior pseudo-reward approaches.  
- Comprehensive empirical evaluation with appropriate baselines, multiple metrics, and ablation studies isolating each auxiliary task’s contribution.  
- Provides sufficient implementation details for reproducibility (architectures, hyperparameters, training procedures).  
- Ablations show pixel control and related tasks outperform input reconstruction, explaining which auxiliaries drive gains.  
- Impressive quantitative results: large Labyrinth and Atari performance improvements and data-efficiency gains.  
- Suggestion: add theoretical justification linking auxiliary tasks to representation learning principles.  
- Suggestion: include computational cost analysis (wall-clock time/overhead) to complement data-efficiency claims.  
- Suggestion: include representative feature/activation visualizations in the paper (not just video).  
- Suggestion: provide more detail on when and why feature control performs relative to pixel control.  
- Suggestion: define how “human-normalized” scores are computed for clarity.  
- Q1: Request more detail/sensitivity analysis on UNREAL’s claimed hyperparameter robustness.  
- Q2: Ask how sensitive reward-prediction performance is to the skewed sampling ratio (e.g., 50/50 vs. 60/40).  
- Q3: Ask how UNREAL’s exploration differs from standard methods and how reward prediction enhances exploration in procedurally-generated Labyrinth.  
- Q4: Ask why input reconstruction speeds initial learning but harms final performance and whether this is general.  
- Q5: Request a more detailed comparison with Prioritized Dueling DQN, including specific Atari metrics.
