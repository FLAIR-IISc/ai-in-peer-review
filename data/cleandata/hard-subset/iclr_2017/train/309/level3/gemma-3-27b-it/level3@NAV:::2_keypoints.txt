- Proposes domain-independent auxiliary tasks (control, reward prediction, value replay) shared with CNN+LSTM features to improve data efficiency in deep RL.  
- Well-motivated, aligns with self-supervised learning trends, and clearly written with accessible methodology.  
- Strong empirical results: significant gains over A3C on Atari and Labyrinth, including a reported 10x speedup in Labyrinth.  
- UNREAL agent presents a meaningful advance in efficiency for complex environments; reviewer recommends acceptance.  
- Ablation studies indicate auxiliary tasks contribute to gains, and human-normalized scores are a useful evaluation metric.  
- Quantify additional computational cost: report training speed loss and identify the most expensive auxiliary components.  
- Clarify in abstract/introduction that separate policies are learned for each auxiliary task (not a single policy optimizing all rewards).  
- Feature-control auxiliary task validation is weak; Figure 5 is unconvincing and task changes raise stability/convergence concerns—needs stronger evaluation.  
- Consider training agents longer and report maximum achieved performance since learning was still improving in some cases.  
- Investigate sensitivity to auxiliary task weights (λ*), treat them as hyperparameters, and explicitly report λPC (assumed to be 1).  
- Explicitly state that auxiliary tasks are trained off-policy and not using “true” Q-learning due to multi-step empirical rewards.  
- Minor corrections needed: Equation 1 lambda_c placement, r_t^(c) → r_{t+k}^{(c)}, typographical/figure reference fixes, grammar fix (“is”→“are”), define L_PC in Eq.2, explain “Clip” in Figure 3 legend.  
- Conduct more ablation studies on Atari to verify that per-component contributions observed in Labyrinth generalize.  
- Clarify that percentages in Figure 3’s legend refer to Labyrinth.  
- Specify the location of the supplementary materials/appendix.
