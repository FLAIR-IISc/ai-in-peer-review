- Proposes UNREAL: augment A3C with auxiliary tasks (pixel control, feature control, and reward prediction via skewed replay) to learn stronger representations and improve RL performance.  
- Introduces two pseudo-control tasks (controlling pixel intensity changes and latent feature activations) plus a supervised offline reward regression task.  
- Empirical results: UNREAL significantly outperforms baselines (880% expert human on Atari, 87% on Labyrinth) and reaches baseline in ~10x fewer iterations.  
- Recommendation: accept.  
- Importance: addresses representation learning for data efficiency and performance, arguing controllable internal feature learning complements reward maximization.  
- Motivation: well-situated in literature, contrasts with passive unsupervised learning by using the agent’s influence as a learning signal.  
- Experimental strength: robust empirical evaluation with thorough ablations; authors acknowledge hyperparameter differences and use top-3 reporting to mitigate bias.  
- Main weakness: insufficient in-depth experimental analysis of the auxiliary tasks themselves, beyond RL performance metrics.  
- Presentation issue: need to present low-level details of auxiliary tasks earlier (e.g., introduce L_PC before appendix).  
- Recommendations for improvement: provide detailed analysis of each auxiliary task’s contribution to representation learning, visualize learned features, run ablations isolating effects on representation quality, and clarify links to partial observability.  
- Questions for authors: how do these representations differ from standard unsupervised methods, and is there evidence the feature control task learns meaningful features rather than memorizing pixels?
