- Introduces UNREAL: a unified RL agent that learns multiple auxiliary tasks alongside the main reward objective to improve data efficiency and robustness.
- Key contribution: unified framework that learns separate policies for pseudo-rewards (pixel changes, network features) plus reward prediction and value replay.
- Shared CNN-LSTM representation across tasks leads to a more robust, generalizable feature space for the primary task.
- Strong recommendation for acceptance based on conceptual novelty and practical effectiveness.
- Empirical results: substantial improvements over A3C — 880% human-normalized on Atari and 87% on Labyrinth.
- Learning speedup: mean 10× faster on Labyrinth, needing <10% of data to match A3C final performance.
- Ablation studies show each auxiliary task contributes, with the full set performing best.
- Comparison to unsupervised baselines shows controlling features outperforms mere prediction/reconstruction.
- Results robust across hyperparameter variations; UNREAL consistently outperforms A3C.
- Request more detailed analysis of computational costs (training time, memory) and which components are most expensive.
- Feature-control evidence is weak in Fig.5; strengthen results or clarify limitations of feature control.
- Clarify that separate policies are learned for each auxiliary task (not a single policy optimizing all rewards).
- Justify fixing auxiliary task weights (λ=1) or include experiments validating this choice and impact.
- Clarify how off-policy, multi-step empirical rewards for auxiliary tasks affect the true Q-learning process.
- Minor corrections needed: Eq.1 placement of λ_C, grammar ("features... are shared"), and figure reference inconsistencies.
- Clarify value/reporting of λ_PC (assumed 1) and explain "Clip" in Figure 3 legend for dueling networks.
