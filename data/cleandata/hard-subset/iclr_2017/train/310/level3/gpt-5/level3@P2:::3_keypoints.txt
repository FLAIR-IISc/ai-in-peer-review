- Introduces a referential game where a sender sees two images and a target indicator and emits a discrete symbol for a receiver to identify the target; agents are neural nets trained end-to-end with REINFORCE on precomputed visual features.

- Agents reliably learn to coordinate to near-perfect accuracy across architectures and vocabulary sizes.

- Induced symbols correlate with meaningful visual/conceptual properties: symbols align above chance with semantic categories and reflect CNN embedding geometry.

- Removing low-level common knowledge (showing different instances per class) increases symbol-category alignment.

- Alternating interactive play with supervised object naming grounds symbols in natural language, substantially raising purity and enabling partial human interpretability.

- Human evaluation on ReferIt: subjects chose sender-intended image 68% of the time even when the target did not literally contain the supervised label.

- Paper is well-motivated as an alternative to purely supervised seq2seq dialog learning and connects to signaling games and emergent communication literature.

- Decision: Accept.

- Key reasons for acceptance: clear motivation, simple scalable framework, convincing empirical evidence including human evaluation, and strong demonstration of semantically meaningful emergent symbols.

- Problem tackled: designing multi-agent environments that elicit communication and assessing portability and human interpretability of emergent codes.

- Approach placement: convincingly positioned relative to prior multi-agent communication work and supervised+self-play paradigms.

- Convergence/performance evidence: consistent near-perfect accuracy; informed sender converges faster and uses more symbols.

- Semantic structure evidence: category purity above chance (including living/non-living split with two symbols); SVD and t-SNE analyses show nontrivial symbol usage aligned with visual similarity.

- Environment shaping evidence: different-instance setting boosts category-level symbol alignment as predicted.

- Grounding evidence: supervised naming grows symbol repertoire and purity (e.g., 88/100 symbols used, purity ~70%).

- Methodology: appropriate use of policy gradients and clear architectures; main claims supported despite some qualitative analyses.

- Suggested evaluation: test cross-play to assess protocol portability between independently trained agents.

- Suggested robustness checks: report variance over seeds, learning curves with confidence intervals, and sample efficiency; include ablations on vocabulary size, temperature/annealing, and image feature types.

- Reporting improvements: unify percentage scales across tables, fix obs-chance purity inconsistency, add ARI/NMI and statistical significance tests, and consider topographic similarity metrics.

- Human study details requested: provide per-word accuracy, confidence intervals, failure examples, and clarify handling of trials with literal supervised labels.

- Suggested tests: evaluate human listeners on the original image set, explore multi-symbol/compositional messages, and discuss extending message length and compositionality.

- Architecture clarity: justify the “informed” sender inductive bias and provide ablations showing whether an agnostic sender can achieve similar semantics with more capacity or training.

- Reproducibility: release code and data splits and detail preprocessing, train/test independence, and full hyperparameters (including temperature schedule).

- Questions for authors: how is target t input to the sender (ordering vs explicit input) and any positional-bias checks?

- Questions for authors: do independently trained sender/receiver pairs interoperate (cross-play), and does supervised grounding improve cross-play?

- Questions for authors: report variance (mean±std) of final accuracy, purity, and symbol usage across runs.

- Questions for authors: how do purity, symbols used, and interpretability scale with vocabulary sizes beyond 10 and 100?

- Questions for authors: sensitivity to feature type (fc vs sm) and whether lower-layer features reduce semantic purity?

- Questions for authors: how is purity computed (clustering method, images per object) and how was chance estimated (permutations vs symbols)?

- Questions for authors: human evaluation specifics — were trials with literal supervised labels excluded, and can accuracy be split into “literal” vs “metonymic” cases?

- Questions for authors: training details — any baseline/variance reduction for REINFORCE, entropy regularization, and was τ fixed or annealed?

- Questions for authors: role/input order randomization — did randomizing the sender’s input order affect learning speed or semantics?

- Minor edits: fix typos (“assest”→“assess”, “usufal”→“usual”) and clean up reference formatting.
