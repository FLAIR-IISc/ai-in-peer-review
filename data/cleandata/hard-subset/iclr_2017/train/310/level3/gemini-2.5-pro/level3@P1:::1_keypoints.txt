- Introduces a referential game framework where sender and receiver learn a discrete communication protocol via RL to identify a target image.  
- Shows simple feed-forward networks can learn to communicate and reach near-perfect task accuracy.  
- Finds architectural inductive biases matter: the "informed" sender yields a richer, more multifaceted vocabulary than the "agnostic" sender.  
- Demonstrates that altering the game environment can encourage more abstract, conceptual symbol meanings.  
- Grounds emergent messages in natural language by combining the game with supervised image labeling, producing partially interpretable signals that generalize to humans.  
- Decision: reject, primarily due to concerns about the central claims of emergent communicative language.  
- Main concern 1: the task is overly simplified and often reduces to distributed classification rather than true referential communication.  
- Main concern 2: using RL (REINFORCE) is inefficient and unnecessary if the protocol mainly encodes object class—supervised methods could solve it more directly.  
- Supporting argument: dataset sampling (different high-level categories as target/distractor) lets the sender simply signal class labels, avoiding compositional/descriptive language.  
- Supporting argument: purity analysis shows symbols cluster by semantic categories, supporting the classification interpretation.  
- Supporting argument: interactive multi-agent RL does not convincingly outperform or justify itself versus a decoupled supervised approach.  
- Suggested improvement: create harder referential tasks where target and distractor are from the same category to force descriptive signaling.  
- Suggested improvement: further analyze the informed sender (e.g., visualize filters, identify influential feature differences) to explain its richer vocabulary.  
- Suggested improvement: emphasize and deepen the grounding experiment and analyze when humans fail to interpret agent signals.  
- Questions for authors: report performance when target and distractor are from the same McRae category and hypothesize outcomes for both sender types.  
- Questions for authors: provide intuition/analysis for why the convolutional informed sender encourages a wider vocabulary and whether it encodes inter-image differences.  
- Questions for authors: compare to a baseline where the sender is pretrained as a supervised classifier and frozen, and then a receiver is trained—report sample efficiency and final performance.
