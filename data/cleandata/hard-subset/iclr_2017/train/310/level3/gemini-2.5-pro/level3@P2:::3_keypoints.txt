- Introduces a referential game where two neural agents learn a communication protocol via reinforcement learning to identify a target image.  
- Main contributions: show communication can emerge from scratch; emergent symbols capture high-level semantics; grounding with supervised image-labeling makes agent language human-interpretable.  
- Decision: Accept.  
- Novel problem formulation: frames language acquisition as interactive, goal-oriented coordination beyond passive supervised learning.  
- Grounding strategy: self-play emergence followed by supervised grounding is simple, effective, and yields interpretable agent language.  
- Well-motivated approach connecting to linguistics/game theory and contrasting prior supervised and human-in-the-loop methods.  
- Strong empirical support: agents converge to near-perfect coordination; sender architecture biases affect learning speed and protocol nature.  
- Semantic structure evidence: cluster purity, t-SNE visualizations, and class-level coordination experiments indicate symbols map to high-level categories.  
- Grounding results: mixing with supervised labeling raises semantic purity (~17% → ~70%) and human participants interpret agent messages at ~68% accuracy, including metonymic uses.  
- Minor issues: inconsistent percentage/proportion reporting across tables and a few typographical errors.  
- Request for clarification: confirm whether the sender is informed of the target solely via fixed input ordering (e.g., concatenation [target_feature, distractor_feature]).
