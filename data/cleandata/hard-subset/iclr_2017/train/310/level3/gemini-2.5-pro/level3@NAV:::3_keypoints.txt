- Introduces a framework for studying emergent communication between AI agents via a referential game.  
- Sender generates a discrete symbol to help the receiver identify a target image from a pair.  
- Agents are neural networks trained with REINFORCE and learn to coordinate and develop a communication protocol from scratch.  
- Analysis shows learned symbols capture meaningful semantic concepts beyond mere task success.  
- Main contribution: multi-game training where the sender alternates between the referential game and supervised image classification to ground agent language.  
- Grounding via supervised task makes emergent symbols more human-understandable, enabling interactive AI that can communicate with people.  
- Strong recommendation for acceptance; addresses important/challenging problem of grounded multi-agent communication.  
- Method offers a valuable alternative to standard sequence-to-sequence approaches by combining game-based learning with supervised grounding.  
- Provides strong empirical support with well-designed experiments that increase in complexity.  
- Shows agents converge to optimal policies and that environment changes (e.g., image classes, multi-game setup) improve semantic richness and interpretability.  
- Final experiment demonstrates humans can interpret the grounded agent language even for novel objects.  
- Task is simple but appropriate as a justified starting point for this research direction.  
- Minor issues to fix: inconsistency in purity reporting (Table 1 scale vs. Table 2), and two typos ("assest"→"assess", "usufal"→"usual").  
- Request clarification: how the target variable t is provided as input to the sender network (text mentions it, architecture description focuses on image features).
