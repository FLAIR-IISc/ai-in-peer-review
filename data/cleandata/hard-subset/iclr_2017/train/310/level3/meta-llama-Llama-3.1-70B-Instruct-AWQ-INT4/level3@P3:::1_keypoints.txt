- Tackles understanding agent-agent communication and whether machine problem-solving mirrors human strategies.  
- Proposes a multi-agent communication framework for language learning aimed at building conversational machines.  
- Main contribution: a simple, interpretable referential game enabling intuitive post-learning analysis.  
- Task: sender and receiver see image pairs; sender sends a message to identify the target image.  
- Two simple neural networks successfully learn to coordinate in the referential game.  
- Agents’ emergent language appears to capture general conceptual object properties rather than low-level visuals.  
- Informed agents use multiple, more word-like symbols; agnostic agents rely on only two symbols.  
- Agents’ language can be grounded in natural language by combining the game with supervised image labeling.  
- Approach is well-motivated, with thorough literature review and clear evaluation metrics.  
- Major limitation: when images come from different categories, the task reduces to image classification, questioning linguistic generality.  
- Symbols largely encode image class, making the setup similar to a transfer-learning/supervised solution.  
- Recommendation: accept with revisions to address noted limitations.  
- Requested revisions: provide stronger evidence that agents learn language rather than classifiers and discuss implications for conversational machines.  
- Key open questions: behavior when images are from the same category; ability to communicate abstract concepts (actions/events); generalization to multi-step communication or dialogue.  
- Additional feedback: include more visualizations of agent symbols and mappings to images.  
- Additional feedback: provide detailed description of the supervised labeling task and its integration with the referential game.  
- Additional feedback: expand discussion of real-world implications and envisioned uses for conversational systems.
