

This paper makes a compelling contribution to the field by proposing a multi-agent communication framework for language learning, moving beyond passive supervised learning to explore interactive language development. The authors focus on referential games where a sender and receiver must coordinate to identify target images using a shared vocabulary, demonstrating that agents can develop meaningful communication protocols through interaction. The work is well-motivated, placing itself in the tradition of linguistic and cognitive studies of language emergence while addressing practical AI challenges. The experimental setup is clear and the results provide evidence that agents can develop language with semantic properties aligned with human intuition.

I recommend rejection of this paper. While the work is conceptually interesting and well-executed, the core contribution is limited by fundamental issues with the experimental setup and interpretation. The key issue is that the referential game effectively reduces to a simple image classification task when the two images are from different categories, which undermines the paper's claim that agents are developing a "language" with semantic properties. The task becomes one of learning to associate a symbol with an image class, rather than developing a meaningful communication protocol. This is particularly problematic because the agents are explicitly given information about which image is the target (via the sender's input structure), making the task fundamentally different from real-world communication where such information is not available.

The paper claims that agents develop symbols that reflect "intuitive semantic properties" of images, but this interpretation is misleading. The symbols merely encode image classes, which is a straightforward transfer learning problem. The authors' "semantic purity" metrics measure how well symbols align with pre-existing image categories (from McRae et al.), not whether the agents have developed a meaningful linguistic system. The fact that agents using the informed sender architecture use more symbols (88 vs. 2 for agnostic) is simply a consequence of the larger vocabulary size and the task's inherent simplicity, not evidence of richer language development.

The grounding experiment, where agents are trained on supervised image classification, further highlights this limitation. The paper claims that this "grounds" the communication in natural language, but the results show that agents primarily use symbols corresponding to supervised labels for images in those categories. The 68% human interpretability rate is impressive but is largely due to the agents using the exact supervised labels for images in the training categories, not because they've developed a truly interpretable language. The "metonymic" examples shown (e.g., using "dolphin" for a stretch of sea) are interesting but represent a very narrow form of language use that doesn't generalize to the broader communication challenges the paper purports to address.

The paper's claims about language development are overstated. The agents aren't developing language; they're learning to solve a classification problem using a discrete communication channel. This is a significant distinction, as true language development would involve more complex semantic relationships, context sensitivity, and the ability to handle novel situations. The paper's framework lacks the necessary complexity to model these aspects.

The paper would benefit from addressing several critical issues. First, it should remove the information about which image is the target from the sender's input, as this fundamentally changes the problem from communication to classification. Second, the authors should demonstrate that the agents' language can be used to communicate about novel objects not seen during training, rather than just matching pre-defined categories. Third, the evaluation should move beyond semantic purity metrics to assess more sophisticated linguistic properties like compositionality, ambiguity resolution, and context sensitivity.

The paper's claim that "agents can develop their own language interactively out of the need to communicate" is not supported by the results. The communication is trivially solved by mapping symbols to image classes, and the "language" has no functional value beyond this simple mapping. The paper would be significantly stronger if it could demonstrate communication that goes beyond simple category labeling, such as describing spatial relationships between objects or using contextual information to disambiguate references.

I have two key questions for the authors to help clarify their approach:
1. If the task is reduced to image classification (due to the target information given to the sender), how does this differ from simply training a classifier with a discrete output layer? What specific communication challenges does the framework address that would be absent in a standard classification task?
2. How does the paper distinguish between "language" and "discrete classification" when the emergent symbols are merely labels for pre-defined image categories? What evidence would be needed to demonstrate that the agents have developed a true language rather than a simple classification system?

The paper makes an interesting contribution to the literature on language emergence, but the experimental setup and interpretation significantly overstate its claims. The work would be much stronger if the authors could demonstrate communication that goes beyond simple category labeling, such as describing relationships between objects or using contextual information to disambiguate references. As presented, the results do not provide convincing evidence of language development, but rather of a clever way to solve a classification problem using a discrete communication channel. The paper's potential is limited by this fundamental mischaracterization of the problem it claims to address.
