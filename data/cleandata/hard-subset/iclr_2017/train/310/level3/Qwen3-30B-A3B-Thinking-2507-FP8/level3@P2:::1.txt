

# Review for ICLR 2017

## 1. Summary of Contributions

This paper makes a compelling contribution to the emerging field of emergent communication in multi-agent systems. The authors propose a novel framework for language learning through multi-agent communication games, specifically focusing on referential games where a sender and receiver must coordinate to identify a target image. The paper demonstrates that simple neural networks can learn to communicate effectively in this setting, and further shows how modifying the environment can lead to communication protocols that better reflect human semantic understanding. The authors also present a promising approach to grounding agents' communication in natural language through a combination of interactive game play and supervised learning. The work is well-motivated, well-executed, and makes a significant contribution to understanding how communication can emerge in artificial systems.

## 2. Decision

I recommend **rejecting** this paper. The core problem of understanding how communication emerges in multi-agent systems is important and well-motivated, but the paper's claims are not sufficiently supported by its experimental results. The main issue is that the task effectively reduces to image classification, which undermines the significance of the communication learning process.

## 3. Supporting Arguments

The paper's main claim is that agents develop "language" through interaction, but the experimental setup reveals a fundamental limitation. When the two images in the referential game are from different categories (which is the majority case given the image selection process), the task reduces to a simple image classification problem. The sender's job becomes identifying which image belongs to which category, and the receiver's job becomes identifying which category the sender is referring to. This means the agents are not truly developing a communication system that requires coordination, but rather learning to classify images and then use the category name as a symbol.

The paper notes that the agnostic sender converges to using only two symbols (which likely correspond to "living" vs. "non-living" categories), while the informed sender uses more symbols. However, this is not evidence of "language" but rather evidence of the agents learning to solve the image classification problem using the minimal number of symbols necessary. The paper's own experiments confirm this when they show that when the game is modified to use class-level properties instead of specific images, purity increases, which is exactly what would be expected when the problem becomes a simpler classification task.

The grounding experiment using supervised learning is interesting, but it doesn't fundamentally address the core issue that the underlying communication task is not truly emergent. The agents are simply learning to map between image categories and words, not developing communication strategies that require coordination. The paper's claim that this is "necessary steps towards developing machines that are able to communicate with humans productively" is overstated given the limitations of the task.

## 4. Additional Feedback

The paper would benefit from:
1. A more thorough analysis of why the communication task is not truly emergent, and how it differs from standard image classification.
2. A clearer demonstration that the communication process requires coordination beyond simple classification (e.g., by creating a scenario where the two images are from the same category).
3. More explicit discussion of the relationship between the authors' work and the broader literature on emergent communication, particularly the distinction between "language" and "communication" in these settings.

## 5. Questions for Authors

1. Could you clarify how the referential game setup differs from a standard image classification task when the target and distractor images are from different categories? If the task effectively reduces to classification, what is the role of communication in this process?

2. The paper shows that the agnostic sender uses only two symbols (likely corresponding to "living" vs. "non-living"). How would you modify the game to force agents to develop communication that goes beyond such basic category distinctions?

3. In the grounding experiment, the authors note that the language developed in the referential game is understandable by humans even when referring to images not in the supervised set. What specific mechanisms allow the agents to generalize beyond the supervised categories, and how does this compare to human language generalization?

4. How would you modify the game setup to ensure that communication requires true coordination rather than just classification? For example, what if both images were from the same category, and the sender had to distinguish between them using a more nuanced description?

The paper addresses an important problem, but the experimental results do not fully support the claimed contribution of "language emergence" through multi-agent communication. The work would be significantly strengthened by addressing these limitations.
