- Positive contribution: proposes a multi-agent communication framework for interactive language learning using referential games.
- Setup: sender and receiver coordinate with a shared vocabulary to identify target images; experiments are clear and well-motivated.
- Evidence claim: agents develop communication protocols with semantic properties aligned with human intuition.
- Recommendation: reviewer recommends rejection despite conceptual interest and solid execution.
- Core criticism: the referential game reduces to simple image classification when images are from different categories.
- Fundamental issue: sender input explicitly contains target information, turning the task into class association rather than emergent communication.
- Semantic interpretation critique: symbols merely encode image classes; semantic purity metrics measure alignment with pre-existing categories, not genuine language.
- Vocabulary effect: informed sender uses more symbols (88 vs. 2) due to larger vocabulary and task simplicity, not richer language development.
- Grounding experiment limitation: supervised training causes symbols to map to supervised labels; 68% human interpretability largely reflects label reuse.
- Metonymy examples are narrow and do not generalize to broader language use.
- Overstated claims: agents learn classification over a discrete channel, not true language with complex semantics, context sensitivity, or novelty handling.
- Suggested fixes: remove target information from sender input to prevent reduction to classification.
- Suggested evaluation improvements: test communication about novel objects and assess compositionality, ambiguity resolution, and context sensitivity beyond semantic purity.
- Two clarifying questions: how does this differ from a classifier with a discrete output layer, and how does the paper distinguish "language" from "discrete classification" and what evidence would prove true language emergence?
- Overall: paper’s claims are overreaching; it demonstrates a clever classification-based communication solution rather than convincing language development.
