### **1. Summary**

This paper investigates the emergence of communication between two neural network agents, a "sender" and a "receiver," tasked with coordinating in a cooperative referential game. The core problem is to design environments that foster the development of a communication protocol that is not only effective for the agents but also interpretable by humans.

The authors first demonstrate that tabula rasa agents can successfully learn to communicate and solve the game using reinforcement learning, achieving near-perfect coordination. They analyze the emergent protocol and show that it captures abstract, conceptual properties of the visual inputs, rather than just low-level features. This is evaluated by measuring the "purity" of symbol clusters against ground-truth semantic categories. The paper also explores how architectural inductive biases (an "informed" sender) and modifications to the game's common knowledge (showing agents different image instances of the same object class) can encourage a more abstract and richer language.

The main contribution is a method for grounding the agents' emergent language in human language. Inspired by AlphaGo's training regime, the authors combine the interactive referential game with a standard supervised image-labeling task. This encourages the sender to align its arbitrary symbols with conventional English words. The effectiveness of this grounding is rigorously tested in a crowdsourced experiment where human participants, acting as the receiver, must interpret the agent's messages. The results show that humans can successfully understand the agent's language at a rate significantly above chance, even when the agent is referring to objects for which it has no direct supervised label, demonstrating a capacity for generalization and establishing a meaningful common ground between agent and human.

### **2. Decision and Key Reasons**

**Decision:** Accept

1.  **Novel and Effective Grounding Method:** The paper presents a clear and well-motivated method for grounding an emergent communication protocol in natural language by combining interactive multi-agent learning with supervised learning. This is a significant contribution towards making agent communication systems interpretable and useful for human-AI interaction.
2.  **Rigorous Empirical Validation:** The claims are supported by a strong set of experiments, culminating in a human evaluation study. This final experiment provides direct, quantitative evidence for the primary claim—that the emergent language is human-interpretable—which adds considerable weight and credibility to the findings.

### **3. Supporting Arguments**

The paper makes a compelling case for using multi-agent games as a framework for developing functional communication in AI. While prior work has shown that agents can develop communication protocols, this work takes the crucial next step of addressing how to make these protocols align with human language.

The proposed method of mixing a cooperative game with a supervised task is both simple and effective. It provides a practical solution to the grounding problem without requiring constant human intervention in the loop, which is a key advantage for scalability. The results from this experiment are strong: purity scores increase dramatically, and more importantly, the resulting language becomes demonstrably useful for communicating with humans.

The empirical evaluation is a major strength. The progression of experiments is logical, starting from establishing baseline communication, analyzing its properties, and then introducing modifications to improve it. The inclusion of the human study in Section 5 is particularly important. By replacing the agent receiver with a human, the authors directly test the primary goal of their research program. The finding that humans can interpret the agent's messages 68% of the time, especially when the agent generalizes beyond its supervised vocabulary through "metonymic" links (e.g., using "dolphin" to refer to the sea), is a powerful and insightful result. This provides strong evidence that the grounding method is not simply memorizing labels but is fostering a more flexible, semantic-based communication system. The technical setup is sound, using standard reinforcement learning techniques (REINFORCE) and well-described neural architectures.

### **4. Additional Feedback for Improvement**

These points are intended to help strengthen the paper and are not critical factors in the decision.

*   **Architectural Intuition:** The "informed" sender architecture, which uses 1-D convolutions, is an interesting inductive bias. A little more discussion on the intuition behind this specific design would be helpful. For example, why was a dimension-by-dimension comparison between image embeddings hypothesized to be beneficial for this task?
*   **Hyperparameter Sensitivity:** The temperature parameter for the Gibbs distributions was set to a high value of 10. This encourages exploration but might slow convergence or lead to less precise protocols. A brief discussion on the sensitivity to this parameter, or justification for its choice, would add to the paper's completeness.
*   **Qualitative Analysis:** The observation that the 2-symbol agnostic sender converges on a "living-vs-non-living" distinction is fascinating and aligns with findings in cognitive science. If possible, a more quantitative analysis to support this (e.g., by mapping the McRae categories to this binary distinction) would strengthen this intriguing point.
*   **SVD Cross-Validation:** The paper mentions that "cross-validated SVD suggests 50 dimensions" to analyze vocabulary redundancy. A brief clarification in the text or an appendix on how cross-validation was applied to SVD in this context would be beneficial for reproducibility.

### **5. Questions for the Authors**

Your answers to these questions would help clarify some aspects of the work and provide a more complete understanding.

1.  In the grounding experiment (Section 5), the sender alternates between the game and the supervised task with equal probability. How sensitive are the grounding results to this mixing ratio? Have you explored other schedules, such as a curriculum that starts with more supervision and gradually shifts towards more game interaction?
2.  The human evaluation result of 68% accuracy is a key finding. While significantly above the 50% chance level, it also indicates room for improvement. Could you provide a brief analysis of the failure cases? What kinds of semantic leaps or "metonymic" links were too difficult for humans to interpret? Understanding these failures could be highly valuable for guiding future research.
3.  The agent architectures are intentionally simple. Have you considered whether more complex architectures, particularly for the receiver (e.g., using an attention mechanism where the symbol directs attention over the image features), might facilitate the emergence of a more compositional or fine-grained language?
