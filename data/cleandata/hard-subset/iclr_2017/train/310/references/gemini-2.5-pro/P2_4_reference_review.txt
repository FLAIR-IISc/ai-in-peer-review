### Review

**Summary of Contributions**

This paper investigates the emergence of communication between two neural network agents—a sender and a receiver—tasked with coordinating in a referential game. The agents start as blank slates (*tabula rasa*) and must develop their own communication protocol to successfully identify a target image from a distractor. The paper's primary contributions are threefold. First, it demonstrates that simple feed-forward agents trained with reinforcement learning can successfully develop an effective communication protocol for this task using discrete symbols. Second, it explores how architectural choices (an "informed" vs. "agnostic" sender) and game mechanics (manipulating common knowledge by showing different exemplars of the same object class) can influence the semantic properties of the emergent language, pushing it towards more abstract, conceptual representations. Third, and most significantly, it proposes and validates a method for grounding the agents' emergent language in human language. By combining the interactive referential game with a supervised image-labeling task, the authors show that the resulting communication protocol not only improves in semantic structure but also becomes partially interpretable by humans, even when referring to objects not seen during the supervised phase. This last claim is compellingly supported by a crowdsourced experiment where human participants successfully interpret the agent's messages.

**Decision and Key Reasons**

**Decision:** Accept

This is a strong paper that I am happy to recommend for acceptance. It addresses a fundamental question in AI—how to develop agents that can learn to communicate meaningfully—with a well-designed and insightful set of experiments. The two key reasons for my decision are:

1.  **Thoughtful Experimental Design and Analysis:** The authors do not simply show that agents can solve a task. They systematically investigate the *properties* of the emergent communication system. The comparison between sender architectures, the manipulation of common knowledge, and the use of metrics like semantic purity and SVD on the symbol-usage matrix go beyond simple task accuracy and provide valuable insights into what drives the structure of the learned language.
2.  **Novel and Effective Grounding Technique:** The final experiment in Section 5 is the paper's highlight. The idea of combining interactive self-play with supervised learning, inspired by AlphaGo, provides a practical and scalable path toward making agent communication human-understandable. The validation with a human study on a held-out dataset (ReferItGame) provides strong evidence for the central claim that this grounding generalizes, allowing for creative, "metonymic" uses of language that humans can still comprehend.

**Supporting Arguments**

The paper is well-motivated, drawing connections to cognitive science, game theory, and philosophy of language, which provides a rich context for the work. The problem is framed clearly, contrasting the interactive, functional approach with the limitations of supervised learning on static conversational datasets.

The experiments are methodologically sound. The use of REINFORCE is appropriate for the discrete communication channel. The initial results in Section 4 convincingly show that agents learn to communicate and that the "informed" sender architecture, which has an inductive bias for comparing image features, learns faster and uses a richer vocabulary. The "object-level reference" experiment is a clever way to force agents away from coordinating on low-level visual features and toward more abstract class-level properties, which is supported by the increase in symbol purity.

The main strength, as noted, is the grounding experiment. Many papers on emergent communication stop at showing that a protocol emerges, but this work tackles the critical next step of making that protocol meaningful to us. The fact that the sender uses supervised labels correctly 47% of the time for in-domain objects is a good start, but the real test is the generalization to the ReferItGame dataset. The 68% accuracy achieved by human guessers is a powerful result. It suggests that the shared embedding space learned through both tasks captures a semantic structure that is flexible enough to be used creatively, much like human language. The qualitative examples of "dolphin" for sea and "fence" for land are excellent illustrations of this emergent semantic flexibility.

**Additional Feedback for Improvement**

These points are intended to help strengthen the paper and are not critical for my decision to accept.

*   **Analysis of the "Informed" Sender:** While the intuition behind the informed sender's 1-D convolutions is clear (encouraging dimension-wise comparison), a more in-depth analysis of what is learned would be beneficial. Does it learn to focus on specific feature dimensions that are highly discriminative? A brief probe into the learned filters could provide more insight into *why* this architecture performs better.
*   **Complexity of Language:** The current framework is limited to single-symbol utterances for referring to a single object. This is a necessary simplification for a first step, but a discussion of the path toward more complex, compositional language would strengthen the paper's conclusion. How might this framework be extended to handle descriptions involving multiple properties, or to generate multi-symbol utterances?
*   **Training Details:** The paper states that REINFORCE was used for training. Given that this algorithm is known for its high variance, it would be useful to briefly mention if any training stability issues were encountered and if techniques like baselines were used. Also, reporting results averaged over multiple random seeds would increase confidence in the stability of the findings.
*   **Human Evaluation Context:** The 68% accuracy in the human study is impressive. To better contextualize this number, it would be useful to know the "human ceiling" or inter-annotator agreement. For instance, if one human generated a one-word clue for another human, what would the accuracy be on this task? This would help gauge how close the agent's communication is to human-level performance in this constrained setting.

**Questions for Authors**

1.  In the experiment where the agnostic sender converges to a 2-symbol vocabulary, you note it seems to represent a "living-vs-non-living" distinction. This is a fascinating observation. Was this based purely on qualitative inspection, or did you attempt to quantify this alignment?
2.  The supervised grounding task uses 100 labels. How was this subset of labels chosen from the full set of 463 concepts? Does the choice of grounding vocabulary significantly impact the quality of the resulting generalized communication?
3.  Have you considered the case where the receiver, rather than the sender, is exposed to the supervised objective? Or perhaps both? It seems plausible that if the receiver also grounds symbols in a shared human-centric semantic space, coordination might be even more robust and human-aligned.
