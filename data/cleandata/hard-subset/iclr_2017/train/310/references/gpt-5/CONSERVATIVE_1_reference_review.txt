Summary and contributions
This paper studies emergent communication in two-agent referential games over natural images. Agents (a sender and a receiver) are trained with REINFORCE to coordinate by sending a single discrete symbol to identify a target image among distractors. The authors compare an “agnostic” sender to an “informed” sender with an inductive bias for dimension-wise comparisons. They assess: (i) whether tabula rasa agents can learn to communicate; (ii) whether the induced codes align with human-relevant semantics via category purity, t-SNE visualizations, and a manipulation removing low-level common knowledge (sender/receiver see different images of the same class); and (iii) whether adding supervised object-name prediction encourages human-interpretable symbols, including a small human study on ReferItGame image pairs. The main claims are that agents reliably learn to communicate, that the learned symbols capture high-level semantic regularities beyond low-level pixel correlations, that environment design can nudge more abstract codes, and that light supervised grounding improves human interpretability.

Decision: Reject
Key reasons: (1) The empirical evidence does not convincingly support the stronger claims about semantic abstraction and human interpretability; core evaluations lack rigorous baselines, ablations, and statistical analysis. (2) The setup leans heavily on pre-trained supervised visual features and a highly simplified, one-symbol protocol, making the “blank slate” and “natural-language-like” conclusions insufficiently substantiated.

Supporting arguments
- Problem and positioning: The general question—how to induce portable, human-aligned communication via interaction—is important and timely. The paper is reasonably placed in prior multi-agent communication work (e.g., Sukhbaatar et al., Foerster et al.) and language emergence/game-theoretic literature. However, quantitative comparisons to concurrent learning-to-communicate baselines are absent, and the paper’s novelty primarily resides in a particular image-based instantiation plus two design tweaks (object-level reference and joint supervised grounding).

- Methodological concerns:
  • Perception is not tabula rasa: agents operate on pre-trained VGG features (including the 1000-D softmax), which already encode object categories closely aligned with human semantics and the McRae categories used for purity. This risks circularity: increased “category purity” could simply reflect the imposed supervised visual prior, not emergent semantics. A rigorous test would include lower-level conv features, self-supervised features, or pixels, and show similar trends.
  • Lack of baselines/controls: No comparison to simple non-interactive baselines such as k-means clustering over image embeddings followed by symbol assignment, or nearest-centroid labeling, to contextualize purity/t-SNE results. No comparison to standard discriminative referential models trained with supervised targets. Without these, it is hard to attribute observed structure to the interactive setup.
  • Limited protocol expressivity: Only single-symbol messages are permitted. This constrains conclusions about language-like structure, compositionality, or systematicity. Claims about “word-like” usage based on symbol counts and SVD spectra are tenuous without stronger analyses (e.g., topographic similarity, productivity on held-out compositions).
  • Analysis rigor: Purity improvements are stated to be “significantly above chance” but exact numbers, variance across seeds, and statistical tests are not consistently reported. The SVD dimensionality argument is informal and does not translate to interpretable communicative capacity. The t-SNE figures are qualitative and susceptible to projection artifacts.
  • Object-level reference manipulation: The idea to remove low-level common knowledge by using different images of the same class is interesting, but again confounded by pre-trained class-aware perception. It is unsurprising that this setting yields modestly higher category purity; a control where class labels are obfuscated or where features are class-agnostic would be informative.
  • Human study: The evaluation uses pairs selected ex post where the agent already succeeded, creating selection bias toward easy, high-signal cases. Only 8% of pairs contained the supervised word in ReferIt annotations, yet the study still uses that word as the prompt, which may cue background correlations. Details about subject demographics, quality control, and alternative-choice baselines are limited. A stronger test would randomize pairs independent of agent success, include adversarial distractors, and compare to human-only baselines.

- Claims vs evidence:
  • Communication success is expected in such simple referential games; demonstrating near-perfect accuracy does not, by itself, substantiate language-like abstraction.
  • The increase in symbol purity under “informed” architecture and object-level reference is plausible but not disentangled from the semantics inherent in the VGG features.
  • The “grounding via supervised labels” section shows that supervision increases symbol usage and purity, and a 68% human choice accuracy on selected pairs. However, given confounds and selection, this only weakly supports the claim that the emergent code generalizes to human-interpretable communication.

Additional feedback (to improve the paper)
- Add strong, non-interactive baselines:
  • k-means over embeddings with symbol assignment; a nearest-neighbor symbolizer; a supervised classifier used directly as a sender.
  • Report purity, cross-play accuracy, and human interpretability for these baselines.
- Report robustness:
  • Multiple random seeds, mean±std for all metrics, and convergence variance.
  • Ablate temperature, embedding size, vocabulary size (controlling for effective number of symbols), and the informed vs agnostic sender under matched expressivity.
- Reduce perception confounds:
  • Use self-supervised or lower-layer conv features; test with raw pixels on a smaller dataset; include a domain shift (e.g., different dataset classes) to probe generalization.
- Evaluate portability:
  • Cross-play between independently trained agents; zero-shot partner swap; generalization to unseen categories.
- Strengthen analyses:
  • Use established metrics (e.g., topographic similarity between symbol distances and semantic/visual distances), mutual information between symbols and categories, and compositional generalization tests.
  • Provide exact statistical tests for purity gains and spectrum analyses.
- Human evaluation:
  • Avoid selection on agent success; predefine image pairs; add foils; include human-human and text-only baselines; report inter-annotator agreement and error analysis.
- Presentation:
  • Clarify training schedule inconsistencies (10k vs 50k iterations). Detail variance reduction for REINFORCE. Correct and standardize references.

Questions for the authors
- How much of the observed category purity persists if you replace VGG softmax/fc features with self-supervised features or early conv layers? Can you report results for such settings?
- What are the mean±std over ≥5 seeds for coordination accuracy, purity, symbol usage, and SVD spectra? Do different runs converge to compatible codes (cross-play results)?
- How does a simple clustering baseline over image embeddings compare on purity and human interpretability?
- In the human study, what happens if you evaluate on randomly sampled pairs independent of agent success, and include adversarial distractors from the same coarse category?
- Can you quantify topographic similarity between symbol space and semantic space, and test generalization to unseen object classes?
- What variance-reduction techniques (baselines, entropy regularization) were used with REINFORCE, if any, and how sensitive are results to τ and vocabulary size?
