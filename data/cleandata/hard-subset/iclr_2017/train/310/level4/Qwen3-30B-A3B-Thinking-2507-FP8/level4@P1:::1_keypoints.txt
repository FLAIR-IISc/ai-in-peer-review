- Paper proposes a multi-agent referential game where agents co-evolve communication and shows simple neural networks can learn semantically meaningful symbols.  
- Decision: Reject — core contribution is interesting but undermined by a critical experimental limitation.  
- Major critique: The standard setup effectively reduces to class-level image classification because the sender receives ImageNet class info, so symbols map directly to class labels.  
- This reduction makes the emergent "language" trivial and not evidence of context-dependent, human-like communication.  
- Grounding critique: The supervised grounding step uses ImageNet labels, importing human language rather than demonstrating autonomous emergence of labels.  
- Language-analysis critique: Evidence for abstract semantic properties is weak — symbols mainly reflect visual similarity/dataset artifacts (t-SNE, purity metrics insufficient).  
- Scalability critique: The paper does not show how the framework scales to open-world, ambiguous, or novel referents; semantic alignment appears predetermined by task design.  
- Improvement suggestion: Clarify novelty by distinguishing classification from context-dependent communication and test on datasets without predefined class labels (e.g., ReferItGame).  
- Improvement suggestion: Strengthen semantic analysis with generalization tests to unseen objects/contexts (e.g., cartoons, variations).  
- Improvement suggestion: Address scalability and vocabulary size (e.g., handling 10k+ words or out-of-vocabulary referents).  
- Clarifying Q1: How does using ImageNet labels in the sender’s input avoid making communication de facto supervised — could agents learn from raw pixels only?  
- Clarifying Q2: How would the grounding experiment handle much larger vocabularies or labels outside the supervised set?  
- Clarifying Q3: Does the agnostic sender’s living vs. non-living cluster align with cognitive science claims about early semantic distinctions, and how does that support human-like communication?
