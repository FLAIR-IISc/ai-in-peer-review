

Thank you for submitting this interesting work. The paper addresses a fundamental challenge in AI: developing communication protocols that emerge from interaction rather than relying solely on supervised language data. The authors propose a multi-agent referential game framework where agents co-evolve a communication system through coordination, demonstrating how simple neural networks can learn to use symbols with meaningful semantic properties. Their approach is grounded in game-theoretic language evolution literature and makes concrete progress toward grounding machine communication in human language via supervised pre-training. The experiments are well-designed, with clear metrics for semantic alignment and human interpretability, and the results on symbol usage and grounding represent significant steps toward interactive language learning.

**Decision: Reject**  
The paper’s core contribution—demonstrating emergent communication in a referential game with semantic alignment—is compelling. However, the experimental setup creates a critical limitation: the game effectively reduces to class-level image classification when using ImageNet categories, undermining the claim that agents develop *natural* language. The authors acknowledge this by modifying the environment to remove low-level visual common knowledge, but the baseline experiments (Table 1, Figure 3) already operate within a highly constrained setting where symbols directly map to class labels. This renders the "emergent language" largely trivial and fails to demonstrate the functional, context-dependent communication required for human-like interaction. The paper does not sufficiently address how this framework scales to more complex, open-world scenarios where semantic alignment is not predetermined by the task design.

**Supporting Arguments**  
1. **The task reduces to classification**: In the standard setup (Section 3), the sender’s input explicitly includes the target image’s class (via ImageNet labels), and the receiver learns to associate symbols with classes. The agents’ "language" is thus a direct proxy for class labels (e.g., "dog" for all dog images), not an emergent system for *referencing* objects in a shared context. The paper’s own purity metrics (Table 1) confirm symbols map to semantic categories—exactly as expected when the task is class prediction. This does not advance beyond existing supervised classification approaches, contradicting the paper’s claim that agents develop language *without* pre-defined semantics.  
2. **Grounding lacks novelty**: The supervised grounding step (Section 5) uses ImageNet labels as ground truth, so the sender’s symbols are simply conventional labels (e.g., "dolphin" for dolphin images). While the human interpretability test is well-executed, the language is not *emergent* but *pre-specified*—the grounding step imports human language rather than enabling its emergence. The paper conflates "using human labels" with "developing human-interpretable language," ignoring the key challenge of *how* agents might autonomously arrive at such labels without supervision.  
3. **Lack of rigor in language analysis**: The paper claims symbols capture "general conceptual properties" (Section 4), but the evidence (e.g., t-SNE plots in Figure 3) shows symbols primarily correlate with visual similarity, not semantic categories. The "living vs. non-living" cluster for the agnostic sender (Table 1) is a trivial artifact of the dataset, not evidence of meaningful abstraction. Without testing on unstructured visual data (e.g., natural scenes without class labels), the semantic claims remain unsupported.

**Additional Feedback for Improvement**  
- **Clarify the novelty**: Explicitly distinguish between *learning to classify* (which is trivial) and *developing context-dependent communication* (which is the goal). Test the framework on a dataset without predefined class labels (e.g., ReferItGame images without ImageNet categories) to avoid the classification shortcut.  
- **Strengthen semantic analysis**: Beyond purity metrics, quantify how symbols generalize to unseen objects or contexts (e.g., "If the sender uses 'dolphin' for a dolphin image, does it also use it for a dolphin cartoon?").  
- **Address scalability**: Discuss how the framework would handle ambiguous or novel referents (e.g., "a blue car" vs. "a car in a blue parking lot"), which are central to human communication but absent in the current setup.

**Clarifying Questions**  
1. The paper states the agents "develop their own language interactively" (Abstract), but the baseline setup (Section 3) relies on ImageNet labels for the sender’s input. How does this avoid making the communication protocol *de facto* supervised? Could the agents learn to reference objects without any class-level input (e.g., using raw pixels only)?  
2. In the grounding experiment (Section 5), the sender’s vocabulary is limited to 100 ImageNet labels. How would the framework handle a vocabulary of 10,000+ words, or words not in the supervised set (e.g., "sailboat" for a boat in a non-ImageNet category)?  
3. The authors note that the "living vs. non-living" cluster for the agnostic sender is "intriguingly basic" (Section 4). Does this align with cognitive science evidence that this is the *first* semantic distinction humans learn? If so, how does this support the paper’s goal of developing human-like communication?
