

This paper proposes a novel referential communication game between two agents observing images, where a sender encodes a binary target into a message for a receiver to recover, with both agents rewarded for successful target prediction. The authors train neural network agents using REINFORCE, leveraging pretrained image representations as features. Key findings demonstrate that agents converge to optimal policies, learning semantically meaningful symbolic communication protocols. The work extends to image-classification variants, where communication becomes more interpretable, and introduces a multi-task setup where the sender alternates between communication and supervised image classification. Crucially, grounding communication in a supervised task yields even more meaningful symbolic representations.  

The paper makes a significant contribution by establishing a framework for studying *functional* communication in multi-agent systems, moving beyond statistical language modeling paradigms. The core insight—that communication can be learned end-to-end and later grounded in human language—is novel and addresses a key limitation of sequence-to-sequence models, which often overlook semantic grounding. The thorough analysis of learned symbols (e.g., via visualization of concept alignment) provides valuable empirical evidence for the emergence of meaningful communication, making this a strong stepping stone for future work on grounded language learning. The task design is well-justified as a controlled first step toward more complex communication scenarios, and the results convincingly demonstrate the feasibility of the approach.  

**Strengths:**  
1. **Novel Framework:** The referential game with image observations and symbolic communication offers a clean, testable paradigm for studying *functional* language emergence.  
2. **Empirical Rigor:** Experiments systematically demonstrate the progression from basic communication to grounded, interpretable symbol use across variants (binary, classification, multi-task).  
3. **Interpretability:** Visualization of learned symbols (e.g., "edge" or "color" concepts) provides compelling evidence of meaningful communication, far exceeding typical black-box RL results.  
4. **Clear Motivation:** The contrast with sequence-to-sequence models is well-articulated and highlights the paper’s unique value.  

**Minor Concerns:**  
1. **Target Integration:** The manuscript does not clarify how the binary target *t* is incorporated into the sender’s network architecture. Is *t* concatenated with image features, or is the sender conditioned on *t* via a separate module? This detail is critical for reproducibility and understanding the learning dynamics.  
2. **Table Presentation:** Tables 1 and 2 report percentages inconsistently—Table 1 uses [0, 100] (e.g., 85.2%), while Table 2 uses [0, 1] (e.g., 0.852). This is confusing and must be standardized. Additionally, the "obs-chance purity" values in Table 1 appear implausibly small (e.g., 0.0003), suggesting a possible data error or mislabeling.  
3. **Terminology:** "Asses" (Table 3) should be "Assess," and "usufal" (Section 3.2) should be "usual."  

**Recommendation:**  
This work presents a compelling, well-executed study of grounded communication in multi-agent systems. The task design, empirical results, and analysis of symbolic meaning significantly advance the field beyond prior work on language emergence. While the minor issues noted above require correction, they do not undermine the paper’s core contributions. The approach offers a promising pathway for future research in interpretable, functional language learning. **I strongly recommend acceptance.**  

**Summary of Required Revisions:**  
- Clarify the sender’s architecture (how *t* is integrated).  
- Standardize percentage reporting across tables (all [0, 1] or all [0, 100]) and correct "obs-chance purity" values.  
- Fix spelling errors ("asses" → "assess," "usufal" → "usual").
