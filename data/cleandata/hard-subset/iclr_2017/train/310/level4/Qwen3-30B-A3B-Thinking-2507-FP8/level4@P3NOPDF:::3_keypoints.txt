- Proposes a novel referential communication game where a sender encodes a binary target into a message and a receiver predicts the target from images.  
- Trains neural agents with REINFORCE using pretrained image features.  
- Agents converge to optimal policies and learn semantically meaningful symbolic communication protocols.  
- Extends to image-classification variants with more interpretable communication.  
- Introduces multi-task setup where sender alternates between communication and supervised classification.  
- Grounding communication in a supervised task yields more meaningful symbolic representations.  
- Framing establishes a framework for studying functional communication, moving beyond statistical sequence-to-sequence paradigms.  
- Core insight: end-to-end learned communication can be grounded in human language, addressing semantic grounding limitations of seq2seq models.  
- Thorough analysis and visualizations (concept alignment) provide empirical evidence for meaningful symbol emergence.  
- Task design is a well-justified controlled first step toward complex communication scenarios.  
- Strength: novel, testable paradigm for language emergence.  
- Strength: systematic experiments showing progression from basic to grounded interpretable symbols.  
- Strength: strong interpretability via visualization of learned symbol concepts.  
- Strength: clear motivation distinguishing this work from seq2seq approaches.  
- Minor concern: unclear how binary target t is integrated into the sender’s architecture (concatenation vs. separate conditioning).  
- Minor concern: inconsistent percentage reporting across tables (some [0,100], some [0,1]) and implausible "obs-chance purity" values suggesting errors.  
- Minor concern: typographical errors ("asses"→"assess", "usufal"→"usual").  
- Recommendation: strongly recommend acceptance.
