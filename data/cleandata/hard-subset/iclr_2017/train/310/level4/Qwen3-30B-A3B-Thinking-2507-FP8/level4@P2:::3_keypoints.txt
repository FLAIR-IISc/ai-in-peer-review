- Introduces a novel multi-agent framework for language learning via referential games (sender/receiver coordinate to identify a target image).
- Shows simple neural networks trained with reinforcement learning can develop effective communication protocols.
- Emergent protocols form symbolic codes that capture semantic image properties rather than low-level visual features.
- Removing low-level common knowledge in the environment encourages more abstract, human-like semantic representations.
- Proposes a grounding strategy: alternating referential game play with supervised image classification to improve human interpretability.
- Reports strong empirical results: near-perfect coordination across configurations and semantic alignment verified by purity metrics and t-SNE visualizations.
- Grounding experiment yields substantially improved human interpretability (68% accuracy in crowdsourced evaluation vs. 1% chance).
- Paper shifts focus from statistical language modeling to functional communication, offering a path toward human-interpretable conversational agents.
- Recommendation: accept — addresses a fundamental limitation of current language-learning approaches and provides compelling evidence.
- Well-motivated placement in literature, connecting multi-agent systems, linguistic evolution, and cognitive science while building on signaling-game theory.
- Clear methodology and reproducible experimental descriptions, with appropriate statistical rigor.
- Analysis comparing informed vs. agnostic sender architectures provides insights on inductive biases affecting language emergence.
- Requested clarification: specify how the target variable (t) is provided to the sender network, as current description is ambiguous.
- Presentation fixes needed: standardize percentage formatting in Tables 1 and 2 and correct apparent formatting errors (obs-chance purity values).
- Minor typos to correct: "assest" → "assess" and "usufal" → "usual."
- Suggested extension: discuss scalability to larger vocabularies and more complex environments for practical applicability.
- Key questions for authors: expected performance with 1,000+ symbols and complex visuals; which semantic patterns humans used to interpret symbols and how to improve them; provide quantitative evidence that the informed sender uses more varied, word-like symbols.
