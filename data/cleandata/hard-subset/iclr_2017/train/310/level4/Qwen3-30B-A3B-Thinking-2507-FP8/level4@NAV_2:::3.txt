

This paper introduces a novel framework for developing language-capable agents through multi-agent referential games, where two agents (sender and receiver) interactively learn to coordinate using symbolic communication without pre-defined language. The authors demonstrate that simple neural networks, trained via REINFORCE with image representations from pretrained VGG, successfully converge to communication protocols that associate symbols with high-level semantic categories (e.g., "animal" vs. "vehicle") rather than low-level visual features. Crucially, they propose grounding this emergent language in human natural language by combining the referential game with a supervised image classification task, showing that agents learn to use conventional object labels (e.g., "dolphin") that humans can interpret with 68% accuracy in a crowdsourced experiment. The approach represents a significant step toward functional language learning that captures communication's interactive purpose, rather than merely statistical language patterns.

Accept. The paper makes two substantial contributions: (1) it establishes a scalable paradigm for emergent language development through multi-agent interaction, and (2) it provides the first systematic method to ground this language in human semantics via supervised pre-training. Both are necessary for building human-interpretable conversational agents.

The paper rigorously supports these claims. First, the agents consistently achieve near-perfect coordination (98–100% success) across architectures and vocabulary sizes, with the informed sender using more semantically diverse symbols (88/100) than the agnostic sender (2 symbols). Second, the semantic analysis—using McRae category purity (70% vs. 33% chance), t-SNE visualizations, and human interpretability tests—demonstrates that the language reflects high-level conceptual properties. Third, the grounding experiment with supervised learning on ImageNet labels directly bridges the gap between emergent symbols and human language, with 68% human interpretation accuracy (vs. 1% chance), and reveals flexible metonymic usage (e.g., "dolphin" for sea scenes). The framework is well-motivated by game theory and cognitive science, and the experiments address the central question of how to design environments fostering portable, human-compatible language.

To further strengthen the paper:  
1. Clarify how the target variable (t) is encoded in the sender network (e.g., is it a one-hot vector, or does the sender infer it from image pair context?).  
2. Standardize table values: Table 1 uses [0,100] percentages while Table 2 uses [0,1]; correct all to [0,100] with consistent labeling. The "obs-chance purity" values in Table 1 (e.g., 0.014) appear to be a formatting error; these should be percentages (e.g., 1.4%).  
3. Replace "usufal" with "usual" in Section 5.  
4. Correct "assest" to "assess" in Section 4.  
5. Briefly address why the informed sender’s 50-dimensional SVD decomposition (Fig 2) implies semantic richness despite potential synonymy.
