- Introduces a cooperative two-player referential game (sender generates symbolic message; receiver picks target image) with agents trained from scratch via reinforcement learning.  
- Demonstrates agents can develop a discrete communication protocol and converge to near-optimal policies.  
- Uses analyses (purity, SVD, t-SNE) showing the emergent language captures meaningful semantic image properties beyond low-level features.  
- Shows altering the environment to force coordination on object-level properties yields more abstract, generalizable symbol meanings.  
- Proposes a hybrid grounding strategy: interleave the referential game with supervised image labeling to ground emergent language in human language.  
- Hybrid grounding increases interpretability of agent code and enables some successful human-agent communication on novel tasks.  
- Decision: Accept.  
- Primary reason 1: Novel and important problem formulation focusing on emergent, grounded communication rather than supervised pattern matching.  
- Primary reason 2: Strong, insightful empirical results with analyses that support the paper’s claims.  
- Paper is well-situated in related work (multi-agent RL, game theory, cognitive science) and motivated to move beyond static conversational data.  
- Empirical validation is thorough: establishes tabula rasa baseline and dissects what is learned using purity and SVD analyses.  
- Section 4.1 demonstrates how task/common-knowledge design shapes the properties of emergent language.  
- Section 5’s grounding experiment is a key contribution: purity increases and crowd-sourced tests show grounding generalizes (including metonymic uses).  
- Suggestion: Standardize table formatting (percentage scales inconsistent across tables).  
- Suggestion: Fix minor typos (e.g., “assest” → “assess”, “usufal” → “usual”).  
- Suggestion: Briefly speculate on how more complex architectures (e.g., recurrent nets) might enable multi-turn or compositional communication.  
- Question: Clarify how the target indicator t is provided to the sender (concatenated binary flag vs implicit via input order).
