- Paper introduces a referential game where sender and receiver agents develop a discrete-symbol communication protocol via reinforcement learning to identify a target image among two images.
- Core contribution: demonstrates that two neural agents, initialized from scratch, can learn to coordinate and solve the task through RL.
- Experiments explore how architecture (informed vs. agnostic sender) and environment variations affect emergent language semantics.
- Presents a method to partially ground emergent symbols in natural language by combining the interactive game with supervised image labeling.
- Experimental setup is intentionally simple, enabling clear, interpretable analysis of learned behaviors and symbol usage.
- Decision: Reject.
- Main reason for rejection: the task appears functionally equivalent to supervised image classification rather than a true emergent-communication problem, reducing novelty.
- Primary concern: the game’s one-shot discrimination incentivizes simple class-label transmission, not richer language properties like compositionality or generalization.
- Sender effectively acts as an image classifier and receiver as a label-matching function, so the communication channel transmits a single class identifier.
- Reinforcement learning seems unnecessarily complex and potentially less efficient than supervised or transfer-learning baselines for this task.
- The paper does not convincingly argue why the RL multi-agent setup is more insightful or powerful than simpler alternatives.
- Positive points: the research direction is important, simplicity aids interpretability, and the informed vs. agnostic sender analysis is insightful.
- Suggested experiment: use target and distractor from the same semantic category to force communication of fine-grained features and encourage compositionality.
- Suggest clarifying relation to standard supervised classification and identifying properties that distinguish the learned protocol from class labels.
- Baseline recommendation: compare against a pretrained sender classifier with only the receiver trained via RL to justify the multi-agent RL framework.
- Grounding suggestion: test supervised labels for abstract attributes (e.g., "shiny," "big") to encourage more descriptive and generalizable communication.
