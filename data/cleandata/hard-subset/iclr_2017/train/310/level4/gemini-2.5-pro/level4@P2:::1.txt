### **Review Summary**

This paper presents a compelling framework for studying the emergence of language through multi-agent communication. The authors use a referential game where a "sender" agent must communicate the identity of a target image to a "receiver" agent using a vocabulary of arbitrary symbols. The core contributions are showing that agents can successfully learn to coordinate from scratch, investigating how changes to the game environment (such as agent architecture or the information agents share) can induce more "semantically natural" language, and demonstrating a method for grounding the emergent language in human language by combining interactive gameplay with a supervised labeling task. The finding that this grounding generalizes to new objects and is partially interpretable by humans is particularly noteworthy.

### **Decision: Reject**

My primary reasons for this decision are:
1.  The experimental setup, while clean and intuitive, simplifies the communication problem to a degree that it closely resembles a supervised classification task, potentially limiting the claims about emergent *language*.
2.  The task does not necessitate the development of compositional or descriptive language, as coordination can be achieved by simply learning discriminative labels for object categories.

### **Supporting Arguments**

The central issue is that in the primary experimental setup, the target and distractor images are sampled from different object concepts (e.g., a 'cat' vs. a 'car'). In this scenario, the sender's optimal strategy is to generate a unique symbol corresponding to the target's category, and the receiver's task is to map that symbol back to the correct image. This process is functionally equivalent to the sender acting as an image classifier and the receiver using its output to make a selection. While implemented within a reinforcement learning framework, the underlying problem structure is that of supervised learning. The communication protocol becomes a set of learned class labels rather than a language with descriptive or compositional properties.

This is supported by several observations in the paper. The "agnostic" sender, for instance, converges to using only two symbols, which the authors suggest might represent a "living-vs-non-living" distinction—a high-level classification. The "informed" sender, which uses more symbols, is architecturally biased to perform dimension-wise comparisons between the image embeddings, which is an effective strategy for discrimination but not necessarily for developing a descriptive language.

The work's claims would be substantially stronger if the task required more nuanced communication. If the target and distractor were from the same category (e.g., two different dog breeds, or two different cars), the agents would be forced to communicate specific visual attributes rather than just the object's class. The current setup does not pressure the agents to develop a protocol that can describe *what* is in the image, only to label *which* of the two distinct categories it belongs to.

### **Additional Feedback and Questions for the Authors**

The following points are intended as constructive feedback to strengthen the paper and are not the primary basis for my decision. This is a very interesting research direction, and I believe addressing these points could make the work much more impactful.

**1. Increasing Task Complexity:**
The most significant improvement would be to design experiments where a simple classification strategy is insufficient.
*   **Intra-category distinction:** How do the agents perform when the target and distractor images are drawn from the *same* semantic category? This would force the emergence of symbols that correspond to finer-grained visual attributes (e.g., color, shape, orientation).
*   **More distractors:** What happens if the receiver must choose the target from a set of three or more images? This increases the required precision of the sender's message.

**2. Analysis of Language Properties:**
The analysis of the emergent language is a good start. It could be extended to investigate properties more characteristic of human language.
*   **Compositionality:** Have you considered extending the framework to allow for multi-symbol messages? This would be a crucial step to see if agents can learn to combine symbols to create new meanings (e.g., one symbol for "red" and another for "car" to refer to a red car).
*   **Systematicity:** Does the learned symbol for "cat" in a 'cat' vs. 'car' game have any relation to the symbol for "cat" in a 'cat' vs. 'dog' game? Analyzing the consistency of symbol usage across different contexts would provide deeper insight into whether the agents are learning stable meanings.

**3. Grounding Experiment Details:**
The grounding experiment in Section 5 is the most compelling part of the paper.
*   Could you elaborate on the design choice of having only the sender receive supervised training? Was training both agents or only the receiver considered? It seems the receiver would also benefit from grounding to better interpret the sender's (now human-aligned) messages.
*   The metonymic links discovered (e.g., "dolphin" for "sea") are fascinating. Is there a way to quantify this effect? For example, could you measure the semantic similarity (e.g., using WordNet or word embeddings) between the word produced by the sender and the actual contents of the image?

**Questions to Clarify My Understanding:**

1.  Could you clarify if any experiments were run where the target and distractor were from the same McRae category? If so, how did the communication success and symbol usage metrics change?
2.  The paper argues that this framework moves beyond supervised learning by capturing the "functional aspects of communication." Given that the primary task can be solved with a discriminative labeling strategy, could you elaborate on what functional aspects (beyond discrimination) you believe are being captured in the current setup?
