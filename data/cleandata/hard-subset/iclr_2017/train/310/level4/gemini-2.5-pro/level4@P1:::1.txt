### Summary

This paper introduces a framework for studying emergent communication between AI agents. The authors design a referential game where a "sender" agent must communicate the identity of a target image to a "receiver" agent from a pair of images, using a vocabulary of arbitrary symbols. The core contribution is demonstrating that two neural network agents, starting as blank slates, can successfully develop a communication protocol to solve this task through reinforcement learning.

The paper investigates several interesting questions within this framework. First, it shows that agents can indeed learn to coordinate effectively. Second, it explores how architectural choices (e.g., "informed" vs. "agnostic" senders) and modifications to the game environment (e.g., varying image instances for the same object class) can influence the emergent language to better align with human-like semantic concepts. Finally, it presents a promising method for grounding the agents' ad-hoc language in natural language by combining the interactive game with a supervised image labeling task, showing that this can lead to a communication protocol that is partially interpretable by humans. The experimental setup is commendably simple, allowing for intuitive and insightful analysis of the learned behaviors, such as the difference in symbol usage between the informed and agnostic architectures.

### Decision and Key Reasons

**Decision:** Reject

My recommendation is based on the following key reason:

1.  The core task, as formulated, appears to be functionally equivalent to a supervised image classification problem rather than a genuine exploration of emergent communication. This significantly reduces the novelty of the findings and the justification for the reinforcement learning approach.

### Supporting Arguments

The primary concern with this work is that the proposed referential game may not be sufficiently complex to necessitate the emergence of a rich communication protocol. Instead, it seems to reduce to a simpler, well-understood machine learning paradigm. In the described setup, the sender is given a target and a distractor image (from different categories) and must produce a symbol. The receiver uses this symbol to identify the target.

This process can be reframed as follows: the sender acts as an image classifier, learning to map the target image to a discrete label (a symbol). The receiver then acts as a matching function, learning to associate that label with the corresponding image. Because the agents' only goal is to succeed at this one-shot discrimination task, there is little incentive to develop a language with properties like compositionality or generalization beyond simple labeling. The communication channel is not used to convey nuanced information but rather to transmit a single class identifier.

Consequently, the reinforcement learning framework, while functional, seems unnecessarily complex and potentially less efficient than a more direct approach. The entire system could likely be framed as a supervised or transfer learning problem, where the sender is explicitly trained to classify the target and the receiver learns to use this output. Such a setup would likely train faster and more stably than the REINFORCE algorithm used here. While the paper successfully demonstrates that agents *can* learn in this RL setting, it does not convincingly argue why this is a more insightful or powerful approach than simpler alternatives for this specific task. The findings, therefore, feel less like a discovery about emergent language and more like a demonstration that a neural network can learn to classify images and another can learn to use those classifications, connected by a discrete communication channel.

### Additional Feedback and Questions for the Authors

Despite the concerns that led to my recommendation, I believe the paper explores a very important and exciting research direction. The motivation is strong, and the goal of understanding how to build interactive, communicating agents is crucial for the future of AI. The points below are intended as constructive feedback to help strengthen the work.

**Positive Points:**
*   The problem of fostering communication between agents is of fundamental importance, and this paper brings a fresh perspective to it.
*   The simplicity of the task is a double-edged sword; while it raises the concerns above, it is also a strength, as it allows for clear, interpretable post-hoc analysis of the learned communication protocol.
*   The analysis comparing the symbol usage of the "informed" versus "agnostic" sender was particularly insightful.

**Questions and Suggestions for Improvement:**
To strengthen the claim that the agents are truly developing a language, I would encourage the authors to consider experiments that cannot be easily solved by simple classification.

1.  What happens if the target and distractor images are from the same semantic category (e.g., two different dog breeds, or two different chairs)? This would force the agents to communicate more subtle, distinguishing features rather than just a high-level class label, potentially fostering a more complex and compositional language.
2.  Could you comment on the relationship between your task and a standard supervised classification task? What fundamental properties of the learned protocol differentiate it from a set of learned class labels?
3.  Have you considered a baseline where the sender is a pre-trained image classifier and only the receiver is trained with RL? A comparison against more direct, non-RL baselines would help justify the necessity and benefits of the current multi-agent RL framework for this problem.
4.  The experiment grounding the communication in human language is very interesting. How does the system perform when the supervised labels are for more abstract concepts (e.g., "shiny," "big") rather than object names? This might encourage the agents to develop a more descriptive and generalizable communication system.
