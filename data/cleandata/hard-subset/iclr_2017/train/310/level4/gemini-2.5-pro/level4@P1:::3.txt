### Review

**Summary**

This paper proposes a multi-agent framework for language learning based on a cooperative referential game. In this game, two agents, a "sender" and a "receiver," observe a pair of images. The sender is informed which image is the target and must send a single symbol from a fixed vocabulary to the receiver. The receiver must then identify the target image based only on this symbol. Both agents receive a reward for successful communication. The authors parameterize the agents as neural networks, using pre-trained image features, and train them with reinforcement learning (REINFORCE).

The experiments demonstrate that the agents successfully learn to communicate and converge to optimal policies. More importantly, the analysis shows that the emergent symbolic language captures meaningful semantic concepts. The authors further explore a game variant where the sender and receiver are shown different instances of the same object classes, which encourages the agents to learn more abstract, generalizable concepts. Finally, a multi-task setup is introduced where the sender alternates between the referential game and a supervised image classification task. This combined training effectively grounds the emergent language in human-defined labels, resulting in a communication protocol that is significantly more interpretable, even generalizing to novel concepts in a way that human evaluators can understand.

**Decision and Key Reasons**

I recommend that this paper be **accepted**.

The research direction explored in this paper—learning shared representations for communication in a multi-agent setup—is both interesting and important. This problem is considerably more complex than standard supervised learning or single-agent RL, which justifies the paper's focus on a simplified, well-defined task. To the best of my knowledge, the approach of first allowing agents to develop a communication protocol from scratch and then grounding it in human language is novel and compelling. As the authors argue, this presents a promising alternative to standard sequence-to-sequence models, which often learn the statistical properties of language without capturing its functional, goal-oriented aspects. The proposed framework, along with the thorough analysis and visualization of the emergent communication, represents a valuable contribution and a solid foundation for future work in this area.

**Supporting Arguments**

The paper's primary strength lies in its novel paradigm for language learning. By focusing on the functional goal of coordination, the agents develop a communication system that is inherently meaningful within the context of their shared task. The experiments are well-designed and convincingly show that:
1.  Simple neural agents can successfully develop a communication protocol without any prior linguistic knowledge.
2.  Modifications to the game environment (e.g., showing different image instances) can steer the emergent language towards more abstract and semantically meaningful concepts.
3.  Combining interactive gameplay with a supervised task is an effective strategy for grounding the agents' language in natural language, making it interpretable to humans. This final point is particularly powerful, as it demonstrates a path toward building agents that can communicate productively with people.

The analysis of the learned communication protocol, using metrics like purity and t-SNE visualizations, provides valuable insights and strengthens the paper's claims.

**Additional Feedback for Improvement**

The following points are intended to help improve the clarity of the paper and are not critical to my decision.

*   **Clarification on Target Input:** Could you please clarify how the target information `t` is incorporated into the sender network? Section 3 states that the target and distractor representations are provided "always in this order," implying that the input order signals the target. However, the formal description in Section 2 suggests `t` is a distinct input, `s(θS(iL, iR, t))`. Please clarify if the input order is the sole mechanism for identifying the target to the sender.

*   **Table Formatting Consistency:** There appears to be an inconsistency in the formatting of Tables 1 and 2. The percentages in Table 1 seem to be on a [0, 100] scale, whereas the values in Table 2 appear to be on a [0, 1] scale. For consistency, it would be best to use the same format throughout.

*   **Purity Values in Table 1:** Related to the previous point, the "obs-chance purity" values in Table 1 seem unusually small (e.g., 0.12). If these are intended to be percentage point differences, the values are quite low. Could you please double-check and clarify these figures?

*   **Minor Typos:** The paper contains a few minor typos (e.g., "assest" instead of "assess"). A final proofread would be beneficial.
