- Proposes a referential game with a sender and receiver observing two images; sender receives a binary target and sends a symbol for the receiver to recover the target.  
- Agents are neural networks using pretrained image representations and trained with REINFORCE.  
- Experiments show agents converge to optimal policies and learn communications that reflect meaningful concepts.  
- A variant grounded on different image classes leads to even more meaningful learned concepts.  
- A multi-game setup where the sender alternates between the game and a supervised task produces more symbolic, meaningful communications.  
- Recommendation: paper should be accepted; approach is an interesting and novel direction for grounding communication in human language.  
- Contributions: task/framework and analysis/visualization of communicated tokens are useful stepping stones for future work.  
- Clarification needed on how the target (t) is incorporated into the sender networks—this is not clearly explained.  
- Inconsistency in percentage reporting: Table 1 appears to use [0,100] while Table 2 uses [0,1].  
- Possible error: "obs-chance purity" column in Table 1 contains extremely small values that seem incorrect.  
- Minor typos to fix: "assest" → "assess" and "usufal" → "usual".  
- Requested author responses/questions: details on target incorporation; plans for extending to >2 agents; consideration of other network types (e.g., RNNs); evaluation strategy for more complex tasks like conversational dialogue systems.
