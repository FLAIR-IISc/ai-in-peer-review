- Proposes a referential game with two agents observing two images where a sender receives a binary target t and sends a symbol so the receiver can recover t.  
- Agents are parameterized as neural networks using pretrained image features and trained with REINFORCE.  
- Empirically shows agents converge to optimal policies and learn meaningful symbolic communications.  
- Image-class-grounded variant leads to even more meaningful concepts in the learned communication.  
- Multi-game setup alternates sender between the game and a supervised classification task, which further grounds symbols to meaningful concepts.  
- Claims novelty in learning agent communication first and then grounding it in human language as an alternative to seq-to-seq approaches.  
- Argues the multi-agent communication problem is challenging and justified to begin with a simple task.  
- Reviewer recommends acceptance based on task, framework, analysis, and visualizations.  
- Request for clarification on how the target t is incorporated into the sender network.
