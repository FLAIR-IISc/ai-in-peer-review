- Introduces a referential game where the sender (with a binary target) communicates to a receiver so the receiver can infer the target; both are rewarded for correct prediction.
- Uses neural networks with pre-trained image features and trains agents using REINFORCE.
- Agents converge to optimal policies and develop meaningful communication concepts.
- Variant games grounded in different image classes yield even more meaningful concepts.
- Multi-game setup (alternating game play and supervised learning) produces more meaningful symbolic communications.
- Learning shared representations for multi-agent communication is promising and more challenging than standard supervised/single-agent RL.
- Novel approach: learn inter-agent communication before grounding in human language as an alternative to seq2seq models.
