This paper proposes a referential game between two agents, both of which observe two images. The first agent, referred to as the sender, receives a binary target variable (t) and must send a symbol (message) to the second agent, called the receiver, such that the receiver can recover the target. Both agents receive a reward if the receiver agent can predict the target. The authors propose parameterizing the agents as neural networks, utilizing pretrained representations of the images as feature vectors, and training them using REINFORCE. In this setting, it is demonstrated that the agents converge to optimal policies, and their learned communications (e.g., the symbolic code transmitted from the sender to the receiver) contain meaningful concepts. Furthermore, the paper presents experiments on a variant of the game grounded in different image classes, in which the agents appear to learn even more meaningful concepts. Finally, a multi-game setup is proposed, where the sender agent alternates between playing the game and a supervised learning task (classifying images). Not surprisingly, when anchored to the supervised learning task, the symbolic communications have even more meaningful concepts.

Learning shared representations for communication in a multi-agent setup is an interesting research direction to explore. This task is significantly more challenging compared to standard supervised learning or single-agent reinforcement learning tasks, which justifies starting with a relatively simple task. To the best of my knowledge, the approach of first learning communication between two agents and then grounding this communication in human language is novel. As the authors remark, this may be an alternative paradigm to standard sequence-to-sequence models, which tend to focus on statistical properties of language rather than their functional aspects. I believe the contributions of the proposed task and framework, as well as the analysis and visualization of what the communicated tokens represent, are a useful stepping stone for future work. For this reason, I think the paper should be accepted.

Other comments:

* The paper could benefit from a clarification on how the target (t) is incorporated into the sender networks.
* There is an inconsistency in the use of percentage values in Table 1 and Table 2. Specifically, the first table appears to use percentages in the interval [0, 100], while the second table uses the interval [0, 1]. This discrepancy should be corrected. Perhaps related to this, the "obs-chance purity" column in Table 1 contains extremely small values, which seems to be a mistake.
* There are a few minor typos: "assest" should be replaced with "assess," and "usufal" should be replaced with "usual."
