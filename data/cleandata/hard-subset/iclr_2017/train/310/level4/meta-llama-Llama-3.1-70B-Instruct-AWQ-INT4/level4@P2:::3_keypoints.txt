- Proposes a novel multi-agent framework for language learning via a referential game.  
- Game setup: sender and receiver observe two images and communicate the target image using a symbolic code.  
- Agents are neural networks trained with REINFORCE to learn optimal communication policies.  
- Empirical results show agents converge to successful communication and develop meaningful concepts.  
- Learning shared representations in a multi-agent setup is an interesting and suitably challenging direction.  
- Using a simple task as a starting point is justified given the challenge relative to supervised/single-agent RL.  
- Novel paradigm: learn inter-agent communication first, then ground it in human language as an alternative to seq2seq approaches.  
- Contributions include the task, framework, and analysis/visualization of communicated tokens.  
- Experiments explore image-class variants and a multi-game setup where the sender alternates between game and supervised learning.  
- Reviewer recommends accepting the paper.

- Ask how the approach will scale to more complex tasks (longer symbol sequences, nuanced semantics).
