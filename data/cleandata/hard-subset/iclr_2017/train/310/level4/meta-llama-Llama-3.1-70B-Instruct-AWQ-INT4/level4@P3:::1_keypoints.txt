- Tackles understanding communication between two agents in RL using referential games.  
- Proposes a multi-agent language learning framework where agents develop codes to coordinate and obtain payoffs.  
- Studies whether tabula rasa agents can develop human-like language features and which environmental factors induce semantic structure.  
- Experimental setup uses ImageNet images with manipulated pairings to test coordination while preserving class labels.  
- Results: agents still coordinate under modifications and show a small increase in symbol usage purity.  
- Grounds communication by combining the game with a supervised task; supervised learning provides some human-language grounding that generalizes beyond trained denotations.  
- Main limitation: the task effectively reduces to image classification/transfer learning, so symbols map to image classes and could be learned faster with supervised methods.  
- Despite the limitation, the paper’s focus on emergent language development and semantic analysis is a valuable contribution.  
- Recommendation: explore more complex game environments that require richer, more sophisticated communication.  
- Suggest investigating scalability to more realistic scenarios, especially human-agent communication.  
- Advise providing more analysis of limitations and avenues for future work.  
- Decision: Accept — for addressing a crucial problem and presenting a well-designed, effective experimental evaluation.
