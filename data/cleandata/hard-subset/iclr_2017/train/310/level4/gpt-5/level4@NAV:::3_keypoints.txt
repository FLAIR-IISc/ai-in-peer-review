- Introduces a referential game where a sender and receiver see two images; the sender knows a binary target t and sends a symbol so the receiver can infer the target.  
- Both agents are neural networks using pretrained image features and are trained with REINFORCE; reward given for correct receiver predictions.  
- Empirical results show agents converge to optimal policies and the emergent symbol communication encodes meaningful concepts.  
- A variant grounded in distinct image classes produces even more interpretable induced concepts.  
- A multi-game setup (alternating the referential game with supervised image classification) grounds symbols further, yielding more semantically meaningful content.  
- Reviewer finds shared representations for multi-agent communication a compelling, challenging direction and supports starting with a simple environment.  
- Novelty: learning communication first and then grounding it in human language is noted as a potentially new approach and alternative to sequence-to-sequence methods.  
- Reviewer values the task, framework, analysis, and visualization of token semantics and recommends acceptance.  
- Request for clarification: how is the target variable t injected into the sender’s network architecture?
