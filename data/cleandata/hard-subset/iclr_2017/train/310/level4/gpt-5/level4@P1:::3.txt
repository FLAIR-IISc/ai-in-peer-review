1. Summary of contributions
The paper proposes a simple yet compelling framework for language learning via multi-agent communication, instantiated as a referential game between a sender and a receiver. Both agents see two images; the sender is informed which image is the target and must transmit a discrete symbol so the receiver can identify it. Agents are parameterized by neural networks and trained with REINFORCE using pretrained visual features as input. The authors show that agents reliably learn to communicate and coordinate, and that the emergent symbol usage exhibits semantically meaningful structure. They also explore a variant in which the agents are shown different images of the same classes to reduce low-level common knowledge, which leads to more conceptually coherent symbol meanings. Finally, they introduce a multi-task setup where the sender alternates between the game and a supervised image classification task, grounding the emergent code in natural language labels and yielding more interpretable communications. Overall, the work offers a promising alternative paradigm to purely supervised sequence models by emphasizing the functional, interactive aspects of language.

2. Decision: Accept
Key reasons:
- Novel and well-motivated approach to learning communication protocols from interaction first, then grounding them in human language via supervision.
- Clear empirical evidence that communication emerges, is semantically structured, and becomes more interpretable with simple environmental tweaks and light supervision.

3. Supporting arguments
- Problem and approach: The paper addresses how to train interactive agents that develop their own communication protocols and, crucially, how to nudge these protocols toward human-interpretable semantics. The referential game setting is a clean testbed that isolates core challenges of reference and symbol emergence while remaining scalable.
- Motivation and placement: The work is well-positioned with respect to supervised seq2seq approaches (which capture statistical structure but not interaction), prior multi-agent communication without pre-coded protocols, and human-in-the-loop or scripted environments that face scalability or engineering bottlenecks. The focus on interaction, simple environments, and realistic visual inputs makes the contributions timely and relevant.
- Empirical support: Across configurations, agents converge to successful coordination using discrete symbols. Analyses suggest that symbol usage reflects broad semantic organization rather than low-level visual cues; furthermore, removing low-level common knowledge (by using different images from the same classes for sender and receiver) increases semantic coherence. Grounding via a concurrent supervised task further improves interpretability, consistent with the idea that light supervision can align emergent codes with natural language while preserving interactive learning benefits. The analyses and visualizations are informative and help substantiate the claims.

4. Additional feedback and questions (to help improve the paper; not all are decision-critical)
- Clarify sender input and target incorporation:
  - How exactly is the target t provided to the sender? The text notes the sender receives inputs “always in this order,” implicitly marking the target. Please make this explicit in the main description and diagrams, and explain why the receiver’s order is randomized.
- Table formatting and units:
  - Table 1 and Table 2 appear to use different percentage scales (0–100 vs. 0–1). Please standardize. Relatedly, in Table 1 the “obs-chance purity” values look implausibly small—likely a formatting error. Please verify and correct.
- Typos:
  - “assest” → “assess”
  - “usufal” → “usual”
- Policy learning details:
  - Do you use a baseline or variance-reduction technique for REINFORCE? If so, specify; if not, comment on variance and training stability.
  - Please report sensitivity to vocabulary size, temperature, and embedding dimensions. Does increasing the vocabulary encourage synonymy, or does it yield finer-grained semantics?
- Emergent semantics:
  - You note that the emergent code sometimes collapses to a living vs. non-living distinction when few symbols are used. Could you quantify how often this occurs across random seeds and settings?
  - For the SVD analysis of symbol usage, consider adding a measure (e.g., rank estimate or explained variance threshold) to more clearly distinguish synonymy vs. genuinely multidimensional semantics.
- Data and generalization:
  - Please clarify the training/test split to avoid potential leakage from image sampling (e.g., ensuring different images or classes across phases where intended).
  - When moving to class-level reference (different instances of the same ImageNet class), a brief ablation on the degree of instance variation would help illustrate how much low-level common knowledge needs to be removed to see the semantic effect.
- Broader scope:
  - Briefly discuss how this framework might extend to multi-symbol messages or multi-round interaction, and whether you anticipate similar grounding benefits when combining with supervision.

Questions for the authors
- How is the target t encoded for the sender, and could alternative encodings (e.g., explicit binary indicator vs. ordered inputs) affect learning speed or emergent semantics?
- Did you use any baseline in REINFORCE (e.g., moving average, learned critic)? If yes, please detail; if not, how stable was training across seeds?
- How sensitive are results to vocabulary size and temperature τ? Does a larger vocabulary lead to more meaningful subcategories or redundant symbols?
- In the object-level reference variant, can you quantify the purity gains and provide examples of categories whose representations improved most?
- In the grounded multi-task setup, how often does the sender use supervised labels when they are applicable versus resorting to other symbols? Any evidence of over-reliance on supervised words?
- Do you observe consistent convergence across random initializations, and do the learned codes differ substantially across runs? If so, how does this variability affect interpretability?

Overall, this is a well-motivated and carefully executed study that advances interactive, communication-centric learning and offers a pragmatic path toward human-aligned semantics.
