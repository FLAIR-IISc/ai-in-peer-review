Summary of contributions
The paper proposes a simple but compelling framework for learning language through multi-agent interaction, instantiated as referential games. A sender observes two images and a target indicator, emits a single symbol from a fixed vocabulary, and a receiver must identify the target based solely on this symbol. Both agents are neural networks trained end-to-end with REINFORCE, using pretrained CNN image features as input. The paper shows: (i) agents reliably learn to coordinate; (ii) an “informed” sender architecture with an inductive bias toward dimensionwise image comparison learns faster and uses a richer symbol inventory; (iii) the emergent code aligns with intuitive semantics, as measured by category purity and structure in the visual space; (iv) modifying the game to remove low-level common knowledge (by showing sender/receiver different images of the same classes) encourages more category-level semantics; and (v) mixing the game with a supervised image-labeling task grounds symbols in natural language, improving interpretability and enabling limited human-agent communication in a crowdsourced study. The work is well motivated, connects to literature on signaling games and emergent communication, and aims toward practical human-interpretable agents.

Decision: Accept
Reason: The paper presents a novel and well-motivated framework showing that interactive, multi-agent learning can induce meaningful communication protocols aligned with human semantics, with clear evidence from both analysis and human evaluation. The approach is timely, scientifically sound, and provides a valuable foundation for future research toward human-compatible communication.

Supporting arguments
- Clarity of the question and motivation: The paper articulates why passive supervised learning is insufficient for interactive agents and positions referential games as a minimal, scalable testbed. The emphasis on portability and human interpretability is appropriate and well grounded in prior work on signaling games and multi-agent RL.
- Methodological soundness: The use of VGG features focuses the learning problem on communication rather than perception, which is appropriate for isolating the core question. The sender/receiver architectures are simple and transparent. The informed sender’s inductive bias is justified and empirically beneficial.
- Empirical support: 
  - Coordination performance is strong and consistent across settings, with faster convergence for the informed sender.
  - The emergent code exhibits non-trivial semantics: higher category purity than chance, meaningful structure in t-SNE visualizations, and interesting coarse splits (e.g., living vs. non-living) even with a small vocabulary.
  - Tweaking common knowledge via different-image same-class setups yields higher-level semantics, supporting the claim that environment design shapes emergent meaning.
  - Grounding through supervised image labels increases symbol usage, raises purity markedly, and—crucially—enables humans to often infer the intended referent even when images are outside the supervised label set, suggesting flexible, metonymic use of words akin to natural language.
- Positioning in literature: The paper is well placed relative to concurrent work on emergent communication (e.g., Sukhbaatar et al., Foerster et al.) and classic signaling-game theory, with a distinctive emphasis on realism (natural images) and human interpretability.

Additional feedback to improve the paper (helpful suggestions; not all affect my decision)
- Clarify target encoding: Please clearly specify how the target (t) is provided to the sender. The text implies the sender sees a fixed-ordered pair (target first, distractor second), while the receiver sees a random order. Make this explicit and discuss whether order leakage contributes to performance. Have you tried randomizing sender input order and passing an explicit target bit instead?
- Reporting and formatting:
  - Table 1 vs. Table 2: Percentages should be reported consistently (prefer [0, 100]%). The “obs–chance purity” values in Table 1 appear implausibly small; please check and correct.
  - Typos: “assest” → “assess”; “usufal” → “usual”.
  - Provide standard deviations or confidence intervals across multiple runs/seeds for success rates, purity, and symbol usage to assess stability.
- Analysis depth:
  - Quantify synonymy/homonymy in the code beyond SVD spectra (e.g., entropy of symbol distribution, type–token ratio, clustering consistency, symbol–category mutual information).
  - Report sensitivity to the temperature τ, vocabulary size, and embedding dimensionality. How brittle is convergence?
  - Expand on the “informed vs. agnostic” claim that the agnostic architecture nests the informed one: if so, why does the agnostic sender converge to a 2-symbol solution? Is this due to optimization or implicit regularization?
  - Assess generalization to new partners: can a receiver trained with one sender coordinate with a different (held-out) sender without finetuning?
- Controls for low-level cues:
  - In the standard setting where both agents see the same images, low-level correlations could be exploited. The “different-images same-class” control is excellent; consider also adding augmentations or adversarial distractors to more stringently rule out pixel-level shortcuts.
- Human evaluation:
  - Provide more details on the human study: instructions, interface, exact prompt wording, demographics, and quality control. Include confidence or response-time analyses if available.
  - Analyze failure cases and semantic drift: when do humans disagree with the agent and why?
- Reproducibility: Include random seeds, number of runs, and, if possible, release code and trained models.

Questions for the authors
- Precisely how is t encoded for the sender? Did you try sender-side randomization plus an explicit target flag? What happens to performance and emergent semantics?
- How sensitive are results to τ and to the sampling vs. argmax decision rule during training/inference?
- Does rotating roles (sender/receiver) or co-training multiple receivers/senders improve robustness or interpretability?
- Can the learned code support limited compositionality (e.g., with multi-symbol messages), and does this improve human interpretability?
- In the grounded setting, what proportion of supervised labels are actually used during game play, and how often are they used metonymically versus denotationally?
- How well do agents generalize to new object categories not seen during supervised grounding?
- Have you tested cross-play between independently trained agents to evaluate convergence to shared protocols vs. idiosyncratic codes?

Overall, this is a clear, carefully executed paper that advances emergent communication with an eye toward human interpretability. The study’s design choices, analyses, and results convincingly support its claims. I recommend acceptance.
