Summary of contributions
The paper studies interactive language learning via multi-agent communication. It introduces a referential game in which a sender sees which of two images is the target and sends a discrete symbol from a fixed vocabulary; the receiver, seeing both images and the symbol, must identify the target. Both agents are neural networks trained end-to-end with REINFORCE, using pretrained CNN image features as inputs and a discrete communication bottleneck. The authors show that:
- Agents reliably learn to coordinate and achieve near-perfect accuracy.
- The emergent symbols exhibit semantic regularities: symbol usage correlates with higher-level object categories and visual similarity, supported by purity analyses, SVD, and t-SNE.
- Modifying the game to remove low-level common knowledge (showing different instances of the same ImageNet class to sender and receiver) nudges the protocol toward more abstract, class-level semantics.
- Combining self-play with supervised object naming (shared embeddings) grounds symbols in natural language, improving purity and interpretability, and enabling partial human–agent communication in a crowdsourced study.

The paper addresses an important problem (emergent communication for interactive AI), motivates the approach against purely supervised sequence models, situates it within work on signaling games and multi-agent RL, and provides empirical support for its claims.

Decision
Accept. Reasons:
1) Novel and well-motivated framework that combines emergent communication with simple grounding in natural language, addressing functional aspects of language.
2) Convincing empirical evidence that tabula rasa agents learn to communicate and that symbol meanings align with intuitive semantics, with a promising human evaluation.

Supporting arguments
- Problem and motivation: The paper tackles how to induce portable, interpretable communication protocols via interaction, a clear step toward conversational agents. The limitations of passive supervised learning for functional communication are well articulated and connected to prior work in linguistics and game theory.
- Approach: Architectures (agnostic vs informed sender) are simple yet appropriate for isolating inductive biases; the discrete symbol bottleneck and REINFORCE training reflect key aspects of language. Using strong, widely adopted visual features (VGG) focuses the study on communication rather than perception.
- Empirical support: Agents consistently converge to high coordination accuracy. Analyses suggest genuine semantic structure rather than degenerate two-symbol solutions: the informed sender uses more symbols; SVD indicates multi-dimensional structure; category purity exceeds chance across settings. The object-level reference variant increases abstraction as hypothesized. The supervised grounding experiment substantially improves purity and interpretability, and the human study (68% success) provides initial evidence that the learned code can be partially understood by humans beyond directly supervised labels.
- Positioning: The work complements and extends emergent communication studies (e.g., Sukhbaatar et al., Foerster et al.) by explicitly steering and evaluating interpretability and human alignment, and by proposing a simple, scalable route to grounding without requiring humans in the loop during training.

Additional feedback to help improve the paper (not central to the accept decision)
- Clarify exactly how the target t is provided to the sender (ordered inputs vs an explicit target indicator), and whether the receiver sees images in random order while the sender always sees target first.
- Reporting consistency: Standardize percentage reporting across tables (use either [0,1] or [0,100] consistently). In Table 1, the “obs–chance purity” values appear implausibly small—please verify the scaling and the chance baseline computation.
- Typos: “assest” → “assess”; “usufal” → “usual”.
- Training details: State whether you used a variance-reduction baseline for REINFORCE, number of random seeds and variance across runs, and whether VGG features were frozen. Report sensitivity to temperature τ, vocabulary size, and embedding dimensions.
- Symbol usage: Quantify effective vocabulary size (e.g., entropy, type–token ratio) and distinguish true polysemy/synonymy from redundancy. The SVD analysis is helpful; consider adding a controllable synonymy experiment or mutual information between symbols and categories.
- Statistical testing: For purity and human-study results, include confidence intervals and significance tests; detail the permutation procedure for chance purity.
- Human study: Provide task instructions, demographics, quality control, agreement statistics, and examples of successes/failures. Clarify how words were mapped to symbols and whether participants saw labels that mismatched obvious image content.
- Scope: Discuss limitations (e.g., single-symbol messages, two-image choice) and outline how the framework extends to sequences, multi-turn dialogs, or larger candidate sets.

Questions for the authors
- Sender input: Precisely how is t incorporated into the sender’s input? Is the target always presented first, and does the sender receive any explicit target flag?
- REINFORCE: Did you use a baseline or other variance-reduction techniques? How stable are results across random seeds?
- Hyperparameters: How sensitive are convergence, purity, and human interpretability to τ, vocabulary size (e.g., 10 vs 100), and embedding dimensionality?
- Symbol semantics: Beyond SVD, can you quantify the number of “effective” symbols (e.g., via clustering of symbol–context distributions)? How much synonymy emerges?
- Grounding generalization: When supervised labels do not overlap with game categories, how does interpretability and communication accuracy change? Any negative transfer?
- Chance baselines: How exactly is chance purity computed (permutation over symbol–object assignments)? Are reported “obs–chance” values scaled in [0,1] or [0,100]?
- Human study details: How were image pairs sampled per symbol? What was the exact prompt? Did you analyze per-word performance (e.g., metonymic vs literal uses)?
- Feature usage: Were CNN features frozen? Have you tried learning visual features jointly, and if so, does this harm or help semantic alignment?
- Generalization: Do agents trained on one image distribution transfer to another without retraining, especially in the object-level reference variant?
