This paper proposes a reference game between two agents. Both agents observe two images. The first agent, called the sender, obtains a binary target variable (t) and must send a symbol (message), called the receiver, to the second agent so that this agent can retrieve the target. If the receiver agent can predict the target, both agents receive a reward. This paper proposes parameterizing the agents as neural networks – with feature vectors as pre-trained representations of images – and training them using REINFORCE. In this setting, it is shown that the agents converge on optimal policies and that their learned communication (such as the symbolic code sent from the sender to the receiver) contains some meaningful concepts. Furthermore, this paper presents an experiment on a type of game based on different image classes. In this setting, the agents seem to learn even more meaningful concepts. Finally, a multi-game setup is proposed, where the sender agent alternately plays the first game and a supervised learning task (image classification). Not surprisingly, when combined with a supervised learning task, the symbolic communication contains even more meaningful concepts.

Learning shared representations for communication in a multi-agent setup is an interesting research direction that should be explored. This is a much more difficult task than standard supervised learning or single-agent reinforcement learning tasks, which justifies starting with a relatively simple task. To my knowledge, learning communication between the first two agents and then establishing this communication in human language is a new approach. As the authors noted, this could be an alternative paradigm to the standard sequence-to-sequence model that focuses on statistical properties rather than the functional aspects of language. I believe that the contribution of the proposed task and framework, and the analysis of what the sent token represents, is a useful step for future work. For this reason, I think this paper should be accepted.
