- Proposes a referential game where a sender, given a binary target t and two images, sends a symbol so a receiver can predict the target.  
- Agents are neural networks using pretrained image features and trained with REINFORCE.  
- Experiments show agents converge to optimal policies and learned communications develop meaningful concepts.  
- A variant grounded in different image classes produces even more interpretable concepts.  
- A multi-game setup alternates the referential game with supervised image classification, grounding communications and improving concept meaning.  
- Learning shared communication representations in a multi-agent setting is an interesting, challenging direction and justifies the simple task choice.  
- Claimed novelty: first to learn communication between agents then ground it in human-understandable concepts, offering an alternative to sequence-to-sequence approaches.  
- Contributions, analysis, and token visualizations are valuable; reviewer recommends acceptance.  
- Please clarify how the target variable t is incorporated into the sender network architecture.  
- Correct table formatting: percentage values are used inconsistently across Table 1 and Table 2, and the “obs-chance purity” values in Table 1 look suspiciously small.  
- Fix typos: “assest” → “assess” and “usufal” → “usual.”
