- Recommendation: paper should be accepted.
- Task & method: proposes a referential game where sender and receiver neural agents learn a symbolic code via reinforcement learning.
- Main result: agents learn to coordinate effectively and produce emergent communication with semantic structure tied to object categories.
- Grounding: combines game-playing with supervised learning to align emergent codes with human language.
- Contribution: provides a framework for studying emergent communication without pre-defined protocols or large labeled datasets.
- Insight: explores how environmental factors (e.g., focusing on object classes) shape emergent language properties.
- Analysis: uses SVD and t-SNE to illustrate and support the semantic structure of learned codes.
- Clarification requested: specify how the target variable (t) is fed into the sender network (separate input vs. image presentation order).
- Table consistency: ensure consistent percentage formatting across tables and verify unusually small “obs-chance purity” values.
- Typos: correct “assest” → “assess” and “usufal” → “usual.”
- Ablation: add studies on hyperparameters (embedding dimensionality, vocabulary size) to assess impact on performance and interpretability.
- Qualitative examples: include more image-symbol pairs with interpretations to aid understanding.
