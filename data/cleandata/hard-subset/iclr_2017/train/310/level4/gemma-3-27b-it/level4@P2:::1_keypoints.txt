- Addresses an important problem: developing inter-agent communication beyond reactive behavior toward more human-like interaction.  
- Proposes a multi-agent communication framework using referential games where agents learn to coordinate and develop a language.  
- Strength: simple referential-image experimental setup allows intuitive analysis of learned communication strategies.  
- Insight: informed agents used a more diverse vocabulary than agnostic agents.  
- Recommendation: Reject.  
- Core critique: results look like sophisticated transfer learning rather than true emergent communication.  
- Argument: when images represent different object categories the task reduces to image classification, so symbols map to pre-defined categories.  
- Reliance on pre-trained CNN features means agents bootstrap on existing semantic knowledge, undermining the claim of emergent language from scratch.  
- Questioning the necessity of RL: a supervised classification approach could likely achieve similar or better results more efficiently.  
- Suggestion: Clarify novelty and explicitly distinguish this work from standard transfer learning, especially by examining same-category image cases.  
- Suggestion: Conduct ablation studies comparing the RL approach to a supervised baseline with the same architecture and data.  
- Suggestion: Provide deeper analysis of symbol usage, including learned-embedding analyses and redundancy exploration.  
- Suggestion: Explicitly discuss limitations, notably reliance on pre-defined object categories.  
- Suggestion: Explore alternative, more challenging environments where distinctions depend on relationships or subtler features to better test emergent communication.  
- Questions for authors: effects when both images are same category; RL vs supervised performance comparison; visualization of learned symbol embeddings; sensitivity to hyperparameters (learning rate, temperature); any human-in-the-loop experiments and their outcomes.
