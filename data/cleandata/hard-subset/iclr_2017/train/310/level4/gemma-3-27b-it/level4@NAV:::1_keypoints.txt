- Addresses a fundamental question in RL: how agents learn to communicate and whether this mirrors human strategies.
- Uses a simple task that enables clear post-learning analysis and interpretation.
- Finds a notable difference in symbol usage: informed agents use a richer symbol set, agnostic agents collapse to a two-symbol vocabulary.
- Critique: the task effectively reduces to image classification/transfer learning (symbols act as class labels), so more efficient non-RL training methods might be applicable.
