- Introduces a referential game where the sender sees two images and a binary target, sends a symbol so the receiver can predict the target; both agents rewarded for correct prediction.  
- Agents implemented as neural networks using pre-trained image features and trained with REINFORCE.  
- Demonstrates agents converge to optimal policies and develop symbolic communication encoding meaningful concepts.  
- Variant using different image classes produces even more interpretable concepts.  
- Multi-game setup alternating referential game and supervised classification grounds communication, yielding more semantically rich symbols.  
- Novel and compelling research direction: learning shared communicative representations in a multi-agent framework and grounding them in human-understandable concepts as an alternative to seq2seq models.  
- Thorough analysis and visualizations; overall contribution is valuable and the paper is recommended for acceptance.
