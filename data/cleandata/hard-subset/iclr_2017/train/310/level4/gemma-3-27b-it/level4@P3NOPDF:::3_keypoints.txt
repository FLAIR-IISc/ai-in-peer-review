- Introduces a referential game between a sender and receiver to study emergence of meaningful communication.
- Game setup: both agents see two images; sender receives binary target t and sends a symbol so receiver can predict t; both rewarded for prediction success.
- Agents are neural networks using pre-trained image features and trained with REINFORCE.
- Results: agents converge to optimal policies and the learned symbolic code develops meaningful concepts.
- Class-grounded variant produces even more interpretable concepts.
- Multi-game setup alternates the referential game with supervised image classification, which anchors communication and yields more semantically coherent symbols.
- Claimed novelty: grounding emergent agent communication in human-understandable concepts is a novel and valuable alternative to seq2seq paradigms.
- Overall recommendation: paper should be accepted.
