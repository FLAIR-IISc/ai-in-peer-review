This paper proposes a reference game between two agents. Both agents observe two images. The first agent, called the sender, receives a binary target variable (t) and is forced to send a symbol (message) to the second agent (called the receiver) so that this agent can retrieve the target. If the receiver agent can guess the target, both agents receive a reward. This paper proposes parameterizing the agents as neural networks – where pre-trained representations of the images are used as feature vectors – and training them using REINFORCE. In this scenario, it is shown that the agents converge on an optimal policy and their learned communications (such as the symbolic code transmitted from the sender to the receiver) have some meaningful concept. Furthermore, this paper presents an experiment on a variation of the game based on different image classes. In this scenario, it seems that the agents can learn more meaningful concepts. Finally, a multi-game setup is proposed, where the sender agent first plays the game and then performs a supervised learning task (classifying the images). It is not surprising that, combined with the supervised learning task, there are more meaningful concepts of symbolic communication.

Learning shared representations for communication in a multi-agent setup is an interesting research area. It is a much more difficult task than standard supervised learning or single-agent reinforcement learning tasks, which support starting with relatively simple tasks. As far as I know, it is a new approach to first learning communication between two agents and then grounding this communication in human language. As the authors note, it could be an alternative paradigm to the standard sequence-to-sequence model, which focuses more on statistical properties than the functional aspects of language. I believe the proposed work and the contribution of the framework and the analysis and visualization of what the communicating tokens represent are a useful step for future work. For this reason, I believe this paper should be accepted.
