This paper proposes a reference game between two agents. Both agents observe two images. The first agent, called the sender, receives a binary target variable (t) and is obliged to send a symbol (message) to the second agent, called the receiver, so that this agent can retrieve the target. If the receiver agent can guess the target, both agents receive a reward. This paper proposes parameterizing the agents as neural networks – where pre-trained representations of the images are used as feature vectors – and training them using REINFORCE. In this scenario, it is shown that the agents converge on an optimal policy and their learned communications (such as the symbolic code transmitted from the sender to the receiver) have some meaningful concept. Furthermore, this paper presents an experiment on a variation of the game based on different image classes. In this scenario, it seems that the agents can learn more meaningful concepts. Finally, a multi-game setup is proposed, where the sender agent first plays the game and then performs a supervised learning task (classifying the images). It is not surprising that, when combined with a supervised learning task, the symbolic communications have a more meaningful concept.

Learning shared representations for communication in a multi-agent setup is an interesting research area. It is a much more difficult task compared to standard supervised learning or single-agent reinforcement learning tasks, which supports starting with a relatively simple task. As far as I know, it is a new approach to first learn communication between two agents and then ground this communication in human language. As the authors note, this could be an alternative paradigm to the standard sequence-to-sequence model, which focuses more on statistical characteristics than the functional aspects of language. I believe the contribution of the proposed task and framework, and the analysis and visualization of what the communicating tokens represent, is a useful step for future work. For this reason, I believe this paper should be accepted.
