Training natural language systems by involving multiple agents in an interactive, referential communication game is very promising. As the authors note, there has been some (though not much, it seems) prior work on using multi-agent games to teach communication, and this certainly seems like an area worth pursuing. Furthermore, the changes in these games and some supervised learning methods, as described in the tests in section 5 and proposed in section 6, appear particularly effective.
