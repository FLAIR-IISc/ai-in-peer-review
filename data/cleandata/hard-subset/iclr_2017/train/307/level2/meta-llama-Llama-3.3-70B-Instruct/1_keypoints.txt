- Proposes a testbed to evaluate end-to-end goal-oriented dialog systems in the restaurant reservation domain.  
- Designs five tasks requiring sentence/symbol manipulation, API calls, and use of API outputs.  
- Evaluates multiple methods: rule-based systems, classical IR models, supervised embeddings, and end-to-end Memory Networks.  
- Main claim: the testbed can break down strengths and shortcomings of end-to-end dialog systems.  
- Empirical support: task-level results covering issuing/updating API calls, displaying options, providing extra info, and full dialogs.  
- Decision to accept: paper is well-motivated and the testbed is well-designed and clearly described.  
- Addresses a significant gap: provides a systematic, controlled evaluation for goal-oriented dialog learning.  
- Key finding: Memory Networks outperform other models on most tasks but struggle with entity reasoning and presenting results.  
- Suggested improvement: include deeper analysis of results and implications for model design.  
- Requests for clarification: plans for releasing the testbed/datasets, intended future uses, and limitations of the testbed.  
- Overall assessment: significant contribution with potential impact; revisions to address analysis and clarity would strengthen the paper.
