- Proposes a testbed for evaluating end-to-end goal-oriented dialog systems in restaurant reservation.
- Designs five tasks requiring sentence/symbol manipulation, API calls, and use of call outputs.
- Evaluates a Memory Networks end-to-end dialog system and compares it to a hand-crafted slot-filling baseline on DSTC2 data.
- Finds Memory Networks achieve promising but imperfect performance and can learn non-trivial operations.
