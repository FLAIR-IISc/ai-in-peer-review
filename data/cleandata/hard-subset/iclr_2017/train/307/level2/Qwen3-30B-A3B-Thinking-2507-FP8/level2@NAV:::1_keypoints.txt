- Proposes a valuable testbed of five restaurant reservation tasks to evaluate end-to-end goal-oriented dialog systems.  
- Demonstrates Memory Networks achieve promising performance while exposing limitations in interpreting API results and handling OOV entities.  
- Validates findings across synthetic data, DSTC2, and a real concierge dataset, showing consistent patterns.  
- Recommends acceptance, citing the paper’s significant contribution as the first systematic testbed for goal-oriented end-to-end dialog systems.  
- Tasks are well-designed to isolate key capabilities required for goal-directed conversations.  
- Comparison across multiple models, including a rule-based baseline, provides strong evidence for the testbed’s utility.  
- Methodological rigor: reproducible testbed, well-motivated tasks, and evaluation spanning synthetic and real-world data.  
- Shows end-to-end models can handle dialog management and API calls but struggle with displaying/interpreting knowledge-base results.  
- Inclusion of match type features to address OOV challenges is particularly insightful.  
- Suggests more detailed analysis on why Memory Networks with match type features underperform on Task 2 (updating API calls).  
- Requests clarification on the training/validation split for the concierge dataset and how temporal data was handled.  
- Recommends a brief discussion on extending the testbed to domains beyond restaurant reservation.  
- Asks how the authors handled distribution shift between synthetic and real-world data and whether synthetic-trained models suffered performance degradation on real datasets.
