- Introduces a novel simulated restaurant reservation dataset as a resource for evaluating end-to-end goal-oriented dialog systems.
- Creates a controlled testbed with five tasks that decompose required dialog capabilities (API calls, state updates, option display, info provision).
- Grounds the simulation in a knowledge base and uses rule-based NL generation to enable reproducibility.
- Shows Memory Networks can learn non-trivial operations and outperform simpler baselines in per-response accuracy.
- Notes per-dialog performance of models remains limited despite per-response gains.
- Evaluates on real-world datasets (DSTC2 and a concierge service) and finds the synthetic data is a reasonable proxy for real-world patterns.
- Recommends acceptance, citing the paper fills a significant gap by providing a well-structured, reproducible testbed for goal-oriented dialog.
- Claims the task decomposition enables meaningful error analysis not possible with existing datasets.
- Suggests publishing the full NL-generation rules to ensure full reproducibility.
- Recommends exploring generation-based approaches (instead of ranking) to better match real-world deployment.
- Asks authors to address practical concerns about creating candidate response lists in new domains and clarify how these lists would be generated beyond restaurants.
