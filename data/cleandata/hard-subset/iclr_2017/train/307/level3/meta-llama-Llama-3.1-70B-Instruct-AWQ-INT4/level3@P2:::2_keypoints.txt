- Introduces a novel testbed for evaluating end-to-end goal-oriented dialog systems in restaurant reservation settings.  
- Proposes five tasks that decompose strengths and weaknesses of end-to-end dialog models.  
- Introduces "match type" features as a simple modification to Memory Networks to better handle entities and OOV words.  
- Provides a thorough evaluation across models (rule-based, IR, supervised embeddings, Memory Networks).  
- Shows end-to-end models can learn non-trivial operations but struggle with goal-oriented dialog, especially entity interpretation and presenting results.  
- Decision: accept.  
- Key reasons for acceptance: contribution of the testbed and a well-designed experimental protocol.  
- Paper includes helpful appendix and informative tables that clarify results and implications.  
- Suggests the paper should better discuss limitations of current dialog evaluation metrics and propose application-driven loss functions.  
- Recommends more insight into real-world applications of the testbed and match type features and more comparisons with state-of-the-art methods.  
- Asks authors for details on match type feature design/implementation, envisioned real-world use of the testbed, and ways to address limitations of current evaluation metrics.
