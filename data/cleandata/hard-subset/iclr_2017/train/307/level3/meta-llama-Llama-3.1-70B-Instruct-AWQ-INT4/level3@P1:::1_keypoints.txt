- Proposes a novel testbed and synthetic dataset for evaluating end-to-end goal-oriented dialog, focused on restaurant reservations.  
- Introduces tasks covering API calls, option display, and full dialogues to evaluate different dialog capabilities.  
- Evaluates multiple learning methods: rule-based systems, classical IR models, supervised embeddings, and end-to-end Memory Networks.  
- Main contribution is a publicly available dataset and tasks to facilitate future goal-oriented dialog research.  
- Shows end-to-end Memory Networks can outperform other baselines on the proposed tasks.  
- Tasks are deterministic, so they do not test handling of noisy or ambiguous user interactions.  
- Lacks a non-neural baseline that incorporates word order, which may make neural models appear unnaturally strong.  
- Recommends adding a non-neural benchmark (e.g., logistic regression with embeddings, bi-gram and match-type features).  
- Suggests introducing tasks that evaluate robustness to noise and ambiguity (e.g., inferring distributions over user goals, dialogue repair).  
- Claims are supported by a thorough evaluation, but generalizability is limited by deterministic tasks and missing baselines.  
- Decision: Accept with minor revisions due to useful contribution and addressable limitations.  
- Supporting reasons: public dataset is significant, evaluation is comprehensive, and limitations can be fixed with minor changes.  
- Additional feedback: provide Memory Network implementation details (architecture, hyperparameters) and more result analysis/error analysis.  
- Questions for authors: supply MemNet architecture/hyperparameters, plan to address missing non-neural baseline and determinism, and provide deeper result/error analysis.
