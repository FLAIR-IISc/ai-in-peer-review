- Paper proposes a reproducible, rule-based synthetic testbed and dataset for evaluating end-to-end goal-oriented dialog (restaurant reservation) covering API calls, option display, and full dialogues.
- Authors release a new public dataset and tasks and validate patterns on two real-user datasets, showing synthetic tasks are a reasonable proxy for real data.
- Strong motivation and clear problem definition for evaluating goal-oriented dialog systems.
- Extensive experiments compare rule-based, IR, supervised embeddings, and end-to-end Memory Networks across tasks.
- Memory Networks outperform classical IR and supervised embeddings on all tasks but still fail on key aspects.
- Concern: unclear whether deep learning models are genuinely suitable versus traditional/rule-based or shallow models, especially given task determinism.
- Concern: missing non-neural baselines that capture word order (e.g., logistic regression with embeddings, bi-grams, match-type features), making neural models appear overly strong.
- Concern: inaccurate claim that prior work lacks well-defined performance measures; related end-to-end and task-oriented evaluation work should be acknowledged.
- Writing feedback: overall clear but introduction could be more concise and related work should be more comprehensive.
- Dataset feedback: paper should include more dataset/task details (number of examples, vocabulary size, evaluation metrics).
- Results feedback: add more visualizations, summaries, and deeper analysis of strengths/weaknesses of each method.
- Questions for authors: provide dataset/task statistics, plans to introduce noisy/ambiguous interactions, deeper result analysis, and extensions to more complex real-world tasks.
