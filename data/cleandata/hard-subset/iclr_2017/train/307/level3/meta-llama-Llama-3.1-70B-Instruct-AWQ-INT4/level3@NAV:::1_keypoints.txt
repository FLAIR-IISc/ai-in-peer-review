- Paper introduces a novel, public, rule-based synthetic dataset and five tasks for goal-oriented dialogue (API calls, option display, full dialogues) focused on restaurant reservations.
- Decision: Accept — valuable, well-motivated testbed with sound approach and rigorous results.
- Addresses a gap in resources for training/testing end-to-end goal-oriented dialog models by providing a new dataset and task suite.
- Thoroughly analyzes multiple model types (rule-based, IR, supervised embeddings, Memory Networks), showing Memory Networks generally outperform others but have limitations.
- Demonstrates the testbed's usefulness by evaluating models on both synthetic and real-world datasets to reveal areas needing improvement.
- Suggestion: add a non-neural benchmark (e.g., logistic regression with embeddings, bi-gram, match-type features) for broader comparison.
- Suggestion: clarify the claim that "existing work has no well-defined measures of performance" with more context and references.
- Suggestion: provide more details on model hyperparameters and optimization procedures.
- Suggestion: include more example predictions from the Memory Network to illustrate strengths and weaknesses.
- Question: provide details on KB entity types used in match-type features and selection criteria.
- Question: describe hyperparameter optimization process and best-performing values for each model.
- Question: explain why Memory Networks struggled on tasks T3 and T5 and outline plans to address these limitations.
