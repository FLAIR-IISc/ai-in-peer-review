- Introduces a novel, publicly available rule-based synthetic dataset and tasks for goal-oriented dialogue focused on restaurant reservations.
- Provides a useful testbed to analyze strengths and shortcomings of end-to-end dialog systems in goal-oriented settings.
- Motivates studying whether end-to-end models can replace traditional hand-crafted dialog systems in narrow domains.
- Experiments use multiple methods: rule-based, classical IR, supervised embeddings, and end-to-end Memory Networks.
- Results show Memory Networks outperform IR and supervised embeddings across tasks but struggle with entity interpretation for result presentation.
- Paper is well-written with thorough, well-designed experiments and is a valuable community resource.
- Reviewer recommends acceptance primarily for the dataset and systematic evaluation contribution.
- Limitation: tasks 1–5 are deterministic and do not evaluate handling of noisy or ambiguous interactions.
- Limitation: lacks a non-neural baseline that incorporates word order, making neural models appear overly strong.
- Reviewer suggests adding a non-neural benchmark (e.g., logistic regression with embeddings, bi-grams, match-type features).
- Reviewer disputes the claim that prior work lacks well-defined performance measures, citing existing evaluations (Wen et al., Liu et al., Yu et al.).
- Requests more analysis and implementation details, especially on Memory Networks, match type feature usage, hyperparameters, and comparisons with/without match types.
- Asks how authors will address deterministic task limitations and how the testbed can be extended to more complex goal-oriented applications.
