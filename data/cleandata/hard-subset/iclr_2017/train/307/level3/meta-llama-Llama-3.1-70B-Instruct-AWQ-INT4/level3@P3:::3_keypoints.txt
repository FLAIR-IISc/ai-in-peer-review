- Introduces a new simulated restaurant-domain dataset for end-to-end goal-oriented dialog using a knowledge base and rule-based NLG.  
- Task defined as ranking candidate next-turn responses, evaluated by per-response and per-dialog accuracy.  
- Authors show Memory Networks outperform bag-of-words baselines and a slot-filling rule-based baseline on real DSTC2 dialogues.  
- Paper accepted because it provides an interesting contribution and useful benchmark for end-to-end goal-oriented dialog learning.  
- Reviewer initially skeptical about end-to-end methods in narrow domains where hand-engineered systems may do better.  
- Appreciates the detailed analysis, including match type features to mitigate exact-match and OOV problems.  
- Thorough evaluation with comparisons to multiple baselines and analysis across different tasks.  
- Suggests improving task definition by exploring generation tasks for greater flexibility and domain generalization.  
- Requests more discussion on practicality—specifically how candidate response lists would be obtained in new domains.  
- Asks for details on the rule-based NLG: the 43 user and 20 bot patterns and how they were created.  
- Asks how the method will address out-of-vocabulary words beyond match type features.  
- Requests clearer definitions and calculations for per-response and per-dialog accuracy metrics.  
- Recommends more related-work discussion comparing strengths/weaknesses of approaches in goal-oriented dialog.  
- Suggests adding visualizations (plots/tables) to better present performance across methods and tasks.  
- Requests more implementation details of the Memory Networks (architecture and hyperparameters).  
- Recommends discussing potential real-world applications and deployment considerations for the proposed method.
