- Introduces an open-source testbed with five synthetic restaurant-reservation tasks to evaluate end-to-end goal-oriented dialog systems.
- Breaks full dialogs into subtasks for detailed error analysis and provides an OOV test set to assess generalization to new entities.
- Validates the synthetic tasks by showing performance patterns correlate with two real-world datasets (DSTC2 and a concierge service).
- Provides comprehensive baselines (IR, supervised embeddings, Memory Networks) and shows current end-to-end models are promising but not fully solved.
- Decision: Weak Accept — leaning toward acceptance primarily for its dataset/benchmark contribution.
- Positive reason: the publicly available, well-designed benchmark fills a community need for reproducible, analytical evaluation.
- Negative reason: experimental evaluation lacks a strong non-neural baseline that incorporates word order (e.g., n-grams), weakening claims about Memory Network advantages.
- Key weakness: current baselines ignore word order, making Memory Network’s superiority unclear; a logistic regression with n-gram and match-type features is missing.
- Data concern: training sets are small (1,000 dialogs), favoring simpler regularized models over data-hungry neural architectures, so stronger baselines are critical.
- Additional feedback: tasks are deterministic and should acknowledge limitations versus real-world stochastic, noisy user behavior and possible extensions to model that.
- Additional feedback: authors overstate lack of prior evaluation work (e.g., PARADISE, Wen et al., Liu et al.); should soften claims and better situate contribution.
- Questions for authors: please add experiments with a logistic-regression baseline using TF-IDF, bigrams, and match-type features; and comment on extending the simulator to model more realistic, noisy user behaviors.
