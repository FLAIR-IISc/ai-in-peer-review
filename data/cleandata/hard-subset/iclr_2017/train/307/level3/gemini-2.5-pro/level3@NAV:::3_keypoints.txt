- Introduces a valuable simulated, open-source dataset/testbed for end-to-end goal-oriented dialogue in the restaurant reservation domain.  
- Uses a knowledge base and rule-based simulator to generate dialogues and tasks probing API calls, dialog state management, and KB usage.  
- Provides strong baselines showing Memory Networks outperform classical IR and supervised embedding models.  
- Evaluation includes per-response and per-dialog accuracy metrics, highlighting remaining challenges.  
- Recommends acceptance because the dataset is reproducible, well-structured, fills a resource gap, and enables standardized comparisons.  
- Experimental analysis is thorough and benchmarks a modern neural model, showing promising per-response accuracy but poor multi-turn goal completion.  
- Primary reservation: framing the task as ranking from a pre-defined candidate response set limits practical applicability and scalability to new domains.  
- Skeptical about the immediate value of end-to-end training for narrow domains where hand-engineered systems may still perform better, supported by low per-dialog accuracy.  
- Suggests publishing the natural language generation rules used by the simulator to increase transparency and assess linguistic diversity/biases.  
- Requests clarification on (1) how a candidate response set would be generated in a live new-domain system, and (2) more details on the complexity/variety of the 20 bot-side NLG rules.
