- Introduces an open-source benchmark of five simulated goal-oriented dialog tasks in the restaurant reservation domain.
- Tasks are designed to test specific capabilities: dialog state tracking, state updates, KB query interpretation, and information provision.
- Provides and evaluates multiple baselines (IR, supervised embeddings, Memory Networks) showing neural models are promising but not yet reliable.
- Memory Networks outperform simpler models but still fail to complete full dialogs at acceptable per-dialog accuracy.
- Includes an out-of-vocabulary (OOV) test set to measure generalization to unseen entities.
- Proposes simple "match type features" that improve handling of unseen entities.
- Validates simulation findings on real-world datasets (DSTC2 and a proprietary concierge dataset), showing consistent model behavior.
- Decision: Accept — primary contribution is a valuable, reproducible community resource and testbed for goal-oriented dialog.
- Experimental design is rigorous, with task decomposition enabling fine-grained error analysis.
- Main motivation: end-to-end systems need standardized testbeds to evaluate applicability to structured, goal-driven conversations.
- Limitation: framing the task as ranking from a fixed candidate set simplifies the problem and avoids true generative response modeling.
- Limitation: simulator uses limited NL patterns (43 user, 20 bot), risking models learning surface cues rather than deep semantics.
- Limitation: paper should discuss where end-to-end models fit versus meticulously engineered slot-filling systems in narrow, high-value domains.
- Questions for authors: justify ranking vs. generative evaluation, publish the NL-generation patterns for reproducibility, and specify which architectural/methodological improvements are most critical (memory, multi-turn reasoning, symbolic KB integration).
