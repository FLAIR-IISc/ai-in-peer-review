- New publicly available rule-based synthetic dataset and tasks for evaluating end-to-end goal-oriented dialogue systems, providing a controlled, reproducible testbed.  
- Tasks decompose dialogue complexity into capabilities (issuing API calls, displaying options, full conversations).  
- Recommendation: Weak Accept — dataset is valuable but experimental evaluation is insufficient.  
- Main weakness: lack of a sufficiently strong non-neural baseline undermines assessment of neural model performance.  
- Provided baselines (TF-IDF, supervised bag-of-embeddings) ignore word order.  
- On deterministic Tasks 1–5, a simple logistic regression with n-gram features and embeddings could likely match neural results.  
- Without stronger baselines, Memory Network gains may reflect captured sequential information rather than true reasoning.  
- Deterministic nature of tasks fails to evaluate robustness to ambiguity or noisy user input found in real-world dialogue.  
- Strong recommendation: include a competitive non-neural baseline (logistic regression with word embeddings, bi-gram and match-type features).  
- Minor correction: claim that prior work lacks well-defined performance measures is inaccurate; prior evaluation literature should be acknowledged.  
- Questions for authors: provide results for a stronger non-neural baseline and consider extending tasks to assess robustness to noise/ambiguity.
