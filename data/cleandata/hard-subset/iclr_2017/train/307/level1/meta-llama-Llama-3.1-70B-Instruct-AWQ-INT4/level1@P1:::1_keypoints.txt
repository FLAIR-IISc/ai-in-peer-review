- Proposes a lightweight testbed for evaluating end-to-end goal-oriented dialog systems in the restaurant reservation domain.  
- Designs five tasks to test dialog management, knowledge-base querying, and output interpretation.  
- Compares an end-to-end Memory Networks system to a hand-crafted slot-filling baseline on DSTC2 and an online concierge dataset.  
- Reviewer recommends acceptance.  
- Acceptance reason 1: addresses the lack of a systematic, controlled evaluation framework for end-to-end goal-oriented dialog learning.  
- Acceptance reason 2: provides a thorough evaluation of models (including Memory Networks) with useful insights into strengths and weaknesses.  
- Paper is well-motivated, clearly explained, and the testbed is accessible and easy to use.  
- Evaluation is comprehensive with detailed analysis of results.  
- Weakness: insufficient discussion of the testbed’s limitations and potential biases in evaluation metrics.  
- Weakness: limited insight into Memory Networks’ ability to generalize to new entities and tasks.  
- Suggested improvements: add details on testbed design, task generation, and evaluation metrics.  
- Suggested improvements: discuss limitations and metric biases, include more baseline models, and provide details on the online concierge dataset collection/annotation.  
- Questions for authors ask for more detail on testbed design/metrics, plans to address limitations/biases, Memory Network generalization, additional baselines, and dataset collection/annotation.
