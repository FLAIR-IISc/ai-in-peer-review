- Proposes a testbed for evaluating end-to-end goal-oriented dialogue systems in restaurant reservation tasks.  
- Designs five tasks requiring sentence/symbol manipulation, API calls, and use of API outputs.  
- Evaluates rule-based, IR, supervised embeddings, and end-to-end Memory Networks on those tasks.  
- Finds Memory Networks outperform baselines but struggle with entity understanding and result presentation.  
- Decision: accept with minor revisions.  
- Contribution: provides a significant, well-motivated testbed addressing limitations of traditional dialog systems.  
- Evaluation: thorough comparison across methods using per-response and per-dialog accuracy metrics.  
- Suggests more analysis of results to clarify strengths and weaknesses of each method.  
- Requests more implementation details for Memory Networks (hyperparameters, number of hops).  
- Requests additional example predictions from Memory Networks, especially for Tasks 3 and 5.  
- Asks how hyperparameters were selected and how varying hops affected performance.  
- Asks how the testbed could be extended to other goal-oriented applications and what challenges that entails.
