- Introduces an open-source testbed for evaluating end-to-end, goal-oriented dialog systems, providing a reality check for current neural approaches.  
- Designs five synthetic restaurant-reservation tasks that break down dialog into subproblems (API calls, user updates, KB option presentation).  
- Adapts two real-world datasets (DSTC2 and a proprietary concierge dataset) to validate the synthetic tasks' relevance.  
- Evaluates multiple models, including a Memory Network (MemNN) with a novel "match type" feature for OOV entity handling.  
- Finds MemNNs outperform IR and embedding baselines but still fail to reliably complete complex multi-turn dialogs.  
- Shows relative performance on synthetic tasks predicts model performance on real data, validating the testbed's usefulness.  
- Highlights the OOV test set as a strong feature addressing generalization to unseen entities.  
- Demonstrates the "match type" feature improves OOV performance but depends on exact string matches to KB entities.  
- Identifies multi-step reasoning and robust KB integration as major remaining challenges for end-to-end models.  
- Praises the experimental protocol and clear empirical analysis that yield actionable insights for future work.  
- Critiques reliance on per-response and per-dialog accuracy as potentially misleading and recommends user-centric metrics or interactive evaluation.  
- Questions robustness of entity matching when user vocabulary differs from KB terms or contains errors, and suggests learning alignments between user language and KB terminology.
