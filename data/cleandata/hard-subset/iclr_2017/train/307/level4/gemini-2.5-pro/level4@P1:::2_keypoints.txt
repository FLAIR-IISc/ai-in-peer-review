- Introduces a rigorous, bAbI-style benchmark/testbed for goal-oriented dialog (restaurant booking) that decomposes the task into key sub-skills.
- Core contributions are an open-source benchmark and a thorough experimental analysis.
- Experiments show current end-to-end approaches are insufficient for reliably conducting goal-oriented dialogs (low per-dialog success rates).
- Performance on the synthetic tasks predicts model performance on real-world datasets (DSTC2 and a proprietary concierge dataset).
- Paper is exceptionally well-written and accessible.
- Decision: Accept — addresses a timely problem with high-quality methodology and provides a valuable community resource.
- Benchmark is foundational and enables precise error analysis beyond single, monolithic metrics.
- Empirical comparisons across models (IR to Memory Networks) clearly identify where modern end-to-end systems succeed and fail.
- Novel algorithmic contribution is modest (the "match type" feature) but is outweighed by the benchmark’s value and insights.
- First systematic evaluation of Memory Networks on goal-oriented dialog; results provide illuminating analysis.
- The work effectively bridges synthetic and real-world conditions and includes clear explanatory appendices/tables (e.g., attention hops).
- Current dialog metrics are brittle: per-response accuracy can be misleading by penalizing necessary preparatory turns.
- Per-dialog accuracy is highly sensitive to task design (e.g., T3 vs T5 discrepancy due to differing protocol requirements).
- Suggestion to develop metrics better aligned with real goals (e.g., weighted loss combining API correctness, number of turns, and options rejected) using interactive evaluation.
- KB grounding remains a challenge: the match-type feature depends on exact string matches and fails with semantic/vocabulary mismatches (e.g., “British food” vs “English”); methods to learn semantic associations automatically are needed.
