- Paper provides a crucial reality check on chatbot capabilities, highly significant for dialog/NLP and ML communities.  
- Two key conjectures: (1) end-to-end ML still insufficient for goal-oriented dialogs, and (2) relative performance on synthetic data predicts performance on natural data.  
- Algorithmic novelty is modest (match type addition to memory networks) but paper is notable as the first thorough deployment/testing of this architecture on goal-oriented tasks.  
- Experimental protocol, clarity, and appendix on memory networks are excellent and accessible to a broad audience.  
- Current dialog metrics are brittle and sometimes arbitrary; per-response accuracy can be misleading because success may depend on multiple exact responses.  
- Per-dialog accuracy is sensitive to protocol details (illustrated by the T3 vs T5 discrepancy), so metric choice strongly affects results.  
- Question raised whether evaluation should consider the single best response (not top-5) for datasets like concierge.  
- Proposes more meaningful, utility-driven metrics (e.g., revenue-weighted loss) which require real users or realistic simulators for measurement.  
- Raises the problem of vocabulary mismatch between user input and knowledge base: match type needs exact matches, so representation learning should learn associations rather than rely on explicit annotation.
