- Paper provides a reality check on current chatbots in goal-oriented dialogs; significant for Dialog/NLP and ML venues.  
- Proposes a systematic testbed, open dataset, and task set for evaluating end-to-end goal-oriented dialog systems.  
- Evaluates systems in a controlled way that breaks down strengths and shortcomings.  
- Paper clarity is excellent and accessible beyond ML/dialog researchers.  
- Experimental protocol is well-designed and uses both synthetic and natural data.  
- Effort to match synthetic and natural conditions is notable; synthetic performance can predict natural performance.  
- Algorithmic contribution: adding match type features to memory networks — simple yet effective.  
- Provides thorough evaluation across models: rule-based, IR, supervised embeddings, and memory networks.  
- Shows end-to-end ML approaches still struggle with complex tasks requiring entity-knowledge interpretation.  
- Highlights the need for more realistic validation of ML applied to dialogs.  
- Raises concerns that per-response accuracy can be overly pessimistic.  
- Notes per-dialog accuracy is sensitive to experimental protocol.  
- Does not address representation learning when user vocabulary does not exactly match the knowledge base; match type requires exact matches.  
- Suggests discussing limitations of match type and ways to learn associations between words and entities.  
- Recommends exploring alternative evaluation metrics (e.g., weighted sum of errors and turns).  
- Decision: Accept — justified by significance, strong protocol, and clarity.
