- Introduces a valuable, reproducible testbed for evaluating end-to-end goal-oriented dialog systems, focused on restaurant reservations.
- Fills a critical gap by proposing a systematic framework to assess goal-directed conversation challenges.
- Decomposes the problem into five subtasks testing dialog management, KB querying, and handling new entities.
- Shows Memory Networks achieve promising results but struggle with interpreting and presenting API query results to users.
- Validates findings on real-world datasets (DSTC2 and an online concierge service), showing consistent patterns across synthetic and natural data.
- Recommends acceptance: the testbed is well-motivated and addresses a significant evaluation gap.
- Provides rigorous empirical evidence across multiple datasets and metrics that end-to-end systems still struggle on core goal-oriented dialog capabilities.
- Experimental design and evaluation are strong; synthetic testbed isolates capabilities and conversion of real data is methodologically rigorous.
- Memory Networks outperform classical IR and supervised embedding baselines across tasks.
- Match type features significantly improve handling of out-of-vocabulary entities.
- Conclusion that end-to-end models need further improvement is well-supported by the results.
- Paper is clear and accessible, with a good Memory Networks implementation explanation and well-structured task definitions.
- Suggests adding more discussion of practical implications and specific dialog failures and mitigation strategies.
- Asks whether evaluation could focus on minimum turns to complete a reservation rather than requiring all responses correct.
- Proposes extending match type features to handle semantic variations in entity names (e.g., "Indian" vs. "Indian food").
- Recommends adapting the testbed to evaluate more complex goal-oriented dialog scenarios beyond restaurant reservation.
- Encourages expanding the conclusion on the matching performance patterns between Concierge and the testbed, highlighting proxy potential.
