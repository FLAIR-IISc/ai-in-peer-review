- Introduces a diagnostic benchmark of five simulated restaurant reservation tasks plus evaluations on DSTC2 and a proprietary concierge dataset.  
- Addresses the lack of standardized benchmarks for evaluating end-to-end goal-oriented dialogue systems.  
- Breaks the reservation goal into subtasks (API calling, KB querying, information presentation, OOV handling) for fine-grained error analysis.  
- Demonstrates Memory Networks can learn some skills but struggle with full task completion.  
- Simulated tasks generalize to real-world data, making them a meaningful proxy for practical performance.  
- Decision: Accept with minor revisions.  
- Contribution is primarily a diagnostic dataset and benchmark rather than a novel model.  
- Well-motivated: clearly articulates limitations of current dialogue evaluation and slot-filling scaling issues.  
- Strong experimental setup: task design isolates sub-skills and uses both simulated and real-world comparisons and baselines.  
- Clear results and analysis: highlights gap between per-response and per-dialog accuracy and includes a useful ablation study.  
- Reproducibility prioritized: dataset and code are provided.  
- Suggest more detailed discussion of single-domain limitations and simulation biases for generalization.  
- Recommend expanding error analysis with concrete failure examples (e.g., wrong API calls, KB misinterpretations).  
- Suggest briefly summarizing Memory Network hyperparameter choices (e.g., hop number) in the main paper for readability.  
- Advise comparing and contrasting findings with recent advances (transformers, pre-trained LMs).  
- Questions for authors: consider varying task complexity (more constraints, ambiguity) to probe model limits.  
- Questions about KB scalability and how models would perform with much larger, complex KBs.  
- Request more details on the proprietary concierge dataset (size, user characteristics, request types).  
- Ask about future plans to expand the benchmark with new tasks, complexity, or additional knowledge sources.
