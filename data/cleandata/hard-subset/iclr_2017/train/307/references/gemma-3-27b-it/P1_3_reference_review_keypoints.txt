- Summary: Proposes five synthetic restaurant-domain tasks plus DSTC2 and concierge evaluations to probe goal-oriented dialog, and evaluates Memory Networks showing strengths and limitations.  
- Decision: Accept with minor revisions — well-motivated benchmark with comprehensive experiments.  
- Clear problem statement & motivation: Nicely articulates need for better evaluation of goal-oriented dialog and limitations of existing benchmarks.  
- Well-designed tasks: Tasks isolate dialog management, KB querying, and entity handling; OOV split probes generalization.  
- Comprehensive evaluation: Compares rule-based, IR, supervised embeddings, and Memory Networks on synthetic and real-world data.  
- Significant contribution: Public release of dataset/tasks and Memory Network baselines useful to the community.  
- Sound methodology: Rigorous experimental setup with hyperparameters reported and both per-response and per-dialog accuracy.  
- Discuss benchmark limitations: Recommend more discussion on domain representativeness and omitted complexities of other goal-oriented tasks.  
- Need deeper error analysis: Suggest qualitative examples and categorization of model failures for insight.  
- KB size & complexity: Ask for discussion on scaling tasks to larger, more realistic knowledge bases.  
- Hyperparameter tuning details: Request brief justification for chosen hyperparameters.  
- Memory Network variants: Recommend evaluating variants (e.g., key-value memory) and alternative end-to-end architectures.  
- Scalability question: How expensive/feasible is generating much larger datasets for these tasks?  
- Generalization to other domains: Can task definitions be adapted to domains like travel or movies, and what changes are needed?  
- Error categorization question: Request detailed breakdown of Memory Network error types, especially for KB interpretation.  
- Concierge data details: Ask for elaboration on converting concierge logs to task format and encountered challenges.  
- KB interaction design question: Did authors consider richer KB interactions (e.g., natural-language KB queries) and their effect on task complexity?  
- Conclusion: The benchmark and analyses are a significant, reproducible contribution likely to stimulate future research.
