- Introduces a new open-source diagnostic dataset for goal-oriented dialogue (restaurant reservation) with controlled sub-tasks and OOV testing.  
- Breaks the task into Tasks 1–5 (API calling, KB interpretation, state tracking, etc.) for granular evaluation.  
- Benchmarks end-to-end models (Memory Networks) and compares simulated vs. real-world (DSTC2, Concierge) performance.  
- Demonstrates a clear performance gap between end-to-end approaches and traditional handcrafted systems.  
- Decision: Accept with minor revisions; dataset and analysis are significant contributions despite no performance breakthrough.  
- Clear problem definition and strong motivation for a dedicated benchmark beyond noisy or limited datasets.  
- Well-designed tasks that improve validity and generalizability by including OOV and real human-machine dialogs.  
- Rigorous experiments using multiple baselines, systematic hyperparameter tuning, and ablation studies.  
- Results compellingly show difficulties in full-dialog completion and KB integration for current models.  
- Public release of dataset and code boosts reproducibility and community impact.  
- Recommend adding a task for error recovery to test correction and clarification handling.  
- Suggest additional metrics (booking success rate, turns-to-completion, human evaluations) for more comprehensive assessment.  
- Request including computational resources and training time details for Memory Network reproducibility.  
- Advise expanding error analysis with concrete failure examples and qualitative attention-weight inspection.  
- Questions for authors: plans to expand to other goal-oriented domains (e.g., travel, movies)?  
- Question scalability: can the simulation framework generate larger, more complex datasets?  
- Ask whether an explicit error-recovery task was considered or could be added.  
- Request attention visualizations from Memory Networks to interpret model focus during dialogs.  
- Seek details on Concierge dataset anonymization and privacy safeguards to prevent sensitive data leakage.
