- Presents an end-to-end optimized deep network for lossy image compression (analysis transform → quantizer → synthesis transform).
- Jointly optimizes rate-distortion using the classical R + λD objective for a principled learning-based compression framework.
- Introduces a differentiable proxy for quantization by replacing rounding with additive uniform noise during training to enable SGD.
- Uses a biologically-inspired architecture with cascaded convolutions and Generalized Divisive Normalization (GDN) nonlinearities.
- Demonstrates state-of-the-art results beating JPEG and JPEG 2000 on standard datasets, especially on perceptual MS-SSIM.
- Claims substantially better visual quality with fewer blocking/ringing artifacts and more visually pleasing, painterly simplification.
- Paper is well-written and connects deep learning with information-theoretic principles and VAE-related perspectives.
- Decision: Accept.
- Primary reason 1: Elegant integration of rate-distortion theory and a continuous relaxation enabling true end-to-end optimization.
- Primary reason 2: Superior empirical performance and visual quality, with fair rate comparisons using an actual entropy coder (CABAC).
- Supporting argument: Direct minimization of the rate-distortion Lagrangian avoids ad-hoc objectives used in prior work.
- Supporting argument: Additive uniform noise is a well-motivated, empirically validated approximation for quantization (Figure 4).
- Supporting argument: GDN is justified by prior density-modeling work and biological vision links, outperforming standard nonlinearities in practice.
- Supporting argument: Comprehensive evaluation using Kodak, real entropy coding, PSNR and MS-SSIM, and strong qualitative comparisons.
- Suggestion: Add comparison with modern classical codecs (e.g., BPG/HEVC) to situate performance against current state-of-the-art.
- Suggestion: Include an ablation study comparing GDN-based architecture to a Conv-ReLU baseline with similar parameter count.
- Suggestion: Report results on classic test images (Lena, Barbara, Mandrill) for broader comparability with prior literature.
- Question: How does the method compare quantitatively to BPG (HEVC-based) codecs?
- Question: Provide quantitative estimates on ReLU vs GDN performance or how much larger a ReLU network must be to match GDN.
