- Proposes an on-policy deep RL method with auxiliary intrinsic variables, essentially a special case of universal value function approximators, applied to 3D navigation.  
- Primary technical contribution is distilling existing ideas for 3D navigation, but the paper should state contributions more clearly in the abstract/introduction.  
- Lacks analysis of failure modes and when the model fails to generalize to changing goals.  
- Conceptual concern: as an on-policy method, catastrophic forgetting is likely if the agent does not repeatedly train on distant-past goals.  
- Needs broader empirical validation (e.g., Atari) to strengthen claims; using the ROM as intrinsic variables is suggested future work.  
- Empirically demonstrates clear advantages and provides useful experimental insights for future agent development.
