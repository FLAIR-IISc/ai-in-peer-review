This paper proposes a nonparametric neural network model that can automatically learn the model size during the training process. The core idea is to randomly add zero units and use a sparse regularizer to automatically discard irrelevant weights. The concept seems to be a random search method in an isolated space with the help of sparse regularization, which eliminates unnecessary units. This is an important problem, and this paper has yielded interesting results. My main comments are listed below:

What is the algorithm's additional computational complexity? Converting each fan-in weight into a parallel component and an orthogonal component, and converting it into radial-angular coordinates can take a lot of additional computational time. The authors may need to discuss the amount of additional operations compared to parametric neural networks. Furthermore, some running time tests need to be shown.

It has been observed that nonparametric networks provide smaller networks in convex datasets, so it is inferior to parametric networks. Is there any idea on this matter?
