**Summary of the Paper's Claims and Contributions**

This paper addresses the significant challenge of reinforcement learning in environments with sparse rewards and long horizons. It proposes a general, two-stage framework for hierarchical reinforcement learning. In the first stage, a diverse set of skills is learned in a pre-training environment using a simple, generic "proxy reward" that requires minimal domain knowledge. In the second stage, these pre-trained skills are used as temporally extended actions for a high-level "manager" policy, which is then trained on specific downstream tasks.

The core technical contributions are threefold. First, the work proposes using a single Stochastic Neural Network (SNN) to represent the entire repertoire of low-level skills. This allows for efficient learning of multiple skills simultaneously through weight sharing, significantly improving sample efficiency compared to training separate policies. Second, to ensure the learned skills are distinct and cover a wide range of behaviors, an information-theoretic regularizer is introduced. This regularizer adds a reward bonus proportional to the mutual information between the latent variable conditioning the skill and the resulting state, effectively encouraging predictability of the skill from the agent's trajectory. Third, the paper investigates architectural choices for integrating the latent skill variable with the state observation, demonstrating empirically that a bilinear integration is more effective than simple concatenation for learning a rich set of skills.

The framework is evaluated on a set of challenging continuous control benchmarks involving locomotion and navigation in maze and food-gathering environments. The experiments convincingly show that the proposed method can learn a wide span of interpretable skills, that these skills lead to substantially improved exploration, and that the resulting hierarchical agent can solve sparse-reward tasks where standard RL algorithms and simpler baselines fail.

**Decision and Key Reasons**

**Decision:** Accept

**Key Reasons:**
1.  The paper presents a novel and well-motivated framework for unsupervised skill discovery that effectively combines Stochastic Neural Networks with an information-theoretic objective. This approach provides a principled and sample-efficient method for learning a diverse skill set with minimal supervisory signal, which is a key problem in hierarchical reinforcement learning.
2.  The empirical evaluation is thorough and provides strong support for the claims. The experiments include clear ablation studies demonstrating the importance of each component (bilinear integration, MI bonus), compelling qualitative visualizations of the learned skills and exploration patterns, and strong quantitative results on challenging sparse-reward tasks.

**Supporting Arguments**

The problem of automatically discovering reusable skills is a cornerstone of advancing reinforcement learning towards more complex, long-horizon problems. This work makes a significant contribution by proposing a practical and general framework. The use of a single proxy reward in a pre-training phase is an elegant way to decouple skill discovery from specific downstream tasks, making the learned skills broadly applicable.

The combination of SNNs for multi-modal policy representation and a mutual information bonus for encouraging diversity is the primary strength of the proposed method. This is a well-grounded approach inspired by successes in generative modeling (e.g., InfoGAN) and is shown to be highly effective in the RL context. The ablation studies in Figure 4 clearly demonstrate that both the SNN architecture (specifically, bilinear integration) and the MI regularizer are crucial for learning a rich and non-overlapping set of skills.

Furthermore, the paper provides excellent empirical evidence that these well-differentiated skills translate directly into improved performance. The random exploration plots in Figure 5 are particularly striking, showing a dramatic increase in state-space coverage for the hierarchical agent compared to naive exploration. This enhanced exploration is shown in Figure 6 to be the key to solving the sparse-reward maze and gather tasks, where baseline methods that learn from scratch or use only the proxy reward make little to no progress. The rigor of the evaluation, which includes running multiple seeds for both pre-training and downstream training to avoid "cherry-picking" a good set of skills, adds significant confidence to the results.

**Additional Feedback for Improvement**

The paper is well-written and the experiments are strong. The following points are intended as constructive suggestions for further improving the work.

*   **Sensitivity of the MI Bonus Calculation:** The estimation of the posterior `p(Z|c)` for the mutual information bonus relies on discretizing the state space. It would be beneficial to include a brief discussion on the sensitivity of the method to the granularity of this discretization. For instance, how was the mesh density of 10 divisions/unit chosen, and how does performance change with this hyperparameter? This would provide valuable practical guidance for applying the method to new environments.
*   **Fixed Skill Duration:** The manager policy commits to a selected skill for a fixed number of timesteps, `T`. The appendix includes a good analysis of this hyperparameter, but a brief mention of this design choice and its potential limitations in the main text (perhaps in the discussion section) would be helpful. Acknowledging that learning a termination condition is a possible extension would further contextualize the current approach within the broader options framework.
*   **Stability on Complex Agents:** The appendix transparently discusses failure modes for the unstable "Ant" agent, where switching skills can lead to falls. This is an important practical finding. Incorporating a sentence about this challenge into the main discussion of limitations would strengthen the paper by clearly delineating the conditions under which the current framework is most effective.

**Clarifying Questions**

*   The results, particularly in Appendix C.4, suggest that the final performance on a specific task can be highly dependent on the set of skills learned during pre-training. For example, solving Maze 0 appears to require a good "backward motion" skill. Does this imply that the proxy reward needs to be implicitly tailored to the kinds of skills that might be useful later, or is the expectation that a sufficiently general proxy reward (like CoM speed) combined with the MI bonus will eventually produce a universally useful set of skills?
*   The posterior for the MI bonus is estimated using only the trajectories from the current batch. This means the bonus reward function is re-estimated at each iteration. Was any instability observed due to this? Were alternative estimation strategies, such as using a replay buffer of recent trajectories to smooth the visitation counts, considered?
