- Contribution: Introduces recursion into the Neural Programmer-Interpreter to achieve provable perfect generalization for learned programs from very small training sets.  
- Main insight: Recursion enables verification via base cases and reduction rules, solving poor generalization in neural program learning.  
- Empirical strength: Recursive models achieve perfect generalization across four tasks (addition, bubble sort, topological sort, quicksort) where non-recursive models fail.  
- Notable result: Recursive topological sort attains 100% accuracy on graphs of 5–70 vertices after training on only six traces, while non-recursive fails beyond size 8.  
- Writing and presentation: Paper is well-written, uses consistent prior notation, and shows a small architectural change with large practical impact.  
- Novelty: Claims first provable guarantees of perfect generalization for neural programs, advancing over prior empirical-only approaches.  
- Decision: Recommend acceptance with minor revisions.  
- Revision request 1: Provide more concrete detail on the verification procedure—how base cases and reduction rules are determined and why the verification set suffices.  
- Revision request 2: Expand discussion of limitations for continuous input spaces (e.g., 3D inputs) and possible future directions to address them.  
- Revision request 3: Clarify how recursive training traces differ from non-recursive ones and what additional information they provide.  
- Reproducibility: Strongly encourage releasing source code to increase impact.  
- Clarification questions: Explain construction and completeness guarantees for the verification set V; justify the small bubble-sort verification set relative to observed generalization failures; and state if/when code will be released.
