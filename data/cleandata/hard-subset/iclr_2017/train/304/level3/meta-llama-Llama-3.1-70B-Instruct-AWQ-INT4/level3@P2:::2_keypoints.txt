- Proposes adding recursion to Neural Programming Interpreters (NPI) to improve generalization from fewer execution traces.  
- Demonstrates improved generalizability and interpretability on grade-school addition, bubble sort, topological sort, and quicksort.  
- Introduces a verification procedure intended to prove perfect generalization of the learned recursive programs.  
- Recommendation: accept, citing novelty of recursion in NPI and strong empirical evaluation.  
- Well-motivated and clearly explains limitations of prior neural programming approaches and how recursion addresses them.  
- The recursion extension is simple, effective, and the paper is well-written and accessible.  
- Key contribution: enables learning correct programs from fewer traces, increasing practicality of neural programming.  
- Verification procedure strengthens reliability and trustworthiness of results.  
- Critique: generalization proofs are somewhat vague and their applicability to more complex or continuous input spaces is unclear.  
- Request clarification on applying the approach to high-dimensional continuous tasks (e.g., 3D car racing).  
- Suggestion: include more implementation details on how recursive calls and the call stack are implemented and managed.  
- Suggestion: release source code to enable replication and further research.  
- Questions for authors: how to apply the method to complex/continuous tasks; how to handle high-dimensional continuous inputs; detailed implementation of recursion/stack; will the code be released?
