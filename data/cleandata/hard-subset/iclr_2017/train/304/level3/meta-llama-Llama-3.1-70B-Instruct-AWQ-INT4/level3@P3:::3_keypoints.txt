- Proposes incorporating recursion into neural architectures to improve generalizability and interpretability.
- Applies recursive approach to the NPI framework on four tasks: grade-school addition, bubble sort, topological sort, and quicksort.
- Reports learned recursive programs achieve perfect generalization from few examples and outperform prior results.
- Main contribution: introduction of recursion enabling provably perfect generalization, with a formal proof claimed as a first in neural program induction.
- Approach is well-motivated: recursion reduces problem complexity and makes program behavior easier to reason about.
- Experiments support claims: recursive models generalize perfectly while non-recursive counterparts do not.
- Reviewer accepts the paper, praising results and the significance of the formal proof.
- Requests more discussion of limitations and future work, e.g., verification procedure inadequacy for very large input domains.
- Requests more implementation details on how the NPI training was modified to learn recursive programs.
- Asks three specific questions: provide implementation details, plans to address verification limits for large domains, and ways to extend to more complex tasks (e.g., MNIST, speech).
- Additional feedback: include more related-work discussion and clarify differences from existing neural program induction methods.
- Additional feedback: provide more details on program sets and trace-generating functions for each task.
- Additional feedback: add more visualizations (e.g., accuracy vs input length) and discuss real-world applications like programming by demonstration or few-shot learning.
