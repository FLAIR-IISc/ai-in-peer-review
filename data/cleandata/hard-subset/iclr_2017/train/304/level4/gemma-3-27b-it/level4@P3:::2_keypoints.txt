- Proposes incorporating recursion into the NPI framework to improve neural program synthesis generalization.  
- Identifies existing methods’ poor generalization beyond training complexity due to spurious correlations.  
- Argues recursion breaks problems into smaller subproblems, reducing each network’s domain and improving semantic learning.  
- Empirical evaluation shows strong generalization on addition, bubble sort, topological sort, and quicksort using very small training sets.  
- Reports instances of 100% test accuracy after limited training, demonstrating broad applicability.  
- Introduces an initial verification procedure toward provable generalization, a notable departure from purely empirical evaluation.  
- Acknowledges the verification approach is currently limited to discrete cases but is a promising first step.  
- Maintains notation consistency with the original NPI paper, aiding clarity.  
- Decision: Accept with minor revisions.  
- Suggests elaborating limitations and potential extensions for continuous input spaces (e.g., formal methods or sampling-based verification).  
- Requests more detailed description and a concrete example of the verification procedure (e.g., identifying base cases and reduction rules for addition).  
- Recommends providing implementation hyperparameters (learning rate, batch size, LSTM size) to improve reproducibility.  
- Asks whether recursion can be learned without explicit recursive traces (e.g., via reward shaping or curriculum learning).  
- Asks about sensitivity of verification to incomplete sets of valid step-input sequences V and its impact on guarantees.  
- Proposes exploring abstract interpretation or compositional verification to handle large/infinite state spaces.  
- Requests confirmation on releasing source code to facilitate reproducibility.
