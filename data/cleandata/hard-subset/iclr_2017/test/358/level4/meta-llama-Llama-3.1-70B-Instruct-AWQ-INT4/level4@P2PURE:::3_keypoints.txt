- Introduces a VAE variant that models dataset-level latent variables; idea is clearly motivated and well described.  
- Main contribution is extending the standard VAE graphical model to richer dataset-level structure.  
- Terminology issue: the term "statistician" (and "statistic network") is unclear and unnecessary; prefer describing it as an "approximate posterior" or similar.  
- Experiments are well-conducted and the authors’ response on one-shot generation is appreciated.  
- One-shot generation description needs clearer wording (end of page 6); Fig. 5 workflow should be explicitly stated: input set → approximate posterior over context c → generate samples from the forward model using samples from that posterior.  
- Clarification requested: are the data-point latent vectors z sampled from the forward model or taken from the approximate posterior?  
- Recommend quantitative evaluation for one-shot generation (e.g., report log p(x|c) with c sampled from the approximate posterior or log p(x) on held-out data) and comparison to a vanilla VAE.
