- Proposes a VAE variant with dataset-level latent variables to capture shared structure across data points.  
- Core idea is well-motivated, clearly described, and a valuable contribution to hierarchical VAE models.  
- Strength: extends the standard VAE graphical model to a more complex, relevant hierarchical structure.  
- Presentation clarity suffers from unconventional terminology (e.g., "statistician", "statistic network").  
- The "statistic network" is essentially an inference network for the approximate posterior over the global context vector c.  
- Authors should situate this component within standard VAE/inference-network terminology and note alternative summarization approaches.  
- Experimental results are promising, and the clarification of the one-shot generation task is helpful.  
- Need explicit description of generation procedure: are per-instance latents z sampled from q(z|x,c) or from the prior p(z)?  
- Current sample evaluation is qualitative; quantitative evaluation using log-probabilities is needed.  
- Recommend reporting log p(x|c) on held-out data (or marginal log p(x)) with c inferred from a context set, and comparing to a standard VAE.
