- Introduces Rectified Factor Networks (RFNs) for biclustering as a scalable extension of FABIA enabling many code units via rectified posteriors and fast GPU computation.
- Demonstrates RFNs on synthetic and real datasets (gene expression, 1000 Genomes) showing improved biclustering performance.
- Decision: Accept — addresses an important biclustering problem and is well-motivated with thorough experiments.
- Clearly identifies FABIA’s computational limits and explains RFN model, advantages, and biclustering application in detail.
- Experimental results and comparisons suggest RFNs outperform other biclustering methods and could become state-of-the-art.
- Suggests clarifying terminology around “one shot generation” and data-dependent z vectors.
- Recommends quantifying one-shot generation performance by reporting log p(x|c) or log p(x) on held-out data.
- Requests more detail on RFN computational complexity and scalability to larger datasets.
- Asks for clarification on the rectification process and how it differs from other rectification methods.
- Asks how the optimal number of code units is chosen and how varying this number affects performance.
- Requests more insight into the RFN application to the 1000 Genomes data and implications for human evolution.
