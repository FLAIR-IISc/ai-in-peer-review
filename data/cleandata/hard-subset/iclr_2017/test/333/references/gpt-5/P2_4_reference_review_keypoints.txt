- Paper asks what minimal ingredients are needed for high-quality natural texture synthesis and claims single-layer random conv filters + Gram matrices + ReLU often rival prior work (Gatys et al.).
- Authors establish a minimal baseline: single-layer CNNs (stride 1, ReLU, zero-padding) with various filter banks (random, Fourier, k-means, PCA), mainly 11×11 filters and a multi-scale variant up to 55×55.
- Synthesis performed by optimizing images to match feature Gram matrices using L-BFGS-B with practical tweaks (loss/gradient scaling, box constraints).
- Propose using VGG-based multi-layer Gram loss as a quantitative proxy for perceptual similarity and present qualitative results on ~20 textures.
- Analyze roles of number of feature maps, filter scale, and nonlinearity; discuss MaxEnt perspective and trade-off between perceptual similarity and variability.
- Central finding (random features + Gram suffice) is an interesting, minimal and scientifically useful baseline.
- Decision: Reject (borderline).
- Main rejection reason: empirical claims that the method “rivals state-of-the-art” are not rigorously substantiated—evaluation relies mostly on visuals and a single imperfect proxy metric.
- No human perceptual study or robust complementary metrics (e.g., LPIPS) are provided to validate perceptual quality.
- Important experimental details and quantitative summaries are missing, limiting reproducibility and confidence (e.g., exact multi-scale architecture, dataset selection, variability assessment).
- Methodological positives: useful ablations on feature counts, filter types, and nonlinearity; clear optimization description.
- Methodological concerns: reliance on the same Gram/VGG loss family for evaluation and synthesis, noted paradoxes where metric favors the proposed method, and lack of statistical analysis across a standardized texture benchmark.
- Missing specifics: multi-scale composition (filter sizes/counts/initialization), padding/ boundary handling, texture set selection, and code release.
- Claims exceed evidence: statements like “often rivals state of the art” need human studies or stronger aggregated metrics and careful optimization parity with baselines.
- Suggested evaluation improvements: small-scale human 2AFC study, aggregate quantitative results across larger texture sets, multiple metrics (VGG-loss, LPIPS, SSIM) and report means/variances.
- Suggested baselines: ensure optimization parity with VGG baseline and compare to scattering-based fixed multiscale filter methods.
- Suggested method/analysis details: fully specify multi-scale architecture, examine other nonlinearities (leaky-ReLU, abs, tanh), quantify diversity (pairwise distances/entropy proxies), and provide runtime/memory comparisons.
- Reproducibility suggestion: release code and scripts to reproduce figures and tables.
- Specific questions for authors: exact multi-scale model composition, texture set size/selection, whether a user study was attempted, comparison to scattering-based synthesis, quantified sample diversity, diagnosis of the VGG-loss paradox, and effect of padding choices.
- Overall: core insight is promising but requires stronger, more rigorous evaluation and clearer methodological specification to be a convincing contribution.
