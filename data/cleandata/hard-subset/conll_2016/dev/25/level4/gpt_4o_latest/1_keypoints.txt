- Paper introduces a method to annotate WordNet word senses with temporal labels (past, present, future, atemporal) using a graph-based semi-supervised classifier leveraging gloss features, WordNet structure, and unlabeled data.  
- Task decomposed into two stages: binary temporal vs. atemporal, then fine-grained past/present/future for temporal synsets.  
- Authors annotate entire WordNet, train on a prior labeled dataset, and crowdsourced intrinsic evaluation on a subset of synsets.  
- System outperforms baselines: ~11% accuracy improvement over prior systems using only 400 labeled examples, and ~10% F1 improvement on TempEval-3 across four labels.  
- Paper is well-organized, clear, and presents a valuable fine-grained temporal resource with potential NLP applications.  
- Major concern: extrinsic evaluation description lacks detail—unclear what entity pairs represent, what entity attributes/features are, how POS/lemmas are handled, and there are no illustrative examples.  
- Labels and relations in extrinsic evaluation are ambiguous—phrases like “event to document creation time” need clarification, and the footnote about 14 relations is unexplained (why exclude others, what makes mappings “too complex”).  
- Unclear whether reported scores are macro- or micro-averaged.  
- Ablation study suggests redundancy between Lexica and Entity features—authors should explain the similar performance and potential overlap.  
- SVM usage needs clarification: which parameters were optimized for extrinsic evaluation, was the SVM inside MinCut optimized, and which SVM implementation was used (LibSVM citation missing if applicable).  
- Request for class distributions: provide number of examples per fine-grained label (only coarse counts given).  
- Suggest reporting estimate of temporally ambiguous words (e.g., senses labeled “present”).  
- Claim of “significantly better” in Table 3 requires statistical significance test details or p-values.  
- Related work clarifications needed: specify the task in Filannino and Nenadic (2014); add reference for “post-calibration procedure” and clarify calibration explanation; specify how other models differ from the authors’.  
- Presentation and formatting suggestions: enlarge Table 3, move metric labels to caption, possibly report fewer metrics; convert Table 4 to a graph; order scores in Table 6 in one column.  
- Typos and minor corrections: correct “1064” to “1264” on p.7 and change “atemporal)” to “atemporal.”  
- TempEval-3 mention needs a proper reference.
