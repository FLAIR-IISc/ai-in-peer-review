- Proposes annotating word senses with temporal labels (past, present, future, atemporal) using a graph-based semi-supervised classifier.
- Integrates item-specific signals (temporal indicators in glosses) with WordNet synset structure.
- Leverages unlabeled WordNet data plus a labeled training set to annotate the rest of WordNet.
- Frames the task as a binary temporal vs. atemporal classification followed by a finer past/present/future split.
- Intrinsic evaluation uses crowdsourced annotations of a WordNet subset.
- Achieves ~11% accuracy improvement over SUTime and prior work, and outperforms previous systems using only 400 labeled examples.
- Extrinsic TempEval-3 evaluation shows a ~10% F1 improvement across four labels.
- Paper is well-structured, clear, and presents a significant resource-level contribution for temporal word-sense information.
- Extrinsic evaluation lacks illustrative examples clarifying the prediction target (entity pairs, features, attributes, POS, lemma acquisition).
- Labels such as “event to document creation time” and “event to same sentence event” are ambiguous; unclear if they denote pairs or relations.
- Footnote about using 14 relations needs justification for excluding others and criteria for a “complex” mapping.
- Must specify whether reported scores are macro- or micro-averaged.
- Ablation study hints at redundancy between Lexica and Entity features and needs further investigation.
- Specify SVM parameters optimized for extrinsic evaluation and clarify if/how the SVM in the MinCut framework was optimized.
- If LibSVM is used, include a proper citation.
- Provide counts per label in the gold data for fine-grained labels and estimate the number of temporally ambiguous words.
- Support the “significantly better” claim in Table 3 with a statistical significance test and p-value.
- Clarify the task in Filannino and Nenadic (2014), add a reference for the post-calibration procedure, and explain how the proposed model differs from others.
- Formatting suggestions: simplify Table 3 (remove parentheses, indicate (p,r,f1) in caption, present only two scores), condense captions.
- Recommend presenting Table 4 graphically for clarity.
- Several typographical and reference corrections needed (TempEval-3 reference, ordering of scores in Table 6, and other minor edits).
