- Proposes combining cognitive features from annotators’ eye-movements with traditional sentiment features for improved sentiment analysis.
- Argues cognitive features help capture nuanced lexical, syntactic, semantic, and pragmatic subtleties in text.
- Demonstrates improvements in polarity detection of up to 3.7% and 9.3% on two public datasets.
- Main contribution: a novel feature set that fuses traditional sentiment features with eye-tracking–derived cognitive features.
- Main contribution: defines specific cognitive features extracted from eye-movement patterns to inform sentiment models.
- Main contribution: empirical evidence showing the cognitive+traditional feature set boosts polarity classification performance.
- Strength: introduces an innovative, cognitively grounded approach to sentiment analysis.
- Strength: presents an effective feature set validated on two public datasets.
- Strength: uses a robust evaluation methodology, including comparisons to traditional methods and feature importance analysis.
- Weakness: limited generalizability since the method requires eye-tracking data that is not widely available.
- Weakness: feature extraction is complex and may require specialized equipment and expertise.
- Weakness: limited, example-based analysis of cognitive features; deeper investigation of their role is needed.
