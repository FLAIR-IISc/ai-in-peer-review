- Proposes using cognitive features from annotators' eye-movement patterns for sentiment analysis and sarcasm detection, claiming novelty in NLP.
- Introduces a new feature set derived from eye movements as the main methodological contribution.
- Evaluates the effectiveness of cognitive features in improving sentiment and sarcasm classification accuracy.
- Explores the feasibility of applying cognitive features in real-world applications.
- Paper is well-written and accessible.
- Using cognitive features for these tasks is an innovative idea with potential to inspire broader NLP research.
- Provides a thorough evaluation including comparisons to traditional linguistic features and feasibility analysis.
- Motivation for why cognitive features help detect sentiment polarities is not fully justified.
- Accuracy improvements from adding cognitive features are marginal, questioning practical significance.
- Feasibility claims for real-world deployment (Section 7.2 example) are not entirely convincing.
- Requests a more detailed explanation of why cognitive features help detect sentiment polarities.
- Asks how authors will address marginal accuracy gains and for concrete examples of real-world integration with linguistic features.
