- Introduces a novel set of cognitive features derived from eye-tracking data for sentiment analysis.  
- Claims cognitive features notably improve handling of complex sentiment cases (sarcasm/irony).  
- Presents chi-squared feature-significance analysis showing gaze features among top contributors.  
- Novel approach that departs from traditional linguistic features and shows meaningful contribution of gaze data.  
- Strong empirical validation on complex sentiment with a 6.1% F-score gain for ironic texts.  
- Comprehensive feature engineering combining traditional, sarcasm-specific, and cognitive features with detailed analysis.  
- Lacks sufficient methodological detail on how cognitive features are derived from raw eye-tracking data (fixation detection, segmentation, duration calculations).  
- Omits comparison with recent deep-learning state-of-the-art models (e.g., BERT, RoBERTa), weakening claims of outperforming SOTA.  
- Underestimates practical deployment challenges—user consent and feasibility of eye-tracking are not evidenced.  
- Potential overfitting concerns due to small datasets, many features, PCA use, and few annotators.  
- Request detailed procedure for computing cognitive features from raw gaze data (how fixations and first fixations are identified).  
- Ask how overfitting was mitigated given dataset sizes and feature dimensionality (regularization, validation beyond 10-fold CV).  
- Ask why deep-learning baselines were not included and how cognitive features could integrate with neural models.  
- Ask for evidence that users would consent to eye-tracking in real-world applications (user studies or surveys).  
- Overall: promising and interesting but lacking rigor; recommend rejection with invitation to revise addressing methodological, comparative, and practical concerns.
