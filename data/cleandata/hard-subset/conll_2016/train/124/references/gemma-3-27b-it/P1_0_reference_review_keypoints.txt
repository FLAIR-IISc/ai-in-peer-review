- Paper investigates using eye-tracking-derived cognitive features to improve sentiment analysis, especially for irony/sarcasm, showing improved accuracy over baselines.
- Contribution: Novel integration of eye-tracking cognitive features into a sentiment analysis pipeline beyond basic fixation stats.
- Contribution: Empirical validation on two datasets rich in sarcasm and irony, providing a challenging evaluation.
- Contribution: Demonstrated statistically significant accuracy improvements when adding cognitive features, especially on complex text.
- Strength: Strong motivation and clear problem definition linking linguistic complexity to the need for cognitive signals.
- Strength: Comprehensive feature set combining textual, sarcasm/irony, and diverse cognitive (eye-tracking) features.
- Strength: Rigorous experimental evaluation using 10-fold cross-validation, McNemar tests, and a held-out complex-dataset.
- Strength: Feasibility discussion addressing mobile eye-tracking availability and real-world application prospects.
- Strength: Detailed feature descriptions that support transparency and reproducibility.
- Weakness: Cognitive features may not generalize due to dependence on eye-tracker hardware, sampling rate, and annotation protocol differences.
- Weakness: Feature complexity reduces interpretability; unclear which cognitive features drive gains and why.
- Weakness: Comparison limited to simpler classifiers (NB, SVM); lack of deep learning baselines leaves open how features integrate with modern models.
- Weakness: Small dataset sizes (994 and 1059) risk overfitting and limit statistical power.
- Question: How robust are cognitive features to variations in eye-tracking hardware and annotation protocols; can feature stability across setups be shown?
- Question: Can authors provide deeper analysis linking top cognitive features to specific cognitive processes (e.g., regressions → reanalysis)?
- Question: Have the authors considered incorporating cognitive features into deep learning models; what challenges/benefits do they anticipate?
- Question: Did individual annotator differences in eye movements affect model performance; was this variability analyzed?
- Question: Can the authors provide detailed error analysis with examples showing remaining system failures and their relation to feature limitations?
- Additional: Paper is generally well-written and organized.
- Additional: References are comprehensive and relevant.
- Additional: Formatting needs minor improvements (e.g., consistent table presentation).
- Overall: Promising and novel contribution with demonstrated gains; recommend acceptance if authors address generalizability, interpretability, deep-learning comparisons, and dataset/analysis concerns.
