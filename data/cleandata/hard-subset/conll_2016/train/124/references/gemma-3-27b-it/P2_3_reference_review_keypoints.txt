- Investigates using eye-tracking–derived cognitive features to improve sentiment analysis, showing accuracy gains on two datasets focused on sarcasm/irony.

- Novel application: demonstrates that cognitive (eye-tracking) features can provide useful signal for sentiment classification.

- Empirical validation: experiments on challenging sarcasm/irony datasets show the approach is meaningful beyond simple sentiment cases.

- Detailed feature engineering: presents comprehensive traditional and cognitive features, including a novel gaze-saliency graph.

- Strong motivation: clearly frames limitations of traditional sentiment methods for linguistic nuance and motivates cognitive features.

- Solid experimental setup: uses cross-validation, baselines, and statistical significance testing.

- Demonstrated improvement: consistent F-score gains, especially on complex instances and in held-out experiments.

- Feature analysis: provides significance analysis indicating cognitive features contribute, though depth is limited.

- Feasibility discussion: addresses practical concerns about eye-tracking data availability and future implementation scenarios.

- Limited novelty of core methods: cognitive features largely use established eye-tracking metrics; saliency graph’s impact is modest.

- Data limitations: datasets are relatively small with moderate inter-annotator agreement, potentially introducing noise.

- Missing ablation: lacks detailed ablation study to identify which gaze features are most impactful or redundant.

- Limited SOTA comparison: does not compare against deep learning sentiment models, reducing contextual benchmarking.

- Superficial error analysis: error analysis is brief and offers little insight into failure modes.

- Questions for authors: request ablation studies for gaze features, elaboration on annotator disagreements and eye-tracking patterns, intuitive interpretation of saliency graph features, and rationale/challenges for not comparing to deep learning approaches.

- Additional minor points: writing is generally clear but dense in places, references are comprehensive, and figures need clearer visuals.

- Overall: solid, interesting contribution with promising results; recommend acceptance pending satisfactory responses to the raised questions.
