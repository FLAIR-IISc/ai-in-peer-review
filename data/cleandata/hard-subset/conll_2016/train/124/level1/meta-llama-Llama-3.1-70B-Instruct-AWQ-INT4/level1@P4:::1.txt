**Summary of the Paper**

The paper proposes a novel approach to sentiment analysis by incorporating cognitive features derived from eye-movement patterns of human annotators. The authors argue that traditional sentiment analysis methods struggle to capture nuances at lexical, syntactic, semantic, and pragmatic levels, and that cognitive features can help address these challenges. The paper presents a comprehensive feature set that combines traditional sentiment features with cognitive features and evaluates its effectiveness on two publicly available datasets.

**Main Contributions**

1. **Novel Feature Set**: The paper introduces a new feature set that combines traditional sentiment features with cognitive features derived from eye-movement patterns. This feature set is shown to improve the performance of sentiment analysis on two datasets.
2. **Cognitive Features**: The paper proposes a set of cognitive features that capture the eye-movement patterns of human annotators, including fixation duration, saccade length, and regression count. These features are shown to be effective in capturing linguistic nuances that traditional features miss.
3. **Evaluation on Challenging Datasets**: The paper evaluates the proposed feature set on two datasets that contain complex constructs like irony and sarcasm, which are challenging for traditional sentiment analysis methods.

**Strengths**

1. **Innovative Approach**: The paper proposes a novel approach to sentiment analysis that incorporates cognitive features, which is a significant departure from traditional methods.
2. **Comprehensive Feature Set**: The paper presents a comprehensive feature set that combines traditional sentiment features with cognitive features, which is shown to be effective in improving the performance of sentiment analysis.
3. **Evaluation on Challenging Datasets**: The paper evaluates the proposed feature set on two datasets that contain complex constructs like irony and sarcasm, which is a challenging task for sentiment analysis methods.

**Weaknesses**

1. **Limited Generalizability**: The paper evaluates the proposed feature set on only two datasets, which may not be representative of all possible sentiment analysis tasks.
2. **Dependence on Eye-Tracking Data**: The paper relies on eye-tracking data, which may not be readily available for all sentiment analysis tasks.
3. **Feature Engineering**: The paper proposes a set of cognitive features that require significant feature engineering, which may be time-consuming and require expertise.

**Questions to Authors**

1. How do you plan to address the issue of limited generalizability of the proposed feature set to other sentiment analysis tasks?
2. Can you provide more details on how the eye-tracking data was collected and preprocessed?
3. How do you plan to make the proposed feature set more accessible to practitioners who may not have access to eye-tracking data or expertise in feature engineering?

**Additional Comments**

The paper presents a novel approach to sentiment analysis that incorporates cognitive features, which is a significant contribution to the field. However, the paper could benefit from more detailed analysis of the results, including error analysis and discussion of the limitations of the proposed feature set. Additionally, the paper could provide more details on the practical applications of the proposed approach and how it can be used in real-world sentiment analysis tasks.
