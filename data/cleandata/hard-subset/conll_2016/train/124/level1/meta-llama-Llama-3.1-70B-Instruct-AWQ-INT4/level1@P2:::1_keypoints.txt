- Proposes using cognitive features from annotators' eye movements to improve sentiment analysis, especially for nuanced language.
- Evaluates the approach on two public datasets using several machine learning classifiers.
- Introduces a novel set of eye-movement–derived features for sentiment polarity detection.
- Demonstrates that adding cognitive features improves accuracy, notably on irony and sarcasm.
- Novel approach that brings cognitive signals into sentiment analysis.
- Provides a fairly comprehensive evaluation across two datasets and multiple classifiers.
- Empirical improvements in accuracy for complex linguistic constructs.
- Limited to only two datasets, which may not generalize across text types.
- Experiments use only three classifiers, limiting evidence across model families.
- Lacks comparison against existing state-of-the-art sentiment analysis methods.
- Asks how cognitive features compare to linguistic or stylistic features.
- Asks whether the method generalizes to longer texts or other domains.
- Asks how eye-movement features relate to other cognitive processes like attention or memory.
- Results are promising but limitations need addressing; paper could use more discussion of implications and applications.
