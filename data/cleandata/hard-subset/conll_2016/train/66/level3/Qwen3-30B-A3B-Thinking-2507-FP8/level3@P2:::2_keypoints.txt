- Presents a transition-based joint syntactic and semantic dependency parser using stack LSTMs to encode entire parser state and a greedy linear-time inference algorithm.  
- Reviewer views the primary contribution as demonstrating that learned neural representations can replace expensive hand-crafted features for semantic role labeling; greedy inference is a secondary practical benefit.  
- Novel application: effectively uses stack LSTMs to capture long-range dependencies and replace predicate-argument path features, simplifying the pipeline while maintaining performance.  
- Practical performance: achieves fast end-to-end runtime (177.6±18s on CoNLL 2009 English single core), ~2.5× faster than comparable pipeline systems.  
- Clear presentation and technical soundness: well-motivated architecture, thorough implementation details, and evaluation on multiple datasets.  
- Missing beam search results: claims of linear time apply only to greedy decoding and no beam-search experiments are reported, so tradeoffs between speed and accuracy are unclear.  
- Limited comparison to recent SOTA: competes with older systems but lags behind more recent approaches using extra features, pipelines, or reranking, and integration with those techniques is not explored.  
- Lack of ablation studies: no experiments isolating the impact of stack LSTM representations versus greedy inference or other components.  
- Question: How do you reconcile claiming to outperform all previous joint parsing models when Table 3 shows being behind Zhao et al. (2009)?  
- Question: Did you run beam-search experiments, and if so what were the results; if not, why were they omitted?  
- Question: Why concatenate word and predicate sense vectors (Eq.5) rather than using interactions like element-wise multiplication or learned transformations?  
- Question: How does your model's runtime compare to other neural joint parsers such as Dyer et al. (2015)?
