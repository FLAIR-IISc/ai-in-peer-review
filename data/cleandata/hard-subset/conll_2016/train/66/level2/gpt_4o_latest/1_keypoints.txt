- Introduces a transition-based parser that jointly predicts syntactic and semantic dependencies using stack LSTMs to represent the full algorithmic state.  
- Eliminates handcrafted features by learning representations of the entire parser state and uses a greedy linear-time inference algorithm.  
- Achieves state-of-the-art results among joint parsers on CoNLL 2008/2009 English and competitive multilingual performance.  
- Demonstrates strong runtime efficiency, substantially faster end-to-end than comparable pipeline systems.  
- Novel use of stack LSTMs to encode parser state, reducing reliance on expert feature engineering.  
- Strong empirical performance on both syntactic and semantic tasks, rivaling feature-rich pipeline systems.  
- Comprehensive evaluation across multiple datasets with detailed comparisons to joint and pipeline baselines.  
- Limited novelty in the transition-based parsing framework itself; main contribution is in representation learning.  
- Joint model shows overfitting to in-domain (WSJ) data and weaker out-of-domain (Brown) performance compared to hybrid models.  
- Heavy reliance on pretrained embeddings for strong results in some languages (e.g., Chinese); robustness without such embeddings is untested.  
- Lack of morphological or character-based features hurts performance on morphologically rich languages (e.g., Czech).  
- Asks how the model performs without high-quality pretrained embeddings, especially for low-resource languages.  
- Requests clarification on scalability to longer sentences and larger datasets and whether linear runtime holds in practice.  
- Suggests exploring incorporation of morphological features or character-based embeddings to improve cross-lingual robustness.  
- Paper is well-written, includes an open-source implementation, and is a practical, promising contribution despite areas needing further work.
