- Proposes cross-lingual NER using wikification to extract language-independent features (FreeBase types and Wikipedia categories) grounded to English Wikipedia.  
- Method enables direct transfer across languages without parallel text or language-specific processing.  
- Evaluated on high-resource (CoNLL) and low-resource (Turkish, Tagalog, Yoruba, Bengali, Tamil) languages, analyzing the effect of Wikipedia size.  
- Practical implementation requiring only multilingual Wikipedia dumps, making it accessible and easy to reproduce.  
- Wikifier features consistently yield strong empirical gains, outperforming strong baselines by large margins.  
- Novel conceptual shift: use wikification as feature generation for NER rather than a downstream task after NER.  
- Comprehensive evaluation across language types provides valuable insights for low-resource settings.  
- Primary limitation: dependence on FreeBase/Wikipedia coverage, which is uneven and hurts languages like Yoruba.  
- Lacks comparison with recent transformer-based or multilingual pretrained models (e.g., multilingual BERT).  
- Analysis is superficial on why wikifier features help; mapping between FreeBase types and NER tags is asserted but not rigorously analyzed.  
- Asks authors to address FreeBase coverage for non-Latin languages and propose improvements for languages like Bengali and Tamil.  
- Requests detailed analysis of FreeBase type-to-NER tag correlations across languages.  
- Suggests exploring integration or complementarity with modern transformer-based cross-lingual models.  
- Overall recommendation: strong, novel, practically useful paper; recommend acceptance with minor revisions to address comparisons and deeper analyses.
