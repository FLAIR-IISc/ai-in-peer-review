

# Review: Cross-Lingual Named Entity Recognition via Wikification

## Summary and Main Contributions
This paper proposes a cross-lingual NER approach that uses a cross-lingual wikifier to ground words to English Wikipedia, extracting language-independent features (FreeBase types and Wikipedia categories) for NER. The main contributions are: (1) demonstrating that wikifier features serve as strong language-independent signals for NER, (2) showing that these features enable effective direct transfer to low-resource languages without parallel data, and (3) providing a thorough evaluation across 9 languages with varying script types and resource levels.

## Strengths

1. **Novel and Practical Approach to Cross-Lingual NER**: The paper's core contribution is using wikification as a feature source rather than a downstream task. The insight that grounding words to English Wikipedia provides useful entity typing information (even with imperfect disambiguation, as shown in Figure 1) is compelling. This approach requires only multilingual Wikipedia, eliminating the need for parallel text or language-specific heuristics that many prior methods rely on. The paper successfully demonstrates this on languages with non-Latin scripts (Bengali, Tamil), which is a significant contribution.

2. **Strong Empirical Validation**: The evaluation across 9 languages (4 CoNLL languages and 5 low-resource languages) with thorough ablation studies (Table 2, Table 3) provides convincing evidence. The paper shows that wikifier features alone outperform lexical features for non-Latin script languages (Bengali and Tamil), and that combining all features yields the best results for Latin script languages. The comparison to Täckström et al. (2012) and Zhang et al. (2016) demonstrates clear superiority on all languages, with particularly significant gains on Turkish, Tagalog, and Bengali.

3. **Insightful Analysis of Feature Utility**: The paper provides valuable practical insights about when to use different feature types. The finding that wikifier features are essential for non-Latin script languages (where lexical features fail) while being complementary to lexical features for Latin script languages is highly useful for practitioners. The analysis of how Wikipedia size affects performance (Table 3) is also particularly valuable for understanding the limitations of the approach.

## Weaknesses

1. **Lack of Comparison to Recent Multilingual Models**: The paper doesn't compare to modern multilingual transformer models (e.g., mBERT, XLM-R) that have achieved state-of-the-art results in cross-lingual NLP tasks. These models would likely outperform the proposed approach, as they're trained on massive multilingual data. The absence of such comparison makes it difficult to assess the current state of the art for cross-lingual NER.

2. **Insufficient Analysis of Wikifier Limitations**: While the paper acknowledges that Wikipedia size affects performance (Table 3), it doesn't thoroughly analyze the quality of the wikification process itself. For example, it doesn't discuss how often the wikifier fails to disambiguate correctly or how these errors propagate to NER performance. A more detailed error analysis would strengthen the paper.

3. **Computational Cost Not Addressed**: The paper doesn't mention the computational cost of running the wikifier on large documents, which could be prohibitive for real-world applications. The current approach relies on a specific wikifier implementation (Tsai and Roth, 2016), but the authors don't discuss how this would scale or how it compares to more efficient alternatives.

## Questions to Authors

1. How would the approach perform if using a more recent cross-lingual wikifier (e.g., based on multilingual BERT) instead of the Tsai and Roth (2016) system? Would this significantly improve performance on low-resource languages?

2. Can the authors provide more details about the training data sizes for the low-resource languages (Bengali, Tamil, etc.) compared to CoNLL languages? How does this impact the observed performance differences?

3. The paper mentions that the model can be trained on multiple languages. Could the authors elaborate on how the performance scales with additional source languages, and whether there are diminishing returns after including a certain number of languages?

## Overall Assessment

This paper presents a novel and practical approach to cross-lingual NER that makes effective use of language-independent features derived from Wikipedia. The methodology is sound, the evaluation is comprehensive, and the insights about feature utility across languages are valuable. While the paper would benefit from comparisons to more recent state-of-the-art methods and a deeper analysis of the wikifier's limitations, the core contribution of using wikification for language-independent NER features is significant and well-supported by the evidence provided.

The paper makes a meaningful contribution to the field of cross-lingual NLP, particularly for low-resource language scenarios. The approach is simple yet effective, requiring only widely available multilingual Wikipedia resources, which makes it highly accessible for practitioners. I recommend acceptance with minor revisions to address the comparison to recent models and provide more detail on computational considerations.
