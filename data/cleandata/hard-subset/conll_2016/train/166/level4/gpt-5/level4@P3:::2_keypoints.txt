- Paper studies direct-transfer cross-lingual NER using a cross-lingual wikifier to generate language-independent Freebase types and Wikipedia categories for every n-gram (n ≤ 4).  
- Model combines wikifier features with standard lexical/non-lexical features and Wikipedia gazetteers, trained on source languages and applied to many targets without parallel data.

Contributions:
- Empirical demonstration that wikification-derived, delexicalized features enable strong direct-transfer NER, outperforming prior baselines on CoNLL languages and competitive on low-resource languages with no parallel or target supervision.  
- Analysis linking wikifier feature quality to Wikipedia size and interlanguage/Freebase coverage, including ablations that explain cross-lingual performance variation.  
- Evidence that the same language-independent features improve monolingual NER and provide modest gains from multi-source training.

Strengths:
- Strong cross-lingual results using minimal resources (no parallel text or native speakers), with large F1 gains from wikifier features on multiple languages.  
- Clear, coherent methodology and broad evaluation across nine languages and both monolingual and cross-lingual settings.  
- Informative ablation studies (removing Freebase/interlanguage links) that explain lower gains for small-Wikipedia languages.  
- General, simple approach requiring only multilingual Wikipedia and lightweight wikifier modifications.  
- Multi-source training experiments show pooling languages is feasible and can help.

Weaknesses:
- Limited methodological novelty: mainly integration of an existing wikifier into a standard NER pipeline; contribution is primarily empirical.  
- Incomplete comparisons with orthogonal methods (projection, multilingual clustering, silver-data); no combined or strong projection baselines reported.  
- POS features not empirically evaluated despite discussion of tagger issues.  
- Section 3.3 is terse and unclear about wikifying every n-gram; needs a step-by-step example.  
- Multi-source training gains are small and significance is not reported; practical value is unclear.

Questions to authors:
- Can you add an ablation with POS-tag features (where universal POS is available) for both monolingual and direct-transfer settings?  
- Can you report preliminary combined results with Täckström-style multilingual clusters or WIKI-derived silver data to substantiate claimed orthogonality?  
- Please provide a detailed running example of the wikification-for-every-n-gram pipeline (candidate generation, ranking, overlap handling, top-k aggregation).  
- Why do specific training languages help particular targets (e.g., Dutch→Spanish, Spanish→Yoruba): data/annotation style, typology, Wikipedia overlap, or gazetteer lexical overlap?  
- For Table 4, can you report statistical significance for multi-source improvements and identify cases where multi-source hurts?

Additional comments:
- Expand related work to include Kazama & Torisawa (2007) and other Wikipedia-based NER, clarifying methodological differences.  
- Position the approach relative to projection methods that use bilingual semantic spaces (e.g., Peirsman & Padó; Vulić & Moens).  
- Clarify the MISC→LOC label mapping and report sensitivity analysis to maintain comparability.  
- Fix typos and awkward phrases throughout the paper for readability.  
- Add notes on computational cost/latency of wikifying all n-grams and common failure modes (NIL rates, ambiguous titles).

Overall evaluation:
- A solid empirical paper with a clear story and strong results showing the value of wikification-derived features for cross-lingual NER with minimal resources; addressing missing comparisons, Section 3.3 clarity, and significance testing would strengthen the submission.
