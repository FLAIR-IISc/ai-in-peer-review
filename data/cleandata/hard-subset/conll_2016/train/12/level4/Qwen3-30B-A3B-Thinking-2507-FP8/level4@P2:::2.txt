

Summary: This paper proposes a novel approach for low-resource POS tagging that jointly learns from a small amount of gold-standard annotations and cross-lingually projected annotations. The authors introduce a noise layer in the neural network architecture that explicitly models the mapping between projected tags and true tags, enabling the model to learn when to trust the noisy projected data. The method is evaluated on eight simulated low-resource languages and two real low-resource languages (Malagasy and Kinyarwanda), using only 1000 gold tokens.

Main contributions:
1. A noise layer that explicitly models the mapping between projected tags and true tags, enabling effective joint learning from gold and noisy projected annotations.
2. State-of-the-art results on both simulated and real low-resource languages using only 1000 gold tokens.
3. A modular approach that can be applied to various low-resource NLP tasks beyond POS tagging.

Strengths:
1. The paper addresses a critical challenge in low-resource NLP: effectively leveraging noisy distant supervision. The noise layer is simple yet highly effective, achieving state-of-the-art performance across eight simulated languages and two real low-resource languages (Malagasy and Kinyarwanda) with only 1000 gold tokens. This demonstrates significant practical utility for truly low-resource settings where annotated data is scarce.

2. The approach is modular and generalizable. The noise layer can be added to any sequence tagging model with a softmax output layer, making it applicable to a wide range of low-resource NLP tasks including dependency parsing and named entity recognition. The authors' clear description of the model architecture and training process would facilitate adoption by the broader NLP community.

3. The experimental evaluation is comprehensive and well-designed. The authors provide results on both simulated and real low-resource settings, with careful attention to the details of the parallel data and POS projection process. The comparison with strong baselines, including the state-of-the-art method by Duong et al. (2014a), provides convincing evidence of the proposed method's effectiveness.

Weaknesses:
1. The experimental design lacks exploration of the trade-offs between gold and projected data. It would be valuable to see how performance varies with different amounts of gold data (e.g., 100, 500, 1000, 5000 tokens) to determine the minimum amount of gold data required for this approach to be beneficial. This would help establish the practical utility of the method across different resource scenarios.

2. The paper only evaluates the method with cross-lingual projections using the same tag set. It would be more compelling to show results with different tag sets (e.g., using a different tag set for projection than for the target language) and with other sources of noisy labels (e.g., crowd-sourced annotations), which would better demonstrate the broader applicability of the approach.

3. The training objective gives equal weight to gold and projected data, which is problematic given that gold data is limited. The authors should investigate whether tuning the relative weights of the two loss terms (gold vs. projected) could further improve performance, especially when gold data is scarce. This would provide more nuanced guidance for practitioners.

4. The description of the projected data representation (line 357) is unclear. The paper describes projected data as pairs of word tokens (x_t) and vector representations \tilde{y}, but does not explicitly explain what the vector representation looks like (e.g., is it a distribution over projected tags for a word type or a token-level tag?). Additionally, the paper states that only one-to-one alignments are preserved, but it's unclear how \tilde{y} is constructed for words with no alignments.

Questions to authors:
1. How does the model perform when the amount of gold data is increased beyond 1000 tokens? Is there a point where using the noise layer becomes less beneficial?

2. Could the approach be extended to handle different tag sets between the source and target languages, and how would this affect performance?

3. How sensitive is the model to the relative weighting of the gold and projected loss terms? What optimal weights did you find during training?

4. How would you construct \tilde{y} for words that have no alignments (i.e., are not mapped to any English word)? Would the approach still work if only one-to-one alignments are preserved?

The paper makes a valuable contribution to low-resource NLP by providing an elegant and effective solution for leveraging noisy distant supervision. While the experimental design could be strengthened by exploring the trade-offs between gold and projected data, the core idea is well-motivated, the implementation is clear, and the results demonstrate significant practical value. The authors should address the clarification questions about the projected data representation to improve the paper's clarity and reproducibility.
