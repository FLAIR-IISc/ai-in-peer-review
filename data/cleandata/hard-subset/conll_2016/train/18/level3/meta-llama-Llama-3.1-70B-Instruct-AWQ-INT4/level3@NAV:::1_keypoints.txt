- Proposes a transition-based graph parser (BSP) for UCCA that supports multiple parents, non-terminal nodes, and discontinuous units.  
- Also explores conversion-based parsing by converting UCCA to constituency or dependency trees for existing parsers.  
- Evaluates approaches on UCCA-annotated corpora reporting labeled precision, recall, and F-score for primary and remote edges.  
- Main contribution: novel BSP transition-based parser designed for broad-coverage semantic structures.  
- Main contribution: conversion-based parsing pipeline as an alternative approach.  
- Main contribution: empirical experiments and reported metrics on UCCA parsing.  
- Strength: approach is novel in proposing a parser handling UCCA’s structural requirements.  
- Strength: evaluation uses multiple metrics (precision, recall, F-score) for primary and remote edges.  
- Strength: compares results against existing parsers (e.g., UPARSE, MaltParser) and shows competitive performance.  
- Weakness: claim of being the first semantic-oriented parser is weak because the transition set does not exploit UCCA-specific phenomena.  
- Weakness: missing baselines—does not compare to existing dependency DAG parsers.  
- Weakness: evaluation metrics are limited and omit metrics from related SDP shared tasks.  
- Question: request for details on UCCA-specific features used in BSP.  
- Question: how authors will address methodological concerns about the transition set not exploiting UCCA phenomena.  
- Question: request for more information on conversion procedures from UCCA to constituency/dependency trees.  
- Question: how evaluation will be extended to include metrics from related SDP shared tasks.
