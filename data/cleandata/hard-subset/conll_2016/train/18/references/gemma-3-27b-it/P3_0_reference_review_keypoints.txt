- Introduces BSP, the first dedicated transition-based parser for UCCA that handles multiple parents, non-terminals, and discontinuities.  
- Evaluates conversion-based approaches (constituency/dependency) and highlights information loss from conversion.  
- Releases code and a clear experimental setup, providing a solid baseline for future UCCA work.  

- Fills a tooling gap by making UCCA parsing accessible and provides publicly available code.  
- Provides a comprehensive empirical evaluation, including in-domain and out-of-domain comparisons.  
- Builds on established transition-based parsing techniques and adds UCCA-specific features (e.g., remote edge handling).  
- Presents a clear, accessible introduction to UCCA with helpful illustrative figures.  
- Includes an informative ablation study analyzing feature importance.  

- Performance gains over tuned conversion methods (notably LSTM dependency parser) are limited, questioning complexity tradeoffs.  
- Lacks a detailed error analysis identifying specific failure modes (e.g., remote edges, coordination).  
- Hyperparameter tuning appears limited and some components used default settings, suggesting room for improvement.  
- Limited exploration of alternative transition systems tailored to DAGs and non-projectivity.  
- Evaluation metric treats all edge types equally and fails to distinguish primary vs. remote edges or hierarchical importance.  

- Request for a detailed error analysis comparing BSP’s errors to the best conversion method.  
- Ask for intuition and details about the ratio feature and whether other regularization methods were tried.  
- Inquire about specific challenges and solutions when adapting transition-based techniques to UCCA’s DAG structure.  
- Ask authors to justify the advantages of a dedicated UCCA parser given modest gains over LSTM dependency parsing.  
- Suggest developing a more nuanced evaluation metric that distinguishes primary and remote edges.  

- Paper is well-written, with comprehensive related work and useful supplementary material.  
- Suggest comparing computational cost of BSP versus conversion-based approaches.  

- Overall: valuable foundational contribution to UCCA parsing; recommend acceptance with minor revisions focused on error analysis, tuning, and evaluation improvements.
