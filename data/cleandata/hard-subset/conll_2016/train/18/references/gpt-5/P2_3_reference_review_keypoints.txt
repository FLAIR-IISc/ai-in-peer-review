- Paper targets broad-coverage grounded semantic parsing supporting reentrancies, non-terminals, and discontinuities, focusing on UCCA.
- Proposes two strategies: deterministic UCCA↔constituency/dependency conversions and a new transition-based parser (BSP) that directly predicts UCCA DAGs.
- BSP introduces a practical transition set (NODE, SWAP), remote-edge handling, a dynamic oracle, and a novel “ratio” feature to control non-terminal generation.
- Conversion plus a strong LSTM dependency parser yields the best primary-edge F-score (~69.2); BSP achieves lower primary LF (~59) but is the only model predicting remotes (remote LF ~13.3).
- Paper provides in-domain (Wiki) and out-of-domain (20K Leagues) evaluations, separate remote-edge scoring, and feature ablations.
- Contribution: first direct transition-based parser supporting non-terminals, reentrancies, and discontinuities for grounded semantic DAGs.
- Contribution: explicit, deterministic UCCA↔constituency/dependency conversion framework enabling strong, reproducible baselines and upper bounds.
- Contribution: empirical benchmarking with OOD evaluation, yield-based evaluation protocol, and ablation studies.
- Strength: clear problem framing motivating the need for non-terminals, reentrancies, and discontinuities using UCCA as a cross-linguistic target.
- Strength: technically sound BSP design, principled extension of prior transitions, appropriate constraints, dynamic oracle, and useful ratio feature.
- Strength: conversion baselines are transparent and strong, showing much primary structure can be recovered via dependency parsing plus conversion.
- Strength: evaluation breadth, transparency, and promises of code release support reproducibility.
- Weakness: BSP underperforms the conversion+LSTM baseline on primary structure; stronger local/neural models for BSP are needed.
- Weakness: remote-edge performance is low and lacks qualitative/error analysis to diagnose failures.
- Weakness: unclear preprocessing details (e.g., POS tagging source/accuracy) and limited comparison to other DAG/semantic dependency parsers; no significance testing reported.
- Weakness: metric/task scope caveats—the yield-based metric and removed phenomena (linkage, inter-sentential) narrow the task and deserve more discussion.
- Weakness: minor presentation issues (typos, notation) and lack of a worked example for dynamic oracle behavior on remotes.
- Questions: which POS tagger and tag quality were used, and are features based on gold or automatic tags?
- Questions: how many sentences contain remote edges and discontinuities, and can per-phenomenon performance be reported (coordination, phrasal verbs, shared arguments)?
- Questions: can BSP be reported with a small beam or neural (bi-LSTM) features to test closing the gap to conversion baselines?
- Questions: how does BSP behave when trained/tested with/without remote edges—evidence of confusion between primary vs. remote?
- Questions: any preliminary qualitative error analysis for remote edges and discontinuities (typical false positives/negatives, SWAP interactions)?
- Overall: timely, well-motivated, technically sound work that advances grounded semantic parsing; leans toward acceptance pending clarification of evaluation details and added error analysis for remote edges.
