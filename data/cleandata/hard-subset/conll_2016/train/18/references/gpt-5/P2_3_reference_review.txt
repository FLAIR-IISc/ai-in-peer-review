Summary
The paper targets broad-coverage semantic parsing for grounded formalisms that require (i) multiple parents (reentrancies), (ii) non-terminal units, and (iii) discontinuities. It focuses on UCCA as a representative scheme and presents two strategies: (a) conversion-based parsing by mapping UCCA to (discontinuous) constituency and dependency trees, using existing parsers and converting back; and (b) a new transition-based parser (BSP) that directly predicts UCCA DAGs with non-terminals and discontinuities, including specialized transitions for remote edges and SWAP for gaps. The paper proposes deterministic conversion algorithms, a novel “ratio” feature, a dynamic oracle, and reports results on in-domain (Wiki) and out-of-domain (20K Leagues) data. Conversion to dependency plus a strong LSTM parser yields the best primary-edge F-score (~69.2), while BSP directly predicts both primary (LF ~59) and remote edges (LF ~13.3), with reasonable OOD robustness. Feature ablations are provided.

Contributions (as I see them)
1) A transition-based parser (BSP) for grounded semantic DAGs that jointly supports non-terminals, reentrancies (multiple parents), and discontinuities, with a practical transition set, dynamic oracle, and task-specific features. This is novel and addresses a clear gap.
2) A conversion-based evaluation framework with explicit, deterministic UCCA↔constituency/dependency conversions, enabling strong baselines (including a modern LSTM dependency parser) and clear upper bounds; this both contextualizes the difficulty and offers reproducible baselines.
3) Empirical benchmarking of UCCA parsing (including OOD evaluation), with a reasonable evaluation protocol (yield-based labeled precision/recall for primary vs. remote edges) and feature ablations.

Strengths
- Clear problem framing and well-motivated target: The paper crisply articulates why grounded semantics needs non-terminals, reentrancies, and discontinuities (coordination, MWEs, phrasal verbs), and shows that existing grounded parsers typically miss at least one. Targeting UCCA makes the setting concrete and cross-linguistically relevant.
- Technical novelty and soundness of BSP: The transition set extends prior work on discontinuous constituency and DAG dependencies in a principled way (NODE, SWAP, and primary vs. remote edge distinctions). The constraints (one primary parent, remote edges allowed) and the use of dynamic oracles are appropriate. The novel ratio feature is simple, intuitive regularization against over-generation of non-terminals and shows empirical utility.
- Strong and fair baselines via conversion: The conversion algorithms are explicit and simple, yielding transparent upper bounds. Using UPARSE and Malt/LSTM parsers with default settings provides informative reference points, especially the gap between conversion upper bounds and achieved performance. The LSTM result (69.2) illustrates that much of UCCA’s primary structure can be recovered via dependency parsing plus lossy conversion, which is a useful empirical finding for the community.
- Evaluation breadth and transparency: In-domain and OOD (20K Leagues) evaluations are included; remote edges are evaluated separately; feature ablations indicate which design choices matter (e.g., unigrams/bigrams and ratio). The reported upper bounds make the information loss in conversions explicit (e.g., zero remote edges).
- Reproducibility: The paper promises public release of converters and parsers, reports hyperparameters (iterations, MINUPDATE, IMPORTANCE), and uses standard toolkits, which should facilitate replication and extension.

Weaknesses
- Limited absolute performance of BSP vs. conversion baseline: While BSP is the only model predicting remotes, its primary-edge LF (~59) lags substantially behind the dependency+LSTM conversion baseline (69.2). The paper argues that adopting an LSTM classifier in BSP would help, but as-is, the main direct parser underperforms the best conversion-based alternative on primary structure. A stronger direct model (even small-beam or neural features) would strengthen the case for the direct approach.
- Remote edge performance is low and under-analyzed: Remote LF ~13.3 (and even lower OOD) indicates the hardest aspect remains largely unsolved. There is no qualitative error analysis (types of reentrancies, confusion between primary/remote, interaction with discontinuity), which limits insight into failure modes and how to improve them.
- Evaluation details and fairness: It is not fully clear what preprocessing is used (e.g., POS tagging source/accuracy), which affects feature quality. Also, existing DAG parsers were not included due to code availability; however, a stronger effort to compare to recent semantic dependency parsers (even via conversion) would situate results better. Statistical significance is not reported.
- Metric and task scope caveats: The yield-based metric is reasonable, but it conflates some structural distinctions and ignores inter-sentential and linkage phenomena (explicitly removed), narrowing the task. The consequences of high- vs. low-attachment decisions lost in conversion (and reintroduced heuristically) deserve more discussion vis-à-vis downstream semantics.
- Presentation and polish: There are minor typos (e.g., passage indices “701–695”), some reference formatting issues, and occasional notational clutter. A brief, worked example of the dynamic oracle behavior for remote edges would aid clarity.

Questions to Authors (optional)
- What POS tagger and accuracy were used? Are features based on gold or automatic tags? How sensitive are results to tagging quality?
- How many sentences exhibit remote edges and discontinuities, respectively? Can you provide per-phenomenon performance (e.g., coordination, phrasal verbs, shared arguments)?
- Can you report BSP with a small beam or with a neural classifier (e.g., bi-LSTM features) to substantiate the claim that better local models will close the gap?
- How does BSP’s performance change if trained to ignore remote edges but at test time allow them (or vice versa)? Any evidence of confusion between primary vs. remote?
- Any preliminary qualitative error analysis for remote edges and discontinuities (e.g., typical false positives/negatives, interaction with SWAP frequency)?

Overall evaluation
This is a timely and well-motivated paper that advances grounded semantic parsing by both proposing a first direct parser meeting key structural requirements and establishing strong, transparent baselines via conversion. While the direct model’s accuracy lags the best conversion baseline for primary structure and remote edges remain challenging, the contributions are technically sound, clearly presented, and likely to stimulate follow-up research. I lean toward acceptance, contingent on clarifying evaluation details and adding some error analysis for remote edges.
