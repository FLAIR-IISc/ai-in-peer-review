### Summary and Contributions

This paper introduces a method for automatically discovering string-level correspondences between multiple languages using the Minimum Description Length (MDL) principle. The core idea is to find a set of correspondence rules that provides the most succinct description of a given set of cognate words. The method avoids making prior assumptions about the distribution or shape of these rules. The authors propose a deterministic, EM-like algorithm to infer the model, where the E-step aligns the data using the current rules and the M-step greedily merges rules to improve the overall description length.

The main contributions of this work are:
1.  **A novel MDL formulation for multi-language correspondence discovery:** The paper presents a principled, non-parametric framework for learning correspondence rules from N languages simultaneously, which is a significant step beyond common pairwise approaches.
2.  **A detailed, information-theoretic similarity metric:** The proposed "shared Description Length" (sDL) allows for a fine-grained quantitative analysis of the similarities between all possible subsets of the analyzed languages, offering a more nuanced perspective than traditional phylogenetic trees.
3.  **An effective, deterministic algorithm for rule inference:** The proposed training procedure provides a practical way to find good sets of correspondence rules, with the additional flexibility of a data over-weighting parameter to control the granularity of the discovered patterns.

### Strengths

1.  **Principled and Well-Motivated Framework:** The use of the Minimum Description Length (MDL) principle is an excellent fit for this unsupervised discovery task. It provides a formal and objective criterion for model selection, naturally balancing the complexity of the learned rule set against its ability to explain the data. This avoids the need for arbitrary distributional assumptions (e.g., priors in a Bayesian model) or hyperparameters that limit the form of correspondences.

2.  **Novel Multi-Language Analysis:** The paper's key strength lies in its ability to handle more than two languages simultaneously. The subsequent analysis using shared Description Lengths (sDL) in Section 5.3 is particularly insightful. Figure 3 provides a compelling visualization that goes beyond a simple hierarchical clustering, quantifying the complexity shared by specific subgroups (e.g., CS-PL, RU-BG) as well as the unique complexity of each language within the context of the larger group. This is a powerful tool for computational historical linguistics.

3.  **Strong Empirical Validation:** The experiments are well-designed. The pairwise analysis on 13 Slavic languages serves as a strong sanity check, successfully recovering the established linguistic phylogeny (Figure 1). The authors provide plausible linguistic explanations for the observed distances and outliers, demonstrating a good understanding of the domain. The qualitative examples of learned rules (Tables 3 and 4) are convincing, showing that the method can capture a wide range of phenomena, from single-character phonological shifts (`g`,`h`) to complex orthographic clusters (`szcz`,`št`) and morphemic patterns.

4.  **Pragmatic Flexibility:** The authors are transparent about the limitations of a purely objective statistical approach on limited data. The introduction of the data over-weighting parameter (`α`) is a simple but clever and practical mechanism. It empowers a domain expert (e.g., a linguist) to guide the discovery process towards finer-grained, linguistically interesting patterns that might not meet the strict threshold of statistical significance, without abandoning the ranked, significance-driven nature of the search.

### Weaknesses

1.  **Significant Scalability Concerns:** The primary weakness is the method's computational complexity. The authors admit that the current implementation is limited to roughly N=5 languages. The model encoding `L(Π)` involves summing over `2^N - 1` language subsets, and the alignment procedure (shortest path on a multi-dimensional grid) is known to scale exponentially with the number of sequences (languages). This severely limits the applicability of the method for larger-scale comparative studies, which is a major goal in historical linguistics. The paper would be stronger if it included a more formal analysis of this complexity and discussed potential avenues for approximation or optimization.

2.  **Greedy and Potentially Suboptimal Search:** The M-step of the algorithm employs a greedy search, merging the single best pair of rules at each iteration. While this makes the algorithm deterministic and efficient for a single step, it is highly susceptible to local optima. An early, seemingly optimal merge could preclude the later discovery of a globally better, longer correspondence rule. The paper acknowledges this possibility but does not investigate its impact. It is unclear how sensitive the final model is to the initialization or the greedy merge decisions.

3.  **Clarity of the MDL Formulation:** While the overall idea is clear, the specific formulation of the model code `L(M)` in Section 4.1 is dense and difficult to parse. In particular, the equation for `L(Π)` contains multiple terms whose interaction is not fully justified. For instance, it encodes rule counts using a data-to-model code (`LN(TΠ) + log(...)`), which seems to encode model parameters (usage probabilities) as part of the model structure cost, rather than being implicitly handled by the data encoding `L(D|M)`. A clearer, step-by-step motivation for each term in the code, perhaps framed as a literal transmission process, would greatly improve the paper's reproducibility and comprehensibility.

4.  **Limited Comparison with Probabilistic Models:** The related work section is quite brief. While MDL is positioned as an alternative to Bayesian methods, the paper does not engage deeply with the relevant literature on non-parametric Bayesian models for sequence alignment and pattern discovery (e.g., models based on Pitman-Yor or Dirichlet Processes). These models can also discover an unbounded number of patterns without strong prior assumptions and often employ more sophisticated inference techniques (like Gibbs sampling) that can escape local optima more effectively than a greedy search. A discussion of the relative advantages and disadvantages of the MDL approach compared to these probabilistic counterparts would strengthen the paper's contribution.

### Questions to Authors

1.  Could you elaborate on the computational bottlenecks as N increases? Is the primary issue the `O(2^N)` complexity of iterating through language subsets in the model code, or the exponential state space of the alignment graph in the E-step? Are there pruning strategies or approximations (e.g., beam search during alignment) that could extend the method's reach to a larger number of languages?

2.  Regarding the greedy M-step: have you explored how much the final rule sets vary under different search strategies? For example, would a beam search for the best *k* merges at each step, or a randomized process like simulated annealing, lead to significantly different models with better overall description lengths?

3.  Could you provide more intuition for the structure of `L(Π)`? Specifically, why are the rule usage counts (`TΠ`, `|Π|`) encoded as part of the model description `L(M)` using a data-to-model code, instead of simply defining `L(D|M) = -∑_d ∑_{π∈d} log(p(π))` where `p(π)` is the ML estimate `count(π) / TΠ`? The standard two-part code would typically place the cost of encoding the parameters (`p(π)`) with the model, and the cost of the data using those parameters would be `L(D|M)`. Your formulation seems to blend these.
