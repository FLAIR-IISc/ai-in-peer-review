### Summary and Contributions

This paper addresses the problem of discovering systematic string-level correspondences among multiple languages. The authors propose a novel approach based on the Minimum Description Length (MDL) principle, which posits that the best model (i.e., the best set of correspondence rules) is the one that provides the most compact description of the given data (cognate lists). The method is implemented via an efficient, deterministic EM-like algorithm that iteratively aligns the data with the current rule set and improves the rule set by greedily merging patterns that yield the best compression. The framework is demonstrated on a set of 13 Slavic languages, showing its ability to generate linguistically plausible phylogenetic trees from pairwise analyses and to perform a novel, fine-grained quantitative analysis of similarities among subgroups in a four-language experiment.

In my view, the main contributions of this work are, in order of importance:

1.  **A principled and novel framework for multi-language correspondence discovery:** The paper successfully formulates the problem in terms of MDL, providing an objective, data-driven method for discovering correspondence rules of arbitrary shape for more than two languages simultaneously. This moves beyond the common limitation of pairwise analysis in prior work.
2.  **A new method for fine-grained analysis of language similarity:** The introduction of "shared Description Length" (sDL) is a direct and powerful benefit of the MDL approach. It offers a quantitative tool to measure the amount of shared structure between any subset of the languages being analyzed, providing a more detailed and nuanced view of linguistic relationships than is possible with a single phylogenetic tree.
3.  **A convincing empirical demonstration on Slavic languages:** The experiments show that the method can recover known linguistic groupings (West, East, South Slavic) and also uncover more subtle relationships, such as the relative similarity of Bulgarian to Russian within a four-way analysis.

### Strengths

1.  **Methodological Soundness and Novelty:** The application of the MDL principle to this problem is a key strength. It provides a theoretically grounded, non-parametric approach that avoids imposing strong prior assumptions on the nature or distribution of correspondence rules. This allows the patterns to emerge directly from the data, which is a significant advantage for objective linguistic discovery. The generalization to handle N>2 languages in a single model is a non-trivial and important contribution.

2.  **Powerful Analytical Capabilities:** The proposed sDL metric is a highlight of the paper. Unlike traditional methods that produce a single distance matrix, this approach provides a rich, multi-faceted quantification of similarity. Figure 3, which breaks down the total description length into components shared by all possible subsets of the four languages, is an excellent demonstration of the method's analytical power. It allows researchers to ask more sophisticated questions about language group structure.

3.  **Clear and Well-Motivated Algorithm:** The paper presents a clear, deterministic EM-like algorithm for optimizing the MDL objective. The process of starting from a "null alignment" and greedily merging rules based on their compression gain is intuitive and well-explained. The fact that the algorithm is deterministic enhances reproducibility, which is a valuable feature for a scientific discovery tool.

### Weaknesses

1.  **Lack of Intrinsic Evaluation for Correspondence Rules:** The primary output of the model is a set of correspondence rules, but their quality is only assessed qualitatively through cherry-picked examples. The paper lacks a quantitative, intrinsic evaluation against a gold standard. For well-studied families like Slavic, there exist established sound correspondences from historical linguistics. Comparing the discovered rules against such a reference set would provide a much stronger validation of the method's ability to find linguistically correct patterns. Without this, it is difficult to assess whether the model is merely finding statistical string patterns or genuine linguistic correspondences.

2.  **Unclear Scalability and Computational Limits:** The authors note that their implementation can handle "up to five languages... within a few hours," which raises significant concerns about scalability. The method's key advantage is its ability to analyze multiple languages, but the practical limit of N seems low. The complexity analysis provided is abstract (e.g., dependent on `|R|`, the number of possible rule applications), and a more concrete discussion of how complexity scales with N (the number of languages) and L (cognate length) is needed. A clear characterization of the computational bottlenecks would be crucial for understanding the method's applicability to larger-scale comparative linguistics.

3.  **Clarity of the MDL Formulation:** While the overall idea is clear, the technical description of the MDL code in Section 4 is dense and difficult to follow. For instance, the role and calculation of the "data-to-model code" for the rule table L(Π) are not explained with sufficient intuition. A small, concrete example walking through the calculation of L(M) and L(D|M) for a toy dataset (e.g., two cognates across three languages) would vastly improve the paper's clarity and reproducibility.

4.  **Ad-hoc Nature of "Data Over-Weighting":** The introduction of the α parameter to over-weight the data term (Section 4.5) seems to contradict the principled, objective nature of MDL, which is presented as a major advantage of the work. This technique feels like an ad-hoc mechanism to deal with data sparsity. Presenting the results from this modified objective as the preferred outcome (e.g., in Table 3) can be confusing. It raises the question of whether the "pure" MDL approach is sufficient on its own. The paper would be stronger if it either provided a more formal justification for this technique or focused its evaluation solely on the results of the unmodified, objective MDL procedure.

### Questions to Authors

1.  Could you provide a more detailed analysis of the algorithm's scalability? How do the key factors in the complexity, such as the size of the alignment graph and the set of candidate rules for merging, grow as the number of languages N increases? What are the primary obstacles to scaling beyond five languages?

2.  The evaluation of the learned rules is currently qualitative. To what extent do the high-ranking discovered correspondences (i.e., those with the highest counts or compression gains) align with known sound laws in Slavic historical linguistics? Even a small-scale comparison against a gold standard would substantially strengthen the paper's claims.

3.  Regarding the data over-weighting technique: is this primarily a tool to counteract data sparsity? Could similar results (i.e., discovering more granular but less frequent rules) be achieved within the standard MDL framework, for example by refining the model code L(M) to have a different complexity penalty?
