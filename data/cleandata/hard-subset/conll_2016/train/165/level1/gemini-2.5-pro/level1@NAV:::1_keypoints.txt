- Introduces a multi-language method for discovering orthographic correspondences using MDL and a novel shared Description Length (sDL) metric.  
- Proposes a deterministic EM-like algorithm that iteratively refines rule sets to minimize description length and includes a data-weighting mechanism to control rule granularity.  
- Major contribution: extends correspondence discovery beyond pairwise comparisons to quantify shared structure among arbitrary subgroups of languages.  
- Methodology strength: MDL provides an objective, information-theoretic foundation that avoids ad-hoc tuning and strong distributional assumptions.  
- Empirical strength: Reconstruction of a phylogenetically plausible Slavic tree in pairwise analysis and a compelling four-way analysis revealing subtle inter-group relationships.  
- Scalability weakness: Practical limit of ~5 languages, possible exponential components in N, and no analysis of runtime/memory scaling with number of languages.  
- Baseline comparison weakness: Paper lacks comparisons to relevant baselines (other sequence-based phylogenies or cognate-analysis tools) to contextualize advantages.  
- Clarity/reproducibility weakness: Section 4’s MDL code description is dense and would benefit from a clearer, more intuitive explanation or a running example.  
- Question for authors: Please detail the scalability limits—how alignment and maximization complexity scale with N and what heuristics/approximations could enable analysis for N > 10.
