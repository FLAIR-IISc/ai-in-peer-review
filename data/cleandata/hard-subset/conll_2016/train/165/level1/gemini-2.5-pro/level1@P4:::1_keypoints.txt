Paper proposes an MDL-based, EM-like method to discover character-level correspondences and align cognates jointly across multiple languages; evaluated on 13 Slavic languages with pairwise and four-way analyses using a new shared Description Length (sDL) metric.

Applies MDL to multilingual correspondence discovery, allowing rules that apply to any subset of languages rather than only pairwise.

Introduces shared Description Length (sDL) as an information-theoretic metric to quantify regularity shared by arbitrary language subsets.

Presents a deterministic, greedy EM-like algorithm that iteratively aligns data and merges patterns to reduce total description length.

Method is technically sound with a clear MDL objective and a detailed two-part code formulation.

Generalizes beyond pairwise analyses; four-way experiments reveal plausible subgroup structures (e.g., CS-PL, RU-BG).

sDL and the breakdown visualization provide nuanced, quantitative insights into language similarities and per-language complexity.

Paper is clearly written with sufficient detail on initialization, Dijkstra-based alignment, and the greedy merging procedure.

Scalability is limited (current implementation up to ~5 languages); alignment complexity likely grows rapidly with N and the dependence of |R| on N is not analyzed.

Core algorithmic novelty is limited—builds on existing MDL/greedy pattern-mining approaches—and the greedy M-step risks convergence to local optima.

The introduced data over-weighting parameter α undermines the paper’s MDL objectivity and lacks rigorous information-theoretic justification.

Experiments rely on orthographic data, risking conflation of orthography/script effects with true phonological correspondences.

Request detailed complexity analysis of the alignment graph as a function of N and suggestions for scalable approximations or alternative search strategies.

Ask for sensitivity analysis of the greedy M-step and consideration of alternatives (e.g., beam search) to avoid premature convergence.

Request stronger theoretical justification for α or principled alternatives (e.g., modifying L(M)) to encourage discovery of non-singleton rules.
