- Presents a novel CNN-based method for MT training data selection, scoring and ranking general-domain sentences.  
- Thorough comparison to monolingual selection methods (n-gram and continuous-space LMs).  
- Lacks comparison to established bilingual selection methods (e.g., cross-entropy difference).  
- Strong motivation for using CNNs over recurrent models to capture salient n-grams.  
- CNN architecture description is clear (one-hot and pre-trained embeddings) but needs a diagram.  
- Missing experimental ablation comparing BOW vs SEQ representations and their combination.  
- Unclear methodology for selecting the optimal number of sentences via validation — which validation set was used?  
- Need clarification on CNN hyperparameter tuning and model sensitivity to those choices.  
- Table 2 should report all methods using an identical number of selected sentences for a fair comparison.  
- Minor writing issue: citation formatting (use textual \citet where appropriate).  
- Suggest adding qualitative analysis (example sentences) to support claims in Section 5.4.  
- Recommend comparing to Axelrod et al. (2015) POS-abstraction baseline to test whether embeddings add benefit.  
- Note that summing source and target scores resembles Moore & Lewis (2010); add a reference and discussion.  
- Suggest experiments with more in-domain data to test robustness under different data conditions.  
- Propose extending the model to a bilingual/parallel CNN to learn weights for combining source and target scores.
