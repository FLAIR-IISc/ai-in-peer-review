- Proposed semi-supervised CNN to select in-domain sentence pairs for SMT using tiny in-domain data augmented with word embeddings from a general corpus.  
- Primary contribution: demonstrates CNN-based data selection outperforms LM-based methods when in-domain data is very limited.  
- Secondary contribution: shows the CNN can be trained with as few as 100 in-domain sentences for topic-specific adaptation.  

- Strength: Effective data selection with limited resources—consistent improvements even with ~100 sentences.  
- Strength: Robustness—CNN approach more stable than LM-based methods on very small in-domain sets, leveraging learned features and pretrained embeddings.  
- Strength: Clear experimental setup and baselines across diverse language pairs/domains with statistical significance testing.  

- Weakness: Overstated performance gains—the reported +3.1 BLEU may largely reflect in-domain data already present in the general training set (baseline gains of +2.8 BLEU).  
- Weakness: Limited generalizability—the 100-sentence experiment was only on zh→en where in-domain data existed in the training set, leaving unclear performance when no in-domain data is present.  
- Weakness: Lack of comprehensive ablation study—missing analysis of embedding sizes, CNN architectures, pooling strategies, etc.  

- Q: Provide results for all four language directions using only 100 in-domain sentence pairs to demonstrate generalizability?  
- Q: What preprocessing/filtering was applied to social media datasets to handle OOVs, usernames, hashtags, emoticons?  
- Q: Provide concrete examples of the types of improvements the data selection yields (e.g., domain-specific terminology vs. other gains)?  
- Q: Clarify how word embeddings, segment vectors, and one-hot vectors are combined; include architecture details and specify the loss function used.  

- Additional: Explicitly state how SMT system weights were tuned in Section 5.2 beyond “standard settings.”  
- Additional: Related work is solid but should briefly mention recent unsupervised domain adaptation advances.
