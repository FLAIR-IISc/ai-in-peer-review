- Proposes two dictionary-based multilingual embedding methods (multiCluster and multiCCA) across 59 languages and an improved evaluation metric (multiQVEC-CCA) plus a web evaluation portal.  
- multiQVEC-CCA strongly correlates with downstream tasks (Pearson 0.896 with document classification), outperforming prior intrinsic metrics.  
- The web portal provides standardized, reproducible evaluation across nine metrics with data downloads and dataset registration.  
- Demonstrates creating multilingual embeddings from dictionaries and monolingual data, useful for low-resource languages.  
- multiCCA shows consistently strong performance across multiple evaluation metrics.  
- multiQVEC-CCA is a substantive methodological improvement over previous intrinsic evaluation approaches.  
- Lacks analysis explaining why multiCluster works for 12 languages but degrades for 59, missing deeper theoretical insight.  
- Does not systematically evaluate sensitivity to key parameters (e.g., dictionary threshold τ, cluster size limits).  
- Related work discussion is incomplete, failing to situate multiCluster relative to prior clustering-based multilingual methods.  
- Paper favors breadth over depth, offering wide evaluations but limited in-depth analysis of conflicting results.  
- Asks authors to explain causes of multiCluster performance drop when scaling to 59 languages.  
- Requests analysis of sensitivity to dictionary-extraction threshold τ and guidance for setting it across languages.  
- Seeks recommendations for addressing inconsistent supersense annotations across languages and standardizing them for multilingual evaluation.
