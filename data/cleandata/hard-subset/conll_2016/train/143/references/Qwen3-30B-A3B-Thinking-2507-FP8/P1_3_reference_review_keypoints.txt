- Proposes two dictionary-based multilingual embedding methods (multiCluster, multiCCA) that use monolingual data and pairwise dictionaries rather than full parallel corpora.  
- Introduces improved evaluation metrics (multiQVEC, multiQVEC-CCA) that better correlate with downstream tasks than existing metrics.  
- Provides a web portal for standardized evaluation of multilingual embeddings with development and test modes.  
- Strongest contribution: evaluation framework demonstrating substantially higher correlation with downstream tasks (e.g., multiQVEC-CCA vs prior metrics).  
- Comprehensive evaluation across multiple intrinsic and extrinsic tasks, enabling systematic comparison of metrics.  
- Web portal is a valuable community resource to standardize multilingual embedding evaluation.  
- Limited novelty of embedding methods: multiCCA extends prior work and multiCluster adapts clustering, with marginal improvements over baselines.  
- Evaluation framework relies on supersense annotations available for very few languages (English, Italian, Danish), limiting applicability.  
- Missing comparison to recent state-of-the-art (e.g., Artetxe et al. 2018), making relative performance unclear.  
- Vocabulary coverage differences across methods may bias results; intersection-based controls were not fully integrated into main results.  
- Claims about effectiveness on 59 languages are overstated given evaluations mostly on 3–12 languages and observed performance drop when scaling.  
- Question: How will the evaluation framework be extended to languages without supersense annotations?  
- Question: Why was Artetxe et al. (2018) not included in comparisons?  
- Question: How will the web portal handle vocabulary coverage differences, especially for 59-language experiments?  
- Overall: Valuable contributions via evaluation metrics and web portal warrant acceptance with minor revisions, but embedding-method novelty and evaluation scope need strengthening.
