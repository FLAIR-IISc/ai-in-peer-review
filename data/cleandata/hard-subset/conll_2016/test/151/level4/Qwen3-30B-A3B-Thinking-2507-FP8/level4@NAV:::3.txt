

This paper proposes a method for parsing universal dependency treebanks using only part-of-speech (POS) tags, leveraging a modified PageRank algorithm that assigns higher weights to meaningful words (excluding stop words). While the core idea is interesting, it closely parallels Søgaard (2012). The personalization factor prioritizing the main predicate is well-motivated, though extending this approach beyond the main predicate would strengthen the contribution. The current implementation restricts personalization to the main predicate with an arbitrary weight of 5. Regarding evaluation, additional visualizations (e.g., charts) would enhance clarity compared to the current tabular presentation. Furthermore, reporting POS tagging performance in prediction mode would clarify whether parsing performance degradation correlates with tagging degradation—a key expectation. Overall, the paper presents an interesting approach, but the incremental advancement over Søgaard (2012) is modest.
