- Proposes parsing UD treebanks using only POS tags via a modified PageRank that weights meaningful words (excluding stop words).  
- Core idea is interesting but closely parallels Søgaard (2012), so the novelty is modest.  
- Personalization that prioritizes the main predicate is well-motivated and beneficial.  
- Limiting personalization to only the main predicate weakens the contribution; extending it beyond the main predicate would strengthen the work.  
- The personalization weight is arbitrarily fixed at 5 without justification.  
- Evaluation would benefit from additional visualizations (charts) rather than only tables.  
- Report POS tagging performance in prediction mode to determine if parsing degradation stems from tagging errors.  
- Minor issue: typo at l.207 ("The the main idea" → "The main idea").
