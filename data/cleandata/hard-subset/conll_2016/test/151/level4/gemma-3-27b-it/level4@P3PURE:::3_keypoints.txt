- Proposes POS-only dependency parsing using a modified PageRank that downweights stop words.  
- Modification emphasizes meaningful words; idea is interesting but closely related to Søgaard (2012).  
- Introduces a personalization factor to weight the main predicate, which is a useful addition.  
- Personalization appears applied only to the main predicate and the chosen weight of 5 seems arbitrary and underexplored.  
- Evaluation lacks visual charts, making it hard to extract key insights from tables.  
- Should report POS tagging scores in prediction mode to check whether parsing degradation follows tagging degradation.  
- Overall contribution is incremental improvement over Søgaard (2012).  
- Minor typo: line 207 “The the main idea” → “The main idea.”
