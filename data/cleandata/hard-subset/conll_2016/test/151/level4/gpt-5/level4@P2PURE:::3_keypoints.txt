- Proposes parsing UD treebanks using only POS tags and a modified PageRank that favors meaningful words over stop words.  
- Idea is interesting but very close to Søgaard (2012), so novelty is limited.  
- Personalization that boosts the main predicate is a nice enhancement but underdeveloped.  
- Personalization appears applied only to the main predicate and the chosen weight (5) seems arbitrary.  
- Evaluation lacks clear visual analyses—charts would help extract the main message from tables.  
- Should report POS tagging scores in prediction mode to check correlation between tagging drop and parsing drop.  
- Overall contribution over Søgaard (2012) is small.  
- Minor issue: typo on line 207 (“The the main idea” -> “The main idea”).
