- Paper proposes parsing UD treebanks using only POS tags and a modified PageRank that upweights content words.
- Idea is appealing but very close to Søgaard (2012).
- Personalization that boosts the main predicate is a useful addition but underdeveloped.
- Personalization appears applied only to the main predicate and the chosen weight (5) seems arbitrary.
- Improvement over Søgaard (2012) is modest.
- Evaluation lacks helpful figures; tables alone make takeaways hard to extract.
- Should report POS tagging scores in prediction mode to check correlation between tagging and parsing accuracy drops.
- Minor typo: l.207 “The the main idea” → “The main idea.”
