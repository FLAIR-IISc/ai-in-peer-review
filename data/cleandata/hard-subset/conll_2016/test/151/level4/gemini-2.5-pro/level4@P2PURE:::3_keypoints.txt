- Proposes UD parsing using only POS tags with a modified PageRank to weight lexical words over function words.
- Core idea is interesting but closely resembles Søgaard (2012), limiting novelty.
- Personalization factor (boosting the main predicate) is a useful addition but underdeveloped.
- Personalization is applied only to the main predicate and the chosen weight of 5 is arbitrary and unjustified.
- Evaluation tables are often hard to interpret; charts or visualizations would improve clarity.
- Should include POS tagging accuracy in prediction mode to assess how tagging errors affect parsing performance.
- Overall contribution is incremental relative to prior work.
- Minor typo: l.207 "The the main idea" → "The main idea."
