- Proposes parsing UD treebanks using only POS tags and a modified PageRank that weights meaningful words over stop words.
- The idea of weighting meaningful words closely resembles Søgaard (2012), so the novelty is limited.
- Adds a personalization factor to emphasize the main predicate, but the idea is underdeveloped.
- Personalization appears applied only to the main predicate and the chosen weight of 5 seems arbitrary.
- Evaluation is presented only in dense tables; inclusion of charts would improve interpretability.
- Should report POS tagging scores in prediction mode to check if parsing degradation correlates with tagging degradation.
- Overall contribution is incremental over Søgaard (2012) and could use further refinement.
- Minor issue: typo on Line 207 ("The the main idea" → "The main idea").
