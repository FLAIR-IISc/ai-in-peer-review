- Proposes parsing UD treebanks using only POS tags with a modified PageRank that prioritizes meaningful words over stop words.
- Method strongly resembles Søgaard (2012); incremental contribution is limited.
- Adds a personalization factor to emphasize the main predicate, which is a welcome idea.
- Personalization is applied only to the main predicate and the chosen weight (5) seems arbitrary and underexplored.
- Evaluation/analyses need improvement—include charts because tables are sometimes hard to interpret.
- Report POS tagging scores in prediction mode to check if parsing decline correlates with tagging degradation.
- Overall interesting approach but limited novelty over prior work.
- Minor typo: "The the main idea" → "The main idea" (l.207).
