**Summary of the Paper**

The paper presents an unsupervised dependency parsing algorithm, UDP, that uses PageRank and a small set of specific dependency head rules to parse sentences without training. The algorithm is designed to work with the Universal Dependencies (UD) project, which aims to provide a consistent representation of dependency structures across languages. The authors evaluate UDP on 32 languages and compare its performance to a multi-source delexicalized transfer parser.

**Contributions of the Paper**

Based on the paper, I identify the following contributions:

1. **Unsupervised dependency parsing algorithm**: The authors propose a novel unsupervised dependency parsing algorithm that uses PageRank and a small set of specific dependency head rules to parse sentences without training.
2. **UD-compliant parsing**: The algorithm is designed to work with the Universal Dependencies (UD) project, which aims to provide a consistent representation of dependency structures across languages.
3. **Competitive performance**: The authors evaluate UDP on 32 languages and show that it achieves competitive performance compared to a multi-source delexicalized transfer parser.

**Strengths of the Paper**

**Weaknesses of the Paper**

1. **Limited handling of multiword expressions**: The authors acknowledge that UDP does not handle multiword expressions, which can be a limitation of the algorithm.
2. **No handling of coordination**: The authors also acknowledge that UDP does not handle coordination, which can be another limitation of the algorithm.
3. **No partial edge labeling**: The authors note that UDP only handles unlabeled attachment, and that partial edge labeling is not considered.
