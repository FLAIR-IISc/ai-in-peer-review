- Proposes UDP, an unsupervised UD parser using PageRank and hard-coded tagging rules that requires no training data.  
- Uses personalized PageRank to improve main predicate identification.  
- Incorporates UD constraints when building POS and dependency annotations.  
- Competitive with delexicalized transfer systems and robust across domains/languages.  
- Less affected by deteriorating POS tags and more resilient to domain changes than supervised methods.  
- Provides thorough evaluation on multiple languages and domains.  
- Authors release the parser and in-domain annotated test sets.  
- Relies on hard-coded rules that may not generalize to all languages or domains.  
- PageRank may be unsuitable for some sentence types or languages.  
- Does not handle multiword expressions, coordination, or proper names.  
- Evaluation is limited to unlabeled attachment score; lacks partial edge labeling.  
- Paper would benefit from a more detailed error analysis.  
- Asks how authors will extend the parser to handle MWEs, coordination, and proper names.  
- Requests more detailed error analysis including types and frequencies.  
- Questions performance on languages with non-standard word orders or grammatical structures.  
- Requests information on computational resources required.  
- Asks for comparisons to other unsupervised dependency parsing systems in performance and robustness.
