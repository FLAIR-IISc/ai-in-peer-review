- Proposes UDP, a deterministic dependency parser combining POS-based head rules with a biased PageRank to define permitted dependencies, evaluated on 32 languages and competitive with delexicalized transfer parsing.  
- Novel unsupervised deterministic parsing algorithm that rivals supervised parsers.  
- Extensive multilingual evaluation across 32 languages demonstrating robustness and generalizability.  
- Analysis showing the algorithm is less affected by degrading POS tags than a supervised parser.  
- Competitive performance makes UDP a viable option for low-resource languages.  
- Greater resilience to domain shift compared to a supervised parser.  
- Clear, well-written presentation.  
- Limited novelty—method is only slightly different from Søgaard (2012).  
- Method tuning was performed on the test set (head directionality), which is methodologically problematic.  
- Insufficient analysis of head rules relative to the corpus (e.g., frequency of disconnected graphs, component sizes).  
- Question about PageRank tie-breaking and its effect on performance.  
- Request for oracle results per constraint (head rules, directionality, distance) to show upper-bound impacts.  
- Request for more detailed corpus-level analysis of head rules (disconnected graphs, component sizes).  
- Minor editorial suggestions: fix phrasing in 4.1, reorder Table 5 to match Table 7, and fix wording in 6.1.
