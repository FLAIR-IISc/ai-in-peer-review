- Proposes UDP: a deterministic, unsupervised dependency parser using POS-based head rules and biased PageRank, evaluated on 32 languages.
- Core algorithm links each word to its nearest allowed head that preserves tree structure and directionality.

Contributions:
- Novel unsupervised parsing algorithm requiring no training data, competitive with delexicalized transfer.
- Linguistically motivated cross-lingual UD parsing approach claimed to be robust to domain shifts.
- Thorough multilingual evaluation across 32 languages with analyses over POS setups and domains.

Strengths:
- Algorithm is clearly presented and easy to understand.
- Comprehensive, wide-ranging evaluation across languages and POS configurations.
- Results show competitiveness with delexicalized transfer and robustness across domains.

Weaknesses:
- Limited novelty: algorithm only slightly different from Søgaard (2012).
- Head directionality was tuned on the test set, risking overfitting.
- Insufficient analysis of head rules relative to the corpus (e.g., disconnected graphs, component sizes).

Recommended fixes:
- Provide a detailed comparison with Søgaard (2012) to clarify differences and advantages.
- Tune head directionality on training/dev data rather than the test set for generalizability.
- Add detailed analysis of head-rule effects: frequency of disconnected graphs and component size distributions.

Questions for authors:
- How is PageRank tie-breaking handled, and does it affect performance?
- Can you report oracle results for each constraint (head rules, directionality, distance) to show upper bounds?
- Can you provide deeper analysis of head rules on the corpus, including disconnected graph frequency and component sizes?

Minor editorial suggestions:
- Fix phrasing in §4.1 ("we obtain [the] rank").
- Reorder Table 5 columns to match Table 7 (prefer Table 7 order).
- Fix wording in §6.1 ("isolate the [contribution] of both").
