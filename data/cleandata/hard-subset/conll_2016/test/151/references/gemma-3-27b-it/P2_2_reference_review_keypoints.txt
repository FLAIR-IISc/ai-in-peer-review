- Presents UDP, a fully unsupervised UD parser using UD constraints and personalized PageRank, avoiding any training.  
- Dynamically estimates adposition direction and relies on POS tags plus head-dependent rules.  
- Evaluated on 32 languages and compared to a multi-source delexicalized transfer parser.  
- Demonstrates that a reasonably accurate dependency parser can be built without annotated training data.  
- Shows broad multilingual applicability with substantial cross-lingual evaluation.  
- Finds UDP more robust to domain shift than a supervised baseline.  
- Achieves strong UAS scores for an unsupervised system, especially with gold POS tags.  
- Effectively exploits UD constraints (e.g., function words as leaves, single root) to guide parsing.  
- Shows consistent cross-language and cross-domain generalizability and reduced domain sensitivity.  
- Paper is clearly written, well-structured, and includes helpful example parse traces.  
- Authors release the parser and datasets, supporting reproducibility.  
- Many individual components (PageRank, head-dependent rules) are not novel; novelty is in their orchestration.  
- Performance degrades notably with predicted POS tags, limiting low-resource applicability.  
- Head-dependent rules appear somewhat ad-hoc and potentially incomplete; some exclusions (e.g., NOUN -> VERB) lack strong justification.  
- Evaluation focuses on UAS only; additional metrics (relation F1, qualitative error analysis) would give a fuller picture.  
- Requests elaboration on POS-tagging error analysis and mitigation strategies (e.g., rule-based corrections).  
- Asks for explanation of how head-dependent rules were derived and justification for excluding NOUN -> VERB.  
- Requests more detailed error analysis identifying commonly mispredicted dependency relations.  
- Asks for concrete computational cost numbers, especially PageRank runtime per sentence.  
- Suggests adding implementation details (language, libraries) to aid reproducibility.  
- Recommends exploring more sophisticated PageRank personalization strategies.  
- Suggests comparing to state-of-the-art multilingual parsers to better situate the work.  
- Overall assessment: strong, well-evaluated contribution to unsupervised parsing; recommends acceptance with minor revisions.
