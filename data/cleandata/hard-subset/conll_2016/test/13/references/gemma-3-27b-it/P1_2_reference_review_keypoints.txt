- Summary: Proposes a two-stage event linking system that uses CNN-extracted sentential features for mention representation and logistic regression for pairwise coreference, reporting strong results on ACE and ACE++.

- Contribution 1: Shows effective event linking without heavy reliance on external semantic resources by learning sentential representations.

- Contribution 2: Provides thorough empirical evaluation and ablation studies demonstrating the value of sentential features and their interactions.

- Contribution 3: Creates and releases the larger ACE++ dataset for event linking research.

- Strength 1: Strong empirical results competitive with state-of-the-art while avoiding external resources.

- Strength 2: Clear motivation and well-explained model architecture and training, with analysis highlighting cross-sentential importance.

- Strength 3: Comprehensive experimental evaluation including ablations and clustering strategy comparisons.

- Strength 4: Detailed error analysis identifying key challenges like pronoun resolution and annotation issues.

- Strength 5: Public release of ACE++ benefits the community.

- Weakness 1: Core CNN sentence encoder lacks novelty; main novelty is application to event linking without external features.

- Weakness 2: Model performance is sensitive to clustering strategy (BestLink reduces performance) and needs deeper analysis.

- Weakness 3: Missing comparisons to recent neural/transformer-based approaches (e.g., BERT), limiting contextual positioning.

- Weakness 4: Possible dataset bias and annotation inconsistencies in ACE++, with inter-annotator agreement not reported.

- Question 1: Why does BestLink clustering degrade performance and what specific errors does it cause?

- Question 2: Have transformer-based encoders (e.g., BERT) been tried in place of the CNN, and what were the results?

- Question 3: What is the inter-annotator agreement on ACE++ and which event types show annotation disagreements?

- Question 4: Have explicit pronoun resolution methods been considered to address pronoun-related errors?

- Additional: Writing is clear, figure and tables are helpful, and dataset link is provided.

- Overall assessment: Solid, publishable work; recommend acceptance with minor revisions to address noted weaknesses.
