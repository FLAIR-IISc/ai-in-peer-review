- Paper presents a CNN-based approach that learns sentential features for event linking, reducing reliance on external semantic resources.
- System combines a CNN mention representation with a pairwise classifier and standard pairwise features.
- Demonstrates competitive or state-of-the-art performance on ACE and a newly created ACE++ dataset.
- Main contribution: shows CNNs can effectively learn sentential features for event linking without expensive external knowledge bases.
- Main contribution: provides comprehensive experimental evaluation including a larger ACE++ dataset and detailed ablation studies.
- Main contribution: offers insightful error analysis highlighting issues like pronoun resolution and annotator disagreement.
- Strength: strong empirical performance and fair comparisons to prior work.
- Strength: novelty in applying sentential CNN features combined with local trigger features for event linking.
- Strength: detailed ablation studies clearly show the contribution of each feature class.
- Strength: releasing ACE++ is a practical dataset contribution for the community.
- Strength: thoughtful error analysis links errors to specific linguistic phenomena and annotation issues.
- Weakness: core CNN and pairwise classification methodology is incremental rather than fundamentally novel.
- Weakness: paper lacks discussion of computational cost, scalability, and efficiency.
- Weakness: reliance on gold event mentions limits applicability to real-world joint extraction/linking scenarios.
- Weakness: justification for using transitive closure clustering over alternatives is underdeveloped.
- Question: what are the model’s training and inference times and how does cost scale with document/dataset size?
- Question: have the authors considered using entity-level coreference resolution to address pronoun-related errors?
- Question: how could the model be adapted for joint event extraction and linking with imperfect mention detection?
- Question: please provide more details about ACE++ annotation procedures, annotators, and consistency controls.
- Additional comment: paper is well-written, clearly organized, and includes helpful figures and a comprehensive reference list.
