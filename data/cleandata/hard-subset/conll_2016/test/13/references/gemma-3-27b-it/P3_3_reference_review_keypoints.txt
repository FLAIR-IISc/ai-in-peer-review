- Paper proposes using CNN-extracted sentential features for event linking (coreference between event mentions).
- Represents event mentions via surrounding sentences processed by a CNN, combined with pairwise features for coreference decisions.
- Demonstrates competitive/state-of-the-art performance on ACE and a newly created ACE++ dataset.
- Contribution: Shows CNNs can learn effective sentential features for event linking without external knowledge resources.
- Contribution: Introduces and releases the ACE++ dataset, expanding available annotated web documents for event linking.
- Contribution: Provides comprehensive evaluation including ablation studies and detailed error analysis.
- Strength: Strong empirical results with notable gains over methods relying on external resources.
- Strength: Well-motivated design; argues sentential context is necessary when shared arguments/triggers are absent.
- Strength: Clear, detailed methodology and helpful figures explaining architecture and feature engineering.
- Strength: Insightful error analysis highlighting pronoun resolution and annotator disagreement issues.
- Strength: Public release of ACE++ enhances reproducibility and research opportunities.
- Weakness: Limited novelty in core CNN architecture and training—methodology is relatively standard.
- Weakness: Performance sensitive to clustering strategy; BestLink underperforms transitive closure without clear explanation.
- Weakness: Lacks comparison to stronger or more recent neural baselines to contextualize improvements.
- Weakness: Dataset bias toward specific event types (Conflict, Movement, Life) limits demonstrated generalizability.
- Question: Why did BestLink perform worse here compared to Liu et al. (2014) — model or dataset characteristics explaining the difference?
- Question: Can you provide deeper ablation results on convolution filter sizes and pooling strategies?
- Question: Have you considered integrating an explicit pronoun resolution module to address pronoun-related errors?
- Question: How would the model generalize to other domains (biomedical, financial) and what adaptations are needed?
- Question: What are the training/inference computational costs and memory requirements for sentence-level CNN processing?
- Additional comment: Code and ACE++ dataset should be publicly released for reproducibility.
- Additional comment: References section is comprehensive.
- Overall assessment: Solid, well-executed paper with valuable contributions (ACE++ and sentential feature demonstration); recommend acceptance with minor revisions.
