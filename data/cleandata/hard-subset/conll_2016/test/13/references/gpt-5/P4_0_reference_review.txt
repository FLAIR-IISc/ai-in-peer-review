Summary and main contributions
The paper addresses within-document event coreference (“event linking”). It proposes a two-step neural model: (a) a CNN-based encoder that builds sentential representations for individual event mentions by combining word and position embeddings with piecewise max-pooling and trigger-local lexical features; (b) a pairwise scorer that concatenates two mention representations with handcrafted pairwise features (type compatibility, distance, modality, argument overlap) and computes a distributed similarity (square activation) followed by logistic regression. Pairwise scores are converted to clusters via transitive closure. Experiments on ACE 2005 and a larger ACE++ variant show competitive or state-of-the-art performance, without external semantic resources.

As I see them, the primary contributions are:
1) Demonstration that sentential features from CNNs enable strong event coreference performance without external semantic resources, matching or surpassing prior systems that rely on WordNet/FrameNet-like features.
2) An empirical study isolating the impact of feature classes (pairwise, trigger-local, sentential) and clustering strategies (transitive closure vs. BestLink), with analyses of data properties, ablations, and error patterns.
3) A practical pairwise neural architecture for event mention encoding and similarity scoring, including a simple polarity-invariant similarity nonlinearity (square) that empirically outperforms alternatives.

Strengths
- Strong empirical results without external resources: On ACE, the model achieves higher BLANC, MUC, and positive-link F1 than Liu et al. (2014), while being competitive on B-CUBED, despite using no external lexical-semantic features. The result supports the claim that sentential neural features can replace costly domain-specific resources.

- Careful experimental analyses: The paper goes beyond headline metrics. It compares clustering strategies, showing that transitive closure yields better F1 than BestLink in this setup; conducts systematic ablations that quantify contributions of pairwise, trigger-local, and sentential features (sentential features substantially improve both precision and recall); and reports naive baselines (all-singletons, one-cluster, same-type) to contextualize task difficulty.

- Clear modeling and implementation details: The architecture and training are specified with equations, layer shapes, activation choices, and hyperparameter search strategy. The description of convolution, piecewise pooling, and position embeddings is sufficiently detailed for reproduction.

- Data-driven motivation and error analysis: Preliminary corpus analysis justifies modeling choices (many coreferent pairs lack argument overlap and have different triggers; long-distance links are common). The manual error analysis identifies challenging phenomena (pronoun/event anaphora) and annotation inconsistencies that bound achievable performance, adding credibility to the empirical findings.

- Sensible metric choice and discussion: The paper prioritizes BLANC to mitigate class imbalance issues common in within-document coreference and still reports MUC and B-CUBED for completeness, with caveats about their limitations.

Weaknesses
- Comparability and realism of evaluation: The model uses gold-annotated event mentions (and appears to use gold arguments for overlap features), whereas some prior work uses predicted arguments (e.g., via SRL). This reduces comparability and may overestimate performance for end-to-end pipelines. The paper acknowledges split/tooling differences but does not quantify performance with predicted triggers/arguments.

- Limited novelty in architecture: While applying CNN-based sentential features to event coreference is a useful extension, the architectural components (PCNN-style pooling, pairwise features, pairwise classification with transitive closure) are largely standard by now in relation/event tasks. The square activation is a minor twist with limited theoretical grounding beyond empirical observation.

- Dataset availability and reproducibility: ACE++ is a major part of the analysis and yields notable gains, but it is not clearly stated whether it will be released or how it was annotated “analogously.” Lack of access limits verification and reuse. Code availability is not discussed.

- Missing baselines and statistical testing: The paper does not report CEAF metrics (commonly used in coreference), nor significance tests across runs. Stronger neural baselines (e.g., BiLSTM encoders for mentions) are absent; the comparison set on ACE is limited and affected by split/tool differences, making it harder to attribute gains unambiguously to the proposed features.

- Scope limitations: The model only uses each mention’s sentence to build representations. Given that over half of positive links are long-distance, document-level context modeling (beyond distance features) might be required for harder cases (e.g., pronoun-like event anaphora). Clustering is a simple transitive closure over independent pairwise decisions; no global consistency or joint inference with entities is explored.

Questions to authors
- Do you use gold event arguments for the “argument overlap” feature? If yes, can you report results with predicted arguments (or without argument overlap) to assess robustness in realistic settings?

- Will ACE++ (documents, annotations, and splits) and code be released? If not, can you provide enough details to replicate annotation guidelines and preprocessing?

- How does the model perform with predicted triggers/mention boundaries? A small-scale experiment (even on ACE) would help quantify end-to-end impact.

- Can you report CEAF (phi-4) and/or CoNLL F1 to facilitate comparison with coreference literature, and provide significance tests across multiple random seeds?

- What is the effect of replacing the CNN with a BiLSTM encoder, or augmenting with document-level context (e.g., mention-aware attention over surrounding sentences)? Even a brief ablation would clarify whether the gains are specific to CNN sentential features.

Overall assessment
The paper is clear and technically sound, and the empirical evidence that CNN-based sentential features can obviate external semantic resources for event linking is valuable. Novelty is moderate but the thorough analyses and resource-light setup are impactful. Addressing evaluation comparability (gold vs. predicted annotations), reproducibility (ACE++), and broader baselines/metrics would strengthen the work.
