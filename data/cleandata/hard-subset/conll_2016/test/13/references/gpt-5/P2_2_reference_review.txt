Summary and contributions
The paper tackles within-document event coreference (event linking). It proposes a two-step neural model: (a) a CNN-based encoder that builds an event-mention representation from the trigger’s sentence using word and position embeddings with piecewise max pooling, augmented by trigger-local embeddings; (b) a pairwise module that concatenates two mention representations with simple pairwise features (type compatibility, distance, modality/realis agreement, argument overlap), passes them through a square-activation “similarity” layer, and predicts a coreference score via logistic regression. Clustering is obtained by transitive closure over positive links. Experiments on ACE 2005 and a larger ACE++ dataset show competitive or state-of-the-art results without external semantic resources; ablations indicate sentential features are important; an error analysis highlights pronoun cases and annotation inconsistency.

As I see them, the main contributions are:
1) Demonstrating that CNN-derived sentential features for event mentions enable competitive event coreference without external semantic resources, achieving state-of-the-art or on-par performance on ACE.
2) An empirical study dissecting feature contributions and design choices (sentential vs trigger-local vs pairwise features; BestLink vs transitive closure; data size), with clear gains from sentential features and more data.
3) A qualitative error analysis that identifies challenging phenomena (pronominal linking) and annotation ambiguity.

Strengths
- Empirical effectiveness without external semantic resources: The model matches or surpasses prior systems on ACE in BLANC, MUC, and positive-link F1 while avoiding WordNet/FrameNet/etc. This directly supports the portability argument and is valuable given domain coverage issues of external resources.
- Solid ablations and design analysis: Systematic removal of feature classes clarifies that sentential features drive most of the gains; the comparison of clustering strategies (BestLink vs transitive closure) is informative and shows that their classifier already filters low-confidence links, making BestLink harmful to recall. The data-scaling result (ACE to ACE++) plausibly explains the recall boost.
- Clear problem motivation and dataset analysis: The authors quantify that many coreferent pairs lack shared arguments and often have different triggers, and that many positive links are far apart. This justifies sentence-based semantic modeling, not just argument overlap.
- Thoughtful metric discussion: Reporting BLANC (with its balance of positive/negative links) alongside MUC/B-CUBED/positive links is appropriate for the class imbalance in ACE, and the paper discusses limitations of the latter metrics.
- Error analysis with actionable insights: Identifies pronoun-based linking and boundary/ambiguous cases, suggesting future directions (discourse modeling, coref with events-as-arguments).

Weaknesses
- Fairness and clarity of preprocessing assumptions: The paper states all systems use gold triggers, but is ambiguous about argument features. “Overlap in arguments” suggests using gold event arguments, while some compared systems use predicted SRL instead. If gold arguments are used, this inflates performance and weakens the claim of resource-lightness and the fairness of comparisons. Similarly, “modality/realis” sourcing is unclear (gold vs predicted). This needs explicit clarification and, ideally, results under matched settings.
- Incremental novelty: The architecture largely combines established components (CNN sentence encoder with position embeddings and piecewise pooling; shallow pairwise MLP; simple clustering). The main novelty is the application to event coreference and the finding that sentential features suffice without external semantics. Methodologically, this is a modest, though practically useful, step.
- Limited scope and reproducibility: Experiments are confined to within-document, English ACE. The new ACE++ is not publicly available, limiting verification and reuse; code availability is not mentioned. No statistical significance testing is reported, and differences across prior works’ data splits further muddy SOTA claims.
- Sentence-local mention representations for a cross-sentential task: While justified as capturing context around each mention, the model does not encode cross-sentence discourse beyond coarse pairwise features (e.g., distance). Given many positive links are >5 sentences apart, lack of document-level context, entity coreference integration, or global inference likely caps performance. The error analysis corroborates this (pronoun linking failures).
- Some missing details: Exact definitions for pairwise features (e.g., “bagged distance”), hyperparameters (referenced table), class imbalance handling, negative sampling, and training dynamics are under-specified, hindering reproducibility.

Questions to authors
- Do you use gold event arguments to compute “argument overlap”? If not, what system provides arguments and with what accuracy? Please report performance without the argument-overlap feature to assess reliance on gold information.
- How are “event type” and “modality/realis” obtained (gold vs predicted)? If predicted, specify the tool and accuracy; if gold, please provide matched comparisons or an end-to-end setting.
- Can you provide results with the same ACE split as a key prior work (e.g., Liu et al., 2014), or at least significance tests on your split?
- How do you handle severe class imbalance in training (all anaphor-antecedent pairs, mostly negative)? Any reweighting, sampling, or threshold tuning?
- Will ACE++ (or a subset/annotation guidelines) and code be released? If not, can you detail its composition (genres, sizes) and annotation quality control?

Overall evaluation
This is a clear, well-executed empirical paper showing that CNN-based sentential features enable competitive event coreference without external semantic resources. The methodological novelty is incremental, and there are caveats in fairness and reproducibility that should be clarified. Nonetheless, the experimental evidence, ablations, and analysis are strong and likely to interest the community. I lean toward weak accept, contingent on clarifying the gold/predicted status of arguments and attributes and, if possible, adding a matched or end-to-end setting.
