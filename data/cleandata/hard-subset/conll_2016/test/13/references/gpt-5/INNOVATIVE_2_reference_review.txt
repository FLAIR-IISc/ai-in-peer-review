Summary and main contributions
This paper addresses within-document event coreference (event linking). It proposes a two-step neural model: (a) a CNN-based sentence encoder with position embeddings and piecewise max-pooling to build an event-mention representation that combines sentential and trigger-local features; (b) a pairwise scorer that augments these mention representations with standard pairwise features and computes a distributed similarity (square nonlinearity), followed by logistic regression. Clustering is done via transitive closure of positive links. The model is evaluated on ACE 2005 and an extended ACE++ dataset, claims state-of-the-art performance without external semantic resources, and includes ablations and error analysis.

As I see them, the main contributions are:
1) A resource-light, sentence-representation approach to event linking that avoids WordNet/FrameNet and still achieves competitive/state-of-the-art performance on ACE.
2) A practical, empirically grounded analysis: feature ablations showing the importance of sentential features for linking, comparison of clustering strategies, and an error analysis highlighting discourse/pronominal challenges.
3) An expanded dataset (ACE++), used to show that more training data improves recall; document splits for ACE are released.

Strengths
- Novel application of sentential CNN features to event coreference. Prior neural work largely targeted event/relation detection and argument extraction; using sentence-level CNN representations to support event linking is an appealing and timely idea. The model demonstrates that strong linking performance is possible without external semantic lexicons, which is valuable for portability to new domains.
- Competitive results with fewer external dependencies. On ACE, the model matches or exceeds prior state-of-the-art under several metrics (notably BLANC F1) while foregoing WordNet/FrameNet/SRL-based features used by competitors. This supports the paper’s thesis that learned sentence representations can substitute for handcrafted semantic resources in linking.
- Careful ablation and analysis. The feature ablations cleanly separate the contributions of pairwise, trigger-local, and sentential features; results show sentential features are critical for recall and overall F1. The comparison of transitive closure vs. BestLink is informative and contradicts previous findings, with a reasonable explanation (classifier already prunes low-confidence links). The error analysis highlights real discourse phenomena (pronouns) and annotation inconsistencies, adding credibility and future directions.
- Solid empirical setup and transparency on caveats. The paper acknowledges split/preprocessing differences across prior work, motivates BLANC as a balanced metric, and provides additional metrics for completeness. Hyperparameter search is reported, and implementation details are clear.
- Practical significance. By minimizing reliance on scarce external resources, the approach lowers the barrier to deploying event linking in under-resourced domains. The finding that adding more data (ACE++) primarily boosts recall is also practically useful.

Weaknesses
- Heavy reliance on gold event mentions and types (and likely arguments). The pairwise features include type compatibility, argument overlap, and modality agreement. If these are gold, the task is considerably simplified and may not reflect realistic end-to-end performance. While using gold triggers is common in this subtask, the paper’s core claim of portability would be much stronger with experiments using predicted triggers/types/arguments (or at least a degradation study).
- Modest methodological novelty. The architecture repurposes known components: PCNN-style sentence encoding with position embeddings, pairwise feature concatenation, and a simple classifier; clustering via transitive closure. The square activation for similarity is interesting but a small tweak. The novelty primarily lies in applying CNN-based sentential features to event linking, rather than in a fundamentally new coreference formulation.
- Comparability and resource contribution concerns. Results depend on differing ACE splits and preprocessing across systems; the paper notes this but it still clouds SOTA claims. The ACE++ extension is not clearly described (annotation process, quality control, licensing) and seems not publicly released, limiting its community impact and reproducibility.
- Limited exploration of global inference. The model does pairwise scoring plus simple clustering. It does not leverage global constraints or jointly model entities and events, which has been argued to help in prior work. The error analysis suggests discourse-level issues (pronouns) where more global modeling could help.
- Metric coverage. While BLANC is justified and important, it would be useful to include CEAF metrics for a more holistic picture, given their prevalence in coreference evaluation.

Questions to authors
- What supervision was used for pairwise features? Are event types, modalities, and arguments gold-annotated or predicted? If gold, can you report performance with automatically extracted triggers/types/arguments (even with off-the-shelf ACE event extractors) to quantify robustness?
- For ACE++: who performed the annotations, what guidelines were used, and what inter-annotator agreement was achieved? Can you release the resource or at least the annotations for research (subject to licensing), and provide a detailed description?
- How sensitive are results to the choice of sentence encoder? Have you compared the CNN to a BiLSTM or self-attentive encoder for the sentential features? If not, can you provide preliminary comparisons or rationale?
- The square activation for similarity: did you compare to absolute value, ReLU on symmetric inputs (e.g., differences/products), or elementwise products between mention embeddings? Please share quantitative comparisons.
- Can you report end-to-end document-level performance including upstream event detection errors, or at least a simulated noise study (e.g., perturbing triggers/types) to justify the portability claim?

Overall recommendation
Borderline accept leaning accept. The paper’s main value is conceptual and practical: demonstrating that sentence-level neural representations can effectively support event coreference without external semantic resources. Despite modest architectural novelty and some evaluation limitations (gold inputs, non-public ACE++), the work opens a useful direction and achieves strong results with solid analysis. Strengthening with predicted-input experiments and more details on ACE++ would further solidify its impact.
