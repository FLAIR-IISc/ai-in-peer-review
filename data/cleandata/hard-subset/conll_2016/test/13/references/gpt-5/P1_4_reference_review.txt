Summary and claimed contributions
The paper tackles within-document event coreference resolution (“event linking”). It proposes a two-stage neural model: (a) a CNN-based encoder that builds an event-mention representation from the entire sentence using word and position embeddings with piecewise max pooling around the trigger, augmented with local trigger embeddings; (b) a pairwise scorer that combines two mention embeddings with simple pairwise features (type compatibility, distance, modality/realis match, argument overlap) and outputs a coreference score. Clustering is done via transitive closure over positive pairwise links. The model avoids external lexical-semantic resources (e.g., WordNet/FrameNet). Experiments on ACE 2005 and an expanded ACE++ dataset show performance on par with or better than prior work and ablations indicate sentential features are key. The paper also compares clustering strategies and provides an error analysis.

As I see them, the main contributions (strongest first):
1) Demonstration that sentential CNN features yield competitive, resource-lean event coreference performance, matching/pruning back dependence on external semantic resources.
2) A careful empirical study: ablations of feature classes, comparison of clustering strategies (BestLink vs transitive closure), and an error analysis highlighting unresolved discourse phenomena (e.g., pronominal event references).
3) A larger training set (ACE++) and analysis showing improved recall with more data.

Strengths
- Practical, resource-light modeling: The approach achieves state-of-the-art or comparable results without external semantic knowledge bases. This directly addresses domain portability concerns and reduces engineering overhead, a clear practical advance for event coref.

- Clear motivation grounded in data: The authors’ analysis (argument sharing is rare; many positives have different triggers; long-distance links dominate) compellingly motivates sentential modeling and not relying solely on intrinsic properties.

- Solid empirical evidence and ablations: The controlled ablations isolating pairwise, trigger-local, and sentential components convincingly show that sentential features substantially boost both precision and recall. The BestLink comparison is informative and counters prior claims, suggesting the classifier already suppresses low-confidence links.

- Sensible evaluation choices and transparency: Using BLANC (appropriate under heavy class imbalance) alongside standard coref metrics is well-justified. The paper candidly discusses evaluation caveats (different train/test splits, preprocessing differences), which helps interpret results.

- Readable and technically clear: The model description is accessible, with equations, architectural choices, and training details specified (e.g., position embeddings, piecewise pooling, similarity-layer motivation).

Weaknesses
- Reliance on gold annotations in pairwise features: Several pairwise features (event type agreement, realis/modality, argument overlap) appear to use gold labels/arguments. This greatly simplifies the task and can inflate performance relative to practical settings. Comparisons are also uneven if baselines differ in gold availability. Please clarify exactly which labels are gold and report results with predicted types/arguments (or without these features).

- Limited novelty in modeling: CNN sentence encoders with position embeddings and pairwise classification plus transitive closure are well-established in relation/event extraction. The contribution is primarily an application and engineering consolidation rather than a fundamentally new algorithm or learning objective (no global inference, no joint learning with extraction, no structured loss).

- ACE++ dataset opacity and reproducibility: The paper relies heavily on ACE++ for tuning and main insights but provides limited details on its annotation process, quality control, inter-annotator agreement, or plans for release. This hinders reproducibility and independent verification. Without public availability, the strongest empirical claims (ablation gains and recall improvements) cannot be replicated.

- Comparability and statistical rigor: Results on ACE use non-identical splits and preprocessing across systems. No statistical significance tests or variance across random seeds are reported, which is important given small datasets and neural training variability.

- Scope limitations: The model is evaluated with gold triggers; robustness with predicted triggers (and arguments) is unknown. Also, evaluation is confined to ACE/ACE++; no results on widely used event coref corpora (e.g., ECB+ within-document) to substantiate generality. Long-distance cases are common; an analysis by sentence distance or document position would further validate the approach.

Questions to authors (to be addressed in rebuttal)
- Which pairwise features use gold supervision? Specifically: event type, realis/modality, and argument overlap—are these gold? If so, please provide results with predicted types/arguments (or ablations removing these features) to assess the benefit purely attributable to the neural sentence encoder.

- How was ACE++ annotated? Who annotated it, using what guidelines, and what inter-annotator agreement was achieved on coref links, event types, and arguments? Will you release ACE++ (or at least de-identified mention pairs with labels) to enable replication?

- Hyperparameter tuning: You state hyperparameters were optimized on ACE++ valid and then used for ACE. Could tuning on ACE++ bias ACE results? Please report a setting tuned only on ACE dev or show that ACE performance is robust to this choice.

- Multiple triggers per sentence: Please clarify how position embeddings and piecewise pooling are computed when multiple triggers occur in a sentence. Is the relative position indexing re-centered per trigger, and are left/right segments defined per candidate mention?

- Variance and significance: How stable are results across random seeds? Can you report mean±std over multiple runs and significance tests vs. the strongest baseline?

Overall, this is a well-executed application of CNN-based sentential features to event coreference with clear empirical benefits and useful analysis. Addressing the reliance on gold annotations, clarifying ACE++ and reproducibility, and adding robustness experiments would substantially strengthen the paper. I lean toward borderline accept if the concerns about gold supervision and dataset transparency are convincingly addressed.
