- Contribution: Two-step event coreference model using CNN sentence-level mention representations + lightweight pairwise similarity, claiming SOTA on ACE without external semantic resources.
- Contribution: Empirical study showing sentential features materially improve performance, more data (ACE++) mainly boosts recall, and transitive-closure clustering outperforms BestLink; includes ablations and baselines.
- Contribution: Error analysis highlighting common failure modes (pronominal/gapped mentions) and ACE annotation ambiguities.

- Strength: Strong empirical results matching/exceeding prior resource-heavy systems using only sentence-level neural features and simple pairwise module.
- Strength: Careful experimental analysis of dataset properties, clustering strategies, data scaling, and multiple ablations.
- Strength: Clear, simple, and reproducible model architecture with motivated design choices and ablation-backed decisions.
- Strength: Qualitative error analysis that highlights dataset ambiguity and difficult cross-sentential phenomena.
- Strength: Practical approach avoiding curated semantic lexicons and relying on commonly available inputs.

- Weakness: Limited novelty—essentially a standard mention-pair model with incremental tweaks and no global/joint inference or discourse modeling.
- Weakness: Evaluation realism/fairness concerns due to use of gold triggers/possibly gold arguments and missing CEAF/CoNLL metrics, making SOTA claim hard to substantiate.
- Weakness: ACE++ extension lacks protocol, IAA, genre mix, availability, and license details, harming reproducibility and interpretability of gains.
- Weakness: Missing stronger contemporary baselines (pure feature-based, BiLSTM/transformer encoders, simple embedding cosine) and no significance testing for modest deltas.
- Weakness: Model does not address major error sources (pronominal/abstract references, discourse phenomena), limiting practical advance.
- Weakness: Code release not mentioned.

- Question: Do you use gold event arguments/types for argument overlap, type compatibility, and realis match; if not, what upstream systems/accuracies were used?
- Question: Will ACE++ (documents, annotations, splits) be released, and what were the annotation protocol, annotator training, IAA, and genre composition?
- Question: Please report CEAF_e and CoNLL average (MUC, B3, CEAF) in addition to BLANC.
- Question: What is performance without all hand-crafted pairwise features (CNN-only) and conversely with pairwise features but no CNN encoder?
- Question: Can you provide variance across random seeds and significance tests for key comparisons (ACE SOTA, clustering, ACE vs. ACE++)?
- Question: Runtime and complexity details: parameter count, average inference time per document, and any pruning strategies for long documents.

- Overall: Solid empirical paper showing sentential neural features enable competitive event coreference without external lexicons, but novelty is modest and evaluation/reproducibility issues should be addressed.
