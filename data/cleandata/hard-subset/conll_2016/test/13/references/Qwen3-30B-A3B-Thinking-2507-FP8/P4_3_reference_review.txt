

# Review of "Event Linking with Sentential Features from Convolutional Neural Networks"

## Summary of the Paper and Main Contributions

This paper addresses event coreference resolution (event linking), the task of resolving coreferences between recognized event mentions within a document. The authors argue that previous approaches either rely on external semantic features (like WordNet or FrameNet) with limited cross-domain availability or show low performance. They propose a novel approach that uses sentential features derived from convolutional neural networks to represent event mentions without requiring external semantic resources.

The main contributions of this work are:

1. **Demonstration that sentential features from CNNs can effectively replace external semantic resources for event linking**: The authors show that their CNN-based approach achieves state-of-the-art performance without relying on external knowledge sources like WordNet or FrameNet, which previous approaches required.

2. **Systematic analysis of feature contributions for event linking**: The authors conduct a thorough ablation study showing that the combination of pairwise features, trigger-local lexical features, and sentential features all contribute to model performance, with sentential features providing the most significant improvement.

3. **Empirical validation of the importance of sentence context for event linking**: The authors provide evidence (through analysis of the ACE 2005 corpus) that a significant portion of coreferential event mentions share no arguments and have different trigger words, justifying their approach of using full sentence context rather than relying solely on event arguments.

## Strengths

1. **Strong motivation with empirical justification**: The paper provides compelling evidence for the importance of event linking by analyzing the ACE 2005 corpus, showing that approximately 23% of event mentions are incomplete on the argument level. This analysis effectively establishes the need for better event linking approaches.

2. **Novel application of CNNs for event linking**: The paper makes a significant contribution by applying sentential features from CNNs to event linking, a task that has not been previously addressed with this approach. The authors demonstrate that these features can replace external semantic resources while maintaining state-of-the-art performance, which addresses a key limitation of prior work.

3. **Comprehensive experimental evaluation**: The authors conduct a thorough evaluation including comparisons to state-of-the-art systems, ablation studies of different feature classes, analysis of dataset size effects, and a detailed error analysis. The ablation study particularly demonstrates the value of each component of their model.

4. **Clear technical contribution to feature engineering**: The paper provides a well-structured approach to generating sentential features using CNNs, with careful attention to the architecture details (including relative position embeddings, convolution, and pooling). This technical contribution is clearly presented and replicable.

5. **Useful error analysis**: The error analysis identifies specific challenges for the model (particularly pronoun resolution) and shows that many errors are due to ambiguous cases even for human annotators, providing valuable insights for future work.

## Weaknesses

1. **Limited novelty in model architecture**: While the application to event linking is novel, the overall model architecture (CNN for representation learning followed by a classifier) is not particularly novel in the context of neural network approaches for coreference resolution. Similar architectures have been used in relation extraction and other coreference tasks, and the paper doesn't sufficiently distinguish its approach from these prior works.

2. **Comparison to recent neural approaches is insufficient**: The paper compares to state-of-the-art systems from 2014 (Liu et al.) but doesn't compare to more recent neural approaches to event linking or coreference resolution. For example, the paper doesn't compare to models using BERT or other transformer-based architectures that have become dominant in NLP in recent years, which would provide a more meaningful benchmark.

3. **ACE++ dataset construction lacks detail**: The paper introduces a new dataset (ACE++) by adding web documents, but provides minimal detail about how these documents were annotated or how they compare to the ACE 2005 annotations. Without this information, it's difficult to assess whether the performance improvements on ACE++ are due to the model or to better data.

## Questions to Authors

1. The paper states that 55.42% of coreferential event mentions are more than five sentences apart. How does this affect the practical applicability of the model in real-world scenarios where event mentions might be even further apart, potentially across multiple documents?

2. The authors claim to achieve state-of-the-art performance, but the comparison is limited to systems from 2014. Could they provide a comparison to more recent neural approaches (e.g., using BERT or other transformer models) that have been proposed for coreference resolution in the last few years?

3. The ACE++ dataset was created by adding web documents. Could the authors provide more details about the annotation process for these additional documents, including how they ensured consistency with the ACE 2005 annotation guidelines?

4. In the error analysis, the authors identify pronoun resolution as a significant challenge. How might the model be adapted to better handle pronoun-based coreference in event linking, particularly in cases where the pronoun refers to an event rather than an entity?
