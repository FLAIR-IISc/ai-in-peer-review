- Summary: Proposes CNN-based sentential features for event linking to avoid external semantic resources, evaluated on ACE 2005 and a new ACE++ dataset with reported state-of-the-art results.

- Contribution 1: Novel CNN-based sentential feature approach that eliminates dependence on WordNet/FrameNet, improving domain generalizability.

- Contribution 2: Empirical finding that ~23% of event mentions are argument-incomplete, motivating the need for event linking.

- Contribution 3: Achieves state-of-the-art performance on ACE 2005 without external semantic features, outperforming prior methods like Liu et al. (2014).

- Strength 1: Well-motivated, generalizable approach that avoids domain-specific external resources.

- Strength 2: Strong empirical results with thorough ablation showing sentential features are critical to performance.

- Strength 3: Comprehensive ablation and clustering analysis, showing BestLink clustering harms performance for this model.

- Weakness 1: Lacks comparison with more recent neural approaches to event linking and relation extraction.

- Weakness 2: Limited analysis of why sentential features work and no inspection of CNN-learned feature interpretations.

- Weakness 3: Insufficient details about ACE++ construction, annotation process, and comparability to ACE 2005.

- Weakness 4: Superficial error analysis that only lists error types (pronoun resolution, boundary cases) without deeper categorization or remedies.

- Weakness 5: No discussion of computational efficiency or feasibility for large-scale/real-world deployment.

- Question 1: Please compare against recent neural baselines (e.g., Chen et al., 2015; Zeng et al., 2015) to validate state-of-the-art claims.

- Question 2: Provide details on ACE++ annotation procedure, domain coverage, and annotation quality relative to ACE 2005.

- Question 3: How do sentential features handle long-distance coreference (55.42% > five sentences apart), and performance on those pairs specifically?

- Additional comment 1: Provide examples of coreferential pairs that share no arguments (~50.97%) to illustrate the phenomenon.

- Additional comment 2: Clarify how the model handles long-distance cases beyond noting the statistic.

- Additional comment 3: Report exact metric numbers for Liu et al. (2014) comparisons for clearer context.

- Additional comment 4: Justify or provide evidence that using a square activation is preferable for measuring similarity.

- Recommendation: Accept — novel, impactful work demonstrating that high-performance event linking can be achieved without external semantic resources despite some evaluation and analysis gaps.
