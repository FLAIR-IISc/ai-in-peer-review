

# Peer Review: "Event Linking with Sentential Features from Convolutional Neural Networks"

## Summary of the Paper and Main Contributions

This paper addresses the task of event linking (event coreference resolution), which aims to resolve coreferences between recognized event mentions within a document. The authors demonstrate that approximately 23% of event mentions in ACE 2005 are incomplete with respect to information in other mentions of the same event instance, highlighting the importance of this task. They propose a model that uses convolutional neural networks to generate sentential features for event mentions, eliminating the need for external semantic features like WordNet or FrameNet. The main contributions of this work are:

1. A novel approach to event linking using sentential features from convolutional neural networks, achieving state-of-the-art performance without relying on external semantic resources.
2. Comprehensive analysis of the impact of different feature classes (pairwise, trigger-local lexical, and sentential) on event linking performance.
3. Creation of a new dataset (ACE++) by adding 1351 web documents to ACE 2005, which demonstrates the benefits of increased training data for event linking.

## Strengths

1. **Novel use of sentential features without external resources**: The authors make a strong case for the importance of sentential features, showing that 50.97% of coreferential event mention pairs share no arguments and 47.29% have different trigger words. Their model effectively captures contextual semantics through CNNs without relying on expensive external resources, which is a significant contribution as it removes a major barrier to applying event linking systems in new domains.

2. **State-of-the-art performance without external features**: The model achieves superior results on ACE 2005 (BLANC F1: 73.31% vs. Liu et al.'s 70.43%) and ACE++ (76.90%) without using the 105 semantic features employed by the previous state-of-the-art system. This empirical validation is compelling, demonstrating that neural sentential features can replace traditional semantic resources.

3. **Thorough ablation study**: The authors conduct a comprehensive analysis of feature classes, showing that pairwise features alone yield poor performance (56.69% BLANC F1), while adding trigger-local lexical features (64.12%) and then sentential features (76.90%) progressively improves results. This provides valuable insights into what aspects of event mentions are most important for coreference resolution.

4. **Creation of ACE++ dataset**: The authors' extension of ACE 2005 with 1351 web documents creates a more substantial resource for event linking research. Their experiments show that increased training data significantly improves recall (from 75.16% to 83.21%), demonstrating the benefits of larger datasets for this task.

## Weaknesses

1. **Lack of comparison with recent neural approaches**: The paper compares with previous state-of-the-art systems but fails to address more recent neural approaches for event coreference resolution. For example, the paper mentions "Other approaches to the upstream task of event extraction" but doesn't compare with contemporary neural models that might also use sentential features. This omission weakens the claim that their specific neural architecture is novel and superior.

2. **Insufficient error analysis**: The error analysis (100 false positives and 100 false negatives) is too brief to draw meaningful conclusions. The authors note that 17 false negatives correspond to cases where annotators made different decisions, but don't provide systematic analysis of these cases or their prevalence. A more detailed analysis would help identify specific model weaknesses and guide future improvements.

3. **Limited discussion of computational complexity**: The paper doesn't address the computational cost of their approach compared to previous methods. Given that they're using convolutional neural networks, it's important to know whether their solution is significantly more expensive than previous methods, especially since they claim to eliminate the need for external semantic features (which are often expensive to process).

4. **Unclear justification for clustering strategy**: While the authors claim their model "already weeds out many low confidence links," this assertion isn't well-supported. The paper states that BestLink performs worse than transitive closure (71.09% vs. 76.90% F1), but doesn't provide evidence that their model's confidence scores are inherently better than those from other systems.

5. **Lack of comparison with simpler baselines**: The paper includes naive baselines (all singletons, one instance, same type) but doesn't compare with more sophisticated baselines that might use simpler neural approaches or other feature combinations. For example, it would be valuable to see how a model using only sentence embeddings (e.g., BERT) would perform.

## Questions to Authors

1. How does your CNN-based approach compare to recent neural models for event coreference resolution that might also use sentential features (e.g., using BERT or other transformer-based architectures)?

2. Could you provide more details about the 17 cases where false negatives correspond to annotator disagreements? How common are these ambiguous cases across the dataset?

3. The paper claims that "our model already weeds out many low confidence links," but this isn't thoroughly justified. Could you provide evidence such as confidence score distributions for different types of event mention pairs?

4. The authors mention that "the use of the square function as the network's non-linearity is backed by the intuition that for measuring similarity, an invariance under polarity changes is desirable." Could you elaborate on this intuition and provide evidence that this choice was indeed better than other activation functions?

5. The paper states that 55.42% of coreferential event mentions are more than five sentences apart. How does this affect the model's ability to capture long-range dependencies, and what architectural changes might help address this limitation?

## Overall Recommendation

This paper presents a well-executed approach to event linking that achieves state-of-the-art results without relying on external semantic resources. The use of sentential features from convolutional neural networks is a promising direction for event coreference resolution. While the paper has several strengths, addressing the weaknesses noted above would significantly strengthen the contribution. I recommend acceptance with minor revisions.
