- Paper tackles event coreference (event linking) using CNN-based sentential features without external semantic resources.
- Main contributions: neural model for event linking without external resources, analysis showing sentential features are critical, and creation of ACE++ dataset (+1351 web docs).
- Strength: Demonstrates practical, domain-agnostic performance comparable or better than prior systems (BLANC F1 73.31% vs. 70.43%).
- Strength: Thorough feature analysis showing all feature classes help and removing sentential features causes the largest drop (76.90% → 64.12%).
- Strength: Strong empirical results on ACE 2005 and scalability shown with ACE++ (BLANC F1 improves with more data).
- Weakness: Model architecture offers limited novelty, closely following prior CNN relation-extraction work without clearly motivating event-linking-specific adaptations.
- Weakness: Inadequate comparison to recent neural event coreference approaches (e.g., Zhang et al. 2017, Chen et al. 2018), weakening the state-of-the-art claim.
- Weakness: Potential data leakage in ACE++—unclear if added web documents overlap domain/content with ACE 2005.
- Question: How does this approach differ architecturally and empirically from Zeng et al. (2014) and Nguyen & Grishman (2015) for relation extraction?
- Question: Please clarify the performance gain from sentential features versus trigger-local features (specific comparison referenced in Table 5).
- Question: Explain the observed precision-recall tradeoff (high precision, lower recall) relative to the high negative-link ratio in ACE.
- Overall: Valuable, well-written contribution with strong results and analysis, but needs clarifications on novelty, comparisons to recent work, and ACE++ data provenance; recommend acceptance with minor revisions.
