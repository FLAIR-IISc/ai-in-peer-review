

# CONLL Review

## Summary
This paper proposes a novel approach to event linking (event coreference resolution) that uses sentential features generated by convolutional neural networks (CNNs), eliminating the need for external semantic resources like WordNet or FrameNet. The authors design a two-step model: first, a CNN generates representations for individual event mentions using sentential features; second, a neural network makes coreference decisions based on these representations plus pairwise features. The model achieves state-of-the-art performance on ACE 2005 without relying on external semantic features.

## Primary Contributions (as I see them)
1. **Novel application of sentential features for event coreference resolution**: The key contribution is applying a new feature class (sentential features from CNNs) to event coreference resolution, which has not been explored before. Previous work used these features for event extraction, not for coreference resolution.
2. **Demonstration that external semantic resources are unnecessary for SOTA performance**: The authors show that their model achieves SOTA results without relying on external semantic features, challenging a fundamental assumption in the field.
3. **Insightful error analysis identifying specific challenges**: The analysis reveals pronoun resolution as a fundamental challenge that affects all current approaches, providing valuable direction for future work.

## Strengths

1. **Truly novel conceptual approach**: The core contribution—applying sentential features from CNNs to event coreference resolution rather than relying on external semantic resources—is genuinely novel. Previous work has used these features for event extraction, but not for coreference resolution. This represents a significant conceptual shift that challenges the field's assumptions about necessary resources for good performance.

2. **Strong empirical validation of the approach**: The model achieves SOTA results on ACE 2005 without external semantic features, which is a compelling demonstration that the authors' conceptual shift is not just theoretically interesting but practically effective. The ablation studies (Table 5) clearly show that each feature class contributes to performance, and the performance on ACE++ (with 1950 documents) shows the model scales well with more data.

3. **Important practical implications**: By showing that external semantic resources aren't necessary, the authors make event linking more accessible for new domains where these resources aren't available. This has significant practical implications for deploying event linking systems across different domains without requiring extensive resource development.

4. **Valuable error analysis**: The error analysis identifies pronoun resolution as a fundamental challenge that affects all current approaches, not just the authors' model. This provides crucial direction for future work and demonstrates the authors' deep understanding of the problem space.

5. **Clear demonstration of feature importance**: The ablation studies (Table 5) effectively show how each feature class contributes to performance, providing valuable insights into what aspects of the model are most important.

## Weaknesses

1. **Limited novelty compared to the feature class itself**: While the application to event coreference is novel, the use of sentential features from CNNs has been explored in other NLP tasks (like relation extraction). The paper doesn't sufficiently distinguish how this application to event coreference is fundamentally different from prior work using similar techniques.

2. **Comparison to recent neural approaches is inadequate**: The paper compares primarily to older systems (Liu et al., 2014; Bejan and Harabagiu, 2010), but doesn't compare to more recent neural approaches for event coreference resolution that might have already used similar techniques. This weakens the SOTA claim.

3. **Lack of justification for CNN choice**: The paper uses CNNs for sentential features but doesn't explain why CNNs were chosen over other architectures (RNNs, Transformers) that might be better suited for this task. This is a significant omission for a neural network-based approach.

4. **ACE++ dataset creation details are insufficient**: The paper creates ACE++ by adding 1351 web documents, but doesn't provide enough detail about the annotation process for these additional documents. How were they annotated to ensure consistency with ACE 2005? This affects the validity of the larger dataset.

5. **The "state-of-the-art" claim needs more context**: The paper claims SOTA results, but without comparing to recent work that might have already achieved similar or better performance using similar techniques, this claim is somewhat questionable.

## Questions to Authors

1. The paper claims that previous approaches relied on external semantic resources, but doesn't explain why the same approach (using sentential features from neural networks) wasn't tried for coreference resolution before. Could you clarify why this wasn't attempted previously?

2. You mention that your model outperforms Liu et al. (2014), but doesn't compare to more recent neural approaches for event coreference resolution. Could you provide a comparison to at least one recent neural approach that might have used similar techniques?

3. Why did you choose CNNs over other neural architectures (like RNNs or Transformers) for generating sentential features? What specific advantages did CNNs offer for this particular task?

## Recommendation

I recommend **acceptance**.

This paper introduces a genuinely novel approach to event coreference resolution that challenges a fundamental assumption in the field (that external semantic resources are necessary for good performance). The core contribution is bold and represents a significant new direction in the field. While there are some methodological weaknesses (particularly around the comparison to recent work and justification for CNN choice), these are outweighed by the novelty and significance of the conceptual contribution. As a reviewer who prioritizes novel ideas over methodological perfection, I believe this paper makes a valuable contribution to the field that should be accepted.
