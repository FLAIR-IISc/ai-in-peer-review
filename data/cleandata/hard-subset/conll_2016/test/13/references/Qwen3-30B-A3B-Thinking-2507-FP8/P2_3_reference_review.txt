

# Review for "Event Linking with Sentential Features from Convolutional Neural Networks"

## Summary
This paper proposes a neural network-based approach for event linking (coreference resolution) that eliminates the need for external semantic resources like WordNet or FrameNet. The authors design a two-step system: first, a CNN processes the full sentence context around event triggers to generate event mention representations using sentential features; second, a coreference classifier combines these representations with pairwise features to determine if event mentions refer to the same real-world event. The model achieves state-of-the-art performance on ACE 2005 and ACE++ datasets without relying on external semantic features.

## Main Contributions
1. A novel neural approach for event linking that uses sentential features from CNNs without external semantic resources, eliminating domain-specific dependency
2. Comprehensive empirical validation showing state-of-the-art performance on event linking tasks
3. Thorough analysis of feature contributions and error patterns, including identification of pronoun resolution as a key challenge

## Strengths

1. **Elimination of dependency on external semantic resources:** This is the paper's strongest contribution. Previous state-of-the-art systems (Liu et al., 2014) relied on a "rich set of 105 semantic features" tied to resources like WordNet, which are domain-specific and costly to create. The authors demonstrate that their CNN-based approach achieves comparable performance without these resources, making the system more broadly applicable across domains. This is a significant methodological advancement that addresses a key practical limitation in event linking.

2. **Strong empirical results with thorough ablation studies:** The paper provides comprehensive evaluation on ACE 2005 and ACE++, showing clear improvements over previous approaches (73.31% BLANC F1 vs. 70.43% for Liu et al.). The ablation studies in Table 5 are particularly valuable, demonstrating the specific contribution of each feature class (sentential, trigger-local lexical, pairwise) to model performance. The authors show that removing any feature class degrades performance, confirming their design choices.

3. **Insightful error analysis:** The authors go beyond standard performance metrics to analyze 100 false positives and 100 false negatives. They identify two key challenges: pronoun resolution (e.g., "It" referring to a previous event) and boundary cases where even human annotators disagree. This analysis provides valuable context for interpreting the results and suggests concrete directions for future work, strengthening the paper's contribution beyond just the model itself.

## Weaknesses

1. **Limited comparison to recent neural approaches for event linking:** While the paper cites neural approaches for relation extraction, it doesn't directly compare to recent neural models specifically designed for event coreference resolution. The authors claim "state-of-the-art" performance, but the comparison is primarily to older systems. A comparison to more recent neural approaches (e.g., using RNNs or transformers for event linking) would better justify this claim and strengthen the paper's contribution.

2. **ACE++ dataset construction lacks justification:** The authors create ACE++ by adding 1351 web documents to ACE 2005, but the paper doesn't sufficiently explain how these documents were annotated or why they're a valid extension of ACE 2005. The annotation process, domain coverage, and quality control of these additional documents are not discussed, making it difficult to assess whether ACE++ is a meaningful extension of the original dataset.

3. **Limited discussion of computational efficiency:** The paper doesn't address the computational requirements of the model compared to previous approaches. For a practical application, it's important to know if the neural approach is significantly more computationally expensive than simpler feature-based methods. This omission is particularly notable given the authors' claim about the practicality of their approach.

## Questions to Authors

1. Could you provide more details about the annotation process for the ACE++ dataset? Specifically, how were the additional 1351 web documents annotated to match ACE 2005's event coreference annotation scheme?

2. How does your model compare to more recent neural approaches for event coreference resolution (e.g., using transformer architectures) that might have been published since the cited works?

3. Could you provide computational complexity analysis showing the training and inference time of your model compared to previous approaches, especially those relying on external semantic resources?

## Overall Assessment

This is a strong paper that makes a significant contribution by demonstrating that event linking can be performed effectively without external semantic resources, using a CNN-based approach that leverages sentential features. The empirical results are convincing, the error analysis is insightful, and the model design is well-justified. While the paper has some limitations regarding comparison to recent neural approaches and dataset justification, these do not significantly undermine the core contribution. The work advances the field by providing a more practical, domain-agnostic approach to event linking that maintains state-of-the-art performance.

I recommend acceptance with minor revisions addressing the questions above.
