- Proposes a CNN-based neural approach for event linking that avoids external semantic resources (WordNet/FrameNet).  
- Model uses CNN-generated sentential features plus a classifier with pairwise features to decide coreference.  
- Main claim: comparable or better performance can be achieved without external semantic resources.  
- Provides thorough empirical evaluation on ACE 2005 and a new ACE++ dataset with ablation studies.  
- Includes a detailed error analysis of 200 examples highlighting key failure modes.  
- Strong motivation: analysis shows ~23% of event mentions are argument-incomplete, justifying the task.  
- Comprehensive comparison to Liu et al. (2014) with competitive BLANC F1 (73.31% vs. 70.43%).  
- Ablation studies convincingly show each feature class contributes to performance.  
- Error analysis identifies pronoun resolution and annotator disagreement as major challenges.  
- Justifies CNNs by noting many coreferential mentions are >5 sentences apart, reducing locality effectiveness.  
- Shows practical benefit of more data: ACE++ yields notable recall improvements over ACE 2005.  
- Overstates "state-of-the-art" claim since improvements are metric-dependent and mixed across measures.  
- Limited comparison to more recent neural methods (transformers/attention-based approaches).  
- No empirical comparison to alternative architectures (RNNs/LSTMs/transformers) to justify CNN choice.  
- Insufficient description and analysis of the new ACE++ dataset (domain/annotation differences unclear).  
- Limited hyperparameter analysis (filters, window size, embedding dims) reducing reproducibility insight.  
- Asks how the model compares to recent transformer/attention-based event linking methods.  
- Asks why CNNs were chosen over RNNs/transformers and whether alternatives were tried.  
- Requests explanation for metric discrepancies (better on BLANC but worse on B-CUBED).  
- Queries how long-distance event coreference compares to entity coreference and affects architecture choice.  
- Suggests incorporating discourse-aware/pronoun-resolution features to address error types.  
- Recommendation: accept with minor revisions — temper "state-of-the-art" claims and include comparisons to recent neural approaches.
