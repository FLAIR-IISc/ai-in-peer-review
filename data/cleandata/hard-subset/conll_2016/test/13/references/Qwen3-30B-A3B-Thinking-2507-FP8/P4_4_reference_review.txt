

# Review of "Event Linking with Sentential Features from Convolutional Neural Networks"

## Summary of the Paper
The paper addresses event linking (event coreference resolution), which involves resolving coreferences between recognized event mentions in a document. The authors note that previous approaches often rely on external semantic features like WordNet or FrameNet, which have limited domain coverage. They propose a model that uses sentential features generated from convolutional neural networks (CNNs) to represent individual event mentions, followed by a coreference decision module that combines these representations with pairwise features. The model does not rely on external semantic resources, but rather uses a combination of local and sentential features to describe event mentions.

## Main Contributions (in order of strength)
1. **Novel application of sentential features for event linking**: The authors demonstrate that event linking can be effectively performed using sentential features generated by CNNs without relying on external semantic resources, addressing a key limitation of previous approaches.
2. **State-of-the-art performance without external semantic features**: The model achieves competitive results on two datasets (ACE and ACE++) without using external semantic resources, showing that high performance in event linking doesn't require domain-specific semantic knowledge.
3. **Comprehensive feature analysis and error analysis**: The paper provides detailed ablation studies showing the contribution of different feature classes, and conducts a thorough error analysis that identifies pronoun resolution as a key challenge for future work.

## Strengths
1. **Strong technical contribution with clear novelty**: The paper makes a significant contribution by demonstrating that high-quality event linking can be achieved without external semantic resources. The authors provide compelling evidence from dataset analysis (Section 3) that sentential features are necessary, as 50.97% of coreferential event mention pairs share no arguments and 47.29% have different trigger words. This analysis clearly motivates their approach and distinguishes it from previous methods.

2. **Thorough experimental evaluation**: The authors conduct comprehensive experiments comparing their model to state-of-the-art systems on two datasets (ACE and ACE++). Their ablation study (Table 5) clearly shows that all feature classes (pairwise, trigger-local lexical, and sentential features) contribute positively to the model's performance. The model achieves state-of-the-art results on the ACE dataset (Table 3) and demonstrates the impact of dataset size (Table 4) and clustering strategy (Table 4).

3. **Clear demonstration of the necessity of sentential features**: The paper's analysis of dataset properties (Section 3) is particularly strong, showing that relying solely on intrinsic event properties would be insufficient for event linking. The authors provide specific statistics showing that 55.42% of coreferential event mentions are more than five sentences apart, indicating that locality-based heuristics would not work well.

## Weaknesses
1. **Limited novelty compared to prior work on sentential features**: While the authors claim novelty, there is significant prior work on using sentential features for relation and event extraction (Zeng et al., 2014; Nguyen and Grishman, 2015; Chen et al., 2015; dos Santos et al., 2015; Zeng et al., 2015). The paper doesn't sufficiently distinguish their specific contribution to event linking from these prior works, making the novelty somewhat incremental rather than groundbreaking.

2. **Lack of comparison to recent neural approaches**: The paper compares to some prior work (Liu et al., 2014; Bejan and Harabagiu, 2010), but doesn't compare to more recent neural approaches to event linking or coreference resolution that might use similar or better techniques. Given the rapid advancement in neural architectures for NLP tasks, this comparison would be valuable for establishing the significance of their contribution.

3. **Unclear broader impact**: The paper doesn't clearly articulate how this work advances the broader field of information extraction or natural language processing beyond the specific task of event linking. A stronger position statement would help readers understand the wider significance of the work.

## Questions to Authors
1. How does your approach specifically differ from prior work that used sentential features for relation extraction (e.g., Zeng et al., 2014; Chen et al., 2015)? What specific challenges in event linking necessitate your particular architecture?

2. Why didn't you compare your model to more recent neural coreference resolution systems (e.g., those using BERT or transformer architectures)? Given the success of transformer models in other NLP tasks, it would be interesting to see how your approach stacks up against these state-of-the-art methods.

3. The error analysis identifies pronoun resolution as a significant challenge. How might your approach be extended to better handle pronoun resolution, and do you think incorporating discourse structure would be beneficial?
