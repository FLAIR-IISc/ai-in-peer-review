

# Review: "Event Linking with Sentential Features from Convolutional Neural Networks"

## Summary and Main Contributions

This paper addresses event linking (event coreference resolution), the task of determining whether different event mentions in a document refer to the same real-world event. The authors analyze the ACE 2005 corpus and find that approximately 23% of event mentions are incomplete on the argument level, making event linking an important task. They propose a neural network-based approach that uses sentential features from convolutional neural networks instead of relying on external semantic resources like WordNet or FrameNet.

The main contributions of this work (as assessed objectively) are:

1. A novel neural architecture for event linking that uses sentential features from CNNs, achieving state-of-the-art results without external semantic features. The authors demonstrate that external semantic resources are not necessary for good performance on event linking, which is significant for practical applications where such resources may not be available.

2. A comprehensive analysis of dataset properties showing that 50.97% of coreferential event pairs share no arguments and 55.42% of coreferential mentions are more than five sentences apart, motivating the need for sentential features rather than just local features.

3. An empirical demonstration that a model using only sentential features and pairwise features can outperform previous approaches that relied on extensive external semantic features, with a clear ablation study showing the contribution of each feature class.

## Strengths

1. **Strong empirical evidence for the effectiveness of sentential features**: The authors provide compelling evidence through comprehensive analysis that sentential features are crucial for event linking. Their analysis showing that 50.97% of coreferential pairs share no arguments and 55.42% of coreferential mentions are more than five sentences apart demonstrates why previous approaches relying solely on local features would fail, and why sentential features are necessary. The ablation study (Table 5) clearly shows that removing sentential features causes a significant drop in performance (from 76.90% to 64.12% BLANC F1), providing strong evidence for the contribution of this component.

2. **State-of-the-art performance without external resources**: The model achieves 73.31% BLANC F1 on ACE 2005, outperforming Liu et al. (2014) (70.43% F1) and Bejan and Harabagiu (2010) (45.1% F1) without using any external semantic resources. This is a significant contribution as it demonstrates that high performance on event linking does not require expensive external semantic resources, which are often domain-specific and hard to maintain. The authors provide a clear comparison showing their model's advantage over previous work that relied on "a rich set of 105 semantic features."

3. **Comprehensive dataset analysis and model evaluation**: The authors create a new dataset (ACE++) by adding 1351 web documents to ACE 2005, which allows for a more robust evaluation. The results show that increasing the training data improves recall from 75.16% to 83.21% while maintaining strong precision (73.31% vs. 76.90% BLANC F1). The authors also provide a thorough error analysis of 200 false predictions (100 false positives and 100 false negatives), revealing that a significant portion of errors involve pronoun resolution (a known challenging problem) and that many boundary cases exist where even expert annotators disagree.

## Weaknesses

1. **Lack of comparison with recent neural approaches**: The paper claims state-of-the-art results but only compares with Liu et al. (2014), which is from 2014. Since then, there have been multiple neural approaches to event coreference resolution published in the last 2-3 years that should be compared against (e.g., from ACL 2019, EMNLP 2020, etc.). Without these comparisons, the claim of "state-of-the-art" performance is not fully supported, as the paper doesn't demonstrate how it compares to more recent methods.

2. **Limited domain coverage and dataset diversity**: The experiments are conducted solely on ACE 2005 and ACE++, which are both news corpora. The paper doesn't evaluate on other domains or datasets (e.g., biomedical, social media, or other text types), making it unclear how well the model generalizes beyond news articles. While the authors mention testing on "more datasets" as future work, this limitation is significant for the current contribution.

3. **Ambiguity regarding the use of pre-trained embeddings**: The paper claims that the model does not rely on external semantic features, but it uses pre-trained word embeddings (from Mikolov et al., 2013). This is a form of external semantic knowledge, which contradicts the authors' claim that "the model does not depend on resources with restricted domain availability." The paper does not clarify how this use of pre-trained embeddings affects the claim of not relying on external semantic resources.

## Questions to Authors

1. The paper claims state-of-the-art performance but does not compare with more recent neural approaches to event coreference resolution (e.g., from ACL 2019-2021). Could the authors provide these comparisons to better establish their model's standing in the current literature?

2. The authors mention that the model uses pre-trained word embeddings (from Mikolov et al., 2013) during initialization. How does this use of external semantic knowledge affect their claim that "the model does not depend on resources with restricted domain availability"? Is this considered acceptable within the community, or should the authors clarify this distinction?

3. The ACE++ dataset was created by adding 1351 web documents to ACE 2005. Could the authors provide more details about how these additional documents were annotated and how the annotation quality was ensured? This is important for understanding the validity of the ACE++ dataset for evaluation purposes.
