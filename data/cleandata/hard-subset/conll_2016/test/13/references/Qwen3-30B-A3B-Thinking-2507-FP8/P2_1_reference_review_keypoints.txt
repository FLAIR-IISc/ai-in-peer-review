- Paper proposes CNN-based sentential features for event coreference resolution, avoiding external semantic resources.  
- Claims state-of-the-art performance without WordNet/FrameNet by representing event mentions via sentence-level CNN encodings.  
- Provides a systematic ablation study showing importance of pairwise, trigger-local lexical, and sentential features.  
- Expands ACE 2005 into a larger ACE++ dataset by adding 1351 web documents.  
- Motivates the task with evidence that ~23% of ACE event mentions are argument-incomplete, justifying event linking research.  
- Demonstrates large performance drop when removing sentential features (76.90% → 64.12% BLANC F1), showing their value.  
- Strong empirical evaluation against relevant baselines using appropriate metrics (BLANC) and improved results on ACE and ACE++.  
- Error analysis identifies hard cases like pronoun resolution and annotation boundary issues, noting human difficulty on some examples.  
- Novelty is limited: applying sentential CNN features to coreference is similar to prior relation/event extraction work and not clearly distinct.  
- Insufficient comparisons to more recent neural coreference models, weakening the state-of-the-art claim.  
- Error analysis is somewhat superficial—lacks concrete examples and proposals for addressing pronoun-resolution failures.  
- Paper omits discussion of computational complexity and resource requirements of the CNN approach.  
- Asks authors to clarify novelty over prior sentential-feature work, explain handling of long-distance coreference given local CNN windows, reconcile mixed metric results (BLANC vs B-CUBED), and consider testing on EventCorefBank/ProcessBank.  
- Overall recommendation: accept with minor revisions to address novelty framing, broader comparisons, deeper error analysis, and computational cost discussion.
