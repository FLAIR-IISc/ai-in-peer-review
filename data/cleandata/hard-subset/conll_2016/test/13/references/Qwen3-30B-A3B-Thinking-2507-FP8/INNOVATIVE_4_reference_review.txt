

# Review of "Event Linking with Sentential Features from Convolutional Neural Networks"

## Summary
This paper proposes a novel approach for event linking (event coreference resolution) that uses sentential features generated from convolutional neural networks (CNNs) instead of relying on external semantic resources like WordNet or FrameNet. The authors design a two-step architecture: first, they use a CNN to generate representations for individual event mentions based on their surrounding sentences; second, they use these representations along with pairwise features to make coreference decisions. The approach achieves state-of-the-art performance on the ACE 2005 dataset and a newly created ACE++ dataset, without requiring external semantic resources.

## Main Contributions (in order of strength)
1. **Novel use of sentential features from CNNs for event coreference resolution** - The most significant contribution, which challenges the common practice of relying on external semantic resources with limited domain coverage.
2. **State-of-the-art performance without external semantic features** - Demonstrating that high-quality event linking can be achieved without domain-specific resources.
3. **Thorough analysis of feature importance** - Providing clear evidence for the contribution of each feature class to the model's performance.

## Strengths

1. **Exceptional Novelty and Bold Approach**: The core contribution of this paper is highly novel and represents a bold departure from current practice in event linking. Previous work has heavily relied on external semantic resources (WordNet, FrameNet) with limited domain coverage (as acknowledged by the authors), while this paper demonstrates that rich sentential features generated directly from text using CNNs can achieve state-of-the-art performance without these resources. This is precisely the kind of innovative approach that should be prioritized in a conference like CoNLL.

2. **Strong Performance Validation**: The paper demonstrates clear empirical evidence that their approach matches or exceeds state-of-the-art performance without external resources. The results on ACE++ (76.90 F1 on BLANC) are particularly compelling, showing that the model generalizes well with additional training data. The ablation study (Table 5) provides convincing evidence that the sentential features are not just a novelty but a meaningful contribution to performance.

3. **Well-Motivated Technical Approach**: The paper provides a strong theoretical justification for their approach based on analysis of the ACE 2005 dataset (50.97% of coreferential pairs share no arguments, 47.29% have different trigger words, 55.42% are more than five sentences apart). This analysis directly supports the need for sentential features and makes the approach more compelling.

4. **Practical Impact**: By removing the dependency on external semantic resources, the approach has significant practical implications for deploying event linking systems across different domains without requiring domain-specific resource creation. As the authors state: "our approach does not depend on resources with restricted domain availability."

5. **Rigorous Error Analysis**: The error analysis (Section 5.3) identifies a key limitation (pronoun resolution) and demonstrates that some errors stem from annotation inconsistencies, providing a nuanced understanding of the task's challenges. This goes beyond typical error analysis in the field.

## Weaknesses

1. **Limited Comparison to Recent Neural Approaches**: The paper doesn't compare to more recent neural approaches for event coreference resolution, such as those using attention mechanisms or transformer models. While the paper acknowledges the use of CNNs, the field has evolved significantly since the cited works (Zeng et al., 2014; Chen et al., 2015), and the absence of this comparison weakens the claim of "state-of-the-art" performance.

2. **Insufficient Justification for ACE++ Dataset**: The paper creates a new ACE++ dataset by adding 1351 web documents to ACE 2005, but provides minimal details about how these additional documents were annotated or how they differ from the original ACE 2005 corpus. This is a significant concern because the performance boost (Table 4) might be partly due to the additional data rather than the model itself.

3. **Limited Comparison to Recent Event Linking Work**: The comparison is limited to older approaches (Bejan and Harabagiu, 2010; Liu et al., 2014), while there has been significant work on event linking since then (e.g., Sachan et al., 2015; Cybulska and Vossen, 2015; Araki and Mitamura, 2015). This makes the "state-of-the-art" claim less compelling.

## Questions to Authors

1. Why didn't the paper compare to more recent neural network-based approaches for event coreference resolution (e.g., using attention mechanisms or transformer models)?

2. Can you provide more details about how the ACE++ dataset was created and annotated, and how it differs from ACE 2005 in terms of domain coverage, quality, and annotation consistency?

3. The paper claims to achieve state-of-the-art performance, but the comparison is limited to older approaches. Can you provide a comparison to more recent work on event linking (e.g., Sachan et al., 2015; Cybulska and Vossen, 2015)?

4. The paper uses multiple metrics (BLANC, MUC, B-CUBED, Positive links) without clear justification. Why did the authors choose these specific metrics, and are they standard for event coreference resolution?

5. The error analysis is based on a small sample (100 false positives, 100 false negatives). Can the authors provide a more systematic analysis of the model's limitations, particularly with respect to pronoun resolution?

## Overall Recommendation

**Accept**

This paper presents a highly novel and bold approach to event linking that challenges the field's reliance on external semantic resources. The core contribution—using sentential features from CNNs for event coreference resolution—is a significant conceptual advancement that deserves recognition. While the paper has some methodological weaknesses (limited comparison to recent neural approaches, insufficient justification for ACE++), these do not outweigh the novelty and strong empirical validation of the core idea.

The paper demonstrates that high-quality event linking can be achieved without external semantic resources, which is a substantial contribution to the field. The performance results are compelling, and the error analysis provides valuable insights into the model's limitations. Given the reviewer's criteria of prioritizing novel ideas over methodological perfection, this paper represents a significant contribution that should be accepted.
