

# Review: Event Linking with Sentential Features from Convolutional Neural Networks

## Summary
This paper proposes a novel approach to event coreference resolution (event linking) that replaces reliance on external semantic resources (e.g., WordNet, FrameNet) with sentential features generated by convolutional neural networks (CNNs). The authors design a two-step architecture: first, a CNN generates representations for individual event mentions by processing the surrounding sentence; second, the model makes coreference decisions based on these representations combined with pairwise features. The approach achieves state-of-the-art performance on two datasets without using external semantic resources, demonstrating that the need for domain-specific semantic knowledge is unnecessary for high-quality event linking.

## Main Contributions (as I see them)
1. **Novel application of CNN-generated sentential features for event linking without external semantic resources** (primary contribution)
2. **Demonstration that external semantic resources are unnecessary for state-of-the-art event linking performance** (secondary contribution)
3. **Thorough analysis of feature classes and their impact on event linking performance** (tertiary contribution)

## Strengths

1. **Bold conceptual shift that challenges the field's conventional wisdom**: The paper makes a compelling case that the field has been unnecessarily reliant on external semantic resources (like WordNet and FrameNet) for event linking. The authors demonstrate that their CNN-based approach achieves state-of-the-art performance without these resources, which is a significant conceptual contribution. This directly addresses the key limitation cited by the authors (that external resources have "limited domain coverage" and are "created at high cost"), and provides a more generalizable approach to event linking.

2. **Strong empirical evidence for the core claim**: The paper provides convincing evidence that their approach outperforms prior state-of-the-art systems (Liu et al., 2014) on the ACE dataset, while eliminating the need for external semantic resources. The comparison to Liu et al. is particularly compelling because their system "involves random-forest classification with best-link clustering and propagation of attributes between event mentions, and is grounded on a manifold of external feature sources, i.e., it uses a 'rich set of 105 semantic features'." The authors' system achieves comparable or better performance without these resources, which is a major contribution to the field.

3. **Thorough feature analysis that provides valuable insights**: The paper's analysis of different feature classes (Table 5) is particularly strong. It demonstrates that all feature classes (pairwise, trigger-local lexical, and sentential features) contribute positively to performance, and provides a clear understanding of how the model works. The finding that adding sentential features significantly improves performance (from 62.24/76.23/64.12 to 73.22/83.21/76.90) is particularly compelling evidence for the value of their approach.

## Weaknesses

1. **Limited novelty in the technical implementation**: While the application of CNNs to event linking is novel, the CNN architecture itself is standard (using convolution, max-pooling, and a hidden layer) and has been used in many prior NLP tasks. The paper doesn't introduce any new architectural innovations to the CNN itself, which limits the technical novelty. This is a relatively minor weakness given the novelty of the application, but it's worth noting.

2. **Insufficient comparison to recent neural approaches**: The paper compares to Liu et al. (2014), which is a strong baseline, but doesn't compare to more recent neural approaches to coreference resolution or event linking (e.g., those using attention mechanisms, transformers, or more advanced neural architectures). This makes it difficult to assess where the proposed approach stands relative to the current state-of-the-art in neural methods.

3. **Limited analysis of the error cases beyond pronoun resolution**: While the error analysis (Section 5.3) is good, it focuses primarily on pronoun resolution as a key challenge. A more comprehensive analysis of error patterns would strengthen the paper's contribution. For example, the authors could analyze whether certain event types or sentence structures are particularly challenging for their model.

## Questions to Authors

1. The paper demonstrates that their approach outperforms Liu et al. (2014), but Liu et al. used a different clustering strategy (BestLink). How would your model perform if you used BestLink instead of transitive closure? Would this change the comparison to Liu et al.?

2. The paper mentions that "55.42% of coreferential event mentions are more than five sentences apart," which motivates the use of sentential features. Could you elaborate on how the CNN-based sentential features specifically capture information across these longer distances, compared to other approaches that might use attention or other mechanisms?

3. The paper doesn't compare to recent neural approaches to coreference resolution (e.g., those using BERT or other transformer-based models). Have you considered testing your approach against these more recent neural architectures, and how would you expect it to perform relative to them?

## Recommendation

Accept

As a reviewer who highly values novelty and bold approaches, I find this paper to be a strong submission. The core contribution—demonstrating that external semantic resources are unnecessary for state-of-the-art event linking performance—is a significant conceptual shift in the field. While the technical implementation of the CNN architecture is standard, the application to event linking is novel and well-motivated. The paper provides compelling empirical evidence for their claim, and the thorough feature analysis adds significant value.

The minor methodological limitations (standard CNN architecture, limited comparison to recent neural approaches) are outweighed by the novelty of the core idea. This paper introduces a new direction for event linking that has the potential to simplify and generalize the field, making it a valuable contribution.
