**Summary of the Paper and Main Contributions**

This paper addresses the task of within-document event coreference resolution (event linking). The authors propose a two-stage neural model that first learns a dense representation for each event mention and then uses these representations in a pairwise classifier to decide coreference. The primary motivation is to achieve strong performance without relying on external semantic resources like WordNet or FrameNet, which have limited domain coverage and can be costly to create. The model uses a Convolutional Neural Network (CNN) with piecewise max-pooling to capture sentential features, which are combined with trigger-local lexical features to represent an event mention. These representations, along with standard pairwise features (e.g., event type agreement, distance), are fed into a final classifier. The authors evaluate their model on the ACE 2005 corpus and a larger, proprietary extension (ACE++), demonstrating performance competitive with previous state-of-the-art systems.

In my view, the main contributions of this work are:
1.  A successful demonstration that learned sentential features from a neural network can effectively replace hand-engineered features from external knowledge bases for event linking. The work shows that a system can achieve competitive, state-of-the-art performance while being more domain-independent, which is a significant practical contribution.
2.  A thorough empirical analysis of the proposed model. This includes a well-designed ablation study that clearly isolates the contribution of different feature types (sentential, local, pairwise), an investigation into the impact of training data size, and a useful comparison of clustering strategies.
3.  An insightful qualitative error analysis that identifies key remaining challenges in the field, such as pronominal event anaphora and annotation inconsistencies, providing valuable directions for future research.

**Strengths**

1.  **High-Impact Methodological Direction**: The core strength of this paper is its compelling argument for moving away from systems dependent on external knowledge bases. By achieving performance on par with a state-of-the-art system (Liu et al., 2014) that uses a "rich set of 105 semantic features," the authors convincingly demonstrate the power of learning representations directly from text. This makes the proposed approach significantly more portable to new domains where resources like WordNet may be unavailable or incomplete.
2.  **Thorough and Rigorous Evaluation**: The experiments are well-conducted and provide strong support for the authors' claims. The comparison on the standard ACE 2005 benchmark is crucial. Furthermore, the experiments on the larger ACE++ dataset, while not fully reproducible, provide valuable insights into the model's ability to generalize with more data (Table 4). The ablation study (Table 5) is particularly effective, clearly demonstrating that all three feature components (pairwise, local lexical, and sentential) contribute positively to the final performance, with the sentential features providing the largest boost.
3.  **Clarity and Quality of Analysis**: The paper is well-written and the model is described clearly, aided by an effective diagram (Figure 1). Beyond presenting performance numbers, the authors provide insightful analysis. For example, their finding that simple transitive closure outperforms the BestLink strategy is an interesting contrast to previous work and provides practical guidance. The error analysis in Section 5.2 is excellent, highlighting difficult cases that are not easily solved by current models and even questioning the consistency of the gold standard annotations, which helps to contextualize the performance of all systems on this challenging task.

**Weaknesses**

1.  **Limited Architectural Novelty**: The CNN architecture used to generate event representations (convolution + piecewise max-pooling) is a well-established method for sentence-level tasks, particularly relation classification, as acknowledged by the authors. The novelty lies in its application to event linking. While this is a valid and useful contribution, the paper would be stronger if it discussed *why* this architecture is particularly suitable for this task over other possible neural architectures (e.g., LSTMs) or if it had explored novel modifications tailored to representing events.
2.  **Slight Overstatement of "State-of-the-Art" Performance**: The performance comparison in Table 3 shows the proposed model is competitive, but a definitive claim of superiority is difficult to sustain. The model slightly outperforms Liu et al. (2014) on BLANC F1 but underperforms on B-CUBED F1. Given the authors' own caveats about differing data splits and preprocessing steps across papers, framing the results as "on-par with the state-of-the-art while using fewer resources" would be more precise and equally impactful.
3.  **Reproducibility Concerns with ACE++**: A significant portion of the analysis, including the ablation study and the impact of data size, relies on the proprietary ACE++ dataset. While the creation of this resource is commendable, its unavailability to the public limits the verifiability of these important findings and makes it difficult for other researchers to build directly upon this aspect of the work. The paper's contribution would be stronger if these key analyses could be replicated on a public dataset, even if smaller.

**Questions to Authors**

1.  The use of the `square` activation function in the similarity layer (Equation 4) is an unusual choice. Could you elaborate on the intuition that "invariance under polarity changes is desirable" for similarity? Did you empirically compare this against more standard activation functions like ReLU or tanh for this layer, and if so, what were the results?
2.  Your error analysis correctly identifies pronominal event coreference as a major source of false negatives. Your current model's scope is the sentence containing the event trigger. How do you envision extending this model to capture the document-level context needed to resolve such anaphora? For example, would it be feasible to incorporate representations of preceding sentences into the event mention representation?
3.  The ablation result in Table 5 (row 4), where removing local features boosts precision but hurts recall, is interesting. You hypothesize that the sentential component adapts to focus more on the trigger. An alternative interpretation could be that the local features are critical for grounding the representation and preventing over-generalization from sentential context alone. Could you comment on this alternative view? Does this suggest an inherent tension between local and sentential features that the model struggles to balance?
