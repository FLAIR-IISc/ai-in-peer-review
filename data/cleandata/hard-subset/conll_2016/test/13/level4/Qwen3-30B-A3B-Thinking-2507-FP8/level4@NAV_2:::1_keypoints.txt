1. Introduces a neural event coreference model that uses CNN-derived sentential features instead of external semantic resources.  
2. Proposes a two-step architecture: CNNs generate latent sentential representations, then lexical/pairwise features feed a similarity scorer.  
3. Reports state-of-the-art results on ACE and ACE++ without relying on domain-specific knowledge bases.  
4. Novel CNN-based architecture removes dependence on resources like WordNet/FrameNet while maintaining SOTA performance.  
5. Shows sentential/contextual features are critical via ablation, with significant performance drops when removed.  
6. Provides a detailed error analysis highlighting unresolved challenges (e.g., pronoun resolution) and dataset limitations.  
7. Resource independence enables application in domains lacking curated KBs, achieving competitive ACE++ F1 (76.90) using only lexical/sentential features.  
8. Ablation study convincingly demonstrates necessity of sentential features: removing them reduces F1 by 15.6% (76.90 → 61.28); pairwise-only yields poor recall (56.70).  
9. Error analysis finds systematic failure modes (17% of false negatives involve pronoun resolution), offering concrete directions for future work.  
10. Comparison to Liu et al. (2014) is unfair because this work uses gold triggers while Liu used predicted triggers, invalidating the F1 comparison.  
11. ACE++ is not publicly released despite claims, impeding reproducibility and fair comparison.  
12. Position embedding description is ambiguous (initialization and sharing unclear), harming interpretability and replicability.  
13. Question: How does performance change with predicted event mentions instead of gold triggers, and how does that compare to Liu et al. (2014)?  
14. Question: Will ACE++ be released publicly; if not, how is the claim of a "publicly available" dataset justified?  
15. Question: Could the model be extended with discourse-level features (e.g., pronoun antecedent heuristics) to address pronoun-resolution errors?
