- Proposes a resource-agnostic two-step neural model (CNN sentential representations + similarity scoring) for event coreference that avoids external semantic resources.  
- Achieves state-of-the-art results on ACE 2005 and a larger ACE++ dataset without using external semantic features.  
- Empirically shows sentential context is necessary: 55.42% of coreferential mentions span more than five sentences, making locality heuristics insufficient.  
- Error analysis highlights unresolved challenges, especially pronoun resolution and annotation inconsistencies.  

- Strength: Demonstrates high-performance event linking without curated knowledge bases, increasing applicability to low-resource domains.  
- Strength: Strong empirical validation, outperforming prior SOTA (Liu et al., 2014) on BLANC F1 and showing robustness with larger training data.  
- Strength: Ablation study confirms all feature classes (pairwise, lexical, sentential) contribute; removing sentential features sharply hurts recall.  

- Weakness: Comparison is unfair because the paper uses gold event triggers while prior work uses predicted triggers, undermining SOTA claims.  
- Weakness: Inconsistent ACE 2005 train/test splits across systems make direct comparisons unreliable and weaken the performance claim.  
- Weakness: Position embedding description is ambiguous (initialization and rationale unclear), hindering reproducibility.  

- Question: How does the model perform end-to-end with predicted triggers instead of gold triggers?  
- Question: Will the ACE++ dataset be publicly released to allow replication and fair comparison?  

- Comment: The claim that single-word context/window choice is insufficient is supported by results, but the rationale for the specific context window size is unclear.  
- Comment: The square activation in Eq. (4) is noted but not contextualized against prior work—unclear if novel or minor.  
- Comment: Rename Table 6’s "same type" baseline to "same event type" for clarity.  
- Comment: The error analysis note about 17 false negatives tied to annotation inconsistencies needs more detail on annotator ambiguity.
