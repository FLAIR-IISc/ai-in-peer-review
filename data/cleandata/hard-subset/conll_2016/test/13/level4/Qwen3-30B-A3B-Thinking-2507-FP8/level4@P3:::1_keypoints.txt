- Proposes a neural event linking model that replaces external semantic resources with CNN-generated sentential features combined with lexical and pairwise features, claiming SOTA on ACE2005 and ACE++.  
- Main contribution: a framework achieving strong event linking without external semantic features.  
- Shows empirical evidence that sentential features are important: 50.97% of coreferential pairs share no arguments and 47.29% have different triggers.  
- Provides an error analysis highlighting persistent challenges like pronoun resolution and annotation boundary cases.  
- Strength: removes dependency on costly domain-specific resources (e.g., WordNet, FrameNet), improving cross-domain applicability.  
- Strength: strong empirical validation—many coreferences are distant or argument-less, and ablation shows sentential features substantially boost F1 (76.90 → 64.12 when removed).  
- Strength: actionable manual error analysis (100 errors) identifying pronoun resolution and annotator inconsistencies.  
- Weakness: unfair comparison due to use of gold-standard triggers while key prior work uses predicted triggers; no predicted-trigger experiments.  
- Weakness: ACE++ dataset is not publicly released, limiting reproducibility and future comparisons.  
- Weakness: lacks statistical significance testing for small reported improvements.  
- Question: please provide results with predicted event triggers for fair pipeline comparison.  
- Question: when will the ACE++ dataset be released?  
- Question: have you considered applying the approach to entity coreference (e.g., OntoNotes) for broader comparison?  
- Additional: error analysis should be tied to concrete model fixes (e.g., discourse-level features for pronouns).  
- Additional: the square activation in the similarity layer is under-motivated—unclear novelty or rationale.  
- Additional: baselines in Table 6 need clearer labeling/distinction (e.g., "same type" vs. "same event").
