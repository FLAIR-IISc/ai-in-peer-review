- Proposes a two-part model using CNNs to generate sentential features for event mentions and combine them with pairwise features for coreference scoring, achieving SOTA on two datasets.
- Contribution: CNN-based architecture for sentential event representations that avoids external semantic resources.
- Contribution: Comprehensive evaluation showing SOTA performance without external semantic features, arguing domain limitations can be overcome.
- Contribution: Error analysis that identifies key failure modes and directions for future work.
- Strength: Strong motivation from ACE 2005 analysis showing 55.42% of coreferential mentions are >5 sentences apart, making locality heuristics inadequate.
- Strength: Creation and analysis of ACE++ dataset showing increased training data improves recall substantially (75.16% → 83.21%).
- Strength: Insightful error analysis highlighting pronoun resolution (e.g., “It”) as a major source of false negatives.
- Weakness: No adequate justification for choosing CNNs over RNNs/LSTMs; lacks ablation/architectural comparisons.
- Weakness: Novelty claim is weak—insufficient differentiation from prior neural methods for relation/event extraction and continuous-space linking.
- Weakness: ACE++ dataset is under-described (event types, domain coverage), undermining claims about generalizability.
- Weakness: Evaluation uses only gold event mentions; lacks experiments with non-gold mentions to reflect end-to-end error propagation.
- Weakness: Does not explore integration with existing coreference systems (e.g., for pronoun resolution) or augmentation strategies.
- Question: What event types does ACE++ include and how does it differ from ACE 2005 in domain coverage?
- Question: Can you provide results using non-gold event mentions to simulate real-world pipelines?
- Question: How does the model perform when integrated with standard coreference systems for pronoun handling?
- Question: Have you tested on other event coreference datasets (e.g., EventCorefBank, ProcessBank) to validate generalizability?
- Question: Can you provide ablation studies comparing different neural architectures for the sentential representation component?
