- Contribution: Proposes event linking using CNN-extracted sentential features, achieving SOTA without external semantic resources.  
- Contribution: Releases ACE++, an expanded dataset for event linking, motivated by dataset-size and argument-completeness analyses.  
- Contribution: Provides detailed error analysis highlighting challenges in event coreference resolution.

- Strength: Achieves competitive (2–3 F1) improvements without domain-specific external knowledge, improving portability.  
- Strength: ACE++ dataset expansion is well-motivated and valuable for future research.  
- Strength: Manual error analysis identifies pronoun resolution and annotator disagreement issues.  
- Strength: Paper is clearly written with reproducible model and experimental descriptions.

- Weakness: Choice of CNN architecture is insufficiently justified; lacks empirical comparison to alternatives like RNNs/LSTMs.  
- Weakness: Limited novelty positioning and incomplete related-work discussion on continuous-space/sentential-feature methods.  
- Weakness: ACE++ lacks detailed dataset documentation (e.g., event types), weakening claims about generalization and domain-independence.  
- Weakness: Ablation study is incomplete; more granular analysis of pairwise features and feature selection rationale is needed.

- Question: Provide a detailed breakdown of event types included in ACE++.  
- Question: Were alternative architectures (RNNs/LSTMs) explored; what were the results or reasons for not exploring them?  
- Question: How were the specific pairwise features selected, and which features were considered but discarded?  
- Question: Have you considered using an off-the-shelf coreference resolver to mitigate pronoun-resolution errors?

- Additional: Report model performance without gold event mentions to assess robustness to upstream extraction errors.  
- Additional: Evaluate on EventCorefBank to test generalizability beyond ACE/ACE++.  
- Additional: Pairwise features remain important; further feature-engineering investigation could improve performance.
