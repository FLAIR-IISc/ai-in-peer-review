- Proposes event linking using sentential features extracted with CNNs as an alternative to external knowledge sources.  
- Model has two stages: generate event mention representations, then compute coreference scores for mention pairs.  
- Paper is well-written with clear presentation and insightful analysis of argument sharing in ACE corpus.  
- Motivates and creates the expanded ACE++ dataset to address ACE limitations.  
- Primary weakness: inadequate justification for choosing CNNs over alternatives (e.g., RNNs) and no comparative analysis.  
- Performance gains are modest (~2–3 F1), requiring stronger evidence that the CNN architecture is superior.  
- Limited novelty: applying CNNs to sentential features for event linking is not well contextualized within existing continuous-space methods.  
- Related work comparison is incomplete; competing methods in Table 3 appear insufficient.  
- Major omission: insufficient details on ACE++ construction, especially event type coverage, risking biased results.  
- Releasing ACE++ publicly would increase the work’s impact and reproducibility.  
- Error analysis is superficial; deeper examination of model error types and causes is needed.  
- Questions for authors: rationale for CNN choice and comparative analysis, ACE++ event type details, plans to release ACE++, and specifics on recurring error cases beyond pronoun resolution.
