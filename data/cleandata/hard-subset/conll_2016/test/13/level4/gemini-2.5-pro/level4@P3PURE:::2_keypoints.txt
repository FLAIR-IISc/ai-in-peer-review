- Paper proposes CNN-generated sentential features for event entity linking with a two-part model (mention representation + pairwise coreference scoring).  
- Strengths: well-written, clear; insightful ACE corpus analysis on argument sharing; dataset size analysis motivates ACE++.  
- Major issue: no motivation or analysis for using a CNN over alternatives (e.g., RNN); modest F1 gains require stronger architectural justification and comparisons.  
- Major issue: novelty unclear—using sentential CNN features is not sufficiently positioned against prior continuous-space/event-linking methods; baselines in Table 3 are incomplete and should include more recent relevant work.  
- Major issue: ACE++ dataset methodology and coverage are not described—must report collection process, event types, how it addresses ACE limitations, evidence that it breaks resource-dependent methods, and plans for public release.  
- Minor: report model performance without gold mentions to make Table 3 evaluation more comprehensive.  
- Minor: consider augmenting with standard coreference systems (e.g., CoreNLP) as this could strengthen baselines and model performance.  
- Minor: evaluate on additional resources (e.g., EventCorefBank) to strengthen claims and motivate joint entity/event coreference modeling.  
- Minor: pairwise features are effective (Table 5)—paper should analyze which features matter most and justify the chosen feature set vs. alternatives.
