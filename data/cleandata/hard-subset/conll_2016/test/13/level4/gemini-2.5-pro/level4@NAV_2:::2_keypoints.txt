- Proposes a two-stage neural model for event coreference that uses a CNN to encode sentential context and a trainable similarity function with pairwise features, avoiding external KBs like WordNet/FrameNet.  
- Demonstrates state-of-the-art performance on ACE 2005 for event linking without domain-specific semantic resources, suggesting improved portability.  
- Introduces a new larger dataset (ACE++) of 1,351 additional documents and shows the model benefits significantly from more training data.  

Strengths:
- Shows learned sentential features can replace handcrafted KB-derived features, achieving strong results and arguing for domain portability.  
- Paper is well-written and clearly presents the two-stage model architecture with helpful figures.  
- Provides dataset analysis motivating sentential context use: many coreferent pairs lack shared arguments (50.97%) or identical triggers (47.29%).  

Weaknesses:
- Choice of CNN for sentence encoding is not well-motivated or compared to alternatives (e.g., RNNs); lacks architectural ablation to justify CNN benefits.  
- Technical novelty is limited since CNN-based sentence representations are established; comparisons to other neural coreference methods could be more comprehensive.  
- ACE++ dataset construction details (source, domain, event distribution) are missing, hindering claims about portability and reproducibility; public release recommended.  

Questions for authors:
- Which pairwise features contributed most in the ablation (Table 5), and how was the final feature set chosen?  
- Have you considered integrating general-purpose coreference resolvers (e.g., Stanford CoreNLP) to handle pronoun-related false negatives?  
- How do you expect performance to change in an end-to-end setting using predicted triggers/arguments rather than gold annotations?
