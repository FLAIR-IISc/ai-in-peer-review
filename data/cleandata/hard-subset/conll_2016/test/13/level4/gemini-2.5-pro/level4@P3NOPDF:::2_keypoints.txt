- Paper is well-written and clearly presented.  
- Provides valuable ACE corpus analysis, especially argument sharing and dataset-size effects, motivating ACE++.  
- Proposes a two-part model using CNN-based sentential features for event mention representation and pairwise coreference scoring.  
- Reported improvement over state-of-the-art is modest (2–3 F1 points).  
- Lacks motivation and analysis for choosing a CNN over other sequence models (e.g., RNNs); needs ablation/comparative studies.  
- Novelty is unclear since CNNs for sentential feature extraction are established; must position against other neural/continuous methods.  
- Baseline comparisons are incomplete; Table 3 should include more recent and relevant baselines.  
- ACE++ dataset details (collection, annotation, covered event types, diversity) are missing and must be provided.  
- Authors should demonstrate domain-independence (e.g., on datasets where resource-heavy methods fail) and release ACE++ publicly.  
- Experimental results should report performance with predicted mentions, not only gold mentions.  
- Discuss potential integration with general-purpose coreference systems (e.g., leveraging CoreNLP links).  
- Evaluate on other resources (e.g., EventCorefBank) to show generalizability and motivate joint entity-event coreference work.  
- Pairwise features appear crucial; include analysis of which features matter most and the feature-selection rationale.
