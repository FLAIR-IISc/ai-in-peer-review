- Summary: Proposes a CNN-based neural model that builds event mention vectors from context, concatenates neighbor embeddings, and combines them with pairwise linguistic features in a single-layer NN, evaluated on ACE2005 and ACE++ with competitive results.
- Contribution: Introduces a neural architecture that combines learned embeddings with a small curated set of linguistic features, showing embeddings alone are insufficient and lightweight features yield large gains.

- Major concern: Experiments use gold-standard event triggers, making results unrealistic for end-to-end use and creating an unfair comparison to systems using predicted triggers—reporting results with predicted triggers is needed.
- Major concern: Uses a new train/test split instead of established splits, hurting reproducibility and fair comparison; should adopt commonly used splits.

- Clarification needed: Rephrase unclear final statement in the intro about the necessity of cross-sentential information.
- Clarification needed: Explain how position embeddings are initialized/trained and justify using the same embedding for a relative position across different anchors.
- Clarification needed: Specify how left vs. right neighbor distinctions are preserved in the context representation (is max-pooling the only mechanism?).
- Clarification needed: Justify choosing a ±1-word context window versus larger windows or attention mechanisms.
- Clarification needed: Clarify the role and usage of the initial event-mention representation v_e versus the augmented v_{sent+lex}.
- Clarification needed: Detail encoding of pairwise features (binary/continuous/integer), how distance is represented, and whether feature vectors are trainable.

- Suggestion: Evaluate/apply the architecture to entity coreference (e.g., OntoNotes) to broaden impact and comparisons.
- Suggestion: Discuss the novelty and properties of using a square function as a non-linearity and its applicability elsewhere.
- Suggestion: Release the ACE++ dataset or at least include baseline comparisons on ACE++; private data impedes reproducibility.
- Suggestion: Perform statistical significance testing for reported performance differences.
- Suggestion: Add missing related work, notably Wiseman et al. (2015) on neural mention-ranking for coreference.

- Minor: Remove redundant "that" in "...assume that for an anaphoric mention..." (line 143).
