- Proposes a two-stage neural event coreference model using CNN-derived sentential features plus pairwise features and a trainable similarity function, evaluated on ACE2005 and a new ACE++ dataset with state-of-the-art results.
- Claims the model shows learned sentential features can replace external knowledge bases and improve portability across domains.
- Introduces ACE++, a larger dataset intended to demonstrate better generalization for event linking.
- Paper is well-written with clear motivation and a useful preliminary analysis showing low argument/trigger overlap in coreferential pairs.
- Ablation study effectively shows sentential features are critical and quantifies contributions of pairwise, local, and sentential feature classes.
- Reduces reliance on external semantic resources, improving portability to domains lacking WordNet/FrameNet.
- Insufficient justification for choosing a CNN over alternatives (e.g., RNN/LSTM); lacks empirical comparison given modest performance gains.
- Limited novelty: using CNN sentence representations for classification is incremental and related neural approaches are not fully compared.
- ACE++ lacks crucial documentation about source documents, annotation process, event type coverage, and public release plans, limiting reproducibility and claims of domain-independence.
- Request for details on pairwise feature engineering: which features were most effective and which were discarded, since pairwise features still contribute strongly.
- Suggests integrating event pronoun resolution with a standard pronoun coreference system and asks if that was considered.
- Asks how performance would change in end-to-end settings with predicted triggers/arguments and suggests evaluating on other resources (e.g., EventCorefBank) given dataset size concerns.
