- Proposes a two-part event entity linking model using a CNN to produce sentential mention representations and a pairwise coreference scorer.

- Paper is well-written and clear; ACE argument-sharing analysis is insightful and dataset-size discussion motivates ACE++.

- Major weakness: no justification for choosing a CNN over alternatives (e.g., RNN), nor analysis of trade-offs given modest F1 gains.

- Major weakness: novelty is unclear because CNN feature extraction is standard; authors must position against existing continuous-space event linking methods and expand baselines.

- Major weakness: ACE++ dataset creation methodology, event types covered, and release plan are not described; claims of domain-independence are unsubstantiated without this information.

- Suggestion: include model performance without gold references in Table 3 for a more realistic evaluation.

- Suggestion: explore augmenting the model with standard coreference systems (e.g., CoreNLP) since off-the-shelf coreference could aid event linking.

- Suggestion: evaluate on additional resources (e.g., EventCorefBank) given ACE’s small size to better demonstrate effectiveness and motivate joint entity/event coreference work.

- Observation: Table 5 shows pairwise features are highly effective; authors should analyze which features matter most and how the feature set was chosen.
