- Proposes a neural event-linking model using CNN-generated sentential features and pairwise features, evaluated on ACE and ACE++ with comparable performance.
- Introduces a neural approach combining word embeddings and linguistic features that avoids external resources.
- Shows sentential features from CNNs capture relevant information for event coreference decisions.
- Provides a comprehensive evaluation against state-of-the-art systems, reporting competitive results.
- Effectively uses CNNs to generate sentential features for event linking.
- Achieves performance comparable to previous feature-rich systems on ACE.
- Presents a novel approach that does not rely on manually crafted external features.
- Relies on gold trigger words rather than predicted ones, limiting real-world applicability.
- Uses different train/test splits than prior work, potentially undermining fair comparison.
- Lacks clear explanation of position embedding generation and left/right neighbor usage.
- Requests clarification on how position embeddings are generated and applied in the model.
- Asks how the authors will address using predicted triggers instead of gold triggers in future work.
- Seeks more insight on sentential features’ utility and potential applications beyond event linking.
- Recommends a more detailed description of the neural architecture and sentential features.
- Suggests performing error analysis to reveal model limitations.
- Advises a more comprehensive comparison to other neural network approaches for event linking.
