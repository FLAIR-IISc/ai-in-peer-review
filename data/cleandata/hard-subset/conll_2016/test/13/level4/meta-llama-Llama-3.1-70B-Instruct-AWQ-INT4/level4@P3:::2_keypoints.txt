- Proposes CNN-based sentential features for event entity linking to avoid external knowledge sources.  
- Introduces a two-part model: event mention representation learning and pairwise coreference scoring.  
- Reports state-of-the-art performance on the ACE dataset without external features.  
- Paper is well-written, clearly presented, and easy to follow.  
- Provides insightful analysis of argument sharing between event coreferences on ACE.  
- Creates the ACE++ dataset, offering a larger resource for event entity linking.  
- Model performance on ACE is competitive with existing systems.  
- Fails to motivate or analyze why CNNs were chosen; lacks comparison with RNNs or alternatives.  
- Novelty of using sentential features is not clearly established; needs explicit comparison to other continuous-space methods.  
- Baseline/method choices in Table 3 are incomplete; more state-of-the-art comparisons needed.  
- No details on how ACE++ was collected or which event types it includes.  
- Lack of dataset details makes it hard to assess the model’s generalizability.  
- Requests: provide collection methodology and event types covered by ACE++.  
- Requests: clarify plans for public release of ACE++.  
- Requests: compare model performance with other SOTA systems on ACE++.  
- Asks how the model could be augmented with a vanilla coreference system and potential trade-offs.  
- Suggests reporting model performance without gold references in Table 3.  
- Recommends exploring integration with other coreference resolution systems.  
- Recommends testing on other datasets (e.g., EventCorefBank) to evaluate generalizability.  
- Suggests reporting feature engineering details and which features were most effective.
