- Proposes a novel event entity linking approach using sentential CNN features and a two-part model (mention representation + coreference scoring).  
- Paper is well-written, with insightful analysis of argument sharing in ACE and a compelling motivation for creating the ACE++ dataset.  
- Major concern: insufficient motivation/analysis for choosing CNNs over alternatives (e.g., RNNs) and no architecture comparisons despite only modest (2–3 F1) gains.  
- Major concern: unclear novelty of using sentential features for event linking—authors must explicitly state contributions and compare to existing continuous-space methods.  
- Major concern: ACE++ dataset collection process and covered event types are poorly documented, undermining claims of domain-independence; public release is needed.  
- Minor: report model performance without gold references in Table 3 for completeness.  
- Minor: consider augmenting with a vanilla coreference resolver (e.g., CoreNLP) as a baseline or hybrid approach.  
- Minor: evaluate on other datasets (e.g., EventCorefBank) to demonstrate robustness and generalizability beyond ACE.  
- Minor: pairwise features remain important—report the most effective features, feature-selection process, and alternative features tried.
