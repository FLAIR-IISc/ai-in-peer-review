- Proposes a novel event entity linking approach using sentential CNN features and a two-part model (mention representation + coreference scoring).  
- Paper is well-structured, clearly written, and easy to follow.  
- Good analysis of argument sharing in ACE and a compelling motivation for creating the ACE++ dataset.  
- Insufficient justification for choosing CNNs over alternative architectures (e.g., RNNs); needs analysis of pros/cons given modest F1 gains.  
- Novelty of using sentential features for event linking is not clearly established; approach resembles prior classification methods.  
- Lacks explicit comparison to existing continuous-space event linking methods and a more thorough evaluation in Table 3.  
- ACE++ dataset collection process and characteristics are not described; unclear which event types are covered and how original ACE biases are addressed.  
- Authors should make ACE++ publicly available to enable future comparisons.  
- Table 3 should include model performance without gold references to provide additional insight.  
- Consider augmenting the model with a vanilla coreference resolution system, as suggested by examples.  
- Model should be tested on other resources (e.g., EventCorefBank) to demonstrate robustness and motivate joint entity/event coreference.  
- Pairwise features remain effective; the paper needs a more detailed discussion of feature selection and exploration of alternative features.
