- Paper proposes event entity linking using sentential CNN features with a two-part model: event mention representation + pairwise coreference scorer.
- Paper is well-written, easy to follow, with strong analysis of argument sharing in ACE and dataset-size motivation leading to ACE++.
- Major omission: no motivation or analysis for choosing CNNs over alternatives (e.g., RNNs) and no ablation showing architectural superiority despite small F1 gains.
- Novelty unclear: using sentential CNNs for classification is not new; the paper fails to position itself against existing continuous-space event linking methods.
- Baselines in Table 3 are incomplete; comparison set is not thorough enough.
- No details on ACE++ collection or covered event types; this undermines claims about domain-independence and reproducibility.
- Need empirical evidence that prior resource-heavy methods fail where this approach succeeds, and ACE++ should be publicly released for future comparison.
- Minor: report model performance without gold references in Table 3.
- Minor: explore augmenting with an off-the-shelf coreference system (e.g., CoreNLP) as it may help linking (example cited).
- Minor: evaluate on other corpora (e.g., EventCorefBank) and consider joint entity+event coreference experiments given ACE’s small size.
- Minor: pairwise (engineered) features are effective — report which features mattered, how they were chosen, and what was tried/omitted.
