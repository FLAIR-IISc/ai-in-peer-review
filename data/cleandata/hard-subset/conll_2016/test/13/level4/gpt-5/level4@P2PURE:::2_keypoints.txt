- Proposes CNN-based sentence-level features for event entity linking with two modules: event-mention representation and pairwise coreference scoring.

- Paper is well written; ACE corpus analysis on argument sharing and dataset size effects motivates creating ACE++.

- Major: Choice of CNNs is insufficiently motivated or analyzed; no comparison to alternatives (e.g., RNNs) despite modest 2–3 F1 gains.

- Major: Novelty unclear—using sentential CNN features for classification is not new; must identify and compare to existing continuous-space event linking approaches and strengthen baselines.

- Major: ACE++ dataset collection is not described; need details on included event types, public release, and evidence that prior methods fail where this approach succeeds to support resource-independence claims.

- Minor: Report model performance in Table 3 without gold references.

- Minor: Explore augmenting the model with an off-the-shelf coreference system (e.g., CoreNLP) as it can provide useful links (e.g., pronoun resolution).

- Minor: Evaluate on additional resources beyond ACE (e.g., EventCorefBank) and consider joint entity/event coreference experiments.

- Minor: Table 5 implies pairwise features are highly effective—report which features were most effective, how the feature set was selected, and what alternatives were tried.
