- Presents a competitive neural model for event linking using CNN-derived sentential features without external knowledge bases.  
- Empirically validates that sentential features can substitute for domain-specific external resources.  
- Introduces the ACE++ dataset as a larger testbed for event linking (not yet public).

- Achieves state-of-the-art or comparable performance on ACE and strong results on ACE++.  
- Novel application of CNNs to generate sentential representations for event coreference.  
- Effectively shows sentential features compensate for lack of external semantic resources.  
- Thorough experimental evaluation including ablations and error analysis.

- Core components (CNNs, embeddings, pairwise features) lack novelty; contribution is integration/application.  
- ACE++ is not publicly available, hindering reproducibility and comparison.  
- Evaluation uses gold trigger words, making comparisons to systems using predicted triggers unfair.  
- No statistical significance testing provided for close result differences.  
- Several design choices and implementation details are unclear or missing (window size, pairwise feature encoding, use of v_e).

- Will ACE++ be released publicly, and when?  
- How are pairwise features encoded and are their parameters learned?  
- How is the event-mention representation v_e used later in the model?  
- Is the square function as a nonlinearity novel and why was it chosen over common activations?  
- What is the impact on performance when using predicted trigger words?

- Minor editorial issues: redundant "that" on line 143 and inconsistent baseline naming (“same type” vs “same event”).  
- Suggest citing Wiseman et al. (2015) on neural coreference resolution.

- Overall recommendation: accept with minor revisions contingent on addressing data availability, fairness of comparisons, significance testing, and clarifying design details.
