- Proposes a two-part event coreference model using sentential features from CNNs to score mention pairs.  
- Achieves state-of-the-art results on ACE 2005 and a newly created ACE++ dataset.  
- Main contribution includes creating and initially evaluating the ACE++ dataset to address ACE's size limitations.  
- Writing is clear and the error analysis provides useful insights into failure cases.  
- No justification provided for choosing CNNs over RNNs or Transformers; experiments with alternatives are not reported.  
- Reported 2–3 point F1 gains are modest and insufficient without comparative studies against other sequence models.  
- Novelty is limited: applying CNNs to sentence-level features is not clearly a novel methodological contribution.  
- Baseline comparisons are incomplete; missing comparisons to contemporary continuous-space event linking methods.  
- ACE++ creation methodology is poorly described: lacks details on event-type coverage, annotation guidelines, and bias mitigation.  
- Releasing ACE++ would be valuable only if accompanied by thorough documentation and validation that it addresses ACE shortcomings.  
- Requests: justify CNN choice or compare to RNN/Transformer alternatives; detail ACE++ data collection and bias control.  
- Requests: confirm ACE++ release and documentation plan; provide an ablation study of pairwise, local, and sentential features.  
- Additional suggestions: evaluate on other datasets (e.g., EventCorefBank), explore integrating off-the-shelf coreference tools, and consider joint entity/event coreference experiments.
