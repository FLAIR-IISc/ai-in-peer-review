- Proposes a two-part event entity linking model that uses CNN-extracted sentential features to represent mentions and score coreference pairs.  
- Demonstrates state-of-the-art performance on ACE and a newly created ACE++ dataset.  
- ACE++ dataset creation is a major contribution, motivated by analysis of argument sharing in ACE.  
- Expanded ACE++ corpus could be valuable if publicly released.  
- Empirical results provide a strong baseline and show effectiveness of the proposed model.  
- Provides an empirical exploration of CNN-based sentential features and their impact on event coreference.  
- Clear problem definition and motivation, emphasizing incomplete arguments and cross-sentential inference.  
- Thorough experimental evaluation, including ablation studies and comparisons to existing methods (though limited).  
- Detailed model description with helpful figures and component rationale.  
- Insightful error analysis highlighting pronoun resolution and annotation inconsistencies.  
- Well-written, organized paper with clear presentation of results.  
- Weak justification for choosing CNNs; lacks discussion of alternatives (RNNs/Transformers) and empirical comparisons.  
- Limited novelty: CNN-over-sentence approach is not new and baseline comparisons are insufficient.  
- ACE++ lacks detailed description of collection, annotation procedures, event type coverage, and bias mitigation.  
- Unclear whether ACE++ will be publicly released—critical for the dataset's value.  
- Reporting performance with gold event mentions (Table 3) reduces interpretability; missing evaluation with automatically extracted mentions.  
- Questions for authors: rationale and experiments comparing CNNs to RNNs/Transformers; detailed ACE++ collection/process and release plans; methods to address pronoun/coreference errors; evaluation using automatic mention detection.  
- Additional suggestions: integrate off-the-shelf coreference systems, evaluate on EventCorefBank, consider joint entity/event coreference, and provide a more detailed feature-importance analysis for Table 5.
