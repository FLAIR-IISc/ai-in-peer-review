- Proposes a novel event linking approach using sentential CNN features and a two-part model for mention representation and coreference scoring.
- Reports state-of-the-art performance on two datasets, one publicly available.
- Avoids reliance on external semantic features by using CNN-derived sentential features.
- Paper is well-written with clear model description and insightful error analysis.
- Provides a thorough analysis of argument sharing in ACE and motivates creation of ACE++.
- Does not justify why CNNs were chosen for mention representations or analyze alternatives (e.g., RNNs).
- Lacks comparisons with other architectures; the reported 2–3 point F1 gains need stronger architectural evidence.
- Novelty is unclear since sentence-level CNN classification is not new; needs comparison to existing continuous-space event-linking methods.
- Baselines in Table 3 are insufficient and should be expanded.
- ACE++ data collection is under-described—event-type coverage and dataset biases are unclear.
- Authors should demonstrate where earlier methods fail and consider releasing ACE++ for reproducibility.
- Should report model performance without gold references in Table 3.
- Suggests augmenting with off-the-shelf coreference systems (e.g., CoreNLP) and investigating joint entity/event coreference given ACE’s small size.
- Recommend evaluating on additional resources (e.g., EventCorefBank).
- Table 5 indicates pairwise features remain important; authors should report which features mattered, selection criteria, and alternative features tried.
