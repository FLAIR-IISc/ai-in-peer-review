- Proposes a neural event linking model using CNN-generated sentential features combined with pairwise features to score coreference.
- Claims state-of-the-art performance on ACE and ACE++ without relying on external semantic features.
- Contribution: integrates word embeddings with CNN-derived sentential features for event linking.
- Contribution: relies on a small set of linguistic features to achieve strong performance without extensive external resources.
- Contribution: empirically evaluates the approach on ACE and ACE++ datasets.
- Strength: novel neural approach that leverages CNNs to produce sentential features for event coreference.
- Strength: achieves strong empirical results (claimed SOTA) on ACE and ACE++.
- Strength: provides thorough evaluation, including analysis of feature classes and clustering strategies.
- Weakness: uses gold-annotated event mentions, which may not reflect real-world use with predicted mentions.
- Weakness: unclear justification or evidence for claims about the need for cross-sentential information.
- Weakness: lacks significance testing to support performance claims.
- Question: how will the model perform with predicted (non-gold) event mentions and how will the authors address this?
- Question: request for detailed CNN architecture specifics (convolution window sizes, number of filters).
- Question: how will the model be extended to cross-document event coreference and what challenges are expected?
- Additional comment: paper needs a more detailed explanation of the CNN architecture for sentential features.
- Additional comment: recommend evaluating on more datasets (e.g., OntoNotes) to show generalizability.
- Additional comment: suggest a more thorough related-work discussion comparing approaches in event coreference.
- Rating: 7/10 — promising and novel with strong results but limited by gold mentions and missing statistical validation.
