- Proposes a neural event-linking model using sentential CNN features that avoids external semantic resources and claims state-of-the-art performance.
- Shows that domain-specific semantic resources are not necessary for high event-linking performance.
- Provides an ablation study isolating contributions of sentential, lexical, and pairwise features.
- Performs error analysis highlighting challenges like pronoun resolution and boundary cases in event coreference.
- Achieves competitive performance on ACE without external resources and generalizes on ACE++ with more training data.
- Ablation results demonstrate substantial performance drops when removing any feature class, indicating their combined necessity.
- Error analysis finds pronoun resolution as a major limitation and notes 22% of false positives were plausible to human raters.
- Uses gold-standard triggers in experiments, making comparisons to prior work unfair and likely inflating performance.
- ACE++ dataset is not publicly available, limiting reproducibility and comparison to feature-rich systems.
- No statistical significance testing is reported, so small reported improvements may be meaningless.
- Several model details are unclear (positional embedding generation, neighbor representation, pairwise feature encoding) and lack justification.
- Fails to compare with recent neural coreference approaches (e.g., Wiseman et al., 2015) which would strengthen the claim for neural methods.
- Requests results with predicted triggers or justification for using gold triggers to ensure fair comparisons.
- Asks whether the authors will release ACE++ to enable reproducibility and further research.
- Questions the choice and theoretical justification for using the square function as the network non-linearity versus common activations like ReLU.
- Asks how the 100 false positives/negatives were sampled for error analysis and requests comparison of ACE++ results to prior work.
