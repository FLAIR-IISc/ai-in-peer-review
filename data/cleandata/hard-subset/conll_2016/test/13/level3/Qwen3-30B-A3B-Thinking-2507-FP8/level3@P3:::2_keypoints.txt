- Proposes a CNN-based model for event linking using sentential features instead of external semantic resources and reports state-of-the-art results on ACE and ACE++.
- Introduces ACE++, an expanded dataset (1,950 docs) augmenting ACE 2005 with 1,351 web documents for more robust evaluation.
- Provides detailed feature importance analysis and error analysis highlighting challenges for future work.

- ACE++ addresses ACE 2005's small size and shows many coreferential mentions are >5 sentences apart, improving benchmark robustness.
- Error analysis pinpoints hard cases like pronoun resolution and contextual difficulty, guiding future research.
- Feature ablation convincingly shows all feature classes contribute meaningfully to performance.

- No justification or comparison for choosing CNNs over RNNs/LSTMs; architectural choice lacks empirical support.
- Novelty is unclear because CNNs for sentence tasks are standard and comparisons omit recent neural continuous-space baselines.
- ACE++ collection/annotation details (event-type coverage, selection criteria, potential biases) are insufficiently described.
- Evaluation omits standard reporting with gold references, hindering proper comparison due to sensitivity to reference sets.
- The paper doesn't explore simple augmentations (e.g., off-the-shelf coreference systems) that could improve performance.

- Please compare the CNN approach to equivalent RNN/LSTM models on ACE++ and report performance differences.
- Clarify ACE++ data collection: selection criteria for the 1,351 web docs and distribution of event types across the corpus.
- Will you release ACE++ for the community, and if not, what prevents its release?
