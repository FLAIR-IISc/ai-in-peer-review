- Paper proposes a within-document event coreference model using a CNN with piecewise max-pooling to learn sentential event representations, combined with local lexical and pairwise features and a squared-activation similarity network, evaluated on ACE 2005 and ACE++ achieving competitive results without external KBs.

- Contribution: shows learned sentential CNN features can replace external semantic resources, improving portability while reaching state-of-the-art performance.

- Contribution: provides a thorough empirical analysis (ablation study) demonstrating complementary value of pairwise, local-lexical, and sentential features, plus insightful qualitative error analysis.

- Strength: addresses dependency on external knowledge bases with a portable, text-driven approach that attains strong empirical results.

- Strength: comprehensive evaluation against strong baselines on ACE and ACE++, with convincing ablation results showing each feature class matters.

- Strength: valuable practical comparisons (transitive closure vs BestLink) and excellent qualitative error analysis highlighting hard phenomena and annotation ambiguities.

- Strength: clear model description and justified design choices (piecewise pooling, squared activation) with helpful figures.

- Weakness: limited methodological novelty—uses established CNN/piecewise pooling architecture applied to a new task, making the contribution mainly empirical.

- Weakness: reliance on proprietary ACE++ dataset for key analyses reduces reproducibility and long-term community impact.

- Weakness: state-of-the-art claim is somewhat ambiguous—mixed metric results versus prior work (better on BLANC/MUC, worse on B-CUBED) warrant deeper discussion.

- Question: please detail other similarity-layer variants tested (e.g., ReLU/tanh, bilinear, cosine) and report the performance gap versus the squared activation.

- Question: how are pronouns represented by embeddings in the model, and how might the CNN architecture be adapted to better capture long-distance/pronominal event coreference?

- Question: can the authors release the document IDs for the ACE 2005 train/validation/test splits to improve reproducibility (train split confirmation beyond provided valid/test links)?
