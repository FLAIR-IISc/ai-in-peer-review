- Model uses CNN-generated sentential features for event linking with a two-part architecture: event mention representation and pairwise coreference scoring.  
- Achieves state-of-the-art performance on ACE 2005 without external semantic features.  
- Provides extensive evaluation and error analysis that highlights event-linking challenges.  
- Novel application of sentential features makes the approach more domain-independent.  
- Demonstrates effective use of CNNs to generate sentential features for event linking.  
- Evaluation is comprehensive and includes insightful error analysis.  
- Limited generalizability since evaluation is confined to ACE 2005.  
- Depends on gold-annotated event mentions, reducing real-world applicability.  
- Error analysis reveals difficulty resolving pronouns to prior event mentions.  
- Question: How does the model perform on other datasets (e.g., OntoNotes, EventCorefBank)?  
- Question: Can the model be adapted to use automatically generated event mentions?  
- Question: How can the model be improved to better resolve pronouns to previous events?  
- Overall: Promising, domain-independent approach but constrained by dataset scope and reliance on gold annotations.
