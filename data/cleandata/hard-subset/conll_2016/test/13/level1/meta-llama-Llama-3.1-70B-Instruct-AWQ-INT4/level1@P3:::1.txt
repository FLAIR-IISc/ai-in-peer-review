**Summary of the Paper**

This paper proposes a model for event linking that utilizes sentential features from convolutional neural networks (CNNs). The model consists of two parts: the first part generates a representation for a single event mention, and the second part calculates a coreference score for a pair of event mentions. The model is trained on the ACE 2005 dataset and achieves state-of-the-art performance without relying on external semantic features.

**Main Contributions**

1. **Novel use of sentential features for event linking**: The paper proposes a new approach to event linking by utilizing sentential features generated by CNNs. This approach does not rely on external semantic features, making it more domain-independent.
2. **State-of-the-art performance on ACE 2005**: The model achieves state-of-the-art performance on the ACE 2005 dataset, outperforming previous systems that rely on external semantic features.
3. **Extensive evaluation and analysis**: The paper provides an extensive evaluation and analysis of the model's performance, including an error analysis that highlights the challenges of event linking.

**Strengths**

1. **Effective use of CNNs for sentential feature generation**: The paper demonstrates the effectiveness of using CNNs to generate sentential features for event linking.
2. **Domain-independent approach**: The model's ability to perform well without relying on external semantic features makes it a more domain-independent approach.
3. **Comprehensive evaluation and analysis**: The paper provides a thorough evaluation and analysis of the model's performance, including an error analysis that highlights the challenges of event linking.

**Weaknesses**

1. **Limited generalizability**: The model is only evaluated on the ACE 2005 dataset, and its performance on other datasets is unknown.
2. **Dependence on gold-annotated event mentions**: The model relies on gold-annotated event mentions, which may not be available in all scenarios.
3. **Error analysis highlights challenges**: The error analysis highlights the challenges of event linking, including the difficulty of resolving pronouns to previous event mentions.

**Questions to Authors**

1. How does the model perform on other datasets, such as OntoNotes or EventCorefBank?
2. Can the model be adapted to work with automatically generated event mentions instead of gold-annotated ones?
3. How can the model be improved to better handle the challenge of resolving pronouns to previous event mentions?

**Additional Comments**

The paper provides a novel approach to event linking that demonstrates the effectiveness of using sentential features generated by CNNs. The comprehensive evaluation and analysis provide valuable insights into the challenges of event linking. However, the model's limited generalizability and dependence on gold-annotated event mentions are limitations that need to be addressed in future work.
