- Proposes an event linking model using sentential CNN features to resolve coreferences between event mentions.  
- Model generates single-event representations via CNNs and computes pairwise coreference scores using those representations and pairwise features.  
- Achieves state-of-the-art performance on two datasets, including one public dataset.  
- Novel use of sentential CNN features instead of traditional external semantic features for event linking.  
- Effective two-part architecture that captures relationships between event mentions for coreference resolution.  
- Demonstrates improved performance compared to prior state-of-the-art systems.  
- Thorough evaluation across two datasets showing effectiveness and generalizability.  
- Direct comparison to state-of-the-art systems clarifies strengths and weaknesses.  
- Includes error analysis that highlights areas for improvement and task challenges.  
- Evaluation limited to only two datasets, potentially restricting domain and genre coverage.
