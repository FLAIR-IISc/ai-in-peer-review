Summary and contributions
- The paper proposes a two-stage neural model for within-document event coreference that avoids external semantic resources. A CNN encodes sentential representations of each event mention using word and position embeddings with piecewise max pooling around the trigger, augmented by trigger-local lexical features; a learned similarity function over paired mention representations plus pairwise features yields a coreference score, followed by transitive-closure clustering.
- Contributions as assessed:
  1) Demonstration that sentence-level neural representations can replace external semantic resources while achieving competitive or state-of-the-art results on ACE 2005 (BLANC/MUC/positive-link F1) and a larger ACE++ variant.
  2) Careful empirical analysis: ablations of feature classes (pairwise vs lexical vs sentential), comparison of clustering strategies (BestLink vs transitive closure), dataset-size effects (ACE vs ACE++), and baselines.
  3) Error analysis identifying key failure modes (e.g., pronoun-mediated links and annotation edge cases).

Strengths
- Empirical strength with reduced dependence on external resources: On ACE, the model matches or surpasses prior work that relies on extensive semantic resources (e.g., WordNet/FrameNet, 105 semantic features in Liu et al., 2014). The BLANC F1 improvement over Liu et al. (2014), together with gains on MUC and positive-link F1, meaningfully supports the claim that external semantic features are not strictly necessary for strong performance.
- Well-justified modeling choices and analysis: The use of sentence-level CNN representations aligns with observed data properties (nearly half of positive pairs lack argument overlap and nearly half differ in triggers), and the architecture encodes both positional and lexical cues relevant for event linking. The square non-linearity for similarity is motivated (polarity invariance) and compared against alternatives (reported to outperform cosine and other activations).
- Thorough ablations: Removing sentential and/or lexical features degrades performance in expected ways; the analysis that sentential features can collapse into trigger-local signals when lexical features are removed is insightful. The dataset-size study (ACE → ACE++) shows substantial recall gains, underscoring scalability.
- Clustering strategy comparison: The finding that transitive closure outperforms BestLink in this model (opposite to Liu et al., 2014) is notable and suggests the classifier already filters low-confidence links, making additional pruning counterproductive.
- Clear problem framing and metrics: The discussion of metric validity (BLANC in the presence of severe class imbalance) is appropriate, with complementary reporting of B^3, MUC, and a positive-link metric.

Weaknesses
- Use of gold annotations and comparability: The setup uses gold event triggers; it is unclear to what extent event types, modality/realis, and argument overlaps are gold vs. predicted. Since pairwise features include type agreement, realis match, and argument overlap, reliance on gold labels could inflate performance and complicate comparison with systems using predicted arguments/SRL (e.g., Liu et al., 2014). A controlled experiment with predicted upstream outputs would strengthen claims of robustness and portability.
- Dataset and reproducibility: The ACE++ corpus is not described in sufficient detail for reproducibility (annotation process, guidelines, inter-annotator agreement, license). Without releasing ACE++ or a way to reproduce it, the strongest claims rely on a non-standard resource. Code, model checkpoints, and exact splits for ACE/ACE++ are not provided.
- Metrics and reporting: CEAF (particularly CEAF_e/CEAF_m) and the CoNLL average are standard in coreference evaluation and would facilitate broader comparability, even if BLANC is argued to be more informative here. Threshold selection for link decisions is fixed at 0.5; sensitivity analysis would help assess stability.
- Limited document-level context: The model encodes each mention using only its sentence, then relies on pairwise features and a learned similarity for cross-sentential pairs. The error analysis highlights pronoun-mediated links as a major source of false negatives; this suggests that integrating discourse/coreference signals for events and entities or lightweight document-level context could yield further gains.
- Novelty scope: While applying CNN-based sentential features to event linking is a timely and useful extension, the architectural novelty is incremental relative to prior CNN-based representations for relation/event extraction; the contribution is primarily empirical and integrative rather than algorithmically groundbreaking.

Questions to authors
- What is the supervision source for pairwise features: are event types, modality/realis, and argument spans/roles gold or predicted? If any are predicted, which tools and with what accuracy? If gold, can robustness be shown with predicted upstream outputs?
- Will ACE++ (documents, annotations, splits) be released? Please provide annotation protocol, quality control, and inter-annotator agreement. If release is not possible, can a subset or detailed conversion pipeline be shared?
- How are multi-token triggers handled in position embeddings and pooling? For sentences with multiple event mentions, how is the target mention indexed and how are overlapping contexts disambiguated?
- How was the 0.5 decision threshold selected? Was it tuned on validation? Can precision–recall curves or threshold sensitivity be provided?
- The square activation reportedly outperforms alternatives. Could quantitative comparisons (e.g., cosine, ReLU/tanh, bilinear) be added to the ablation table?

Additional comments and suggestions
- Add CEAF metrics and (optionally) the CoNLL average to facilitate comparison with a broader set of coreference work.
- Provide per-type results and performance as a function of sentence distance and trigger agreement to better characterize where sentential features help most.
- Consider a lightweight document-level extension (e.g., attention over nearby sentences, entity coreference features, or joint modeling of entities and events) to address pronoun-mediated links.
- Release code, trained models, and ACE split lists; for ACE++, provide as much reproducible detail as licensing allows.

Overall, the paper presents a solid, well-executed application of sentential CNN features to event linking with competitive results and meaningful analyses. Clarifying the role of gold vs. predicted annotations, improving reproducibility, and expanding evaluation metrics would strengthen the submission.
