- Proposes a two-stage neural model using sentential CNN features, position embeddings, and trigger-local lexical features to score pairwise event coreference; reports SOTA on ACE and ACE++ with ablations and error analysis.

Main contributions:
- Shows sentential CNN features plus lightweight pairwise features can match or outperform resource-heavy baselines.
- Provides a systematic empirical study isolating sentential vs. lexical vs. pairwise features and comparing clustering strategies (transitive closure vs. BestLink).
- Constructs and evaluates an expanded ACE++ dataset and analyzes its properties and model behavior.

Strengths:
- Strong empirical improvements across metrics and clear ablations attributing gains to sentential features.
- Careful experimental design: multiple coreference metrics, data analyses (argument overlap, trigger mismatch, distance), and motivated architecture choices.
- Practical: avoids reliance on WordNet/FrameNet/KBs, improving cross-domain portability.
- Transparent methodology with training details, hyperparameters, example generation, clustering procedure, and published ACE splits.

Weaknesses:
- Uses gold event triggers and partial gold arguments, reducing realism; no evaluation with predicted triggers.
- Comparability on ACE weakened by varying splits/preprocessing and lacks statistical significance testing.
- Unclear whether ACE++ and code will be released, hindering reproducibility of ablations and conclusions.
- Limited global inference — transitive closure can propagate local errors; no learned agglomeration or cluster-level constraints explored.
- Fails on pronominal/event anaphora and lacks discourse/coreference signals (entity coref, lexical chains) to address these errors.

Questions for authors:
- Will ACE++ and the code be released, or can you provide detailed reproduction guidelines for the annotations?
- How does performance change when using predicted triggers and arguments?
- Did you test alternative similarity functions (e.g., cosine, bilinear) and run significance tests?
- What are runtime and model size compared to feature-rich baselines?
