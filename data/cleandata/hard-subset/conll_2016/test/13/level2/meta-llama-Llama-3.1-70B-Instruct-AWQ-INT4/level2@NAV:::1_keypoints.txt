- Paper proposes a two-part CNN-based model using sentential features for event linking, producing single-event and pairwise representations.  
- Claims state-of-the-art performance on two datasets (one public) without external semantic features.  
- Contribution: novel sentential-CNN approach for event linking that avoids external semantic resources.  
- Contribution: model accounts for local and sentential features and aims to address pronoun resolution.  
- Strength: novel and effective approach achieving SOTA on two datasets.  
- Strength: thorough performance analysis including error analysis highlighting pronoun resolution issues.  
- Strength: demonstrates neural models can generate useful latent sentence representations.  
- Weakness: heavy reliance on ACE 2005 dataset, limiting domain generality.  
- Weakness: lacks detailed comparison with other state-of-the-art systems.  
- Weakness: unclear how the model handles multiple event mentions within a single sentence.  
- Question: please clarify how the model handles multiple event mentions in one sentence.  
- Question: how will the model be extended to cross-document event linking?  
- Question: provide more detailed error analysis — most common error types and potential fixes.  
- Additional: paper would benefit from clearer comparisons to other systems and more explicit handling of pronoun/multiple-mention cases.
