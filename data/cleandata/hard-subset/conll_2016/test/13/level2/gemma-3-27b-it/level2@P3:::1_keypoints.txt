- Paper proposes an event linking system using CNN-derived sentential features to reduce dependence on external semantic resources.
- Core contribution: adapting CNN-based feature extraction methods for event coreference resolution.
- Secondary contribution: state-of-the-art results on ACE and competitive performance on the newly released ACE++ dataset.
- Contribution: public release of the ACE++ dataset.
- Strength: strong empirical results, with notable gains on ACE++.
- Strength: novel application of CNNs to generate sentential features for event linking without external KBs.
- Strength: comprehensive error analysis identifying pronoun resolution issues and annotation inconsistencies.
- Strength: model design choices are well-motivated by dataset characteristics (e.g., non-overlapping arguments, mention distances).
- Weakness: limited comparison to recent transformer- or attention-based event linking approaches.
- Weakness: insufficient detail on hyperparameter search, optimization strategy, and compute resources.
- Weakness: unexpected performance drop with BestLink clustering is noted but not deeply investigated.
- Weakness: interpretation of feature ablation (trigger-focus claim) lacks analysis of learned weights/representations.
- Question: Why did BestLink perform worse than transitive closure—what dataset or model properties explain this?
- Question: Please provide specific hyperparameter ranges, search procedure, and computational budget.
- Question: Provide concrete examples from the error analysis illustrating pronoun resolution failures.
- Additional: writing is clear and figures/tables are helpful.
- Additional: related work should be updated to include newer methods.
- Additional: public release of ACE++ is a valuable resource for the community.
