

Summary: The paper proposes a neural network-based approach for event coreference resolution that utilizes sentential features from convolutional neural networks instead of relying on external semantic resources like WordNet or FrameNet. The main contributions are a two-step model architecture where CNNs generate event mention representations from sentential context, and a coreference scoring function combining these representations with pairwise features. The model achieves state-of-the-art performance on two datasets without external semantic features.

Strengths:
1. The avoidance of external semantic resources is a significant contribution, as it addresses a key limitation of previous approaches that require domain-specific resources. The model demonstrates that high performance can be achieved without these restrictive features, enabling broader application across domains.

2. The experimental evaluation is thorough, with clear ablation studies showing the contribution of each feature class (Table 5) and a detailed error analysis identifying specific limitations (e.g., pronoun resolution challenges). The results on ACE++ show meaningful improvements over the ACE dataset, demonstrating the model's ability to generalize with more training data.

3. The model design is well-motivated by empirical observations about the data (e.g., 50.97% of coreferential event pairs share no arguments), and the architecture effectively combines local and sentential features in a way that addresses these observations.

4. The work provides valuable evidence that neural approaches can generate effective latent representations for sentential features, contributing to the growing body of literature on neural methods for information extraction tasks.

Weaknesses:
1. The ACE++ dataset construction lacks sufficient detail. The paper states it "encompasses these documents and additionally contains 1351 more web documents annotated in an analogous fashion," but provides no information about the annotation process, guidelines, or quality control measures. Without this information, it's difficult to assess the validity of the results on this dataset.

2. The comparison with prior work (Table 3) has significant methodological inconsistencies that undermine the claim of state-of-the-art performance. Different train/test splits across systems and varying preprocessing pipelines (e.g., some use gold-annotated event mentions while others use semantic-role labelers) make direct comparisons questionable without proper discussion.

3. The novelty claim regarding sentential features is overstated. While the paper cites recent work using sentential features for relation extraction (Zeng et al., 2014; Nguyen and Grishman, 2015), it doesn't adequately distinguish how this work differs from those approaches for the specific task of event coreference resolution. The connection to prior work needs to be more precisely articulated.

4. The model architecture is conventional (CNN for sentential features followed by a similarity function) without sufficient justification for why this particular configuration was chosen over alternatives. The paper doesn't discuss why recurrent networks or other architectures weren't explored, despite mentioning them in the related work.

Questions to Authors:
1. Please provide details about the annotation process for the ACE++ dataset. How were the 1351 additional web documents annotated? What guidelines were used, and how were annotation disagreements resolved?

2. The paper claims state-of-the-art performance, but the comparison in Table 3 has methodological inconsistencies. How can the community properly assess these results given the varying train/test splits and preprocessing pipelines across systems?

3. How does the use of sentential features for event linking differ from the approaches used by Zeng et al. (2014) and Nguyen and Grishman (2015) for relation extraction? What specific challenges of event linking necessitate this adaptation?

Additional Comments:
The error analysis identifying pronoun resolution as a key limitation is valuable, but the paper could better contextualize this within the broader challenges of event coreference resolution. The observation that 17 false negatives corresponded to cases where annotators made different decisions on the same examples suggests that annotation guidelines may need refinement, which is an important consideration for future work.
