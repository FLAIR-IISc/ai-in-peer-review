- Summary: Proposes a CNN-based, feature-agnostic event linking model that replaces external semantic resources with sentential features and provides an ACE++ dataset.  
- Contribution 1: A feature-agnostic event linking model achieving reported SOTA results.  
- Contribution 2: Empirical validation via ablation showing sentential features are critical.  
- Contribution 3: Public release of the ACE++ dataset.  
- Strength 1: Achieves 73.31 BLANC F1 on ACE without domain-specific resources, aiding cross-domain deployment.  
- Strength 2: Robust ablation study showing sentential features dominate performance (large drop when removed).  
- Strength 3: Actionable manual error analysis identifying pronoun resolution and annotation ambiguities.  
- Weakness 1: Misleading SOTA comparison due to dataset mismatch (ACE vs. larger ACE++), invalidating direct comparisons.  
- Weakness 2: Unrealistic reliance on gold event triggers; performance with predicted triggers is unaddressed.  
- Weakness 3: Unexplained ablation anomaly where removing lexical features improves precision without analysis.  
- Question 1: How does performance change when triggers are predicted rather than gold-annotated?  
- Question 2: Why does removing lexical features increase precision despite claims about trigger-word overlap?  
- Question 3: Could adding discourse-level features (e.g., coreference chains) address pronoun resolution failures?
