This paper provides a useful approach for jointly segmenting and transcribing handwritten paragraphs into text. The authors use 1) encoding of the input and 2) an attention network to learn MDLSTMs that effectively segment each line of text. A bidirectional-LSTM decoder is trained to output text corresponding to the combination of encoded sequences. A CTC loss is minimized over the entire paragraph. This paper is a useful advancement of previous work that goes beyond models that require line segmentation, demonstrating the model's real-world applicability. Some feedback on how the paper could have been stronger: a baseline using only convolutional or affinity [5] layers would have been useful because these would have been much simpler to train compared to using MDLSTMs. If another layer of MDLSTMs with CTCs is used to find paragraph boundaries, the model's complexity will increase. While the authors get improvements on their own baseline and better performance on rhyme benchmarks, performance on the IAM benchmark was not better than previous results.
