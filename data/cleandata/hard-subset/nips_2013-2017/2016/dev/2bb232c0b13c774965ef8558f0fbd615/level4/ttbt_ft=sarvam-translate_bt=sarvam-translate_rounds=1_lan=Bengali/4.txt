This work presents a modification of the collapse layer of the standard MD-LSTMS, which is used to recognize handwriting. This work is also closely related to [6], which performs the same function – decoding characters at the level of letters at each time step. The modification is the conversion of global average pooling to a weighted average pooling (Equation 2). Furthermore, these weights are learned using a recurrent attention layer (MD-LSTM) that takes both the previous weights and the encoded feature vectors. A CTC loss is used to calculate the loss for direct transcription without character or line-level alignment. Moreover, the attention mechanism can be interpreted as a "soft" line segmentation. This work compares with other methods, including [6], and achieves better/comparable performance. From my understanding, the main change from [6] to this paper is that a complete line is decoded against a single character at each time step. Although this results in improved speed and performance, I am not entirely sure there is enough technological novelty. (If I am missing something, please correct me, and if so, I would be happy to increase my rating.) Also, it would have been interesting to imagine the attention using more than the number of lines in the input image than the time steps set for running the algorithm – because it has been mentioned that the method performs even better in these situations.
