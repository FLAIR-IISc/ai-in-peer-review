This work presents a modification of the standard MD-LSTMS collapse level, which is used to recognize handwriting. This work is also closely related to [6], which does the same thing – decoding characters at the level of letters at each step. The modification is converting global average pooling to a weighted average pooling (Equation 2). Furthermore, these weights are learned using a recursive attention layer (MD-LSTM) that takes both the previous weights and the encoded feature vector. A CTC loss is used to calculate the loss for direct transcription without character or line-level alignment. Moreover, the attention process can be interpreted as a "soft" line segmentation. This work achieves better/comparable performance compared to other methods, including [6], and from my understanding, the main change in this paper from [6] is that a complete line is decoded against a single character at each step. Although this provides improved speed and performance, I am not entirely sure there is enough technological novelty. (If I miss something, please correct me, and if so, I would be happy to increase my rating.) Also, it would have been interesting to imagine the algorithm running using more attention than the number of lines in the input image than the number of steps – because it has been mentioned that the method works better in these situations.
