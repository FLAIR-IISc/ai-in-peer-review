The paper provides a useful method for segmenting and copying handwritten paragraphs into text. The authors use MLSTM to 1) encode the input and 2) an attention network that effectively segments each line of text. A bidirectional-MLSTM decoder is trained to output the text in line with the connection of the encoded sequences. A CTC loss is minimized across the entire paragraph. This paper is a useful advancement of previous work that extends beyond models requiring line segmentation. The model has real-world applicability. Some feedback on how the paper could have been stronger: there should have been a baseline using only convolutional or affinity [5] layers, as these would have been much easier to train compared to using MLSTM. If another layer of MLSTM with CTC is used to find paragraph boundaries, the model complexity will increase. Although the authors found improvements and better performance on rhyme benchmarks compared to their own baseline, the performance on the IAM benchmark was not better than the previous results.
