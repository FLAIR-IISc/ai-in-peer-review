This research paper is related to the use of tensor networks for supervised learning. Specifically, the authors demonstrate how tensor networks can be adapted to effectively represent data and solve classification problems. The main idea involves considering data feature maps created from the tensor product of local features and representing them as higher-order tensors. Then, a classification problem involves viewing the model parameters as tensor networks and learning them using matrix-product state (MPS) decomposition. An optimization algorithm is described to find MPS decomposition, and an experiment is given on the MNIST digit data set. Finally, a discussion is provided on the class of tasks for which the solution is sought. The idea of using tensor networks for machine learning problems is interesting. Unfortunately, there are several significant issues that prevent the manuscript from being published. My main concerns are as follows: 1- Some parts of the paper are difficult to read. The paper implements and optimizes some ideas about tensor networks that have been used/developed by the physics community for ML problems. The authors have done a good job of capturing ideas and intuition through several figures. However, I think providing more information for an ML audience, tensor networks, MPS decomposition algorithms, and their use for physics problems (in short, more information on papers cited and published in physics journals) would improve the readability and accessibility of the paper. A section could be dedicated to this. 2- The notation is also difficult to follow and does not help with understanding. For example: In equation 2, what is the relationship between x and xj? Is xj a component of the vector x? What is N: is it the number of features? In equation 3, \phi is indexed by sj but depends only on xj. For this example, is it the same \phi for all s_j? Equation 5 needs to define A in line 68: 'labeled data' should be * ... 3- The experiments are weak. To assess the performance of the method, a large data set should be used, and not just simple MNIST digit data. Also, some information about the experimental setting is missing. For example, how was the data set divided into training and test data? 4- I think the choice of local feature map is important. It is not clear how this feature map can be chosen. The equation given in equation 3 seems to be optimized only for image data. The same is true for the parameters d and m. How can these parameters be chosen? 5- Is MPS decomposition unique? If not, how does it affect the learning process? 6- MPS decomposition was used for feature extraction and classification in [4]. I think the novel contributions need to be stated more clearly than in [4]. Line 210 is too short and does not provide enough information to see the difference between the two tasks. 7- ANOVA kernels are based on the product of a set of 'local' kernels. The product of the kernels allows finding a solution in a Hilbert space constructed from the tensor product of the reconstructive Hilbert space associated with each local kernel. I think it is interesting to study the relationship between the location of the tasks obtained by such kernels and the tensor network representation proposed here.
