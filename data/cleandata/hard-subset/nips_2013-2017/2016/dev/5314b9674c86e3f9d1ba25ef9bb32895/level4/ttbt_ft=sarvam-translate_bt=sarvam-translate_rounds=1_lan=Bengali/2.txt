This paper discusses the use of tensor networks for supervised learning. Specifically, the authors demonstrate how tensor networks can be used to create useful representations of data and solve classification problems. The core idea involves considering data feature maps generated from the tensor product of local features and representing them as high-order tensors. Subsequently, a classification problem involves viewing the model's parameters as tensor networks and learning them using matrix-product state (MPS) decomposition. An optimization algorithm is described to find MPS decomposition, and an experiment is conducted on the MNIST digit data set. Finally, a discussion is provided on the functions whose classes the solution is sought in. The concept of using tensor networks for machine learning problems is appealing. Unfortunately, there are many important issues that currently hinder the publication of the manuscript. My main concerns are: 1- Some parts of the paper are difficult to read. The paper applies some concepts about tensor networks and uses them for ML problems. The authors have done a good job of explaining the concepts and intuition through many figures. However, I believe that for ML audiences, providing more detailed information (in short, more detailed information on research papers cited and published in physics journals) about tensor networks, MPS decomposition algorithms, and their use for physics problems would increase the readability and accessibility of the paper. A section could be dedicated to this topic. 2- The notation is also difficult to follow and does not help in understanding. For example: In equation 2, what is the relationship between x and xj? Is xj an element of vector x? What is N: is it the number of attributes? In equation 3, \phi is denoted by sj but it depends only on xj. For this example, is \phi the same for all s_j? In line 68 of equation 5, A should be defined as: instead of 'in the case of labeling data classified', it should be written as 'unlabeled data' (from a given training set) * ... 3- The experiment is weak. To evaluate the effectiveness of the method, more data sets should be used, and not just the standard MNIST digit data. Also, some information about the experimental settings is missing. For example, how was the data set divided into training and test data? 4- I think the choice of local feature map is very important. It is not clear how this feature map can be chosen. The map given in equation 3 seems to be suitable only for image data. The same is true for parameters d and m. How can these parameters be chosen? 5- Is MPS decomposition unique? If not, how does it affect the learning process? 6- MPS decomposition has been used for feature extraction and classification [4]. I think new contributions should be clearly mentioned compared to [4]. Line 210 is too short and does not provide enough detail to see the difference between the two tasks. 7- ANOVA kernels are based on the product of a set of 'local' kernels. The product of the kernels helps to find a solution in a Hilbert space constructed from the tensor product of the reproducing Hilbert space associated with each local kernel. I think it is interesting to study the connection between the space of the functions obtained by such kernels and the tensor network representation proposed here.
