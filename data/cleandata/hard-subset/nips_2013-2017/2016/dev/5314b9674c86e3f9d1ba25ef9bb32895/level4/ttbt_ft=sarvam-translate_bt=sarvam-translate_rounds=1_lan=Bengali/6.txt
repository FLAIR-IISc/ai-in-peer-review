This paper develops an efficient algorithm for image classification. The authors demonstrate how a tensor network optimization algorithm can be used for supervised learning using the matrix product state. Initially, the authors map a very large vector X into a higher-dimensional space using a feature map Φ(x), and then use a decision function f(x)=W×Φ(x) to classify these vectors. Since the feature vector Φ(x) and weight vector W can be exponentially large or even infinite, the authors propose a new method that differs from kernel tricks (reference 9). This method extracts hidden information from the trained model and creates a "sweeping" optimization algorithm (Eq.7) using the structure of W, which minimizes the classification task using a quadratic cost function. This paper develops an efficient algorithm for image classification. The authors demonstrate how a tensor network optimization algorithm can be used for supervised learning using the matrix product state. Initially, the authors map a very large vector X into a higher-dimensional space using a feature map Φ(x), and then use a decision function f(x)=W×Φ(x) to classify these vectors. Since the feature vector Φ(x) and weight vector W can be exponentially large or even infinite, the authors propose a new method that differs from kernel tricks (reference 9). This method uses a tensor network to minimize the classification task using a quadratic cost function, which extracts hidden information from the trained model and creates a "sweeping" optimization algorithm (Eq.7) using the structure of W. Overall, I believe this paper is well-organized and clearly written; the proposed method is new and makes an important contribution to the field of machine learning. It is good to create a link between tensor networks and feature selection. However, a major problem with this paper is its experimental part. To demonstrate the effectiveness of the proposed method, the authors use a single dataset, i.e., MINST, and achieve a classification error of less than 1% on the test set. It would be better if the authors could apply the proposed method to other commonly used datasets in this area and show the results. Most importantly, there is a lack of comparison with state-of-the-art learning algorithms and feature selection methods, which makes it difficult to judge the effectiveness of the proposed method. I hope the authors will clarify these questions through the rebuttal process. Overall, my opinion on this paper is weak.
