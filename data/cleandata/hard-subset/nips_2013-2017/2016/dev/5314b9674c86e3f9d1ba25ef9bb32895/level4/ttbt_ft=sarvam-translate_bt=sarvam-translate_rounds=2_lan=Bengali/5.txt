This paper provides a supervised learning algorithm that discusses tensor networks from quantum physics. A major drawback of this work is that it is not intended to use tensor networks to define feature maps. It is clear that the use of tensor networks creates several difficulties in the optimization process. We suggest studying the authors' kernel techniques, which can reduce computational complexity, as they have already been widely used in kernel machines, such as support vector machines. Within the proposed classification model, as given in section 3, the concept of labels is difficult to understand, especially how the label index is placed in the position of a tensor and how/why it can be moved to another tensor. It can be considered as affecting a pixel label in the image, not the entire image. This is because the model (4)-(5) is not suitable for multiclass classification. The authors used quadratic loss (6), which is motivated to get the best test results. Other loss functions used should also be mentioned in their comparison. It is worth noting that quadratic loss provides relatively easy calculations compared to some other loss functions due to its derivative. There are many problems related to the test: - Several details are missing, such as the number of data used in the test, i.e., N, and the number of classes, i.e., L. - To reduce training time, 28x28 images were subsampled to 14x14. However, nothing has been said about the computational complexity of the algorithm. This is a major problem to solve in this paper. - It is not clear how training and test errors are estimated. - The proposed classification algorithm has not been compared with state-of-the-art classification methods. It should be compared in terms of estimated error and computational complexity.
