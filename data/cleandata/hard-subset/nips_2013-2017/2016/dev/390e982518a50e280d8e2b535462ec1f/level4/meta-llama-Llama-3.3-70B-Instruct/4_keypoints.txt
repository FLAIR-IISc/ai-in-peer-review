- Introduces a novel diffusion convolutional network (DCNN) for multilevel graph-structured classification (nodes, edges, graphs).  
- DCNN maps nodes and their features to outcomes of a diffusion process initiated from those nodes.  
- Parameter tying is based on search depth rather than positional grid locations, differing from traditional CNNs.  
- Theoretically proven that DCNN activations are identical for isomorphic input graphs, so representations are invariant to node indexing.  
- Empirically achieves state-of-the-art performance on node classification and comparable results on graph classification.  
- Edge classification is not evaluated; model potential on edge-level tasks remains unexplored.  
- Paper is comprehensive but would benefit from clearer explanations and simple illustrative examples (e.g., clearer Figures 1a–c) to improve readability.  
- Overall strong for node tasks; graph classification performance is less distinctive, indicating areas for future improvement.
