- Addresses learning optimal Bayesian network structures with non-decomposable ancestral constraints, which break many efficient exact learning algorithms.  
- Extends an A* search over the EC tree framework that uses an oracle for decomposable subproblems.  
- Main contribution 1: a method to prune the EC tree search space directly using ancestral constraints.  
- Main contribution 2: "constraint projection" technique to infer decomposable edge and ordering constraints from ancestral constraints.  
- Projected constraints are fed to the oracle, substantially improving oracle and overall search efficiency.  
- Provides algorithms and formal theorems (Theorems 1–4) giving a solid theoretical foundation.  
- Empirical evaluation is thorough, varying network size, dataset size, and amount of background knowledge.  
- Demonstrates orders-of-magnitude speedups over a state-of-the-art ILP method (GOBNILP).  
- Paper is technically sound, complete, and the combined pruning + oracle-empowering approach is powerful.  
- Writing and structure are clear; a small illustrative example effectively shows non-decomposability.  
- Suggested improvements: describe how ancestral constraints are expanded to a "maximum set" (transitive closure) and give more intuition for the MaxSAT acyclicity clause.  
- Minor suggestion to include proof sketches for main theorems (e.g., in an appendix) to improve self-containedness.  
- Original and non-trivial extension of prior EC tree work; projection algorithms and MaxSAT ordering formulation are novel.  
- Significant practical impact: makes optimal learning with ancestral constraints feasible up to ~20 variables and is likely useful when data is scarce.  
- Argument against acceptance: primarily an extension of an existing framework rather than a wholly new paradigm, though the extension is substantial.
