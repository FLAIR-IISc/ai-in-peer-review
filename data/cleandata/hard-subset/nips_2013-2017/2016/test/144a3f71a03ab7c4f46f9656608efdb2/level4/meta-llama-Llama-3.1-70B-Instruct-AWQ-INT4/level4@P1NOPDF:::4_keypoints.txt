- Proposes projecting non-decomposable ancestral constraints onto decomposable edge and topological-ordering constraints to enable decomposable-score-based structure learning.  
- Novelty centers on Section 5, which introduces the projection method from ancestral constraints to decomposable constraints.  
- Paper is well-written with clear theoretical and experimental exposition.  
- Concern: Table 3 shows improvements for EC trees but not ILP, raising questions about the method’s generalizability beyond EC trees.  
- Request for stronger evidence that inferred decomposable constraints can improve learning approaches that cannot natively handle non-decomposable constraints.  
- Observation: Table 2 shows an easy–hard–easy pattern in EC tree performance as constraint proportion increases; asks for explanation and whether performance is highly sensitive to amount of domain knowledge.  
- Question whether the number of ancestral constraints must be carefully tuned to achieve optimal EC tree performance, unlike ILP.  
- Missing comparison: Table 2 reports structural Hamming distance (SHD) only for EC trees; request to include ILP SHD for a comprehensive comparison.  
- Minor suggestion: remove Section 4.1 (BN foundations) and instead expand explanations of EC trees and the A* algorithm in the EC-tree context, since experiments focus on EC-tree search with ancestral constraints.
