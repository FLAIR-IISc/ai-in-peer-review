The authors present a method that leverages a given set of ancestral constraints to inform the heuristic for an exhaustive search algorithm. I found the experimental results somewhat perplexing. The existing approaches of EC and GOB are compared in Table 1, but the comparison is between (EC + new work) and (GOB + new work'). If I understand this correctly, Table 3 then compares EC to GOB. It seems that it would be more informative to include an explicit comparison between (EC + new work) and EC. Although the comparison between (GOB + new work') and GOB is interesting, given that EC significantly outperforms GOB, it is likely that (EC + new work) would still be preferable even if GOB benefits more from the new method.
