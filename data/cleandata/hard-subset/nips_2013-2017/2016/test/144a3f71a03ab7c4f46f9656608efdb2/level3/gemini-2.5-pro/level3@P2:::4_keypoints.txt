- Addresses learning optimal Bayesian network structures under non-decomposable ancestral constraints.  
- Builds on an A*-based EC-tree search framework that handles non-decomposable scores via an oracle for decomposable problems.  
- Main contribution: project ancestral constraints into decomposable edge presence/absence and ordering constraints to empower the oracle.  
- The projection method is theoretically supported (Theorems 2–4) and implemented using a MaxSAT solver for ordering constraints.  
- Empirical evaluation shows orders-of-magnitude speedups over the state-of-the-art ILP solver (GOBNILP) on benchmarks.  
- Experimental methodology is solid, varying variables, dataset sizes, and amounts of background knowledge.  
- Paper is well-written and organized; technical preliminaries and core contributions are clearly presented with examples.  
- Minor clarity suggestion: reduce basic BN graph exposition and add more detail on the A* process or the URLEARNING oracle.  
- Novel and significant idea: projecting non-decomposable constraints into decomposable ones to improve A*-based search heuristics.  
- Potential broader applicability: technique may extend to other types of non-decomposable constraints.  
- Weakness: unclear generalizability—benefit inconsistent for ILP, so show utility for classic DP-based learners that cannot handle ancestral constraints.  
- Weakness: observe an "easy-hard-easy" runtime pattern as constraint percentage increases; lacks analysis or explanation.  
- Weakness: SHD accuracy reported only for the EC-tree method; should report ILP baseline SHD to complete comparison.  
- Overall: strong technical contribution with minor empirical/discussion gaps that can be addressed in a revision.
