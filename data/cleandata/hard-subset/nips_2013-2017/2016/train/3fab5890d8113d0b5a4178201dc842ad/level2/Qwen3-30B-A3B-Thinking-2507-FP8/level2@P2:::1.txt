

This paper introduces Sparse Access Memory (SAM), a memory-augmented neural network architecture designed to overcome the computational and memory scalability limitations of existing models like Neural Turing Machines (NTMs). The key innovation is constraining memory reads and writes to a sparse subset of memory words (K=4-8), using efficient data structures for content-based addressing, and introducing a memory management scheme that tracks usage to find unused memory blocks. The authors claim SAM achieves asymptotic lower bounds in space and time complexity, demonstrating up to 1,000× faster training and 3,000× less memory usage compared to dense models, while maintaining comparable performance on synthetic tasks and Omniglot.

The paper's main strength lies in its practical demonstration of significant scalability improvements for memory-augmented networks. The experimental results are well-structured, showing SAM's advantages across multiple tasks: NTM-style algorithmic tasks (Copy, Associative Recall, Priority Sort), Babi question answering, and Omniglot one-shot learning. The speed and memory benchmarks (Figure 1) convincingly demonstrate SAM's computational advantages, particularly for large memory sizes. The authors also successfully generalize SAM to create a Sparse Differentiable Neural Computer (SDNC), which achieves state-of-the-art results on Babi tasks without supervising memory access.

The paper is well-written, clearly organized, and provides sufficient technical detail about the sparse read/write operations and memory management. The theoretical claims about asymptotic optimality are supported by supplementary material, though the main paper could better highlight the key assumptions behind these claims. The discussion of related work appropriately contextualizes SAM within the MANN literature, acknowledging prior approaches to scaling memory (e.g., reinforcement learning for addressing).

However, the paper has several weaknesses that need addressing. First, the claim that SAM is "asymptotically optimal" requires stronger justification in the main text. The paper references Supplementary A for proofs, but the main text should summarize the key insights about why SAM achieves these bounds, especially since this is a central contribution. The paper also doesn't thoroughly address the limitations of the sparse approach. For instance, when the relevant memory content is not among the top K nearest neighbors (which could happen in some distributions), how does SAM perform? The authors should provide an analysis of this failure case and discuss how it might be mitigated.

Another significant gap is the lack of comparison with other approaches to scaling MANNs. The paper mentions "prior work aimed at scaling Neural Turing Machines [22] used reinforcement learning to train a discrete addressing policy" but doesn't compare SAM's performance with this alternative approach. The authors claim SAM is "no more efficient than SAM in space and time complexity" for content-based addressing, but this is a strong assertion that needs empirical support.

The paper's claim of "1,000× faster" needs qualification. The speedup depends on specific parameters (e.g., memory size, ANN implementation), but the paper doesn't clearly state the conditions under which this ratio holds. The comparison with the dense DAM variant (which is itself a sparse approximation) might overstate the gains over the original NTM.

Additionally, the paper doesn't fully explore the trade-offs of sparsity. While the authors show SAM learns effectively on the tested tasks, it's unclear whether this would hold for more complex tasks where memory access patterns are less predictable. The paper could benefit from a more detailed analysis of how the sparsity level (K) affects performance across different tasks.

Despite these weaknesses, the paper makes a valuable contribution to the field by demonstrating a practical, scalable approach to memory-augmented networks. The results on Babi tasks and Omniglot show the method's applicability to meaningful real-world problems. The authors have provided a clear, well-executed solution to a significant problem in MANNs.

In summary, this paper presents a novel and effective approach to scaling memory-augmented neural networks with strong empirical results. While the theoretical claims need better justification in the main text and the limitations of the sparse approach require more thorough discussion, the work is technically sound, clearly presented, and makes a meaningful contribution to the field. The paper should be accepted with minor revisions to address the theoretical justification and limitations.
