- Introduces Sparse Access Memory (SAM), scaling MANNs by constraining reads/writes to sparse memory subsets.  
- Uses approximate nearest neighbor (ANN) data structures to reduce per-operation time/space from O(N) to O(log N) while keeping end-to-end differentiability.  
- Empirical results show up to 1,000× speedup and 3,000× memory reduction vs. dense baselines with comparable task performance.  
- Demonstrates applicability to multiple tasks: synthetic NTM tasks, Babi QA, Omniglot one-shot learning, and long-sequence associative recall.  
- Generalizes to Differentiable Neural Computer (SDNC), achieving state-of-the-art Babi results (9/10 tasks) without supervised memory access.  
- Solid theoretical analysis (Supplementary A) arguing asymptotic optimality and empirical verification of O(1) memory per time step.  
- Technical approach is well-motivated: sparse reads via top-K ANN retrieval and writes to read locations or least-recently-used slots.  
- Ablation study (SAM vs. DAM) convincingly isolates sparsity’s impact on learning efficiency.  
- Writing and supplementary material are generally clear and comprehensive, though some definitions (U^(2)_T) need more intuition.  
- Minor weaknesses: main text lacks explicit lower-bound proof for the claimed optimality and clearer rationale/comparison for ANN choices (k-d tree vs LSH).  
- Omniglot results claim outperforming others but lack quantified margins; write scheme (Eq 5) should be contrasted with alternative sparse write methods.  
- High significance: enables training MANNs with 100,000s of slots/time steps and makes long-context applications practical; recommendation: accept.
