- Tackles scalability limits of MANNs (e.g., NTM) whose read/write ops scale linearly with memory size.  
- Proposes Sparse Access Memory (SAM) achieving O(log N) read time and constant per-timestep space.  
- Introduces a sparse read using an approximate nearest neighbor (ANN) index to attend a constant number of locations.  
- Uses sparse writes targeting previously-read or least-recently-used slots.  
- Presents an efficient backpropagation scheme that avoids caching the full memory each step, reducing space.  
- Novel and elegant integration of ANN data structures into an end-to-end differentiable model.  
- Theoretical asymptotic efficiency is justified and empirically validated with large speed and memory gains.  
- Extensive experiments: synthetic algorithmic tasks, QA, one-shot learning, and curriculum learning.  
- Strong empirical generalization (trained on 10k steps, generalizes to 200k steps).  
- Paper is exceptionally clear, well-motivated, and figures effectively illustrate results.  
- Main weakness: reliance on ANN adds implementation complexity and hyperparameters (e.g., rebuild timing).  
- Minor recommendation: discuss sensitivity of training dynamics to ANN choice/configuration.  
- Arguments for acceptance: breakthrough scalability, novelty, rigorous evaluation, and orders-of-magnitude empirical improvements.  
- No significant arguments against acceptance; reviewer recommends acceptance.
