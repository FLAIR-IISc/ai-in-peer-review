

This paper introduces Sparse Access Memory (SAM), a novel architecture for memory-augmented neural networks (MANNs) that enables efficient scaling of external memory through sparse read/write operations and approximate nearest-neighbor (ANN) data structures. SAM addresses a critical bottleneck in prior MANNs (e.g., Neural Turing Machines [7] and Memory Networks [21]), which suffer from O(N) space/time complexity per time step as memory size N grows. The authors demonstrate that SAM achieves asymptotic optimality (O(log N) time, O(N) initialization, O(1) per-step space) while maintaining end-to-end differentiability. Key results include a 1,000× speedup and 3,000× memory reduction over dense baselines, successful scaling to 100,000+ time steps, and state-of-the-art performance on Babi tasks (SDNC variant solves 19/20 tasks without supervision). Crucially, SAM generalizes to sequences 20× longer than training (e.g., 10,000 → 200,000 steps), a capability absent in prior MANNs. This work builds directly on NIPS 2014’s NTM [7] and 2016’s DNC [8], but fundamentally advances the field by resolving the scalability barrier that has hindered practical deployment of MANNs.

**Strengths**  
*Quality:* The technical contributions are rigorously validated. The authors provide asymptotic complexity proofs (Supplementary A), comprehensive benchmarks across synthetic tasks (Copy, Associative Recall, Priority Sort), and real-world applications (Omniglot, Babi). SAM’s sparse read/write mechanism (Section 3.1–3.2) is elegantly designed: it retains the K most relevant memory words (K ≈ 4–8) via ANN-based nearest-neighbor search (O(log N) time), and uses a usage counter (U^(2)_T) to manage sparse writes without disrupting gradient flow. Experiments confirm SAM matches dense models’ data efficiency (Fig. 2) while drastically reducing runtime/memory (Fig. 1). The SDNC extension (Supplementary D) further demonstrates generality to DNC-like architectures.

*Originality:* SAM introduces a fundamentally new approach to MANN scaling—sparse access via efficient data structures—distinct from prior work. While RL-based sparse addressing [22] exists, SAM’s differentiable, gradient-based scheme avoids high-variance RL updates. Crucially, it is the first to achieve *asymptotic optimality* in space/time complexity for content-based addressing, resolving a key limitation noted in NIPS 2014 (NTM) and 2016 (DNC) papers. The authors clearly contextualize SAM against prior NIPS work, emphasizing its novelty over incremental improvements.

*Significance:* This paper makes MANNs practically viable. Prior MANNs struggled with memory overhead (e.g., 29GiB for 64K words [Fig. 1]), limiting real-world use. SAM’s orders-of-magnitude efficiency gains (1,000× faster, 3,000× less memory) enable tasks previously infeasible (e.g., BPTT over 4,000 steps with 2M memory words). The generalization to 200,000-step sequences (Section 4.3) and superior Babi performance (19/20 tasks) demonstrate tangible advances over prior work [17, 20]. This could catalyze adoption in applications like long-context language modeling or temporal reasoning.

*Clarity:* The paper is exceptionally well-organized. Sections 3.1–3.5 concisely explain SAM’s innovations, with clear mathematical formulations and algorithmic details. Figures 1–4 effectively visualize speed/memory gains and generalization. The supplementary materials provide thorough implementation details (e.g., ANN tradeoffs, usage counter design). Minor suggestions: Clarify why LSH outperforms k-d trees for large word sizes (Section 3.5) and specify SAM’s memory overhead in absolute terms (e.g., "53MiB initialization" vs. NTM’s "29GiB" [4.1]).

**Weaknesses**  
The discussion of *why* SAM enables such strong generalization (e.g., 200k-step generalization) is underdeveloped. Section 5 notes sparsity may act as a regularizer but lacks empirical analysis (e.g., comparing SAM’s internal representations to dense models). While the authors posit sparsity reduces overfitting (similar to sparse autoencoders), a brief ablation (e.g., "What if K=1?") would strengthen this claim. Additionally, the comparison to non-NTM MANNs (e.g., End-to-End Memory Networks [16]) is limited to Babi tasks; extending to Omniglot or synthetic tasks would further validate generality.

**Recommendation**  
This is a high-impact contribution that resolves a critical scalability barrier in MANNs. The technical rigor, clear presentation, and transformative practical implications align perfectly with NIPS’ emphasis on significant, scientifically sound advances. The minor weakness in explaining generalization mechanisms does not undermine the paper’s core contributions. I recommend **strong acceptance**. The authors should briefly expand Section 5 to address the generalization mechanism (e.g., "Sparsity may prevent overfitting to short sequences by limiting the model’s reliance on specific memory locations, as evidenced by SAM’s consistent performance across curriculum levels [Fig. 3]"), which would significantly enhance the paper’s theoretical depth without major revisions. This work is a major step toward deploying MANNs in real-world applications.
