- Introduces Sparse Access Memory (SAM), a MANN that uses sparse read/write over a constant-sized subset (K=4–8) of memory words per timestep.  
- Uses approximate nearest neighbor (ANN) data structures for content-based addressing to achieve sparse access.  
- Proves asymptotic optimality (O(1) time/space per timestep) under reasonable assumptions (Supplementary A).  
- Eliminates the O(N) memory/time overhead of prior MANNs, enabling training with 100,000+ memory words.  
- Demonstrates massive efficiency gains: ~1,000× runtime reduction and ~3,000× physical memory reduction versus dense variants.  
- Maintains near-identical data efficiency and task performance on synthetic tasks (Copy, Associative Recall, Priority Sort).  
- Shows superior scalability in curriculum learning, generalizing to sequences 20× longer and to 200k-sequence settings.  
- Achieves state-of-the-art results on Babi tasks without supervised memory access; SDNC extension outperforms prior end-to-end MANNs.  
- Experiments are comprehensive and comparisons against dense baselines (DAM, NTM) and LSTMs are rigorous.  
- Contextualizes SAM relative to prior work and contrasts with reinforcement-learning-based sparse approaches.  
- Original contribution: first end-to-end differentiable sparse memory access achieving asymptotic optimality for MANNs.  
- Significance: moves MANNs toward practical large-scale applications (e.g., Wikipedia-length sequences) and is broadly applicable to NTM-like architectures.  
- Clarity: paper is well-organized and explanations are intuitive; key contributions are emphasized early.  
- Minor suggested clarifications: include a simple visual workflow of SAM and move ANN trade-off discussion (k-d trees vs. LSH) into main text.  
- Strengths: solves scalability bottleneck with theoretical and empirical rigor, large efficiency gains without performance loss, practical impact, and general framework.  
- Weaknesses: lacks deep analysis of why sparsity improves long-sequence generalization beyond increased capacity; ANN implementation details are mostly in supplements.  
- Recommendation: Strong Accept — substantial contribution with clear technical soundness and practical impact.
