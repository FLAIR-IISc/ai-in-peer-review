- Problem: Dense read/write in MANNs scales poorly with memory size, causing prohibitive time and BPTT space costs.  
- Contribution: Proposes Sparse Access Memory (SAM) that restricts reads/writes to a constant number of locations per timestep.  
- Mechanism: Uses an approximate nearest neighbor (ANN) index for content-based reads and a policy to update recently read or least-recently-used slots for writes.  
- Complexity gains: Reduces per-step time complexity to O(log N) and BPTT space complexity to O(1), approaching asymptotic optimality.  
- Empirical efficiency: Reports dramatic improvements (up to ~1000x speedup and ~3000x memory reduction).  
- Empirical performance: Matches or exceeds dense MANNs on synthetic tasks (Copy, Associative Recall, Priority Sort), bAbI QA, and Omniglot one-shot learning.  
- Scalability highlight: Demonstrates training on sequences exceeding 100,000 steps for associative recall.  
- Novelty and clarity: Integrates efficient data structures (ANN indices) into forward pass; paper is well-written and original.  
- Weakness — accessibility: Background could better explain MANN memory operations (e.g., "soft" write) for non-experts.  
- Weakness — experiment framing: Omniglot comparison fixes physical memory, which conflates larger capacity with sparsity benefits.  
- Suggested experiment: Compare models with the same number of memory slots to isolate the effect of sparsity vs. increased capacity.  
- Overall: Strong, practical, and well-validated contribution that substantially advances scalable MANNs, with minor presentation and experimental clarifications needed.
