- Proposes Sparse Access Memory (SAM), a neural memory architecture that scales more efficiently than prior memory-augmented networks.
- Introduces sparse read/write scheme using efficient data structures to reduce compute and memory overhead.
- Key innovation: uses fast approximate nearest neighbors and arg-top-K for attention to enable training on larger problems.
- Demonstrates SAM on algorithmic tasks, question answering, and Omniglot one-shot learning.
- Results: SAM matches or outperforms dense models while substantially lowering computational and memory costs.
- Provides time and space complexity analysis claiming asymptotic optimality.
- Novelty is somewhat limited because related scalable-memory ideas exist, though writable-memory application is a valuable extension.
- Recommendation: add more detailed task and experimental descriptions and commit to releasing code for reproducibility.
- Recommendation: analyze limitations and failure cases, including effects of argmax/K-argmax and conditions where SAM fails.
- Recommendation: remove section 3.6 as it is not essential to the main contribution.
- Recommendation: stop calling Omniglot "non-synthetic" or "real-world"—treat it as a standard benchmark.
- Strength: practical solution to scalability in memory-augmented neural networks.
- Strength: thorough time and space complexity analysis.
- Strength: strong experimental evidence of SAM’s effectiveness across tasks.
- Weakness: limited novelty relative to prior memory network work.
- Weakness: lacks detailed analysis of limitations and failure cases.
- Weakness: experiments need clearer descriptions and published code to improve reproducibility.
- Overall decision: accept conditional on the recommended revisions.
