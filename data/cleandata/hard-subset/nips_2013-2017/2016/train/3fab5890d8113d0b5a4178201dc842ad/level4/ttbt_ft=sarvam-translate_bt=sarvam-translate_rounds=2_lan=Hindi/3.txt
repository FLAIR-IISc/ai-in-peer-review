This paper introduces a new memory access scheme called sparse access memory (SAM) in a memory augmented neural network architecture. This research area is quite new, with the invention of the neural Turing machine. This enhancement allows neural networks to learn beyond the limitations of traditional LSTMs. The contribution of this paper is that all writes and reads from external memory are limited to a certain size, thus the memory is a subset of words rather than an unlimited size. The authors have proven the methodology and its optimality. Good read and write performance is likely to be achieved due to the reduction in memory size shown in Section 4.1. However, the authors also included the learning cost of introducing sparsity using standard NTM functions in Section 4.2: 1. Copy, 2. Associated call, and 3. Priority sequence. The combined cost for some functions (i.e., 2 and 3) is lower than NTM. I also like the fact that the authors have included real-world examples in Section 4.4. The benchmark is using Torch7. The idea is straightforward and novel. The speed is significant compared to NTM, which is impressive. The fact that this sparse read and write can actually benefit initial learning in some cases by a linear or an ANN is very interesting (Section 4.2). It would be very helpful for the audience if the authors could elaborate and think about how to generalize for other cases (in which learning can be improved), or access different ANN approaches with SAM. Overall, this is a well-written paper in which all concepts are clearly stated and the architecture is given in the appendix. It is interesting to read.
