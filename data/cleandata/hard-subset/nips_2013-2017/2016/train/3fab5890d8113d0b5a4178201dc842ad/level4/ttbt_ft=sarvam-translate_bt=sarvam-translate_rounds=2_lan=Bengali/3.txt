This paper introduces a new memory access scheme called Spars Access Memory (SAM) in memory augmented neural network architecture. This research area is quite recent, with the discovery of the neural Turing machine. This augmentation allows neural networks to learn beyond the limitations of traditional LSTML models. The contribution of this paper is that writing and reading from external memory are limited to a specific size, so a subset is used instead of an unlimited size of memory words. The authors have proven the method and its superiority. It will likely achieve better reading and writing performance due to the reduction in memory size shown in section 4.1. However, the authors have also included the learning cost of introducing Spars in section 4.2 using standard NTM tasks: 1. Copy, 2. Associated call, and 3. Priority selection. For some tasks, the combined cost (such as 2 and 3) is lower than NTM. I also like that the authors have included real-world examples in section 4.4. The benchmark is created using Torch7. The concept is simple and innovative. Its speed is significantly higher than NTM, which is very impressive. This Spars reading and writing can benefit the initial level of learning done by a linear or an ANN in some cases, which is very interesting (section 4.2). It would be very helpful for the audience if they could think in detail about how to generalize to other cases (where learning can be improved), or about accessing different ANN methods using SAM. Overall, it is a well-written paper, where all assumptions are clearly stated and the architecture is provided in the appendix. It is interesting to read.
