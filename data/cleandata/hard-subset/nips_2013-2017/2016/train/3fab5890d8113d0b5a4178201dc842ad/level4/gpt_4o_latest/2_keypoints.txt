- Memory-augmented neural networks tend to scale inefficiently in both space and time as memory size increases.
- The paper proposes a memory access mechanism called Sparse Access Memory (SAM).
- SAM is claimed to preserve representational capacity while enabling efficient training, with reported 1000× speedup and 3000× memory savings.
- The reviewer finds the technical contributions insufficiently robust for acceptance.
- The core technical elements—efficient backpropagation and approximate nearest neighbor search via SAM—are viewed as relatively straightforward.
- Despite interesting experimental results, the reviewer does not recommend acceptance.
