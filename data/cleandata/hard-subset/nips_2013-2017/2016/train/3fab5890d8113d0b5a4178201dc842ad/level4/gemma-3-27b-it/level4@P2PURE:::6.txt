This paper introduces a Neural Turing Machine (NTM) with sparse read and write access, addressing a key limitation of NTMs that hinders their applicability to real-world problems: the requirement for prohibitively large memory and computational cost during backpropagation. The proposed architecture aims to resolve these issues while maintaining performance comparable to state-of-the-art models. The paper is well-organized and clearly written. To enhance accessibility for readers less familiar with the field, a brief overview of memory-augmented neural networks in the background section – or a clearer explanation of the memory mechanism itself – would be beneficial. Specifically, detailing the intended function of the write operator compared to conventional memory operations could improve understanding. Furthermore, the comparison presented in Figure 4 may not be fully representative. While comparing performance under a fixed physical memory constraint is a valid and justifiable approach, evaluating performance with the same *logical* memory size (rather than the same physical memory size) would likely provide more informative insights. Despite this, the algorithm’s strong performance, approaching theoretical bounds, is very encouraging.
