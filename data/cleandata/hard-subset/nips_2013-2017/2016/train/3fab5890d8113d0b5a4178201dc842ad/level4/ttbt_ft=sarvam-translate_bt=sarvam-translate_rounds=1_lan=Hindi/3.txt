This paper introduces a new memory access scheme called Sparce Access Memory (SAM) in a memory augmented neural network architecture. This research area is quite new, with the invention of the neural Turing machine. This enhancement allows neural networks to learn beyond the limitations of traditional LSTMs. The contribution of this paper is that all writes and reads from external memory are limited to a fixed size, thus the memory is a subset of words rather than an unlimited size. The authors have proven the methodology and its optimality. It is likely to achieve good read and write performance due to the reduction in memory size as shown in Section 4.1. However, the authors also included the learning cost of introducing sparsity using standard NTM functions in Section 4.2: 1. Copy, 2. Associated Call, and 3. Priority Sequence. The combined cost for some functions (i.e., 2 and 3) is lower than NTM. I also like the fact that the authors have included real-world examples in Section 4.4. The benchmark is using Torch7. The idea is straightforward and novel. The speed compared to NTM is significant, which is impressive. The fact that this sparse read and write can actually benefit early learning in some cases by a linear or an ANN is very interesting (Section 4.2). If the authors could elaborate and think about how to generalize to other cases (in which learning can be improved), or reach different ANN approaches with SAM, it would be very helpful for the audience. Overall, this is a well-written paper in which all concepts are clearly stated and the architecture is given in the appendix. It is interesting to read.
