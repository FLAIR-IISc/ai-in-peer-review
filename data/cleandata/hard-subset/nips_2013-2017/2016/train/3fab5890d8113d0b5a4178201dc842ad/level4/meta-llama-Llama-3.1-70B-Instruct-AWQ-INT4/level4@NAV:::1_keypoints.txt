- Proposes using fast approximate nearest-neighbors and arg-top-K for attention in a Neural Turing Machine.  
- Experiments show this approach preserves training performance and enables scaling to larger problems.  
- Novelty is limited given prior work in memory networks, but applying these techniques to writable memories is a useful contribution.  
- Recommends acceptance pending revisions because the paper’s practical value merits reporting.  
- Authors should provide detailed task descriptions and release experiment code to enable replication.  
- Paper should discuss limitations and potential failure cases of the model.  
- Argmax/K-argmax is not smooth—this should be analyzed and addressed.  
- Compare with MemNN-WSH and analyze the impact of K=1 on performance.  
- Discuss which task types and setups allow the proposed training to work, and clarify handling of discrete actions.  
- Suggest removing section 3.6.  
- Avoid describing Omniglot data as “non-synthetic” or “real-world.”
