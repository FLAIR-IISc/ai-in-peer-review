- Proposes Sparse Access Memory (SAM): sparse read/write plus approximate nearest neighbor search to scale writable memory-augmented neural networks.
- Core idea restricts memory modifications to a sparse subset and uses efficient data structures for content-based access.
- Reports large scalability gains: up to ~1000x speedups and ~3000x physical memory reductions over traditional MANNs.
- Maintains comparable task performance on synthetic algorithmic tasks, BAbi reasoning, and one-shot Omniglot.
- Practical and implementable approach leveraging ANN indexing and data structures for real-world applicability.
- Demonstrates generality by applying SAM to the Differentiable Neural Computer with significant improvements.
- Evaluation is thorough across synthetic, reasoning, and recognition benchmarks.
- Builds on prior MANN work (NTM, Memory Networks) and addresses their scalability/backpropagation bottlenecks.
- Related hashing/ANN ideas exist for read-only memory networks, but writable-memory application is less explored here.
- Replicability concerns: experimental task descriptions lack detail and code release is needed for reproducibility.
- Limited analysis of failure cases and sensitivity (e.g., non-smooth argmax, choice of K) weakens understanding of limitations.
- Claiming Omniglot as "real-world" is misleading; it is a curated benchmark rather than true real-world data.
- Section 3.6 on ANN indexing feels tangential and could be streamlined or integrated better.
- Recommendation: accept — significant, well-written contribution addressing a key MANN bottleneck, pending improvements to reproducibility and failure-case analysis.
