- Proposes Sparse Access Memory (SAM), a memory access scheme constraining reads/writes to a fixed-size sparse subset of memory words.  
- Builds on Neural Turing Machines to extend neural network memory beyond LSTM limits.  
- Main contribution: restrict operations to sparse subsets rather than entire memory, claimed optimal.  
- Provides theoretical/supporting analysis arguing for SAM’s optimality.  
- Section 4.1 shows substantial read/write efficiency gains via reduced computation and memory overhead.  
- Section 4.2 studies learning effects of induced sparsity on standard NTM tasks (Copy, Associative Recall, Priority Sort).  
- Empirical results: overall cost lower than NTM on some tasks, notably Associative Recall and Priority Sort.  
- Includes a real-world example in Section 4.4 demonstrating applicability.  
- Implementation provided in Torch7.  
- Concept is straightforward yet novel, with striking speedups over NTM.  
- Observation: sparse reads/writes using linear search or ANN can sometimes improve early-stage learning.  
- Reviewer requests further analysis of when/why ANN/linear-search improvements occur and how different ANN choices interact with SAM.  
- Paper is clearly written, assumptions explicit, and architectural details included in the appendix.
