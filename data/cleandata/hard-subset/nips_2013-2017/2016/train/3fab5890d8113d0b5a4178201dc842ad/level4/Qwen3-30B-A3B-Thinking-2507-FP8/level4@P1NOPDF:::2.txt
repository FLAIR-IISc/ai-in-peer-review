

Most existing neural network models incorporating memory exhibit poor scalability in both memory usage and computational cost as memory size increases. The authors propose a memory access mechanism called Sparse Access Memory (SAM), claiming it maintains representational power while enabling highly efficient training—achieving a 1000x speedup and 3000x memory reduction. However, I find the technical contribution insufficient for acceptance. The core innovations, namely efficient backpropagation and approximate nearest neighbor search leveraging SAM, are relatively straightforward and lack significant novelty. While the experimental results appear promising, the foundational technical arguments do not demonstrate sufficient depth or rigor to justify acceptance. Given the simplicity of the proposed approach, I cannot recommend acceptance.
