- Proposes an NTM with sparse read/write access to fix impractical memory and computational demands of traditional NTMs.  
- Claims the new architecture preserves performance comparable to state-of-the-art methods.  
- Paper is well-organized and easy to follow.

- Suggests clarifying the intended role of the write operator versus conventional memory systems.  
- Recommends reconsidering Figure 4’s comparison: matching physical memory is defensible, but comparing performance with the same memory might be more informative.
