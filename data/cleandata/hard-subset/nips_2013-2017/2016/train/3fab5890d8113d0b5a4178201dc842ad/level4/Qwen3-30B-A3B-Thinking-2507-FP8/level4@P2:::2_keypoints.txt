- Proposes Sparse Access Memory (SAM), a differentiable MANN using sparse reads/writes to address scalability limits of existing models.  
- Constrains memory access to a small constant number of words per operation while preserving end-to-end differentiability.  
- Theoretically proves SAM achieves asymptotic optimality in time and space complexity under reasonable assumptions.  
- Implements sparse reads via approximate nearest neighbor search and sparse writes using a least-recently-accessed memory management scheme.  
- Reports empirical gains: ~1,000× speedup and ~3,000× memory reduction vs. non-sparse models.  
- Validated across synthetic tasks (Copy, Associative Recall, Priority Sort) and real tasks (Omniglot, Babi QA).  
- Scales to very long sequences (100k–200k+ time steps) and memory sizes in the millions using curriculum learning.  
- Demonstrates generality by adapting SAM to Differentiable Neural Computer (SDNC) and achieving state-of-the-art on Babi without supervised memory access.  
- Paper is well-written, clearly organized, and provides reproducible architecture and implementation details with helpful figures and supplements.  
- Experimental validation is comprehensive with comparisons to relevant baselines (NTM, DAM, LSTM).  
- Originality: builds on prior sparse/efficient ideas but contributes a novel differentiable, asymptotically optimal SAM architecture.  
- Significance: addresses a critical bottleneck in MANNs, enabling practical large-scale applications and long-range dependency modeling.  
- Limitations: innovations may seem straightforward to reviewers familiar with sparse structures; lacks in-depth discussion of theoretical limits of the sparse approximation.  
- Missing comparisons to more recent MANN variants beyond NTM are noted as an experimental gap.  
- Recommendation: strong accept — practical and theoretical contributions with thorough validation, despite some expected caveats.
