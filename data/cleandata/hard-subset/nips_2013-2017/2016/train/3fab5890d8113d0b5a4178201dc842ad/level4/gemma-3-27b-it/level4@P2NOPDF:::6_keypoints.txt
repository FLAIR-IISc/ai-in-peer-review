- Proposes Sparse MTN, an NTM variant that reduces backpropagation memory and computation while aiming for comparable performance to standard NTMs.  
- Paper is well-written and logically organized, making it easy to follow.  
- Recommends adding a brief background on memory-augmented networks, clarifying the memory component and the write operator for non-expert readers.  
- Primary concern: Figure 4’s comparison fixes physical memory size; suggests instead comparing performance with the same number of memory locations.  
- Comparing equal memory capacity (locations) would better demonstrate Sparse MTN’s efficiency for storing information.  
- Results are compelling: performance close to theoretical bounds despite much lower memory footprint and compute.  
- Overall, a promising contribution that could make NTMs more practical for real-world applications.
