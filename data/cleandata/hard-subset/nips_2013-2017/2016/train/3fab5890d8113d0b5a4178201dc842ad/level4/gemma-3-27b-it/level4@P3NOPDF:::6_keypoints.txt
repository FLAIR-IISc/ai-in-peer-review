- Introduces an NTM variant using sparse read/write access to reduce memory and computation, especially for backpropagation.  
- Core contribution: mitigates large memory footprint and computational cost of standard NTMs, improving practical viability.  
- Sparse access mechanism is a significant step toward practical NTM implementations.  
- Paper is well-structured and clearly written.  
- Recommend adding a concise background on memory-augmented neural networks for readers unfamiliar with the field.  
- Ask for clearer explanation of the write operator: how it differs from conventional memory writes and why that matters.  
- Experimental results are promising, achieving performance close to theoretical bounds.  
- Figure 4 comparison (fixed physical memory) is justified, but a complementary comparison using identical memory capacity is suggested.  
- A same-capacity comparison would clarify trade-offs between sparsity and raw memory capacity.  
- Strengths: clear presentation, addresses NTM scalability, strong experimental results, defensible experimental design.  
- Weaknesses: limited background for non-experts and the need for complementary analysis in Figure 4.  
- Overall assessment: strong, well-motivated paper with compelling results; recommend acceptance.
