Although most proposed neural network models have been improved with memory scales, they grow weakly with memory increases in both space and time. The authors presented a memory access scheme called Spars Access Memory and showed that it retains representativeness while training efficiently, being 1000 times faster and 3000 times more memory-efficient. I don't think this paper is strong enough to be technically acceptable. Efficient backpropagation and nearest neighbor approximation using Spars Access Memory form the main technical part, but these are quite simple. Therefore, although the experiments look interesting, I am not in favor of acceptability.
