- Proposes an NTM with sparse read/write access to overcome NTMs’ prohibitive memory and backpropagation costs.  
- Aims to retain performance comparable to state-of-the-art memory-augmented models.  
- Paper is well-structured and clearly written.  
- Suggest adding a brief background on memory-augmented neural networks for readers unfamiliar with the subfield.  
- Recommend a more detailed explanation of memory mechanisms, especially the write operator and how it differs from conventional memory operations.  
- Raises fairness concern in Figure 4: compare algorithms using the same memory size (information capacity) rather than the same physical memory footprint.  
- Results are promising, achieving performance close to theoretical bounds and demonstrating strong potential.
