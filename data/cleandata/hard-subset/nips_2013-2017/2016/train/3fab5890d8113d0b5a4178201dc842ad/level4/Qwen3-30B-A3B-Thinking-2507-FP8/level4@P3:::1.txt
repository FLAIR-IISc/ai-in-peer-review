

The paper introduces Sparse Access Memory (SAM), a novel memory architecture for memory-augmented neural networks (MANNs) that significantly improves computational and memory efficiency through sparse read and write operations. SAM achieves asymptotic optimality in space and time complexity (O(N) initialization, O(log N) per forward/backward step, O(1) space per time step) by constraining memory operations to a small, fixed number of memory slots (K=4-8) and utilizing approximate nearest neighbor (ANN) data structures. The authors demonstrate SAM's effectiveness on synthetic tasks (Copy, Associative Recall, Priority Sort), Babi question answering tasks, and Omniglot one-shot learning, showing comparable performance to dense models while achieving 1,000× faster training and 3,000× reduced memory usage. The approach is generalizable to other MANNs, as demonstrated through a Sparse Differentiable Neural Computer (SDNC).

This work builds upon the Neural Turing Machine (NTM) and Memory Networks literature [7, 21], where external memory has been used to extend neural network capabilities beyond traditional LSTM limitations. While sparse memory access has been explored in some Memory Networks variants [17, 20], the authors make the important contribution of extending this approach to writeable memory systems with content-based addressing—a significant practical challenge not previously addressed in the literature. The paper's most valuable contribution is the demonstration that sparse memory operations do not degrade learning performance while enabling scalability to memory sizes previously infeasible for MANNs.

The paper excels in quality, with thorough experimental validation across multiple domains. The speed and memory benchmarks (Figure 1) provide compelling empirical evidence of SAM's efficiency gains, with the authors carefully documenting the trade-offs between different ANN implementations (k-d trees vs. LSH). The scalability experiments with curriculum learning (Figure 3) demonstrate SAM's ability to handle tasks requiring 100,000s of time steps and memory slots, a significant advancement over previous MANNs. The Babi task results (Section 4.4) are particularly noteworthy, as SDNC achieves the best reported performance without supervising memory access—a key advancement over prior work.

The paper is generally well-written and clear, though some sections could benefit from refinement. The technical explanations in Section 3 are precise, though the description of the memory management scheme (Section 3.2) could be clarified with a simple example. The authors' claim that SAM "retains the representational power of the original approaches" is well-supported by their experimental results showing comparable data efficiency across multiple tasks.

The novelty is somewhat incremental but practically significant. While sparse memory access has been used in some Memory Networks papers [17, 20], this is the first to successfully apply it to writeable content-addressable memory systems with end-to-end differentiable training. The authors acknowledge related work appropriately, though they could more clearly distinguish their contribution from the Memory Networks paper [21] (Section 3.3 reference), which discusses similar sparse memory approaches but for read-only settings. The paper's true innovation lies in demonstrating that sparse writes can be effectively implemented in a differentiable MANN framework.

The significance of this work is substantial. By enabling MANNs to scale to memory sizes that were previously infeasible (millions of slots), SAM opens the door to applications requiring long-term memory and complex algorithmic reasoning—areas where MANNs have struggled to compete with conventional neural network architectures. The demonstration of learning on the Omniglot dataset (Section 4.5) and the ability to generalize to sequences far longer than seen during training (Supplementary Figure 8) provide strong evidence that SAM enables meaningful progress in memory-augmented learning.

However, the paper has several weaknesses that need addressing. First, the lack of sufficient detail for replication is concerning. The paper describes task construction only minimally (Section 4.2), and without access to the original NTM task implementations, it will be difficult for other researchers to reproduce the results. The authors should provide detailed task specifications, including all hyperparameters and data generation procedures, and commit to releasing their code.

Second, the paper lacks a thorough analysis of the limitations and failure cases of the sparse approach. The authors acknowledge that "argmax is not smooth" but do not explore how this affects performance across different tasks or whether certain task types are particularly sensitive to the discrete approximation. A more detailed analysis of when SAM succeeds or fails (compared to fully continuous models) would significantly strengthen the paper.

Third, the characterization of Omniglot as "real-world" or "non-synthetic" is misleading. While Omniglot is a standard benchmark, it remains a synthetic dataset designed for rapid learning evaluation. The authors should revise this language to avoid overstating the practical relevance of their results.

Finally, Section 3.6 (which appears to discuss a specific implementation detail not referenced elsewhere) should be removed, as it adds little value to the main contribution.

In conclusion, this is a strong practical contribution that addresses a significant limitation of current MANNs. While the novelty is incremental, the empirical results are compelling and the scalability improvements are substantial. With the suggested revisions—particularly providing sufficient detail for replication and clarifying the limitations of the approach—the paper would be an excellent contribution to NIPS. The authors have demonstrated that sparse memory access is not just a theoretical possibility but a practical solution that enables previously infeasible applications of MANNs. I recommend acceptance with minor revisions.
