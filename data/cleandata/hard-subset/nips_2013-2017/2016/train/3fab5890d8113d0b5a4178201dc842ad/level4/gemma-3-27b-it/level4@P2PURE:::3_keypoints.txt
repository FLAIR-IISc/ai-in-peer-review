- Introduces Sparse Access Memory (SAM), a memory access scheme for memory-augmented neural nets that limits reads/writes to a fixed-size subset of memory.
- Key contribution: constraining both read and write operations to a sparse subset rather than the entire memory.
- Authors provide a proof of optimality for the proposed sparse access methodology.
- Reduced memory access size plausibly yields improved read/write performance (demonstrated in Sec. 4.1).
- Learning costs of sparsity are analyzed (Sec. 4.2) and evaluated on standard NTM tasks: copy, associative recall, and priority sort.
- Overall computational cost is lower than a standard NTM for certain tasks (notably associative recall and priority sort).
- Empirical results show a significant and impressive speedup compared to NTM implementations.
- Notable finding: sparse reads/writes (via linear layer or ANN) can benefit early-stage learning in some scenarios (Sec. 4.2).
- Includes a real-world example (Sec. 4.4) and an implementation in Torch7.
- Suggestion: authors should clarify how to generalize the learning benefits—conditions under which learning improves and interactions with different ANN architectures.
- Paper is well written with clear assumptions and a detailed architectural description in the appendix.
