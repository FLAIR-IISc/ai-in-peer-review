- Addresses scalability limits of neural networks with external memory as memory size grows.  
- Proposes Sparse Access Memory (SAM) to improve computational and memory efficiency.  
- Claims SAM preserves representational power while delivering up to 1000x training speedup and 3000x memory reduction.  
- Empirical results are impressive and practically valuable.  
- Primary concern: limited technical novelty in the contributions.  
- Core components (efficient backprop for sparse memory and use of ANN for retrieval) are viewed as straightforward applications of existing techniques.  
- Backprop derivation follows standard principles; ANN retrieval is common engineering practice.  
- Conclusion: an effective engineering solution that lacks sufficient fundamental innovation for NIPS acceptance; reviewer cannot recommend acceptance.
