- Addresses scalability in memory-augmented neural networks and proposes Sparse Access Memory (SAM).
- Reports major empirical gains: ~1000x speedup and ~3000x reduction in memory consumption.
- Reviewer does not recommend acceptance, citing insufficient novelty.
- Core technical contributions (efficient backprop through SAM and nearest-neighbor approximation) are viewed as straightforward extensions of existing techniques.
- Paper lacks a clear, novel insight explaining why these techniques work particularly well within the SAM framework.
- Reviewer requests stronger justification of technical contributions, e.g., nuanced theoretical analysis or identification of a previously unaddressed challenge.
- Ask for comparison to alternative sparsity patterns (random/structured) via a comprehensive ablation study.
- Request a formal theoretical analysis of SAM’s representational power and how it scales with memory size and sparsity.
- Request a detailed computational complexity analysis showing how sparse access cost scales with memory size and memory-vector dimensionality.
- Ask for a sensitivity analysis of key hyperparameters (sparsity level, nearest-neighbor parameters) to demonstrate robustness.
- Conclusion: practical benefits are clear, but technical novelty is insufficient for NIPS; encourage revisions to address concerns.
