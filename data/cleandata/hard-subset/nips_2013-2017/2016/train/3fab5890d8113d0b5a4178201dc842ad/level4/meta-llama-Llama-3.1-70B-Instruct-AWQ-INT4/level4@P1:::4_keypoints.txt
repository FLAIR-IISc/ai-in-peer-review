- Introduces Sparse Access Memory (SAM), an end-to-end differentiable memory with representational power comparable to Neural Turing Machines but optimized for very large memory.  
- Well-motivated: addresses scalability limitations of existing memory-augmented neural networks (MANNs) like NTMs and Memory Networks.  
- Technical novelty: sparse read using approximate nearest neighbor data structures to compute read weights in O(log N) time.  
- Technical novelty: sparse write scheme that writes to previously read locations or the least recently accessed location.  
- Theoretical claim: SAM achieves asymptotic lower bounds in space and time complexity for memory access.  
- Empirical efficiency: implementation runs ~1,000× faster and uses ~3,000× less physical memory than dense baselines.  
- Evaluation: compares SAM and dense variant DAM against NTM and LSTM on algorithmic tasks, bAbi reasoning, and Omniglot one-shot classification.  
- Performance: SAM matches data efficiency of existing models and scales to tasks with 100,000s of timesteps and memory slots.  
- Generality: extends to temporal-association models (Sparse Differentiable Neural Computer, SDNC).  
- SDNC results: ~400× speedup at 2,000 memory slots and best reported bAbi performance without supervised memory access.  
- Quality: paper is technically sound with theoretical analysis and experiments supporting claims.  
- Clarity: well-written and organized, with sufficient detail for expert reproduction.  
- Originality: introduces original concepts of sparse reads/writes for external memory in neural networks.  
- Significance: contributes a scalable memory architecture that can influence future MANN research and applications.  
- Arguments for acceptance: significant contribution, clear exposition, strong empirical and theoretical support, high potential impact.  
- Arguments against acceptance: requires expert background to follow, lacks detailed analysis of limitations, and limited comparison to other state-of-the-art models.
