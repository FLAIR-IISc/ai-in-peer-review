- Proposes Sparse Access Memory (SAM), a memory access scheme for memory-augmented neural networks.
- Main contribution: restricts read/write to a fixed-size subset of memory words rather than full memory.
- Provides a proof of methodology and argues its optimality.
- Constrained access reduces memory footprint and improves read/write performance (evidence in Sec. 4.1).
- Evaluates learning costs of sparsity on standard NTM tasks (copy, associative recall, priority sort) in Sec. 4.2.
- Combined computational cost is lower than NTM for some tasks (associative recall and priority sort).
- Includes a real-world example (Sec. 4.4); implementation in Torch7 with significant speedup relative to NTM.
- Finds sparse reads/writes via a linear layer or ANN can aid early-stage learning in some cases (Sec. 4.2).
- Recommends elaborating when SAM improves learning and testing SAM with alternative ANN architectures.
- Overall assessment: well-written, clear assumptions, detailed appendix, and a compelling contribution.
