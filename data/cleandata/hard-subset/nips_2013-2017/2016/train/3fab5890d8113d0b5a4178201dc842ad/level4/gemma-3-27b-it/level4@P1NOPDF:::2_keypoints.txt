- Problem: Existing memory-augmented neural nets scale poorly in space and time as memory size increases.
- Contribution: Proposes Sparse Access Memory (SAM) that maintains representational power while enabling efficient training.
- Empirical claim: SAM achieves large speedups (1000x) and memory efficiency improvements (3000x).
- Reviewer verdict: Not technically strong enough for acceptance despite interesting results.
- Technical novelty: Core contributions (efficient backprop through sparsity and using approximate nearest neighbor search) are viewed as straightforward and incremental.
- Missing analysis: Paper lacks deeper investigation into why the specific sparse access pattern works (e.g., role of hashing function or sparsity level).
- Suggested experiments: Needs ablation studies to isolate impact of components like hashing and sparsity.
- Experiments strength: Demonstrates performance on several tasks (image classification, language modeling) with clear presentation of speed/memory gains.
- Baseline comparisons: Comparisons to existing memory networks are incomplete—should include recent/optimized Neural Turing Machine or DNC implementations and vary baseline memory sizes.
- Presentation: Sparse access mechanism is clearly presented with helpful figures, but mathematical notation is sometimes inconsistent and confusing.
- Recommendation: Encourage more in-depth theoretical analysis and more exhaustive experiments to substantiate technical contribution.
