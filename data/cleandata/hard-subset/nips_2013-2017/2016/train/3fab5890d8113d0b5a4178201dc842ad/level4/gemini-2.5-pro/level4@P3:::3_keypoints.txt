- Paper addresses scalability bottleneck in Memory-Augmented Neural Networks by replacing dense O(N) memory access with sparse access restricted to K slots.
- Proposes Sparse Access Memory (SAM) using Approximate Nearest Neighbor (ANN) structures to find K relevant memory locations in O(log N) time.
- Introduces a backpropagation scheme that reverts sparse memory modifications to reduce training space overhead from O(N) to O(1) per step.
- Theoretical analysis shows reduced per-step time complexity from O(N) to O(log N) and constant additional space per step.
- Empirical benchmarks report empirical speedups >1000x and memory reductions ~3000x.
- Experiments on synthetic algorithmic tasks show sparse approximation does not degrade and can improve learning vs dense models.
- Curriculum learning experiments demonstrate scaling to extremely long sequences (up to 200,000 steps) and large memories.
- Applying SAM principles to the Differentiable Neural Computer yields state-of-the-art results on the bAbI tasks, showing generality.
- Paper is well-written, clearly motivated, and backed by detailed appendices aiding reproducibility.
- Novelty and technical quality: integrating ANN data structures into end-to-end differentiable MANNs is elegant and technically sound.
- Clever efficient BPTT implementation that "rolls back" memory state addresses often-overlooked training memory overhead.
- Minor weakness: lack of deeper analysis explaining why sparsity sometimes improves learning (potential regularization effect not explored).
- Minor weakness: limited discussion on trade-offs and challenges of ANN choices in dynamic online memory settings and potential adaptive alternatives.
- Overall recommendation: strong accept — significant, well-supported contribution likely to influence future work.
