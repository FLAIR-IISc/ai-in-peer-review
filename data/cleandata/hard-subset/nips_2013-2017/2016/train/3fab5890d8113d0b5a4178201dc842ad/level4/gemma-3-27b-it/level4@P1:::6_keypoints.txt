- Presents Sparse Access Memory (SAM), a novel MANN that restricts read/write to a sparse subset of memory locations to improve scalability.
- Targets the scalability bottleneck of MANNs during training, especially BPTT overhead.
- Achieves large empirical speedups and memory reductions (reported ~1000x speed, ~3000x memory) while maintaining competitive performance.
- Provides theoretical justification by establishing asymptotic lower bounds in space and time complexity.
- Empirically validated across synthetic tasks (copy, associative recall, priority sort), Omniglot, and bAbI tasks.
- Demonstrates generality by adapting SAM to the Differentiable Neural Computer (SDNC).
- Paper is well-structured and clearly explains technical details.
- Suggests expanding the background to better explain core MANN concepts, especially the write operator versus conventional memory operations.
- Recommends augmenting Figure 4 with comparisons at fixed memory size (not only fixed physical memory) for clearer performance scalability insights.
- Requests more detail on the ANN implementation used (specific algorithms and parameter tuning).
- Argument for acceptance: novel, theoretically sound, empirically strong, and addresses an important field bottleneck.
- Argument against acceptance: potential incremental novelty since ANN techniques have been used elsewhere; SAM may not decisively outperform on bAbI.
- Overall recommendation: strong paper with valuable contribution; reviewer recommends acceptance.
