

Most existing neural network models augmented with memory exhibit poor scalability in both space and time as memory size increases. The authors propose a memory access scheme termed Sparse Access Memory (SAM), claiming it preserves representational power while enabling efficient training—achieving a 1000× speedup and 3000× memory efficiency gain. However, the technical contribution is insufficient for acceptance. The core innovations—efficient backpropagation and approximate nearest neighbor search using SAM—are conceptually straightforward and lack substantial novelty. Despite potentially interesting experimental results, the paper's technical depth is inadequate, and I recommend rejection.
