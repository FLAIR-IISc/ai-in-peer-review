- Proposes Sparse Access Memory (SAM), a novel MANN architecture using sparse read/write to scale memory with asymptotic lower bounds in space and time while retaining representational power.
- Key idea: restrict read/write to a sparse subset of memory words for efficient computation and reduced memory usage.
- Demonstrates effectiveness via experiments: speed/memory benchmarks, learning with sparse access, curriculum scaling, and bAbI question answering.
- Paper is well-structured and clear, with good explanations of related work and organized figures/tables.
- Suggestion: include a brief description of MANNs and the memory architecture in the background to make the paper more self-contained.
- Critique: Figure 4 comparison uses equal physical memory usage across models; a comparison at equal memory size would be more informative/fair.
- Strength: novel and efficient approach to scaling MANNs with solid theoretical analysis of time and space complexity.
- Strength: experiments are well-designed and support the claims of improved efficiency and scalability.
- Weakness: background lacks sufficient detail on MANNs/memory architecture for non-experts.
- Weakness: potential unfairness in experimental comparison (different effective memory sizes).
- Originality: introduces sparse read/write operations and SAM, achieving asymptotic complexity improvements.
- Significance: work has important implications for practical, scalable MANNs and real-world applications.
- Quality: well-written, thorough analysis, and convincing empirical results.
- Arguments for acceptance: novelty, theoretical analysis, and convincing experiments.
- Arguments for rejection: insufficient background detail and possibly unfair comparison metric in Figure 4.
- Overall: significant contribution with minor issues that do not undermine the paper’s value.
