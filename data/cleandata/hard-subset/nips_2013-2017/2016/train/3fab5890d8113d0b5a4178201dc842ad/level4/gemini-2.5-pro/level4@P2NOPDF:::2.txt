This paper addresses the poor scalability of neural network models augmented with external memory, a significant challenge that limits their application as memory size increases. The authors propose a novel memory access scheme, Sparse Access Memory (SAM), designed to improve both computational and memory efficiency. They claim that their method retains the representational power of denser memory architectures while achieving dramatic performance gains, reporting up to a 1000x speedup in training and a 3000x reduction in memory footprint.

Despite these impressive empirical results, my primary concern is the limited technical novelty of the proposed contributions. The core technical components of the paper consist of an efficient backpropagation algorithm for the sparse memory system and the use of approximate nearest neighbor (ANN) search for memory retrieval. While functional, these elements appear to be relatively straightforward applications of existing, well-established techniques rather than fundamental innovations. The derivation for backpropagation follows standard principles applied to this new architecture, and leveraging ANN for efficient retrieval from large-scale databases is a common engineering practice.

In conclusion, while the paper presents a valuable engineering solution to a practical problem and the experimental results are certainly interesting, the work's core contributions seem to lack the sufficient technical depth and novelty expected for a NIPS publication. The paper effectively combines existing methods to build an efficient system, but it does not appear to introduce new fundamental concepts to the field. For this reason, I cannot recommend acceptance at this time.
