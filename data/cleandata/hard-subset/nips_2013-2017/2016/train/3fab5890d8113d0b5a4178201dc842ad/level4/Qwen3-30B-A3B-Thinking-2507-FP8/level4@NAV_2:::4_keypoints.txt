- Introduces Sparse Access Memory (SAM): a MANN using sparse reads/writes (K=4–8) with ANN indexing to achieve O(log N) time and O(1) space per step.  
- Scales to 100k+ memory words, enabling training on much larger problems while remaining end-to-end differentiable and compatible with existing MANNs.  
- Reported practical gains: ~1,000× faster and ~3,000× less memory than dense implementations while maintaining near-identical data efficiency on synthetic tasks.  
- Demonstrates strong empirical results across Copy, Associative Recall, Priority Sort, Omniglot one-shot, and Babi QA tasks.  
- Sparse DNC variant attains state-of-the-art Babi performance without supervised memory access.  
- Theoretical analysis rigorously proves SAM’s asymptotic optimality and the sparse operations’ negligible gradient error for large N.  
- Novelty: end-to-end differentiable sparse memory access using efficient data structures (k-d trees/LSH), distinct from prior RL-based sparse approaches.  
- Significance: enables training on previously infeasible sequence/memory scales (e.g., 4,000+ sequence length), advancing real-world long-term memory applications.  
- Clarity: architecture, formalism, and experiments are well-explained with precise equations and illustrative figures.  
- Weakness: ANN index rebuild every N insertions introduces runtime overhead that is acknowledged but not fully quantified against alternatives (e.g., Mondrian forests).  
- Weakness: Simple “least recently accessed” write policy may not generalize to all access patterns; adaptive write metrics were not explored.  
- Weakness: Omniglot evaluation lacks comparison to recent meta-learning baselines, limiting contextual significance of one-shot claims.  
- Pro-accept: resolves a long-standing MANN scalability bottleneck, achieves strong empirical/theoretical rigor, and includes clear ablations.  
- Con-accept: remaining concerns about ANN overhead, write strategy generality, and limited Omniglot baselines warrant further work.  
- Overall: SAM is a significant, well-executed, and technically sound advance with minor issues that do not undermine the core contribution.
