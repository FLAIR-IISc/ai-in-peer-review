- Addresses scalability of memory-augmented neural networks where large memory increases time and space costs.  
- Proposes Sparse Access Memory (SAM) that uses approximate nearest neighbor (ANN) search to access a small relevant subset of memory.  
- Claims SAM preserves dense memory representational power while achieving up to 1000x training speedup and 3000x memory reduction.  
- Tackles an important, practical problem with potentially high impact for large-scale memory models.  
- Empirical results are impressive and experiments appear well-executed and convincing on the tasks shown.  
- Main technical contributions are: (1) applying ANN for memory retrieval, and (2) an efficient backpropagation scheme for sparse accesses.  
- Concern that these components are established techniques and their combination is more an engineering integration than a fundamental algorithmic innovation.  
- Efficient backpropagation follows directly from sparsity (gradients only for accessed slots), so novelty is limited.  
- Paper would be stronger with demonstrations of non-obvious challenges or new theoretical insights on approximation–sparsity–performance trade-offs.  
- Reviewer leans to reject due to limited technical novelty; recommends a workshop or applications-focused venue instead.
