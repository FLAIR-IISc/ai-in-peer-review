- Problem: Neural network models with external memory incur large computational and memory costs as memory size grows.  
- Contribution: Introduces Sparse Access Memory (SAM), a novel memory access method.  
- Claims: SAM maintains representational capacity while enabling much more efficient training.  
- Results: Reports up to 1000× speedups and 3000× memory reductions.  
- Criticism: Core techniques (efficient backpropagation and approximate nearest neighbor search) are viewed as relatively incremental.  
- Verdict: Experimental findings are promising but technical contribution insufficient for acceptance; reviewer recommends rejection.
