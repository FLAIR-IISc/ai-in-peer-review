- Proposes Sparse Access Memory (SAM), a novel architecture using sparse read/write to reduce compute and memory overhead in MANNs.
- Paper is well-structured with clear explanation of the architecture and its components.
- Background covers existing MANNs (NTMs, Memory Networks) but needs a brief description of memory-augmented networks and memory operations (e.g., write operator) for non-experts.
- Thorough experimental evaluation (speed/memory benchmarks, sparse access learning, curriculum scaling) demonstrating significant speedups and memory reductions while maintaining comparable performance.
- Examines learning cost of inducing sparsity and the effect of using approximate nearest neighbor indexes.
- Concern: Figure 4 comparison may be unfair—using equal physical memory vs equal memory size across models could alter conclusions.
- Strength: Novel solution addressing scalability issues of traditional MANNs.
- Strength: Comprehensive empirical validation showing efficiency gains.
- Weakness: Insufficient self-contained explanation of memory operations for readers unfamiliar with MANNs.
- Weakness: Potentially misleading experimental comparison in Figure 4.
- Originality: Introduces a sparse read/write scheme that meaningfully reduces computational and memory overhead.
- Significance: Could enable real-world adoption of MANNs by overcoming prior scalability limits.
- Quality: Well-written and easy to follow with promising results.
- For acceptance: Novel, well-evaluated, and potentially high-impact contribution.
- Against acceptance: Need for clearer exposition for non-experts and fairer experimental comparisons.
