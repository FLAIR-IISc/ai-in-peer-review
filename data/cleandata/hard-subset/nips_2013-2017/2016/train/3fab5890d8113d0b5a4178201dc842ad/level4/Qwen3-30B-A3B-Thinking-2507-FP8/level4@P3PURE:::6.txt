

Review: The paper introduces a Neural Turing Machine (NTM) variant featuring sparse read and write access to address a critical limitation of NTMs—namely, their prohibitive memory and computational requirements for backpropagation, which hinder real-world applicability. The proposed architecture aims to resolve this issue while maintaining performance comparable to state-of-the-art methods. The paper is well-structured and clearly written. To enhance accessibility for non-specialists, a brief background on memory-augmented neural networks in the introduction, or a clearer explanation of the memory mechanism (e.g., explicitly defining the intended behavior of the write operator relative to conventional memory systems), would be beneficial. While the comparison in Figure 4 is justifiably grounded in equal physical memory constraints—a reasonable approach for resource-limited scenarios—the analysis might be more informative if systems were compared under identical memory capacity rather than identical physical memory size, as this better reflects practical performance trade-offs. Nevertheless, it is particularly noteworthy that the algorithm achieves performance near theoretical bounds while operating with such sparse access.
