- Neural network models with external memory scale poorly in time and space as memory grows.  
- Authors propose Sparse Access Memory (SAM) to enable efficient memory access and training.  
- SAM reportedly preserves representational power while giving ~1000x speedups and ~3000x memory reduction.  
- Core technical contributions are efficient backpropagation and approximate nearest-neighbor lookup using sparse access.  
- Reviewer views these contributions as relatively straightforward and not technically strong.  
- Despite interesting experimental results, reviewer does not recommend acceptance.
