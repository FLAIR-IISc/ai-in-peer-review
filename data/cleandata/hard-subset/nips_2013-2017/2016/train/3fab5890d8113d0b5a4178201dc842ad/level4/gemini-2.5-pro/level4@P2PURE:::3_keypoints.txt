- Introduces Sparse Access Memory (SAM), a novel memory access scheme for memory-augmented neural networks.  
- SAM constrains reads and writes to a fixed-size subset of memory words instead of dense full-memory access used by NTMs.  
- Provides a theoretical justification and claims optimality for the sparse access methodology.  
- Demonstrates superior read and write performance due to accessing fewer memory locations (Section 4.1).  
- Evaluates impact of sparsity on learning convergence using standard NTM tasks (copy, associative recall, priority sort).  
- Shows lower combined learning cost for associative recall and priority sort compared to the NTM baseline (Section 4.2).  
- Includes a real-world benchmark implemented in Torch7 (Section 4.4).  
- Finds sparse access controlled by a linear layer or ANN can improve early-stage learning in some cases, but lacks analysis of conditions for this effect.  
- Suggests exploring different controller architectures combined with SAM as a valuable extension.  
- Paper is well-written with clear assumptions, architectural details in the appendix, and a compelling, well-evaluated contribution.
