- Existing neural memory models scale poorly in space and time as memory size increases.
- The paper proposes Sparse Access Memory (SAM) as a novel memory access approach.
- Authors claim SAM maintains representational capabilities while enabling efficient training.
- Reported empirical gains include ~1000× speedup and ~3000× memory efficiency improvements.
- Reviewer finds the core technical components (efficient backprop and approximate nearest neighbor search) straightforward and lacking in complexity.
- Despite intriguing experimental results, the reviewer does not support acceptance due to overall technical weaknesses.
