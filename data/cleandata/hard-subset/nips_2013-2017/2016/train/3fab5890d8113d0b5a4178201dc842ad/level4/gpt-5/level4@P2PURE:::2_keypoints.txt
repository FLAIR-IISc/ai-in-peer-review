- Problem: Neural network models with external memory scale poorly in time and space as memory size increases.
- Contribution: Authors introduce Sparse Access Memory (SAM) as a memory access scheme.
- Claim: SAM preserves representational power while enabling efficient training.
- Empirical results: Reported up to 1000× speedups and 3000× improvements in memory efficiency.
- Reviewer judgment: Technical contributions are not strong enough for acceptance.
- Rationale: Key components (efficient backpropagation and approximate nearest neighbor via sparse memory access) are relatively straightforward.
- Conclusion: Despite interesting experiments, reviewer is not in favor of acceptance.
