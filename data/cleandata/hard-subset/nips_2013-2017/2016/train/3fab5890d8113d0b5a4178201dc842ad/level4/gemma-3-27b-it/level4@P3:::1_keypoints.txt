- Introduces Sparse Access Memory (SAM) to scale memory-augmented networks via sparse read/write and approximate nearest neighbor search.  
- Core idea: limit memory modifications to a sparse subset and use ANN to reduce time/space complexity of MANNs/NTMs.  
- Reports large empirical gains: ~1,000x speedup and ~3,000x memory reduction while maintaining competitive performance.  
- Sound technical approach with theoretical asymptotic analysis supporting sparse operations and efficient data structures.  
- Empirical validation across diverse tasks (synthetic copy/associative recall/priority sort, bAbI, Omniglot one-shot) and scaling to long sequences.  
- Demonstrates generality by adapting approach to the Differentiable Neural Computer (SDNC).  
- Paper is clearly written and well organized.  
- Limited novelty: use of hashing/ANN for memory access has prior precedent, especially for read-only memories; writable-memory extension needs clearer positioning.  
- Reproducibility concerns: insufficient task detail and reliance on unpublished NTM code; code release and more detail required.  
- Lacks analysis of failure cases and conditions where SAM performs poorly; arg-top-K non-smoothness and its effects need discussion.  
- Mischaracterizes Omniglot as “non-synthetic/real-world”; dataset is curated and differs from real-world data.  
- Arguments for acceptance: tackles a key scalability bottleneck, delivers strong empirical/resource improvements, clear presentation, and adaptable method.  
- Arguments for rejection: limited core novelty, reproducibility issues, and insufficient analysis of limitations/failures.  
- Overall recommendation: accept with revisions — release code, add task and failure-case details, analyze K-argument impact, move ANN details to supplementary material, and clarify dataset characterization.
