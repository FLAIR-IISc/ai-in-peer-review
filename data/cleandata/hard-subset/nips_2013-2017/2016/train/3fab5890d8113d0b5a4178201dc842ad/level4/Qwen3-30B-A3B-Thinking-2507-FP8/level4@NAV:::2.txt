

Most existing neural network models incorporating memory exhibit poor scalability in both space and time as memory capacity increases. The authors propose a memory access mechanism termed Sparse Access Memory, demonstrating it preserves representational capacity while enabling efficient training—achieving a 1000× speedup and 3000× memory efficiency improvement. However, the technical contribution lacks sufficient depth for acceptance; the core innovations—efficient backpropagation and approximate nearest neighbor search via sparse access—are relatively straightforward. Despite the compelling experimental results, the foundational technical novelty is insufficient to warrant acceptance.
