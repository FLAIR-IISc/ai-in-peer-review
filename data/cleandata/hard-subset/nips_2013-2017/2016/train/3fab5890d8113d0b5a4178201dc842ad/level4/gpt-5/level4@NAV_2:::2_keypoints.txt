- Problem: Neural networks with external memory scale poorly in time and space as memory size increases.  
- Proposal: Authors introduce Sparse Access Memory (SAM) as a memory access mechanism to address scaling issues.  
- Claim: SAM preserves the expressive power of prior models while enabling efficient training.  
- Results: Reported up to 1000× speedups and ~3000× reductions in memory footprint.  
- Reviewer critique: Paper is not technically strong enough for acceptance.  
- Technical assessment: Main contributions (sparse backpropagation updates and approximate nearest neighbor indexing for sparse reads/writes) are viewed as relatively straightforward.  
- Overall: Experiments are appealing, but reviewer does not recommend acceptance.
