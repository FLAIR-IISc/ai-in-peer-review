- Problem: MANNs scale poorly since computation and memory grow linearly with external memory size, limiting large-scale use.  
- Proposal: Introduces Sparse Access Memory (SAM) that restricts reads/writes to a small constant subset of memory locations.  
- Mechanism: Replaces dense attention with sparse attention; uses ANN structures (k-d trees, LSH) to find top-K relevant memory words in sub-linear time.  
- Writes: Performs sparse writes to recently read locations or the least recently used slot.  
- BPTT: Presents an efficient backpropagation method that tracks/reverts sparse updates, reducing per-timestep space from O(N) to O(1).  
- Relation to prior work: Builds on MANNs/NTM but focuses on scalability rather than new learning algorithms.  
- Quality: Technically sound with thorough experiments supporting asymptotic and empirical efficiency claims.  
- Empirical gains: Reports up to 1000x speedups and 3000x memory reduction while maintaining or improving learning performance.  
- Clarity: Manuscript is well-written, motivated, and clearly explains architecture and experiments.  
- Originality concern: Core techniques (ANN, top-K sparsification, sparse BPTT) are standard and the contribution is mainly system-level integration.  
- Significance: High practical impact by enabling training on long sequences and millions of memory words, opening new applications.  
- For acceptance: Effective, well-engineered solution to an important scalability problem with strong empirical validation across diverse tasks.  
- Against acceptance: Limited conceptual novelty—primarily application of known data structures and heuristics to MANNs.  
- Recommendation: Solid engineering and significant results but judged borderline due to lack of core technical novelty; reviewer leans toward rejection.
