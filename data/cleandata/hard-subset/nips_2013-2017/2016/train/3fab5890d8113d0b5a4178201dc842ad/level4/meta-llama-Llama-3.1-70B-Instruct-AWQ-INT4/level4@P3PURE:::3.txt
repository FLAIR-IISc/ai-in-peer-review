This paper introduces a novel memory access scheme called Sparse Access Memory (SAM) in memory-augmented neural network architectures. This research area is fairly recent, with the invention of Neural Turing Machines (NTMs), which allow neural networks to learn beyond the limits of traditional Long Short-Term Memory (LSTM) models. The primary contribution of this paper is that all writes to and reads from external memory are constrained to a fixed size, specifically a subset of the memory words, rather than an unlimited size.

The authors demonstrate the methodology and its optimality, which conceivably achieves good read and write performance due to the reduced memory size, as shown in Section 4.1. Furthermore, the authors assess the learning costs of introducing sparsity in Section 4.2 using standard NTM tasks, namely copy, associated call, and priority sort. The combined costs are lower than those of NTMs for some tasks (i.e., tasks 2 and 3). The inclusion of real-world examples in Section 4.4 is also appreciated. The benchmarking is performed using Torch7, which provides a suitable framework for evaluation.

The idea presented is straightforward and novel, with significant speedup compared to NTMs, which is impressive. The fact that sparse reads and writes, achieved through either linear or artificial neural network (ANN) approaches, can benefit early-stage learning in some cases is particularly interesting (Section 4.2). To further enhance the paper, it would be helpful if the authors could elaborate on how to generalize SAM to other cases, including scenarios where learning could be improved, and explore different ANN approaches that can be used in conjunction with SAM. This would provide valuable insights for the audience.

Overall, the paper is well-written, with clear explanations of assumptions and a detailed architecture description in the appendix, making it an interesting read.
