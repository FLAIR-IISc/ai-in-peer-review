This paper introduces a new memory access scheme called Spars Access Memory (SAM) in memory augmented neural network architecture. This research area is quite recent, with the discovery of the neural Turing machine. This augmentation allows neural networks to learn beyond the limitations of traditional LSTM models. The contribution of this paper is that writing and reading from external memory are limited to a specific size, so a subset is used instead of an unlimited size of memory words. The authors have proven the method and its superiority. It is likely to achieve better read and write performance due to the reduced memory size, as shown in section 4.1. However, the authors have also included the learning cost of introducing Spars in section 4.2 using standard NTM tasks: 1. Copy, 2. Associated Call, and 3. Priority Sort. The combined cost for some tasks (such as 2 and 3) is lower than NTM. I also like that the authors have included real-world examples in section 4.4. The benchmark is created using Torch7. The concept is simple and innovative. Its speed is significantly higher than NTM, which is very impressive. This Spars read and write can benefit the primary level learning in some cases done by a linear or an ANN, which is very interesting (section 4.2). It would have been very helpful for the audience if the authors could have thought in detail about how to generalize to other cases (where learning improvement can be made), or about accessing different ANN approaches using SAM. Overall, this is a well-written paper, where all assumptions are clearly stated and the architecture is provided in the appendix. It is interesting to read.
