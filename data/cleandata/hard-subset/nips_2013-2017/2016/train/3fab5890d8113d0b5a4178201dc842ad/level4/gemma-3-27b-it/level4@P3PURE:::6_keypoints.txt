- Proposes a Neural Turing Machine with sparse read/write access to reduce required physical memory and backprop compute.
- Addresses a major NTM limitation, enabling applicability to larger, real-world tasks while keeping performance similar to state-of-the-art.
- Paper is well-structured and clearly written.
- Recommend adding a brief background overview of memory-augmented neural networks for less-familiar readers.
- Recommend a more detailed explanation of memory operations, especially clarifying the intended function of the write operator versus conventional memory systems.
- Suggest adding comparisons using identical memory sizes (not just identical physical memory constraints) to better reveal algorithmic efficiency.
- Empirical results are strong, approaching theoretical bounds, indicating significant potential.
