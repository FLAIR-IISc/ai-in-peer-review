- Introduces Sparse Access Memory (SAM), a novel memory access scheme for memory-augmented neural networks.
- Positions work in context of recent NTM research and as an extension beyond LSTM capabilities.
- Main contribution: constrain all reads/writes to a fixed-size subset of memory words instead of unbounded access.
- Provides proofs of the methodology and claims of optimality.
- Section 4.1: reduced effective memory size yields strong read and write performance.
- Section 4.2: analyzes learning costs from sparsity using standard NTM tasks (Copy, Associative Recall, Priority Sort).
- Combined learning cost is lower than NTM for some tasks (notably Associative Recall and Priority Sort).
- Includes a real-world example (Section 4.4) and Torch7 benchmark implementations.
- Observes significant and impressive speedup relative to NTM; idea is straightforward and novel.
- Finds that sparse reads/writes driven by linear controllers or ANNs can aid early-stage learning in some cases.
- Requests more analysis on when early-stage learning benefits generalize and evaluation of different ANN approaches with SAM.
- Overall assessment: well-written, clear assumptions, detailed appendix, and an interesting contribution.
