- Introduces Sparse Access Memory (SAM), a neural memory architecture using sparse reads/writes to scale efficiently with large external memories.  
- Addresses poor space/time scaling of existing MANNs by using efficient data structures for sparse memory management.  
- Proposes tracking of memory usage and finding unused memory blocks for efficient recording of new information.  
- Introduces an efficient backpropagation-through-time algorithm that reduces MANNs’ space complexity.  
- Provides thorough time and space complexity analysis and claims significant training speedups and memory reductions.  
- Experimental results show SAM outperforms dense variants (DAM) and original NTM in speed and memory usage.  
- Demonstrates comparable data efficiency to existing models on synthetic tasks and one-shot Omniglot recognition.  
- Paper is well-written, well-organized, and provides sufficient detail for expert reproduction.  
- Originality: novel sparse read/write scheme enabling scalable MANNs; builds on but extends NTMs/Memory Networks.  
- Significance: could enable training with large memories, impacting language modeling and machine translation.  
- Pro-acceptance: novel scaling solution, solid analysis, technical soundness, and promising experimental evidence.  
- Con-acceptance: contributions may be seen as incremental over prior MANN work.  
- Con-acceptance: lacks comprehensive comparison with other state-of-the-art MANN architectures.  
- Con-acceptance: assumes reader familiarity with MANN/NTM basics, limiting accessibility to non-experts.
