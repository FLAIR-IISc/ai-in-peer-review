This paper proposes Sparse Access Memory (SAM), a memory-augmented neural network that scales external memory to millions of slots by enforcing sparsity in both read and write operations and by integrating efficient approximate nearest neighbor (ANN) data structures. The key ideas are: (1) sparse reads that query only the top-K memory slots returned by an ANN index (k-d trees or LSH) and then perform content-based soft attention over this small set; (2) sparse writes that target previously read locations and a least-recently-used (LRU) location, with a constant number of modified slots per step; and (3) a reversible, in-place memory update scheme to enable backpropagation through time (BPTT) using O(1) space per step by logging sparse deltas and rolling memory states back during the backward pass. The authors claim O(log N) time per step (with ANN) and O(1) space per time step (O(N) initialization), arguing asymptotic optimality under content-based access assumptions. They also show how to adapt the approach to the Differentiable Neural Computer (DNC), producing a Sparse DNC (SDNC).

Empirically, SAM is 100–1600× faster and uses orders of magnitude less physical memory than dense NTMs at large N; it matches or exceeds dense baselines (NTM, a dense approximation DAM, LSTM) on synthetic NTM tasks; scales via curriculum to >4000 time steps and millions of memory slots; achieves strong unsupervised results on bAbI (especially with SDNC); and improves one-shot Omniglot performance on long episodes, presumably due to larger memory capacity. The work is grounded in prior MANNs (NTM, Memory Networks) and DNC, and relates to RL-based sparse addressing (RL-NTM). The use of ANN within end-to-end training connects to FLANN/LSH literature.

Quality:
- Theory/algorithms: The sparsified read/write mechanism and the reversible update scheme are well-motivated. Complexity arguments (O(1) space per step; O(log N) query/insert/delete) are plausible and relegated to the supplement; it would help to clarify assumptions (e.g., fixed small K; word dimension regimes for which k-d trees vs LSH achieve sublinear behavior; periodic rebalancing costs).
- Differentiability: The neighbor set selection via ANN is non-differentiable; gradients do not flow through the index. The model computes a soft attention over the selected K items, ignoring the dependence of the selection on q. This approximation is central; empirically it works well, but a brief discussion of stability and potential bias (e.g., missed neighbors due to ANN recall) would strengthen the theoretical story.
- Experimental rigor: The speed/memory benchmarks are compelling and directly address the paper’s goal. Synthetic tasks reproduce NTM-style results and the scaling curriculum convincingly demonstrates long-horizon BPTT. bAbI results (especially SDNC) are strong and competitive without supervision. Omniglot experiments show better long-episode generalization. Some concerns: (i) Dense vs sparse models are sometimes matched on physical memory by severely limiting dense memory size (e.g., 64 slots) while giving SAM 2×10^6 slots; this is fair for a scalability claim but obscures algorithmic efficiency comparisons independent of capacity. (ii) Ablations are limited: the effect of K, choice of ANN (kd-tree vs LSH), ANN recall/latency trade-offs, and the write policy (mix of previous-read and LRU) are not explored. (iii) For bAbI, it would be useful to report SAM-only (without SDNC) on the full suite in the main text and detail training budgets and hyperparameters across models to ensure fairness.
- Reproducibility: Many implementation details appear in the supplement (ANN maintenance, usage tracking, reversible updates, curricula). It would help to include pseudocode for forward/backward memory updates and concrete ANN hyperparameters (number of trees/hashes, rebuild cadence) in the main paper.

Clarity:
The paper is generally clear, with a good high-level narrative and helpful complexity intuition. Some notation (e.g., I_U, usage definitions, thresholds) could be streamlined, and the read/write equations and control flow would benefit from a concise algorithmic summary in the main text. The role of the gates (α_t, τ_t) and their ranges could be explained more explicitly. Figures clearly illustrate scaling benefits.

Originality:
While MANNs, content-addressable memory, and usage-based allocation exist (NTM, Memory Networks, DNC), this paper’s combination of (i) hard sparse top-K access via an external ANN inside an end-to-end trained neural system, and (ii) reversible sparse updates enabling O(1) memory for BPTT, is novel and impactful in 2016. The SDNC adaptation is a timely extension.

Significance:
High. Scaling external memory to millions of slots with large speed and memory savings addresses a key bottleneck for MANNs, potentially enabling applications in long-context reasoning and continual learning. The empirical demonstrations (long-horizon curricula, bAbI without supervision, large-episode Omniglot) substantiate practical value.

Pros (arguments for acceptance):
- Strong, well-motivated scalability contribution with clear complexity improvements.
- Large empirical speed/memory gains; practical engineering that others can adopt.
- Comparable or better learning performance relative to dense baselines on multiple domains.
- Extension to DNC yields state-of-the-art bAbI results without supervision.
- Generally clear presentation and good coverage of related work.

Cons (arguments against acceptance):
- Non-differentiable ANN selection is central; theoretical implications and failure modes are under-discussed.
- Limited ablations on K, ANN choice/quality, and write policy; lack of sensitivity analyses.
- Some comparisons confound capacity with algorithmic efficiency (dense baselines constrained to tiny memories).
- Real-world tasks beyond Omniglot are limited; no large-scale NLP/LM or MT despite motivation.
- Practical details (ANN hyperparameters, rebuild frequency, memory dimension effects) could be more explicit in the main text.

Suggestions for improvement:
- Add ablations: vary K; compare exact vs ANN; kd-tree vs LSH vs linear; measure ANN recall and its effect on learning; analyze the trade-off between speed and accuracy.
- Provide pseudocode for forward/backward sparse memory updates and the usage/eviction data structures.
- Report SAM-only bAbI results comprehensively in main text and clarify training budgets.
- Discuss when kd-trees degrade (high-dimensional words) and how often ANN rebuilds occur; report amortized costs.
- Explore a sparse content-based write variant as an alternative to the previous-read/LRU heuristic.

Overall, this is a strong and timely contribution that substantially improves the scalability of memory-augmented models with thoughtful algorithmic and systems design. Despite some missing ablations and theoretical discussion around the non-differentiable ANN, the paper meets NIPS criteria on quality, originality, clarity, and significance. I lean toward acceptance.
