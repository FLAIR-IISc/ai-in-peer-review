- Addresses scalability of MANNs by eliminating dense read/write operations that scale linearly with memory size.  
- Proposes Sparse Access Memory (SAM): restricts access to a constant-sized subset using ANN indices for O(log N) reads and a sparse write scheme.  
- Introduces an efficient backpropagation that tracks only sparse memory modifications, reducing per-timestep space from O(N) to O(1).  
- Reports dramatic empirical improvements (up to ~1000x speed and ~3000x memory efficiency), enabling training with millions of memory slots.  
- Strong experimental validation across algorithmic tasks, bAbI QA, and Omniglot one-shot learning; includes a dense baseline (DAM) to isolate sparsity effects.  
- Novel contribution: integrating classic ANN data structures into an end-to-end differentiable model for efficient sparse attention.  
- Paper is well-written and clearly explains motivation, architecture, and results.  
- Minor weakness: sensitivity to ANN index and sparsity parameter K not thoroughly explored.  
- Minor weakness: write mechanism is heuristic (write-to-read or least-used); a learned sparse write policy might be preferable.  
- Minor weakness: lacks deeper analysis of ANN index dynamics (rebuild costs and effects on training stability).  
- Overall assessment: significant, practical, and principled advance for memory-augmented models; strengths far outweigh minor weaknesses.
