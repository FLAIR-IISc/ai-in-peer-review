- Introduces Sparse Access Memory (SAM), a memory architecture to solve scalability limits in memory-augmented neural networks.  
- SAM restricts reads and writes to a small sparse subset (K non-zero entries) using approximate nearest neighbor (ANN) search.  
- Theoretical analysis shows SAM achieves asymptotic optimal complexity: O(log N) read, O(1) write, O(N) initialization.  
- Empirical results report up to 1,000× speedup and 3,000× memory reduction compared to dense models.  
- Maintains comparable performance to dense models on synthetic algorithmic tasks (copy, associative recall, priority sort).  
- Matches performance on Babi reasoning tasks and Omniglot one-shot classification while scaling to 100,000s of steps and memory entries.  
- Extends to Differentiable Neural Computer (SDNC) and achieves state-of-the-art results on Babi without supervised memory access.  
- Builds on and differentiates clearly from prior work (NTMs, Memory Networks, DNC) with theoretical complexity emphasis.  
- Experimental evaluation is comprehensive with clear figures, tables, and extensive supplementary implementation details.  
- Theoretical claims are rigorous and well-supported by experiments.  
- Minor weakness: limited exploration of how the sparsity parameter K affects performance across tasks.  
- Potential empirical limitation: ANN index overhead might reduce practical speedups in some settings.  
- Lacks analysis of scenarios where dense memory access could be necessary or advantageous.  
- Clarity is strong; architecture and backpropagation methods are well-explained, though a diagram of SAM is suggested.  
- Originality: first to achieve end-to-end differentiable MANN with asymptotic optimal space/time complexity via sparse access.  
- Significance: enables training and application of MANNs on much larger, more realistic problems requiring long-term memory.  
- Overall conclusion: technically strong, significant work with minor weaknesses; reviewer recommends acceptance.
