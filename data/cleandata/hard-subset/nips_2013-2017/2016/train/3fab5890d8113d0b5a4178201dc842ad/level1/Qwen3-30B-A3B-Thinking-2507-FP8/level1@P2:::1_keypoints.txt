- Paper introduces Sparse Access Memory (SAM), a memory architecture enabling efficient training of MANNs with very large external memories.  
- Addresses scalability limits of existing MANNs (e.g., NTMs) that scale poorly in space and time as memory size grows.  
- SAM constrains reads/writes to a sparse subset of memory words (K=4–8) and uses efficient data structures for content-based addressing.  
- Introduces a memory management scheme that tracks usage and locates unused memory blocks.  
- The authors prove SAM achieves asymptotically optimal time and space complexity and provide detailed supplementary proofs.  
- Extensive experiments compare SAM to dense variants (DAM), NTMs, and LSTMs on synthetic tasks (Copy, Associative Recall, Priority Sort), Babi, and Omniglot.  
- Empirical results show SAM matches or exceeds dense models while achieving orders-of-magnitude improvements in speed and memory efficiency.  
- Reported performance gains include ~1,000× faster runtime and ~3,000× less memory usage than non-sparse models.  
- Paper is technically sound, well-written, and organized with clear mathematical formulations and helpful figures.  
- Original contribution: first differentiable MANN achieving asymptotic optimality in space/time via sparse access, distinct from prior sparse ideas.  
- Significant impact: enables scaling MANNs to 100k+ words and long sequences (200k+), opening real-world application possibilities.  
- Strengths: solves a core scalability problem, strong theory and empirical validation, practical applicability, generalizable approach.  
- Weaknesses: limited exploration of different approximate nearest neighbor (ANN) index types and minor clarifications needed in memory management details.  
- Overall recommendation: strong contribution, well-suited for NIPS 2016, recommended for acceptance.
