- Summary: Introduces SAM, a memory-augmented network using sparse reads/writes and approximate nearest neighbors to achieve scalable, end-to-end differentiable memory with O(log N) read time and O(1) per-step space.
- Key empirical claim: SAM achieves up to 1600x speedup over NTMs while maintaining or improving task performance.
- Theoretical quality: Provides rigorous asymptotic analysis showing optimal time and space complexity.
- Empirical quality: Extensive experiments demonstrate dramatic memory and runtime savings (e.g., 53MB init vs 29GB for NTM at 64k words).
- Learning robustness: Sparse approximations do not impair learning and can improve performance on tasks like priority sort and associative recall.
- Curriculum/scale: Curriculum experiments show SAM handling sequences >4,000 words and millions of memory slots, scaling beyond prior MANNs.
- Babi/Omniglot: SDNC achieves state-of-the-art on Babi without supervised memory access and strong one-shot results on Omniglot.
- Clarity and reproducibility: Paper is well-written, organized, and supplemented with implementation details sufficient for reproduction.
- Originality: Novel end-to-end differentiable use of sparse access and ANN data structures within MANNs, distinct from RL-based sparse approaches.
- Significance: Solves a fundamental scalability barrier for MANNs, enabling practical large-memory applications and broader adoption.
- Strengths: Solves scalability, delivers large efficiency gains, preserves/improves learning, provides theory and thorough validation, and is clearly presented.
- Weaknesses: Limited discussion of potential failure modes for the sparse approximation and some ANN implementation details could be expanded.
- Recommendation: Accept — high-quality, significant contribution advancing the state of the art in memory-augmented neural networks.
