- Introduces Sparse Access Memory (SAM) to scale MANNs/NTMs by restricting reads/writes to a sparse subset of memory.  
- SAM plus efficient data structures reduces quadratic time/space complexity, yielding up to ~1000x improvements while preserving performance.  
- Strong empirical results across algorithmic tasks, reasoning (bAbI), and Omniglot, including ability to handle longer sequences via curriculum.  
- Applies the sparse approach to the Differentiable Neural Computer (DNC), showing the method generalizes and speeds up other memory architectures.  
- Theoretical analysis claims optimal space/time complexity with supporting supplementary material.  
- Paper is clear, well-written, and provides comprehensive evaluation and consistent notation.  
- Performance depends heavily on approximate nearest neighbor (ANN) search; the choice and effect of ANN inaccuracies are not thoroughly explored.  
- Method may be sensitive to hyperparameters (e.g., sparsity K and threshold ε), but lacks detailed sensitivity analysis.  
- Components (sparse ops, ANN, memory management) are individually not novel; primary contribution is their integration to solve scalability.  
- Use of curriculum learning complicates attribution of gains solely to the sparse memory access scheme.  
- Reviewer recommends acceptance with minor revisions.  
- Requested revisions: analyze ANN accuracy impact and alternatives, add hyperparameter sensitivity analysis, and better situate novelty relative to prior sparse/approximate work.
