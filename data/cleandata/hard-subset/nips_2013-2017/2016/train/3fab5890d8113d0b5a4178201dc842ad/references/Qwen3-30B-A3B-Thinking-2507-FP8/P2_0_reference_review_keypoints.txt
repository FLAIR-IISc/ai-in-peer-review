- Introduces SAM (Sparse Access Memory), a MANN that scales external memory via sparse reads/writes (K=4–8) using ANN for content-based addressing.
- SAM achieves O(log N) read time and O(1) space per timestep, with theoretical proof of asymptotic optimality (Supplementary A).
- Empirically reports up to 1600x speedup and ~1000x memory reduction vs. vanilla NTMs while maintaining near-identical performance.
- Evaluated across algorithmic tasks, bAbI reasoning, and Omniglot classification, with thorough ablation studies against DAM, NTM, and LSTM.
- Extends approach to SDNC (Sparse DNC), achieving state-of-the-art on bAbI without memory access supervision.
- Paper is technically sound with rigorous theoretical analysis and comprehensive supplementary material (hyperparams, implementation details).
- Exceptionally well-written and organized, with clear architecture descriptions, precise math, and informative figures (Fig. 1–4).
- Novel contribution: integrates efficient ANN data structures into MANNs to achieve sublinear content addressing, differentiating from prior dense/limited-memory models.
- Significant impact: enables scaling to ~1M-word memories, generalizes to sequences 20x longer than training, and opens new applications for large-scale MANNs.
- Strengths: solves a major scalability bottleneck, strong empirical gains, solid theory, comprehensive experiments, and clear presentation.
- Limitation: “optimal” claim could use more discussion about required assumptions for optimality.
- Limitation: ANN implementation/synchronization with memory updates and lack of gradient flow through the ANN index need more explicit treatment.
- Limitation: Trade-offs between ANN accuracy and memory efficiency (how ANN choice affects performance) are not fully explored.
- Recommendation: Strong accept — technically strong, empirically validated, significant contribution, with only minor issues to address.
