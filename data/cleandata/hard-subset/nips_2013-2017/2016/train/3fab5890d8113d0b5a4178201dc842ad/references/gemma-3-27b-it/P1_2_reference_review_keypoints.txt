- Proposal: Introduces SAM, a memory-augmented neural network using sparse reads/writes to scale MANNs.
- Core idea: Constrains memory access to a sparse subset to dramatically cut compute and memory overhead.
- Empirical gains: Reports up to three orders of magnitude speedups and substantial memory savings.
- Performance: Sparsity does not significantly degrade accuracy; SAM matches or outperforms dense MANNs on varied tasks.
- Tasks evaluated: Tests include algorithmic tasks, bAbI reasoning, and Omniglot one-shot classification with curriculum learning.
- Extension: Presents Sparse DNC (SDNC), showing the sparsity approach generalizes to other memory architectures.
- Implementation: Uses efficient data structures (CSR) and approximate nearest neighbor (ANN) search for practical viability.
- Writing/presentation: Paper is clear, well-organized, and accessible.
- Relation to prior work: Builds on NTMs, Memory Networks, and DNCs; leverages ANN techniques from IR/database literature.
- Strength of evaluation: Comprehensive empirical evaluation across synthetic and real-world datasets.
- Weakness — ANN details: Specific ANN implementations (FLANN vs. LSH) and parameter choices are insufficiently detailed.
- Weakness — hyperparameters: Sensitivity to key hyperparameters (e.g., sparsity level K, usage parameter) is not fully explored.
- Weakness — comparisons: Limited comparison to alternative scaling methods (e.g., discrete addressing via RL).
- Weakness — theory: Asymptotic optimality claims rely on assumptions detailed only in the supplement and need clearer exposition.
- Recommendations: Add ANN implementation details, analyze hyperparameter sensitivity, compare to alternative scalability approaches, and clarify theoretical claims.
- Verdict: Strong contribution that addresses a critical bottleneck; reviewer recommends acceptance (high confidence).
