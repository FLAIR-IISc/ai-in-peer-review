- Overall assessment: SAM proposes sparse read/write MANNs that significantly reduce compute and memory costs and is recommended for acceptance.
- Problem/positioning: Addresses quadratic scaling of MANNs by restricting access to a sparse subset and using ANNs/CSR for efficient content-based reads, positioned relative to NTM, Memory Networks, and DNC.
- Key idea: Sparse memory access plus efficient data structures enables practical BPTT without full memory duplication.
- Efficiency gains: Reports up to three orders of magnitude speedup and large memory reductions.
- Theoretical basis: Claims asymptotic optimality in space and time, with further details in supplement.
- Empirical validation: Strong results across algorithmic tasks, Babi reasoning, and Omniglot one-shot learning; scales to longer sequences via curriculum learning.
- Extensibility: Successfully adapts sparse access to DNC (SDNC), showing generality.
- Clarity: Paper is well-written with useful supplementary implementation and experimental details.
- Novelty: Applies sparsity and ANNs specifically to MANNs and efficient BPTT—a notable contribution.
- Weakness — ANN analysis: Insufficient comparison of ANN methods (FLANN vs LSH) and lack of detail on ANN parameter choices.
- Weakness — hyperparameters: Limited discussion of hyperparameter sensitivity and tuning procedures.
- Weakness — ablation: Lacks granular ablations (sparse reads vs writes, ANN vs linear search) to isolate component contributions.
- Weakness — controller: Unclear how controller complexity scales with memory size and whether simpler controllers would suffice.
- Significance: Enables scaling MANNs to long-term memory tasks and real-world applications; SDNC achieves SOTA on Babi without supervised memory access.
- Acceptance arguments: Solves a core MANN bottleneck, demonstrates large efficiency gains, strong empirical results, clear exposition, and generalizability.
- Rejection arguments: Missing detailed ANN sensitivity analysis, hyperparameter tuning details, and more comprehensive ablations.
- Final recommendation: Accept, with a request to address ANN performance analysis and hyperparameter sensitivity in the camera-ready.
