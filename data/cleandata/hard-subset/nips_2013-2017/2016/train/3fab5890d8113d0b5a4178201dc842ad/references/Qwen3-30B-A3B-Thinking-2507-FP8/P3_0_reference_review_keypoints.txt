- Proposes SAM, a memory-augmented neural network that constrains reads/writes to a sparse subset (K=4–8) while keeping end-to-end differentiability.  
- Claims SAM achieves O(log N) time and O(1) per-step space vs O(N) for standard MANNs, with large empirical speed/memory gains.  
- Uses approximate nearest neighbor (k-d trees, LSH) for sparse reads to avoid full memory scans and a usage-tracking scheme for writes.  
- Provides theoretical analysis (Supplementary A) arguing SAM’s asymptotic optimality in time/space.  
- Extensive experiments on NTM tasks, bAbI, and Omniglot show SAM matches/exceeds dense MANNs with up to 1600x speedups and ~1000x memory reduction.  
- Demonstrates strong generalization via curriculum learning to sequences 200x longer and scaling to 1M memory words / 200k sequence lengths.  
- Weakness: The “asymptotically optimal” claim lacks a formal lower-bound proof for content-based addressing.  
- Weakness: Paper observes sparse models sometimes outperform dense ones (e.g., priority sort) but provides no explanation.  
- Weakness: ANN implementation choices (k-d tree vs LSH) lack justification and no ablation on ANN quality or its effect on performance.  
- Weakness: No direct main-text comparison to the original DNC despite its relevance; only a supplemental mention of Sparse DNC.  
- Relation to prior work: Correctly situates relative to NTMs and Memory Networks but should better contrast SAM vs DNC addressing/scaling.  
- Strengths: Significant practical impact, rigorous multi-task evaluation, clear and implementable sparse memory management.  
- Recommendation: Strong accept with minor revisions to strengthen theoretical justification and ANN implementation details.
