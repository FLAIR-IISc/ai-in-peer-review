- Addresses MANN scalability by replacing dense memory access with Sparse Access Memory (SAM) that touches only a constant number of slots per step.
- Uses an Approximate Nearest Neighbor (ANN) index for efficient content-based reads within the training loop.
- Employs a write scheme targeting previously read or least-recently-used locations to maintain memory utility.
- Reduces time complexity of memory access from linear to sub-linear (logarithmic) with memory size.
- Makes it feasible to train models with very large external memories (e.g., millions of words), enabling long-form QA, document modeling, and lifelong learning.
- Novel, technically strong contribution with end-to-end differentiable sparse access and clever backpropagation that reverts sparse modifications instead of caching full memory.
- Extensive experiments and speed benchmarks show orders-of-magnitude improvements and preserved or improved learning performance.
- Exceptionally clear and well-written presentation with sufficient technical detail for experts.
- Minor weakness: lacks deeper discussion of trade-offs and failure modes when using ANN structures whose data distributions shift during training.
- Overall recommendation: strong accept; impactful work likely to be built upon by others.
