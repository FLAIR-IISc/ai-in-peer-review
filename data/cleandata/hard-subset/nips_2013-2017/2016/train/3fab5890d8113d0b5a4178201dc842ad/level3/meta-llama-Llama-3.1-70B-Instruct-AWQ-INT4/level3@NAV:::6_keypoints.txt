- Proposes Sparse Access Memory (SAM), an NTM variant using sparse read/write to improve scalability.
- Claims SAM attains state-of-the-art performance while reducing physical memory and backpropagation operations.
- Paper is well-structured and accessible to a broad audience.
- Provides clear explanation of SAM architecture and components (sparse read/write, controller, efficient BPTT).
- Empirically demonstrates scalability: handles large external memories with improved runtime and memory overhead versus traditional NTMs.
- Matches or approaches existing models in data efficiency and task performance on synthetic tasks and one-shot Omniglot.
- Could be more self-contained by adding a brief background on memory-augmented neural networks.
- Figure 4 comparison may be unfair—suggest comparing using the same logical memory instead of the same physical memory.
- Original contribution: novel approach to NTM scalability, and shows generality by applying SAM ideas to other models (e.g., DNC).
- Significance: addresses a difficult problem more effectively than prior work and advances the state of the art.
- Overall assessment: strengths outweigh weaknesses; paper is a strong candidate for acceptance.
