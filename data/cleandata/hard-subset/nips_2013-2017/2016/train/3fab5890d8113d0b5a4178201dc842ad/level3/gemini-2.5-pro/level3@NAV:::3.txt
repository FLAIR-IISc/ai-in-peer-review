This paper addresses a critical scalability bottleneck in memory-augmented neural networks (MANNs), building upon recent work like the Neural Turing Machine (NTM). Whereas NTMs perform dense read and write operations across the entire memory at each timestep, incurring prohibitive computational and memory costs for large memories, this work introduces Sparse Access Memory (SAM). The main contribution is a novel memory access scheme that constrains reads and writes to a small, fixed-size subset of memory words. This is achieved through an efficient, sub-linear time content-based read using approximate nearest neighbor (ANN) data structures and a simple, effective write scheme targeting recently-read or least-used memory slots.

The paper’s strengths are numerous. The core idea is elegant, novel, and highly significant, directly tackling a well-known barrier to the practical application of MANNs. The authors provide strong support for their claims through both theoretical justification of the method’s asymptotic optimality and extensive empirical validation. The reported performance gains are outstanding, with experiments demonstrating orders-of-magnitude improvements in speed and memory efficiency. The quality of the evaluation is high, covering standard NTM tasks (copy, associative recall), real-world data (Omniglot), and impressive curriculum-based scaling, all while maintaining or even improving learning efficiency compared to dense models. The paper is well-written, its assumptions are clear, and architectural details are provided for reproducibility.

The primary weakness is a lack of deeper analysis into some of the paper’s more surprising findings. The authors note that sparse access can benefit early-stage learning on tasks like associative recall (Section 4.2). This is a fascinating result, but the paper would be strengthened by a more detailed discussion exploring *why* this might be the case. Does this sparsity act as a helpful learning bias or a form of regularization? While the chosen ANN integrations are effective, a brief exploration of other structures could also be beneficial.

In conclusion, this is an excellent and important paper. It presents a high-quality, well-executed solution to a major challenge in the field, significantly advancing the state-of-the-art for scalable memory systems in neural networks. The strengths far outweigh the minor areas for improvement, and the work is likely to be highly influential. I strongly recommend acceptance.
