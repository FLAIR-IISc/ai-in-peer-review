- Addresses scalability bottleneck in MANNs by proposing Sparse Access Memory (SAM) to reduce per-step time/space complexity.
- Core innovation: sparse read attending to K locations using ANN indices (k-d trees/LSH) to drop search from O(N) to O(log N).
- Core innovation: sparse write that updates only read locations or the least-recently-used slot to keep updates localized.
- Core innovation: efficient BPTT by recording sparse modifications and rolling back memory instead of caching full O(N) states.
- Empirical evaluation across algorithmic tasks, scaled curricula, bAbI, and Omniglot demonstrates comparable or better learning performance.
- Massive efficiency gains reported: up to ~1600x faster and ~3000x more memory-efficient than dense baselines.
- Demonstrated scalability: training with millions of memory slots, BPTT over 4000+ steps, and generalization to sequences of ~200,000 steps.
- Technical analysis supports asymptotic optimality; implementation and benchmarks (Figure 1) strongly validate efficiency claims.
- Minor experimental weakness: Omniglot comparison matches physical memory but gives SAM larger logical memory; lacking a controlled logical-size comparison.
- Clarity: paper is well-written and structured; suggestion to add a brief explanation of dense NTM erase/add write matrices for broader accessibility.
- Originality: novel integration of classic ANN data structures into end-to-end differentiable models and a clever memory rollback BPTT technique.
- Significance: high — enables practical use of MANNs at scales previously infeasible, opening real-world applications like long-context QA and language modeling.
- Arguments for acceptance: solves a critical problem, novel and sound method, huge empirical gains without performance loss, excellent presentation.
- Argument against acceptance: single minor experimental caveat on fair logical-memory comparison; does not undermine core contributions.
- Overall recommendation: strong recommendation for acceptance based on technical quality, empirical impact, and significance.
