- Paper tackles scalability bottleneck in memory-augmented neural networks (MANNs) by improving space/time complexity as memory grows.
- Introduces Sparse Access Memory (SAM): restricts reads/writes to sparse subsets of memory locations.
- Uses efficient data structures (approximate nearest neighbor search) for content-based sparse access.
- Authors claim large efficiency gains: ~1000x speedup and ~3000x memory reduction versus dense models.
- Builds on prior work (Neural Turing Machines, Memory Networks) and shows adaptability to the Differentiable Neural Computer (DNC).
- Experiments on synthetic tasks (copy, associative recall, priority sort) and Omniglot show comparable performance with much improved efficiency.
- Maintains gradient flow and efficient BPTT by caching sparse memory modifications.
- Technical contribution viewed as incremental: sparse updates and ANN are established techniques applied to MANNs.
- Core novelty is the engineering combination within SAM rather than a major theoretical advance.
- Arguments for acceptance: practical, scalable improvements demonstrating feasibility for larger datasets and tasks.
- Arguments against acceptance: insufficient originality/technical depth for NIPS; better suited to applied ML or systems venues; reviewer recommends rejection.
