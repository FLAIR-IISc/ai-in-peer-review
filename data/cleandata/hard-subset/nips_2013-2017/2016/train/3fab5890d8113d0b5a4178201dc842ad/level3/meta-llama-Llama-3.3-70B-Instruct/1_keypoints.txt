- Proposes Sparse Access Memory (SAM): a neural memory using approximate nearest-neighbors and arg-top-K to compute sparse attention for large external memories.  
- Main innovation: applying fast ANN and top-K to writeable external memory enabling training on much larger problems.  
- Novelty judged thin relative to prior memory network work, but unique in writeable memory context and worth reporting.  
- Paper is well-written and clearly organized with a good introduction and related work review (NTM, Memory Networks).  
- SAM architecture is thoroughly described: sparse read/write ops, ANN indexes, and efficient BPTT.  
- Time and space complexity analyzed and claimed asymptotically optimal.  
- Strong experimental results: comparable efficiency to dense models on synthetic/natural tasks (one-shot Omniglot, bAbI) and scalability to 100Ks time steps/memories.  
- Missing detailed analysis of the argmax/top-K choice and its impact on performance.  
- Does not sufficiently discuss limitations/trade-offs of using approximate nearest neighbor indexes (accuracy vs. efficiency).  
- Recommends more detailed task descriptions and public release of code for reproducibility.  
- Suggests analyzing failure cases/limitations (including argmax/K and ANN behavior) and removing section 3.6; rephrase Omniglot as not “non-synthetic” or “real-world.”  
- Overall recommendation: accept, with minor revisions to address noted limitations.  
- Pro-accept: novel/efficient architecture, impressive experiments, clear presentation.  
- Con-accept: limited novelty, insufficient analysis of argmax/K choice, and ANN limitations.
