- Introduces Sparse Access Memory (SAM) to scale memory-augmented neural networks by constraining reads/writes to a sparse subset of memory locations.
- Motivation: existing MANNs (NTMs, Memory Networks) cannot scale due to linear cost of smooth read/write and memory duplication in backpropagation.
- SAM uses sparse reads/writes and sparse memory management to drastically reduce compute and memory overhead while preserving representational power.
- Uses approximate nearest neighbor data structures (e.g., LSH, randomized k-d trees) for efficient content-based reads.
- Optimizes backpropagation by tracking sparse memory modifications instead of caching the full memory state.
- Demonstrates large efficiency gains: up to ~1000x faster and ~3000x less memory than dense models.
- Shows maintained or improved task performance on NTM benchmarks, bAbI QA, and one-shot Omniglot recognition.
- Scalability experiments succeed on tasks with 100,000s of time steps, highlighting real-world applicability.
- Generality shown by adapting SAM to a Sparse Differentiable Neural Computer (SDNC) with improved bAbI results.
- Technical soundness: rigorous, well-motivated approach with clear implementation details.
- Originality: novel combination of sparse access and efficient data structures for MANN scaling.
- Clarity: paper is exceptionally well-written and accessible.
- Weakness: limited comparison to other state-of-the-art methods, especially those using memory supervision.
- Weakness: insufficient analysis of hyperparameter choices and sensitivity in the main text.
- Weakness: brief discussion of alternative ANN implementations and their trade-offs beyond k-d trees vs LSH.
- Arguments for acceptance: significant, original contribution with comprehensive experiments and high potential impact.
- Arguments against acceptance: need for more thorough comparisons and deeper hyperparameter sensitivity analysis.
- Overall assessment: strong paper, clever effective solution to MANN scalability, recommended for acceptance.
