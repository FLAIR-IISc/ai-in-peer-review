- Introduces Sparse Access Memory (SAM), a MANN that uses sparse read/write to scale external memory efficiently.
- Uses approximate nearest neighbor search and arg-top-K to limit access, achieving O(log N) read time and O(1) space per time step.
- Empirical results claim SAM runs ~1,000× faster and uses ~3,000× less memory than dense models while matching performance on synthetic tasks, Omniglot, and bAbI.
- Demonstrates strong engineering: rigorous time/space analysis, scalable implementation (100k+ memory words), and efficient backpropagation through time.
- Shows sparse approximations do not degrade learning and sometimes outperform dense variants (e.g., priority sort).
- Practical contribution: effective integration of approximate nearest neighbors and sparse data structures into writeable differentiable memories.
- Novelty is overstated: core techniques (hashing, sparse attention) have prior use; need clearer distinction from related work.
- Authors should clarify whether the main novelty is integrating these techniques with writeable memories or specific implementation details.
- Synthetic task descriptions lack sufficient detail for replicability (curriculum parameters for associative recall/copy tasks).
- Authors should release experiment code and complete task construction details to ensure reproducibility.
- Paper lacks analysis of failure cases and limitations—need discussion of when SAM fails and comparisons to other sparse approaches.
- Non-smooth argmax/K-argmax selections require careful treatment—discuss implications for differentiability and compare to discrete-addressing methods (e.g., RL).
- Omniglot should not be called "non-synthetic" or "real-world"; use more precise language about its benchmark status.
- Section 3.6 appears redundant and should be removed; discussion of SDNC should be better integrated with the SAM framework.
- Overall assessment: strong technical and practical contribution worth acceptance after revisions addressing replicability, novelty clarification, and limitation analysis.
