

This paper introduces Sparse Access Memory (SAM), a memory-augmented neural network (MANN) architecture that enables efficient scaling of external memory systems through sparse read and write operations. Building on Neural Turing Machines (NTMs) and Memory Networks, SAM uses approximate nearest neighbor search and arg-top-K operations to constrain memory access to a small subset of memory words, achieving O(log N) time complexity for reads and O(1) space per time step. The authors demonstrate SAM's effectiveness through extensive experiments, showing it runs 1,000× faster and uses 3,000× less memory than non-sparse models while maintaining comparable performance on synthetic tasks, Omniglot one-shot classification, and Babi question-answering tasks.

The work makes a valuable practical contribution by demonstrating how established computational techniques (approximate nearest neighbors, sparse data structures) can be effectively integrated into differentiable memory architectures. While the core technical approach builds upon prior work (e.g., the use of hashing for scalable lookups in Memory Networks), the authors provide a thorough engineering implementation for writeable memories—a less explored area that significantly advances the field's ability to scale MANNs. The experimental results are compelling, with SAM scaling to memory sizes of 100,000+ words and demonstrating superior performance on large-scale curriculum learning tasks.

The paper excels in technical quality, with rigorous theoretical analysis of time and space complexity and thorough experimental validation across multiple domains. The authors provide strong evidence that the sparse approximation does not degrade learning performance, with SAM even outperforming dense variants on some tasks (e.g., priority sort). The implementation details for efficient backpropagation through time are particularly well-executed, addressing a critical bottleneck in training MANNs.

However, the novelty claim requires refinement. While the specific application to writeable memory systems is less explored, the core techniques (approximate nearest neighbors, sparse attention) have been previously used in related contexts (e.g., Memory Networks with hashing, as noted in Section 3.3). The paper would benefit from a more precise discussion of how SAM differs from prior work that used similar techniques. The authors should clarify whether the key innovation is the specific integration with writeable memories or the particular implementation details.

The paper also has some important weaknesses that need addressing. The task descriptions for synthetic benchmarks are insufficiently detailed to ensure replicability. For instance, the exact parameters for the curriculum learning setup on associative recall and copy tasks are not clearly specified. The authors should provide complete task construction details and release all experiment code to address this concern. Additionally, the paper lacks analysis of failure cases and the method's limitations—under what conditions does SAM fail, and how does it compare to other sparse approaches? A discussion of when sparsity becomes detrimental would strengthen the paper.

The non-smoothness of the argmax/K-argmax operations warrants more careful treatment. The paper uses discrete selection of memory locations, which creates a disconnect with the otherwise differentiable framework. The authors should discuss the implications of this non-smoothness, compare it to fully smooth models, and address how it affects gradient estimation. The claim that "the sparse approximations do not impair learning" would be more convincing with a direct comparison to methods that handle this discreteness (e.g., reinforcement learning approaches for discrete addressing).

The Omniglot section should be revised to avoid calling it "non-synthetic" or "real-world" data; while it's more complex than the synthetic tasks, it's still a controlled benchmark dataset. The authors should use more precise language to describe its relationship to real-world applications.

The paper contains a section (3.6) that appears redundant and should be removed. The discussion of the Sparse Differentiable Neural Computer (SDNC) is valuable but should be integrated more tightly with the main SAM framework.

Overall, this is a strong technical contribution with significant practical implications for scaling memory-augmented networks. The empirical results are impressive, and the engineering insights are valuable for the community. While the novelty is somewhat limited compared to the paper's claims, the practical implementation details and thorough scaling analysis make this an important step toward making MANNs viable for real-world applications. The paper should be accepted with revisions to address the replicability concerns, clarify the novelty, and provide more thorough analysis of limitations and failure cases. The authors have made a valuable contribution that advances the field's ability to scale memory-augmented systems, and with the requested revisions, this paper will be a useful reference for future work in the area.
