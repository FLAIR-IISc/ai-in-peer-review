- Introduces Sparse Access Memory (SAM), a novel sparse read/write scheme for MANNs constraining accesses to a fixed small subset (K=4–8) of memory words.  
- Achieves asymptotically optimal complexity: O(N) initialization, O(1) per-timestep space, and O(log N) time per operation via ANN indexing.  
- Builds on Neural Turing Machine framework but removes prohibitive O(N) per-step overheads that limited prior MANNs to toy tasks.  
- Empirical speed/memory gains: ~1,000× faster training and ~3,000× reduced physical memory compared to dense NTMs.  
- Uses approximate nearest neighbor structures (k-d trees/LSH) to enable sublinear scaling in memory size.  
- Maintains near-identical data efficiency and task performance on standard MANN benchmarks (copy, associative recall, priority sort).  
- Outperforms dense models on associative recall and priority sort, and scales to >200,000-step sequences with 100k+ memory slots.  
- Sparse Differentiable Neural Computer (SDNC) variant attains state-of-the-art bAbI results without supervised memory access.  
- Strong theoretical backing: matches asymptotic lower bounds and supports efficient backpropagation by tracking sparse memory modifications.  
- Clear architecture description and implementation details, with reproducible Torch7 code and hyperparameters in the appendix.  
- Key strengths: simplicity, generality, avoidance of complex RL-based addressing, and practical impact moving MANNs beyond toy problems.  
- Minor weaknesses: limited analysis of why sparsity improves learning on certain tasks (e.g., associative recall).  
- Future/limitation note: evaluation limited to classic ANN methods (k-d tree/LSH); newer ANN techniques could further optimize performance.  
- Overall assessment: technically sound, well-presented, high significance for scaling MANNs to real-world applications; strong candidate for acceptance.
