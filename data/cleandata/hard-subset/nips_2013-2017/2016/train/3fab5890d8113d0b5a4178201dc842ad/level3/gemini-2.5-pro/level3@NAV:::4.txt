This paper addresses a critical and well-known bottleneck in Memory-Augmented Neural Networks (MANNs): their poor scalability with respect to memory size. Building on prior work like the Neural Turing Machine (NTM) and Differentiable Neural Computer (DNC), which use dense attention mechanisms that access the entire memory at each timestep, the authors introduce Sparse Access Memory (SAM). The core idea is to replace the computationally expensive dense read/write operations with sparse ones that touch only a constant number of memory slots. This is achieved by integrating an efficient Approximate Nearest Neighbor (ANN) index for content-based reads and a clever write scheme that targets previously read or least-recently-used locations, reducing the time complexity from linear to sub-linear (logarithmic) with memory size.

**Strengths:**
- **Significance & Impact:** The problem of scaling memory systems for neural networks is of high importance. By making it feasible to train models with massive external memories (e.g., millions of words), this work opens up new possibilities for long-form question answering, document modeling, and lifelong learning. The approach demonstrably advances the state of the art in a practical and significant way.
- **Originality & Quality:** The paper proposes several original and technically strong concepts. The main novelty lies in creating a fully end-to-end differentiable memory access scheme that is sparse by design. The technique for efficient backpropagation, which reverts sparse memory modifications rather than caching the entire memory at each step, is particularly clever and effective. Claims are rigorously supported by extensive experiments, including speed benchmarks showing orders-of-magnitude improvements and learning tasks demonstrating that sparsity does not impair—and sometimes even helps—performance.
- **Clarity:** The paper is exceptionally well-written. The problem is clearly motivated, the proposed architecture is explained concisely, and the results are presented effectively. The authors provide enough detail for an expert to understand the core mechanics and appreciate the contributions.

**Weaknesses:**
The paper is a very complete and well-executed piece of work, and it is difficult to identify significant weaknesses. One minor point could be a deeper discussion on the potential trade-offs and failure modes of different ANN structures within the learning loop, as these indices were not originally designed for a context where the underlying data distribution shifts during training.

**Recommendation:**
This paper presents a clear, technically sound, and highly impactful solution to a major challenge in the field. It is a novel contribution that other researchers will likely build upon. I strongly recommend acceptance.
