- Introduces Sparse Access Memory (SAM), scaling MANNs by restricting read/write to sparse memory subsets.  
- Achieves asymptotic optimality: O(1) space per timestep and O(log N) time, enabling large-scale memory use.  
- Reports substantial empirical gains: ~1,000× speedup and ~3,000× reduced physical memory vs. dense models.  
- Maintains near-state-of-the-art performance on synthetic tasks, Babi QA, and Omniglot one-shot learning.  
- Builds on NTMs and DNCs, addressing scalability via efficient sparse operations and approximate nearest-neighbor indexing.  
- Strength: strong technical contribution with rigorous complexity analysis and convincing experimental validation.  
- Strength: paper is clear, well-structured, with thorough ablation studies showing comparable data efficiency to dense models.  
- Strength: SDNC variant achieves best reported Babi results without memory access supervision, showing practical impact.  
- Weakness: background on MANNs (NTM, Memory Networks, DNC) is insufficient for non-expert readers; add a brief overview.  
- Weakness: Figure 4’s physical memory comparison is misleading; logical memory capacity is the more relevant scalability metric and should be clarified.  
- Overall: significant, practical advance that overcomes a major MANN bottleneck; technically sound and highly relevant, pending minor clarifications.
