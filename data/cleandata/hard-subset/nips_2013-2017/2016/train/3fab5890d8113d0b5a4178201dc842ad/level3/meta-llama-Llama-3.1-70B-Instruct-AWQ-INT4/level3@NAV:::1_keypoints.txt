- Introduces Sparse Access Memory (SAM), a neural memory architecture using sparse reads/writes to scale memory-augmented neural networks.  
- Demonstrates SAM on algorithmic tasks, question answering, and one-shot learning (Omniglot).  
- Achieves significant speedups and memory reductions versus traditional MANNs, enabling large-scale training.  
- Proposes an efficient backpropagation scheme that lowers space complexity for longer sequences and larger memories.  
- Obtains comparable or better empirical performance than baseline MANNs across tasks.  
- Novelty is limited: sparse read/write ideas overlap with prior work (e.g., Memory Networks).  
- Paper lacks in-depth analysis of failure cases and limits of the proposed method.  
- Potential non-smoothness from argmax/K-argmax operations is not sufficiently discussed.  
- Authors should provide detailed task construction and evaluation procedures.  
- Release experiment code to enable replication and follow-up work.  
- Analyze and report failure cases and practical limits of SAM.  
- Discuss non-smoothness issues and possible mitigations; consider removing unnecessary Section 3.6 for clarity.  
- Recommendation: accept with revisions and clarifications addressing the above weaknesses.
