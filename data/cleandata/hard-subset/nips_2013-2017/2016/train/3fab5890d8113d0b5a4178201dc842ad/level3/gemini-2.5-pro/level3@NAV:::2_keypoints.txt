- Problem: MANNs (e.g., NTMs, Memory Networks) scale poorly because computational and memory costs grow linearly with external memory size.  
- Proposed solution: Sparse Access Memory (SAM) constrains reads/writes to a small, constant-sized subset of memory slots.  
- Read mechanism: Uses an approximate nearest neighbor (ANN) index to find relevant memory locations in logarithmic time.  
- Write mechanism: Writes target recently accessed or least-used slots to keep updates sparse and efficient.  
- Claims: Achieves strong asymptotic improvements and massive empirical gains (up to 1000× speed, 3000× memory efficiency).  
- Experiments: Thorough evaluation on algorithmic tasks, QA, and one-shot learning; curriculum learning enables scaling to sequences of hundreds of thousands.  
- Performance: Sparse approximation does not harm learning and can sometimes improve it.  
- Clarity: Paper is clear, well-motivated, and describes the architecture in sufficient detail.  
- Weakness — novelty: Core contributions are mainly combinations of existing techniques rather than new fundamental concepts.  
- Two central technical components: ANN-based approximate attention and a sparse backpropagation/update tracking scheme.  
- Assessment: Strong systems/engineering contribution but lacks the conceptual/theoretical leap expected for a top-tier venue.  
- Recommendation: Empirically impressive and practically significant, but reviewer does not recommend acceptance due to insufficient technical novelty.
