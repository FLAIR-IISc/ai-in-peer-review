

Review for "Scaling Memory-Augmented Neural Networks with Sparse Reads and Writes"

This paper introduces Sparse Access Memory (SAM), a novel memory access scheme for memory-augmented neural networks (MANNs) that addresses the critical scalability limitations of existing approaches like Neural Turing Machines (NTMs) and Memory Networks. The authors identify that standard MANNs suffer from linear computational and memory overhead with respect to memory size, making them impractical for real-world applications requiring large memory capacities. SAM solves this by constraining memory reads and writes to a fixed-size sparse subset of memory words, rather than operating over the entire memory. The paper provides theoretical justification for SAM's asymptotic optimality in time and space complexity, and demonstrates empirically that SAM achieves orders-of-magnitude improvements in computational efficiency while maintaining comparable learning performance to existing MANNs.

The work builds on foundational MANN approaches including the Neural Turing Machine [7] and Memory Networks [21], which decoupled memory capacity from model parameters but still suffered from linear complexity with memory size. SAM represents a significant step toward making MANNs practical for real-world applications by addressing this fundamental scalability limitation.

Quality: The paper is technically sound with strong theoretical support. The authors provide a clear theoretical justification for SAM's optimality (Supplementary A) and demonstrate substantial empirical improvements in Section 4.1 (1,000× faster and 3,000× less memory for large memories). The evaluation is comprehensive, covering synthetic tasks (copy, associative recall, priority sort), Babi question-answering tasks, and Omniglot one-shot classification. Results consistently show SAM achieves comparable or better learning performance than dense alternatives while dramatically improving computational efficiency. The authors carefully analyze when sparsity benefits learning (Section 4.2), acknowledging that this varies across tasks.

Clarity: The paper is exceptionally well-written and organized. The authors clearly introduce the problem, motivate their approach, and systematically present methodology, results, and analysis. Mathematical notation is consistent and well-explained. The paper flows logically, with each section building on the previous one, and key contributions are well-highlighted. Supplementary materials provide thorough implementation details and theoretical analysis, making the work highly reproducible. Figures effectively illustrate key points.

Originality: The core idea of sparse memory access for MANNs is novel and addresses a significant gap in the literature. While prior work has explored sparse memory access (e.g., reinforcement learning approaches [22]), SAM's end-to-end differentiable approach with theoretical optimality guarantees represents a distinct contribution. The paper clearly differentiates SAM from previous work and provides thorough coverage of related NIPS papers. The authors have appropriately cited relevant prior work including recent NIPS contributions.

Significance: This work addresses a critical bottleneck in MANN applications. By demonstrating that MANNs can scale to memory sizes of 100,000s with orders-of-magnitude computational improvements, SAM makes these models practical for real-world applications like language modeling and machine translation. The results on Babi tasks and Omniglot show SAM can learn meaningful representations and generalize effectively. The scalability enabled by SAM opens new possibilities for MANNs in applications requiring large memory capacities, with the authors' claim of asymptotic optimality representing a significant theoretical contribution.

Strengths: The paper's core idea is novel and well-executed. The theoretical justification is strong, and the empirical results are compelling. The implementation details are thorough, and the paper is exceptionally well-written with clear organization. The authors demonstrate scalability to tasks requiring 100,000s of time steps and memories, a significant advancement over prior work.

Weaknesses: While the paper notes that sparsity benefits learning for some tasks (e.g., priority sort), it could provide deeper analysis of when and why this occurs. The authors briefly mention alternative ANN implementations (k-d trees vs. LSH) but could explore how different choices impact performance more thoroughly. The theoretical analysis in Supplementary A could be strengthened with more detailed proofs of the optimality claims.

Pro Arguments for Acceptance: Solves a critical scalability bottleneck in MANNs; provides theoretical justification for asymptotic optimality; demonstrates orders-of-magnitude improvements in efficiency; comprehensive evaluation across multiple tasks; clear writing and well-organized structure.

Con Arguments Against Acceptance: Limited analysis of when sparsity improves learning; some theoretical claims could be more thoroughly supported; alternative ANN implementations could be explored more deeply.

This paper makes a significant contribution to the field of memory-augmented neural networks by providing a scalable, efficient, and theoretically justified memory access scheme. The results are compelling, the writing is excellent, and the work addresses a critical practical limitation of existing MANNs. I recommend acceptance.
