- Introduces Sparse Access Memory (SAM), a MANN that uses sparse reads/writes (K=4–8) and ANN-based content addressing to achieve asymptotic time/space optimality.  
- SAM reports up to 1,000× runtime speedup and 3,000× less physical memory than dense MANNs while matching performance on synthetic tasks, Babi, and Omniglot.  
- Core innovations: sparse content-based addressing via ANN, sparse write using least-recently-accessed slots, and efficient sparse backpropagation enabling 100k+ memory words.  
- Theoretical guarantees provided (asymptotic optimality proofs in Supplementary A) and empirical validation across multiple tasks.  
- Substantially departs from prior dense MANNs and outperforms reinforcement-learning-based sparse NTMs in scalability without degrading performance.  
- Significance: removes a practical scalability barrier, enabling training with 1M+ words and applications like long-document processing.  
- Presentation is clear and well-structured, with effective figures and thorough empirical methodology.  
- Weakness: Omniglot (Fig. 4) comparison is misleading—physical memory normalized instead of logical memory, conflating sparsity benefits with sheer memory size.  
- Weakness: Background section lacks a brief explanation of NTM memory write operator (erase/add), reducing accessibility for non-MANN readers.  
- Pro acceptance points: solves known scalability bottleneck, combines theoretical and empirical strength, demonstrates real-task feasibility, and provides reproducible implementation details.  
- Con acceptance points: figure comparison misrepresents advantage and minor background clarity gap.  
- Overall assessment: high-impact, technically sound contribution to scalable MANNs; recommendation: Accept.
