- Introduces Sparse Access Memory (SAM), a memory access scheme restricting reads/writes to a fixed-size subset of memory words.  
- Builds on NTMs and Memory Networks, using sparse access plus approximate nearest neighbor (ANN) data structures for efficiency.  
- Theoretical analysis shows SAM achieves O(log N) read time and O(1) space per time step, giving asymptotic improvements over dense MANNs.  
- Empirical results report up to 1,000× speedup and 3,000× memory reduction versus NTMs while maintaining comparable task performance.  
- Demonstrates effectiveness on standard MANN tasks (copy, associative recall, priority sort) and real-world tasks (Omniglot classification, Babi reasoning).  
- Authors acknowledge both strengths (computational gains) and potential weaknesses (learning costs in some scenarios), noting sparsity can sometimes improve learning.  
- Paper is well-written, well-organized, and provides thorough technical details and implementation (CSR formats, ANN indexing) in the appendix.  
- Experimental design is robust, with careful comparisons to dense variants (DAM) and baselines (NTM, LSTM).  
- Main limitation: insufficient systematic analysis of when sparsity helps versus hinders learning across different task types.  
- Suggestion to explore other sparse memory management strategies beyond ANN to broaden applicability.  
- Overall assessment: original, technically sound, significant practical and theoretical contribution; recommend acceptance with minor revisions to address sparsity effects.
