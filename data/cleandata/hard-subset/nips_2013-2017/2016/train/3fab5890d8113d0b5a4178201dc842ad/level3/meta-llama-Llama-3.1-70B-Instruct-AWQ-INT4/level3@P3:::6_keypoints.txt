- Proposes Sparse Access Memory (SAM), a novel neural memory architecture using sparse read/write to improve scalability.  
- SAM aims to reduce huge physical memory and backpropagation costs that limit MANNs.  
- Authors claim SAM achieves comparable performance to state-of-the-art models while greatly reducing memory and computation.  
- Paper is well-structured, clearly written, and accessible to a broad audience.  
- Provides clear background on MANNs (NTMs, Memory Networks) and their scalability limitations.  
- SAM’s sparse read/write design reduces computational and memory overhead in practice.  
- Experiments (algorithmic tasks, bAbI, Omniglot one-shot) show significant speedups and memory reductions with similar performance.  
- Incorporates approximate nearest neighbor (ANN) indexes to make the forward pass sublinear in memory size.  
- Discusses trade-offs between different indexing approaches but leaves room for improvement and deeper analysis.  
- Concern: Figure 4 comparison may be unfair because it matches physical rather than logical memory sizes.  
- Paper is technically sound with claims supported by theory and experiments.  
- The approach is original and a meaningful contribution to scaling MANNs.  
- Significance: approach is general and applicable to many differentiable memory architectures.  
- Arguments for acceptance: novelty, strong empirical/theoretical results, clear presentation, broad applicability.  
- Arguments against acceptance: potential unfair comparison in Figure 4, insufficient detail on indexing trade-offs, lack of computational resource reporting.  
- Recommendation: accept, but authors should address the noted concerns to strengthen the work.
