- Addresses scalability bottleneck in memory-augmented neural networks (NTMs) by avoiding dense read/write across entire memory.  
- Proposes Sparse Access Memory (SAM) that constrains reads/writes to a small, fixed subset of memory words.  
- Implements sub-linear content-based reads using approximate nearest neighbor (ANN) data structures.  
- Uses a simple, effective write scheme targeting recently-read or least-used memory slots.  
- Provides theoretical justification for asymptotic optimality.  
- Presents extensive empirical validation showing orders-of-magnitude improvements in speed and memory while maintaining or improving learning efficiency.  
- Evaluation covers standard NTM tasks (copy, associative recall), real-world data (Omniglot), and curriculum-based scaling; paper is well-written and reproducible.  
- Main weakness: lacks deeper analysis of surprising findings (e.g., why sparsity benefits early-stage learning—bias vs. regularization).  
- Minor suggestion: brief exploration of alternative ANN structures could strengthen the work.  
- Overall: excellent, important, likely influential paper; recommend acceptance.
