- Proposes Sparse Access Memory (SAM) that replaces dense soft attention with sparse top-K access using ANN for reads and recent/LRU writes to scale memory-augmented networks.  
- Introduces an efficient backpropagation scheme that tracks sparse memory changes to avoid caching full memory during BPTT.  
- Demonstrates massive empirical speed and memory gains (up to 1000x), enabling training with millions of memory slots and sequences >100k steps.  
- Shows sparse approximation does not harm—and can improve—learning on several tasks, and scales with curriculum training.  
- Extends the idea to a Sparse Differentiable Neural Computer (SDNC) achieving state-of-the-art on bAbI QA tasks.  
- Significance: addresses a major bottleneck in memory-augmented models, enabling practical use on large real-world knowledge/storage tasks.  
- Quality: technically sound with strong empirical evaluation and relevant baselines (NTM, DAM) compared.  
- Practical contribution: combines ANN data structures with an engineering BPTT solution as a blueprint for scalable memory systems.  
- Originality concern: conceptual novelty limited—similar ideas (hashing/NN search) exist in prior Memory Networks; novelty mainly in integrating these into read-write NTM-style memory.  
- Reproducibility concern: task descriptions, curriculum details, and hyperparameters are insufficient—more appendix detail or released code is requested.  
- Sparse-approximation analysis lacking: needs discussion of non-smooth top-K selection, gradient handling (e.g., straight-through), effects at extreme K (K=1), and comparisons to prior hashing methods.  
- Failure-mode analysis missing: authors should investigate tasks or conditions where sparse approximation degrades relative to dense models.  
- Comparison to RL-based discrete addressing: discuss trade-offs between the sparse differentiable approximation and reinforcement-learning approaches for discrete memory access.  
- Minor wording: calling Omniglot "real world" is overstated—suggest rephrase to “a non-synthetic dataset of real characters.”  
- Overall recommendation: strong, practically impactful paper recommended for acceptance, contingent on addressing reproducibility and deeper analysis of approximations.
