- Proposes Sparse Access Memory (SAM): a sparse read/write neural memory using efficient data structures (approximate nearest neighbor indexes) for fast, scalable memory access.  
- Strong motivation and clear exposition of limitations of traditional MANNs and advantages of the proposed approach.  
- Technical contributions are sound with thorough time and space complexity analysis.  
- Experiments show impressive speedups and memory reductions compared to traditional MANNs.  
- Novelty is somewhat limited because related work (e.g., Memory Networks) already uses hashing for scalable lookups.  
- Practical contribution: applying scalable lookup tools to writable memories is less explored and valuable.  
- Reviewer recommends acceptance contingent on revisions.  
- Requests improved task descriptions, detailed task construction, and release of experiment code for reproducibility.  
- Asks for analysis of failure cases and limits—when and under what conditions the method fails.  
- Raises concern about non-smoothness of argmax/K-argmax (especially K=1); requests discussion and comparison to fully smooth models and MemNN-WSH.  
- Requests discussion of implications/limits of training with discrete actions without explicitly accounting for discreteness.  
- Suggests removing section 3.6 and not labeling Omniglot as “non-synthetic” or “real-world.”  
- Overall: significant contribution with potential for strong impact after requested revisions.  
- Rating: 8/10; Confidence: 8/10; Recommendation: Accept with revisions.
