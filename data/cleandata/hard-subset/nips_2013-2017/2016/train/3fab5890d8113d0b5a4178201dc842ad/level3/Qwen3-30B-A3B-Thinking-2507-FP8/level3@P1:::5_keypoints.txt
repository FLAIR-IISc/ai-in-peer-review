- Introduces Sparse Access Memory (SAM), a memory architecture achieving O(log N) per-step complexity and O(N) initialization.  
- Uses sparse reads via approximate nearest neighbors and sparse writes via least-recently-accessed slot selection to avoid linear overheads.  
- Demonstrates massive efficiency gains: ~1,000× speedup and ~3,000× memory reduction over dense models.  
- Maintains near-identical data efficiency on synthetic benchmarks (Copy, Associative Recall, Priority Sort) and Omniglot one-shot classification.  
- Scales to very long sequences (up to 200,000 steps), far beyond prior MANNs.  
- Achieves state-of-the-art bAbI reasoning performance without supervised memory access, outperforming dense variants and prior MANNs.  
- Strong theoretical foundation proving asymptotic optimality (Supplementary A).  
- Well-designed sparse write mechanism and efficient backpropagation validated by experiments.  
- Effective memory management (usage tracking) that avoids simpler sparse-approach pitfalls.  
- Clear, well-organized paper with sufficient detail for reproducibility and a solid dense baseline (DAM).  
- Limited real-world validation—lacks experiments on large-scale natural language tasks (e.g., MT, document QA).  
- Underexplored why sparsity improves long-sequence generalization; needs memory-content analysis and write-strategy ablations.  
- ANN implementation trade-offs (k-d trees vs. LSH, ANN accuracy vs. speed) are pragmatic but not thoroughly evaluated.  
- Highly original and significant contribution that makes scalable, end-to-end differentiable MANNs practical; recommendation: Accept.
