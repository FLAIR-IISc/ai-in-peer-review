- Introduces Sparse Access Memory (SAM), a memory-augmented neural network that constrains reads/writes to a sparse subset using ANN indexing.  
- Claims asymptotic optimality: O(log N) time for access, O(N) initialization, and O(1) per-step memory overhead.  
- Demonstrates massive empirical gains: up to 1,600× faster and 3,000× less physical memory than vanilla NTMs on large memories (e.g., 64K words).  
- Matches or exceeds dense MANNs on synthetic tasks (Copy, Associative Recall, Priority Sort), Babi QA, and Omniglot one-shot learning.  
- Scales to very long sequences (100,000+ time steps) without truncation and generalizes to sequences 20× longer than training.  
- Introduces Sparse Differentiable Neural Computer (SDNC) variant that achieves state-of-the-art Babi results without supervised memory access.  
- Provides strong theoretical justification for SAM’s complexity bounds (Supplementary A) and supports claims with thorough experiments.  
- Paper is clear and well-structured, with effective figures and explanations of sparse read/write and ANN integration.  
- Original contribution: novel sparse access scheme with ANN indexing that preserves end-to-end differentiability and avoids high-variance RL-based addressing.  
- Demonstrates practical significance by enabling MANNs to handle real-world scale problems previously deemed impractical.  
- Minor clarity gap: the memory write operator (Eq 3) and its sparse variant (Eq 5) need a brief, self-contained explanation for non-NTM readers.  
- Figure 4 comparison nuance: using identical physical memory across models conflates capacity with efficiency; a logical-memory-fixed comparison would better isolate scalability benefits.  
- Overall verdict: technically sound, significant scalability breakthrough with strong empirical validation; recommended for acceptance with minor clarifications.
