- Proposes Sparse Access Memory (SAM), a neural memory architecture using sparse read/write and approximate nearest-neighbors for efficient large external memories.  
- Well-motivated with clear explanation of limitations in prior MANNs (NTMs, Memory Networks).  
- Detailed description of SAM’s sparse read/write operations, controller, and efficient backpropagation through time.  
- Thorough time and space complexity analysis showing asymptotic optimality.  
- Strong experimental results: significant speedups and memory reductions versus dense MANNs while maintaining comparable performance.  
- Demonstrates scalability on tasks with large memories and long sequences.  
- Paper is well-written and provides sufficient detail to enable replication.  
- Task descriptions and experiment setup are insufficiently detailed and need expansion.  
- Lacks analysis of failure cases and limits—authors should discuss when SAM fails or degrades.  
- Novelty is somewhat limited because related work (e.g., Memory Networks with hashing) already addresses scalable lookups.  
- Practical contribution is valuable: applying scalable lookup tools to writable memories is less explored and useful.  
- Reviewer recommends acceptance conditional on revisions and clarifications.  
- Specific requested revisions: add task construction details, analyze failure modes, discuss implications of discrete actions, remove Section 3.6, and stop calling Omniglot “non-synthetic” or “real-world.”  
- Overall assessment: technically sound, significant contribution to MANNs, likely to be useful to the community despite some originality concerns.
