- Proposes Sparse Access Memory (SAM): constrain reads/writes to a constant K memory words and use ANN search to select top-K, with sparse writes to recently read or LRU slots.  
- Solves the MANN scaling bottleneck where dense reads/writes incur O(N) time and memory per timestep.  
- Uses Approximate Nearest Neighbor structures (e.g., k-d trees, LSH) to find K nearest memory words in sublinear O(log N) time.  
- Implements sparse backpropagation that tracks only sparse memory modifications, achieving asymptotically O(log N) time and O(1) space per timestep.  
- Strong empirical speed/memory gains (e.g., ~1600x faster, ~3000x less memory for large N) validating theoretical claims.  
- Efficiency does not degrade learning; SAM matches or improves data efficiency on standard NTM tasks.  
- Curriculum scaling experiments show handling of sequence lengths in the hundreds of thousands, impossible for dense MANNs.  
- Demonstrates generality with tasks beyond algorithms: bAbI question answering and one-shot Omniglot classification.  
- Novelty: first to integrate end-to-end differentiable sparse memory access with ANN data structures instead of RL-based discrete addressing.  
- Solid technical backing: theoretical analysis in appendix plus extensive empirical validation.  
- Presentation strength: paper is well-written, clearly organized, and architecture/complexity details are thorough.  
- Weakness: limited analysis of why sparsity sometimes improves learning (needs deeper hypotheses or experiments).  
- Weakness: insufficient discussion of practical engineering trade-offs for ANN integration (e.g., index imbalance, rebuild frequency, different ANN methods).  
- Acceptance rationale: addresses a major problem with a novel, sound approach, strong evidence, high impact, and high-quality presentation.  
- Main counterpoint: only minor—lack of deeper analysis of surprising empirical phenomena; not a fatal flaw.
