- Introduces Sparse Access Memory (SAM), a sparse memory access scheme for memory-augmented neural networks (MANNs).
- Motivates SAM by citing poor scalability of traditional MANNs (NTMs, Memory Networks) due to dense read/write operations with linear overhead.
- Proposes sparse read/write and efficient data structures to achieve better time and space asymptotics while retaining representational power.
- Connects to prior MANN work (NTMs, Memory Networks) and argues broader applicability to real-world domains.
- Claims SAM achieves asymptotic lower bounds in space and time complexity compared to dense approaches.
- Main technical components: efficient backpropagation by tracking sparse memory modifications and approximate nearest-neighbor search for sparse reads.
- Approximate NN search implemented with standard data structures (e.g., k-d trees, locality-sensitive hashing).
- Reviewer judges the technical components straightforward and not particularly novel.
- Experiments compare SAM to dense variant (DAM), NTMs, LSTMs on synthetic tasks, one-shot Omniglot, and bAbI question answering.
- Results show SAM matches performance of dense models while being faster and more memory-efficient.
- Reviewer feels experimental results are interesting but do not compensate for limited technical novelty.
- Suggests more in-depth analysis and comparison with related work (e.g., Differentiable Neural Computer) is needed.
- Paper quality: well-written, clear architecture description, and well-executed experiments.
- Clarity issues: some notation and terminology are unclear, particularly variable/function usage.
- Originality: presents a new memory access scheme but core techniques are not seen as novel.
- Significance: demonstrates potential scalability gains, but reviewer views the overall impact as limited.
- Arguments for acceptance: new scheme for MANNs, promising scalability results, and good writing.
- Arguments against acceptance: limited novelty, insufficient comparative analysis, and exaggerated claims.
- Overall verdict: solid contribution but not sufficiently novel or impactful; recommends toning down claims and adding deeper analysis/comparisons.
