- Problem: Standard MANNs use dense soft attention, causing O(N) per-step time and O(NT) BPTT space that prevents large-memory / long-sequence training.  
- Proposal: Sparse Access Memory (SAM) restricts reads/writes to a constant K memory slots to reduce cost.  
- Read mechanism: Uses ANN structures (k-d trees, LSH) to retrieve K nearest memory words in O(log N) time instead of O(N).  
- Write mechanism: Sparse writes directed to previously read slots plus the least-recently-used slot.  
- BPTT innovation: Efficiently reverses sparse updates to avoid caching full memory each step, reducing space from O(NT) to O(N + T).  
- Empirical results: Massive speedups (up to 1600x) and memory reductions (up to 3000x) while matching or improving performance on algorithmic tasks, bAbI, and Omniglot one-shot.  
- Relation to prior work: Builds on hashing/ANN ideas from Memory Networks; novelty is in integrating and engineering these for a full read/write differentiable memory system.  
- Strength — significance: Solves a major practical bottleneck, enabling MANNs with millions of slots and very long sequences.  
- Strength — quality: Thorough, convincing experiments and strong curriculum scaling demonstrations.  
- Strength — clarity: Well-written with helpful diagrams and complexity analysis.  
- Weakness — originality: Conceptual novelty limited; paper should better position itself relative to prior hashing/ANN work.  
- Weakness — sparsity discussion: Replacing smooth softmax with hard top-K via ANN is discrete and non-smooth; needs deeper theoretical discussion and comparisons to discrete-action approaches.  
- Weakness — reproducibility: Experimental details (curriculum schedules, hyperparameters) are too high-level; code release recommended.  
- Weakness — limitations analysis: Missing critical discussion of failure modes (ANN retrieval failures, top-K misses, limits of LRU writes).  
- Minor wording: Calling Omniglot “real world data” is misleading; the task is a synthetic benchmark using real images.  
- Recommendation: Accept — practical impact and empirical strength outweigh concerns, but revision should address discussed issues.
