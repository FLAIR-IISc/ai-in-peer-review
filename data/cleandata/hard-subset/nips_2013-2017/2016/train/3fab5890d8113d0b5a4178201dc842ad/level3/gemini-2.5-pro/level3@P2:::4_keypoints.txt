- Problem: Dense soft read/write in MANNs scales linearly with memory size N, making large-memory tasks impractical.
- Proposal: Sparse Access Memory (SAM) enforces sparsity in memory operations while retaining end-to-end differentiability.
- Sparse reads: Restricts reads to a constant K nearest memory locations found via ANN data structures (e.g., kd-tree, LSH) in O(log N) time.
- Sparse writes: Writes target previously read locations or the least-recently-used slot to maintain sparsity.
- Efficient BPTT: “Memory rollback” stores only sparse O(K) modifications per timestep, reducing per-timestep space from O(N) to O(1).
- Theoretical soundness: Asymptotic efficiency claims are well-justified and supported by analysis.
- Empirical performance: Demonstrates orders-of-magnitude gains (e.g., 1,000x faster and 3,000x less memory for 1M words) and scales to sequences/memories far beyond dense models.
- Experimental rigor: Thorough comparisons to NTM and a dense variant (DAM) show comparable or better data efficiency and robust results on long-range tasks.
- Clarity: Paper is exceptionally well-written, well-illustrated, and provides sufficient detail for experts.
- Originality: Novel contributions include enforcing sparse differentiable addressing, integrating ANN as a non-differentiable search component, and the memory rollback BPTT mechanism.
- Significance: High impact—breaks a major scalability bottleneck and enables new applications for large external memories.
- Arguments for acceptance: Strong impact, novelty, empirical performance, effectiveness on hard tasks, and clear presentation.
- Main weakness: Sensitivity to hyperparameter K is underexplored; an ablation study would clarify trade-offs.
