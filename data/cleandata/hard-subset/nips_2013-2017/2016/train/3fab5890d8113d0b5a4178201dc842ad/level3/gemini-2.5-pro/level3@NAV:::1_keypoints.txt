- Proposes Sparse Access Memory (SAM) replacing dense attention with a sparse alternative to scale MANNs.  
- Uses fast approximate nearest-neighbor (ANN) lookups to read top-K memory slots and sparse writes to recent/least-used locations.  
- Enables training with millions of memory words and sequences hundreds of thousands of steps long, yielding orders-of-magnitude speed and memory improvements.  
- Empirical results show the sparse approximation does not degrade learning and can be beneficial.  
- Conceptual novelty is moderate since hashing/ANN for scalable lookups has prior work (e.g., Memory Networks); related work discussion should be expanded.  
- Practical/engineering contribution is significant: applying ANN-based retrieval to dynamic, writeable memory is non-trivial.  
- Replicability concern: task descriptions lack detail; authors should thoroughly document task construction and release code.  
- Limitation analysis needed: examine failure modes and when sparse approximation breaks down (e.g., small K, specific tasks or hyperparameters).  
- Non-smoothness concern: argmax/top-K discreteness and implications for backpropagation should be discussed (why standard backprop works without RL).  
- Clarity issue: avoid imprecise characterization of Omniglot as "non-synthetic" or "real-world."  
- Recommendation: accept contingent on addressing reproducibility and providing a more critical analysis of limitations.
