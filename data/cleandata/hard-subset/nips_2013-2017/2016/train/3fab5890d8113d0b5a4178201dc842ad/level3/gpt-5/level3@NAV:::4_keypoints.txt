- Introduces Sparse Access Memory (SAM): a differentiable memory-augmented neural network with sparse reads/writes that retains NTM representational power.  
- Core ideas: top-K content-based reads via an in-network ANN index (randomized k-d trees or LSH), sparse writes to previously read locations and an LRU slot, and reversible memory updates for O(1) BPTT space.  
- Claims asymptotically optimal time/space for content-based access relative to prior work (NTM, Memory Networks, DNC, RL-based addressing).  
- Empirical results show up to 1000× speedups and 3000× memory savings, scaling to millions of slots and >100k steps, with strong bAbI and one-shot Omniglot performance.  
- Technical contributions are sound, supported by complexity analysis and careful engineering of reversible BPTT and sparse memory management.  
- Some optimality claims rely on assumptions deferred to the supplement; a brief statement of conditions in the main text would help.  
- Additional ablations (sensitivity to K, ANN recall, write policy) are needed to improve rigor.  
- Presentation is exceptionally clear and concise; problem, algorithms, data structures, and training setup are well explained.  
- Minor clarity suggestion: centralize ANN hyperparameters and rebuild schedules in the main text.  
- Originality is high: novel integration of sparsity and ANN into a fully differentiable MANN with reversible BPTT and extension to DNC.  
- Significance is high: enables efficient scaling of differentiable memory for long-context sequence modeling and algorithmic reasoning.  
- Pros: elegant end-to-end differentiable sparse access, strong asymptotic/practical gains, clear writing, and compelling empirical demonstrations.  
- Cons: limited evaluation on real-world large-scale tasks (e.g., language modeling, MT) and underanalysis of ANN nondifferentiability and hyperparameter sensitivity.  
- Cons (cont.): comparisons may be unfair when memory sizes differ; write policy design choices (prev-read/LRU) lack ablation.  
- Overall recommendation: clear accept.
