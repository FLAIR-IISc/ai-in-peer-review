

This paper introduces Sparse Access Memory (SAM), a novel memory access scheme for memory-augmented neural networks (MANNs) that constrains reads and writes to a fixed-size subset of memory words rather than the entire memory. Building on Neural Turing Machines (NTMs) and Memory Networks, SAM enables efficient training with very large memories by using sparse access patterns combined with approximate nearest neighbor (ANN) data structures. The authors provide theoretical justification for SAM's optimality in space and time complexity, demonstrating asymptotic improvements over dense MANNs.

The paper excels in several areas. The theoretical foundation is sound, with clear analysis of SAM's O(log N) time complexity for reads and O(1) space per time step. Empirical results are compelling: SAM achieves 1,000× speedup and 3,000× memory reduction compared to NTMs while maintaining comparable performance on standard MANN tasks (copy, associative recall, priority sort) and real-world applications like Omniglot classification and Babi reasoning tasks. The authors appropriately address both strengths (significant computational gains) and weaknesses (potential learning cost in some scenarios), showing SAM's sparse access can actually improve learning for certain tasks.

The paper is exceptionally well-written with clear organization and thorough technical details in the appendix. The experimental design is robust, including careful comparisons with dense variants (DAM) and baselines (NTM, LSTM), and the real-world application on Omniglot demonstrates practical utility. The SAM architecture is elegantly simple yet powerful, with efficient implementation using CSR formats and ANN indexing.

The main limitation is that the paper doesn't sufficiently explore *when* sparsity improves learning versus when it might hinder performance. The authors suggest this as a future direction, but a more systematic analysis of sparsity effects across different task types would strengthen the contribution. Additionally, while the ANN approach is effective, exploring other sparse memory management strategies could further broaden applicability.

Overall, this is a high-quality contribution that significantly advances MANNs by enabling scalable memory access. The work is original, technically sound, and demonstrates clear significance for both theoretical understanding and practical applications. The paper meets all NIPS quality standards and represents a meaningful advancement in the field. I recommend acceptance with minor revisions to address the sparsity effects analysis.
