This research paper proposes a new method for training a deep neural network (DNN) (or any other gradient-based approach) using an oracle (which knows which member of the network is best for the current example) on each data instance used for training, and applying backprop only to the network with the selected output. The advantage of this approach is that it allows the network to focus on specific topics, creating a group that well covers all (or most) of the potential outputs of interest. The results have been reported in various fields, with excellent overall performance. I think this is a very good paper. It presents an interesting idea for training a DNN with an oracle, which directs the backprop update to the net with the best output (according to the true label). It seems to apply expertise well in terms of classification. The overall presentation is really good, and the experiments have been conducted in several areas. Overall, this is a strong work that should be published. If the issue of exploiting a DNN trained with the proposed approach had also been addressed, I would have considered this paper excellent. Currently, training is done with an oracle, but the use of the resulting group has not been further developed, except that it can provide a wider pool of potential results, and this approach is consistent with the proposed top-k metric for some problems (i.e., ImageNet). This is a crucial missing piece to make the proposed approach fully useful in practice. Still, there is a contribution in the current stage, the missing piece is relatively problem-specific.
