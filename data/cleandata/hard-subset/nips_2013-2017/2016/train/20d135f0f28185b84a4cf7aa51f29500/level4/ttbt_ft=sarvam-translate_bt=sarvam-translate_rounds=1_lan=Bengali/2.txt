This paper proposes a new approach for training an ensemble of deep neural networks (DNNs) (or any other gradient-based method). It uses an oracle (which knows which member of the ensemble is best for the current instance) on each data instance used for training, and applies the backprop only to the network with the selected output. The advantage of this method is that it allows the networks to create special themes, which results in an ensemble that better covers all (or most) of the possible outputs of interest. The results have been reported with excellent performance in various domains. I think it is a pretty good paper. It presents an interesting idea of training an ensemble of DNNs with an oracle, where the oracle guides the network to the best output (according to the actual label) with the backprop update. It seems to apply specialization quite well in a classification context. The overall presentation is really good, and the experiments have been done in several domains. Globally, it is a strong work that deserves to be published. I would have considered this paper outstanding if the use of the proposed method for training the ensemble of DNNs had also been discussed. Currently, it is trained with an oracle, but the use of the resulting ensemble has not been further developed, only that it can provide a wide pool of possible results, and that this method is consistent with the proposed top-k metrics (e.g., ImageNet) for some problems. This is an important missing part to make the proposed method fully usable. Nevertheless, there is a contribution at the current stage, the missing part is relatively problem-specific.
