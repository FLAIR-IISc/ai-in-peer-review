- Introduces sMCL, a method that trains diverse deep ensembles by minimizing oracle loss via SGD, avoiding costly MCL retraining.  
- Positions the work within ensemble/diversity literature and highlights its link to handling human-like ambiguity.

- Technical innovation: backpropagate gradients only to the lowest-loss predictor per example, yielding a simple, powerful modification.  
- Computationally efficient and practical: reportedly ~5× faster than MCL, loss- and architecture-agnostic, with minimal implementation changes.

- Comprehensive evaluation across classification, segmentation, and captioning using oracle accuracy, IoU, and CIDEr-D, showing consistent 6–10% relative gains.  
- Strong analysis of emergent specialization: ensemble members naturally specialize by class, segment, or n-gram without explicit design.  
- High practical relevance for ambiguous real-world applications (e.g., autonomous vehicles, image captioning).

- Lacks theoretical analysis: no convergence or optimality guarantees for the stochastic approach.  
- Insufficient implementation detail: unclear justification for the 5× speedup and sensitivity to number of meta-iterations.  
- Incomplete comparisons: should include more recent diverse-prediction methods (e.g., Kirillov et al., Batra et al.).  
- Missing discussion of hyperparameter sensitivity (ensemble size, learning rate schedule).

- Overall assessment: strong empirical contribution with insightful analysis; strengths outweigh weaknesses.  
- Recommendation: Accept.

- Pro: solves a real problem, simple and efficient algorithm, consistent multi-domain improvements, insightful specialization analysis, broadly applicable.  
- Con: missing theoretical convergence analysis, some implementation and comparison details need clarification.
