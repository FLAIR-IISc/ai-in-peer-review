- Proposes AutoBDMC: uses Bidirectional Monte Carlo (AIS and rAIS) to assess MCMC convergence via stochastic bounds on log marginal likelihood.
- Shows the gap between AIS and rAIS bounds gives a quantifiable upper bound on the Jensen–Shannon divergence between true posterior and MCMC sample distribution.
- Produces a single scalar diagnostic for MCMC convergence quality.
- Strengths: clear motivation, readable presentation, and practically useful for diagnosing MCMC behavior.
- Experimental validation on synthetic and real-world datasets supports method efficacy and utility for model selection and debugging.
- Section 3 is hard to follow; Section 3.1 is abrupt and lacks sufficient context/motivation for introduced variables and equations.
- If the divergence bound in Section 3.1 is novel, the paper should explicitly state this as a core theoretical contribution.
- Paper never concisely defines “AutoBDMC”; the term should be clearly defined before being used (e.g., prior to Section 3.2).
- Minor notational issues: likely typo p(θ, z | y) should be p(θ, z, y) (line 149) and inconsistent use of y vs D for data.
- Supplemental WebPPL code needs specified compatible WebPPL versions and integration instructions; packaging as an npm WebPPL package is recommended.
- Recommend also releasing the Stan implementation in a user-friendly form to broaden impact.
- Overall assessment: promising and well-written; addressing clarity in Section 3, defining AutoBDMC, and improving code releases would substantially strengthen the paper.
