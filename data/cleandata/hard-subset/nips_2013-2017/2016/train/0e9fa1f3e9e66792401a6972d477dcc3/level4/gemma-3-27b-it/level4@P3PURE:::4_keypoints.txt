- Paper introduces AutoBDMC, a method using Bidirectional Monte Carlo to assess MCMC convergence via AIS-based stochastic bounds on the log marginal likelihood.  
- The gap between upper and lower bounds yields a bound on the Jensen-Shannon divergence between the true posterior and approximate samples, providing a single scalar convergence metric.  
- Method currently only analyzes convergence on synthetic data drawn from the joint, though experiments suggest synthetic analyses predict real-data behavior.  
- Demonstrates practical applications for model selection and debugging.  
- Overall manuscript is readable, well-motivated, and experimentally reasonably compelling.  
- Section 3 is harder to follow; transition into Section 3.1 is abrupt and lacks motivation for introduced variables/equations.  
- If the divergence bound in Section 3.1 is novel, this novelty should be explicitly stated, as it is a core contribution.  
- The paper lacks a precise, early definition of “AutoBDMC”; it should explicitly state that AutoBDMC uses AIS and reverse AIS to analyze MCMC convergence.  
- Notational issues: possible typo at Line 149 (p(θ, z | y) vs p(θ, z, y)) and inconsistent use of y vs D for data, causing confusion.  
- Supplemental WebPPL code needs: (a) the WebPPL versions it works with, and (b) instructions for integrating into a WebPPL distribution.  
- Recommend releasing the WebPPL code as an npm-installable package and, if possible, providing a similar package for the Stan code.
