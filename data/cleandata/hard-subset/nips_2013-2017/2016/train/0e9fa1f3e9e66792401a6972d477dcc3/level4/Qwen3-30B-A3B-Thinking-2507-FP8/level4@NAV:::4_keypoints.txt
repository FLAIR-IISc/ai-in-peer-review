- Introduces AutoBDMC: uses Bidirectional Monte Carlo with AIS/rAIS to compute stochastic upper and lower bounds on the log marginal likelihood.  
- Shows the gap between bounds upper-bounds the Jensen–Shannon divergence between true posterior and approximate sample distribution, yielding a single scalar convergence metric.  
- Method requires synthetic data from the joint; authors provide experiments showing synthetic analyses predict real-world performance.  
- Demonstrates applications for model selection and debugging.  
- Paper is well-motivated, readable, and presents a useful method with a reasonably compelling evaluation.  
- Section 3 is notably harder to follow; the transition into 3.1 is abrupt and introduces variables/equations without sufficient motivation.  
- If the divergence bound in Section 3.1 is novel, the paper should explicitly state this novelty, since it is central to the contribution.  
- The paper never explicitly defines “AutoBDMC”; an early clear definition is needed, especially before 3.2 references it.  
- Minor error: Line 149 likely should read p(theta, z, y) instead of p(theta, z | y).  
- Notation is inconsistent: uses y and D interchangeably for data/observations, causing confusion.  
- Supplement issues: WebPPL code should state compatible WebPPL versions and installation instructions (preferably npm); releasing corresponding Stan code would be beneficial.
