- Proposes AutoBDMC, using Bidirectional Monte Carlo with AIS to compute stochastic upper/lower bounds on log marginal likelihood for MCMC convergence assessment.
- Shows the gap between bounds upper/lower bounds the Jensen-Shannon divergence between the true posterior and approximate samples, giving a single scalar convergence diagnostic.
- Demonstrates method is limited to synthetic data drawn from the joint but provides experiments suggesting synthetic analyses predict real-data behavior.
- Illustrates utility of the method for model selection and debugging.
- Paper is readable, well-motivated, and experimentally convincing overall.
- Section 3 is hard to follow; the transition to Section 3.1 abruptly introduces variables/equations without sufficient motivation.
- If the divergence bound in Section 3.1 is novel, this should be explicitly stated because it is central to the paper’s contribution.
- The definition of “AutoBDMC” is not explicitly stated; the authors should clearly define it (e.g., using AIS and rAIS to analyze MCMC convergence).
- Minor error: Line 149 likely uses p(theta, z | y) but should be p(theta, z, y).
- Notational inconsistency between using y and D for data/observations creates confusion.
- Supplemental WebPPL code needs documentation on compatible WebPPL versions and integration instructions, or better: release as an npm-installable WebPPL package.
- Suggestion to similarly release the Stan code to improve reproducibility and usability.
