- Introduces AutoBDMC, leveraging BDMC and AIS to produce stochastic upper and lower bounds on the log marginal likelihood for assessing MCMC convergence.  
- Proves the gap between these bounds upper-bounds the Jensen–Shannon divergence between the true posterior and approximate samples, yielding a single scalar convergence metric.  
- Method can only analyze convergence on synthetic data from the joint model, but experiments indicate synthetic results predict real-world behavior.  
- Demonstrates uses for model selection and debugging; overall paper is clear, well-motivated, and empirically convincing.  
- Section 3 is difficult to follow: the transition into 3.1 is abrupt and introduces variables/equations without sufficient motivation.  
- If the divergence bound in Section 3.1 is novel, that novelty should be stated explicitly.  
- “AutoBDMC” is never clearly defined; the paper should explicitly state early that it refers to using AIS and reverse AIS (rAIS) to analyze MCMC convergence.  
- Section 3.2’s claims presume AutoBDMC is already defined—clarify the term before using it in later sections.  
- Minor: Line 149 likely typo—p(theta, z | y) should be p(theta, z, y).  
- Minor: Inconsistent notation for observed data (y vs D); standardize across the paper.  
- Minor: Supplemental WebPPL code should list compatible WebPPL versions and integration instructions; prefer releasing as an npm-installable WebPPL package and provide a similar packaged Stan implementation if possible.
