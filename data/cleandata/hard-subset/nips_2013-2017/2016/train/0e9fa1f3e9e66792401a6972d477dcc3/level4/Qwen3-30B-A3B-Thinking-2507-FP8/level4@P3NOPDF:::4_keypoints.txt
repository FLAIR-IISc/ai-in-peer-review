- Introduces AutoBDMC: using BDMC with AIS to compute stochastic bounds on the log marginal likelihood for MCMC convergence assessment.
- Core contribution: gap between BDMC bounds gives a provable upper bound on the Jensen–Shannon divergence, yielding a single scalar convergence metric.
- Empirical validation: synthetic-data analysis predicts real-world behavior and demonstrates utility for model selection and debugging.
- Writing and experiments: well-motivated, overall clear, and experimentally compelling.
- Critical omission: "AutoBDMC" is never explicitly defined in the main text, undermining clarity of the method.
- Section 3.2 is meaningless without a prior definition of AutoBDMC; definition must precede that discussion.
- Exposition issues in Section 3: abrupt transition into Section 3.1, with equations and variables presented without motivation.
- Theorem 1’s novelty and role should be explicitly stated; Section 3.1 is difficult to follow and obscures the technical contribution.
- Need clearer motivation linking the divergence bound (Theorem 1) directly to the scalar metric used in experiments.
- Notation inconsistency: use of `y` vs `D` for data (e.g., `p(theta, z | y)` vs `p(theta, z, y)`) causes confusion.
- Supplemental code issues: WebPPL code should specify compatible versions and installation instructions; consider releasing as an npm package; provide analogous Stan implementation and version notes.
- Experimental robustness: claim that synthetic data predicts real-world behavior rests on a single empirical test; more diverse real-data validation would strengthen the claim.
- Required fixes for acceptance: explicitly define AutoBDMC, state Theorem 1’s novelty, fix notation (e.g., `p(theta, z, y)`), and improve WebPPL code documentation.
- Recommendation: Major Revision Required — method is sound and valuable but presentation omissions currently obscure its significance.
