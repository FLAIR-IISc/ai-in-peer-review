- Proposes BREAD (Bounding Divergences with Reverse Annealing) to quantitatively assess MCMC convergence by extending BDMC/AIS.  
- Main theoretical contribution: the gap between forward and reverse AIS bounds yields an expected upper bound on the Jensen–Shannon (Jeffreys) divergence between true posterior and AIS sample distribution.  
- Provides a single, interpretable scalar metric for sample quality, improving on common heuristics like R-hat.  
- Builds directly on Grosse et al. (2015) BDMC work but extends it to evaluate sample quality rather than only marginal likelihood.  
- Practical utility shown: compares model representations (e.g., collapsed vs. uncollapsed) and discovered a real WebPPL bug, demonstrating value for debugging and model design.  
- Implements method in WebPPL and Stan, indicating applicability in probabilistic programming systems.  
- Main limitation: reverse AIS requires an exact posterior sample, restricting formal guarantees to synthetic data drawn from the joint.  
- Proposed protocol to apply method to real data: fit hyperparameters, simulate data from fitted model, then run BREAD; experiments show similar inference behavior between real and simulated data.  
- Experimental validation is sound and convincing, especially Section 5.1 supporting the simulation protocol.  
- Presentation weaknesses: Section 3 (theory) is hard to follow; notation for forward/reverse chains and the AIS history variable v needs clearer intuition and gradual derivation.  
- The core contribution (Jeffreys divergence bound) should be emphasized more explicitly rather than presented as a straightforward BDMC extension.  
- Terminology clarity: introduce and define the name BREAD concisely when first used.  
- Reproducibility: released code is good but needs version specifications, clearer integration instructions, and ideally easy-install packages (e.g., npm for WebPPL, Stan library inclusion).  
- Minor issues: inconsistent notation for data (y vs. D) and occasional notational inconsistencies between joint and conditional distributions in equations.
