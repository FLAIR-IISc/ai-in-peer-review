- Proposes AutoBDMC: using BDMC (AIS and reverse AIS) to bound Jensen-Shannon divergence between true posterior and approximate MCMC samples via marginal likelihood bound gap.
- Offers a scalar convergence metric for diagnosing MCMC quality, valuable for black-box probabilistic programming.
- Paper is well-motivated, generally clear, and demonstrates utility in WebPPL and Stan, including discovering a WebPPL Gaussian sampling bug.
- Extends prior work on convergence diagnostics and marginal likelihood estimation by using BDMC to assess sample quality directly.
- Appropriately cites and distinguishes from related work (e.g., Cusumano-Towner & Mansinghka).
- Section 3 (especially 3.1) has clarity issues: variables and equations introduced abruptly without enough context or motivation.
- The novelty and significance of the divergence bound in Section 3.1 are not explicitly stated and should be emphasized if novel.
- “AutoBDMC” is never formally defined in the manuscript and should be explicitly stated as AIS + rAIS applied for MCMC analysis.
- Minor issues: possible typo (line 149) and inconsistent data notation (y vs. D).
- Code release is positive, but the supplemental WebPPL code needs better documentation about compatible versions and integration.
- Validation shows consistency between simulated and real-data behavior, which is reassuring.
- Arguments for acceptance: novel BDMC application, theoretically grounded scalar metric, practical debugging utility, overall good writing.
- Arguments against acceptance: clarity problems, absence of formal AutoBDMC definition, and rigorous bounds limited to simulated data.
- Recommendations: improve Section 3 clarity, define AutoBDMC, fix notation, enhance code documentation, and release WebPPL and Stan code as installable packages.
