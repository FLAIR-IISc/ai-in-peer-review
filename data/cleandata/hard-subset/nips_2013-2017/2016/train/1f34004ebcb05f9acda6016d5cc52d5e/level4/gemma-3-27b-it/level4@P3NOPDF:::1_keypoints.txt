- Studies convergence for a non-convex loss in a graph-based ranking model defined by random-walk-based node and edge weights.  
- Problem is challenging because exact function values are intractable in this random-walk setting.  
- Proposes a two-level algorithm: first level uses a linearly convergent method to approximate the stationary distribution of the Markov walk.  
- Shows the stationary distribution approximation is valid and the loss can be approximated to arbitrary precision.  
- Develops a gradient method for constrained, non-convex optimization with an inexact oracle and proves convergence to a stationary point.  
- Main technical contribution: adapting convergence analysis to constrained optimization where function values are known only to specified precision.  
- Proofs appear sound, though the reviewer did not fully verify all details.  
- Concern: many optimization techniques seem adapted from Nesterov & Spokoiny (2015) on random gradient-free minimization.
