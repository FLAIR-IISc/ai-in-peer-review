- Main contribution: Apply triangular Cholesky factorization to CMA-ES to remove the matrix square root and reduce matrix inversion cost.  
- Theoretical claim: Modification is theoretically justified to not degrade algorithm performance.  
- Complexity/runtime: Method lowers runtime complexity and reduces wall-clock time, especially as dimensionality increases.  
- Evaluation cost: Maintains roughly the same number of objective function evaluations as standard CMA-ES.  
- Practical value: Valuable application of Cholesky factorization within the CMA-ES framework.  
- Request for clarification: Ask authors to elaborate on theoretical reasons for observed improved numerical stability.  
- Suggestion: Consider integrating low-dimensional approximation techniques (e.g., Loshchilov 2015) to better handle very high-dimensional problems like neural network policy search.
