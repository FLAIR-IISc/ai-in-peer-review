- Primary contribution: apply triangular Cholesky factorization to CMA-ES to avoid matrix square root and reduce matrix inversion cost.  
- Theoretical justification provided that algorithm performance is not compromised by the change.  
- Method reduces runtime complexity while keeping a comparable number of objective function evaluations.  
- Paper offers a valuable implementation of Cholesky factorization for CMA-ES.  
- Request for more detailed theoretical explanation on how the method achieves greater numerical stability.  
- Empirical results: similar objective evaluations to standard CMA-ES but reduced wall-clock time, increasingly so with higher dimensionality.  
- Suggestion to consider combining the framework with low-dimensional approximation methods (e.g., Loshchilov 2015) for high-dimensional applications like neural network policy search.
