- Introduces a Cholesky-factor-based covariance update for CMA-ES that adapts the triangular factor directly.  
- Eliminates explicit matrix inversion and directly yields covariance eigenvalues.  
- Reduces per-iteration computation from O(d³) to O(d²).  
- Lowers memory usage by storing only the lower-triangular Cholesky factor (O(d²) storage).  
- Maintains identical convergence behavior (same number of function evaluations) across benchmark problems.  
- Substantially reduces wall-clock time per iteration, with gains scaling for higher dimensions (d ≥ 50); e.g., ~30% faster on 256-D Ellipsoid.  
- Thorough empirical evaluation against the state-of-the-art Python `cma` implementation across standard test functions and multiple dimensions.  
- Particularly well-suited for medium-to-large-scale ML tasks, reinforcement learning/policy search, and resource-constrained/high-dimensional settings.  
- Manuscript is clearly written with effective figures showing scaling and performance gains.
