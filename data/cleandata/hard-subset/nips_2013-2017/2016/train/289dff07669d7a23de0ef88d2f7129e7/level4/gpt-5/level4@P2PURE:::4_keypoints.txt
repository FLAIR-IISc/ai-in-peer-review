- Main contribution: use triangular Cholesky factorization to remove the matrix square root in CMA-ES, reducing cost of matrix inverses.  
- Theoretical justification: modification does not deteriorate algorithm performance.  
- Empirical effect: runtime complexity reduced without significant change in number of objective function evaluations.  
- Overall assessment: valuable Cholesky-based implementation for CMA-ES.  
- Request for clarification: please elaborate theoretically why the computation is “numerically more stable.”  
- Observation: Cholesky-CMA-ES retains same objective evaluation count but reduces wall-clock time, with greater gains at higher dimensionality.  
- Suggestion: consider integrating the low-dimensional approximation of Loshchilov (2015) into this framework.  
- Concern: high dimensionality (e.g., neural network policy search) remains a critical practical issue that should be addressed.
