- Proposes Cholesky-CMA-ES replacing covariance matrix square root with Cholesky factor to reduce computational complexity.  
- Targets the known CMA-ES bottleneck where matrix square root dominates runtime and memory in high dimensions.  
- Notes prior work either applied to specialized variants or lacked theoretical justification for performance preservation.  
- Provides rigorous theory that replacing the square root with the Cholesky factor in step-size adaptation introduces vanishing error (Lemma 1).  
- Achieves same asymptotic time complexity O(µ d²) as the fastest existing implementations.  
- Maintains the same number of objective function evaluations as standard CMA-ES.  
- Empirical results show significant wall-clock speedups (up to 20× at d=64) with increasing gains at higher dimensions.  
- Practical benefits include optimal storage complexity, free eigenvalue computation, and numerically stable inversion of the Cholesky factor.  
- Main contribution is a theoretically justified, practical Cholesky implementation filling a gap in prior approaches.  
- Empirical evaluation is thorough: multiple benchmark functions, dimensions up to 256, and rotation-invariance tests.  
- Minor weakness: insufficient explanation or evidence supporting the claim of improved numerical stability.  
- Opportunity for improvement: discuss integration with low-dimensional approximation methods for extremely high-dimensional problems.  
- Paper is well-written, algorithm description precise (implementable), and experiments are well-designed.  
- Significant practical impact for applications like RL policy search and hyperparameter tuning, enabling larger-scale CMA-ES usage.  
- Overall assessment: strong paper with theoretical and empirical validation; recommended for acceptance and suggested as replacement when covariance update/storage is a bottleneck.
