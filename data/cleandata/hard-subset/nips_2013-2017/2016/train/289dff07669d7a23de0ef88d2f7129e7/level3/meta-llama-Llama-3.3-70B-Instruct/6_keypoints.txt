- Proposes Sparse Access Memory (SAM), a neural memory architecture using sparse read/write to scale large external memories efficiently.
- Sparse read and write operations reduce computational and memory overhead relative to traditional MANNs.
- Shows significant speedups and memory reductions compared to NTMs and Memory Networks while preserving comparable learning performance.
- Clearly explains SAM’s architecture, read/write mechanisms, and efficient backpropagation through time.
- Provides a thorough time and space complexity analysis and claims asymptotic optimality.
- Experimental results demonstrate SAM matches dense models on synthetic tasks, NLP tasks, and one-shot learning.
- Paper is well-written, clearly organized, and includes a detailed related-work discussion.
- Lacks comparison to other sparse memory architectures, limiting evaluation of relative advantages.
- Needs deeper analysis of trade-offs between sparse and dense memory architectures.
- Should analyze more thoroughly the potential impact of sparsity on learning performance and generalization.
- Reviewer recommendation: strong accept due to significant contribution and strong experimental support.
