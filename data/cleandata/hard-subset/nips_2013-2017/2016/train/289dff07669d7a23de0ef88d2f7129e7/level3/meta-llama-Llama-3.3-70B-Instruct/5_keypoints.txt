- Introduces Sparse Access Memory (SAM), a novel neural memory architecture using sparse read/write for large external memories.  
- Key innovation: sparse operations reduce computational and memory overhead of traditional MANNs.  
- Empirically achieves significant speedups and memory reductions versus NTMs and Memory Networks while keeping comparable task performance.  
- Well-written paper with clear explanation of SAM components (sparse read/write, approximate nearest neighbor index, efficient BPTT).  
- Experimental results are thorough and demonstrate effectiveness on synthetic and some real-world datasets.  
- Strong balance of theory and practice, including rigorous time and space complexity analysis showing asymptotic optimality.  
- Provides clear comparisons with existing MANNs, highlighting relative advantages and limitations.  
- Concern: sparse approximations may be ineffective when memory has strong structure or complex dependencies.  
- Discussion of the impact of sparse approximations on learning and trade-offs is limited and needs deeper investigation.  
- Evaluation is limited on more realistic and challenging domains (e.g., large-scale NLP or computer vision) and should be expanded.  
- Originality: significant contribution by combining sparse access operations with efficient data structures for scalable neural memory.  
- Overall strong contribution with practical and theoretical merits, but applicability and evaluation scope need further work.
