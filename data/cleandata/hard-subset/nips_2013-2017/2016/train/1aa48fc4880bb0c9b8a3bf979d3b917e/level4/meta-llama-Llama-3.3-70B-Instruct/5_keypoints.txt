- Proposes first linearly convergent algorithms for saddle point problems using variance reduction (SVRG, SAGA).  
- Notes conventional convex minimization analysis is inapplicable; uses monotone-operator framework to establish convergence.  
- Analysis extends to a broader class of problems, including variational inequality problems.  
- Shows formulation can handle non-separable loss functions and regularizers as saddle point problems.  
- Main significance: first algorithms achieving linear convergence for saddle point problems relevant to many ML models.  
- Proof technique—interpreting saddle points as zeros of a monotone operator—is intriguing and broadly applicable.  
- Provides simpler proofs for FB splitting and its stochastic variant.  
- Algorithms with non-uniform sampling are competitive and often outperform prior alternatives.  
- Since primary novelty is the proof technique, suggests adding more proof details or a proof sketch in the main paper.  
- Requests clarification on the type of non-uniform sampling used beyond bilinear problems and the precise definition of π.  
- Asks for clarification on how step size σ was set in experiments and intuition on why non-uniform sampling is essential for saddle point problems but not for convex minimization.
