Summary and relation to prior work
The paper proposes Supervised Word Mover’s Distance (S-WMD), a supervised variant of Word Mover’s Distance (WMD) that learns a task-specific ground metric and word-importance weights. The method learns (i) a linear transformation A applied to pre-trained word embeddings (e.g., word2vec), and (ii) a nonnegative reweighting w of document histograms, and uses these to define a new optimal-transport distance between documents. The learning objective is a stochastic neighbor loss (as in NCA) that approximates leave-one-out kNN classification error in the learned distance.

To make training tractable, the paper replaces the exact WMD (EMD) with the entropically regularized OT (Sinkhorn distance), using the Sinkhorn-Knopp scaling to obtain a unique optimal transport T*λ in O(q^2) time for q unique words per document. Gradients w.r.t. A use the optimal transport plan; gradients w.r.t. w use the dual potentials (α, β), approximated from the Sinkhorn scaling vectors (u, v). The authors also propose an efficient training scheme: initialize A with NCA learned on word centroid distances (WCD), use mini-batches, and restrict neighborhood computations using WCD-based nearest neighbors.

Empirically, on 8 datasets spanning categorization and sentiment tasks, S-WMD allegedly outperforms 26 baselines (unsupervised representations, WCD/WMD, and three supervised metric learning methods applied to various document representations) in kNN classification accuracy, with code released.

This work builds on: WMD (Kusner et al.), entropic OT (Cuturi), and NCA (Goldberger et al.). Related supervised EMD ground-metric learning has been explored in vision (e.g., ITML/LMNN variants for histogram distances; Cuturi & Avis; Wang & Guibas; Wang et al.), but those did not target text with word embeddings nor include word-importance reweighting.

Quality (technical soundness)
- Formulation: The objective—NCA-style loss using a learned OT ground metric—matches the goal of reducing kNN error. The introduction of both A and w is well-motivated: A adapts semantic geometry; w promotes task-specific vocabulary weighting while preserving normalized histograms.
- Optimization: Replacing exact WMD by entropic OT is standard and yields uniqueness and differentiability. Using T*λ in the gradient w.r.t. A is appropriate for the regularized problem; the paper, however, blurs whether the loss uses the regularized distance consistently (it appears so in practice, but clarification would help).
- Gradients: The derivative w.r.t. A via T*λ is standard. The derivative w.r.t. w via dual potentials is reasonable; recovering (α, β) from (u, v) is known for entropic OT, but the paper gives only a brief expression (including a centering term). A short derivation or citation to a precise source (e.g., Cuturi & Peyré) would improve rigor.
- Constraints on w: Since w reweights histograms that are then renormalized, w should be constrained to be nonnegative; this is implied but not stated nor enforced in Algorithm 1. A practical positive reparameterization (e.g., w=exp(θ)) would avoid invalid weights and improve stability.
- Regularization: There is no explicit regularization on A or w. Given the potentially large vocabulary, the model can overfit; an L2 penalty or sparsity encouragement on w, and Frobenius/nuclear regularization on A, would be advisable. It is unclear whether early stopping suffices; ablation on regularization would strengthen the case.
- Computational claims: The complexity analysis is plausible: O(q^2) for Sinkhorn, plus costs for gradient assembly. The neighbor pruning using WCD is sensible and likely essential. Reporting absolute training and inference times per dataset (and λ, N sensitivity) would be helpful; Table 3 is referenced but not fully visible in the excerpt.
- Experiments: The evaluation uses kNN classification across 8 datasets with many baselines and supervised comparators (NCA/ITML/LMNN after PCA). Hyperparameters are tuned via Bayesian optimization; code is released. That said:
  - Fairness: S-WMD leverages pre-trained word embeddings; some baselines do not. Including additional strong text baselines with embeddings (e.g., average word2vec + supervised metric learning; paragraph vectors) would isolate gains due to OT versus representation.
  - Ablations: The contribution of w versus A is not disentangled. Reporting performance with only A, only w, and both would clarify their roles. Sensitivity to λ and neighborhood size N is also missing.
  - Task scope: Results are only for kNN classification; retrieval/ranking metrics (e.g., MAP, nDCG) would better match the motivation of document distances.

Clarity
- The paper is generally understandable, with a clear high-level story and algorithmic description. However:
  - Notation clashes: p is used for both the Lp ground cost exponent and the number of unique words; d denotes embedding dimension; r is projection dimension. Standardize symbols to avoid ambiguity.
  - Minor typos and formatting: Section numbering (“.2 Optimization”), missing or malformed references (e.g., “[5, 21?]”), and some incomplete citations should be fixed.
  - Derivations: Provide a precise reference or appendix derivation for mapping (u, v) to (α, β) in the entropic dual, and state explicitly that the training loss uses the entropic OT distance.
  - Constraints: State w ≥ 0 and whether A is constrained (e.g., low-rank), and describe any regularization used.

Originality
- Combining supervised metric learning with WMD by learning both a ground metric in embedding space and word-importance weights appears novel in NLP. While supervised EMD ground-metric learning exists in vision, applying entropic OT and NCA to text with word embeddings and an explicit histogram reweighting is a meaningful new combination. The WCD-based initialization for A is a practical and sound heuristic.

Significance
- If the reported improvements over strong baselines hold, S-WMD provides a useful supervised distance for text that is computationally feasible and easy to adopt with pre-trained embeddings. It should be relevant for classification and retrieval tasks that benefit from OT-style document comparison. The approach is likely to be built upon (e.g., with regularization, nonlinear A, or task-specific embeddings).

Pros and cons for acceptance
Pros:
- Clear problem motivation; principled use of entropic OT for efficiency.
- Joint learning of A and w tailored to labels; practical training scheme with sensible initialization and neighbor pruning.
- Broad empirical evaluation with many baselines; code release.
- Conceptually simple, leverages widely used word embeddings.

Cons:
- Limited ablations and sensitivity analyses (role of w vs A; λ, N; regularization).
- Missing/implicit constraints on w; lack of explicit regularization may invite overfitting.
- Clarity issues in notation, derivations, and references.
- Evaluation limited to kNN classification; lacks retrieval metrics or non-kNN classifiers using the learned distance.

Suggestions for improvement
- Add ablations for A-only, w-only, and both; include sensitivity to λ and neighbor set size; introduce and tune regularization on A and w.
- Enforce w ≥ 0 via reparameterization; report the effect of sparsity on interpretability and performance.
- Clarify whether the loss uses entropic OT distances consistently; provide derivations or citations for dual potentials from Sinkhorn scalings.
- Expand baselines to include embedding-based representations with supervised metric learning, and report retrieval metrics.
- Clean up notation, section numbering, and references.
