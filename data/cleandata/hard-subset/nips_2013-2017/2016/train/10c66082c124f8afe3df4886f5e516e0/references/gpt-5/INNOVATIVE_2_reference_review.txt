Summary
The paper proposes Supervised Word Mover’s Distance (S-WMD), a supervised extension of WMD for document comparison. Rather than learning document embeddings, S-WMD learns (i) a linear transformation A of the pre-trained word embedding (ground metric) and (ii) a nonnegative word-importance vector w that reweights term histograms, so that the OT-based distance aligns with class labels. The learning objective is an NCA-style stochastic leave-one-out kNN loss using exp(−DA,w). The key computational idea is to replace the exact OT plan with entropic OT (Sinkhorn), which yields a unique, differentiable transport plan and gives access to efficient gradients—both via the primal plan for ∂/∂A and via dual potentials (extracted from Sinkhorn’s u,v) for ∂/∂w. The method initializes A by running NCA on word centroids (S-WCD), then optimizes with batched gradients and a WCD-based neighbor pruning. Across 8 datasets and 26 baselines, S-WMD achieves best kNN accuracy on 5/8 datasets and the best average rank; code is released.

Strengths
- Conceptual contribution: A clean, principled unification of metric learning and OT for text. Learning both the ground metric (A) and term weights (w) directly in the OT framework is a compelling direction that goes beyond applying off-the-shelf metric learning to document vectors.
- Computational practicality: Using the entropic regularization for both efficiency and differentiability is well-motivated. The use of Sinkhorn’s fixed-point iterations to obtain dual variables for w’s gradient is elegant.
- Objective choice: The NCA-style loss is a natural surrogate for nearest-neighbor performance, which is how WMD is typically used.
- Implementation details that matter: Sensible initialization (S-WCD), neighbor filtering based on WCD, and minibatching make the approach practical. Complexity analysis is provided and training times are reasonable.
- Empirical results: Broad evaluation against many competitive baselines, including supervised metric learning applied to standard document representations. Clear qualitative visualizations, and code availability.

Weaknesses
- Novelty is incremental in places: Learning ground metrics for EMD/OT and using entropic OT for differentiability are established; the primary novelty is their integration for text with a learned word-importance vector. This is a solid synthesis, but not a fundamentally new OT formulation.
- Missing or unclear constraints/regularization: The method relies on w to reweight histograms; w must be nonnegative to keep normalized histograms valid. The paper does not specify how nonnegativity is enforced (projection, exponential parameterization, or penalties), nor any regularization on A or w to avoid degeneracy/sparsity collapse.
- Evaluation scope: Results are exclusively with kNN. Since many baselines produce fixed-length representations, it would be informative to report standard linear classifiers (e.g., logistic regression/SVM) on those representations and to compare against a classifier built atop the learned A and w (e.g., nearest-centroid), to ensure the gains are not kNN-specific. Some strong text baselines (e.g., linear SVM on TF-IDF) are not explicitly reported.
- Ablations and sensitivity: It is unclear how much each component contributes. Missing: ablation of w vs A (learn only A, only w), the impact of initialization quality, sensitivity to λ (entropic temperature), neighbor set size N and batch size, and ground metric choice (L2 vs L1).
- Train/test discrepancy: Training uses the entropic OT objective; it is not clearly stated whether test-time distances for kNN use exact WMD or the same Sinkhorn-regularized distance. This matters for both speed and fidelity to the trained objective.
- Theoretical aspects: While gradients are justified under strong convexity of entropic OT, there is no discussion of approximation bias introduced by λ, nor generalization considerations for the NCA surrogate in this OT setting.

Questions for the authors
1. How do you ensure w ≥ 0 during training? Is there projection, reparameterization (w = exp(θ)), or a constraint penalty? Any regularization on A (e.g., Frobenius/trace) or on w (e.g., L1/L2) to prevent trivial solutions?
2. What is used at test time for kNN distances—exact WMD or entropic OT with the same λ? If the latter, please report λ-sensitivity and the trade-off between accuracy and speed.
3. Ablations: (a) learn A only vs w only vs both; (b) remove WCD-based neighbor pruning; (c) different λ and p; (d) using L1 ground cost. These would strengthen the empirical narrative.
4. Fairness of baselines: Can you include logistic regression or linear SVM on TF-IDF and on learned representations, and compare to a classifier using the learned metric (beyond kNN)?
5. Dimensionality r: What r is used for A? Any insights into the structure of A (e.g., spectrum) and interpretability of w across datasets?

Clarity
Generally clear and well organized. The gradient derivations are concise and helpful. Please clarify the constraints/regularization on w and A, the test-time distance, and provide a brief note on how α,β are obtained from u,v (with constants’ indeterminacy handled by centering), with references.

Originality
Moderate. The combination—supervised ground metric + histogram reweighting within OT for text, trained via NCA and Sinkhorn—is a noteworthy and practically useful synthesis that extends WMD meaningfully.

Significance
High practical significance for OT-based text similarity. The approach is simple, efficient, and shows robust gains over strong baselines on multiple datasets, with released code. I expect others to adopt S-WMD or its variants.

Pros for acceptance
- Clear, useful extension of WMD with supervision and efficient training
- Solid empirical improvements and code release
- Method likely to influence follow-on work at the intersection of OT and metric learning for NLP

Cons for acceptance
- Somewhat incremental relative to known components
- Missing constraints/regularization details and ablations
- Evaluation limited to kNN and lacks some standard text baselines

Recommendation
Accept (weak accept). While built from known pieces, the synthesis is timely, practically impactful, and opens a useful supervised OT direction for document distances. With clarifications and ablations addressed, this would be a strong contribution.

Confidence
High.
