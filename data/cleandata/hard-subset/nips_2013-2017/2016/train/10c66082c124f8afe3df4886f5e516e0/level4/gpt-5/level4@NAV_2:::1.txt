This paper introduces a supervised metric learning framework built atop the recent Word Mover’s Distance (WMD) [19], which formulates document similarity as an optimal transport problem over word2vec embeddings. The method can be used with any transport distance that employs a squared Euclidean ground cost after a linear projection. Beyond learning a linear projection to improve performance, the approach also learns per-word importance weights: it reweights each word’s frequency by a learned scalar and re-normalizes the document histogram accordingly. The authors present an efficient training strategy for this parameterized metric, leveraging a tractable approximation to avoid nested optimization. A comprehensive empirical study demonstrates effectiveness versus numerous strong baselines across multiple document classification benchmarks, using kNN with the learned metric.

+ This work builds directly on WMD and constitutes a technically meaningful and practically useful extension, with clear gains on several kNN text classification tasks.

- The potential impact may be tempered by the observation that an Earth Mover’s-based approach initialized via a “word centroid distance” (WCD) metric learning step performs comparably to the fully trained model. See the gap between the last two rows in Table 3. In short, the main technical advances here seem to add relatively modest improvements beyond this initialization using existing tools.
- Table 3 would benefit from including results for the “word centroid embedding” representation (document as the weighted mean of word embeddings) combined with standard metric learning methods (ITML, LMNN, NCA).
- An ablation assessing the separate contributions of the word-specific weights versus the projection matrix A is missing. Reporting performance when each component is used in isolation would clarify their relative importance.

o The primary novelty lies in the optimization strategy (Lines 156–175). While the notion of learning a linear metric within the EMD framework is appealing, by itself it represents a relatively limited contribution.
