- Proposes a supervised extension of WMD that learns a linear transform A and word importance weights w for document metric learning.  
- Core contribution is well-motivated: supervised metric learning can improve document similarity measurement.  
- Equation (5) lacks intuition; authors should explain how w modifies the distance computation relative to A and which component drives performance gains.  
- Need clarification/evidence on whether w primarily handles lexical variation (synonyms) while A handles semantic alignment.  
- Missing baseline: report kNN performance using unsupervised WMD to contextualize supervised gains (authors note this was later addressed).  
- Ambiguity in regularization λ: explicitly define its role and whether the λ in Equation (2) and line 170 are the same or distinct hyperparameters.  
- Claim that the method is “forgiving” of syntactic variation (line 316) lacks evidence; provide a concrete example or analysis (e.g., “car” vs. “automobile”).  
- Presentation issues: incorrect table reference (line 257) — fix to reference the correct table/figure.  
- Table 2 uses undefined terms “n” and “ne” — define these in the caption.  
- k for kNN classification is unspecified — state whether k is fixed or optimized per dataset for reproducibility.  
- Strengths: clear problem statement, logical extension of WMD, thorough empirical validation across datasets, promising w component, and promised code release.  
- Authors were responsive to queries; recommend acceptance pending minor revisions: explain w in Eq.(5), define symbols (λ, k, n/ne), add syntactic-variation example, and correct table references.
