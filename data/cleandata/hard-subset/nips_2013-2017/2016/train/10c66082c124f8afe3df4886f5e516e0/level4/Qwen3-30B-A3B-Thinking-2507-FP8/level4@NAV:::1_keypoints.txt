- Introduces supervised metric learning for word-movers distance that jointly learns a linear projection and word-specific frequency weights for document representations.  
- Method generalizes to any transport distance based on squared Euclidean distances after linear projection.  
- Provides an efficient optimization procedure for the parameterized metric (claimed as the main originality).  
- Comprehensive experiments show strong performance versus many baselines on multiple document classification datasets using kNN.  
- Technically interesting and effective extension for kNN text classification tasks.  
- Limited impact: earth-movers distance with metric learning initialized by word-centroid distance achieves comparable performance, so gains appear marginal.  
- Table 3 omits evaluation of word-centroid embeddings combined with standard metric learning methods (ITML, LMNN, NCA).  
- Evaluation lacks ablation testing to separate the contributions of word-specific weights versus the projection matrix A.  
- Originality primarily resides in the optimization details (Lines 156–175); the overall contribution is constrained.  
- Paper is exceptionally well written and clear.  
- Minor clarifications requested: clarify which paper “The authors” refers to at Line 152; indicate if α and β in Equation 11 should carry stars; explicitly recall that Sinkhorn scaling (from Line 56) is used around Line 170.
