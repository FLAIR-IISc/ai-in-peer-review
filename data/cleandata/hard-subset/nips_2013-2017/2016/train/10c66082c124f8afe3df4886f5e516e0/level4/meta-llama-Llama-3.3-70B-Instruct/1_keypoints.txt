- Proposes a supervised metric learning framework using the word-movers distance on word2vec embeddings.  
- Framework applies to any transport distance based on squared Euclidean distances after a linear projection.  
- Method jointly learns a linear projection and word-specific weights that re-weight and re-normalize word frequencies.  
- Introduces an efficient optimization method to learn the metric parameters.  
- Extensive experiments show effectiveness on several document classification datasets using kNN with the learned metric.  
- Limitation: an earth-movers variant initialized with a word-centroid metric learning yields similar performance, questioning added impact.  
- Table 3 omits a comparison of the word-centroid embedding (weighted mean) against existing metric learning methods (ITML, LMNN, NCA).  
- Missing ablation study evaluating the individual contributions of word-specific weights versus the projection matrix A.  
- Originality mainly in the metric optimization (linear metric learning within earth-movers distance), but the contribution is limited.  
- Minor writing/clarity issues: ambiguous "The authors" reference (Line 152), Equation 11 may need alpha/beta starred, and optimization should explicitly be tied to the earlier Sinkhorn scaling.
