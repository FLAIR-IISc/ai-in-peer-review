This paper introduces the Supervised Word Mover's Distance (S-WMD), a novel method for learning a task-specific document distance metric. The work builds directly upon the recently proposed Word Mover's Distance (WMD), which defines the distance between documents as an optimal transport problem over their respective word embeddings. While WMD is entirely unsupervised, S-WMD incorporates supervision from class labels to learn a more discriminative metric. This is achieved by simultaneously learning a linear transformation of the underlying word embedding space and a vector of word-importance weights. To overcome the computational challenge of differentiating through the WMD optimization, the authors propose an efficient approximation based on the regularized Sinkhorn distance, making the approach practical for training. The authors promise to make the code available.

**Strengths / Arguments for Acceptance:**

*   **Originality and Significance:** The paper addresses a natural and important extension of the powerful WMD framework to the supervised setting. The proposed method for learning both a transformation of the embedding space and word-specific importance weights is novel and well-motivated.
*   **Technical Quality:** The use of the regularized Sinkhorn distance to create a differentiable and efficient approximation of the WMD objective is a sound and clever technical contribution. This makes an otherwise computationally prohibitive metric learning problem tractable. The initialization strategy using a supervised word centroid distance (S-WCD) is also a sensible approach to handle the non-convex optimization.
*   **Thorough Evaluation:** The experimental evaluation is extensive, comparing S-WMD against 26 competitive baselines across eight diverse real-world datasets. The results are compelling, showing that S-WMD achieves state-of-the-art k-nearest neighbor classification performance on a majority of the tasks.

**Weaknesses / Questions for Improvement:**

*   **Ablation Study:** The method learns two components: a linear transformation `A` and a word importance vector `w`. It would significantly strengthen the paper to include an ablation study that isolates the contribution of each component. This would provide valuable insight into which part of the learned metric is responsible for the performance gains.
*   **Lack of Intuition:** More intuition could be provided for the re-weighting scheme defined in Eq. (4). A brief explanation of how this formulation was chosen and its expected effect would improve the paper's clarity.
*   **Experimental Context:** While evaluating a distance metric using kNN error is appropriate, the results would be better contextualized by including a comparison to a standard supervised classifier (e.g., an SVM with TF-IDF features) on the same datasets. This would help benchmark the performance of this metric-learning-based approach against more direct classification methods.
*   **Minor Clarifications:**
    *   In Section 3.1, the paper introduces a regularization parameter λ for the Sinkhorn distance. Is this the same λ used to derive the dual variables (line 170)? Clarification on this and how its value (λ=10) was chosen would be helpful.
    *   The conclusion claims S-WMD is "particularly forgiving when two documents use syntactically different but conceptually similar words." While this property is inherited from WMD, the paper could substantiate this claim with a qualitative example in the supervised context.
    *   The description of results contains a minor inconsistency: the text (line 257) mentions an "average error across datasets," but Table 2 appears to report the "average rank." Please clarify.
    *   In Table 2, the column headers "n" and "ne" are not defined.
    *   The procedure for selecting the hyperparameter `k` for kNN classification is not described. It is unclear if `k` was tuned for each method and dataset or fixed.
