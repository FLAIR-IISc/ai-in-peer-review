This paper proposes the Supervised Word Mover’s Distance (S-WMD), an extension of the recently introduced Word Mover’s Distance (WMD). The original WMD provides an effective unsupervised distance metric between documents by calculating the minimum cost to "transport" words from one document to another in a pre-trained word embedding space. The key contribution of this work is to incorporate supervision into this framework. The authors propose learning two components simultaneously: (1) a linear transformation `A` of the word embedding space, and (2) a word importance vector `w` that re-weights the bag-of-words representations. These parameters are learned by minimizing a stochastic leave-one-out k-NN classification error, inspired by Neighborhood Components Analysis (NCA). A significant technical challenge is that the gradient of the WMD objective is computationally prohibitive. The authors address this by using an entropy-regularized version of the optimal transport problem (the Sinkhorn distance), which allows for an efficient and differentiable approximation. The proposed method is evaluated on eight text classification datasets, where it demonstrates state-of-the-art or highly competitive performance against 26 baselines.

**Strengths:**

1.  **Originality and Significance:** The paper addresses a clear and important limitation of the original WMD: its unsupervised nature. The proposed method for introducing supervision is novel and well-motivated. Extending WMD, which is already a very strong baseline, to a supervised setting is a significant contribution to the field of document similarity and classification. The two-pronged approach of learning both a transformation of the embedding space (`A`) and word-specific importance weights (`w`) is elegant and powerful.

2.  **Technical Quality:** The technical approach is sound. Recognizing the computational bottleneck in differentiating through the WMD optimization is key, and the use of the entropy-regularized Sinkhorn distance as a surrogate is a clever and practical solution. The paper provides the necessary derivations for the gradients with respect to both `A` and `w`. The proposed optimization strategy, including a sensible initialization using a supervised version of the Word Centroid Distance (WCD) and batch gradient descent with neighbor filtering, makes the overall algorithm practical.

3.  **Thorough Evaluation:** The empirical evaluation is extensive. The method is tested on eight diverse datasets against a very large set of 26 baselines, including both classic document representations (BOW, TF-IDF, LSI, LDA) and more modern ones (mSDA), combined with several standard metric learning algorithms (NCA, LMNN, ITML). The results are compelling, showing that S-WMD achieves the lowest error on 5 of the 8 datasets and the best average rank overall. The promise to release the code is a valuable addition for reproducibility.

**Weaknesses and Suggestions for Improvement:**

While this is a strong paper, there are several areas where clarity could be improved, which would strengthen the final manuscript.

1.  **Intuition and Ablation:** The paper could provide a clearer intuition for the role of the word importance vector `w`. While Eq. (4) defines it mathematically as a re-weighting of the document's bag-of-words vector, a more explicit explanation of its function would be helpful. More importantly, the relative contributions of the two learned components (`A` and `w`) are unclear. The performance gains are attributed to their combination, but an ablation study would be highly valuable. For instance, what is the performance of learning only `A` (with `w` fixed to ones) or only `w` (with `A` fixed to identity)? This would provide crucial insight into which component is driving the improvements.

2.  **Clarity on Technical Details:** There is a minor point of confusion regarding the regularization parameter `λ`. Equation (2) introduces `λ` in the context of the regularized transport problem from prior work. Later, on line 170, the authors state they use `λ = 10`. It should be made explicit that the `λ` from the background section (Cuturi, 2013) is the same parameter used in their gradient approximation. A brief sentence connecting these two mentions would resolve any ambiguity.

3.  **Justification of Claims:** The conclusion states that S-WMD is "particularly forgiving when two documents use syntactically different but conceptually similar words." While this is an intuitive benefit inherited from the original WMD, the claim would be much stronger with a concrete, qualitative example. Showing a pair of documents from the test set with different keywords that S-WMD correctly identifies as close (and perhaps WCD or TF-IDF fails on) would provide excellent support for this claim.

4.  **Clarity in Reporting Results:**
    *   The text (approx. line 261) mentions that S-WMD "on average across all datasets [...] outperforms all other 26 methods." However, Table 2 only shows the "Avg. Rank". While rank is a good summary statistic, the text seems to imply an average error was computed. Please clarify if this is a misstatement or if the average error metric is missing from the table.
    *   In Table 2, the meaning of the columns "n" and "ne" is not defined in the caption or text. Please add this information.
    *   The experimental setup for k-NN classification does not specify how the value of `k` was chosen. Was it tuned per dataset via cross-validation, or was a single value used for all experiments? This is an important detail for reproducibility.

**Note on Evaluation Scope:**
I note from my initial reading and the authors' subsequent clarification in the rebuttal that the goal of the paper is to learn a superior distance metric for k-NN classification, not necessarily to build the best overall text classifier. This framing is appropriate and justifies the choice of k-NN as the evaluation method.

**Summary of Arguments:**

*   **Pro-acceptance:**
    *   Highly novel and significant extension of the influential WMD metric to a supervised setting.
    *   Technically sound solution to a non-trivial optimization problem.
    *   Extensive and convincing experimental results showing state-of-the-art performance.
    *   The work is likely to be impactful and built upon by others.

*   **Con-acceptance (minor, addressable):**
    *   Lack of an ablation study to disentangle the effects of the learned matrix `A` and weights `w`.
    *   Several minor points of unclarity in the experimental reporting and technical descriptions that should be addressed.

Overall, this is a very high-quality paper presenting a novel, well-executed, and significant contribution. The weaknesses are minor and primarily relate to clarity and completeness of the analysis, all of which can be addressed in a final version. I recommend acceptance.
