- Introduces a spectral approach using the Koopman operator to analyze nonlinear dynamical systems via function composition.
- Applies Koopman decomposition to forward dynamics defined as x(t+2) = f(f(x(t))).
- Uses system trajectories to perform eigen-decomposition of the operator.
- Draws parallels to kernel PCA and subspace analysis techniques.
- Demonstrates the method on embedding and recognition tasks with promising empirical results.
- Key contribution: framing function composition as a linear operator that can be decomposed, which suits dynamical systems.
- Novelty appears limited because of substantial prior work on Dynamic Mode Decomposition (DMD).
- Essentially presents a kernelized extension of established DMD-like techniques.
- Notes alternative approaches (e.g., recurrent neural networks) can address similar tasks, and applications don’t clearly show a unique advantage.
