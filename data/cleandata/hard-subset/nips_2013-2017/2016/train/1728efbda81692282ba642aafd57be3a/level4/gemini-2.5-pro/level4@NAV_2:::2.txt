This paper presents a method for spectral analysis of nonlinear dynamical systems by extending Dynamic Mode Decomposition (DMD) to Reproducing Kernel Hilbert Spaces (RKHS). The core idea is to analyze the Koopman operator, an infinite-dimensional linear operator that governs the evolution of observables of a nonlinear system, within the context of an RKHS. By doing so, the method can approximate the Koopman eigenfunctions using a rich, nonlinear set of basis functions implicitly defined by a kernel function, which overcomes the limitations of standard DMD that typically relies on linear observables. The authors propose an algorithm that identifies the system's dynamic modes by performing an eigendecomposition of the Koopman operator projected onto a Krylov subspace constructed in the feature space. A second, more robust algorithm is also presented, which first projects the data onto its principal components in the feature space (via kernel PCA) before performing the decomposition. The paper situates this work in the context of decomposition techniques for dynamical systems like POD and DMD, and correctly identifies its relationship to spectral methods in machine learning.

**Strengths:**
*   The paper provides an elegant and theoretically sound extension of the popular DMD method to RKHSs. This is a natural step that allows for a principled, non-parametric approach to capture complex nonlinear dynamics without requiring the pre-selection of an explicit basis, as in methods like Extended DMD.
*   The derivation of the algorithm is clear and well-grounded in the theory of Koopman operators and kernel methods. The provided theorems connect the computed Ritz values and vectors to the true Koopman eigenvalues and eigenfunctions under reasonable assumptions.
*   The proposed robustification of the algorithm using a projection onto kernel principal components is a valuable practical contribution. This step addresses the well-known numerical sensitivity of Arnoldi-type methods like DMD, particularly when dealing with rank-deficient data matrices.
*   The paper successfully demonstrates the versatility of the learned Koopman modes as features for various downstream tasks, including state prediction, sequential change-point detection, and the embedding and recognition of dynamics.

**Weaknesses:**
*   **Originality:** The primary weakness is the limited discussion of the method's novelty with respect to prior work, specifically Extended DMD (eDMD) [35]. The eDMD framework provides a general procedure for Koopman analysis using a user-specified dictionary of observables. The proposed kernel DMD can be interpreted as a specific instance of eDMD where the dictionary is implicitly the feature map of the chosen kernel. The paper would be significantly strengthened by an explicit discussion of this relationship, articulating the practical and theoretical advantages of the kernel formulation over explicitly defining a basis.
*   **Experimental Evaluation:** While the applications are interesting, the empirical evaluation lacks rigorous comparison against relevant state-of-the-art baselines from the machine learning community. For example:
    *   The prediction experiments should be compared against established nonlinear time-series models such as Gaussian Process Dynamical Models or Recurrent Neural Networks (RNNs) to demonstrate a competitive advantage.
    *   The change-point detection application is compared to a single method but would be more convincing if benchmarked against a wider array of modern sequential change detection algorithms.
    Without such comparisons, the significance of the results and the practical utility of the method remain difficult to assess.
*   **Clarity on Practical Implementation:** The paper lacks detail on crucial practical aspects of implementing a kernel method. There is little discussion on how the kernel function and its hyperparameters (e.g., kernel width) were chosen for the experiments. Given the sensitivity of kernel methods to these choices, guidance on parameter selection or a sensitivity analysis would greatly improve the paper's reproducibility and utility for practitioners.
