- Paper develops spectral analysis of the Koopman operator in RKHS and an algorithm to estimate it from measurement data.
- Demonstrations include a damped linear 1D system, a chaotic map, and real locomotion data, with comparisons to DMD, PCA, and kernel PCA that yield unclear outcomes.
- The work is an extension of dynamic mode decomposition similar to earlier "kernelizing" methods framed as linear-operator approaches.
- Paper is clearly written and has educational value for readers unfamiliar with fluid-dynamics concepts.
- Experimental evaluation is largely on artificial problems and does not yet convince ML readers of practical usefulness.
- Reviewer questions relevance to ML because many ML dynamical-system models include stochastic components (noise/innovations).
- Authors should identify concrete use cases and attempt to compete with state-of-the-art application domains (e.g., robotics) to show practical benefit.
- The theoretical results are well-executed but unsurprising given existing kernel PCA/ICA and related kernel methods.
- Unclear whether the modal decomposition algorithm requires explicit feature-space representations; if it does, this is a limitation versus kernel PCA and should be clarified.
- Reviewer asks for baseline results using standard DMD on the toy examples and the Henon map to demonstrate differences.
