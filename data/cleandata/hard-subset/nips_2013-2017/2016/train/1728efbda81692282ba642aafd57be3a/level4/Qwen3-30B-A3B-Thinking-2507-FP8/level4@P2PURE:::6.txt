

The paper presents a spectral analysis of the Koopman operator within reproducing kernel Hilbert spaces (RKHS) and introduces an algorithm for this analysis using measurement data from dynamical systems. While the Koopman operator originates from fluid dynamics, the authors suggest its relevance to machine learning. They demonstrate their approach on two simple dynamical systems—damped linear 1D dynamics and a chaotic map—and apply it to real-world locomotion data, comparing performance against standard dynamic mode decomposition (DMD), PCA, and kernel PCA; however, the results for the locomotion data are not clearly interpretable. The work constitutes an interesting extension of dynamic mode decomposition, conceptually aligned with prior "kernelized" techniques for linear operators (e.g., kernel PCA, kernel ICA, kernel independence tests, GP bandits). The experimental results are primarily on artificial problems. The paper is clearly written and holds educational value for machine learning researchers unfamiliar with fluid dynamics concepts. I broadly support efforts to introduce machine learning practitioners to cross-disciplinary techniques. The method addresses function decompositions from \(M \to M\) for deterministic dynamical systems (without stochastic components). While I am not aware of such deterministic setups being prevalent in machine learning (where stochastic elements like noise are typically incorporated), I may be overlooking relevant applications. The authors must strengthen their case for practical utility in machine learning by: (1) identifying concrete use cases, (2) demonstrating competitive performance against state-of-the-art methods, and (3) specifying advantages for a domain where dynamical system inference is critical, such as robotics (e.g., "What specific benefits would Koopman analysis offer robotics practitioners?"). The theoretical derivations are sound, but the results are unsurprising given established kernel methods (e.g., kernel PCA, kernel ICA, kernel independence tests, kernel Bayes rule), as the framework leverages standard RKHS properties to yield eigendecomposition-based algorithms. A critical technical point: it is unclear whether the modal decomposition algorithm requires explicit feature space representations. If so, this represents a limitation compared to kernel PCA, and the authors should explicitly address this. Additionally, including standard DMD performance on the toy examples and Henon map would clarify the methodological differences.
