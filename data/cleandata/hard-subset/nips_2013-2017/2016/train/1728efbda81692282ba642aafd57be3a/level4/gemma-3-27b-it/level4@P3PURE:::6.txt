In this paper, the authors present a spectral analysis of the Koopman operator within reproducing kernel Hilbert spaces and develop an algorithm for performing this analysis from measurement data of a dynamical system. The Koopman operator originates in fluid dynamics, and the authors suggest its relevance to machine learning. The proposed algorithm and analysis are demonstrated on two simple dynamical systems – a damped linear 1D system and a chaotic map – and applied to real-world locomotion data, with comparisons to dynamic mode decomposition (DMD), PCA, and kernel PCA. The results of the latter comparison are somewhat inconclusive. The work represents an interesting extension of DMD, and conceptually aligns with prior efforts in “kernelizing” techniques commonly found in machine learning (e.g., kernel PCA, kernel ICA, kernel independence tests, Gaussian process bandits). The experimental validation primarily utilizes largely artificial problems.

The paper is clearly written and provides educational value for readers unfamiliar with concepts from fluid dynamics. I generally support work that bridges techniques across disciplines, and here, the authors present a decomposition technique for functions mapping from a space M to itself, used to construct a dynamical system without stochastic components. While I am unaware of prevalent use cases for such deterministic dynamical systems in machine learning (typically, applications involve random effects like innovations and noise), my knowledge may be incomplete. Currently, the authors need to more convincingly demonstrate the practical utility of these techniques to a machine learning audience. Specifically, what potential applications exist? And can the method compete, even partially, with state-of-the-art approaches in those areas? 

Robotics is one machine learning discipline where inference and learning in dynamical systems are crucial, and where substantial work has already been done. What advantage would Koopman analysis offer to a roboticist? Alternatively, the authors could focus on any other relevant domain. Furthermore, while the theoretical derivations are well-executed, they are not particularly surprising given familiarity with kernel PCA, kernel ICA, or subsequent work on kernel independence tests and “kernel Bayes rule”. The core idea involves applying standard techniques—eigendecomposition on finite-dimensional matrices—to a linear operator acting on functions within a reproducing kernel Hilbert space.

Regarding implementation details, it is unclear whether the modal decomposition algorithm requires explicit representation in the feature space. If so, this is a limitation compared to kernel PCA, and the authors should explicitly acknowledge this. Additionally, it would strengthen the analysis to include the performance of standard DMD on both the toy example and the Henon map, allowing for a clearer demonstration of the differences between the methods in a simple setting.
