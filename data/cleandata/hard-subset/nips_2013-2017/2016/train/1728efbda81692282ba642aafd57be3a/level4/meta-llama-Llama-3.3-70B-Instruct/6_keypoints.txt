- Presents spectral analysis of the Koopman operator in RKHS and an algorithm to perform this analysis from measurement data.  
- Demonstrates the method on a damped linear 1D system, a chaotic map, and real-world locomotion data; real-data comparisons to DMD, PCA, and kernel PCA are somewhat ambiguous.  
- Offers an interesting extension of dynamic mode decomposition, fitting into prior "kernelizing" of linear-operator methods (kernel PCA, kernel ICA, etc.).  
- Experiments are largely artificial; the paper is well-written and has educational value for readers unfamiliar with fluid-dynamics-derived techniques.  
- Promising for introducing ML practitioners to cross-disciplinary techniques, but practical utility for ML is not yet convincingly shown.  
- Main limitation: focuses on deterministic dynamical systems without stochastic components common in ML settings.  
- Authors should explore concrete ML use cases and show competitiveness with state-of-the-art methods (suggestion: robotics/inference in dynamical systems).  
- Theoretical derivations are sound but not surprising given foundations in kernel PCA/ICA and kernel independence tests.  
- Method reduces to eigendecomposition on finite-dimensional matrices arising from a linear operator in an RKHS.  
- Clarification needed: does the modal decomposition require an explicit feature-space representation? If so, this is a limitation versus kernel PCA and should be highlighted.  
- Suggestion to include baseline results from standard dynamic mode decomposition on the toy example and the Henon map to clarify performance differences.
