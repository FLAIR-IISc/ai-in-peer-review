- Introduces Koopman spectral analysis in RKHS and an algorithm using measurement data from dynamical systems.
- Demonstrated on damped 1D dynamics, a chaotic map, and real locomotion data with comparisons to DMD, PCA, and kernel PCA, but results are ambiguous.
- Extension of DMD resembles prior “kernelized” linear-operator methods, raising novelty concerns relative to existing kernel techniques.
- Paper is clearly written and provides useful educational value for ML readers unfamiliar with fluid-dynamics concepts.
- Method focuses on deterministic dynamical systems; reviewer questions relevance to ML where stochasticity/noise is common and requests concrete ML use cases.
- Theoretical derivations are sound but expected given that linear operators in RKHS naturally yield eigendecomposition-based algorithms.
- Major technical concern: unclear whether modal decomposition requires explicit feature-space representations—if so, this is a limitation versus kernel PCA and must be clarified.
- Recommend adding DMD performance comparisons on the toy examples and the Henon map to better delineate methodological differences.
