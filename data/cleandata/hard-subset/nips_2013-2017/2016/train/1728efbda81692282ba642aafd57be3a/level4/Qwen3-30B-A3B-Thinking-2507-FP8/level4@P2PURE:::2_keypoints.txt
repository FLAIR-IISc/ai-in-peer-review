- Introduces a spectral method using the Koopman operator to linearize function composition in nonlinear dynamical systems.
- Applies Koopman operator decomposition to second-order forward dynamics x(t+2) = f(f(x(t))) using collected trajectories.
- Uses eigen-decomposition and draws parallels to kernel PCA and subspace analysis techniques.
- Demonstrates embedding and recognition applications and reports strong empirical performance.
- Key novelty for kernel-method readers: treating function composition as a linear operator and applying new decompositions.
- Application is relevant because dynamical systems naturally involve function composition.
- Novelty appears limited because Dynamic Mode Decomposition (DMD) is already established and this work mainly kernelizes that framework.
- Existence of alternative methods (e.g., recurrent neural networks, change-detection techniques) and the presented applications do not convincingly show superiority or uniqueness.
