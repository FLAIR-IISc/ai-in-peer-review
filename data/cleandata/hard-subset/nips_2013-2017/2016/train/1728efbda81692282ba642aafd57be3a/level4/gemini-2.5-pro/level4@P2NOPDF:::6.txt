This paper presents a spectral analysis of the Koopman operator within the framework of reproducing kernel Hilbert spaces (RKHS), and introduces a corresponding algorithm for its application to measurement data from dynamical systems. The authors draw upon the Koopman operator, a concept from fluid dynamics, and propose its application to machine learning. The method is demonstrated on a simple damped linear system and a chaotic map, and applied to real-world locomotion data, where its performance is compared against standard Dynamic Mode Decomposition (DMD), PCA, and kernel PCA, yielding results that are difficult to interpret.

The main contribution is an interesting extension of Dynamic Mode Decomposition, conceptually similar to established "kernelized" techniques like kernel PCA, kernel ICA, and kernel-based independence tests. The paper is clearly written and holds educational value for machine learning researchers unfamiliar with concepts from fluid dynamics. The effort to introduce techniques from other disciplines to the machine learning community is commendable.

However, the paper has several weaknesses that should be addressed.

The primary concern is the limited motivation for a machine learning audience. The analysis focuses on deterministic dynamical systems, whereas many applications in machine learning involve stochastic processes that incorporate noise and other random effects. The authors need to do more to convince readers of the practical utility of this technique. What specific problems can this method solve more effectively than existing approaches? In which domains could it offer a competitive advantage? For example, a concrete discussion of its potential benefits in a field like robotics, which relies heavily on learning and inference in dynamical systems, would significantly strengthen the paper's impact. Without clear use cases or compelling performance against state-of-the-art methods in an established ML domain, the contribution remains isolated.

Furthermore, while the theoretical derivations are well-executed, the core idea may appear incremental to readers familiar with the "kernel trick." The general principle of representing a linear operator in an RKHS, which ultimately leads to an eigendecomposition of a finite-dimensional Gram matrix, is a well-established pattern in machine learning (e.g., in kernel PCA, kernel ICA, etc.). The novelty of applying this pattern to the Koopman operator needs to be situated more carefully within this broader context.

The experimental evaluation could also be strengthened. The results on the real-world locomotion data are presented as somewhat unclear, which fails to make a strong case for the method's superiority. To better demonstrate the advantages of the proposed kernel-based approach over its linear counterpart, it would be highly beneficial to include the performance of standard Dynamic Mode Decomposition on the toy examples (the 1D linear system and the Hénon map). This direct comparison in a simple, controlled setting would more clearly illustrate the differences.

Finally, a key technical detail requires clarification. It is not clear whether the proposed modal decomposition algorithm requires an explicit representation of the data in the feature space. If it does, this would be a significant practical limitation compared to methods like kernel PCA that rely solely on the kernel matrix. The authors should explicitly state whether their algorithm depends on the feature map and discuss the implications.

In summary, the paper introduces an interesting theoretical connection between Koopman operator theory and kernel methods. However, to be a valuable contribution to the machine learning literature, it requires a much stronger motivation regarding its practical applications, a more compelling experimental comparison against relevant baselines, and clarification on key implementation details.
