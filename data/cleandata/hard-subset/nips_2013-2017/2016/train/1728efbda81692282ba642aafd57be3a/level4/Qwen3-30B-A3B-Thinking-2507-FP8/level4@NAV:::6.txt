

The paper presents a spectral analysis of the Koopman operator within reproducing kernel Hilbert spaces (RKHS) and introduces an algorithm for this analysis using measurement data from dynamical systems. While the Koopman operator originates in fluid dynamics, the authors suggest its relevance to machine learning. The method is demonstrated on two simple dynamical systems—damped linear 1D dynamics and a chaotic map—and applied to real-world locomotion data, with comparisons to dynamic mode decomposition (DMD), PCA, and kernel PCA yielding somewhat ambiguous results. The work extends dynamic mode decomposition in a manner reminiscent of prior "kernelized" linear operator techniques (e.g., kernel PCA, kernel ICA, kernel independence tests, GP bandits). The paper is clearly written and holds educational value for readers unfamiliar with fluid dynamics concepts. I generally support efforts to introduce machine learning practitioners to cross-disciplinary techniques. However, this approach focuses on deterministic dynamical systems (functions from M to M without stochastic components), and I am unaware of such setups being prevalent in machine learning (where stochastic elements like noise are typically central), though I may be overlooking relevant applications. The authors must strengthen the case for practical utility in machine learning: what specific use cases exist? Can the method compete with state-of-the-art approaches in domains like robotics, where dynamical systems inference is critical? While the theoretical derivations are sound, the results are not unexpected given established kernel methods (e.g., kernel PCA, kernel ICA, kernel independence tests, kernel Bayes rule), as linear operators in RKHS naturally yield eigendecomposition-based algorithms. A key technical concern: it remains unclear whether the modal decomposition algorithm requires explicit feature space representations, which would constitute a limitation relative to kernel PCA and should be explicitly addressed. Additionally, the authors should include DMD performance comparisons on the toy examples and Henon map to clarify methodological distinctions.
