- Introduces a spectral approach using the Koopman operator to analyze nonlinear dynamical systems by treating function composition as a linear operator.
- Specifically applies Koopman operator decomposition to forward dynamics x(t+2) = f(f(x(t))).
- Uses collected system trajectories and performs eigen-decomposition inspired by kernel PCA and subspace analysis.
- Applies the method to dynamical system embedding and recognition, showing promising empirical results.
- Novelty for kernel-method readers is framing function composition as a linear operator enabling useful decompositions.
- Demonstration on dynamical systems is an appropriate application of the idea.
- Concern that dynamic mode decomposition (DMD) already covers related ground, limiting novelty to kernelizing existing methods.
- Notes that alternative methods (e.g., recurrent neural networks) exist for recognition and change detection in dynamical systems.
- Concludes that the applications shown do not strongly justify the method’s uniqueness or clear advantages.
