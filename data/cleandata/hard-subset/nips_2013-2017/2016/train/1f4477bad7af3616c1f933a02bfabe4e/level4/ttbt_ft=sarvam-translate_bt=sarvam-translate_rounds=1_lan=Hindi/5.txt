This paper discusses and defines a specific type of feedforward neural network for which a unique global optimizer can be obtained. As a product, it defines the algorithm that finds the global optimizer and has a linear convergence rate in the algorithm. The mathematical derivation is instructive, although some parts are not very clear to me. 1. Theorem 3 involves calculating the first-order derivatives, as shown in those inequalities, but theorem 4 reaches the second-order derivative level. What is the reason for this? 2. The design of this type of network considers the last layer to be separate from any hidden layer. Since u is also a matrix representing the parameters connecting to the hidden layer, is the gradient in G^\Phi (3) with respect to u just a gradient in terms of the vectorization of u? (Similar questions arise in many other places, such as limiting u within a ball). One concern (am I missing something?) is why the model needs to maximize the 1-norm of the connection parameters in (2). In most NN models, the 1-norm needs to be minimized for the weight decay. It seems that this is because arbitrarily small epsilon in (2) is only necessary to make the gradient strictly positive, so it is in V_++. This only seems necessary for their proof but is not well justified in practical NNs. Is it justified that there is no free parameter here? All the data needs to be non-negative for this, and then the model uses all positive weights (not even zero), so it keeps adding. How practical is this design? In experiments, the proposed NN is compared to the SVM (because the SVM also has a global optimizer?). However, in practice, do these types of NNs bring additional value in any aspect compared to other NNs (more commonly used NNs, for example, with ReLU)?
