- Proposes SGRRLD: use Richardson-Romberg extrapolation on two parallel SG-MCMC chains (step sizes γ and γ/2) to reduce bias.
- Correlates Brownian increments between chains (Z^γ_k = (Z^(γ/2)_{2k-1} + Z^(γ/2)_{2k})/√2) to control variance.
- Shows bias improves from O(γ) (SGLD) to O(γ²) under the RR scheme.
- Proves MSE rate improves to O(K^-4/5) versus SGLD’s O(K^-2/3), matching higher-order integrators using only first-order updates.
- Bridges numerical analysis (RR extrapolation) and ML, applying an established technique in a novel SG-MCMC context.
- Method is simple to implement with minor changes to existing SG-MCMC code and preserves parallelizability.
- Theoretical analysis is thorough: provides CLT and non-asymptotic bias and MSE bounds.
- Empirical results (synthetic Gaussians and large-scale matrix factorization) show consistent accuracy improvements over SGLD and SGHMC.
- Strengths: clear presentation, solid comparisons to prior work, and practical applicability.
- Unclear applicability: RR requires expansion π_γ(f)=π(f)+Cγ+O(γ²); conditions for which SG-MCMC methods satisfy this need clarification.
- Extension to SGHMC and other methods beyond the shown case is not fully established.
- Computational efficiency concern: running one chain twice as long may cause idle time in parallel implementations.
- Suggests exploring alternatives (e.g., three-chain schemes) to better utilize parallel resources without increasing variance.
- Experiments could be strengthened by adding a standard SGD baseline in the matrix factorization task to contextualize Bayesian benefits.
- Lacks detailed comparison of computational cost versus true higher-order integrators.
- Novelty and significance are high; reviewer recommends acceptance.
