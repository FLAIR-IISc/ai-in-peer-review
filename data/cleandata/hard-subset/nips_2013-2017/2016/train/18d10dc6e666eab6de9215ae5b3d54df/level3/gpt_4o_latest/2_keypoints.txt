- Introduces q-KG, a novel batch Bayesian optimization algorithm that computes Bayes-optimal batches for parallel evaluations.
- Derives a decision-theoretic acquisition function with rigorous Bayes-optimality for batch settings.
- Uses infinitesimal perturbation analysis (IPA) and discretization to make q-KG computationally feasible and enable efficient gradient computation.
- Shows superior performance to parallel EI and UCB-based methods, especially in noisy settings.
- Provides clear, well-organized writing and open-source code for reproducibility.
- Experimental evaluation is narrow: only two datasets (MNIST, CIFAR10), one task (classification), and two models (logistic regression, CNN).
- Uses test set error as the optimization metric, which is suboptimal for classification compared to AUC or log loss.
- Assumes independent normally distributed errors, which may be invalid due to structural biases in hyperparameter spaces.
- Lacks benchmarking against a broader suite of tasks (e.g., HPOlib), limiting empirical generalizability.
- Recommendation: accept with revisions to expand empirical validation and adopt more appropriate evaluation metrics.
