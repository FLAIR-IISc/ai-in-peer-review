- Proposes a novel, elegant parallel Bayesian optimization algorithm (parallel knowledge gradient) that computes Bayes-optimal batches.
- Writing is clear, well-organized, and easy to follow, making the paper pleasant to read.
- Strong theoretical foundation with solid analysis supporting the algorithm’s claims.
- Uses infinitesimal perturbation analysis to efficiently compute gradients of the parallel knowledge gradient acquisition function.
- Empirical evaluation is limited: only two datasets and a single machine learning task, which is not sufficiently convincing.
- Uses test set error as the optimization criterion, which is a suboptimal score function for hyperparameter tuning.
- Assumes independent normally distributed errors, which may be unrealistic for hyperparameter optimization and could limit applicability.
- Reproducibility is good for expert readers; the paper provides enough information to reproduce results.
- Originality is reasonable: a novel combination of familiar techniques with adequate related-work discussion.
- Significance is high: addresses a difficult problem better than prior work and is likely to be used or built upon by others.
- Recommendation: accept, but the authors should address experimental limitations and the independent-normal-error assumption.
- Suggested improvement: perform a more comprehensive evaluation using varied benchmarks (e.g., HPOlib).
