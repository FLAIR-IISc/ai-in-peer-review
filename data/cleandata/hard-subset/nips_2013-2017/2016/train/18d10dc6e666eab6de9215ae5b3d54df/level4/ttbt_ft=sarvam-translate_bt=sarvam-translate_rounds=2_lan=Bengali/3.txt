This paper proposes a new version of a stochastic gradient algorithm combined with Bayesian modeling for optimizing multivariate objective functions over a finite domain. The new part of the algorithm is that, in each iteration, it identifies a set of points based on which the objective will be calculated, and this calculation can be done in parallel. The problem is identifying sampling points in the best way. To this end, the authors adopt a knowledge gradient framework, which estimates the expected improvement of the objective value based on the use of an additional sampling point and makes decisions about which sampling point to use based on this. This method is generalized here so that decisions can be made based on multiple sampling points in a batch.
