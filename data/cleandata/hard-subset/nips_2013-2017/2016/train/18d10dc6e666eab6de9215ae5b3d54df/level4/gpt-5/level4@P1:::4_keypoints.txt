- Proposes q-KG, a Bayes-optimal acquisition function for selecting a single remaining batch in parallel Bayesian optimization.  
- Develops an efficient Monte Carlo + IPA stochastic gradient estimator to optimize q-KG jointly over batch points.  
- Handles continuous domains by discretizing the feasible set for q-KG computation.  
- Empirically outperforms or matches parallel EI/UCB baselines on synthetic and hyperparameter-tuning tasks, especially under noise.  
- Main novelty: principled KG-based batch objective with a practical gradient estimator enabling joint batch optimization.  
- Strength: clear decision-theoretic formulation that preserves one-step Bayes-optimality in the batch setting.  
- Strength: practical stochastic-gradient implementation (IPA) makes joint batch optimization computationally feasible.  
- Strength: strong empirical performance and public code release for reproducibility.  
- Weakness: lacks theoretical guarantees on error introduced by discretizing the domain for q-KG.  
- Weakness: no quantitative complexity analysis as functions of batch size q, discretization size M, and input dimension d.  
- Weakness: scalability of the IPA-based optimizer in high dimensions is not characterized.  
- Weakness: reliance on Gaussian process priors may limit applicability to very high-dimensional hyperparameter spaces.  
- Clarity issues: ambiguous notation for A between equations and need distinct symbols for continuous vs. discretized sets.  
- Clarity issues: brief treatment of baseline parallel EI variants—request self-contained descriptions and implementation details.  
- Requests: report Monte Carlo sample counts, gradient steps, and runtime comparisons for representative q and d.  
- Requests: discuss sensitivity to M, kernel hyperparameter and noise-variance estimation, and include ablation experiments if possible.  
- Minor fixes: several line-specific clarifications/rephrasings and minor typos noted.  
- Overall: solid, timely contribution with persuasive experiments; would be strengthened by theoretical/complexity analyses and clearer exposition.
