- Introduces q-KG: a Bayes-optimal batch acquisition function extending knowledge gradient to parallel Bayesian optimization.  
- Jointly optimizes the entire batch of q points rather than using iterative greedy selection.  
- Derives a decision-theoretic acquisition function that targets minimizing the expected posterior minimum when one batch remains.  
- Presents an efficient computation method using infinitesimal perturbation analysis to estimate the acquisition and its gradient.  
- Builds on prior KG and parallel BO literature but distinguishes itself by non-greedy, collective batch selection.  
- Theoretical derivation is clear, well-motivated, and provides a strong foundation for the method.  
- Empirically outperforms benchmarks (parallel EI, GP-UCB variants) on synthetic tasks and ML hyperparameter tuning, especially under noise.  
- Demonstrates convincing practical gains (e.g., CNN tuning improving test error from 14% to 11%).  
- Paper is well-written and reproducible with clear algorithm description.  
- Experimental evaluation is limited: only MNIST and CIFAR10 and two ML tasks, lacking broader HPO benchmarks.  
- Uses test set error as the optimization objective, which is a suboptimal criterion for hyperparameter selection.  
- Assumes independent normal errors in evaluations, which may be unrealistic for hyperparameter optimization and needs discussion/adaptation.  
- Recommendation: accept with minor revisions to expand benchmarks (e.g., HPOlib), justify/replace test-error objective, and discuss realistic error models.
