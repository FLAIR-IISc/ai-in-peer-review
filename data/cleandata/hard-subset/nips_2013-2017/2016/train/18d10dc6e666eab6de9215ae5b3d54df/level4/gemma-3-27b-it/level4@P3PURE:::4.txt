This paper considers the parallel knowledge gradient method for batch Bayesian Optimization, focusing on scenarios where function evaluation is computationally expensive. The authors propose a new acquisition function, q-KG, designed to output multiple points for parallel evaluation, and present an efficient approach for evaluating this function. Empirical results are provided to validate their theoretical findings. The problem addressed is important; distributing the computational workload is crucial for improving the performance of data-driven applications. Instead of sequential point evaluation, the algorithm considers a knowledge gradient over *q* points, proposing *q* evaluations per iteration, which significantly improves efficiency. This is an interesting and worthwhile direction for research. 

However, it is unclear whether the assumption of a Gaussian process is appropriate, particularly when applied to neural network architectures with high-dimensional input spaces. Providing theoretical results regarding error bounds for the method, and a quantitative analysis of its complexity with respect to input dimension, would strengthen the paper. 

Nevertheless, this is a solid contribution to the field of Bayesian Optimization. The method is sound, the experiments are well-designed, and the paper is generally well-written. Minor revisions are suggested: on line 36, rephrase “the set of points to evaluate next that is” for conciseness; on line 95, define the variable A; on line 104, clarify the input to μ<sup>n</sup>; on line 128, briefly discuss the parallel Expected Improvement (EI) algorithm for context; and on line 140, consider using distinct notation for A to avoid potential confusion.
