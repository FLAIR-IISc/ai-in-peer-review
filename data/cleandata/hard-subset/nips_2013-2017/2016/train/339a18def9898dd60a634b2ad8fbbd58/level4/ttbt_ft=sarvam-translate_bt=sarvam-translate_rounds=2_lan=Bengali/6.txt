This work provides a theoretical limit on the budget-feasibility tradeoff for label density using rate-distortion theory and the channel coding theorem. The theoretical limit is well-defined by this paper. If this is the first attempt to apply rate-distortion theory and the channel coding theorem to the density problem, it could be a reasonably significant contribution to the field. However, the novelty and practical value of this paper are still questionable, as the emergence of the theoretical limit is relatively straightforward and there is a risk of the limit being too loose. The main problem is that unlike the channel coding problem, where we have a lot of freedom in designing a codebook or codewords (we can use repeat codes, Hamming codes, convolutional codes, RS codes, Low Density Parity Check (LDPC) codes), it is not clear whether we have much freedom in designing the order of the channel input u^n. In equation (3), almost all mappings from B(X)^n to U^(\sum mi) (except for the degenerates) should be valid or implementable codebooks, otherwise the channel capacity could be a loose limit. Clearly, for density, something like repeat codes is possible where the natural decoding method is the majority vote. However, beyond that, it is unclear how we can design u^n to improve error correction. In particular, there is a significant difference between the channel coding problem and the density problem in that we cannot give the same question to the same person multiple times, as this would violate the IID assumption in equations (1) or (2), because the same person is likely to produce the same answer to the same question, whereas in a communication channel we can assume that the channel is IID and the same question can be used multiple times for the same channel. This is just one example, and in general, the density problem is likely to have a lot of fundamental limitations in equation (3), unlike channel coding, where there are slight restrictions on an encoding mapping {0,1}^{nR} -> {0,1}^n. Therefore, equation (3) should have significant limitations on the set of possible encoding schemes, and due to such limitations, the proposed theoretical limit is likely to be too loose. Finally, it would be good to compare the performance of the theoretical limit obtained with state-of-the-art methods.
