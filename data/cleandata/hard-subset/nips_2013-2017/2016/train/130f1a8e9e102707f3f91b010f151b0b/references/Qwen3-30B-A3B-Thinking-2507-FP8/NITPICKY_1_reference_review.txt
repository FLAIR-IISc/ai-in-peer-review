

# Review of "Learning brain regions via large-scale online structured sparse dictionary learning"

This paper proposes Smooth-SODL, an extension of online dictionary learning with Laplacian regularization to learn spatially structured brain regions from neuroimaging data. The authors aim to define brain regions based on inter-subject variability rather than anatomical segmentation, which is a reasonable direction for neuroimaging analysis.

The paper has several technical inconsistencies and minor flaws that undermine its technical rigor:

1. In equation (2), the objective function contains a limit as n→∞, but the problem is formulated for finite n. This is inconsistent and should be clarified.

2. The constraint set definition in equation (3) is inconsistent with the text. The paper defines Sp(τ) as "Bp,`1(τ) ∩ Rp+" but then uses it as a constraint set in the algorithm, which is confusing since the simplex Sp(τ) is typically defined as {v | sum(v_i) = τ, v_i ≥ 0}, not as the intersection of an l1-ball and the positive orthant.

3. In equation (4), the Laplacian regularization is defined as ΩLap(v) = (1/p) sum_k [(∇x v)_k^2 + ...], but the standard Laplacian penalty should be v^TΔv without the 1/p factor. This normalization is inconsistent with the definition of the Laplacian operator.

4. Algorithm 1, step 5 has a clear typo: "At ← At−1 + UtUTt" should be "At ← At−1 + UtU^T_t" (missing transpose symbol).

5. Equation (7) contains unclear notation: "At[j, j]−1(Bjt −VAjt)" appears to have multiple errors. The correct form should be something like "At[j,j]^{-1}(Bt^j - V A_{t,j})" but the current notation is ambiguous.

6. The paper repeatedly misspells "interpretable" as "intepretable" (e.g., "more intepretable" in abstract, "intepretable" in results section).

7. The paper claims "the dictionary atoms are piece-wise smooth" but never defines what "piece-wise smooth" means in this context or provides mathematical justification for how the Laplacian penalty achieves this property.

8. The paper states "we use mini-batches of size η = 20" but provides no justification for this specific batch size choice.

9. In the experimental section, the paper claims "CanICA [22] for this experiment took 530s (≈ 8.8 minutes) to run" but provides no hardware specifications or implementation details, making this comparison difficult to verify.

10. The paper claims "the Laplacian penalty is meant to impose blobs" but doesn't provide any theoretical justification for why this specific penalty would create blob-like structures rather than other spatial patterns.

While the paper addresses a meaningful problem in neuroimaging, the technical inconsistencies and formatting errors undermine the paper's credibility. The authors make several claims about algorithmic properties (e.g., convergence guarantees, numerical complexity) that are not properly supported with references or proofs. The experimental results are presented without sufficient statistical validation (e.g., the claim about "reproducible major structures" lacks statistical evidence).

The paper would benefit from:
- Correcting the mathematical notation throughout
- Providing proper definitions for all technical terms
- Justifying all algorithmic choices (e.g., batch size, Laplacian normalization)
- Fixing the repeated misspellings
- Providing more details about experimental setup

These minor technical issues accumulate to create a paper that is not technically rigorous, which is especially concerning for a NIPS submission where technical precision is paramount. While the problem addressed is relevant to neuroimaging, the paper's execution fails to meet the high technical standards expected at NIPS.

Pro: Clear motivation for the problem, reasonable experimental setup, potential utility for brain imaging community.

Con: Technical inconsistencies, unclear notation, unsupported claims, repeated misspellings, lack of proper justification for methodological choices.

Given the cumulative effect of these minor but significant technical flaws, I recommend rejection. The paper has a promising idea but requires substantial technical refinement before it can be considered for publication.
