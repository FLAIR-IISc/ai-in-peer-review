This paper proposes a convex framework for partitioning a dataset into *k* subsets, where each subset has a semantic class distribution that is as similar as possible and distinct from each other. The difference between the subsets is maximized by a selected kernel-induced RKHS, which is called maximum distinctiveness. Each of these subsets is assigned a secret domain. The average learning ability (how well a classifier can learn) within each secret domain is maximized to determine *k* (the number of secret domains).

Although each cluster may not be influenced by a single class, if the class distribution within each domain is as similar as possible, it can be a limiting assumption for secret domain detection in some tasks. For example, if we consider both horses and cows as secret domains for an animal classification task, the class distribution within each class may not be the same. For instance, we see horses and cows both standing in a left-right posture, but horses are often not depicted in a sitting posture, while cows are often seen in that posture.

Although secret domain detection is not new, the idea of controlling the class label distribution to better identify secret domains within a (loosely) convex framework is new. Identifying the number of secret domains by testing the quality of classification within each secret domain is also an important practice that ensures there are enough samples in the secret domain to learn generalization and differentiation better between each class.

In several places, the concepts of datasets, domains, and secret domains are not clear and can be easily confused. These concepts should be clearly defined, possibly with some helpful examples. Clarity is needed, especially in section 4.2 of the paper. As far as I understand, the terms datasets and domains are used in the same sense, such as Si being called both a dataset and a source domain. Nevertheless, the concept definitions should be clarified in the experimental setting and section 4.2. Furthermore, maxk r(U_k,B) is not defined but is used in equation 7.

Experimental validity seems sufficient. The results show a reasonable improvement over the baselines. However, it is not clear why the current selection of source and target datasets was preferred for reporting. For example, adaptation without excluding a dataset could be a more reasonable evaluation. Qualitative results are helpful to see what the algorithm ostensibly achieves.

In line 074, the problem (secret domain learning) is stated as an unsupervised learning problem, which can be misleading, as the methods use semantic class labels. Secret domain detection is an important technical contribution by encouraging similar class distributions within each secret domain formulated within a (loosely) convex framework. However, some concepts and experimental validity of the paper need to be clarified.
