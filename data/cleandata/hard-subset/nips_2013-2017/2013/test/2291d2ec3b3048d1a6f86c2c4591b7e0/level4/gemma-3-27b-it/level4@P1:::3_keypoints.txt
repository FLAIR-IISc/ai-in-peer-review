- Proposes a convex framework to partition a dataset into K “latent domains” by maximizing pairwise mean differences in an RKHS.
- Identifies K by maximizing average “learnability” (DWCV) — how well classifiers can be learned within each domain.
- Novel contribution: enforcing controlled class label distributions during latent domain discovery within a relaxed convex formulation.
- Novel contribution: using per-domain classification quality to select the number of latent domains.
- Limitation: assuming similar class distributions within each domain can be unrealistic (e.g., pose-based domains with imbalanced class presence).
- Terminology clarity problem: “dataset,” “domain,” and “latent domain” are not consistently or precisely defined and need illustrative examples.
- Specific ambiguity: Section 4.2 treats Si inconsistently (dataset vs domain) and Eq.6 uses max_k r(U_k,B) without prior definition.
- Experimental results: quantitative improvements over baselines and useful qualitative visualizations support the method’s effectiveness.
- Experimental design concern: rationale for chosen source/target datasets is weak; a leave-one-dataset-out scheme could be more comprehensive.
- Claim issue: calling the problem “unsupervised” is misleading because semantic class labels are used in the method.
- Arguments for acceptance: principled convex approach, useful class-distribution control, practical DWCV method, and improved empirical performance.
- Arguments against acceptance: restrictive class-distribution assumption, unclear terminology and experimental setup, weak dataset selection justification, and misleading unsupervised claim.
