- Proposes a convex framework to partition data into K latent domains with similar semantic class distributions while maximizing pairwise distinctiveness in RKHS.  
- Determines K by maximizing average learnability of classifiers within each latent domain.  
- Forcing similar class distributions per domain can be limiting (e.g., pose-based domains where class-specific pose distributions differ).  
- Novelty lies in controlling class label distributions within a relaxed convex formulation and selecting K via domain-wise classification quality.  
- Definitions of dataset, domain, and latent domain are unclear and conflated; paper needs explicit definitions and examples.  
- Experiments section (4.2) is confusing (terms like Si used inconsistently) and contains an undefined term "maxk r(U_k,B)" in equation 7.  
- Experimental validation shows improvements and useful qualitative results, but dataset selection rationale is unclear; suggest leave-one-dataset-out evaluation.  
- Calling the problem unsupervised is misleading since semantic class labels are used; overall contribution is notable but requires conceptual and experimental clarifications.
