

This paper proposes a convex framework for partitioning datasets into K subsets (termed "latent domains") such that each subset exhibits similar semantic class distributions while maximizing pairwise distinctiveness between subsets. Distinctiveness is achieved by maximizing pairwise mean differences of subsets in the reproducing kernel Hilbert space (RKHS) induced by a selected kernel. The number of latent domains K is determined by maximizing the average learnability—measuring how well a classifier can be trained—within each latent domain.  

While constraining class distributions to be similar within each latent domain prevents clusters from being dominated by single classes, this assumption may limit applicability for certain tasks. For instance, in an animal classification task where poses represent latent domains (e.g., horses vs. cows), the class distributions within pose-based domains may not be similar. Specifically, both horses and cows appear in left-right standing poses, but horses are rarely depicted in sitting poses whereas cows are commonly shown sitting.  

Although latent domain discovery itself is not novel, the paper’s approach of controlling class label distributions within a (relaxed) convex framework to better identify latent domains constitutes a novel contribution. Similarly, determining K via classification quality within each latent domain—which ensures sufficient samples per class for robust generalization—is a notable methodological practice.  

Critical ambiguities persist regarding the concepts of "dataset," "domain," and "latent domain," which are frequently conflated. These terms must be rigorously defined with illustrative examples. Section 4.2 particularly requires clarification: S_i is referred to as both "datasets" and "source domains," creating confusion in the experimental setup. Additionally, the term max_k r(U_k, B) appears in Equation 7 without definition.  

The experimental validation demonstrates reasonable improvements over baselines, and qualitative results effectively illustrate the algorithm’s visual outcomes. However, the rationale for selecting specific source and target datasets for evaluation is unclear. A "leave-one-dataset-out" adaptation protocol would provide a more robust assessment.  

Line 74 incorrectly characterizes the latent domain discovery problem as unsupervised learning, which is misleading since the method leverages semantic class labels. The core technical contribution—formulating latent domain discovery via convex optimization with controlled class distributions—is significant. However, conceptual clarity and experimental justification require substantial revision before acceptance.
