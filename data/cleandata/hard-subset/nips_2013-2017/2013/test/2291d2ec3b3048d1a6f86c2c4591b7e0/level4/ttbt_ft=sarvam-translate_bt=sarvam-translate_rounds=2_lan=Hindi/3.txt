This paper proposes a convex framework for partitioning datasets into K subsets, ensuring that each subset has a similar semantic class distribution and is as distinct from each other as possible. This differentiation, called maximum specificity, is achieved by maximizing the pairwise mean difference between the subsets in the RKHS induced by a selected kernel. Each of these subsets is designated as a hidden domain. The identification of K (the number of hidden domains) is obtained by maximizing the average learning capacity (how well a classifier can be learned) within each hidden domain.

While forcing the distribution of classes within each domain to be as similar as possible can prevent groups from being dominated by each class, it can also be a limiting assumption for finding hidden domains for certain tasks. For example, if we consider a pose as a hidden domain for an animal classification task (such as horse and cow), the distribution of hidden domains within each class may not be equal. For instance, we see both the horse and the cow in a left-right standing pose, although the horse is not often shown in a sitting pose, while the cow is.

Although the search for hidden domains is not new, the idea of controlling the class label distribution for better identification of hidden domains within the (convex) framework is new. Identifying the number of hidden domains by examining the quality of classification within each hidden domain is also a notable exercise, ensuring that there are sufficient samples in the hidden domain for each class so that better generalization and discrimination can be learned between classes.

In many places, the concepts of datasets, domains, and hidden domains are not clear and can be easily confused. These concepts should be clearly defined, preferably with some supporting examples. In particular, the experiment section 4.2 needs clarification. As far as I understand, the terms datasets and domains are used interchangeably because Si is designated as both datasets and source domains. However, the experimental setting and concept definition should be clarified in section 4.2. Furthermore, maxk r(U_k,B) is not defined, but it is used in equation 7.

The experimental validation seems sufficient. The results have reasonable improvement from the baseline. However, it is not clear why the current selection of source and target datasets is preferred for reporting. For example, omitting a dataset for optimization may be a more appropriate evaluation. Qualitative results are helpful for seeing the algorithm, to see what it visually achieves.

At line 074, the problem (hidden domain learning) is being described as an unsupervised learning problem, which can be misleading because the methods use semantic class labels. Finding hidden domains by encouraging a similar class distribution in each hidden domain prepared in the (convex) framework is a notable technical contribution. However, some concepts and experimental validation need to be clarified in the paper.
