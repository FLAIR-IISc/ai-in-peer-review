This paper proposes a convex framework for partitioning a dataset into *k* subsets, where each subset has a semantic class distribution that is similar and as distinct from each other as possible. The difference is achieved by maximizing the pairwise mean difference between the subsets in an RKHS induced by a chosen kernel, which is called maximum distinctness. Each of these subsets is named a latent domain. The average learning capacity (how well a classifier can learn) within each latent domain is maximized to determine *k* (the number of latent domains).

Although every cluster may not be dominated by a single class if the class distribution within each domain is tried to be as similar as possible, this can be a limiting assumption for latent domain discovery in some tasks. For example, if we consider horses and cows as latent domains for an animal classification task, the class distribution within each class may not be the same. For instance, we see both horses and cows standing in a left-right posture, but horses are often not depicted in a sitting posture, while cows are often seen in that posture.

While latent domain discovery is not new, the idea of controlling the class label distribution for better identification of latent domains within a (loosely) convex framework is new. Identifying the number of latent domains by testing the quality of classification within each latent domain is also a significant exercise that ensures there are enough samples in the latent domain to learn generalization and difference better between each class.

In several places, the concepts of datasets, domains, and latent domains are not clear and can be easily confused. These concepts should be clearly defined, possibly with some helpful examples. Clarity is needed, especially in the section 4.2 of the test. As far as I understand, the terms datasets and domains are used in the same sense, as Si is called both datasets and source domains. Nevertheless, the experimental setting and the definition of the concept in section 4.2 should be clarified. Furthermore, maxk r(U_k,B) is not defined but is used in equation 7.

The experimental validity seems sufficient. The results show a reasonable improvement over the baselines. However, it is not clear why the current selection of source and target datasets was preferred for reporting. For example, the adaptation might be a more reasonable evaluation by excluding a dataset. Qualitative results are helpful to see what the algorithm ostensibly achieves.

In line 074, the problem (latent domain learning) is stated as an unsupervised learning problem, which can be misleading, as the methods use semantic class labels. Latent domain discovery is a significant technical contribution by encouraging similar class distributions in each latent domain formulated within a (loosely) convex framework. However, some concepts and experimental validity of the paper need to be clarified.
