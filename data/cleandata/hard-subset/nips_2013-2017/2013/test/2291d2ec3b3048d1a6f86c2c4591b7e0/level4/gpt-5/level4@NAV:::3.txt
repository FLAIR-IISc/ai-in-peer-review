This paper presents a convex formulation that partitions the dataset(s) into K subsets such that each subset has similar semantic class distributions while being as distinct from the others as possible. Distinctiveness, termed maximum distinctiveness, is realized by maximizing the pairwise mean differences of the subsets in the RKHS induced by a chosen kernel. These subsets are referred to as latent domains. The value of K (the number of latent domains) is determined by maximizing average learnability (how well a classifier can be trained) within each latent domain.

While enforcing similar class distributions within each domain helps prevent clusters from being dominated by a single class, it can be a restrictive assumption for certain latent domain discovery tasks. For example, if pose constitutes the latent domain in an animal classification setting (e.g., horses vs. cows), the latent-domain distributions within each class may not align. Both horses and cows appear in left–right standing poses, but cows are frequently pictured sitting whereas horses rarely are.

Although latent domain discovery is not new, the idea of constraining class-label distributions to improve the identification of latent domains within a (relaxed) convex framework is novel. Selecting the number of latent domains by evaluating within-domain classification quality is also a noteworthy practice that helps ensure each latent domain has sufficient samples per class to generalize and learn class discrimination.

In several places the notions of dataset, domain, and latent domain are unclear and easily conflated. These concepts should be explicitly defined, preferably with illustrative examples. In particular, the experimental Section 4.2 needs clarification. As far as I can tell, dataset and domain are used interchangeably since S_i is referred to as both datasets and source domains. The experimental setup and the conceptual definitions in Section 4.2 should be clarified. Additionally, maxk r(U_k, B) is used in eq.7 but not defined.

The experimental validation appears adequate, with results showing reasonable gains over baselines. However, the rationale for the chosen source and target datasets is not clear. For instance, a leave-one-dataset-out adaptation protocol might be a more appropriate evaluation. The qualitative results are useful for visualizing what the algorithm achieves.

On line 074, describing the problem (learning latent domains) as unsupervised may be misleading given that semantic class labels are used. The discovery of latent domains by encouraging similar class distributions within each latent domain in a (relaxed) convex framework is a notable technical contribution. However, some conceptual definitions in the paper and aspects of the experimental validation require clarification.
