

This paper introduces a convex framework for partitioning a dataset into K subsets (termed "latent domains") with two key properties: (1) each subset exhibits similar class distributions across semantic categories, and (2) subsets are maximally distinct from one another. Distinctiveness is quantified by maximizing pairwise mean differences between subsets in a reproducing kernel Hilbert space (RKHS) induced by a chosen kernel. The number of latent domains K is determined by optimizing average learnability—measuring how effectively a classifier can be trained within each subset—ensuring sufficient samples per class for robust generalization.

While constraining class distributions within subsets to be similar prevents clusters from being dominated by single classes, this assumption may limit applicability for certain tasks. For example, in animal classification where pose constitutes the latent domain (e.g., horses vs. cows), class distributions *within* each pose domain may not be similar. Horses rarely appear in sitting poses, whereas cows frequently do, meaning the same pose (e.g., "sitting") would not yield comparable class distributions across the latent domains.

Though latent domain discovery is not novel, the paper’s use of a (relaxed) convex framework to explicitly control class label distributions for improved discovery is a significant contribution. Similarly, determining K via classification quality within each latent domain—ensuring adequate samples per class—is a noteworthy approach to avoid overfitting.

Conceptual clarity requires substantial improvement. The terms "dataset," "domain," and "latent domain" are inconsistently defined and easily conflated. These should be rigorously distinguished with concrete examples. Section 4.2 particularly needs clarification: the text refers to S_i as both "datasets" and "source domains," creating confusion about the experimental setup. Additionally, the notation max_k r(U_k, B) in Equation 7 lacks definition.

The experimental validation is generally adequate, showing reasonable improvements over baselines. However, the rationale for selecting specific source and target datasets for reporting is unclear. A leave-one-out adaptation strategy would provide a more rigorous evaluation. Qualitative results effectively illustrate the algorithm’s visual outcomes.

Line 74 states that "learning latent domains" is framed as an unsupervised problem, which is misleading since the method leverages semantic class labels. The core technical contribution—discovering latent domains by encouraging similar class distributions within a convex framework—remains valuable, but the paper’s conceptual foundations and experimental presentation require significant refinement to ensure clarity and reproducibility.
