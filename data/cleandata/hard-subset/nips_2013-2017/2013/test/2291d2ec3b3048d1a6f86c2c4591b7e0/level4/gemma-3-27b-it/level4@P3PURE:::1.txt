The goal of this work is to automatically discover latent domains within a training set, subsequently leveraging this discovery in a domain adaptation framework to improve classification performance on a test set. The paper defines a function measuring the difference between two feature vectors using a specified kernel, and aims to partition the data points into domains such that this function is maximized across all pairs of domains. This problem is formulated as an integer programming problem subject to two constraints: each data point is assigned to exactly one domain, and the distribution over class labels within each domain matches the input distribution over the entire dataset. This integer program is relaxed to a continuous optimization problem involving a quadratic cost function with linear constraints, and the optimal number of domains is determined via cross-validation.

The approach is evaluated on two datasets: static images from [2] and the IXMAS multi-view action dataset [15]. A key result is the demonstration of improved performance compared to the domain adaptation approach of [19].

Positives: The paper is well-written and, to the best of my knowledge, presents a novel approach to domain adaptation (although I am not a specialist in this field). The performance gains over [19] are noteworthy.

Negatives: Currently, I lean slightly towards rejection. However, I have two main concerns that, if addressed in the rebuttal, could lead me to revise my score:

(i) The motivation for this work is unclear. On line 77, the paper states that “simply clustering images by their appearance is prone to reshaping datasets into per-category domains.” First, what evidence supports this claim? Second, how does the model formulation in Section 2 prevent this issue – i.e., how is the formation of per-category domains avoided?

(ii) Relatedly, on line 154, the rationale for the second constraint ("label prior constraint") is not explained. I am curious about the performance impact of removing this constraint. Furthermore, a baseline comparison using k-means clustering or unsupervised object discovery techniques (e.g., Sivic et al. ICCV '05) applied to the appearance vectors would be valuable. Finally, what is the performance when the dataset is randomly partitioned into equally sized sets?
