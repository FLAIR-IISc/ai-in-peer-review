- Proposes a convex framework to partition data into K "latent domains" by maximizing pairwise mean differences in RKHS for maximum distinctiveness.  
- Selects K by maximizing average learnability within each latent domain using classifier performance as a criterion.  
- Assumes class distributions within each latent domain should be similar, which may be limiting or unrealistic for some tasks (e.g., animal poses differing by class).  
- Latent domain discovery itself is not novel, but controlling class label distributions in a relaxed convex formulation is a novel contribution.  
- Determining K via per-domain classification quality is a useful idea that helps ensure sufficient per-class samples and better generalization.  
- Presentation issues: unclear definitions and distinctions between "dataset", "domain", and "latent domain", and lack of supporting examples.  
- Section 4.2 is confusing (Si used inconsistently as datasets vs source domains) and equation 7 uses undefined notation maxk r(U_k,B).  
- Experimental results show reasonable improvements over baselines, but choice of source/target datasets is not well justified.  
- Suggests alternative evaluation (e.g., leave-one-dataset-out adaptation) as potentially more appropriate.  
- Notes a misleading statement calling the problem "unsupervised" when semantic class labels are used.  
- Overall: notable technical contribution in formulation and K-selection, but several conceptual and experimental clarifications are needed.
