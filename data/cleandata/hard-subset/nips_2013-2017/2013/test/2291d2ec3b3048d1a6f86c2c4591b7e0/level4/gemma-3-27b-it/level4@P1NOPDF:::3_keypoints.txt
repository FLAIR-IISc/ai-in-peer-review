- Proposes a convex framework to partition data into K “latent domains” by maximizing pairwise mean differences in an RKHS while enforcing similar class distributions within each subset.  
- Chooses K by maximizing average learnability, i.e., classifier performance, within each latent domain.  
- Enforcing similar class distributions across domains can be limiting for tasks where class proportions naturally vary by domain (example: animal poses with uneven class presence).  
- Novelty: controlling class-label distributions within a (relaxed) convex framework for latent-domain discovery is a new contribution.  
- Novelty: selecting the number of latent domains via per-domain classification quality is a useful practical contribution.  
- Terminology is unclear—“dataset,” “domain,” and “latent domain” are used interchangeably and need precise definitions and examples, especially in Section 4.2.  
- Equation 7 uses max_k r(U_k,B) without prior definition and needs clarification.  
- Experiments show reasonable improvements over baselines, but the choice of reported source/target datasets is not well justified; recommend leave-one-dataset-out evaluation; qualitative results are helpful.  
- Framing the task as unsupervised learning is misleading because the method uses semantic class labels.  
- Overall assessment: technical contribution is noteworthy but several concepts and experimental details require clearer explanation.
