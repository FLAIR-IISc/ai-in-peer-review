- Proposes a convex framework to partition data into K latent domains with similar class distributions and maximal distinctiveness via RKHS pairwise mean differences.
- Selects the optimal K by maximizing average learnability, i.e., how well a classifier can be learned within each latent domain.
- Enforcing similar class distributions per domain prevents single-class dominated clusters but may be limiting for factors (e.g., pose) where class distributions differ across domains.
- Latent domain discovery itself is not new, but controlling class label distributions within a relaxed convex framework is a novel contribution.
- Using classification quality to choose K is valuable because it ensures sufficient per-class samples for generalization and discrimination.
- Key concepts ("dataset", "domain", "latent domain") need clearer definitions and illustrative examples.
- Experiments (Sec. 4.2) are confusing: terms like Si are used inconsistently as datasets and source domains; experimental setting and concept usage must be clarified.
- Equation 7 uses "maxk r(U_k,B)" without a clear definition and should be explicitly defined.
- Experimental validation shows reasonable improvements over baselines and helpful qualitative visualizations.
- Rationale for chosen source and target datasets is unclear; a more comprehensive protocol (e.g., leave-one-dataset-out) is recommended.
- Calling the problem "unsupervised" is misleading because the method uses semantic class labels.
- Overall, the technical contribution—discovering latent domains by encouraging similar class distributions in a relaxed convex framework—is notable but would benefit from clearer exposition and experimental detail.
