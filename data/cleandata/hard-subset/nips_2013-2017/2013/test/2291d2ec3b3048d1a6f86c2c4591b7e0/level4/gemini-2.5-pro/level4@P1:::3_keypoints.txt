- Proposes a framework to partition source data into K latent domains by maximizing inter-domain distinctiveness (RKHS pairwise mean differences) while enforcing domain learnability via a class label prior constraint (LPC).

- Novel, principled (relaxed) convex formulation for latent domain discovery with an explicit constraint on class label distributions.

- Automatic selection of K by maximizing average cross-validated classification accuracy ensures discovered domains are learnable, not just statistically distinct.

- Experimental validation is solid, showing consistent improvement on object and action recognition; qualitative results effectively illustrate learned domains.

- LPC may be overly restrictive when class and domain are correlated (e.g., pose distributions differ by class), potentially preventing discovery of true domains.

- Authors should discuss LPC limitations and possible relaxations or alternatives for real-world scenarios.

- Terminology is unclear—terms like "dataset," "domain," and "latent domain" are used interchangeably; symbols such as r(A → B) should be explicitly defined.

- Experimental protocol lacks rationale for chosen source/target combinations; suggest a more standard evaluation (e.g., leave-one-dataset-out) for robustness.

- Describing the problem as "unsupervised" is misleading since the method uses class labels for the LPC; better phrasing is unsupervised w.r.t. domain labels but supervised w.r.t. class labels.
