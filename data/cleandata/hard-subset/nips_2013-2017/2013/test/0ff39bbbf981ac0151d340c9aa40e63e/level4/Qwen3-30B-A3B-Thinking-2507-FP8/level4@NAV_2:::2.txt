

Summary: This paper introduces a two-step approach for cross-lingual representation learning that leverages matrix completion followed by latent semantic indexing. The method constructs a unified document-term matrix using parallel bilingual documents, solves a matrix completion problem with projected gradient descent optimization, and then applies latent semantic analysis to obtain low-dimensional cross-lingual representations. Extensive experiments on cross-lingual sentiment classification using multilingual Amazon product reviews demonstrate that the proposed method consistently outperforms both monolingual baselines and cross-lingual representation learning approaches, particularly when the number of unlabeled parallel documents is limited.

Quality: The experimental evaluation is comprehensive and well-designed, thoroughly validating the method's effectiveness across 18 cross-lingual sentiment classification tasks. The authors carefully compare their two-step learning algorithm against multiple relevant baselines including Target Bag-Of-Words (TBOW), Cross-Lingual Latent Semantic Indexing (CL-LSI), Cross-Lingual Oriented Principal Component Analysis (CL-OPCA), and Cross-Lingual Kernel Canonical Correlation Analysis (CL-KCCA), providing convincing evidence for their claims.

Clarity: The paper is well-structured and clearly written, with detailed explanations of both the theoretical framework and implementation details. All parameter selection procedures are thoroughly documented, making the work highly reproducible. A minor suggestion: the motivation for using matrix completion could be more explicitly articulated in the Introduction section, as it is currently better explained in Section 3.

Originality: While cross-lingual representation learning is an established research area, the application of matrix completion to bridge language feature spaces represents a novel approach. The paper clearly demonstrates how this method differs from previous work, particularly by first completing the document-term matrix before applying latent semantic indexing, rather than directly applying dimensionality reduction techniques to the incomplete matrix as in prior methods.

Significance: The consistent performance improvements across all 18 classification tasks, especially the substantial gains over baselines (up to 14.61% accuracy improvement on the EFB task), highlight the practical value of this approach. The method's effectiveness with limited parallel data (demonstrated by experiments with varying numbers of unlabeled parallel documents) suggests broader applicability to resource-constrained cross-lingual learning scenarios. This work provides a valuable framework for cross-lingual representation learning that could be extended to other natural language processing tasks beyond sentiment classification.
