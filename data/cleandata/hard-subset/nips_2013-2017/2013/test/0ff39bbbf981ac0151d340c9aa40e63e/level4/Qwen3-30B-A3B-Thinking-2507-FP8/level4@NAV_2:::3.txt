

The paper introduces a two-step approach for cross-lingual representation learning that bridges disjoint language feature spaces using unlabeled parallel documents. It first constructs a unified document-term matrix where parallel documents are fully observed and non-parallel documents have missing entries for the other language, then solves a matrix completion problem to recover missing entries via projected gradient descent with convergence guarantees. Finally, it applies latent semantic indexing (LSI) on the recovered matrix to produce low-dimensional cross-lingual representations, enabling standard monolingual classifiers for cross-lingual sentiment classification. This work builds upon NIPS contributions like CL-LSI [10], CL-OPCA [15], and CL-KCCA [20] but differs by first completing the document-term matrix before applying LSI, rather than directly processing the sparse matrix.

**Quality:** The technical formulation is sound, with a well-motivated matrix completion problem and a rigorous convergence proof for the projected gradient descent algorithm. However, the experimental evaluation relies solely on linear SVMs, which are sensitive to feature scaling—a concern given that LSI inherently scales features. The observed performance gains (up to 14.61% over baselines) may partly stem from LSI's scaling properties rather than the core matrix completion step. To validate robustness, the authors should compare results using classifiers invariant to feature scaling (e.g., non-linear SVMs with RBF kernels or gradient-boosted trees) across all methods. Without this, the significance of the method's contribution to representation learning remains uncertain.

**Clarity:** The paper clearly articulates the two-step framework and its relation to prior work. However, Line 113's description of the "unified document-term matrix" lacks precision: it is unclear whether the vocabulary treats language-specific surface forms (e.g., "Merkel" vs. "Merkelen") as distinct features or uses a shared lexicon. Additionally, the experimental setup is fragmented across Table 1 and text, requiring readers to cross-reference details. A consolidated description of the dataset and protocol in one section would improve readability.

**Originality:** The method is novel in its two-step pipeline—matrix completion followed by LSI—which differs from prior approaches like CL-LSI [10] (direct LSI on sparse matrix), CL-OPCA [15], and CL-KCCA [20] (projection-based methods). The matrix completion step explicitly leverages co-occurrence patterns across languages, addressing a gap noted in the related work section.

**Significance:** If validated with scaling-insensitive classifiers, the method’s robustness with minimal parallel data (e.g., 200 documents) could be impactful for low-resource cross-lingual tasks. However, the current results do not conclusively establish this, as the linear SVM baseline may inadvertently amplify LSI’s inherent scaling effects. The technique’s potential for principled non-linear inference via matrix completion is promising but requires stronger evidence to demonstrate broader significance beyond the specific experimental setup.

**Pro/Con Summary for Acceptance:**  
*Pro:* Novel two-step framework, rigorous convergence analysis, strong empirical results on Amazon reviews, effective performance with small parallel datasets.  
*Con:* Limited classifier evaluation (linear SVM only), unclear vocabulary treatment, fragmented experimental description, unverified robustness to feature scaling.  

The paper presents a technically grounded idea with clear potential, but the current evaluation lacks sufficient rigor to confirm its significance. Addressing the classifier robustness concern would substantially strengthen the contribution.
