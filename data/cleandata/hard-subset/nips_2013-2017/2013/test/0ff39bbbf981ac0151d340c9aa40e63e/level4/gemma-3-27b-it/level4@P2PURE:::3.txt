The paper introduces a translingual representation learning method that leverages matrix completion to generate multilingual vectors for each row, followed by Canonical Correlation Analysis with Latent Semantic Indexing (CL-LSI). 

Quality: Although the proposed approach appears promising, the robustness of the results is questionable. The observed improvements may be attributable to the inherent scaling properties of LSI being more favorable for a linear SVM than those of Orthogonal Procrustes Canonical Correlation Analysis (OPCA) or Canonical Correlation Analysis (CCA). 

To address this concern, I recommend that the authors evaluate their system with a machine learning model less sensitive to feature scaling, such as boosted decision trees (readily available in Weka). If the two-step system consistently outperforms existing techniques when used with boosted decision trees, the reported results will be more convincing.

Significance: A robust and validated result would be a valuable contribution, as matrix completion provides a well-defined, non-linear inference mechanism.

Novelty: The proposed idea appears novel.

Clarity: The paper contains some clarity issues. Specifically, line 113’s description of “construct[ing] a unified document-term matrix for all documents” is ambiguous. It is unclear whether this refers to using a single term to represent all surface forms (e.g., “Merkel”), or to individually labeling each surface form from each language (e.g., “Merkelen” and “Merkelde”). Clarification is needed. Furthermore, the experimental setup is fragmented across Table 1, which hindered my understanding. 

Overall, this is an interesting idea. I encourage the authors to explore classifiers beyond linear SVMs to determine whether the observed performance gains are robust or an artifact of feature scaling.
