- Proposed method: two-step translingual representation learning—matrix completion to create multilingual vectors, then Cross-Lingual LSI (CL-LSI).
- Experimental robustness concern: positive results may be an artifact of the classifier choice rather than better representations.
- Linear SVM sensitivity: because linear SVMs are sensitive to feature scaling, LSI’s scaling properties might explain superior performance over OPCA/CCA.
- Suggested experiment: evaluate with a rescaling-insensitive classifier (e.g., boosted decision trees) to validate robustness.
- Significance: if robust, using matrix completion as a controlled non-linear inference is a valuable contribution.
- Novelty: the combination of matrix completion and CL-LSI for this task appears novel.
- Clarity issue 1: ambiguous phrase “unified document-term matrix”—clarify whether surface forms are canonicalized or treated as distinct features.
- Clarity issue 2: experimental setup description is split between text and Table 1—consolidate details for readability.
- Overall: current evaluation is insufficient; conclusions would be more convincing with results from classifiers not sensitive to feature scaling.
