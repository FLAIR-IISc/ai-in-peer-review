- Proposed method: two-step approach that fills a unified document-term matrix via matrix completion (projected gradient descent) then applies LSI to get cross-lingual document representations.  
- Novelty: casting cross-lingual representation learning as a matrix completion problem combined with LSI is a new contribution.  
- Theoretical: convergence analysis provided for the projected gradient descent optimization.  
- Main experimental weakness: evaluation may be confounded by the classifier choice, so reported gains may not reflect the matrix completion step.  
- Scaling confound: LSI produces scaled features while OPCA/KCCA have different scalings; a linear SVM is sensitive to feature scaling and may favor LSI representations.  
- Clarity issue 1: unclear whether terms from different languages are treated as distinct features (e.g., separate columns for “book” and “livre”).  
- Clarity issue 2: experimental setup is fragmented across text and table caption; data splits and train/test sizes should be presented in one place.  
- Significance: addresses an important problem and could be valuable for low-resource scenarios if results are robust.  
- Suggested fix: validate results with a scale-invariant classifier (e.g., boosted decision trees/AdaBoost with trees) to rule out scaling artifacts.  
- Pro: elegant, novel idea; relevant to the community; promising initial results, especially with few parallel documents.  
- Con: potential experimental flaw undermines the central empirical claim about the method’s effectiveness.  
- Overall: interesting and original work that requires stronger, scale-invariant empirical validation to be a complete, convincing contribution.
