- Proposes a novel two-step method (matrix completion then LSI) to learn cross-lingual document representations using unlabeled parallel documents.  
- Builds a unified document-term matrix with missing entries for non-parallel docs and formulates a convex matrix completion problem with trace-norm and L1 regularization.  
- Solves the optimization via projected gradient descent and provides a convergence proof.  
- Applies LSI to the completed matrix to obtain low-dimensional cross-lingual representations for standard monolingual classifiers.  
- Empirically outperforms baselines (CL-LSI, CL-KCCA, CL-OPCA) on Amazon reviews, especially when parallel data is scarce.  
- Main theoretical concern: the low-rank assumption for document-term matrices is not well-justified and needs more intuition or evidence.  
- Unclear motivation for the LSI step after completion — why LSI over alternatives like PCA and whether M* is dense despite L1 penalty.  
- Experimental setup is ambiguous; request confirmation of data splits (4000 labeled source, 100 labeled target, 2000 unlabeled parallel, 1900 test) and varying unlabeled counts in Sec. 5.3.  
- Need clearer distinction of data usage: labeled source data for supervised training vs. unlabeled parallel data for representation learning.  
- Missing runtime/computational-cost comparison with CL-LSI, CL-KCCA, and CL-OPCA to assess practical trade-offs.  
- Paper would benefit from more intuition on why enforcing global low-rank structure outperforms CCA-based methods.  
- Apparent inconsistency between Figure 1 and Table 1 for tasks EFM and EGM with 2000 parallel docs requires clarification.  
- Minor editorial fixes suggested for notation and phrasing around Eq. (2), Eq. (4)/(5), and Section 4.1 wording.  
- Originality: combination of matrix completion and LSI for cross-lingual representation is novel despite using established components.  
- Significance: addresses practical cross-language classification, with robustness to limited parallel data; recommendation leans toward acceptance if concerns are addressed.
