- Proposes Deep Recurrent Neural Networks (DRNNs): each layer is a recurrent network that takes the previous layer’s hidden state as input to create an explicit temporal hierarchy.  
- Claims standard RNNs only implicitly learn temporal hierarchies, while DRNNs explicitly embed different time scales at different layers.  
- Demonstrates SOTA performance for recurrent networks on Wikipedia character-level language modeling using simple SGD, with the DRNN-1O variant outperforming standard RNNs.  
- Provides analysis of emergent time scales across layers and experiments showing layer-specific contributions to long-term dependencies (e.g., parentheses closing task).  
- Formal analysis of temporal hierarchies in deep recurrent architectures is a significant conceptual contribution, extending prior stacked-LSTM work.  
- Experimental design and validation are strong, effectively showing DRNNs capture multiple time scales and layer-specific memory properties.  
- Writing and presentation are clear, with helpful equations and figures explaining the architecture.  
- Strength: strong technical quality, meaningful empirical and analytical insights into time-scale emergence.  
- Weakness: limited comparison to more recent non-recurrent methods (e.g., transformers).  
- Weakness: architectural novelty is somewhat incremental given prior stacked RNN/LSTM work (e.g., [8]).  
- Weakness: would benefit from more thorough ablation studies on layer-specific contributions.  
- Recommendation: acceptance—valuable contribution to understanding and modeling multi-scale temporal dependencies.
