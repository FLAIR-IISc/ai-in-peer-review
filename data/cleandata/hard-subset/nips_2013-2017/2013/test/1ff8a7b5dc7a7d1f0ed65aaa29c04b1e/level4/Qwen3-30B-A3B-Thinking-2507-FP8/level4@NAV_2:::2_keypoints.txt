- Paper studies deep recurrent neural networks (DRNNs) where each layer is a recurrent network taking the previous layer’s hidden state, applied to character-level language modeling.  
- Claims state-of-the-art performance for recurrent networks on character-level modeling using only SGD (BPC 1.47 vs. 1.63 baseline).  
- Demonstrates DRNNs can learn long-term structure (e.g., balancing parentheses) without Hessian-free optimization.  
- Provides novel, well-supported time-scale analysis showing deeper layers retain longer context (top-layer output dominates long-range predictions).  
- Uses layer-wise ablation (disabling bottom-layer output degrades text quality) to add interpretability.  
- Architecture novelty is overstated: similar/identical setups (stacked bidirectional LSTMs / DRNN-1O) were used by Graves et al. (NIPS 2013).  
- Evaluation scope is narrow (only character-level modeling); extending to speech tasks (e.g., TIMIT) would improve significance.  
- Comparison to Hessian-free training is misleading; truncated BPTT (e.g., Mikolov’s work) may achieve competitive results more cheaply.  
- Overall: valuable analysis of DRNN temporal hierarchy but limited impact due to overstated novelty and narrow experimental scope; revise to clarify prior work and add broader tasks.
