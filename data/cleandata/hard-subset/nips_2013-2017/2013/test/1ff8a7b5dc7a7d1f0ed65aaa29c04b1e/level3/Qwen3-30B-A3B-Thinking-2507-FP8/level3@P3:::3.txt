

# Review of "Training and Analyzing Deep Recurrent Neural Networks"

This paper introduces Deep Recurrent Neural Networks (DRNNs), where each layer is a recurrent network that receives the hidden state of the previous layer as input. The authors argue that this architecture naturally creates a temporal hierarchy, with lower layers processing short-term dependencies and higher layers capturing longer-term patterns. The paper demonstrates that DRNNs achieve state-of-the-art results on character-level language modeling using only SGD, without the need for more complex training algorithms like Hessian-free optimization. The work builds on prior hierarchical RNN approaches [3,5,8] but provides a more thorough analysis of the temporal hierarchy that emerges in DRNNs.

**Strengths:**
The paper makes a compelling case for DRNNs as a natural way to model time series with multiple temporal scales. The text-generation experiment (Table 2) is particularly persuasive, showing that DRNN-AO produces coherent text that maintains grammatical structure and meaningful phrases. The analysis of time scales through perturbation experiments (Figure 3) effectively demonstrates how different layers operate at different temporal resolutions. The authors successfully show that DRNNs can maintain memory of hundreds of characters, as evidenced by their parentheses-closing experiment (Figure 4). The paper is well-written, clear, and makes a solid contribution to understanding RNN architectures.

**Weaknesses and Suggestions:**

1. **Layer Architecture and Analysis:** The paper's core insight that older data benefits from more processing layers is valuable. However, the authors should consider adding extra nonlinear layers between timesteps (e.g., between the output of one layer and the input of the next) to better separate processing from temporal propagation. It would be helpful to know whether this was attempted and why it was not adopted.

2. **Gradient Normalization:** The paper mentions gradient normalization as a "simple heuristic" to prevent gradient explosions. It would be valuable to understand how critical this is to the method's success. Specifically, how does this normalization avoid "bifurcations" (a term not clearly defined)? This is particularly relevant as the authors cite the ICML 2013 U. of Montreal paper on gradient truncation [1], which provides a more formal treatment of this issue.

3. **SGD Averaging Behavior:** The paper does not address whether gradient normalization disrupts SGD's natural averaging behavior, which is known to help with generalization. This is an important concern that should be discussed.

4. **Comparison to Baselines:** The comparison to Mikolov et al. on Wikipedia compression tasks would strengthen the paper's claims about language modeling performance. Additionally, the parameter count differences between the DRNNs and baseline RNNs (4.9 million vs. 2.1 million parameters) make the comparison potentially unfair. The authors should include baselines with the same number of parameters or units to isolate the architectural benefit.

5. **Key Missing Experiment:** A critical missing experiment is whether higher layers genuinely gain from upstream processing versus simply relocating units to the first layer. The paper should test if a single-layer network with the same number of units as the full DRNN performs similarly, which would clarify whether the depth itself provides benefits.

6. **Figure 2 Analysis:** Instead of Figure 2 (which shows performance for different DRNN depths without matching parameter counts), the authors should plot performance for various DRNN depths trained from scratch, including wider layers to match parameters/units. This would provide a fairer comparison of depth versus width.

7. **Minor Clarifications:** The terms DRNN-AO and DRNN-IO should be defined in the text, not just in Figure 1. The paper refers to "DRNN-IO" in the introduction but never defines it, creating confusion.

**Overall Assessment:**
This paper makes a worthwhile contribution to understanding RNN architectures and temporal hierarchies. The core idea that DRNNs create natural time scales is compelling and well-supported by the experiments. The paper is readable, well-organized, and largely convincing. However, it would be significantly strengthened by addressing the concerns above, particularly the parameter count comparison and the key missing experiment regarding whether depth genuinely provides benefits beyond parameter count. With these additional experiments and clarifications, the paper would be even more impactful.

The paper is of high quality and would be appropriate for NIPS. While the current version is already strong, the suggested additions would make it excellent. I recommend acceptance with minor revisions.
