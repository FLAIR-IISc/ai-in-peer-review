

This paper proposes Deep Recurrent Neural Networks (DRNNs), where each layer is a recurrent network that receives the hidden state of the previous layer as input. The authors argue that this architecture naturally captures temporal hierarchies in time series data, with lower layers handling short-term dependencies and higher layers processing longer-term patterns. They demonstrate state-of-the-art performance on character-level language modeling using Wikipedia text, with DRNN-AO (all layers connected to output) and DRNN-1O (only top layer connected) architectures.

The paper makes several strong contributions: it provides a clear architectural innovation that combines deep learning with recurrent structures, offers convincing empirical evidence through text generation (Table 2) showing DRNNs' ability to handle long-range dependencies (e.g., parenthesis closing), and analyzes emergent time scales across layers (Figure 3). The explanation of how hierarchical processing emerges during training is particularly compelling.

However, several important concerns need addressing. The comparison to prior work is problematic: Mikolov et al.'s Wikipedia compression results (2010) should be included for fair benchmarking, and the parameter count differences between DRNNs and baselines (e.g., 4.9M vs. MRNN's 4.9M but trained with Hessian-free optimization) make performance comparisons misleading. The claim that higher layers "benefit from more processing" requires a key missing experiment: testing whether relocating units to the first layer would achieve similar performance, rather than genuine hierarchical processing.

The gradient normalization technique (Equation 3) is used without sufficient justification. The authors should clarify how this avoids "bifurcations" and whether it disrupts SGD's averaging behavior. The suggestion to add nonlinear layers between timesteps (beyond the current architecture) merits exploration. While the ICML 2013 paper by Bengio et al. on gradient truncation is relevant, it's not cited.

Minor issues include not defining DRNN-AO/IO in the text (only in Figure 1) and the layer-surgery interpretation in Figure 2 being confounded by complex interdependencies. A more thorough analysis of DRNN depth performance (with parameter-matched networks trained from scratch) would strengthen the results.

Overall, this is a well-written, significant contribution that advances understanding of hierarchical temporal processing. With additional experiments addressing the parameter count issue, gradient normalization justification, and comparative benchmarks, it would be a strong paper. The core idea is promising, but the current evidence needs strengthening to fully convince about the necessity of the hierarchical architecture.
