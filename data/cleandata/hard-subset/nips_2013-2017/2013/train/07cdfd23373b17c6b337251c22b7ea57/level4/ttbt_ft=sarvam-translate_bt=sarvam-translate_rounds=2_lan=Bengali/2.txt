This paper proposes a technique for learning hidden structures in social network data. This model matches the triangle counting statistics obtained from a pre-processing step, similar to word-bag representations for text documents. The main advancement in modeling was inspired by aMMSB: instead of parameterizing interactions between all "communities," they consider the configuration of triangles to parameterize. This results in fewer parameters in the model and, consequently, makes it more usable. They apply recent advances in Stochastic Variational Inference – creating a global update and a local update for the proposed model.

The new modeling contribution is a simple combination of aMMSB with MMTM, although some tedious calculations were required. The presentation of this component could be further improved.

Although most learning algorithms are a direct application of Stochastic Variational Inference, the authors provide an intuitive and appealing approximation for the local update rule.

Considering the breadth of the proposed network, a detailed discussion of how these methods can be extended is welcome.

Persistent Concerns:

- "Our Persimoneous Triangular Model (PTM) is essentially a hidden space representation of ERGM, which represents the processes of creating triangle motifs." This seems like a curious statement: 1) ERGM can include many different statistics besides triangles, and 2) I don't understand how PTM is a "hidden space representation" unless all the hidden spaces are network models "hidden space representations."
- Can the assumed hidden space be explained? If so, what can we learn about the similarities and differences between these graphs?
The difference between delta^1 and delta^2 is not clear (at least to this reader).
- Why does the subsampling of triangles (using delta) for each node not provide sufficient approximation (in this paper)?
- If the goal is to recover the "non-empty set of roles," then the model should probably have a more binary concept of "role."
- How strong are the results of hidden space recovery for a threshold of .125 (which seems somewhat arbitrary)?
- Different hidden space models can create power-law networks. Which one did you use for the experiment?
- Why is PTM CGS K^3 complex? Didn't you argue that there are O(K) parameters instead of O(K^3)?
- One concern is how it will perform when fewer edges are seen. In the experiment, they only retained 10% of the edges, probably because the distribution of adjacent triangles would be quite biased if many edges were removed before the pre-processing step.
- It is intuitive that the method defeats MMSB in a synthetic network created through MMSB (Figure 1, left 4 panel).

This work uses Stochastic Variational Inference to learn a mixed-membership block model (with a restricted parameter set) on triangle counting in a graph. Their modeling choice and approximation allow for a more easily usable O(NK) algorithm.
