This paper proposes a technique for learning latent structures in social network data. This model fits the triangular counting statistics obtained from a pre-processing step, similar to the word-bag representation for text documents. The main advancement in the modeling is inspired by aMMSB: instead of parameterizing interactions across all "communities," they consider the configuration of triangles to identify K roles to be parameterized. This results in the model having fewer parameters and, consequently, being more easily usable. They apply recent advances in Stochastic Variational Inference – creating a global update and a local update for the proposed model.

The new modeling contribution is a simple combination of aMMSB with MMTM, although some tedious calculations were required. The representation of this component could be further improved.

Although most learning algorithms are a direct application of Stochastic Variational Inference, the authors provide an intuitive and appealing approximation for the local update rule.

Given the breadth of the directed network, a detailed discussion of how these methods can be extended is welcome.

Persistent Concerns:

- "Our Persimoneous Triangular Model (PTM) is essentially a latent space representation of ERGM, which represents the processes of creating triangular motifs." This seems like an intriguing statement: 1) ERGM may include many different statistics besides triangles, and 2) I don't understand how PTM is a "latent space representation," unless all the latent spaces are network models "latent space representations."
- Is the assumed latent space explainable? If so, what can we learn about the similarities and differences between these graphs?
The difference between delta^1 and delta^2 is not clear (at least to this reader).
- Why does the subsampling of triangles (using delta) for each node not provide sufficient approximation (in this paper)?
- If the goal is to recover the "non-empty set of roles," then the model should probably have a more binary concept of "role."
- How strong are the latent space recovery results for the threshold of .125 (which seems somewhat arbitrary)?
- Different latent space models can create power-law networks. Which one did you use for the experiment?
- Why is the complexity of PTM CGS K^3? Didn't you argue that there are O(K) parameters instead of O(K^3)?
- One concern is how it will perform when fewer edges are seen. In the experiment, they only retained 10% of the edges, probably because the distribution of adjacent triangles would be quite biased if too many edges were removed before the pre-processing step.
- It is counterintuitive that the method defeats MMSB in a synthetic network created through MMSB (Figure 1, left 4 panels).

This work uses Stochastic Variational Inference to learn a mixed-membership block model (with a restricted parameter set) on triangle counting in a graph. Their modeling choices and approximations allow for a more easily usable O(NK) algorithm.
