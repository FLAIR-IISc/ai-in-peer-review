This paper proposes a technique for learning a hidden structure in social network data. This model is suitable for triangle computation data obtained from the preprocessing step, which is similar to a bag-of-words representation for text documents. The main modeling advancement is inspired by aMMSB: instead of parameterizing all "community" interactions, they consider the configuration of triangles to identify a set of K roles to parameterize. This results in fewer parameters in the model and, in turn, a more scalable implementation. They apply recent advances in Stochastic Variational Inference – obtaining a global update and local updates for the proposed model.

The new modeling contribution is a direct combination of aMMSB with MMTM, although some extent of tedious bookkeeping was required. The presentation of this material could be improved.

Although most learning algorithms are direct applications of Stochastic Variational Inference, the authors have provided a intuitive and interesting approximation for the local update rule.

Given the breadth of directed networks, a detailed discussion on how these methods can be extended would be welcome.

Ongoing Concerns:
- "Our Parsimonious Triangulate Model (PTM) is essentially a hidden space equivalent to ERGM, which represents processes for generating triangular motifs". This seems like a strange statement: 1) ERGM can include much more diverse figures than just triangles, and 2) I don't see how PTM is a "hidden space equivalent", unless all hidden space network models are "hidden space equivalents".
- Are the inferred hidden spaces interpretable? If so, what do we learn about the similarities and differences between these graphs?
The difference between delta^1 and delta^2 is not clear (at least to this reader).
- It is not clarified (in this paper) why the subsampling of triangles (using delta) for each node provides sufficient approximation.
- If the goal is to recover "a set of non-zero roles", then perhaps the model should have a more binary notion of "role".
- How robust are the hidden space retrieval results for the .125 limit (which seems somewhat arbitrary)?
- Different types of hidden space models can produce power-law networks. Which one did you use for the experiments?
- Why is PTM complexity K^3 CGS? Didn't you argue that there are O(K) parameters instead of O(K^3)?
- One concern is how it will perform when fewer edges are observed. In the experiments, they only kept 10% of the edges, possibly because the distribution of adjacent triangles would be quite biased if too many edges were removed before the preprocessing step.
- It is not intuitive that the method beats MMSB on a synthetic network generated through MMSB on aMMSB (Figure 1, left 4 panel).

This work uses Stochastic Variational Inference to learn a mixed-membership block model (with a restricted set of parameters) on triangle computation in a graph. Their modeling options and approximations allow for a more scalable O(NK) algorithm.
