Following the authors' feedback, the following has been added:

*   Please include a simulation where the loss function is not strongly convex, but the PG algorithm converges linearly (for various levels of stability), and discuss significantly when the theory fails. Also, discuss how this can be extended for the analysis of ADMM.

*   This paper proves the linear convergence of the proximal gradient method applied to a trace-norm regular learning problem when the loss function has the form f(X) = h(A(X)), where A is linear and h is strongly convex on any compact set and is a Lipschitz gradient.

This paper is an extension of Tseng [20], Tseng and Yoon "A coordinate gradient descent method for non-smooth differentiable optimization" and Zhang et al. [22], which established similar results using the "error-bound condition" for Lasso and group Lasso for trace norms. This is a non-trivial extension but the contribution seems purely technical.

The presentation of the proofs is mostly clear.

Strengths:

*   Demonstrates the linear convergence of the proximal gradient algorithm, extending the results of Tseng et al.

Weaknesses:

*   The contribution is purely technical.
*   I would like to see a numerical example showing linear convergence.

Further details:

1.  The outline of the proofs for theorems 3.1 and lemma 3.2 seems similar to Tseng's proofs. The authors should refer to the original work more precisely to clarify their contribution.
2.  I would like to see a numerical example that actually shows linear convergence (a semilog plot of function value versus number of iterations).
3.  It is not clear how the constants \ kappa1, ..., \ kappa4, \underbar{\alpha} and \bar{\alpha} depend on the choice.

Minor issue:

4.  The order of inequalities before inequality (13) is misleading. The last inequality is due to the convexity of the trace norm and the fact that -\bar{G}\in \tau\partial\|X\|_{\ast} and not due to the intermediate inequality (see page 289 in [20]).

An extension of the previous linear convergence result based on the "error-bound condition" for trace norms from Lasso and group Lasso.
