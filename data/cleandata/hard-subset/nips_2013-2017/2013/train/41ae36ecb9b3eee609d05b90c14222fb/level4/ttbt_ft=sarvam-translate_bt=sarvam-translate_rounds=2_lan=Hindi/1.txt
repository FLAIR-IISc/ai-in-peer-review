Following is an addition to the response:

*   Please include a simulation where the loss function is not strongly convex, but the PG algorithm converges linearly (for various levels of stability), and discuss significantly when the theory fails. Also, discuss how it can be extended for ADMM analysis.

*   This paper proves the linear convergence of the proximal gradient method applied to a trace-norm regular learning problem when the loss function has the form f (X) = h (A (X)), where A is linear and h is strongly convex over any compact set and is a Lipschitz gradient.

This paper is an extension of Tseng [20], Tseng and Yoon "A coordinate gradient descent method for non-smooth differential optimization" and Zhang et al. [22], which established similar results using the "error-bound condition" for group Lasso for Lasso and trace norms. This is a non-trivial extension but the contribution seems purely technical.

The presentation of the proofs is mostly clear.

Strengths:

*   Demonstrates the linear convergence of the proximal gradient algorithm, extending the results of Tseng et al.

Weaknesses:

*   The contribution is purely technical.
*   I would like to see a numerical example showing linear convergence.

Further Information:

1.  The outline of the proofs for Theorem 3.1 and Lemma 3.2 seems similar to Tseng's proofs. The authors should refer to the original work more precisely to clarify their contribution.
2.  I would like to see a numerical example that actually shows linear convergence (a semilog plot of the function value versus the number of iterations).
3.  It is not clear how the constants \ kappa1, ..., \ kappa4, \underbar{\alpha} and \bar{\alpha} depend on the choice.

Minor Issue:

4.  The order of inequalities before inequality (13) is misleading. The last inequality is due to the trace norm's convexity and the fact that -\bar{G}\in \tau\partial\|X\|_{\ast} and not due to the intermediate inequality (see page 289 [20]).
