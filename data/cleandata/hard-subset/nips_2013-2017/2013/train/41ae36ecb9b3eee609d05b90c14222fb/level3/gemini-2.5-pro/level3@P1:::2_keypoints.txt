- Summary: Proves linear convergence of proximal gradient method for trace-norm regularized problems when f(X)=h(A(X)) with h strictly convex/smooth, without requiring strong convexity; key tool is a new Lipschitzian error bound using spectral analysis.

- Key technical contribution: A novel Lipschitzian error bound relating distance to the optimal set to the proximal residual ||R(X)||_F, obtained via matrix perturbation and SVD structure near optima.

- Strength — Significance: Addresses an important practical setting (low-rank recovery, matrix completion) where strong convexity fails, giving the first linear-rate guarantee for the standard PGM.

- Strength — Originality/Technical quality: Theorem 3.1 (error bound) is novel and the proof—showing shared singular vectors between X* and −∇f(X*) and using that structure—is sound and clever.

- Strength — Clarity: Paper is well written, motivated, situated in literature, and includes experiments that corroborate theory.

- Weakness — Comparison to l1: Needs a more explicit comparison to l1-norm results, clearly explaining why non-separability of singular values breaks existing analyses and how the SVD approach fixes this.

- Weakness — Novelty-in-context: Should state that the l1-norm error bound is a special/degenerate case of the trace-norm bound (diagonal matrices) to better position the contribution.

- Weakness — Intuition for lemmas: Provide higher-level intuition for Lemma 4.1 (Restricted Invertibility), relate it to concepts like RIP, and explain the contradiction/limiting-direction proof strategy.

- Minor — Title: Consider highlighting the composite loss structure in the title (e.g., "...for Composite Loss Functions").

- Minor — Definitions: Briefly recall Q-linear and R-linear convergence in the main text for reader convenience.

- Minor — Residual map: Clarify how the paper's residual map R(X) relates to the proximal residual in the Luo–Tseng framework.

- Minor — Experiments: Clarify whether the sampling operator P in matrix completion is a fixed random mask or drawn per iteration.

- Minor — References: Fix the broken/incorrect citation noted (line ~337).
