- Proves linear convergence of proximal gradient method for trace-norm regularized problems when f(X)=h(A(X)) with h strictly convex, without requiring strong convexity.
- Main technical contribution: a new Lipschitzian error bound linking distance to the optimal set with the norm of a residual map.
- Extends the Luo–Tseng analysis framework to the trace norm by exploiting shared singular vector structure (Prop. 4.2), resolving an open question.
- Significance: provides a rigorous explanation for observed linear-like convergence in non-strongly-convex ML problems (e.g., matrix completion, multi-task learning).
- Originality: non-trivial adaptation because the trace norm is non-separable and prior l1 techniques do not apply; requires novel spectral analysis.
- Quality: technically sound, well-written, complete proofs and experiments that empirically demonstrate the predicted linear convergence.
- Suggestion: expand the discussion contrasting the l1-norm analysis and explicitly show why variable-splitting/grouping arguments fail for the trace norm.
- Suggestion: remark that the l1-norm is a special case (diagonal matrices) and clarify that the result generalizes l1 while using fundamentally different techniques.
- Suggestion: improve accessibility by briefly explaining Q- vs R-linear convergence and relating the residual map R(X) to Luo–Tseng’s error bound formulation.
- Suggestion: provide more intuition for Lemma 4.1 and the Restricted Invertibility property, explaining how progress in A-space implies progress in X-space.
- Minor details: clarify whether the matrix-completion sampling mask P is fixed or re-sampled across trials and check citations for correctness.
- Recommendation: strongly accept — significant, novel, and well-executed work with mainly presentation-level improvements suggested.
