This paper introduces LoSST, a novel online algorithm for learning the structure of graphical models in non-stationary environments. The key assumption is that the data-generating process is locally stationary, meaning the underlying graphical model (structure and/or parameters) can change over time. The algorithm works by tracking sufficient statistics (mean, covariance, and an "effective" sample size) using a dynamic weighting scheme. New data points that fit the current model poorly, as measured by the Mahalanobis distance, signal a potential change. This triggers an increase in the weight of subsequent data points, causing the algorithm to adapt to the new regime while "forgetting" the old one. A standard batch structure learning algorithm (here, the PC algorithm) is then periodically applied to the tracked statistics to output a graph.

The work is well-positioned relative to prior art. It correctly identifies the limitation of standard batch-mode structure learning algorithms (e.g., Spirtes et al., 2000; Chickering, 2002), which assume global stationarity. It also distinguishes itself from online parameter estimation techniques (e.g., Sutton, 1988) which do not learn structure, and from changepoint detection methods (e.g., Adams & MacKay, 2007) which are often not designed for real-time operation on high-dimensional covariance structures. The problem of tracking a time-varying graphical model structure in an online fashion is highly significant and has been underexplored, and this paper presents one of the first practical solutions.

**Strengths:**

1.  **Significance and Originality:** The paper tackles a fundamental and challenging problem that is highly relevant to real-world applications, where the i.i.d. assumption rarely holds. The proposed LoSST algorithm is a novel and intuitive approach that elegantly combines online parameter tracking with batch structure learning. The idea of using the Mahalanobis distance to modulate data point weights for tracking a changing covariance matrix is a clever and effective core mechanism.

2.  **Technical Quality:** The algorithm is technically sound. The update rules for the weighted mean and covariance are correctly derived. The use of Liptak's method for pooling p-values from the Mahalanobis distances is a principled way to build a robust change signal over time, preventing overreactions to single outliers. The discussion in Section 3.1 about the trade-off between "convergence" and "diligence" and how it relates to the algorithm's parameters (γ, T) adds theoretical depth and is a valuable insight.

3.  **Experimental Evaluation:** The empirical evaluation is thorough and convincing. The simulations clearly demonstrate that LoSST performs comparably to a standard batch learner (PC) in a stationary environment, which is a crucial result showing there is little cost to using the online method. More importantly, it dramatically outperforms the batch learner when the underlying model changes, validating the core contribution. The experiments thoughtfully consider both structure changes and parameter-only changes, revealing interesting differences in performance. The application to US price index data is a compelling real-world case study that illustrates the algorithm's potential utility and ability to detect known macroeconomic shifts.

**Weaknesses and Suggestions for Improvement:**

1.  **Heuristic Components and Parameter Sensitivity:** The algorithm relies on several heuristic choices and hyperparameters that are not fully justified.
    *   The weight update rule in Equation (4) is a key component, but its specific linear form is chosen without justification. What is the impact of other functional forms?
    *   Similarly, the probabilistic relearning schedule uses a noisy-OR gate, which is a reasonable choice but one among many.
    *   The paper states that key parameters (T=0.05, γ=3) were found via "unsystematic investigation." This is a significant weakness, as the performance of the algorithm likely depends heavily on these settings. A more systematic sensitivity analysis is needed to understand how these parameters affect the algorithm's ability to detect changes of different magnitudes and speeds. Providing guidance to practitioners on how to set these parameters would greatly increase the paper's impact.

2.  **Baseline for Comparison:** While comparing LoSST to a batch-mode PC algorithm effectively highlights the problem with the stationarity assumption, it is a somewhat weak baseline in the non-stationary setting. A more informative comparison would include a simple but plausible alternative, such as applying the PC algorithm on a sliding window of recent data. This would help to better isolate the contribution of LoSST’s adaptive weighting scheme over a more naive online approach.

3.  **Scope of Applicability:** The paper focuses on recursive SEMs over continuous variables, for which the covariance matrix is a sufficient statistic. This is a reasonable starting point, but the claim that the "core algorithm, are readily adaptable to learn structure for models based on undirected graphs" or other model classes is not fully substantiated. For instance, extending this to discrete-variable Bayesian networks is highly non-trivial, as tracking the sufficient statistics (conditional count tables) in a weighted, online fashion would be memory-intensive and computationally complex. The paper would be strengthened by either toning down these claims or providing more concrete details on how such extensions would be realized.

**Overall Assessment:**

This is a well-written and well-motivated paper that makes a significant contribution to the field of graphical model learning. It presents a novel, practical, and effective algorithm for the important problem of structure learning in non-stationary environments. The empirical results are strong and the real-world application is compelling. While the method has some heuristic elements and its sensitivity to hyperparameters needs further exploration, these limitations do not detract from the core value of the work. The paper addresses a clear gap in the literature and is likely to inspire follow-up research. I recommend acceptance.
