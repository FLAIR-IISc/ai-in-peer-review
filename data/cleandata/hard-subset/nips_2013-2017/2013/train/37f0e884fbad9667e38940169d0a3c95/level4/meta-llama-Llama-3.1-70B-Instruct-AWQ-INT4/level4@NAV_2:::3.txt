The paper presents a novel hybrid strategy for stochastic gradient descent, which employs both stochastic and batch gradients. The algorithm, referred to as Epoch Mixed Gradient Descent (EMGD), is guaranteed to converge to an epsilon accurate solution using O(log(1/eps)) full gradients, and O(k^2 log(1/eps)) stochastic gradients.

The convergence proof appears correct and novel, with some minor mistakes detailed below. However, my main concern is about the relevance of the proposed algorithm in the machine learning setting, which is the focus of the conference. In typical machine learning algorithms, strong convexity is often introduced by the regularizer, which implies that the strong convexity parameter lambda is of the order of the number of samples N, i.e., lambda = C N, where C does not depend on N. Under this assumption, the proposed method is faster than batch gradient only if the number of samples is bounded by O(C^3/L^3), which does not seem to be an interesting regime. Furthermore, the convergence rate for the proposed algorithm holds only in high probability, whereas the ones for batch gradient descent are deterministic.

This point is crucial and must be carefully discussed to demonstrate that the algorithm has a real advantage over batch gradient descent and to prove the relevance of the paper for the machine learning community. The authors should provide a clear explanation of the benefits of EMGD in practical machine learning optimization problems.
