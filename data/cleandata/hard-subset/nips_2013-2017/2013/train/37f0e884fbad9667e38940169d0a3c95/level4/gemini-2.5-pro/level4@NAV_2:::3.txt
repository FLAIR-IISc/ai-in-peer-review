This paper proposes a novel algorithm, Epoch Mixed Gradient Descent (EMGD), for smooth and strongly convex optimization. The key idea is to combine full-gradient and stochastic-gradient updates to reduce the dependency on the condition number, κ, for the number of expensive full-gradient computations. The authors present a theoretical analysis showing that EMGD finds an ε-accurate solution by computing O(log(1/ε)) full gradients and O(κ² log(1/ε)) stochastic gradients. This work aims to bridge the gap between deterministic methods, which require O(√κ log(1/ε)) full-gradient evaluations, and purely stochastic methods, by replacing the bulk of full-gradient steps with cheaper stochastic ones, thereby making the number of full-gradient calls independent of κ.

**Strengths:**
*   The paper addresses an important problem: reducing the computational burden of full-gradient methods for ill-conditioned problems.
*   The proposed EMGD algorithm, which uses a mixed-gradient update within a shrinking local domain, is novel.
*   The convergence proof appears technically sound and provides a high-probability linear convergence guarantee, which is a strong theoretical result.

**Weaknesses:**
My primary concern relates to the practical significance of the proposed algorithm within the context of typical large-scale machine learning problems. The main theoretical result is a trade-off: the number of full gradients is reduced from O(√κ) to being independent of κ, but at the cost of introducing an O(κ²) dependency on the number of stochastic gradients.

*   The total computational complexity of EMGD, assuming a stochastic gradient is 1/n of the cost of a full gradient, is O((n + κ²) log(1/ε)). This should be compared to the O(n√κ log(1/ε)) complexity of accelerated full gradient methods. As the authors note, EMGD is more efficient only when κ ≤ n²/³, approximately. The paper would be significantly strengthened by a more thorough discussion of practical machine learning scenarios where this condition holds. In many applications, strong convexity is introduced via a regularizer, and the regularization coefficient is often small, leading to a very large condition number κ that may not fall into this favorable regime.
*   The convergence guarantee for EMGD holds with high probability, whereas the guarantees for deterministic competitors like accelerated gradient descent are deterministic. This is a weaker type of guarantee that should be acknowledged in the comparison.

To establish the paper's relevance for the machine learning community, it is crucial that the authors more clearly delineate the problem settings where their algorithm provides a concrete advantage over standard accelerated batch gradient descent.

**Minor Comments and Suggestions for Clarification:**
*   It would be helpful to explicitly state early on that L ≥ λ, from which it follows that the condition number κ = L/λ ≥ 1.
*   In the analysis leading to equation (14), the equality `max_w (...) = η/2 * ||∇gt(wt)||^2` seems to add complexity without aiding comprehension. Simply using an inequality might improve clarity.
*   In the bound for BT (after eq. 18), the notation `|Vt|` is used. To be precise, this appears to be the absolute value of the inner product, which is bounded using norms. Clarifying this could prevent confusion.
*   The notation in equation (6) in Lemma 1 is standard, but the reviewer of this paper seems to have made a typo in their draft. The equation in the paper is correct. It would be beneficial if the authors double-checked all equations for notational consistency (e.g., consistent use of `w`, `x`, `F`, `f`).
*   When using equation (7) to derive (16), it might be clearer to explicitly state that the property F(w̄) - F(w*) ≤ λ/2 * ∆² from (7) is being applied.
