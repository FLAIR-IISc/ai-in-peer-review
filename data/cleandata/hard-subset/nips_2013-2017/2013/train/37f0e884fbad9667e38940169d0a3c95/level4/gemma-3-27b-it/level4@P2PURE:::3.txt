The paper introduces a novel hybrid stochastic gradient descent strategy that leverages both stochastic and batch gradient evaluations. The algorithm is theoretically guaranteed to converge to an ε-accurate solution using O(log(1/ε)) full gradients and O(k<sup>2</sup> log(1/ε)) stochastic gradients. 

The convergence proof appears correct and represents a novel contribution, aside from minor points detailed below. 

My primary concern lies with the practical relevance of this algorithm within the typical machine learning context, which is the focus of this conference. In many machine learning applications, strong convexity is primarily enforced by the regularization term, implying a value of μ proportional to the number of samples N (μ = CN, where C is independent of N). Under this assumption, the proposed method offers speedups over batch gradient descent only when the number of samples is bounded by O(C<sup>3</sup>/L<sup>3</sup>), a regime which does not seem particularly compelling. Furthermore, the convergence rate for the proposed algorithm is established only in high probability, while batch gradient descent offers deterministic convergence guarantees. 

This distinction is crucial and requires careful discussion to demonstrate a clear advantage of the proposed algorithm over standard batch gradient descent and to justify the paper's relevance to the machine learning community.
