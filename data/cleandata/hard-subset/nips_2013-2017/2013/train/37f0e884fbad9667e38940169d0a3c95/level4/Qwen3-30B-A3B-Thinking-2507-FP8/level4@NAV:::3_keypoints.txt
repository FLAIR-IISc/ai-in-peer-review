- Proposes a novel hybrid stochastic/batch gradient descent algorithm combining stochastic and batch gradients.  
- Claims convergence to an ε-accurate solution using O(log(1/ε)) full gradients and O(k² log(1/ε)) stochastic gradients.  
- Convergence proof appears correct and novel, though minor errors exist.  
- Major concern: in ML settings with regularization (μ = Θ(N)), the method outperforms batch GD only when N = O(C³/L³), a practically unlikely regime.  
- Additional concern: the method’s convergence guarantee is only with high probability, whereas batch GD has deterministic convergence — this must be addressed to demonstrate a clear advantage.  
- Minor corrections: use ||w* − ŵ||² in Eq.(6); specify which function in line 288 applies to (7); remove the equality in line 286; replace absolute values with norms in (13); explicitly state L ≥ λ.  
- More minor fixes: in (4) change x* to w* and f to F (first max term dominates per Lemma 1); precisely define condition number κ = L/λ.  
- Overall: unclear whether the algorithm provides a practical advantage over standard batch gradient descent for ML problems.
