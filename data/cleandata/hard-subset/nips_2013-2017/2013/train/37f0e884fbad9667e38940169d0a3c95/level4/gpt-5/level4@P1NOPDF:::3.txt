This paper proposes a hybrid stochastic/batch gradient method that interleaves stochastic and full-gradient steps. The algorithm is proven to achieve an ε-accurate solution using O(log(1/ε)) full gradients and O(k^2 log(1/ε)) stochastic gradients. Apart from a few minor issues noted below, the convergence analysis appears correct and, to my knowledge, novel.

My primary concern is the relevance of the algorithm to typical machine learning settings, which is the focus of this venue. In standard ML formulations, strong convexity typically arises from the regularizer. Consequently, the strong convexity parameter μ scales with the number of samples N; more precisely, μ ≈ C N, where C does not depend on N. Under this assumption, the proposed method is faster than batch gradient descent only when the number of samples satisfies N = O(C^3 / L^3), which does not seem to be an interesting or practically common regime. Moreover, the stated convergence rate for the proposed algorithm holds only with high probability, whereas the convergence guarantee for batch gradient descent is deterministic.

This issue is crucial. The paper should discuss this regime carefully, quantify when (in realistic ML problems) the hybrid method yields a concrete advantage over batch gradient descent, and clarify the implications for practice. Without such a discussion, it is difficult to assess the method’s practical significance for the ML community.

Minor comments:
- In Eq. (6), the expression should be ||w^* − \hat{w}||^2.
- Please specify on line 288 which function Eq. (7) is applied to.
- The equality on line 286 should be removed; it does not aid comprehension and instead obscures it.
- In Eq. (13), absolute values should be norms.
- Please explicitly state and justify that L >= lambda; even if standard, it should be clearly noted.
- In Eq. (4), x^ should be w^, f should be F, and by Lemma 1 the first term in the max is always larger than the second.
- Please define the condition number precisely as a function of lambda and L.

Overall assessment: The paper introduces a novel hybrid stochastic/batch gradient method with an interesting analysis. However, it is currently unclear whether it offers a practical advantage over standard batch gradient descent in typical ML optimization problems.
