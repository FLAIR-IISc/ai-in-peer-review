- Proposes a hybrid stochastic/batch gradient algorithm with convergence using O(log(1/ε)) full gradients and O(k^2 log(1/ε)) stochastic gradients.  
- Convergence proof is largely correct and novel, aside from minor errors.  
- Major concern: relevance to ML — with regularizer-induced strong convexity mu = C·N, the method is faster than batch GD only if N = O(C^3 / L^3), which is an uninteresting regime.  
- Convergence result holds only with high probability, whereas batch gradient descent has deterministic guarantees.  
- Authors must carefully discuss and demonstrate a real advantage over batch GD for typical ML settings.  
- Eq.(6) should read ||w^* - \hat{w}||^2.  
- Specify which function is used in line 288 for equation (7).  
- Remove the equality in line 286; it reduces clarity.  
- In Eq.(13) replace absolute values with norms.  
- Explicitly state that L ≥ λ (even if obvious).  
- In Eq.(4), x^ should be w^ and f should be F; also note the first term in the max is always larger by Lemma 1.  
- Precisely define the condition number as a function of λ and L.  
- Overall: novel hybrid algorithm, but unclear practical advantage for standard machine-learning optimization problems.
