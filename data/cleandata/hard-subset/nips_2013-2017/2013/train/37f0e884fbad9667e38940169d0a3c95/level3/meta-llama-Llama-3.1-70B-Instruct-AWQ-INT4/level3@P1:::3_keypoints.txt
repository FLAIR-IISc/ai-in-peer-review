- Proposes a novel hybrid stochastic/batch algorithm (Epoch Mixed Gradient Descent, EMGD) for smooth, strongly convex optimization.  
- Claims convergence rate O(log(1/ε)) using O(log(1/ε)) full gradients and O(κ^2 log(1/ε)) stochastic gradients.  
- Provides a thorough convergence analysis with a detailed proof of the main theorem.  
- Proof appears correct but contains minor mistakes/typos (e.g., equation (6) should read ||w^* − ŵ||^2; remove equality in line 286).  
- Major concern: relevance to ML is unclear because strong convexity λ often comes from regularization and scales with sample size N.  
- If λ ∝ N, the algorithm’s advantage over batch gradient descent is only significant for very small N, which is atypical in ML.  
- The result is high-probability only, whereas batch gradient descent gives deterministic convergence, raising practical advantage questions.  
- Authors need a more careful comparison to batch gradient descent and justification of practical advantages in ML settings.  
- Recommend adding context on typical values of λ and L in ML and discussing trade-offs with batch gradient descent.  
- Minor clarity/presentation issues to fix (specify function in line 288 for eq. (7); explicitly state L ≥ λ).  
- Arguments for acceptance: novel algorithm, dependence on well-studied condition number κ, and thorough analysis/proof.  
- Arguments against acceptance: unclear ML relevance, only high-probability convergence, and advantage limited to unusually small N.  
- Overall quality: 7/10.  
- Clarity: 8/10.  
- Originality: 8/10.  
- Significance: 6/10.  
- Impact: 6/10.
