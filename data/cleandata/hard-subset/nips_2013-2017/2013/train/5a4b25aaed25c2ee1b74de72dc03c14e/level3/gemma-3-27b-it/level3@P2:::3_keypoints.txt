- Proposes projecting Ising model parameters onto a set guaranteed to be fast-mixing for Gibbs sampling.
- Core idea: favor parameter sets where MCMC converges rapidly, trading some accuracy for faster, reliable sampling.
- Implements projection using multiple divergences (Euclidean, KL, piecewise KL, reversed KL).
- Experiments show projected models consistently outperform standard variational methods.
- For limited computation budgets, projected models can beat Gibbs sampling on original slow-mixing parameters.
- Well-motivated and grounded in existing approximate inference and mixing-time theory.
- Clearly connects to fast-mixing conditions via the dependency matrix spectral norm.
- Euclidean projection derivation (dual formulation + L-BFGS-B) is technically sound.
- Comparative analysis of divergences is valuable; reversed KL performed particularly well.
- Empirical validation is strong but limited to small models (8x8 grids, 10-node random graphs).
- Scalability to larger, realistic problems remains uncertain.
- Computational cost of the projection step (notably reversed KL) is a concern; SGD required intensive tuning without full details.
- Description of projected-gradient algorithm (Section 5.1) lacks clarity on inner L-BFGS-B stopping criteria.
- Figure 2 x-axis labeled “number of samples” is misleading; runtime/absolute seconds for all methods should be reported.
- Minor editorial fixes: replace “onto the tree” with “onto graph T” and use “subgradient” instead of “derivative.”
- Overall recommendation: accept with minor revisions to clarify computational details, stopping criteria, runtime reporting, and scalability.
