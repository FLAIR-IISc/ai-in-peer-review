- Proposes projecting Ising model parameters onto a parameter set satisfying the Dobrushin criterion to speed Gibbs mixing via a constrained optimization formulation.  
- Addresses an under-explored combination of stochastic and deterministic inference; core idea is interesting.  
- Technical description of the dual projection problem is unclear, especially how the zij * dij = 0 constraints preserve the original parameter space.  
- Risk that constraints could restrict the parameter space rather than preserve it; need clarification.  
- Ambiguity about the overall procedure: whether projected gradient descent runs to completion before sampling or is interleaved with Gibbs; an algorithm box would help.  
- No discussion or guarantee of convergence for the proposed projected gradient descent strategy.  
- Experimental: using 30,000 Gibbs iterations is arbitrary; comparisons should be based on wall-clock time and include error bars for statistical significance.  
- Experiments omit the computational cost of the projection step, so net practical benefit (time saved vs. projection overhead) is unassessed.  
- Missing stronger MCMC baselines (e.g., blocked Gibbs, Gogate’s Lifted Gibbs) to demonstrate comparative advantage.  
- Overall: promising idea but requires clearer technical development and a more thorough, fair experimental evaluation.
