- Paper proposes projecting Ising model parameters to satisfy the Dobrushin criterion to guarantee faster Gibbs mixing via a constrained optimization formulation.
- Combining stochastic and deterministic inference principles is an interesting and under-explored idea.
- Description of the dual/projection problem is unclear — specifically how constraints z_ij * d_ij = 0 guarantee the new parameterization stays in the same space.
- Concern that the z_ij * d_ij = 0 constraint might inadvertently restrict the parameter space to a smaller subset.
- Overall algorithmic procedure is ambiguous: are projections done to completion before sampling, or is sampling interleaved with gradient updates?
- An explicit algorithm box would greatly improve clarity.
- Unclear if the projected gradient descent strategy has convergence guarantees.
- Experimental setup issues: rationale for running Gibbs for 30,000 iterations is not explained.
- Comparisons to deterministic methods should be based on wall-clock time for fairness.
- Plots lack error bars, so statistical significance of results is hard to assess.
- Computational cost of computing the projection is not accounted for when claiming faster mixing.
- Missing comparisons to stronger baselines such as blocked Gibbs samplers and Gogate’s “Lifted Gibbs Sampler.”
- Overall assessment: promising idea but needs clearer development of the projection method and more comprehensive experiments.
