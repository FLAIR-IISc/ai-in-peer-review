- Proposed method: reparameterize Ising model via constrained optimization to satisfy Dobrushin criterion and improve Gibbs mixing, explored under various distance/divergence measures.
- Novelty: integration of stochastic and deterministic inference ideas is interesting and relatively unexplored.
- Clarity issue: explanation of the dual projection problem is unclear.
- Technical concern: unclear how the constraint z_{ij} * d_{ij} = 0 ensures the new parameterization spans the original parameter space rather than a subspace.
- Algorithmic ambiguity: unclear whether projected gradient updates are run to completion before Gibbs sampling or interleaved; an explicit algorithmic description is needed.
- Convergence concern: unclear whether the proposed projected gradient descent procedure has convergence guarantees.
- Experimental issue: comparisons run for a fixed number of iterations (30,000) rather than wall-clock time, which is unfair when comparing sampling to deterministic methods.
- Experimental reporting: absence of error bars makes result significance difficult to assess.
- Missing cost accounting: time to perform the projection is not included in comparisons, which could negate practical benefits.
- Baselines omitted: no comparison to blocked Gibbs samplers or Gogate’s Lifted Gibbs Sampler.
- Overall assessment: idea is promising but projection formulation, method refinement, and a more comprehensive experimental evaluation are needed.
