- Proposal: Project Ising-model parameters onto a Dobrushin-criterion-satisfying parameter set via constrained optimization to speed Gibbs mixing; evaluated under several divergences.  
- Positive: Combining stochastic and deterministic inference and augmenting parameters to improve mixing is an intriguing, under-explored idea.  
- Clarity issue: The dual formulation is unclear—specifically how the zij * dij = 0 constraints ensure the new parameterization spans the same space rather than a smaller one.  
- Procedural ambiguity: It is not specified whether projected gradient updates are run to convergence before sampling or interleaved with Gibbs sampling; an algorithm box is needed.  
- Theoretical concern: No guarantee or discussion of convergence for the projected gradient descent strategy is provided.  
- Experimental choices unclear: Running Gibbs for 30,000 iterations is unexplained; comparisons to deterministic methods should be based on runtime rather than iteration count.  
- Presentation weakness: Plots lack error bars, making result significance hard to assess.  
- Cost omission: Comparisons showing faster mixing under projection ignore the time required to compute the projection, which could negate benefits.  
- Missing baselines: Should compare to blocked Gibbs samplers and Gogate’s Lifted Gibbs Sampler for a more comprehensive evaluation.  
- Overall assessment: Promising idea but requires clearer development of the projection, convergence analysis, and more robust, time-aware experiments.
