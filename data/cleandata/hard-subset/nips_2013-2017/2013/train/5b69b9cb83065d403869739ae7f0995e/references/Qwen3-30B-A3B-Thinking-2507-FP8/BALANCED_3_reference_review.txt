

# Review: Low-rank matrix reconstruction and clustering via approximate message passing

## Summary
This paper proposes an approximate message passing (AMP) algorithm for Bayesian low-rank matrix reconstruction that extends previous work to handle both marginalization and MAP estimation problems. The authors further apply this algorithm to K-means clustering, presenting a novel approach that directly handles the constraint that each datum should be assigned to exactly one cluster within the low-rank matrix framework. The work builds on recent advances in AMP algorithms [6, 7, 9] while differentiating itself by fixing rank r rather than r/m in the asymptotic limit and not assuming statistical independence among row components.

## Strengths

1. **Technical Contribution**: The paper makes a meaningful extension of AMP to handle both marginalization and MAP problems for low-rank matrix reconstruction, which is not covered by previous AMP algorithms [9]. This unified framework is a significant technical contribution.

2. **Novel Application**: The connection between low-rank matrix reconstruction and clustering is well-motivated, and the proposed AMP-based clustering algorithm represents a genuine novelty in handling the exact cluster assignment constraint directly within the low-rank framework, unlike previous approaches [10, 11].

3. **Theoretical Grounding**: The paper provides a solid theoretical foundation, including Proposition 1 which establishes a connection between AMP fixed points and the ICM algorithm, and shows that AMP can avoid getting stuck in poor local minima.

4. **Experimental Validation**: The numerical experiments on both artificial and real data (ORL faces) are thorough and demonstrate clear advantages of the proposed method over standard K-means++, VBMF-MA, and AMP-MA in terms of clustering accuracy and convergence speed.

## Weaknesses

1. **Limited Complexity Comparison**: While the paper states the algorithm has O(mN) computational complexity per iteration, it lacks a detailed comparison with existing methods (e.g., how the constant factors compare to ICM or variational Bayes approaches). This is a significant omission for a paper claiming computational efficiency.

2. **State Evolution Analysis**: The paper mentions that state evolution could be used to predict performance (as in [6, 8, 18]) but does not develop this analysis. For AMP-based methods, state evolution is a key theoretical contribution that would strengthen the paper significantly.

3. **Scalability with Rank**: The experiments show good results for small to moderate ranks (r ≤ 16), but the paper doesn't address how the algorithm scales with increasing rank r, which is crucial for practical applications where r might be large.

4. **Convergence Guarantees**: While Proposition 1 provides some convergence properties, the paper doesn't address whether the algorithm converges to a global optimum or how sensitive it is to initialization, which is important for a clustering algorithm.

5. **Practical Implementation Details**: The paper mentions using m⁻²N⁻¹‖A - Uᵗ(Vᵗ)⊤‖²F as a temporary estimate of τ, but doesn't explain how this affects performance compared to other estimation methods. The initialization method for real data (using K-means++) is mentioned but not compared to other initialization strategies.

## Overall Assessment

The paper presents a technically sound contribution that extends AMP algorithms to a broader class of low-rank matrix reconstruction problems and applies them effectively to clustering. The work is well-motivated, theoretically grounded, and supported by compelling experimental results.

The main weaknesses are the lack of detailed computational complexity comparison, absence of state evolution analysis (which is a hallmark of AMP papers), and insufficient discussion of scalability with rank. These are significant omissions for a NeurIPS paper, as they prevent a full assessment of the algorithm's practical advantages.

## Recommendations

1. Add a detailed computational complexity comparison with existing methods (ICM, variational Bayes) to demonstrate the efficiency gains.

2. Include state evolution analysis for the proposed AMP algorithm to provide theoretical performance guarantees, as this is a key strength of AMP-based methods.

3. Extend the experiments to show how the algorithm scales with increasing rank r (e.g., up to r=100 or more) and discuss the practical limitations.

4. Provide more details on the initialization strategy for real-world applications, including how it affects performance.

5. Clarify the distinction between the proposed approach and previous work on matrix reconstruction for clustering [10, 11] by explicitly stating why their method is the first to handle the exact cluster assignment constraint directly.

The paper makes a valuable contribution to the field, and with these improvements, it would be a strong candidate for NeurIPS. As it stands, it's a solid technical contribution but lacks some of the theoretical depth and practical scalability discussion expected at NeurIPS. I recommend acceptance with minor revisions to address the weaknesses noted above.
