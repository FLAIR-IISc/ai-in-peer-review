

# Peer Review: "Low-rank matrix reconstruction and clustering via approximate message passing"

## Summary
This paper proposes an approximate message passing (AMP) algorithm for Bayesian low-rank matrix reconstruction, extending prior AMP work in compressed sensing to matrix factorization problems. The authors develop an algorithm that handles both marginalization (posterior mean estimation) and MAP estimation, and apply it to K-means clustering by recognizing clustering as a special case of low-rank matrix reconstruction with structural constraints. The key contribution is an AMP-based algorithm that addresses the MAP problem for low-rank matrix reconstruction (previously limited to posterior mean estimation in [9]), and a novel clustering algorithm that directly handles the constraint that each datum must be assigned to exactly one cluster.

## Strengths

1. **Theoretical grounding**: The paper builds on solid theoretical foundations, extending the AMP framework to matrix factorization. The derivation of the algorithm using the β-parameterized posterior distribution is elegant and provides a unified treatment of marginalization and MAP problems.

2. **Computational efficiency**: The algorithm has a low computational cost (O(mN) per iteration), which is a significant advantage for large-scale problems. The authors correctly note that the additional terms compared to variational Bayes are computationally insignificant.

3. **Practical application**: The application to clustering is well-motivated and well-executed. The AMP-based clustering algorithm (Algorithm 3) is theoretically grounded and demonstrates clear empirical advantages over Lloyd's K-means in the experiments.

4. **Theoretical insights**: Proposition 1 provides an important theoretical guarantee showing that any fixed point of the AMP algorithm is also a fixed point of the ICM algorithm, with potential benefits for avoiding local minima.

## Weaknesses

1. **Incremental novelty**: While the extension to MAP estimation is useful, it's not clear how substantial this contribution is compared to prior work. [9] already proposed an AMP algorithm for the general-rank case, though limited to posterior mean estimation. The paper should more clearly articulate what makes their extension novel beyond a straightforward generalization.

2. **Theoretical limitations**: The paper mentions state evolution but does not provide a detailed analysis of the algorithm's convergence properties or theoretical guarantees. The claim that "required iteration numbers are O(1)" needs stronger justification beyond a brief reference.

3. **Clustering claim overstatement**: The paper claims to be "the first that directly deals with the constraint that each datum should be assigned to exactly one cluster in the framework of low-rank matrix reconstruction." This is questionable, as document clustering via NMF (e.g., [10]) and convex nonnegative matrix factorization (e.g., [11]) have explicitly handled this constraint. The paper should more carefully position its contribution relative to these works.

4. **Experimental limitations**: The experiments, while promising, lack comparison with state-of-the-art clustering algorithms beyond Lloyd's K-means and K-means++. The comparison with variational Bayes methods (VBMF-MA) is limited to maximum accuracy clustering, but the paper's main contribution is for K-means clustering, which should be compared against more recent clustering approaches.

## Overall Assessment

This is a solid technical contribution that extends the AMP framework to a wider class of low-rank matrix reconstruction problems and applies it effectively to clustering. The algorithm is well-motivated, theoretically grounded, and shows promising empirical results. However, the novelty claim is somewhat overstated, and the theoretical analysis would benefit from more detail.

The paper makes a useful contribution to the growing body of work on AMP for matrix factorization problems. While not revolutionary, it represents a valuable incremental advance that could be useful to practitioners dealing with large-scale low-rank matrix problems and clustering.

## Recommendation

**Accept** with minor revisions.

The paper addresses an important problem with a theoretically sound approach that has practical value. The main issues are the need to more precisely position the novelty relative to [9] and prior clustering work, and to provide more detailed theoretical analysis. These revisions would strengthen the paper significantly.

## Specific Suggestions for Improvement

1. Clarify the relationship between this work and [9] in the introduction and related work sections, explicitly stating what is novel in the extension to MAP estimation.

2. Re-evaluate the claim about being "the first" to handle the cluster assignment constraint in low-rank matrix reconstruction, citing relevant NMF clustering work.

3. Provide more details about the state evolution analysis (even if brief) to support the claim about O(1) iteration complexity.

4. Include comparisons with more recent clustering algorithms (e.g., deep clustering methods) in the experiments to better demonstrate the significance of the approach.

The paper is well-written, technically sound, and makes a valuable contribution to the field. With minor revisions to address the points above, it would be a strong candidate for NIPS.
