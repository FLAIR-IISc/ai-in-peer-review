- Presents an EM-based inference algorithm for binary latent feature models using FAB shrinkage to learn z's and infer the number of latent features K.  
- Algorithm is similar in spirit to nonparametric IBP methods but uses a model-selection FAB criterion instead of priors for parsimony.  
- Experiments show FAB outperforms Gibbs, variational, and MAP IBP methods on some large datasets in runtime and inference quality.  
- Technical approach appears sound and algorithmic details are correct.  
- Method scales to large datasets and can automatically infer the number of latent features.  
- Methodology sections are reasonably clear, but experimental details are insufficiently described.  
- Applying FAB to latent feature models is novel relative to prior FAB work on mixtures/HMMs.  
- Addresses the important problem of efficient inference in LFMs while inferring K.  
- Paper lacks discussion of algorithmic limitations and cases where it may underperform compared to Gibbs sampling.  
- No discussion of potential convergence difficulties for the EM-like procedure.  
- Method is limited to binary latent feature models and cannot be applied to factor analysis or probabilistic PCA.  
- Experimental evaluation is not thorough and contains flaws.  
- Behavior in small-data regimes (where asymptotic arguments may fail) is not explored or discussed.  
- Some recent related work on efficient inference for LFMs is missing from the discussion.  
- MEIBP baseline was inappropriately used: MEIBP requires non-negative W, so some simulation and block-data comparisons are invalid.  
- For fairness, IBP Gibbs should have been given the known noise value in block images experiments instead of a heuristic.  
- VB-IBP experiments used a finite variational version and lack details (e.g., number of restarts); infinite variational version could be used.  
- The accelerated Gibbs result finding ~70 features on 49-d Sonar data is suspicious and likely due to bad initialization or wrong noise variance.  
- Reference [21] is incorrectly described as Gibbs sampling; it performs MAP inference and should be corrected.  
- Suggest adding discussion of MAD-Bayes (ICML 2013) and other small-variance asymptotic work for nonparametric LFMs.  
- Minor correction: recommended std-dev setting for real data (refs [1,22]) is 0.25 times examples' std-dev, not 0.75.  
- After author feedback, experimental methodology and baseline settings still need clearer explanation and justification.  
- Must fix reference description for [21], include MAD-Bayes citation, and improve experimental evaluation before acceptance.
