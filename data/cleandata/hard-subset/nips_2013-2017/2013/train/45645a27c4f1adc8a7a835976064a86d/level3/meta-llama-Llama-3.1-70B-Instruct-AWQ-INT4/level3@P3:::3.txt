This paper proposes an extension of factorized asymptotic Bayesian (FAB) inference to latent feature models (LFMs). The authors present an EM-based inference algorithm for binary LFMs using FAB shrinkage to learn z's and infer the number of latent features K. The algorithm is similar in spirit to nonparametric Indian Buffet Process (IBP) methods but uses a model-selection FAB criterion instead of priors for parsimony.

The paper is well-written, and the technical approach appears sound. The algorithmic details are correct, and the method scales to large datasets and can automatically infer the number of latent features. The methodology sections are reasonably clear, but experimental details are insufficiently described. The application of FAB to latent feature models is novel relative to prior FAB work on mixtures and hidden Markov models.

The paper addresses the important problem of efficient inference in LFMs while inferring K. The experiments show that FAB outperforms Gibbs, variational, and MAP IBP methods on some large datasets in runtime and inference quality. However, the paper lacks discussion of algorithmic limitations and cases where it may underperform compared to Gibbs sampling. There is also no discussion of potential convergence difficulties for the EM-like procedure.

The method is limited to binary latent feature models and cannot be applied to factor analysis or probabilistic PCA. The experimental evaluation is not thorough and contains flaws. For example, the behavior in small-data regimes (where asymptotic arguments may fail) is not explored or discussed. Some recent related work on efficient inference for LFMs is missing from the discussion.

Moreover, the MEIBP baseline was inappropriately used, as MEIBP requires non-negative W, so some simulation and block-data comparisons are invalid. For fairness, IBP Gibbs should have been given the known noise value in block images experiments instead of a heuristic. The VB-IBP experiments used a finite variational version and lack details (e.g., number of restarts); an infinite variational version could be used.

The accelerated Gibbs result finding ~70 features on 49-d Sonar data is suspicious and likely due to bad initialization or wrong noise variance. Reference [21] is incorrectly described as Gibbs sampling; it performs MAP inference and should be corrected. The authors should add a discussion of MAD-Bayes (ICML 2013) and other small-variance asymptotic work for nonparametric LFMs.

A minor correction is needed: the recommended std-dev setting for real data (refs [1,22]) is 0.25 times examples' std-dev, not 0.75. After author feedback, experimental methodology and baseline settings still need clearer explanation and justification. The authors must fix the reference description for [21], include the MAD-Bayes citation, and improve experimental evaluation before acceptance.

Overall, the paper presents a novel and efficient method for inference in LFMs, but it requires more thorough experimental evaluation and discussion of its limitations.
