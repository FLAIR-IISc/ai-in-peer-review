## Review

**Summary:**

This paper introduces an inference algorithm for the binary Latent Feature Model (LFM), where each observation *x<sub>n</sub>* is represented by a binary vector *z<sub>n</sub>* indicating the presence of latent features. The algorithm learns both the latent features (*z*) and model parameters using an Expectation-Maximization (EM) approach incorporating a shrinkage mechanism to simultaneously infer the number of latent features (*K*). This work draws inspiration from nonparametric Bayesian LFM approaches like the Indian Buffet Process (IBP), but instead of employing prior distributions to encourage parsimony, it leverages the Factorized Asymptotic Bayesian Inference (FAB) framework – previously applied to mixture models and Hidden Markov Models. FAB formulates the log marginal likelihood in terms of the expected complete log-likelihood, augmented by a model selection term that promotes shrinkage in *K* through an upper bound.  The authors demonstrate that FAB outperforms Gibbs sampling, variational inference, and MAP estimation methods for the IBP on several large datasets, in terms of both runtime and inference quality.

**Quality:**

The technical quality of the paper appears sound, and the algorithmic details are presented correctly. The ability to handle large datasets while simultaneously learning the number of latent features is a significant strength.

**Clarity:**

The methodology is described clearly. However, details regarding the experimental setup require further clarification (see comments below).

**Originality:**

The proposed method builds upon the recently introduced FAB inference framework for mixture models and HMMs. Applying FAB to latent feature models represents a novel contribution.

**Significance:**

The paper addresses an important problem: efficient inference in latent feature models, coupled with the challenge of inferring the appropriate number of latent features.

**Weak Points:**

*   The paper lacks a discussion of the algorithm’s limitations. It would be valuable to consider scenarios where the proposed method might perform less effectively than standard Gibbs sampling.
*   The potential for convergence issues, inherent in EM-like procedures, is not addressed.
*   The algorithm is currently limited to binary latent feature models and cannot be directly extended to factor analysis or probabilistic PCA.
*   The experimental evaluation is not sufficiently comprehensive and contains some apparent flaws (detailed below).
*   The algorithm’s behavior in the small-data regime, where asymptotic arguments may not hold, is unclear and warrants discussion or empirical investigation.
*   Relevant recent work on efficient inference in LFMs is not discussed.
*   The comparison with the MEIBP algorithm ([22]) is invalid due to the algorithm’s requirement for non-negative linear Gaussian models. The synthetic and block data experiments were conducted with a real-valued loading matrix *W*, which is incompatible with MEIBP. A valid comparison would require generating synthetic data with a positive *W*.

**Other Comments:**

*   For the block image data, given that the noise level is known and provided to the FAB algorithm, fairness dictates that the IBP Gibbs sampler should also be provided with this value, rather than the commonly used 0.75 standard deviation heuristic.
*   For VB-IBP [2], the infinite variational version should be considered, as the experiments utilized a finite variational approximation. The experimental settings for VB-IBP are insufficiently detailed (e.g., the number of restarts used).
*   The discovery of approximately 70 latent features by the accelerated Gibbs sampler on the 49-dimensional Sonar data is surprising. This likely stems from poor initialization and/or an inadequately specified noise variance.
*   Line 70: Reference [21] describes MAP estimation for the IBP, not Gibbs sampling. Please revise the text accordingly.
*   The paper would benefit from a discussion of recent work on efficient inference using small-variance asymptotics in nonparametric LFMs, such as “MAD-Bayes: MAP-based Asymptotic Derivations from Bayes” (ICML 2013).
*   Line 349: For real-data, the standard deviation should be set to 0.25 times the standard deviation of examples across all dimensions, as suggested in [1, 22], rather than 0.75.

**Comments after Author Feedback:**

The authors’ response addressed some of the initial concerns. However, several issues require further attention before acceptance.

*   The experimental methodology, for both the proposed method and the baselines, needs to be more thoroughly explained. Justifications and explanations should be provided for the observed behavior of different algorithms (e.g., the use of a fixed alpha in the Gibbs sampler potentially leading to an overestimated *K*, or issues with hyperparameter settings).
*   Reference [21] describes MAP inference, not sampling; this should be corrected.
*   The reference to MAD-Bayes inference for the IBP should be included.

The proposed inference method for LFMs is promising, particularly its scalability to large datasets and its ability to infer *K*. However, the experimental evaluation requires more careful consideration.
