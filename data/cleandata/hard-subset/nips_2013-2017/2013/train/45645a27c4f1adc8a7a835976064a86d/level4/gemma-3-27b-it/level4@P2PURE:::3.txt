Summary:
The paper presents an inference algorithm for the binary latent feature model (LFM), where each observation x<sub>n</sub> is represented by a K-sized binary vector z<sub>n</sub>, indicating the presence or absence of the k-th latent feature in x<sub>n</sub>. The goal is to learn the z<sub>n</sub> vectors for all observations, along with other model parameters. The proposed EM-based algorithm employs a shrinkage mechanism to learn these vectors and simultaneously infer the number of latent features (K). This approach shares similarities with nonparametric Bayesian latent feature models like the Indian Buffet Process (IBP). However, unlike these models that utilize prior distributions to encourage parsimony, the presented algorithm leverages a model selection criterion – Factorized Asymptotic Bayesian Inference (FAB) – previously introduced for mixture models and Hidden Markov Models. FAB is based on expressing the log marginal likelihood in terms of the expected complete log-likelihood plus a model selection term that encourages shrinkage of K by imposing an upper-bound and learning the optimal value during inference. Empirical evaluation on large datasets demonstrates that FAB outperforms other inference methods, including Gibbs sampling, variational inference, and MAP estimation for the IBP, in terms of both running time and inference quality.

Quality: The technical quality appears sound, and the algorithmic details seem correct. The algorithm's ability to handle large datasets and simultaneously learn the number of latent features is particularly appealing.

Clarity: The methodology section is reasonably clear. However, some experimental details require further clarification (see comments below).

Originality: The proposed method builds upon the recently developed framework of Factorized Asymptotic Bayesian (FAB) Inference, initially applied to mixture models and HMMs. Applying FAB to latent feature models, however, represents a novel contribution.

Significance: The paper addresses an important problem – efficient inference in latent feature models, including the inference of the number of latent features.

Weak Points:
- The paper lacks a discussion of the algorithm’s limitations. Are there scenarios where it might perform worse than standard Gibbs sampling?
- The potential difficulties in convergence, given its EM-like procedure, are not addressed.
- The algorithm is currently limited to binary latent feature models and cannot be readily extended to factor analysis or probabilistic PCA.
- The experimental evaluation is not sufficiently thorough and contains some apparent flaws (detailed in the comments below).
- The algorithm's behavior in the small-data regime (where asymptotic arguments may not hold) is unclear. Discussion or experimentation in this setting would be beneficial.
- Relevant recent work on efficient inference in LFMs is not discussed. (See comments below).
- The MEIBP algorithm referenced in [22] is only applicable to non-negative linear Gaussian models (requiring a positive-definite W matrix, not a real-valued one). Consequently, the synthetic dataset and block data experiments are invalid for the MEIBP baseline; a positive loading matrix W would be necessary for a valid comparison.
Other Comments:
- For the block image data, since the noise level is known and provided to the FAB algorithm, the IBP Gibbs sampler should also be provided with the same value for fairness, rather than the 0.75 standard deviation heuristic.
- For VB-IBP [2], the infinite variational version could be considered instead of the finite version used in the experiments. Furthermore, the experimental settings for VB-IBP lack sufficient detail (e.g., the number of restarts used).
- The approximately 70 latent features discovered by accelerated Gibbs sampling on the 49-dimensional Sonar data are surprising. This may be attributable to poor initialization or an inappropriately specified noise variance.
- Line 70: Reference [21] describes MAP estimation for the IBP, not Gibbs sampling. Please revise the text accordingly.
- Recent work on efficient inference using small-variance asymptotics in nonparametric LFMs ("MAD-Bayes: MAP-based Asymptotic Derivations from Bayes," ICML 2013) should be discussed.
Minor comments:
- Line 349: The suggestion in [1, 22] was to set the standard deviation to 0.25 times the standard deviation of examples across all dimensions, not 0.75.

Comments after the author-feedback: The authors' feedback addressed some of the initial concerns. However, several issues require further attention before acceptance:
- The experimental methodology (for both the proposed method and the baselines) needs to be more comprehensively explained, with justifications for the observed behavior of different algorithms (e.g., a fixed alpha value in the Gibbs sampler potentially leading to an overestimated K, or improper tuning of noise variance hyperparameters).
- Reference [21] describes MAP inference, not sampling. Please correct this.
- Include a citation to the MAD-Bayes inference work for the IBP. The proposed inference method for LFMs is interesting, and its scalability to large datasets (along with K inference) is valuable. However, the experimental evaluation needs to be more carefully considered.
