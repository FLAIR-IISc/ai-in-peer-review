Summary:
The paper presents an inference algorithm for the binary latent feature model (LFM), which represents each observation xn as a K-dimensional binary vector zn. The presence of the k-th latent feature in xn is indicated by z{nk} = 1. The goal of inference is to learn the z's for all observations, along with other model parameters. The proposed EM-based algorithm employs a shrinkage mechanism to learn the z's and infer the number of latent features (K). This approach is similar in spirit to nonparametric Bayesian latent feature models, such as the Indian Buffet Process (IBP). However, the algorithm does not utilize prior distributions to induce model parsimony; instead, it relies on the Factorized Asymptotic Bayesian Inference (FAB) model selection criterion, recently introduced in the context of mixture models and HMMs. The FAB method expresses the log marginal likelihood in terms of the expected complete log-likelihood and a model selection term that encourages shrinkage of the number of latent features. On some large datasets, FAB is shown to outperform other inference methods, including Gibbs sampling, variational inference, and MAP estimation methods for the IBP, in terms of running time and inference quality.

Quality:
The technical quality of the paper appears to be reasonably sound, and the algorithmic details seem correct. The ability to handle large datasets and learn the number of latent features is very appealing.

Clarity:
The paper is reasonably clear in its methodology section. However, some experimental details are somewhat unclear and require further clarification.

Originality:
The proposed method is based on the recently proposed framework of Factorized Asymptotic Bayesian (FAB) Inference, applied to mixture models and HMMs. The application of FAB to latent feature models, however, is novel.

Significance:
The paper addresses an important problem: efficient inference in latent feature models while also inferring the number of latent features.

Weak Points:
- The paper lacks a discussion on the limitations of the proposed algorithm. It would be beneficial to explore cases where the algorithm might perform worse than vanilla Gibbs sampling.
- There is no discussion on the potential difficulties in convergence, given that it is an EM-like procedure.
- The algorithm is limited to binary latent feature models and cannot be applied to factor analysis or probabilistic PCA.
- The experimental evaluation is not thorough enough and appears to be flawed in some aspects.
- In the small-data regime, where the asymptotic argument does not hold, it is unclear how the algorithm will behave. Some discussion or experiments on this topic would be helpful.
- The paper does not discuss some recent works on efficient inference in LFMs.
- The MEIBP algorithm proposed in [22] is applicable only to non-negative linear Gaussian models (W must be positive, not real-valued). Therefore, the artificial simulation and the block data experiments are invalid for the MEIBP baseline. If the authors wish to compare with MEIBP, the synthetic dataset should be generated such that the loading matrix W is positive.

Other Comments:
- For the block images data, since the noise value is known and given to the FAB algorithm, it would be fair to provide the same value to the IBP Gibbs sampler instead of using the 0.75 standard deviation heuristic.
- For VB-IBP [2], the infinite variational version could be used, as the experiments employed the finite variational version. The experimental settings for VB are not described in sufficient detail (e.g., the number of restarts used).
- The accelerated Gibbs sampling discovered approximately 70 latent features on the 49-dimensional Sonar data, which seems suspicious. This might be due to bad initialization and/or a poorly specified noise variance value.
- Line 70: The reference [21] is not about Gibbs sampling but rather MAP estimation for the IBP (similar to reference [22]). Please modify the text and correct the description.
- There is recent work on efficient inference using small-variance asymptotics in nonparametric LFMs (see "MAD-Bayes: MAP-based Asymptotic Derivations from Bayes" from ICML 2013). It would be beneficial to discuss this work as well.

Minor Comments:
- Line 349: For real-data, the suggestion in [1,22] was not to set the standard deviation equal to 0.75, but to set it equal to 0.25 times the standard deviation of examples across all dimensions.

Comments after Author Feedback:
The author feedback addressed some of the questions, but there are still a few things that need to be fixed or improved before the paper can be accepted. In particular:
- The experimental methodology for the proposed method and the baseline needs to be better explained, and there should be justifications or explanations for why certain algorithms behaved in a certain way (e.g., the Gibbs sampler used a fixed alpha that might have led to an overestimated K, or noise variance hyperparameters were not properly set).
- Reference [21] does MAP inference, not sampling. Please fix this.
- Include the reference on MAD-Bayes inference for the IBP. The proposed inference method for LFMs is interesting, and the fact that it scales to large datasets (in addition to inferring K) is appealing. However, the experimental evaluation should have been more carefully done.
