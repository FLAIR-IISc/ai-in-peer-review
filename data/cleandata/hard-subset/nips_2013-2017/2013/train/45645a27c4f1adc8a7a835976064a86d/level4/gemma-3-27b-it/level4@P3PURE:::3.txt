Summary:
The paper presents an inference algorithm for the binary latent feature model (LFM), which expresses each observation x<sub>n</sub> as a K-sized binary vector z<sub>n</sub> where z<sub>nk</sub> = 1 indicates that x<sub>n</sub> possesses the k-th latent feature. The goal is to learn the z's for all observations, along with other model parameters. The presented EM-based algorithm uses a shrinkage mechanism to learn the z's and infer the number of latent features (K). This is similar in spirit to nonparametric Bayesian latent feature models such as the Indian Buffet Process (IBP). However, unlike these models which rely on prior distributions to induce model parsimony, the presented algorithm employs a model selection criterion – Factorized Asymptotic Bayesian Inference (FAB) – recently introduced for mixture models and Hidden Markov Models. The FAB approach expresses the log marginal likelihood in terms of the expected complete log-likelihood plus a model selection term that encourages shrinkage of the number of latent features, achieved by specifying an upper bound on K and learning the optimal K during inference. On several large datasets, FAB outperforms other inference methods, including Gibbs sampling, variational inference, and MAP estimation methods for the IBP, in terms of running time and inference quality.

Quality: The technical quality appears reasonably sound, and the algorithmic details seem correct. The ability to handle large datasets and simultaneously learn the number of latent features is particularly appealing.

Clarity: The paper is reasonably clear in its methodological description. However, some experimental details are unclear (discussed below).

Originality: The proposed method builds upon the recently introduced framework of Factorized Asymptotic Bayesian (FAB) Inference, originally applied to mixture models and HMMs. The application of FAB to latent feature models is novel.

Significance: The paper addresses an important problem: efficient inference in latent feature models, coupled with the ability to infer the number of latent features.

Weak Points:
- The paper lacks a discussion of the algorithm’s limitations. Are there scenarios where it might perform worse than standard Gibbs sampling?
- There is no discussion of potential convergence difficulties, given its EM-like procedure.
- The algorithm is currently limited to binary latent feature models and cannot be directly applied to factor analysis or probabilistic PCA.
- The experimental evaluation is not comprehensive enough and contains some apparent flaws (see detailed comments below).
- The algorithm's behavior in the small-data regime (where the asymptotic argument may not hold) is unclear; some discussion or experiments addressing this would be beneficial.
- Relevant recent work on efficient inference in LFMs has not been discussed. (see comments below).
- The MEIBP algorithm proposed in [22] is applicable only to non-negative linear Gaussian models (requiring a positive, not real-valued, W). Therefore, the artificial simulation and block data experiments are invalid when comparing against MEIBP. A fair comparison with MEIBP would necessitate generating the synthetic dataset with a positive loading matrix W.

Other Comments:
- For the block image data, given that the noise value is known and provided to the FAB algorithm, for fairness, the IBP Gibbs sampler should also be provided with the same value, rather than the commonly used 0.75 standard deviation heuristic.
- For VB-IBP [2], the infinite variational version could be used (the experiments utilized a finite variational version). The experimental settings for VB are not described in sufficient detail (e.g., the number of restarts used).
- I am surprised that accelerated Gibbs sampling identified approximately 70 latent features on the 49-dimensional Sonar data. This likely stems from poor initialization and/or an inadequately specified noise variance value.
- Line 70: Reference [21] describes MAP estimation, not Gibbs sampling. Please correct this description.
- Recent work on efficient inference using small-variance asymptotics in nonparametric LFMs exists (see “MAD-Bayes: MAP-based Asymptotic Derivations from Bayes” from ICML 2013). Including a discussion of this work would be valuable.

Minor comments:
- Line 349: For real-data, previous work [1,22] recommended setting the standard deviation to 0.25 times the standard deviation of examples across all dimensions, rather than 0.75.

Comments after the author feedback: The authors’ response addressed some of the initial questions. However, several points require further attention before the paper could be accepted. Specifically:
- The experimental methodology, for both the proposed method and the baselines, needs more detailed explanation. Justifications and explanations should be provided for why certain algorithms exhibited particular behaviors (e.g., Gibbs sampler using a fixed alpha leading to an overestimated K, or suboptimal noise variance hyperparameters).
- Reference [21] describes MAP inference, not sampling. This requires correction.
- Include a reference to the MAD-Bayes inference method for the IBP.

Overall: The proposed inference method for LFMs is interesting, and its scalability to large datasets alongside K-inference is appealing. However, the experimental evaluation needs to be significantly more carefully executed.
