**Summary**

This paper presents an inference algorithm for the binary latent feature model (LFM), where each observation is represented by a K-dimensional binary vector indicating the presence or absence of K latent features. The goal of inference is to learn these binary feature assignments for all observations, along with other model parameters. The proposed EM-based algorithm employs a shrinkage mechanism to simultaneously learn the feature assignments and infer the number of latent features, K. This approach is similar in spirit to nonparametric Bayesian methods like the Indian Buffet Process (IBP).

However, instead of using parsimony-inducing prior distributions, the proposed algorithm relies on a model selection criterion called Factorized Asymptotic Bayesian (FAB) inference, which was recently introduced for mixture models and HMMs. The FAB framework expresses the log marginal likelihood as the sum of the expected complete log-likelihood and a model selection term. This term encourages shrinkage, effectively learning the number of latent features by starting with a specified upper bound on K and pruning unused features during inference. On several large datasets, the FAB-based approach is shown to outperform other inference methods for the IBP, including Gibbs sampling, variational inference, and MAP estimation, in terms of both runtime and inference quality.

**Quality**

The technical quality of the paper is reasonably sound, and the algorithmic details appear to be correct. The method's ability to handle large datasets while automatically inferring the number of latent features is particularly appealing.

**Clarity**

The methodology section is reasonably clear. However, some experimental details lack clarity, as discussed below.

**Originality**

The proposed method builds upon the recently introduced Factorized Asymptotic Bayesian (FAB) inference framework, previously applied to mixture models and HMMs. Its application to latent feature models, however, is novel.

**Significance**

The paper addresses the important problem of performing efficient inference in latent feature models while simultaneously determining the number of latent features.

**Weak Points**

*   The paper lacks a discussion of the proposed algorithm's limitations. For instance, are there scenarios where it might underperform standard Gibbs sampling?
*   The authors do not discuss potential convergence issues, which are common for EM-like procedures.
*   The algorithm is restricted to binary latent feature models and cannot be directly applied to models with continuous latent variables, such as factor analysis or probabilistic PCA.
*   The experimental evaluation is not sufficiently thorough and appears to have some flaws (see detailed comments below).
*   It is unclear how the algorithm will perform in the small-data regime where its asymptotic assumptions may not hold. This point warrants discussion or further experiments.
*   The paper fails to discuss some recent and relevant work on efficient inference for LFMs (see comments below).
*   A significant issue lies in the comparison with the MEIBP algorithm [22]. This baseline is only applicable to non-negative linear-Gaussian models where the loading matrix W is constrained to be positive. The authors, however, use a real-valued W. This makes the comparisons in the artificial simulation and the block data experiments invalid. To provide a fair comparison, the synthetic data should have been generated with a positive loading matrix W.

**Other Comments**

*   In the block images experiment, the noise variance is known and provided to the FAB algorithm. For a fair comparison, the IBP Gibbs sampler should also have been provided with this value, rather than using a heuristic for the standard deviation.
*   The comparison with VB-IBP [2] could be strengthened by using the infinite variational version instead of the finite one. Furthermore, the experimental setup for the VB baseline is insufficiently detailed (e.g., the number of restarts is not mentioned).
*   The discovery of approximately 70 latent features by the accelerated Gibbs sampler on the 49-dimensional Sonar dataset is surprising. I suspect this result may be an artifact of poor initialization or a poorly specified noise variance.
*   Line 70: Reference [21] is incorrectly described as a Gibbs sampling method. Like [22], it is a MAP estimation method for the IBP. This description should be corrected.
*   The related work section would benefit from a discussion of recent research on efficient inference for nonparametric LFMs using small-variance asymptotics, such as "MAD-Bayes: MAP-based Asymptotic Derivations from Bayes" (ICML 2013).

**Minor Comments**

*   Line 349: The description of the heuristic from [1, 22] for setting the noise standard deviation on real data is inaccurate. The suggestion was to use 0.25 times the standard deviation of the data, not a fixed value.

***

**Comments after the author-feedback**

The authors' feedback has addressed some of my concerns. However, several issues must be resolved before the paper can be accepted:

*   The experimental methodology for both the proposed method and the baselines needs to be explained more clearly. The authors should provide justifications for the observed behavior of certain algorithms (e.g., why the Gibbs sampler overestimated K, which may be due to a fixed alpha parameter or poorly set noise variance hyperparameters).
*   The description of reference [21] must be corrected; it performs MAP inference, not Gibbs sampling.
*   The paper should include and discuss the MAD-Bayes work on IBP inference.

Overall, the proposed inference method for LFMs is interesting, and its ability to scale to large datasets while inferring the number of features is a significant advantage. However, the paper's contribution is currently undermined by an experimental evaluation that needs to be conducted more carefully and reported more thoroughly.
