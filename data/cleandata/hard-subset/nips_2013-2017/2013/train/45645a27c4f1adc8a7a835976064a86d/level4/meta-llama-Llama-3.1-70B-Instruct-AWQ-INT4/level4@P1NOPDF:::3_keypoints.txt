- Proposes an EM-based inference algorithm for binary latent feature models using a FAB-based shrinkage term to infer both Z and the number of features K.  
- Method is similar in spirit to nonparametric IBP approaches but does not use priors for parsimony; it uses Factorized Asymptotic Bayesian (FAB) model selection instead.  
- Empirically outperforms Gibbs sampling, variational inference, and MAP IBP methods on some large datasets.  
- Technical and algorithmic details appear sound and scalable to large datasets.  
- Methodology section is reasonably clear but some experimental details are insufficiently explained.  
- Applying FAB to latent feature models is a novel contribution.  
- Addresses the important problem of efficient inference while estimating the number of latent features.  
- Paper lacks discussion of algorithmic limitations and failure cases (e.g., when it may perform worse than Gibbs).  
- No discussion of convergence issues despite using an EM-like procedure.  
- Method is limited to binary latent feature models and cannot handle factor analysis or probabilistic PCA.  
- Experimental evaluation is not thorough and contains flaws that undermine some baseline comparisons.  
- Behavior in small-data regimes (where asymptotic FAB arguments may fail) is not analyzed or experimentally tested.  
- Recent related work on efficient inference for LFMs is not adequately discussed or cited.  
- The MEIBP baseline is inappropriately used for experiments where its non-negativity constraints are violated; synthetic data should have non-negative W for fair comparison.  
- For block images, FAB was given the true noise value but IBP Gibbs was not; baseline noise settings should be matched.  
- Variational IBP experiments need more detail (e.g., use of infinite variational version, number of restarts).  
- The accelerated Gibbs result on the Sonar data (≈70 features) seems suspicious—possibly due to poor initialization or mis-specified noise variance.  
- Reference [21] is mischaracterized: it is MAP inference for the IBP, not Gibbs sampling, and should be corrected.  
- The MAD-Bayes ICML 2013 work on MAP-based asymptotics for nonparametric LFMs should be discussed and cited.  
- After author feedback some issues were addressed, but experimental methodology, baseline settings, and reference corrections remain to be fixed before acceptance.
