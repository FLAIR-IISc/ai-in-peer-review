**Summary**

This paper presents an inference algorithm for the binary latent feature model (LFM), which represents each observation as a K-sized binary vector. The goal of inference is to learn the latent features for all observations, along with other model parameters. The proposed EM-based algorithm utilizes a shrinkage mechanism to learn the latent features and infer the number of latent features (K). This approach is similar in spirit to nonparametric Bayesian latent feature models, such as the Indian Buffet Process (IBP). However, the algorithm employs a model selection criterion, Factorized Asymptotic Bayesian Inference (FAB), rather than prior distributions inducing model parsimony. FAB expresses the log marginal likelihood in terms of the expected complete log-likelihood plus a model selection term that encourages shrinkage of the number of latent features. The algorithm is shown to outperform other inference methods, including Gibbs sampling, variational inference, and MAP estimation methods for the IBP, on some large datasets.

**Quality**

The technical quality of the paper appears to be reasonably sound, and the algorithmic details seem correct. The ability to handle large datasets and learn the number of latent features is particularly appealing.

**Clarity**

The paper is reasonably clear in its methodology, but some experimental details are unclear. Further clarification is necessary to ensure that the reader can fully understand the experimental setup and results.

**Originality**

The proposed method is based on the recently introduced FAB framework, which is a novel application to latent feature models.

**Significance**

The paper addresses an important problem in efficient inference in latent feature models while also inferring the number of latent features.

**Weak Points**

1. The paper lacks a discussion on the limitations of the proposed algorithm, including potential cases where it may perform worse than vanilla Gibbs sampling.
2. There is no discussion on the possible difficulties in convergence, given that it is an EM-like procedure.
3. The algorithm is limited to binary latent feature models and cannot be applied to factor analysis or probabilistic PCA.
4. The experimental evaluation is not thorough enough and seems flawed in some aspects.
5. The behavior of the algorithm in the small-data regime, where the asymptotic argument does not hold, is unclear and requires further discussion or experiments.
6. Recent works on efficient inference in LFMs are not discussed in the paper.
7. The MEIBP algorithm proposed in [22] is inapplicable to the experiments, as it is limited to non-negative linear Gaussian models.

**Other Comments**

* For the block images data, the IBP Gibbs sampler should be given the same noise value as the FAB algorithm for fairness.
* The infinite variational version of VB-IBP could be used, and the experimental settings for VB should be described in more detail.
* The accelerated Gibbs sampling results on the Sonar data seem suspicious, potentially due to bad initialization or poorly specified noise variance.
* Reference [21] should be corrected, as it is about MAP estimation for the IBP, not Gibbs sampling.
* The recent work on MAD-Bayes inference for the IBP should be discussed.

**Minor Comments**

* Line 349: The suggestion in [1,22] was to set the standard deviation equal to 0.25 times the standard deviation of examples across all dimensions, not 0.75.

**Comments after Author Feedback**

The author feedback addressed some of the concerns, but there are still a few issues that need to be fixed or improved before the paper can be accepted:

* Experimental methodology for the proposed method and baseline needs to be better explained, with justifications for the behavior of certain algorithms.
* Reference [21] should be corrected to reflect that it is about MAP inference, not sampling.
* The reference on MAD-Bayes inference for the IBP should be included.
