Summary:
The paper proposes an inference algorithm for binary latent feature models (LFMs), where each observation is represented as a binary vector of size K, indicating the presence or absence of K latent features. The algorithm, based on Expectation-Maximization (EM) and a shrinkage mechanism, aims to learn the binary vectors and infer the number of latent features (K). This approach is similar in spirit to nonparametric Bayesian latent feature models, such as the Indian Buffet Process (IBP), but instead of using prior distributions to induce model parsimony, it employs the Factorized Asymptotic Bayesian (FAB) inference framework. FAB expresses the log marginal likelihood in terms of the expected complete log-likelihood and a model selection term that encourages shrinkage of the number of latent features. The algorithm is shown to outperform other inference methods, including Gibbs sampling, variational inference, and MAP estimation for the IBP, on some large datasets.

Quality: The technical quality of the paper appears sound, and the algorithmic details seem correct. The ability to handle large datasets and learn the number of latent features is a significant advantage.

Clarity: The paper is reasonably clear in its methodology, but some experimental details are unclear and require further explanation.

Originality: The application of FAB to latent feature models is novel, although the framework itself has been recently proposed for mixture models and Hidden Markov Models (HMMs).

Significance: The paper addresses an important problem in efficient inference for latent feature models while also inferring the number of latent features.

Weak Points:

* The paper lacks a discussion on the limitations of the proposed algorithm, including cases where it might perform worse than standard Gibbs sampling.
* There is no discussion on potential convergence difficulties, given the EM-like procedure.
* The algorithm is limited to binary latent feature models and cannot be applied to factor analysis or probabilistic PCA.
* The experimental evaluation is not thorough and seems flawed in some aspects.
* The behavior of the algorithm in the small-data regime, where the asymptotic argument does not hold, is unclear and requires further discussion or experiments.
* The paper does not discuss recent works on efficient inference in LFMs.
* The comparison with the MEIBP algorithm is invalid, as the synthetic dataset does not meet the required conditions for MEIBP.

Other Comments:

* For the block images data, the IBP Gibbs sampler should be given the same noise value as the FAB algorithm for a fair comparison.
* The experimental settings for VB-IBP are not described in sufficient detail, and the infinite variational version could be used instead.
* The accelerated Gibbs sampling results on the Sonar data seem suspicious, possibly due to bad initialization or poorly specified noise variance.
* The reference [21] is incorrect, as it describes MAP estimation for the IBP, not Gibbs sampling.
* The recent work on MAD-Bayes inference for the IBP should be discussed.
* Minor corrections are needed, such as the suggestion in [1,22] for setting the standard deviation.
