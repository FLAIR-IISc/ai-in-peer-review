- Proposes an EM-based inference algorithm for binary latent feature models that uses FAB model selection to shrink and learn the effective number of features K.
- FAB approximates log marginal likelihood as expected complete log-likelihood plus a shrinkage term, allowing K to be learned from an upper bound during inference.
- Reported to outperform Gibbs sampling, variational inference, and MAP IBP methods on large datasets in runtime and inference quality.
- Technical development and algorithmic details appear sound and consistent.
- Attractive for scaling to large datasets while automatically selecting K.
- Methodological exposition is generally clear but some experimental details are insufficient or unclear.
- Application of FAB to latent feature models is novel, building on prior FAB work for mixtures and HMMs.
- Important problem addressed: efficient inference in LFMs with automatic feature number selection.
- No discussion of limitations or settings where the method might underperform vanilla Gibbs sampling.
- No discussion of potential convergence difficulties for an EM-like procedure.
- Restricted to binary LFMs; cannot be applied to factor analysis or probabilistic PCA.
- Experimental evaluation is insufficiently thorough and contains flaws.
- Behavior in small-data regimes (where asymptotics may fail) is unclear and not evaluated.
- Missing discussion of some recent efficient-inference work for LFMs, including small-variance asymptotics (MAD-Bayes).
- MEIBP baseline comparison is invalid because MEIBP requires nonnegative W; synthetic and block-image experiments used real-valued W.
- For fairness, IBP Gibbs sampler should be given the same known noise variance used by FAB in block-image experiments.
- VB-IBP experiments used a finite variational form; infinite variational formulation could be used and VB experiment details (e.g., restarts) are missing.
- Surprising result: accelerated Gibbs found ~70 features on a 49-dim Sonar dataset, suggesting poor initialization or wrong noise variance.
- Incorrect citation: Reference [21] is MAP estimation for IBP, not Gibbs sampling (also true for [22]); fix text.
- Minor correction: recommended noise std-dev for real data is 0.25 of empirical std-dev, not 0.75.
- After author feedback, experimental methodology still needs clearer description and justification (e.g., Gibbs alpha, noise variance hyperparameters).
- Must include and discuss MAD-Bayes reference for IBP before acceptance.
