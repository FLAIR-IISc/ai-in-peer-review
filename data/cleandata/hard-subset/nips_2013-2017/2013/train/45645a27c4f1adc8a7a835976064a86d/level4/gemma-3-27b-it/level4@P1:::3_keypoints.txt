- Proposes an EM-based FAB inference algorithm for binary latent feature models that learns z_n and automatically shrinks/infers K.  
- Motivated by IBP but leverages the Factorized Asymptotic Bayesian (FAB) framework rather than nonparametric Bayes.  
- Experiments claim FAB outperforms Gibbs sampling, variational inference, and MAP IBP in runtime and inference quality on large datasets.  
- Technical and algorithmic details appear sound; method scales to large datasets and learns the number of features.  
- Paper is generally clear methodologically but several experimental details need clarification.  
- Original contribution: novel application of the FAB framework to latent feature models.  
- Significance: addresses efficient inference and automatic model selection in LFMs.  
- Missing discussion of algorithm limitations and scenarios where it may perform worse than Gibbs sampling.  
- No discussion of potential convergence difficulties inherent to EM-like procedures.  
- Method is restricted to binary latent feature models and cannot be directly applied to factor analysis or probabilistic PCA.  
- Experimental evaluation is insufficient and contains potentially flawed comparisons.  
- Unclear behavior in small-data regimes where asymptotic arguments may not hold; needs discussion or experiments.  
- Fails to discuss some recent relevant work on efficient inference for LFMs (e.g., MAD-Bayes).  
- Comparison with MEIBP is inappropriate because MEIBP requires non-negative W; synthetic data should use positive W for fair comparison.  
- For block image data, IBP Gibbs should use the same noise value as FAB for a fair comparison.  
- VB-IBP experiments used a finite variational version and lack details (e.g., number of restarts); an infinite variational version could be used.  
- Gibbs finding ~70 features on 49-dimensional Sonar data is suspicious and may indicate poor initialization or mis-specified noise variance.  
- Reference [21] is mischaracterized as Gibbs sampling when it describes MAP inference; text should be corrected.  
- Recommendation in refs [1,22] for setting standard deviation was 0.25×data std, not a fixed 0.75; correct this.  
- After author feedback, some concerns remain: experimental methodology still needs more elaboration and justifications for observed behaviors.  
- Authors should correct reference errors and include MAD-Bayes discussion.  
- Strengths for acceptance: novel FAB application to LFMs, scalability, automatic K selection, competitive empirical performance.  
- Reasons against acceptance: incomplete/flawed experiments, limited discussion of limitations and convergence, restricted to binary LFMs, insufficient engagement with recent related work.
