**Summary**

This paper introduces an inference algorithm for the binary latent feature model (LFM), where each observation is represented by a K-dimensional binary vector indicating the presence or absence of K latent features. The primary inference task is to learn these binary feature vectors for all observations, along with other model parameters. The proposed EM-based algorithm employs a shrinkage mechanism not only to learn the feature assignments but also to automatically infer the total number of latent features, K. This goal is similar to that of nonparametric Bayesian methods like the Indian Buffet Process (IBP). However, instead of using priors that induce parsimony, the algorithm leverages a model selection criterion—Factorized Asymptotic Bayesian Inference (FAB)—which was recently applied to mixture models and HMMs. The FAB framework approximates the log marginal likelihood with the expected complete log-likelihood plus a model selection term that penalizes model complexity, thereby shrinking the number of active latent features from a specified upper bound. Experiments on several large datasets suggest that the FAB-based approach can outperform alternative inference methods for the IBP, including Gibbs sampling, variational inference, and MAP estimation, in terms of both runtime and inference quality.

**Quality**

The technical quality appears sound, and the algorithmic details seem correct. The method's ability to handle large datasets while simultaneously learning the number of latent features is a significant advantage.

**Clarity**

The methodology is presented with reasonable clarity. However, several experimental details are unclear, as discussed below.

**Originality**

The proposed method builds upon the recently introduced Factorized Asymptotic Bayesian (FAB) Inference framework. While FAB has been applied to mixture models and HMMs, its application to latent feature models is novel.

**Significance**

The paper addresses the important and challenging problem of performing efficient inference in latent feature models while also determining the model's dimensionality.

**Weak Points**

*   A discussion on the algorithm's limitations is absent. For instance, it is unclear if there are scenarios where it might underperform standard methods like Gibbs sampling.
*   Potential convergence issues, which are common in EM-like procedures, are not discussed.
*   The algorithm is restricted to binary latent features and is not applicable to models with real-valued factors, such as factor analysis or probabilistic PCA.
*   The experimental evaluation lacks thoroughness and contains apparent flaws (detailed below).
*   The paper does not address how the algorithm is expected to perform in a small-data regime, where the asymptotic justification for FAB may not hold. This should be discussed or demonstrated experimentally.
*   Several recent works on efficient inference for LFMs are not discussed or cited (see comments below).
*   The comparison with the MEIBP algorithm [22] appears invalid. MEIBP is applicable only to non-negative linear Gaussian models (i.e., with a positive loading matrix W). Consequently, the experiments on the artificial simulation and the block data, which use a real-valued W, are not suitable for this baseline. A valid comparison would require generating synthetic data with a positive loading matrix.

**Other Comments**

*   For the block images dataset, to ensure a fair comparison, the known noise value provided to the FAB algorithm should also have been provided to the IBP Gibbs sampler, instead of relying on the 0.75 standard deviation heuristic.
*   The experiments with VB-IBP [2] used the finite variational version, whereas the infinite version might have been more appropriate. Furthermore, the experimental details for the VB baseline are insufficient (e.g., the number of restarts is not specified).
*   The discovery of approximately 70 latent features by the accelerated Gibbs sampler on the 49-dimensional Sonar dataset is surprising and may suggest issues with initialization or a poorly specified noise variance.
*   Line 70: The description of reference [21] is incorrect; it pertains to MAP estimation for the IBP, not Gibbs sampling. The text should be corrected.
*   A discussion of recent work on small-variance asymptotic inference for nonparametric LFMs, such as "MAD-Bayes: MAP-based Asymptotic Derivations from Bayes" (ICML 2013), would strengthen the paper's positioning.

**Minor comments**

*   Line 349: The suggestion in [1, 22] for real-world data was to set the standard deviation to 0.25 times the standard deviation of the data across all dimensions, not a fixed value of 0.75.

**Comments after the author-feedback**

While the author feedback addressed some concerns, several issues must be resolved for the paper to be acceptable for publication. Specifically:

*   The experimental methodology for both the proposed method and the baselines requires a more detailed explanation. The paper should also provide justifications for the observed behavior of certain algorithms (e.g., why the Gibbs sampler might have overestimated K, possibly due to a fixed alpha or improperly set noise hyperparameters).
*   The text must be corrected to reflect that reference [21] describes MAP inference, not sampling.
*   The MAD-Bayes paper should be cited and discussed.

In summary, the proposed inference method is compelling due to its novelty and scalability. However, its appeal is currently undermined by an experimental evaluation that requires significant improvement.
