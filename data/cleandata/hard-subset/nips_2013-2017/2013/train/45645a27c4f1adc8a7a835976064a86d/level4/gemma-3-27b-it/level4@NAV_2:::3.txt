The paper presents an inference algorithm for the binary latent feature model (LFM) which expresses each observation xn as a K size binary vector zn where z{nk} = 1 means that xn has the k-th latent feature present. The goal of inference is to learn the z's for all observations (along with the other model parameters). The presented EM-based algorithm uses a shrinkage mechanism to learn the z's as well as infer the number of latent features (K). This is similar in spirit to nonparametric Bayesian latent feature models such as the Indian Buffet Process (IBP). However, the presented algorithm doesn't actually use prior distributions inducing model parsimony but rather uses a model selection criteria – Factorized Asymptotic Bayesian Inference (FAB) – recently introduced in the context of mixture models and HMMs. The FAB idea is based on expressing the log marginal likelihood in terms of the expected complete log-likelihood plus a model selection term that encourages shrinkage of the number of latent features (by specifying an upper-bound on K and learning the correct K during inference). On some large datasets, the FAB is shown to outperform other inference methods including Gibbs sampling, variational inference, and MAP estimation methods for the IBP (in terms of running time and inference quality). 

The technical quality seems reasonably sound and algorithmic details seem correct. The ability to deal with large datasets and also learn the number of latent features is very appealing. 

The paper is reasonably clear in the methodology part. However, some of the experimental details are somewhat unclear.

The proposed method is based on a recently proposed framework on Factorized Asymptotic Bayesian (FAB) Inference applied to mixture models and HMMs. The application of FAB to latent feature models, however, is novel.

The paper is addressing an important problem – efficient inference in latent feature models while also inferring the number of latent features. 

**Weak Points:**

- There is no comprehensive discussion of the limitations of the proposed algorithm. Are there scenarios where the algorithm might perform worse than vanilla Gibbs sampling, and if so, under what conditions?
- The potential difficulties in convergence, given it is an EM-like procedure, are not addressed. What guarantees can be provided regarding convergence to a local optimum, and how sensitive is it to initialization?
- The algorithm is currently limited to binary latent feature models and cannot be directly applied to factor analysis or probabilistic PCA. Is there potential for generalization, and if not, why?
- The experimental evaluation lacks thoroughness. Reporting standard deviations across multiple runs would strengthen the conclusions.
- The behavior of the algorithm in the small-data regime (where the asymptotic argument may not hold) is unclear. Discussion or experiments exploring this regime are needed.
- Several recent works on efficient inference in LFMs are not discussed (see comments below).
- The MEIBP algorithm [22] is only applicable for non-negative linear Gaussian models (W must be positive, not real-valued). Consequently, the artificial simulation and block data experiments are potentially invalid when using MEIBP as a baseline. If comparing with MEIBP, the synthetic dataset should be generated with a positive loading matrix W.

**Other Comments:**

- For the block images data, fairness dictates that the IBP Gibbs sampler should be provided with the same noise value as FAB, rather than the 0.75 std-dev heuristic.
- For VB-IBP [2], the infinite variational version could be considered, as the experiments used a finite variational version. The experimental settings for VB are not detailed enough (e.g., number of restarts).
- The reported ~70 latent features discovered by accelerated Gibbs sampling on the 49-dimensional Sonar data is surprising. This likely stems from poor initialization or an improperly specified noise variance value.
- Line 70: Reference [21] describes MAP inference, not Gibbs sampling. Please correct the text.
- The paper would benefit from discussing "MAD-Bayes: MAP-based Asymptotic Derivations from Bayes" (ICML 2013), a related work on efficient inference using small-variance asymptotics for nonparametric LFMs.

**Minor comments:**

- Line 349: For real-data, the suggestion in [1, 22] was to set the standard deviation to 0.25 times the standard deviation of examples across all dimensions, rather than a fixed 0.75.

**Comments after the author feedback:** The authors’ responses to the initial review addressed several points, providing further clarity and justification for their methodological choices. However, several key areas still require attention before the paper can be considered for acceptance.

- The experimental methodology – both for the proposed method and the baselines – needs to be more comprehensively explained. Justifications for the parameter settings used and explanations for the observed behavior of different algorithms (e.g., why Gibbs sampling overestimated K, or why specific noise variance values were chosen) are crucial.
- Reference [21] still incorrectly describes the method as Gibbs sampling; it performs MAP inference. This must be corrected.
- The reference to "MAD-Bayes" inference for the IBP should be included.

The proposed inference method for LFMs is interesting, and its ability to scale to large datasets while simultaneously inferring the number of latent features is a significant contribution. However, a more rigorous and carefully conducted experimental evaluation is necessary to fully support the claims made in the paper.



**Overall Assessment:** Reject with request for major revisions. The core idea is promising, but the experimental validation and clarity require substantial improvement before this paper can be considered for publication.
