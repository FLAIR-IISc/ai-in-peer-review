**Summary:**
This paper introduces an inference algorithm for binary latent feature models (LFMs), where each observation is represented by a K-dimensional binary vector indicating the presence or absence of K latent features. The central task is to infer these binary feature vectors (z) for all observations and learn the other model parameters. The authors propose an EM-based algorithm that utilizes a shrinkage mechanism to learn the feature assignments while simultaneously inferring the number of latent features (K). This approach is conceptually similar to nonparametric Bayesian methods like the Indian Buffet Process (IBP). However, rather than using priors to induce model parsimony, the algorithm employs a model selection criterion called Factorized Asymptotic Bayesian (FAB) Inference, which was recently developed for mixture models and HMMs. The FAB method works by expressing the log marginal likelihood as the sum of the expected complete log-likelihood and a model selection term that encourages a smaller number of active latent features, thus learning the correct K from a specified upper bound. The experimental results on several large datasets suggest that the FAB approach outperforms alternative inference techniques for the IBP, including Gibbs sampling, variational inference, and MAP estimation, in terms of both runtime and inference quality.

**Quality:**
The technical quality of the work appears to be sound, and the details of the algorithm seem correct. The method's ability to handle large datasets while also learning the model dimensionality (K) is a significant advantage.

**Clarity:**
The methodological exposition is generally clear. However, certain aspects of the experimental setup lack clarity, as detailed in the comments below.

**Originality:**
The proposed method builds upon the recently introduced Factorized Asymptotic Bayesian (FAB) inference framework. While the framework itself is not new, its application to latent feature models represents a novel contribution.

**Significance:**
The paper tackles the significant challenge of performing efficient inference in latent feature models while concurrently determining the number of features, which is an important problem in the field.

**Weak Points:**
*   The paper lacks a discussion on the potential limitations of the proposed algorithm. For instance, it is not clear under what conditions it might underperform relative to standard Gibbs sampling.
*   There is no treatment of potential convergence issues, which can be a concern for EM-like procedures.
*   The algorithm's applicability is restricted to binary LFMs and does not extend to continuous models like factor analysis or probabilistic PCA.
*   The empirical evaluation is not sufficiently rigorous and contains several potential flaws (see specific comments).
*   The behavior of the algorithm in the small-data regime, where the asymptotic assumptions of FAB may not hold, is not discussed or evaluated.
*   The paper omits discussion of some other recent contributions to efficient inference in LFMs.
*   The comparison with the MEIBP algorithm [22] appears to be invalid. MEIBP is designed for non-negative linear Gaussian models (i.e., a positive loading matrix W), whereas the artificial simulation and block data experiments do not adhere to this constraint. A valid comparison would require generating a synthetic dataset with a positive loading matrix.

**Other Comments:**
*   In the block images data experiment, the noise value was known and provided to the FAB algorithm. To ensure a fair comparison, this same value should have been given to the IBP Gibbs sampler instead of using the 0.75 std-dev heuristic.
*   For the VB-IBP baseline [2], the experiments employed a finite variational version, whereas the infinite variational version could have been used. The experimental settings for VB are also insufficiently described (e.g., the number of restarts is not mentioned).
*   The result that accelerated Gibbs sampling discovered approximately 70 latent features on the 49-dimensional Sonar data is surprising and may suggest issues with initialization or a poorly specified noise variance.
*   Line 70: Reference [21] pertains to a MAP estimation method for the IBP, similar to reference [22], not Gibbs sampling. The text should be corrected to reflect this.
*   The discussion would benefit from including recent work on efficient inference for nonparametric LFMs using small-variance asymptotics, such as "MAD-Bayes: MAP-based Asymptotic Derivations from Bayes" (ICML 2013).

**Minor comments:**
*   Line 349: For real-world data, the recommendation in [1, 22] was to set the standard deviation to 0.25 times the standard deviation of the data across all dimensions, not to a fixed value of 0.75.

**Comments after the author-feedback:**
The author feedback has clarified some points. However, several issues must be addressed before the paper is suitable for acceptance. Specifically:
*   The experimental methodology for both the proposed method and the baselines requires more detailed explanation. The paper should also provide justifications for the observed behavior of certain algorithms (e.g., why the Gibbs sampler might have overestimated K due to a fixed alpha, or if noise hyperparameters were improperly set).
*   The description of reference [21] must be corrected to state that it performs MAP inference, not sampling.
*   The paper should incorporate and discuss the MAD-Bayes reference.

In summary, while the proposed scalable inference method for LFMs is promising and its ability to infer K is appealing, its contribution is currently undermined by an experimental evaluation that needs to be performed more carefully.
