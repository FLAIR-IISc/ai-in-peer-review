- Proposes an EM-based inference algorithm with a shrinkage mechanism for binary latent feature models to learn feature vectors and infer K.
- Uses the Factorized Asymptotic Bayesian (FAB) framework to induce model parsimony instead of nonparametric priors like the IBP.
- Claims improved performance over Gibbs sampling, variational inference, and MAP IBP on some large datasets.
- Technical presentation appears sound and algorithmic details seem correct.
- Algorithm scales to large datasets and can infer the number of latent features automatically.
- Methodology is reasonably clear, but several experimental details are insufficiently explained.
- Applying FAB to latent feature models is novel, though FAB itself has prior applications to mixture models and HMMs.
- Addresses a significant problem: efficient inference in LFMs while selecting the number of features.
- Paper omits discussion of algorithmic limitations and scenarios where it might underperform compared to Gibbs sampling.
- No discussion of potential convergence issues inherent to EM-like procedures.
- Approach is restricted to binary LFMs and cannot be applied to continuous-factor models like factor analysis or probabilistic PCA.
- Experimental evaluation is incomplete and contains methodological flaws.
- Behavior in small-data regimes (where asymptotic FAB justifications break down) is unclear and untested.
- Fails to discuss recent related work on efficient inference for LFMs.
- Comparison to MEIBP is invalid because the synthetic data used do not meet MEIBP’s required conditions.
- For block images, IBP Gibbs should use the same noise setting as FAB for fair comparison.
- VB-IBP experimental settings are under-specified; the infinite variational version could be considered instead.
- Accelerated Gibbs results on Sonar look suspicious, possibly due to bad initialization or incorrect noise variance.
- Reference [21] is incorrect—it describes MAP inference for the IBP, not Gibbs sampling.
- Recent MAD-Bayes work for the IBP is not discussed and should be included.
- Minor corrections suggested, e.g., recommended standard-deviation setting from cited works.
- After author feedback some concerns were addressed, but key issues remain: experimental methodology and justification of algorithmic choices need clearer explanation.
- Remaining fixes before acceptance: correct reference [21], include MAD-Bayes citation, and conduct a more careful experimental evaluation.
