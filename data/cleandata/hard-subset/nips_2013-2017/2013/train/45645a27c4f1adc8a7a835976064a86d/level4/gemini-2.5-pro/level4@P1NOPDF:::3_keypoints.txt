- Proposes an EM-based inference algorithm for binary latent feature models that uses FAB shrinkage to learn feature assignments and infer K from an initial upper bound.  
- FAB framework replaces parsimony priors with a model-selection term that prunes unused features; authors report better runtime and inference quality than IBP Gibbs, VI, and MAP on large datasets.  
- Technical execution appears sound and the method scales to large datasets while inferring the number of latent features.  
- Methodology section is reasonably clear, but experimental details are under-specified or unclear.  
- Novelty lies in applying the recently introduced FAB inference framework to latent feature models.  
- Significance: addresses efficient inference and simultaneous model selection for LFMs.  
- Paper lacks discussion of algorithmic limitations and scenarios where it might underperform standard Gibbs sampling.  
- No discussion of potential convergence issues inherent to EM-like procedures.  
- Method is restricted to binary latent features and cannot be directly applied to continuous latent-variable models (e.g., factor analysis, probabilistic PCA).  
- Experimental evaluation is insufficiently thorough and contains flaws.  
- Unclear performance in small-data regimes where FAB’s asymptotic assumptions may fail; needs discussion or experiments.  
- Fails to discuss some recent relevant work on efficient inference for LFMs (e.g., small-variance asymptotics / MAD-Bayes).  
- Invalid comparison with MEIBP: MEIBP assumes non-negative loading matrix W whereas authors used real-valued W, making those baselines unfair for some experiments.  
- In block images experiment, FAB was given known noise variance while IBP Gibbs was not—unfair experimental setup.  
- VB-IBP baseline comparison is weak: should use infinite variational version and report details like number of restarts.  
- Surprising result of ~70 latent features on 49-d Sonar dataset likely an artifact of poor initialization or mis-specified noise variance.  
- Reference [21] is mischaracterized as Gibbs sampling; it is a MAP estimation method and must be corrected.  
- Related work section should include MAD-Bayes (ICML 2013) and other small-variance asymptotics approaches for IBP/LFM inference.  
- Minor correction: heuristic for noise standard deviation from [1,22] was misdescribed—the suggestion was 0.25 times the data standard deviation, not a fixed value.  
- After author feedback: some concerns addressed, but remaining issues include unclear experimental methodology, need to correct ref [21], and inclusion/discussion of MAD-Bayes; overall experimental reporting must be improved before acceptance.
