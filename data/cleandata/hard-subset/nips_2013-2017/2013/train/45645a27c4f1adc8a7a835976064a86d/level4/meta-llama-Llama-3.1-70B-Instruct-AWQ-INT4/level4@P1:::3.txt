Summary:
The paper presents an inference algorithm for binary latent feature models (LFMs), which express each observation as a binary vector indicating the presence or absence of latent features. The algorithm uses a shrinkage mechanism to learn the latent features and infer the number of features (K), similar in spirit to nonparametric Bayesian latent feature models like the Indian Buffet Process (IBP). However, the algorithm employs a model selection criterion, Factorized Asymptotic Bayesian Inference (FAB), recently introduced for mixture models and HMMs. FAB expresses the log marginal likelihood in terms of the expected complete log-likelihood plus a model selection term that encourages shrinkage of the number of latent features. The algorithm is shown to outperform other inference methods, including Gibbs sampling, variational inference, and MAP estimation for the IBP, on large datasets.

Quality:
The technical quality of the paper appears sound, and the algorithmic details seem correct. The ability to handle large datasets and learn the number of latent features is appealing. However, some limitations and potential issues with the algorithm are not discussed, such as cases where it may perform worse than vanilla Gibbs sampling, convergence difficulties, and the restriction to binary latent feature models.

Clarity:
The paper is reasonably clear in the methodology part, but some experimental details are unclear. For example, the experimental evaluation could be more thorough, and some recent works on efficient inference in LFMs are not discussed.

Originality:
The application of FAB to latent feature models is novel, although the FAB framework itself was recently proposed for mixture models and HMMs.

Significance:
The paper addresses an important problem: efficient inference in latent feature models while also inferring the number of latent features.

Weak Points:

* There is no discussion about the limitations of the proposed algorithm, such as cases where it may perform worse than vanilla Gibbs sampling.
* There is no discussion about potential convergence difficulties, given that it is an EM-like procedure.
* The algorithm is limited to binary latent feature models and cannot be applied to factor analysis or probabilistic PCA.
* The experimental evaluation is not thorough enough and seems flawed at some points.
* The behavior of the algorithm in the small-data regime, where the asymptotic argument does not hold, is unclear.
* Some recent works on efficient inference in LFMs are not discussed.
* The MEIBP algorithm proposed in [22] is applicable only to non-negative linear Gaussian models, making the artificial simulation and block data experiments invalid for the MEIBP baseline.

Other Comments:

* For block images data, the IBP Gibbs sampler should also be given the known noise value for fairness.
* For VB-IBP [2], the infinite variational version could be used, and the experimental settings should be described in more detail.
* The accelerated Gibbs sampling discovered about 70 latent features on the 49-dimensional Sonar data, which seems suspicious and may be due to bad initialization or a poorly specified noise variance value.
* Line 70: The reference [21] is not about Gibbs sampling but rather MAP estimation for the IBP, and the text should be corrected.
* Recent work on efficient inference using small-variance asymptotics in nonparametric LFMs, such as "MAD-Bayes: MAP-based Asymptotic Derivations from Bayes" from ICML 2013, should be discussed.

Minor Comments:

* Line 349: For real-data, the suggestion in [1,22] was not to set the standard deviation equal to 0.75 but to set it equal to 0.25 times the standard deviation of examples across all dimensions.

After the author-feedback, some points still need to be addressed:

* Experimental methodology for the proposed method and the baseline should be better explained, and justifications/explanations should be provided for the behavior of certain algorithms.
* Reference [21] should be corrected to reflect that it performs MAP inference, not sampling.
* The reference on MAD-Bayes inference for the IBP should be included.
