Summary:

The paper presents an inference algorithm for the binary latent feature model (LFM), where each observation $x_n$ is represented as a K-dimensional binary vector $z_n$. The $k^{th}$ element of $z_n$, denoted as $z_{nk}$, equals 1 if the $k^{th}$ latent feature is present in $x_n$. The goal of inference is to learn the $z$'s for all observations, along with other model parameters. The proposed EM-based algorithm uses a shrinkage mechanism to learn the $z$'s and infer the number of latent features (K). This approach is similar in spirit to nonparametric Bayesian latent feature models, such as the Indian Buffet Process (IBP). However, the proposed algorithm does not use prior distributions to induce model parsimony. Instead, it relies on a model selection criterion, Factorized Asymptotic Bayesian Inference (FAB), which was recently introduced in the context of mixture models and HMMs. FAB expresses the log marginal likelihood in terms of the expected complete log-likelihood plus a model selection term that encourages shrinkage of the number of latent features. The algorithm is shown to outperform other inference methods, including Gibbs sampling, variational inference, and MAP estimation methods for the IBP, on some large datasets.

Quality:

The technical quality of the paper appears sound, and the algorithmic details seem correct. The ability to handle large datasets and learn the number of latent features is particularly appealing.

Clarity:

The paper is reasonably clear in its methodology section. However, some experimental details are unclear and require further explanation.

Originality:

The proposed method is based on the recently proposed framework of Factorized Asymptotic Bayesian (FAB) Inference, which has been applied to mixture models and HMMs. The application of FAB to latent feature models, however, is novel.

Significance:

The paper addresses an important problem, namely efficient inference in latent feature models while also inferring the number of latent features.

Weak Points:

* There is no discussion on the limitations of the proposed algorithm. Are there cases where the algorithm might perform worse than vanilla Gibbs sampling?
* There is no discussion on the potential difficulties in convergence, given that it is an EM-like procedure.
* The algorithm is limited to binary latent feature models and cannot be applied to factor analysis or probabilistic PCA.
* The experimental evaluation is not thorough enough and appears flawed in some aspects.
* In the small-data regime, where the asymptotic argument does not hold, it is unclear how the algorithm will behave. Some discussion or experiments are needed to address this.
* Recent works on efficient inference in LFMs have not been discussed in the paper.
* The MEIBP algorithm proposed in [22] is applicable only to non-negative linear Gaussian models, which means that the artificial simulation and block data experiments are invalid for the MEIBP baseline. If a comparison with MEIBP is desired, the synthetic dataset should be generated such that the loading matrix W is positive.

Other Comments:

* For the block images data, since the noise value is known and given to the FAB algorithm, it would be fair to provide the same value to the IBP Gibbs sampler instead of using the 0.75 standard deviation heuristic.
* For VB-IBP [2], the infinite variational version could be used, and the experimental settings for VB should be described in more detail (e.g., the number of restarts).
* The performance of accelerated Gibbs sampling on the 49-dimensional Sonar data, which discovered around 70 latent features, seems suspicious. This might be due to poor initialization or an incorrectly specified noise variance value.
* Line 70: Reference [21] is not about Gibbs sampling but rather MAP estimation for the IBP (similar to reference [22]). Please modify the text and correct the description.
* Recent work on efficient inference using small-variance asymptotics for nonparametric LFMs (see "MAD-Bayes: MAP-based Asymptotic Derivations from Bayes" from ICML 2013) should be discussed.
* Minor comment: Line 349: For real data, the suggestion in [1, 22] was not to set the standard deviation equal to 0.75 but to set it equal to 0.25 times the standard deviation of examples across all dimensions.

Comments after Author Feedback:

The author feedback addressed some of the questions, but there are still a few issues that need to be fixed or improved before the paper can be accepted. Specifically:

* Experimental methodology for the proposed method and the baseline needs to be better explained, and justifications or explanations are needed for why certain algorithms behaved in a certain way (e.g., the Gibbs sampler used a fixed alpha that might have led to an overestimated K, or noise variance hyperparameters were not properly set).
* Reference [21] should be corrected to reflect that it performs MAP inference, not sampling.
* The reference on MAD-Bayes inference for the IBP should be included. The proposed inference method for LFMs is interesting, and its ability to scale to large datasets (in addition to inferring K) is appealing. However, the experimental evaluation should have been more carefully done.
