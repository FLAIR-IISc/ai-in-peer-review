Summary:
The paper presents an inference algorithm for the binary latent feature model (LFM), where each observation xn is associated with a K-dimensional binary vector zn, and znk = 1 indicates that xn exhibits the k-th latent feature. The goal is to infer the zn for all observations (along with other model parameters). The proposed EM-based algorithm uses a shrinkage mechanism to infer both the latent features and the number of features K. This is similar in spirit to nonparametric Bayesian LFMs such as the Indian Buffet Process (IBP), but the method does not rely on priors that induce parsimony. Instead, it uses a model selection criterion—Factorized Asymptotic Bayesian (FAB) inference—recently introduced for mixture models and HMMs. FAB approximates the log marginal likelihood as an expected complete log-likelihood plus a model selection term that encourages shrinkage in the number of features, starting from an upper bound on K and learning the effective K during inference. On some large datasets, FAB is reported to outperform Gibbs sampling, variational inference, and MAP estimation methods for the IBP in both runtime and inference quality.

Quality:
The technical development appears sound, and the algorithmic details are consistent. Handling large datasets while automatically learning K is particularly attractive.

Clarity:
The methodological exposition is generally clear, but some experimental details are insufficient or unclear (see below).

Originality:
The method builds on the recently proposed FAB framework for mixture models and HMMs. Its application to latent feature models appears novel.

Significance:
The paper addresses an important problem: efficient inference in LFMs with automatic selection of the number of latent features.

Weak Points:
- No discussion of limitations. Are there settings where the algorithm underperforms vanilla Gibbs sampling?
- No discussion of potential convergence difficulties for an EM-like procedure.
- The algorithm is restricted to binary LFMs and cannot be applied to factor analysis or probabilistic PCA.
- The experimental evaluation is not sufficiently thorough and appears flawed in places (see below).
- In the small-data regime (where asymptotic arguments may not hold), the behavior of the algorithm is unclear; this warrants discussion or experiments.
- Some recent work on efficient inference in LFMs is missing from the discussion (see below).
- The MEIBP algorithm in [22] applies only to nonnegative linear Gaussian models (W must be nonnegative, not real-valued). Thus, the artificial simulation and block-image experiments are not valid comparisons for the MEIBP baseline. If comparing to MEIBP, the synthetic dataset should be generated with a nonnegative loading matrix W.

Other Comments:
- For the block-image data, since the noise variance is known and provided to FAB, the IBP Gibbs sampler should, for fairness, be given the same value (rather than the 0.75 std-dev heuristic).
- For VB-IBP [2], the infinite variational formulation could be used; the experiments used a finite variational version. The VB experimental settings lack detail (e.g., how many random restarts?).
- I am surprised that the accelerated Gibbs sampler discovered about 70 latent features on the 49-dimensional Sonar dataset. I suspect poor initialization and/or an improperly specified noise variance.
- Line 70: Reference [21] is not about Gibbs sampling but MAP estimation for the IBP (as is [22]). Please correct the text and description.
- Please discuss recent work using small-variance asymptotics for nonparametric LFMs, e.g., “MAD-Bayes: MAP-based Asymptotic Derivations from Bayes” (ICML 2013).

Minor Comment:
- Line 349: For real data, [1,22] recommend setting the noise std-dev to 0.25 times the empirical std-dev of examples across all dimensions, not 0.75.

Comments after author feedback:
The feedback addressed some questions. A few issues should be fixed or improved before acceptance:
- The experimental methodology (for both the proposed method and baselines) needs clearer description, with justifications for observed behavior. For example, did the Gibbs sampler use a fixed alpha that might have overestimated K? Were noise variance hyperparameters appropriately set?
- Reference [21] performs MAP inference, not sampling; please correct this.
- Please include and discuss the MAD-Bayes reference for IBP.

Overall, the proposed inference method for LFMs is interesting, and its ability to scale to large datasets while inferring K is appealing. However, the experimental evaluation needs to be more carefully designed and documented.
