**Summary**

This paper introduces an EM-based inference algorithm for the binary latent feature model (LFM), where each observation is represented by a binary vector indicating the presence or absence of a set of K latent features. The primary goal is to infer these binary feature vectors for all observations while also learning the other model parameters. A key contribution of the proposed algorithm is a shrinkage mechanism that allows it to infer the number of latent features, K, from the data. This goal is shared by nonparametric Bayesian approaches like the Indian Buffet Process (IBP).

However, instead of using priors that induce parsimony, this method leverages the Factorized Asymptotic Bayesian (FAB) inference framework, a model selection criterion recently applied to mixture models and HMMs. The FAB approach approximates the log marginal likelihood with the expected complete log-likelihood plus a penalty term that encourages a smaller number of latent features. The algorithm starts with an upper bound on K and learns the appropriate value during inference. Experimental results on several large datasets show that the FAB-based method can outperform other LFM inference techniques, including Gibbs sampling, variational inference, and MAP estimation for the IBP, in terms of both runtime and inference quality.

**Quality**

The technical quality of the paper appears sound, and the algorithmic details seem correct. The ability to handle large datasets while simultaneously learning the number of latent features is a very appealing property.

**Clarity**

The paper's methodology section is reasonably clear. However, some experimental details are ambiguous, as discussed below.

**Originality**

The proposed method is based on the recently introduced Factorized Asymptotic Bayesian (FAB) framework. While FAB has been applied to mixture models and HMMs before, its application to latent feature models is novel.

**Significance**

The paper addresses the important and challenging problem of performing efficient inference in latent feature models while automatically determining model complexity.

**Weak Points**

*   The paper lacks a discussion of the proposed algorithm's limitations. For example, are there scenarios where it might perform worse than standard methods like vanilla Gibbs sampling?
*   There is no mention of potential convergence issues, which can be a concern for EM-like procedures.
*   The algorithm is restricted to binary latent features and cannot be directly applied to continuous models like factor analysis or probabilistic PCA.
*   The experimental evaluation is not sufficiently thorough and appears to have some flaws (see comments below).
*   The algorithm's performance in the small-data regime, where its asymptotic justification may not hold, is unclear. A discussion or experiments on this topic would be beneficial.
*   The paper does not discuss some relevant recent work on efficient inference for LFMs.
*   The comparison with MEIBP [22] seems invalid. The MEIBP algorithm is designed for non-negative linear Gaussian models (i.e., the loading matrix W must be non-negative). The artificial and block data experiments use a real-valued W, making them an inappropriate baseline for comparison. To compare fairly with MEIBP, the synthetic data should have been generated with a non-negative loading matrix.

**Other Comments**

*   For the block images experiment, the true noise value was provided to the FAB algorithm. For a fair comparison, the IBP Gibbs sampler should also have been given this value instead of relying on a heuristic for the noise standard deviation (0.75).
*   The experiments used a finite variational version of VB-IBP [3]. The infinite variational version could have been used instead. Furthermore, the experimental settings for VB are not described in sufficient detail (e.g., how many restarts were used?).
*   I am surprised that accelerated Gibbs sampling found approximately 70 latent features for the 49-dimensional Sonar dataset. I suspect this may be due to poor initialization or an incorrectly specified noise variance.
*   The paper seems to mischaracterize reference [22] (MEIBP) as a Gibbs sampling method in the related work section. This reference, like [1], describes a MAP estimation method for the IBP. Please correct the description.
*   Recent work on efficient inference for nonparametric LFMs using small-variance asymptotics ("MAD-Bayes: MAP-based Asymptotic Derivations from Bayes," ICML 2013) should be discussed.

**Minor Comments**

*   In the experimental setup for real data, the authors state they follow the suggestion in [1, 22] by setting the noise standard deviation to 0.75. The suggestion in those papers was to set it to 0.25 times the standard deviation of the data, not a fixed value of 0.75.

***

**Comments after author-feedback**

The author feedback answered some of my questions. However, several issues need to be addressed before the paper can be accepted. In particular:

*   The experimental methodology for both the proposed method and the baselines needs to be explained more clearly. The paper should also provide justifications for the observed behavior of certain algorithms (e.g., why the Gibbs sampler might have overestimated K, perhaps due to a fixed alpha or improperly set noise hyperparameters).
*   The description of reference [22] needs to be corrected; it performs MAP inference, not sampling.
*   The paper should include a reference to and discussion of MAD-Bayes inference for the IBP.

The proposed inference method for LFMs is interesting, and its ability to scale to large datasets while inferring K is appealing. However, the experimental evaluation must be performed more carefully and described more thoroughly to support the paper's claims.
