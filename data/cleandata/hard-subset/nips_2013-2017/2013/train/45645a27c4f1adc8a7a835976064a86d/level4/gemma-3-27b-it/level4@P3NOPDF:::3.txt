## Review for NIPS 2013 - Binary Latent Feature Model Inference with Factorized Asymptotic Bayesian Inference

**Summary:**

This paper presents an efficient inference algorithm for binary latent feature models (LFMs), where each observation is represented by a binary latent vector indicating the presence or absence of K latent features. The algorithm, based on an Expectation-Maximization (EM) framework, simultaneously learns the latent features and infers the number of latent features (K) using a novel shrinkage mechanism. This approach draws inspiration from nonparametric Bayesian LFM techniques like the Indian Buffet Process (IBP), but instead of relying on prior distributions to encourage parsimony, it utilizes Factorized Asymptotic Bayesian Inference (FAB). FAB leverages the log marginal likelihood, decomposing it into an expected complete log-likelihood and a model selection term that penalizes large K. The authors demonstrate that FAB outperforms existing inference methods—including Gibbs sampling, variational inference, and MAP estimation for the IBP—in terms of both runtime and inference quality on several large datasets.

**Quality:**

The technical quality of the paper appears sound, and the algorithmic details are clearly presented. The ability to scale to large datasets while simultaneously inferring the number of latent features is a significant strength.

**Clarity:**

The paper is generally clear in its description of the proposed methodology. However, the experimental setup and results require further clarification, as detailed in the “Weak Points” and “Other Comments” sections.

**Originality:**

While the paper builds upon the recently proposed FAB framework applied to mixture models and Hidden Markov Models, its application to latent feature models represents a novel contribution.

**Significance:**

The paper addresses an important problem: efficient inference in latent feature models, particularly when the number of latent features is unknown.  Solving this problem effectively has the potential to broaden the applicability of LFMs.

**Weak Points:**

The paper lacks a discussion of potential limitations. It would benefit from exploring scenarios where the proposed algorithm might underperform compared to more traditional methods like Gibbs sampling. Furthermore, a discussion of convergence properties, given the EM-like procedure, is necessary. 

The method is currently limited to binary latent feature models and cannot be directly extended to continuous latent variable models like factor analysis or probabilistic PCA. This limitation should be explicitly acknowledged.

The experimental evaluation, while promising, isn’t comprehensive enough and exhibits some potential flaws.  Specifically, the behavior of the algorithm in the small-data regime, where the asymptotic assumptions underpinning FAB may not hold, is not adequately addressed through either discussion or experimentation. 

Several recent works on efficient inference in LFMs are missing from the related work discussion.  

Finally, the comparison to MEIBP [22] appears flawed. MEIBP is applicable only to non-negative linear Gaussian models, requiring a positive loading matrix. The synthetic and block data experiments, as currently presented, are invalid for this baseline; a comparison would require generating data with a positive loading matrix W.

**Other Comments:**

*   For the block image data, given that the noise level is known and provided to the FAB algorithm, the IBP Gibbs sampler should also be provided with the same value, rather than the commonly used 0.75 standard deviation heuristic, to ensure a fair comparison.
*   For the Variational Bayes-IBP (VB-IBP) [2] experiments, the use of the infinite variational version (as opposed to the finite version used) should be considered. Details regarding the experimental settings for VB-IBP, such as the number of restarts, are currently insufficient.
*   The finding that accelerated Gibbs sampling discovered 70 latent features on the 49-dimensional Sonar data is surprising. This likely indicates issues with initialization and/or the specified noise variance.
*   Line 70: Reference [21] describes MAP estimation for the IBP, not Gibbs sampling. Please correct this description.
*   The paper would benefit from a discussion of “MAD-Bayes: MAP-based Asymptotic Derivations from Bayes” (ICML 2013), which presents related work on efficient inference using small-variance asymptotics in nonparametric LFMs.

**Minor Comments:**

*   Line 349: The standard deviation for real-world data should be set to 0.25 times the standard deviation of examples across all dimensions, as suggested in [1, 22], rather than 0.75.

**Comments after Author Feedback:**

The author feedback addressed several concerns and clarified some aspects of the methodology. However, further improvements are needed before acceptance. 

Specifically, the experimental methodology (for both the proposed method and the baselines) requires more detailed explanation. Justifications or explanations should be provided regarding why certain algorithms exhibited specific behaviors (e.g., the effect of a fixed alpha value in the Gibbs sampler leading to an overestimated K, or the impact of improperly set noise variance hyperparameters). 

The misattribution of [21] to sampling remains and needs to be corrected to reflect that it describes MAP inference. 

The inclusion of a reference to the MAD-Bayes work on IBP inference is also necessary.

**Overall Recommendation:**

The proposed inference method for LFMs is interesting, and its scalability to large datasets, combined with the ability to infer K, is appealing. However, the experimental evaluation needs to be conducted more rigorously. I recommend acceptance pending these revisions.
