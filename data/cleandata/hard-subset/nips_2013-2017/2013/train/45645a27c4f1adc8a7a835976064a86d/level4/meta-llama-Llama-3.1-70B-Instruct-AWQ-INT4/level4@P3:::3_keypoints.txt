- Summary: Proposes a FAB-based inference algorithm for binary latent feature models that uses a shrinkage mechanism to learn features and infer the number of latent features K.  
- Summary: The FAB application to LFMs is novel and the method is reported to outperform Gibbs sampling, variational inference, and MAP/IBP baselines on large datasets.  

- Quality: Technical approach appears sound and suitable for large-scale inference while jointly inferring K.  

- Clarity: Methodology is reasonably clear, but key experimental details (parameter initialization, hyperparameter choices, convergence criteria) are underspecified.  

- Originality: Applying the FAB framework to LFMs is novel, though FAB itself has prior use in other models.  

- Significance: Addresses an important problem (efficiently inferring latent features and their number) and could have broad impact if validated.  

- Weak point 1: Paper lacks discussion of algorithmic limitations and scenarios where it may underperform relative to vanilla Gibbs.  
- Weak point 2: Convergence issues of the EM-like procedure are not discussed.  
- Weak point 3: Method is limited to binary latent feature models and cannot be applied to factor analysis or probabilistic PCA.  
- Weak point 4: Experimental evaluation is insufficient and contains potential flaws (e.g., inappropriate baselines or dataset-model mismatch).  
- Weak point 5: No discussion of behavior in small-data regimes where the asymptotic arguments may not hold.  
- Weak point 6: Recent related work on efficient LFM inference (e.g., MAD-Bayes for IBP) is not cited or discussed.  

- Other comment 1: MEIBP [22] applies only to non-negative linear Gaussian models, so the synthetic dataset should have been generated accordingly.  
- Other comment 2: For block images, the IBP Gibbs sampler should have used the same noise value as FAB for a fair comparison.  
- Other comment 3: VB-IBP experimental settings are under-described; the infinite variational version could have been used.  
- Other comment 4: Accelerated Gibbs found an unusually large number of features on Sonar data, possibly due to bad initialization or mis-specified noise variance.  
- Other comment 5: Reference [21] is about MAP estimation, not Gibbs sampling, and should be corrected.  

- Minor comment: Line 349 incorrectly states a suggestion on setting standard deviation; the suggestion was 0.25 times the examples' standard deviation, not 0.75.  

- After author-feedback 1: Experimental methodology for the proposed method and baselines still needs clearer explanation and justification for observed behaviors.  
- After author-feedback 2: Correct reference [21] to reflect that it addresses MAP inference, not Gibbs sampling.  
- After author-feedback 3: Include a reference and discussion of MAD-Bayes inference for the IBP.  

- Overall: The inference method is interesting and scalable, but the experimental evaluation and discussion of limitations/related work require substantial improvement.
