**Summary:**

This paper presents an estimation algorithm for a binary latent feature model (LFM) that expresses each observation xn as a binary vector zn of size K, where z{nk} = 1 means that the k-th latent feature xn is present. The goal of the estimation is to learn z for all observations (along with other model parameters). The presented EM-based algorithm uses a shrinkage mechanism to estimate the number of latent features (K) while learning z. This is similar to non-parametric Bayesian latent feature models such as the Indian Buffet Process (IBP). However, the presented algorithm does not actually use a prior distribution that inspires model parsing, but rather a model selection criterion – Factorized Asymptotic Bayesian Inference (FAB) – which has recently been introduced in the context of mixture models and HMMs. The FAB idea is based on a model selection term that encourages the shrinkage of the number of latent features (by specifying an upper bound on K and learning the correct K during estimation) along with expressing the log marginal probability in terms of expected full log-probability. On some large datasets, FAB has been shown to outperform other estimation methods, including Gibbs sampling, variational inference, and MAP estimation methods for IBP (in terms of running time and estimation quality).

**Quality:** The technical quality seems reasonably good, and the algorithm descriptions seem correct. The ability to deal with large datasets and learn the number of latent features is very appealing.

**Clarity:** The paper is reasonably clear in the methodology section. However, some experimental details are somewhat unclear (discussed below).

**Originality:** The proposed method is based on a recently proposed framework for Factorized Asymptotic Bayesian (FAB) Inference, which has been applied to mixture models and HMMs. The application of FAB to latent feature models, however, is new.

**Importance:** The paper is addressing an important problem (efficient estimation in latent feature models while also estimating the number of latent features).

**Weak Points:**

*   There is no discussion on the limitations of the proposed algorithm. Are there cases when the algorithm may perform worse than vanilla Gibbs sampling?
*   There is no discussion on potential difficulties in convergence (considering that it is an EM-like process).
*   The algorithm is limited to binary latent feature models (it cannot be applied to factor analysis or probabilistic PCA).
*   The experimental evaluation is not thorough enough (and seems flawed at some points; see comments below).
*   In small-data regimes (when asymptotic arguments do not apply), it is not clear how the algorithm will behave (some discussion or experimentation should be done).
*   The paper does not discuss some other recent works on efficient estimation in LFM (see comments below).
*   The MIBP algorithm proposed in [22] applies only to non-negative linear Gaussian models (W should be positive, not real-valued). Therefore, artificial simulations and block data experiments are only invalid for the MIBP baseline. If you compare with MIBP, a synthetic dataset should be generated so that the loading matrix W is positive.

**Other Comments:**

*   For block image data, since the noise value is known (and given to the FAB algorithm), the IBP Gibbs sampler should also be given the same value for fairness (instead of a 0.75 std-dev estimate).
*   For VB-MIBP [2], an infinite variational version can be used (a finite variational version was used in the experiments). The experimental settings for VB are not described in sufficient detail (for example, how many restarts were given?).
*   I am surprised that the quick Gibbs sampling discovered approximately 70 latent features on 49-dimensional sonar data! I suspect this is due to poor initialization and/or poorly specified noise variance values.
*   Line 70: The reference [21] is not about Gibbs sampling, but about MAP estimation for IBP (just like reference [22]). Please revise the text and correct the details.
*   Efficient estimation using small-variance asymptotes has recently worked in the case of non-parametric LFM (see "MAD-based: MAP-based Asymptotic Derivation from a Base" from ICML 2013). It would be good to discuss this work as well.

**Minor Comments:**

*   Line 349: For real-data, the suggestion in [1,22] was not to set std-dev equal to 0.75, but rather to set it equal to 0.25 times the std-dev of the examples in all dimensions.
*   **Author-response after comments:** The response answered some questions. There are some things that need to be fixed/corrected before the paper is accepted. In particular:

*   The experimental methodology (for the proposed method and baseline) should be better explained, and there should be justification/clarification for why some algorithms behaved in a certain way (for example, the Gibbs sampler may have used a certain alpha that resulted in a high estimate of K, or the noise variance hyperparameters were not set correctly).
*   Reference [21] estimates MAP, not sampling. Please correct this.
*   Include a reference on MAB-based estimation for IBP. The proposed estimation method for LFM is interesting, and the fact that it scales to large datasets (besides estimating K) is appealing. However, the experimental evaluation should have been done more carefully.
