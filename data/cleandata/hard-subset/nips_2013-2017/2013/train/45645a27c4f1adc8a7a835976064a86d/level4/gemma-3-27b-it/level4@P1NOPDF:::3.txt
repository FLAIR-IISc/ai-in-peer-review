Summary:
The paper presents an inference algorithm for the binary latent feature model (LFM), where each observation x<sub>n</sub> is represented as a K-dimensional binary vector z<sub>n</sub>, with z<sub>nk</sub> = 1 indicating the presence of the k-th latent feature in x<sub>n</sub>. The goal is to learn the z’s for all observations, along with the other model parameters. The proposed Expectation-Maximization (EM)-based algorithm employs a shrinkage mechanism to learn the z’s and simultaneously infer the number of latent features (K). This approach shares similarities with nonparametric Bayesian latent feature models like the Indian Buffet Process (IBP). However, instead of relying on prior distributions to induce model parsimony, the algorithm utilizes a model selection criterion – Factorized Asymptotic Bayesian Inference (FAB) – recently introduced for mixture models and Hidden Markov Models (HMMs). The FAB idea leverages the log marginal likelihood, expressed as the expected complete log-likelihood plus a model selection term that encourages shrinkage in the number of latent features by specifying an upper bound on K and learning its optimal value during inference. The authors demonstrate that FAB outperforms other inference methods, including Gibbs sampling, variational inference, and MAP estimation for the IBP, on several large datasets in terms of both running time and inference quality.

Quality: The technical quality appears sound, and the algorithmic details seem correct. The algorithm’s ability to handle large datasets while also inferring the number of latent features is particularly appealing.

Clarity: The paper is generally clear in its description of the methodology. However, some experimental details require further clarification (discussed below).

Originality: The proposed method builds upon the recently developed framework of Factorized Asymptotic Bayesian (FAB) Inference, initially applied to mixture models and HMMs. Applying FAB to latent feature models, however, represents a novel contribution.

Significance: The paper addresses an important problem: efficient inference in latent feature models, coupled with the challenge of inferring the number of latent features.

Weak Points:
- The paper lacks a discussion of the algorithm’s limitations. Are there scenarios where it might perform worse than standard Gibbs sampling?
- There is no discussion regarding potential convergence issues, given the EM-like nature of the procedure.
- The algorithm is currently limited to binary latent feature models and cannot be readily extended to factor analysis or probabilistic PCA.
- The experimental evaluation is not comprehensive enough and contains some potential flaws (see comments below).
- The algorithm's behavior in the small-data regime, where the asymptotic approximations may not hold, is unclear. Discussion or experiments addressing this would be beneficial.
- Several recent publications on efficient inference in LFMs are not discussed. (see comments below).
- The MEIBP algorithm proposed in [22] is only applicable to non-negative linear Gaussian models (requiring a positive definite W, not a real-valued one). Consequently, the artificial simulation and block data experiments are invalid when comparing against MEIBP. If a comparison to MEIBP is intended, the synthetic dataset should be generated with a positive definite loading matrix W.

Other Comments:
- For the block image data, as the noise value is known and provided to the FAB algorithm, fairness dictates that the IBP Gibbs sampler should also be provided with the same value, rather than the 0.75 standard deviation heuristic.
- For VB-IBP [2], the infinite variational version could be used (the experiments utilized a finite variational version). Furthermore, the experimental settings for VB are not described in sufficient detail (e.g., the number of restarts performed).
- I am surprised that the accelerated Gibbs sampling discovered approximately 70 latent features on the 49-dimensional Sonar data. This may be due to poor initialization and/or an inappropriately specified noise variance.
- Line 70: Reference [21] describes MAP estimation, not Gibbs sampling, for the IBP. Please revise the text accordingly.
- Recent work on efficient inference using small-variance asymptotics in nonparametric LFMs (e.g., “MAD-Bayes: MAP-based Asymptotic Derivations from Bayes” from ICML 2013) should be discussed.

Minor comments:
- Line 349: Previous work [1, 22] suggests setting the standard deviation to 0.25 times the standard deviation of examples across all dimensions, rather than 0.75.

Comments after the author feedback: The authors’ response addressed some of the initial questions. However, several issues require further attention before the paper can be accepted. In particular:
- The experimental methodology (for both the proposed method and the baselines) needs to be more thoroughly explained, with justifications for observed behaviors (e.g., a fixed alpha value in Gibbs sampling potentially leading to an overestimated K, or improper hyperparameter tuning for noise variance).
- Reference [21] performs MAP inference, not sampling. Please correct this.
- Include a citation to the MAD-Bayes inference work for the IBP.

The proposed inference method for LFMs is interesting, and its scalability to large datasets alongside its ability to infer K is appealing. However, the experimental evaluation needs to be more rigorous and carefully considered.
