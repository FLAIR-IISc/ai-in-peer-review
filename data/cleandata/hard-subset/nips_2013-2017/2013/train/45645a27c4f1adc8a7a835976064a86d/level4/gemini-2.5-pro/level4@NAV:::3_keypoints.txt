- Proposes an EM-based inference algorithm with a shrinkage mechanism to learn binary latent feature assignments and infer the number of features K.  
- Uses Factorized Asymptotic Bayesian (FAB) inference as a model selection term instead of nonparametric priors like the IBP.  
- FAB expresses log marginal likelihood as expected complete log-likelihood plus a penalty that encourages fewer active features.  
- Experimental results claim FAB outperforms IBP Gibbs, variational inference, and MAP in runtime and inference quality on large datasets.  
- Technical details and algorithmic correctness appear sound; scalable handling of large datasets and learning K is a key advantage.  
- Methodological exposition is generally clear, but some experimental setup details are unclear.  
- Applying FAB to latent feature models is novel, though FAB itself is not new.  
- The problem tackled (efficient inference in LFMs while determining K) is significant.  
- Paper lacks discussion of potential limitations and failure conditions compared to Gibbs sampling.  
- No treatment of convergence issues for the EM-like procedure.  
- Approach is limited to binary LFMs and does not extend to continuous models like factor analysis or PPCA.  
- Empirical evaluation is insufficiently rigorous and contains potential flaws.  
- No analysis of algorithm behavior in small-data regimes where FAB’s asymptotics may fail.  
- Omits discussion of some recent related work on efficient inference for LFMs.  
- Comparison with MEIBP [22] appears invalid because MEIBP assumes a non-negative loading matrix not respected in some experiments.  
- Block images experiment gave FAB the true noise value but not the IBP Gibbs baseline, creating an unfair comparison.  
- VB-IBP baseline used a finite variational version and lacked details (e.g., number of restarts); infinite variational VB was not used.  
- Surprising result of ~70 features on 49-dimensional Sonar data suggests possible initialization or noise-variance mis-specification.  
- Reference [21] was mischaracterized as Gibbs sampling but actually describes MAP estimation; the manuscript should correct this.  
- Recommend citing and discussing small-variance asymptotics work (e.g., MAD-Bayes ICML 2013).  
- Minor correction: recommended noise std-dev in prior work is 0.25× data std-dev, not 0.75 fixed.  
- After author feedback, authors clarified some points but experimental methodology still needs more detail and justification.  
- Paper needs to correct reference descriptions and incorporate MAD-Bayes before acceptance; current experimental weaknesses undermine the contribution.
