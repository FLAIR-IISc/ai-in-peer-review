The paper presents an inference algorithm for the binary latent feature model (LFM), which represents each observation as a K-dimensional binary vector. The goal of inference is to learn the latent features for all observations, along with other model parameters. The proposed EM-based algorithm utilizes a shrinkage mechanism to learn the latent features and infer the number of latent features (K). This approach is similar in spirit to nonparametric Bayesian latent feature models, such as the Indian Buffet Process (IBP). However, the algorithm does not use prior distributions to induce model parsimony, instead relying on a model selection criterion called Factorized Asymptotic Bayesian Inference (FAB). The FAB approach expresses the log marginal likelihood in terms of the expected complete log-likelihood plus a model selection term that encourages shrinkage of the number of latent features.

The technical quality of the paper appears to be reasonably sound, with correct algorithmic details. The ability to handle large datasets and learn the number of latent features is particularly appealing. The methodology is clearly explained, but some experimental details are unclear. The application of FAB to latent feature models is novel, and the paper addresses an important problem in efficient inference.

However, there are several weak points that need to be addressed. Firstly, there is no discussion on the limitations of the proposed algorithm, including cases where it might perform worse than vanilla Gibbs sampling. Additionally, the paper does not discuss potential convergence issues, given that it is an EM-like procedure. The algorithm is also limited to binary latent feature models and cannot be applied to factor analysis or probabilistic PCA. The experimental evaluation is not thorough enough and appears to be flawed in some cases.

Furthermore, the paper does not discuss recent works on efficient inference in LFMs, including the MEIBP algorithm proposed in [22]. The experimental settings for the VB-IBP baseline are not described in sufficient detail, and the use of a fixed alpha value in the Gibbs sampler may have led to an overestimated K. The noise variance hyperparameters were not properly set, which may have affected the results.

In the experimental evaluation, the MEIBP algorithm was applied to invalid datasets, as it is only applicable to non-negative linear Gaussian models. The synthetic dataset should have been generated with a positive loading matrix W to make the comparison valid. For the block images data, the IBP Gibbs sampler should have been given the same noise value as the FAB algorithm for a fair comparison. The infinite variational version of VB-IBP could have been used instead of the finite variational version.

I am also surprised that the accelerated Gibbs sampling discovered 70 latent features on the 49-dimensional Sonar data, which may be due to bad initialization or a poorly specified noise variance value. The reference [21] is not about Gibbs sampling but rather MAP estimation for the IBP, and the text should be corrected accordingly. Recent work on efficient inference using small-variance asymptotic in nonparametric LFMs, such as "MAD-Bayes: MAP-based Asymptotic Derivations from Bayes" from ICML 2013, should be discussed.

Minor comments include correcting the suggestion in [1,22] for setting the standard deviation to 0.25 times the standard deviation of examples across all dimensions instead of 0.75.

After receiving the author feedback, I believe that the experimental methodology needs to be better explained, and justifications should be provided for the behavior of certain algorithms. The reference [21] should be corrected to reflect that it performs MAP inference, not sampling. The reference on MAD-Bayes inference for the IBP should be included. While the proposed inference method for LFMs is interesting, the experimental evaluation should have been more carefully done.
