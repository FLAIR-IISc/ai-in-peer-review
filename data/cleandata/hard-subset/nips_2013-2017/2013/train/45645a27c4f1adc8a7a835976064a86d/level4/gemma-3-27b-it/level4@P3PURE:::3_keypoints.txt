- Proposes an EM-based inference algorithm using Factorized Asymptotic Bayesian (FAB) shrinkage for binary latent feature models to learn latent feature assignments z and infer K.  
- FAB-based approach replaces nonparametric priors (e.g., IBP) with a model-selection term that encourages feature shrinkage given an upper bound on K.  
- Empirical results claim faster runtime and better inference quality than Gibbs sampling, variational inference, and MAP methods on several large datasets.  
- Technical formulation and algorithmic details appear sound and scalable to large datasets.  
- Paper is reasonably clear methodologically, but several experimental details are unclear or omitted.  
- Novelty: applying FAB to latent feature models is a novel contribution.  
- Significance: addresses important problem of efficient LFM inference with automatic K selection.  
- Missing discussion of algorithm limitations and scenarios where it might underperform compared to Gibbs sampling.  
- No analysis of potential convergence difficulties for the EM-like procedure.  
- Method is restricted to binary LFMs and cannot be directly applied to factor analysis or probabilistic PCA.  
- Experimental evaluation is not comprehensive and contains methodological flaws.  
- Unclear how the method behaves in small-data regimes where asymptotic arguments may fail; experiments/discussion needed.  
- Failed to discuss relevant recent work on efficient inference for LFMs (e.g., MAD-Bayes from ICML 2013).  
- Comparison to MEIBP is invalid because MEIBP requires a non-negative loading matrix W; synthetic/block experiments did not generate data accordingly.  
- For block image experiments, IBP Gibbs should be given the same known noise value for fairness.  
- VB-IBP experiments used a finite variational version; the infinite variational alternative and experimental settings (e.g., restarts) are not described.  
- Surprising result of ~70 features from accelerated Gibbs on Sonar likely due to poor initialization or mis-specified noise variance.  
- Reference [21] is incorrectly described as Gibbs sampling; it actually describes MAP inference and should be corrected.  
- Minor comment: previous work recommends setting noise std to 0.25 of data std, not 0.75.  
- After author response, some issues remain: experimental methodology needs clearer justification and more detailed explanation of baseline behaviors, and missing citations (MAD-Bayes) should be added.  
- Overall: method is interesting and scalable with appealing K-inference, but experiments must be substantially improved before acceptance.
