Review- Summary:
The paper proposes an inference algorithm for the binary latent feature model (LFM), representing each observation xn by a K-dimensional binary vector zn where z{nk} = 1 indicates that the k-th latent feature is present for xn. The inference goal is to learn the z’s for all observations (along with other model parameters). The EM-based method introduces a shrinkage mechanism to estimate the z’s and infer the number of latent features (K). This is conceptually related to nonparametric Bayesian latent feature models such as the Indian Buffet Process (IBP). However, instead of using priors to induce parsimony, the approach relies on a model selection criterion—Factorized Asymptotic Bayesian Inference (FAB)—recently developed for mixture models and HMMs. FAB rewrites the log marginal likelihood as the expected complete log-likelihood plus a model selection term that promotes shrinking the number of latent features (by imposing an upper bound on K and learning the appropriate K during inference). On several large datasets, the FAB approach is shown to outperform alternatives including IBP-based Gibbs sampling, variational inference, and MAP methods in terms of runtime and inference quality.

Quality:
The technical development appears reasonably solid, and the algorithmic details are correct. The ability to handle large datasets while simultaneously estimating the number of latent features is compelling.

Clarity:
The methodological presentation is largely clear. Some aspects of the experimental setup, however, are insufficiently explained (see below).

Originality:
The method builds on the recently proposed FAB framework for mixture models and HMMs. Its application to latent feature models is novel.

Significance:
The paper tackles an important problem: efficient inference in latent feature models while automatically determining the number of latent features.

Weak Points:
- No discussion of the algorithm’s limitations. Are there settings in which it underperforms vanilla Gibbs sampling?
- No treatment of potential convergence challenges (given the EM-like nature of the procedure).
- The method targets binary latent feature models only (it does not extend to factor analysis or probabilistic PCA).
- The experimental evaluation is not sufficiently thorough and appears flawed in places (see comments below).
- In small-data regimes (where asymptotic arguments may not hold), the behavior of the algorithm is unclear; discussion or experiments are needed.
- Some recent work on efficient inference for LFMs is not discussed (see below).
- The MEIBP method in [22] applies only to non-negative linear Gaussian models (W must be non-negative, not real-valued). Therefore, the artificial simulation and block data experiments are not valid baselines for MEIBP. If comparing to MEIBP, the synthetic data should enforce a non-negative loading matrix W.

Other Comments:
- For the block image data, since the noise level is known (and provided to FAB), fairness requires giving the IBP Gibbs sampler the same value rather than using the 0.75 std-dev heuristic.
- For VB-IBP [2], the infinite variational formulation could be used (the experiments appear to use a finite variational version). The VB settings are under-specified (e.g., number of restarts).
- It is surprising that accelerated Gibbs discovered ~70 latent features on the 49-dimensional Sonar dataset; this likely reflects poor initialization and/or mis-specified noise variance.
- Line 70: Reference [21] concerns MAP estimation for the IBP, not Gibbs sampling. Please correct the text and description.
- Please discuss recent work on efficient inference using small-variance asymptotics for nonparametric LFMs (e.g., “MAD-Bayes: MAP-based Asymptotic Derivations from Bayes,” ICML 2013).

Minor comments:
- Line 349: For real data, [1,22] recommend setting the noise std-dev to 0.25 times the empirical std-dev of the examples across all dimensions, not 0.75.

Comments after the author-feedback:
The feedback addressed several questions. A few items must be corrected or improved before acceptance:
- The experimental methodology (for both the proposed approach and baselines) needs clearer exposition, including justifications for observed behaviors (e.g., a fixed alpha in Gibbs possibly inflating K, or mis-specified noise variance hyperparameters).
- Reference [21] performs MAP inference, not sampling—please fix.
- Include and discuss the MAD-Bayes reference for IBP.
The proposed LFM inference method is interesting, and its scalability alongside automatic K selection is attractive. However, the experimental evaluation requires more careful design and reporting.
