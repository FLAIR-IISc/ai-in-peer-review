- Tackles covariance selection at extreme scale (millions of features, trillions of parameters).  
- Centers on the CLIME estimator, enabling column-wise precision estimation and easier/distributed analysis compared to Graphical-Lasso.  
- Introduces a novel block-wise CLIME algorithm combined with an inexact ADMM approach.  
- Proves theoretical convergence rates for both objective value and distance-to-optimality.  
- Achieves computational efficiency by exploiting sparse and low-rank matrix structure.  
- Provides a scalable parallel implementation for both shared- and distributed-memory systems.  
- Presents numerical experiments comparing runtime against competitors on synthetic and real datasets.  
- Analyzes speedup behavior with respect to block size and core count across different architectures.  
- Paper is exceptionally well-written, with clearly articulated novelty and thorough literature context.  
- Algorithmic exposition balances practical numerical analysis with rigorous convergence guarantees.  
- Implementation shows remarkable performance gains in experiments.  
- Main concern: lack of publicly released source code threatens reproducibility; code release recommended for publication.
