This paper addresses covariance selection in high-dimensional settings, scaling to problems with millions of features and trillions of parameters. The authors focus on the CLIME estimator, which offers a computationally attractive alternative to Graphical Lasso due to its column-wise solvability and potential for distributed computation. They present a novel algorithm for fitting CLIME that leverages a block-wise solution approach coupled with an inexact alternating direction method of multipliers. The paper includes theoretical analysis demonstrating convergence rates with respect to both the objective function and distance to optimality. Furthermore, the authors exploit sparse and low-rank matrix structures to reduce computational complexity. The algorithm is implemented within a scalable parallel computation framework supporting both shared and distributed memory systems.  Experimental results compare the algorithm's runtime to competing methods on both synthetic and real datasets, and a detailed performance study analyzes speedup as a function of block size and core count for different memory architectures.

This is a well-written and clear paper. The novelty and contributions of the method are clearly articulated in the introduction, and the work is appropriately situated within the existing literature with thorough referencing. The algorithm is presented effectively, striking a good balance between technical details of numerical analysis/algebra and theoretical convergence guarantees. The numerical analysis is appropriate, and the implementation demonstrates impressive performance. 

My primary concern is the accessibility of this method. Without public release of the source code – which relies heavily on open-source libraries – the impact of this work is significantly limited. 

Overall, this is a strong algorithmic paper that tackles the important and practical problem of covariance selection in very high dimensions. If the code is made available to the community, this powerful tool warrants publication.
