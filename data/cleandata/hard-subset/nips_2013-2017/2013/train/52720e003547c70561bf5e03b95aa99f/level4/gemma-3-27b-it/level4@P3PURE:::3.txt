Summary of the paper
This paper addresses the problem of covariance selection in very large-scale settings, involving up to millions of features and trillions of parameters. The authors focus on the CLIME estimator, which provides an estimate of the precision matrix for a Gaussian vector and is well-suited for distributed computation due to its column-wise solution structure – offering computational advantages over Graphical Lasso. The paper introduces a novel algorithm for fitting CLIME by solving the problem block-wise, coupled with an inexact alternating direction method of multipliers. Theoretical results regarding the convergence rate of this algorithm are presented, expressed in terms of both objective function value and distance to optimality. The authors leverage special matrix structures, specifically sparsity and low rank, to reduce the computational burden. The algorithm is implemented within a scalable parallel computation framework supporting both shared and distributed memory systems. The numerical studies compare runtime performance against direct competitors using both synthetic and real datasets. A detailed performance analysis is also provided, evaluating the impact of block sizes and the number of cores on speedup in both shared and distributed memory environments.

Comments
This is a well-written and clear paper. The novelty and contributions of the method are clearly articulated in the introduction, and connections to existing work are thoroughly established with strong bibliographic support. The algorithm is presented effectively, balancing technical details of numerical analysis and algebra with theoretical convergence guarantees. The numerical analysis is appropriate, and the implementation demonstrates impressive performance.

My primary concern is the availability of the proposed method. Without public release of the source code – which is largely based on open-source libraries – the impact of this work is limited. 

Overall, this is a strong algorithmic paper that tackles the important practical problem of covariance selection with very high-dimensional data. Such a powerful tool merits publication, contingent upon making the code available to the broader research community.
