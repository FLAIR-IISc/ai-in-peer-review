- Addresses large-scale sparse precision matrix estimation by scaling CLIME to millions of dimensions with distributed computation.
- Proposes solving CLIME in column-blocks (not column-by-column) combined with an inexact ADMM algorithm.
- Introduces a block-cyclic distribution strategy to achieve optimal load balancing across cores.
- Demonstrates scalability to extreme sizes (e.g., ~1M features, ~1T parameters) using hundreds of cores.
- Establishes O(1/T) convergence rates for both the objective function and optimality conditions.
- Provides rigorous theoretical analysis extending prior inexact ADMM results.
- Empirically shows significant scalability advantages and near-linear speedup on synthetic and real datasets.
- Compares favorably to state-of-the-art baselines (CLIME, Tiger, DC-QUIC) in scalability while maintaining comparable statistical performance.
- Clear, well-organized exposition with detailed algorithmic description and helpful pseudocode (Algorithm 1).
- Careful implementation for both shared-memory (OpenMP) and distributed-memory (OpenMPI) architectures.
- Demonstrated practical impact on real datasets (e.g., Leukemia, Climate) and potential for large-scale applications.
- Novel contribution relative to prior CLIME work through the column-block approach and distributed framework.
- Main weakness: lack of explicit statement about public code availability and implementation details.
- Overall recommendation: strong, publishable contribution (recommend acceptance), with expectation authors release code to maximize impact.
