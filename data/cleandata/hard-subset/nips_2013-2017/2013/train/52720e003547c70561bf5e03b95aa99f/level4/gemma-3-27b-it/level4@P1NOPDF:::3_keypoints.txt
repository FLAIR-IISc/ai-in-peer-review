- Addresses covariance selection at massive scale (millions of features, trillions of parameters).
- Focuses on the CLIME estimator as a computationally attractive alternative to Graphical Lasso.
- CLIME’s column-wise solution structure makes it well-suited to distributed computation and more tractable theoretical analysis.
- Proposes a novel block-wise algorithm for CLIME, using an inexact ADMM scheme.
- Provides theoretical convergence rates for the algorithm in terms of objective value and distance to optimality.
- Exploits matrix structures (sparsity and low-rank) to reduce computational burden.
- Implements the algorithm in a scalable parallel framework for both shared and distributed memory systems.
- Empirical evaluation compares runtime performance against direct competitors on synthetic and real datasets.
- Detailed experiments measure speedup effects of varying block sizes and number of cores on shared and distributed systems.
- Paper is well-written, clear, and situates contributions with thorough bibliographical references.
- Good balance between numerical/numerical-algebraic details and theoretical convergence guarantees.
- Implementation demonstrates impressively strong performance in experiments.
- Primary concern: method’s value is significantly diminished unless source code is made available to the community.
- Recommendation: strong paper deserving publication contingent on making the code accessible.
