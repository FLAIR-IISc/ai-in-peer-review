- Addresses covariance selection in ultra-high-dimensional settings, focusing on the CLIME estimator.
- Highlights CLIME’s column-wise decomposability as beneficial for distributed computation and analysis versus Graphical Lasso.
- Introduces a novel block-wise CLIME algorithm using inexact ADMM to solve block subproblems instead of columns.
- Exploits sparsity and low-rank structure to substantially reduce computational overhead.
- Provides theoretical convergence guarantees for objective value and distance to the optimal solution.
- Implements the algorithm in a scalable parallel framework supporting shared and distributed memory systems.
- Presents extensive numerical experiments on synthetic and real datasets comparing runtime to state-of-the-art methods.
- Includes a detailed parametric study (block size, core count) analyzing speedup across parallel architectures and offering deployment guidance.
- Paper is exceptionally well-written, clear, and balances numerical linear algebra and convergence theory for diverse audiences.
- Main concern: lack of released source code undermines reproducibility and community impact; code availability is essential for validation and adoption.
- Recommendation: acceptance contingent on authors committing to release the full implementation (e.g., via a public repository) upon acceptance.
