- Tackles covariance selection at very large scale (millions of features, trillions of parameters).
- Builds on CLIME estimator, which is amenable to distributed/column-wise solution.
- Proposes a new block-wise algorithm for CLIME coupled with an inexact direction method of multipliers.
- Provides theoretical convergence-rate guarantees for both objective value and distance to optimality.
- Exploits sparsity and low-rank structure to reduce computational burden.
- Implements a scalable parallel framework for both shared- and distributed-memory systems.
- Empirical evaluation compares runtime against competitors on synthetic and real datasets, showing impressive performance.
- Presents a detailed study of speedup as a function of block size and number of cores.
- Paper is well written, clear, and situates contributions with solid bibliographic connections.
- Algorithm presentation balances numerical/algebraic detail with theoretical guarantees.
- Main concern: source code availability—release of implementation is necessary for community impact.
- Overall verdict: sound, practically important work recommended for publication provided the code is made available.
