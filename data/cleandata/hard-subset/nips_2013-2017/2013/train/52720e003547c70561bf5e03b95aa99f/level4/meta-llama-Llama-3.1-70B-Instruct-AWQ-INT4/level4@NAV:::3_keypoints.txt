- Addresses covariance selection at extreme scale (millions of features, trillions of parameters).  
- Focuses on the CLIME estimator, which is solved column-wise and is more amenable to distributed computation than Graphical-Lasso.  
- Claims CLIME has simpler theoretical analysis compared to Graphical-Lasso.  
- Proposes a novel block-wise algorithm using an inexact direction method of multipliers (inexact ADMM-like approach).  
- Provides theoretical convergence rates for both objective value and distance to optimality.  
- Exploits special matrix structure (sparsity, low rank) to reduce computational burden.  
- Implements a scalable parallel computation framework supporting shared and distributed memory systems.  
- Presents numerical experiments comparing runtime to competitors on synthetic and real data.  
- Includes detailed speedup analysis with respect to block sizes and number of cores.  
- Manuscript is well-written, clear, and situates contributions within existing literature.  
- Balances numerical analysis, algebraic exposition, and theoretical guarantees effectively.  
- Demonstrates impressive implementation performance, but the main concern is lack of publicly available source code limiting impact.  
- Overall assessment: solid, publishable algorithmic contribution contingent on making the method/code available to the community.
