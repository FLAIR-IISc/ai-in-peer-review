- Addresses covariance selection at extreme scale (millions of features, trillions of parameters).
- Focuses on the CLIME estimator, which decomposes column-wise and is more tractable than graphical lasso.
- Proposes a new block-wise algorithm for CLIME using an inexact ADMM approach.
- Provides theoretical convergence rates for objective value and distance to the optimum.
- Algorithm exploits special matrix structures (sparsity, low-rank) to reduce computation.
- Implements a scalable parallel framework for both shared- and distributed-memory systems.
- Numerical experiments compare runtime against competitors on synthetic and real datasets.
- Detailed analysis of speedup as a function of block size and number of processing cores.
- Paper is well-written, clearly motivated, and well-situated in related literature.
- Strong balance between numerical algorithm details and theoretical guarantees; implementation shows impressive performance.
- Main concern: practical impact depends on making the high-performance source code available to the community.
- Overall assessment: strong algorithmic contribution that merits publication if the implementation is released.
