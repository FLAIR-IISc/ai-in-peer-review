- Addresses covariance selection at extremely large scale (millions of variables, trillions of parameters).
- Focuses on the CLIME estimator, which is column-wise solvable and theoretically simpler than Graphical Lasso.
- Proposes a new block-wise algorithm for CLIME using an inexact direction method of multipliers.
- Provides theoretical convergence-rate results for objective value and distance to optimality.
- Explicitly exploits matrix structure (sparsity and low rank) to reduce computational cost.
- Implemented in a scalable parallel framework for both shared- and distributed-memory systems.
- Numerical evaluation compares runtime with competitors on synthetic and real data.
- Includes detailed speedup analysis as a function of block size and number of cores.
- Paper is well written, clear, and situates contributions within related work.
- Good balance between numerical/algebraic details and theoretical guarantees.
- Implementation demonstrates very impressive performance.
- Main concern: source code must be released for community benefit.
- Overall: strong algorithmic contribution deserving publication if code is made available.
