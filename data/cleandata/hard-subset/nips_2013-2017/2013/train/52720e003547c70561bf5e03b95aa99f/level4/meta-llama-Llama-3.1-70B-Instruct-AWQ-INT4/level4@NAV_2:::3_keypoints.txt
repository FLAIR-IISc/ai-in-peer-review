- Tackles large-scale covariance selection (millions of features, trillions of parameters) using the CLIME estimator.  
- Proposes a novel block-wise algorithm for CLIME coupled with an inexact direction method of multipliers (inexact ADMM-like approach).  
- Theoretically characterizes algorithm convergence rates in terms of objective value and distance to optimality.  
- Exploits special matrix structures (sparsity and low-rank) to reduce computational burden.  
- Implements the algorithm in a scalable parallel framework for both shared-memory and distributed-memory systems.  
- Provides extensive numerical comparisons on synthetic and real data, demonstrating impressive runtime performance.  
- Detailed empirical study of speedups with varying block sizes and numbers of cores on both shared and distributed systems.  
- Paper is well written, clear, and situates contributions relative to prior work with solid references.  
- Good balance between numerical analysis, algebraic details, and theoretical guarantees.  
- Novelty: first to solve CLIME block-wise at this scale and couple it with an inexact multiplier method.  
- Significance: offers a practical scalable alternative to column-wise CLIME and arguably easier analysis than Graphical Lasso.  
- Main weakness: method utility depends on releasing source code; lack of code would undermine impact.  
- Argument for acceptance: novel, theoretically grounded, and empirically strong solution to a challenging large-scale problem.  
- Argument for rejection: conditional—reject if source code is not made available to the community.  
- Recommendation: accept conditional on making the source code publicly available.
