- Tackles massive-scale covariance selection (up to millions of features and trillions of parameters).
- Focuses on the CLIME estimator, which is column-wise solvable and more amenable to distributed computation than Graphical-Lasso.
- Proposes a novel block-wise algorithm for CLIME using an inexact method of multipliers.
- Provides theoretical convergence rates for both objective value and distance to optimality.
- Exploits matrix structure (sparsity and low-rank) to reduce computational burden.
- Implements the algorithm in a scalable parallel framework supporting shared and distributed memory.
- Numerical comparisons of runtime against direct competitors on synthetic and real data.
- Detailed empirical study of speedups with varying block sizes and numbers of cores in both shared and distributed settings.
- Paper is well-written, with clear exposition, solid bibliographic context, and balanced technical/theoretical treatment.
- Implementation shows impressive performance in experiments.
- Major concern: impact is limited if source code is not released (mostly built on open-source libraries).
- Overall assessment: sound, practically significant algorithmic contribution that would be highly valuable if the code is made available.
