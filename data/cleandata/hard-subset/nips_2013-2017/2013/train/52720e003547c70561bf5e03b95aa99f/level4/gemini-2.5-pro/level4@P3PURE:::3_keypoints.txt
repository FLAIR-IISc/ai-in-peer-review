- Addresses large-scale covariance selection for up to millions of features and trillions of parameters.
- Focuses on the Constrained L1-Minimization Estimator (CLIME) for precision matrix estimation.
- Proposes a new block-wise algorithm for CLIME paired with an inexact ADMM solver.
- Provides theoretical convergence rates for objective value and distance to the optimal solution.
- Exploits matrix structures (sparsity and low-rank) to reduce computational costs.
- Implements a scalable parallel framework for both shared- and distributed-memory systems.
- Empirical evaluation compares runtime against competitors on synthetic and real data.
- Detailed analysis of speedup relative to block size and number of cores on different architectures.
- Paper is well-written, clear, and situates contributions with sound bibliographic references.
- Main concern: value to the community depends on making the source code publicly available; recommends publication if released.
