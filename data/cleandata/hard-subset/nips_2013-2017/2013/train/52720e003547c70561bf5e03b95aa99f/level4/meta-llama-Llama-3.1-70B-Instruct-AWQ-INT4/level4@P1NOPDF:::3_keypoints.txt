- Tackles covariance selection at extremely large scale (millions of features, trillions of parameters).
- Focuses on the CLIME estimator, which is amenable to column-wise and distributed solution.
- Proposes a new block-wise algorithm for CLIME combined with an inexact method of multipliers.
- Provides theoretical convergence rates measured by objective value and distance to optimality.
- Exploits matrix structures (sparsity and low-rank) to reduce computational cost.
- Implements a scalable parallel framework for both shared-memory and distributed-memory systems.
- Presents numerical comparisons of runtime against direct competitors on synthetic and real datasets.
- Includes detailed experiments on speedup relative to block size and number of cores for both memory models.
- Paper is well-written, clear, and places contributions in context with solid references.
- Algorithm presentation balances numerical analysis, algebraic details, and theoretical guarantees.
- Numerical analysis is appropriate and the implementation shows impressive performance.
- Main concern: lack of publicly available source code would limit the work's impact; publication recommended if code is released.
