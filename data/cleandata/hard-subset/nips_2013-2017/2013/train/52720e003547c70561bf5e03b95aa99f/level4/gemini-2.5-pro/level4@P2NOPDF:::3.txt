### Summary of the Paper

This paper addresses the problem of covariance selection in large-scale settings, particularly for datasets with millions of features and trillions of parameters. The authors focus on the Constrained L1-Minimization for Inverse Matrix Estimation (CLIME) estimator. CLIME is advantageous as it estimates the precision matrix of a Gaussian random vector by solving for each column independently, a property that makes it highly suitable for distributed computation. The authors also note that CLIME offers more straightforward theoretical analysis compared to competing methods like the Graphical Lasso.

The primary contribution is a novel algorithm for fitting the CLIME estimator. This algorithm innovatively solves the problem block-wise, rather than column-wise, and is coupled with an inexact alternating direction method of multipliers (ADMM). The paper provides theoretical results on the convergence rate of this algorithm, establishing bounds in terms of both the objective function value and the distance to the optimal solution. To enhance computational efficiency, the algorithm is designed to exploit special matrix structures, such as sparsity and low-rank properties.

The authors implement their algorithm in a scalable parallel framework, demonstrating its compatibility with both shared and distributed memory architectures. The numerical studies benchmark the algorithm's runtime against its direct competitors using both synthetic and real-world datasets. A detailed analysis is also presented to evaluate the algorithm's speedup as a function of block size and the number of processing cores on both shared and distributed memory systems.

### Comments

The paper is well-written and clearly presented. The introduction effectively articulates the novelty and contributions of the proposed method. The authors successfully position their work within the existing literature, establishing clear connections to related research with a solid set of bibliographical references. The algorithmic description strikes an excellent balance between the technical details of numerical analysis and the theoretical guarantees of its convergence. The empirical evaluation is appropriate, and the implementation demonstrates very impressive performance.

My primary concern relates to the reproducibility and practical utility of this work. The value of such a powerful computational tool would be immensely amplified if the source code were made publicly available. Given that the implementation is based on open-source libraries, releasing the code should be feasible and would be a significant service to the research community.

In summary, this is a strong algorithmic paper that tackles the important practical problem of covariance selection in very high dimensions. The proposed method represents a powerful new tool for practitioners. This work is highly deserving of publication, and its impact would be greatly enhanced if the software were made available to the research community. I recommend acceptance, with the strong suggestion that the authors release their code.
