- Addresses covariance selection at extreme scale (millions of variables, trillions of parameters).  
- Focuses on the CLIME estimator, highlighting column-wise solvability and easier theoretical analysis than Graphical Lasso.  
- Proposes a novel block-wise CLIME algorithm solved via an inexact augmented Lagrangian method.  
- Provides theoretical convergence rates for both the objective function and distance to optimality.  
- Leverages special matrix structures (sparsity, low rank) to reduce computational cost.  
- Implements the algorithm in a scalable parallel framework for shared and distributed memory systems.  
- Presents numerical comparisons of runtime against direct competitors on synthetic and real datasets.  
- Analyzes speedup with respect to block sizes and number of cores on both shared and distributed systems.  
- Paper is well-written, clear, and situates contributions with solid bibliographical references.  
- Algorithm exposition balances numerical analysis, algebraic detail, and theoretical guarantees.  
- Implementation demonstrates impressive empirical performance.  
- Main concern: impact is limited if the source code (largely based on open-source libraries) is not released.  
- Recommendation: publish provided the implementation is made available to the community.
