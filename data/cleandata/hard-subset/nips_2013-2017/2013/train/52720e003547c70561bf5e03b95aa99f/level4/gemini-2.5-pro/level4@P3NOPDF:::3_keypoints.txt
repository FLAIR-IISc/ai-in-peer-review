- Tackles large-scale covariance selection (up to millions of features, trillions of parameters) using the CLIME estimator.  
- Proposes a novel block-wise algorithm for CLIME based on inexact ADMM, departing from standard column-wise methods.  
- Provides theoretical convergence guarantees for objective value and distance to the optimum.  
- Exploits matrix structure (sparsity, low-rank) to reduce computational cost.  
- Implements the method in a scalable parallel framework for shared and distributed memory architectures.  
- Empirical evaluation is thorough, showing strong performance and scalability on synthetic and real datasets.  
- Detailed study of speedup versus block size and number of processor cores.  
- Well-written paper that situates the contribution clearly within related work and provides good references.  
- Algorithm presentation balances numerical/linear-algebra details with theoretical analysis.  
- Primary concern: source code not publicly available, which would hinder reproducibility and impact.  
- Recommendation: accept, strongly urging authors to release the implementation; publication should be contingent on code release.
