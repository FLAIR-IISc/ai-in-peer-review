

This paper addresses covariance selection problems in extremely large-scale settings (up to millions of features and trillions of parameters). It focuses on the CLIME estimator, which enables column-wise precision matrix estimation for Gaussian vectors—facilitating distributed computation and offering theoretically simpler analysis compared to Graphical-Lasso. The authors introduce a novel block-wise algorithm for CLIME, combined with an inexact ADMM approach, and establish theoretical convergence rates for both the objective function and distance to optimality. Special computational efficiency is achieved through structured handling of sparse and low-rank matrix properties. The implementation leverages a scalable parallel framework supporting both shared and distributed memory systems. Numerical experiments demonstrate runtime comparisons against competitors on synthetic and real data, alongside a detailed analysis of speedup dependencies on block size and core count across different parallel architectures.  

The paper is exceptionally well-written and clear. The novelty and contributions are precisely articulated in the introduction, with strong contextualization to existing literature and thorough bibliographical references. The algorithmic presentation balances technical numerical analysis with rigorous convergence guarantees. The numerical evaluation is appropriate, and the implementation achieves remarkable performance gains.  

My sole reservation pertains to reproducibility: if the source code (primarily built on open-source libraries) is not publicly released, the practical value of this significant contribution is substantially diminished. A robust algorithmic solution to large-scale covariance selection—addressing a critical real-world challenge—deserves publication only if made accessible to the research community.
