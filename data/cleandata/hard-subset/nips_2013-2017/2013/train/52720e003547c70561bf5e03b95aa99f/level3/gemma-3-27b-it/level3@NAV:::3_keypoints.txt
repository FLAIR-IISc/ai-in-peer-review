- Tackles large-scale sparse precision matrix estimation using the CLIME estimator, addressing millions of variables and trillions of parameters.
- Proposes a novel block-wise solver combined with an inexact ADMM approach leveraging CLIME’s column-wise solvability.
- Strong theoretical contribution: convergence rates are established for the proposed algorithm.
- Well-structured and clearly written with solid literature contextualization.
- Implementation emphasizes scalability across shared and distributed memory systems.
- Experimental results on synthetic and real data show substantial performance gains over state-of-the-art methods.
- Detailed scalability analysis exploring block sizes and core counts.
- Practical implementation details include block cyclic data distribution to mitigate load imbalance.
- Insights on exploiting sparsity and low-rank structure to reduce computation are valuable.
- Theoretical analysis of perturbation strategies to induce sparsity in the sample covariance matrix is a clever addition.
- Major reproducibility concern: lack of publicly available source code limits independent verification and impact.
- Acceptance recommendation contingent on authors releasing source code to enable reproduction and broader adoption.
