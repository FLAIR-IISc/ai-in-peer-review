- Presents a novel block-wise inexact ADMM algorithm for CLIME to solve large-scale covariance selection via column-block updates, exploiting sparse and low-rank structure.
- Provides rigorous theoretical convergence rates for the objective and distance to optimality.
- Implements a scalable parallel framework supporting shared- and distributed-memory systems and demonstrates strong runtime performance on synthetic and real data.
- Technical novelty: block cyclic distribution and column-block ADMM for scalable precision matrix estimation.
- Theoretical guarantees: strong convergence results that bolster credibility.
- Scalability: can handle ultra-large problems (e.g., ~1M dimensions, ~1T parameters) with near-linear core scaling.
- Comprehensive experiments: robust comparisons to state-of-the-art methods and detailed speedup analysis by block size and core count.
- Clarity and presentation: well-written with effective figures and tables.
- Weakness: source code is not publicly available, limiting reproducibility and adoption.
- Weakness: practical distributed-system issues (communication overhead, fault tolerance) are not thoroughly addressed.
- Weakness: comparisons could better explain why competing methods (e.g., Tiger, DC-QUIC) perform differently across scenarios.
- For acceptance: addresses a significant high-dimensional problem with a novel, scalable, theoretically backed, and empirically validated solution.
- Against acceptance: lack of released code and incomplete treatment of practical distributed implementation concerns reduce immediate impact.
- Recommendation: strong scientific contribution conditional on releasing source code to maximize impact; if code is released, the paper deserves acceptance.
