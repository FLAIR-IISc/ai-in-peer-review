- Overall: Strong positive recommendation for a significant, scalable contribution to sparse precision matrix estimation in ultra-high dimensions.
- Problem: Tackles estimation of sparse precision matrices when p is extremely large (millions) and n << p, addressing computational bottlenecks of existing methods.
- Method: Proposes a distributed CLIME-ADMM algorithm using inexact ADMM, column-block updates, and optimized matrix multiplication for scalability.
- Theory: Provides convergence analysis and theoretical guarantees building on existing literature.
- Empirics: Strong empirical validation with substantial speedups and demonstration of a 1M-dimensional precision estimate in 11 hours on 400 cores.
- Implementation: Detailed distributed implementation (block cyclic data distribution) and practical considerations for scaling.
- Related Work: Clear positioning against prior scaling approaches and constrained optimization limitations.
- Importance: Solves a crucial problem relevant to genomics, finance, and other high-dimensional fields.
- Methodological Soundness: CLIME choice justified; inexact ADMM and block updates are well-motivated and effective.
- Comparison: Comprehensive runtime comparisons with CLIME, Tiger, and DC-QUIC.
- Clarity: Paper is dense and technical; mathematical derivations and heavy notation hinder accessibility.
- Parameter Sensitivity: Limited exploration of sensitivity to parameters (λ, ρ, η, block size k) and tuning strategies.
- Structural Assumptions: Discussion of exploiting sparse/low-rank structure is disjoint and lacks practical implementation details and prevalence analysis.
- Real-world Evaluation: Reliance on synthetic data for accuracy; real datasets used mainly for runtime without ground truth comparisons.
- Memory Analysis: Insufficient detailed analysis of memory footprint and scalability with problem size.
- For Acceptance: Novel, scalable algorithm with theory and strong experiments; important and timely problem addressed.
- Against Acceptance: Density/clarity issues, limited parameter studies, and dependence on synthetic data for main evaluations.
- Recommendation: Strong accept, with suggested revisions to improve clarity, streamline notation, and add parameter sensitivity analysis.
