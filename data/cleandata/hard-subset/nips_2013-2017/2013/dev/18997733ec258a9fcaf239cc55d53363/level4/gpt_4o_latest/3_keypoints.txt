- Proposes contextual linear bandits on a graph where neighboring nodes have similar weight vectors — a compelling model for social-network recommendations.
- Algorithm runs standard contextual linear bandits per node but shares information across nodes using the graph Laplacian to update estimates and confidence intervals.
- Regret analysis is unsatisfactory because it analyzes a different algorithm (TCB) than the one proposed/evaluated (CB) with no justification for this discrepancy.
- No discussion of how differences between CB and TCB affect algorithm performance, evaluation, or deployment; theory should match the proposed method or be omitted.
- Unclear how key terms ln|M_t| and 2L scale with T and n, and no analysis of the effect of graph noise on these quantities.
- Analysis gives no guidance for setting the exploration parameter α and does not clarify how the regret bound depends on α.
- Experimental section is excellent: thorough descriptions, use of real-world data, and strong reproducibility details.
- The algorithm lacks a parameter to quantify how rewards influence neighboring nodes; suggest regularizing the Laplacian (e.g., adding a scalar multiple of the identity) as a principled approach.
- Question why the regret bound uses L(u1,…,un) instead of the regularized Laplacian used in the algorithm.
- Post-rebuttal view: including theory for a different algorithm creates confusion; removing the theoretical analysis would likely improve the paper’s reception.
- Overall: promising and relevant problem with excellent experiments, but the theoretical analysis is disappointing and does not add clear value.
