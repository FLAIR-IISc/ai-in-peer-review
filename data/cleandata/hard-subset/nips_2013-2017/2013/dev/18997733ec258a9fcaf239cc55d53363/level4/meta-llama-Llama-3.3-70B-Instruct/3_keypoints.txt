- Proposes contextual linear bandits on a graph where adjacent nodes have similar weight vectors, suitable for social recommendation systems.
- Algorithm runs classical contextual linear bandits per node, with nature selecting the node and neighbors updated via Laplacian-regularized sharing.
- Regret analysis is weak: it analyzes a different algorithm than the one proposed and evaluated, with no justification for the mismatch.
- Fails to discuss differences and implications between contextual bandits (CB) and transductive contextual bandits (TCB) for algorithm design, evaluation, and deployment.
- Analysis lacks clarity on key terms (e.g., ln|M_t|, 2L) and how they scale with T and n; effect of graph noise is not examined.
- Provides no practical guidance on choosing the exploration parameter α, nor clear exposition of how the bound depends on α.
- Experiments are strong: reproducible, well-described, use real data, and include useful practical considerations.
- Paper lacks a tunable parameter to control the degree/informativeness of graph-based updates; regularizing the Laplacian (vs identity) is suggested but not formalized.
- Inconsistency: regret bound uses unregularized L(u1,...,un) while the algorithm regularizes the Laplacian.
- Theoretical analysis pertains to a modified algorithm with unclear relation to the original, making the paper confusing; dropping the theory might improve the manuscript.
- Overall: promising graph-based extension with excellent experiments but underwhelming and potentially misleading theoretical analysis.
