This paper creates a new type of covariance function for multi-output Gaussian processes. Without explicitly mentioning cross-covariance functions, the authors propose using cross-power spectral density and applying inverse Fourier transforms to find the corresponding cross-covariance functions. This paper follows a similar concept to the single-output Gaussian process proposed by [6], where the power spectral density was represented by a mixture of Gaussians and then back-transformed to obtain a robust kernel function with extrapolation capabilities. The authors applied a synthetic data example and two real-world data examples, a weather dataset and the Swiss Jura dataset for metal density, using the newly established covariance.

Extending the concept from [6] seems like a clever way to create a new type of covariance function for multiple outputs. One of the objectives of [6] was to create a kernel function with extrapolation capabilities. It would be interesting to provide this covariance function for multiple outputs with the same capabilities. The experimental section only provides results for very simple datasets, and the results obtained are similar to those obtained with other covariance functions for multiple outputs. The impact of this paper can be enhanced by including tests for datasets for which no other model is suitable.
