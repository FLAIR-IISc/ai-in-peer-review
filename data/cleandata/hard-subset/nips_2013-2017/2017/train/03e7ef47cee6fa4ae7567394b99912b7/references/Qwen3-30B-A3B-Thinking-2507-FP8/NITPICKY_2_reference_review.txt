

# Review of "Query Complexity of Clustering with Side Information"

This paper studies the query complexity of clustering with side information, where the clustering algorithm has access to a similarity matrix that provides side information about whether pairs of points belong to the same cluster. The authors establish information-theoretic lower bounds and design algorithms that nearly match these bounds, showing significant reduction in query complexity compared to the standard O(nk) bound.

## Major Issues

1. **Critical Typo in Lower Bound**: The lower bound theorem (Theorem 2) states "Assume H2(f+‖f−) ≤ 118" which should be "H2(f+‖f−) ≤ 1/18". This is a serious error as the bound's validity depends on this condition. The rest of the paper incorrectly uses "118" instead of "1/18" in multiple places.

2. **Inconsistent Notation**: The paper uses inconsistent notation for the Hellinger divergence. Sometimes it's written as H2(f+‖f−), sometimes as H(f+‖f−), and sometimes as H^2(f+‖f−). The squared Hellinger divergence is defined as H2(f‖g) = 1/2 ∑(√f(i)-√g(i))^2, but the paper uses H2(f+‖f−) without consistency.

3. **Algorithm Notation Errors**: The algorithm description contains critical notational errors. The query complexity is stated as "O(k logn H2(f+‖f−))" which should be "O(k / (log n) H2(f+‖f−))" or "O(k / (H2(f+‖f−) log n))" to be meaningful. Similarly, "O(n log n + min(nk, k2 logn H2(f+‖f−)))" should be "O(n log n + min(nk, k^2 / (H2(f+‖f−) log n)))".

4. **Confusing Algorithm Description**: The algorithm's Membership function is defined as "Membership(v, C) = -H2(pv,C‖pC)" which is confusing since Hellinger divergence is non-negative. The negative sign isn't explained, and the description of how this relates to cluster assignment is unclear.

5. **Inconsistent Terminology**: The paper uses "side information" and "similarity matrix" interchangeably without clear distinction. The side information is described as "a noisy weighted upper-triangular similarity matrix W = {wi,j}", but later it's treated as a binary matrix in the algorithm.

6. **Experimental Results**: The experimental section claims "the Hellinger square divergence between the two distributions is 0.6" without showing how this value was computed or providing the actual distributions. The reference to "Figure 1 (middle)" is missing in the text.

## Technical Concerns

1. **Algorithm Complexity**: The claim that the algorithms are "parameter free" is questionable since the algorithms implicitly require knowledge of the support size of f+ and f- to compute the Hellinger divergence.

2. **Lower Bound Justification**: The proof of Lemma 1 has a logical gap. The calculation |J' ∩ J''| > (1+β)k/2 + (2-β)k/2 - k = k/2 is correct, but the interpretation of this result in the context of the algorithm is not clearly justified.

3. **Convergence Analysis**: The convergence analysis in Section 3 relies on adapting Sanov's theorem, but the paper doesn't explain why Hellinger distance (rather than KL divergence) is the appropriate measure for this application.

## Strengths

1. The paper addresses an important problem in interactive clustering with practical applications in entity resolution.

2. The theoretical framework is well-motivated and builds on previous work in stochastic block models.

3. The experimental results provide some empirical validation of the algorithm's performance.

## Overall Assessment

The paper presents an interesting theoretical framework for clustering with side information, but the technical execution contains multiple critical errors that undermine the validity of the main results. The consistent notational inconsistencies, critical typo in the lower bound condition, and confusing algorithm description significantly weaken the paper's contribution. 

While the problem is important and the theoretical approach is promising, these technical flaws prevent the paper from being accepted as a solid scientific contribution. The authors have made a good start but need to thoroughly revise the paper to correct these fundamental issues.

## Recommendation

Reject. The paper contains too many technical errors and inconsistencies that prevent the reader from verifying the main claims. The critical typo in Theorem 2 (H2(f+‖f−) ≤ 118 vs. H2(f+‖f−) ≤ 1/18) is particularly concerning as it fundamentally changes the meaning of the lower bound. The inconsistent notation throughout the paper further compounds the problem, making it difficult to assess the validity of the results.
