- Studies query complexity of exact clustering with adaptive same-cluster queries plus noisy similarity matrix W drawn from unknown f+ / f−; provides information-theoretic lower bounds and parameter-free algorithms.  
- Lower bounds: any algorithm needs Ω(min{nk, k^2/H^2}) expected queries (constant prob.) and Ω(n + min{nk, k^2/H^2}) for certainty, with H^2 = Hellinger^2(f+||f−) in the regime H^2 ≤ 1/18.  
- Algorithms: Monte Carlo (no parameters) with O(min{nk, (k log n)/H^2}) queries w.h.p. and Las Vegas with O(n log n + min{nk, (k^2 log n)/H^2}) queries; both run in O(n^2) time.  
- Technique: new lower-bound framework for adaptive queries and an iterative-update algorithm that estimates H^2 using Hellinger divergence and Sanov large-deviations tools.  
- Generalization: extends beyond SBM to arbitrary unknown discrete f+, f− and adversarial cluster sizes; connects query savings to H^2.  
- Theoretical soundness: lower bound reduction to multi-hypothesis testing is careful and addresses adaptive-query gaps; stated assumptions are reasonable though some constants are not optimized.  
- Upper bounds: parameter-free algorithms are nontrivial, use Hellinger for concentration and triangle inequalities, and nearly match lower bounds up to log factors.  
- Completeness: proofs sketched with full details in appendices; extension claims for larger supports are deferred; lower bound covers discrete and continuous cases.  
- Empirics: limited experiments (one ER dataset) mainly showing greedy feasibility; scaling with k and H^2 not stress-tested.  
- Clarity issues: typos, notational inconsistencies, and inconsistent query-complexity/success-probability expressions that need standardization.  
- Notational ambiguity: unclear assumptions about support size of f+, f− (fixed q vs. arbitrary pmfs) and some misformatted/incomplete references.  
- Presentation suggestion: provide concise pseudocode for Membership subroutine and summarize constants.  
- Originality: novel adaptive-query lower-bound framework and adaptive, parameter-free Hellinger-based algorithm.  
- Significance: advances interactive clustering theory and quantifies query savings via H^2; lower-bound technique has broader applicability.  
- Practical limitations: requires full O(n^2) independent similarity matrix and has limited empirical validation, tempering immediate practical impact.  
- Strengths: clear problem, near-tight rigorous bounds, parameter-free efficient algorithms, novel techniques, and links to SBM and crowdsourced ER.  
- Weaknesses: clarity/notation problems, unrealistic full independent W assumption for some applications, scalability/memory concerns, and minimal experiments.  
- Technical concern: lower bound assumes H^2 ≤ 1/18 — need discussion on extension beyond this regime.  
- Model assumptions: algorithm assumes discrete fixed-support distributions; continuous or growing-support settings need elaboration.  
- Experimental requests: run additional experiments varying k and synthetic f+/f− to validate 1/H^2 scaling and compare baselines.  
- Questions for authors: confirm intended query-complexity formulas, robustness to model misspecification, adaptation to sparse W, and changes needed for continuous f+/f−.  
- Minor fixes: clean references, define constants where first used, unify success-probability notation, and state memory requirements.  
- Recommendation: Accept (borderline strong) — solid, original theoretical contributions with near-optimal guarantees but needs clarity and stronger experiments.  
- Reviewer confidence: High.
