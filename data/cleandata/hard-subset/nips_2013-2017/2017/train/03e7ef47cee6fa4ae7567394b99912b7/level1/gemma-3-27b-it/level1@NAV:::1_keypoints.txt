- Core contribution: shows how using a similarity matrix (side information) dramatically reduces query complexity for clustering with pairwise oracle queries.  
- Main theoretical result: query complexity drops from Θ(nk) to O(k² log n · H(...)), with H representing the Hellinger divergence between similar/dissimilar pair distributions.  
- Provides both algorithmic upper bounds and matching lower bounds, yielding near-optimal query complexity results.  
- Positions the problem in relation to the stochastic block model (SBM) and generalizes to arbitrary similarity distributions with unknown parameters.  
- Motivation: well-grounded in real-world tasks like entity resolution where similarity functions are imperfect and human queries are used.  
- Novelty: fills a gap by systematically analyzing query complexity with side information, beyond prior heuristic approaches.  
- Generality: framework handles arbitrary similarity distributions and an unknown number of clusters.  
- Practical relevance: results could improve existing heuristics and impact interactive clustering applications.  
- Strength: rigorous, strong theoretical proofs and clear linkage to existing literature.  
- Weakness: presentation is dense and requires strong background in information theory and clustering; needs more intuitive explanations.  
- Weakness: experimental evaluation is limited to a single dataset and lacks broader comparisons to state-of-the-art heuristics.  
- Weakness: parameter-free algorithm is relegated to the appendix; explanation should be moved into the main text.  
- For acceptance: significant theoretical advance, well-motivated, likely to stimulate follow-up work.  
- Against acceptance: accessibility concerns due to technical depth and limited empirical validation.  
- Overall recommendation: accept with minor revisions to improve clarity and expand experiments.
