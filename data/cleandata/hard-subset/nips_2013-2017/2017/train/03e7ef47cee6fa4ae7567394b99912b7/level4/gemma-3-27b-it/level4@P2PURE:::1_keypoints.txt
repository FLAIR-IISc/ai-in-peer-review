- Problem studied: query complexity of clustering when given side information in a similarity matrix W whose entries follow f+ if same-cluster and f- otherwise.
- Baseline: without side information, Θ(nk) queries are necessary.
- Main upper bound: with W, only O(k log n / H^2(f+ || f-)) queries suffice, where H is the Hellinger distance between f+ and f-.
- Tightness: the bound is shown to be information-theoretically tight up to a log n factor.
- Novelty/connection: links query-based clustering to the Stochastic Block Model and studies how query access shifts the clustering threshold.
- Conceptual contribution: identifies the Hellinger distance as the key metric quantifying W’s usefulness and rigorously shows side information can dramatically reduce query complexity.
