- Problem setup: clustering with side information represented by similarity matrix W, where Wij ~ f+ if same cluster and ~ f- otherwise.
- Main theoretical result: with side information, O(k log n / H^2(f+||f-)) queries suffice for clustering (H = Hellinger distance).
- Baseline comparison: without side information clustering requires Θ(nk) queries.
- Optimality: the O(k log n / H^2) upper bound is tight up to a log(n) factor.
- Novelty: studies how the ability to make queries changes the necessary threshold for successful clustering.
- Relation to prior work: closely related to the Stochastic Block Model (SBM) literature but asks a new query-driven question.
- Impact: demonstrates side information can dramatically reduce query complexity.
- Contribution: provides a quantitative measure (Hellinger distance) of similarity-matrix quality and its effect on query complexity.
