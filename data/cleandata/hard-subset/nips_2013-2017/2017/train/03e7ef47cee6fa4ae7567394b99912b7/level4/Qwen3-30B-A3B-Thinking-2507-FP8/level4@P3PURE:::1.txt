

This paper investigates the query complexity of clustering problems when provided with side information in the form of a similarity matrix $W$. Specifically, the entry $W_{ij}$ is distributed according to $f_+$ if vertices $i$ and $j$ belong to the same cluster, and according to $f_-$ otherwise. The main contribution establishes that $\Theta(nk)$ queries are necessary without side information, while only $O\left(k \log n / H^2(f_+ \| f_-)\right)$ queries suffice with side information, where $H$ denotes the Hellinger distance. The authors further demonstrate that this bound is tight up to logarithmic factors in $n$.

The result presents a theoretically interesting advance for several reasons. It closely relates to the stochastic block model (SBM) literature, which has been highly active in recent years, but addresses a novel question: how does the introduction of queries alter the computational threshold for clustering? The work rigorously quantifies the significant reduction in query complexity achievable through side information. While it is not unexpected that side information could drastically reduce complexity (e.g., if it provides complete cluster assignments), the paper's key contribution lies in precisely characterizing how to assess the quality of the similarity matrix via the Hellinger distance between the underlying distributions. This provides a fundamental framework for understanding the utility of side information in clustering.
