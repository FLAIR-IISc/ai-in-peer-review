- Problem setup: studies clustering query complexity with side information given by a similarity matrix W where W_ij ~ f+ if i,j same cluster and ~ f- otherwise.
- Baseline: without side information clustering requires Θ(nk) queries.
- Main result: using W reduces query complexity to O(k log n / H^2(f+ || f-)).
- Metric: H is the Hellinger distance, which quantifies the required quality of W for query reduction.
- Optimality: the claimed query complexity is asymptotically optimal up to a log n factor.
- Novelty: frames a new query-based question related to SBM rather than directly applying existing SBM results.
- Methodological contribution: adapts SBM techniques to analyze the impact of queries on clustering thresholds.
- Tightness: provides tightness results that strengthen the theoretical guarantees of the reduction.
