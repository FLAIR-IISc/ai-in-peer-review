- Proposes "Aggressive Sampling for Multi-class to Binary Reduction" with a (π, κ)-double sampling strategy to handle long-tailed class distributions.  
- First step balances classes by oversampling small classes and subsampling large classes (π).  
- Second step reduces dyadic examples by considering only κ adversarial classes per example.  
- Provides theoretical analysis with generalization bounds using local fractional Rademacher complexity.  
- Analysis accounts for class distribution shift and dependencies between sampled dyadic examples.  
- Empirical evaluation on large text datasets (up to 100K classes) shows strong performance and efficiency.  
- Compares comprehensively to strong baselines (OVA, M-SVM, FastXML, PfastReXML, RecallTree, PD-Sparse).  
- Demonstrates significant reductions in computational complexity and memory while maintaining predictive performance.  
- Addresses limitations of existing methods (tree-based error propagation, OVA/M-SVM linear K complexity).  
- Well-written, clearly organized, with consistent notation and clear algorithm pseudocode.  
- Could better justify the chosen loss function (Eq. 1) and its relation to large-scale ranking challenges.  
- Lacks practical guidance on selecting hyperparameters (π, κ).  
- Doesn't discuss applicability limits outside text classification domains.  
- Comparison to the latest state-of-the-art could be more exhaustive (authors justify some exclusions).  
- Novel contribution: aggressive double sampling specifically targeting long-tailed distributions.  
- Significant practical impact: enables classification at 100K classes with favorable time/memory trade-offs.  
- Recommendation: Accept—rigorous theory, strong experiments, and meaningful advancement for large-scale classification.
