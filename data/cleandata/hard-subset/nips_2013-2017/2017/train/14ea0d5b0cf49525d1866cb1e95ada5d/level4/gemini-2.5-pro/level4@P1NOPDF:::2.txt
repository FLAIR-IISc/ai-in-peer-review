### Summary

This paper proposes a novel reduction from multi-class classification to binary classification, particularly well-suited for problems with a very large number of classes. The method is based on a hypothesis that maps (input, class) pairs to a score. The underlying loss function then measures the fraction of incorrect classes that are scored higher than the true class. More specifically, the authors assume a feature transformation φ that maps (input, class) pairs into a p-dimensional feature space, and then learn a scoring function over this space.

This reduction extends the work of Joshi et al. (2015), which, for each data point (x, y), creates K-1 transformed points, where each transformed point represents a comparison between the true label y and an incorrect label y'. As this transformation introduces correlations among the training examples, standard generalization bounds are not applicable. The paper's first contribution is an improved generalization analysis over Joshi et al. (2015), leveraging new bounds on the fractional Rademacher complexity for this setting.

The second main contribution is to show that sampling techniques can be incorporated into the reduction to balance class distributions and reduce the training set size. Specifically, the authors employ two sampling strategies: first, they sample classes with replacement to ensure a more balanced representation in the transformed data; second, rather than generating transformed points for all K-1 incorrect classes, they use only a random subset of these classes for comparison. The authors demonstrate that these two sampling procedures do not significantly degrade their generalization bounds. Finally, the paper presents strong empirical results, showing that the proposed procedure performs competitively against several strong baselines.

### Comments

This paper presents an interesting reduction from multi-class classification with a large number of classes to binary classification. A key strength is the development of corresponding generalization bounds that account for the correlated data generated by this reduction. The primary novelty lies in the proposed sampling techniques for rebalancing class distributions and reducing the transformed dataset's size, as the underlying reduction framework was previously introduced by Joshi et al. The empirical comparison to existing algorithms is very strong.

One area for improvement is the lack of theoretical analysis for the prediction-time algorithm described in Section 2.3. This algorithm accelerates inference by comparing an input point only against the *k* classes with the nearest centroids. The paper does not formally discuss the impact of this approximation on performance or how it relates to the derived generalization bounds.

Additionally, the paper would be strengthened by a more detailed discussion of the sample complexity bounds. For instance, comparing these bounds to standard generalization bounds for the original (non-transformed) multi-class problem would provide valuable context and better highlight the trade-offs inherent in this reduction method.
