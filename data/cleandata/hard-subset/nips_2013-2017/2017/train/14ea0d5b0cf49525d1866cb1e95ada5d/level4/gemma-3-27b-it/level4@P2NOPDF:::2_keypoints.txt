- Proposes a reduction from multi-class to binary classification using a score function on (input, class) pairs and a loss equal to the fraction of incorrect classes scoring higher than the true class.  
- Uses a feature transformation φ mapping (input, class) to R^p and learns a mapping R^p → scores.  
- Builds on Joshi et al. (2015) which creates K−1 correlated comparison examples per instance, causing standard generalization bounds to fail.  
- Main contribution 1: refined generalization analysis with novel fractional Rademacher complexity bounds tailored to correlated transformed data, improving on Joshi et al.  
- Main contribution 2: sampling techniques to balance class representation and reduce training set size (sampling classes with replacement and sampling a subset of negatives instead of all K−1).  
- Shows theoretically that the proposed sampling procedures preserve the derived generalization guarantees.  
- Empirical results: competitive performance against several strong baselines.  
- Strength: effective reduction for large-K problems, with significant novelty in sampling to address class imbalance and dataset size.  
- Weakness: lacks detailed theoretical analysis of the prediction algorithm in Section 2.3 that limits comparisons to the k nearest class centroids.  
- Suggestion: provide a theoretical study of how restricting to k nearest centroids affects performance.  
- Suggestion: compare sample complexity bounds to standard non-transformed multi-class bounds, especially how complexity scales with number of classes K.
