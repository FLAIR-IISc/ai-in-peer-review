- Proposes a reduction from multi-class to binary classification tailored for many classes, using a scorer on (input, class) pairs.
- Defines loss as fraction of incorrect classes scored higher than the true class and uses a feature map phi: (input,class) -> R^p.
- Learns a mapping from R^p to scores for class comparison.
- Builds on Joshi et al. (2015) which creates K-1 transformed points per example representing pairwise comparisons.
- Notes transformed dataset contains correlated examples, making many standard generalization bounds inapplicable.
- First contribution: improved generalization analysis over Joshi et al. via new bounds on fractional Rademacher complexity.
- Second contribution: sampling procedures to rebalance class distributions and reduce transformed dataset size (sample classes with replacement; sample a subset of K-1 comparisons).
- Shows the sampling procedures do not substantially worsen the generalization bounds.
- Empirically competitive against several strong baselines.
- Main novelty is the sampling-based rebalancing/reduction; the basic reduction itself follows prior work.
- Omissions: no theoretical analysis of the Section 2.3 prediction algorithm that compares only the k nearest class centroids.
- Omissions: lacks a deeper discussion/comparison of sample complexity bounds to those for non-transformed multi-class methods.
