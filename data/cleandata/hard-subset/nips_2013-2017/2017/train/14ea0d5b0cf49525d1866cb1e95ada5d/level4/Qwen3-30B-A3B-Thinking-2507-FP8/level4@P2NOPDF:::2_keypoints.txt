- Introduces a reduction from multi-class to binary classification for very large K, using a score function on (input, class) pairs and a ranking loss counting incorrect classes above the true class.  
- Implements the reduction via a feature transform φ:(x,y)→ℝ^p and a learned map ℝ^p→scores, extending Joshi et al. (2015).  
- Notes that the transformed dataset exhibits label correlations, breaking standard generalization bounds.  
- Contribution 1: provides improved generalization analysis using tighter fractional Rademacher complexity bounds for correlated data.  
- Contribution 2: proposes a sampling strategy (class sampling with replacement and random class comparisons) to rebalance classes and reduce training size while preserving generalization guarantees.  
- Empirical results show competitive performance, often outperforming strong baselines on large-scale multi-class tasks.  
- Strength: the sampling approach is practical, reduces computational overhead, and is theoretically supported.  
- Theoretical gap 1: no analysis of the prediction step that restricts comparisons to the k nearest class centroids—missing bounds on accuracy/generalization trade-offs.  
- Theoretical gap 2: lacks comparative sample complexity analysis against standard multi-class methods (e.g., one-vs-all, softmax), leaving data-efficiency benefits unclear.  
- Writing suggestions: clarify wording (e.g., "fraction of incorrect classes ranked above the true class") and more explicitly connect sampling to the fractional Rademacher complexity (e.g., effect on effective VC-dimension).  
- Overall verdict: strong, NIPS-worthy contribution; recommend acceptance pending minor revisions to address the two theoretical gaps.
