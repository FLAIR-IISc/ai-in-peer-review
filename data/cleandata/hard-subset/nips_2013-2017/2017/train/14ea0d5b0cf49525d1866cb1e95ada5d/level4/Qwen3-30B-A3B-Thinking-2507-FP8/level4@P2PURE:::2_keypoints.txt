- Introduces a reduction from multi-class to binary classification for very large K using a score function on (input, class) pairs and a loss based on fraction of incorrect classes ranked above the true class.  
- Assumes a feature transform φ mapping (input, class) pairs to ℝ^p and learns a mapping from ℝ^p to scores.  
- Builds on Joshi et al. (2015) by creating K−1 comparison examples per datapoint, which yields correlated examples in the transformed dataset.  
- Main theoretical contribution: new fractional Rademacher complexity bounds to handle correlated examples and improve generalization analysis.  
- Main algorithmic contribution: sampling procedures that rebalance class representation and reduce the number of comparisons by sampling classes with replacement and using only a random subset of comparisons.  
- Proves the sampling procedures have negligible effect on the derived generalization bounds.  
- Empirical results show the method competes strongly against multiple strong baselines.  
- Reviewer notes the primary novelty is the sampling/rewighting techniques; the core reduction structure largely follows prior work (Joshi et al.).  
- Missing: no theoretical analysis of the prediction algorithm in Section 2.3 that restricts comparisons to the K nearest class centroids.  
- Missing: insufficient discussion of sample complexity bounds and lacking comparison to generalization bounds for non-transformed multi-class methods.
