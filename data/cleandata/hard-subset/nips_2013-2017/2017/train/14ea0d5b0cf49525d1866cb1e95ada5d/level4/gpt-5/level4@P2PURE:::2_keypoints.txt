- Paper proposes a reduction from multiclass to binary classification for very large class sets that learns a scoring function over (input, class) pairs using a p-dimensional feature map φ.
- Loss is defined as the fraction of incorrect classes that score higher than the true class, and a function from R^p to scores is learned.
- Method extends Joshi et al. (2015) which creates K−1 correlated transformed examples per original example, making standard generalization bounds inapplicable.
- First contribution: improved generalization analysis for the correlated transformed dataset using new bounds on fractional Rademacher complexity.
- Second contribution: a sampling scheme that rebalances class distributions (sampling each class with replacement) and reduces transformed-data size (keep a random subset of classes instead of all K−1 comparisons).
- Authors prove these sampling procedures do not significantly degrade the generalization bounds.
- Empirical results show the method is very competitive with several strong baselines.
- Main novelty is the sampling/rebalancing scheme; the core reduction follows prior work (Joshi et al.).
- Omitted: theoretical analysis of the Section 2.3 prediction procedure that restricts comparisons to the k nearest centroids.
- Omitted: further discussion and comparison of sample-complexity bounds to generalization guarantees for the original multiclass formulation.
