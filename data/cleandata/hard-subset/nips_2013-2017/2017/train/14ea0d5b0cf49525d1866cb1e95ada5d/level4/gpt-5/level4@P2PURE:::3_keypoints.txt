- Paper proposes reducing extreme multiclass classification to pairwise binary classification via a scalable, sampling-based reduction.  
- Critiques existing methods: label trees can be suboptimal and cause cascading errors.  
- Critiques label-embedding methods: suffer when the true label matrix is not low-rank.  
- The proposed method comes with theoretical consistency guarantees relative to the multiclass ERM objective.  
- Empirical results on five text datasets show higher accuracy and F1, faster training, and much lower memory use once classes exceed ~30k.  
- Prediction is somewhat slower due to pairwise classification at inference.  
- Aggressive Double Sampling uses two parameters: per-class sampling frequency set inversely to empirical class frequency, and a number of uniformly drawn adversarial examples.  
- The sampling procedure repeats to train a final dyadic classifier, helping handle long-tail classes.  
- Inference uses a heuristic: build class centroids, select candidate classes by centroid similarity, then run pairwise classification among candidates.  
- Theoretical analysis handles dependence induced by dyadic examples and distribution shift from oversampling.  
- Proof strategy splits the dependent sum using the dyadic-example graph structure and applies Janson’s concentration for partially dependent variables.  
- Finite-sample risk bound is biased by the ratio of true to oversampled class probabilities, but the bias decreases linearly with sampled/resampled training set sizes.  
- Overall assessment: method is theoretically grounded, practically useful, and particularly strong on very large label spaces.
