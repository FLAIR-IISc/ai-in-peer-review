- Proposes a reduction from multi-class to binary classification using a feature transform φ mapping (input, class) pairs to a p-dimensional space and scoring function on R^p.  
- Uses a loss that counts fraction of incorrect classes scoring higher than the true class.  
- Builds on Joshi et al. (2015) which creates K-1 transformed comparison points per example.  
- Identifies correlated training examples in the transformed dataset where standard generalization bounds may fail.  
- Contribution 1: provides improved generalization analysis using new bounds on fractional Rademacher complexity tailored to the correlated setting.  
- Contribution 2: introduces sampling techniques to rebalance class representation and reduce transformed dataset size by sampling classes with designed probabilities and using random subsets of classes.  
- Proves the sampling procedures do not significantly degrade the generalization bounds.  
- Empirically competitive with strong baselines on large-scale datasets.  
- Primary novelty is the sampling-based rebalancing and dataset-size reduction; the core reduction echoes prior work by Joshi et al.  
- Missing theoretical analysis of the prediction/candidate-selection algorithm (section 2.3) that evaluates only k nearest centroids.  
- Requests more discussion and comparison of sample complexity bounds against non-transformed multi-class generalization results.
