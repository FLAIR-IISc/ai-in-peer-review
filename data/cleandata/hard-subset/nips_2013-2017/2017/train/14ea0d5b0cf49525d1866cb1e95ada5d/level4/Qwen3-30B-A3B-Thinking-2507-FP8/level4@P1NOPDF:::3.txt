

This paper introduces a novel learning reduction for extreme classification, where the output space can involve up to 100,000 classes. While many existing approaches rely on either learning a latent tree structure (e.g., hierarchical softmax) or label embeddings, the authors correctly identify key limitations: tree-based methods risk error propagation due to suboptimal structures, and embedding approaches often fail when the true label matrix is not approximately low-rank.  

The authors instead adopt a reduction to pairwise binary classification, proposing a scalable sampling-based method with theoretical guarantees for consistency with the original multiclass ERM formulation. Experiments on five text datasets demonstrate that their Aggressive Double Sampling (ADS) method consistently outperforms competitors in accuracy and F1-score when the number of classes exceeds 30,000. It also achieves significant gains in training speed and memory efficiency, though prediction speed is slightly slower than some alternatives.  

The ADS reduction operates with two key parameters: one controlling class sampling frequency (inversely proportional to empirical class frequency to prevent missing rare classes in the long tail), and another setting the number of adversarial examples drawn uniformly. The sampling procedure is iterated to train the final dyadic classifier.  

At prediction time, the intractability of computing pairwise features for all classes is addressed via a heuristic: each class is represented by the centroid of its input-space vectors, and prediction proceeds through pairwise classifications only with candidate classes identified by this centroid proximity.  

The theoretical analysis faces two challenges: (1) the reduction transforms independent random variables into a dependent structure via dyadic examples, and (2) oversampling rare classes shifts the empirical distribution. The authors resolve these by decomposing the dependent sum into independent components using the graph structure induced by the sampling process, then applying Janson’s concentration inequalities for partially dependent variables. While the finite-sample risk bound incorporates a bias proportional to the ratio of true to oversampled class probabilities, this bias decreases linearly with training set size. The analysis appears sound, though I have not verified the full proof.  

Overall, this work presents a compelling solution for extreme classification. The method delivers strong empirical performance—particularly on large-scale datasets—with minimal memory and computational overhead. The binary reduction framework is theoretically rigorous, conceptually clear, and offers practical value for practitioners dealing with massive output spaces. The contributions are well-motivated, technically solid, and the experimental validation is thorough.
