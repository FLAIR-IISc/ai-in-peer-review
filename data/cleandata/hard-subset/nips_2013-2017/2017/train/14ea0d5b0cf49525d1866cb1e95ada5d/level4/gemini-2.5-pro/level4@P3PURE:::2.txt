### Summary

This paper proposes a reduction from multi-class to binary classification, designed for problems with a very large number of classes. The approach is based on a hypothesis class that maps (input, class) pairs to real-valued scores, with a prediction corresponding to the class that maximizes this score. The underlying loss function is a ranking loss that counts the fraction of incorrect classes scored higher than the correct class. More specifically, the authors define a feature transformation `phi` that maps an (input, class) pair to a p-dimensional feature space, and then learn a function from this feature space to the score.

This reduction builds upon the work of Joshi et al. (2015), where each data point (x,y) is expanded into K-1 training examples for a binary classifier, with each new example representing a comparison between the true class y and an incorrect class y'. A key challenge with this approach is that the resulting transformed dataset contains highly correlated examples, rendering standard generalization bounds inapplicable.

The paper's contributions are as follows. First, it provides an improved generalization analysis for this setting, deriving novel bounds on the fractional Rademacher complexity that account for the data correlation. Second, it introduces sampling techniques into the reduction process to balance class distributions and reduce the overall size of the transformed training set. Specifically, the authors propose sampling classes with replacement to create a more balanced representation and, for each data point, sampling a random subset of incorrect classes for comparison rather than all K-1 of them. The authors demonstrate theoretically that these sampling procedures can be employed without significantly weakening the generalization bounds. Finally, empirical results show that the proposed procedure is highly competitive with several strong baseline methods.

### Comments

This paper presents an interesting and effective reduction from multi-class to binary classification, particularly for settings with a large number of classes. The corresponding generalization bounds, which are designed to handle the correlated data generated by the reduction, are a key strength. The primary novelty lies in the proposed sampling techniques for rebalancing class distributions and reducing the training set size, as the foundational reduction framework was introduced in prior work by Joshi et al. The empirical comparison against existing algorithms is very strong.

However, the paper has a few weaknesses. A notable omission is a theoretical analysis of the approximate prediction algorithm presented in Section 2.3. This algorithm restricts comparisons to only the *k* classes whose centroids are nearest to the input point, but its impact on the theoretical guarantees is not discussed.

Additionally, the paper would be strengthened by a more detailed discussion of the sample complexity bounds. In particular, a comparison of these new bounds to existing generalization bounds for standard (non-transformed) multi-class classification would provide valuable context and better highlight the trade-offs of this reduction method.
