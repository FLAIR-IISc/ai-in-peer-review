Summary:

This paper proposes a novel reduction from multi-class classification to binary classification, particularly suited for scenarios with a large number of classes. The authors consider a hypothesis that maps input-class pairs to scores, with the underlying loss function counting the fraction of incorrect classes scored higher than the true class. Specifically, they assume a feature transformation phi that maps input-class pairs to a p-dimensional feature space, and learn a mapping from R^p to scores.

Their reduction builds upon the work of Joshi et al. (2015), which creates K-1 transformed points for each data point (x,y), where each transformed point intuitively corresponds to the comparison of label y with some incorrect label y'. However, the transformed dataset contains correlated training examples, rendering many standard generalization bounds inapplicable. The paper's first contribution is an improved generalization analysis over Joshi et al. (2015) using new bounds on the fractional Rademacher complexity in this setting. The second main contribution is demonstrating the feasibility of employing sampling techniques during the reduction to balance class distributions and decrease the training set size. Specifically, they sample each class (with replacement) to ensure roughly equal representation in the transformed training data, and use only a random sample of classes for transformed points, rather than all K-1 possible class comparisons. The authors show that these sampling procedures do not significantly affect their generalization bounds. Finally, they demonstrate that this new procedure performs competitively against several strong baselines.

Comments:

This paper presents an interesting reduction from multi-class classification with many classes to binary classification, along with corresponding generalization bounds for handling correlated data in the reduced dataset. The main novelty of the reduction lies in the sampling technique for rebalancing class distributions and reducing the size of the transformed dataset, while the basic reduction was introduced in an earlier paper by Joshi et al. The empirical comparison to existing algorithms is strong.

One notable omission is a theoretical analysis of the impact of the prediction algorithm outlined in section 2.3, where comparisons are only performed for the k classes whose centroids are closest to a given input point.

Additional discussion of the sample complexity bounds, possibly comparing them to the generalization bounds obtained for non-transformed multi-class problems, would have been appreciated.
