

This paper addresses extreme classification, a multiclass setting where the output space reaches up to 100,000 distinct classes. Existing approaches often rely on inferring a tree structure among output labels (e.g., hierarchical softmax), requiring only logarithmic binary predictions, or label embedding methods learned via least-squares or sampling. The authors identify key limitations: inferred tree structures may be suboptimal, causing cascading errors, while label embedding can yield prediction inaccuracies when the true label matrix is not low-rank.  

An alternative framework reduces extreme classification to pairwise binary classification. The authors adopt this approach, proposing a scalable sampling-based method with theoretical guarantees for consistency with the original multiclass ERM formulation.  

Experiments across five text datasets demonstrate that the method consistently outperforms competitors in accuracy and F1 score once the class count exceeds 30,000. It also trains faster and consumes significantly less memory, though prediction speed is slightly reduced. The Aggressive Double Sampling reduction employs two parameters: one governs class sampling frequency (inversely proportional to observed class frequencies to prevent missing rare classes in the long tail), and the other specifies the number of uniformly drawn adversarial examples. This process trains the final dyadic classifier through repeated iterations.  

At prediction time, pairwise feature generation with all classes is intractable. A heuristic identifies candidate classes by computing centroid representations (class averages in input space), followed by pairwise classification only over these candidates.  

Theoretical analysis is complicated by two factors: the reduction from multiclass to dyadic binary classification transforms a sum of independent random variables into a sum with specific dependence structures across dyadic examples, and oversampling rare classes shifts the empirical distribution. The proof resolves this by partitioning the dependent sum into independent sums using the graph structure induced by example construction, then applying Janson's concentration inequalities for partially dependent variables. Although the finite-sample risk bound exhibits bias proportional to the ratio of true to oversampled class probabilities, this bias diminishes linearly with the size of the sampled and re-sampled training sets. The analysis appears sound, though the reviewer did not verify the proof in detail.  

Overall, this paper presents a robust extreme classification method with minimal memory and computational demands, achieving superior performance—particularly on the largest datasets. The binary reduction algorithm is theoretically well-founded, novel, and highly applicable to practitioners.
