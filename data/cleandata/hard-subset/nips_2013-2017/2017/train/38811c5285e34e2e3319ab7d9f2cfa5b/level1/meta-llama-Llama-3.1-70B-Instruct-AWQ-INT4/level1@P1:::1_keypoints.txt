- Proposes a novel, black-box parallelisation scheme for ML that needs no extra derivations or dedicated code.  
- The scheme reduces runtime of many learning algorithms to polylogarithmic time using quasi-polynomially many processors.  
- Theoretical result: algorithms meeting certain conditions can be parallelised to achieve (ε,∆)-guarantees in time polylogarithmic in N0(ε,∆).  
- The runtime reduction cost is bounded by a quasi-polynomial in 1/ε and log(1/∆).  
- Practical design: run base learner on random data subsets, aggregate hypotheses in a tree using Radon points at internal nodes.  
- Empirical results: with 150 processors the Radon machine is 80–700× faster than the base algorithm on one processor with the same data.  
- Empirical evaluation is thorough and demonstrates practical potential.  
- Paper is well-written, well-organized, and technically sound.  
- Novelty: uses Radon points for aggregation, a new approach to parallelising learning algorithms.  
- Significance: advances understanding of efficient parallelisation for ML and has implications for large-scale learning.  
- Weakness: assumes abundant data and a fixed finite-dimensional Euclidean parametrisation of hypothesis space.  
- Weakness: lacks a detailed analysis of communication complexity.  
- Weakness: does not address applicability to non-linear models.  
- Recommendation: accept with minor revisions.  
- Requested revisions: add a detailed communication-complexity analysis and discuss applicability to non-linear models.
