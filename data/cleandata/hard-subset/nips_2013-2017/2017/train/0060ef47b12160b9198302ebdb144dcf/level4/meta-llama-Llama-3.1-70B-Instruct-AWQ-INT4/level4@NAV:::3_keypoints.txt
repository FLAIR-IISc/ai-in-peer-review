- Proposes a novel image saliency mask generation method achieving 100 images/sec on a standard GPU for 224×224 images.  
- Trains a masking model to predict the tightest rectangular crop containing the entire salient region for a specified class using a black-box classifier (AlexNet/GoogleNet/ResNet).  
- Model architecture uses multi-scale image feature maps (e.g., ResNet-50); final-scale features go through a feature filter for initial localization and upsampling blocks refine masks.  
- Experimental results show the method outperforms other weakly supervised techniques on ImageNet localization.  
- References and related work appear sufficient (not exhaustively checked).  
- Technical correctness seems satisfactory, though not comprehensively verified.  
- Authors provide insightful design discussions and intuitions.  
- Presentation is well-organized and clear.  
- Reviewer questions the real-time speed claim and applicability to real-world (larger) images.  
- Asks whether the 100 images/sec includes image resizing and requests timing for larger images (e.g., 640×480 from an iPhone 6s).  
- Questions how the method handles small objects and whether 224×224 input is sufficient for small-class instances.
