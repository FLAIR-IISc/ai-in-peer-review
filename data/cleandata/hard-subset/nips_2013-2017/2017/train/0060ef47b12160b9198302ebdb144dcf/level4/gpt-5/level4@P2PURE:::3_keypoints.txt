- Proposes a fast image saliency mask generator that processes ~100 224×224 images/s on a standard GPU.
- Trains a masking model to find the tightest rectangular crop containing the entire salient region for a requested class using black-box classifiers (AlexNet/GoogLeNet/ResNet).
- Architecture uses multi-scale feature maps (e.g., ResNet-50); final-scale features pass through a feature filter for localization and upsampling blocks refine masks.
- Shows improved performance over other weakly supervised methods on the ImageNet localization task.
- Paper includes sufficient references, appears technically correct, provides motivating intuitions, and is well presented.
- Reviewer concerned about the real-time speed claim and applicability to real-world (larger) images.
- Asks whether the 100 images/s throughput includes image resizing and requests runtime for larger images (e.g., 640×480 from an iPhone 6s).
- Questions how the method handles small requested-class objects and whether a 224×224 input is sufficient.
- Requests whether Table 3 includes corresponding metrics for works listed in Table 2 (e.g., Feed [2]).
