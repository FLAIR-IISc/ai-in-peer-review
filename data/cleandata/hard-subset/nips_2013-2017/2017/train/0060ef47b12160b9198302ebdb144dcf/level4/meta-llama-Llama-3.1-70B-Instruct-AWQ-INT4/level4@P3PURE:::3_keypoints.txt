- Proposes a saliency mask generation method that processes ~100 224×224 images/s on a standard GPU.
- Trains a masking model to find the tightest rectangular crop containing the entire salient region of a requested class using a black-box classifier (e.g., AlexNet, GoogleNet, ResNet).
- Architecture leverages multi-scale feature maps (e.g., ResNet-50); final-scale feature filter does initial localization and upsampling blocks refine masks.
- Demonstrates improved performance over other weakly supervised techniques on the ImageNet localization task.
- References and related work appear sufficient; technical correctness and methodology explanations are adequate and well presented.
- Concerns about the real-time speed claim and its applicability to real-world (larger) images remain.
- Unclear how runtime scales to larger images (e.g., 640×480 iPhone 6s images) despite including resizing time.
- Uncertainty about performance when the requested-class object is small—question if 224×224 inputs suffice.
