- Presents a fast saliency mask generator that processes ~100 224×224 images/sec on a standard GPU.  
- Trains a masking model to find the tightest rectangular crop containing the salient region for a requested class using black-box classifiers (AlexNet, GoogleNet, ResNet).  
- Architecture uses multi-scale feature maps (e.g., ResNet-50), a feature filter for initial localization, and upsampling blocks to refine masks.  
- Experimental results show the method outperforms other weakly supervised techniques on ImageNet localization.  
- References and related work appear sufficient (not exhaustively checked); technical correctness seems sound (not exhaustively checked).  
- Paper includes useful intuitions and design discussions and is well presented.  
- Concern: Unclear whether the 100 images/sec speed includes image resizing.  
- Concern: No reported runtime for larger images (e.g., 640×480 from an iPhone 6s).  
- Concern: Unclear performance when the requested-class object is small—are 224×224 inputs sufficient?
