- Proposes a fast image saliency mask generator claimed to process ~100 224×224 images/sec on a standard GPU.
- Trains a masking model to predict the tightest rectangular crop containing the entire salient region for a requested class using a black-box classifier (e.g., AlexNet, GoogleNet, ResNet).
- Architecture leverages multi-scale image feature maps (e.g., ResNet-50); a feature filter localizes the final-scale map and upsampling blocks refine masks.
- Experiments show the method outperforms other weakly supervised approaches on ImageNet localization.
- Paper includes sufficient references, appears technically sound, and is well-written with insightful design intuitions.
- Concern: unclear whether the 100 images/sec timing includes image resizing and how runtime scales to larger images (e.g., 640×480 from an iPhone 6s).
- Concern: performance on small, within-image objects is unclear—question whether 224×224 inputs suffice for small-object localization.
- Concern: Table 3 lacks corresponding metrics for some methods listed in Table 2 (e.g., Feed [2]).
- Minor typos: Line 273 “More over” → “Moreover”; Line 151 “optimise” → “optimizes” (and grammar fix).
