- Proposes an efficient masking model for image saliency that reportedly processes ~100 224x224 images/sec on a standard GPU.
- Trains the model to predict the tightest rectangular bounding box for a specified class using a black-box classifier (e.g., AlexNet/GoogleNet/ResNet).
- Architecture leverages multi-scale feature maps (e.g., ResNet-50) with a feature filter on fine-scale maps and upsampling blocks to refine masks.
- Demonstrates superior performance to other weakly supervised methods on the ImageNet localization task.
- Strong related-work coverage, plausible technical correctness, clear rationale for design choices, and well-structured presentation.
- Question about real-time claim: need clarification if 100 imgs/sec includes resizing and performance on larger, realistic resolutions (e.g., 640x480).
- Unclear handling of small objects: 224x224 inputs may lack detail for tiny targets; needs evaluation across a wider range of object scales.
- Comparative metrics concern: ensure consistent evaluation (e.g., include comparable metrics for methods listed in Table 2 such as Feed [2]).
- Generalizability concern: test performance on more complex, uncurated real-world datasets with noise, clutter, and lighting variation.
