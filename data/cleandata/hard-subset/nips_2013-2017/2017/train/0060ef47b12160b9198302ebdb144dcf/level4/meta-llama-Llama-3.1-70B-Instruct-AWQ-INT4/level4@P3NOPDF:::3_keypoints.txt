- Proposes a fast image saliency mask generator achieving ~100 224x224 images/sec on a standard GPU.  
- Trains a masking model to output the tightest rectangular crop containing the salient region for a requested class using black-box classifiers (AlexNet/GoogLeNet/ResNet).  
- Leverages multi-scale feature maps (e.g., ResNet-50) with a final feature filter for initial localization and upsampling blocks for mask refinement.  
- Demonstrates superior performance over other weakly supervised methods on ImageNet localization.  
- References and related-work coverage appear adequate (not exhaustively checked).  
- Technical correctness seems satisfactory though not deeply verified by the reviewer.  
- Paper includes intuitive design discussions and is well organized and easy to follow.  
- Concern: reported 100 imgs/sec includes resizing, but runtime for larger images (e.g., 640x480 from iPhone 6s) is not provided—scalability unclear.  
- Concern: salient objects evaluated are relatively large; unclear how the method handles small objects or whether 224x224 input suffices.  
- Concern: Table 3 comparisons lack clarity—missing corresponding metrics for some works listed in Table 2 (e.g., Feed [2]).  
- Minor corrections: Line 273 "More over, because our model" → "Moreover, because our model." Line 151 "difference that [3] only optimise the mask" → "difference that [3] only optimises the mask."
