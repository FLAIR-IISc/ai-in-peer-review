- Paper proposes a fast method to generate image saliency masks (tightest rectangular crop for a class), claiming ~100 224x224 images/sec on a standard GPU.
- Model is trained using a pre-trained black-box classifier (e.g., AlexNet/GoogLeNet/ResNet) and uses multi-scale features from a backbone (e.g., ResNet-50) with coarse-to-fine upsampling to refine masks.
- Experiments on ImageNet localization show the method outperforms other weakly-supervised techniques.
- Paper is technically sound, well-referenced, provides good intuition for design choices, and is clearly presented.
- Major concern: unclear whether the 100 imgs/sec includes image resizing, and how runtime scales to larger, real-world resolutions (e.g., 640x480).
- Major concern: experiments mainly use images with large salient objects—how does the method handle small targets, and is 224x224 input sufficient for small-object localization?
- Request: report the same performance metrics in Table 3 for the other methods listed in Table 2 to enable direct comparison (e.g., method [2]).
- Minor typos to fix: Line 151 “optimise” -> “optimises”; Line 273 “More over” -> “Moreover.”
