- Introduces a saliency mask generator that processes 100 224x224 images/sec on a standard GPU.  
- Trains a masking model to output the tightest rectangular crop covering the salient region for a specified class using a black-box classifier (AlexNet/GoogLeNet/ResNet).  
- Architecture uses multi-scale image feature maps (e.g., ResNet-50) with a final-scale feature filter and upsampling blocks to refine masks.  
- Experimental results claim outperforming other weakly supervised methods on ImageNet localization.  
- References and related work appear adequate, though not exhaustively verified.  
- Technical correctness appears sound but was not exhaustively checked.  
- Paper provides useful design intuitions and discussions.  
- Presentation quality is good.  
- Overall compelling technical results, but reviewer has reservations about the real-time speed claims and real-world applicability.  
- Question: Does the 100 images/sec include image resizing, and what is runtime for larger images (e.g., 640x480)?  
- Question: How does the method handle small objects in the image— is 224x224 resolution sufficient?  
- Question: In Table 3, are there corresponding metrics for methods listed in Table 2 (e.g., Feed [2])?
