Summary
The paper proposes a “high-order attention” framework for multimodal learning, instantiated on VQA. The model defines unary, pairwise, and ternary “potentials” over modalities (image V, question Q, answer A), computing correlations via inner products: C2 for pairwise (e.g., Q–V) and C3 for ternary (Q–V–A). Attention distributions over each modality (PV, PQ, PA) are produced by a learned linear combination of the corresponding marginalized potentials followed by a softmax, which the authors describe as a single mean-field iteration. Attended modality vectors are then combined for prediction via compact polynomial pooling (MCB/MCT) using Tensor Sketch. The authors claim state-of-the-art results on VQA (multiple-choice), show qualitative attention maps, and argue that the approach offers a general way to extend attention to any number of modalities.

Relation to prior work
The paper is positioned against co-attention methods (e.g., Lu et al., ECCV’16/NIPS’16), bilinear/multimodal pooling (MCB; Fukui et al., 2016), and bilinear/structured attention variants (e.g., Kim et al., structured attention; MLB/MUTAN). Pairwise attention via similarity matrices is well established; this work adds an explicit ternary correlation involving answers and a simple linear combination of potentials claimed to be mean-field. For prediction, it relies on MCB/MCT, which are standard.

Strengths
- Simple, unified formulation that cleanly enumerates unary/pairwise/ternary interactions, with a plausible extension path to more modalities.
- Practical use of answer-guided attention for multiple-choice VQA is sensible and can improve discrimination.
- Uses compact polynomial pooling to control parameter growth for decision making and gives some comparison between MCT and stacked MCB.
- Qualitative visualizations suggest the pairwise/ternary signals focus attention appropriately. Some implementation details and a code link are provided.

Weaknesses and concerns
Technical soundness/quality
- The “mean-field” interpretation is not rigorous. No explicit energy/CRF is specified, no variational objective is derived, and the resulting update is a learned linear combination followed by softmax. This looks more like a learned fusion layer than a principled inference step. If the probabilistic framing is a key contribution, a precise derivation is needed.
- Computational complexity of ternary potentials is unclear. C3 naively scales as O(nq·nv·na·d). The paper appears feasible only because for multiple-choice VQA, na=18 per question, but the text elsewhere defines na as the number of answers (e.g., 3000). Please clarify what is actually computed, how C3 is materialized or approximated, and give time/memory breakdowns.
- Insufficient ablations. It is crucial to isolate the contribution of each potential: unary only; +pairwise; +ternary. Also ablate the learned weights (α,β,γ), report what the model actually learns, and compare MCT vs MCB with controlled capacity.
- Lack of strong, fair baselines. The 2-modality baseline uses lower-dimensional embeddings (512 vs common 2048), making the comparison unfavorable. A robust set of baselines with matched capacity/parameters is needed (e.g., SA/Show-Ask-Attend-Answer, co-attention, MCB/MLB/MUTAN), plus a conventional “score-each-answer” baseline that uses Q–V attention without answer-guided attention.
- Statistical rigor: no confidence intervals or multiple runs; no category-wise breakdown (Yes/No, Number, Other); no significance tests.
- Reproducibility gaps: it is unclear whether CNN features are fine-tuned; exact preprocessing, question length, and masking details are partially specified; the decision head architectures are not fully described (e.g., widths, non-linearities after pooling).

Clarity
- Several typographical/notation issues (e.g., aV written twice when defining aA; inconsistent convolution notation; missing/extraneous symbols in figures; numerous citation and formatting errors). The description of “convolutions” over C2/C3 with 1×1 kernels along a dimension is confusing—are weights shared across positions? Please formalize these operations.
- The quantitative results are referred to but not clearly presented in the text (no explicit test-dev/test-std numbers; only a single validation number for MCT vs MCB). Claims of state-of-the-art are not verifiable from the provided excerpt.

Originality
- Pairwise attention via affinity matrices and bilinear interactions is well studied. The central novelty appears to be using ternary correlations involving the answer choices to guide attention and wrapping the fusion in a “mean-field” view. Given the reliance on MCB/MCT for decision making and the lack of a precise probabilistic derivation, the incremental novelty is modest.

Significance
- If convincingly demonstrated, answer-guided high-order attention could matter for multiple-choice VQA. However, the current empirical evidence is not strong enough: the comparison set is limited, capacity is not matched, and key ablations are missing. Generalization to open-ended VQA and other datasets (e.g., VQA v2.0, Visual7W) is not reported. The claim of easy extensibility to more modalities is not supported by scaling experiments.

Questions for the authors
- Please provide full quantitative results (test-dev/test-std, with category breakdown) and compare to strong baselines with matched parameter counts and embedding sizes.
- Clarify the exact complexity and implementation of C3. For MC VQA, do you compute over 18 answers per question? How do you handle open-ended settings?
- Provide ablations removing ternary potentials and/or answer-guided attention. How much does each component contribute?
- Justify the “mean-field” claim with a defined energy and variational update, or tone down the probabilistic framing.

Suggestions for improvement
- Strengthen empirical validation: complete tables, multiple seeds, significance, breakdowns, and cross-dataset results.
- Fair baselines with equal capacity; include MLB/MUTAN and co-attention variants.
- Clean up notation and fix typos; precisely define the convolutional reductions over C2/C3.
- Report learned α/β/γ to interpret reliance on each potential; add runtime/memory profiles.

Recommendation: Lean reject
Pros: conceptually simple high-order attention view; plausible benefits of answer-guided attention; some encouraging qualitative results.
Cons: weak novelty relative to established bilinear/co-attention work; insufficiently rigorous probabilistic justification; missing critical ablations and fair, comprehensive comparisons; unclear computational story for ternary potentials; incomplete quantitative evidence. With a stronger, more careful empirical and theoretical treatment, this could be a solid contribution.
