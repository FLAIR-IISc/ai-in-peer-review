- Proposes using RNNs to represent implicit distributions via gamma distributions for count vectors, with shape parameters capturing long-time dependencies.  
- Shape parameters are approximated by neural networks that integrate information from both prior (higher layer) and likelihood (lower layer).  
- Network parameters are updated using expectations of latent variables to reduce sampling variance and improve computational efficiency.  
- Offers an interesting approach to addressing long-term dependency issues in dynamic models.  
- Contains critical typos (e.g., Equation 10) that must be corrected.  
- Treats the gamma scale parameter as fixed and only updates the shape; requires clear justification and analysis of implications for performance.  
- Experimental evaluation is limited to two-layer models; should compare against single-layer and deeper variants to assess the effect of layer depth.  
- Needs analysis of other hyperparameters (window size, number of factors) and their influence on results.  
- Should include detailed graphical illustrations of the full model (e.g., analogous to Figure 1 in Chung et al., NIPS 2015).  
- Must discuss recent deep Poisson factor analysis work (Zhou et al., JMLR 2016; Cong et al., ICML 2017) in the Introduction for broader context.
