- Proposes DDPFA: a deep dynamic Poisson factorization model using a neural network to model sequential count vectors.  
- Uses variational inference to learn network parameters, improving over MCMC/approximate inference methods.  
- Strong technical contribution with thorough analysis of prior dynamic PF limitations in capturing long-range dependencies.  
- Introduces a hierarchical deep structure to learn complex temporal relationships in count data.  
- Extensive experiments on synthetic and real-world datasets.  
- Outperforms baselines (PGDS, LSTM, PFA) on MSE and PMSE.  
- Contains a factual error regarding the Poisson Randomized Gamma distribution that must be corrected.  
- Experimental reporting should include held-out perplexity as a stronger evaluation metric alongside MSE/PMSE.  
- Table 2 formatting needs improvement and best results should be bolded for readability.  
- LSTM comparison is informative but not definitive; need a dynamic PF baseline with a single-level deep architecture to justify multi-level hierarchy.  
- Manuscript has numerous grammatical errors and typos requiring careful proofreading.  
- Originality: novel idea of using a hierarchical deep network for sequential count data.  
- Significance: addresses an important problem with potential impact on applications like text, recommendations, and networks.  
- Arguments for acceptance: novel approach, addresses important problem, extensive experiments showing effectiveness, generally well-written analysis.  
- Arguments against acceptance: factual error on PRG, incomplete experimental metrics (perplexity), insufficient baseline comparisons, and many grammatical mistakes.  
- Reviewer score: 8/10; Confidence: 8/10.
