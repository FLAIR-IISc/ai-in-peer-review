- Proposes Deep Dynamic Poisson Factorization Model (DDPFA) using RNNs to model latent temporal factors for sequential count data.  
- Uses RNNs to parameterize gamma shape parameters by combining higher-level priors and lower-level likelihoods.  
- Aims to capture complex, long-range temporal dependencies beyond simpler Markovian models (e.g., PGDS).  
- Inference updates neural network parameters based on expectations of latent variables for improved efficiency.  
- Contribution is original and significant, combining neural flexibility with interpretable Bayesian matrix factorization.  
- Paper is rushed and needs substantial polishing; critical typos (e.g., Eq. 10) undermine technical confidence.  
- Methodological choice to fix gamma scale parameters and only update shape parameters requires stronger justification and analysis.  
- Experimental validation is preliminary and should include a single-layer baseline and ablation studies (window size, number of factors).  
- Introduction should cite recent related work in deep Poisson factor analysis (e.g., augmentable gamma belief networks).  
- Model complexity hampers understanding; add a detailed graphical illustration of the full architecture.  
- Overall: strong core idea but presentation, methodological justification, and experimental rigor need significant improvement.
