- Summary: Proposes Deep Dynamic Poisson Factorization (DDPFA) extending PFA with a simplified recurrent deep structure and variational inference using neural nets, claiming implicit distributions.  
- Core idea: Potentially useful for modeling temporal count data but undermined by significant technical shortcomings.  
- RNN critique: The proposed "RNN-like" structure is not a true RNN—hidden state h_t depends only on top-layer memory vectors, not same-layer previous time steps.  
- Distribution claim: The paper incorrectly labels distributions as "implicit"—they are explicit Gamma distributions with closed-form conditionals.  
- Evaluation metrics: Uses inappropriate metrics (MSE/PMSE) for Poisson models; predictive log-likelihood should be reported.  
- Latent dimensionality: Experiments use too few latent factors (K=3–10); should test scalability with larger K (≈100).  
- Clarity and organization: Paper omits key equations/parameterizations, contains unclear phrases (e.g., "fanatical data and disaster data"), and has disconnected discussions and figures.  
- Writing quality: Excessive passive voice, grammatical errors, typos, and an unreadable Fig.4 caption; variational updates should be moved to the supplement.  
- Originality: Limited novelty—deep structure is superficial and fails to convincingly distinguish from prior hierarchical/dynamic models.  
- Significance: Problem is important, but weak execution, poor metrics, and experimental limits reduce the paper’s impact.  
- Strengths: Conceptually interesting use of neural nets for temporal count dependencies; better long-term pattern capture than PGDS on some datasets; synthetic experiments show ability to learn complex time patterns.  
- Weaknesses: Technical architecture flaws, incorrect claims, inappropriate evaluation, poor writing, insufficient experiments, and unclear explanations.  
- Acceptance pros: Addresses a meaningful problem with a potentially useful approach if properly fixed.  
- Acceptance cons: Fundamental technical errors, poor presentation, and inadequate experiments argue against acceptance.  
- Recommendations: Fix technical errors (clarify RNN, correct "implicit" claims, use predictive log-likelihood), expand experiments (larger K, depth ablations), improve clarity and writing, move updates to supplement, and address pruning via alternative inference (e.g., MCMC).
