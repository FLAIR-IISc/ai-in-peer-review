- Proposes DDPFA: extends Poisson Factor Analysis by using RNNs to model complex, long-range temporal dependencies in latent count factors.  
- Novelty: using RNNs to parameterize Gamma shape parameters moves beyond first-order Markov assumptions and is a significant conceptual contribution.  
- Inference: hybrid scheme combining coordinate-ascent variational updates for latents with gradient-based training of RNNs using expectations, yielding a stable and efficient "probability autoencoder" approach.  
- Clarity: manuscript is poorly written and difficult to follow; model description and hierarchical information flow need substantial revision.  
- Specific clarity issues: unclear roles of feedforward/backward networks (ffeed/fback) and inconsistent description of the top layer h(N).  
- Suggestion for clarity: include a detailed graphical model illustrating dependencies across layers and time.  
- Technical justification lacking: authors fix Gamma scale parameters (βθ, βh) while only modeling shapes with RNNs, with no discussion of the impact on expressiveness or posterior approximation.  
- Experimental weaknesses: evaluation is limited and does not convincingly support claims about deep architecture benefits.  
- Missing baselines: no comparison to a single-layer RNN-PFA variant to isolate the value of depth.  
- Layer-depth finding needs analysis: authors report added layers do not help (and can hurt), but offer only a brief, unconvincing explanation (e.g., "pruning").  
- Missing ablations: no study of key hyperparameters such as window size c or number of factors K.  
- Related work: insufficient comparison and discussion relative to other deep generative/count models (e.g., Augmentable Gamma Belief Networks, other deep Poisson models).  
- Pros for acceptance: novel model for long-range dynamics, clever integration of RNNs with PFA, and a pragmatic inference strategy.  
- Cons for acceptance: poor presentation, unjustified modeling choices, and weak experimental validation; needs substantial revision before acceptance.
