

This paper proposes a novel computational model of human question asking by formalizing questions as executable programs in a probabilistic context-free grammar (PCFG). The authors model questions as lambda calculus expressions that, when executed on a game state, yield single-word answers (e.g., "True," "3," "Blue"). Key innovations include: (1) treating question generation as program synthesis over a compositional space, (2) defining question quality via a log-linear model incorporating informativeness (Expected Information Gain), complexity (grammar-based log probability), answer type, and relevance, and (3) demonstrating that human-like question preferences balance informativeness with simplicity—a critical insight resolving prior failures to predict human questions using informativeness alone. The model is evaluated on 605 human questions from a Battleship-like game domain [19], where players inferred ship configurations from partial board states. The full model (using all features) significantly outperforms variants (e.g., complexity-agnostic models yield negative correlations), and generates novel human-like questions not present in training.

**Strengths:**  
*Quality:* The work is technically rigorous, with thorough model variants (lesioning key features), cross-validation, and clear statistical validation (bootstrap comparisons). The model directly addresses a gap in active learning literature—most prior work focuses on label queries, not natural-language questions.  
*Clarity:* The paper is exceptionally well-structured, with precise definitions of grammar, features, and evaluation. The grammar design (e.g., using lambda expressions for set operations) and program execution examples (e.g., `(> (size Red) (size Blue))`) make the abstract framework tangible.  
*Originality:* This is the first model to unify linguistic compositionality (e.g., Fodor 1975, Piantadosi et al. 2015) with probabilistic program synthesis for *open-ended* human question asking. It distinguishes from prior work: (a) VQG systems (Mostafazadeh et al. 2016) require image-question pairs; (b) goal-driven dialog (Serban et al. 2016) uses canned questions; (c) active learning (Settles 2010) lacks natural-language expressiveness.  
*Significance:* The findings resolve a key cognitive puzzle—why informativeness alone fails to predict human questions (complexity is equally critical)—advancing self-directed learning theory (Gureckis & Markant 2012). The program-based framework offers a path toward machines that ask creative, context-aware questions (e.g., "Are all ships horizontal?"), moving beyond stereotyped queries.

**Weaknesses:**  
*Scope:* The Battleship domain is highly constrained (small state space, few features), limiting generalizability. While the authors note domain-general constructs (e.g., logical operators, set operations), the approach requires significant knowledge engineering for new domains. This modestly reduces significance for NIPS’ broader audience.  
*Scalability:* The PCFG-based approach becomes computationally intensive for complex domains (e.g., real-world scenes), though the paper acknowledges this as a limitation. Future work should explore scaling to richer environments (e.g., visual question generation with dynamic contexts).  
*Contextualization:* The discussion of related work could better engage with NIPS 2015–2017 work on program induction (e.g., Lake et al. 2015 on concept learning) and formal semantics (Ginzburg 1995, Rojas-Esponda 2013), which address similar question-answering dynamics.

**Pro/Con Acceptance Summary:**  
*Pro:* First model to successfully predict human question *compositionality* and *novelty* via program synthesis; resolves a key gap in active learning; strong empirical validation; advances cognitive science via a formal model of human question asking.  
*Con:* Domain specificity limits immediate applicability to broader ML challenges; computational constraints in complex settings; minimal discussion of how to automate the grammar design for new domains.

**Recommendation:** Accept. The paper makes a significant, well-executed contribution to computational cognitive science and active learning. While the Battleship domain is narrow, the core framework—balancing informativeness and complexity via program synthesis—is broadly applicable and directly addresses a longstanding gap in modeling human-like question asking. The model’s ability to generate novel, contextually relevant questions (e.g., "What is the column of the top left of the tiles that have the color of the bottom right corner of the board?") demonstrates genuine innovation. The authors’ acknowledgment of scope limitations and domain-general components (e.g., set operators, lambda expressions) suggests a clear path for future work. Minor revisions (e.g., clarifying scalability constraints, citing Ginzburg/Rojas-Esponda in related work) would strengthen the paper further, but the current work meets all NIPS quality criteria. This is a high-impact contribution to NIPS’ mission of advancing machine learning through interdisciplinary insights.
