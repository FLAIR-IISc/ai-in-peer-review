- Problem: Robust submodular maximization in a single-pass streaming setting where an adversary may remove any m elements and one must still find a k-element solution from a small summary.  
- Proposed method: STAR-T, a streaming algorithm using a partitioned data structure with exponentially decreasing thresholds and bucket sizes, producing summary size O((k + m log k) log k).  
- Query stage: Use STAR-T-GREEDY — run standard greedy on the remaining summary to obtain the k-element solution.  
- Main theoretical result: First constant-factor approximation guarantee for this streaming robust submodular maximization problem with near-optimal memory.  
- Technical novelty: Clever, non-trivial algorithm design and analysis that handles multiple cases; proofs appear correct.  
- Practical significance: Single-pass operation and sublinear memory with logarithmic dependence on k make it applicable to large-scale problems.  
- Clarity: Well-written, clearly organized, with precise algorithm descriptions and a helpful proof sketch; experiments are clearly presented.  
- Narrow formulation: Two-stage model forbids further data access or resampling, which may be too restrictive for some real applications.  
- Weak motivation: Example applications (e.g., movie recommendations) do not convincingly justify adversarial removals; need more realistic adversarial scenarios like sensor failures or withdrawn items.  
- Conference fit: Heavily theoretical focus makes the paper more suitable for algorithms venues (SODA/ICALP) than NeurIPS unless links to ML applications are strengthened.  
- Experimental gap: Missing comparisons to non-robust streaming baselines (e.g., run SIEVE-STREAMING then remove m items) to quantify the “price of robustness.”  
- Overall assessment: Strong, novel theoretical contribution with sound analysis and good memory bounds, but somewhat weak practical motivation and limited fit for the NeurIPS audience.
