This paper studies the problem of robust submodular maximization in a streaming setting. The goal is to design a single-pass algorithm that constructs a small summary set S from a large ground set V. This summary must be robust, meaning that after an adversary removes any m elements, one can still efficiently find a k-element solution Z from the remaining items in the summary that achieves a constant-factor approximation of the optimal solution on the original, post-removal ground set. The authors propose STAR-T, a streaming algorithm based on a novel partitioned data structure and an exponentially decreasing thresholding rule. The algorithm produces a summary of size O((k + m log k) log k). For the query stage, they propose running a standard greedy algorithm (STAR-T-GREEDY) on the remaining summary to find the solution. The main theoretical result is a constant-factor approximation guarantee for this two-stage procedure. This work builds upon the literature of streaming submodular maximization, particularly SIEVE-STREAMING [8], and robust submodular optimization [9, 11].

**Strengths:**

*   **Technical Quality and Originality:** The paper presents a novel and technically sound solution to a challenging problem. The algorithm's design, which combines a partitioned structure where bucket sizes grow exponentially with exponentially decreasing thresholds, is clever and non-trivial. The analysis appears correct and successfully navigates a complex proof structure by considering different cases (e.g., whether partitions contain many full buckets or not). Providing the first constant-factor approximation for this problem with a near-optimal memory footprint is a significant theoretical achievement.

*   **Significance of the Core Result:** The problem of achieving robustness in streaming settings is important and practical. The algorithm's single-pass nature and sublinear memory complexity make it applicable to large-scale scenarios. The theoretical guarantee is strong, and the memory bound's logarithmic dependence on k is particularly noteworthy.

*   **Clarity:** The paper is generally well-written and clearly organized. The algorithms are described precisely, and the proof sketch in the main body provides valuable intuition for the more detailed analysis in the appendix. The experimental section is also clear and effectively demonstrates the algorithm's empirical performance.

**Weaknesses:**

*   **Problem Formulation and Motivation:** While technically sound, the problem formulation could be seen as somewhat narrow. The two-stage model strictly separates summary construction from the query phase, precluding any further access to the data stream. While this is true for some applications, others might allow for resampling or limited additional passes, which could lead to better solutions. More importantly, the motivation for this specific model feels underdeveloped. The application examples, such as movie recommendations and dominating sets, seem somewhat forced into the adversarial removal framework. In the movie recommendation scenario, "removals" are framed as the user having already seen movies or filtering by genre. These are not truly adversarial removals; they are preference constraints. A more convincing case would involve scenarios where the removals are genuinely adversarial or unexpected, such as sensor failures in a network or withdrawn items in an e-commerce setting. The paper would be stronger if it better justified why this specific robust formulation is the most relevant one for the target machine learning applications.

*   **Relevance to NIPS:** The paper is a strong contribution to the field of algorithms and submodular optimization. However, its direct connection to the core topics of the NIPS conference is questionable. The work is heavily theoretical, and its framing reads more like a paper for an algorithms conference (e.g., SODA, ICALP) than one for a machine learning conference. To improve its relevance for the NIPS audience, the authors could have dedicated more space to exploring the implications and applications of their method within specific machine learning domains, going beyond the brief examples provided. Strengthening the link between the theoretical model and tangible challenges in neural information processing would significantly increase the paper's impact.

*   **Experimental Comparison:** The experiments compare STAR-T against SIEVE-STREAMING running on the data *after* the removal is known. While this is a strong baseline, it doesn't fully capture the landscape. It would be beneficial to also compare against a non-robust streaming baseline. For instance, what happens if one runs a standard streaming algorithm like SIEVE-STREAMING to get a summary S', and then runs greedy on S' after the same adversarial removal? This would help quantify the "price of robustness"—i.e., how much performance is gained by using a dedicated robust algorithm versus a standard one that is subjected to removals.

**Summary of Pros and Cons:**

**Pros:**
*   Provides the first constant-factor approximation algorithm for the streaming robust submodular maximization problem.
*   The algorithm and its analysis are technically deep, novel, and appear correct.
*   The memory complexity of the summary is well-controlled, with a favorable logarithmic dependence on k.
*   The paper is clearly written and structured.

**Cons:**
*   The motivation for the specific two-stage, adversarial problem formulation is not entirely convincing for the applications presented.
*   The paper's focus is heavily theoretical, making its relevance to the primary NIPS audience somewhat borderline.
*   The experimental evaluation could be enhanced by including comparisons to non-robust streaming baselines to better isolate the value of the proposed robust approach.

Overall, this is a high-quality paper presenting a significant theoretical result in submodular optimization. The technical contributions are strong. The main weaknesses are the somewhat tenuous connection to practical ML applications and its potential mismatch with the typical NIPS audience. Despite these concerns, the technical novelty and importance of the problem itself warrant consideration.
