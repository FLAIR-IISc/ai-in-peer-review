- Problem studied: streaming robust submodular maximization—construct T from ground set E so that after deleting any m elements one can still select k elements S with high f(S).
- Motivation/applications: robustness useful when chosen items may later be removed (e.g., movie recommendations), but application examples and motivation are weak.
- Main result: a constant-factor approximation algorithm that outputs T of size (m log k + k) log^2 k.
- Technical quality: results appear correct; the algorithm and analysis are non-trivial and technically challenging.
- Concern: relevance to the conference is questionable.
- Critique: the problem formulation is not general enough.
