The paper studies the problem of finding a set of elements T from a ground set E in a streaming fashion, such that even if any set of m elements is deleted from T, a set of k elements S can still be found to maximize a submodular function f. This problem is an extension of the well-known submodular function maximization problem, with an added robustness requirement.

The maximization of submodular functions has numerous applications, and streaming algorithms for this problem already exist. The proposed robust version is useful in scenarios where some elements may need to be removed after an initial solution is found. For instance, in a recommendation system, a set of movies may be computed for a user, only to find that the user is not interested in some of them.

The paper presents a constant factor approximation algorithm for this problem, which returns a set T of size (m log k + k) log^2 k. To the best of my understanding, the results presented in the paper are correct. The solution is non-trivial, and both the algorithm and its analysis are technically challenging.

However, my main concern is the relevance of this paper to the conference. While the problem is interesting, the applications and motivations presented are not compelling, and the problem itself may not be general enough to warrant significant attention. The connection to real-world applications is not strongly established, which may limit the paper's impact.
