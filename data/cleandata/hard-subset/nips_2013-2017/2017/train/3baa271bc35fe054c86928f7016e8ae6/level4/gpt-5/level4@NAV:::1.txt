Review- 
Problem studied: The paper investigates the following problem. From a ground set E, the goal is to construct, in a streaming setting, a set T such that, after deleting any set of m elements from T, one can still extract a set S of k elements for which f(S) is maximized, where f is a submodular function.

Applications: Submodular maximization has many applications, and streaming algorithms for it are well-studied. A robust variant is useful when a computed solution may later require removing some elements (e.g., recommending a set of movies and subsequently learning the user is uninterested in some of them).

Results: The paper presents a constant-factor approximation for this problem, returning a set T of size (mlogk+k)log^2k.

Comments: To the best of my understanding, the results are correct. The solution is nontrivial, and the algorithm and its analysis are technically challenging. Although the problem is interesting, my primary concern is the paper’s relevance to the conference: the applications and motivation feel weak, and the problem does not appear sufficiently general.
