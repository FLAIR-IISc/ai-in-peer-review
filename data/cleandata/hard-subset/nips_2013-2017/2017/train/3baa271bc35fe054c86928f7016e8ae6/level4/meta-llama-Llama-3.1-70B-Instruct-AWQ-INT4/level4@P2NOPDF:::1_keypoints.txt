- Problem: find T in a streaming setting so that after any m deletions from T, one can still extract a k-element set S maximizing a submodular function f (robust streaming submodular maximization).
- This is an extension of standard submodular maximization with an added robustness-to-deletions requirement.
- Prior streaming algorithms exist for submodular maximization; this work targets the deletion-robust variant.
- Practical motivation claimed: useful when selected elements may later be removed (example: recommendation lists where a user rejects some items).
- Main contribution: a constant-factor approximation algorithm that returns T of size (m log k + k) log^2 k.
- Results appear correct; the algorithm and its analysis are non-trivial and technically challenging.
- Primary concern: relevance to the conference is questionable.
- Weakness: applications and motivations are not compelling and the problem may lack sufficient generality, limiting impact.
