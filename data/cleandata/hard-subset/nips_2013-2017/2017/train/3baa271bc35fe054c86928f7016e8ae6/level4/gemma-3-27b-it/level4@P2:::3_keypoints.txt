- Tackles robust submodular maximization in a streaming setting, combining streaming efficiency with post-removal robustness.  
- Claims novelty as the first algorithm addressing streaming and robustness together.  
- Proposes STAR-T with a constant-factor approximation of 0.149.  
- Output set size guarantee: O((m log k + k) log^2 k).  
- Uses a partitioning and thresholding framework with a novel bucket-filling strategy.  
- Empirical evaluation on dominating set (influence) and personalized movie recommendation tasks.  
- Shows competitive or superior performance vs. baselines that know removals in advance.  
- Significant because it provides provable guarantees for a practically relevant, large-scale problem.  
- Missing lower bounds, so optimality of the approximation is unclear.  
- Proof sketch and analysis lack sufficient detail and do not clearly highlight novel analytical insights.  
- Influence maximization experiment uses neighborhood size objective, which is a simplistic proxy for real propagation models.  
- Recommends acceptance contingent on improving analytical clarity and strengthening the influence maximization evaluation.
