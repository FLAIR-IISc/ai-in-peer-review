**Problem Studied:** This paper addresses the problem of finding a robust set of elements in a streaming context. The objective is to construct a set `T` from a ground set `E` such that after an adversary deletes any `m` elements from `T`, one can still find a `k`-element subset `S` that maximizes a given submodular function `f`.

**Applications:** The authors situate their work within the well-established field of streaming submodular maximization. The motivation for this robust variant is drawn from scenarios where elements of an initially computed solution might be invalidated post-computation. An example provided is a recommendation system where a user may discard a subset of suggested items, requiring a high-quality solution to be recoverable from the remainder.

**Results:** The primary contribution is a constant-factor approximation algorithm for the problem. The algorithm outputs a solution set `T` of size `O((m log k + k) log^2 k)`.

**Comments:** The technical contributions of this paper appear to be sound. The proposed solution is non-trivial, and both the algorithm and its analysis are technically demanding. My main reservation, however, relates to the paper's fit for this conference. The motivation for this particular problem formulation seems weak, and the supporting applications are not sufficiently convincing. As a result, the problem, while technically interesting, may be too narrowly defined to capture the broad interest of the audience.
