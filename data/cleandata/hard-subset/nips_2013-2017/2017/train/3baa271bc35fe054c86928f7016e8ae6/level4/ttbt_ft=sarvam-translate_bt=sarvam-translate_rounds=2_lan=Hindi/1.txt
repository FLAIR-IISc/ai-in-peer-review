**The Problem Studied:** This paper examines the following question. Its goal is to find a set T of elements from a base set E in a streaming manner, such that if we remove a set of m elements from T, we can still find a set S of k elements, where f(S) is maximal, where f is a sub-modular function.

**Application:** Sub-modular functions have many applications, and we know streaming algorithms for this problem. A robust version is useful because if we find a solution and then need to remove some elements (say you create a set of movies for a user and then find that the user is not interested in some of them).

**Result:** This paper gives a constant-factor approximation for this problem, returning a set T of size (mlogk+k)log^2k.

**Note:** In my understanding, the results of the paper are correct. The solution is non-trivial, and the algorithm and its analysis are technically challenging. While the problem is interesting, my main concern is the relevance of this paper for the conference. The applications and their motivation are quite weak, and the problem is not sufficiently general.
