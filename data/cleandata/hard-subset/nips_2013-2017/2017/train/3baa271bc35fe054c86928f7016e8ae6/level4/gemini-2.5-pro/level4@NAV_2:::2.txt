This paper addresses the problem of maximizing a monotone submodular function with a cardinality constraint in a setting that combines two significant challenges: the data arrives in a stream, and the solution must be robust to the adversarial removal of `m` elements. The authors propose a two-stage procedure. In the first stage, a single-pass streaming algorithm, STAR-T, constructs a robust summary of the data. This algorithm is notable for its novel partitioned structure, where elements are assigned to buckets based on an exponentially decreasing thresholding rule. In the second stage, after an arbitrary set of `m` elements is removed from the summary, a simple greedy algorithm (STAR-T-GREEDY) is run on the remaining summary elements to produce the final solution. The paper provides a theoretical analysis showing that this procedure achieves a constant-factor approximation guarantee. This work extends the thresholding ideas from the streaming algorithm SIEVE-STREAMING [8] to the robust setting, while also adapting a partitioned structure similar to the one used in the non-streaming robust algorithm PRO [11].

**Strengths:**

*   **Originality and Significance:** The paper tackles a novel and practical problem at the intersection of streaming algorithms and robust submodular optimization. Providing a constant-factor approximation for robust submodular maximization in the streaming setting is a significant contribution. This work, along with the concurrent paper by Mirzasoleiman et al. [18], pioneers this important area. The proposed algorithm's logarithmic dependence on `k` for memory complexity offers an advantage over the linear dependence in [18].
*   **Technical Quality:** The proposed STAR-T algorithm, which combines a multi-level partitioning scheme with an exponentially decreasing thresholding rule, is a clever and well-motivated design. The theoretical analysis, which culminates in a constant-factor approximation guarantee (Theorem 4.1), appears sound. The paper also provides a practical solution for handling the unknown optimal value needed to set the algorithm's threshold parameter `τ` (Theorem 4.5), which is crucial for real-world application.
*   **Clarity:** The paper is generally well-written and organized. The algorithmic approach is presented clearly, with an intuitive explanation of the partitioned summary structure. The proof sketch provides helpful insights into the main theoretical arguments.

**Weaknesses:**

*   **Experimental Evaluation:** The empirical results demonstrate that STAR-T-GREEDY performs comparably to, and sometimes slightly better than, the SIEVE-STREAMING baseline. While this is a positive result, it is perhaps not as compelling as it could be, given that the baseline is granted the powerful advantage of knowing the removed set `E` in advance. A key motivation for robust optimization is to guard against a worst-case scenario, but the experiments do not fully showcase a setting where STAR-T's robustness provides a clear and substantial advantage over a non-robust method.
*   **Clarity on Objective Function:** In Section 5.1, the objective function for the dominating set experiment is defined as f(Z)=|N(Z) ∪ Z|. While this is a standard objective for dominating set (which is monotone and submodular), the text frames it as an "influence maximization" problem and cites the seminal work by Kempe et al. (2003). The models in that paper (e.g., Independent Cascade, Linear Threshold) are stochastic and distinct from the deterministic dominating set objective. It would strengthen the paper to either clarify why dominating set is used as a proxy for the influence maximization problem described in the cited work, or to perform experiments using one of the standard stochastic influence models. This would provide a more direct evaluation of the algorithm's performance on the canonical influence maximization task.

In summary, this is a strong paper that introduces a novel algorithm with theoretical guarantees for an important and challenging problem. The technical contributions are solid, but the experimental section could be strengthened by clarifying the choice of objective function and by more forcefully demonstrating the practical benefits of the proposed robust approach over strong, non-robust baselines.
