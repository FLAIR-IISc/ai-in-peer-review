

Problem Studied: The paper addresses the problem of identifying a set of elements \(T\) from a ground set \(E\) in a streaming setting. The objective is to ensure that, after deleting any subset of \(m\) elements from \(T\), the remaining elements still contain a subset \(S\) of size \(k\) that maximizes the value of a submodular function \(f\).  

Applications: Submodular maximization has well-established applications, and existing streaming algorithms exist for the standard problem. A robust variant is motivated by scenarios where a computed solution may require post-hoc removal of elements—such as when recommending a set of movies to a user, only to later discover the user has no interest in certain items.  

Results: The paper achieves a constant-factor approximation for this problem while outputting a set \(T\) of size \((m \log k + k) \log^2 k\).  

Comments: The technical results appear correct to the best of my understanding. The algorithm and its analysis are non-trivial and technically challenging. While the problem is inherently interesting, the primary concern lies in the paper's relevance to this conference. The applications and their motivation are weak, and the problem scope is insufficiently general.
