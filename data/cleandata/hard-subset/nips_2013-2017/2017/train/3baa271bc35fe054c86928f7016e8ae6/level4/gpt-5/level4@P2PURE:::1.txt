Problem studied: The paper addresses the following task. In a streaming setting, select a set T from a ground set E such that, after deleting any set of m elements from T, one can still extract a set S of k elements that maximizes a submodular function f (over the remaining elements).

Applications: Maximizing submodular functions has many applications, and there are known streaming algorithms for the standard (non-robust) setting. A robust variant is useful when, after producing a solution, some elements may later need to be removed (e.g., recommending a set of movies to a user and then finding the user is not interested in some of them).

Results: The paper achieves a constant-factor approximation to this problem while returning a set T of size (m log k + k) log^2 k.

Comments: To the best of my understanding, the results are correct. The solution is nontrivial, and both the algorithm and its analysis are technically challenging. While the problem is interesting, my main concern is its relevance to the conference: the applications and their motivation are quite weak, and the problem does not appear sufficiently general.
