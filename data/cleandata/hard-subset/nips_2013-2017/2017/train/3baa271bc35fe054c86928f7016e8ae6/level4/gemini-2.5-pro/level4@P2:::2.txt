This paper addresses the problem of maximizing a monotone submodular function subject to a cardinality constraint `k` in a streaming setting. The work introduces an additional, practical challenge: the solution must be robust to the subsequent removal of up to `m` elements. The authors propose a two-stage procedure to tackle this problem. In the first stage, a single-pass streaming algorithm, STAR-T, processes the data to build a robust summary `S`. STAR-T's novelty lies in its use of a partitioned data structure, where elements are assigned to buckets based on an exponentially decreasing thresholding rule. In the second (query) stage, after an arbitrary set `E` of `m` elements is removed, a simple greedy algorithm (STAR-T-GREEDY) is run on the remaining elements of the summary (`S \ E`) to find a solution of size `k`.

The paper's main theoretical contribution is a proof that this two-stage procedure achieves a constant-factor approximation guarantee. This result builds on the ideas of thresholding-based streaming algorithms like SIEVE-STREAMING and partitioned structures used in non-streaming robust maximization. To my knowledge, this is the first work to provide a streaming algorithm with provable guarantees for the robust submodular maximization problem. The authors also demonstrate the algorithm's performance on two data summarization tasks: a dominating set problem and personalized movie recommendation.

**Strengths:**

1.  **Originality and Significance:** The paper tackles a well-motivated and practically significant problem. The combination of streaming data processing and the need for robust solutions is highly relevant in many large-scale applications where data cannot be stored entirely and solutions must be resilient to failures or adversarial attacks. Providing the first streaming algorithm for this problem is a notable and timely contribution.

2.  **Technical Quality:** The proposed STAR-T algorithm is an elegant and novel combination of a partitioned data structure and an exponentially decreasing thresholding scheme. The theoretical analysis appears sound and provides a constant-factor approximation guarantee (Theorem 4.1), which is a strong result for this challenging setting.

3.  **Practical Considerations:** The authors thoughtfully address the practical issue of the threshold parameter `τ` depending on the unknown optimal value. They show how to overcome this by running multiple instances of the algorithm in parallel, a standard but important technique for making the algorithm usable in practice. Furthermore, the experimental results suggest that the memory parameter `w` can be set to 1, simplifying hyperparameter tuning.

**Weaknesses and Questions for Clarification:**

1.  **Experimental Evaluation:** The empirical results demonstrate that the algorithm is effective, but they could be more compelling. In several plots (e.g., Figure 2b, 2d), the proposed method performs comparably to or is slightly outperformed by the SIEVE-STREAMING baseline. While this baseline is granted the significant advantage of knowing the removed set `E` in advance, it would be beneficial to see experiments in settings where the robust nature of STAR-T provides a more decisive advantage.

2.  **Clarity on the Influence Maximization Objective:** The objective function used in the dominating set experiment is defined as `f(Z) = |N(Z) U Z|`, representing the size of the union of a set `Z` and its neighborhood. While this is a valid submodular function, the paper frames this experiment in the context of influence maximization and cites the seminal work of Kempe et al. (2003). The standard influence maximization models (e.g., Independent Cascade, Linear Threshold) are computationally different and represent the expected number of influenced nodes, not just the immediate neighborhood. It would be helpful if the authors could clarify why this simpler objective was chosen and comment on how they expect their algorithm to perform on standard influence spread models.

3.  **Adversarial Model in Experiments:** The problem is defined for an adversary that removes any `m` elements from the ground set `V`. However, in the experiments, the removals seem to be restricted to the summary `S` generated by STAR-T. For instance, the greedy adversarial removal in Section 5.1 iteratively removes elements from `S` to maximally decrease the objective. While this is a strong test of the summary's robustness, it would be interesting to also see performance against an adversary that can remove any `m` elements from the entire dataset `V`, some of which may not have been selected for the summary `S`.

In summary, this paper presents a solid and valuable contribution by introducing the first provably-correct streaming algorithm for robust submodular maximization. The algorithmic design is novel and the theoretical analysis is rigorous. The main areas for improvement lie in strengthening the experimental evaluation and clarifying the choice of objective function in the influence maximization experiment.
