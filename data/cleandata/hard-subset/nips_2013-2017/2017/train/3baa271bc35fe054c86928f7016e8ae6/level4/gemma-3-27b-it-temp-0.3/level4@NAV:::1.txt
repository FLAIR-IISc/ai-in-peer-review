The paper investigates the problem of robust submodular maximization in a streaming setting. Specifically, the authors aim to identify a set *T* from a ground set *E* such that, even after removing any *m* elements from *T*, a near-optimal set *S* of *k* elements can still be found to maximize a submodular function *f*.

The motivation stems from the practical utility of submodular maximization and the need for solutions that remain effective even with element failures or removals – a scenario relevant in applications like recommendation systems where user preferences can change. Existing streaming algorithms address standard submodular maximization, but this work tackles the more realistic robust variant.

The authors present a constant-factor approximation algorithm for this problem, achieving a solution set *T* of size *O(mlogk + k)log²k*. 

My assessment is that the technical content is sound; the results appear correct, and the algorithm and its analysis demonstrate significant technical depth. However, my primary reservation concerns the paper’s fit for this conference. While the problem itself is interesting, the provided applications feel somewhat tenuous and the problem’s generality appears limited, raising questions about its broad relevance to the conference audience.
