Problem studied: This paper investigates the problem of maintaining a set T of elements chosen from a ground set E in a streaming setting. The objective is to identify a set T such that, even after removing any m elements from T, a subset S of k elements can still be found within the remaining elements that maximizes a submodular function f.

Applications: Maximizing submodular functions arises in numerous applications, and streaming algorithms for this task are well-established. This work introduces a robust version of the problem, which is particularly relevant when solutions require potential element removal – for example, recommending items to a user and subsequently needing to adjust the recommendations based on user disinterest. 

Results: The authors present a constant-factor approximation algorithm for this problem, returning a set T of size (mlogk+k)log^2k.

Comments: The results appear correct and the solution is non-trivial, with a technically challenging algorithm and analysis. However, my primary concern relates to the paper’s relevance to the conference. The presented applications and their motivating rationale are somewhat weak, and the problem itself lacks broad generality.
