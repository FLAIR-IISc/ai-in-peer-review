Problem studied: The paper addresses robust submodular maximization in the streaming setting. The goal is to construct a summary set T from a ground set E such that, after the deletion of any m elements from T, one can still extract a set S of k elements that (approximately) maximizes a submodular objective f.

Applications: Maximizing submodular functions has many applications, and streaming algorithms are known for the non-robust setting. A robust formulation is useful when elements selected by an algorithm may later need to be removed (e.g., recommending a set of movies to a user and subsequently excluding items the user is not interested in).

Results: The paper presents a constant-factor approximation for this robust streaming problem and returns a set T of size (m log k + k) log^2 k.

Comments: To the best of my understanding, the results are correct. The solution is non-trivial, and both the algorithm and its analysis are technically challenging. While the problem is interesting, my main concern is the paper’s relevance to the conference: the applications and motivation are relatively weak, and the problem formulation does not appear sufficiently general to have broad impact.
