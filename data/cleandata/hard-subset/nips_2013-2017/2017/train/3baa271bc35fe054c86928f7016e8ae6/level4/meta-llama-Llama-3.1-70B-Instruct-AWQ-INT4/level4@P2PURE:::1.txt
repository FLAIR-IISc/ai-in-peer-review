The paper studies the problem of finding a set of elements T from a ground set E in a streaming fashion, such that even if any set of m elements is deleted from T, a subset S of k elements can still be found that maximizes a submodular function f. 

Maximizing submodular functions has numerous applications, and streaming algorithms for this problem already exist. However, the robust version of this problem is useful in scenarios where a solution is found, but some elements need to be removed later (e.g., recommending movies to a user, only to discover they are not interested in some of them).

The paper presents a constant factor approximation algorithm for this problem, returning a set T of size (m log k + k) log^2 k.

To the best of my understanding, the paper's results are correct. The proposed solution is non-trivial, and both the algorithm and its analysis are technically challenging. However, my main concern is the paper's relevance to the conference. While the problem is interesting, the applications and their motivation seem weak, and the problem is not general enough.
