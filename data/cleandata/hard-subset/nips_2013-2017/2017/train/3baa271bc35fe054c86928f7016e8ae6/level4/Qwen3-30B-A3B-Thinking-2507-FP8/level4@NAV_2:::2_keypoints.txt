- Introduces a two-stage framework (STAR-T streaming + STAR-T-GREEDY) for robust submodular maximization under cardinality constraints against removal of up to m elements.  
- Theorem 4.1 proves a constant-factor (0.149) approximation with summary size O((k + m log k) log² k).  
- Novelty: first streaming algorithm for robust submodular maximization using partitioned thresholding and exponential decay.  
- Builds on prior work (PRO, SIEVE-STREAMING) and improves memory efficiency compared to concurrent work [18] (O(m log² k) vs O(mk log k/ε)).  
- Theoretical analysis (Lemmas 4.2–4.4) rigorously supports the constant-factor guarantee and summary size bounds; extension to unknown τ via parallel instances is solid.  
- Practical significance: applicable to influence maximization, recommendations, and other large-scale summarization tasks; supports one summary for multiple removals.  
- Clarity: algorithms and partition structure are well-defined and experiments support claims.  
- Experiments: STAR-T-GREEDY matches SIEVE-STREAMING in some plots but underperforms in greedy removals (Figures 2b,d); reasons for SIEVE-STREAMING's edge need clarification.  
- Experimental validity concern: the influence objective used (f(Z)=|N(Z)∪Z|) deviates from standard spread models, requiring justification.  
- Hyperparameter sensitivity: empirical w=1 works, but theory requires w≥⌈4m log k / k⌉; paper should discuss when w=1 suffices versus tight cases.  
- Pro: strong theoretical contribution, first streaming robust solution, practical implementation with few hyperparameters.  
- Con: experimental comparisons lack statistical rigor in key cases and the influence objective inconsistency weakens empirical validation.
