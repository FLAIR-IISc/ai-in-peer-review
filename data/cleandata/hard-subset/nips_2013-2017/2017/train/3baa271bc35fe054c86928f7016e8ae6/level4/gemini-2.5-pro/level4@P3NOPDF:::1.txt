### **Summary**

This paper addresses the problem of robust submodular maximization in the streaming setting. The goal is to select a summary set `T` from a ground set `E` arriving in a stream. This set `T` must be robust to adversarial deletions: for any `m` elements removed from `T`, the remaining set must contain a `k`-element subset `S` whose value `f(S)` is a good approximation of the optimal offline solution, where `f` is a monotone submodular function. The authors present a streaming algorithm that achieves a constant-factor approximation for this problem, returning a summary set `T` of size O((m log k + k) log² k).

### **Strengths**

*   **Technical Depth:** The paper tackles a challenging and theoretically interesting problem that combines concepts from streaming algorithms, submodular optimization, and robust optimization. The proposed algorithm and its analysis are non-trivial and technically sophisticated. The proofs appear to be sound, and the technical arguments are rigorous and well-presented.
*   **Novelty:** To the best of my knowledge, this is the first work to study this specific formulation of robust submodular maximization in a single-pass streaming context, providing a constant-factor approximation. The result is a solid contribution to the theoretical literature on submodular optimization.

### **Weaknesses**

My main concerns are related to the motivation for the problem and its relevance to the NIPS community.

*   **Motivation and Applications:** While the paper gestures toward applications like robust recommender systems (e.g., finding a set of movies where, even if the user dislikes `m` of them, a high-quality set of `k` remains), this motivation feels underdeveloped. The adversarial deletion model seems particularly ill-suited for this scenario. It is unclear why a user's preferences would manifest as a worst-case removal of `m` items. A more natural model for such applications might involve stochastic feedback, online learning, or a different objective function altogether. The paper would be significantly strengthened by providing more compelling, concrete, real-world scenarios where this specific adversarial robustness model is critical and well-justified.
*   **Relevance and Problem Scope:** The primary concern is the paper's fit for the NIPS conference. While technically sound, the work reads more like a theoretical computer science paper (e.g., for SODA/FOCS) than a machine learning paper. The connection to core machine learning tasks is not made sufficiently clear. Without stronger ties to practical machine learning problems where this precise formulation is necessary, the problem risks being perceived as a theoretical construction with limited immediate applicability for the NIPS audience. The paper could be improved by discussing how the proposed techniques might apply to other models of uncertainty or failure more common in machine learning (e.g., stochastic noise, corrupted data) or by drawing clearer connections to existing work in robust machine learning.

### **Conclusion**

In summary, this paper presents a technically impressive and novel result in the area of streaming submodular optimization. The algorithm and its analysis are of high quality. However, the work is hampered by weak motivation and a questionable fit for the NIPS conference. The practical relevance of the specific adversarial model is not convincingly established, making the contribution feel more theoretical than applied. While I appreciate the technical depth, I believe the paper would be better suited for a more theory-focused venue unless the authors can substantially revise it to better motivate the problem and demonstrate its significance to the machine learning community.
