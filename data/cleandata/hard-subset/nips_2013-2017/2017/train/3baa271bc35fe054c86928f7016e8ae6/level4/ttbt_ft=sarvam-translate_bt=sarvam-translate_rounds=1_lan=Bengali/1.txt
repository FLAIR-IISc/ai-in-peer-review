The problem addressed in this paper explores the following question: Given a set of elements T from a set of ground sets E, if we remove m elements from T, can we still find a set of k elements of S such that the value of f(S) is maximized, where f is a submodular function?

Application: Submodular functions have multiple applications to maximize their values, and we know streaming algorithms for this problem. A powerful version is useful because if we find a solution and then need to remove some elements (say you create a list of movies for a user and later see that the user is not interested in some of them).

Result: This paper provides a constant-factor approximation of this problem and provides a set T of size (mlogk+k)log^2k.

Comments: As far as I understand, the results of the paper are correct. The solution is non-trivial, and the algorithm and its analysis are technically challenging. Although the problem is interesting, my main concern is how relevant this paper is for the conference. Its application and purpose are quite weak, and the problem is not sufficiently general.
