- Problem: Finds a robust set T in a streaming setting so that after any m deletions one can still extract a k-element subset S maximizing a submodular function f.  
- Setting: Builds on streaming submodular maximization with robustness to adversarial removals of selected elements.  
- Motivation/example: Motivated by scenarios like recommendation systems where some suggested items may be discarded post-computation.  
- Main result: Presents a constant-factor approximation algorithm for the robust streaming problem.  
- Output size: The algorithm returns T of size O((m log k + k) log^2 k).  
- Technical quality: The algorithm and analysis are non-trivial and appear technically sound.  
- Concern: The problem motivation is weak and supporting applications are not convincing.  
- Fit: The formulation may be too narrow to attract broad interest from the conference audience.
