Problem studied: The paper investigates a robust streaming variant of submodular maximization. Given a ground set E presented as a stream, the goal is to construct a summary set T such that, after the deletion of any set of m elements from T, we can still select a set S of k elements that maximizes a submodular function f.

Applications: Submodular maximization has many applications, and streaming algorithms for the standard (non-robust) setting are known. A robust version is useful when some chosen items might later need to be removed—for example, recommending a set of movies to a user and subsequently finding that the user is not interested in some of them.

Results: The paper provides a constant-factor approximation for this robust streaming problem while returning a summary T of size (m log k + k) log^2 k.

Comments: To the best of my understanding, the results are correct. The solution is nontrivial, and the algorithm and its analysis are technically challenging. While the problem is interesting, my main concern is its relevance to the conference. The applications and their motivation are quite weak, and the problem does not seem general enough.
