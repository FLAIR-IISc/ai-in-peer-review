The paper studies the problem of finding a set of elements T from a ground set E in a streaming fashion, such that if any set of m elements is deleted from T, a set of k elements S can still be found to maximize a submodular function f. This problem is a variant of the traditional submodular function maximization problem, which has numerous applications.

The authors motivate the study of this robust version by noting that in practice, elements may need to be removed from the solution set after it has been computed. For example, if a set of movies is recommended to a user, some of these movies may not be of interest to the user and thus need to be removed. While this motivation is reasonable, I found the applications and their explanation to be somewhat weak and not thoroughly explored in the paper.

The paper's main contribution is a constant factor approximation algorithm for this problem, which returns a set T of size (m log k + k) log^2 k. To the best of my understanding, the results presented in the paper are correct, and the solution is indeed non-trivial. The algorithm and its analysis are technically challenging, demonstrating a clear mastery of the relevant technical tools.

However, my main concern is the relevance of this paper to the conference. While the problem studied is interesting and the solution is technically sound, I am not convinced that the paper addresses a problem of sufficient generality or importance to warrant presentation at NIPS. The applications, while plausible, are not thoroughly explored, and the paper does not provide a compelling case for why this particular variant of submodular function maximization is worthy of study.
