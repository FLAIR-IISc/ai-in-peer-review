Problem studied: This paper investigates the problem of robust submodular maximization in a streaming setting. Specifically, the goal is to identify a set of elements *T* from a ground set *E* such that, even after deleting any *m* elements from *T*, a subset *S* of *k* elements can still be found within the remaining set to maximize a submodular function *f*.

Applications: Maximizing submodular functions arises in numerous applications, and streaming algorithms for this problem are well-established. This work addresses a robust variant that is valuable when solutions may require modification – for example, selecting a set of movies for a user and subsequently needing to remove options the user dislikes. 

Results: The authors present a constant-factor approximation algorithm for this robust submodular maximization problem, returning a set *T* of size *O( (m log k + k) log<sup>2</sup> k )*. 

Comments: The technical results appear correct, and the proposed solution is non-trivial. The algorithm and its analysis demonstrate a significant technical effort. However, my primary concern is the relevance of this work to NIPS. While the problem itself is interesting, the provided applications and their motivating rationale feel weak, and the problem setting lacks broad generality.
