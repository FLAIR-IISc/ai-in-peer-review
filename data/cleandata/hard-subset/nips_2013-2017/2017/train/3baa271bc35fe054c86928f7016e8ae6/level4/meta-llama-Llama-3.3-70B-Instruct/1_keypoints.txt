- Problem: Select T from ground set E in a streaming setting such that after removal of any m elements from T, one can still retrieve a subset S of size k that approximately maximizes a submodular function f.  
- Motivation: Robust variant useful when initial solutions need later adjustments (e.g., changing user preferences); grounded in well-studied submodular maximization literature.  
- Contribution: A constant-factor approximation algorithm for the robust streaming problem.  
- Result: Algorithm returns T of size (m log k + k) log^2 k.  
- Technical quality: Results appear technically sound; the algorithm and analysis are sophisticated and nontrivial.  
- Main concern: Relevance to the conference is questionable — motivations and applications are weak and the problem may not be broadly applicable.
