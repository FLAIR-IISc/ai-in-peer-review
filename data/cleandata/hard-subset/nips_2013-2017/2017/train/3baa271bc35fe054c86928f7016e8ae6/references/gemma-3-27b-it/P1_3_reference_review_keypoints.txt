- Proposes STAR-T: a two-stage streaming algorithm using partitioned thresholding to build a robust summary and then greedy selection after removals.  
- Core innovation is the partitioned thresholding scheme enabling a provable constant-factor approximation in streaming settings.  
- Achieves a constant-factor theoretical guarantee (approximation 0.149) for robust submodular maximization.  
- Improves memory complexity compared to concurrent work, highlighted in comparisons with prior streaming algorithms (e.g., SIEVE-STREAMING).  
- Situates the work well within prior literature on submodular, streaming, and robust maximization and differentiates contributions clearly.  
- Empirically validated on dominating set and personalized movie recommendation tasks, showing competitive performance and robustness to removals.  
- Interactive experiments convincingly demonstrate robustness across different element-removal scenarios.  
- Paper is generally well-written, organized, and includes clear algorithm descriptions.  
- Addresses limitations such as dependence on estimating f(OPT(k, V\E)) and provides a practical solution in Section 4.1.  
- Main weakness: the proved constant-factor (0.149) is relatively small and may limit practical usefulness.  
- Performance appears sensitive to the memory parameter w; theory provides a bound but experiments show w=1 often suffices.  
- Experimental scope could be broader—additional datasets and applications would strengthen generalizability.  
- Some proof details are deferred to supplementary material; a concise main-paper proof summary would aid readers.  
- Recommendation: accept with minor revisions to expand discussion on parameter selection and summarize key proof ideas.  
- Reviewer confidence: 8/10.
