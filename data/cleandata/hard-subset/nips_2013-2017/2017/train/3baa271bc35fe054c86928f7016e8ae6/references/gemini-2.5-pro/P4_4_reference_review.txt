This paper tackles the problem of robust submodular maximization in a streaming setting. The goal is to design a single-pass algorithm that uses sublinear memory to compute a summary set `S` from a large dataset `V`. This summary must be robust, meaning that even after an arbitrary set `E` of up to `m` elements is removed from `S`, one can still extract a high-quality solution of size `k` without re-accessing the original data stream `V`.

The authors propose a two-stage procedure. In the first stage, a novel streaming algorithm called STAR-T (Streaming Robust submodular algorithm with Partitioned Thresholding) constructs the summary `S`. The core idea of STAR-T is to maintain a set of partitions, where each partition `i` is associated with an exponentially decreasing threshold `τ/2^i`. Each partition contains multiple buckets, and an incoming stream element is placed into the first available bucket where its marginal gain exceeds the corresponding partition's threshold. This structure is inspired by the non-streaming algorithm PRO [11], but the mechanism of filling buckets via thresholds is adapted from the streaming algorithm SIEVE-STREAMING [8]. In the second stage, after `m` elements are removed from `S`, a simple greedy algorithm (STAR-T-GREEDY) is run on the remaining elements of the summary to find the final `k`-sized solution.

The main theoretical contribution is a proof that this procedure achieves a constant-factor approximation guarantee. Specifically, the algorithm constructs a summary of size `O((k + m log k) log k)` and, for any removal of `m` elements, the final solution `Z` satisfies `f(Z) ≥ c · f(OPT(k, V \ E))` for a constant `c ≈ 0.149`. The paper also addresses the practical issue of the threshold `τ` depending on the unknown optimal value by using a standard technique of running parallel instances with guessed values, at the cost of a logarithmic increase in memory. The approach is validated empirically on dominating set and movie recommendation tasks, where it performs comparably to or better than the non-robust SIEVE-STREAMING algorithm, even when the latter is given the advantage of knowing the removed elements in advance.

### Strengths

**Quality:** The paper is technically sound. The theoretical claims are well-supported by a detailed proof sketch that insightfully breaks the analysis into two main cases depending on the "fullness" of the summary's partitions. The proof then cleverly balances the lower bounds derived from each case to establish the final constant-factor approximation. The work is a complete contribution, presenting the algorithm, theoretical guarantees, and compelling experimental validation.

**Originality:** The work is highly original. It addresses a novel and practical problem formulation that combines the challenges of submodular maximization, streaming computation, and robustness. The proposed STAR-T algorithm is a novel contribution that thoughtfully merges and extends ideas from prior work. The combination of an exponentially partitioned structure [11] with an exponentially decreasing thresholding scheme [8] is a non-trivial and creative adaptation for the robust streaming context. The analysis technique is also tailored to this new algorithm and appears novel. The authors are commendably clear about concurrent work [18], highlighting that their approach achieves a better memory dependence on the solution size `k`.

**Significance:** The results are important and likely to be impactful. The need for algorithms that produce robust summaries from massive, streaming data is a pressing concern in many large-scale machine learning applications. The two-stage framework is particularly practical: the computationally intensive streaming part is done once to create a robust summary, which can then be queried efficiently and repeatedly for different removal scenarios (as demonstrated effectively in the interactive recommendation experiment). The empirical results are strong, showing that the algorithm's robustness does not come at a high performance cost; it is competitive with a strong, non-robust baseline that has an oracle-like advantage. This demonstrates the practical viability of the approach.

### Weaknesses

**Clarity:** The paper is generally very well-written and organized. However, a few points could be clarified to improve readability. The choice for the value of the threshold `τ` in Theorem 4.1 appears to be derived directly from algebraic balancing in the proof. A brief sentence of intuition explaining *why* this value is chosen (i.e., that it is the point that balances the two worst-case bounds from Lemma 4.2 and the combined Lemmas 4.3/4.4) would be helpful. Furthermore, there is a noticeable gap between the theoretical requirement for the memory parameter `w` (`w ≥ ⌈4dlogke * m/k⌉`) and its practical setting (`w=1`). While it is common for theoretical bounds to be looser than what is needed in practice, a short discussion on this discrepancy would strengthen the experimental section and provide valuable insight for practitioners.

**Quality:** The derived approximation constant `c ≈ 0.149` is somewhat small. While achieving any constant factor for this challenging problem is a significant result, it would be beneficial for the authors to briefly discuss whether this constant is believed to be tight for their analysis or if it could be improved with a more refined argument. This is a minor point, as the empirical performance is strong regardless of the theoretical constant.

### Arguments for Acceptance:
*   Addresses a novel, well-motivated, and significant problem.
*   Proposes a novel and elegant algorithm by creatively combining ideas from related fields.
*   Provides a sound theoretical analysis with a constant-factor approximation guarantee.
*   Demonstrates strong empirical performance on real-world tasks, validating the practical utility of the method.

### Arguments against Acceptance:
*   The theoretical approximation guarantee is a relatively small constant.
*   The connection between the theoretical requirements for parameters (e.g., `w`) and their practical settings could be discussed more thoroughly.

Overall, the strengths of this paper significantly outweigh its minor weaknesses. It makes a solid and original contribution to an important and challenging area. The proposed algorithm is both theoretically principled and practically effective.
