**Summary**
This paper addresses the problem of robust submodular maximization in a streaming setting. The goal is to design a single-pass, low-memory algorithm that produces a summary set `S` from a large dataset. This summary must be robust, meaning that even after an adversary removes an arbitrary subset `E` of `m` elements from `S`, one can still efficiently find a `k`-sized solution `Z ⊆ S \ E` that provides a constant-factor approximation to the optimal solution on the original dataset `V \ E`.

The authors propose a novel two-stage procedure. In the first stage, a streaming algorithm called STAR-T constructs the robust summary `S`. The core idea is a new partitioned thresholding scheme: the summary `S` is structured into partitions, each associated with an exponentially decreasing greedy threshold. An incoming stream element is added to the first available bucket in a partition whose threshold it exceeds. In the second stage, after `m` elements are removed, a simple greedy algorithm (STAR-T-GREEDY) is run on the remaining summary to find the final solution.

The main theoretical contribution is a proof that this procedure yields a constant-factor approximation (≈0.149) while using `O((k + m log k) log k)` memory. This memory complexity is a significant improvement over concurrent work, as it depends logarithmically on `k`. The authors validate their approach on dominating set and personalized movie recommendation tasks, showing that their method performs comparably to or better than a strong baseline (SIEVE-STREAMING) that is given the unfair advantage of knowing which elements will be removed in advance.

**Strengths**

1.  **High Originality and Novelty:** The paper introduces a compelling and novel approach to a challenging and practical problem. Combining the streaming and robust settings for submodular maximization is timely and important. The core algorithmic idea—a summary set structured by partitions with exponentially decreasing thresholds—is an elegant and clever synthesis of ideas from robust submodular optimization (the partitioned structure, e.g., [11]) and streaming algorithms (thresholding, e.g., [8]). This is not merely an incremental extension but a significant new conceptual contribution.

2.  **Significance of Results:** The contribution is highly significant for several reasons:
    *   **Problem Formulation:** The two-stage model (build a robust summary once, then query it multiple times for different removal sets) is extremely well-motivated for real-world applications like interactive recommendation systems or dynamic influence maximization, where re-processing the entire dataset is infeasible.
    *   **Theoretical Guarantees:** Providing the first constant-factor approximation for this problem with such favorable memory complexity is a major achievement. The logarithmic dependence on `k` in the memory bound is a substantial improvement over the linear dependence in concurrent work [18], making this approach far more scalable for tasks requiring large summary sizes `k`.
    *   **Practical Performance:** The experiments are well-designed and convincingly demonstrate the method's effectiveness. Showing that STAR-T can compete with an oracle-like baseline (SIEVE-STREAMING that knows the deletions beforehand) strongly validates the quality of the robust summary it constructs. The interactive movie recommendation experiment is a particularly effective illustration of the practical power of the proposed framework.

3.  **Bold Approach:** This work is ambitious. It tackles a complex problem setting and provides a clean, novel algorithmic solution. The willingness to propose a new structure and develop the analysis to support it, rather than just tweaking existing methods, is commendable.

**Weaknesses**

1.  **Low Approximation Constant:** The theoretical approximation guarantee of 0.149 is quite low. While establishing any constant-factor guarantee is a strong result for a new and difficult problem, this value is far from what is achieved in simpler, non-robust or non-streaming settings. However, the experimental results suggest that the practical performance is much stronger than this worst-case bound, which mitigates this concern significantly. For a foundational paper in a new direction, the existence of a constant-factor guarantee is more important than its precise value.

2.  **Dependence on Unknown Optimal Value:** The algorithm, in its basic form, requires an estimate `τ` of the optimal solution value, `f(OPT(k, V \ E))`. The authors address this using the standard technique of running multiple instances in parallel with geometrically spaced guesses for `τ`. While this is a common and accepted workaround in the literature, it adds a logarithmic factor to the memory complexity and is a slight practical inconvenience.

3.  **Clarity:** The paper is generally well-written, but some details could be clarified. For instance, the threshold in Algorithm 1 is given as `τ/min{2i, k}`, while the text describes it as exponentially decreasing (`τ/2i`). This transition to a constant threshold for later partitions is a subtle but important detail whose motivation could be explained more explicitly. Similarly, the intuition behind the complex form of the memory parameter `w` could be developed further for the reader. These are minor points that do not detract from the core contribution.

**Recommendation**

I strongly recommend accepting this paper. It exemplifies the kind of research that NIPS should showcase: it is bold, novel, and significant. The paper introduces a new, highly relevant problem formulation and presents an elegant algorithm with strong theoretical backing and compelling empirical support. Its key strength is the introduction of a new algorithmic idea—partitioned thresholding—that leads to a provably superior memory complexity compared to concurrent approaches. While the theoretical approximation constant is low and there are minor points of clarification, these are far outweighed by the novelty and importance of the contribution. This work opens up a new and promising direction for research in large-scale data summarization and is likely to inspire considerable follow-up work.
