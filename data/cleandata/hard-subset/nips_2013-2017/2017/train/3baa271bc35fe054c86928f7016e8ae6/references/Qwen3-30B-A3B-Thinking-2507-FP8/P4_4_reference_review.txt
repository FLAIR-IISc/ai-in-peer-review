

# Review of "Streaming Robust Submodular Maximization: A Partitioned Thresholding Approach"

This paper proposes a two-stage approach for robust submodular maximization in the streaming setting. The authors address the problem of creating a small summary set S that remains effective even after any m elements are removed, with the query stage then producing a solution of size k from S \ E. The key innovation is a partitioned thresholding algorithm (STAR-T) that structures the summary into partitions with exponentially increasing bucket sizes and exponentially decreasing thresholds, allowing robustness to m element removals while maintaining a compact summary.

The paper builds on previous work in submodular optimization, particularly SIEVE-STREAMING [8] for standard streaming submodular maximization and PRO [11] for robust submodular maximization. The authors properly contextualize their work, noting that previous robust approaches [9, 10, 11] did not address streaming settings, and their work differs from concurrent work [18] in memory complexity (O((k + m log k) log² k) versus O(mk log k/ε)).

**Quality:** The paper demonstrates strong technical soundness. The theoretical results are well-supported with detailed proofs, including key lemmas that handle two cases (partitions with many full buckets versus partitions with few full buckets). The authors carefully address the challenge of not knowing f(OPT(k, V\E)) in advance by providing a parallel variant that increases memory by a log factor, achieving a (1+ε)-approximation. The experimental validation on two applications (dominating set and movie recommendation) shows competitive performance against SIEVE-STREAMING, which knows in advance which elements will be removed. The paper acknowledges limitations, such as the need for τ approximation, and provides a solution.

**Clarity:** The paper is well-organized and clearly written. The problem statement, algorithm descriptions, and theoretical results are presented logically. Pseudocode for STAR-T and STAR-T-GREEDY is precise and well-explained. The main technical contribution (partitioned thresholding) is effectively illustrated. The paper would benefit from more intuitive explanation of why the exponentially decreasing thresholds work, but the technical details are sufficient for reproducibility.

**Originality:** The partitioned thresholding approach is novel in the context of robust streaming submodular maximization. Previous robust approaches [9, 10, 11] did not address streaming settings, while streaming approaches [8, 12, 13] did not consider robustness. The authors' use of exponentially increasing bucket sizes with exponentially decreasing thresholds represents a significant technical innovation that enables their improved memory complexity. The paper properly differentiates from concurrent work [18] by showing better memory scaling with k.

**Significance:** The problem of robust data summarization is important for large-scale applications where data is too large to store or process repeatedly. The two-stage approach (precompute robust summary, then query) is practical for real-world applications like social network influence maximization and personalized recommendations. The results demonstrate that their method matches or outperforms SIEVE-STREAMING even when SIEVE-STREAMING knows in advance which elements will be removed. The memory complexity improvement is significant for large-scale applications.

**Strengths:**
- Novel partitioned thresholding approach for robust streaming submodular maximization
- Strong theoretical guarantees with constant-factor approximation
- Practical experimental validation in two application domains
- Better memory complexity than concurrent work
- Clear distinction from related work

**Weaknesses:**
- The approximation factor (0.149) is lower than standard greedy (1-1/e ≈ 0.63), which is expected but worth noting
- The paper doesn't fully address how to set parameter τ in practice beyond the theoretical approximation
- The computational complexity of the algorithms beyond memory usage isn't discussed
- The connection to broader NIPS themes could be strengthened with more discussion of how this advances the field

**Recommendation:**
This paper makes a significant contribution to the problem of robust data summarization in large-scale settings. The theoretical results are sound, the algorithms are well-designed, and the experimental validation supports the claims. While the approximation factor is lower than standard submodular maximization, this is expected given the added robustness constraint. The memory complexity advantage over concurrent work is a meaningful contribution for large-scale applications.

The paper is well-written, properly contextualized, and addresses an important practical problem. The authors have carefully considered the technical challenges and provided a thorough analysis. The experimental results convincingly demonstrate the practical value of their approach.

**Arguments for acceptance:**
- Novel algorithmic approach for robust streaming submodular maximization
- Strong theoretical guarantees with constant-factor approximation
- Practical applications with competitive performance against strong baselines
- Better memory complexity than concurrent work
- Clear and well-structured presentation

**Arguments against acceptance:**
- Lower approximation factor than standard submodular maximization
- Some practical implementation details (like τ setting) could be more thoroughly discussed
- Limited comparison with other robust submodular maximization algorithms

Overall, this paper presents a technically sound, novel contribution that advances the state of the art in robust submodular maximization for streaming data. The theoretical and empirical results are compelling, and the problem addressed is significant for large-scale machine learning applications. The paper meets all the criteria for acceptance at NIPS.
