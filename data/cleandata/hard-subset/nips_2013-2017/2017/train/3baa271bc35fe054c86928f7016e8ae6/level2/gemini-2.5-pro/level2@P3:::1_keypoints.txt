- Problem: Maximizing a monotone submodular function under a cardinality constraint in a single-pass streaming setting with robustness to removal of m items.
- Method overview: Two-stage procedure — STAR-T builds a robust summary S in the stream, then STAR-T-GREEDY runs greedy on S \ E after up to m removals.
- Core contribution: STAR-T, a novel streaming algorithm using O(log k) partitions with exponentially decreasing thresholds and multiple buckets per partition.
- Algorithmic idea: Incoming items are placed in the first bucket whose partition threshold is exceeded, combining streaming thresholding and partitioned-structure robustness.
- Theoretical guarantee: For summary size O((k + m log k) log k), STAR-T-GREEDY obtains a constant-factor approximation ≈0.149.
- Unknown OPT handling: Analysis uses parallel algorithm instances to deal with unknown optimal value, a standard practical technique.
- Empirical validation: Experiments on dominating set and personalized movie recommendation show comparable or better performance than SIEVE-STREAMING even when SIEVE knows the removed set.
- Quality: High technical quality with rigorous proofs, clear proof sketch in main text, full details in supplement, and careful evaluation.
- Clarity: Very clearly written and well-organized; problem, algorithms, and theory are presented precisely and accessibly.
- Originality: Highly original — among the first to address robust submodular maximization in streaming and to combine partitioned structures with exponential thresholding.
- Memory advantage: Memory complexity depends logarithmically on k, improving over linear dependence in a concurrent work.
- Significance: Addresses a practically important problem (single-pass efficiency + robustness) with a simple, effective algorithm likely useful to practitioners and researchers.
- Arguments for acceptance: Novel, well-motivated problem; elegant algorithmic design; strong theoretical guarantees; convincing real-world experiments.
- Argument against acceptance: Proven approximation constant (~0.149) is modest, though any constant is notable in this setting.
- Suggestion — explain constant gap: Add discussion on why the theoretical constant is low versus strong empirical performance (analysis artifact vs. worst-case behavior).
- Suggestion — compare concurrent work: Explicitly note that Mirzasoleiman et al. [18] achieve a higher approximation (1/2 − ε) to clarify trade-offs.
- Suggestion — clarify experiments: State from which set elements are greedily removed in figures (summary S or solution on S) and comment on empirical robustness when w=1 vs. theoretical requirements.
