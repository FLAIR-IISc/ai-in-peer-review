- Addresses robust streaming submodular maximization, a relevant problem with practical applications.
- Proposes STAR-T, an efficient streaming algorithm that builds robust summaries using partitioned thresholds.
- Provides rigorous theoretical analysis with a constant-factor approximation guarantee (0.149) even after removals.
- Novel partitioning structure and exponentially decreasing thresholding rule underpin the algorithmic design.
- Paper is clearly written, well-organized, and presents the algorithm concisely.
- Empirical results on dominating set and movie recommendation show STAR-T’s robustness versus baselines like SIEVE-STREAMING and RANDOM.
- Highlights applicability in interactive settings (e.g., user filters for recommendations) and benefits of precomputed robust summaries.
- Approximation constant (0.149) is rather small and worsens with dependence on log k, limiting practical appeal.
- Multiple parameters (k, m, τ, w) introduce sensitivity; τ requires estimating f(OPT(k, V\E)), which is challenging in practice.
- Space complexity O((k + m log k) log k) can be large for big datasets and high m, potentially a bottleneck.
- Related work discussion could better emphasize trade-offs and more clearly contrast memory dependence (e.g., linear k dependence in [18]).
- Experimental details are sparse: matrix completion method, SIEVE-STREAMING parameters, and variance measures (CIs/stddev) are not reported.
- Arguments for acceptance: problem relevance, novel provable algorithm, strong theoretical foundations, and supportive experiments.
- Arguments against acceptance: small approximation constant, potentially prohibitive space, and reliance on estimating f(OPT).
- Overall recommendation: solid contribution warranting acceptance with minor revisions to expand experimental details and discuss constant/space implications and τ sensitivity.
