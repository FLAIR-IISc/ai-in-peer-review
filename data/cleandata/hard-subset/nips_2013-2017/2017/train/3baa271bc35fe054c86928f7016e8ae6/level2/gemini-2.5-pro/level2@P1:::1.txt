**Summary**

This paper addresses the problem of maximizing a monotone submodular function subject to a cardinality constraint `k` in a challenging setting that combines two practical constraints: the data arrives in a single pass (streaming), and the solution must be robust to the removal of an arbitrary set of `m` elements. The authors propose a two-stage framework. In the first stage, a novel streaming algorithm, STAR-T, makes one pass over the data to construct a robust summary `S`. In the second "query" stage, after any `m` elements are removed from `S`, a simple greedy algorithm (STAR-T-GREEDY) is run on the remaining summary to find a solution of size `k`.

The key innovation of the STAR-T algorithm is its summary structure, which is organized into partitions and buckets. Partitions are associated with exponentially decreasing thresholds, and incoming elements are placed into buckets based on their marginal gain. This structure is inspired by prior work in non-streaming robust maximization (PRO) and streaming submodular maximization (SIEVE-STREAMING), but the combination and analysis are new.

The main theoretical contribution is a proof that this procedure provides a constant-factor approximation guarantee (approximately 0.149) for the robust problem. The algorithm constructs a summary of size `O((k + m log k) log k)`, which notably has only a logarithmic dependence on `k`, contrasting with concurrent work. The authors also provide a standard method for handling the unknown value of the optimal solution by running parallel instances of the algorithm. The theoretical claims are supported by experiments on two applications: the dominating set problem and personalized movie recommendation. These experiments show that the proposed method performs comparably to, and sometimes better than, the SIEVE-STREAMING algorithm, even when the latter is given the significant advantage of knowing the removed elements in advance.

**Quality**

The paper is of high technical quality. The proposed algorithm, STAR-T, is a clever and non-trivial extension of existing ideas to a more difficult problem setting. The theoretical analysis appears sound, and the proof sketch provided in the main paper is helpful for understanding the core arguments, which involve balancing two different bounds depending on the state of the summary's buckets. The constant approximation guarantee, while not numerically high, is a strong theoretical result for this combined streaming and robust setting. The claims are well-supported by both the theoretical analysis and the empirical results. The experimental design is thoughtful, particularly the choice to compare against a strong baseline (SIEVE-STREAMING) that is given an oracle-like advantage, which makes the strong performance of the proposed method very convincing.

**Clarity**

The paper is very well-written and organized. The problem statement is clear, and the motivation for the two-stage robust streaming setting is compelling and illustrated with good examples. The algorithms are presented with clear pseudocode, and the structure of the summary `S` is helpfully visualized in a figure. The theoretical results are stated precisely. The experimental section clearly describes the datasets, objective functions, and baselines, making the results easy to interpret.

One minor point for clarification: In Section 3, the text describes the threshold for partition `i` as `τ/2^i`, while Algorithm 1, line 5, uses `τ/min{2^i, k}`. This is likely because the bucket size is also capped at `k`, so the per-element gain is based on this capped size. Clarifying this explicitly in the text would be helpful.

**Originality**

The paper is original and makes a significant contribution. While it builds upon concepts from prior work in streaming submodular maximization (thresholding from [8]) and robust submodular maximization (partitioning structure from [11]), the synthesis of these ideas into a single-pass streaming algorithm for the robust setting is novel. The analysis required to prove an approximation guarantee for this new algorithm is also a novel contribution. The paper correctly positions itself relative to related work, including a concurrent and independent paper [18], and clearly highlights the key difference in memory complexity (logarithmic vs. linear dependence on `k`), which is an important trade-off.

**Significance**

The problem addressed by this paper is highly significant. As datasets grow, the need for single-pass algorithms that produce robust summaries becomes increasingly critical in many applications, such as network monitoring, data summarization, and interactive recommender systems. The proposed method provides a practical and theoretically-grounded solution. The fact that a single precomputed summary can handle multiple, unforeseen removal queries makes it particularly useful for interactive applications, as demonstrated in the movie recommendation experiment. Researchers and practitioners working on large-scale data summarization are likely to find these ideas useful and build upon them. The work demonstrably advances the state of the art in submodular optimization.

**Arguments for acceptance:**
*   Addresses a practical and important problem at the intersection of streaming algorithms and robust optimization.
*   Proposes a novel algorithm (STAR-T) with a clever partitioning and thresholding structure.
*   Provides the first constant-factor approximation guarantee for this problem with a memory complexity that is logarithmic in the solution size `k`.
*   The paper is technically sound, clearly written, and well-organized.
*   Strong experimental results that show the method is competitive with a powerful baseline that has access to more information.

**Arguments against acceptance:**
*   The proven constant approximation factor (0.149) is somewhat low, although obtaining any constant factor in this setting is a significant achievement. The practical performance is shown to be much better.
*   There is a gap between the theoretical requirement for the memory parameter `w` and its practical setting in the experiments (`w=1`), which could benefit from more discussion.

Overall, the strengths of this paper far outweigh its minor weaknesses. It represents a solid and significant contribution to the field of large-scale machine learning and submodular optimization.
