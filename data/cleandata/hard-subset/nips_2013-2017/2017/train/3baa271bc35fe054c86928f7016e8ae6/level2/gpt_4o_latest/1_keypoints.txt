- Problem: Robust streaming maximization of a monotone submodular function under a cardinality constraint, resilient to removal of up to m selected elements.  
- Proposed method: Two-stage algorithm STAR-T (streaming summary) + STAR-T-GREEDY (query-stage greedy) using partitioned memory and exponential thresholds.  
- Novelty: Introduces a novel partitioning structure and exponentially decreasing thresholding rule extending SIEVE-STREAMING ideas.  
- Theoretical results: Provides constant-factor approximation guarantees with rigorous proofs.  
- Memory efficiency: Achieves summary size O((k + m log k) log^2 k), improving scalability.  
- Robustness: Handles arbitrary removals of up to m elements without prior knowledge, supported by theory and experiments.  
- Empirical validation: Demonstrated competitive or superior performance on dominating set and personalized movie recommendation tasks.  
- Practical relevance: Applicable to data summarization, influence maximization, and recommendation systems.  
- Clarity concern: Presentation is dense with heavy notation; partitioning and thresholding explanations need simplification.  
- Limited experiments: Evaluation limited to two applications; broader tasks (e.g., facility location, clustering) would strengthen claims.  
- Baseline comparisons: Missing comparisons with other robust submodular methods (e.g., PRO and recent algorithms).  
- Parameter assumptions: Theoretical analysis assumes knowledge/approximation of values like f(OPT); needs discussion of tuning and sensitivity.  
- Recommendation: Accept with minor revisions to improve clarity and expand experimental validation.  
- Suggested fixes: Simplify key explanations, add diverse tasks and baselines, and provide practical parameter-tuning guidance.
