- Proposes a novel CI test by framing conditional independence as a classification task using nearest-neighbor bootstrap to approximate the conditional product distribution.
- Clever and potentially powerful approach, especially promising in high-dimensional settings where traditional CI tests fail.
- Places work well within existing literature and relates clearly to classifier two-sample tests.
- Strong theoretical contributions: extends classifier two-sample tests to conditional setting with guarantees.
- Rigorous analysis of Rademacher complexity for non-i.i.d. samples is a significant technical strength.
- Empirical evaluation on synthetic and real data shows performance gains over KCIT and RCIT.
- Demonstrates practical applicability to causal inference tasks (e.g., testing unconfoundedness).
- Code is available, aiding reproducibility and adoption.
- Proofs are dense and hard to follow, reducing accessibility for some readers.
- Missing discussion on a discrete-variable analog and how bootstrap behaves with discrete Z.
- Lacks thorough sensitivity analysis on classifier choice and hyperparameter robustness.
- Minor presentation issues: redundant text in Section 1.1 and an overly verbose Algorithm 2.
- Contains typos (e.g., “classifier” and “graphs”) that should be fixed.
- Arguments for acceptance: novel, theoretically sound, empirically validated, practical impact, generally well-written.
- Arguments against acceptance: difficult-to-follow proofs, no discrete-variable discussion, limited sensitivity analysis, minor presentation/typo issues.
- Overall recommendation: accept contingent on improving proof clarity and adding sensitivity analysis.
