The main contribution of this paper is a convergent proof for minimizing sum fi(x) + g(x) in asynchronous settings, where fi(x) is smooth and g is rough. The problem is significant; to my knowledge, there is no known proof.

An important aspect of this work is the block separatability of variables, which allows for some decomposition gain in the proxy step (since the gradient of each fi can only affect specific indices of x). This is an important practical assumption, otherwise, it could create proxy locks, which could be unavoidable. (It would be interesting to see how element-based proxies, such as compression, can be decomposed.)

There are two main theoretical results here. The first theorem gives a convergent rate for the proxy SAGA, which is slower but better than the previous result. The second theorem gives a rate for asynchronous settings, which is more groundbreaking.

Overall, I think this is a good paper with an excellent theoretical contribution that is also very useful in practice. I think it would have been even stronger without the sparsity assumption, but the authors also state that researching asynchronousity without sparsity for rough functions is an open problem, so this is acknowledged.
