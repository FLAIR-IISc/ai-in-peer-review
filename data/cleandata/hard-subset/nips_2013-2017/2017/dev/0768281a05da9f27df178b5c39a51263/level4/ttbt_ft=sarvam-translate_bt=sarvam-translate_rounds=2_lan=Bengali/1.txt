Here is a summary.

This paper proposes a hyperalignment method based on using a neural network instead of a base function under a structure similar to kernel hyperalignment. This method has been applied to several real-world FMRI datasets. Improved classification accuracy has been shown at a lower computation time compared to another deep learning architecture.

Comments:

1. The novelty of this paper is unclear. From the text, the proposed method seems to be a combination of previous contributions. Is it difficult to create an objective that uses a neural network instead of a base function, which is much harder to optimize? The optimization method seems to be "inspired" by previous work.
2. George Lang's work on functional alignment should be mentioned.
3. The main purpose of alignment is to establish subject-based correspondence so that subject-based variability can be modeled. The best way to display better correspondence is to visually show that the activation patterns of subjects are better aligned. Using a classification task for evaluation in this work, it is difficult to say whether good classification results from good alignment or whether good performance only comes from good functional representation. In fact, it is unclear how data aligned in the original physiological position can be mapped for the purpose of explanation.
4. Since G is not unique, will it have a major impact on the results?
5. The results may reflect overfitting, i.e., be biased, because the best performance has been reported within the range of parameter values, as opposed to conducting proper nested cross-validations. Models with greater flexibility will tend to perform better. Also, the range of parameter values will have a great impact on the results.
6. Functional hyperalignment probably does not require pre-physiological alignment. Will performance gains be maintained if spatial normalization is not done?
