- Proposes PG2, a novel two-stage framework to generate person images in arbitrary poses from a reference image and target pose.
- Stage 1 produces a coarse image integrating reference and pose; Stage 2 refines details via an adversarial network.
- Key contributions: task formulation, pose integration using pose heatmaps, two-stage architecture with masked L1 loss.
- Well-motivated problem with practical applications (movie making, data augmentation for pose estimation).
- Builds on and cites relevant prior work in conditional image generation and multi-view synthesis.
- Novelty: explicitly conditioning on both reference image and pose is a meaningful extension; pose heatmaps are sensible.
- Technical soundness: uses established components (U-Nets, GANs); two-stage approach justified for pose + appearance complexity.
- Masked L1 loss addresses background change issues effectively.
- Experimental validation: extensive experiments on DeepFashion and Market-1501 with quantitative (SSIM, IS) and qualitative results.
- Ablation studies clarify contributions of pose embeddings, loss functions, and two-stage vs one-stage design.
- Clarity: paper is well-written; architecture diagram and stage explanations are clear.
- Limited comparison to related work—needs more thorough benchmarking against pose-transfer/image-manipulation methods.
- Quantitative metrics (SSIM, IS) are questionable for realism; user study is limited and partly contradictory.
- Failure cases discussion is brief; more analysis on causes (complex poses, clothing types) is needed.
- Hyperparameter sensitivity (e.g., lambda) analysis is insufficient; robustness to settings unclear.
- Arguments for acceptance: novel, technically sound, convincing experiments, useful ablations.
- Arguments against acceptance: limited related-work comparison, imperfect quantitative evaluation, brief failure-case discussion.
- Overall assessment: reviewer recommends "strong accept" while suggesting more thorough comparisons and analyses.
