- Problem: Studies active query-based clustering where the learner queries an oracle about pairwise same-cluster membership or more generally any fixed binary function of instance labels.
- Objective: Characterizes the query complexity required to perfectly or approximately recover the target clustering.
- Approach: Takes an information-theoretic perspective to derive bounds on query complexity.
- Results: Provides both upper and lower bounds on the number of queries needed.
- Robustness: Extends the analysis to handle noisy oracle responses.
- Novelty: Uses interesting and novel information-theoretic tools for the machine learning community.
- Motivation: Well-motivated by practical applications like record deduplication and crowdsourcing.
- Limitation: Lacks structural assumptions (no hypothesis class), so bounds mainly depend on label marginals and noise, making the setting distant from typical learning-theory frameworks.
- Significance: Represents an important first step toward understanding fundamental information-theoretic limits, particularly establishing relevant lower bounds.
