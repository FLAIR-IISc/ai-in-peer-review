This paper considers a relevant bandit problem with budget constraint B and known time horizon T. For each trial, the agent observes a context j chosen according to a probability distribution \pi, which is based on a context set {1,..,J} and takes a move k from {0} \cup {1,..,K}, where 0 means "discard". If k = 0, the agent receives no reward and incurs no cost. If k > 0, the agent receives a reward y and incurs a cost c{j,k}, where y is chosen according to a probability distribution based on [0,1] that depends on j and k, and c{j,k} > 0 is a parameter known to the agent. The agent's goal is to maximize the reward, provided that the increasing cost never exceeds the budget B.

The main part of the paper is dedicated to a special case where \pi is known and c{j,k} = 1 for all j and k, and the paper gives an algorithm with an O(log T) regret bound under a mild condition. The paper extends the algorithm to the general case where \pi is unknown and c{j,k} are arbitrary.

[16] also considers a more general situation, but it gives an algorithm with O(\sqrt T) regret. Thus, the paper is probably the first to give an algorithm with O(log T) regret, although the mild condition should be met. The algorithm is based on an approximate algorithm that computes the optimal sequence of moves if all the statistics are known. The algorithm is simple but very interesting.

The paper is well-motivated and very clearly written. It would be more interesting and significant if the O(log T) regret was always true, otherwise, the lower bound \Omega(\sqrt T) is proven.
