This paper considers a fairly simple budget-constrained CB problem with isolated contexts and activities. Its main contribution is the computational efficiency of the algorithm (using LP approximation) and the fact that it achieves a log(t) regress bound.

This paper is very well written and easy to follow.

Suggestion: In future work, consider using a parametric model for the reward. This is useful when the number of contexts is large. In section 4.1, when incorporating ALP with UCB, be more explicit about the logic used in the UCB-ALP algorithm. For example, I think you are following optimism in the face of uncertainty. This is probably equivalent to the principle update of URPPL2. It would be great to include some numerical tests in the extended version of the paper.

You can see this paper: http://arxiv.org/abs/1506.03374 This paper is worthy of acceptance because it provides an efficient algorithm for constrained CB with optimal regress in most cases. It is interesting and a real improvement over existing work.
