This paper considers a contextual bandit problem with a budget constraint *B* and a known time horizon *T*. In each trial, the agent observes a context *j* sampled from a fixed probability distribution π over a context set {1, …, *J*}, and then selects an action *k* from {0} ∪ {1, …, *K*}, where *k* = 0 represents skipping the trial. If *k* = 0, the agent receives no reward and incurs no cost. Otherwise (*k* > 0), the agent receives a reward *y* and incurs a cost *c<sub>j,k</sub>*, where *y* is drawn from a fixed probability distribution over [0, 1] dependent on *j* and *k*, and *c<sub>j,k</sub>* > 0 is a known parameter. The agent’s goal is to maximize cumulative reward while ensuring the cumulative cost remains within the budget *B*.

The main contribution of the paper focuses on a special case where π is known and *c<sub>j,k</sub>* = 1 for all *j* and *k*. An algorithm is presented with a regret bound of O(log *T*) under a mild condition. The paper then extends this algorithm to the more general scenario where π is unknown and the *c<sub>j,k</sub>* values are arbitrary.

A related, more general setting is explored in [16], but that work achieves a regret of O(√*T*). Thus, this paper potentially represents the first work to achieve O(log *T*) regret, contingent upon the mild condition holding. The algorithm is based on an approximation algorithm for computing the optimal action sequence given complete knowledge of the problem statistics. The algorithm itself is simple yet interesting.

The paper is well-motivated and clearly written. It would be more impactful if the O(log *T*) regret guarantee held universally, or if the authors could demonstrate an Ω(√*T*) lower bound for the cases where the condition is not met.

Initially, I was concerned about the correctness of Lemma 1, as I wasn’t convinced that the optimal action sequence always satisfies condition (2) for all trials. This led me to question whether using the average constraint truly provides a valid relaxation of the hard constraint. However, my concerns were addressed in the rebuttal, and I am now satisfied with the lemma’s proof.

Minor comments and questions remain: Is it NP-hard to exactly compute the optimal action sequence? Can the results be generalized to scenarios where *T* is unknown? And, can we assume *u<sup>1</sup>* > *u<sup>2</sup>* > … > *u<sup>*</sup><sub>J</sub>* without loss of generality, even when the context distribution is unknown? 

Overall, I strongly recommend this paper for acceptance.
