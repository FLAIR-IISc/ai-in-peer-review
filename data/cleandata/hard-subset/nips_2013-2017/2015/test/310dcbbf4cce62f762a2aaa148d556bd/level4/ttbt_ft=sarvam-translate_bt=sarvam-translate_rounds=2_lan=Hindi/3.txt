This paper considers a relevant bandit problem with budget constraint B and known time horizon T. For each trial, the agent observes a reference j chosen according to a certain probability distribution \pi on a reference set {1,..,J} and takes an action k from {0} \cup {1,..,K}, where 0 means "leave." If k = 0, the agent receives no reward and incurs no cost. If k > 0, the agent receives a reward y and incurs a cost c{j,k}, where y is chosen according to a certain probability distribution on [0,1] that depends on j and k, and c{j,k} > 0 is a parameter known to the agent. The agent's goal is to maximize the cumulative reward, subject to the condition that the cumulative cost never exceeds the budget B.

The main part of the paper is dedicated to a special case where \pi is known and c{j,k} = 1 for all j and k, and the paper gives an algorithm whose regret bound is O(log T) under a mild condition. The paper extends the algorithm to the general case where \pi is unknown and c{j,k} is arbitrary.

[16] considers a more general setting, but it gives an algorithm with O(\sqrt T) regret. Therefore, the paper is probably the first work with O(log T) regret, although the mild condition must be satisfied. The algorithm is based on an heuristic algorithm that calculates the optimal action sequence when all data is known. The algorithm is simple but very interesting.

The paper is well-motivated and very clearly written. It would be more interesting and important if the O(log T) regret always held, otherwise the lower bound \Omega(sqrt T) is proven in the limiting cases.

All the results seem to be based on Lemma 1, but I am not sure about its correctness.
