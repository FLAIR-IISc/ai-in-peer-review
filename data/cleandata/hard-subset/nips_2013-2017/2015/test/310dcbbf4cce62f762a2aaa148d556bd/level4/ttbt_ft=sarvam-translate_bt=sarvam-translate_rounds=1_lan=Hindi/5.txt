This paper explores a relatively simple budget constraint constrained CB problem with discrete contexts and tasks. The main contributions are the computational efficiency of the algorithm (through LP approximation) and the fact that it achieves a log(t) regress bound.

The paper is very well-written and easy to understand.

Suggestion: In future work, consider using a parametric model for reward. This is useful when the number of contexts is large. In section 4.1, when combining UCB with ALP, be more explicit about the logic used in the UCB-ALP algorithm. For example, I think you are following optimism in the face of uncertainty. This is probably equivalent to a policy update in URL2. It would be great to include some numerical experiments in an expanded version of the paper.

You might want to check out this paper: http://arxiv.org/abs/1506.03374 This paper deserves acceptance because it provides an efficient algorithm for constrained CB with optimal regress in most cases. It is interesting and is indeed an improvement over existing work.
