- Studies budget-constrained contextual bandits with discrete contexts/actions and proposes an LP-approximation approach.
- Introduces Adaptive Linear Programming (ALP) algorithm approximating the oracle, achieving near-optimal performance (O(1) regret except boundary cases).
- Proposes UCB-ALP that combines ALP with UCB to handle unknown rewards, achieving O(log T) regret except boundary cases.
- Paper is well-written, well-organized, and easy to follow; technical proofs are provided in the supplement.
- Technical analysis is sound and results are significant for constrained contextual bandits.
- Algorithm is computationally efficient via LP approximation.
- Assumes a non-parametric reward model, which may not scale well with many contexts.
- Suggests considering parametric reward models to improve scalability when contexts are numerous.
- Recommends clarifying the rationale for combining UCB with ALP (explicitly state optimism-in-the-face-of-uncertainty and ties to UCRL2).
- Recommends adding numerical experiments in an extended version to demonstrate practical performance.
- Suggests referencing related work (e.g., arXiv:1506.03374).
- Original contribution: novel combination of LP approximation with UCB for constrained contextual bandits.
- Significance: provides an efficient, improved algorithm with near-optimal regret guarantees.
- Overall recommendation: strengths outweigh weaknesses; paper recommended for acceptance.
