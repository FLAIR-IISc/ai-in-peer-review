A major problem with the Markov network is weight learning, which is often defined as finding a set of weights (connected to the features) that maximize the likelihood. Typically, a gradient descent algorithm is used to learn the weights, and a crucial sub-task of this algorithm is to compute the gradient. Since the latter is generally NP-hard, the MCMC algorithm is often used instead of the exact algorithm to compute it.

This paper proves from theoretical results that if the weights (parameters) are such that MCMC converges to a stable distribution over them quickly, then the weight learning algorithm (which uses MCMC to compute the gradient) is an FPRS. This paper considers two cases: the convex case and the non-convex case, and creates separate limits for each.

The paper is generally well-written, but there are some errors here and there. I couldn't verify all the evidence, but it seems credible.

Although the theoretical results seem impressive, they have almost no practical relevance (I like that the authors acknowledge this). I think it is practically impossible to ensure that the parameters meet the limitations required by the theorems. Have the authors given any thought to this? Specifically, is it possible to artificially enforce this condition (this condition can be seen as a form of regularization, and using ridge regression alongside it can create an over-regularized model). If this last problem can be solved, you will have a very powerful paper with these theoretical results.

Overall, this is a reasonably well-written paper that provides an FPRS for weight learning using MCMC (with and without regularization). However, this condition has almost no practical relevance. The results obtained in the paper need to solve non-trivial research problems before creating a practical/exact algorithm for weight learning.
