

# Review: "Maximum Likelihood Learning With Arbitrary Treewidth via Fast-Mixing Parameter Sets"

This paper proposes a novel approach to maximum likelihood learning in exponential families by restricting parameters to a "fast-mixing" set Θ, where Markov chain Monte Carlo (MCMC) sampling mixes rapidly. The authors show that gradient descent with MCMC-based gradient estimates converges to the optimal solution within Θ with high probability, providing theoretical guarantees on the number of samples and Markov chain transitions required. This work differs from previous approaches (e.g., tree-structured models, pseudolikelihood) by focusing on fast-mixing properties rather than treewidth constraints.

**Strengths:**
1. The paper provides a clean theoretical framework for analyzing gradient descent with MCMC-based gradient estimates, filling a gap in the literature on practical maximum likelihood learning.
2. The results are well-structured, with clear theorems and proof sketches that capture the essential arguments.
3. The connection between fast-mixing parameter sets and tractable learning is conceptually interesting and potentially useful for practitioners.

**Weaknesses:**

1. **Notation and Terminology Inconsistencies:**
   - The paper defines θ as a parameter vector but uses θ to denote a set of parameters (e.g., "Θ = {θ : ∃tree T, ∀(i, j) ∉ T, θij = 0}"), which is confusing. The set should be denoted differently (e.g., Θ).
   - In equation (1), the regularization term is written as "λ ∥θ∥22" without the standard 1/2 factor, which is inconsistent with standard convex optimization notation. This minor but significant notational issue could confuse readers.
   - The paper uses "R2" for the norm bound on sufficient statistics, but in the example section, it states "∥t(x)∥2 ≤ R2 = √24," which is inconsistent with standard notation (R2 should be the bound, not the norm).

2. **Theoretical Errors and Ambiguities:**
   - In the Introduction, the paper states "evaluation of the partition function (which can easily be computed from the likelihood)" – this is incorrect. The likelihood requires the partition function, but the partition function cannot be easily computed from the likelihood.
   - Theorem 1 claims the Lipschitz constant L = 4R2^2 + λ, but the proof sketch states "∥dA/dθ - dA/dφ∥2 ≤ 2R2∥pθ - pφ∥TV" and then "∥pθ - pφ∥TV ≤ 2R2∥θ - φ∥2." The second bound is not justified and appears incorrect; the standard bound is ∥pθ - pφ∥TV ≤ R2∥θ - φ∥2 (not 2R2).
   - In Theorem 4, the expression "v = log acβ1β3ϵ/(− logα)" is ambiguous; it should be log(1/(acβ1β3ϵ)) or similar to ensure positivity.
   - In Theorem 5, the expression "log 4DR2C/β1β3ϵf" should be log(4DR2C/(β1β3ϵf)) to be mathematically correct.

3. **Example Section Errors:**
   - The example claims "the sufficient statistics are t(x) = {xixj |(i, j) ∈ Pairs}" for the Ising model, but this is incomplete; the sufficient statistics should include both pairwise terms (xixj) and unary terms (xi).
   - The example states "ϵθ = 2" as the desired accuracy, but ϵθ should be a small positive number (e.g., 0.01), not 2. This is a significant error in the example's parameters.
   - The paper mentions using "L = 10" (lower than the theoretical L = 97), but doesn't explain why this is valid or how it affects convergence guarantees.

4. **Formatting and Typographical Errors:**
   - "the Efron-Stein inequality can bounds its variance" (Theorem 2 proof sketch) should be "can bound."
   - "Dept. of Communications" (Acknowledgements) should be "Department of Communications."
   - The paper states "the existence of a FPRAS for the partition function would imply that RP = NP" – this is incorrect. A FPRAS for a #P-complete problem would imply RP = NP, but the statement is misleading.
   - In the Example, "a lower value of L = 10 is used, which converged faster in practice" – this is concerning because using a lower Lipschitz constant than theoretically justified could lead to instability or divergence.

5. **Missing Context:**
   - The paper claims its approach "applies generally to any exponential family with bounded sufficient statistics," but provides no general method for constructing the fast-mixing set Θ beyond the Ising model example. This limits the generality of the claim.
   - The discussion of contrastive divergence (Section 7) is superficial and doesn't properly connect to the main results.

**Overall Assessment:**
While the paper presents an interesting theoretical approach to maximum likelihood learning with MCMC, the numerous technical inconsistencies and errors significantly undermine the credibility of the results. The notational errors, theoretical misstatements, and example inaccuracies make it difficult to trust the correctness of the main theorems. The paper's contribution is novel but its execution is flawed.

**Recommendation:**
Reject. The paper's theoretical contribution is potentially interesting but the numerous technical errors and inconsistencies in the presentation are too severe to be overlooked. A careful revision addressing these issues would be required for acceptance. The minor nature of the errors (relative to the main contribution) is irrelevant to the quality of the paper – the reviewers' role is to ensure technical correctness, not to overlook minor flaws.
