This paper addresses the problem of testing membership of an unknown distribution. This problem is fundamentally based on [n] (or [n]^d, for linearity), where the distributions are linear, log-convex, unimodal, and linear hazard rate distributions. A beautiful general framework is followed for each problem. First, an approximate value q of the unknown distribution p is created, which effectively assumes that p belongs to the class being considered, in order to reduce the complexity of the sample. Then, a modified chi-squared test is applied to test whether p and q are close. The required approximate values for achieving the first step are new and unique for each class distribution.

This two-step approach avoids obvious alternatives, such as testing for equality within a net on p and, let's say, the class of interest, which carries huge computational and statistical costs. The proposed method provides efficient algorithms with improved (and optimal) sample complexity.

It is perhaps somewhat noteworthy that the rates for the four classes are not only identical (in the case of d=1, for linear distributions), but also the construction of matching lower bounds for each problem is identical.

Incorporating some definitions would increase clarity for readers, such as the partial order in the definition of linear distributions on hypergrids. (Also, a definition of distance is promised in the appendix on line 161, which seems to be missing.) This paper makes an important contribution to the field of testing size-restricted discrete distributions.
