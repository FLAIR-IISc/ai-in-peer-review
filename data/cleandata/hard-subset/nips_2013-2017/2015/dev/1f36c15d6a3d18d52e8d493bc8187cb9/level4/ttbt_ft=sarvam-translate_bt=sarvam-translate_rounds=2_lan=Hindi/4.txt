This paper addresses the problem of testing an unknown distribution for [n] (or [n]^d for [n]), for a set of similar, log-convex, unimodel, and equal-risk-rate distributions. The approach for each problem follows a good general framework – first, an estimate q of the unknown distribution p is created, which effectively assumes that p is related to the class being considered to reduce sample complexity. Then, a modified Chi-square test is applied to test whether p and q are close. The estimators required to obtain the first step are new and unique for each class distribution.

This two-step process avoids the need for explicit choices, such as testing for similarity, e.g., a net on the class in question, which involves large computational and statistical costs. The proposed approach gives efficient algorithms with better (and optimal) sample complexities.

It is perhaps somewhat remarkable that not only are the (optimal) rates the same for each of the four classes (in the case of d=1, for the same distribution), but also the construction used in the lower bound for each problem is the same.

Incorporating some definitions will improve clarity for readers who are not familiar with this particular topic, for example, the partial ordering in the definition of the same distribution on a hypergrid. (Also, line 161 promises a definition of distances in the appendix, which seems to be missing.) This paper makes a significant contribution to the testing of shape-restricted discrete distributions.
