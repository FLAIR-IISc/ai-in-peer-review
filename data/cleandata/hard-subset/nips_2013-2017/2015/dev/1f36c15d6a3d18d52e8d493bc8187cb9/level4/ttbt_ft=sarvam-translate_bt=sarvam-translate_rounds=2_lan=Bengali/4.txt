This paper addresses the problem of testing the membership of an unknown distribution. This problem is fundamentally based on [n] (or [n]^d, for linearity), where the distributions are linear, log-convex, unimodal, and linear hazard rate distributions. A beautiful general framework is followed for each problem. First, an estimated value q of the unknown distribution p is created, which effectively assumes that p belongs to the considered class, so that the sample complexity is reduced. Then, a modified Chi-squared test is applied to check whether p and q are close. The estimated values required to achieve the first step are new and unique for each class distribution.

This two-step method avoids obvious alternatives, such as testing for equality within a net of p and, let's say, the class of interest, which carries huge computational and statistical costs. The proposed method provides efficient algorithms with improved (and optimal) sample complexity.

It is perhaps somewhat remarkable that the rates for the four classes are not only uniform (in the case of d=1, for the linear distribution), but also the construction of matching lower bounds for each problem is uniform.

Some definitions would increase clarity for readers if they were included, such as the partial order in the definition of linear distributions in Hypergrid. (Also, the distance definition is promised in the appendix on line 161, which seems to be missing.) This paper makes an important contribution to the field of testing size-bounded discrete distributions.
