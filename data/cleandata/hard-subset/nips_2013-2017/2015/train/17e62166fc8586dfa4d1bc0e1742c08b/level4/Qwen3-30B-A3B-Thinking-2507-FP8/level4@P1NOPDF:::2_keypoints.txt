- Novel architecture for retrieving coherent sentence sequences to describe image sequences from blog posts.  
- Comprehensive quantitative, qualitative, and human evaluations showing clear superiority over baselines that ignore sentence coherence.  
- Ablation study (removing the local coherence model) validates the contribution of the coherence component.  
- Authors commit to releasing source code and the dataset.  

- Related-work claim: paper asserts [5] lacks sentence-coherence mechanism, but [5] models coherent intermediate semantic/topic representations—this claim needs clarification.  
- Figure–text inconsistency: Fig.2b implies per-sentence fully connected layers, while Eq.(2) uses shared parameters tied to a single bi-LSTM output—needs resolution.  
- Evaluation metric: BLEU is suboptimal for single-reference sentence generation; METEOR is recommended as a more appropriate automatic metric.  
- Redundant linear layers: sequential W_{f1} and W_{f2} appear equivalent to one linear transform; an ablation is needed to justify the two-layer design.  
- Parameter sharing: identical FC parameters are used for bi-LSTM output (o_t) and local coherence output (q); the rationale for this shared parameterization should be justified.  
- Paragraph vector unclear: need to state whether the paragraph vector model [16] is fine-tuned or kept fixed for reproducibility.  

- Post-rebuttal: reviewer recommends acceptance—authors addressed formulation, evaluation, and related-work concerns; final paper must explicitly state the paragraph-vector training status.
