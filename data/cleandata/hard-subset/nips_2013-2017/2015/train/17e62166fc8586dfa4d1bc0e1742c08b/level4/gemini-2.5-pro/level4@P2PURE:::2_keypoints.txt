- Task and approach: generates coherent multi-sentence descriptions for image sequences by retrieving K=5 nearest training images/sentences, embedding sentences, and using a Bi-LSTM over sentence embeddings with a ReLU+FC and image–sentence compatibility scoring plus a local coherence model.  
- Core contribution: a method to select and sequence retrieved sentences to form a coherent description for an entire image stream.  
- Strength: proposes a novel, effective architecture that models coherence across sentences.  
- Strength: extensive quantitative, qualitative, and human evaluations show superiority over baselines, with an ablation study removing the coherence model.  
- Strength: authors plan to release source code and dataset.  
- Issue: claim that prior work [5] has “no mechanism for coherence” is inaccurate since [5] models intermediate semantic representations that promote topic-level coherence.  
- Issue: apparent discrepancy between Figure 2b and Section 3.3/Equation (2) about whether the fully-connected layer is applied jointly to all sentences or shared and applied per time step—needs clarification.  
- Issue: evaluation metric choice—METEOR may be more appropriate than BLEU when only a single reference exists; ask if METEOR was considered.  
- Issue: Equation (2) uses two consecutive linear transforms (Wf1, Wf2) whose composition is linear, so the benefit of the two-layer structure is unclear and warrants an ablation.  
- Issue: unclear motivation for sharing the same FC layer parameters for both the Bi-RNN output (o_t) and the local coherence model output (q_t) in Equation (2).  
- Issue: need to state whether the paragraph vector model [16] is fine-tuned during training or kept fixed.  
- Recommendation (post-rebuttal): after rebuttal the reviewer recommends acceptance, noting authors addressed concerns, and requests the camera-ready clarify whether paragraph vectors are fine-tuned.
