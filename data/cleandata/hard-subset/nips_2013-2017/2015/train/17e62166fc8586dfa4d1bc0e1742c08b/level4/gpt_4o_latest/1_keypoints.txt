- Addresses retrieving natural sentences for image sequences and forming coherent sentence sequences; closely related to multi-sentence video description work.  
- Fails to cite prior related work on video/multi-sentence description (e.g., Rohrbach et al. 2014).  
- Paper is well-written, clear, and the problem is well defined.  
- Uses a set of robust techniques to address the problem.  
- Proposes CRCN neural model that captures relationships between sentence sequences and image sequences.  
- Model architecture is technically sound and incorporates discourse via entity-based coherence models.  
- Sentence-generation method is simplistic/ad hoc: link each sentence to visually similar training images and concatenate by image order.  
- Dataset provenance is unclear, data not disclosed, and description suggests potential anonymity breach (authors likely same as referenced [11]).  
- Equation (2) seems incorrect, implying st derives solely from ot and suggesting the network is not fully connected.  
- Reviewer recommendation: suitable for acceptance as a poster in current form.
