- Task & approach: Retrieve top K=5 similar training images/sentences per query image and select coherent sentences for an image sequence using sentence embeddings and a bidirectional LSTM.

- Architecture detail: BRNN outputs go through ReLU and fully connected layers, then scored by an image–sentence compatibility function; a local coherence model is added to enforce sentence-to-sentence compatibility.

- Main contribution: Method to select the most relevant, coherent sentences for a sequence of images.

- Strength: Proposes a novel, effective architecture for retrieving coherent multi-sentence descriptions.

- Strength: Extensive quantitative, qualitative, and human evaluations showing superiority over baselines and an ablation removing the coherence model.

- Strength: Authors plan to release source code and dataset.

- Weakness 1: Incorrect claim about prior work [5] lacking coherence mechanism—[5] models coherent intermediate semantic representations/topics across sentences.

- Weakness 2: Figure 2b appears inconsistent with Section 3.3/Equation (2)—visual suggests FC layers connect to all sentences while equation implies parameter sharing and single-sentence connections.

- Weakness 3: BLEU is a suboptimal automatic metric with single references; METEOR would be more suitable.

- Weakness 4: Using two sequential linear functions (Wf2, Wf1) is unclear since they are equivalent to one linear transform; an ablation is needed to justify them.

- Weakness 5: Unclear why the same FC layer parameters are used for both BRNN outputs (o_t) and the local coherence model q in Equation (2).

- Weakness 6: Unclear whether the paragraph vector [16] is fine-tuned or kept fixed.

- Post-rebuttal: Reviewer recommends acceptance—authors addressed formulation, evaluation, related work, and clarified paragraph vector usage—final version still needs several clarifications.
