- Task & contribution: Proposes retrieving K=5 similar image-sentence pairs per image and selecting coherent sentences to describe an image sequence.
- Model overview: Embeds sentences, models sentence sequence with a bidirectional LSTM, applies ReLU and a fully connected layer, then computes image-sentence compatibility; includes a local coherence model.
- Main novelty: Defines a method to select relevant, coherent sentences for an image sequence.
- Evaluation strengths: Extensive quantitative, qualitative, and human evaluations showing superiority over baselines lacking coherence.
- Ablation & release: Performs an ablation removing coherence and plans to release code and dataset.
- Related work concern: Claim that [5] has “no mechanism for coherence” is inaccurate because [5] models topics/semantic representations that induce coherence.
- Figure/equation mismatch: Figure 2b implies the FC layer connects to all sentences, while Eq.(2)/Section 3.3 indicates shared parameters applied per sentence—needs clarification.
- Metric choice: BLEU may be inappropriate with a single reference; suggest using METEOR for automatic evaluation.
- Redundant linear layers: Two consecutive linear functions (Wf2 and Wf1) can be combined; authors should justify or ablate their use.
- Shared parameters question: Using the same FC parameters for BRNN output (o_t) and the local coherence model q is unexplained and should be justified.
- Paragraph vector training: Unclear whether the paragraph vector [16] is fine-tuned or fixed during training; needs clarification.
- Recommendation: Reviewer recommends acceptance after the authors address formulation, evaluation, related-work issues, and clarify the design choices above.
