- Proposes a novel method to describe image sequences by retrieving K=5 similar image–sentence pairs and selecting a coherent sentence sequence via sentence embeddings and a bidirectional LSTM.
- Model architecture: sentence embeddings → biLSTM → ReLU + fully connected layer → image–sentence compatibility scoring, with an added local coherence model.
- Strength: novel, effective architecture for retrieving coherent sentences for image streams.
- Strength: extensive quantitative, qualitative, and human evaluations showing superiority over baselines without coherence.
- Strength: ablation study removing the coherence model supports the claimed contribution.
- Strength: authors plan to release source code and dataset, benefiting the community.
- Related-work correctness concern: incorrect claim that [5] has “no mechanism for coherence,” since [5] models topic-level semantic coherence across sentences.
- Clarity/correctness issue: Figure 2b appears inconsistent with Section 3.3/Equation (2) regarding FC layer connections versus parameter sharing.
- Evaluation metric concern: BLEU is used but METEOR would be more appropriate, especially with single reference sentences.
- Design-choice question: using two linear functions in Equation (2) (Wf2, Wf1) seems redundant—an ablation should justify this choice.
- Design-choice question: unclear why the same FC parameters are used for BRNN outputs (o_t) and the local coherence model q; needs explanation.
- Implementation detail missing: unclear whether paragraph vectors [16] are fine-tuned or kept fixed.
- Verdict: reviewer recommends acceptance after authors address the above clarifications, particularly the paragraph vector question.
