- Task: Generate coherent multi-sentence descriptions for image streams, extending single-image captioning to sequential visual input.  
- Model: Proposes Coherent Recurrent Convolutional Network (CRCN) combining CNNs for images, BRNNs for language, and a local coherence model.  
- Novelty: Key contribution is extending captioning to sequential output (multiple sentences) with an explicit coherence model.  
- Data: Leverages large-scale blog posts (NYC and Disneyland) as image–text parallel training data.  
- Evaluation: Shows improved performance over baselines using BLEU/Recall metrics and Amazon Mechanical Turk user studies.  
- Strength: Technical soundness—well-designed integration of established components and clear training/loss description.  
- Strength: Empirical validation—comprehensive experiments and user studies support effectiveness.  
- Strength: Data sourcing—using blogs is a clever way to obtain sequential image–text pairs.  
- Weakness: Baselines—insufficient implementation/adaptation details make comparisons hard to assess.  
- Weakness: Coherence model explanation—parse-tree/entity-grid description is dense and needs clearer, more intuitive exposition and ablation.  
- Weakness: Limited error analysis—paper lacks detailed failure-case analysis to show model limitations.  
- Recommendation: Overall positive—accept with minor revisions to clarify baseline implementations and the coherence model.
