- Introduces a novel recurrent convolutional encoder-decoder network to learn out-of-plane rotations for 3D objects (faces, chairs).
- Basic model: encoder disentangles input into identity and pose units; action units modify pose; decoder (conv + upsampling) reconstructs rotated image and mask.
- Extended recurrent model: identity units fixed, pose units updated by a sequence of action units to support longer rotation trajectories.
- Primary contribution: learning disentangled identity/appearance and pose representations that yield view-invariant identity features.
- Identity units shown to be discriminative for cross-view object recognition (quantitative evidence).
- Disentangling enables better rendering predictions when using longer rotation trajectories via curriculum training.
- Paper is well-written with clear motivation and strong qualitative and quantitative results supporting the method.
- Weakness: model only supports discrete rotation angles tied to the training set; extension to continuous angles is not addressed.
- Weakness: trained with fixed-length sequences, which conflicts with typical variable-length recurrent network behavior.
- Overall: novel end-to-end recurrent conv encoder-decoder with generative disentangling is well-demonstrated.
