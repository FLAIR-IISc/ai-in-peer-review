This paper proposes an algorithm for learning decision trees with linear splits at each node. Instead of the standard greedy, node-by-node approach, this algorithm optimizes all tree parameters—the weights of the split functions and the output distributions at the leaves—in a global manner. The authors formulate the problem by proposing an L2-regularized upper bound on the tree's empirical loss, which results in a smoother objective function. The algorithm then uses stochastic gradient descent (SGD) to find the tree parameters that minimize this upper bound.

The formulation of the tree optimization problem is elegant, drawing a clear connection to structured prediction with latent variables. However, the empirical results presented are not fully convincing that this global optimization approach is superior to traditional greedy methods. The proposed method significantly outperforms the greedy baseline on only a few datasets, while incurring higher computational costs.

The paper is well-structured, but several points require clarification:
- The description of the "stable SGD" variant is unclear. Specifically, the mechanism for assigning data points to leaves needs further explanation. Including pseudo-code for this procedure would be beneficial.
- In Figure 2, it is not specified whether the results were generated using the standard SGD or the stable SGD variant. Additionally, the term "active" leaf should be precisely defined.
- In Algorithm 1 (line 10), the parameter Θ is updated and then projected onto a simplex. The paper should describe the practical method used to solve this projection problem. It would also be helpful to explicitly state that this projection ensures that each row of Θ sums to one, enforcing a valid probability distribution.
