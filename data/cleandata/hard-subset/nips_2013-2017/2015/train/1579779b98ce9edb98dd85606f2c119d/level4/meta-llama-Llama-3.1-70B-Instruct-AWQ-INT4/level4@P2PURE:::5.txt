This paper proposes an algorithm for learning a decision tree with linear split functions at the nodes, optimizing the tree parameters (i.e., the weights in the split functions and the distribution of the output variable at each leaf) globally, in contrast to the standard greedy approach. The authors derive an upper bound on the tree's empirical loss and regularize it using L2 regularization of the split function weights to achieve a smoother optimization problem. They then employ stochastic gradient descent to identify the tree parameters that minimize the upper bound.

The tree optimization problem is elegantly formulated by establishing a connection with structured prediction with latent variables. However, the empirical results presented in the paper do not convincingly demonstrate that global optimization is a better alternative to greedy learning of the tree. The globally optimized tree significantly outperforms the greedy tree on only a couple of datasets, while being more computationally intensive.

The paper is well-structured, but some points require clarification. Firstly, the stable version of stochastic gradient descent (SGD) is not clearly described, particularly in terms of data point assignment to leaves. Including a pseudo-code would likely improve understanding. Secondly, Figure 2 lacks clear explanations: it is unclear whether the results were obtained using SGD or stable SGD, and the definition of an "active" leaf is not provided.

In Algorithm 1, line 10, the update and projection of \Theta onto a simplex could be better explained. Specifically, it would be helpful to describe how this projection problem is solved in practice and to mention that this projection ensures each line of \Theta sums up to one.

Minor comments and typos include:

- Lines 407-408: The text mentions tuning the "number of features evaluated when building an ordinary axis-aligned decision tree." However, in standard decision tree construction, all features are evaluated at each node. 
- Equations (7), (8), and (9): "argmax" should be replaced with "max."
- Figure 1 is not mentioned in the main text.
- Line 141: The sentence "...to the index of the leaf on by this path" appears to be incorrect.
- Line 238: The text should read "the solutions to loss-augmented inference" instead of "the solution."
- Line 301: The phrase "An key observation" should be corrected to "A key observation."
- Line 360: The acronym FSGD is not defined.
- Line 404: The text likely means "tuning" errors instead of "training errors."

In the supplementary material:

- Line 29: The correct notation should be "max{g = sgn...}" instead of "max{g \in sgn...}".
- Line 30: The correct equation should be "sgn(Wx)^T Wx + l(\Theta^Tf(sgn(Wx)), y)" instead of "sgn(Wx)^T Wx + l(\Theta^Tf(g), y)".

Overall, while the tree optimization problem is elegantly formulated, the results presented do not convincingly demonstrate the superiority of global optimization over greedy learning of the tree.
