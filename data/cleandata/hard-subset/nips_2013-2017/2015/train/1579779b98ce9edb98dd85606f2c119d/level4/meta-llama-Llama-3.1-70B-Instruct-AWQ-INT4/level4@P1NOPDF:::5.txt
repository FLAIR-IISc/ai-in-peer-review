This paper presents an algorithm for learning a decision tree with linear split functions at the nodes. The algorithm optimizes the tree parameters, specifically the weights in the split functions and the distribution of the output variable at each leaf, in a global manner, contrasting with the standard greedy approach to learning trees. An upper bound on the tree's empirical loss is proposed, and to achieve a smoother optimization problem, this upper bound is regularized via L2 regularization of the split function weights. The algorithm then employs stochastic gradient descent to identify the tree parameters that minimize the upper bound.

The tree optimization problem is elegantly formulated by establishing a link with structured prediction with latent variables. However, the empirical results presented in the paper do not convincingly demonstrate that global optimization is a better alternative to greedy learning of the tree. The globally optimized tree significantly outperforms the greedy tree on only a couple of datasets, while being more computationally intensive.

The paper is well-structured, but some points require clarification:

- The stable version of stochastic gradient descent (SGD) is not clearly described. Specifically, it would be helpful to explain how data points are assigned to leaves, and a pseudo-code would be beneficial.
- Figure 2: It is unclear whether these results are obtained using SGD or stable SGD. Additionally, the definition of an "active" leaf is not provided.
- In line 10 of Algorithm 1, \Theta is updated and then projected onto a simplex. In practice, it would be helpful to describe how this projection problem is solved. It is also worth mentioning that this projection ensures that each line of \Theta sums up to one.
