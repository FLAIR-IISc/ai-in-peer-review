This research paper presents an algorithm for learning a decision tree with linear partition functions on nodes. This algorithm optimizes the tree's parameters (i.e., weights in partition functions and the distribution of the output variable on each leaf) in a global manner, which is in contrast to the standard greedy approach for learning trees. An upper bound on the empirical loss of the tree is proposed, and to obtain a smooth optimization problem, this upper bound is regularized through L2 regularization of the partition function weights. The algorithm then involves using a stochastic gradient descent to identify the tree's parameters that minimize the upper bound.

The tree optimization problem is elegantly formulated by establishing a link with a hidden variable with structured prediction. However, the empirical results shown in the paper do not actually convince us that global optimization is a better alternative to greedy learning of a tree. Globally optimized trees perform significantly better than greedy trees on only a few datasets, while being computationally more intensive.
