- Proposes a global algorithm for learning decision trees with linear split functions instead of the standard greedy method.
- Introduces an upper bound on empirical loss and regularizes it with L2 on split weights to smooth optimization.
- Uses stochastic gradient descent to minimize the regularized upper bound and learn tree parameters.
- Elegant formulation linking tree optimization to structured prediction with latent variables.
- Empirical results show global optimization beats greedy on some datasets but improvements are not consistently significant.
- Global optimization is more computationally intensive than greedy training.
- Stable SGD variant is not well explained; a clear description and pseudocode for leaf assignment/updates are needed.
- The meaning and determination of an “active” leaf are unclear and should be explicitly defined.
- Figure 2 lacks indication of whether results use SGD or stable SGD—each result should state which algorithm was used.
- Experimental setup omits key details (number of iterations, learning rates, etc.) that should be reported.
- Algorithm 1’s Θ update (line 10) is unclear; clarify how projection onto the simplex is implemented and note that it makes each row of Θ sum to one.
- Multiple minor errors/typos in the main text that need correction (specific line-by-line fixes provided).
- Supplementary material contains incorrect notation in lines 29–30 that should be corrected.
- Overall: strong theoretical formulation but insufficient empirical evidence and clarity; more experiments and clarifications required.
