- Introduces an algorithm for decision trees using linear split functions with global optimization of split weights and leaf output distributions versus greedy construction.
- Proposes a smooth upper bound on empirical loss with L2 regularization on split weights and optimizes it via stochastic gradient descent.
- Formulation connects tree optimization to structured prediction with latent variables.
- Empirical evaluation shows limited performance gains on a few datasets and increased computational complexity, so benefits of global optimization are not convincingly demonstrated.
- Implementation details of the stable SGD variant are unclear; leaf assignment for data points needs an explicit description and pseudocode.
- Figure 2: unclear whether results use standard SGD or the stable variant; definition of “active” leaf is missing.
- Algorithm 1 projects Θ onto a simplex; practical details for solving this projection and the fact that each row of Θ sums to one should be stated.
- Lines 407–408: statement about tuning “number of features evaluated” is misleading because standard trees typically evaluate all features at each node.
