This paper introduces an algorithm for learning decision trees using linear split functions at each node. The key innovation lies in a global optimization approach to determining tree parameters – the weights defining the split functions and the leaf-specific output distributions – contrasting with the traditionally employed greedy methods. To facilitate optimization, the authors propose an upper bound on the empirical loss, which is further regularized using L2 regularization on the split function weights, resulting in a smoother optimization landscape. The algorithm then leverages stochastic gradient descent (SGD) to minimize this regularized upper bound.

The formulation of the tree optimization problem is elegant, drawing a compelling connection to structured prediction with latent variables. However, the empirical evaluation fails to definitively demonstrate the benefits of this global optimization strategy over greedy tree learning. While the globally optimized tree exhibits significant performance improvements on a limited number of datasets, these gains are accompanied by a substantial increase in computational cost. This raises questions about the practical advantages of the proposed approach.

The paper is generally well-structured, but several points require further clarification:

*   **Stable SGD Implementation:** The description of the stable SGD variant lacks detail. Specifically, the process of assigning data points to leaves needs to be explained more thoroughly. Including pseudocode would greatly improve understanding.
*   **Figure 2 Interpretation:** The figure’s results should clarify whether they are derived from standard SGD or the stable SGD variant. Furthermore, the definition of an “active” leaf requires more precise articulation.
*   **Simplex Projection:** Algorithm 1 updates \Theta and then projects it onto a simplex. The practical implementation of this projection step should be described. It would also be beneficial to explicitly state that this projection enforces the constraint that each row of \Theta sums to one, representing a valid probability distribution.

**Minor Comments and Typos:**

*   Lines 407-408: The phrase “number of features evaluated when building an ordinary axis-aligned decision tree” is unclear, as standard decision tree algorithms typically evaluate all features at each node.
*   Equations (7), (8), and (9): Replace “argmax” with “max”.
*   Figure 1: This figure is referenced in the paper but not discussed in the main text.
*   Line 141: “…to the index of the leaf on by this path.” – This sentence fragment requires correction.
*   Line 238: “…the solution to loss-augmented inference…” – Change to “…the solutions to…”
*   Line 301: “An key observation…” – Change to “A key observation…”
*   Line 360: Define the acronym “FSGD”.
*   Line 404: “…a tree with minimum training errors.” – This likely refers to “tuning” errors.

**Supplementary Material:**

*   Line 29: `max{g \in sgn...}` should be `max{g = sgn...}`
*   Line 30: `sgn(Wx)^T Wx + l(\Theta^Tf(g), y)...` should be `sgn(Wx)^T Wx + l(\Theta^Tf(sgn(Wx)), y)`

In summary, while the theoretical formulation of the tree optimization problem is a valuable contribution, the empirical results presented are not sufficiently compelling to demonstrate a clear advantage over existing, greedy tree learning methods. The clarity of the implementation details could be improved with additional explanations and pseudocode. Addressing the points raised above would strengthen the paper considerably.
