This paper proposes an algorithm for learning decision trees with linear split functions at the nodes, where the tree parameters are optimized globally, in contrast to the standard greedy approach. The authors introduce an upper bound on the tree's empirical loss, which is regularized via L2 regularization of the split function weights to facilitate a smoother optimization problem. The algorithm employs stochastic gradient descent to identify the optimal tree parameters that minimize the upper bound.

The formulation of the tree optimization problem is elegant, leveraging a connection to structured prediction with latent variables. However, the empirical results presented in the paper fail to convincingly demonstrate the superiority of global optimization over greedy learning. The globally optimized tree significantly outperforms the greedy tree on only a few datasets, while being more computationally intensive.

The paper is well-structured, but some points require clarification:

- The stable version of SGD is not clearly described. Clarification on data point assignment to leaves and a pseudo-code would be beneficial.
- Figure 2: It is unclear whether the results are obtained using SGD or stable SGD. Additionally, the definition of an "active" leaf is not provided.
- In line 10 of Algorithm 1, the update and projection of \Theta onto a simplex require further explanation. It would be helpful to describe the practical solution to this projection problem and mention that it ensures each line of \Theta sums up to one.

Minor comments and typos:

- Lines 407-408: The description of tuning the "number of features evaluated when building an ordinary axis-aligned decision tree" is unclear, as all features are evaluated at each node in a standard decision tree.
- Equations (7), (8), and (9): "argmax" should be replaced with "max."
- Figure 1 is not mentioned in the main text.
- Line 141: The sentence "...to the index of the leaf on by this path" appears to be incorrect.
- Line 238: The phrase "...the solution to loss-augmented inference..." should be pluralized to "...the solutions to..."
- Line 301: "An key observation" should be corrected to "A key..."
- Line 360: The definition of FSGD is not provided.
- Line 404: The phrase "...a tree with minimum training errors" likely means "...tuning errors."

In the supplementary material:

- Line 29: The notation "max{g \in sgn...}" should be corrected to "max{g = sgn...}"
- Line 30: The equation "sgn(Wx)^T Wx + l(\Theta^Tf(g), y)..." should be corrected to "sgn(Wx)^T Wx + l(\Theta^Tf(sgn(Wx)), y)"

While the tree optimization problem is elegantly formulated, the results presented are not convincing, and further clarification is needed to fully appreciate the proposed algorithm.
