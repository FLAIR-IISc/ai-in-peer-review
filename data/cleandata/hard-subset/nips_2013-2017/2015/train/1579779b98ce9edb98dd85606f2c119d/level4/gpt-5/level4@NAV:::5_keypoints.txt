- Proposes globally optimized decision-tree learning with linear split functions, using an upper bound on empirical loss and L2 regularization, optimized by SGD.  
- Elegant formulation connecting tree optimization to structured prediction with latent variables.  
- Empirical results are unconvincing: global optimization only beats greedy baselines on a few datasets and is more computationally expensive.  
- “Stable” SGD variant is not clearly described—unclear how data points are routed to leaves; pseudocode requested.  
- Figure 2: unclear whether results use standard SGD or stable SGD and what constitutes an “active” leaf.  
- Algorithm 1 (line 10): projection of Theta onto a simplex is not specified—need practical solution and note that it enforces each row sums to one.  
- Line 407–408: unclear phrasing about “number of features evaluated” for axis-aligned trees—standard trees evaluate all features at each node.  
- Equations (7), (8), (9): use “max” instead of “argmax.”  
- Figure 1 is never cited in the main text.  
- Several wording/typo issues: line 141 (incorrect sentence ending), line 238 (“solution” → “solutions”), line 301 (“An key” → “A key”), line 360 (define FSGD), line 404 (“minimum training errors” likely should be “tuning errors”).  
- Supplementary fixes: line 29 replace max{g in sgn...} with max{g = sgn...}; line 30 correct sgn(Wx) usage to l(Theta^T f(sgn(Wx)), y).  
- Overall assessment: formulation is elegant but experimental validation is weak.
