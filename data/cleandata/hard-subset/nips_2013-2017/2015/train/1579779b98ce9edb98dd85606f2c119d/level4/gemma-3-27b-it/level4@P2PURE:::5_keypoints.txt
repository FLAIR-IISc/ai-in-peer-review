- Introduces algorithm for learning decision trees with linear split functions and globally optimized split weights and leaf distributions via SGD on a regularized upper bound.  
- Uses an L2-regularized upper bound on empirical loss to enable smoother optimization.  
- Elegant formulation connecting tree learning to structured prediction with latent variables.  
- Empirical results show substantive gains on only a few datasets and incur substantial computational cost; advantage over greedy methods not convincingly demonstrated.  
- Implementation of the stable SGD variant is unclear—specifically how data points are assigned to leaves; pseudo-code requested.  
- Clarify whether Figure 2 uses standard SGD or stable SGD and define what constitutes an “active” leaf.  
- Algorithm 1 (line 10): projection of Θ onto the simplex is described but the practical projection solution is not provided; note that projection enforces a sum-to-one constraint per row.  
- Line 407–408 confusingly refers to “number of features evaluated” when standard trees typically evaluate all features at each node.  
- Equations (7), (8), (9): replace “argmax” with “max.”  
- Figure 1 is referenced but not discussed in the main text.  
- Line-specific wording and typo fixes needed: line 141 fragment, line 238 pluralization, line 301 “An key” → “A key,” line 360 define “FSGD,” line 404 clarify “tuning” vs “minimum training errors.”  
- Supplementary corrections: line 29 use `max{g = sgn...}` instead of `max{g \in sgn...}`, and line 30 use `l(Θ^T f(sgn(Wx)), y)` instead of `l(Θ^T f(g), y)` with sgn(Wx) inserted.  
- Overall: strong theoretical contribution but experimental evidence and computational trade-offs need stronger justification.
