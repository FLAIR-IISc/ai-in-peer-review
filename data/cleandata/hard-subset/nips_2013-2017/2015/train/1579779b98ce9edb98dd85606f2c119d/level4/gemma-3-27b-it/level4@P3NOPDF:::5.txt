This paper presents an algorithm for learning decision trees using linear split functions at each node. A key contribution is the formulation of tree learning as a global optimization problem, contrasting with the standard greedy approaches. The authors propose an upper bound on the tree’s empirical loss and regularize it with an L2 penalty on the split function weights to encourage smoother optimization. The algorithm then employs stochastic gradient descent (SGD) to minimize this regularized upper bound and learn the tree parameters.

The formulation of the tree optimization problem is elegant, particularly the connection drawn to structured prediction with latent variables. However, the empirical evaluation does not strongly support the claim that global optimization consistently outperforms greedy tree learning. While the globally optimized tree exhibits significant improvements on a limited number of datasets, this comes at a considerable computational cost. This trade-off is not fully justified by the presented results.

The paper is generally well-structured, but several points require clarification to enhance understanding. First, the implementation details of the “stable” version of SGD are insufficiently described. Specifically, the method for assigning data points to leaves is unclear. Providing pseudocode would be highly beneficial. Second, Figure 2’s results lack context. It is unclear whether these results are obtained using standard SGD or the stable variant. Further, the definition of an “active” leaf needs to be explicitly stated. Third, Algorithm 1 updates and projects the matrix Θ onto a simplex in line 10. The practical method used to solve this projection problem should be specified. It would also be valuable to mention that this projection ensures each row of Θ sums to one, thereby representing a probability distribution over leaves.

Several minor comments and typos detract from the presentation:

*   Lines 407-408 mention tuning the “number of features evaluated when building an ordinary axis-aligned decision tree.” This is potentially misleading, as standard decision tree algorithms typically evaluate all features at each node.
*   Equations (7), (8), and (9) should use “max” instead of “argmax.”
*   Figure 1 is referenced but not discussed within the main text.
*   Line 141 contains an incomplete sentence: “…to the index of the leaf on by this path.”
*   Line 238 should read “…the solutions to loss-augmented inference…”
*   Line 301 contains a grammatical error: “An key observation…” should be “A key observation…”
*   Line 360: FSGD is undefined; please clarify this abbreviation.
*   Line 404: “…a tree with minimum training errors” likely refers to “tuning” errors.

In the supplementary material:

*   Line 29: “max{g \in sgn…” should be “max{g = sgn…”
*   Line 30: “sgn(Wx)^T Wx + l(\Theta^Tf(g), y)...” should be “sgn(Wx)^T Wx + l(\Theta^Tf(sgn(Wx)), y)”

In summary, while the paper presents a theoretically sound and elegantly formulated approach to global decision tree optimization, the empirical results are not sufficiently compelling to demonstrate a clear advantage over established greedy methods. Addressing the clarity issues raised above and providing a more convincing empirical evaluation would significantly strengthen the contribution of this work. The potential for computational savings and robustness improvements with this approach remains an open question.
