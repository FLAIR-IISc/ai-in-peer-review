This paper introduces an algorithm for learning decision trees utilizing linear split functions at each node. The core innovation lies in a global optimization approach to tree parameter learning – encompassing split function weights and leaf-level output distributions – contrasting with the conventional greedy tree learning paradigm. To facilitate smoother optimization, an upper bound on empirical loss is proposed and regularized using L2 regularization on the split function weights. The algorithm then employs stochastic gradient descent to minimize this regularized upper bound and identify optimal tree parameters.

The formulation of the tree optimization problem is particularly elegant, establishing a connection to structured prediction with latent variables. However, the empirical evaluation does not strongly demonstrate the benefits of global optimization over greedy tree learning. While the globally optimized tree exhibits significant performance gains on a limited number of datasets, this improvement is accompanied by increased computational cost.

The paper is well-structured, but several points require further clarification:

- The implementation details of the stable SGD variant are unclear. Specifically, the method for assigning data points to leaves should be elaborated upon, and the inclusion of pseudo-code would be beneficial.
- Regarding Figure 2, it is unclear whether the presented results were obtained using standard SGD or the stable SGD method. Additionally, the definition of an “active” leaf requires clarification.
- In Algorithm 1, line 10, the projection of \Theta onto a simplex is performed. A description of the method used to solve this projection problem would be valuable, along with a note that this projection enforces the constraint that each row of \Theta sums to one.

Minor comments and potential typos include:

- Lines 407-408: The phrase “number of features evaluated” is potentially misleading, as standard decision tree construction typically evaluates all features at each node.
- Equations (7), (8), and (9): “argmax” should be replaced with “max”.
- Figure 1 is referenced in the figures but not discussed in the main text.
- Line 141: The final phrase “…to the index of the leaf on by this path” appears grammatically incorrect.
- Line 238: “…the solution to loss-augmented inference…” should be “…the solutions to…”.
- Line 301: “An key observation…” should be “A key observation…”.
- Line 360: The abbreviation “FSGD” is undefined.
- Line 404: “…a tree with minimum training errors” likely refers to “tuning” errors.

In the supplementary material:

- Line 29: “max{g \in sgn…” should be “max{g = sgn…”.
- Line 30: “sgn(Wx)^T Wx + l(\Theta^Tf(g), y)… should be “sgn(Wx)^T Wx + l(\Theta^Tf(sgn(Wx)), y)…”.

In summary, while the tree optimization problem is elegantly formulated, the empirical results do not provide compelling evidence to support the advantages of the proposed global optimization approach.
