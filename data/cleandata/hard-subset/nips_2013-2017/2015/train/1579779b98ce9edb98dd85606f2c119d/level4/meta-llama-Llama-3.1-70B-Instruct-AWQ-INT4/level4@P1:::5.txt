This paper presents an algorithm for learning a decision tree with linear split functions at the nodes. The algorithm optimizes the tree parameters (i.e., the weights in the split functions and the distribution of the output variable at each leaf) in a global way, as opposed to the standard greedy way of learning trees. An upper bound on the tree's empirical loss is proposed, and to achieve a smoother optimization problem, this upper bound is regularized via a L2 regularization of the split function weights. The algorithm then consists of using a stochastic gradient descent to identify the tree parameters that minimize the upper bound.

The tree optimization problem is elegantly formulated, by establishing a link with structured prediction with latent variables. However, the empirical results that are shown in the paper do not entirely convince that a global optimization is a better alternative to a greedy learning of the tree. The globally optimized tree outperforms the greedy tree on a couple of datasets, but the improvement is not significant across all datasets. Furthermore, the global optimization approach is more intensive computationally.

The paper is well-structured, but some points need to be clarified. The stable version of SGD, in particular, is not described very clearly. It would be helpful to have a pseudo-code to illustrate how data points are assigned to leaves and how the assignments are updated. Additionally, it would be beneficial to provide more details on what an "active" leaf means exactly and how it is determined.

In Figure 2, it is unclear whether the results are obtained using SGD or stable SGD. A clear indication of which algorithm is used for each result would be helpful. Furthermore, it would be beneficial to provide more details on the experimental setup, such as the number of iterations used for each algorithm and the learning rate.

In Algorithm 1, the update of Θ in line 10 is not entirely clear. It would be helpful to provide more details on how the projection onto a simplex is performed in practice. Additionally, it would be beneficial to mention that this projection ensures that each line of Θ sums up to one.
