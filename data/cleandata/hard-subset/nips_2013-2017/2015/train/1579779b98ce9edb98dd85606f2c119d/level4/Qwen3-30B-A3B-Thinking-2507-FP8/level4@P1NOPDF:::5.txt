

This paper proposes a global optimization approach for learning decision trees with linear split functions at nodes, contrasting with standard greedy tree learning. The algorithm jointly optimizes split function weights and leaf output distributions by minimizing a regularized upper bound on empirical loss (using L2 regularization on split weights), implemented via stochastic gradient descent. The problem is elegantly framed as structured prediction with latent variables.

While the formulation is mathematically sound, the empirical results fail to convincingly demonstrate that global optimization offers meaningful advantages over greedy learning. The globally optimized tree significantly outperforms the greedy variant on only a small number of datasets, while being computationally more intensive. The limited performance gains do not justify the increased complexity for most practical scenarios.

The paper is well-structured, but several clarifications are needed:
- The stable SGD variant requires better explanation. Specifically, how are data points assigned to leaves during optimization? A pseudocode would greatly improve clarity.
- Figure 2: Are these results from standard SGD or stable SGD? Additionally, the meaning of an "active" leaf should be explicitly defined.
- In Algorithm 1, line 10, Θ is updated and projected onto a simplex. The paper should specify how this projection is implemented (e.g., using a standard simplex projection algorithm) and note that it ensures each row of Θ sums to one.

Minor issues and typos:
- Lines 407–408: The phrase "number of features evaluated when building an ordinary axis-aligned decision tree" is confusing. In standard axis-aligned trees, all features are evaluated at every node; the phrasing implies a subset is considered, which is inaccurate.
- Equations (7), (8), (9): "argmax" should be "max".
- Figure 1 is referenced in the text but never mentioned in the main body.
- Line 141: "...to the index of the leaf on by this path" contains an error; likely should be "on which this path ends."
- Line 238: "...the solution to loss-augmented inference..." should be "solutions to...".
- Line 301: "An key observation" → "A key observation".
- Line 360: "FSGD" is undefined.
- Line 404: "a tree with minimum training errors" → "a tree with minimum training error" (or "a tree minimizing training error").

Supplementary material corrections:
- Line 29: "max{g ∈ sgn...}" → "max{g = sgn...}".
- Line 30: "sgn(Wx)^T Wx + l(Θ^Tf(g), y)..." → "sgn(Wx)^T Wx + l(Θ^Tf(sgn(Wx)), y)...".

The core contribution—global optimization via structured prediction with latent variables—is theoretically interesting. However, the empirical evidence supporting its practical utility is insufficient. The marginal performance gains on select datasets do not adequately offset the computational cost, and the presentation of results lacks sufficient detail to assess the method's true potential. The paper would benefit significantly from additional experiments comparing scalability, clearer algorithmic descriptions, and a more thorough analysis of when global optimization provides tangible benefits over greedy approaches.
