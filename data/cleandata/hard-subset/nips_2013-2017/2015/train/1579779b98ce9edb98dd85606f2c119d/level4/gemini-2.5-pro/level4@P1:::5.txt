This paper presents a non-greedy algorithm for learning decision trees with oblique (linear) split functions. In contrast to standard greedy methods that optimize one node at a time, this algorithm globally optimizes all tree parameters—the weights of the split functions and the class distributions at the leaves—jointly. The core of the method is to formulate the tree optimization problem as structured prediction with latent variables. The authors then derive a convex-concave upper bound on the tree's empirical loss, which serves as a surrogate objective function. This objective, regularized by constraining the L2 norm of the split function weights, is then optimized using stochastic gradient descent (SGD).

The primary strength of the paper is its elegant formulation. Establishing a link between decision tree learning and structured prediction with latent variables is a novel and principled approach to the challenging problem of global tree optimization.

However, the main weakness is that the empirical results are not entirely convincing. The experiments show that the globally optimized trees significantly outperform greedy baselines on only a couple of the datasets presented. For the other datasets, the performance gains are marginal. Given that the proposed method is considerably more computationally intensive than standard greedy algorithms, the results do not provide a strong enough case that this global optimization approach is a clear practical alternative.

The paper is well-structured, but several points require clarification to improve the manuscript:

*   The description of the Stable SGD (SSGD) variant is unclear. Specifically, the process for assigning data points to leaves needs to be explained in more detail. Providing pseudocode for the SSGD algorithm would be very helpful.
*   In Figure 2, it is not specified whether the results for the proposed method were obtained using the standard SGD or the SSGD variant.
*   The term "active leaf" is used in the text but is not precisely defined.
*   In Algorithm 1 (line 10), the parameter matrix Θ is updated. The authors should clarify the practical implementation of this step, particularly how the projection onto the probability simplex is solved. It would also be worth explicitly stating that this projection ensures each row of Θ corresponds to a valid probability distribution.

**Minor Comments & Typos:**

*   In equations (7), (8), and (9), the `argmax` operator should be replaced with `max`, as the goal is to find the maximum value, not the argument that produces it.
*   Figure 1 is not referenced in the main body of the text.
*   Line 141: The end of the sentence appears to be incomplete or contain a typo: "...to the index of the leaf on by this path."
*   Line 238: "the solution to loss-augmented inference..." should likely be "the solutions to..." if multiple solutions are possible.
*   Line 301: "An key observation..." should be "A key observation..."
*   Lines 407-408: The text mentions tuning the "number of features evaluated when building an ordinary axis-aligned decision tree." This is confusing, as standard decision tree algorithms typically evaluate all features at each node. Please clarify if this refers to a method akin to Random Forests, where a random subset of features is considered for each split.
*   Line 404: In the description of hyperparameter tuning, the text states the goal is to find "a tree with minimum training errors." This should probably be "validation error," as using the training error for tuning would lead to overfitting.

**In the supplementary material:**

*   Line 29: `max{g \in sgn...}` should be `max{g = sgn...}` or simply state the expression for `g = sgn(Wx)`.
*   Line 30: The expression should be `sgn(Wx)^T Wx + l(Θ^T f(sgn(Wx)), y)`.

In summary, the paper introduces an elegant and novel formulation for non-greedy decision tree optimization. However, its practical significance is diminished by experimental results that do not consistently demonstrate a compelling advantage over simpler, greedy alternatives.
