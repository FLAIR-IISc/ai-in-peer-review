This paper presents a decision tree learning algorithm using linear split functions in nodes. This algorithm optimizes the tree parameters (i.e., the weights of the split function and the distribution of the output variable at each leaf) in a global manner, unlike the standard greedy approach of tree learning. An upper bound on the empirical loss of the tree is proposed, and to achieve a smooth optimization problem, this upper bound is regularized through L2 regularization of the weights of the split function. The algorithm then uses a stochastic gradient descent to identify the tree parameters that reduce the upper bound.

The tree optimization problem is nicely formulated by establishing a link with structured predictions with linear variables. However, the empirical results shown in the paper are not truly convincing as an alternative to learning a tree using the global optimization greedy approach. The globally optimized tree only significantly surpasses the greedy tree on a few datasets, where it is computationally more intense.
