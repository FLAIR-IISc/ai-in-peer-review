

# Review of "Efficient Non-greedy Optimization of Decision Trees"

## Summary
This paper proposes a non-greedy approach for decision tree learning by formulating the problem as structured prediction with latent variables. The authors introduce a convex-concave upper bound on the empirical loss and optimize it using stochastic gradient descent. The key technical contribution is reducing the complexity of loss-augmented inference from O(2^dp) to O(d^2p) for deep trees. The authors compare their method against several greedy baselines on benchmark datasets, claiming improved generalization performance.

## Strengths
The paper presents a novel connection between decision tree learning and structured prediction with latent variables, which is an interesting theoretical contribution. The reduction of computational complexity from exponential to quadratic in tree depth is a significant practical improvement. The experimental results demonstrate consistent improvements over greedy baselines across multiple datasets, suggesting the approach has practical value.

## Weaknesses

### Critical Notational Error
Equation (5) contains a fundamental notational error: "sgn(Wx) = argmax h∈H^m (h^TWx)". This is incorrect because sgn(Wx) is a vector (of dimension m), while argmax h∈H^m (h^TWx) is a single element of H^m (a binary vector of length m). The left-hand side is a vector, while the right-hand side is a single vector. This error fundamentally undermines the paper's theoretical foundation.

### Missing Reference
Reference [22] (Microsoft Kinect application) is cited in the introduction but is missing from the reference list. This is a critical omission in a scientific paper.

### Incomplete References
Several references are incomplete (e.g., "Global tree optimization: A non-greedy decision tree algorithm, K.P. Bennett, Computing Science and Statistics, pages 156–156, , "). This violates basic academic standards.

### Unsupported Claim
The paper claims "our latent variable formulation of decision trees enables efficient learning, it can handle any general loss function" (Section 2). However, the paper only demonstrates log loss and squared loss. The authors do not provide any theoretical justification for why their approach would work with arbitrary loss functions, nor do they provide experimental results with other loss functions.

### Incomplete Complexity Analysis
The paper claims "the fast loss-augmented inference can be solved in O(d^2p) time" (Section 6). However, the explanation is insufficient. For each of the d+1 leaves, computing the path to the leaf requires O(d) operations, and for each leaf, the inner product requires O(p) operations. Thus, the total complexity should be O(d^2p), but the paper's explanation does not clearly connect these steps. The paper should explicitly state why this is O(d^2p) rather than O(d^2p) or other complexity.

### Missing Supplementary Material
The paper references "supplementary material" for the proof of Proposition 1, but no supplementary material is provided in the submission. This is unacceptable for a scientific paper.

### Inconsistent Notation
The paper uses "m" for both the number of internal nodes and the row index in the matrix W ∈ R^(m×p). This is confusing and should be clarified. The paper should use a different symbol for the number of internal nodes (e.g., M) to avoid confusion with the matrix row index.

### Experimental Details Missing
The paper states "Connect4 only has 55,000 training points" but does not provide the test set size or validation set size. The paper also fails to provide statistical significance of the improvements over baselines (e.g., p-values or confidence intervals).

### Incomplete Algorithm Description
Algorithm 1 has a line 6: "W^(tmp) ← W^(t) − η ĝx^T + η ĥx^T" followed by a loop over rows in line 8. This suggests W^(tmp) is a matrix, but the operation is vectorized. The paper should clarify how the vectorized operation relates to the row-wise projection in line 8.

## Recommendation
While the paper presents an interesting connection to structured prediction and achieves practical improvements over greedy methods, the critical notational error in Equation (5) fundamentally undermines the theoretical foundation. The missing reference [22], incomplete references, unsupported claim about general loss functions, and missing supplementary material are serious issues that prevent the paper from meeting NIPS's quality standards.

The paper would be significantly strengthened by:
1. Correcting the fundamental notational error in Equation (5)
2. Providing complete references
3. Demonstrating the approach with a loss function beyond log loss and squared loss
4. Including the supplementary material for Proposition 1
5. Clarifying the complexity analysis

Without these corrections, the paper's technical contributions cannot be properly evaluated. I recommend rejection.
