

1) The authors must explicitly state in Section 3.2 that the algorithm is substantially similar to the approaches in references [14] and [15], as it represents a hybrid derived using identical approximations.  
2) The initial discussion paragraph incorrectly implies that the "constrain-then-train" methodology (learning a weight distribution followed by sampling for ensemble inference) is novel, despite its prior proposal and validation in reference [15].  
3) The novelty claim (i) in the introduction's concluding paragraph is misleading: reference [15] introduced "expectation backpropagation," a backpropagation update rule using expectation propagation for binary neurons (with sign activation, where zero input is never exact) and binary weights, which inherently supports constrained connectivity.  
4) The cited previous state-of-the-art results are inaccurate; to the best of my knowledge, reference [15] achieved the prior best results for binary weights and neurons on MNIST (without data augmentation, as implemented in this work).
