- Proposes a unified two-stage object detection framework combining a Region Proposal Network (RPN) and a Fast R-CNN detector with shared convolutional layers.  
- Sharing convolutional layers yields an efficient, end-to-end system that runs near real-time (~5 fps) and achieves state-of-the-art accuracy.  
- Concern: The alternating optimization appears to be performed only once; authors should justify not iterating and provide an ablation on number of alternating steps.  
- Concern: Region proposals are generated from a coarse convolutional feature map—clarify the effective stride when mapped to input coordinates and its impact on localization, especially for small objects.  
- Concern: Need more discussion/analysis on whether bounding-box regression can fully compensate for a potentially coarse initial anchor grid.  
- Clarification: Variable h_a is used (lines 173–174) without a prior definition.
