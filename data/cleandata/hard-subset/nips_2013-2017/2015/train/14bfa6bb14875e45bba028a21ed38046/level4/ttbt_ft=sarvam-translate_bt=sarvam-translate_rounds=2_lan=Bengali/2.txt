This paper presents a region proposal network (RPN) that shares the conv features with R-CNN and includes two fully connected layers that predict object boundaries and object scores at each location. The RPN combines with R-CNN in a heuristic manner for object detection and achieves state-of-the-art performance in VOC07 and VOC12. The region proposal generation is significantly reduced because the RPN shares conv layers with the detection network.

Advantages:
- A smart idea to share conv layers between RPN and R-CNN, which has greatly reduced the runtime of object proposals.
- State-of-the-art performance on object detection.
- A well-written paper.

Disadvantages:
- RPN and R-CNN are weakly combined through rapid R-CNN training optimization changes, and there is no convergent evidence.
- RPN can be used independently as a region proposal method. However, it does not seem to perform better than selective search.
- It is a very nice work, which improves both the efficiency and accuracy of R-CNN in object detection. Well-written paper and thorough experimental evaluation. The only concern is how to combine RPN and R-CNN for object detection. The proposed alternative optimization method is somewhat heuristic.
