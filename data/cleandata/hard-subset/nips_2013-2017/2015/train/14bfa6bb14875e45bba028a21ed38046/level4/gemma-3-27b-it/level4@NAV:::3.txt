This paper presents a two-stage object detection algorithm built upon convolutional neural networks (CNNs), effectively extending existing CNN-based object detection approaches. The method employs a region proposal network (RPN) operating on the high-level feature map of a CNN, followed by a fast R-CNN stage for refined detection. Leveraging shared convolutional layers and GPU acceleration, the algorithm demonstrates near real-time performance (5fps). 

However, several points require further clarification. First, the description of the alternating optimization process between the RPN and R-CNN suggests a single iteration. If so, the approach implicitly assumes the initial RPN proposals are optimal, which is unlikely to hold true in practice.

Second, the application of a pixel-wise sliding window to the top layer of the CNN raises a concern regarding the effective step size at the original image resolution. Due to the downsampling introduced by max-pooling layers within the CNN, a sliding window step on the feature map corresponds to a potentially large step size on the input image, potentially leading to inaccuracies in bounding box localization. 

Finally, the definition of the variable $h_a$ in lines 173 and 174 remains unclear and should be explicitly stated.
