This paper considers linear estimation with a non-linear link function. Inspired by recent work by Plan and Vershinin, it studies the asymptotic estimation error using generalized Lasso for arbitrary link functions with \mu \ne 0. It is shown that the asymptotic error is exactly the same, whether the link function is linear or non-linear, as shown in Theorem 2.1.

Theorem 2.1 applies when the linear measurement vector is sampled from a normal distribution, which can be strict in some cases. The author may need to explain why this assumption is necessary to justify the conclusion.

Some questions: 1. How to choose the regularizer based on the set \kai, where x_0 is located? While the paper discusses the notion of a convex regularizer, it seems that the above problem has not yet been solved. 2. While the result shows that the non-linear is equivalent to the linear measurement, the conclusion includes \mu and \sigma, which indicate the robustness of the estimate from the non-linear function g. I would like to know if the authors have any thoughts on the need for these dependencies.

Some references are missing. 1. "Study of linear vector/matrix estimation under unknown link function single index model and reduction in insufficient dimension." However, the paper does not review any papers in this area. 2. "Optimal linear estimation under unknown linear estimation" by Yi et al., a recently written paper, introduces a new algorithm for sparse recovery even with a link function with zero \mu. The paper establishes the equivalence of high-dimensional linear estimation with linear and non-linear link functions under appropriate conditions. The conclusion is carefully supported by theoretical evidence and empirical results. It is a well-written paper with quite interesting results.
