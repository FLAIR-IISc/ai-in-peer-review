In many class-based multi-class classification problems, semantic similarity between classes is often observed. This paper proposes an algorithm called Top-K SVM, which explicitly addresses learning situations where you only get one value when the ground truth is outside the proposed Top-K class (such as the ImageNet challenge). After extending the loss function to Top-K loss, an efficient optimization method has been created. Experiments on large datasets have shown the scalability of the algorithm and observed clear improvements compared to comparable algorithms.

The paper is clearly inspired and easy to follow. And the problem they work on (similar classes in multi-class learning) is an important problem that occurs in many practical applications. This paper has created the Top-K SVM algorithm in the case of multi-class classification when some classes may be very similar to each other. The algorithm is compatible with very large datasets and has shown improvements compared to comparable methods.
