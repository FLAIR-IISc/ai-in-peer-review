- Problem: Multiclass classification with many classes often has semantic class similarity, motivating top-k evaluation (cost only if true class is outside top k).
- Contribution: Introduces top-k SVM that explicitly optimizes for top-k accuracy.
- Loss formulation: Extends the loss function to a top-k loss tailored to the setting.
- Optimization: Derives an efficient optimization procedure for the top-k SVM.
- Experiments: Large-scale experiments show scalability and substantial improvements over baselines.
- Presentation: Paper is well-motivated, accessible, and addresses an important practical challenge.
