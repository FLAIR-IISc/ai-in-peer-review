- Proposes a novel gossip-style decentralized algorithm using a secondary state (initial observations performing random walks) to compute U-statistics (variance, correlations) across a network.  
- Key innovation: secondary state variable enables computing higher-order statistics via random-walked initial observations.  
- Provides convergence analysis with rates O(1/t) (synchronous) and O(log t / t) (asynchronous).  
- Numerical experiments claim superiority over existing methods (e.g., U2-gossip).  
- Lacks clear connection to prior gossip literature (e.g., Dimakis et al.) and lifting constructions (Jung, Shah, Shin, LADA).  
- Unclear specification of the induced edge distribution from the “node wakes up, contacts neighbor” process; needs more detail.  
- Should consider and compare alternative designs (more memory per node, flooding/contact-based approaches) as baselines.  
- Analysis appears to assume worst-case initial data distributions; this should be clarified in problem statements and theorems.  
- Experiments should include stronger baselines and comparisons to lifting-based accelerations (e.g., LADA).  
- Contains minor errors, typos, and unclear directed/undirected graph implications that need correction.  
- Overall: significant and promising contribution to decentralized estimation, but requires revisions addressing clarity, prior work, assumptions, and experimental baselines.
