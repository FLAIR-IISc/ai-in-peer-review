- Paper is well-motivated and clearly written.  
- Proposes synchronous and asynchronous randomized gossip algorithms for decentralized U-statistic estimation; core contribution is algorithm development and analysis.  
- Provides convergence bounds for the proposed methods.  
- Experimental results on two datasets show significant outperformance of the U2-gossip baseline across various graph structures using average relative AUC error.  
- Reviewer requests run-time per iteration for both methods; high per-iteration cost could limit practical utility.  
- Experiments need more detail on dataset/graph characteristics (sparsity, degree distribution) and how these affect results.  
- Clarify how convergence bounds scale with problem size (number of nodes, data dimension) and how many iterations are needed for target accuracy.  
- For the asynchronous algorithm, explicitly state asynchrony-model assumptions (e.g., bounded delays, i.i.d. delays).  
- Improve notation readability by defining variables explicitly when first introduced.  
- Add brief discussion of related work on decentralized U-statistic estimation for context.  
- Overall: solid contribution likely suitable for NIPS if runtime and theoretical-scaling concerns are addressed.
