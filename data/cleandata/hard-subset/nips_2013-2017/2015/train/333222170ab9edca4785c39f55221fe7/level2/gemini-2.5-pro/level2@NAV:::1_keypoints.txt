- Proposes "on-the-job learning": deploy high-accuracy systems from zero labeled data using real-time crowdsourcing.  
- Leverages acquired crowd labels to train a model that becomes increasingly self-sufficient.  
- Formalizes the problem as a stochastic game and uses Bayesian decision theory to define a utility balancing accuracy, cost, and latency.  
- Optimal policy is intractable; authors propose an approximation using Monte Carlo Tree Search.  
- Clearly distinguishes this work from online learning and active learning.  
- High-quality, significant problem formulation with strong practical impact.  
- Technically sound solution with strong empirical evaluation across three tasks.  
- Demonstrates order-of-magnitude cost reduction on NER while achieving super-human performance.  
- Paper is clearly written, well-organized, and provides open-source code and reproducible experiments.  
- Minor weakness: environment model is simplistic, assuming fixed worker accuracy.  
- Could be improved by learning worker reliability and latency over time.  
- Would benefit from sensitivity analysis of the utility function's cost and time weights.  
- Overall judgment: clear accept — strengths substantially outweigh minor limitations.
