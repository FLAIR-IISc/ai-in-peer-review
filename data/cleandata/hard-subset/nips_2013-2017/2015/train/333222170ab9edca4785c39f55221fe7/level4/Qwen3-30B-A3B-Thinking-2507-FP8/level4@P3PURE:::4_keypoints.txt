- Proposed "on-the-job learning" scenario is interesting, but its application to the face recognition experiment is ambiguous (unclear whether queries target only the instance).  
- Algorithm behavior is clear for structured prediction (can query sequence elements), but non-structured use lacks clarity and propagation between adjacent examples.  
- In the non-structured setting the utility function appears ill-defined because the model does not propagate information across examples.  
- The threshold baseline uses two unexplained parameters; the choice and sensitivity of these values are not justified and require analysis.  
- A baseline using uncertainty sampling (standard in active learning) is missing and would be a more meaningful comparison.  
- The authors’ claim of novelty over active online learning is weak: standard active learning can be adapted to this domain (treat sequence as active set and update on responses), which undermines the novelty argument.  
- Overall: framework seems potentially novel but suffers from unclear implementation details and limited applicability.
