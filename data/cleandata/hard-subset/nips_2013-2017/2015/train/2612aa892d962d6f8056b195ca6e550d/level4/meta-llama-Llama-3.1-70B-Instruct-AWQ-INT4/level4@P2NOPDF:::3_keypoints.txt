- Proposal: train a neural network offline to mimic Contact-Invariant Optimization (CIO) outputs, alternating CIO and NN optimization with noise injection (data augmentation + additive hidden-layer noise).  
- Empirical claim: network yields realistic, stable real-time control policies across diverse character models.  
- Novelty: joint training with CIO and noise injection are significant contributions that improve generalization.  
- Dataset limitation: base training used only 100–200 trials with 5 branched segments, likely causing overfitting in the no-noise variant; larger dataset needed.  
- Weak comparative evaluation: only a brief comparison to MPC; lacks systematic comparison between direct methods and trajectory-optimization-based methods.  
- Math and notation: explanations generally clear but some points are superficial and Section 3.2/time index t notation is confusing and needs clarification and extra references.  
- Reproducibility: authors should release code for community validation.  
- Performance profiling: real-time computational time measurements are missing and necessary (potential bottleneck for applications like games).  
- Terminology fixes: clarify “biped location” as locomotion; change “unity” to “unite/unify.”  
- Target specification: current target appears to be only x,y,z positions—consider adding target orientation (angle).  
- Terminology precision: “noise injection” is misleading if it refers to data augmentation rather than denoising autoencoders.  
- Algorithm clarity: clarify initial network weights θ in Alg.1 and justify using the same hyperparameter η in both Alg.1 steps.  
- Minor issues: numerous small errors (missing articles, incorrect notation) throughout the paper.  
- Overall assessment: promising application of NNs to character control, but practical benefits and broader applicability remain to be established.
