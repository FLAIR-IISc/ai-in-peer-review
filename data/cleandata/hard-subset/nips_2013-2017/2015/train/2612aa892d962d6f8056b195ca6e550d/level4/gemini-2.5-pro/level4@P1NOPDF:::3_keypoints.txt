- Proposes an NN trained offline to reproduce Contact-Invariant Optimization (CIO) planner output for real-time 3D character control.
- Key contribution: interleaved/alternating joint optimization where CIO and NN training inform each other.
- Uses noise via data augmentation (input perturbations) and additive hidden-layer noise to improve robustness.
- Experiments show realistic, stable control policies across diverse character models.
- Joint optimization and noise injection are critical for generalization and avoiding overfitting.
- Training data is limited (100–200 trials with five branched segments), weakening validation.
- “No noise” ablation likely overfits due to limited data; “no joint” variant might improve with more varied data.
- Comparative evaluation is weak—only a brief MPC comparison and missing reference.
- Some mathematical components (especially CIO step and Equation 6) are under-specified and hard to reproduce.
- Section 3.2 notation/procedure is confusing (time index t dropped), hindering clarity.
- Reproducibility would benefit from releasing source code and fuller implementation details.
- No discussion of computational performance or latency scaling with model complexity, important for practical use.
- Minor wording/typo issues (e.g., “biped location” vs locomotion; “unity” vs unite/unify; missing articles).
- Clarify whether control targets include orientation or only XYZ coordinates.
- Terminology: calling data perturbation “noise injection” is misleading—should distinguish data augmentation from denoising autoencoders.

- Justify using the same hyperparameter η for both optimization steps in Algorithm 1.
