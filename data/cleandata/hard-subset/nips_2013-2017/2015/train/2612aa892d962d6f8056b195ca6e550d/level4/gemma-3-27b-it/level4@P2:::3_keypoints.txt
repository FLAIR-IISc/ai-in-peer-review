- Proposes training a neural network offline to reproduce Contact-Invariant Optimization (CIO) outputs for real-time 3D model control.  
- Key contribution is an alternating training loop where CIO and NN iteratively refine each other’s outputs.  
- Incorporates noise injection via input data augmentation and hidden-layer noise to improve robustness.  
- Demonstrates stable, realistic control policies for diverse character models using a single network architecture.  
- Joint training plus noise injection notably enhances generalization across character morphologies.  
- Dataset size is small (≈100–200 trials with 5 branches each), risking overfitting and limited generality.  
- Overfitting observed in the “no noise” variant and limited performance in the “no joint” variant likely stem from dataset size.  
- Comparative evaluation is weak: only a brief, uncited comparison to MPC; needs broader benchmarking against prior methods.  
- Mathematical descriptions are mostly clear but omit some notation/details (e.g., Eq. 6) requiring readers to consult cited works.  
- Section 3.2 on optimal feedback gains for noisy inputs is hard to follow; notation and dropped time indices are confusing.  
- Reproducibility is hindered by missing code and unclear algorithmic details (weight initialization, single hyperparameter η).  
- Lacks reported computation times for real-time control, which is important for practical deployment (e.g., games).  
- Minor issues: terminology and word-choice errors (“biped location,” “unity”), misuse of “noise injection” for augmentation, typos and grammar.  
- Suggestion: include angular target constraints in addition to x,y,z to potentially improve control performance.  
- Arguments for acceptance: novel CIO+NN combination, single approach for varied morphologies, joint training and noise injection advance generalization, good application potential.  
- Arguments against acceptance: limited experiments/comparisons, small dataset, unclear math, no code, and unreported computational cost.  
- Overall: promising and valuable contribution but needs stronger experiments and clearer presentation; recommend borderline accept contingent on revisions.
