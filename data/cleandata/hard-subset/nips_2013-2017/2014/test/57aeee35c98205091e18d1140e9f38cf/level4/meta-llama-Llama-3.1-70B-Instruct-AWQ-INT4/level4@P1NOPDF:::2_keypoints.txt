- Proposes a method to bound the average causal effect (ACE) using inverse Entner's Rule 1, relaxed faithfulness (allowing some path cancellation), and inequality-based assumptions.
- Develops a linear programming computational scheme to compute the ACE bounds.
- Algorithm unoptimized but produces notably narrow and accurate bounds versus state-of-the-art.
- Some criteria used to define the bounds are not comprehensively explained.
- The validity of applying Entner's Rule 1 in the inverse direction is not clearly justified; the paper should explicitly assume and discuss the Fig. 2 structure.
- No general principle is provided for choosing relaxation parameters, though these strongly affect results.
- Theoretical exposition is mostly clear except for the issues above.
- Experimental presentation and implementation details are inadequately described; parameter definitions in Table 1 are unclear.
- Contains typos and incorrect table references (missing algorithm table and erroneous "Table 5" citation).
- Originality is limited by prior work, but the unified combination of the three ideas has some novelty.
- Significance is promising for tighter ACE bounds, but experiments are limited and theoretical/general validation is insufficient.
