- Proposes a new ACE-bounding method using inverse Entner's Rule 1, relaxed faithfulness (allowing path cancellation), and inequality constraints solved by linear programming.
- Preliminary results indicate much narrower and more accurate ACE bounds than current methods, though the algorithm is not fully optimized.
- Strength: novel and compelling synthesis of several principles into a unified computational framework.
- Critical theoretical gap: the inverse application of Entner's Rule 1 lacks clear justification and likely requires explicit structural assumptions.
- Missing rigorous argument showing when the inverse rule is valid; authors should state necessary assumptions and proofs.
- Parameter selection problem: no principled guidance or criteria for choosing relaxation parameters that control path cancellation.
- Sensitivity analysis on those parameters is missing but necessary to assess credibility of bounds.
- Experimental reporting is inadequate: implementation details are insufficient for replication.
- Table/parameter clarity issues: parameters in Table 1 are not well defined, making results hard to interpret.
- Typographical and referencing errors (e.g., missing algorithm table, incorrect table numbers) reduce professionalism and clarity.
- Originality: problem is standard, but novelty lies in the specific combination of the three ideas; validity of that combination remains unestablished.
- Significance: promising for practical causal quantification but currently limited by narrow experiments and lack of theoretical/general empirical validation.
- Recommendation: strengthen theoretical foundations, provide principled parameter selection and sensitivity analysis, and improve experimental description and proofreading.
