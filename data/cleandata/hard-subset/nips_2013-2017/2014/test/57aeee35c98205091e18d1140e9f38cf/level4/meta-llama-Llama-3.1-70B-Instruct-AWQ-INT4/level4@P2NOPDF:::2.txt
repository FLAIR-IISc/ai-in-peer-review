This paper proposes a novel approach to derive bounds on the average causal effect (ACE) from variable X to variable Y. The method's core principles include applying Entner's Rule 1 in its inverse direction, relaxing the faithfulness assumption to allow for path cancellation to some degree, and introducing assumptions represented by inequality constraints. The authors formulate a computational scheme for the ACE bound using linear programming under these principles. Although the algorithm has not been optimized, the resulting bounds appear narrow and accurate compared to state-of-the-art approaches.

Quality:
The authors have established a new method to derive more accurate ACE bounds than existing state-of-the-art methods. By combining several feasible principles, they have successfully provided tighter bounds, making the paper's content very interesting. However, some criteria defining the bounds are not comprehensively explained. Specifically, the validity of using Entner's Rule 1 in the inverse direction is not clearly discussed. To justify this use, the authors should explicitly assume the structure depicted in Fig. 2, which requires clear explanation and discussion.

Furthermore, the paper lacks generic criteria for choosing appropriate parameters for relaxation, which significantly affect the bounds' results. The principle for determining these parameters should be more thoroughly investigated and included in the work.

Clarity:
The paper's theoretical explanation is generally comprehensive, except for the essential discussion mentioned above. However, the demonstration of experimental results is not well described. The actual implementation of the algorithm is not comprehensively explained, and the definition of parameters shown in Table 1 (or Table 5) is not clearly provided. Additionally, there are some typos, such as "The full algorithm is shown in Table 1" on page 6, where no table of the algorithm is present, and "Results are summarized in Table 5" on page 8, where Table 5 is not found (possibly referring to Table 1).

Originality:
While the problem setting addressed in this paper has been explored in earlier work, as acknowledged by the authors, the scope of this paper does not exhibit high originality. However, combining the three ideas into a unified framework contains some originality. Nevertheless, the validity of this combination is not clearly discussed, as mentioned earlier.

Significance:
This method shows some significance in deriving tighter bounds of ACE. However, its performance has been evaluated through limited experiments, and its generality has not been strictly assessed in both theory and extensive experiments. The paper addresses an important problem – evaluating the bound of the strength of causal relations – but the validity of the presented idea should be more thoroughly described in both theory and experiments.
