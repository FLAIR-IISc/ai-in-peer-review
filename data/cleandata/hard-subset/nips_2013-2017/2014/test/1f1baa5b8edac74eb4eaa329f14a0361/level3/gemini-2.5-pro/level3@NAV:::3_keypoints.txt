- Proposes a random forest ZSL method that explicitly models attribute classifier unreliability.  
- Trains SVMs per attribute and builds a random forest per unseen class using class-attribute signatures.  
- Leverages an attribute-labeled validation set propagated down trees to estimate node-specific, ROC-based reliability statistics.  
- Integrates reliability estimation into split selection, favoring attributes that are both discriminative and reliably predictable.  
- Captures joint attribute error statistics during tree construction, unlike prior two-stage approaches (e.g., DAP).  
- Paper is exceptionally well-written, clear, and provides a good summary of related work.  
- Novelty is modest; claim of unique handling of unreliable attributes is overstated.  
- Contribution is better framed as an improved technique for modeling attribute noise and correlations, not a new ZSL paradigm.  
- Reliance on recursively subdividing the validation set risks sparse data and unstable error estimates in deeper tree nodes.  
- Proposed heuristic fix for sparse-validation issues is insufficiently justified.  
- Experimental results show positive but only moderate gains over strong baselines.  
- Few-shot learning extension is underdeveloped and the information-gain combination appears ad-hoc.
