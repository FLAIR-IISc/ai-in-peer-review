- Proposes a random-forest-based zero-shot method that explicitly models attribute prediction unreliability using attribute signatures and a validation set, with a few-shot extension using category-labeled images.  
- Paper is well-written and provides a clear summary of related work.  
- Overall novelty is modest; the claim of uniquely handling unreliable attributes is overstated because prior methods (e.g., DAP) already address attribute uncertainty.  
- Key technical differences are ROC-based reliability estimates instead of Platt-sigmoids and propagating validation data down trees to capture joint attribute error statistics.  
- Method should be framed as improved modeling of attribute correlations/noise rather than a fundamentally new way to handle unreliability.  
- Requires a large validation set because data gets subdivided in trees; the proposed heuristic fix is insufficiently justified.  
- Few-shot extension is brief and less polished; the weighted information-gain combination (Eq. 6) is unintuitive compared to alternatives.  
- Experiments on three datasets are appropriate and support claims, but quantitative gains are modest and not decisive.  
- Technically sound and well-presented, but main contribution is weakened by similarities to prior work and experimental/validation shortcomings.  
- Recommendation: Accept with minor revisions.  
- Suggested minor revisions: clarify relationship to prior work, address validation-set-size concerns and few-shot section, and rephrase the claim about uniquely handling unreliable attributes.  
- Scores: Originality 6/10, Clarity 9/10, Significance 7/10, Quality 8/10.
