

# Review of "Zero-Shot Recognition with Unreliable Attributes"

## Summary
This paper addresses a critical gap in zero-shot learning: the assumption of reliable attribute predictions, which is often violated in practice. The authors propose a novel approach that accounts for attribute prediction unreliability by incorporating receiver operating characteristics (ROC) of attribute classifiers during random forest training. The method trains trees using class attribute signatures rather than image data, and propagates signatures through the tree according to estimated attribute classifier error rates. The authors also extend their approach to handle signature uncertainty and few-shot learning. Experiments on three standard datasets (AwA, aPY, SUN) show consistent improvements over strong baselines like DAP and SIGNATURE RF.

## Strengths

1. **Significant Problem Identification**: The paper correctly identifies a crucial limitation in existing zero-shot learning work—ignoring the unreliability of attribute predictions. This is a meaningful gap that has been overlooked in the literature.

2. **Technical Soundness**: The proposed method is theoretically well-founded, with clear explanations of how ROC characteristics are integrated into the random forest training process. The information gain formulation in Section 3.2.2 properly accounts for attribute unreliability.

3. **Thorough Experimental Validation**: The paper provides comprehensive experiments across three diverse datasets with controlled noise experiments, real-world attribute predictions, and a few-shot extension. The ablation studies (Table 1) clearly demonstrate the contribution of each component.

4. **Strong Baseline Comparisons**: The comparison with DAP (the leading attribute-based zero-shot method) is particularly convincing, showing significant improvements (e.g., 10.2% gain on "giant panda" in AwA) even when using only 80% of the attribute-labeled images.

5. **Practical Value**: The method's ability to work with discovered attributes (Section 4.1) demonstrates broader applicability beyond hand-designed attributes.

## Weaknesses

1. **Validation Set Limitation**: The paper relies on a validation set of attribute-labeled images to estimate ROC characteristics, but doesn't adequately address how to select this validation set to best represent the unseen classes. The authors acknowledge this limitation ("If that data perfectly represented the true attribute errors on images from the unseen classes..."), but the method's performance is fundamentally tied to the quality of this validation set, which is problematic in true zero-shot scenarios.

2. **Scalability Concerns**: The approach is evaluated on only 10-12 unseen classes. It's unclear how the method would scale to hundreds or thousands of unseen classes, which is a common scenario in practical zero-shot applications. The paper doesn't provide analysis or discussion of this important scalability aspect.

3. **Limited Comparison to Recent Work**: While the paper compares to DAP, it doesn't adequately address recent advances in zero-shot learning (e.g., the Devise model [4] or recent graph-based approaches [14, 26]) that might handle attribute uncertainty differently.

4. **Few-Shot Extension Insufficiently Explored**: The few-shot extension (Section 3.3) is promising but not thoroughly evaluated. The paper doesn't explore how the method performs with varying numbers of labeled examples across different attribute sets or how it compares to recent few-shot learning methods.

5. **Insufficient Discussion of Limitations**: The paper acknowledges the validation set limitation but doesn't provide concrete suggestions for mitigating it in practice, such as using transfer learning to estimate ROC characteristics from seen classes.

## Recommendations

1. **Address Validation Set Limitation**: The authors should conduct additional experiments to explore how to select the validation set most effectively. For instance, they could investigate whether focusing validation data on seen classes with attribute signatures most similar to the unseen classes improves performance.

2. **Expand Scalability Analysis**: The paper should include a scalability analysis showing how the method performs with increasing numbers of unseen classes (e.g., 50+ classes), which would be more representative of practical applications.

3. **Update Baseline Comparison**: Include comparisons with recent state-of-the-art zero-shot methods that handle semantic relationships or attribute uncertainty more directly.

4. **Clarify Few-Shot Extension**: Provide more detailed analysis of the few-shot extension, including how the optimal λ parameter varies across different attribute sets and how the method compares to dedicated few-shot learning approaches.

5. **Discuss Practical Implementation**: Add a section discussing practical considerations for implementing the method, particularly regarding the acquisition of the validation set for ROC estimation.

## Overall Assessment

This is a strong paper that addresses a meaningful gap in zero-shot learning. The proposed method is technically sound, well-motivated, and demonstrates significant improvements over existing approaches. While there are some limitations regarding the validation set dependency and scalability, these are common challenges in zero-shot learning that the authors have appropriately identified and partially addressed.

The paper makes a valuable contribution to the field by showing that zero-shot learning can be practical even with unreliable attribute predictions, which is a critical insight for real-world applications. The method's elegance and effectiveness make it a promising approach that could influence future work in zero-shot learning.

I recommend acceptance with minor revisions to address the scalability and validation set limitations more thoroughly. The paper is well-written, technically sound, and makes a significant contribution to the field.
