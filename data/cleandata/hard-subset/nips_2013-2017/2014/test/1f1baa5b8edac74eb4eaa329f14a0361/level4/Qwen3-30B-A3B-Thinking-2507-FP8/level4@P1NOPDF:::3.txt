

This paper proposes a method for attribute-based zero-shot learning using random forests. Attribute classifiers are trained via SVMs, and a random forest is built on class-attribute signature vectors. The approach uses a validation set of attribute-labeled images to estimate attribute reliability at each tree node, with an extension to few-shot learning also provided.  

Clarity: The paper is clear and well-written, with a strong literature review that effectively contextualizes the work.  

Originality: The novelty is modest but not negligible. The claim that the method uniquely handles unreliable attribute predictions—while prior work does not—is overstated. For instance, the baseline DAP method [7] (used widely in the field) shares the same foundational approach:  
(a) Training attribute classifiers on image features (identical to this work),  
(b) Using a validation set to model attribute reliability (via Platt scaling and probabilistic attribute combination in [6], versus the proposed random forest approach).  
Thus, the core strategy for addressing attribute uncertainty is not fundamentally new. The key distinction lies in *how* attribute noise is modeled:  
1. The proposed method uses an ROC-based reliability estimate (instead of a sigmoidal model like Platt scaling),  
2. It explicitly models joint attribute error statistics (unlike [6], which assumes attribute independence), as the validation set propagates down the tree structure.  
This suggests a more accurate framing: the method offers a better way to model attribute correlation (e.g., as in Scheirer et al.’s "Multi-Attribute Spaces," which could be cited).  

A significant concern is the need for a large validation set to maintain reliable reliability estimates. As the validation set subdivides through the tree structure, insufficient data could degrade performance. The paper glosses over this issue and proposes a heuristic solution (line 323) without justification or empirical validation. Additional experiments exploring the impact of training/validation set sizes would strengthen the argument.  

Quality: The technical execution is sound, with well-motivated choices and clear descriptions. Experiments support the paper’s claims, though the quantitative gains are modest. The few-shot extension (Section 3.3) feels underdeveloped relative to the rest of the paper. While the main sections provide thorough explanations, this section is brief and lacks intuitive grounding. Specifically, a method combining class-attribute priors with image-based attribute estimates (to compute p(a|...)) would be more natural than the weighted combination of information gains in Equation 6.  

In summary, the paper presents a technically competent approach with a plausible refinement for modeling attribute correlation. However, the claimed novelty regarding attribute uncertainty handling is overstated, and the validation set size challenge remains inadequately addressed. The few-shot extension requires deeper elaboration. While the work is well-executed and the experiments are appropriate, the marginal improvements and unresolved methodological gaps make it difficult to justify acceptance solely on empirical merit. The primary contribution—better modeling of attribute correlations—deserves clearer articulation and stronger justification.
