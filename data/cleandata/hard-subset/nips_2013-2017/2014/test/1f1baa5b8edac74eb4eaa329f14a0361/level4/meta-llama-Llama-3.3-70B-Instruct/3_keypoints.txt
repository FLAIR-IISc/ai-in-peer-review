- Proposes attribute-based zero-shot learning using SVM-trained attribute classifiers and a random forest on class-attribute signature vectors, with attribute reliability estimated per node via a validation set; also extends to few-shot learning.
- Manuscript is well-structured, clearly written, and provides a comprehensive related-work summary.
- Novelty is limited; some novel elements exist but the central claim of uniquely addressing unreliable attribute predictions is overstated.
- The approach shares core steps with prior methods (e.g., DAP): train attribute classifiers, then train class probability estimator using validation-based attribute reliability.
- Key methodological difference: models attribute noise via ROC-based estimates and captures joint attribute-error statistics through tree propagation, unlike independent-attribute treatments.
- Potential drawback: requires a large validation set for reliable probability estimates, especially as the set is subdivided during tree propagation; this limitation is not thoroughly addressed and proposed fixes seem heuristic.
- Experiments are appropriate and technically sound but yield only modest quantitative improvements insufficient alone to justify acceptance.
- Few-shot extension is interesting but underdeveloped and less clearly explained than the main method.
- Suggests exploring alternatives (e.g., combining class-attribute priors with image attribute estimates) and conducting experiments on training/validation set size effects.
