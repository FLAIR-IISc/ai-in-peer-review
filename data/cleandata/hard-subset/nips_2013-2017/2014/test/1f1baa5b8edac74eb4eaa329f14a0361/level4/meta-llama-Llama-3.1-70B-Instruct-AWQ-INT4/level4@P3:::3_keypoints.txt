- Paper proposes attribute-based zero-shot learning using random forests; attribute classifiers trained with SVMs and a validation set is used to estimate attribute reliability at each tree node.  
- Extension to few-shot learning is proposed via a weighted combination of information gains.  
- Paper is well-written with a clear explanation and good related work summary.  
- Experimental results on three challenging datasets demonstrate effectiveness.  
- Main novelty is modest but includes a more complex model for attribute noise.  
- Key technical contribution: ROC-based reliability estimation (vs. Platt scaling) and modeling joint attribute error statistics by propagating validation data down trees.  
- Claim that earlier work does not handle unreliable attribute predictions is overstated—baselines like DAP also handle uncertainty to some extent.  
- Potential weakness: validation set must be large to obtain reliable probability estimates as it gets subdivided in trees.  
- Authors propose heuristics to address small validation subsets, but lack experiments evaluating training/validation size effects.  
- Few-shot extension feels less polished; rationale for weighted information gain over alternatives is unclear.  
- Recommendation: Accept with minor revisions; address experiments on validation/training size and clarify few-shot method.  
- Reviewer score: 7/10.
