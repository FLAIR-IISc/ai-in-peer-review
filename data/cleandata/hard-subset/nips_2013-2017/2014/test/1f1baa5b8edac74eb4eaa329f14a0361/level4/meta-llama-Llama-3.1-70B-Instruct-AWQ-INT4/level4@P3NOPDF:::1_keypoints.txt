- Bridges theory and practice of attribute-based zero-shot learning, noting attribute classifiers can be as hard to learn as object classes.  
- Proposes using random forests to predict unseen classes from attribute vectors.  
- Training incorporates attribute detector reliability by propagating a validation set through each tree.  
- Extends the method to handle few-shot training examples for test categories.  
- Achieves state-of-the-art results on multiple attribute datasets.  
- Presents an elegant, theoretically sound random forest framework for the zero-shot problem.  
- Conducts thorough experiments: ablations, artificial noise, zero- vs few-shot settings, and literature comparisons.  
- Paper is well-written and easy to follow.  
- Minor clarity issue: threshold t in Section 3.2.1 is unclear for binary attribute signatures (later clarified).  
- Originality is moderate (random forest application not entirely novel) but the contribution is significant and impactful.
