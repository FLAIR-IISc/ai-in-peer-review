This paper introduces a random forest-based method for attribute-based zero-shot learning, building upon initial attribute classifiers trained with SVMs. A random forest is constructed using class-attribute signature vectors, with attribute reliability estimated at each node via a validation set of images possessing attribute labels. The work also extends to the few-shot learning setting.

The paper is exceptionally clear and provides a strong summary of relevant prior work. While the novelty is present, it is somewhat incremental. The claim of fundamentally addressing unreliable attribute predictions in a way distinct from existing methods appears overstated. Specifically, the baseline Direct Attribute Prediction (DAP) method – and others like it – already incorporate mechanisms for handling attribute uncertainty. Both DAP and the proposed approach share a common framework: training attribute classifiers on image features, followed by learning a class probability estimator based on these attribute predictions, utilizing a validation set to assess attribute classifier reliability (Platt scaling and probabilistic models in [6], versus the random forest approach here). 

The contribution lies in a more sophisticated model for attribute noise: utilizing an ROC-based reliability estimate instead of a sigmoidal model, and effectively modeling joint attribute error statistics through validation set propagation within the decision trees. Framing the work as a refined method for modeling attribute correlations may be more accurate; related work such as Scheirer et al. "Multi-Attribute Spaces: Calibration for Attribute Fusion and Similarity Search" could be considered.

A potential limitation is the need for a substantial validation set to ensure robust probability estimates as data is subdivided during tree construction. This concern is briefly addressed in the paper (line 323) with a heuristic approach, but further investigation into the impact of training and validation set size would be beneficial. 

Overall, the paper is technically sound, with well-justified design choices and appropriate experiments that generally support the claims. However, the quantitative improvements are not substantial enough for acceptance based on empirical results alone. The few-shot learning extension (Section 3.3) feels comparatively underdeveloped, lacking the detailed explanation present elsewhere. A method integrating class-attribute priors and image-based attribute estimates for probability estimation might be more intuitive than the current information gain weighting (Eq. 6). In conclusion, this is a well-written and technically competent paper, but the central argument regarding a novel approach to handling unreliable attributes is debatable.
