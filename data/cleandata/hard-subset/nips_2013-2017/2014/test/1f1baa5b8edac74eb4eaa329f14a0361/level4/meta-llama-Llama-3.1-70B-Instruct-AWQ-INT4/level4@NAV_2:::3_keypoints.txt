- Introduces attribute-based zero-shot learning using random forests: SVM-trained attribute classifiers, random forests on class-attribute signatures, and a validation set to estimate attribute reliability; includes a few-shot extension.

- Paper is very clear, well-written, logically organized, and summarizes related work effectively.

- Novelty is somewhat low but not negligible.

- Claim that the method uniquely handles unreliable attribute predictions is overstated/misleading.

- Prior methods (e.g., DAP) already address attribute uncertainty using validation-based calibration (e.g., Platt scaling); the overall pipeline is similar.

- Key differences: uses ROC-based reliability estimates instead of sigmoidal models and implicitly models joint attribute error statistics via tree propagation.

- Recommend citing related calibration work (e.g., Scheirer et al.) to situate contributions on attribute correlation modeling.

- Potential weakness: needs a large validation set because samples get subdivided down trees; current remedy is heuristic and under-justified.

- Paper lacks experiments studying the effect of training/validation set size on performance.

- Experiments are appropriate and support claims, but quantitative improvements are modest.

- Few-shot learning extension is underdeveloped and less polished than the main contribution.

- Suggestion: combining class-attribute priors and image attribute estimates directly might be more intuitive than the proposed weighted information-gain approach.

- Overall verdict: technically sound and well-presented, but the main selling point about handling unreliable attributes is questionable.
