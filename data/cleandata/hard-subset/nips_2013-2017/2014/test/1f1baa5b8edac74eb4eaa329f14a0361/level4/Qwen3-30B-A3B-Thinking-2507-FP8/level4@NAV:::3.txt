

This paper introduces a method for attribute-based zero-shot learning leveraging random forests. Attribute classifiers are initially trained via SVMs, and a random forest is subsequently trained on class-attribute signature vectors. Attribute reliability at each tree node is estimated using a validation set of images with attribute-labels, and an extension to few-shot learning is provided.  
Clarity: The paper is exceptionally clear and well-written, with a strong summary of related work.  
Originality: The novelty is modest but acceptable for acceptance. However, the claim that the proposed method uniquely handles unreliable attribute predictions—while prior work does not—is overstated. For instance, the baseline DAP method [7] (used in this paper and others) shares similar capabilities for managing attribute uncertainty. Both approaches:  
a) Train attribute classifiers on image features (identical setup),  
b) Employ a validation set to estimate attribute classifier reliability for a second-stage class probability estimator (where [6] uses Platt scaling and probabilistic attribute combination, while the proposed method uses a random forest-based approach).  
Thus, the core methodology for handling attribute uncertainty is not fundamentally novel. The distinction lies in the modeling of attribute noise: (1) the paper uses ROC-based reliability estimation instead of sigmoidal models like Platt scaling, and (2) it models joint attribute error statistics via tree propagation (unlike [6], which assumes attribute independence). A more accurate description would frame this as an improved method for modeling attribute correlation (e.g., citing Scheirer et al.’s "Multi-Attribute Spaces" for related work). Additionally, the proposed approach likely requires a large validation set to ensure reliable probability estimates when validation data is subdivided across tree nodes—a critical issue glossed over in the paper. The heuristic solution at line 323 lacks empirical justification, and further experiments on dataset size effects would strengthen the claim.  
Quality: The paper is well-written, technically sound, and experiments appropriately support its claims. However, the quantitative improvements are marginal, and the few-shot extension (Section 3.3) is underdeveloped compared to the rest of the paper. The brief treatment of this section—contrasting with the thorough explanations elsewhere—feels inconsistent. The proposed weighted combination of information gains (Eq. 6) for integrating class-attribute priors and image estimates appears less intuitive than a direct probability combination approach. Overall, the paper is technically competent with solid experiments, but its primary contribution as a novel solution for unreliable attributes is questionable.
