This paper attempts to bridge the gap between the theory and practice of feature-based zero-shot learning. The theory is that new classes can be automatically identified using pre-trained feature predictors; however, in practice, learning these feature classifiers can be as difficult or even more difficult than learning object classes.

A random forest is trained to predict unseen classes from a feature vector, and the training process takes into account the reliability of feature detectors by propagating a validation set through each decision tree during training. The authors have shown how the method can be extended to handle training with some training examples of test categories. This method achieves state-of-the-art results on several feature datasets.

Quality: The authors have done a really good job of handling this fundamental zero-shot learning problem using a random forest framework. The model is elegant and theoretically sound. The results on 3 standard datasets are robust. The authors have done a good job of conducting ablation studies, introducing artificial noise, evaluating multiple settings (zero- versus few-shot learning), and comparing with the literature.

Clarity: This paper is very well written. There was a confusion in section 3.2.1: when the threshold t is introduced, the feature signature is still binary, so without changing anything in equation (2), t can be any value 0 < t < 1. Then, in lines 207-212, it is not clear how a new test example can be propagated in the tree, as t seems undefined. This is clarified in later sections.

Originality: This is a good application of the random forest framework to an important problem.

Significance: This paper solves a fundamental problem in zero-shot learning.

This paper is robust, interesting, and solid. The results on 3 datasets in different settings are convincing. This paper is worthy of publication.
