This paper attempts to bridge the gap between the theory and practice of attribute-based zero-shot learning. The theory states that new classes can be automatically identified using previously trained attribute predictors; however, in reality, these attribute classifiers can be as difficult or even more difficult to learn than learning object classes.

Random forests are trained to predict unseen classes from attribute vectors, and the reliability of attribute detectors is considered in the training process. During training, a validation set is propagated through each decision true. The authors demonstrate how this method can be extended to match training examples in several test categories. This method achieves state-of-the-art results in several attribute datasets.

Quality: The authors have handled this fundamental zero-shot learning problem very well using the Random Forest framework. The model is elegant and theoretically sound. The results for 3 standard datasets are robust. The authors have done an excellent job of conducting ablation studies, introducing artificial noise, evaluating several settings (zero- versus few-shot learning), and comparing with literature.

Clarity: The paper is written very well. There was one confusion in section 3.2.1: when the threshold t is introduced, the attribute signatures are binary, so t can be any value 0 < t < 1, which can be done without any change to equation (2). Then, in lines 207-212, it is not clear how a new test example can be propagated below true, since t seems to be undefined. This has been clarified in subsequent sections.

Originality: This is an excellent application of the Random Forest framework for an important problem.

Significance: The paper discusses a fundamental problem of zero-shot learning.

The paper is robust, interesting, and accurate. The results for 3 datasets in different settings are credible. The paper is worthy of publication.
