

This paper proposes an attribute-based zero-shot learning method using random forests. It first trains attribute classifiers via SVMs, then constructs a random forest on class-attribute signature vectors, leveraging a validation set with attribute-labels to estimate attribute reliability at each tree node. An extension to few-shot learning is also presented.  

Clarity: The paper is exceptionally clear and well-written, with a thorough summary of related work.  

Originality: The novelty is modest but not insufficient for acceptance. However, the claim that the proposed approach uniquely handles unreliable attribute predictions—while earlier work does not—is overstated. For instance, the baseline method Direct Attribute Prediction (DAP) [7], used in this paper and others, shares similar capabilities for addressing attribute uncertainty. Both approaches follow the same core workflow:  
(a) Train attribute classifiers on image features (identical for both),  
(b) Train a second-stage class probability estimator using attribute predictions, with a validation set to estimate classifier reliability ([6] employs Platt scaling and probabilistic attribute combination, while this work uses a random forest-based method).  
Thus, the approach is not fundamentally new in handling attribute uncertainty. Its distinction lies in two aspects: (1) using an ROC-based reliability estimate instead of a sigmoidal model like Platt scaling, and (2) modeling joint attribute error statistics via tree propagation (unlike [6], which assumes attribute independence). Consequently, a more precise characterization would frame the contribution as an improved method for modeling attribute correlation (Scheirer et al.’s "Multi-Attribute Spaces" [8] offers relevant context for this direction).  
A notable weakness is the requirement for a large validation set to reliably estimate probabilities at tree nodes, as the validation data is recursively subdivided during tree propagation. The paper glosses over this issue; the proposed solution (line 323) appears heuristic, and more rigorous experiments on the impact of training/validation set sizes would strengthen the argument.  

Quality: The paper is technically sound, with well-motivated choices and clear exposition. Experiments appropriately support the claims, though the magnitude of quantitative gains is modest. The few-shot learning extension (Section 3.3) lacks the polish of the main framework: it is overly concise compared to the detailed rest of the paper. The proposed weighted combination of information gains (Eq. 6) for integrating class-attribute priors and image attribute estimates feels less intuitive than a probabilistic fusion of \( p(a \mid \cdot) \). Overall, this is a well-written paper with sensible technical decisions and adequate experiments, but its core contribution—positioned as a novel solution for unreliable attributes—is questionable.
