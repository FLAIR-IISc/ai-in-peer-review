This paper presents a method of attribute-based zero-shot learning using a random forest. Attribute classifiers are first trained using SVMs. A random forest is trained on the attribute signature vectors of the class using a validation set of attribute-labeled images, and used to estimate the reliability of the attributes at each node of the tree. An extension of multi-shot learning is also proposed.

Clarity: This paper is very clear and well-written. It provides a good summary of related work.

Novelty: The novelty of this paper is somewhat reduced, but it is not too little to be accepted. One problem is that the proposed method works with flawed attribute prediction, which previous work did not. This claim is somewhat exaggerated or misleading. For example, Direct Attribute Prediction [7] (DAP) – the baseline method used in this paper and many others – has some similar capabilities to address attribute uncertainty. Both methods use basic methods:
a) Attribute classifier training on image features (identical for both papers)
b) Second-class probability estimation as a function of attribute prediction, using a validation set to estimate the reliability of attribute classifiers ([6] platform scaling and a probabilistic model to integrate attributes, where the proposed method uses a new random forest-based method)

Therefore, I do not consider the proposed method to be fundamentally a new way to address attribute uncertainty. What is different from previous work is that it uses a more complex model for attribute noise modeling: 1) it uses an ROC-based estimator instead of a sigmoid-like model like platform scaling, and 2) while [6] considers attributes as independent, the proposed method effectively models the joint statistics within the attribute errors, since the validation set extends below the tree. Therefore, I think it would be more appropriate to describe the method as a good way to model attribute relationships (I do not know this field well, but Scheirer et al. "Multi-Attribute Space: Calibration for Attribute Fusion and Similarity Search" may be a related paper that could be cited).

At the same time, a potential weakness of the proposed method compared to methods that consider attributes as independent is that a larger validation set will be required to obtain a valid probability estimate, as it becomes subdivided as it extends below the decision tree. This problem is avoided in the paper, and the proposed method to address it (line 323) seems heuristic. It would have been good to see more tests evaluating the impact of the size of the training and validation sets.

Quality: In general, the quality of the paper is good, and the technical choices and descriptions are meaningful. The tests are appropriate and support the paper's claims, although the level of quantitative improvement is not sufficient for the paper to be fully accepted based on empirical results. The extension of multi-shot learning (section 3.3) seems less refined than the rest of the paper. Although the rest of the paper is clear and the explanations are long, this section was very brief. To me, a method that can combine class-attribute prior probabilities and image attribute estimates when estimating probability p(a|...) may seem more intuitive than a weighted combination of information gain (equation 6). It is a well-written paper, where the technical decisions are meaningful and there are sufficient tests; however, the paper's main attraction as a new way to address flawed attributes is somewhat questionable.
