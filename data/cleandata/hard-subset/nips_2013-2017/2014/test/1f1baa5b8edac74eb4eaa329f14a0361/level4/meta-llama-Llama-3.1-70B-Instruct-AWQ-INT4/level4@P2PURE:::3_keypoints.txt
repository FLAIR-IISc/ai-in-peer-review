- Proposes attribute-based zero-shot learning using SVM-trained attribute classifiers and a random forest trained on class-attribute signature vectors, with a validation set to estimate attribute reliability; also extends to few-shot learning.

- Paper is well-written, clear, and summarizes related work effectively.

- Novelty is limited; claim that this uniquely handles unreliable attribute predictions is overstated because prior methods (e.g., DAP) also address attribute uncertainty.

- Both this work and prior methods follow the same two-stage approach: train attribute classifiers, then train a class estimator using validation-based reliability estimates.

- Key technical differences: this method uses ROC-based reliability estimates and models joint attribute-error statistics via tree propagation, rather than Platt-scaling and independent-attribute assumptions.

- The approach should be characterized as improved modeling of attribute correlation rather than a fundamentally new solution to unreliable attributes; suggest citing Scheirer et al. "Multi-Attribute Spaces..."

- Potential weakness: requires a large validation set because validation data is subdivided across tree nodes; the proposed remedy is heuristic and not thoroughly evaluated.

- Experiments are suitable and support claims but quantitative gains are modest and not sufficient alone for strong acceptance.

- Few-shot extension (Section 3.3) is underdeveloped and less polished; combining class-attribute priors with image attribute estimates might be more intuitive than the proposed weighted information-gain approach.

- Overall: technically sound and well-presented, but the main selling point—novel handling of unreliable attributes—is questionable.
