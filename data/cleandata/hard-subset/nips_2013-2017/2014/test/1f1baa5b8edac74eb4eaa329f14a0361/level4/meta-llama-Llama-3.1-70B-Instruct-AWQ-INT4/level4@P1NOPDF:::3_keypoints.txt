- Proposes attribute-based zero-shot learning using SVM-trained attribute classifiers and random forests trained on class-attribute signature vectors, with a validation set to estimate attribute reliability per tree node; includes a few-shot extension.
- Paper is well-written, clear, and provides a good summary of related work.
- Novelty is modest — interesting but not groundbreaking.
- Claim that the method uniquely handles unreliable attribute predictions is overstated or misleading.
- Direct Attribute Prediction (DAP) and related prior work already address attribute uncertainty using similar two-stage approaches (train attribute classifiers, then a class estimator using validation data).
- Main technical differences are using an ROC-based estimate instead of Platt scaling and implicitly modeling joint attribute error statistics via validation data propagation in trees.
- The method is better characterized as improved modeling of attribute correlations/noise rather than a fundamentally new way to handle unreliability; additional relevant citations (e.g., Scheirer et al.) could be added.
- Potential weakness: method may require a large validation set because validation data is subdivided down trees, and the proposed heuristic fix is under-motivated.
- The paper should include experiments analyzing the effect of training and validation set sizes.
- Technical choices and experiments are sensible and support claims, but quantitative gains are modest and not sufficient alone to justify acceptance.
- The few-shot learning extension (Section 3.3) is underdeveloped and brief; a probabilistic combination of class-attribute priors and image attribute estimates might be more intuitive than the weighted information-gain approach (Eq. 6).
- Overall: solid, well-written work with adequate experiments, but the main selling point about handling unreliable attributes is questionable.
