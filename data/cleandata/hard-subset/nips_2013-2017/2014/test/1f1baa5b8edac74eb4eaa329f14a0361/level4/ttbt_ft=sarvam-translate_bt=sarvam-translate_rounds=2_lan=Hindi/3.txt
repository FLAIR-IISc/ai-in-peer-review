This paper presents a method for feature-based zero-shot learning using random forests. A feature classifier is first trained using an SVM. A random forest is trained on a class-feature signature vector, while feature reliability is estimated at each node of the tree using a validation set of feature-labeled images. An extension of zero-shot learning is also proposed.

Clarity: This paper is very clear and well-written. It does a good job of summarizing the related work.

Novelty: The novelty of this paper is somewhat low, but not too low to accept. One issue is that the proposed approach deals with unreliable feature predictions, whereas earlier work did not, which is a bit exaggerated or misleading. For example, direct attribute prediction [7] (DAP) – the underlying method used in this paper and many others – has some similar capabilities to deal with feature uncertainty. Both approaches use the basic approach:
a) Train a feature classifier on image features (same for both papers)
b) Train a second-order probability estimator as a function of feature predictions, using a validation set to estimate the reliability of the feature classifier ([6]; uses Platt scaling and a probabilistic model to combine features, while the proposed approach uses a new random forest-based method)

Thus, I do not find the proposed approach fundamentally new for handling feature uncertainty. What distinguishes it from earlier work is that it uses a more complex model to model feature noise: 1) it uses an ROC-based estimator instead of a sigmoid-like model like Platt scaling, and 2) while [6] considers features to be independent, the proposed approach effectively models the combined statistics in feature errors, as the validation set spreads down the trees. Thus, I think it would be a little more appropriate to describe the approach as a better way to model feature correlation (I am not well-versed in this area, but Scheirer et al. "Multi-Attribute Spaces: Calibration for Attribute Fusion and Similarity Search" may be a relevant paper that could be cited).

Furthermore, a potential weakness of the proposed approach, compared to methods that consider features to be independent, is that it seems a larger validation set would be needed to obtain valid probability estimates because it is sub-divided when it spreads down the decision trees. This issue was briefly shown in the paper, and the proposed approach to deal with it (line 323) seems self-evident. It would be good to see more experiments evaluating the impact of training and validation set sizes.

Quality: Generally, the quality of the paper is good, and the technical options and details are understandable. The experiments are appropriate and support the paper's claims, although the level of quantitative improvement is not sufficient to fully accept the paper based on empirical results. The extension of zero-shot learning (section 3.3) felt less refined compared to the rest of the paper. While the rest of the paper was clear and the explanation lengthy, this section was very concise. For me, a method that can combine class-feature predictors and image feature estimates when probabilities p(a|...) are estimated may have a more intuitive understanding than a weighted combination of information gains (equation 6). This is a well-written paper, with technical decisions that are understandable, and sufficient experiments; however, the paper's main selling point as a new way to handle unreliable features is somewhat questionable.
