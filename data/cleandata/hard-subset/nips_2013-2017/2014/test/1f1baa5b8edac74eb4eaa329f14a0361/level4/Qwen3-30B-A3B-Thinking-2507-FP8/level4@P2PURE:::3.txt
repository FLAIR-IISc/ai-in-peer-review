

This paper introduces an attribute-based zero-shot learning method utilizing random forests. Attribute classifiers are initially trained via SVMs. A random forest is then trained on class-attribute signature vectors, with attribute reliability estimated at each tree node using a validation set of images annotated with attribute-labels. An extension to few-shot learning is also presented.  

Clarity: The paper is exceptionally clear and well-written, providing a concise summary of related work.  

Originality: The novelty is modest but acceptable for acceptance. However, the claim that the proposed approach uniquely handles unreliable attribute predictions—while prior work does not—is overstated. For instance, the baseline method Direct Attribute Prediction (DAP) [7], widely adopted in this field, shares similar capabilities for managing attribute uncertainty. Both approaches share the same foundational steps:  
a) Training attribute classifiers on image features (identical across methods),  
b) Training a second-stage class probability estimator using attribute predictions, with reliability estimated via a validation set (where [6] employs Platt scaling and probabilistic attribute combination, while this work uses a random forest-based method).  
Thus, the approach does not fundamentally differ in handling attribute uncertainty. The key distinctions lie in:  
1) Using an ROC-based reliability estimate instead of a sigmoidal model (Platt scaling),  
2) Modeling joint attribute error statistics (via validation set propagation through trees), unlike [6]’s assumption of attribute independence.  
A more accurate characterization would position this as an improved method for modeling attribute correlation (e.g., related to Scheirer et al.’s "Multi-Attribute Spaces," which could be cited). Additionally, the proposed method likely requires a large validation set to yield reliable probability estimates when the set is subdivided during tree propagation—a limitation glossed over in the paper. The heuristic solution at line 323 does not adequately address this, and further experiments on validation set size effects would strengthen the work.  

Quality: The paper is technically sound, with well-justified choices and clear descriptions. Experiments appropriately support the claims, though the quantitative improvements are marginal. The few-shot learning extension (Section 3.3) is less developed than the main contribution: it lacks the clarity and depth of other sections, being overly brief. The proposed weighted combination of information gains (Eq. 6) for integrating class-attribute priors and image attributes appears less intuitive than a direct probability-based approach. Overall, the paper is well-written with sensible technical decisions and adequate experiments, but its central claim—that it offers a novel solution for unreliable attributes—is questionable.
