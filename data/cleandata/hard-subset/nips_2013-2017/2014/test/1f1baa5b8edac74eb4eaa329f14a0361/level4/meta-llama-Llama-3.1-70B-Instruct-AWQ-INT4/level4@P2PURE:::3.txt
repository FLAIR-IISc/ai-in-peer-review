This paper proposes a method for attribute-based zero-shot learning using random forests. Attribute classifiers are first trained using SVMs. A random forest is then trained on class-attribute signature vectors, utilizing a validation set of images with attribute labels to estimate attribute reliability at each node of the tree. An extension to few-shot learning is also proposed.

The paper is well-written and clear, effectively summarizing related work.

The novelty of this paper is somewhat limited, but not to the extent that it would preclude acceptance. However, the claim that the proposed approach uniquely addresses unreliable attribute predictions, while earlier work does not, is overstated. For instance, direct attribute prediction (DAP), the baseline method employed in this paper and others, also has some capabilities to handle attribute uncertainty. Both approaches follow a similar methodology: 
a) Training attribute classifiers on image features (identical for both papers)
b) Training a second class probability estimator as a function of attribute predictions, using a validation set to estimate the reliability of attribute classifiers ([6] uses Platt scaling and a probabilistic model to combine attributes, whereas the proposed approach uses a new random forest-based method)

In this context, the proposed approach does not fundamentally differ from earlier work in handling attribute uncertainty. Instead, it uses a more complex model to capture attribute noise: 1) employing an ROC-based estimate rather than a sigmoidal model like Platt scaling, and 2) modeling joint statistics in attribute errors, since the validation set is propagated down trees, unlike [6] which treats attributes as independent. A more accurate description of the approach would be as an improved method for modeling attribute correlation (a related paper that could be cited is Scheirer et al.'s "Multi-Attribute Spaces: Calibration for Attribute Fusion and Similarity Search").

However, one potential weakness of the proposed approach, compared to methods that treat attributes independently, is the need for a large validation set to obtain valid probability estimates, as it gets subdivided when propagated down decision trees. This issue is not thoroughly addressed in the paper, and the proposed solution (line 323) appears heuristic. Further experiments evaluating the effect of training and validation set size would have been beneficial.

The overall quality of the paper is good, with sound technical choices and descriptions. The experiments are suitable and support the paper's claims, although the level of quantitative improvement is not substantial enough to warrant acceptance solely based on empirical results. The extension to few-shot learning (Section 3.3) feels less polished than the rest of the paper. While the rest of the paper is clear and explanations are detailed, this section is brief. A method that combines class-attribute priors and image attribute estimates when estimating probabilities p(a|...) might be more intuitive than the weighted combination of information gains (Eq. 6). This is a well-written paper with sensible technical decisions and adequate experiments; however, the main selling point of the paper as a new way of handling unreliable attributes is somewhat questionable.
