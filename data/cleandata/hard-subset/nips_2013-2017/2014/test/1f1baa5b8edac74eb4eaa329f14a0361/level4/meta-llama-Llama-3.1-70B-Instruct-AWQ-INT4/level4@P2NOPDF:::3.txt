This paper proposes a method for attribute-based zero-shot learning using random forests. The approach begins by training attribute classifiers using Support Vector Machines (SVMs). A random forest is then trained on class-attribute signature vectors, utilizing a validation set of images with attribute labels to estimate attribute reliability at each node of the tree. An extension to few-shot learning is also presented.

The paper is well-written and clear, effectively summarizing related work. However, the originality of the paper is somewhat limited, although not to the extent that it would preclude acceptance. One concern is that the claim that the proposed approach addresses unreliable attribute predictions, whereas earlier work does not, may be overstated or misleading. For instance, direct attribute prediction (DAP) - the baseline method employed in this paper and many others - has similar capabilities to handle attribute uncertainty. Both approaches follow a basic framework: 

a) Training attribute classifiers on image features (identical for both papers)
b) Training a second class probability estimator as a function of attribute predictions, using a validation set to estimate the reliability of attribute classifiers ([6] uses Platt scaling and a probabilistic model to combine attributes, whereas the proposed approach uses a new random forest-based method)

In this context, the proposed approach does not seem to be a fundamentally new way of handling attribute uncertainty. Instead, it differs from earlier work by employing a more complex model for modeling attribute noise: 1) it uses an ROC-based estimate instead of a sigmoidal model like Platt scaling, and 2) whereas [6] treats attributes as independent, the proposed approach effectively models joint statistics in attribute errors, as the validation set is propagated down trees. A more accurate description of the approach might be that it provides a better way to model attribute correlation (a related paper that could be cited is Scheirer et al.'s "Multi-Attribute Spaces: Calibration for Attribute Fusion and Similarity Search").

One potential weakness of the proposed approach, compared to methods that treat attributes independently, is that it appears to require a large validation set to obtain valid probability estimates, as the set is subdivided when propagated down decision trees. This issue is glossed over in the paper, and the proposed approach for addressing it (line 323) seems heuristic. It would be beneficial to see more experiments evaluating the effect of training and validation set size.

The overall quality of the paper is good, with sound technical choices and descriptions. The experiments are suitable and support the paper's claims, although the level of quantitative improvement is not substantial enough to justify acceptance based solely on empirical results. The extension to few-shot learning (Section 3.3) feels less polished than the rest of the paper. While the majority of the paper is clear and explanations are detailed, this section is brief. A method that combines class-attribute priors and image attribute estimates when estimating probabilities p(a|...) might be more intuitive than a weighted combination of information gains (Eq. 6). This is a well-written paper with sensible technical decisions and adequate experiments; however, the main selling point of the paper as a new way of handling unreliable attributes is somewhat questionable.
