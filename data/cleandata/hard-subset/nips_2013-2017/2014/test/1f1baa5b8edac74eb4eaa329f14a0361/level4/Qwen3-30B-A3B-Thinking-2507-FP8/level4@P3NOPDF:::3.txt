

Clarity: The paper is exceptionally clear and well-written. The introduction and related work are effectively summarized, providing a strong foundation for the proposed method. Technical descriptions are precise and accessible.

Originality: The paper's novelty is modest but not negligible. The primary claim—that the proposed method handles unreliable attribute predictions while prior work does not—is somewhat overstated. Specifically, the baseline Direct Attribute Prediction (DAP) approach [7] employs a fundamentally similar strategy to manage attribute uncertainty: both train attribute classifiers on image features and use a validation set to estimate attribute reliability for a second-stage class predictor. The key difference lies in the *modeling of attribute noise*:  
1.  The paper uses an ROC-based reliability estimate instead of Platt scaling (as in [6]), and  
2.  It models joint attribute error statistics (via tree propagation on the validation set), whereas [6] treats attributes as independent.  
This positions the contribution more accurately as a refined method for modeling *attribute correlation* rather than a novel approach to uncertainty handling. A citation to Scheirer et al.'s "Multi-Attribute Spaces" [8] would strengthen this context, as it directly addresses correlated attribute errors. The paper would be more compelling if it framed its contribution this way, rather than implying prior work ignores attribute uncertainty entirely.  

Quality: The technical execution is sound, with well-motivated choices and clear explanations. Experiments are appropriate and support the core claims, though the quantitative gains are marginal. The few-shot learning extension (Section 3.3) is underdeveloped relative to the rest of the paper. It lacks sufficient detail, justification, and connection to the main framework. For instance, the weighted combination of information gains (Eq. 6) feels ad hoc; a probabilistic fusion of class-attribute priors and image-based attribute estimates would be more intuitive and theoretically grounded.  

A critical methodological gap concerns the validation set requirements. The paper relies on propagating a validation set down the random forest tree to estimate attribute reliability at each node. This approach *requires a large validation set* to yield reliable estimates at deeper tree nodes (as the set is partitioned across splits). The paper does not adequately address this sample size constraint or evaluate how validation set size impacts performance (e.g., via ablation studies on validation set size). The heuristic solution proposed in Line 323 ("if the number of samples in a node is below a threshold, use the root node estimate") is not rigorously justified and risks introducing bias. Such an analysis would be essential to validate the method’s practicality, especially given the common scarcity of labeled data in zero-shot settings.  

Overall, the paper presents a technically competent method with a reasonable contribution to modeling attribute correlation. However, the overstated novelty claim and insufficient treatment of validation set requirements weaken its impact. The marginal empirical gains further reduce the paper’s appeal for a top-tier venue like NIPS. While the work is solid, it does not advance the state-of-the-art sufficiently to warrant acceptance over more innovative submissions. The few-shot extension’s lack of depth is an additional concern.  

**Recommendation**: Weak Reject. The paper makes a modest technical contribution but fails to demonstrate sufficient novelty or address key practical limitations. The core idea—leveraging random forests to model correlated attribute errors—is interesting but not compelling enough for publication given the current context of zero-shot learning research. The validation set size issue alone is a significant drawback that the authors should resolve before reconsidering acceptance.
