- Proposes attribute-based zero-shot learning using SVM-trained attribute classifiers and a random forest over class–attribute signatures, with a validation set used to estimate attribute reliability per tree node; includes a few-shot extension.

- Paper is very clear, well written, and provides a solid related-work summary.

- Novelty is limited; the claim of uniquely handling unreliable attributes is overstated because prior work (e.g., DAP) already addresses attribute uncertainty via a similar two-stage recipe.

- Both this work and prior methods train attribute classifiers and then a second-stage class estimator using a validation set to estimate attribute reliability; the new method mainly swaps a probabilistic/Platt calibration for a random-forest estimator.

- Main technical differences are: (1) using ROC-based estimates rather than sigmoidal Platt scaling, and (2) implicitly modeling joint attribute-error statistics via tree propagation rather than assuming independent attributes.

- The contribution is better described as improved modeling of attribute correlations and non-sigmoidal calibration, not a fundamentally new mechanism for handling unreliable attributes.

- Recommend citing Scheirer et al., “Multi-Attribute Spaces: Calibration for Attribute Fusion and Similarity Search,” as a relevant reference.

- Potential weakness: likely requires a large validation set because node-wise subdivision reduces estimate reliability; the paper downplays this and the proposed mitigation appears heuristic.

- Experiments are sensible and generally support claims, but quantitative gains are modest and may not suffice on empirical grounds alone.

- Few-shot extension (Section 3.3) is underdeveloped and brief; an alternative combining class–attribute priors with image-based attribute estimates might be more intuitive than the proposed weighted information-gain combination.

- Overall: technically reasonable and well presented, but the primary claim is questionable and the true contribution is a richer error model/correlation-aware calibration.
