This paper proposes an attribute-based zero-shot learning method built on random forests. Attribute classifiers are first trained using SVMs. A random forest is then learned over class–attribute signature vectors, while a held-out set of images with attribute labels is used to estimate attribute reliability at each node of the trees. The paper also presents an extension to few-shot learning.

Clarity: The manuscript is clear, well organized, and does an effective job summarizing related work.

Originality: The contribution is moderately novel. The claim that the method uniquely addresses unreliable attribute predictions overstates the gap with prior work. For instance, direct attribute prediction (DAP) [7], which the paper uses as a baseline, also accounts for attribute uncertainty. Both this paper and DAP follow a similar two-stage structure:
a) Train attribute classifiers on image features (identical setup).
b) Learn a second-stage class probability estimator as a function of attribute predictions, using a validation set to calibrate attribute reliability. In [6], this is done with Platt scaling and a probabilistic model for attribute fusion; here it is done with a new random forest–based approach.

Thus, the proposed method is not a fundamentally new way to handle attribute uncertainty. Its main difference is a richer model of attribute noise: (1) it uses ROC-based reliability estimates rather than a sigmoidal calibration like Platt scaling, and (2) unlike [6], which assumes independent attributes, the random forest implicitly models joint error statistics because the validation data are propagated down the trees. Framing the contribution as improved modeling of attribute correlations would be more accurate. A potentially relevant related work to cite is Scheirer et al., “Multi-Attribute Spaces: Calibration for Attribute Fusion and Similarity Search.”

A potential weakness relative to methods that treat attributes independently is data efficiency. Because reliability is estimated at tree nodes, the validation set is repeatedly subdivided, which may require a large validation pool to obtain stable probability estimates. This issue is not analyzed in depth, and the mitigation described (line 323) appears heuristic. Experiments varying the sizes of the training and validation sets would strengthen the paper.

Quality: Overall, the technical choices are sound and the experimental design is appropriate. The results support the claims, though the quantitative gains are not large enough to justify acceptance on empirical improvements alone. The few-shot extension (Section 3.3) feels underdeveloped relative to the rest of the paper. While most sections are thorough, this one is brief. Conceptually, a probabilistic mechanism that combines class–attribute priors with image-based attribute evidence when estimating p(a|·) seems more natural than the proposed weighted combination of information gains (Eq. 6).

In summary, this is a well-written paper with reasonable technical decisions and adequate experiments. However, the central selling point—that it uniquely addresses unreliable attribute predictions—is overstated; the contribution is better characterized as a more expressive treatment of attribute calibration and correlation. Additional analysis of data requirements and a more polished few-shot section would significantly strengthen the work.
