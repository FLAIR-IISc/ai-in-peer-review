- Problem: Attribute classifier unreliability hurts two-stage zero-shot learning; authors target making ZSL robust to noisy attribute predictions.  
- Method: Novel random forest framework trained on ideal class attribute signatures to be robust to attribute prediction errors.  
- Key technical idea: Tree splits use estimated ROC stats (TPR/FPR) of attribute classifiers from a validation set instead of standard information gain.  
- Fractional propagation: Class signatures are fractionally propagated down tree paths to simulate noisy attribute predictions at test time.  
- Extensions: Method handles uncertainty in class signatures and supports few-shot learning.  
- Strength — novelty/significance: Explicit modeling of attribute classifier reliability is a significant and practical contribution to attribute-based ZSL.  
- Strength — technical elegance: Using validation ROC stats to guide splits is an intuitive, technically sound way to prefer reliably predictable attributes.  
- Strength — empirical validation: Extensive experiments on AwA, aPY, SUN show substantial gains over DAP and robustness in controlled noise studies; works with human and discovered attributes.  
- Strength — clarity: Paper is well-written, clearly motivated, and method/experiments are well-presented.  
- Weakness — qualitative analysis: Lacks deeper qualitative insights (e.g., which attributes IG_zero favors, per-category failure/success cases).  
- Weakness — hyperparameter justification: No sensitivity analysis for the 80/20 train/validation split of attribute-labeled data and its trade-offs.  
- Recommendation: High-quality, impactful paper; minor analysis gaps only — recommend acceptance.
