This paper presents a method for attribute-based zero-shot learning using random forests. Attribute classifiers are first trained using SVMs. A random forest is trained on class-attribute signature vectors, while attribute reliability is estimated at each node of the tree using a validation set of attribute-labeled images. An extension of few-shot learning is also proposed.

Clarity: This paper is very clear and well-written. It does a good job of summarizing related work.

Novelty: The novelty of this paper is somewhat low, but not too low to accept. One issue is that the proposed approach deals with unreliable attribute predictions, whereas earlier work did not, which is a bit exaggerated or misleading. For example, direct attribute prediction [7] (DAP) – the underlying method used in this paper and many others – has some similar capabilities to deal with attribute uncertainty. Both approaches use the basic approach:
a) Train the attribute classifier on image features (same for both papers)
b) Train a second-order probability estimator as a function of attribute predictions, using a validation set to estimate the reliability of the attribute classifier ([6]; uses Platt scaling and a probabilistic model to combine attributes, while the proposed approach uses a new random forest-based method)

Thus, I do not find the proposed approach to be fundamentally new in handling attribute uncertainty. What distinguishes it from earlier work is that it uses a more complex model to model attribute noise: 1) it uses an ROC-based estimator instead of a sigmoid-like model like Platt scaling, and 2) while [6] considers attributes to be independent, the proposed approach effectively models the combined statistics in attribute errors, as the validation set is propagated down the trees. Thus, I think it would be a little more appropriate to describe the approach as a better way to model attribute correlation (I am not well-versed in this area, but Scheirer et al. "Multi-Attribute Spaces: Calibration for Attribute Fusion and Similarity Search" may be a relevant paper that could be cited).

Also, a potential weakness of the proposed approach, compared to methods that consider attributes to be independent, is that it seems a larger validation set would be needed to obtain valid probability estimates because it is subdivided when propagated down the decision trees. This issue was briefly shown in the paper, and the proposed approach to deal with it (line 323) seems heuristic. It would be good to see more experiments evaluating the impact of training and validation set sizes.

Quality: Generally, the quality of the paper is good, and the technical options and details are understandable. The experiments are appropriate and support the paper's claims, although the level of quantitative improvement is not sufficient to fully accept the paper based on empirical results. The extension of few-shot learning (section 3.3) felt less refined compared to the rest of the paper. While the rest of the paper was clear and the explanation lengthy, this section was very concise. For me, a method that can combine class-attribute predictors and image attribute estimates when probabilities p(a|...) are estimated may have a more intuitive understanding than a weighted combination of information gains (equation 6). This is a well-written paper, with technical decisions that are understandable, and sufficient experiments; however, the paper's main selling point as a new way to handle unreliable attributes is somewhat questionable.
