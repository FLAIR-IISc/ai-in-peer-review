- Proposes attribute-based zero-shot learning using SVM-trained attribute classifiers and a random forest trained on class-attribute signature vectors.  
- Uses a validation set with attribute labels to estimate attribute reliability at each decision node; also presents a few-shot learning extension.  
- Manuscript is exceptionally clear, well-written, and situates the work within prior literature.  
- Novelty is marginal; claim of uniquely handling unreliable attributes is overstated because prior methods like DAP perform a similar two-stage process.  
- Key technical differences are: ROC-based reliability estimation instead of sigmoidal (Platt) scaling, and implicit modeling of joint attribute error statistics via validation-set propagation down trees.  
- Better described as an improved method for modeling attribute correlation rather than a fundamentally new paradigm for attribute uncertainty.  
- Should cite related work such as Scheirer et al.'s "Multi-Attribute Spaces."  
- Potential weakness: method likely requires a large validation set because it repeatedly subdivides validation examples down the trees; paper does not adequately address this.  
- Proposed remedy for small validation sets (line 323) appears heuristic and under-justified.  
- Experiments are well-designed and support claims, but empirical performance gains are modest and may not alone justify acceptance.  
- Few-shot learning extension is underdeveloped and lacks the depth and clarity of the main contribution.  
- Suggests alternative formulation: directly combine class-attribute priors with image-based attribute estimates (e.g., compute p(a|...)) instead of using a weighted information-gain combination (Eq. 6).
