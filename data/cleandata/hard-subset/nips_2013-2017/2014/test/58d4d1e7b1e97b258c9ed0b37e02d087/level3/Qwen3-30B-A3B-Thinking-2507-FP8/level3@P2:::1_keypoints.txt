- Addresses non-identifiability in Bayesian perception models and proposes a framework for a priori testing of identifiability.
- Demonstrates framework utility with two case studies: time-interval estimation and speed perception.
- Correctly highlights the degeneracy problem where different model components yield identical behavior.
- Uses a general Bayesian observer model and quantifies expected model support via KL divergence.
- Provides experimental-design insights (e.g., measuring motor parameters independently improves prior recovery).
- Fails to engage with substantial prior work on model distinguishability (e.g., Pitt, Navarro et al.).
- Does not clearly articulate how the framework advances beyond existing model comparison techniques.
- Key modeling assumptions (parametric sensory map, mixture models) lack sufficient justification.
- No sensitivity or robustness analysis to show how identifiability results depend on modeling choices.
- Some technical details are unclear (e.g., a "trivial mathematical fact") and minor grammatical/legend issues exist.
- Contribution is important but diminished by weak connection to prior literature and insufficient justification of assumptions.
- Recommendation: Weak reject; merits consideration if prior work, assumptions, and robustness are addressed.
