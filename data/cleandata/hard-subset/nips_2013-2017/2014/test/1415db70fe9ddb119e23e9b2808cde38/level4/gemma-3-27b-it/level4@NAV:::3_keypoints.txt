- Introduces a sentiment classification method using encoding and decoding vectors from an RNN auto-encoder.  
- Labels the method “belief propagation,” which is misleading because it does not match the standard inference algorithm.  
- Paper is generally well-written but has unclear motivation and weak theoretical grounding.  
- Evaluation is limited: reporting an ensemble result is hard to interpret against single-model baselines and may obscure performance.  
- Experimental details are missing: hyperparameter tuning process and criteria for selecting ensemble/best models need clarification.  
- Unclear whether benchmarks in Table 2 were tuned equivalently or used the same data augmentation as BP-RNN.  
- Unclear if RNN and Bi-RNN baselines used the same hybrid word vectors as BP-RNN.  
- Given missing details, the second entry in Table 3 seems the most direct comparison to the core BP-RNN architecture.  
- Minor concern: using an off-the-shelf tweet parser may produce inaccuracies; a discussion is needed.  
- Minor concern: several citations (e.g., Irsoy and Cardie) are incomplete.  
- Summary: approach is promising but requires stronger rationale and much clearer experimental methodology.
