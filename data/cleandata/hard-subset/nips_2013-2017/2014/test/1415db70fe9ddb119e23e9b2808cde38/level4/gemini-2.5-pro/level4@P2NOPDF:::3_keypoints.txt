- Proposes a recursive neural network auto-encoder using both encoding and decoding phrase vectors for phrasal sentiment classification (called "belief propagation" — misleading name).
- The term "belief propagation" is inappropriate and likely to confuse readers because the model is not the standard belief propagation algorithm.
- Paper is well-written and easy to follow.
- Motivation for using both encoding and decoding vectors is weak and lacks theoretical justification for improved performance.
- Experimental evaluation lacks clarity and rigor.
- Main comparison table reports an ensemble for the proposed model, which is not an apples-to-apples comparison against single-system benchmarks.
- Missing experimental details on hyperparameter tuning and model selection (procedure for tuning, how ensemble or single best model was chosen).
- Unclear whether benchmark models were re-implemented and tuned on the same validation data or if results were taken from prior work.
- Unclear whether benchmark models had access to the same augmented data as BP-RNN.
- Unclear whether baseline RNN and Bi-RNN used the hybrid word vectors or if that was unique to BP-RNN.
- Ambiguity over which result (ensemble vs single best model) is the appropriate comparison undermines the paper’s claims.
- Minor: using an off-the-shelf tweet parser is surprising for noisy data; discussion of parser accuracy on this dataset is needed.
- Minor: several incomplete citations in the bibliography (e.g., Irsoy and Cardie).
- Overall: novel and promising architecture but hindered by misleading naming, weak motivation/theory, and insufficient experimental reporting; addressing these methodological questions would strengthen the paper.
