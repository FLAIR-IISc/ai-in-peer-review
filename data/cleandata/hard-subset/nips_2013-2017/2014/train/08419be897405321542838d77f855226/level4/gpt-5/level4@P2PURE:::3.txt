This paper proposes an interesting approach to discovering more efficient algebraic identities. It is particularly noteworthy that distributed, recursive neural representations can reliably classify these identities.

However, the problem formulation appears quite restricted.

Clarification needed:
- The statement “weight tensor W3 (of size l x l x l, so that the output is also size l)” is not well explained. Does this imply you are using a neural tensor network rather than a simple linear layer? Bowman reported that tensor networks substantially outperform RNNs on these tasks.

Comments on Section 7.2 (Efficient Identity Discovery):
- It is unclear whether you are still using the RNN for classification in this section, while primarily evaluating different search strategies.
- I did not fully follow how proposal candidates are generated during the search; this should be explained in more detail.
