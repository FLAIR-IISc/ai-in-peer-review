- Proposes an ML-driven method to discover computationally efficient mathematical identities using an attribute grammar and learning-guided search.  
- Grammar framework where each rule carries an associated computational complexity.  
- Two learning-based search strategies: n-gram models for common substructures and a recursive neural network (RNN) for continuous expression representations.  
- Trains on solutions to simpler expressions to guide search for more complex identities.  
- Demonstrates discovery of identities that reduce computational complexity (e.g., from O(n³) to O(n²)).  
- Shows RNNs can learn useful continuous representations for symbolic expressions, enabling ML techniques in symbolic domains.  
- Experiments show learning-based methods significantly outperform random search; n-gram models excel on repetitive patterns.  
- Problem scope is narrowly limited to homogeneous polynomials of fixed degree with a single matrix/vector variable, with insufficient justification.  
- Insufficient comparison to established symbolic-computation systems (e.g., Mathematica, Maple).  
- Technical exposition unclear around the weight tensor (W3) — ambiguity between neural tensor network and standard RNN; relevant prior work (e.g., Bowman 2013) not cited.  
- Experimental description ambiguous about the RNN’s role (classification vs. search) and how proposals are generated.  
- Contains several minor textual errors and a figure label mistake (Figure 3 should reference "RNN" not "TNN").  
- Contributions are valuable and well-executed for the domain, with potential for broader extension.  
- Overall recommendation: accept with minor revisions to clarify scope, RNN details, comparisons, and correct errors.
