

This paper introduces an interesting solution for discovering more efficient algebraic identities. It is particularly intriguing to observe that distributed, recursive neural representations successfully classify these identities.

The problem definition appears quite restricted.

The following point requires clarification:
"weight tensor W3 (of size lxlxl, so that the output is also size l)"
Does this indicate the use of a neural tensor network instead of a simple linear network? Bowman's work demonstrates that tensor networks significantly outperform RNNs in such contexts.

Section 7.2 (Efficient Identity Discovery) raises concerns. It is unclear whether the RNN remains the classifier while evaluating different search strategies. Additionally, the proposal generation mechanism during the search process lacks sufficient explanation.
