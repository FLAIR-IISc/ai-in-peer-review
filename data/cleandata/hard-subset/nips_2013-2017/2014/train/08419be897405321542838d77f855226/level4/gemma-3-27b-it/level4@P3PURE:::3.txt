This paper introduces an interesting solution to finding more efficient algebraic identities. It is compelling to see that distributed, recursive neural representations can successfully classify these identities.

However, the problem definition appears quite restricted in scope.

Section 7.2, “Efficient Identity Discovery,” and the description of the weight tensor W3 (of size lxlxl, resulting in an output of size l) are unclear. Does this imply the use of a neural tensor network rather than a simple linear network? Bowman previously demonstrated that tensor networks outperform RNNs in similar tasks. 

Furthermore, it is unclear whether the RNN is still utilized for classification while evaluating different search strategies. The method for generating proposals during search is not sufficiently detailed and would benefit from a clearer explanation.
