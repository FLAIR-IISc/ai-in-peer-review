

This paper presents a novel framework for discovering computationally efficient mathematical identities using attribute grammars combined with machine learning techniques. The authors introduce a grammar-based representation of symbolic expressions and propose two learning approaches—n-gram models and recursive neural networks—to guide the search for equivalent expressions with lower computational complexity. The work builds upon attribute grammar concepts from compiler construction and applies machine learning techniques to symbolic computation, drawing inspiration from NLP approaches like n-gram language models and recursive neural networks used for sentence parsing. The paper demonstrates the framework's effectiveness through several case studies, including efficient computation of matrix expressions that would require exponential time using naive evaluation.

Strengths of the paper include the clear problem definition, the practical demonstration of the framework discovering previously unknown mathematical identities (as shown in the supplementary material), and the well-executed experiments comparing different search strategies. The paper makes a significant contribution by applying machine learning to a domain that has traditionally relied on rule-based systems, and the learned representations for symbolic expressions could potentially enable broader applications of machine learning to symbolic computation. The evaluation shows that the learning-based approaches significantly outperform random search for complex expressions, as evidenced by the experimental results in Section 7.2.

The paper has several weaknesses that need addressing. The problem definition is quite restrictive, limiting the framework to homogeneous polynomials of fixed degree with a single matrix variable, which may limit the applicability of the approach. The explanation of the recursive neural network architecture requires clarification—specifically, the description "weight tensor W3 (of size l×l×l, so that the output is also size l)" suggests the use of a neural tensor network rather than a standard recursive neural network, which contradicts the paper's claim of using a recursive neural network. Bowman's work [3] has demonstrated that tensor networks outperform standard RNNs for logical reasoning tasks, and it would be helpful to clarify why the authors chose this approach.

The paper's explanation of the search strategy could be improved. It is unclear how the RNN-based search strategy generates proposals for the next grammar rule, and the description of the scheduler and strategy components could be more precise. The paper states "the vector is presented to the next operation" but doesn't specify how the vector is transformed between operations. Additionally, the paper mentions "the final vector is pass to a softmax classifier" which contains a grammatical error.

The paper would benefit from more detailed explanation of the relationship between the RNN's learned representation and the search strategy, particularly in Section 6.3 where the authors transition from the classification task to the search strategy. The claim that "the RNN-based representation is employed for identity discovery" without further explanation is insufficient.

The paper contains several minor errors that need correction: "has contains a single matrix" should be "contains a single matrix," "listed in table Table 1" should be "listed in Table 1," "vector is the presented to the next operation" should be "vector is presented to the next operation," "The final vector is pass to a softmax classifier" should be "The final vector is passed to a softmax classifier," "as to how explore space of possible" should be "as to how to explore the space of possible," and "The scheduler is is fixed" should be "The scheduler is fixed." Figure 3 should reference RNN, not TNN.

Overall, the paper presents an interesting application of machine learning to symbolic computation with promising results, but would benefit from clarifying the technical details and addressing the restrictive problem definition. The discovery of novel mathematical identities that reduce computational complexity from exponential to polynomial is significant and could have practical applications in machine learning (as discussed in Section 8), making this a valuable contribution to the field.

Pro: Novel framework combining attribute grammars with machine learning for discovering efficient mathematical identities; Demonstrates discovery of previously unknown identities with significant computational complexity reduction; Clear experimental evaluation showing learning approaches outperform random search for complex expressions.

Con: Problem definition is overly restrictive; Technical details about the recursive neural network architecture need clarification; Explanation of the search strategy is insufficient; Contains several minor errors that need correction.
