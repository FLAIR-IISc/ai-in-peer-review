

This paper proposes an innovative approach to discovering computationally efficient mathematical identities using machine learning techniques. The authors introduce an attribute grammar framework for representing symbolic expressions and propose learning-based search strategies to navigate the exponentially large space of possible expression trees. The core idea is to train models on solutions to simpler expressions to guide the search for more complex identities, with the goal of finding expressions that compute the same result as a target but with lower computational complexity.

The paper makes several noteworthy contributions. First, it develops a grammar framework for symbolic expression search, where each grammar rule has an associated computational complexity. Second, it introduces two learning-based search strategies: n-gram models that capture common substructure patterns in solutions, and a recursive neural network (RNN) that learns a continuous representation of symbolic expressions. Third, it demonstrates the discovery of new mathematical identities that significantly reduce computational complexity, such as finding O(n²) implementations for expressions that naively require O(n³) time. The paper also makes an interesting contribution by showing how RNNs can learn effective continuous representations for symbolic expressions, enabling the application of machine learning techniques to symbolic domains.

The paper is well-motivated and addresses an interesting problem at the intersection of symbolic computation and machine learning. The technical approach is sound, with careful attention to the challenges of symbolic equivalence checking (using integer-based descriptors to avoid numerical issues) and representation learning. The experiments demonstrate that the learning-based approaches significantly outperform random search for complex expressions, with the n-gram models particularly effective for patterns with repetitive structure.

However, the paper's scope is quite restricted, which is not sufficiently justified. The authors limit their problem to homogeneous polynomials of fixed degree with a single matrix or vector variable, which significantly narrows the applicability of their approach. While the authors acknowledge this limitation, they do not sufficiently explain why this restriction is reasonable or how it might be overcome in future work. The paper also lacks sufficient comparison with related work in symbolic computation, particularly with systems like Mathematica and Maple, which have extensive capabilities for symbolic manipulation despite not using learning approaches.

The paper's technical exposition contains some unclear points. The description of the weight tensor in Section 4.2 is confusing: "weight tensor W3 (of size l×l×l, so that the output is also size l)" suggests the authors are using a neural tensor network rather than a standard RNN, but they don't clarify this distinction. It's worth noting that Bowman's work (2013) demonstrated that tensor networks outperform standard RNNs for similar tasks, which would strengthen the authors' claims about their RNN's effectiveness.

The experimental section (Section 7.2) is somewhat unclear about the exact role of the RNN. The paper states they use the RNN for classification (Section 4.2) but then uses it for search strategy (Section 6.3), which could be explained more clearly. The description of how the search strategy generates proposals is also insufficiently detailed.

The paper contains several minor errors that should be corrected: "has contains a single matrix," "listed in table Table 1," "vector is the presented to," "final vector is pass to," "as to how explore space of possible," and "scheduler is is fixed." Additionally, Figure 3 should reference "RNN" not "TNN."

Despite these issues, the paper makes a valuable contribution to the field. The demonstration that machine learning can guide symbolic search to discover new mathematical identities is significant, particularly for applications in machine learning (e.g., efficient computation of dropout and RBM partition functions). The paper's framework could potentially be extended to broader symbolic domains with additional grammar rules.

Strengths:
- Novel application of machine learning to symbolic identity discovery
- Effective demonstration of computational complexity reduction
- Clear technical framework with well-defined problem statement
- Interesting exploration of continuous representations for symbolic expressions
- Demonstrated success on non-trivial mathematical problems

Weaknesses:
- Restricted problem scope with insufficient justification
- Unclear technical details about the RNN implementation
- Inadequate comparison with symbolic computation systems
- Minor technical errors and unclear explanations
- Limited discussion of the generalizability of the approach

Overall, the paper presents an interesting and potentially valuable contribution to the field. While the restricted scope is a limitation, the core idea of using learning to guide symbolic search is compelling and well-executed for the problem domain. The technical approach is sound, and the results demonstrate clear advantages over random search. With improved clarity and justification of the problem restrictions, this paper would be a strong contribution to NIPS. I recommend acceptance with minor revisions.
