This paper introduces a novel approach for discovering efficient algebraic identities using distributed, recursive neural representations to classify them. This is an interesting and promising direction, and it is impressive to see that this method can be successful.

However, the paper could be significantly improved with clarifications in several key areas.

**Major Comments**

*   **Problem Definition:** While the approach is interesting, the scope of the problem as defined in the paper appears somewhat narrow.

*   **Model Architecture:** The description of the model architecture is unclear. The paper states the use of a "weight tensor W3 (of size lxlxl, so that the output is also size l)." Does this imply the use of a Neural Tensor Network rather than a standard linear layer? This is a critical detail, as prior work (e.g., Bowman et al.) has shown that tensor networks can significantly outperform standard RNNs on similar tasks. A more explicit description of this component is needed for clarity and to properly contextualize the results.

*   **Efficient Identity Discovery (Section 7.2):** Further clarification is needed for the experiments in this section.
    *   It is unclear whether the RNN classifier is still employed at this stage while the different search strategies are being evaluated.
    *   The mechanism for generating proposals during the search is not explained in sufficient detail. A more thorough description of this process would greatly improve the paper's clarity and reproducibility.
