This paper introduces an interesting solution to finding more efficient algebraic identities. It is intriguing to see that distributed, recursive neural representations can successfully classify these identities.

However, the problem definition seems quite restricted. The explanation of the weight tensor W3 (of size lxlxl, such that the output is also of size l) is unclear. Specifically, it is not apparent whether the authors use a neural tensor network instead of a simple linear network. This distinction is important, as Bowman has shown that tensor networks outperform RNNs in certain tasks.

In Section 7.2, Efficient Identity Discovery, it is unclear whether the authors are still using the RNN for classification while evaluating different search strategies. The proposal generation process in the search is also not well-explained and could benefit from additional clarification.
