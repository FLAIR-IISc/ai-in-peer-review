This paper proposes an interesting solution for discovering more efficient algebraic identities, leveraging distributed and recursive neural representations to classify these identities successfully.

However, the problem definition appears to be somewhat restricted, and certain aspects of the methodology could be clarified. Specifically, the description of the weight tensor W3, which is of size lxlxl to produce an output of the same size, is unclear. It is not explicitly stated whether a neural tensor network is employed instead of a simple linear network. This distinction is crucial, as Bowman's work has demonstrated that tensor networks significantly outperform RNNs in similar tasks.

In Section 7.2, Efficient Identity Discovery, it is ambiguous whether the RNN is still utilized for classification while evaluating different search strategies. Furthermore, the proposal generation process in the search could be explained more clearly.
