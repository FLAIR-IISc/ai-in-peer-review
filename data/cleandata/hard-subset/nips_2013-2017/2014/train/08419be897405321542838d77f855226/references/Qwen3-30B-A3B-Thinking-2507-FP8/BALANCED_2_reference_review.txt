

# Review of "Learning to Discover Efficient Mathematical Identities"

This paper proposes a novel framework for discovering computationally efficient versions of symbolic mathematical expressions using attribute grammars combined with machine learning techniques. The authors address the problem of finding alternative expressions that compute the same result as a target expression but with lower computational complexity. The framework uses attribute grammars to represent expressions as trees, numerical descriptors for efficient equivalence checking, and machine learning (n-gram models and recursive neural networks) to guide the search for efficient expressions. The approach is demonstrated on several families of polynomial expressions, showing significant computational improvements over naive evaluation.

The paper makes several important contributions: (1) a grammar framework for symbolic expression search, (2) integration of machine learning into this framework, (3) a novel application of recursive neural networks for continuous representation of symbolic expressions, and (4) discovery of new mathematical identities with computational advantages.

**Quality**: The paper presents a technically sound approach with clear methodology. The experimental design is reasonable, with well-defined families of expressions and appropriate evaluation metrics. The authors carefully describe the RNN training procedure with important details (identity initialization, rectified linear units, curriculum learning). However, the computational complexity analysis is somewhat hand-wavy (e.g., claiming O(n^3) complexity without clear justification of operations). The paper acknowledges limitations in the grammar constraints but could better address how these constraints might affect the generality of the approach.

**Clarity**: The paper is well-organized and clearly written, with good examples (Examples 1 and 2) that effectively illustrate the approach. The mathematical notation is generally clear, though the grammar definitions could be more accessible to non-experts. The supplementary material is mentioned as containing the discovered identities, which is helpful but could be better integrated into the main text. The description of the RNN architecture and training details is particularly thorough.

**Originality**: This work presents a novel application of machine learning to symbolic computation. While attribute grammars have been used in other domains (e.g., compiler construction, bioinformatics), their application to discovering mathematical identities with learning-based search strategies appears original. The use of n-grams over mathematical operations and the application of recursive neural networks to symbolic expressions are innovative. The paper adequately addresses related work in theorem proving, program induction, and NLP, though the comparison to symbolic computation systems like Maple and Mathematica is somewhat limited.

**Significance**: The discovered identities offer significant computational improvements (e.g., reducing O(n^3) to O(n^2) complexity), which could have practical applications in machine learning (as noted in Section 8 regarding dropout and RBM partition functions). The framework demonstrates how machine learning can guide search in combinatorially large spaces, with potential broader applicability to other AI problems involving combinatorial explosions. However, the paper could better articulate how these specific discoveries translate to practical benefits in real-world machine learning systems.

**Strengths**:
- Novel integration of grammars with machine learning for symbolic computation
- Demonstrated success on challenging mathematical identities
- Thorough technical description of the RNN representation learning
- Clear experimental results showing the advantage of learning-based search strategies
- Potential for practical applications in machine learning

**Weaknesses**:
- Problem restricted to homogeneous polynomials of fixed degree with single matrix/vector inputs
- Limited discussion of how discovered identities scale to larger problems
- Computational complexity claims lack detailed justification
- Insufficient comparison to other symbolic computation approaches
- Representation learning evaluation limited to classification accuracy without showing broader utility

**Recommendation**: I recommend acceptance with minor revisions. The paper presents a technically sound, novel approach with significant potential impact on symbolic computation and machine learning. The main weaknesses are the restricted problem scope and insufficient justification of complexity claims, which could be addressed with minor revisions.

**Suggestions for Improvement**:
1. Provide more detailed justification for the computational complexity claims (e.g., explicitly count operations in the efficient expressions)
2. Expand the discussion of limitations to better address how the framework might be extended to more general expressions
3. Include a more thorough comparison to existing symbolic computation techniques beyond Maple/Mathematica
4. Add a brief discussion of how the learned representations could be used for other symbolic tasks beyond identity discovery
5. Consider including a small example showing how the discovered identities could be integrated into a machine learning system (e.g., for more efficient dropout computation)

The paper makes a valuable contribution to the intersection of machine learning and symbolic computation, and with the suggested improvements, it would be a strong fit for NIPS. The work demonstrates how machine learning can help solve combinatorial search problems, which has broader implications for AI research.
