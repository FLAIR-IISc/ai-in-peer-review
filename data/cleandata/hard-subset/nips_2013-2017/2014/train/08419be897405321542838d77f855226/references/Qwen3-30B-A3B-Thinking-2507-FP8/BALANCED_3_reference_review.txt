

# Review of "Learning to Discover Efficient Mathematical Identities"

## Summary
This paper proposes a novel framework for discovering computationally efficient mathematical identities using machine learning. The authors formulate the problem of finding equivalent expressions with lower computational complexity as a search problem over a grammar of symbolic operations. They introduce a grammar-based representation of expressions, develop numerical and learned continuous representations for efficient equivalence checking, and employ machine learning techniques (n-grams and recursive neural networks) to guide the search strategy. The approach is evaluated on several families of polynomial expressions, demonstrating the discovery of identities that reduce computational complexity from O(n³) to O(n²) or from exponential to polynomial time.

The work builds upon attribute grammars (Knuth, 1968) and borrows techniques from NLP (n-gram models, recursive neural networks) to apply machine learning to symbolic computation. It distinguishes itself from related work in theorem proving, program induction, and symbolic computation engines by focusing on learning search strategies rather than relying on hand-designed heuristics.

## Strengths

**Originality:** The paper presents a genuinely novel application of machine learning to discover mathematical identities. The integration of attribute grammars with learning-based search strategies is innovative, and the paper clearly differentiates itself from previous work in theorem proving and symbolic computation. The discovery of previously unknown identities (e.g., for matrix expressions with degree k=6) represents a significant contribution to the field.

**Quality:** The technical approach is well-justified, with careful consideration of representation methods (numerical vs. learned) and a thorough experimental evaluation. The RNN-based representation for symbolic expressions (Section 4.2) is particularly well-executed, with appropriate attention to training details (curriculum learning, weight initialization, dropout) that prevent overfitting. The experimental results (Section 7) demonstrate clear success in discovering identities that random search cannot find.

**Significance:** The paper addresses a meaningful problem in computational efficiency with potential practical applications in machine learning (e.g., for approximating dropout or RBM partition functions). The approach represents a promising step toward applying machine learning to symbolic computation, a domain where such methods have been largely unexplored.

## Weaknesses

**Scope Limitations:** The restricted problem formulation (homogeneous polynomials of fixed degree, single matrix variable) significantly limits the applicability of the approach. While the authors acknowledge this in Section 8, they don't sufficiently address how to extend the framework to more general mathematical expressions (e.g., with trigonometric functions or recursion), which would be necessary for broader impact. The paper would benefit from a more detailed discussion of these extension challenges.

**Evaluation Depth:** The evaluation focuses primarily on success/failure rates within a fixed time limit, but doesn't quantify the actual computational savings achieved by the discovered identities. For example, it would be valuable to show the ratio of original complexity to discovered complexity for the identities found, rather than just stating the complexity classes.

**Comparison with Existing Tools:** The paper mentions symbolic computation engines like Maple and Mathematica but doesn't provide a meaningful comparison of their capabilities on the same tasks. It would strengthen the paper to show how the discovered identities compare to what these tools can produce (or fail to produce) for the same expressions.

**Reproducibility:** While the authors provide a GitHub repository, the paper lacks sufficient detail on key implementation choices (e.g., specific hyperparameters for the RNN, the exact numerical parameters for the integer-based descriptor). More details on these would improve reproducibility.

## Overall Assessment

This paper presents a technically sound and innovative approach to discovering efficient mathematical identities using machine learning. The core contributions—integrating learning with grammar-based search, developing a continuous representation for symbolic expressions, and discovering novel identities—are significant and well-executed. The work represents a promising direction for applying machine learning to symbolic domains.

However, the restricted scope of the problem and the limited evaluation depth prevent it from reaching the highest level of impact expected at NIPS. The paper would be substantially strengthened by addressing the scope limitations, providing more detailed evaluation metrics, and offering a clearer path to broader applicability.

## Specific Suggestions for Improvement

1. Expand the evaluation to include quantitative metrics of computational savings (e.g., actual speedup factors) alongside success rates.

2. Add a comparison with symbolic computation tools (Maple, Mathematica) on the same identities to demonstrate the added value of the proposed approach.

3. Provide more detailed discussion of how to extend the framework to more general mathematical expressions (e.g., with trigonometric functions, recursion), including potential challenges.

4. Include more implementation details in the main paper (e.g., hyperparameters for the RNN, specific integer parameters for the descriptor) to improve reproducibility.

5. Consider adding a small case study showing how one of the discovered identities could be integrated into a practical ML system (e.g., for faster dropout computation).

## Recommendation

The paper makes a novel contribution to the intersection of machine learning and symbolic computation, with well-executed experiments demonstrating the feasibility of the approach. While the scope is somewhat limited, the core ideas are strong and potentially impactful. I recommend acceptance with minor revisions addressing the points above. The work represents a valuable step toward applying machine learning to symbolic domains and deserves to be shared with the NIPS community.
