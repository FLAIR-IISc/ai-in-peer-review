- Overall assessment: Novel, promising approach combining attribute grammars and machine learning; recommend acceptance with minor revisions.
- Core idea: Use attribute grammars to define expression space and learn search strategies (n-gram, RNN) to discover efficient mathematical identities.
- Empirical results: Demonstrates substantial computational improvements (e.g., O(n^3) → O(n^2)) and finds identities often beyond manual discovery.
- Novelty: First application of learned search strategies and RNN-based symbolic embeddings in this specific automated identity discovery problem.
- Technical soundness: Framework is well-defined; integer-based descriptors effectively address floating-point precision issues.
- Learned representation: RNN embeddings for symbolic expressions are an important contribution enabling further ML integration.
- Code and data: Public release (GitHub) promotes reproducibility and follow-up work.
- Clarity: Paper is generally well-written and organized, making complex ideas accessible.
- Limited scope: Framework currently restricted to homogeneous polynomials with a single matrix/vector variable, limiting generality.
- RNN performance: RNNs do not consistently outperform simpler n-gram baselines, especially on simpler families (Fig. 3).
- Hyperparameter tuning: Insufficient detail on hyperparameter selection (n-gram order, learning rates, etc.) beyond RNN hidden size.
- Presentation: Many discovered identities are relegated to supplementary material; a few should be moved into the main text.
- Failure analysis: More in-depth discussion needed on why framework struggles with hard families (e.g., (RBM-2)k).
- Validation method: Using a linear system to verify solutions is clever and strengthens empirical claims.
- Significance: Work has potential wide impact (computer algebra, optimization, physics/engineering) and opens new research directions in learned symbolic methods.
