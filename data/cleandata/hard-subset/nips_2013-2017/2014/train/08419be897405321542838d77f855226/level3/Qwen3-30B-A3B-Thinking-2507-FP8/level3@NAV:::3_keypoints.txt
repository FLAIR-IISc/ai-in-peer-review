- Proposes a grammar-based framework that discovers efficient mathematical identities using attribute grammars and learning.
- Introduces recursive neural networks to learn continuous representations of symbolic expressions.
- Demonstrates discovery of new identities that reduce computational complexity (e.g., O(n³) to O(n²)).
- Shows learning-based search significantly outperforms random search on complex, repetitive patterns.
- Novel application of neural methods to symbolic computation with clear NLP connections.
- Problem scope is overly restricted to homogeneous polynomials of single matrix/vector inputs, limiting applicability.
- Explanation of the weight tensor W3 (l×l×l) is unclear—uncertain if a tensor network or just a tensor-parameterized linear network is used.
- Fails to justify choosing RNNs over tensor networks despite evidence (Bowman 2013) that alternatives can outperform RNNs on logical tasks.
- Section 7.2 is confusing about whether the RNN is still used for classification during search; evaluation of search strategies lacks clarity.
- Proposal generation in the search procedure is insufficiently explained.
- Paper contains multiple typos and grammatical errors, including a Figure 3 labeling error (TNN should be RNN).
- Opening sentence is duplicated in the manuscript.
- Technical execution and clarity issues weaken the paper’s contribution and suitability for NIPS.
- Recommends addressing clarity, model-justification, and expanding problem definition to show broader impact.
