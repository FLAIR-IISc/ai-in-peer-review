- Proposes a novel framework for discovering efficient mathematical identities using machine learning.  
- Introduces an attribute grammar to represent symbolic expressions.  
- Explores two learning approaches: n-gram models and recursive neural networks (RNNs).  
- Uses an RNN to learn continuous representations of symbolic expressions for guiding search.  
- Discovers new mathematical identities that significantly reduce computational complexity.  
- Evaluates on several expression families and shows discoveries beyond brute-force or human derivation.  
- Learning approaches outperform random search; 3-gram, 4-gram, and 5-gram perform particularly well.  
- Problem definition seems overly restricted; broader settings should be explored.  
- Explanation of weight tensor W3 is unclear; unclear if a neural tensor network is used instead of a linear network.  
- Proposal generation in the search procedure lacks sufficient detail.  
- Unclear whether the RNN is still used for classification in Section 7.2.  
- Manuscript contains typos, grammatical errors, and duplicated/repeated sentences that should be fixed.  
- Figure 3 incorrectly labels TNN as RNN and should be corrected.  
- Overall: interesting and promising work that would be stronger with clarifications and revisions.
