Here is a summary:

This paper proposes a model for solving discriminatory tasks using video input. This model consists of two convolutional networks. The input to one network is an appearance frame. The input to the second network is a stack of densely computed optical flow features. Each pathway is trained separately to classify its input. The predictions (weighted) made by each network are averaged to obtain a prediction for the video.

The model has been evaluated on two datasets. The results are impressive and match (or are close to) state-of-the-art methods (which use closely hand-crafted features).

Strengths:
- This model is a simple application of convolutional networks that produces good results.
- Previous deep learning models did not attempt to use optical flow (or other hand-crafted features) for vision tasks, preferring to learn all features directly from pixels. This paper shows that, at least for action recognition, the optical flow field is a useful input representation. This is a significant contribution.

Weaknesses:
- The model does not consider the overall sequence of events in the entire video, but only models adjacent frames of a specific length (L=10).

Quality:
The experiments were well-designed. It would have been more insightful if the authors had included some analysis of error modes. For example, are there any classes or groups of classes that the model is unable to classify well? Is it possible to identify what types of videos the model does not perform well on?

Clarity:
The paper is written clearly. The model is well-explained.

Novelty:
The application of convolutional networks for optical flow features is new.

Significance:
This approach can have a significant impact on research using video and motion to solve various vision problems. The model is a simple application of convolutional networks for video data and produces very promising results. The use of optical flow features as input to convolutional networks is a new contribution.
