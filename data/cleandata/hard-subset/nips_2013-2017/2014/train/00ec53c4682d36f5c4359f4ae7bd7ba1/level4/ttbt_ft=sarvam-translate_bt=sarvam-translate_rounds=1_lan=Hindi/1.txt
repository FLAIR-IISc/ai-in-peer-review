Summary:

This paper proposes a model for solving differential tasks with video input. This model includes two convolutional nets. The input to one net is a sequence of frames. The input to the second net is a stack of densely computed optical flow features. Each pathway is trained separately to classify its input. The prediction for a video is obtained by taking the (weighted) average of the predictions made by each net.

The model is evaluated on two datasets. The results are impressive and match (or come close to) state-of-the-art methods that use deeply handcrafted features.

Strengths:

- This model is a simple application of convolutional nets that produces good results.
- Previous deep learning models have not tried to use optical flow (or other handcrafted features) for vision tasks, preferring to learn all features directly from pixels. This paper shows that, at least for action recognition, the optical flow field is a useful input representation. This is a significant contribution.

Weaknesses:

- The model does not take into account the overall sequence of actions across the entire video, but only models adjacent frames of a fixed length (L=10).

Quality:

The experiments are well-designed. It would be more informative if the authors also included some analysis of error modes, for example, whether there are certain classes or groups of classes that the model is unable to classify well. Is it possible to identify the types of videos on which the model does not perform well?

Clarity:

The paper is written clearly. The model is well explained.

Novelty:

The application of convolutional nets for optical flow features is novel.

Significance:

This approach can have a significant impact on research using video and motion for various vision problems. The model is a simple application of convolutional nets for video data and achieves very promising results. Using optical flow features as input to a convolutional net is a novel contribution.
