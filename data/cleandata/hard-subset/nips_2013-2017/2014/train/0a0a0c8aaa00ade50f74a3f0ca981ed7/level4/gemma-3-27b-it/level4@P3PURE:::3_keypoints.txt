- Unified framework addressing three neglected biological constraints (Dale’s law, non-sparse representations, and continuous-rate vs. binary memory mismatch) in associative memory models.
- Leverages a recent control-theoretic, spectral-bounding approach to construct networks that satisfy these constraints.
- Introduces a cost function optimized by gradient descent with three terms: enforcing fixed-point memories, ensuring stability, and penalizing large weights to avoid saturation.
- Enforces Dale’s principle directly via weight parameterization, yielding natural excitation–inhibition balance.
- Memory is implemented using only excitatory neurons while treating inhibitory neurons as optimizable resources.
- Single-neuron firing-rate saturation emerges from optimization rather than being imposed explicitly.
- Numerical simulations show robustness to noise and demonstrate associative memory performance.
- Provides an explanation for reduced trial-to-trial variability after stimulus onset as convergence to the nearest attractor.
- Does not propose a biologically plausible learning rule but acknowledges and cites recent proposals that could lead to one.
