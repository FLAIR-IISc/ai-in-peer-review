- Fresh perspective and significant new insights on the associative memory problem in neural networks.  
- Identifies overlooked biological aspects in prior work: Dale’s law, binary rate-based models, and non-sparse memory representations.  
- Proposes a comprehensive strategy that addresses these biological concerns simultaneously.  
- Employs a recent control-theoretic approach using spectral bounding techniques.  
- Introduces a three-term cost function balancing fixed-point enforcement, stability, and small weights to prevent saturation.  
- Enforces Dale’s principle directly via weight parameterization, producing natural excitation–inhibition balance.  
- Single-neuron saturation emerges from the optimization rather than being explicitly enforced.  
- Uses excitatory neurons to implement memories while treating inhibitory neurons as optimized resources.  
- Numerical simulations validate results, showing robustness to noise and good associative memory properties.  
- Provides an explanation for decreased trial-to-trial variability after stimulus onset via convergence to the nearest attractor.  
- Paper is clear, well-written, and an important contribution, though it does not yet propose a biologically plausible learning rule (mentions possible directions).  
- Minor issue: sensitivity analysis for the choice γ = 0.04 in equation (3) is needed.  
- Minor issue: provide motivation for using a log-normal distribution for memory states.  
- Minor issue: quantify and discuss how cost function (4) may violate exact fixed-point conditions.  
- Minor issue: clarify the unclear statements on lines 254–256.  
- Minor issue: discuss capacity limitations or provide preliminary statements about capacity.  
- Minor issue: discuss whether Dale’s law affords computational benefits under specific constraints or is primarily evolutionary/biophysical.
