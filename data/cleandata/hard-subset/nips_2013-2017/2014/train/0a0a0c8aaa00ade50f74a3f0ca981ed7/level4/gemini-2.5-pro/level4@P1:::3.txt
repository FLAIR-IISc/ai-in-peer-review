This paper revisits the classic problem of associative memory in recurrent neural networks, offering a fresh perspective that integrates multiple biological constraints into a unified framework. The authors correctly point out that many previous models have fallen short by violating Dale's law, relying on binary memory representations, or requiring neurons to fire at saturating rates. While some of these issues have been addressed individually in prior work, this paper presents a principled method to tackle them all simultaneously.

The core of the contribution is a novel control-theoretic approach to construct networks that can store and recall graded, analog memory patterns. The authors formulate a cost function that is optimized via gradient descent to achieve three goals: ensuring the target memory patterns are fixed points of the dynamics, guaranteeing the stability of these fixed points, and regularizing synaptic weights. Stability is achieved using a recently developed technique from robust control theory, the minimization of the Smoothed Spectral Abscissa (SSA).

A key aspect of the model is that it enforces Dale's law through its parameterization and defines memories only in the activity of the excitatory population. The activity of inhibitory neurons is treated as a set of auxiliary variables to be optimized, effectively using inhibition as a resource to stabilize the excitatory memory states. The authors validate this approach through numerical simulations, demonstrating that the resulting networks are robust to noise in both the initial cue and the ongoing dynamics. The optimized networks naturally operate in a balanced excitation-inhibition regime, exhibit sparse connectivity, and reproduce the experimentally observed reduction in trial-to-trial variability following stimulus onset.

**Strengths and Contributions**

*   **Novel Synthesis and Methodology:** The primary strength of this paper is its novel application of modern control theory (specifically, SSA minimization) to design biologically plausible associative memory networks. This provides a powerful, principled method for building networks that satisfy a host of important biological constraints in a unified manner, moving beyond previous models that typically address these constraints in isolation.
*   **Biologically Realistic Model:** The resulting network model is compellingly realistic. It incorporates Dale's law, supports graded (analog) memories, operates with non-saturating firing rates through a dynamic E/I balance, and develops sparse connectivity—all features observed in cortical circuits.
*   **Insightful Use of Inhibition:** The treatment of inhibitory neurons' activities as optimizable auxiliary variables is an elegant and insightful idea. This reframes the role of inhibition as a dynamic resource that can be marshaled to sculpt the network's dynamics and ensure the stability of multiple, graded memory states.
*   **Clarity and Quality:** The paper is clearly written, well-organized, and presents a complete piece of work. The claims are well-supported by the numerical experiments, which convincingly demonstrate the network's performance and properties.

**Weaknesses and Areas for Improvement**

*   **Lack of a Biologically Plausible Learning Rule:** The proposed method is a direct optimization algorithm for setting the network's parameters, not a local, synaptic learning rule. As the authors acknowledge, how a biological system could achieve such a solution remains an open question. While this does not detract from the paper's value as a proof of concept, it is an important limitation.
*   **Absence of Capacity Analysis:** The paper demonstrates successful storage for a given network size and memory load (e.g., 30 memories in a 150-neuron network). However, it does not investigate the network's storage capacity. A preliminary analysis of how recall performance scales with the number of stored memories would significantly strengthen the paper's contribution and allow for better comparison with classic associative memory models.

**Detailed Comments and Questions**

Overall, this is a clear, concise, and important contribution to computational neuroscience. The following are minor points and suggestions for improvement:

1.  The parameter γ in the neuronal gain function (Eq. 1) is set to 0.04. It would be helpful if the authors could comment on the sensitivity of the results to this specific choice.
2.  Memory patterns are generated from a log-normal distribution. While the motivation to match observed firing rate statistics is clear, could the authors provide more justification for this choice and discuss whether the framework is robust to other pattern distributions?
3.  The cost function (Eq. 5) minimizes ||dv/dt||², which may result in "slow points" rather than exact fixed points. The paper notes that true fixed points are found nearby. Could this deviation from the target patterns be quantified more rigorously? For instance, what is the typical magnitude of the drift at the target state, and what is the distance to the actual learned fixed point?
4.  The statements on lines 254-256 were unclear. Could the authors please rephrase this part for clarity?
5.  The paper does not explore the computational benefits of Dale's law itself. A brief discussion on this topic would be a welcome addition. For example, does enforcing this constraint provide any computational advantages (e.g., regarding stability or robustness) within this framework, or is it primarily a biological constraint that the optimization must work around?

In summary, this is an excellent paper that tackles a classic problem with modern tools, yielding new insights into how biologically realistic neural circuits might support memory.
