- Introduces a control-theoretic framework that enables associative memory with Dale’s law, analog (non-binary) representations, and non-saturating neural activity.  
- Key technical innovation: uses the Smoothed Spectral Abscissa (SSA) to enforce both fixed-point conditions and stability for arbitrary analog patterns.  
- Enforces Dale’s law via weight parameterization and treats inhibitory neurons as optimizable resources rather than fixed components.  
- Single-neuron saturation emerges naturally from the optimization rather than being imposed.  
- Cost function (three terms: fixed-point condition, stability, small weights) effectively balances competing requirements.  
- Resulting networks operate in a balanced regime, show robust noise tolerance, and reproduce reduced trial-to-trial variability after stimulus onset.  
- Paper is technically novel, well-validated numerically, clear, and well-organized.  
- Major gap: no biologically plausible learning rule is provided; inhibitory plasticity is only hinted at.  
- Parameter choice concern: γ = 0.04 appears arbitrary and needs sensitivity analysis and physiological justification.  
- Assumption of log-normal distribution for memory states lacks strong motivation; alternatives (Gaussian, Poisson) should be considered.  
- Quantify how much the cost function violates exact fixed-point constraints (‖dv/dt‖ remains small but non-zero).  
- Clarify statements about the relationship between the baseline state and memory patterns (lines 254–256).  
- Include a preliminary analysis of memory capacity (how many analog patterns can be stored before degradation).  
- Discussion of Dale’s law should be deepened to explain specific computational benefits (how excitation/inhibition balance enhances memory function).  
- Recommendation: accept with minor revisions to address the above concerns.
