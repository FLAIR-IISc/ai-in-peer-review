- Proposes a novel framework for biologically plausible auto-associative memory networks using rate-based recurrent networks that obey Dale’s Law and use non-saturating graded units.  
- Trains the network to store analog memory patterns as stable fixed points, departing from binary Hopfield-style models.  
- Uses a training procedure that combines gradient descent with the Smoothed Spectral Abscissa to directly optimize stability of desired memory states.  
- Emergent properties include E/I balance, sparse synaptic weight distributions centered near zero with long tails, and stimulus-driven quenching of trial-to-trial variability.  
- Manuscript is technically rigorous, clearly written, well-organized, and supported by effective simulations and figures.  
- Main weakness: insufficient justification for selecting the Smoothed Spectral Abscissa and lack of discussion about whether simpler or alternative methods could achieve similar stability.  
- Minor suggestion: include (or at least discuss) comparisons of memory capacity (stored memories vs. network size) with prior attractor network models.  
- Overall verdict: excellent, novel contribution; strong recommendation for acceptance.
