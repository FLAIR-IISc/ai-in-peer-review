

This paper addresses a longstanding challenge in computational neuroscience: building biologically plausible associative memory networks that satisfy key physiological constraints (Dale's law, graded activity, sparse connectivity, low firing rates). Prior NIPS work on memory networks, such as Hopfield (1982) and Treves (1990), either violated Dale's law (requiring neurons to be both excitatory/inhibitory), enforced binary patterns, or relied on saturating neuron models. Recent advances (Roudi et al., 2007; Hennequin et al., 2014) incorporated Dale's law but still required binary states or global inhibition. This paper makes a significant contribution by unifying these constraints through a novel control-theoretic framework. The core innovation is optimizing network parameters (via Smoothed Spectral Abscissa minimization) to stabilize arbitrary analog memory patterns as fixed points while maintaining balanced excitation-inhibition dynamics—a feat not achieved in prior work.

The paper’s strengths are substantial. It directly addresses multiple physiological constraints simultaneously: Dale’s law is enforced via a reparameterized weight matrix (Eq. 2), graded activity is preserved without artificial saturation (Fig. 1c), and sparse connectivity emerges naturally (Fig. 4b). The optimization cost function (Eq. 5) elegantly balances three objectives: fixed-point stability (minimizing |dv/dt|), robust stability (SSA minimization), and sparse weights (L2 regularization). Crucially, the model explains reduced trial-to-trial variability after stimulus onset—a ubiquitous experimental finding (Churchland et al., 2010)—by showing convergence to attractors (Fig. 6b). The numerical validation is thorough: recall robustness to noise (Fig. 3), sparse connectivity matching cortical data (Fig. 4), and balanced E/I inputs (Fig. 5) all demonstrate strong empirical support. The writing is exceptionally clear, with well-justified choices (e.g., log-normal memory distributions matching in vivo data) and effective figures.

Key weaknesses require attention. First, while the paper correctly identifies the absence of a biologically plausible learning rule as a limitation (Section 4), it offers only a vague suggestion ("inhibitory synaptic plasticity"). A brief discussion of how local rules could approximate the global optimization would strengthen the biological relevance. Second, the choice of γ=0.04 in the gain function (Eq. 1) lacks motivation; the authors should cite specific cortical rate data (e.g., from Anderson et al., 1998) supporting this value. Third, the log-normal distribution for memory patterns (Fig. 1a) is used without justification—why not Gaussian or another distribution? A sentence on this would clarify the modeling choice. Fourth, the fixed-point violation (|dv/dt| > 0 after optimization, Fig. 2) is acknowledged but not quantified. The authors should report the average |dv/dt| across patterns and discuss its impact on stability. Finally, the storage capacity claim ("0.2" in Fig. 2) is ambiguous; they should define it as memories per neuron (e.g., 30/150 = 0.2) and compare to theoretical limits (e.g., Treves, 1990).

Minor concerns include unclear phrasing on lines 254–256 (e.g., "the network was initialized prior to learning...") and insufficient discussion of Dale’s law’s computational role. While the paper correctly notes Dale’s law is biophysical, it could briefly address whether the constraint *enables* the observed properties (e.g., robustness) versus being merely evolutionary. This would deepen the significance.

Overall, this is a high-quality, original contribution that advances the field by resolving critical gaps in memory network modeling. The control-theoretic approach is elegant and well-executed, with results that bridge theory and experiment (e.g., explaining trial-variability reduction). The paper’s clarity and thorough validation make it a strong candidate for NIPS. The weaknesses are addressable in revision: the learning rule gap is the most critical, followed by parameter justifications and capacity discussion. Given the significance of the problem and the paper’s technical soundness, it merits acceptance with minor revisions. The work provides a crucial foundation for future studies on learning rules and cortical memory mechanisms.
