- Summary: Proposes a control-theoretic framework that uses Smoothed Spectral Abscissa (SSA) to optimize rate-based E-I networks to store stable analog memory fixed points under biological constraints.

- Key technical contribution: Uses SSA as a differentiable surrogate to optimize stability, overcoming non-differentiability of spectral abscissa.

- Biological constraints: Enforces Dale's law, sparse connectivity, graded activity, low firing rates, and balanced E-I dynamics in the optimized networks.

- Empirical validation: Demonstrates robust recall of log-normal analog memories under cue and ongoing noise, synaptic sparsity analysis, and comparison to an ideal observer.

- Biological interpretation: Provides evidence (e.g., Fig. 5a–c) that the optimized networks operate in a balanced E-I input regime with clear biological relevance.

- Advances prior work: Extends Hopfield-like and balanced-network literature by enabling analog memory storage in biologically constrained E-I networks.

- Strength — novelty: SSA-based stability optimization is a technically sophisticated and elegant solution to a key challenge.

- Strength — comprehensive constraints: Simultaneously satisfies multiple biological constraints not typically handled together in prior models.

- Strength — strong experiments: Thorough empirical analyses including noise robustness and ideal-observer comparisons.

- Strength — clear mapping to biology: Good connection between mathematical framework and biological principles, especially E-I balance.

- Strength — addresses limitation: Directly tackles the longstanding problem of biologically plausible analog memory storage.

- Weakness — scalability: Limited analysis of how capacity and optimization time scale with network size and memory load.

- Weakness — rate vs spiking gap: Insufficient evidence or demonstration that optimized rate-based connectivity would preserve stability in spiking implementations.

- Weakness — comparison to recent work: Inadequate contrast with recent analog-memory models (e.g., spike-timing based approaches such as [12]).

- Weakness — theoretical capacity: Lacks theoretical analysis or upper bounds on storage capacity under the imposed constraints.

- Weakness — hyperparameter choices: Hyperparameters (ηs, ηF, ε) were set heuristically with no presented sensitivity analysis.

- Weakness — learning plausibility: Does not address how optimized weights could be learned via local, biologically plausible rules.

- Suggestion — scalability analysis: Add experiments or analysis showing how optimization time and memory capacity scale with network size and number of memories.

- Suggestion — spiking translation: Provide a stronger argument or a small spiking-network simulation to justify transfer from rate to spiking models.

- Suggestion — comparative table: Include a table comparing this work to prior models on key properties (binary vs analog, Dale’s law, stability method, etc.).

- Suggestion — capacity bounds: Add a brief theoretical analysis of maximum storage capacity given Dale’s law and balanced dynamics.

- Suggestion — hyperparameter sensitivity: Report sensitivity analysis or justification for chosen optimization hyperparameters.

- Overall assessment: Strong, technically sound contribution that meaningfully advances biologically constrained analog-memory models despite limitations in scalability and biological learning plausibility.

- Recommendation: Accept — valuable and timely contribution likely to interest the NeurIPS community.

- Final comment: The work provides a promising framework for analog memory in balanced E-I networks and should stimulate further experimental and theoretical follow-ups.
