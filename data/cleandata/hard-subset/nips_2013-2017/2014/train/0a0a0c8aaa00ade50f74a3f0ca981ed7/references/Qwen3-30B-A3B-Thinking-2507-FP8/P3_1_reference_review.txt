

# Peer Review: "Analog Memories in a Balanced Rate-Based Network of E-I Neurons"

## Summary

This paper presents a novel control-theoretic framework for constructing neural networks that can store and recall analog memory patterns as stable fixed points, addressing several fundamental limitations of previous memory models in theoretical neuroscience. The authors propose optimizing network parameters to satisfy two conditions: (1) the desired activity states must be fixed points of the dynamics, and (2) the dynamics must be stable around those fixed points. They overcome the challenge of stability optimization by using the Smoothed Spectral Abscissa (SSA) as a differentiable surrogate for the spectral abscissa. The model respects biological constraints including Dale's law (separation of E and I neurons), sparse connectivity, graded neuronal activity, low firing rates, and a dynamic balance of excitation and inhibition. The authors demonstrate that their optimized networks can robustly recall analog memories (drawn from log-normal distributions) despite noise in both the recall cue and ongoing dynamics, with a balanced E-I input regime.

This work builds on the classical Hopfield model [1] and subsequent extensions that addressed binary memory patterns [2,3,4,5,6], but importantly differs by focusing on analog memory patterns within a biologically constrained E-I network framework. It extends recent work on balanced networks [7,8,9] and the use of robust control theory in neural networks [10,11] by directly optimizing for stable analog memory storage.

## Strengths

1. **Novel Approach to Stability Optimization**: The paper's most significant contribution is the use of SSA as a differentiable surrogate for stability optimization, which elegantly sidesteps the non-differentiability of the spectral abscissa. This is a technically sophisticated solution to a well-known challenge in the field.

2. **Comprehensive Biological Constraints**: The authors successfully incorporate multiple biological constraints simultaneously (Dale's law, graded activity, low firing rates, balanced E-I dynamics) that previous models have struggled to address. This represents a substantial advance over prior work.

3. **Strong Empirical Validation**: The paper provides thorough experimental validation, including memory recall performance under noise, analysis of synaptic sparsity, demonstration of balanced E-I inputs, and robustness to ongoing noise. The comparison to an "ideal observer" is particularly insightful.

4. **Clear Biological Interpretation**: The paper successfully connects the mathematical framework to biological principles, with Figures 5a-c providing compelling evidence of the balanced E-I regime that the authors claim to achieve.

5. **Addressing a Fundamental Limitation**: The paper directly addresses a longstanding limitation in theoretical neuroscience: the inability to construct biologically plausible networks that store analog memories with stable, balanced dynamics.

## Weaknesses and Critiques

1. **Limited Scalability Analysis**: The paper demonstrates results for a 150-neuron network storing 30 memories (capacity 0.2). While this is a proof-of-concept, the paper lacks analysis of how the method scales with network size or memory load. For instance:
   - How does storage capacity (memories per neuron) change as network size increases?
   - How does the optimization time scale with the number of memories?
   - What is the theoretical upper bound on storage capacity under these constraints?

2. **Rate-Based vs Spiking Model Gap**: The authors acknowledge the gap between rate-based and spiking networks in the discussion but don't sufficiently address how the optimized rate-based connectivity would transfer to a spiking model. While they mention that their gain function matches physiological input-output relationships, they don't provide evidence that the optimized network would maintain stability in a spiking implementation. This is a critical limitation for neuroscience applications.

3. **Comparison to Recent Analog Memory Work**: The paper cites [3,4] but doesn't adequately compare to more recent work on analog memory models. For instance, [12] (cited as "spike timing-based graded memory network") is mentioned but not properly contrasted with the authors' approach. A more thorough comparison would strengthen the claim of novelty.

4. **Lack of Theoretical Capacity Analysis**: The paper reports a capacity of 0.2 (30 memories for 150 neurons) but doesn't provide theoretical analysis of the maximum achievable capacity under their constraints. How does this compare to theoretical upper bounds for analog memory storage in E-I networks?

5. **Optimization Methodology Details**: While the paper describes the optimization framework, it lacks details on how the hyperparameters (ηs, ηF, ε) were chosen. The paper states these were "set heuristically" (Table 1), but doesn't explain the sensitivity analysis performed to justify these choices.

6. **Biological Plausibility of Optimization**: The authors optimize network parameters to achieve stable memories, but don't address how this could be achieved through local learning rules. The discussion mentions "local synaptic learning rules" as future work, but this is a significant gap for a neuroscience paper. How could a network learn these optimized weights through biologically plausible mechanisms?

## Specific Suggestions for Improvement

1. **Add a scalability analysis**: Include a brief analysis of how the optimization time and memory capacity scale with network size and number of memories. This would be valuable for assessing the practicality of the approach.

2. **Expand the spiking network discussion**: Provide a more concrete argument for how the rate-based connectivity could be translated to a spiking model. Consider simulating a small spiking version of the network to demonstrate stability.

3. **Improve comparison to related work**: Add a table comparing the key properties (binary vs analog, Dale's law, stability method, etc.) of the current work with previous models, including [3,4,12].

4. **Include theoretical capacity bounds**: Add a brief theoretical analysis of the maximum storage capacity under the constraints of Dale's law and balanced dynamics.

5. **Clarify hyperparameter choices**: Include a sentence or two about the sensitivity analysis performed for ηs, ηF, and ε, or add a small sensitivity analysis figure.

## Overall Assessment

This is a strong paper that makes a significant contribution to the theoretical neuroscience literature on memory models. The authors successfully address multiple longstanding limitations in the field by combining control theory with neuroscience constraints to create a biologically plausible model of analog memory storage. The methodological innovation in using SSA for stability optimization is particularly noteworthy.

The paper is technically sound, well-written, and makes a clear contribution to the field. While there are some limitations in scalability and biological plausibility of the optimization method, these are common challenges in theoretical neuroscience work and don't detract from the core contribution. The paper advances the state of the art in a demonstrable way by providing a framework that simultaneously satisfies multiple biological constraints while achieving robust analog memory storage.

## Recommendation

**Accept**

This paper represents a significant advance in the theoretical understanding of memory storage in neural networks. It successfully addresses multiple limitations of previous models while maintaining biological plausibility, and the experimental validation is thorough. While there are some limitations regarding scalability and the path to biological implementation, these are not fatal flaws for a theoretical contribution of this nature. The work provides a valuable framework that will likely influence future research on neural memory circuits.

The paper is well-suited for NeurIPS as it combines theoretical neuroscience with computational methods in a novel and rigorous way, addressing a fundamental question in neural computation. It represents a significant step forward in building biologically constrained models of memory that can handle the graded activity patterns observed in real neural circuits.

## Final Comments

The authors have made an important contribution to the field by demonstrating that analog memory storage with balanced E-I dynamics is possible within biologically constrained networks. This work should inspire new experimental approaches to studying memory circuits and provide a theoretical foundation for future computational neuroscience models. The paper is well-written, technically sound, and makes a clear, novel contribution to the field. I believe it will be of significant interest to the NeurIPS community.
