This paper introduces the expectation-loss SVM (e-SVM), a novel formulation for learning from data with continuous-valued labels in the [0, 1] range, which the authors term "positiveness". The core idea is to model the binary class label as an unobserved Bernoulli random variable, where the success probability is the known positiveness value. The loss function then becomes the expectation of the standard hinge loss over this random variable. This elegantly reframes the problem, allowing a single data point to contribute fractionally to both the positive and negative class constraints. The paper extends this to a weakly supervised setting (latent e-SVM) where the positiveness itself is unknown (e.g., when only bounding boxes are available for segmentation). In this case, the positiveness is treated as a latent variable and optimized alternately with the SVM weights. The method's efficacy is demonstrated on semantic segmentation and object detection, where it is used to learn classifiers for segment proposals. Notably, when integrated as a supplementary feature for the R-CNN object detector, it provides a significant performance boost on the PASCAL VOC 2007 benchmark.

**Strengths:**

1.  **Elegant and Principled Formulation:** The primary strength of the paper is the e-SVM formulation itself. It provides a theoretically sound and intuitive way to handle continuous labels between 0 and 1 within a max-margin framework. It avoids the ad-hoc thresholding required to use standard SVC and is arguably more suitable than SVR, which does not inherently account for the bounded nature of the positiveness score (e.g., IoU).

2.  **Practicality and Simplicity:** A major advantage of the basic e-SVM is that its final objective function (Equation 1) is equivalent to a weighted SVM problem. This means it can be optimized using standard, highly efficient off-the-shelf SVM solvers like LibLinear, making the approach easy to implement and deploy. This is a significant practical contribution.

3.  **Strong Empirical Validation:** The paper provides compelling experimental results. The newly proposed evaluation metrics for segment classification (TAPC and NDCG) are well-motivated and show that e-SVM outperforms both SVC and SVR on this sub-task. More importantly, the application to object detection demonstrates a non-trivial improvement (1.5% average AP) over R-CNN, which was a state-of-the-art baseline at the time. This shows that the learned segmentation cues provide valuable, complementary information even to powerful deep learning features.

4.  **Relevance of the Problem:** The work addresses a crucial problem in computer vision and machine learning: how to effectively learn from weak supervision. As datasets grow, the cost of dense, pixel-level annotation becomes prohibitive. Methods that can leverage cheaper annotations like bounding boxes are highly significant. The paper's problem setting is practical and of broad interest.

**Weaknesses and Areas for Improvement:**

1.  **Latent e-SVM Optimization:** The optimization for the latent e-SVM (Algorithm 1) is a coordinate descent-style approach. While standard, this method is susceptible to local minima. The paper acknowledges this but does not provide much analysis of this issue. The performance heavily relies on the initialization (ubb) and the KL-divergence regularizer `R(u)`. The choice of this specific regularizer, while sensible, feels somewhat arbitrary. A discussion of alternative regularizers or an ablation study on the effect of the regularization parameter λR would strengthen the paper.

2.  **Clarity on Novelty vs. Weighted SVMs:** The paper correctly relates its work to Weighted SVMs (WSVM). However, the distinction could be articulated more sharply. The key insight of e-SVM is that a single sample `xi` with positiveness `ui` contributes `ui` to the positive loss term and `(1-ui)` to the negative loss term. This is different from standard WSVM, which typically assigns a single weight to the entire hinge loss for a given sample. This subtle but important difference is the core of the contribution and could be emphasized more clearly in the related work section.

3.  **Details of the Object Detection Integration:** The mechanism for integrating the e-SVM output into the R-CNN pipeline—the "spatial confidence map feature"—is a key component of the detection results. However, its description is brief. For instance, the paper mentions encoding the feature "by additive kernels approximation mapping" but provides no further details on why this was done or how it was implemented. It is also slightly unclear whether the performance gain comes purely from the superiority of the e-SVM model or from the simple act of fusing segmentation-based features (regardless of how they are trained) with the R-CNN features. A baseline that uses SVR-trained confidence maps in the same feature fusion pipeline would make the contribution of e-SVM itself clearer.

4.  **Minor Notational Issues:** The notation for the regularization term in Equation 1, `λw · 1/wTw`, is non-standard. The conventional form is `(λ/2) ||w||^2` or `(λ/2) w^T w`. While the intent is clear, using standard notation would improve clarity.

**Summary of Arguments for/against Acceptance:**

*   **For Acceptance:**
    *   Proposes a novel, elegant, and well-motivated loss function (e-SVM) for an important problem.
    *   The core model is simple to implement using existing tools, enhancing its practical value.
    *   Demonstrates strong empirical results, including a significant improvement over a state-of-the-art object detection system.
    *   The problem of learning from weak supervision is highly relevant to the NIPS community.

*   **Against Acceptance:**
    *   The optimization for the latent variable version is a standard procedure with potential local minima issues that are not deeply analyzed.
    *   The experimental validation for the object detection task could be strengthened with more ablation studies to isolate the contribution of the e-SVM model versus the feature fusion strategy.
    *   The contribution's distinction from prior work on weighted SVMs could be made more explicit.

**Overall Assessment:**

This is a strong paper with a clear and valuable contribution. The e-SVM formulation is a clever idea that elegantly solves a real-world problem. The weaknesses identified are primarily related to a need for deeper analysis of the latent model and more detailed experimental ablation, but they do not undermine the core contribution. The paper is well-written, the experiments are convincing, and the results are significant. It advances the state of the art in a demonstrable way and presents ideas that are likely to be built upon by other researchers. The work is a good fit for NIPS, balancing a novel machine learning model with a high-impact application. I recommend acceptance.
