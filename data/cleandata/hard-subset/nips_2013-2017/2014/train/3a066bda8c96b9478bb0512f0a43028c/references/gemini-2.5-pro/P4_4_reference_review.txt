### Summary of the Paper

This paper introduces a novel learning algorithm called the expectation loss SVM (e-SVM) for weakly supervised learning, particularly in computer vision contexts where labels are continuous values within a bounded interval [0, 1]. The core idea is to model the label of a training example (e.g., an image segment) as a Bernoulli random variable, where the probability of being a positive instance is a function of its "positiveness" score (e.g., Intersection over Union with a ground truth object). The e-SVM loss function is then the expectation of the standard SVM hinge loss over this random label.

The authors present two versions of the model:
1.  **Basic e-SVM:** For the strongly supervised case where pixel-level annotations are available, the positiveness `u_i` of each segment is known. The loss function becomes a weighted combination of the positive and negative hinge loss terms for each sample, which can be solved using standard SVM solvers.
2.  **Latent e-SVM:** For the weakly supervised case where only bounding box annotations are given, the exact positiveness `u_i` is unknown. It is treated as a latent variable and optimized jointly with the SVM weights `w` through an alternating optimization procedure. A KL-divergence-based regularization term is introduced to guide the estimation of `u_i` and prevent trivial solutions.

The paper validates the proposed method on two fundamental computer vision tasks: semantic segmentation and object detection. For segment proposal classification, the e-SVM is shown to outperform standard Support Vector Classification (SVC) and Support Vector Regression (SVR) baselines on newly proposed metrics (TAPC and NDCG). For object detection, the authors use the latent e-SVM to learn a segment-based confidence score, which is then aggregated into a "spatial confidence map feature." This feature is used to augment the state-of-the-art R-CNN detector, leading to a notable 1.5% mean Average Precision (mAP) improvement on the PASCAL VOC 2007 dataset.

This work relates to prior work on weakly supervised learning, such as Multiple Instance Learning (MIL), but addresses a different setting where no single proposal is guaranteed to be fully positive. It also connects to the literature on weighted SVMs (WSVM), differentiating itself by weighting the positive and negative loss components for each sample independently, rather than weighting the entire sample.

### Strengths and Weaknesses

**Quality:** The paper is technically sound. The proposed e-SVM formulation is a principled extension of the standard SVM to handle continuous-valued labels representing "positiveness." The claims are well-supported by a thorough experimental evaluation. The authors test their method on standard, challenging benchmarks (PASCAL VOC 2011 and 2007) and compare against appropriate baselines (SVC, SVR). A key strength is the demonstration of improving a very strong, state-of-the-art object detection system (R-CNN), which substantiates the significance of the contribution. The introduction of new evaluation criteria (TAPC and NDCG) for the segment proposal ranking task is a thoughtful addition that allows for a more direct assessment of the model's performance on its primary task. The work appears complete and presents a well-rounded contribution from formulation to application.

**Clarity:** The paper is generally well-written and logically structured. The motivation for moving beyond thresholded SVC or standard SVR is clear. The progression from the basic e-SVM to the latent e-SVM is easy to follow. However, clarity could be improved in a few areas concerning reproducibility:
*   In Section 3.1, the paper states that a standard linear SVM solver can be used. While true, it would be beneficial to explicitly state how the expectation loss in Equation 1 is transformed into a weighted classification problem. For example, explaining that each sample `(x_i, u_i)` can be represented as two instances—a positive one `(x_i, +1)` with weight `g(u_i)` and a negative one `(x_i, -1)` with weight `1-g(u_i)`—would make implementation more straightforward.
*   In the latent e-SVM optimization (Algorithm 1, line 4), the minimization step with respect to `u` is not detailed. Given the regularization term, it is unclear if there is a closed-form solution for `u(new)` or if it requires a numerical solver. Providing this detail is crucial for others to reproduce the work.
*   The description of the spatial confidence map feature in Section 4.2 could be more detailed, particularly regarding the "additive kernels approximation mapping" step.

**Originality:** The core contribution—formulating the SVM loss as an expectation over a random label parameterized by a continuous "positiveness"—is novel. While related to weighted SVMs, the specific formulation where each sample contributes to both positive and negative loss components based on its positiveness `u_i` is a unique perspective. The extension to a latent variable model for weak supervision is a natural but effective next step. The application of this idea to generate supplementary features for a modern object detector is also a novel combination of techniques. The related work section adequately positions the paper with respect to MIL and WSVM literature, highlighting the key differences in problem setup and formulation.

**Significance:** The paper addresses a highly significant and practical problem: reducing the reliance on expensive pixel-level annotations by effectively learning from weaker forms of supervision like bounding boxes. The results are important for two reasons. First, the e-SVM provides a more principled way to handle continuous or noisy labels than ad-hoc thresholding (SVC) or direct regression (SVR). Second, the demonstrated ability to improve a state-of-the-art object detector like R-CNN shows that segment-based cues, even when learned from weak supervision, can provide complementary information to powerful deep feature representations. This suggests a promising direction for future work in combining different levels of supervision and feature types. The proposed method is general and could likely be applied to other domains beyond computer vision where labels are not strictly binary.

### Arguments for Acceptance:
*   **Pro:** Presents a novel and well-motivated e-SVM framework for learning with continuous-valued labels.
*   **Pro:** Addresses the significant and practical problem of learning from weak supervision, which is a major bottleneck in scaling up machine learning models.
*   **Pro:** The experimental results are strong, demonstrating clear improvements over relevant baselines on the direct task and, more importantly, boosting the performance of a state-of-the-art object detection system on a standard benchmark.
*   **Pro:** The introduction of new, suitable evaluation metrics for segment proposal classification is a valuable minor contribution.

### Arguments against Acceptance:
*   **Con:** The paper lacks some key implementation details regarding the optimization steps for both the basic and latent e-SVM, which hinders reproducibility.
*   **Con:** The choice of the linear function `g(u_i) = u_i` and the specific KL-divergence regularizer are not deeply justified or ablated, leaving questions about their optimality.
*   **Con:** The convergence properties of the alternating optimization for the latent e-SVM are not discussed.
