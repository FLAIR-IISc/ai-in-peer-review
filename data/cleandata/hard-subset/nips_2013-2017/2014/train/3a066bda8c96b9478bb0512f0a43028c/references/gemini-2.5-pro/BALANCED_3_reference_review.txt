### Review of "Learning From Weakly Supervised Data by The Expectation Loss SVM (e-SVM) algorithm"

**Summary**

This paper introduces a novel Support Vector Machine formulation, termed the expectation-loss SVM (e-SVM), designed to learn from data with continuous labels in the [0, 1] range, which the authors refer to as "positiveness." This scenario is common in computer vision tasks where, for instance, a segment proposal's quality is measured by its Intersection over Union (IoU) with a ground truth object. The core idea is to model the binary class label as a latent Bernoulli random variable, with its success probability equal to the known positiveness `u`. The e-SVM objective function is then the expectation of the standard SVM hinge loss over this latent variable.

The paper further extends this model to a weakly supervised setting where the positiveness `u` is unknown (e.g., when only bounding box annotations are available, not pixel-level masks). In this "latent e-SVM," the positiveness values are treated as latent variables and are optimized jointly with the SVM model parameters via an alternating minimization procedure. A key component of the latent model is a KL-divergence based regularization term that encourages the learned positiveness values to stay close to an initial estimate (e.g., IoU with the bounding box).

The authors demonstrate the effectiveness of their method on two fundamental computer vision tasks. First, on segment proposal classification, they introduce new evaluation metrics (TAPC and NDCG) and show that e-SVM outperforms standard Support Vector Classification (SVC) and Support Vector Regression (SVR). Second, and more significantly, they apply the latent e-SVM to improve a state-of-the-art object detection system (R-CNN). By generating a novel "spatial confidence map feature" from the e-SVM classifier and concatenating it with R-CNN features, they achieve a notable 1.5% average precision improvement on the PASCAL VOC 2007 dataset.

**Strengths (Arguments for Acceptance)**

1.  **Novel and Principled Formulation:** The e-SVM formulation is an elegant and intuitive way to handle continuous-valued labels within the max-margin framework. Framing the problem as an expectation over a latent binary label is a principled approach that leads to a simple, convex loss function. The extension to the latent e-SVM for weakly supervised learning is a logical and powerful generalization, addressing a highly relevant problem.

2.  **Significant and Timely Application:** The paper tackles the critical problem of reducing the reliance on expensive pixel-level annotations by effectively learning from weaker bounding-box supervision. The application to object detection is particularly compelling. Demonstrating a significant performance gain (1.5% average AP) over a very strong and contemporary baseline like R-CNN is a substantial achievement and clearly shows the practical value of the proposed method.

3.  **Thorough Experimental Validation:** The experiments are well-designed and convincing. The authors first validate the core e-SVM model on the sub-task of segment classification, even proposing new and appropriate metrics (TAPC, NDCG) for this specific problem. This direct comparison against SVC and SVR isolates and highlights the benefit of the proposed loss function. The subsequent application to semantic segmentation and object detection shows the utility of the method in complex, end-to-end vision pipelines.

4.  **Generality and Potential Impact:** The proposed e-SVM is a general model not limited to computer vision. It could be useful in any domain where supervision signals are uncertain or expressed as a continuous "degree of confidence." The idea of the spatial confidence map is also a valuable contribution, providing a concrete method for integrating segmentation-based cues into region-based object detectors.

**Weaknesses and Constructive Suggestions**

While this is a strong paper, there are a few areas where it could be improved for clarity and completeness.

1.  **Relation to Weighted SVMs:** The paper distinguishes e-SVM from weighted SVMs (WSVM) by stating it "weights labels instead of data points." This is a good conceptual distinction. However, to strengthen the paper's positioning, it would be beneficial to elaborate on the practical implementation. The e-SVM loss function, `Σ [u_i * l_i^+ + (1-u_i) * l_i^-]`, appears equivalent to a standard SVM formulation where each sample `x_i` is duplicated: once as a positive example `(x_i, +1)` with sample weight `u_i`, and once as a negative example `(x_i, -1)` with sample weight `1-u_i`. If this is the case, stating it explicitly would enhance clarity and reproducibility, as it shows how to implement e-SVM using off-the-shelf solvers that support per-sample weighting.

2.  **Justification for the Regularization Term:** In the latent e-SVM, the KL-divergence based regularizer `R(u)` is crucial for preventing the latent positiveness `u` from collapsing to binary values. The authors state this, but the paper would be stronger with an empirical ablation study. Showing the performance of the latent e-SVM with `λ_R = 0` or with a simpler regularizer (e.g., L2 distance `||u - u_bb||^2`) would provide concrete evidence for the superiority of the chosen KL-divergence regularizer and validate this important design choice.

3.  **Details on the Spatial Confidence Map Feature:** The creation of the spatial confidence map feature is a key component for the object detection results, but some implementation details could be clarified.
    *   When training the final detector for a specific class (e.g., 'car'), is the full `M x K` dimensional feature used, or only the `M` dimensional part corresponding to the 'car' confidence map? Clarifying this would help in understanding the feature's richness and the final classifier's complexity.
    *   The spatial binning parameter `m=5` is mentioned. It would be useful to know how this was chosen and how sensitive the final performance is to this parameter.

4.  **Minor Details:** A few of the citations in the reference list appear to be incomplete (e.g., [6], [8], [9]). A pass to clean up the bibliography would be appreciated for the final version.

**Decision**

Overall, this paper presents a novel, well-motivated, and empirically validated contribution to the field of weakly supervised learning. The strengths—particularly the elegant formulation and the significant improvement over a state-of-the-art object detection model—far outweigh the weaknesses, which are largely related to clarifying details. The work addresses an important problem and is likely to be impactful for researchers working on learning with limited supervision. I recommend acceptance.
