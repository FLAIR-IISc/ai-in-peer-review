- Overall: Proposes e-SVM, a novel SVM-based method for learning from weakly supervised continuous “positiveness” labels; recommend accept with minor revisions.  
- Core idea: Formulates an SVM loss that weights labels by positiveness values and extends to a latent-variable setting for unobserved positiveness.  
- Related work: Related to MIL and weighted SVMs; positioning is reasonable but should include more recent proposal-based weakly supervised methods (e.g., EM, contrastive approaches).  
- Novelty: e-SVM is a clever extension handling continuous label values and latent positiveness.  
- Theory: Provides clear mathematical formulation and analysis (convexity, relation to standard SVM solvers).  
- Empirics: Consistent improvements over SVC/SVR on segmentation and object detection; coupling with RCNN yields notable gains.  
- Practicality: Method is flexible and compatible with different proposal detectors, feature extractors, and detectors.  
- Writing: Paper is generally clear and well written.  
- Incremental gains: Improvements (e.g., ~1.5% AP) are modest and not revolutionary; more analysis on where gains occur is needed.  
- Regularization sensitivity: Latent e-SVM depends on a regularization term, but guidance on selecting it and its performance impact is lacking.  
- Comparative scope: Comparisons are limited to SVC/SVR; should include newer weakly-supervised methods (pseudo-labeling, self-training, etc.).  
- Feature fusion: Concatenating confidence maps with CNN features seems ad-hoc; other fusion strategies should be explored.  
- KL regularizer: Need a clearer, more intuitive explanation of why the KL divergence regularization prevents poor local minima.  
- Evaluation metrics: TAPC and NDCG are appropriate but require more justification versus standard metrics like AUC.  
- ImageNet pretraining: Claim that ImageNet would boost results is vague—authors should explain how they’d leverage ImageNet.  
- Figure 2: Spatial confidence map illustration is helpful but should include an overlaid visual example on an image.  
- Acceptance pros: Novel, theoretically sound, consistent empirical gains, general method, clear presentation.  
- Acceptance cons: Incremental performance, limited comparisons to recent work, insufficient hyperparameter tuning guidance.  
- Recommendation actions: Expand related work, detail parameter tuning (especially regularization), add comparisons to recent methods, elaborate on ImageNet usage, and add a visual example for Figure 2.
