The paper proposes a novel learning algorithm, called expectation loss SVM (e-SVM), that can effectively handle weakly supervised data where the labels of training data are continuous values with a bounded interval. The algorithm is well described, and the experimental results indicate that it may be valuable for semantic segmentation and object detection tasks.

The main idea of the e-SVM algorithm is to modify the loss function of the SVM by assigning a per-example weight to each training example. This weight is a function of the "positiveness" of the example, which is a continuous value between 0 and 1 that quantifies the affiliation of each training data to the positive class. The algorithm can be extended to the case when the per-example weight is not observed, i.e., it is a latent variable.

The experimental results show that the e-SVM algorithm outperforms the traditional SVM and Support Vector Regression (SVR) methods in the segment proposal classification task. The algorithm is also applied to two major tasks in computer vision: semantic segmentation and object detection. For semantic segmentation, the algorithm achieves comparable results to the state-of-the-art segment confidence prediction model using SVR. For object detection, the algorithm improves the average precision by 1.2 and 1.5 before and after applying bounding box regression, respectively.

The strengths of the paper include:

* The algorithm is well described, and the experimental results are convincing.
* The extension to the case when the per-example weight is not observed is interesting and has potential applications in weakly supervised learning.
* The algorithm is flexible and can be applied to various tasks in computer vision.

However, there are some weaknesses and concerns:

* The idea of assigning a per-example weight to each training example is not new and has been widely used in machine learning as "example dependent costs." The authors should clarify the difference between their approach and existing methods.
* The extension to the case when the per-example weight is not observed is non-convex, and no theoretical analysis is provided. The authors should comment on the use of a strong regularization term and its implications.
* The claim that the problem is not a standard regression problem because the positiveness belongs to a bounded interval [0, 1] is not accurate. Logistic regression is a standard method that is bounded to [0, 1].
* There is a minor spelling mistake in the abstract: "continues" should be "continuous."

In conclusion, while the paper proposes an interesting algorithm and demonstrates its effectiveness in various tasks, it lacks clarity on the novelty of the approach and the theoretical analysis of the extension to the case when the per-example weight is not observed.
