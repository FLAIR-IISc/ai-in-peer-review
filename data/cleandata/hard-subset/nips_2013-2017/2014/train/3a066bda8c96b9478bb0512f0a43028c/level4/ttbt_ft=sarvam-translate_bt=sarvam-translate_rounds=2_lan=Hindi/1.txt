The authors propose an Expectation Loss SVM (e-SVM) that can handle the continuous features of positive examples in a learning problem and apply the model to learn a visual segmentation classifier under weak supervision. This method is valid for both semantic segmentation and object recognition and has promising results. This paper addresses a significant problem in visual learning under weak supervision, the presentation is clear and well-motivated, and the experiments are interesting and well-executed. Although the technical content is an extension of the standard Multiple Instance Learning framework (by changing the binary with soft weighting of the positive), it applies in both seen and unseen settings and represents a valid contribution to the NIPS. An Expectation Loss SVM method that can handle the continuous features of positive examples in a learning problem and apply it to learn a visual segmentation classifier under weak supervision. Good motivation, clear presentation, and relevant results for semantic segmentation and object recognition.
