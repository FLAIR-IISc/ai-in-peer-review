The paper proposes a modification to the SVM learning algorithm to handle weakly supervised data by assigning a per-example weight, between 0 and 1, to each training example, and subsequently modifying the loss function to incorporate this weight. This approach is then extended to the case where the per-example weight is a latent variable.

The method is well-described, and the experimental results suggest that this approach may offer value over traditional methods that treat each example with equal weight, particularly for semantic segmentation and object detection tasks. However, the distinction between the proposed method and existing "example dependent costs" is unclear, as these are widely recognized and utilized in the machine learning community. For instance, SVMlight provides a similar mechanism.

The extension to the case where the per-example weight is unobserved is intriguing, albeit straightforward. This scenario results in a non-convex problem, yet no theoretical analysis is provided. Notably, a strong regularization term is employed, implying that the per-example weights do not deviate significantly from their initial values. It would be beneficial for the authors to comment on this aspect.

On a minor note, the claim on line 50 that "it is also not a standard regression problem since the positiveness belongs to a bounded interval [0, 1]" is inaccurate, as logistic regression is a standard approach that is indeed bounded to the interval [0, 1].

Additionally, there is a minor spelling mistake in the abstract, where "continues" should be replaced with "continuous". Overall, the paper proposes a modified SVM learning algorithm with a per-example weight, but it is unclear how this differs from existing example-dependent costs, which are widely used in machine learning.
