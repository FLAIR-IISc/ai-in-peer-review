The authors propose an Expectation Loss SVM (e-SVM) that can handle the continuous features of positive examples in a learning problem and can be applied to model visual segmentation classifiers to learn under weak supervision. The method is valid for both semantic segmentation and object recognition and has yielded promising results. This paper discusses an important problem in visual learning under weak supervision. The presentation is clear and well-motivated, and the experiments are interesting and well-conducted. Although the technical content is an extension of the standard Multiple Instance Learning framework (using a soft weighting of positives instead of binaries), it applies to both observed and latent cases, and presents a valid contribution to NIPS. An Expectation Loss SVM method that can handle the continuous features of positive examples in a learning problem and can be applied to model visual segmentation classifiers to learn under weak supervision. Good motivation, clear presentation, semantic segmentation, and relevant results for object recognition.
