- Proposes Expectation Loss SVM (e-SVM) to handle weak supervision where label "positiveness" is a continuous value in a bounded interval.  
- Core idea: modify SVM loss to be the expectation over unobserved binary labels modeled as Bernoulli variables with probability µi from positiveness ui.  
- Introduces latent e-SVM to handle unobserved positiveness ui by treating it as a latent variable optimized with a regularization term.  
- Demonstrates superior performance to SVC and SVR on segment proposal classification.  
- Shows effectiveness on semantic segmentation and object detection, achieving state-of-the-art detection on PASCAL VOC 2007.  
- Strength: clear, concise algorithm description and thorough experimental evaluation.  
- Strength: discusses related work and connections to instance-weighted SVMs.  
- Weakness: unclear distinction from existing "example-dependent costs" approaches; needs clearer comparison.  
- Weakness: latent-extension is straightforward with no provided theoretical analysis.  
- Weakness: heavy regularization in latent model suggests per-example weights hardly deviate from initial values.  
- Minor: spelling error in abstract ("continues" → "continuous").  
- Minor: incorrect claim that logistic regression is not standard for bounded [0,1] outputs; logistic outputs are bounded and commonly used.
