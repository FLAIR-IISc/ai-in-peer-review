- Proposes expectation-loss SVM (e-SVM) modeling binary labels as Bernoulli variables with known continuous "positiveness" scores and using expected hinge loss.  
- e-SVM reduces to a weighted SVM where each sample contributes soft positive/negative weights.  
- Extends to latent e-SVM for weak supervision by treating positiveness scores as latent and optimizing them jointly with classifier weights via alternating optimization.  
- Latent e-SVM is a sensible MIL-like extension that replaces binary-instance assumptions with continuous soft weights.  
- Validated on semantic segmentation and object detection; e-SVM produces spatial confidence map features that, when combined with R-CNN, improve PASCAL VOC 2007 detection.  
- Addresses an important, practical weak-supervision problem and naturally handles continuous-quality labels like IoU.  
- Presentation is exceptionally clear and the progression from e-SVM to latent e-SVM is logical and easy to follow.  
- Experimental evaluation is thorough, introduces useful metrics (TAPC, NDCG) for segment proposal classification, and shows consistent gains over SVC/SVR baselines.  
- Method is practical since it can be solved with standard SVM solvers and integrates well with existing detectors.  
- Main novelty is incremental: e-SVM is essentially a specific weighted SVM and latent e-SVM resembles existing latent/alternating SVM approaches.  
- Alternating optimization only guarantees local minima; paper lacks deeper analysis of optimization landscape, initialization sensitivity, and convergence behavior.  
- Combining segmentation cues with R-CNN via simple feature concatenation is ad-hoc; more principled integration could be explored.  
- Pro-accept: important problem, elegant/practical solution, strong experimental gains, and clear writing.  
- Con-accept: core idea is an incremental combination of familiar techniques rather than a fundamental breakthrough.  
- Conclusion: solid, well-executed contribution with convincing results; recommendation for acceptance.
