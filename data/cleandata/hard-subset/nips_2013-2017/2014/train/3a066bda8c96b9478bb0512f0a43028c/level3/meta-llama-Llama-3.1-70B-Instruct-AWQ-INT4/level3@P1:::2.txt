This paper proposes a novel learning algorithm called expectation loss SVM (e-SVM) that can effectively learn from weakly supervised data where the labels are continuous values within a bounded interval. The algorithm is designed to address the problem of segment proposal classification, where the positiveness of each segment proposal is quantified by a continuous value between 0 and 1. The authors demonstrate the effectiveness of their approach on two major computer vision tasks: semantic segmentation and object detection.

The e-SVM algorithm is an extension of the classical Support Vector Classification (SVC) model, where the label of each segment is treated as an unobserved random variable. The algorithm uses a linear function as the prediction function and adopts a loss function that is the expectation over the random variables. The loss function is convex with respect to the model parameters, making it solvable using standard SVM solvers.

The authors also propose a latent extension of the e-SVM model, which can handle the situation where the positiveness of each segment proposal is unobserved. This is achieved by treating the positiveness as a latent variable and using a regularization term to prevent the algorithm from getting stuck in local minima.

The experimental results show that the e-SVM algorithm outperforms the SVC and Support Vector Regression (SVR) methods on the segment proposal classification task, using two new evaluation criteria: Threshold Average Precision Curve (TAPC) and Normalized Discounted Cumulative Gain (NDCG). The authors also demonstrate the effectiveness of their approach on semantic segmentation and object detection tasks, achieving state-of-the-art object detection performance on the PASCAL VOC 2007 dataset.

* The paper proposes a novel learning algorithm that can effectively learn from weakly supervised data.

* The latent extension of the e-SVM model is an interesting contribution, allowing the algorithm to handle unobserved positiveness.

Weaknesses:

* The novelty of the method is unclear, as it appears equivalent to existing "example-dependent costs" methods (e.g., SVMlight).
* The optimization of the latent extension is non-convex, and no theoretical analysis is provided.
* Heavy regularization is used, which may limit the algorithm's ability to learn from the data.
