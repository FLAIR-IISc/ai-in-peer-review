This paper proposes a novel learning algorithm called expectation loss SVM (e-SVM) that can handle situations where only the "positiveness" of a binary label is available, rather than the binary label itself. The "positiveness" is a continuous value that quantifies the affiliation of each training data to the positive class. The e-SVM algorithm can also be extended to learn segment classifiers under weak supervision where the exact positiveness value of each training example is unobserved.

The main idea of the e-SVM algorithm is to treat the label of each segment as an unobserved random variable and assume that it follows a Bernoulli distribution. The probability of the label being positive is denoted as µ, which is a function of the positiveness. The loss function of the e-SVM is the expectation over the random variables, which is equivalent to "weighting" each sample with a function of its positiveness.

The authors evaluate the performance of the e-SVM algorithm on segment proposal classification, semantic segmentation, and object detection tasks. They show that the e-SVM algorithm outperforms the alternative approaches, including Support Vector Classification (SVC) and Support Vector Regression (SVR), in both strong and weak supervision scenarios. They also demonstrate that the e-SVM algorithm can be applied to object detection tasks by coupling the appearance cues with the base detector.

The strengths of the paper include:

* The e-SVM algorithm is a novel and well-motivated extension of multiple instance learning, which can handle continuous attributes of positive instances via soft weighting.
* The algorithm is validated on several tasks, including semantic segmentation and object detection, with promising results.
* The presentation is clear and well-motivated, and the experiments are interesting and well-executed.

However, there are also some weaknesses:

* The algorithm is not thoroughly compared to other state-of-the-art methods in the field, such as deep learning-based approaches.
* The regularization term used in the latent e-SVM model is not thoroughly justified, and its impact on the performance of the algorithm is not fully explored.
* The paper could benefit from a more detailed analysis of the computational complexity of the algorithm and its scalability to large datasets.

Overall, the paper presents a novel and well-motivated algorithm that can handle continuous attributes of positive instances via soft weighting. The algorithm is validated on several tasks, including semantic segmentation and object detection, with promising results. However, further comparisons to other state-of-the-art methods and a more detailed analysis of the computational complexity of the algorithm are needed to fully assess its potential.
