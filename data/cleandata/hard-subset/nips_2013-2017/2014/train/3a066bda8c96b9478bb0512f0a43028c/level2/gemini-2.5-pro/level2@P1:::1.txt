This paper introduces a novel SVM formulation, the expectation loss SVM (e-SVM), designed for learning problems where training labels are not binary but are continuous "positiveness" values within a bounded interval (e.g., [0, 1]). This situation is common in computer vision, where the Intersection over Union (IoU) between a segment proposal and a ground truth object provides such a continuous label. The core idea is to treat the true binary label of each sample as an unobserved Bernoulli random variable, where the success probability is a function of the known positiveness. The e-SVM then minimizes the expectation of the standard hinge loss over this latent binary label. This formulation results in a convex optimization problem that can be solved using standard SVM solvers.

The authors further extend this model to a weakly supervised setting, the latent e-SVM, where the positiveness values themselves are unobserved. This is particularly relevant for tasks like object detection where only bounding box annotations are available. In this case, the positiveness values are treated as latent variables and are optimized jointly with the model weights via an alternating minimization procedure, regularized to stay close to an initial estimate (e.g., IoU with the bounding box).

The proposed method is validated on segment proposal classification, semantic segmentation, and object detection. For object detection, the authors demonstrate that features derived from a latent e-SVM trained on segmentation proposals can be used to augment the R-CNN detector, achieving a new state-of-the-art result on the PASCAL VOC 2007 dataset.

**Strengths:**

1.  **Novel and Elegant Formulation:** The e-SVM provides a principled and well-motivated approach to handle continuous-valued supervision in a classification setting. Framing the problem as minimizing an expected hinge loss is an elegant idea that connects regression-like labels (the positiveness `u_i`) to a classification loss.
2.  **Practicality and Generality:** The method addresses a common and practical problem, especially in computer vision where pixel-level annotations are expensive to obtain. The extension to the latent e-SVM for weakly supervised learning from bounding boxes is a key contribution that significantly broadens the applicability of the method. The fact that it can be solved with standard solvers is a major practical advantage.
3.  **Strong Empirical Results:** The paper demonstrates the effectiveness of the proposed method through comprehensive experiments. The introduction of new, direct evaluation metrics for segment proposal classification (TAPC and NDCG) is a good contribution. Most importantly, the improvement over the very strong R-CNN baseline on the PASCAL VOC 2007 object detection benchmark is significant and demonstrates the practical value of the proposed approach. A 1.5% mAP gain on this competitive task is a substantial result.
4.  **Significance:** This work provides a useful tool for the weakly supervised learning community. The ideas presented are likely to be built upon by other researchers, and the method could be applied to other domains where continuous confidence scores are available instead of hard labels.

**Weaknesses and Questions for the Authors:**

1.  **Comparison to Weighted SVMs:** The paper states that e-SVM is different from weighted SVMs (WSVM) because it "weights labels instead of data points." While intuitively clear, this distinction could be made more formal. The e-SVM loss function in Eq. (1) can be interpreted as applying different weights to the positive and negative slack variables for a single data point. This is closely related to cost-sensitive SVMs or WSVM formulations where one might create two copies of each data point `x_i`—one with label +1 and weight `g(u_i)`, and another with label -1 and weight `1-g(u_i)`. A more detailed discussion or derivation clarifying the exact relationship and potential advantages over such a WSVM baseline would strengthen the paper.
2.  **Details of the Latent Model:** The optimization of the latent e-SVM relies on a regularization term R(u) to prevent `u_i` from collapsing to 0 or 1. The choice of a KL-divergence based regularizer is reasonable but seems somewhat arbitrary. It would be beneficial to discuss the sensitivity of the model to this choice and to the hyperparameter `λ_R`. Are there other plausible regularizers, and how might they affect performance?
3.  **Reproducibility of Features:** The description of the "spatial confidence map feature" used for object detection could be more detailed. For a given test image, how are the segment proposals generated? Is it the same mechanism (e.g., CPMC) as in training? Providing more precise details about the feature extraction process (e.g., the number of segments, the pooling strategy, the kernel approximation) would improve the paper's clarity and make the results easier to reproduce.
4.  **Clarity of Evaluation:** In Section 5.1.2, the results are presented for models trained with "clean pixel-wise object class labels" and "weakly-labelled bounding-box annotation". Is the latter case referring to the latent e-SVM? This could be stated more explicitly for clarity.

In summary, this paper presents a valuable and novel contribution to the field of weakly supervised learning. The proposed e-SVM and latent e-SVM models are well-motivated, technically sound, and empirically shown to be effective on challenging computer vision tasks, leading to state-of-the-art results. The weaknesses are minor and mainly relate to requests for additional clarification and analysis that would further strengthen an already solid paper. I recommend acceptance.
