

This paper introduces a novel deep recursive neural network architecture that achieves state-of-the-art results on a challenging sentiment classification dataset. The contributions are well-motivated, the experimental validation is thorough, and the writing is exceptionally clear. The proposed architecture demonstrates significant improvements over prior recursive neural network (RNN) approaches through four key innovations: (1) untied weights for leaf nodes (representing individual words) and nonterminal nodes (representing phrase compositions), enabling more flexible modeling; (2) the adaptation of rectified linear units (ReLUs) to the recursive setting, which enhances representational capacity; (3) the use of large unsupervised word vectors (e.g., pre-trained embeddings) alongside compact hidden states for nonterminal nodes, balancing efficiency and expressiveness; and (4) a deep architecture that introduces explicit inter-layer connections between tree nodes rather than merely stacking layers, facilitating richer hierarchical information flow.  

The experimental results are compelling, with the architecture outperforming strong baselines across multiple metrics. However, a critical methodological issue requires attention: Table 1(a) reports results on the test set rather than the development set, implying that hyperparameters were tuned using the test data. This practice invalidates the evaluation and must be corrected. The authors should re-run experiments using the development set for tuning and report final results on a held-out test set.  

Additionally, the design choice to include connections from matrix *V* (which maps hidden states to the next layer) raises an interesting question. It would be valuable to explore whether the performance would remain robust if this connection were removed, using only the last hidden layer’s output as input to the parent node. This would clarify whether the inter-layer connections are essential or if a simpler design suffices. While the current architecture’s design is well-justified, this comparison would strengthen the analysis.  

The paper’s clarity and technical soundness are its strongest assets. The motivation for each modification is thoroughly explained, and the ablation studies convincingly demonstrate their individual contributions. The authors effectively address common pitfalls in recursive neural networks, such as overparameterization and suboptimal activation functions. The use of unsupervised word vectors while keeping hidden states small is particularly pragmatic, aligning with recent trends in efficient NLP model design.  

Minor improvements would further enhance the paper. The description of the deep architecture’s connectivity (Section 3.2) could be clarified to explicitly distinguish it from naive layer stacking. A brief discussion of computational efficiency (e.g., training time or memory usage) would also be beneficial, as these factors are increasingly important for practical deployment.  

In summary, this is a high-quality contribution that advances the state of the art in recursive neural networks for sentiment analysis. The core ideas are innovative, the experiments are rigorous, and the writing is exemplary. The only critical revision required is to correct the evaluation protocol (development set for tuning, test set for final reporting). With this fix, the paper would be suitable for acceptance at NIPS. The work provides a strong foundation for future research on hierarchical neural models and demonstrates how careful architectural choices can yield substantial gains on difficult NLP tasks.
