- New deep recursive neural architecture achieving state-of-the-art on a challenging sentiment classification dataset.  
- Results are impressive and the paper’s exposition is very clear.  
- Untying leaf and nonterminal node parameters in the model.  
- Employing rectified linear units (ReLUs) within the recursive framework.  
- Using large unsupervised word vectors together with smaller hidden nonterminal nodes.  
- Introducing a deep architecture with connections between layers and tree nodes (not just replacing a single-layer RNN with a deep layer); suggest comparing to a variant that drops the connection from matrix V (only last hidden layer output feeds parent).  
- Methodological concern: Table 1(a) results appear tuned on the test set and should have been reported on the development set to avoid compromised evaluation.  
- Minor typo: "classificationd" → "classification".
