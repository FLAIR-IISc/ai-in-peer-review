- Paper presents a deep recursive neural architecture that achieves state-of-the-art sentiment classification and is clearly written.
- Proposed modifications are thoughtful, well motivated, and empirically validated.
- Untying parameters for leaf nodes and nonterminal (internal) nodes.
- Using rectified linear units (ReLUs) in the recursive composition.
- Leveraging large unsupervised word vectors while keeping smaller hidden representations at internal nodes.
- Introducing a genuinely deep recursive architecture with cross-layer connections and connections between tree nodes (not just a deeper single RNN layer).
- Recommend an ablation that passes only the last hidden layer to the parent (dropping the V matrix) to isolate the effect of cross-layer connections.
- Evaluation protocol concern: Table 1(a) should report development-set results; using the test set for model selection risks overfitting and invalidates test evaluation.
- Minor wording/typos to fix: hyphenate “state-of-the-art”; correct “classificationd” → “classification”; change “comprise a class of architecture” → “comprise/constitute a class of architectures”; avoid contractions and ensure consistent capitalization/punctuation.
- Overall assessment: exciting, well-executed work with strong empirical gains; addressing the evaluation issue and including the suggested ablation would further strengthen the paper.
