This paper presents a novel deep recursive neural architecture that achieves state-of-the-art results on a challenging sentiment classification dataset. The proposed architecture yields impressive performance, and its explanation is remarkably clear.

The modifications to existing RNN models are ingenious, well-motivated, and experimentally validated:

* Decoupling leaf nodes and non-terminal nodes
* Employing rectified linear units in the recursive setting
* Utilizing large unsupervised word vectors alongside smaller hidden non-terminal nodes
* Implementing a deep architecture that introduces connections between layers and tree nodes, rather than simply replacing a single RNN layer with a deep layer

It would be intriguing to compare the results if the output of the last hidden layer at each node was the sole input to the next parent node, effectively removing the connection from matrix V.
