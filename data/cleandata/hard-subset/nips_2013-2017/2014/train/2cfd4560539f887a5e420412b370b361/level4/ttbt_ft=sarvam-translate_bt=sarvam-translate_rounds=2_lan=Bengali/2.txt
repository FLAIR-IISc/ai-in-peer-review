This fascinating paper introduces a new deep recurrent neural architecture that achieves state-of-the-art performance on challenging sentiment classification datasets.

The results of their new architecture are impressive, and the presentation is very clear!

The changes they propose to previous RNN models are clever, well-inspired, and experimentally work well:

- Separating leaf nodes and non-terminal nodes.
- Using modified linear units in the recurrent setting.
- Using large unsupervised word vectors but using small hidden non-terminal nodes.
- Using a deep architecture that not only replaces a single neural network RNN layer with a deep layer but also establishes connections between the layers and tree nodes. -- An interesting comparison would be if the output of the last hidden layer in each node were the sole input to the next parent, i.e., the connection was removed from the matrix V.
