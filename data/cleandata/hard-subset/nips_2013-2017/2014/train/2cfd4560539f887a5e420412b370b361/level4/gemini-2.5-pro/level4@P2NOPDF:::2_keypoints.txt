- Presents a novel deep recursive neural network achieving state-of-the-art sentiment results; paper is clear and well-written.
- Untied parameters: separate matrices for leaf (words) and non-terminal (phrases) nodes let the model learn distinct compositional functions.
- ReLUs: successful application of rectified linear units in a recursive architecture.
- Efficient representation sizing: combines large pre-trained word vectors with smaller hidden dims for non-terminals to balance semantics and efficiency.
- Novel deep architecture: introduces cross-layer connections between corresponding parent and child layers across the parse tree for integrated deep compositionality.
- Major concern: hyperparameters appear tuned on the test set (Table 1a), risking inflated performance estimates; requires clarification and re-evaluation using a proper dev/test split.
- Requested ablation: run an experiment removing skip-layer (V) connections so parents receive only children’s final hidden layer to quantify the benefit of deep connections.
- Minor issue: typo—“comprise a class of architecture” is grammatically incomplete.
