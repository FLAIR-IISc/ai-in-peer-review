This exciting paper introduces a new deep recursive neural architecture that achieves state-of-the-art performance on a challenging sentiment classification dataset. The results obtained from this novel architecture are impressive, and the exposition is very clear.

The proposed changes to previous RNN models are clever, well-motivated, and experimentally demonstrated to be effective. These changes include: untying the weights of leaf nodes and nonterminal nodes, utilizing rectified linear units in the recursive setting, leveraging large unsupervised word vectors in conjunction with smaller hidden nonterminal nodes, and employing a deep architecture that introduces connections between layers and tree nodes. It would be an interesting comparison to investigate the impact of dropping the connection from matrix V, such that the outputs of the last hidden layer at each node are the only inputs to the next parent.

However, there are a few issues that need to be addressed. The results in Table 1 (a) should have been reported on the development set, not the final test set. This is because the authors are effectively tuning hyperparameters on the test set, which is not a standard practice.

Additionally, there are a few typos that should be corrected, including "comprise a class of architecture" and "sentiment classificationd dataset".
