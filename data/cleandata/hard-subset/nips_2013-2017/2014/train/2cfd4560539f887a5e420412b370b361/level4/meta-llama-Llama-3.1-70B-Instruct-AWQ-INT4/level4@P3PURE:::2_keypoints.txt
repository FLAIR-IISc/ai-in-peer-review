- Introduces a new deep recursive neural architecture achieving state-of-the-art performance on a challenging sentiment classification dataset.  
- Results are impressive and the paper’s exposition is very clear.  
- Unties leaf and nonterminal node parameters, allowing more flexible representations.  
- Uses rectified linear units (ReLU) in the recursive setting to enable more efficient training.  
- Combines large unsupervised word vectors with smaller hidden nonterminal nodes to balance representation capacity and model complexity.  
- Designs a deep architecture with connections between layers and tree nodes rather than a naïve deep replacement; suggests comparing the effect of removing the V-matrix connection so only the last hidden-layer outputs feed the parent.  
- Methodological concern: Table 1(a) results appear reported on the test set rather than the development set, implying hyperparameter tuning on the test set and risking overly optimistic estimates.  
- Minor typos to fix (e.g., "comprise a class of architecture" and "sentiment classificationd").
