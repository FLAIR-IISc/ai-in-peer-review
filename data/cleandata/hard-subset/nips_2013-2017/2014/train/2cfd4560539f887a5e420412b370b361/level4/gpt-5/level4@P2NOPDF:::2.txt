This paper presents a deep recursive neural architecture that achieves state-of-the-art performance on a challenging sentiment classification dataset. The results are compelling, and the paper is clearly written.

The proposed modifications to prior RNN/recursive models are thoughtful, well motivated, and empirically validated:
- Untying parameters for leaf nodes and nonterminal (internal) nodes.
- Using rectified linear units in the recursive composition.
- Leveraging large, unsupervised word vectors while keeping smaller hidden representations at internal nodes.
- Introducing a genuinely deep recursive architecture with connections both across layers and between tree nodes, rather than merely replacing a single RNN layer with a deeper one.

A useful additional comparison would be a variant in which only the last hidden layer at each node is passed to the parent (i.e., dropping the connection via matrix V). This ablation would help isolate the contribution of cross-layer connections to parent nodes.

Evaluation protocol:
- Table 1(a) should report development-set results. Using the final test set for model selection or tuning risks overfitting and compromises the validity of the test-set evaluation.

Minor wording/typos:
- Use “state-of-the-art” (hyphenated).
- “classificationd” → “classification.”
- “comprise a class of architecture” → “comprise a class of architectures” or “constitute a class of architectures.”
- Avoid contractions and ensure consistent capitalization and punctuation throughout.

Overall, this is an exciting and well-executed paper. The architectural choices are justified and effective, the empirical gains are strong, and the presentation is clear. Addressing the evaluation issue in Table 1(a) and including the suggested ablation would further strengthen the work.
