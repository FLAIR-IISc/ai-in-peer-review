This exciting paper introduces a new deep recursive neural architecture that achieves state-of-the-art performance on a challenging sentiment classification dataset. The results are impressive, and the exposition is very clear.

The proposed modifications to prior RNN/recursive models are clever, well motivated, and empirically validated:
- Untying parameters between leaf nodes and nonterminal (internal) nodes.
- Using rectified linear units (ReLUs) in the recursive setting.
- Leveraging large unsupervised word vectors while using smaller hidden dimensions for nonterminal nodes.
- Employing a deep architecture that does more than simply replace a single RNN layer with a deeper layer; it also introduces connections between layers and tree nodes. It would have been an interesting comparison if the outputs of only the last hidden layer at each node were passed to the parent—i.e., dropping the connection from matrix V.

One concern: Table 1(a) results should be reported on the development/validation set rather than on the final test set. As presented, this amounts to tuning on the test set, which can inflate reported performance and undermines fair evaluation.
