- Introduces a new deep recursive neural architecture that achieves state-of-the-art on the Stanford Sentiment Treebank.
- Exposition is clear and the paper is well-written.
- Key modification: unties leaf-node parameters from non-terminal node parameters.
- Key modification: uses rectified linear units (ReLUs) in the recursive setting.
- Key modification: employs large unsupervised word vectors with smaller hidden non-terminal nodes.
- Key modification: uses a deep architecture with inter-layer and tree-node connections (not merely a deeper single RNN layer).
- Missing/necessary comparison: evaluate dropping the V matrix connection so only the last hidden layer feeds the parent.
- Empirical result: deep RNNs outperform shallow RNNs of the same size and baselines (multiplicative RNN, paragraph vectors).
- Qualitative finding: different layers capture different compositionality aspects—layer 1: local word features; layer 2: sentiment intensity; layer 3: most abstract sentiment.
- Minor criticism: hyperparameters appear tuned on the test set (Table 1a should report dev set), risking overfitting.
- Minor issue: a few typos (e.g., "comprise" → "comprises").
- Strengths: novel architecture, strong empirical performance, clear writing, and insightful qualitative analysis.
- Weaknesses: insufficient comparisons/ablations (e.g., removing V) and questionable hyperparameter tuning practice.
- Overall verdict: strong paper; rating 9/10, confidence 8/10.
