- New deep recursive neural network achieves state-of-the-art performance on a challenging sentiment classification dataset.  
- Results are impressive and the paper is exceptionally well-written and clear.  
- Authors’ modifications to previous RNNs are insightful, well-motivated, and effective.  
- Disentangling weights of leaf and non-terminal nodes is a key architectural change.  
- Using rectified linear units (ReLU) within the recursive framework improves performance.  
- Leveraging large pre-trained word vectors with smaller hidden representations for non-terminals is beneficial.  
- Introducing a genuinely deep architecture with connections between layers and tree nodes is novel; compare to a variant that only passes the last hidden layer’s output to the parent (removing V-mediated connection).  
- Correction needed in Table 1(a): reported results should be from the development set, not the test set, since hyperparameter tuning appears to use the test set.  
- Minor issues: grammatical error in "comprise a class of architecture" and typo "classificationd".
