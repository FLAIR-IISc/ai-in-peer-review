

This paper presents an exciting new deep recursive neural architecture that achieves state-of-the-art results on a challenging sentiment classification dataset. The proposed model is well-motivated and delivers compelling empirical performance, with exceptionally clear exposition throughout.

The authors introduce four key innovations that effectively address limitations of prior recursive neural networks: (1) untied parameters for leaf and non-terminal nodes, (2) the use of rectified linear units (ReLU) in the recursive context, (3) leveraging large unsupervised word vectors while maintaining smaller hidden state dimensions for non-terminal nodes, and (4) a deep architecture that establishes meaningful inter-layer connections between tree nodes rather than merely stacking layers. The fourth point is particularly noteworthy—the model's design actively incorporates connections across layers, not just sequential propagation. It would be valuable to see a comparison where only the last hidden layer's output at each node feeds into the parent (i.e., removing the matrix V connections), as this would clarify the contribution of the proposed inter-layer mechanisms.

The evaluation requires one critical correction: Table 1(a) reports results on the test set, not the development set. This constitutes a standard evaluation error, as hyperparameter tuning should occur on a held-out development set to prevent test set leakage. The reported performance likely overestimates generalization ability.

Minor typographical issues include: "classificationd" (should be "classification"), and the fragment "comprise a class of architecture" appearing mid-sentence. These are easily resolved in revision.

Overall, the technical contributions are significant and well-validated. The model design demonstrates thoughtful consideration of architectural trade-offs, and the results provide strong evidence for the proposed approach. The authors should address the evaluation protocol issue before publication. The work represents a meaningful advancement in recursive neural networks for sentiment analysis.
