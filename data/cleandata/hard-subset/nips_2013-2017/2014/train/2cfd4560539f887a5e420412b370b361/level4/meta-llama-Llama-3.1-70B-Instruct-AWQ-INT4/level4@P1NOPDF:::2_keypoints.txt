- Introduces a new deep recursive neural architecture with state-of-the-art results on a challenging sentiment classification dataset.  
- Results are impressive and the paper’s exposition is very clear.  
- Innovations are clever, well-motivated, and experimentally validated.  
- Untying weights for leaf nodes vs. non-terminal nodes to increase model flexibility.  
- Using rectified linear units (ReLUs) in the recursive setting to improve information flow.  
- Leveraging large unsupervised word vectors while using smaller hidden non-terminal nodes for efficiency.  
- Deep architecture that links layers and tree nodes (not just replacing a single RNN layer) to enhance information exchange.  
- Suggestion to compare a variant that drops the matrix V connection so only the last hidden-layer outputs feed the parent.  
- Methodological concern: Table 1(a) reports final test-set results (possible tuning on test set); should report development-set results instead.  
- Minor typos to fix (e.g., “comprise a class of architecture”, “sentiment classificationd dataset”).
