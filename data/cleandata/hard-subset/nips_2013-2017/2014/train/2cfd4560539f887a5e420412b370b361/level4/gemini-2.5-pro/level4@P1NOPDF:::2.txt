This paper introduces a novel deep recursive neural network architecture and applies it to sentiment classification, achieving state-of-the-art performance on a challenging dataset. The results from the new architecture are impressive, and the paper is very clearly written.

The authors propose several clever and well-motivated modifications to previous recursive neural network models, and their effectiveness is demonstrated experimentally. The key contributions include:

*   Untying the parameters for leaf nodes and non-terminal nodes, allowing for more specialized representations.
*   The successful application of rectified linear units (ReLUs) within the recursive setting.
*   An effective design that combines large, pre-trained unsupervised word vectors with smaller hidden representations for non-terminal nodes.
*   A novel deep architecture that goes beyond simply stacking RNN layers. The introduction of connections from the input vectors to all layers at each tree node is a particularly interesting design choice.

Regarding this deep architecture, it would have been an interesting comparison to evaluate a model where only the output of the final hidden layer at each node is passed to its parent (i.e., dropping the skip-layer connections from the input word vectors). This ablation study would help clarify the impact of these connections.

My primary concern with the paper lies in the experimental methodology. The results in Table 1(a) appear to be based on tuning performed on the final test set, as performance is not reported on the development set. Using the test set for any form of model selection or hyperparameter tuning invalidates the final performance claims. The authors should clarify this process and, if necessary, report test set results for a model tuned exclusively on the development set.
