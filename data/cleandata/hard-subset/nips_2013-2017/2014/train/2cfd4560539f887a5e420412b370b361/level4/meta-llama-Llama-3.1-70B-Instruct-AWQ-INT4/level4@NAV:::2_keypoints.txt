Paper introduces a novel deep recursive neural architecture that achieves state-of-the-art results on a challenging sentiment classification dataset.  
Exposition of the model and experiments is clear.  
Proposed modifications are ingenious, well-motivated, and experimentally validated.  
Key model changes: decoupling leaf nodes and non-terminal nodes.  
Key model changes: employing rectified linear units (ReLU) in the recursive setting.  
Key model changes: using large unsupervised word vectors with smaller hidden non-terminal nodes.  
Key model changes: a deep architecture that connects layers and tree nodes rather than merely replacing a single RNN layer with a deep layer.  
Suggestion to compare a variant where only the last hidden layer output at each node feeds the parent (removing matrix V).  
Methodological concern: Table 1(a) results should be reported on the development set, not the test set, to avoid tuning on the test set.  
Minor issue: typo in phrase "comprise a class of architecture."
