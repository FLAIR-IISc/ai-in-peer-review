This paper introduces an exciting new deep recursive neural architecture that achieves state-of-the-art performance on a challenging sentiment classification dataset. The results presented are impressive, and the paper is very clearly written.

The proposed modifications to previous recursive neural network (RNN) models are clever, well-motivated, and experimentally demonstrated to be effective. Key contributions include:

*   Untying the parameters for leaf nodes and non-terminal nodes.
*   The application of Rectified Linear Units (ReLUs) in the recursive setting.
*   Combining large, pre-trained unsupervised word vectors for leaf nodes with smaller-dimensional hidden vectors for non-terminal nodes.
*   A novel deep architecture that introduces connections between different layers at each tree node, rather than simply stacking RNN layers.

Regarding the deep architecture, it would have been an interesting comparison to evaluate a model where the output of the final hidden layer at each node is the sole input to its parent (i.e., dropping the connection from the input matrix V to higher layers). This ablation study could further clarify the contribution of these specific connections.

A significant concern is that the results reported in Table 1(a) appear to be based on the final test set. If hyperparameters were selected using these results, it constitutes tuning on the test set, which can lead to over-optimistic performance estimates and compromises the validity of the reported scores. The authors should clarify this point and ensure that all model tuning was performed on a separate development set.
