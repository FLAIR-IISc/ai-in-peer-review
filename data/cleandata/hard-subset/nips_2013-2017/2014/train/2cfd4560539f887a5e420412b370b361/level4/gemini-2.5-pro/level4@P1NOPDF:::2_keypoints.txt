- Introduces a novel deep recursive neural network architecture with state-of-the-art sentiment classification results.  
- Paper is clearly written and results are impressive.  
- Unties parameters for leaf nodes and non-terminal nodes to allow more specialized representations.  
- Successfully applies rectified linear units (ReLUs) within a recursive neural network setting.  
- Combines large pre-trained unsupervised word vectors with smaller hidden representations for non-terminal nodes.  
- Proposes a novel deep design that connects input vectors to all layers at each tree node (skip-layer connections).  
- Suggests an ablation: evaluate a variant that only passes the final hidden layer output to parent nodes (drop skip-layer input connections).  
- Major experimental concern: tuning appears to have been done on the test set (no development-set results reported), which could invalidate performance claims and requires clarification or re-reporting.
