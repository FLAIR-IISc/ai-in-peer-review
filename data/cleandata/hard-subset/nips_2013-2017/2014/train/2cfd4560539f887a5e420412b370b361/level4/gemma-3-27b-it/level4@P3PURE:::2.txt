This paper introduces an exciting new deep recursive neural network architecture that achieves state-of-the-art performance on a challenging sentiment classification dataset. 

The results obtained with this new architecture are impressive, and the exposition is remarkably clear. The changes proposed to previous recurrent neural network (RNN) models are clever, well-motivated, and demonstrably effective through experimentation:

*   Untying the weights of leaf and nonterminal nodes.
*   Utilizing rectified linear units within the recursive setting.
*   Leveraging large, unsupervised word vectors in conjunction with smaller hidden nonterminal nodes.
*   Employing a deep architecture that goes beyond simply replacing a single neural network RNN layer with a deep layer; instead, it introduces connections *between* layers and tree nodes. It would have been a valuable addition to the analysis to compare performance when the outputs of the last hidden layer at each node were the *only* inputs to the next parent node, effectively removing the connection mediated by matrix V.

A potential issue exists with Table 1(a), where the reported results appear to be on the final test set rather than the development set. This suggests tuning is being performed directly on the test set, which is problematic. 

Finally, a minor note: the phrase "comprise a class of architecture" contains a grammatical error.
