This paper introduces a novel deep recursive neural architecture that achieves state-of-the-art performance on a challenging sentiment classification dataset. The results obtained from this new architecture are impressive, and the exposition is very clear.

The proposed changes to previous RNN models are clever, well-motivated, and experimentally demonstrated to be effective. Specifically, the authors introduce the following modifications:

- Untying leaf nodes and nonterminal nodes, allowing for more flexible representations
- Utilizing rectified linear units in the recursive setting, which improves the model's ability to learn complex patterns
- Employing large unsupervised word vectors, while using smaller hidden nonterminal nodes, which strikes a balance between representation capacity and model complexity
- Designing a deep architecture that is not simply a replacement of a single neural network RNN layer with a deep layer, but rather introduces connections between the layers and tree nodes, enabling more nuanced interactions between different levels of representation

It would have been an interesting comparison to investigate the effect of dropping the connection from matrix V, such that the outputs of the last hidden layer at each node are the only input to the next parent. This would help to isolate the contribution of the introduced connections to the model's performance.

However, there are a few issues that need to be addressed. Firstly, the results in Table 1 (a) should have been reported on the development set, rather than the final test set. This is because the development set is typically used for hyperparameter tuning, and reporting results on the test set may give an overly optimistic estimate of the model's performance.

Additionally, there are a few typos that need to be corrected. For example, the phrase "comprise a class of architecture" should be revised for clarity.

Overall, this paper presents a significant contribution to the field of deep learning, and the proposed architecture has the potential to achieve state-of-the-art performance on a range of sentiment classification tasks.
