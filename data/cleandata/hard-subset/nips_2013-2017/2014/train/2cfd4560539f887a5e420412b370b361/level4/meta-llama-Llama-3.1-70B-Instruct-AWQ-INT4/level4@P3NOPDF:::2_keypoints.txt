- Introduces a novel deep recursive neural architecture that achieves state-of-the-art performance on a sentiment classification dataset and is clearly presented.
- Proposed modifications are clever, well-motivated, and shown experimentally to be effective.
- Untying leaf nodes and nonterminal nodes allows more flexible representations.
- Using rectified linear units (ReLU) in the recursive setting improves the model’s ability to learn complex patterns.
- Combining large unsupervised word vectors with smaller hidden nonterminal nodes balances representation capacity and model complexity.
- The deep architecture adds connections between layers and tree nodes (not just replacing a single RNN layer), enabling nuanced interactions across representation levels.
- Suggestion to run an ablation: drop the matrix V connections so only last hidden-layer outputs feed the parent, to isolate their contribution.
- Issue: Results in Table 1(a) should be reported on the development set (not the test set) to avoid optimistic estimates from hyperparameter tuning.
- Issue: Several typos and unclear phrases (e.g., “comprise a class of architecture”) need correction.
- Overall assessment: A significant contribution with strong potential to achieve state-of-the-art performance across sentiment tasks.
