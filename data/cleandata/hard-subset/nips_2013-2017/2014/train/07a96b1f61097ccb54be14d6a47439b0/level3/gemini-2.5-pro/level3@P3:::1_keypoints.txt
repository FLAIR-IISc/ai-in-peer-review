- Addresses RPCA's failure on "coherent" data and links high coherence to clustered/union-of-subspaces structure.
- Observes that increasing number of clusters raises data coherence, breaking RPCA guarantees and performance.
- Proposes using LRR with a low-rank dictionary A whose column space contains the true low-rank component to avoid coherence issues.
- Theoretical result (Theorem 1): a low-rank dictionary eliminates dependence on the problematic µ2 coherence parameter.
- Practical unsupervised two-stage algorithm: run RPCA for an initial estimate, build a low-rank dictionary from it, then run LRR for improved recovery.
- Empirical validation: synthetic phase-transition plots and corrupted motion-sequence experiments show substantially better recovery than RPCA.
- Application benefit: using the method as preprocessing yields ~40% relative reduction in subspace clustering error on motion data.
- Strengths: significant, original problem framing; strong theoretical contribution extending RPCA/LRR analysis; clear presentation and intuitive algorithm.
- Weakness/suggestion: experiments treat dictionary learning as preprocessing — could benefit from integrating A and Z learning jointly for clustering.
- Recommendation: high-quality, practically important paper that should be accepted.
