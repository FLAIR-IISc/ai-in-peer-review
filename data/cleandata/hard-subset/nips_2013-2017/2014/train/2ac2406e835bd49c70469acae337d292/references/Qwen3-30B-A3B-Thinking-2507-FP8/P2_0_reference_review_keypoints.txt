- Proposes a method-of-moments spectral algorithm for learning Mixture of HMMs (MHMMs).
- Key insight: an MHMM can be viewed as a single HMM with a block-diagonal transition matrix (Lemma 1).
- Main technical contribution: recovering individual HMM parameters from permuted global parameters using spectral properties.
- Provides theoretical analysis of spectral properties of the global transition matrix (Lemma 2, Corollary 1).
- Introduces a spectral longevity measure to estimate the number of clusters K.
- Presents Conjecture 1 giving a sample complexity bound, but lacks a rigorous proof tying it to MHMMs.
- Experimental evaluation includes synthetic robustness tests, speed/accuracy comparisons with EM, and a real handwritten-character dataset.
- Results show comparable accuracy to EM while being significantly faster and useful as an EM initialization.
- Paper is well-written with consistent notation, clear figures, and a clearly presented algorithm.
- Transition from noiseless to noisy case and the intuition for the spectral longevity measure need clearer explanation.
- Using a cubic polynomial feature transform in the real experiment satisfies MK < L but should be discussed as a methodological limitation.
- Novelty lies in applying spectral learning to MHMMs and addressing permutation ambiguity; could contrast more with tensor-based methods.
- Practical significance: enables scalable MHMM learning and provides effective EM initialization, but still requires MK × MK matrix operations.
- Overall recommendation: accept — technically sound, well-validated, and practically useful despite some theoretical and presentation weaknesses.
