This paper presents a spectral algorithm for learning the mixture of hidden Markov models (MHMM). It demonstrates that an existing spectral algorithm for learning HMMs (Anandkumar et al., 2012) can be used to form MHMMs as a special case of HMMs. It proposes a stability-based solution for parameter settings, where the assumption is presented that each HMM's transition matrix has a single eigenvalue with a value of 1.

There seems to be a problem with the use of some symbols here. The authors should show more awareness of related work by providing references (e.g., EM started with spectral assumptions has already been explored in Chaganty and Liang (2013)). The experiments are quite preliminary, and I strongly recommend that the results be reported in a table format instead of a graphical format in Figure 4.

However, the proposed scheme to avoid parameter setting problems is interesting and intuitive. The authors have also provided an analysis in the case of noise, which leads to an intriguing use of the eigenvalues in the algorithm (equation (7)).

Comments:
- I was initially confused by the conditions of Corollary 1. It would be good to define 1J^T (a vector of length J rows filled with ones) and explain the structure of lim{e->inf} bar{A}^e in a little more detail (this is a JK x JK block diagonal where the columns of each J x J block are the same).
- In general, define each vector/matrix with its dimensions when first presenting it. This will greatly help the presentation.
- I am having a lot of difficulty reading Figure 4: please replace it with a table. Also, I am not sure why EM is performing so poorly in Table 1. Generally, with random initialization, it beats the spectral method with sufficient iterations (despite the local optima problem). A standard practice is to report the best (not average) accuracy of EM: please include that result.
- Nit: If the paper is in accordance with the symbols established by Hsu et al. (2009), it may help readers. Namely, use T for the transition matrix (not A), use n, m for the number of observations and hidden states (not L, J).

This paper proposes an algorithm for learning the mixture of hidden Markov models. This algorithm consists of an existing spectral algorithm for learning hidden Markov models and a new stability step to eliminate parameter settings.
