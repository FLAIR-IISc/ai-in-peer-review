This paper presents a spectral algorithm for learning a mixture of Hidden Markov Models (MHMMs). It demonstrates that an existing spectral algorithm for learning HMMs (Anandkumar et al., 2012) can be used to learn MHMMs by preparing an MHMM as a special case of the HMM. It proposes a stabilization-based solution for parameter sequences, introducing the notion that each HMM's transition matrix has a single eigenvalue of value 1.

There seem to be some problems with the notation. The authors definitely need to show more awareness of the related work by providing references (for example, introducing EM with spectral estimates has already been explored in Chaganty and Liang (2013)). The experiments are quite basic, and I strongly recommend that the results be reported in a table format rather than in a graphical format in Figure 4.

That being said, the proposed scheme to avoid parameter sequence problems is interesting and intuitive. The authors have also provided an analysis for the case of estimation noise, which leads to an interesting use of eigenvalues in the algorithm (equation (7)).
