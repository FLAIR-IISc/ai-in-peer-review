## Review of LSDA: Large Scale Detection through Adaptation

This paper proposes LSDA, a framework for adapting image classification-trained deep convolutional neural networks for object detection, specifically addressing the scarcity of bounding box annotations for many object categories. The core idea is to learn a transformation from classification features to detection features using categories with both classification and detection data, and then apply this transformation to classifiers for categories lacking bounding box labels. The authors demonstrate strong performance on the ImageNet LSVRC-2013 detection challenge, achieving 4th place and showcasing a 7.6K detector built using readily available classification data. 

The paper is exceptionally well-written and clearly presented, making the methodology easy to follow. The CNN architecture, based on Krizhevsky et al., is a sensible choice and relatively straightforward to implement. The experimental results are promising and highlight the potential of leveraging abundant classification data for scaling up object detection. 

However, while the method achieves impressive results, the underlying adaptation mechanism lacks a strong theoretical justification. The motivation for the specific update rule in Section 3.2 (Wj^c + W_avg) feels ad-hoc, and the physical meaning of the “offset” parameter remains unclear. Further intuition and analysis of this component would greatly strengthen the paper. The superior performance of the kNN variant also requires more explanation; why does averaging the offsets of nearest neighbors work better than other strategies?

A significant concern is the evaluation methodology. The authors acknowledge using part of the validation set (val1) for training and val2 for testing. This is a clear deviation from standard practice, potentially inflating performance. Comparisons to other methods become unfair as they typically report results on a separate, held-out test set. Furthermore, the claim of R-CNN achieving similar performance on val2 is not statistically significant given the small size of this dataset. 

In conclusion, LSDA presents a compelling approach to large-scale detection with good empirical results. However, the adaptation method requires more rigorous justification, and the evaluation needs to be corrected to ensure fair comparison with the state-of-the-art. 

**Arguments for Acceptance:**

*   Strong empirical performance on a challenging benchmark.
*   Clear and well-written presentation.
*   Addresses a significant problem in scaling object detection.

**Arguments against Acceptance:**

*   Lack of principled justification for the adaptation method.
*   Questionable evaluation methodology.
*   Insufficient explanation of key design choices (e.g., kNN).
