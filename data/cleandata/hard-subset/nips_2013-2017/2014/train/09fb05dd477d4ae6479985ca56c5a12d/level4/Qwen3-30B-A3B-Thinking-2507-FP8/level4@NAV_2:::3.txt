

The paper introduces LSDA, a method for adapting image classifiers to object detectors using a subset of categories with bounding box annotations. This work builds upon R-CNN [1] and domain adaptation techniques, specifically addressing the scalability challenge of detection systems by leveraging abundant image-level classification data (e.g., ImageNet) rather than expensive bounding box annotations. LSDA frames the classifier-to-detector transformation as a domain adaptation problem, where classification data serves as the source domain and detection data as the target domain. Key innovations include learning a category-invariant feature transformation (via fine-tuning shared layers 1–7 on bounding box-labeled categories) and a category-specific output layer adaptation (using nearest-neighbor approximation for categories without detection data). The approach enables detection for 7,604 ImageNet categories using only classification data for 7,404 categories, achieving a 50% relative mAP improvement over classification-only baselines on ILSVRC-2013 detection data.

**Quality**: The method is technically sound and well-motivated. The ablation study (Table 1) demonstrates that feature adaptation (layers 1–7 fine-tuning) contributes most significantly to performance, while category-specific output adaptation provides incremental gains (k=10 yields 16.15% mAP vs. 15.85% without). However, the comparison to key baselines is incomplete: (1) A standard CNN fine-tuned on full images as positive examples (with regions from other classes as negatives) should be included to isolate the contribution of bounding box adaptation; (2) The MIL approach of Ali et al. [6] (which uses image-level labels for detection) is not evaluated on ImageNet, despite being a direct competitor; (3) The "centered bounding box" baseline (predicting the mean bounding box per class at test time) is omitted, which would clarify the difficulty of the task. Without these, the claimed 50% improvement over classification-only baselines may partly reflect the inherent strength of CNN features rather than the adaptation mechanism.

**Clarity**: The paper is well-structured and clearly written, with a logical flow from problem statement to experiments. Minor improvements would include explicitly defining the baseline methods in Table 1 (e.g., "Output Layer Adaptation (k=10)" should reference Equation 1) and clarifying the role of background class learning in Figure 3.

**Originality**: The core idea of adapting classifiers to detectors via domain adaptation is novel for large-scale detection (unlike prior domain adaptation work [12–16], which assumes labeled data for all categories). It extends R-CNN [1] by eliminating the need for bounding box data for all categories, and differs from MIL-based approaches [6,7] by enabling direct adaptation without iterative region selection. The 7,604-category detector capability is a significant advancement over previous NIPS detection work (e.g., R-CNN [1] on 20 classes, ImageNet detection [3] on 200 classes).

**Significance**: The problem addressed—scaling detection to tens of thousands of categories—is highly relevant for real-world applications. LSDA’s ability to leverage existing classification data (e.g., ImageNet’s 20,000 categories) to produce a large-scale detector with minimal additional annotation effort is a major practical contribution. The 7.6K detector (Figure 6) and 2fps runtime (Section 4.4) demonstrate tangible progress toward scalable detection systems. While performance still lags behind oracle (full bounding box data) results, the method provides a viable path to overcome the current annotation bottleneck.

**Pro/Con for Acceptance**:  
*Pro*: Novel adaptation framework enabling detection for 7.6K categories; strong empirical results on ImageNet; clear technical contribution to scaling detection; practical software release.  
*Con*: Missing critical baselines (MIL [6], full-image fine-tuning) weaken the novelty claim; the 50% mAP gain could partially stem from CNN feature quality rather than adaptation.  

The paper addresses a significant problem with a technically sound solution, though the evaluation would be strengthened by comparing to the MIL baseline [6] and full-image fine-tuning. Given the novelty of enabling large-scale detection and the quality of the experiments, it merits acceptance.
