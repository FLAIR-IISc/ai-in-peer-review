- Introduces LSDA: convert image classifiers into object detectors using minimal bounding-box annotations.  
- Frames classifier→detector as domain adaptation: classification data = source, detection data = target.  
- Method: train CNN on all categories (classification), fine-tune layers 1–7 on a subset with boxes including a background class.  
- Adapt category-specific output weights for unseen categories via nearest-neighbor transfer (Equation 1).  
- Empirical gains: ~50% relative mAP improvement on held-out ImageNet categories.  
- Scales to a 7,604-category detector using ImageNet hierarchy; reports ~2 fps runtime with fast proposals.  
- Ablation studies show category-invariant adaptation provides the largest performance boost.  
- Missing baseline: full-image training (use whole images as positives and other-class regions as negatives) to test if adaptation is necessary.  
- Missing baseline: Multiple Instance Learning (MIL) comparison to assess weakly supervised localization alternatives.  
- Missing baseline: centered bounding-box (mean-box) predictor as a simple strong weak-supervision baseline.  
- Without those baselines, the 50% mAP gain might reflect improved CNN features rather than the adaptation technique.  
- Presentation is clear and well-organized; however Table 1 and Equation 1 need minor clarifications (which layers adapted, explicit k value, validation split).  
- Originality: novel in handling held-out categories without detection data and adapting both domain and task, distinct from R-CNN and prior domain-adaptation work.  
- Significance: high—enables large-scale detection with reduced annotation cost, though it still requires some bounding boxes and does not close the gap to an oracle detector.  
- Pros: addresses an important large-scale problem, solid empirical results, public code/models, demonstrated scalability.  
- Cons: lacks critical baselines, potential confound with feature quality, limited analysis of failure modes (e.g., poor nearest-neighbor matches).  
- Recommendation: accept, contingent on adding the three suggested baselines to strengthen experimental claims.
