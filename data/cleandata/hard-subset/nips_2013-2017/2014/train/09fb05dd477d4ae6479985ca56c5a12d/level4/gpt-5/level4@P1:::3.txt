Summary and relation to prior work:
This paper addresses the challenge of scaling object detection to many categories when bounding box annotations are scarce. The authors propose LSDA, which learns a generic transformation from image classification to detection using a subset of categories with bounding boxes, then applies this transformation to classifiers of categories without bounding box data. Concretely, they fine-tune a Krizhevsky et al. (AlexNet) CNN pretrained on ILSVRC2012 classification: (1) train a K-way classifier on all categories using image-level labels; (2) fine-tune layers 1–7 plus a new background class using detection data from m categories (set B); and (3) adapt the category-specific fc8 parameters for the remaining K−m categories (set A) by adding to each classifier weight vector the average fc8 change observed in its nearest neighbors from set B. At test time, the model scores region proposals per category and subtracts the background score.

The approach builds on R-CNN, which showed the benefit of fine-tuning classification networks for detection but requires bounding box data for all categories. It also relates to domain adaptation (adapting from classification to detection) and to weakly supervised detection via MIL, though LSDA does not perform MIL on the held-out classes. The authors evaluate on ILSVRC2013 detection, simulating access to detection labels for 100 categories and only classification labels for the other 100. They report a 50% relative mAP gain over using the classification network directly on proposals, show that adaptation mainly reduces localization and background confusions, and scale the method to a 7.6K-class detector with a faster variant.

Quality:
The paper is technically sound and the approach is simple and logical. The ablations convincingly show that category-invariant feature adaptation (layers 1–7 and a background class) is the main driver of improvement, while fc8 adaptation via nearest neighbors provides smaller gains. The error analysis is helpful. The work leverages the strong performance of modern CNNs, and the claim of reduced training time versus full R-CNN is practical.

However, several important baselines are missing, making it hard to assess whether the gains are due to the proposed adaptation or simply to strong CNN features and data preprocessing:

1) Full-image supervision baseline: Train/fine-tune the CNN using full images labeled as positive and use full images and region proposals from other classes as negatives, then apply to region proposals at test time. This mirrors standard classification fine-tuning but uses more comprehensive negatives. It would directly test whether introducing the background class and region-level fine-tuning is essential.

2) Weakly supervised/MIL baseline: As above, but for classes in set A, iteratively select the highest-scoring positive region per image (MIL) and continue fine-tuning. This directly competes with LSDA’s category-specific adaptation and may be more principled for discovering object extents without box labels.

3) Center-box baseline: Train/fine-tune the CNN for classification only; at test time, predict a single bounding box per image (e.g., a mean box) and report detection mAP. This simple baseline would indicate how much of the detection performance comes from recognition alone versus precise localization on this dataset.

It is not clear to me that LSDA would outperform (1), and (2) is likely a stronger competitor for held-out classes. Including (3) would calibrate the difficulty of the dataset and the strength of the classification signal.

Clarity:
The paper is generally clear and well organized. The experiments section would benefit from more explicit descriptions of the ablations in Table 1. In particular:
- Precisely define each row: which layers are fine-tuned, whether the background class is included, and whether fc8 adaptation (nearest-neighbor averaging) is applied, and with what k.
- Make explicit whether the “category-invariant adaptation” corresponds exactly to updating layers 1–7 and adding the background class, while “category-specific adaptation” is the fc8 nearest-neighbor delta in Eq. (1).
- Clarify how nearest neighbors are computed for fc8 (L2 on normalized weights), and provide sensitivity to k.

Originality:
The idea of adapting from classification to detection by learning a shared feature transformation and background modeling is straightforward but novel in its application to categories lacking detection labels. The category-specific fc8 adaptation via nearest-neighbor deltas is a simple, pragmatic contribution. Relative to R-CNN and existing domain adaptation methods, the contribution lies in adapting to categories with no bounding box data.

Significance:
The problem is important, the setup is realistic, and the results are promising. Demonstrating a substantial relative mAP gain on held-out categories and scaling to 7.6K classes with reasonable runtime is compelling. If the method outperforms strong weakly supervised baselines, it could be widely adopted.

Pros:
- Addresses a highly relevant scalability problem in detection.
- Simple, effective pipeline; practical training time.
- Clear ablations showing the value of background modeling and feature adaptation.
- Useful error analysis; public release of models/code; demonstration at 7.6K classes and faster inference.

Cons:
- Missing key baselines (full-image training with stronger negatives, MIL on set A, center-box).
- Heavy reliance on strong CNN features; unclear how much benefit comes specifically from the proposed adaptation vs. alternative weakly supervised fine-tuning.
- Some ambiguity in the experimental descriptions (Table 1 details and the exact components varied).

Suggestions and questions for the authors:
- Add the three baselines above to quantify the value of LSDA beyond strong CNN features.
- Report mAP for varying k in fc8 adaptation with per-class deltas, and analyze failure cases where nearest-neighbor transfer hurts.
- Provide per-class results on the held-out set and confusion matrices to complement the error taxonomy.
- Clarify all ablation settings and explicitly map them to the described components.
- Discuss whether adding SVMs/bbox regression (as in R-CNN) on the adapted features for set B would further close the gap, and how that would affect transfer to set A.

Overall:
A solid and timely contribution with promising results and good potential impact. With the addition of the suggested baselines and clearer experimental descriptions, the paper would be substantially strengthened.
