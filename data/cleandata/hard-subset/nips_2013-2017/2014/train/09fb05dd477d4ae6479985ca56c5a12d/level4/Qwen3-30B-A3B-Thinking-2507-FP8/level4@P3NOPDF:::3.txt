

This paper proposes adapting a Krizhevsky-style CNN trained for classification (without bounding box annotations) to object detection using a small set of classes with bounding box labels. The method involves three key stages: (1) training a classification CNN on full images across all classes; (2) training a region-based CNN on labeled classes (including a background class) using image regions; and (3) integrating the background class into the full CNN model while fine-tuning selected weights. At test time, the network processes multiple image regions per input.  

**Quality and Technical Soundness**  
The approach is logically structured and leverages established CNN architectures effectively. The results are compelling, demonstrating that weakly supervised detection is feasible on large-scale datasets without bounding box annotations during training. However, the experimental evaluation lacks critical baselines that are essential for contextualizing the contribution. Specifically, the paper should compare against:  
1. *Full-image baseline*: Training a CNN on full images (positive examples) and regions from other classes (negative examples), fine-tuning all layers.  
2. *MIL-based baseline*: Iteratively selecting top-scoring regions per image (instead of full images) for training.  
3. *Simple heuristic baseline*: Using a precomputed mean bounding box (e.g., ImageNet class center) with the classification CNN.  

Without these comparisons, it remains unclear whether the proposed method outperforms simpler, more intuitive alternatives. Baseline 1 is particularly relevant as it directly tests whether the region-based training strategy (vs. full-image training) provides an advantage. Baseline 2 assesses whether iterative region selection (a common weak-supervision tactic) would suffice. Baseline 3 establishes the "hardness" of the task by showing how well a naive approach performs. The current results may merely reflect the strength of CNN features rather than a novel detection strategy. Including these baselines would strengthen the paper’s claims and clarify the method’s true value.  

**Clarity**  
The paper is well-written and logically organized. However, the experiments section would benefit from greater precision. For instance, Table 1 lists methods that reference "Eq. 1" or "Eq. 2," but the paper does not explicitly state whether these equations apply to specific CNN layers or the entire network. Clarifying this would prevent ambiguity.  

**Originality**  
The core idea—adapting a classification CNN for detection via weak supervision—is modestly novel. While the approach is simple and specific to certain CNN architectures, it addresses a relevant problem in computer vision. The method is not radical but represents a practical and useful extension of existing work.  

**Significance**  
The problem addressed is significant: enabling detection without bounding box annotations reduces annotation costs, which is critical for large-scale applications. The paper demonstrates that competitive detection performance is achievable under these constraints, which is valuable for the field. However, the significance is partially undermined by the missing baseline comparisons. If the results do not substantially outperform simpler baselines (e.g., Baseline 1), the contribution’s impact diminishes.  

**Recommendation**  
The paper presents a reasonable approach with promising results but requires substantial revision to address experimental gaps. The missing baselines are not merely an oversight—they are fundamental to establishing the method’s novelty and efficacy. Without them, the claims about the approach’s advantages over simpler alternatives remain unsubstantiated.  

**Summary of Required Revisions**  
1. Add the three baselines (full-image, MIL, and mean-bounding-box) to the experiments.  
2. Explicitly clarify the implementation details of methods in Table 1 (e.g., layer-wise application of equations).  
3. Discuss how the results compare to these baselines in the main text.  

As submitted, the paper’s contribution is too closely tied to the success of CNN features rather than the proposed adaptation strategy. With the proposed revisions, it would make a meaningful contribution to weakly supervised detection. I recommend *conditional acceptance* pending these critical additions. The current version risks overstating the method’s novelty and utility.
