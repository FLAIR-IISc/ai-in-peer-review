

This paper introduces LSDA (Large Scale Detection through Adaptation), a method for transforming image classifiers into object detectors using minimal bounding box annotations. The authors address the critical challenge of scaling object detection to thousands of categories by leveraging abundant image-level classification data, rather than requiring expensive bounding box annotations for every category. The approach treats the transformation from classification to detection as a domain adaptation problem, where classification data (source domain) is adapted to detection data (target domain). The key innovation is learning a generic transformation from classifiers to detectors using only a small subset of categories with bounding box annotations, then applying this transformation to extend detection capabilities to categories without bounding box annotations.

The method consists of three key components: (1) training a CNN classifier on all categories using image-level labels, (2) adapting the classifier to detectors by fine-tuning layers 1-7 on a small subset of categories with bounding box annotations (including a background class), and (3) applying category-specific adaptation (via Eq. 1) to extend detection to categories without bounding box annotations. On the ImageNet detection dataset, LSDA achieves a 50% relative mAP improvement over a classification-only baseline on held-out categories, and demonstrates the ability to produce a 7,604-category detector using only classification data for most categories.

Quality: The technical approach is sound and well-motivated. The methodology logically extends previous work on R-CNN [1] and domain adaptation. However, the evaluation lacks several critical baselines that would strengthen the claims. Specifically, the authors should compare to: (1) a method using full images as positive examples and regions from other classes as negative examples, training and fine-tuning all CNN layers; (2) a MIL-based approach that iteratively selects highest-scoring positive regions for each image from set A; and (3) a simple baseline where the classifier runs on full images with a single bounding box at the image center (e.g., mean bounding box for ImageNet). Without these comparisons, it remains unclear whether LSDA's improvements stem from the adaptation technique itself or simply from using CNN features. The paper claims to enable detection for "tens of thousands of categories," but the evaluation is limited to ImageNet's 200 categories, with no demonstration of scalability to larger sets.

Clarity: The paper is well-written and logically organized, with clear figures illustrating the methodology. The experimental section could be improved by more explicitly explaining the baselines in Table 1 (e.g., specifying whether each method applies Eq. 1 or uses different CNN layers) and clarifying the significance of different adaptation techniques.

Originality: The core idea of adapting classifiers to detectors using limited bounding box annotations for large-scale detection is novel. While the approach builds on R-CNN [1] and domain adaptation techniques, the specific adaptation strategy for extending to unseen categories represents a meaningful contribution. However, the method is tightly coupled to CNN architectures like Krizhevsky's, which limits its applicability to other feature representations.

Significance: The problem addressed is highly relevant for scaling object detection to large numbers of categories. The 50% relative mAP improvement on held-out categories demonstrates the method's potential value for practitioners. The ability to use existing classification data for detection could significantly reduce annotation costs for large-scale detection systems. However, the significance would be stronger with a more comprehensive evaluation, particularly on larger datasets and with the suggested baselines.

Pro:
- Addresses a significant problem in scaling object detection to large numbers of categories
- Provides a clear, logical method for adapting classifiers to detectors using minimal bounding box annotations
- Demonstrates substantial improvement over classification-only baselines
- Well-motivated and well-structured methodology

Con:
- Lacks comparison to several important baselines (as detailed above)
- Evaluation limited to ImageNet's 200 categories, with no demonstration of scalability
- Methodology tightly coupled to specific CNN architectures
- The claimed "50% relative mAP improvement" is only for held-out categories, not the full set

Overall, this is a valuable contribution to the field that addresses an important practical problem. With the addition of the suggested baselines and more comprehensive evaluation, this work would be a strong candidate for acceptance. As presented, the results are compelling but require more thorough validation to fully support the claims about scalability and effectiveness.
