This paper addresses the challenge of object detection, specifically the issue of obtaining bounding boxes on a scale similar to which category labels exist for object categorization. The authors propose a novel approach to adapt object classifiers for the detection task, thereby sidestepping the challenge of obtaining bounding boxes.

The paper is well-written, and the quality of the work is good. The experimental protocol uses 100 categories for training (with both category labels and bounding boxes) and tests on 100 left-out categories, which had only category labels. The results obtained are reasonably good, filling about 35% of the gap between an oracle and a baseline on the 100 left-out categories.

The paper combines two existing ideas related to adaptation: adapting object classifiers for detection and adapting from one set of categories to another. While this is a novel and interesting idea, it is not groundbreaking. The main contribution of this paper is the novel approach, but the results-wise significance is hard to judge, and it is likely that they will be surpassed later by more sophisticated methods.

One of the main issues with the results is that they inevitably sacrifice performance (mAP) by using fewer bounding boxes, and there is no objective way to judge if this trade-off is worth it. Additionally, it is not clear if the issue being tackled is really an important long-term challenge, as throwing resources at the problem may be able to fix it.

The authors should address several issues, including clearly stating which methods use category labels to help with detection in Figure 3a. They should also highlight the main result, which is that the proposed approach is about 1/3 between baseline and oracle, rather than the 78% and 50% improvement claims. Furthermore, the authors should devote a table or a sub-section to clearly stating the differences from R-CNN, which seems to be the closest competing alternative.

Minor issues include a typo on line 431 and inconsistent naming of the proposed method as "DNN" or "DDA". Overall, this is a reasonably good paper proposing a reasonably novel approach, and it deserves to be accepted and seen by the NIPS community. While the results will almost certainly be surpassed soon, the long-term significance of the paper and the approach remains to be seen.

Strengths:
- Novel approach to adapting object classifiers for the detection task
- Reasonably good results, filling about 35% of the gap between an oracle and a baseline
- Well-written paper

Weaknesses:
- Results-wise significance is hard to judge, and may be surpassed later by more sophisticated methods
- Sacrifices performance (mAP) by using fewer bounding boxes
- Unclear if the issue being tackled is really an important long-term challenge

Arguments for acceptance:
- Novel approach to adapting object classifiers for the detection task
- Reasonably good results
- Well-written paper

Arguments against acceptance:
- Results-wise significance is hard to judge
- Sacrifices performance (mAP) by using fewer bounding boxes
- Unclear if the issue being tackled is really an important long-term challenge

Recommendation: Accept, but with revisions to address the issues mentioned above.
