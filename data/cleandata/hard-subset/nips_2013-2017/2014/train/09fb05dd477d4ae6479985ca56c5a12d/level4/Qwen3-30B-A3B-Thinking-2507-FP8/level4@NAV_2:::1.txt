

This paper addresses the challenge of scaling object detection to tens of thousands of categories by proposing a method to adapt image classifiers into object detectors without requiring bounding box annotations for all categories. The authors introduce LSDA (Large Scale Detection through Adaptation), which learns a generic transformation from classification to detection using a small subset of categories with bounding box annotations. The approach leverages classification data for all categories while requiring detection data for only a subset, enabling detection for categories that lack bounding box annotations but have image-level labels. The method was evaluated on the ImageNet LSVRC-2013 detection challenge, demonstrating a 50% relative mAP improvement over a classification-only baseline on held-out categories.

Quality and clarity: This is a high-quality, well-written paper with clear organization. The technical content is presented in a logical manner, and the experimental methodology is sufficiently detailed for reproducibility.

Originality: The paper combines two established concepts—adapting classifiers to detectors and transferring adaptation between categories—into a novel framework for large-scale detection. While the core ideas build on existing work (particularly R-CNN and domain adaptation), the specific combination and application to the large-scale detection challenge represents a meaningful contribution. To the reviewer's knowledge, this specific approach has not been previously proposed.

Significance: The primary contribution is a practical method for extending detection to categories lacking bounding box annotations, potentially enabling detection for tens of thousands of categories. While the results show a significant improvement over the baseline (50% relative mAP gain), the approach sacrifices some performance compared to full detection training. The significance lies in reducing the annotation bottleneck for detection systems, though the long-term impact will depend on whether this approach proves useful for practical applications.

Detailed issues that should be addressed:
- In Figure 3a, clarify which methods (besides LSDA and R-CNN) use category labels for detection. The current presentation lacks clear comparison of what methods use classification data versus detection data.
- The main result is the 35% gap reduction between baseline and oracle on held-out categories, not the 78% figure mentioned in the paper. The 50% relative improvement claim (line 425) should be recalibrated to reflect the performance relative to the baseline on held-out categories, not just oracle performance.
- Create a dedicated comparison table or subsection highlighting key differences between LSDA and R-CNN, as the current scattered references to R-CNN make it difficult to assess the novelty and advantages of the proposed approach. The anecdotal examples in Figure 6 could be omitted to focus on quantitative results.

Minor issues:
- The paper uses "DNN" to refer to the baseline method (classification-only network), but this is confusing since the core method also uses deep neural networks. The authors should clarify this terminology.
- Line 431 contains a typo ("classi?ers" instead of "classifiers").
