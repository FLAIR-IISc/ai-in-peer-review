This paper presents a framework for adapting deep neural networks pre-trained for image classification to the task of object detection. The authors demonstrate strong performance on the ImageNet LSVRC-2013 dataset, achieving 4th place in the object detection challenge. The paper is well-written and, for the most part, easy to follow. The proposed convolutional neural network (CNN) structure is both interesting and appears relatively straightforward to implement. Overall, this is a solid contribution, but I have two primary concerns that, if addressed, would significantly strengthen the work.

My first concern relates to the justification of the proposed adaptation method. While the results are promising, the underlying motivations for key design choices are not sufficiently explained. Specifically, Section 3.2 (Page 5, Line 231) describes combining weights – summing W<sub>j</sub><sup>c</sup> and W<sub>avg</sub> – for the adapted detection task. The rationale for this particular combination, and the meaning of this resulting ‘offset parameter’, remains unclear. Understanding *why* this summation is beneficial, and what it represents intuitively, is crucial. This lack of clarity extends to the kNN variant; the paper does not adequately explain *why* using kNN improves performance in this context. A more principled explanation of these choices would significantly enhance the contribution.

My second concern regards the fairness of the comparative evaluation. The authors appear to have used a portion of the validation set for hyperparameter tuning and model selection, rather than reserving the test set for final evaluation as is standard practice. While the authors acknowledge this in Footnote 1 (Line 322) and mention that RCNN exhibits similar performance on the ‘val2’ set, the small size of this validation subset undermines the statistical significance of any observed performance gains. A more rigorous comparison should ideally be conducted using a dedicated, held-out test set to ensure a fair assessment against competing methods. 

In summary, the authors present a compelling framework for adapting pre-trained CNNs to object detection, achieving competitive results. However, the paper would benefit from a more thorough justification of the adaptation method's design choices and a more rigorous, standard evaluation protocol. Addressing these points would elevate the paper's overall impact and clarity.
