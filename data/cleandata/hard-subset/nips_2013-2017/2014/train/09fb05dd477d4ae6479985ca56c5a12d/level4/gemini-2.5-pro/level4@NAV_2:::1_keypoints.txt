- Addresses scaling object detection to many categories by learning a classifier-to-detector transformation to avoid costly bounding-box annotation for every class.
- Proposes LSDA: learn a category-invariant feature transform and category-specific model transform on categories with boxes, then apply to held-out categories with only image labels.
- Frames the problem as domain adaptation from classification (source) to detection (target).
- Experiments on ILSVRC 2013 simulate 100 categories with boxes and 100 held-out, showing LSDA improves substantially over a classifier baseline.
- Reported numbers: baseline 10.31 mAP, full LSDA 16.15 mAP (≈50% relative boost), with an upper bound (oracle) of 32.8 mAP — LSDA closes ≈26% of the gap.
- Strength — Originality: novel idea of a generic classifier-to-detector transformation applicable to held-out classes.
- Strength — Significance: tackles a practically important annotation bottleneck for large-scale detection.
- Strength — Quality/Clarity: well-written, straightforward algorithm and a clear ablation study demonstrating component contributions.
- Weakness — Results framing: should present improvement relative to the oracle (gap closed) for a more balanced interpretation.
- Weakness — Comparison with R-CNN: needs a dedicated comparison detailing data requirements, training pipeline differences, performance, time, and scalability trade-offs.
- Weakness — Experimental clarity: must state which methods use which training data (e.g., same pre-training) to ensure fair “apples-to-apples” comparisons.
- Arguments for acceptance: significant, novel, well-executed approach with clear empirical gains and useful ablation insights.
- Argument against acceptance: despite gains, substantial gap remains to fully supervised performance and the paper should better discuss the performance vs. data-cost trade-off.
