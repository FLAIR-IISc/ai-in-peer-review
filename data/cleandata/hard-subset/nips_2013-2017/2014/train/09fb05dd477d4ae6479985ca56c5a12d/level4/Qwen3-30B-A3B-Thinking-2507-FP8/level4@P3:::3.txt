

This paper introduces LSDA (Large Scale Detection through Adaptation), a method to transform image classifiers into object detectors using minimal bounding box annotations. The core insight treats classifier-to-detector adaptation as a domain adaptation problem: leveraging classification data (image-level labels) as the source domain and detection data (bounding box annotations) as the target domain. LSDA trains a CNN on all categories using classification data, then fine-tunes layers 1–7 on a small subset of categories with bounding boxes (including a background class), and finally adapts category-specific output weights for unseen categories via nearest-neighbor transfer (Equation 1). The approach demonstrates significant gains on ImageNet detection (50% relative mAP improvement on held-out categories) and scales to a 7,604-category detector using ImageNet’s hierarchical structure.

**Quality**  
The work is technically sound, with clear ablation studies validating each component (category-invariant adaptation contributes most to performance). However, the experimental design lacks critical baselines that would strengthen the claims. Specifically:  
1. *Full-image training baseline*: Train a detector by treating full images as positive examples and regions from other classes as negatives (fine-tuning all layers). This directly tests if the proposed adaptation is necessary over simple image-level learning.  
2. *MIL baseline*: Apply Multiple Instance Learning (e.g., Ali et al. [6]) to select top-scoring regions per image for weakly labeled classes. The paper cites MIL as "complimentary" but omits comparison, making it unclear whether LSDA’s gains are due to adaptation or improved feature learning.  
3. *Centered bounding box baseline*: For held-out classes, predict a single bounding box at the image center (e.g., mean bounding box from ImageNet). This establishes a simple, strong baseline for the "weak supervision" challenge.  
Without these, the 50% relative mAP improvement could reflect better CNN features (already proven in R-CNN [1]) rather than novel adaptation. The paper’s claim that "the main benefit is transferring category-invariant information" remains unverified against these alternatives. The method is well-executed but would be more compelling with these comparisons.

**Clarity**  
The paper is exceptionally clear in exposition and organization. The three key insights (background recognition, category-invariant transfer, category-specific adaptation) are logically developed. The description of Equation 1 (category-specific adaptation) is precise, though Table 1 could better clarify which layers were adapted (e.g., "FC layers only" vs. "all layers"). The error analysis (Figure 5) effectively demonstrates reduced background/localization errors post-adaptation. Minor improvements would include explicitly stating the "k" value used in Equation 1 (k=10, per Table 1) and clarifying the validation protocol (e.g., "val2" split).

**Originality**  
LSDA’s core contribution—adapting classifiers to detectors for *unseen categories*—is novel relative to prior work. While R-CNN [1] fine-tunes classifiers for *all* categories, LSDA requires bounding boxes for only a subset. Domain adaptation techniques (e.g., [12–16]) adapt between visual domains but fix the task; LSDA adapts *both* domain *and* task (classification → detection). Crucially, it is the first method to handle *held-out categories* without detection data. The approach is specific to CNNs (e.g., Krizhevsky architecture), but this aligns with the field’s focus on deep learning. Related work on MIL [6,7] and domain adaptation [17–22] is well-cited, though the paper could better distinguish LSDA from [22] (which adapts *all* categories with detection data).

**Significance**  
The significance is high: LSDA enables detection for tens of thousands of categories (e.g., ImageNet’s 20k+ leaf nodes) using only classification data, addressing a major bottleneck in scaling detection systems. The 7,604-category detector demonstrates practical utility, and the 2fps runtime (with fast region proposals) shows real-world viability. This advances the field by reducing annotation costs for large-scale detection—critical for applications like video analysis. However, the paper underemphasizes the *cost-benefit* of LSDA: while it saves bounding box annotation time, the need for *some* bounding box data (for m categories) limits its applicability to scenarios where even limited annotation is prohibitive. The claim that LSDA "minimizes the gap between large-scale classifiers and detectors" is valid but requires acknowledging the remaining performance gap to the oracle (200-category detector in Table 1).

**Pro/Con Acceptance**  
*Pros*:  
- Solves a meaningful large-scale problem with strong empirical results.  
- Clear, well-motivated approach leveraging domain adaptation.  
- Provides public code/model (lsda.berkeleyvision.org), fostering reproducibility.  
- Demonstrates scalability to 7.6K categories—a unique contribution.  

*Cons*:  
- Missing critical baselines (full-image, MIL, center-bbox) weaken the novelty claim.  
- Performance gains could partially stem from CNN feature quality rather than adaptation.  
- Limited discussion of failure modes (e.g., when nearest-neighbor categories are poor matches).  

**Recommendation**  
This paper makes a valuable contribution to scalable object detection. With the addition of the three suggested baselines, the significance would be more definitively established. As-is, it is technically sound and addresses an important problem, but the experimental rigor would be strengthened by addressing the baseline gap. I recommend **acceptance** with the expectation that the authors incorporate these comparisons in the camera-ready version. The work aligns well with NIPS’ focus on scalable, practical methods and advances the state of the art in weakly supervised detection.
