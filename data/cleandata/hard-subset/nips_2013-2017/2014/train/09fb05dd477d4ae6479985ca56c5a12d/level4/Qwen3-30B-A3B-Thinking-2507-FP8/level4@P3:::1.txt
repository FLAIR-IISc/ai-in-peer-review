

This paper tackles the critical challenge of scaling object detection to large numbers of categories by leveraging abundant image-level classification labels instead of scarce bounding box annotations. The authors propose LSDA (Large Scale Detection through Adaptation), which treats the transformation from classification to detection as a domain adaptation problem. Using a subset of categories with bounding box annotations (B), LSDA learns a generic feature transformation (via fine-tuning layers 1–7) and a category-specific adaptation (via nearest-neighbor transfer of output weights) to convert classifiers for unlabeled categories (A) into detectors. Evaluated on ImageNet detection (200 categories), LSDA achieves a 50% relative mAP improvement over a classification-only baseline on held-out categories (A), enabling a 7,604-category detector using only classification data for most categories.

**Quality**  
The work is technically sound and well-executed. The authors provide a thorough ablation study (Table 1) demonstrating that category-invariant adaptation (fine-tuning layers 1–7 + background class learning) contributes most to performance (mAP: 10.31% → 15.85%), while category-specific adaptation (k=10) adds marginal gains (16.15%). The method is rigorously evaluated on the ILSVRC2013 detection dataset, with clear implementation details (e.g., using Caffe, data splits, augmentation protocols). The claim of "50% relative mAP improvement" is correctly framed against the classification-only baseline (not oracle performance), addressing a key ambiguity in the draft. The approach avoids overclaiming, explicitly acknowledging the gap to oracle performance (17.95% mAP vs. LSDA’s 16.15% on held-out categories). The error analysis (Fig. 5) further validates the method’s ability to reduce background confusion and localization errors—critical for detection.

**Clarity**  
The paper is exceptionally well-written and organized. The problem statement (Section 1), related work (Section 2), and algorithmic description (Section 3) logically build the case for LSDA. Key insights (e.g., the importance of background modeling) are clearly articulated with supporting figures (Figs. 2–3). The implementation details (Section 4.1) and experimental results (Section 4.2–4.4) are sufficiently detailed for reproducibility. Minor issues (e.g., "classi?ers" typo, inconsistent use of "DNN" vs. "LSDA" in draft) are easily fixable and do not impede understanding.

**Originality**  
LSDA combines two established concepts—domain adaptation for task transfer and nearest-neighbor-based parameter transfer—but applies them in a novel context: adapting classifiers *to detection* for categories lacking bounding boxes. While domain adaptation (e.g., [12–16]) and weakly supervised detection (e.g., [6–7]) exist, LSDA uniquely addresses the *category-scale* problem (e.g., adapting 7,604 categories with minimal detection data). Crucially, it differs from R-CNN [1] (which requires detection labels for all categories) and MIL-based methods (which lack large-scale validation). The paper adequately contextualizes this in Section 2, distinguishing LSDA from prior work (e.g., "To our knowledge, ours is the first method to adapt to held-out categories that have no detection data"). This is a meaningful contribution, though not revolutionary.

**Significance**  
The significance lies in enabling practical large-scale detection. Current detection systems (e.g., R-CNN) are limited to ~200 categories due to annotation costs. LSDA demonstrates that classification data (available for 20,000+ ImageNet categories) can be leveraged to build detectors for *tens of thousands* of categories—a direct path to scaling detection toward human-level capabilities. The 7,604-category detector (Section 4.4) and 2fps speedup (via spatial pyramid pooling) showcase tangible real-world applicability. While the method may be surpassed by future work, its simplicity and scalability make it a valuable stepping stone. The paper correctly frames the problem: bounding box annotation is *orders of magnitude more expensive* than image-level labeling (e.g., ImageNet classification data vs. detection labels), making this a critical bottleneck. The claim that "this algorithm enables detection for tens of thousands of categories" is well-supported.

**Strengths**  
- **Problem relevance**: Addresses a genuine scalability bottleneck in detection.  
- **Technical execution**: Rigorous ablation, clear error analysis, and reproducible code (lsda.berkeleyvision.org).  
- **Impact**: Enables detection for 7,604 categories using only classification data for most classes.  
- **Practicality**: Fast inference (2fps) and minimal annotation requirements.  

**Weaknesses**  
- **Novelty framing**: The paper could better emphasize *why* the category-scale transfer (vs. standard domain adaptation) is uniquely challenging (e.g., task shift from classification to detection *plus* category shift).  
- **Comparison depth**: While R-CNN is contrasted, a concise table summarizing key differences (e.g., data requirements, scalability) would strengthen the argument (as suggested in draft).  
- **Error analysis**: The "Oth" (other) error type (Fig. 5a) could be expanded to discuss how similar categories (e.g., lemon/orange) affect performance.  

**Recommendation**  
LSDA makes a solid, well-executed contribution to scaling object detection. It is technically sound, clearly presented, and addresses a meaningful problem with practical impact. While not groundbreaking, it provides a simple yet effective solution that significantly advances the state of large-scale detection. The results are compelling (50% relative mAP gain on held-out categories), and the method’s simplicity ensures broad applicability. The paper meets all NIPS criteria: high quality, clarity, and significance for the field. Minor revisions (e.g., clarifying the category-scale novelty, adding a comparison table) would further strengthen it, but these are not essential for acceptance.  

**Pro/Con Acceptance**  
*Pro*: Solves a critical scalability problem with minimal annotation overhead; enables 7.6K-category detection; well-validated experimentally.  
*Con*: Novelty is incremental (combines existing techniques), but the application is highly impactful.  

**Final Recommendation**: **Accept**. This work provides a valuable, practical approach to a major challenge in computer vision, with clear potential for adoption and future extension.
