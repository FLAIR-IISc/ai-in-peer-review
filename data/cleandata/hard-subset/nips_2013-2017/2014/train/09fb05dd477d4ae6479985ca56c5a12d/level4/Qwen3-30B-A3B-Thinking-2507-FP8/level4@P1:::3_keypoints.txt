- Paper introduces LSDA: transforming image classifiers into object detectors using minimal bounding-box annotations via domain adaptation.
- Key idea: learn a generic classification→detection transformation on a small set of annotated categories and apply it to categories without boxes.
- Method steps: (1) train CNN classifier on all categories with image-level labels, (2) fine-tune layers 1–7 on a small subset with bounding boxes (adding background class), (3) apply category-specific adaptation to extend detection.
- Empirical claims: 50% relative mAP improvement over a classification-only baseline on held-out categories.
- Demonstrated ability to produce a 7,604-category detector using classification data for most categories.
- Technical approach is sound, well-motivated, and logically extends R-CNN and domain adaptation ideas.
- Major missing baselines: (1) full-image positives with region negatives and fine-tuning all CNN layers, (2) MIL-based region selection baseline, (3) classifier-on-full-image using a single center/mean bounding box.
- Without those baselines, unclear if gains come from adaptation or from using CNN features.
- Evaluation limited to ImageNet’s 200 categories, so scalability claims to "tens of thousands" are not empirically demonstrated.
- Clarify experimental details: explicitly state which methods in Table 1 use Eq.1 and which CNN layers are adapted.
- Originality: novel adaptation strategy for extending detectors to unseen categories, but builds on existing CNN/R-CNN work.
- Limitation: method is tightly coupled to specific CNN architectures (e.g., Krizhevsky), potentially limiting generality.
- Significance: addresses an important practical problem and could reduce annotation costs, but impact would be stronger with broader evaluations.
- Overall: valuable contribution with compelling initial results, but requires the suggested baselines and more comprehensive evaluation to fully support scalability and effectiveness claims.
