- Proposes a framework to adapt image-classification DNNs for object detection, evaluated on ImageNet LSVRC-2013.  
- Achieves competitive performance (4th place) in the detection challenge.  
- Paper is clearly written and the proposed CNN architecture is straightforward to implement.  
- Code and models are publicly available, supporting reproducibility.  
- Addresses an important problem: scaling object detection with limited bounding-box annotations.  
- Core adaptation method lacks principled justification—why adapted detection weights are computed as W_c^j + δ_avg is unclear.  
- The physical interpretation of the offset parameter δ_avg and the rationale for the k-NN variant are not convincingly explained.  
- Comparison uses a portion of the validation set rather than the standard test set, risking unfair bias.  
- Small validation subset raises concerns about the statistical significance of reported results.  
- Authors’ footnote about R-CNN performance on the validation set does not fully resolve benchmarking concerns.  
- Improving theoretical justification and using standard test-set comparisons would strengthen the paper’s impact.
