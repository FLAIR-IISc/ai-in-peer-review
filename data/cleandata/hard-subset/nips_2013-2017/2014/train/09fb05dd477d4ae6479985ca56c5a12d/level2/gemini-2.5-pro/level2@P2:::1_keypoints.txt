- Problem: scaling object detection to many categories where bounding-box annotations are prohibitively expensive.  
- Proposal: LSDA — learn to transform an image classifier into an object detector via domain adaptation.  
- Core idea: learn the classifier→detector transformation on source categories with bbox+image labels and apply it to target categories with only image labels.  
- Implementation: builds on R-CNN/AlexNet — pretrain for classification then fine-tune with available data.  
- Two-stage adaptation (category-invariant): fine-tune shared layers (conv1–fc7) on detection data to learn localization and background separation.  
- Two-stage adaptation (category-specific): adapt final classification weights; for target classes approximate weight changes by averaging k-NN source deltas in classifier weight space.  
- Relation to prior work: extends R-CNN and offers an alternative to MIL-based weakly supervised methods by transferring from fully supervised categories.  
- Technical quality: sound, logical method with rigorous experiments on ILSVRC2013 and a clear ablation study (Fig.3/Table1).  
- Empirical result: ~50% relative mAP improvement over a strong classification-only baseline on held-out categories.  
- Error analysis: adaptation mainly reduces localization and background confusion errors — key differences between classification and detection.  
- Clarity: paper is exceptionally well-written, well-organized, and easy to follow with helpful figures and experimental details.  
- Originality: novel and insightful formulation of a transferable classifier-to-detector transformation for unseen categories.  
- Significance: high impact for large-scale detection; includes release of a 7,604-category detector and code.  
- Strengths: addresses an important practical problem, proposes an elegant solution, provides strong empirical validation, and is clearly presented.  
- Weaknesses/suggestions: k-NN averaging for category-specific adaptation is ad-hoc — consider learning a regression mapping; evaluate performance as the number of source categories is reduced; discuss integrating SVMs and bbox regression for held-out classes.
