

# Peer Review: LSDA: Large Scale Detection through Adaptation

## Summary
This paper proposes LSDA (Large Scale Detection through Adaptation), an innovative approach to scaling object detection to thousands of categories by leveraging abundant image-level classification data instead of expensive bounding box annotations. The authors frame the transformation from classification to detection as a domain adaptation problem, where the source domain is image classification data (with category labels) and the target domain is detection data (with bounding boxes and category labels). Their key insight is that a transformation learned from a subset of categories with bounding box annotations can be applied to adapt classifiers for categories without any bounding box annotations. This approach significantly outperforms baselines (50% relative mAP improvement on held-out categories) and enables a 7,600-category detector using only classification data for most categories.

The work builds upon R-CNN (Girshick et al., 2014), which demonstrated that fine-tuning image classifiers can produce detectors but requires bounding box data for all categories. LSDA differs fundamentally by transferring adaptation knowledge across categories and tasks, rather than requiring bounding box data for each category. The paper also relates to domain adaptation literature (e.g., Kulis et al., 2011; Hoffman et al., 2013) but extends it to simultaneously adapt both the feature representation and task (from classification to detection).

## Strengths

**Quality**: The paper presents a technically sound approach with rigorous experimental validation. The ablation study (Table 1) clearly demonstrates the contribution of each component (category invariant adaptation, category specific adaptation). The error analysis (Figure 5) provides valuable insights into the types of errors reduced by LSDA, showing significant improvement in localization and background rejection. The implementation details are thorough, and the authors make their models and code publicly available.

**Originality**: The core idea of using domain adaptation to transfer detection capabilities across categories is novel. While MIL-based methods (Ali et al., 2011; Song et al., 2014) have been used for weakly supervised detection, LSDA is the first to adapt to held-out categories with no detection data. The nearest neighbor approach for transferring adaptation parameters (Equation 1) is particularly clever and well-justified.

**Significance**: The paper addresses a critical bottleneck in scaling object detection to real-world applications. The ability to produce a 7,600-category detector using only classification data represents a major step toward practical large-scale detection systems. This could significantly impact applications where bounding box annotation is prohibitively expensive (e.g., medical imaging, satellite imagery, or web-scale content analysis).

**Clarity**: The paper is exceptionally well-written and organized. The problem statement is clear, the methodology is explained step by step, and the figures (Figures 1-6) effectively illustrate the key concepts. The authors provide sufficient implementation details for reproducibility, including the use of Caffe and specific dataset processing protocols.

## Weaknesses

**Limited Comparison with State-of-the-Art**: While the paper demonstrates 50% relative improvement over a classification-only baseline, it doesn't compare against other state-of-the-art detection methods (e.g., Fast R-CNN, Faster R-CNN) that might be applicable to the same setting. It would strengthen the significance claim to show how LSDA compares to these methods, especially since the paper mentions "oracle" performance as an upper bound.

**Nearest Neighbor Selection Justification**: The paper states they use k=10 nearest neighbors but doesn't thoroughly justify why this value was chosen over others. While they mention testing various k values, the paper doesn't provide a comprehensive analysis of how k affects performance, which would help readers understand the sensitivity of this parameter.

**Scalability Concerns**: The paper demonstrates success with 7,600 categories, but doesn't address potential scalability issues. For categories very dissimilar to those with bounding box annotations (e.g., distant leaves in the ImageNet tree), the nearest neighbor approach may not generalize well. The paper should discuss these limitations and potential solutions.

**Speedup Details**: The claim of "2fps for the 7.6K detector" is impressive, but the paper doesn't sufficiently explain how they achieved this speedup. While they mention using "fast region proposals" and spatial pyramid pooling, a brief technical explanation would help readers understand the practical implications.

## Recommendation

**Pro-acceptance arguments**: LSDA addresses a significant practical problem in object detection with a novel, well-supported technical approach. The 50% relative mAP improvement on held-out categories is impressive and demonstrates clear progress toward scalable detection. The paper is technically sound, well-written, and provides substantial empirical validation.

**Con-acceptance arguments**: The paper would benefit from more comprehensive comparisons with state-of-the-art detection methods and a deeper analysis of the nearest neighbor parameter k. The scalability concerns for dissimilar categories should be more thoroughly addressed.

## Final Recommendation

I recommend acceptance of this paper. LSDA makes a significant contribution to the field by enabling large-scale detection with minimal annotation effort, addressing a critical bottleneck in the practical application of object detection systems. While the paper has minor weaknesses in comparing against state-of-the-art methods and explaining the speedup details, these do not undermine the core contribution, which is novel, well-validated, and highly significant. The work has the potential to substantially impact both research and practical applications of object detection.
