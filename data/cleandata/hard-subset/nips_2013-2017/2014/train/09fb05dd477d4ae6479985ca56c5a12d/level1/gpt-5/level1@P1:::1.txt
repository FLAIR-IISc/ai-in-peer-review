Summary and relation to prior work:
This paper proposes LSDA (Large Scale Detection through Adaptation), a method to convert image classifiers into object detectors when bounding box annotations are scarce. The core idea is to learn a category-invariant adaptation from classification to detection using a subset of categories with detection labels (set B), and then transfer that knowledge to held-out categories (set A) that lack bounding box annotations. The adaptation has three components: (1) learning to model background (absent in classification), (2) fine-tuning shared CNN layers (1–7) to induce a generic classifier-to-detector transformation, and (3) a category-specific adjustment of the last layer by averaging the learned “delta” between classification and detection weights over nearest neighbors in set B, identified in the fc8 classification space.

Experiments on ILSVRC-2013 detection simulate the realistic regime where all 200 categories have classification labels but only the first 100 have detection labels. Starting from an AlexNet-like architecture pre-trained on ILSVRC classification, LSDA fine-tunes on detection data for set B (including a learned background class), applies the shared adaptation to all categories, and transfers per-class adjustments to set A via nearest neighbors. On the held-out 100 categories, mAP improves from 10.31% (using classification weights on proposals) to 15.85% with category-invariant adaptation, and to 16.15% with the category-specific transfer (k=10), a statistically significant +50% relative gain over the baseline. Error analysis indicates marked reductions in background confusion and localization errors. The authors also scale to a 7.6K-category detector by adapting 7,404 ImageNet leaf categories without boxes and discuss a faster variant using SPP and fast proposals to reach ~2 fps.

LSDA builds directly on R-CNN (Girshick et al., CVPR’14) by leveraging fine-tuning from classification to detection but removes the need for detection labels for all categories. It is also related to weakly supervised detection (MIL-based methods), domain adaptation (feature/model adaptation), and concurrent CNN-based detection efforts (OverFeat), but differs by explicitly learning a transferable classifier-to-detector transformation for held-out categories.

Strengths:
- Important and timely problem: scaling detection to thousands of categories without collecting bounding boxes for each.
- Clear and simple algorithm with a principled decomposition: background modeling, shared feature adaptation, and per-class last-layer transfer via nearest neighbors.
- Strong empirical support: a well-structured ablation showing where gains come from; statistically significant improvements on held-out categories; careful error analysis diagnosing reduction in BG/Loc errors.
- Scalability demonstrated: a 7.6K-category detector and a fast variant; public release of models/code increases impact and reproducibility.
- Relation to prior work is well-situated; the method is compatible with standard region proposal pipelines and widely used CNN backbones.

Weaknesses and detailed evaluation:
- Quality: While the methodology is sound and experimentally supported, several choices are heuristic and would benefit from deeper analysis:
  - The per-class transfer uses nearest neighbors in l2-normalized fc8 classification space; justification for this metric and sensitivity to k are limited. Alternatives (e.g., WordNet/semantic similarity, metric learning in embedding space, or features from earlier layers) could be explored.
  - Scoring as score_i − score_bg is somewhat ad hoc; calibration (e.g., Platt scaling or sigmoid/softmax calibration) or consistent probabilistic treatment could be considered. The authors note avoiding SVMs and bbox regression for efficiency and adaptability, but a comparison with light-weight calibration would strengthen claims.
  - The evaluation protocol reuses ILSVRC val1 for detection training for set B (augmented with training images) and val2 for testing, following R-CNN. While standard for the time, it differs from test server evaluation and may obscure how well the category-invariant adaptation generalizes more broadly.
  - No direct comparison to weakly supervised MIL methods on ImageNet detection (even on a subset) restricts positioning relative to that line of work, though the authors note this is complementary.
- Clarity: Overall clear, but several details could be expanded to aid reproducibility:
  - Hyperparameters and schedules for fine-tuning (learning rates per layer group, number of iterations, data balancing between BG and positives).
  - Detailed results for varying k (beyond a single reported choice) and alternate neighbor definitions.
  - Exact preprocessing and the effect of classification data noise (training set contains multiple objects per image but only one image-level label).
- Originality: The idea of adapting a classification network for detection via fine-tuning is known (R-CNN), but LSDA’s key novelty is learning a transferable adaptation to held-out categories and explicitly modeling/classifying background. The per-class delta transfer is simple yet new in this context. The work is an incremental but meaningful step bridging classification and detection at scale.
- Significance: High. Practitioners and researchers can leverage large classification corpora to bootstrap detectors across many categories. The 50% relative mAP gain on held-out categories and the 7.6K detector are impactful, and the code release increases adoption. Subsequent work can build on this to integrate better weak supervision, calibration, and joint multi-task training.

Suggestions for improvement:
- Evaluate alternative neighbor metrics for per-class transfer: semantic similarity (WordNet), visual similarity in intermediate feature spaces, or learned metric embeddings.
- Provide sensitivity plots for k and ablations on which layers are adapted; explore partial freezing strategies and multi-task joint training for classification and detection.
- Include a lightweight calibration baseline (temperature scaling or per-class bias tuning) and, if feasible, a comparison to a weakly supervised MIL method on a shared subset for context.
- Report per-class results and analyze which semantic families benefit most; explore long-tail behavior in the 7.6K setting.
- Discuss potential negative transfer and mechanisms to guard against it (e.g., selective transfer based on neighbor confidence).

Arguments for acceptance:
- Addresses a clear, important bottleneck (scarcity of bounding boxes) with a practical, scalable method.
- Solid empirical gains with careful ablation and error analysis; statistically significant improvements on a challenging benchmark.
- Broad impact via released models and demonstration of 7.6K-category detection and a faster inference pathway.

Arguments against acceptance:
- Some heuristic choices insufficiently justified; limited exploration of alternatives for per-class transfer and calibration.
- Missing comparisons to state-of-the-art weakly supervised detection methods on the same benchmark.
- Absolute performance remains well below fully supervised detectors; more analysis on where the gap remains would help.

Overall, this is a well-executed, impactful contribution that meaningfully advances large-scale object detection when bounding box supervision is limited. I recommend acceptance after minor clarifications and expanded analysis as suggested.
