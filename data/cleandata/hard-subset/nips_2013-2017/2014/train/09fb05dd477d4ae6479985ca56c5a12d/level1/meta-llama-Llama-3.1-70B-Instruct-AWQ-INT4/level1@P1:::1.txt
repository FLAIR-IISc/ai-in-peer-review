**Summary**

The paper "LSDA: Large Scale Detection through Adaptation" proposes a novel algorithm that enables the transformation of image classifiers into object detectors without requiring bounding box annotations for all categories. The authors cast the task as a domain adaptation problem, where the classification data is the source domain and the detection data is the target domain. The algorithm, LSDA, learns a generic transformation from the source domain to the target domain, which can be applied to any image classifier to adapt it into a detector.

**Relation to Previous Work**

The paper builds upon the work of Girshick et al. (R-CNN) [1], which demonstrated the importance of adaptation for transferring deep features from classification to detection. However, R-CNN requires bounding box annotations for all categories, whereas LSDA can adapt classifiers to detectors even without bounding box annotations. The paper also relates to Multiple Instance Learning (MIL) methods, which have been used for training detectors using weak labels.

**Strengths**

1. **Novelty**: The paper proposes a novel approach to object detection that can adapt classifiers to detectors without requiring bounding box annotations for all categories.
2. **Effectiveness**: The algorithm demonstrates significant improvement over a baseline of using raw classifier weights on object proposal regions, with a 50% relative mAP boost.
3. **Scalability**: The paper showcases the capabilities of the technique by producing a 7604 category detector, which is a significant improvement over existing object detection systems.

**Weaknesses**

1. **Assumptions**: The algorithm assumes that the classification data is sufficient to train strong classifiers, which may not always be the case.
2. **Limited Evaluation**: The paper only evaluates the algorithm on the ILSVRC2013 detection dataset, which may not be representative of all object detection tasks.
3. **Computational Cost**: The algorithm requires fine-tuning the network on the available labeled detection data, which can be computationally expensive.

**Originality**

The paper proposes a novel approach to object detection that combines domain adaptation and transfer learning. The algorithm is original in that it can adapt classifiers to detectors without requiring bounding box annotations for all categories.

**Significance**

The paper has significant implications for object detection, as it can enable the detection of tens of thousands of categories without requiring bounding box annotations. The algorithm can be used to improve the performance of existing object detection systems and can be applied to various applications, such as image classification, object recognition, and robotics.
