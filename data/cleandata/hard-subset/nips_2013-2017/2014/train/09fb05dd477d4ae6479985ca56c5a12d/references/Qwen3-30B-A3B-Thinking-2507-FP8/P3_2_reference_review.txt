

# Peer Review: LSDA: Large Scale Detection through Adaptation

## Summary

This paper proposes LSDA (Large Scale Detection through Adaptation), an algorithm that transforms image classifiers into object detectors without requiring bounding box annotations for all categories. The core insight is to treat the classifier-to-detector transformation as a domain adaptation problem, where image classification data (source domain) is adapted to detection data (target domain) using a small subset of categories with bounding box annotations. The authors propose two key adaptations: (1) a category-invariant transformation learned by fine-tuning shared convolutional layers using detection data for a subset of categories, and (2) a category-specific adaptation using nearest neighbors in the classification feature space to approximate the detector parameters for categories without bounding box annotations. The authors demonstrate significant improvements (50% relative mAP) on held-out categories in the ImageNet detection challenge, with the ability to produce a 7,604-category detector.

The work builds on R-CNN's insight that fine-tuning is important for transfer from classification to detection, but extends it by enabling adaptation to categories without any bounding box annotations. It also relates to domain adaptation literature, though the authors make a compelling case that their task involves both domain adaptation (between visual domains) and task adaptation (from classification to detection).

## Strengths

1. **Significant Problem Addressed**: The paper tackles a fundamental challenge in computer vision - the enormous annotation cost required for large-scale object detection. The authors convincingly argue that scaling detection to tens of thousands of categories is currently infeasible due to bounding box annotation requirements, making this a highly relevant problem.

2. **Technical Soundness**: The methodology is well-justified and technically sound. The category-invariant adaptation (fine-tuning shared layers) is a reasonable approach to learn a general transformation from classification features to detection features. The category-specific adaptation using nearest neighbors in the classification feature space is a clever and practical solution to the problem of adapting to held-out categories.

3. **Strong Experimental Results**: The ablation study in Section 4.2 provides clear evidence of the value of each component of LSDA. The 50% relative mAP improvement over the classification-only baseline is impressive, and the error analysis in Section 4.3 effectively demonstrates how LSDA reduces localization and background confusion errors.

4. **Practical Impact**: The authors successfully demonstrate the approach on a 7,604-category detector, showcasing the scalability of their method. The release of code and models at lsda.berkeleyvision.org is a significant contribution to the community.

5. **Clear Writing and Organization**: The paper is well-written, clearly organized, and provides sufficient detail for reproducibility (including implementation details with Caffe).

## Weaknesses and Concerns

1. **Lack of Comparison to State-of-the-Art Detection Methods**: This is the most significant weakness. The paper only compares to a very weak baseline (using raw classifier weights on object proposals), which is not a fair comparison to the current state of the art. The authors should compare their method to recent detection frameworks (Faster R-CNN, Mask R-CNN, YOLO, SSD) that use bounding box annotations for all categories. Without this, it's impossible to gauge whether LSDA can compete with current detection methods when using the same amount of bounding box data (only for 100 categories instead of 200).

2. **Questionable Baseline**: The baseline of "using raw classifier weights on object proposal regions" is extremely weak. It's unclear why this would be considered a reasonable baseline for a detection method, as R-CNN itself uses a similar approach (with SVMs on top of the features) but achieves much better performance. The authors should compare to a proper baseline like R-CNN with the same training data.

3. **Limited Analysis of Category-Specific Adaptation**: The nearest neighbor approach for category-specific adaptation lacks thorough analysis. The paper mentions they tried different k values (k=1, 10, |B|), but doesn't explain why the best k=10 was chosen or how this approach might fail for categories that are not well-represented in the set B. A more detailed analysis of when this approach works well and when it fails would be valuable.

4. **Scalability Concerns for Very Large Category Sets**: The paper demonstrates a 7,604-category detector, but doesn't address how the nearest neighbor approach scales to even larger category sets (e.g., 100,000+ categories). The nearest neighbor search could become computationally expensive, and the quality of approximation might degrade as the category set grows.

5. **Lack of Discussion on Modern Detection Architectures**: The paper uses the R-CNN pipeline (with selective search proposals and SVM classifiers), but doesn't explore how their adaptation technique would work with more recent detection frameworks (e.g., Faster R-CNN with region proposals generated by a neural network). This limits the applicability of their approach to current detection systems.

6. **Missing Error Analysis for Large-Scale Case**: The paper shows results for a 200-category detection task, but doesn't provide detailed error analysis for the 7,604-category detector, which is the main demonstration of their method's scalability.

## Specific Technical Comments

1. **Related Work**: The related work section is generally good but misses some key recent developments in weakly-supervised object detection. For example, the paper doesn't cite recent works like "Weakly Supervised Object Detection with Multiple Instance Learning" (2017) or "Object Detection with Weakly Supervised Learning" (2018) which have explored similar problems. Also, the paper should discuss how their approach relates to recent advances in unsupervised domain adaptation for detection.

2. **Methodology**: The authors claim that "fine-tuning transforms all network weights (except for the linear classifiers for set A)" - this is slightly misleading. In standard fine-tuning practice, all weights are updated, including the linear classifiers for set B. The weights for set A are not updated because there's no detection data for those categories, but the weights for set B are updated through fine-tuning.

3. **Experimental Setup**: The authors use a very small detection training set (5,000 images for 100 categories) with augmentation. It's unclear how much of the performance gain is due to the adaptation technique versus the small amount of detection data they used. A fairer comparison would be to train a standard detector on the same 5,000 images for 100 categories and compare to LSDA's performance on the same 100 categories.

4. **Performance Upper Bound**: The paper mentions an "upper bound" performance when detection data is available for all 200 categories (Table 1), but doesn't provide this value. It would be helpful to include this number to better understand the gap between LSDA's performance and the oracle performance.

## Overall Assessment

This paper addresses an important problem in computer vision with a technically sound approach that shows promising results. The core idea of adapting classifiers to detectors using a small subset of bounding box annotations is novel and valuable, and the experimental results on held-out categories are impressive.

However, the lack of comparison to state-of-the-art detection methods is a significant weakness that undermines the paper's claims about the effectiveness of LSDA. Without this comparison, it's impossible to determine whether LSDA is a viable alternative to current detection methods, or if it's simply a way to make a weak baseline slightly better.

The paper would be significantly strengthened by:
1. Adding comparisons to standard detection methods (R-CNN, Fast R-CNN, Faster R-CNN) trained on the same amount of bounding box data
2. Providing a more thorough analysis of the category-specific adaptation approach
3. Including a detailed error analysis for the large-scale 7,604-category detector

## Recommendation

I recommend **acceptance** with major revisions. The paper has significant potential impact and addresses an important problem, but requires substantial revisions to strengthen the experimental evaluation and comparisons. The authors should address the concerns raised above, particularly the comparison to state-of-the-art detection methods.

## Final Thoughts

This is a promising approach that could significantly reduce the annotation bottleneck in object detection. The authors have made a strong case for the importance of the problem and presented a technically sound solution. With the suggested revisions, this paper could make a valuable contribution to the field of object detection and domain adaptation. The authors' release of code and models is particularly commendable and will help the community build upon their work.

The main issue that needs to be addressed is the lack of fair comparison to current detection methods. Once this is resolved, the paper will have a strong case for acceptance.
