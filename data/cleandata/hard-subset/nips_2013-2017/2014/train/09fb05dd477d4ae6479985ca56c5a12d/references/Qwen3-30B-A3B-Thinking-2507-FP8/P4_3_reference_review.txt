

# Review of "LSDA: Large Scale Detection through Adaptation"

This paper proposes LSDA (Large Scale Detection through Adaptation), an algorithm that transforms image classifiers into object detectors using domain adaptation. The authors address the critical bottleneck in scaling object detection to large numbers of categories: the need for extensive bounding box annotations. While previous work like R-CNN [1] demonstrated how to adapt classifiers to detectors, it required bounding box data for all categories. LSDA instead uses bounding box annotations for only a small subset of categories and leverages classification data (image-level labels) for all categories to learn a transformation from classification to detection. The authors frame this as a domain adaptation problem where classification data (source domain) is transformed to detection data (target domain).

The paper relates well to prior work. It correctly positions itself relative to R-CNN [1], which requires bounding box data for all categories, and acknowledges MIL-based weakly supervised detection methods [6,7] but notes they haven't been evaluated on ImageNet. The authors also properly cite domain adaptation work [12-16] but clarify that their approach differs by adapting both from classification to detection tasks and from a large visual domain to a smaller one.

**Quality:** The paper is technically sound with a well-structured algorithm and thorough experimental validation. The three key insights (background recognition, category invariant information transfer, category-specific differences) are well-motivated. The authors provide clear ablation studies showing that category invariant adaptation (fine-tuning layers 1-7 with background learning) contributes most significantly to performance (15.85% mAP vs. 10.31% for classification-only baseline). The statistical significance (p=0.017) of improvements is properly reported. The implementation details are sufficient for reproducibility (Caffe, model definitions, and code released at lsda.berkeleyvision.org). The paper acknowledges limitations, noting the gap to oracle performance (when full bounding box data is available).

**Clarity:** The paper is well-written and organized. The algorithm description is clear, with Figure 2 providing a useful visual summary. The three-stage adaptation process (category invariant, category specific, detection) is logically presented. The error analysis (Figure 5) effectively demonstrates how LSDA reduces background confusion and localization errors compared to the classification-only baseline. The paper would benefit from slightly more detail on the nearest neighbor selection process for category-specific adaptation, but overall provides sufficient information for an expert reader to reproduce the results.

**Novelty:** The paper makes a clear contribution by enabling detection for categories without any bounding box annotations. The authors correctly identify this as novel, noting "to our knowledge, ours is the first method to adapt to held-out categories that have no detection data." While related to domain adaptation, the dual adaptation (task and domain) is distinctive. The paper properly references prior work on domain adaptation [12-16] and MIL [6,7] to establish novelty.

**Significance:** The significance is high. The paper demonstrates a 7604-category detector using only bounding box annotations for 200 categories (vs. 2000+ for full annotation). The 50% relative mAP improvement on held-out categories is substantial, and the method significantly reduces the annotation burden for large-scale detection. This addresses a major practical barrier in computer vision, potentially enabling detection for tens of thousands of categories with minimal additional annotation effort.

**Strengths:**
- Addresses a significant practical problem in scaling object detection
- Clear novelty in adapting to held-out categories without bounding box data
- Strong empirical results with thorough ablation studies
- Well-organized paper with clear explanations
- Code and model release for reproducibility
- Demonstrates practical application (7604-category detector)

**Weaknesses:**
- The paper acknowledges but doesn't directly compare with MIL-based methods [6,7] on ImageNet (though they note these methods haven't been evaluated on ImageNet)
- The category-specific adaptation using nearest neighbors (Equation 1) is somewhat heuristic without exploration of alternative similarity metrics
- The performance gap to oracle (full bounding box data) remains significant (16.15% vs. 24.32% mAP)
- The paper doesn't explore how the method scales to extremely large category sets (e.g., 100,000+ categories)

**Pro and Con for Acceptance:**
Pro: This work solves a major practical bottleneck in computer vision (annotation cost for large-scale detection) with a novel, well-supported approach. The 50% relative mAP improvement is substantial, and the 7604-category detector demonstrates real-world applicability. The paper is technically sound, well-written, and provides valuable resources (code, models) for the community.

Con: The paper doesn't directly compare with MIL-based approaches on ImageNet, which could be relevant to their claimed novelty. The category-specific adaptation method is somewhat heuristic, and the performance gap to oracle remains considerable.

Overall, the paper makes a significant contribution to the field by enabling large-scale detection with minimal bounding box annotations. The method is technically sound, well-validated, and addresses a pressing practical problem. The novelty is clear and significant compared to prior work, and the results demonstrate tangible improvements. The paper meets all NIPS quality criteria and would be of interest to a broad audience in computer vision.
