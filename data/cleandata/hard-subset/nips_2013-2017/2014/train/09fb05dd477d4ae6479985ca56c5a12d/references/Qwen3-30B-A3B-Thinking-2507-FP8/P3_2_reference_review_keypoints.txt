- Proposes LSDA: adapt image classifiers into object detectors using detection data for a subset of categories via domain/task adaptation.  
- Learns category-invariant transformation by fine-tuning shared convolutional layers with detection data.  
- Uses category-specific adaptation via nearest neighbors in classification feature space to approximate detector parameters for unlabeled categories.  
- Demonstrates large improvements (≈50% relative mAP) on held-out ImageNet categories and a scalable 7,604-category detector.  

- Addresses the high annotation cost for large-scale detection and the need for methods that scale without bounding boxes for all categories.  
- Methodology is technically sound: fine-tuning shared layers for domain shift and nearest-neighbor for category-specific adaptation are reasonable.  
- Strong experimental ablation showing value of components and error analysis reducing localization/background confusion.  
- Practical impact validated by a 7,604-category detector and released code/models.  
- Writing is clear and implementation details (Caffe) enable reproducibility.  

- Lacks comparison to state-of-the-art detectors (Faster R-CNN, Mask R-CNN, YOLO, SSD), making effectiveness relative to modern methods unclear.  
- Baseline (raw classifier weights on proposals) is weak; should compare to proper baselines like R-CNN trained on same data.  
- Insufficient analysis of the nearest-neighbor category-specific adaptation (choice of k, failure modes, representativeness of set B).  
- Scalability concerns for nearest-neighbor approach on much larger category sets (e.g., 100k categories) and potential computational/quality degradation.  
- Does not explore integration with modern detection architectures (region proposal networks, end-to-end frameworks).  
- Missing detailed error analysis for the 7,604-category large-scale results.  

- Related work section omits recent weakly/unsupervised detection and unsupervised domain adaptation literature.  
- Slightly misleading phrasing about which weights are fine-tuned; classifiers for set A are not updated due to no detection data.  
- Experimental setup uses a small detection set (5,000 images for 100 categories); unclear how much gains come from adaptation vs. this data.  
- Paper mentions an "upper bound" with full detection data but does not provide the value for comparison.  

- Overall: promising, technically sound approach with significant potential impact but experimental evaluation is currently insufficient.  
- Key required revisions: add comparisons to standard detection methods trained on same bounding-box data, deepen analysis of category-specific adaptation, and provide detailed error analysis for large-scale detector.  
- Recommendation: accept with major revisions.
