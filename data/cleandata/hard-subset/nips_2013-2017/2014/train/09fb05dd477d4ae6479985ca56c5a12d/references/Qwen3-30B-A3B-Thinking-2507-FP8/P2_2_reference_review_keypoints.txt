- Summary: Proposes LSDA to convert image classifiers into object detectors via domain adaptation, learning a transformation from a small set of categories with bounding boxes and applying it to others without boxes.
- Method steps: (1) pre-train CNN on classification data, (2) fine-tune layers 1–7 using detection data for subset B, (3) adapt category outputs for set A via nearest-neighbor techniques.
- Main result: Achieves ~50% relative mAP improvement on ImageNet held-out categories and demonstrates a 7604-category detector.
- Strength — Technical quality: Technically sound with clear algorithmic description, thorough ablation studies, and statistical significance (p=0.017); reduces localization and background errors.
- Strength — Clarity/reproducibility: Exceptionally well-written, informative figures, implementation details provided, and model weights released.
- Strength — Originality: Novel application of domain/task adaptation to scale detection from classification to detection across many categories.
- Strength — Significance: Addresses an important practical problem (scaling detectors with minimal bounding-box annotation) with potential real-world impact.
- Weakness — Limited validation: Experiments restricted to ImageNet; no evaluation on other detection datasets (e.g., PASCAL VOC) to show broader applicability.
- Weakness — Missing MIL comparison: No direct comparison to Multiple Instance Learning or other weakly supervised detection methods on the same benchmarks.
- Weakness — Efficiency details lacking: Claims large training time reduction (3 days to 5.5 hours) but omits technical details explaining how.
- Weakness — Simple category adaptation: Uses a nearest-neighbor approach for category-specific adaptation without exploring more advanced transformation-learning methods.
- Overall assessment: Strong, well-executed paper that makes a significant, practical contribution despite some experimental and comparative gaps.
- Recommendation: Accept.
