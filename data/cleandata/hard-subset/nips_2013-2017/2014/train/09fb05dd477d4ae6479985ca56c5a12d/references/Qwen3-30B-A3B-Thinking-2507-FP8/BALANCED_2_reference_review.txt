

# Review of "LSDA: Large Scale Detection through Adaptation"

This paper proposes LSDA, an algorithm for adapting image classifiers into object detectors without requiring bounding box annotations for all categories. The authors frame the problem as domain adaptation, where image classification data (abundant) serves as the source domain and detection data (scarce) serves as the target domain. The key insight is that a small set of categories with bounding box annotations can be used to learn a transformation that adapts classifiers for categories without any bounding box data into detectors.

**Strengths:**

1. **Significance:** The paper addresses a genuinely important challenge in computer vision - the prohibitive cost of collecting bounding box annotations for large-scale object detection (tens of thousands of categories). The authors demonstrate a 7,604 category detector, which is a compelling practical contribution showing the scalability of their approach.

2. **Technical Soundness:** The method is well-executed with a clear ablation study showing that each component contributes to the 50% relative mAP improvement over the classification-only baseline. The error analysis (Figure 5) effectively demonstrates how LSDA reduces localization errors and background confusion.

3. **Clarity:** The paper is well-written and clearly organized, with helpful figures that effectively illustrate the approach. The description of the three key insights (background recognition, category invariant information, category-specific differences) provides a strong conceptual foundation.

4. **Practical Impact:** The authors release both the 7,604 category model and code, making their work immediately useful to the community. Their implementation with fast region proposals (reducing detection time to half a second per image) shows practical viability.

**Weaknesses:**

1. **Novelty:** While the application to large-scale detection is compelling, the core idea builds heavily on existing approaches (R-CNN, domain adaptation). The authors acknowledge this but don't sufficiently distinguish their method from existing weakly supervised detection techniques like MIL-based approaches [6,7], which have been proposed for similar problems. The paper should directly compare with these methods to establish the novelty of their approach.

2. **Comparative Baselines:** The paper compares only with a "classification only" baseline, but fails to compare with state-of-the-art weakly supervised detection methods. This is a significant omission, as MIL-based approaches have been successfully applied to similar problems (e.g., [6,7]). Without this comparison, it's difficult to assess whether the 50% improvement is truly novel or simply a result of better implementation of existing ideas.

3. **Theoretical Foundation:** The paper lacks a strong theoretical justification for why the nearest neighbor approach for category-specific adaptation works. The authors should provide more insight into why category similarity in the classification space correlates with similarity in detection space, and how the choice of k affects performance.

4. **Implementation Details:** Some key implementation details are missing, such as how the nearest neighbor approach handles categories with no similar categories in set B (e.g., for rare categories), and the exact metric used for determining category similarity.

**Recommendation:**

The paper presents a practical and significant contribution to large-scale object detection. While the core idea builds on existing work, the authors demonstrate a compelling solution to a meaningful problem with strong experimental results. The paper would be significantly strengthened by:

1. Direct comparison with state-of-the-art weakly supervised detection methods (particularly MIL-based approaches)
2. More theoretical grounding for the category similarity assumption
3. Clarification of implementation details for the nearest neighbor approach
4. A more precise discussion of how LSDA differs from existing domain adaptation methods for detection

The paper's significance is high, but the novelty is somewhat limited. The authors have made a valuable contribution to the field by demonstrating how to scale detection to large numbers of categories, but they need to better position their work relative to existing weakly supervised detection methods.

**Pro/Con Acceptance:**

*Pro:*
- Addresses a meaningful, practical problem in computer vision
- Demonstrates significant scalability (7,604 category detector)
- Well-executed experiments with clear ablation study
- Strong practical impact with released code and models

*Con:*
- Limited novelty compared to existing weakly supervised detection methods
- Inadequate comparison with state-of-the-art baselines
- Insufficient theoretical justification for key components
- Some implementation details are unclear

Overall, this is a strong technical contribution with practical significance, but the paper would benefit from a more thorough comparison with existing weakly supervised detection methods to better establish its novelty. With these improvements, this paper would be a valuable addition to the NIPS proceedings. As it stands, I recommend borderline acceptance (weak accept), but the authors should address the comparative baseline issue before final acceptance.
