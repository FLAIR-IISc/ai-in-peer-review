

# Review of "LSDA: Large Scale Detection through Adaptation"

This paper proposes LSDA (Large Scale Detection through Adaptation), an algorithm that aims to transform image classifiers into object detectors without requiring bounding box annotations for all categories. The core idea is to learn a transformation from a classification network to a detection network using a small subset of categories with bounding box annotations (set B), then apply this transformation to adapt classifiers for categories without bounding box annotations (set A). The authors claim a 50% relative mAP improvement over a weak baseline that simply uses raw classifier weights on object proposal regions.

The paper builds on R-CNN [1], which showed that fine-tuning a classifier can produce a detector, but requires bounding box data for all categories. LSDA extends this by using a small subset of categories with bounding box data to learn a transformation that can be applied to categories without such data. The method involves three key components: category invariant adaptation (fine-tuning layers 1-7 using detection data for set B), category specific adaptation (using nearest neighbors to adapt output layers for set A), and detection with LSDA (combining per-category scores with background scores).

**Strengths:**
The paper is well-written, clearly organized, and provides a detailed description of the method. The problem of scaling object detection to many categories is important, and the authors correctly identify the challenge of obtaining bounding box annotations for large numbers of categories. The ablation study demonstrates that the main benefit of their approach comes from adapting the feature representation, which aligns with standard practices in object detection.

**Weaknesses:**

1. **Insufficient Baseline Comparison:** The claimed "50% relative mAP improvement" is against a very weak baseline (using raw classifier weights on object proposals). This is not a fair comparison, as the R-CNN paper itself showed that fine-tuning classifier weights on detection data provides a significant improvement over this baseline. The paper fails to compare against the standard R-CNN fine-tuning approach for the categories with bounding box data, which is the natural baseline for their method.

2. **Lack of State-of-the-Art Comparison:** The paper does not compare against current state-of-the-art detection methods (e.g., Faster R-CNN, Mask R-CNN, etc.). Without such comparisons, it's impossible to assess whether the claimed improvements are meaningful in the context of modern detection systems.

3. **Weak Validation of Key Innovation:** The "category specific adaptation" component (using nearest neighbors to adapt output layers for set A) is presented as a key innovation, but the paper provides no theoretical justification for why this approach should work. The experiments show k=10 is best, but don't explain why or if there's a principled way to choose k. This is a highly ad hoc step that lacks rigorous validation.

4. **Insufficient Demonstration of Scale:** The paper claims to enable detection of "tens of thousands of categories," but the only large-scale demonstration is a 7604-category detector with no meaningful performance metrics. The absolute mAP numbers on held-out categories (15.85%) are quite low compared to state-of-the-art detectors (which achieve over 50% mAP on PASCAL VOC), making it unclear how useful this method would be in practice.

5. **Misrepresentation of Contribution:** The paper claims to be "the first method to adapt to held-out categories that have no detection data," but this is not accurate. Multiple instance learning (MIL) approaches [6,7] have been used for training detectors from weak labels (image-level annotations), which is precisely the scenario the authors are addressing. The paper doesn't adequately differentiate its approach from these existing methods.

6. **Lack of Theoretical Foundation:** The paper presents an empirical approach without providing any theoretical justification for why the proposed transformations should work. This is particularly concerning given the paper's claim of being a "domain adaptation" method.

**Overall Assessment:**

While the problem addressed is important, the paper's contribution is not sufficiently novel or rigorously validated. The core idea of adapting classifiers to detectors is not new (R-CNN [1] already demonstrated this), and the proposed method appears to be a minor extension of existing techniques rather than a significant advance. The lack of comparison against state-of-the-art methods and the weak baseline make it difficult to assess the true significance of the claimed improvements.

The paper's claims about enabling detection of "tens of thousands of categories" are not substantiated with meaningful evidence. The absolute performance numbers are low, and the method doesn't address the key challenge of maintaining high detection quality at scale.

**Recommendation:**

I recommend rejection. The paper presents an interesting extension to existing object detection methods, but fails to provide sufficient evidence that it represents a meaningful advance over current state-of-the-art techniques. The validation is incomplete, the baseline is too weak, and the method's key innovation (nearest neighbor adaptation) lacks theoretical justification. Without rigorous comparison against the current state of the art and more compelling evidence of scalability, the contribution does not meet NIPS's high standards for methodological soundness and significance.
