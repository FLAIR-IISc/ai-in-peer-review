

# Review of "LSDA: Large Scale Detection through Adaptation"

## Summary
This paper proposes LSDA, an algorithm for adapting image classifiers to object detectors without requiring bounding box annotations for all categories. The authors treat the transformation from classification to detection as a domain adaptation problem, using a small subset of categories with bounding box annotations to learn a transformation that can be applied to categories without such annotations. The method achieves a 50% relative mAP improvement over using raw classifier weights on object proposal regions on the ImageNet detection dataset. The work builds on domain adaptation literature and extends the R-CNN framework by reducing the need for bounding box annotations.

## Strengths
1. **Significant Problem Addressed**: The paper correctly identifies the critical bottleneck in scaling object detection to large numbers of categories - the prohibitive cost of collecting bounding box annotations for tens of thousands of categories. This is a well-motivated and important problem in computer vision.

2. **Strong Experimental Results**: The ablation study provides clear evidence that the proposed components (category invariant adaptation and category specific adaptation) contribute to performance gains. The 50% relative mAP improvement over the baseline is compelling.

3. **Practical Impact**: The demonstration of a 7604-category detector showcases the potential of the approach for real-world applications where large-scale detection is needed. The release of the model and code enhances reproducibility.

4. **Clear Methodology**: The paper presents a well-structured approach with a logical progression from problem statement to methodology to evaluation.

## Weaknesses
1. **Limited Comparison with Related Methods**: The paper claims to be the first method to adapt to held-out categories without detection data, but doesn't sufficiently compare with weakly supervised detection methods like MIL-based approaches [6,7] on the same dataset. These methods could provide a more direct comparison point.

2. **Vague Explanation of Key Mechanism**: The "category invariant adaptation" mechanism is not sufficiently explained. The paper states that fine-tuning induces a "generic, category invariant transformation," but doesn't provide evidence for why this transformation generalizes to categories without bounding box data. The relationship between the background class learning and the category invariant transformation needs clarification.

3. **Lack of Analysis on Category Similarity Assumption**: The paper assumes that categories with similar classification features will have similar detection features. However, it doesn't analyze how this assumption holds across different visual domains or how the method performs when set A categories are visually dissimilar from set B categories.

4. **Insufficient Discussion of Limitations**: The paper doesn't thoroughly address the limitations of the approach. For example, how does the method perform when the categories in set A are very different from those in set B? What is the performance degradation when the number of categories in set B is reduced below a certain threshold?

5. **Missing Computational Analysis**: The paper claims a reduction in training time from 3 days to 5.5 hours, but doesn't provide a detailed analysis of computational complexity or scalability with respect to the number of categories.

## Recommendations for Improvement
1. Expand the related work section to include a thorough comparison with weakly supervised detection methods like MIL-based approaches [6,7], including results on the ImageNet dataset if available.

2. Provide more detailed analysis of the "category invariant" adaptation mechanism, including visualizations of feature transformations or statistical analysis of feature distributions across categories.

3. Conduct additional experiments to test the category similarity assumption - for example, by evaluating performance when set B and set A contain categories from different visual domains.

4. Add a section discussing limitations, including a quantitative analysis of performance degradation as the number of categories in set B decreases.

5. Include a computational complexity analysis comparing LSDA with other detection methods, particularly in terms of scalability to very large numbers of categories.

## Overall Assessment
The paper addresses an important problem in computer vision with a technically sound approach that shows significant improvements over the baseline. The results are promising and the method has clear potential for practical applications. However, the paper lacks sufficient comparison with related weakly supervised detection methods, the key mechanism of "category invariant adaptation" is not well-explained, and the limitations of the approach are not thoroughly discussed. 

The work is novel in its specific application of domain adaptation to the problem of scaling detection to large numbers of categories, but the novelty is somewhat incremental given the strong foundation in domain adaptation literature. The significance of the work is high due to the practical implications for large-scale detection systems.

**Decision: Weak Accept**

The paper has strong technical merit and addresses an important problem, but would benefit from the suggested improvements to strengthen the theoretical foundation, provide more thorough comparisons, and discuss limitations. With these revisions, the paper would be an excellent contribution to NeurIPS.
