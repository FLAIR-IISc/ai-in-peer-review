This paper presents a novel algorithm for learning the full kernel matrix of a determinantal point process (DPP) using an expectation-maximization (EM) approach. The authors demonstrate that their method outperforms the naive approach of maximizing likelihood by projected gradient ascent on the entries of the kernel matrix.

**Summary of Main Ideas**

The authors start by introducing DPPs as a probabilistic model of set diversity, compactly parameterized by a positive semi-definite kernel matrix. They note that learning the kernel matrix by maximizing the log-likelihood of the available data is a non-convex problem, conjectured to be NP-hard. Previous work has focused on restricted convex learning settings, such as learning a single weight for each row of the kernel matrix or learning weights for a linear combination of DPPs with fixed kernel matrices.

The authors propose a novel EM algorithm that changes the kernel parameterization from matrix entries to eigenvalues and eigenvectors, and then lower-bounds the likelihood using Jensen's inequality. They develop an efficient optimization procedure that alternates between an "expectation" step, which is concave in the auxiliary distribution, and a "maximization" step, which is concave in the eigenvalues.

**Strengths**

1. **Novel approach**: The authors propose a new EM algorithm for learning the full kernel matrix of a DPP, which is a significant contribution to the field.
2. **Efficient optimization**: The algorithm is designed to be efficient, with a runtime of O(T1nNk2 + T1T2N3), which is asymptotically faster than the naive approach.
3. **Robustness**: The EM algorithm is more robust to initialization and dataset changes, as demonstrated in the experiments.

**Weaknesses**

1. **Limited experiments**: The authors only present results on a single dataset (baby registries from Amazon.com), which may not be representative of other applications.
2. **Comparison to other methods**: The authors only compare their EM algorithm to the naive approach of maximizing likelihood by projected gradient ascent, and do not consider other existing methods for learning DPPs.
