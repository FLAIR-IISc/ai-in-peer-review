This paper proposes using an autoencoder with both deterministic and stochastic units to reconstruct faces of the same person under different 3D perspectives.

It is well-written, and the experiments are quite solid.

Figure 2 shows the model having some similarity to the model given in [25]. The main difference is that v is added and is estimated by both y and h3. It is useful to be able to use a feedforward estimate of v instead of using Bayes' rule. Discussing these differences in the text is probably a good idea.
