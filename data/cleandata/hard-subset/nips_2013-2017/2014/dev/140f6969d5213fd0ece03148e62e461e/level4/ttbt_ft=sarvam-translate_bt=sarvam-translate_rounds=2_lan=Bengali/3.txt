This paper discusses multi-view face recognition in a constrained environment. It proposes a multi-layer multi-view perceptron (MVP) network that uses deterministic and random neurons to estimate identity and view angle representation from a single 2D image. MVP is capable of creating new views of a face through a sequence of view representations from a single 2D face. This paper develops a Monte Carlo Expectation Maximization (MCEM) method for learning MVP from a pair of face and view labels, assuming that the output face follows a conditional diagonal Gaussian distribution. MVP was learned from 200 individuals in a multi-PIE dataset and used identity features to classify 137 individuals reset by LDA, achieving performance similar to recent methods using convolutional neural networks.

This paper proposes a new MVP network to separate face recognition and view angle representation and uses identity features in LDA for multi-view face recognition. New views of a face can be created by changing the view angle feature in MVP. Overall, the technical approach is good with some limitations, and its performance in multi-PIE is very competitive. Some technical concerns are listed below:

This method seems to rely heavily on well-aligned normalized 32x32 face images. Experiments on face recognition in multi-PIE with 137 individuals raise questions about the scalability of the proposed method. It is unclear whether the method can benefit from more training samples and how the method can be extended to real-world applications for seamless face recognition.

How important is it to estimate the conditional diagonal Gaussian distribution for the output y? How many training pairs of 200 individuals are used to learn MVP in multi-PIE? Why does the new face view have to be "sequentially" synthesized?

The claim that this new MVP "simulates how the human brain encodes view representations" is quite weak. This paper does not present any credible evidence or discussion about how the human brain encodes view representations and why this process is similar to MVP. Therefore, I will not list it as a key contribution of the paper.

This is a good work on learning multi-view perceptron (MVP) for multi-view face recognition. Its performance in multi-PIE is competitive. Although I have concerns about the scalability of this method and how it can be applied to a face recognition system.
