In this paper, the authors discuss discriminatory clustering from a different perspective. Their main idea is that they imagine a classifier that separates different (cluster) classes and they set a threshold for the generalization error created by this classifier. They try to optimize this threshold using a clustering algorithm (in this case, a belief-propagation-based method).

The main feature that distinguishes this work from previous work is that they have recreated the cost function as a new type of similarity measure that uses a kernel density estimate to avoid parameter search problems. This does not mean that their formulation does not have parameters: however, these are either balance parameters or a variance parameter for the kernel density function that they estimate.

Overall, I think this is a reasonable idea and has some advantages in terms of reducing the number of parameters for discriminatory clustering. It gives a good idea of what the clustering algorithm is trying to optimize.

In reality, while evaluating the effectiveness of this idea, this paper is somewhat weak. The main comparison is with a different example-based clustering method. I find this a bit odd, because one of the selling points of this paper is that it requires fewer parameters than other discriminatory clustering methods. Would it have been more useful to compare it with MM clustering or even information-theoretical methods? The space spent on proving the consistency of KDE could be used for such comparisons, as the consistency results directly follow the familiar results.

I couldn't read this paper very easily, mainly due to the use of excessive notation to explain the main concepts. It could have benefited from a final "here is our algorithm" explanation (somewhere) that would have consolidated everything. Although I understand how the parts fit together, I spent some time trying to figure out the different parameters (including the variance H) and how they are used in the overall algorithm.

A reasonable idea to consider. However, more thorough experimental evaluation is needed.
