In this paper, the authors have taken a different approach to discriminative clustering. Their central idea is as follows: they imagine a classifier that separates different (cluster) classes, and derive a threshold for the generalization error induced by this classifier. This is the threshold they then try to optimize using a clustering algorithm (in this case, a belief-propagation-based method).

The main feature that distinguishes this work from previous work is that they reform the cost function into a new type of similarity measure that uses a kernel density estimate, in order to avoid the parameter search problem. This does not mean that their formulation does not contain parameters: but these are either equilibrium parameters or a variance parameter for the kernel density function that they estimate.

Overall, I think this is a reasonable idea, and it has some merit in terms of reducing the number of parameters for discriminative clustering. It also provides a better story about what the clustering algorithm is actually trying to optimize.

In terms of the merit of this idea in practice, this paper is a bit weak. The main comparison is with a different example-based clustering approach. This seems a bit odd to me, because one of the paper's alleged selling points is that it requires fewer parameters than other discriminative clustering methods. Would it be more useful to compare with MM clustering or even information-theoretic approaches? The space spent proving the stability of KDE could have been used for such a comparison, as the stability results directly match the known results.

I didn't find this paper very easy to read, mainly due to the abundance of notation used to explain the main ideas. It would have benefited (somewhere) from a final "here's our algorithm" explanation that puts everything together. While I understand how the pieces fit together, I spent some time tracking the different parameters (including the variance H) and how they are being used in the overall algorithm.

A reasonable idea that is worth considering. But more in-depth experimental evaluation is needed.
