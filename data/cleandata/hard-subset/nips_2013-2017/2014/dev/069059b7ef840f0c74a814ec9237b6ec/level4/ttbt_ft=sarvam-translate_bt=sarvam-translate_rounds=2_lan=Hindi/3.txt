In this paper, the authors have adopted a different approach to differential clustering. Their central idea is as follows: they imagine a classifier that separates different (cluster) classes, and derive a threshold for the generalization error induced by this classifier. This is the threshold they then try to optimize using a clustering algorithm (in this case, a belief-propagation-based method).

The main feature that distinguishes this work from previous work is that, to avoid the parameter search problem, they improve the cost function in a new type of similarity measure that uses a kernel density estimate. This does not mean that their formulation does not contain parameters: but these are either equilibrium parameters or a variance parameter for the kernel density function that they estimate.

Overall, I think this is a reasonable idea, and it has some merit in terms of reducing the number of parameters for differential clustering. It also provides a better story about what the clustering algorithm is actually trying to optimize.

In terms of the merits of this idea in practice, the paper is a bit weak. The main comparison is with a different example-based clustering approach. This seems a bit odd to me, because one of the paper's alleged selling points is that it requires fewer parameters than other differential clustering methods. Would it be more useful to compare with MM clustering or even information-theoretical approaches? The space spent on proving the stability of KDE could have been used for such a comparison, as the stability results directly match the known results.

I didn't find this paper very easy to read, mainly due to the abundance of notation used to explain the main ideas. It would have benefited from a final "here's our algorithm" explanation (somewhere) that puts everything together. While I understand how the pieces fit together, I spent some time tracking the various parameters (including the variance H) and how they are being used in the overall algorithm.

A reasonable idea that is worth considering. But more in-depth experimental evaluation is needed.
