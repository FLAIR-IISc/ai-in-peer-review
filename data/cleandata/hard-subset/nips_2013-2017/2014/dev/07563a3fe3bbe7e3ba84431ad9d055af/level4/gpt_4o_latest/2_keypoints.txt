- Proposes discriminative unsupervised feature learning using seed patches as surrogate classes with transformed examples to train a convolutional network.  
- After training, extracts pooled feature maps from each layer for downstream classification.  
- Evaluated on multiple standard image classification benchmarks and shows strong performance across tasks.  
- Analyses on the impact of transformations and invariance properties are insightful.  
- Paper is well-written and provides sufficient detail for reproducibility.  
- Request clarification on the patch extraction procedure (sampling strategy, gradient computation, top-N selection).  
- Suggest explicitly noting that STL-10 images are 96×96 to justify the 32×32 patch size.  
- Ask for intuition or justification for choosing 32×32 patches and guidance on selecting patch size relative to image size.  
- Recommend adding diagrams: one for surrogate-class training and one for the feature-extraction pipeline to aid less-expert readers.  
- Advise including a common baseline: features from an ImageNet-pretrained convnet plus linear classifier in Table 1 for comparison.  
- Ask how well surrogate-task validation correlates with downstream classification validation and suggest reporting both scores to show that relationship.  
- Note a discrepancy: Supplementary Table 2’s diagonal differs from Table 1 for the same network—request explanation.  
- Point out prior best STL-10 result (70.1% ± 0.6% from Swersky et al., NIPS 2013) and suggest a strong baseline: k-means feature maps + convnet with the paper’s transformations.  
- Overall assessment: simple, effective method with promising generality; recommends acceptance.
