- Proposes a simple discriminative unsupervised feature-learning method using seed patches as surrogate classes, transformations to generate intra-class examples, and a convnet trained to predict seed classes; features extracted by pooling conv feature maps.

- Demonstrates strong performance on standard image classification benchmarks and provides useful analysis on the role of transformations and invariance.

- Paper is generally clearly written and reproducible.

- Request for more detail on patch extraction methodology (e.g., are patches sampled, gradients computed, and top-N chosen?).

- Suggest mentioning STL-10’s 96x96 image size earlier to justify the 32x32 patch choice.

- Question whether 32x32 patch size was arbitrary and ask for intuition on selecting patch size relative to image size.

- Recommend adding illustrative figures showing surrogate-class convnet training and the feature-extraction pipeline for clarity.

- Suggest including results using ImageNet-pretrained convnet features + linear classifier (e.g., DeCAF) in Table 1 for practical comparison.

- Ask whether surrogate-task validation performance correlates with downstream classification validation, and recommend reporting both surrogate and classification validation scores.

- Point out an inconsistency: diagonal in supplementary Table 2 differs from results in main Table 1 and needs explanation.

- Recommend trying a strong baseline (k-means feature maps plus convnet with the paper’s transformations) as in Swersky et al., and note a published STL-10 best result (70.1%) for comparison.

- Overall evaluation: nice, simple approach with good results and interesting directions; reviewer recommends acceptance.
