- Proposes Exemplar-CNN: unsupervised discriminative learning using surrogate classes from transformed random patches to learn invariant image features.
- Leverages data augmentations (translation, rotation, scaling, color shifts) and trains a CNN to discriminate surrogate classes.
- Provides a formal analysis showing the objective enforces both discrimination and invariance.
- Achieves state-of-the-art or competitive results on STL-10, CIFAR-10, and Caltech-101, notably 72.8% on STL-10.
- Novelty: introduces a discriminative unsupervised objective distinct from generative approaches with rigorous theoretical insights.
- Empirical strength: strong experimental performance and significant gains over prior unsupervised methods.
- Thorough experimentation: analyzes effects of number of surrogate classes, transformations, and network size.
- Clear presentation: well-written and well-organized paper.
- Scalability concern: does not scale well to larger datasets because many surrogate classes can force similar patches apart.
- Transfer learning gap: no experiments on using the same surrogate classes across related datasets without retraining.
- Missing analysis of patch-size variability: lacks experiments varying patch sizes to test feature robustness.
- Limited invariance: struggles with complex variations like 3D viewpoint changes and inter-instance variability.
- Suggestions: test shared surrogate classes across datasets, include patch-size experiments, and explore scalability fixes (e.g., merging classes or weak supervision).
- Recommendation: accept with minor revisions to address scalability and transferability issues.
